# 其中机械键盘团体发现了电动线性电位计

> 原文：<https://hackaday.com/2017/04/26/wherein-the-mechanical-keyboard-community-discovers-motorized-linear-potentiometers/>

在 Digikey 和 Mouser 数据库的深处，你会发现有史以来最酷的组件。电动线性电位计是一种罕见的、奇特的、非常简洁的输入设备，最常见于巨大的混音板和其他价格相当于一辆汽车的设备上。它们是滑动电位计，有一个锦囊妙计:内部有一个电机，可以将滑块设置到任何位置。

过去几年，机械键盘社区一直在推动输入设备的边界，看起来他们刚刚发现了电动线性锅。[Jack] [为他的键盘](https://imgur.com/gallery/2z31d)创造了一个电动滑动键帽。它就像一个滚轮，但对键盘来说。它很漂亮，很实用，很棒。

这个构建的硬件正是您所期望的。侧面安装的 60 毫米电动线性电位计，或安装在键盘顶部的 100 毫米电动线性电位计，由 Arduino 克隆和小型电机驱动器控制。那只是硬件；这里真正的诀窍是软件。到目前为止，[Jack]已经实现了一个插件系统、配置软件和力反馈。现在，在任何 Adobe 产品中改变时间线都是简单而直观的。这个设备也有一个“不太振动”模式，每当[杰克]在他的桌面上收到通知。

现在，[Jack]正在 reddit 帖子中为这个[进行团购，费用在 55 美元到 75 美元之间，取决于有多少人想要。这是一款非常棒的产品，我们迫不及待地想看到 Corsair 推出一款带有无数 RGB LEDs 的版本。在那之前，我们只能对[杰克]贴在下面的视频垂涎三尺。](https://www.reddit.com/r/MechanicalKeyboards/comments/66zuk8/photos_update_and_group_buy_for_my_mechanical/)
