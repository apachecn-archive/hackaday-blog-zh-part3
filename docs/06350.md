# Hackaday 奖参赛作品:健康监测柔性智能手表

> 原文：<https://hackaday.com/2017/07/15/hackaday-prize-entry-health-monitoring-flexible-smartwatch/>

[Nick Ames]的[柔性智能手表](https://hackaday.io/project/21581-flexible-smartwatch)项目旨在创造一种开源智能手表，由柔性电容电子墨水触摸屏制成，使用表带的整个表面。这款环绕式智能手表显示来自板载脉搏和血氧传感器以及加速度计和磁力计的信息，让你清楚地知道你对即将到来的会议有多紧张。

尼克配备的显示器被称为电泳显示器(EPD)。它是 400×200 像素，115ppi，对角线 4 英寸，可以绕手腕弯曲。它可以用 16 种灰度绘制图形，刷新时间不到一秒，或者用更快的速度绘制 B&W 图形。

[Nick]的项目中描述的智能手表将有 2.5 毫米厚——肯定薄到可以塞进袖子里。我们怀疑这种外形的成功可能取决于[Nick]成功地使它看起来不像医院腕带。虽然这让我们想到生物反馈感应智能腕带可能是住院的未来。
