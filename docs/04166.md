# 泰格磨抹滚珠丝杠(终于！)

> 原文:[https://hack aday . com/2016/11/16/taig-mill-nouncted-with-ball-screw-at-last/](https://hackaday.com/2016/11/16/taig-mill-anointed-with-ball-screws-at-last/)

是的，当我们说话的时候，我们可以听到一群“非黑客”正在给他们的大炮装弹药，但是这台机器在社区里有一个特殊的位置。多年来，Taig 铣床一直是轻型家庭机械商店的首选微型铣床。这些机器往往会按照更高的标准进行装饰和切割，这可能是因为拥有这些工具的社区往往喜欢为加工而加工——或者可能是因为研磨机的每个部件都可以在网上作为替换零件获得。对许多人来说，这台机器是在家制造芯片的起点。(事实上，Other Machine Co .的首席技术官迈克·埃斯特(Mike Estee)是用 Taig 开始他的机械加工冒险的。)

多年来，泰格公司出售的机器带有一个丝杠和一个黄铜螺母，可以拉紧以减少反冲。然而，对于最挑剔的机械师来说，反冲仍然是一个问题；所以，终于， [Taig 发布了一个无反冲滚珠丝杠变体](https://www.youtube.com/watch?v=uaI4yiNW_OA)的两个版本:一个预装滚珠丝杠的一体机和一个为已经用丝杠模型装饰车库的客户提供的升级套件。

在下面的剪辑中，[John]带我们参观了将 3，12 毫米滚珠丝杠塞入原始拓扑结构所面临的挑战。正如我们所料，几块漂亮的金属被切掉，为稍大的滚珠螺母腾出了空间。尽管进行了删减，但这一构建足够整洁，足以让我们误以为滚珠丝杠从一开始就符合原始设计。

困惑为什么滚珠丝杠是丝杠的巨大飞跃？借你的眼睛和耳朵一点时间来了解一下[[Al]对这个问题的概述](http://hackaday.com/2016/11/02/mastering-ball-screws/)。
