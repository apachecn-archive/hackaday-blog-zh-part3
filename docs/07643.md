# 我在你的未来看到了雨

> 原文：<https://hackaday.com/2017/11/10/i-see-rain-in-your-future/>

谁不想拥有一个水晶球呢？不幸的是，我们的计算机不太擅长预测未来。然而，他们偶尔也会得到正确的天气预报，所以[珍妮·哈内尔]做了一个水晶球来显示天气预报。她称之为“Sphaera ”,你可以在下面看到它的运行视频。

用户界面很有趣，依靠 5 个光敏电阻。当你盖住其中一个时，里面的 Raspberry Pi 会检测到，并将其解释为一个命令。一块塑料允许从 LCD 显示器投影到球体内部。[珍妮]称之为全息图，虽然从技术上讲，它不是真正的全息图，当然。

我们希望球体是独立的，但是电子设备在下面的盒子里。然而，我们不得不怀疑，你是否可以用一些电池和无线充电装置密封一个 Pi Zero，使整个事情成为一个真正的水晶球。说到材料，[Jenny]提到了 cellplast。据我们所知，这就是我们所说的泡沫。在这个应用中，你可以使用从聚苯乙烯泡沫塑料到花卉泡沫塑料到泡沫塑料的几乎任何东西。

同时，天气数据来自 OpenWeatherMap 的 API，当然，用另一个服务替换它应该很简单。

我们很乐意看到有人将这个项目与我们最喜欢的驼鹿合并。随着 Kinect 的过时，我们认为[使用这个水晶球](https://hackaday.com/2011/06/04/two-kinnects-plus-one-hd-projector-makes-the-coolest-snowglobe-ever/)为时已晚。
