# RasPi LED 面板库是 Nyan-tastic

> 原文:[https://hack aday . com/2015/11/29/raspi-led-panel-library-is-nyan-tastic/](https://hackaday.com/2015/11/29/raspi-led-panel-library-is-nyan-tastic/)

快速–必要时，让我们拥有一个巨大的 RGB LED 矩阵！虽然听起来很神奇，但人们很容易忘记，选择正确的部件、调试驱动程序和整理我们的意大利面条式布线是一场需要赢得的战斗。请放心，[Hzeller]已经用 Raspberry Pi RGB LED 矩阵实现为我们完成了所有繁重的工作，该实现可扩展到多个面板，并可在迄今为止的任何 Pi 模型上运行！

提供大约 100 Hz 的 24 位颜色，总共 36 个面板，[Hzeller]库毫不逊色。该库可以定制你的面板排列，一个独立的项目(也是赫泽勒的作品)使这种设置与作为网络设备的 T2 像素推进协议兼容。

的确，许多美国的、T2 的、T4 的、T6 的都是为了这些展示，所以你可能会问:“我们以前见过这个吗？有什么大惊小怪的？”像其他人一样，最终的产品也是引人注目的，但是[hzeller]和他的实现仍然很强大，因为他回答了这个问题:*如何实现？事实上，更令人印象深刻的是他的全面的在线文档。在里面，[hzeller]详细介绍了各种硬件配置，用于定制数量的面板或驱动它们的特定风格的 Pi。他还提供了关于引脚排列的参考，并提供了开箱即用的软件演示，以确保任何人都可以将这个项目付诸实施。如果一个写得很差或者根本不存在的读物让你回避构建一个开源项目，不用担心。从引出线怪癖和开箱即用的软件演示，[hzeller]已经涵盖了所有的基础，并给了我们一个项目，人们的所有水平的黑客。*

也许这个项目最好的部分是观众的跨度，可以从它身上带走一些东西。如果您是一个经验丰富的 Linux 爱好者，可以深入源代码，了解一下[hzeller]如何在 Raspi-2 配置中将这个项目推进到一个单核上。如果你是数字电子的新手，让这个项目成为你拿起一个 Pi、一个面板(或四个)并运行的时刻，要知道[hzeller 的] README 是你照亮夜晚所需的唯一一本书。

在今年的[超导](http://hackaday.com/2015/11/16/hackaday-supercon-marks-the-dawn-of-something-new/)展会上，我们有幸通过现场演示欣赏了一些 Nyan-Cat 彩虹。
