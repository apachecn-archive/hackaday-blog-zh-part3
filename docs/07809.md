# Modder 将计算机放入电源中

> 原文:[https://hack aday . com/2017/11/24/mod der-puts-computer-inside-a-power-supply/](https://hackaday.com/2017/11/24/modder-puts-computer-inside-a-power-supply/)

当建立一个定制的电脑钻机，大多数人把 SMPS 电源放在电脑机箱内。[James]又名[Aibohphobia]又名[fearofpalindromes]将它翻了个底朝天，造出了[stx 160.0——一台完全成熟的游戏电脑，塞在 ATX 的一个电源外壳里](https://smallformfactor.net/forum/threads/stx160-0-the-most-powerful-atx-unit-in-the-world.934/)。虽然小外形(SFF)计算机并不新鲜，但他的构建在一个小机箱中提供了强大的冲击力，是计算机改造、黑客独创性和工程学的一个很好的例子。成品电脑使用迷你 ITX 外形主板，配有英特尔 i5 6500T 四核 2.2GHz 处理器、EVGA GTX 1060 SC 显卡、16GB DDR4 RAM、250GB SSD、WiFi 卡和两个 USB 端口，所有这些都由 160 W AC-DC 转换器供电。它的外部尺寸与 ATX-EPS 电源相同，为 150 L x 86H x 230D mm。stx 160.0 由市电供电，而不是来自外部砖块,[James]认为这是作弊。

对于那些想快速，TL；《图片评论博士》首先浏览他在 Imgur 上的[相册，尽情欣赏完成的电脑及其内部的图片。但是魔鬼就在细节中，所以请查看论坛帖子，获取大量有趣的构建信息、组件来源、技巧和琐事。例如，为了将图形卡连接到主板上，他使用了一个“M.2 to powered PCIe x4 适配器”和一个来自一家名为](https://imgur.com/a/A2JP6) [Adex Electronics](http://www.adexelec.com/adexhm.htm) 的古怪公司的柔性电缆扩展器，这家公司仍然喜欢以传统方式做生意，其网站可能会让你想起 Netscape Navigator 占主导地位的浏览器时代。

作为一项基准测试，[James]发布了“在盖板打开的情况下，在满负荷(Prime95 Blend @ 2 线程和 FurMark 1080p 4x AA)下，CPU 风扇以 1700RPM 的速度运转时，CPU 温度约为 65℃，GPU 在 48%的风扇速度下，温度为 64℃。”乍一看，这是一个相当令人印象深刻的电源。

在这个项目中，我们真正感兴趣的两件事是他的细致研究，从大量可用的选择中找到符合他要求的具体零件。第二个是他关于为这个项目设计定制外壳的极其详细的笔记，并使其对 DFM(制造设计)友好，以便可以大规模生产——只要看看他的“[目录](https://smallformfactor.net/forum/threads/stx160-0-the-most-powerful-atx-unit-in-the-world.934/#post-17952)就知道他覆盖的范围有多大了。如果你对定制构建和计算机改造感兴趣，有大量有用的信息嵌入其中。
