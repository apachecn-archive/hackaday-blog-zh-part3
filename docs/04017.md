# 钎焊铝

> 原文：<https://hackaday.com/2016/11/04/brazing-aluminum/>

你对金属加工中的一个永恒问题:铜焊还是焊接持什么态度？作为你的黑客作家，一个铁匠的女儿，这是非常焊接的一面。钎焊接头太容易分离，这就是为什么在该地区，这是写在至少，他们不允许每年的车辆道路性能测试。如果你曾经不得不用角磨机去除铜焊的补丁，你会知道在危机中你该相信哪一个。

如果有问题的金属是铝呢？[乔治·格雷夫斯]给我们发来一个链接，链接到几年前关于这个主题的一个论坛讨论，还有一个链接到[一个 YouTube 视频](https://www.youtube.com/watch?v=jtvOkI_pvMI)，我们已经把它嵌入到休息时间下面了。奇迹铜焊棒声称惊人的韧性，但世界分成了赞成 TIG 强度的人和指出铜焊渗透到被连接金属表面之间的人。经过一段时间的试验，我们承认铝钎焊棒确实能很好地连接断裂的零件。但是，你不会再看到这个黑客作家骑着一辆没有用可靠的 TIG 火炬焊接的自行车。

看一下视频，看看你有什么想法。即使它不是一个你会赌上你的生命的关节，它仍然是一种技术，是你车间武器库的一个有用的补充。
