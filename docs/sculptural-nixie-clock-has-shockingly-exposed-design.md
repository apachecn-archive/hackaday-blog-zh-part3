# 雕塑谢妮时钟有令人震惊的暴露设计

> 原文：<https://hackaday.com/2017/01/17/sculptural-nixie-clock-has-shockingly-exposed-design/>

单管谢妮钟？去过那里，见过那个。带有暴露危险电压的雕刻布线的单管谢妮钟？这可不是你每天都能看到的。

[Andrew Moser]的时钟显然是一个麻醉美学的案例——他是在手术后，在止痛药的影响下建造的。这可能解释了有问题的判断，但我们不会与外观争论。用于谢妮的升压转换器位于弯曲线框架的底部附近，ATmega 328 和 DS1307 RTC 由连接的无源元件和跳线的引线支撑在中部。框架顶部的一个环支撑着谢妮的八进制插座和每个元件的驱动晶体管。

在休息后的视频中，[Andrew]谈到在 PCB 上重建这个。虽然我们以前见过[单管谢妮 PCB 时钟](https://hackaday.com/2014/12/13/simple-and-elegant-single-digit-nixie-tube-clock/)，我们也同意设计需要更加安全，但我们不会放弃这种死 bug 风格。也许只是把整个东西扔在玻璃钟罩或丙烯酸管里。
