# 欧姆？别忘了基尔霍夫！

> 原文:[https://hackaday.com/2017/05/25/ohm-dont-forget-kirchhoff/](https://hackaday.com/2017/05/25/ohm-dont-forget-kirchhoff/)

不知道欧姆定律，就很难深入研究电子学。以[Georg Ohm]命名，它描述了线性电路中的电流和电压关系。然而，有两条更基本的定律没有得到欧姆定律那样的尊重。这些是基尔霍夫定律。

简单来说，基尔霍夫定律实际上是能量守恒的一种表达。基尔霍夫电流定律(KCL)表示，进入单个点(节点)的电流必须有完全相同的电流流出。如果你更数学化，你可以说流入和流出的电流之和将永远为零，因为流出的电流与流入的电流相比将具有负号。

你知道串联电路中的电流总是相同的，对吧？例如，在包含电池、LED 和电阻的电路中，LED 和电阻中的电流相同。那是 KCL。流入电阻的电流最好与流出电阻并流入 LED 的电流相同。

当有两条以上的导线进入一个点时，这是非常有趣的。例如，如果一个电池驱动 3 个神奇一样的灯泡，那么每个灯泡将获得总电流的三分之一。电池导线与 3 个灯泡的导线连接的节点就是节点。所有流入的电流，必须等于所有流出的电流。即使灯泡不一样，总数仍然是相等的。所以如果你知道任何三个值，你可以计算第四个。

如果你想自己玩，可以[模拟](http://tinyurl.com/mprjh3e)以下的电路。

![](../Images/f54ca1b28c36a05dd5f4e4d6f7e2e9ed.png)

来自电池的电流必须等于流入电池的电流。最左侧和最右侧的两个电阻流过相同的电流(1.56 mA)。在模拟器的舍入误差范围内，分流的每个分支都有自己的份额(注意，底部分支具有 3K 总电阻，因此携带的电流较少)。

基尔霍夫电压定律(KVL)说，回路周围的电压总和必须为零。举个简单的例子。12V 电池上有一个 12V 灯泡。灯泡两端的电压是多少？12V。如果有两个相同的灯泡，他们仍然会看到每个灯泡上的 12V 电压。

可以[模拟这个电路](http://tinyurl.com/nxlvuk2)看看效果。有两个灯泡的回路有 12V 电压，每个灯泡得到一半，因为它们是相同的。右边的路径有不同的电压，但它们的总和仍然是 12。

![](../Images/aefc4c02c2122579f4ac5fef4b7d34af.png)

KVL 本身不会很有用，但是有一个叫做叠加的原理。这是一种奇特的说法，你可以把一个复杂的电路分解成几个部分，观察每一部分，然后把结果加回去，得到正确的答案。

## 分析

无论电路有多复杂，你都可以用这两个定律通过节点分析(对于 KCL)或网格分析(对于 KVL)来分析电路。唯一的问题是，你最终会得到大量的方程，并且可能不得不以联立方程组的形式来求解它们。幸运的是，计算机在这方面非常擅长，电路分析软件经常使用这些技术中的一种来寻找答案。

考虑[这个电路](http://tinyurl.com/mcnypby):

![](../Images/6a978ecdd4a56e825e28f43e5f4956ef.png)

这实际上太简单了，因为我们一开始就知道 V1 和 V2(5V 用于电池，0，因为 V2 接地)。此外，人类知道如何计算 R2 和 R3 的等值，但在更复杂的电路中，尤其是对计算机而言，这可能并不明显。

标记为 Vx 的节点有三个电流。I1 是通过电池和 R1 流入的电流。I2 是流经 R2 的电流，I3 是流经 R3 的电流。你可以很容易地写出所有三种电流的方程式:

```
I1=(Vx-V1)/R1

I2=(Vx-V2)/R2

I3=(Vx-V2)/R3
```

当然，我们知道右边所有东西的值，除了 Vx，所以:

```
I1=(Vx-5)/300 

I2=Vx/R2

I3=Vx/R3
```

注意上面第一行是“向后”的，因为 I1 是流入节点 Vx，其他都是流出；有几种方法可以选择来处理这个问题。现在使用 KCL，我们知道:`I1+I2+I3=0`你可以用它们的等式替换所有的 I:

```
(Vx-5)/300 + Vx/500 + Vx/100 = 0

(5Vx+3Vx+15Vx)/1500 = 5/300

23Vx/1500=5/300

23Vx=1500(5/300)

Vx=25/23=1.09V (about)
```

对于上面的第 2 行，300、500 和 100 的最小公倍数是 1500，我们在两边加上 5/300，得到 Vx 项。在第 4 行中，我们将两边都乘以 1500 来得到解。

如果你看仿真，你会看到 Vx 是 1.09V。现在你可以回到方程，得到 I1，I2 和 I3，只需插入值。当然，真正的问题会变得更加棘手，通常会以你必须解决的一系列方程而告终。

如果你真的想追求更高的数学，你可能会喜欢汗学院关于节点分析的视频，如下。请注意，它们明确地处理了负电流的概念。如果你想在我们的例子中使用他们的数学，那么 I2 和 I3 显式为负，I1 来自 5-Vx 而不是 Vx-5。然后你得到-23Vx=-25，最后得到同样的结果。数学就是这样。

## 历史

德国物理学家古斯塔夫·基尔霍夫(Gustav Kirchhoff)在 1845 年发现了这一切，比欧姆(Ohm)发现他的定律晚了大约 20 年。其实[Ohm]不是第一个，他只是第一个说的。[亨利·卡文迪许]在 1781 年用莱顿瓶(大电容器)和他自己的身体作为电流表发现了欧姆定律。他会用自己的身体接通电路，并根据受到的电击量来判断电流的大小。这才是奉献。[Ohm]有一个更好的实验装置，而且——据我们所知——当然不会电击自己。

你可能认为[欧姆]因为他的发现而备受尊敬，但事实并非如此。当权派对他的发现非常不满。一本德国科学评论年鉴称其为“赤裸裸的幻想之网”。德国教育部长称之为“异端邪说”。这与巴洛定律(1825 年由彼得·巴洛提出)相反，巴洛定律认为电流与导线的直径和长度有关。

实际上，[巴洛]并没有完全错。他使用恒定电压，但不理解(像[欧姆]一样)电压源有内阻。事实上，[Ohm]从电池切换到热电偶，因为那时它们有更稳定的输出和可预测的低内阻。

今天很难想象，但当时有很多实验和法律条文——显然不是所有都是正确的。通常我们联系到的人并不是真正的第一个，只是发表的那个人。另一个例子是惠斯通电桥。(查尔斯·惠斯通爵士)让它出名了，但它实际上是(塞缪尔·克里斯蒂)的创意。

## 然后呢。

出于某种原因，每个人都知道欧姆定律，但你不太听说可怜的老[古斯塔夫]。如果你上电气工程课，这些定律是你首先要学的东西。你可能不会每天都使用它，尤其是在计算机模拟的今天。然而，像这样理解分析可以帮助你对电子学有一个直观的理解。
