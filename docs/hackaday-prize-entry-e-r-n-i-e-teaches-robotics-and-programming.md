# Hackaday 奖参赛作品:E.R.N.I.E .教授机器人和编程

> 原文：<https://hackaday.com/2017/08/23/hackaday-prize-entry-e-r-n-i-e-teaches-robotics-and-programming/>

[Sebastian Goscik]的 2017 年 Hackaday 奖参赛作品是一个排队机器人。嗯，不尽然；最终结果是一个循线机器人，但实际项目[是一个简单、廉价的机器人底盘](https://hackaday.io/project/26057-ernie)，用于学校、俱乐部和其他教育、蒸汽教育活动。随着机箱设计而来的是一个课程计划，允许教师在向学生演示该套件时有一个良好的开端。

课程计划是针对循线机器人的，但在设计中是第二堂课——通过总线连接到主基站并同步工作的交通灯。这些课程的理念是对教师和学生来说都相当简单明了，以便让他们对 STEM 科目更感兴趣。

[Sebastian]注意到的其他机器人套件是，它们要么昂贵，要么复杂，要么缺乏教程。有些要么是预先组装好的，要么需要很长时间才能组装好。[Sebastian]简化的东西–在初次组装底盘后，唯一需要的东西是:拉链、绝缘胶带和几个螺钉。PCB 不能拆卸，但是组装好的 PCB 可以重复使用。

[Sebastian]拿出的硬件由大约 3 毫米可以激光切割的材料(丙烯酸或木材)和一个传感器板组成，该传感器板有 5 个红外 led 和相应的红外传感器。只需用一把十字螺丝刀就可以将机箱组装在一起，传感器 PCB 都有详细记录，因此焊接起来尽可能容易。一个 Arduino 被用作这个单位的大脑。

[Sebastian]提出了一个很棒的项目，像这样一个平台，包括几个课程计划的想法是一个很棒的想法。他还在开放硬件许可下发布了硬件，这样其他人也可以共享和添加。当然，还有其他循线机器人，像[这个用模拟电路制造的微型机器人](https://hackaday.com/2014/03/28/a-mini-op-amp-based-line-following-robot/)，还有其他用于教学的开源机器人[，像这个](https://hackaday.com/2016/05/11/open-robots-with-open-roberta/)。但(塞巴斯蒂安)对课程计划的关注是解决问题的一种真正独特的方式——一种有望非常成功的方式。
