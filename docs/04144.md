# 本周失败:谈中国计算器合成器乐队

> 原文：<https://hackaday.com/2016/11/22/fail-of-the-week-talking-chinese-calculator-synth-orchestra/>

有些时候，当你开始做一件事，虽然你没有实现你的目标，但你成功地做了一些其他的事情，只是有点特别。[TheKhakinator]发给我们一些他描述为失败的东西，但即使我们将它作为我们的本周失败系列之一发布，我们认为结果仍然有一些关于它的胜利。这可能不是他所希望的惊人之作，但在这种情况下，这真的无关紧要。

在中国旅行期间，他的注意力被一个日常电子小玩意吸引住了，这是一个电子计算器，当你使用它时，它会用中文说出数字和运算。他买了几个，希望当他把它们放回工作台上时，他会发现一个包含样本的 EEPROM，他可以用自己的来替换它，成为一个便宜但低比特率的采样器。

遗憾的是，这种巧妙的破解并没有成功，因为当[他把令人惊讶的构造良好的计算器拆下来](https://www.youtube.com/watch?v=pYufIaR41S4)时，他发现只有一团环氧树脂隐藏着一个芯片。然而，这并没有失去一切，因为在研究该设备的功能时，他发现除了能说出中文数字和操作数外，它还内置了一系列闹铃，以及一种作为初级电子琴操作的模式。他给我们留下了几个视频，我们已经在休息时间下面发布了，首先是他的拆卸，然后是一个虚拟的[计算器乐队演奏舞曲](https://www.youtube.com/watch?v=jlQxk9IPQ34)，因为他忘记了失败，专注于胜利。
