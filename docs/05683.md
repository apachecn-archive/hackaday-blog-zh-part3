# 发射第一颗开源卫星

> 原文:[https://hack aday . com/2017/04/19/首飞开源卫星/](https://hackaday.com/2017/04/19/flying-the-first-open-source-satellite/)

自由空间基金会是一个致力于自由空间硬件开发的组织。它诞生于 SatNOGS 项目——第一届 Hackaday 奖的获奖者——现在这个基金会在太空中。自由空间基金会[昨天搭乘了 ATK 轨道发射](https://www.linkedin.com/feed/update/urn:li:activity:6259731952476258304/)，现在他们完全开源的立方体卫星正在前往国际空间站的路上。

[问题中的立方体卫星是 UPSat](https://upsat.gr/) ，一个完全开源的 2U 立方体卫星。从机箱到固件的一切都是完全开放的，所有的源文件都托管在 GitHub 上。

UPSat 目前正在前往国际空间站的途中，装载在轨道 ATK 天鹅座货运飞船中。从这里，UPSat 将由当前国际空间站探险队的成员卸载，并在 [NanoRacks](http://nanoracks.com/products/smallsat-deployment/) 的帮助下部署。基本上，第一颗开源卫星将从国际空间站被扔出。如果你想收听 UPSat 向下传送的数据，[建造一个 SatNOGS 地面站](https://hackaday.io/project/1340-satnogs-global-network-of-ground-stations)，调谐到 435.765 MHz。有了好的天线，当国际空间站在天空中时，你应该能听到它，或者一周几次。

你可以在下面的视频中查看 UPSat 搭载的天鹅座飞船的发射情况。美国宇航局还记录了发射台的 360 视频，不幸的是，在发射后的头几秒钟就被切断了。
