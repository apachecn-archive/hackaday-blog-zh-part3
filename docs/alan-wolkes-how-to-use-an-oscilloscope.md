# [Alan Wolke]的如何使用示波器

> 原文：<https://hackaday.com/2016/07/25/alan-wolkes-how-to-use-an-oscilloscope/>

如果你要创建一个黑客读者和示波器拥有者的维恩图，这两个群体很可能会有一个非常重要的交集。不管这个仪器是几十年前的 CRT 老爷车还是闪亮的现代数字示波器，它可能是你经常使用的东西，你会非常熟悉它的操作。

示波器是一种包含大量功能的非常复杂的仪器。特别是“现代”望远镜通过软件带来的功能，仅仅在几年前是不可想象的。所以当你看你的示波器时，你真的知道如何使用它的每一个功能吗？您是否从中获得了最大收益，或者您只是对它的功能略知一二？

[Alan Wolke，w2ew]是 Tektronix 公司的应用工程师，因此，正如您所料，他对示波器略知一二。他在过去的“Dopes 示波器”讲座中谈到过这个问题，他的最新视频是对新泽西州古董无线电俱乐部的演示，是对使用示波器的彻底探索。视频在休息时间下面，时长一小时二十分钟，是一个很长的视频。我们不会为此道歉，因为对于任何示波器的拥有者来说，它的整体应该是令人着迷的。即使你发现自己点头同意他所说的大部分内容，也一定会有你没有意识到的“范围智慧”的珍珠。
