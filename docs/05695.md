# [Hari]打印一个很棒的蜘蛛机器人

> 原文:[https://hack aday . com/2017/04/22/Hari-prints-an-awesome-spider-robot/](https://hackaday.com/2017/04/22/hari-prints-an-awesome-spider-robot/)

虽然我们强烈怀疑模型的设计者在昆虫学上失败了，但是这个蜘蛛机器人非常酷。[Hari Wiguna]做了一个，对结果感到兴奋是理所当然的。(观看下面嵌入的 YouTube 上的[他的总结。)](https://www.youtube.com/watch?v=9Pos9pE8xwU)

多亏了[Regis Hsu]的[漂亮设计](https://www.instructables.com/id/DIY-Spider-RobotQuad-robot-Quadruped/)，所有[Hari]要做的就是以大约 20 美元的价格订购一打六足机器人的 9g 伺服系统，打印出零件，安装一个 Arduino 克隆体，他就完成了。我们真的很喜欢印刷部分的切口，很好地适应伺服喇叭。[Hari]说校准过程很简单；你运行一个草图，将所有的伺服系统设置到一个已知的位置，然后将腿拧紧到位。非常圆滑。

基本上任何打印机都不支持打印部件。[Hari]的有点滑稽，展示了各种层与层之间的不规则性(对不起，伙计！)但是机器人工作起来很完美。这并不是说[Hari]没有组装技能——看看[世界上最小的(？)RGB LED 立方体如果你觉得这家伙不会焊接](http://hackaday.com/2016/02/05/building-the-worlds-smallest-rgb-led-cube/)。当然，你可以完全避开 3D 打印部件，只需[将一堆伺服系统固定在一起，称之为机器人](https://hackaday.io/project/6050-tote)。制造一个四条腿的机器人比这两个项目更容易。你还在等什么？
