# 拿着蓝色药丸往前走

> 原文：<https://hackaday.com/2017/08/04/take-the-blue-pill-and-go-forth/>

Forth 作为一种流行的黑客语言有着悠久的历史。引导很简单。很有表现力。它可以是一个非常强大的系统。[jephthal]将出色的 Mecrisp 放在非常便宜的 STM 32“blue pill”板上，创建了一个开发系统，成本约为 2 美元。你可以看下面的视频。

如果你有三十分钟的时间，你会发现复制他的壮举是多么容易。蓝色药丸板必须使用 STM32 编程器编程一次。之后，您可以使用最标准的 Forth 单词，也可以使用一些可以操作低级微控制器资源的单词。

蓝色药丸板的价格真的非常惊人，只需几美元就能提供强大的计算能力。给它添加一种交互式的高级语言会使它更加有用。我们已经使用 [Arduino IDE](https://hackaday.com/2017/03/30/the-2-32-bit-arduino-with-debugging) 和 [Platform.io](https://hackaday.com/2017/04/07/platformio-and-visual-studio-take-over-the-world/) 覆盖了这个平台几次。

我们已经对这个相同的主题做了自己的研究。Mecrisp 可用于[相当多的其他平台](https://hackaday.com/2015/08/30/go-forth-on-a-breadboard/)，我们已经覆盖了其中相当多的平台。如果你想更多地了解作为一种语言的[第四语言](https://hackaday.com/2017/01/27/forth-the-hackers-language/),【埃利奥特·威廉姆斯】今年早些时候报道过。
