# 改进的伺服系统为望远镜增加了聚焦控制

> 原文：<https://hackaday.com/2016/12/26/modified-servo-adds-focus-control-to-telescope/>

用望远镜扫描天空是度过漫长而晴朗的冬夜的好方法，但使用手动望远镜可能会很麻烦。带有高度和方位控制的电动安装架是严肃的观察者的基本设备，但是[添加伺服系统来控制望远镜的焦点](http://seiferteric.com/?p=375)比普通的现成仪器更进一步。

作为一个高级项目，Eric Seifert 已经将他的普通望远镜的赤道架的两个轴机动化，他决定也将聚焦架机动化。他的第一个倾向是使用步进电机，就像他在其他两个轴上所做的那样，但手头有一个备用的高扭矩伺服系统，他很快就完成了概念验证。伺服系统被修改为以通常的方式在[连续旋转，但是增加了用外部线性电位计替换内部电位计的扭曲。连接到聚焦管，线性锅允许[埃里克]控制修改伺服的位置和速度。听起来控制焦点对[Eric]计划的他的范围的 web 界面很重要；我们很快就会看到那个项目的细节。](http://hackaday.com/2016/10/30/primer-on-servos-hits-all-the-basics/)

我们喜欢这个解决方案的简单性，对于其他项目来说，这是一个值得记住的技巧。但是如果你不喜欢花哨的步进器和伺服系统，不用担心——天体摄影就像用铰链把几块木板拍在一起一样简单。
