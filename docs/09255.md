# 带 Tinkercad 的参数铰链

> 原文：<https://hackaday.com/2018/04/23/parametric-hinges-with-tinkercad/>

简单的工具很好，但有时使用简单的东西更方便，因为它可以完成工作，所以你不需要尝试其他的特性。Tinkercad 是这种程序的一个很好的例子。它实际上相当强大，但许多人只是以最简单的方式使用它。[Chuck]注意到一个关于使用 Tinkercad 制作 3D 打印铰链的视频，在那个视频中[Nerys]使用剪切和粘贴以及箭头键手动放置了一堆铰链用于定位。虽然它有效，但这不是最优雅的方式，所以[Chuck] [制作了一个视频，展示如何参数化地做这件事](https://www.youtube.com/watch?v=OOqH3nxt3XY)。你可以看到下面的视频，以及原始的铰链视频。

查克展示了两种主要的技术。首先，他向 Tinkercad 工具箱添加创建铰链所需的零件。这使得将它们添加到您未来的任何设计中变得非常简单。其次，他使用数字参数和复制的组合来快速准确地将铰链组件放置在另一个对象上——在这种情况下是蝙蝠侠标志。

有问题的铰链组件是一种[销铰链](https://www.thingiverse.com/thing:2840744/#files)。如果你切开它，看看这些层，你会看到一个轴承抓住了一个在里面旋转的销钉。然而，还有其他方法来制作铰链。你可能以前见过的一种流行的铰链是活铰链。你有没有用过塑料盒，盒盖和盒子是由一片薄薄的塑料连接在一起的？那是一个活铰链。他们很难用普通材料 3D 打印，但不是[不可能](https://www.3dhubs.com/knowledge-base/how-design-living-hinges-3d-printing)。你也可以用[激光切割它们](https://hackaday.com/2017/04/06/two-piece-boxes-thanks-to-laser-cut-flex-hinges/)。

就个人而言，我们可能会使用 OpenSCAD 来进行参数化建模，但我们确实喜欢关于使用 Tinkercad 上的中心点的提示，另外——对大多数人来说——看到形状形成比像我们这样编写代码有趣得多。如果你对图形化的参数化建模感兴趣，可以看看效果不错的 [SolveSpace](https://hackaday.com/2016/06/18/solvespace-a-parametric-cad-tool/) 。
