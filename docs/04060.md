# 保护你肺部的宜家台灯

> 原文:[https://hack aday . com/2016/11/15/IKEA-desk-lamp-that-will-defend-your-lungs/](https://hackaday.com/2016/11/15/ikea-desk-lamp-that-will-defend-your-lungs/)

虽然有些人可能喜欢偶尔闻到一股有害气味——汽油、斧头身体喷雾等——但长时间暴露在烟雾中对你的健康没有好处。这也适用于焊接，难道不是你该在你的工作台工具上加些除污剂的时候了吗？

受到我们之前介绍过的一些通风柜的启发——请注意，缺乏项目的叶——[格奥尔格·斯鲁特曼 ]组装了他自己的[宜家灯吸油烟机](https://blog.sman.dk/?p=328)。

最引人注目的特点是，它安装在宜家台灯上，便于放置，最大限度减少杂乱。NeoPixels 灯条照亮了您的焊接空间，而 PIR 传感器在检测到移动时会启动风扇。一个 WeMos D1 迷你版包含了 WiFi 连接，但是这个功能还需要一点时间。现有的功能仍然令人印象深刻；休息之后会有更多的报道。
