# 给你的吐司黄油(有时)

> 原文:[https://hack aday . com/2017/02/24/toast-bot-butters-for-you-something/](https://hackaday.com/2017/02/24/toast-bot-butters-for-you-sometimes/)

有时——尽管不切实际、安全、失败和一般常识——人们有一种为了项目而完成项目的冲动。当你厌倦了每天早上给烤面包涂黄油时，你可能会学学里克·桑德克——咳咳，[威廉·奥斯曼]的书，造一个[烤面包机器人来帮你完成任务](https://www.youtube.com/watch?v=61WGOJ3oYUw)。

[Osman]——选择钉过度杀伤商——是使用往复锯电机来固定黄油，同时烤面包片在由线性步进电机控制的平台上的设备下面移动。Toast-Bot 的框架和底座是用他自制的激光切割机(被亲切地命名为 Retina Smelter 9000’)从木头上切割下来的，经过一些挫折和拉链的应用后组装而成。最后的结果确实是黄油吐司，但是——嗯——你自己看吧。
