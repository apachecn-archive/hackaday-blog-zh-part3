# 微芯片 ICD4 评论

> 原文:[https://hackaday.com/2017/09/08/microchip-icd4-review/](https://hackaday.com/2017/09/08/microchip-icd4-review/)

[Mike]是一名狂热的 PIC 开发人员，他用 ICD4 取代了他的 ICD3 调试器。他用他的印象制作了一个视频，你可以在下面看到。[Mike]发现带有性感 LED 的厚重铝制外壳很有吸引力，但想知道他为什么要购买开发工具。他还对他们用新连接器替换 ICD3 电缆连接感到不满。最后，他希望别针能印在箱子上。

另一方面，ICD4 也将进行 JTAG，并处理 Atmel 部分(微芯片[收购了](https://hackaday.com/2016/10/18/whats-the-deal-with-atmel-and-microchip/))。[Mike]打开盒子，展示设备的内部，然后实际使用它来完成预定的任务。

正如您所料，ICD4 与 MPLAB 集成良好，并具有您所期望的 ICD3 和其他微芯片编程器/调试器的相同功能。[Mike]发现编程算法与以前的设备有些不同，所以他将示波器放在上面，并比较了 PICKit3、ICD3 和 ICD4 的编程速度。

ICD4 是一个非常重要的工具。如果你只是想试验，你可以建立自己的 PIC 程序员或尝试 PICKit 3。你也可以买一块便宜的主板，走在线 IDE 路线。
