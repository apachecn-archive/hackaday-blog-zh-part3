# 老虎钳快速释放锁定你的零件为好

> 原文：<https://hackaday.com/2016/06/14/vise-quick-release-locks-down-your-parts-for-good/>

如果你曾经使用过钻床，你可能也熟悉可怕的空中零件。只要钻得再远一点，然后——咔嚓——一个粉碎的钻头就会让你的零件旋转，或者更糟，穿过车库。我们很快学会了用老虎钳夹紧零件。不幸的是，即使是老虎钳也不能阻止钻头四处跳动，在它想钻的地方钻。

幸运的是[djpolymath]有一个解决办法。他用备用自行车快卸装置拼凑了一个既简单又巧妙的老虎钳。在自行车上，快速释放是一种无痛的机制，可以在紧要关头取下车轮，而无需使用花哨的工具。[djpolymath]只是简单地将一些备件重新放置在一个老虎钳上。用几个垫圈把它们正确隔开，他就搞定了——老虎钳也搞定了。

既然车库里的事情变得更安全了，为什么不尝试一些其他的工具改进，比如这个拼图桌。
