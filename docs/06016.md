# 澳大利亚树莓派教程

> 原文：<https://hackaday.com/2017/07/02/australian-raspberry-pi-tutorials/>

澳大利亚公司 Core Electronics 有一个新的非常详细的关于树莓派的视频教程。共有 30 个视频和 5 个章节。少数介绍视频很短，但详细视频从 3 到 16 分钟不等。

讲师[Michael]从最开始开始——在 Pi 上加载 NOOBS——然后转到 Python、shell 脚本和使用 TkInter 构建 GUI 应用程序。还将介绍如何在与 IFTTT 集成的物联网应用中使用 Particle Pi。

我们确实意识到，大多数阅读 Hackaday 的人可能至少用过一两次树莓派。然而，我们也知道，我们都被要求为初学者推荐材料，或者——在某些情况下——我们在学校或黑客空间使用材料授课。

Core 也有类似的 Arduino 和 3D 打印视频研讨会。当然，还有很多其他的教程，但是有选择总是好的。

如果你想看到一些实用的东西，我们已经看到了其他使用 TkInter 的项目。顺便说一句，如果你正在上课，但没有足够的树莓派，你可能会在 Linux PC 上做很多(但不是全部)，特别是用 [PIXEL live CD](https://hackaday.com/2016/12/22/raspberry-pi-software-comes-to-pc-mac/) 。
