# 树莓派和 Alexa 让泰迪·鲁斯宾比一般的熊聪明

> 原文:[https://hack aday . com/2016/12/24/raspberry-pi-and-Alexa-make-teddy-ru xpin-smart-than-the-average-bear/](https://hackaday.com/2016/12/24/raspberry-pi-and-alexa-make-teddy-ruxpin-smarter-than-the-average-bear/)

看看亚马逊的 Alexa 和 80 年代功能有限的电子动画熊 Teddy Ruxpin 的邪恶结合吧。 [Alexa Ruxpin](http://www.tinkernut.com/portfolio/alexa-ruxpin-raspberry-pi-alexa-teddy-bear/) ？

好像把 Alexa 塞进一条会说话的鱼里面还不够糟糕，现在亚马逊的虚拟助手可以通过令人毛骨悚然的复古毛绒与你交谈，这要归功于[小叮当]的记忆之旅。在易贝以远低于童年圣诞树下 70 美元的原价找到了一只泰迪熊后，[小叮当]很快意识到要让这只泰迪熊复活，必须进行大手术。下面的第一个视频显示了原始伺服系统被切除，现代微型伺服系统被移植，允许通过 Arduino 控制嘴，眼睛和鼻子。

随着熊再次控制了自己的官能，[小叮当]开始让它有话可说。运行 AlexaPi 的 Raspberry Pi 加入了熊最近空出的胸部，音频输出在熊的扬声器和 Arduino 上的模拟输入之间分离。结果是一个合理的动画，虽然我们会说一点点调整 Arduino 脚本可能有助于同步。那些眼睛和鼻子也需要融入比赛。但这是一个不错的开始。

这不是泰迪·鲁斯平第一次以黑客的名义接受手术，也不会是最后一次。还有[玩具制造商正在走的路](https://hackaday.com/2016/01/29/hello-barbie-not-an-iot-nightmare-after-all/)，他们可能会抢先我们黑客一步。
