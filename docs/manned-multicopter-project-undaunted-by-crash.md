# 有人驾驶多翼飞机项目不怕坠毁

> 原文：<https://hackaday.com/2016/02/13/manned-multicopter-project-undaunted-by-crash/>

我们不得不对[令人惊叹的项目]印象深刻，他们完全完成了他们的载人多直升机建造，这已经跨越了八个视频。他在第八段视频中解释了这次事故，并从最近的错误中吸取了教训。

当你认真对待它的时候，它就像看起来一样危险。然而，对于业余爱好者来说，巨型多翼飞机可能是最容易制造的飞行器。这是一种低效的蛮力方法，但肯定比试图从零开始建造一架直升机要好。这台机器是一个非常不符合空气动力学的椅子，它的框架和月球车有很多共同之处；上面有引擎。简单。

有很多方法可以解决这个问题。其中一个更疯狂的是[这个装置](https://www.youtube.com/watch?v=t5JgnMJzCtQ)有一堆愚蠢的电动马达。[amazingdiyprojects]配备了八个汽油发动机。我们对他控制每台发动机转速和处理内燃机节气门非线性响应的方法非常感兴趣。然后将所有这些反馈到可能与普通 diy 无人机完全相同的电子设备中。

老实说，我们很惊讶它居然起作用了，我们等不及让他完成它，这样我们就可以看到他在他的危险椅子上跑来跑去。休息后的视频。
