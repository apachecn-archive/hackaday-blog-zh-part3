# 555 等离子扬声器

> 原文：<https://hackaday.com/2016/03/27/555-plasma-speaker/>

555 无所不能。好吧，这在我们的社区中已经成为一个比喻，但是在它背后有相当多的真理:这个小小的定时器芯片是一个惊人的多功能组件。

[Alexander Lang]为 555 的曲目增加了另一项成就，[他用它创造了一个等离子扬声器](http://langster1980.blogspot.co.uk/2016/03/555-flyback-driver-and-plasma-speaker_15.html)。在 [Hackspace Manchester](https://hacman.org.uk/) 工作时，他将 555 用作脉宽调制器，通过 MOSFET 驱动回扫变压器，为安装在激光切割外壳中的火花隙供电。结果可能还不是高保真的，但它是有效的，非常清晰。

我们已经跟踪这个项目有一段时间了，因为他通过几次迭代更新了他的进度。从[最初的设计想法](http://langster1980.blogspot.co.uk/2016/02/555-flyback-driver-and-plasma-speaker.html)到 [PCB 和外壳设计](http://langster1980.blogspot.co.uk/2016/02/555-flyback-driver-and-plasma-speaker_9.html)，到[第一个工作原型](http://langster1980.blogspot.co.uk/2016/03/555-flyback-driver-and-plasma-speaker.html)和一些音频改进，最后是这个外壳中有火花隙的最新帖子。他仍在完善他的演讲者，所以还会有更多

在休息时间下面的视频中，他演示了他的脉宽调制器，并使用键盘作为输入来测试该设备。

我们在 Hackaday 上只展示了[一个等离子扬声器](http://langster1980.blogspot.co.uk/2016/03/555-flyback-driver-and-plasma-speaker.html),但我们肯定涵盖了 555 个项目。有些特别令人难忘，像这个基于 [555 的计算机](http://hackaday.com/2011/08/05/building-a-computer-out-of-555-chips/)，而其他像这个 [AM 发射机](http://hackaday.com/2011/01/10/555-based-am-radio-transmitter/)和[接收机](http://hackaday.com/2011/02/25/hear-that-its-a-555-timer-am-radio/)则是更有趣的项目。自己浏览它们，[通过查看标签为“555”](http://hackaday.com/tag/555/)的帖子。

与此同时，在 2012 年，我们注意到它的创造者【汉斯·卡门辛德】的去世，我们 555 个人对他感激不尽。
