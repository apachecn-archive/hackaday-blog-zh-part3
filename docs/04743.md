# 无绳电钻不耗电

> 原文:[https://hack aday . com/2017/01/16/cordless-drill-uses-no-electricity/](https://hackaday.com/2017/01/16/cordless-drill-uses-no-electricity/)

关于如何制作自己的无绳电钻的项目很少，但让[Johnny 90 ' s][这个令人惊叹的项目](https://www.youtube.com/watch?v=yX-gKv6pHN0)与众不同的是，他的动力装置是一台硝基发动机。当然，这不是一个简单的任务，但他让它看起来比实际上更容易，我们真的很享受建造过程。

他使用的 RC 阔嘴猴睡衣 GX12 发动机是[之前改装的](https://www.youtube.com/watch?v=gSCbBVSIOpk)，用一个更大的冷却头来更换。发动机驱动一个从另一台钻机上取下的齿轮箱。所有其他部分都是手工制作的。离合器经过精心加工，冷却风扇是用 3D 打印机制造的。其他必要的部分是框架，黄铜垫片，以调整发动机的高度和对齐，油门臂和手柄。最后，甚至连变速箱也不得不为了更高的速度而改装。完成后的钻机看起来和听起来都很棒，似乎完全能够完成它的工作。

与[Johnny 90]的其他机械项目一样，该视频具有良好的时间安排和对细节的关注。他的频道绝对值得一去，尤其是如果你喜欢涡轮机的话。
