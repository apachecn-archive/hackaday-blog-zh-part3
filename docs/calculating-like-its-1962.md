# 像 1962 年一样计算

> 原文：<https://hackaday.com/2017/12/01/calculating-like-its-1962/>

我们有时会忘记，今天我们认为微不足道的事情是昨天极端工程的壮举。想想这个不起眼的袖珍计算器，如今它如此便宜和容易制造，以至于它们基本上是一次性的。但是，在 1962 年建造一个简单的“四个香肠”计算器绝不是一件简单的任务，值得看看我们今天站在巨人的肩膀上几乎一无所有地完成了什么。

如果有什么事情不能让克里夫·斯托尔的热情变得有趣，我们不知道会是什么，他肯定会通过拆卸和分析老式电子计算器来完成这项工作。你会从他的书 [*【布谷鸟的蛋】*](http://hackaday.com/2017/10/19/books-you-should-read-the-cuckoos-egg/) 中记得【克里夫】，记录了他在 80 年代中期揭露一帮为克格勃工作的黑帽黑客的电脑侦探。[Cliff]偶然发现了一对 Friden EC-132 电子计算器，在 1962 年设计它们的工程师[Bob Ragen]的帮助下，其中一个开始工作。这些计算器有一个 PC 板架，巧妙地铰接以节省空间，并塞满了锗晶体管、一个 CRT 显示器和一个声学延迟线存储器。但是当你花一点时间思考这样一件事到底花了多少心血时，它真的会让你想知道老前辈们是如何将产品推向市场的。

顺便说一句，很高兴看到[悬崖]在这么多年后仍然如此充满活力。看着他如此兴奋和充满激情地跳来跳去，真让我们兴奋不已。
