# 门多西诺电机驱动隔间对话

> 原文：<https://hackaday.com/2017/11/05/mendocino-motor-drives-cubicle-conversations/>

Mendocino 电机是依靠伪悬浮的太阳能电机。悬浮来自安装在轴两端的磁铁，磁铁将固定在轴下方的同磁场磁铁排斥到底座中。当光线照射在太阳能电池板上时，电流流经相连的电磁线绕组，产生电磁场，与安装在下方的大型固定磁铁相互作用。这些不断的排斥力使转轴旋转，太阳能电池板之间的间隙提供了使其旋转 360°所需的开关周期。

[正如【康斯坦丁】发现的](http://www.instructables.com/id/Solar-Mendocino-Motor/)，制造这个简单的马达并让它旋转取决于很多因素。绕组的数量、每个太阳能电池板的重量以及磁铁的尺寸都包括在内。然而，康斯坦丁的挣扎是你的收获。他的 Instructable 将猜测排除在公差之外，并设计了一个漂亮的开源 3D 打印结构。

你说得对，这些发动机做不了多少工作。但它放在你的桌子上肯定会很酷，甚至可能引发一两次对话。如果没有，[搅起这个小电磁火车](https://hackaday.com/2014/12/04/amazing-sciences-simple-electric-train/)。
