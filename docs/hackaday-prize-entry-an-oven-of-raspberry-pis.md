# Hackaday 奖品入口:一炉树莓馅饼

> 原文：<https://hackaday.com/2016/08/13/hackaday-prize-entry-an-oven-of-raspberry-pis/>

当 Raspberry Pi 问世时，世界得到了一台非常便宜、可用的 Linux 计算机。便宜是好的，它使一种以前相当昂贵的项目成为可能。当然，这是集群计算，现在我们可以想象在我们的公寓里有一个 Aronofsky 风格的 Beowulf 集群。

[这个 Hackaday 奖项的参赛作品](https://hackaday.io/project/11132-hadoop-on-a-100-board-raspberry-pi-cluster)是一个运行 Hadoop 的 100 板 Raspberry Pis 集群。以前有人做过类似的事情吗？当然可以。诀窍是要正确，能够物理扩展群集，并在其上安装正确的软件。

Raspberry Pi 并没有在所有正确的地方都有连接器。以太网和 USB 在一边，电源输入在另一边，如果您需要直接串行连接到堆栈中间的 Pi，上帝会帮助您的。这是将一簇 pi 放在一起的物理问题。如果你非常聪明，并且正在使用 Pi 零点，你会想出类似于这个的[东西，但是对于普通的 Pi，你需要一个外壳、一个强大、高效的电源和一堆网络开关。](http://hackaday.com/2016/01/25/raspberry-pi-zero-cluster-packs-a-punch/)

对于软件，这个覆盆子盒子背后的团队正在转向 Hadoop。雅虎最近建立了一个具有 32，000 个节点的 Hadoop 集群，用于深度学习和其他计算密集型任务。这个小得多的集群不会用于要求非常高的工作。相反，这个集群将用于教育、培训和训练那些曾经重要的 STEAM 学生。这是一个小包装中的大数据，也是 Hackaday 奖的一个伟大项目。
