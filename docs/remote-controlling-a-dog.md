# 遥控一只狗

> 原文：<https://hackaday.com/2017/01/21/remote-controlling-a-dog/>

直到工业革命，人类利用动物来使我们的劳动更容易。这在一些小众领域仍然可以看到，比如还没有发明出像猪一样善于嗅出松露的机器。[威廉]已经听回了人类早期的根源，但在一个更现代的转折，使一个通用狗的东西，可以可行地做任何可以想象的工作。[现在他的狗是遥控的](http://www.williamosman.com/2017/01/biohacking-dog.html)。

[William]用一种卡通的方式完成了这个巨大的任务，用了一个古老的比喻:在某物面前挂一个热狗，让他们去追它。连接点安装了一个遥控接收器和一个致动器，以使悬挂的热狗向狗的右边或左边摆动更多一点，这取决于“操作者”希望狗去哪里。[威廉]的牛头犬似乎是一个很好的运动，而且也不会磨损。

信不信由你，已经有一些关于遥控动物的实际研究，尽管目前为止仅限于遥控蟑螂。虽然我们喜欢遥控狗的简单性，但不要指望这些装备会很快取代皮带！
