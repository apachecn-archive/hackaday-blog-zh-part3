# 来自音频和纯数据的视频

> 原文:[https://hack aday . com/2015/10/11/video-from-audio-and-pure-data/](https://hackaday.com/2015/10/11/video-from-audio-and-pure-data/)

尽管图形编程语言已经存在了很长时间，但它们在教育环境之外还没有真正的用处。这方面的少数反例之一是纯数据和 Max MSP，它们是可视化编程语言，使音乐和视频开发变得简单，只需放下几个盒子并在它们之间画线。

几年前，[托马斯]和[丹尼]开发了一个非常酷的纯数据视听演示。他们开发的程序只能生成图形，但通过巧妙的编码，他们能够从计算机中输出的任何视频中生成一些音频信号。这个项目被称为 TVestroy ，这是你见过的最酷的视听展示之一。

整个节目在三个大屏幕和九台 CRT 电视上播出。通过一些非常聪明的代码和一个电子黑匣子，视频变成了音频。下面来看看吧。

尽管这是一个相对较旧的版本，但[Thomas]认为现在重新审视这个项目是个好主意。他开源了大部分纯数据文件，所有东西都可以在项目页面上下载。
