# Latskap 半自动酒柜

> 原文：<https://hackaday.com/2017/08/18/latskap-semi-automatic-liquor-cabinet/>

对于喜欢娱乐的绅士或淑女来说，一个储备充足的酒柜是必不可少的。准备好合适的烈酒和调酒器来调制从马提尼到边车的任何东西，总是确保客人在你的鸡尾酒会上玩得开心的可靠方法。在过去，一个制作精美的樱桃或胡桃木酒柜足以让游客对你的富裕印象深刻。如今，如果你不想看起来像个穷光蛋，你必须更进一步。

[伊莱亚斯·巴肯]和他的叔叔[迈克·莫尔顿]决定将酒柜带入 21 世纪，推出一款名为 Latskap 的半自动酒柜。该项目仍在进行中，处于原型阶段，但他们在 Hackaday.io 上的[构建日志](https://hackaday.io/project/26295-latskap)显示出了很大的潜力。不久他们就会完成一个功能齐全的原型。

 [https://www.youtube.com/embed/aFtl-y39Cy0?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/aFtl-y39Cy0?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



Latskap 有几个主要功能:首先是当有人靠近它时，它会自动打开。然后，口渴的客人可以使用触摸屏从菜单中选择他们想要的饮料。橱柜内的瓶子放在 NeoPixels 上，系统会点亮该饮料所需的酒和混合瓶。最后，当配料倒入时，橱柜前面的秤会称出杯子的重量，并告诉口渴的顾客他们已经倒了正确的量。

我们在过去已经看到过[自动售酒机](http://hackaday.com/2013/12/31/barbot-mixes-drinks-perfectly-with-web-interface/)，它们被设计用来混合鸡尾酒，但是 Latskap 采用了一种有趣的新方法。这种设计的主要好处是瓶子的数量只受可用空间的限制。不需要复杂的泵系统。我们非常期待看到成品！
