# 合成生物学创造了活体计算机

> 原文：<https://hackaday.com/2017/08/09/synthetic-biology-creates-living-computers/>

大多数人对 DNA 至少有一个模糊的概念。但是，如果你问的是 RNA，除非你是在和生物学家交谈，否则你可能会更加手足无措。然而，我们黑客可能不得不重读我们的生物教科书，因为研究人员已经用 RNA 建造了逻辑门。

有时我们读了这些大学的新闻稿，意识到结果不太实际。但在这种情况下，亚利桑那州立大学的研究显示了“与”、“或”和“非”门是如何实现的，并显示了使用活细胞的四输入“与”门和六输入“或”门的实际应用。关键是一种被称为 RNA 脚趾开关的结构(见下面的视频)。虽然这是在 2012 年得出的，但最近的这项研究显示了如何实际应用它。

在细胞世界中(这是生物细胞，不是手机细胞)，DNA 几乎就像一个磁盘驱动器——它包含关于该做什么的信息。RNA 具有类似的化学结构，但与 DNA 不同，它通常是单链的。它可以通过转录过程复制 DNA 片段。这种信使 RNA 可以折叠成复杂的三维形状，因为它不与第二条链结合。核糖体可以阅读这种 RNA，并将氨基酸缝合在一起，形成蛋白质，在细胞中实际发挥作用。

我们不是生物学专家，但你可以证明 DNA 就像程序的大容量存储器，RNA 是 ram 存储器，核糖体是整个系统的 CPU 和 3D 打印机。RNA 还被用来将原材料运送到核糖体，就像 RAM 存储器可以保存指令和数据一样。正如 CPU 本身通常包含寄存器形式的内存一样，组成核糖体的组件是蛋白质和 RNA。我们不是生物学家，所以对这个类比要有所保留。

由于细胞的微小尺寸和基于 RNA 的计算工作方式，它为主要的并行性提供了机会，类似于量子计算。此外，由于这可以影响活细胞，研究人员认为这将导致新型智能药物和更好的医疗诊断工具。同一团队的一项类似技术已经证明它可以检测寨卡病毒。

下一步是在细胞内创建神经网络，并形成一个细胞网络(再次强调，不是电话类型)，允许一个细胞中的处理元件与另一个细胞进行通信。我们有物联网。也许有一天我们会有细胞网络。

如果你对生物黑客感兴趣，我们已经讨论过使用 [RNA 转录作为阻止感染的手段](http://hackaday.com/2015/09/08/how-biohackers-are-fighting-a-two-front-war-on-antibiotic-resistance/)。如果你想知道更多关于如何合成寡核苷酸(DNA 和 RNA 的花哨说法)，我们看看[如何在太空中制造它们](http://hackaday.com/2017/04/27/osm-pronounced-awesome/)。
