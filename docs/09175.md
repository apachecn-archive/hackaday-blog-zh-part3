# Tracktorino 让你远离糟糕的界面

> 原文：<https://hackaday.com/2018/04/21/tracktorino-shields-you-from-poor-interfaces/>

数字音频工作站中的屏幕控制扩展了 DJ 或音乐家的能力，但并不是每个人都很直观。按钮、旋钮、滑块和现实世界控制的触感与使用鼠标、触控板甚至触摸屏完全不同。不幸的是，由于位置或成本的原因，本应由 DJ 掌控的设备可能无法获得。[Gustavo Silveira]负责这一情况，以便他可以帮助其他 DJ 和音乐家通过为 Traktor DJ 软件定制的 MIDI 接口控制他们的工作站。

MIDI 是一种广泛使用的串行协议，从 DIN 连接器发展到 USB，现在它也是无线的。这意味着 Traktorino 没有锁定到 Traktor，尽管名称相同。在 Hackaday.io 页面上，甚至有一个它将与之合作的其他工作站的列表，但由于许多工作站，无论如何都是好的工作站，接受这样的 MIDI 硬件，真正的列表要长得多。

定制电路板实际上是一个屏蔽。使用 Arduino UNO，目前 Arduino 世界的典型代表，为许多不了解专业软件的人提供了可访问性。还包括激光切割外壳的矢量图。这意味着，甚至按钮上的标签也没有锁定为英语。

这是另一个结合了激光切割和 MIDI 的项目，制作了一些非常聪明的按钮或[把你的 DIN MIDI 连接器变成 USB](https://hackaday.com/2017/09/26/cheap-diy-midi-to-usb-adapter/) 。
