# 你可以在你的车库里建造一个冷冻干燥机

> 原文：<https://hackaday.com/2018/02/07/a-freeze-dryer-you-can-build-in-your-garage/>

trail mix、宇航员冰淇淋和 cryogel 有什么共同之处？这听起来像是一个老生常谈的谜语，但它们都是你可以在车库里用自制的冷冻干燥机制作的东西。[The Thought Emporium]建造了自己的[冷冻干燥机](https://www.youtube.com/watch?v=obhYSf1ogoo),最大限度地减少了麻烦，只使用了几个外来组件，如真空泵和高质量压力计。该视频也在休息后发布，其中包含零件列表以及在哪里可以购买。

冷冻干燥使用一种叫做冷冻干燥或冻干的过程。在一定的压力下，水跳过液相，直接变成气体，所以冷冻的物品可以从冰变成干的，而不会被弄湿。当你跳过液相时，物体会保持它们冷冻时的形状，因为没有使用热量，你不会使你的糖焦糖化。

像这样的冷冻干燥机有三个部分。首先是不需要任何解释的泵。水泵旁边一定有一个存水弯。这个寒冷的隔间重新冷凝水蒸气，所以它不会进入泵内或浸透你想干燥的东西。最后，有干燥室，你的物品放在那里，让他们的水分带走。
