# 木材店的剪刀升降台，用于木材店

> 原文:[https://hack aday . com/2017/08/02/剪刀升降台从木材店到木材店/](https://hackaday.com/2017/08/02/scissor-lift-table-from-the-wood-shop-for-the-wood-shop/)

商店里可移动的可调节高度的手推车的价值不能被高估。从移动工具到在墙壁和天花板上安装重型固定装置，剪式升降平台是一个很好的工具。不过，商业版本有点贵，所以对于精打细算的人来说，一个商店自制的木制剪刀升降台可能是一个不错的项目。

木材可能不是你制作这种固定装置的首选，但它是[Marius Hornberger]准备使用的材料，通过正确的物种选择和精心的工程设计，它可以制作出非常坚固的桌子。[Marius]选择了白蜡木作为他的零件，这是一种在困难条件下表现良好的木材。当然，桌子不全是木头的；剪式机构中使用金属衬套和销，升降驱动装置是坚固的梯形螺纹螺杆和螺母。我们对[Marius]的细木工技艺以及桌子的坚固程度印象深刻。

出于某种原因，似乎没有多少木工项目出现在我们的提示行中，这是一个遗憾。我们喜欢以木头为特色，就像我们自己的[【John Baichtal】最近指出的](https://hackaday.com/2017/07/28/where-a-wood-shop-goes-a-hackerspace-follows/)，木材店的健康状况通常是黑客空间健康状况的领先指标。
