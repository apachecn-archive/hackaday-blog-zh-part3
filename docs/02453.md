# 不浪费，没有灯

> 原文：<https://hackaday.com/2016/05/27/waste-not-no-lights/>

炼金术士试图将铅变成黄金，但没有成功。如果你能把废物转化为能源会怎么样？这就是[chemicum]在最近的一个视频中所做的——他和一些朋友[建造了微生物燃料电池](https://www.youtube.com/watch?v=_zCsAfEbVRc),将粪便转化为电能(你可以在下面看到视频)。

视频没有给你所有的细节，但它似乎足够简单。你需要一些不锈钢网，一些活性炭，一些环氧树脂，塑料容器，和一些各种各样的金属板和硬件。当然，你还需要排泄物，如果视频有所提示的话，还需要一些衣夹，在你工作时夹住你的鼻子。

制成的电池在 13mA 时产生了 800mV 多一点的电流。当然，你可以增加更多的电池来获得更高的电压或电流。与电池不同，燃料电池只要有燃料来源就能发电。视频中的电池似乎没有任何方法来更新它们的燃料供应。如果你想知道更多关于细菌燃料电池的细节，维基百科总会有。

这不是我们看到的第一个微生物细胞(尽管我们最喜欢的[里面有一条鱼](https://hackaday.com/2014/10/16/microbial-fuel-cell-with-a-side-of-betta-fish/))。这甚至不是我们见到的第一个将[废物转化为电能的电池](https://hackaday.com/2016/03/26/they-put-the-p-in-power/)。
