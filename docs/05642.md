# 廉价简易应急丁烷炉

> 原文：<https://hackaday.com/2017/04/27/cheap-and-easy-emergency-butane-stove/>

自制炉灶是一种非常流行的方式，你可以在 YouTube 上找到无数的视频，大部分是关于酒精炉灶的，它们非常有效。丁烷燃料炉不太常见，但[托马斯·金]上传了一个超级简单便宜的丁烷炉视频。

和其他很多 DIY 炉一样，炉体是苏打铝罐。用铝箔封住顶面后，你只需要在上面钻几个孔就行了。其他必要的部件是金属管和作为流量调节器的注射器针头。[Thomas Kim]制作了一个有趣的固定装置，连接到罐子上，让你控制罐子阀门上的压力，并通过几个螺钉调节炉子的火焰。

炉子很好用。如果你想尝试这些炉灶，这是一个很好的简单的项目。安全当然很重要，在通风区域工作并保护丁烷源免受热影响(在这种情况下，进料管使其远离燃烧器)。其他一些你可能会感兴趣的项目是这个简单的火箭炉，甚至是这个用废料制作煤球的项目。享受并保持安全。
