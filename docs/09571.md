# 醒来的最大好处就是变得更好了

> 原文：<https://hackaday.com/2018/05/31/the-best-part-of-waking-up-just-got-better/>

如果你问我们，生活中最大的乐趣之一就是坐下来喝一杯热腾腾的咖啡、茶或热巧克力。当然，这个仪式最好的部分是当饮料冷却到足够达到最佳饮用温度的短暂时间。

不可思议的事情经常发生——我们喝得太早而被烫伤，或者因看猫视频而分心，或因阅读同事的黑客帖子而完全错过了时间。该怎么办呢？我们希望我们已经想到的东西:通过热动力学的方式，利用饮料的热量来冷却自己。对于参加 2018 年 Hackaday 奖的[Scott Clandinin]，[他希望从饮料中利用足够的热能来驱动风扇，风扇将吹过杯子的顶部](https://hackaday.io/project/96252)。

[Scott]让一个朋友把一块厚铜板锻造成直角。垂直侧面的柔和曲线将热量从陶瓷杯中带走，并将其转移到 CPU 冷却器的散热片上。然后，只需用升压转换器提高热电发电机产生的电压。一旦他拨入这个，他想用超级电容供电，并添加一个温度传感器和一个微控制器来提醒他，他的禅时刻即将到来。我们为此干杯！
