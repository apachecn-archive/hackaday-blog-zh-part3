# 向本田阿西莫挥手告别，它是一个会向你挥手的机器人

> 原文:[https://hack aday . com/2018/07/01/wave-goodbye-to-Honda-Asimo-a-robot-that-wave-back/](https://hackaday.com/2018/07/01/wave-goodbye-to-honda-asimo-a-robot-that-would-wave-back/)

技术爱好者会回想起几年前，本田的人形机器人阿西莫似乎无处不在。除了在研究实验室的日常工作之外，Asimo 还有一份公关兼职，向所有人展示本田不仅仅是汽车和摩托车。从贸易展到电视节目，甚至游乐园和音乐厅，阿西莫的宣传日程排得很满。现在，一个退休派对可能已经准备好了，因为据报道，研究项目已经停止。

近年来，Asimo 的活动逐渐减少，因此这并不令人感到意外。本田的官方网站已经一年多没有更新了。最近媒体上的人形机器人更有可能出现在像 [DARPA 机器人挑战赛](https://hackaday.com/2015/08/23/hilarious-darpa-robots-falling-video/)这样的活动中，或者来自像 [Boston Dynamics](https://www.youtube.com/watch?v=fRj34o4hN4I) 这样的公司。此外，所需的技术已经变得足够容易让我们制造自己的两条腿机器人。因此，它的火炬已经传递下去，但阿西莫将作为一个机器人被人们记住，他开创了许多关于人形机器人如何与有血有肉的人类互动的思考。这是第一个能够识别人类挥手示意的机器人之一，并以挥手回应。

随着本田的研究团队将重点转向更少的人形外形，从 Asimo 开发的许多概念将继续存在。我们可以在他们在 CES 2018 期间发布的概念视频中看到本田的新野心(嵌入如下。)这些机器人仍然被设计成与人类一起生活和工作，但现在它们专门用于不同的领域，它们在轮子上旅行。这实际上离杰森一家的未来更近了一步，因为罗西在轮子上滚动！
