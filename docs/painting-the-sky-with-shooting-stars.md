# 用流星画天空

> 原文：<https://hackaday.com/2016/05/28/painting-the-sky-with-shooting-stars/>

日本公司 ALE 一直在研究一种新型的天空秀，人造流星，字面意思是在 40 到 50 英里(60 到 80 公里)的高度创造一场人造流星雨。在 125 英里(200 公里)范围内的任何人都可以看到这场演出，这意味着纽约市和费城或洛杉矶和圣地亚哥的人都可以观看同一场演出。恰如其名，他们称这个项目为“天空画布”。

该计划是让一颗含有大约 500 到 1000 个源粒子的卫星，用一种特殊设计的装置释放这些粒子。正如下面的视频所示，通过连续喷射粒子，而不是一次全部喷射，它们将创造出相当于流星雨的效果。这些粒子在进入大气层前将绕地球运行 1/3 的距离，形成流星群。通过使用不同的粒子材料，不同的颜色将成为可能，这个火球大炮展示了一些东西。

到目前为止，测试是在电弧加热的风洞中进行的，类似于将测试对象放在喷灯的火焰中。这些风洞，其中一个你可以在裂缝下面看到，产生超音速气流和数千度的温度。他们测试进入地球大气层的物体会经历什么，因为它们被高速与空气相互作用产生的等离子体吞没。在风洞中，ALE 产生了一个明显的-1 级。天狼星是夜空中最亮的星星，视星等为-1.5，这意味着从城市中可以看到流星(数字越小，越容易看到该物体。)

ALE 计划在 2017 年推出他们的第一款淋浴。与此同时，你可以用树莓皮和它漂亮的相机附件来捕捉大自然的流星雨。
