# 剩余 VFD 的快速和肮脏的驱动程序提示

> 原文：<https://hackaday.com/2018/01/21/quick-and-dirty-driver-tips-for-surplus-vfds/>

有时候，易贝似乎是世界上的垃圾箱，这是我们用最好的方式表达的意思。有时，廉价零件的种类令人难以置信，尤其是当零件曾经被大量订购，但已经过时。批量订购这类零件的冲动会让人无法抗拒，迟早你会发现自己有一大把旧东西，却不知道如何使用。

例子:俄罗斯过剩的七段真空荧光显示器(VFD)的盒子，[w_k_fay]不得不弄清楚如何使用。结果是[一个关于快速和肮脏的 VFD 驱动的教程](https://hackaday.io/project/29339-driving-vacuum-fluorescent-display-tubes)看起来非常方便。[w_k_fay]煞费苦心地指出，这些是将多余的 VFD 投入使用的实用技巧，而不是工程解决方案。他从描述你多余的管子开始，以防你没有引出线。一个 1.5 伏的电池将足够热阴极，而一个 9 伏的电池将打开段。VFD 可以像普通的阴极 LED 显示器一样处理，下面可以看到一个用 4026 十进制计数器驱动显像管的简单电路。他还介绍了从微控制器驱动 VFD 的挑战，并承诺完全构建一个频率计数器，其中多路复用的奥秘将得到解决。

听起来是时候储备那些多余的 VFD 并让它们发挥作用了。要获得灵感，看看这个极简的 VFD 时钟，或者 T2 将 VFD 和 Nixies 混合使用，以满足你对所有事物的渴望。
