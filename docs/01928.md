# 一本优秀的机械制图初级读本

> 原文:[https://hack aday . com/2016/03/30/an-excellent-primer-for-sketching-mechanical-drawings/](https://hackaday.com/2016/03/30/an-excellent-primer-for-sketching-mechanical-drawings/)

机械图纸是传达设计信息的一种极好的方式，尽管复杂的 3D 建模正在慢慢接管，一些公司接受文件而不是图纸，但机械图纸仍然是书面合同，可以说是具有公差和非建模特征的复杂零件。

但是如果你没有上过技术制图课(典型的是工程制图 101)，你如何学习呢？嗯，如果你有 15 分钟的时间，[这是一个很好的视频](https://www.youtube.com/watch?v=1Hm5Zyjmjac)，从经验来说，它涵盖了 101 课程的基础知识。

本课程涵盖所有基础知识，从 2D 投影、多视图绘图、等轴投影、横截面、线性尺寸标注、基本公差和替代视图。
