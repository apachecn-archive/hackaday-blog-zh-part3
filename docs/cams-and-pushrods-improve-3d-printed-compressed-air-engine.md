# 凸轮和推杆改进了 3D 打印压缩空气发动机

> 原文：<https://hackaday.com/2018/04/27/cams-and-pushrods-improve-3d-printed-compressed-air-engine/>

有些人就是不能让自己过得足够好，这通常是件好事。这款 3D 打印的压缩空气发动机就是这样，而且越来越好。

自从我们第一次报道[汤姆·斯坦顿]试图从玩具“空气猪”系列的辉煌岁月中恢复动力装置以来，设计已经发生了很大的变化，他随后[围绕](https://hackaday.com/2017/12/11/a-remote-controlled-air-plane/)建造了一架飞机。发动机很简单，有一个球阀，当安装在活塞顶部的弹簧弹出活塞时，球阀允许空气进入气缸。然而，那个春天总是困扰着[汤姆]，迫使他重新开始。他想用一个由凸轮和推杆驱动的球阀来代替球阀。这将大大增加发动机的复杂性，但好处是消除了弹簧的故障点。通过设计中的一些迭代，他能够重新定位球阀，在曲轴上添加凸轮，并使用推杆打开阀门。新设计比以前的版本好得多，听起来更像割草机，而不是 3D 打印引擎。查看下面视频中的设计过程和一些测试。
