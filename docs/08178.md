# 庆祝电路马戏钟显示多样性

> 原文：<https://hackaday.com/2018/01/03/celebrate-display-diversity-for-a-circuit-circus-clock/>

漂亮整洁的项目有很多优点，所有东西都排列整齐，看起来很漂亮。看到直线和令人愉悦的比例，说出了我们的强迫症倾向，往往能抚慰心灵，平静精神。但是无序并不是没有它的魅力，时不时地混合一下，比如用[这个混合媒体数字钟](https://hackaday.io/project/28922-circus-clock-one-to-rule-them-all)，可能是个好主意。

现在，我们知道你在想什么——又一个谢妮钟。没错，但这只是故事的一半，或者更准确地说，是六分之一。在[机身]的马戏团朋克主题时钟中，只有一个谢妮，用于显示小时部分的最低有效数字。其他数字用四个七段设备显示——一个 Numitron，一个真空荧光显示器和一个 LED 点显示器——加上一个真正的怪人，一个旧的机电显示器，每个角色都有单独的幻灯片和一个后屏幕投影仪。该项目的 RTC 部分是标准的 Arduino fare，但正如你可以想象的那样，如此多样化的显示器所需的电源非常复杂，必须提供+5 到-270 伏的电压。每台显示器都需要自己的驱动程序，这让它更像一个动物园，而不是一个马戏团。混乱的外观也很适合马戏团的主题。不过，我们真的很想知道更多关于投影仪显示屏的信息。

为你的下一个时钟制造寻找一个真正的陈述？看看[那稀有如母鸡牙齿的 NIMO 管](https://hackaday.com/2017/10/19/the-nixie-tube-killer-that-never-was/)。
