# 开放硬件在巴布亚新几内亚掌权

> 原文:[https://hack aday . com/2018/07/10/open-hardware-take-charge-in-巴布亚新几内亚/](https://hackaday.com/2018/07/10/open-hardware-takes-charge-in-papua-new-guinea/)

你可能不太想给手机充电。找个插座，插上电源，等一会儿就好了。找不到线缆或壁灯？在灯火通明的街角药店，有一大堆便宜的糖果色选择等着你。

在巴布亚新几内亚这样的第三世界国家，这种情况与现实相去甚远，在那里，生活在偏远丛林中的人们有手机覆盖，但他们必须通过将手机直接连接到廉价的太阳能电池板和旧汽车电池上来充电。

[马里乌斯·塔丘克]想要改变这一切。在他的朋友[Brian]的建议下，他设计了一个中间设备，可以接受任何输入，并通过一个低成本、可靠的降压转换器将其转换为干净的 5 伏电压。输入是一对鳄鱼夹，因此它们可以连接到汽车电池端子、裸线太阳能电池板引线或 9V 连接器。

手机对巴布亚新几内亚人民来说意义重大。它们就像是新闻、医疗建议和教育的第一世界护理包。到了晚上，它们就变成了简单而有价值的灯笼。但是这些肮脏的收费手段经常导致房屋火灾。有人会在早上出去打猎的时候把手机放在那里充电，然后回家时发现一堆灰烬。

这是一个开放的，简单的设备，最终可以挽救某人的生命，这正是我们正在寻找的项目类型。希望有一天能在易贝看到这些，我们也是。冲过休息时间，看[Marius]讨论 Brian Box 和他试图帮助的人。
