# 廉价喂猫器增强睡眠

> 原文:[https://hack aday . com/2016/12/30/cheap-cat-feeder-enhances-sleep/](https://hackaday.com/2016/12/30/cheap-cat-feeder-enhances-sleep/)

我们承认:我们有时会把事情复杂化。例如，看看【彼得·魏斯布罗德的】[自动化猫喂食器](https://nachonachoman.svbtle.com/how-to-keep-hungry-tigers-busy-in-the-early-morning)。它只不过是一个瓶子、一个伺服系统、一些零碎东西和一个 Arduino。然而，它可以让他睡懒觉，而不会被他的猫吵醒。

我们看着代码，心想，“这个东西会一直分发食物！那不是你想要的！”然后我们走近看。[Peter]使用一个普通的家用定时器在早上打开设备，让它运行一会儿，然后关闭它。你可以在下面看到这个机制的视频。

老实说，我们有复杂的感情。因为我们确信有人已经停止阅读来评论:你真的不需要一个 Arduino 来做这个。如果它在计时，这可能会使它更合理。或者让它在早晨感受阳光来进食。不过，Arduinos 很便宜(我们刚刚花了大约 3 美元买了一些微型 Pro ),而且它比驱动伺服系统的 555 更灵活。

这些年来，我们见过很多喂猫的人。其中一些[使用定制组件](http://hackaday.com/2012/04/21/an-easy-to-build-cat-feeder-driven-by-a-diy-linear-actuator/)。其他人用[你手头有的](http://hackaday.com/2015/05/12/super-simple-cat-feeder/)(包括另一个厨房定时器)。不管你怎么做，有一件事是清楚的:当外星人来观察地球上的生命时，他们无疑会得出结论:猫是主宰。
