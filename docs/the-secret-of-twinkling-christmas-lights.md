# 闪烁圣诞灯的秘密

> 原文：<https://hackaday.com/2018/01/07/the-secret-of-twinkling-christmas-lights/>

随着假期的结束，我们中的许多人都在冒着风雨取下所有的节日灯。LED 灯已经在很大程度上占领了市场，但在一些地方，你仍然可以买到经典的白炽灯泡。有些效果 led 还不能完全模仿。其中之一是“闪烁”灯组的魔力，[[Alec Watson]在技术连接视频](https://www.youtube.com/watch?v=zeOw5MZWq24)中解释了这一点。每个人都见过闪光的灯泡和变暗的琴弦。但是闪烁效应直到最近还很难描述。

典型的闪光灯使用双金属片。当灯泡的灯丝加热时，金属条弯曲，从而打开电路。然后铝带冷却并再次闭合电路。闪烁的灯光正好相反。双金属片使灯泡短路，而不是打开电路。闪烁灯组也使用了许多双金属片灯泡——通常每五个灯泡就有一个金属片。灯泡短路的结果是一组中的所有灯泡都有较高的电压。这使得整个条带随着闪烁的时间而闪烁。这就是闪烁魔法的来源。

我们想到，条带上的电压将是随机种子的一个很大来源。当然，你不得不不时地更换灯泡，但是有多少人能说他们是从一组圣诞灯中得到随机数的呢？

如果你想知道白炽灯圣诞灯怎么会爆炸而不取出整条灯带，[看看这篇关于反保险丝的文章](https://hackaday.com/2013/12/17/fixing-christmas-lights-and-shocking-yourself-silly/)。
