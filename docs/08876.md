# Gamecube Dock 的开关 Mods 任天堂与更多的任天堂

> 原文：<https://hackaday.com/2018/03/16/gamecube-dock-for-switch-mods-nintendo-with-more-nintendo/>

[Dorison Hugo]让我们了解一下他刚刚完成的一个项目，该项目不仅用更多的任天堂改进了任天堂，而且突出了一些来自与现有硬件一起工作和围绕现有硬件工作的挑战。该项目是任天堂 Switch 的 [Gamecube 坞站，配有工作 Gamecube 控制器端口。它看起来像一个游戏立方体，上面有一大块，任天堂 Switch 可以无缝对接。不仅如此，由于嵌入式适配器，原始 Gamecube 控制器可以插入端口并与开关配合工作。当开关对接时，Gamecube 顶部原来的橙色 LED 甚至会亮起。它主要是由其他模块遗留下来的零件制成的。](https://www.youtube.com/watch?v=2vJDniqV8V4)

这个项目有趣的地方不仅仅是在整个建造过程中对细节的关注，而是[多里森]用来让一切恰到好处的过程。集成现有的硬件意味着接受不受控制的设计约束，例如电路板的大小和形状、电线的长度以及插头和连接器通常不方便的位置。最重要的是，[多里森]希望这个国防部是非破坏性的，可逆的任天堂 Switch 码头本身。

为了实现这一点，船坞是用 CAD 建模和 3D 打印的。其余的模块都是使用 3D 打印的码头作为真实单元的替身完成的。由于成品不会以任何方式进行喷漆或后期处理，因此必须避免昂贵的基座和 Gamecube 外壳上的任何划痕。有很多切割不足和耐心打磨，以获得正确的切割结果。视频(嵌入在下面)一步一步地介绍了这个过程的每一个部分。将所有东西固定在一起的最后的螺丝钉必须以一个奇怪的角度进入，但最后所有东西都合适了。
