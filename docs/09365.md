# 迈克的机器狗是朝着正确方向迈出的第一步

> 原文：<https://hackaday.com/2018/05/08/mikes-robot-dog-is-a-first-step-in-the-right-direction/>

多亏了他们的腿和快速反应的平衡系统，人类几乎可以穿越任何地形。因此，如果你想要一个具有同等灵活性的机器人，腿是一个很好的选择，波士顿动力公司机器人的所有成就都证实了这一点。因此，[迈克·瑞斯比]很自然地以波斯顿动力公司的狗一样的机器人 [SpotMini](https://hackaday.com/2016/07/01/spotmini-struts-its-stuff/) 为原型[他的机器狗](https://hackaday.io/project/52956-mikes-robot-dog)。

他的 Hackaday.io 页面上的构建日志读起来很有趣。例如，他开始时腿的方向像 SpotMini 一样，但他发现当试图站立时，前腿工作正常，但后腿滑动或狗向后移动或两者都发生。他的解决方案是从他 90 年代的索尼机器狗[艾博](https://en.wikipedia.org/wiki/AIBO)身上得到启示，通过反转后腿的方向。然后，他将伺服马达升级为扭矩加倍的马达，并增加了腿部结构的强度。在下面的第一个视频中，你可以看到他的狗现在完全直立起来。

到目前为止，为了让它更像狗，他安装了谷歌的 AIY 视觉套件，它可以根据一个人微笑的程度来改变灯光的颜色，尽管我们认为摇尾巴也很好。可能性是无穷无尽的，但要一步一步来。请参见下面的第二个视频，了解视觉套件的使用演示。

[迈克]让他的机器狗参加了[黑客日机器人模块挑战赛](https://hackaday.io/submissions/prize2018_robotics/list)，这是理所当然的。他谈到了如何将腿用作其他机器人的模块，但我们认为四条腿的步行平台也可以作为一个模块。一旦他的腿能够行走，各种功能都可以添加到上面。有明显的脖子和头，机器人手臂和承载平台，所有这些都可以帮助花园，但也可以帮助病人或老人。在异想天开的一面，它可以配备[猫反措施](https://hackaday.com/2016/07/08/neural-network-targets-cats-with-a-sprinkler-system/)或充当[啤酒输送系统](http://hackaday.com/2016/06/07/rc-beer-crate-handles-rough-terrain-like-a-pro/)。你会用它做什么？
