# 最灵活的合成器是 DIY，树莓派

> 原文:[https://hack aday . com/2016/11/26/the-most-flexible-synthesizer-is-DIY-raspberry-pi/](https://hackaday.com/2016/11/26/the-most-flexible-synthesizer-is-diy-raspberry-pi/)

[Ivan Franco]给了我们他正在做的这个很棒的合成器项目。或者也许它更像一个合成器元项目:一个合成器构造集。你看， [Pryth](https://prynth.github.io/) 内部有一个运行定制发行版的树莓 Pi，其中包括用于生成声音的[超级碰撞器](https://supercollider.github.io/)、用于通信层的 [OSC](http://opensoundcontrol.org/introduction-osc) 以及一个多达 80(！)多路复用模拟输入，您可以将其连接到您想要的任何硬件。

有了盒子里的电脑——这里提到的树莓派——你可以很容易地把这个系统变成一个独立的乐器，而不用把它拴在你的笔记本电脑上。或者你可以限制它，使用 Pi 上的网络接口，为你的仪器编写新的超级对撞机程序，改变它的行为方式。当然，如果你已经是超级对撞机或覆盆子 Pi 专家，你可以直接研究 Pi。

这个系统是全新的，但是看看【伊万】已经做好的 [Mitt synth 设备](https://prynth.github.io/instruments/mitt/)，下面嵌入了一段视频。这是对一种可能的硬件配置的很好概述，也是几个超级对撞机例程的很好演示。

[Ivan]正试图为您创建一个有趣且易于使用的空容器，将您的硬件和软件合成器梦想倒入其中。一切都是开源的，除了圆周率和电位计的价格，这是一个非常便宜的设备。你不会被超级对撞机限制，我们猜测你也不会被 80 个输入限制。你能想到的最疯狂的发声器是什么？
