# R.O.B .得到一个适当的 RC 复活

> 原文：<https://hackaday.com/2018/03/03/r-o-b-gets-a-proper-rc-resurrection/>

30 多年前，任天堂的 R.O.B .登上了玩具货架，帮助开创了一个视频游戏的时代。对于我们中少数有幸拥有这些遗物之一的人来说，我们会发现 R.O.B .驱动手臂和颈部运动的内部机制只是乞求被修改。这正是[肯尼·斯托姆]所做的，安装几个连续旋转的伺服系统，给 R.O.B [一个自己的新的移动生命](https://www.ez-robot.com/Community/Forum/Thread?threadId=2440)。

最初的 R.O.B .在肩部嵌入了令人惊讶的复杂变速箱配置，用于肩部上下运动和捏手。(关于原始硬件内部的更详细调查，请看这个[拆卸。](https://www.youtube.com/watch?v=_KJK4bnokrU))这个黑客很少被记录，但从我们收集的信息来看，mobile R.O.B .使用了最初支持的所有三个现有自由度，同时还增加了连续旋转伺服的机动性。

看一下这个论坛帖子的日期，这个发现差不多有 8 年了。不过，在这里，年龄从来都不是决定性因素，因为这种黑客的纯粹古怪肯定会经受住时间的考验。看着 R.O.B .在这张桌子上占据了一个移动的位置，让我们想起了童年时代的神秘主义，那就是当我们还是孩子的时候，把这个同伴和我们的任天堂拆开包装。最后(不要脸塞！)，如果你和作者一样兴奋，有机会看到 R.O.B .用家庭制造技术重新回到你的书架上，试着打印我的 1:1 比例的 R.O.B .头部复制品。
