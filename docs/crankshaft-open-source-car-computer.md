# 曲轴:开源汽车电脑

> 原文：<https://hackaday.com/2018/02/26/crankshaft-open-source-car-computer/>

现代汽车和车头部件都是相当精巧的小玩意。但是如果你的车还是 8 轨的呢？没问题。只要拿起一个树莓 Pi 3 和一个 7 英寸的触摸屏，用[曲轴](http://getcrankshaft.com/)把它变成一个 Android 自动设置。

开源项目基于 [OpenAuto](https://github.com/f1xpl/openauto) ，而 OpenAuto 又利用了 [aasdk](https://github.com/f1xpl/aasdk) 。曲轴的优势是它是一个即插即用的分布。然而，如果你愿意，你可以从 [GitHub](https://github.com/htruong/crankshaft) 中自己构建它。

我们可以看到的唯一限制源于 Raspberry Pi 上缺少音频输入，尽管我们想知道 USB 声卡是否会解决这个问题。如果你有一个备用的 Pi 和一个屏幕，这是一个方便的项目。一个 3D 打印的 Pi 外壳和一些 LCD 支架，你可以放弃 8 声道。更不用说下面的 Pi 和所有的源代码了，这应该是非常容易破解的。

也许你会做一点仪表板手术。如果你已经有了双份晚餐，那么[可能不会很难。](https://hackaday.com/2013/01/29/adapting-the-nexus-7-for-a-double-din-car-dashboard-opening/)
