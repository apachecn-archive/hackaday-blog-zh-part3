# 海量柔性 LED 灯带显示屏像素太多？

> 原文：<https://hackaday.com/2016/03/25/massive-flexible-led-strip-display-has-too-many-pixels/>

这个[巨大的 LED 显示屏](http://ws2811softdisplay.blogspot.ca/)被组装在一个 PVC 横幅上(*可以卷起来！)*尺寸为 2 米乘 1.5 米，拥有超过 6000 像素，正如你从照片中看到的——看起来棒极了。

我们最近发表了一篇关于[多少个 LED 太多了](https://hackaday.com/2016/03/12/how-many-leds-are-too-many/)的帖子，引发了许多评论，展示了更多 LED 数量的更令人印象深刻的显示器。这只是其中之一——还能让它变得更灵活？那只是锦上添花。

为了使显示器具有灵活性，[Elektric-Junkys]定制了一个印有条纹的 PVC 横幅，以帮助他们在表面上对齐 58 条平行的 WS2811 LEDs。

 [https://www.youtube.com/embed/e-rdgB_19Fg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/e-rdgB_19Fg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



整个构建过程看起来耗时且相当昂贵，但最终结果却相当惊人。当然，随着平板高清电视价格的上涨，你可能会想，使用大规模生产的面板可能会更便宜。但那更难卷起来随身携带。
