# CRT 白内障手术

> 原文：<https://hackaday.com/2017/08/11/crt-cataract-surgery/>

回到过去的美好时光，人们通过盯着随时可能爆炸的粒子加速器来获取信息，我们喜欢这种方式，口香糖！为了防止灾难，CRT 显示器在显像管的前面安装了安全屏。几十年的使用通常会导致用于固定安全玻璃的胶水降解，从而导致可怕的“CRT 白内障”疾病

对老式终端的爱好者来说，幸运的是，[John Sutley]提出了治疗 CRT 白内障的方法。下面的视频展示了从开始到结束的简单但有些繁琐的过程。你需要遵循[John]的建议，对高压反激部分的任何存储电荷进行放电；我们从痛苦的经历中谈到这一点。将 CRT 从机箱中取出后，拆卸安全屏幕就像用热风枪熔化胶水并用油灰刀轻轻撬动一样简单。我们认为塑料工具不太可能划伤玻璃，但[约翰]成功地将它们分开，没有发生任何事故。丙酮和肘油脂清除了旧胶，修复后的 CRT 在重新组装时看起来很棒。

白内障治愈后，[约翰]说他的下一步是修复他的 [Lear Siegler ADM-3A](https://en.wikipedia.org/wiki/ADM-3A) 终端上不稳定的键盘。也许他应该看看[这个 VT220 键盘维修](http://hackaday.com/2017/08/08/diy-vt220-keyboard/)的想法。
