# 手绕无刷电机复苏接地四

> 原文：<https://hackaday.com/2017/06/06/hand-wound-brushless-motors-revive-grounded-quad/>

你正愉快地穿越蓝色的原野，躲避和穿越你最喜欢的四轴飞行器赛道的障碍。你得到了一个太靠近树枝的阴影，突然你护目镜中的图像抖动了一下，你的鸟撞到了地上。然后你闻到了烟味，你知道发生了什么——一个断裂的叶片使电机失去平衡，烧坏了定子中的一个绕组。

怎么办？一个明智的飞行员可能会把四轴飞行器送去修理台更换发动机。但是【BRADtheRipper】更喜欢抓住机会[用手重绕他烧坏的无刷电机](https://www.youtube.com/watch?v=E5HYyAbF70w)，尽管事实上新的要花 5 美元。他的方法有些疯狂，他在下面的视频中展示了这一点，但也有一些努力的理由。[布拉德]的线圈移植受体，2205 赛车发动机，最初是用不明来源的双 28AWG 磁线缠绕的。他选择用高质量的 25AWG 漆包线缠绕它，在一根更容易处理、更不易碎的导线中提供几乎相同的载流量。此外，通过改变定子每极的匝数，他能够改变电机的性能。

总的来说，这里有一堆不错的技巧，以备不时之需。如果你需要了解 BLDC 汽车的基础知识，请查阅这本入门书。或者你可能只是想开始 [3D 打印你自己的 BLDC 汽车](http://hackaday.com/2017/05/08/powerful-professional-brushless-motor-from-3d-printed-parts/)。
