# 周五黑客聊天:树莓 Pi 首席硬件工程师罗杰·桑顿

> 原文:[https://hack aday . com/2017/03/01/Friday-hack-chat-raspberry-pi-principal-hardware-engineer-Roger-Thornton/](https://hackaday.com/2017/03/01/friday-hack-chat-raspberry-pi-principal-hardware-engineer-roger-thornton/)

[![rpichat1-01](../Images/3ec1e6510567734af475b87a32bbbce4.png)](https://hackaday.com/wp-content/uploads/2017/02/rpichat1-01.png) 你听说过新的树莓派 Zero W 吗，它现在包括 WiFi 和蓝牙功能。你当然有。想知道流行设计中加入了什么吗？现在是时候询问本周的黑客聊天何时由 Raspberry Pi 的首席硬件工程师罗杰·桑顿领导了。

Raspberry Pi 诞生于 2012 年 2 月 29 日，已经经历了大量的硬件版本和修订。自始至终，硬件都是可靠且价格合理的，这并不是一件容易的事情。Roger 将讨论他的团队从概念到用户手中的整个过程。这是一个很好的机会来问你所有的问题。

黑客聊天定于 3 月 3 日星期五中午太平洋标准时间(20:00 GMT)进行。

你不必等到星期五；随时加入，你可以看到社区在谈论什么。
