# 康奈尔大学的学生支持你

> 原文：<https://hackaday.com/2017/01/08/cornell-students-have-your-back/>

背部问题是办公室职员和其他白领工作中最常见的伤害。这些伤害是长期形成的，通常是由不良姿势或不良人体工程学造成的。康奈尔大学的一些电气工程学生认识到了这个问题，并利用他们的毕业设计项目来解决这个问题。[Rohit Jha]、[Amanda Pustis]和[Erissa 伊拉尼]设计并制造了一种[姿势矫正装置](https://people.ece.cornell.edu/land/courses/ece4760/FinalProjects/f2016/alp225_eli8_rj245/alp225_eli8_rj245/alp225_eli8_rj245/Website%20Code/alp225_eli8_rj245.html)，当佩戴者的脊柱不在理想位置时，它会提醒佩戴者。

这个装置可以装进一件紧身衬衫里。传感器本身是来自 Sparkfun 的柔性传感器，可以检测偏转。然后，PIC32 微控制器读取该数据。对佩戴者的反馈是通过一个振动马达和一个带按钮的 TFT 显示器来完成的。当然，他们不只是把所有的东西都连接起来，然后就到此为止；对此有很多生物学研究。学生们努力为典型的人确定最理想的姿势，放置传感器的最佳位置，以及发送舒适用户体验的最佳反馈类型。

我们总是很兴奋地看到来自大学生的高级设计项目。他们经常突破传统思维的界限，这正是下一代工程师需要的技能。一定要看看下面的项目视频，如果你想看更多这学期的其他项目，[我们也为你准备了](http://hackaday.com/2016/12/29/pick-and-place-machine-for-candy/)。
