# 制造你自己的高温超导体

> 原文:[https://hack aday . com/2018/02/28/cook-up-your-own-高温超导体/](https://hackaday.com/2018/02/28/cook-up-your-own-high-temperature-superconductors/)

它看起来更像一个木炭团块，但在[Ben Krasnow]坩埚底部的黑色脆性东西实际上是一种超导陶瓷，当它放在液氮中时可以使磁铁悬浮起来。在本的帮助下，你也可以做一些。

可以在相对较高的液氮温度下工作的超导体，而不是超冷的液氦，在商业上很容易获得，所以如果你想让几个磁体漂浮起来，直接击中易贝会容易得多。但是，到达那里是乐趣的一半，从下面视频中充满活力的反应来看，[本]对此很感兴趣。这里讨论的超导体是钇、钡和氧化铜的混合物，仁慈的缩写是 YBCO。

制造 YBCO 最简单的方法是将氧化钇、氯化钡、碳酸钡和氧化铜一起进行多轮粉碎，然后在熔炉中加热。这在某种程度上是可行的，但[本]想要更多，所以他进行了自燃反应。将这三种成分的水溶液煮沸，会产生一种粘稠的污泥，最终以惊人的方式自燃。YBCO 残渣在窑中用氧气吹制，制成的圆盘具有超导体的所有神奇特性。视频中有很多细节，本用他的 YBCO 做的实验也非常有趣。
