# 他们让 P 掌权

> 原文:[https://hackaday.com/2016/03/26/they-put-the-p-in-power/](https://hackaday.com/2016/03/26/they-put-the-p-in-power/)

燃料电池有点像电池。两者都利用化学反应来发电。不同的是，当电池耗尽反应物时，它就没电了。在某些情况下，你可以给它充电，但通常每次充电你得到的能量会更少。另一方面，只要你不断提供燃料，燃料电池就会发电。哪种燃料？取决于电池，但最常见的是氢气或甲醇。

巴斯大学、伦敦玛丽皇后大学和布里斯托尔机器人实验室的研究人员想要使用一种不同的燃料:[尿液](http://www.sciencealert.com/this-new-urine-based-microbial-fuel-cell-puts-your-liquid-waste-to-good-use?utm_source=feedburner&utm_medium=feed&utm_campaign=Feed%3A+sciencealert-latestnews+%28ScienceAlert-Latest%29)。根据研究人员的说法，这是一种我们永远不会耗尽的资源。燃料电池是一种微生物燃料电池，这并不新鲜。突破是新电池相对便宜，使用碳布和钛丝。钛通常不是你认为便宜的东西，直到你意识到传统电池通常使用铂。

奇怪的是，将尿液转化为动力并不是一个独特的想法(正如你在下面的视频中看到的；显然布里斯托尔的家伙们已经考虑这个问题有一段时间了)。就我们而言，我们已经涵盖了[尿动力](http://hackaday.com/2010/05/20/urine-a-candidate-for-energy-independence/)和[火药生产](http://hackaday.com/2015/06/15/gunpowder-from-urine-fighting-a-gorn/)，所以在这篇文章的评论部分看到很多文字游戏，我们不会感到惊讶。
