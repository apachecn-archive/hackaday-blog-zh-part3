# 蒸汽动力的树莓派零没有得到任何更多的蒸汽朋克

> 原文:[https://hackaday . com/2016/03/17/steam-powered-raspberry-pi-zero-donts-get-any-more-steam punk/](https://hackaday.com/2016/03/17/steam-powered-raspberry-pi-zero-doesnt-get-any-more-steampunk/)

蒸汽朋克通常包括在你的笔记本电脑上粘几个旧阀门，骑一便士硬币，但[Alexzpro]明白真正的事情:他刚刚创造了一个[蒸汽动力的树莓派零](http://elektronikforumet.com/forum/viewtopic.php?f=2&t=81764) ( [翻译](https://translate.google.com/translate?sl=auto&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Felektronikforumet.com%2Fforum%2Fviewtopic.php%3Ff%3D2%26t%3D81764&edit-text=))。

他的装置有点杂乱无章，但很实用这是一种老式发电技术的倒退，理应得到蒸汽朋克的绰号。蒸汽锅炉驱动汽轮机，汽轮机带动电机，产生电能。这将馈入一个调节器和一组电容器，将电压平滑到 5 伏，为 Pi 供电。

这并不十分有效，但使用两个丙烷喷灯运行蒸汽锅炉确实让我们咧嘴笑了。通常我们看到人们试图走相反的方向，用可再生能源为他们的项目供电。我们也可以欣赏这一点，这肯定不是我们第一次看到树莓皮通过电力燃烧而没有明显的收益。
