# 经典汤米玩具得到爱改头换面

> 原文:[https://hack aday . com/2017/12/04/classic-tomy-toy-gets-aiy-makover/](https://hackaday.com/2017/12/04/classic-tomy-toy-gets-aiy-makover/)

几个月前，树莓皮杂志赠送了一个硬件，谷歌 AIY 语音控制套件。订户都收到了一本，但像往常一样，易贝的黄牛把店内所有的书都扫光了，只有极少数幸运的发烧友得到了自己的一套。

[Circuitbeard]是这些沮丧的 Pi 拥有者中的一员，他决定制作自己的工具包。由于一个硬纸板箱缺乏风格，他决定用 20 世纪 80 年代的“我的钱先生玩具新奇银行”的外壳来做这个“T2”。里面有一个树莓派 Zero W 和一个音频 pHat，有一个伺服系统来操作头部，还有一个微型开关连接到玩具的手臂上作为触发器。

运行一切的 Python 代码都包含在文章中，他还发布了一个设备运行的视频，我们把它放在了休息区的下面。
