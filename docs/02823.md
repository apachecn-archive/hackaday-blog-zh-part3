# 可怕的微距钻机拍出好照片

> 原文：<https://hackaday.com/2016/07/01/horrible-macro-rig-makes-good-photos/>

我们喜欢这样可怕的黑客。这是一个透镜和一圈发光二极管，贴在手机上。通过鳄鱼夹供电，也绑在手机上。这里没有什么专业的东西——我们可以想出一百万种方法来调整这个食谱。但是布丁的味道才是最好的证明。

这次攻击的[源被删除了](https://www.reddit.com/r/techsupportmacgyver/comments/4qcuuy/diy_pcb_inspection_camera_no_money_spend/)，但是上面的图片是你真正需要重新创建的。众所周知，好的摄影秘诀是好的照明。对于微距摄影来说，阴影总是出现在你不想看到的地方。环形灯通过从镜头周围的各个角度照亮场景来消除它们。什么是环形灯？一圈灯。

除非你尝试过，否则你可能不知道的是，任何旧的放大镜都可以很好地从普通相机中获得近距离的微距照片。镜头越好，效果自然也就越好，但你会震惊于仅仅用一片廉价的透镜丙烯酸树脂能走多远。

加上胶带就大功告成了。或者你*可能*把事情带到[的另一个极端](http://hackaday.com/2011/11/14/insane-macro-photography-rig/)。
