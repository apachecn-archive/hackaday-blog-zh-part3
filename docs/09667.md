# 电动打火机的闪电发生器

> 原文：<https://hackaday.com/2018/06/05/lightning-generator-from-electric-lighter/>

产生高电压并不难。如果你是一家电力公司，拥有数百万美元的资金和一个安装变压器的变电站，一个像样的变压器会很容易让你达到 100 千伏。如果你想超过这一点，那么事情开始变得困难，大多数人倾向于在其他地方寻找高电压，如电压倍增器。

这些设备只使用电容器和二极管，来自[等离子频道]的[Jay]向我们展示了如何构建一个小型桌面版本的电压倍增器，可以产生近 70 kV 的电压。这足以引发一个巨大的火花，它只由电子打火机中的可充电电池供电。在某种程度上，它们也比变压器便宜，因为它们需要更少的绝缘材料，更少的铜和铁。电压倍增器分阶段工作，每一级都将电压提升到高于前一级的临界水平，类似于[马克思发生器](https://en.wikipedia.org/wiki/Marx_generator)。

实验室使用类似的设计来模拟雷击，可以产生数百万伏的电压。它们是一种产生巨大电压脉冲和研究各种事物的经济有效的方式，从闪电对各种设备的影响到在聚变能测试中产生 X 射线。我们甚至看到它们[被用于激光](https://hackaday.com/2017/12/19/home-brew-ruby-laser-packs-a-wallop/)。
