# 阿米加 2000 紧急维修

> 原文：<https://hackaday.com/2018/07/08/amiga-2000-emergency-repair/>

大公司花了小笔钱来确保他们的电脑保持运行，并且在紧急情况下可以快速修复。不过，你不会想到对 Amiga 2000 进行紧急维修。[RETR-奥-马特]买了一台 Amiga 2000，可以启动，但已知主板上有一个泄漏的电池。他想赶在漏液造成严重损害之前[更换电池。你可以在下面的视频中看到这一切。](https://www.youtube.com/watch?v=uJVkKIRVWfI)

这台电脑在 32 年的使用寿命中看起来很少使用，甚至在外壳脱落时也是如此。不过，电池腐蚀很明显。甚至固定主板的螺栓也明显被泄漏的电池腐蚀，导致它非常难以拆卸。

电池泄漏也使电池的脱焊成为一个挑战。几个芯片和插座——包括 CPU——受到了影响，所以它们必须被移除。您可以看到一个很好的“旧螺丝刀技巧”的演示，如果您只使用过 SMD 芯片，这可能会让您大开眼界。

即使你不太关心 Amiga 2000，看到像这样的旧电脑内部并注意到它与现代设计的差异和相似之处也是很有趣的。该视频既是一个修复故事，也是一个撕裂。如果你不得不从任何一个设备上拆下一个漏液的电池，这也可能是有用的。

当[RETR-奥-马特]提到他用来中和电池腐蚀剂的液体时，音频有点难以理解。这是醋。他把这种腐蚀剂称为“电池酸”，但事实上，从那种电池来看，它是一种碱。根据金霸王的说法:

> 遵循以下建议在电池泄漏后进行清理:在通风良好的地方工作。戴上家用手套和眼镜。使用牙刷或棉签，清除电触点的电池漏电。在试用新电池之前，请确保电子设备完全干燥。
> 
> 要清洁以下类型的电池(碱性电池、镍镉电池和镍氢电池)的任何泄漏，请使用一加仑水中的一汤匙硼酸或等量稀释醋或柠檬汁与水的混合物(50/50 比例)。

虽然金霸王没有提到，但[RETR-O-垫]先用水清洗，然后用酒精清洗(你应该使用 90%或更高的酒精)。如果我们有蒸馏水的话，我们会做同样的事情，但是使用去离子水。

原来视频只是第一部分，因为他还没有做更换插座等 PCB 返工。然而，至少他阻止了大部分伤害的继续，我们也清楚地看到了一辆非常干净的 Amiga 2000 的内部。

我们必须向 Amiga 的设计者和制造者脱帽致敬。这些小盒子的使用寿命很长。我们认为这就是为什么人们[仍然在为他们](https://hackaday.com/2016/04/18/bootstrapping-an-amiga-2000-graphics-card-because-vintage-is-pricey/)制作新卡。z
