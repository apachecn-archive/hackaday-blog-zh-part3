# 3D 打印让电子产品唾手可得

> 原文：<https://hackaday.com/2017/01/26/3d-printing-makes-electronics-a-snap/>

自从有了电子学，人们就一直在寻找一种方法，让学生和业余爱好者不费吹灰之力就能构建项目。带有 Fahnestock 夹子的板子可能是第一次尝试。如今，它更多的是无处不在的无焊试验板。在这两者之间，我们已经看到了铜管件和橡皮筋，安装在磁铁上的组件，用于固定它们并进行连接，以及其他更不可能的方案。几年前，出现了一种新方法:Snap 电路。这个名字几乎说明了一切。基板上有不同组件的安装孔。所有组件通过一个常见的卡扣进行电气连接和机械连接，就像你在衣服上看到的一样。甚至电线也是两端都有搭扣的小段。

任何类似这样的系统都有一个问题，那就是如何集成定制组件。当然，有了快照，这并不是很难，但[查克·埃莱比克]在 TinkerCad 上发挥了创造力，研究出了如何为系统 3D 打印定制模块。你可以在下面看到他的视频。

该系统使用 12 毫米的快照，尽管[Chuck]没有这样做，但你也可以 3D 打印自己的基板，如果你想这样做，就不要使用任何原始的快照电路设备。他不得不使用砂轮将搭扣修成双面，但除此之外，这相当简单。他用记号笔在部件上做了标记，但我们可能已经在塑料上压印或雕刻了符号和指示符。

您可能对使用 Snap Circuits 不感兴趣，但它确实是一个很好的实际例子，说明了如何使用 TinkerCad 来构建一个与现有器件相匹配的器件。你可以在 [Thingiverse](https://www.thingiverse.com/thing:2049512) 上找到最终结果文件。

如果你喜欢更普通的多线连接器，[请选择 Pure](https://hackaday.com/2016/12/25/pure-modules-aim-to-make-prototyping-easier/) 。或者，如果你是 3D 打印，你可以得到一些导电涂料，只是[做整个板](https://hackaday.com/2013/12/13/3d-printed-solderless-circuits/)。
