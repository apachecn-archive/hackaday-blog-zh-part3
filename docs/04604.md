# 激光切割姜饼拖车回家

> 原文：<https://hackaday.com/2017/01/03/laser-cut-gingerbread-trailer-home/>

啊，假日姜饼屋。传统的——虽然乏味——款待；任务是创造，美味是拆除，那么为什么不尝试应用一些制造者的技能来使建造工作变得更容易呢？[威廉·奥斯曼]决定尝试两种非正统的方法来制作姜饼；首先，他选择建造一个姜饼移动房屋。其次，他用激光切割机切割出碎片[。](http://www.williamosman.com/2016/12/laser-cut-gingerbread-trailer.html)

在完成烘烤姜饼的繁重任务后，[Osman]在 SolidWorks 中制作了拖车模型，并开始用他自制的 80W 激光切割机切割。两次。一定要仔细检查你做的任何激光切割的原始位置，以免你破坏你的材料。此外——尽管在任何 CAD 程序中制作食物模型时这可能特别困难——一定要考虑到你的材料的厚度，否则你的手会有很多修剪。至少姜饼很容易切。

热熔胶和皇家糖霜将这些部分固定在一起，以及一些最终细节的即兴创作，从某种角度来看，这是一幅完美的假日场景。

如果你尝试用激光切割你的食物，不要放在冰箱里，否则你会发现所有的东西都有烧焦的味道。尽管如此，姜饼创作中令人眼花缭乱的复杂性的潜力使得这对于任何莫邪人来说都是一个有吸引力的选择。
