# 一张简单的 POV 名片

> 原文:[https://hackaday.com/2018/03/09/a-simple-pov-business-card/](https://hackaday.com/2018/03/09/a-simple-pov-business-card/)

仔细想想，名片是过去的一个奇怪的幸存者。当一名穿着《广告狂人》风格西装的推销员走出他的 Studebaker，走过满是打字员的房间，来到老板用木板装饰的办公室时，他会理所当然地递上一张名片。它会被归档在名片夹里。

如今，让你的卡片从打印店的特价商品中脱颖而出已经成为一种艺术形式。在我们的社区中，这延伸到了集成电子设备的卡，例如[这款卡带有由来自【James Cochrane】](https://www.youtube.com/watch?v=GlTnAOosOy8)的 ATtiny 驱动的视觉暂留显示器，如下面的视频所示。这绝不是第一张这样的卡片，但他带我们详细了解了它的设计和构造，这使得下面的视频值得一看。例如，如果你从未制作过墨粉转印印刷电路板，你可以看看他是如何制作的。

他指出，虽然 POV 旋转器只需要向一个方向显示，但卡片必须前后摆动。因此，它需要改变其显示方向，并需要一个倾斜传感器来激活它。他的构建方法使用通孔元件，但是是表面贴装，因为它们被焊接到电路板的顶面。结果是一个相当有吸引力的 POV 卡，也许你不会把它分发给所有人，但也许这不是重点。
