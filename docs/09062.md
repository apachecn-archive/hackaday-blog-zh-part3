# 汽水罐灯精确定位你的兴趣

> 原文：<https://hackaday.com/2018/04/05/soda-can-lamp-pinpoints-your-interests/>

我们之前已经说过无数次了:3D 打印会拓展你的视野。打印得越多，就越会想到可以打印的内容以及在构建项目的过程中使用打印的新方法。【AHNT】对这种现象了如指掌，因为[他想到了一个办法，用汽水罐做画布，做可定制的像素艺术灯罩](https://www.myminifactory.com/object/3d-print-individualized-soda-can-lamp-with-led-or-candle-56421)。

[AHNT]设计了一种可印刷的套筒，完全适合 250 毫升的罐子。它为用医用针戳小孔提供了一个坚固的网格，并且可以以任何可以想象的方式无限重复使用。他创造了两种不同的可印刷基底来照亮灯:一种是用来装许愿蜡烛的，另一种是为带摇臂开关和 12 VDC 桶形插孔的 LED 条形电路制作的。我们认为使用 RGB LED 并不需要太多——一个小饰品或一个宝石肯定会适合底座。

在广告之后的视频中，[AHNT]讲述了通过干净地移除盖子来准备罐子，他通过锉平顶部边缘直到各层分离。他还讨论了几种去除油漆的方法，并指出喷砂效果最好。

不需要另一盏灯？你可以用那个空汽水罐做无数的事情。你可以做一个[特雷门琴](https://hackaday.com/2014/06/01/soda-can-theremin-made-in-minutes/)，或者一个[电池](https://hackaday.com/2013/09/30/diy-soda-can-battery/)，甚至一个[宝箱](https://hackaday.com/2011/01/21/making-boxes-from-soda-cans/)。切开它，做一个[焊料模板](https://hackaday.com/2013/02/04/diy-solder-stencils-from-soda-cans/)。或者完全做点别的，然后[给我们发个提示](https://hackaday.com/submit-a-tip/)。
