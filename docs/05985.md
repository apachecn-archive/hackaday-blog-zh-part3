# Hackaday 奖参赛作品:模块化开源 AV 接收器

> 原文：<https://hackaday.com/2017/05/30/hackaday-prize-entry-a-modular-open-source-av-receiver/>

Hi-Fi 几十年来变化不大。好吧，我们承认这是一个有争议的说法，当然你家的高保真音响这些年来发生了巨大的变化。例如，以前你可能有一个转盘和一个卡式录音机，现在你可能有一个流媒体播放器和一个环绕声处理器。

但仍然可以肯定地说，高保真复制在几十年里没有太大变化。你仍然可以将最新的音频源连接到几十年前制造的放大器和扬声器上，你仍然可以享受美妙的声音。

但事实并非如此，如果你买了一个内置放大器和处理功能的 AV 接收器，而不是传统的放大器。这是消费电子世界的一个快速变化的角落，设备在其界面和功能变得过时之前的寿命通常可以在短短几年内测量出来。

对(安德鲁·博林)来说，这没什么意义。他的解决方案有一些优点，他生产了一个模块化的开源 AV 处理器，其重点是可升级性，以跟上未来的发展，而不是向用户呈现一个黑盒，随着时间的推移，有一天会变得无用。

他的设计围绕着一个背板，它接受子卡来实现单独的功能，还有一个 Raspberry Pi 来完成繁重的计算。到目前为止，他已经完成了一个概念验证，可以接收 HDMI 音频，并将 S/PDIF 音频输出到他的 DAC，但计划正在进行中。我们可以看到这将成为一个非常有用的开源家庭娱乐系统的中心。

如果你做了一个，请记得用[我们自己的声音改善配件](http://hackaday.com/2017/04/01/introducing-the-hackaday-passive-aligned-ferrite-active-quantum-crystal-nanoparticle-reference-sticker/)来增强它。
