# 国产金属制动器

> 原文:[https://hackaday.com/2016/10/02/home-made-metal-brake/](https://hackaday.com/2016/10/02/home-made-metal-brake/)

有时，适当使用武力是解决问题的必要行动。也许不太雅观，但是如果没有工具，精确地弯曲一块金属是很困难的。也就是说，当制造商面临问题时，很快就会找到解决方案；而且——如果你没有像 YouTuber[makjosher]——[那样的金属刹车，你自己造一个](https://www.youtube.com/watch?v=qOdRej3579M)很快就能完成。

根据大量的网上资料，makjosher 的刹车是由 1/8 英寸的钢条和 1/8 英寸的角钢制成的。角铁固定在一个 3/4 英寸的木制安装板上。在只用钢锯切割所有这些金属的过程中，makjosher 表现出了坚韧不拔的精神，他在钢上刻出了槽来安装铰链，这些铰链原本与木头齐平。他姗姗来迟地意识到，他们需要与弯曲的表面齐平。这导致了一些回溯和重新切割。[Makjosher]然后用螺丝将旋转部件固定在木头底座上。一个盒状的管子作为把手。一层油漆完成了这个项目，并为这个制造商的工具包增加了另一个工具。
