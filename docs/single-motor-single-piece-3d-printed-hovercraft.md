# 单马达，单件 3D 打印气垫船

> 原文：<https://hackaday.com/2018/03/08/single-motor-single-piece-3d-printed-hovercraft/>

钢筋混凝土气垫船提供了各种各样的设计选择，使他们成为有趣的探索项目。有双马达的，其中一个马达提供升力，而另一个提供推力。为了转向，推力马达可以旋转，或者你可以在它后面放一个方向舵。还有单电机的，一个电机做所有的工作。在这种情况下，来自马达叶片的气流必须以某种方式转向气垫船下方，同时也被引导出后部并转向。

[汤姆·斯坦顿]决定制作一艘单引擎气垫船，只用一个 3D 打印件作为主要结构。他的目标是让它尽可能简单、轻便、便宜。来自叶片的一些空气通过印刷在结构中的管道被引导到下侧，而剩余的空气通过转向舵向后流动。他甚至设法在方向舵的伺服系统和马达支架之间共用了一个螺栓。另一个目标是不需要印刷的支撑结构，尽管他确实得到了一些线，他很容易用热风枪清理它们。

从最初的测试中，他发现它转向不好。他怀疑方向舵没有将空气重新定向到足够的横向角度。他想出的解决方案相当巧妙，改用楔形舵。在下面的视频中，他给出了两个方向舵的并排比较，显示了空气应该改变方向的角度的巨大差异，进一步的测试证明它现在转向很好。

他在下面的视频中攻击的另一个问题是气垫船倾向一侧。他通过对裙子的一些迭代变化解决了这个问题，但我们将让您观看视频了解细节。组装的简易性和他在最后展示的 8 字形漂移过程表明，他成功地实现了他的设计目标。

这是我们报道的第一艘使用单个马达的 3D 打印气垫船，但我们以前见过一艘非 3D 打印的气垫船，这是一艘用泡沫板制成的[高速气垫船。当你在这里的时候，看看这个](https://hackaday.com/2013/09/04/screaming-fast-rc-hovercraft/)[玻璃纤维增强泡沫塑料，看起来就像蝙蝠侠电影](https://hackaday.com/2015/01/28/diy-rc-hovercraft-makes-batman-action-figure-envious/)。
