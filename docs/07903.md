# 用蓝牙为您的骰子增添趣味

> 原文：<https://hackaday.com/2017/12/10/spice-up-your-dice-with-bluetooth/>

有很多项目用电子版骰子取代了常规的棋盘游戏骰子，将数字功能带入了现实世界。然而[Jean]却反其道而行之，用他的装有蓝牙的电子骰子将现实世界带入数字世界。

这些骰子是围绕一个 Simblee 模块构建的，该模块在单个芯片上包含蓝牙 le 堆栈和天线以及 ARM Cortex-M0。添加一个用于侧面检测的加速度计和一束指示检测到的侧面的 led，[Jean]将所有这些都放在包裹电池的柔性 PCB 上，并放入一个 3D 打印的外壳中，该外壳仅比标准芯片稍大。

虽然它们将作为普通骰子的简单 LED 灯替代品，但它们最大的价值显然是增加了蓝牙功能。在休息后的项目介绍视频中，[Jean]展示了一个 Yahtzee 的概念验证游戏，在他的手机上显示掷骰子的数值。更进一步，他还演示了将特殊目的和自定义行为映射到选定骰子的场景，并谈到了他对未来的其他想法。

在看到骰子的内部之后，很显然，获得一个蓝牙驱动的 D20 将不幸地在一段时间内仍然是一个梦想——当然，除非你把这个巨大的作为尺寸的灵感。
