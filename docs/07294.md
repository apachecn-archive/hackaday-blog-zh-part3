# 摇滚 Playmobil 婚礼

> 原文：<https://hackaday.com/2017/10/04/rocking-playmobil-wedding/>

我们中的许多人都把自己的制作/黑客/建筑技能作为帮助朋友和家人的工具。[ Boris Werner]也不例外，他开始用 Playmobil 人物和零件为一对即将结婚的朋友打造一个[音乐节舞台。微型表演者是成形家庭的 1/24 比例模型。新娘和新郎弹着吉他，唱着歌，小鼓手在一旁打鼓。](http://boriswerner.eu/tinkering-tuesday-playmobil-stage-introduction/)

把孩子们的玩具变成值得结婚的礼物并不容易，但鲍里斯·沃纳所使用的细节水平是我们都可以借鉴的。休息之后的视频很好地展示了有多少很酷的同步灯光功能可以塞进一个真实表演风格的小舞台，并且经常使用真正的 Playmobil 部件。 自动化是由 MOSFET 控制的舞台照明 led、幕布后可寻址的光环、带步进电机的迪斯科球和音乐混合而成，所有这些都由 Arduino 控制。

除非你是某种 Playmobil 纯粹主义者，否则这比任何开箱即用的东西都酷。这是在 Hackaday 上第一次提到 Playmobil，但[的微型汽车](http://hackaday.com/?s=miniatures)很难像这个[类似比例的太空轿车](http://hackaday.com/2010/01/11/amazing-chassis-hacks/)一样成为新主题。
