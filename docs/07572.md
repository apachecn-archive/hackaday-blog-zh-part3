# 南瓜钢琴承诺一个葫芦时间

> 原文：<https://hackaday.com/2017/10/31/pumpkin-piano-promises-a-gourd-time/>

秋天——这是一年中落叶飘零的时候，万圣节，以及你所珍视的一切的南瓜版本。在这种情况下，这不是一杯拿铁——这是[【罗伯特·沃特曼】的南瓜钢琴。](https://github.com/robertvorthman/capacitive-piano)

[Robert]采用了一种简单的构建方法，将 Raspberry Pi 作为运营的主干投入使用。这与 MPR121 的 Adafruit 分线板相结合，MPR 121 是一种提供 12 个电容触摸敏感输入的芯片。这些都是连接到丰富的产品，构成了这个有趣的假日黑客钢琴键。[Robert]使用一些 Python 代码与 [fluidsynth、](http://www.fluidsynth.org/)一个使用 Soundfont 文件创建不同声音的软件合成器对话。当每种蔬菜被触发时，它都被一些闪烁的新像素包裹着。

这个版本对于任何秋季聚会来说都是一个很好的派对作品，而且[罗伯特]已经做了一个很好的工作，收集了报道中需要的所有硬件和软件。对于另一个以蔬菜为基础的管弦乐队，看看去年的竖琴葫芦。
