# 用于机器人等的可变刚度关节

> 原文:[https://hack aday . com/2016/02/25/机器人变刚度关节-及更多/](https://hackaday.com/2016/02/25/variable-stiffness-joints-for-robots-and-more/)

人体有许多我们通常认为理所当然的神奇能力。其中一个更微妙的是你的关节刚度的变化。用专业术语来说，刚度是指抵抗载荷的能力。例如，微妙地操作艺术家的画笔不需要太多的负载阻力，但需要精细的控制。然而，这位艺术家可能会拿起一个关节更硬的保龄球(通常控制不太好)。

[克里斯托弗·丘吉尔]和一些同事有了一种可以快速改变刚度的新型机械装置。该设备可以应用于机器人和其他设备。它还可以传递或减弱振动，因为非刚性关节不像刚性关节那样容易传递振动。

下面的视频展示了这项新发明背后的基本理念。其概念是将正刚度和负刚度结合起来，使净刚度平衡为零，然后进行细微的改变，以产生所需的刚度。可变刚度系统并不新鲜。例如，你汽车的减震器在静止时是僵硬的，但在运动时是有弹性的。然而，新系统可以在几毫秒内将刚度改变 100 倍。
