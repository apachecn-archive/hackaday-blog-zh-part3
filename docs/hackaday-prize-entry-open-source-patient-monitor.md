# Hackaday 奖参赛作品:开源患者监护仪

> 原文：<https://hackaday.com/2017/07/22/hackaday-prize-entry-open-source-patient-monitor/>

生命体征监测仪通常出现在发达国家；对于不太富裕的社区来说，它们的成本太高了。HealthyPi 项目旨在通过开发一种便宜但精确的监视器来改变这种情况，这种监视器使用了树莓皮、镶有传感器的定制帽子和触摸屏。由此产生的监视器可以用于医疗专业人员以及学生和私人研究人员。

[Ashwin K Whitchurch]和他的团队创造了 HealthyPi，这是一个树莓 Pi 帽子，包括一个用作脉搏血氧计前端的 AFE4490 芯片，一个解释 ECG 和呼吸数据的模数转换器，以及一个 MAX30205 体温传感器。帽子有自己的微控制器，ATSAMD21 Cortex M0+,也可以用 Arduino Zero bootloader 加载。

该项目能够监测患者的脉搏、呼吸、体温和所有其他生命体征，这些都是由其他“医疗级”生命体征监测仪以极低的成本测量的。这是一项大众化的技术，而且[Ashwin]已经在大众供应上有一些可用的硬件[。](https://www.crowdsupply.com/protocentral/healthypi-v3)

[在项目页面了解更多关于 HealthyPi](https://hackaday.io/project/25380-connected-health-open-source-iot-patient-monitor) 的信息，或者[从 GitHub 下载代码](https://github.com/Protocentral/protocentral-healthypi-v3)。
