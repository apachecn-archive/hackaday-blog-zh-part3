# nematoduino:Arduino 上的蛔虫神经模型

> 原文：<https://hackaday.com/2017/10/13/nematoduino-a-roundworm-neural-model-on-an-arduino/>

当谈到建立一个神经网络来模拟复杂的行为时，Arduino 并不是第一个想到的平台。但是当你的目标是[模拟只有少量神经元的有机体的行为](https://github.com/nategri/nematoduino)时，Arduino 带来的限制就开始有意义了。

它可能是你从未听说过的最重要的非分节蠕虫，但*秀丽隐杆线虫*，仁慈地缩写为*秀丽隐杆线虫*，是神经生物学的重要模式生物，其整个神经系统在 2012 年被绘制出来。[Nathan Griffith]使用这个“连接体”来模拟小型线虫的行为子集，特别是朝着吸引物和远离障碍物的运动。Arduino 骑在一个小型机器人底盘上，当模型确定是时候启动虚拟蠕虫的肌肉时，它会向马达发送信号。超声波传感器代表真正蠕虫的“鼻子接触”神经元，当模型不忙于避免接触时，它会利用“趋化性”行为主动寻找食物。这个模型已经上传到 GitHub 上，[Nathan]希望它能为未来的神经机器人专家提供一个平易近人的平台。

这并不是第一次有人在计算机上模拟线虫的连接体*，但是【内森】在 Arduino 提出的限制下完成了这项工作，这是值得称赞的。*
