# 使用这款 3D 打印的自行车辅助工具在镇上四处行驶

> 原文:[https://hack aday . com/2016/05/13/boost-around-town with this-3d-printed-bicycle-assist/](https://hackaday.com/2016/05/13/boost-around-town-with-this-3d-printed-bicycle-assist/)

[MechEngineerMike]的自行车助推器只是看着的一种享受[，而且，我们确信，使用起来是一种解脱。虽然它不会让你在街上飞来飞去，但它肯定会带走一些痛苦。(只是](http://www.instructables.com/id/Bicycle-BoosterPack-a-3D-Printed-Portable-Electric/)[喜欢专业人士](http://hackaday.com/2016/04/25/cyclists-use-tiny-motors-to-cheat/)！)

设计一个可以装一辆自行车的装置是一回事。如果它能支持多个框架，那就是另一个壮举了。最重要的是，它甚至很简单。它附着在一个点上，很容易将动力传递给车轮。甚至只有一根电线可以连接，一根 RCA 电缆，来接合升压。

我们真的很喜欢[Mike]使用外转轮马达的旋转外壳作为压在轮子上的表面的巧妙方式。我们想知道一个[铸造聚氨酯橡胶轮胎](http://www.davehylands.com/Robotics/Marauder/Making-Tires/)对马达是否有帮助，或者只是帮助马达过热？

该设备的零件是 3D 打印的，非常笨重。他们应该撑得住。看看这段视频，它让[迈克]去杂货店，在那里，他可以用休息后节省下来的卡路里少买些东西。
