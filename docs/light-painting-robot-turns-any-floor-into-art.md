# 光绘机器人将任何地板变成艺术

> 原文：<https://hackaday.com/2017/03/20/light-painting-robot-turns-any-floor-into-art/>

[海绵宝宝]是艺术吗？人们会有不同的看法，但毫无疑问的是，用一个流动的光绘机器人来绘制比基尼岛最著名的土著儿子的像素映射延时画像是多么酷。

受到最近光线装饰的 Roombas 在黑暗房间中长时间曝光的趋势的启发，[Hacker House]决定更进一步，制造一个具有较少随机导航倾向的照明机器人。一个 3D 打印的框架和轮子载着一对踏步机和一个树莓皮。顶部的 8×8 新像素矩阵提供光线。该软件能够渲染简单的矢量图像和大面积的光栅图像，从而生成全色图像。在下面的视频中，你会注意到运动和光线之间的精心协调，以及火星车令人印象深刻的急转弯性能，这两者都使产生的图像如此精确。

我们之前已经报道过很多光绘视频，包括[摇动 3D 打印机](https://hackaday.com/2015/06/26/color-light-painting-with-a-3d-printer/)和[使用悬挂式绘图仪](https://hackaday.com/2012/02/27/light-painting-with-a-string-plotter/)进行绘画。但是我们以前还没有见过一个光线画家有着本质上无限的画布。我们也很想看看两个或两个以上的小家伙一起工作会有什么样的成就。
