# 哈德威诺把快乐时光的努力

> 原文：<https://hackaday.com/2016/08/15/hardwino-takes-the-effort-out-of-happy-hour/>

如今很难找到私人调酒师。当一个制造商不得不建造自己的产品时，这个世界会变成什么样？[Pierre Charlier]可以在开源鸡尾酒制造商 hard wino 面前助你一臂之力。

自动酒吧安装在一个六槽旋转饮料架上，由 Arduino Mega 控制，通过 TFT 屏幕接受饮料订单。步进电机和 L298 驱动板由 3D 打印部件支持，由标准的 12V DC 插孔供电。组装 HardWino 有点复杂，所以[Charlier]在休息后的视频中提供了一个完整的分步过程。

[Charlier]还好心地加入了他的 Arduino 代码,以进一步方便您的欢乐时光。最精彩的部分？这甚至不是最终产品。然而——这个功能原型已经可以在漫长的一天中扭转局面。无论你选择什么饮料，在[这个方便的杯垫](http://hackaday.com/2015/12/04/smart-coaster-informs-you-when-your-drink-is-ready/)的帮助下，确保它保持你想要的温度。
