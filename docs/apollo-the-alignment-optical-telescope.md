# 阿波罗:对准光学望远镜

> 原文：<https://hackaday.com/2016/09/06/apollo-the-alignment-optical-telescope/>

阿波罗计划不断提醒我们，我们不需要这么多就能完成任务。当然，使用今天的工具更容易，但是努力工作也能做到。比尔·哈马克阐述了这样一项工程:对准光学望远镜。

望远镜被用来寻找登月舱在太空中的位置，以便它的导航计算机可以进行将登月舱带回家所需的计算。它使用的技术我们已经在陆地上使用了几个世纪，并且今天仍然在太空中使用；虽然现在是用计算机视觉完成的。它被用来将飞船对准恒星。美国宇航局使用恒星作为坐标系的固定参考点，用于定位太空中的物体。但是这是如何非常精确地完成的呢？

对准光学望远镜通过测量制导计算机所需的两个未知数来做到这一点。宇航员将通过将望远镜指向建立读数所必需的大致区域来找到第一个值，然后旋转望远镜上的第一个十字线(一条水平线)，直到它接触到正确的恒星。然后调整一个环形组件，移动蚀刻在取景器上的阿基米德螺线。当螺旋接触到星星时，你可以读出第二个数值，这是由圆环旋转的距离确定的。

如果你曾经亲眼见过登月舱，你的第一印象可能会对它的简陋感到有点好笑。事实是，这种粗糙很难实现。他们需要世界上有史以来最简单、最轻、最可靠的组件。正如[Bill Hammack]在视频结尾所说，将通常使用的复杂工具分解成两个简单的表盘是一项惊人的工程成就。
