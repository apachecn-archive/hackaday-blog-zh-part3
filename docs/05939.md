# 外骨骼旨在防止老年人摔倒

> 原文:[https://hack aday . com/2017/05/17/exoskeleton-aims-to-prevent-falls-for-seniors/](https://hackaday.com/2017/05/17/exoskeleton-aims-to-prevent-falls-for-seniors/)

当我们想到外骨骼时，我们往往会想到漫画中的场景:机械套装赋予反派超级英雄力量。但也许外骨骼还有更实际的用途:例如，恢复行走能力，或者像[这样的外骨骼短裤](https://arstechnica.com/science/2017/05/quick-twisting-pelvic-robot-could-keep-you-upright-when-youre-old/)，通过在跌倒发生前检测和纠正跌倒来防止髋部骨折。

对于老年人来说，跌倒和可能导致的虚弱性损伤是残酷的生活现实，任何可能减轻它们的东西都可能是公共健康的巨大福音。跌倒通常可以归结为由于滑倒而失去平衡，无论是松散的地毯，一块冰，甚至是众所周知的香蕉皮。[由【Vito Monaco】及其同事开发的“主动骨盆矫形器”](https://www.nature.com/articles/srep46721)旨在通过对髋关节施加正确的扭矩来感知滑动并纠正它们。这种短裤的原型看起来有点笨重，还连接着外部计算机，每个臀部都有谐波驱动的电机和角度传感器，还有加速度计来检测滑倒的运动学特征。研究人员发现，迫使向前滑动的腿同时向后推动稳定的腿有助于减少摔倒的可能性。下面的视频展示了短裤在防滑跑步机上防止摔倒的效果。
