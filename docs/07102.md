# 周五黑客聊天:开源创业公司

> 原文:[https://hack aday . com/2017/09/13/Friday-hack-chat-open-source-startups/](https://hackaday.com/2017/09/13/friday-hack-chat-open-source-startups/)

如果你想创办一家公司，你会发现网上有一页又一页的建议告诉你该怎么做。如果你想创建一个开源硬件公司呢？这有点难——你不能做硬件即服务，开源的绰号会赶走投资者。

[![](../Images/a95f69a8f9e48c178b84ed185dae7db5.png)](https://hackaday.com/wp-content/uploads/2017/09/opensourcestartuplessonshackchat-01.png) 【扎克·弗雷丁】是为数不多的让开源硬件公司运转起来的创始人之一。2015 年，他开发了[神经细胞](http://www.neurotinker.com/neurobytes/)，这是一个电神经元系统，其设计方式是，如果你得到大约 200 个，你就可以复制一个扁虫的大脑。NeuroBytes】入围了 2015 年 Hackaday 奖，该团队获得了 NHS 的拨款，现在这些 PCB 神经元将于今年晚些时候上市。

在本周的 Hack Chat 中，我们将与[Zach]讨论从零开始创建一家公司并以开源方式运营所面临的挑战。本周五 Hack Chat 的主题将包括建立开源硬件公司、制造业、围绕产品建立社区以及用美元符号拼写的业务的经验。

这次聊天将在太平洋时间周五中午进行。如果你有问题要问[扎克]。这是一份电子表格，我们将从中抽取问题。
