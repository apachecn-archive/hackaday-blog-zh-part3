# 烦躁的旋转器变成了无刷电机；仍然没有用

> 原文：<https://hackaday.com/2017/12/28/fidget-spinner-becomes-a-brushless-motor-remains-useless/>

你奶奶是好意。但是当她发现某样东西只是一时流行时，它已经是旧闻了。所以，也许今年你的袜子里有一个坐立不安的旋转器。胜过煤炭。在你把它转送给你侄女之前，你可以把它改装成一辆汽车。从技术上来说，[B.Aswinth Raj]制造了一个无刷电机，虽然它不会很快飞你的四轴飞行器，但它仍然是一个很好的演示。

下面可以看到一个视频。这个想法是将磁铁放在旋转器上，并使用电磁体将能量传递到旋转器中，旋转器位于上次 3D 打印机制造遗留下来的一根螺纹杆上。霍尔效应传感器决定何时给电磁铁通电。

有刷电机使用弹簧加载的电刷将电流输送到电机线圈，并保持磁场方向正确。无刷电机的工作原理不同。有几个可行的方案，但[Raj]使用的是最常见的一个。他在转子上增加了固定的磁铁，然后使用电磁铁在正确的时间提供正确的推力。然而，实际的无刷电机可能会有不止一个线圈，控制器必须按照特定的顺序来移动转子。

如果你想看真正的马达内部，我们之前看了如何倒带。如果你想把你的旋转器换成更实用的东西，你总是可以[制造一些音乐](https://hackaday.com/2017/09/16/fidget-spinner-gets-useful-as-midi-controller/)。
