# Hackaday 奖参赛作品:绿色机器

> 原文:[https://hackaday.com/2016/06/08/the-green-machine/](https://hackaday.com/2016/06/08/the-green-machine/)

对于黑客来说，使用基本的构建模块，如 Raspberry Pi、Arduino 和用于家庭酿酒项目的各种附加盾，可以使快速原型制作变得更容易。但是我们没有看到太多真实世界的工业应用或使用这些现成电子产品建造的机器。[SlyScience]制造了[绿色机器](https://hackaday.io/project/11948-the-green-machine)，这是一种工业级自动喷漆设备，有助于对聚碳酸酯管进行一致的喷涂。

绿色机器本质上是一个线性驱动器，它可以移动喷枪穿过旋转的透明管，并将其均匀地涂上所需的颜色。这些灯管被用作滤色器，它们滑过标准的 T5、T8 或 T12 荧光灯，用于广告、特效、电影和类似应用。在这台机器问世之前的近 10 年里，这项工作都是手工完成的。用于该过程的 HPLV(高压、低容量)喷枪需要熟练的手来获得一致的结果。管子很容易损坏，也不可能清洗。[SlyScience]在旅途中解决问题——自学并解决所有软件和硬件难题。焊接钢架是这个建筑中唯一的“定制”部件。其他都是行军床。看看下面的绿色机器运行的视频，如果你有任何帮助改进构建的技巧，请提出你的意见。
