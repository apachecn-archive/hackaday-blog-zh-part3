# 打印一个祭祀磁铁方块

> 原文：<https://hackaday.com/2018/01/02/print-a-sacrificial-magnet-square/>

这是你今天的快速而肮脏的方法。有时候，你只是需要一些对你正在努力做的事情有用的东西，而你不想按规定去做。当这种情况发生时，它是一个便宜的一次性工具，符合要求。不，我们不是在谈论港口货运——我们指的是你自己制作的那些需求驱动的工具，它们可以毫不费力地完成工作。如果你真的很幸运，你可以在它们坏掉之前使用几次。

这是其中一种工具。[杰克的工作室]希望能够在不取出夹钳的情况下快速固定角焊缝，所以他想，[为什么不打印一些磁铁方块](https://www.instructables.com/id/3D-Print-Magnetic-Welding-Holders/)？[杰克]挖空三角形以节省灯丝，但这也使它比商店购买的磁铁方块有一个很好的优势:你可以用手指穿过它，然后挂在钉板上，以备下次使用。

[Jake]幸运的是口袋尺寸合适，能够将磁铁压入合适的位置，但添加一滴 CA 胶来帮助应变是值得的。他似乎忘记了上传他的各种风格的文件，但一个带倒角和磁铁袋的空心三角形应该很容易在 OpenSCAD 或 SolidWorks 中复制，他在下面的视频中使用了它。

你自己制作的廉价工具有些特别。即使你知道它不会永远存在，它也比一些廉价的、引起愤怒的小玩意或来自空气中 85%废气的地方的组合更有意义。我们非常喜欢这里由需求驱动的自建工具，以至于我们给了他们自己的小工具。
