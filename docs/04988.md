# 聚氯乙烯水下机器人

> 原文:[https://hackaday.com/2017/02/10/pvc-submersible-rov/](https://hackaday.com/2017/02/10/pvc-submersible-rov/)

[mark.brubaker.1]和他的团队决定用 PVC 管作为框架，为一个学校项目制作一个潜水器。它的背部有两个电机，提供向前的推力和转向，以及一个水平安装在 PVC 底盘中间的电机，提供向上和向下的推力。他们使用普通的马达，通过将马达插入一个装满水暖工蜡的盒子来防水。我们不确定这能在海底保持多久，但是对于学校在游泳池的项目来说，这很好。这是如何制作的说明。

构建完全是模拟的，控制器是一个带有三个开关的板，分别控制不同的电机。所以如果你想左转，你启动右边的马达。对于右边，你做相反的事情，启动左边的马达。上上下下，嗯，你明白了。如果你附近有一个游泳池、湖泊或一些水体，并且你正在和你的孩子一起寻找一个周末项目，这是一个很好的建议。这不是 Arduino 控制的机器鱼，但这是朝着这个方向迈出的第一步；你可以稍后使用框架来改进设计并添加一些电子设备。
