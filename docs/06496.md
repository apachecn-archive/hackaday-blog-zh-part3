# 微生物躲不过水视镜

> 原文：<https://hackaday.com/2017/07/18/microorganisms-cant-hide-from-droposcope/>

[水滴观测仪](https://www.instructables.com/id/DropoScope/)是一种水滴投射器，它通过投射激光穿过一滴水来工作，理想情况下是*爬满微生物的脏水*。通过正确的调整，一个亮点被投射到附近的墙上，显示出里面微小动物的放大图像。单细胞生物只显示为黑点，但像蚊子幼虫这样的较大生物显示出明确的结构和细节。

虽然在概念上很简单，只需要一个注射器和一个激光笔，但要获得有用的结果可能需要大量细致的调整。但是，多亏了英格斯，所有这些对于任何一个有激光切割机的人来说都是过去的事情了。他的激光切割夹具设计让任何人都可以轻松制作和调整自己的水滴投影仪。

如果你想看一些微生物的活动，下面嵌入的是来自[不同水滴投影仪](http://hackaday.com/2010/08/21/laser-microscope-projection/)的视频(一个在操作上相同，但没有足够幸运地受益于【英格吉斯】的设计)。)
