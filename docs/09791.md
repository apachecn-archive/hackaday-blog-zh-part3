# 机器人组装宜家家具时，桌子被翻转过来

> 原文：<https://hackaday.com/2018/06/19/tables-are-turned-as-robots-assemble-ikea-furniture/>

Hackaday 网页上充斥着用家具零件制造机器人的例子。在这个例子中，桌子被翻转，[机器人是宜家产品](https://gizmodo.com/watch-two-robots-nimbly-assemble-an-ikea-chair-1825333735)的主人。我们不会傻到认为这些机器人展开指令，互相看着对方，抓挠它们的 CPU，然后开始组装。当然，这个过程是由程序员预先决定的，但他们将钉子插入交叉构件末端的方式是一件非常人性化的事情。这让我们想起了在黑暗中寻找手机充电插座。这种行为是由于力反馈，它告诉机器人何时一个零件被正确放置，这意味着它们可以使用视觉将部件装配在一起，而不需要亚毫米精度。

所有用于制造宜家组装机的硬件都是公开的，虽然它可能超出了典型黑客的价格范围，但这是机器人成为家庭一部分的时代标志。目前，家用机器人是洗衣机、智能扬声器和 3D 打印机。十年前，这些都不是联网机器，所以如果机械臂很快加入家用机器人俱乐部，应该不会令人惊讶。你的下一个机器人项目可能会成为一个引爆点，将一类新的机器人带入家庭。

回到我们通常的 hijinks，这里有一个来自 IKEA parts 的[机械臂和一个内置在类似灯](https://hackaday.com/2017/01/04/from-ikea-lamp-to-robot-arm/)中的[投影仪。或者一台装在宜家橱柜](https://hackaday.com/2018/05/16/ikea-lamp-with-raspberry-pi-as-the-smartest-bulb-in-the-house/)里的 [3D 打印机，打造一个优雅的家用机器人。](https://hackaday.com/2013/12/18/custom-rostock-3d-printer-makes-use-of-ikea-components/)
