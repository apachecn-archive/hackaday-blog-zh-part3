# 打印你自己的细丝

> 原文:[https://hackaday.com/2018/06/02/print-your-own-filament/](https://hackaday.com/2018/06/02/print-your-own-filament/)

问问任何一个有 3D 打印机的人，他们最擅长做什么。他们可能会说“垃圾”有额外的部分，从挤出机中渗出的东西，支撑材料，没有粘在床上的部分，或者只是需要稍微调整一下的部分。无论你做什么，你都会得到很多残羹剩饭。如果你能回收所有这些东西，那就太好了，[3D 打印书呆子]看到了 FelFil Evo 细丝挤出机承诺它可以做到这一点。你可以看下面的视频。

正如你所料，该设备是一个电动螺旋挤出长丝通过热端没有什么不同，你的打印机的热端。你必须先用一包特殊材料穿过它来清理塑胶通道。之后，你可以用标准的小球或旧塑料制造灯丝。

这个装置没有马上工作。原来里面有根电线松了。之后，效果如何？观看视频，找出答案。

我们会说，它非常慢，你不能运行机器很长一段时间，以防止过热。我们也不清楚当灯丝出来时，你如何保持它的整洁。该视频也从未显示使用切碎的塑料部件，但你会认为这不会比颗粒更好。

作为一个小玩意，细丝挤压机的想法很酷。对于回收，这是一个伟大的想法。然而，由于长丝相对便宜且容易获得，任何台式长丝挤出机都必须工作良好才能赢得大量用户。

当然，我们见过[自制挤压机](https://hackaday.com/2014/04/20/filament-extruder-pumps-out-1kghour/)。其中一些[相当便宜，使用通用部件](https://hackaday.com/2013/11/01/low-cost-filament-extruder/)。所以，如果你想尝试，没有什么可以阻止你。
