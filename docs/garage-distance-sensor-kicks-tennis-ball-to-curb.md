# 车库距离传感器将网球踢到路边

> 原文：<https://hackaday.com/2017/08/07/garage-distance-sensor-kicks-tennis-ball-to-curb/>

那些有小车库的人可能熟悉把网球挂在天花板上的方法，以确保他们的车不会撞到后墙。但是，如果汽车不在车库里，挂在绳子上的网球就会挡道。为了缓解这个问题，[asaucet]发明了一个距离传感器，可以告诉他[他的车何时距离车库墙壁](http://theembeddedlab.com/tutorials/create-proximity-sensor-garage-part-1-measure-distance-hc-sr04/)最合适。

距离传感器的核心是 HC-SR04 超声波测距仪和 PIC16F88 微控制器。[asaucet]使用一组四个发光二极管来提醒司机他们离车库墙壁有多近。[asaucet]还详细介绍了如何使用 LCD 和该微控制器来设置项目，这些细节足以让任何人开始类似的项目。

虽然这不是一个[新想法](http://hackaday.com/2013/01/09/garage-parking-monitor-guides-you-in-every-time/)，但【asaucet】在设置微控制器、使用距离传感器和使用 LCD 时所涉及的细节绝对值得一看。即使没有这个确切的应用程序，你也一定会在[项目页面](http://theembeddedlab.com/tutorials/create-distance-sensor-garage-part2-proximity-meter/)上找到一些有用的信息。
