这个 Hackaday.io 项目随机数发生器在一个最伟大的项目托管网站独有的属性上工作。所有的链接在 URL 中都有一个数字和项目名称。删除项目名称，链接仍然有效。如果你想在 PCB 上放一个项目的链接，这是一个方便的伪 URL 缩略器，也是一个查看所有项目的好方法。io——您需要的只是一点 Python、Perl 或其他脚本语言

现在，[Greg]有一个 Perl 脚本正在他的一个服务器上运行(当你读到这篇文章时，它肯定已经停机了)，它选择一个随机数，并试图获取那个 Hackaday.io 项目。如果返回 404，它将再次尝试，直到成功。
