# 3D 打印外壳将伺服系统转变为优质线性致动器

> 原文：<https://hackaday.com/2016/03/31/3d-printed-case-turns-servo-into-quality-linear-actuator/>

微型伺服电机是黑客的主食。你可能会在一架遥控飞机上发现四五个，而一个六角机器人可以吸收十几个甚至更多的便宜且容易获得的设备。不幸的是，长行程线性致动器有点难以获得，所以很高兴知道你可以 [3D 打印标准微型 RC 伺服系统的线性齿轮](https://hackaday.io/project/10578-3d-printed-linear-servo-extender)并滚动自己的。

目前在修订版 2 中，[兔子罗杰]的设计不仅仅是一个快速和肮脏的解决方案。他认真考虑了他在第一次修改中观察到的问题，结果是一个健壮、强大的线性致动器。小齿轮适合修剪伺服曲柄臂，配套的齿条是坚固和僵硬的，早期的反冲问题已经得到解决。整个外壳易于组装，正如下面的视频所示，完成的致动器可以举起 300 克。

我们喜欢[Roger]的构建过程，尤其是改进设计的迭代方法。我们将继续关注它的下一步——用于超长投掷的连续旋转伺服系统？在我们等待的时候，如果你想了解一些底层机制的背景知识，你可能想看看[理查德·巴古利]最近的关于伺服系统的[初级读本](http://hackaday.com/2016/03/24/hackaday-dictionary-servo-motors/)。
