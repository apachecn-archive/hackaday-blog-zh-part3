# 数码管时钟不仅仅是一个时钟

> 原文：<https://hackaday.com/2015/09/19/nixie-tube-clock-isnt-just-a-clock/>

鉴于最近新闻报道的所有事情，[Jarek]决定是时候完成他的最新项目了。物联网最近项目激增，这个[时钟也提醒他天气](http://www.soniktech.com/nixienet/index.html)是最新添加的。此外，它还有一个额外的好处，那就是使用每个人都喜欢的显示器:谢妮电子管！

当然，对 nixies 使用高电压可能会引起恐怖，但[Jarek]在易贝发现了一种电源，它能够为电子管供电，而且花费不多。控制器是 HV5622，它可以控制多达 32 个 nixies，同时只使用微控制器上的三个引脚，如果输出引脚数量有限，这非常方便。

这个时钟还有另一个隐藏在所有电子管电线后面的设备:一个 ESP8266，为它提供网络连接。时钟连接到互联网，搜索九小时天气预报。显示屏后面有几个谢妮灯，照亮表壳的切口，指示几种不同的天气状态。这是一个非常精致的项目，因为它被装在一个漂亮的盒子里，不太可能被误认为是任何电影道具。当然，[其他谢妮项目](http://hackaday.com/2009/11/03/wozs-watch-makes-air-travelers-nervous/)没有同样令人欣慰的外观。
