# 大众多用途机器人

> 原文：<https://hackaday.com/2016/06/08/multipurpose-robot-for-the-masses/>

随着几乎所有技术的成本下降，从电子产品到电池，甚至像 3D 打印机这样的工具，建造我们大多数人以前无法企及的东西的成本突然变得非常实惠。至少,[John Choi]花大约 2000 美元制造了一个完全 DIY 的通用机器人。

好吧，所以 2000 美元并不算“便宜”，但考虑到类似的东西(如[巴克斯特](http://www.rethinkrobotics.com/baxter/))的价格高于一辆新车的价格，这意味着【约翰】将通用机器人的价格降低了一个数量级。这个机器人在功能上也不吝啬。它有一个允许它在房间里导航的平台，两个带有大量伺服系统的操纵肢体，一个允许轻松接口、测试和编程的笔记本电脑“头”，以及一个允许它轻松与任何传感器或其他硬件接口的 Arduino Mega。它也是模块化的，所以可以很容易地修复和运输，它使用开源软件和开放硬件，所以很容易建立。

这个机器人是一个令人印象深刻的作品，应该有助于将这项技术带到高端工厂和研究实验室以外的地方。他们已经展示了机器人给植物浇水、弹钢琴、捡东西以及许多其他任务。我们可以说，他们正在朝着增加使用这项技术的学生和爱好者的目标前进。不过，如果 2000 美元的价格仍然太高，还有其他方法进入机器人领域，而不是一头扎进巴克斯特式的机器人。
