# DC 马达旋转发电

> 原文：<https://hackaday.com/2017/07/21/dc-motor-whirligig-generates-power/>

众所周知，如果你转动 DC 发动机的轴，它就会发电。[Vapsvus]已经找到了一种新的方法来做到这一点，不需要与轴直接机械连接。他只是简单地用一圈绳子绑在马达罐上。[这有效地将电机转变为旋转电机](https://www.youtube.com/watch?v=yKjqnSjcsvE)。翻转马达，让绳子扭转几下，然后拉两个环。绳子松开后又卷起来，就像我们一起长大的玩具一样。

有趣的是，当像这样旋转时，马达产生可用的功率。[Vapsvus]将两个发光二极管连接到电机的导线上，以显示正在发生的情况。当电流从正极流向负极时，白色 LED 发光；当电流从负极流向正极时，红色 LED 发光。

引擎盖下发生的一切都与动力有关。当马达开始旋转时，沉重的铁转子保持静止。电力产生了。最终，反电动势产生的摩擦和扭矩也会导致转子旋转。当转子旋转的时候，马达已经在反转方向了。当电流反向流动时，会产生更多的能量。

当然，这不太实际，但我们很想看看它能走多远。加上一个超级电容器，我们打赌它会比几年前流行的磁性摇头灯更有效。

旋转木马是有用的小装置。它们不仅能让孩子们开心，你还可以把它们当成离心机。
