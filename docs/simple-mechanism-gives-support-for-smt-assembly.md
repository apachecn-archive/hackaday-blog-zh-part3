# 简单的机制支持 SMT 组装

> 原文：<https://hackaday.com/2018/04/10/simple-mechanism-gives-support-for-smt-assembly/>

由于表面贴装技术需要精细的工作，大部分工作需要克服人体的限制。几十年前的眼睛需要帮助才能看清正在发生的事情，而能够操作相对较大物体的手指需要机械辅助才能抓住微小的 SMT 元件。但真正可能出问题的是抖动，即我们在现实世界中很少注意到的不自觉的微小肌肉运动，但当我们试图在 PCB 上放置元件时，它会造成严重破坏。

为了对抗震动，你可以做两件事之一:去除人类，或者改善人类。由于无法证明抓放机器人适合前者，【Tom】选择制造[一种用于表面贴装工作的快速手动支架](https://hackaday.io/project/120507-super-simple-smt-placement)，考虑到它完全由废料制成，结果令人印象深刻。它只是一个三件式手臂，关节采用标准对接铰链；安装时，铰链销垂直于工作表面，并配有水平扶手，它将运动限制在 PCB 上方的平面内。手托上有一个小真空吸头孔，允许[Tom]拿起一个零件并将其放在电路板上——他报告说焊膏的粘性足以将 SMD 从吸头上移除。下面的视频显示了它的实际效果，但我们想知道丙烯酸扶手是否可以提供更好的可视性。

还没准备好[自己挑选和放置？](https://hackaday.com/2017/08/29/hackaday-prize-entry-a-manual-cnc-pick-and-place-machine/)这可以理解；不是每个商店都需要这样的生产规模。但是我们认为这是一个让 SMT 面向更广泛受众的好主意。
