# Hackaday 奖参赛作品:Arduino 供电的 LED 视觉暂留可充电 3D 打印坐立不安旋转器

> 原文：<https://hackaday.com/2017/07/27/hackaday-prize-entry-the-arduino-powered-led-persistence-of-vision-rechargeable-3d-printed-fidget-spinner/>

事情不得不发展到这一步。为了参加今年的 Hackaday 奖，[Sean Hodgins] [创造了一个持续视觉坐立不安的 spinner](https://hackaday.io/project/25538-pov-fidget-spinner) 。这不仅仅是一个 PoV 小动作旋转器——这是小动作旋转器技术的极致。它是可充电的，里面有一个 Arduino。外壳是 3D 打印的。它能提高士气。这是你在一个坐立不安的旋转器中想要的一切，也是最后一个坐立不安的旋转器项目。

我们以前见过电子坐立不安纺纱，但从来没有这种程度的抛光。教授编码的[坐立不安旋转器很棒，但是它不像精细坐立不安的鉴赏家所希望的那样精致。](http://hackaday.com/2017/06/23/teaching-steam-with-fidget-spinners/)[坐立不安旋转器的互联网](http://hackaday.com/2017/06/13/hackaday-prize-entry-internet-of-fidget-spinners/)同样是一项有价值的努力，甚至包括 RGB LEDs 和 WiFi，但【肖恩】的 POV 坐立不安旋转器在另一个现实平面上。这种旋转器使用可以充电的电池，甚至还有 3D 打印(烧结，甚至！)将一切都装进一个小巧紧凑的包装中的外壳。这是迄今为止，我们见过的最优雅的坐立不安旋转器，*和*它测量自己的旋转速度。没有比这更好的了。

你可以在[Sean]的 GitHub 上找到这个令人惊讶的坐立不安的旋转器[的所有来源，或者查看他在下面制作的未货币化的演示视频。](https://github.com/IdleHandsProject/POV_Fidget)
