# 我要一杯啤酒和一杯赞美酒

> 原文：<https://hackaday.com/2018/02/02/ill-have-a-beer-with-a-compliment-chaser/>

[安德鲁·麦克弗森]发现称赞，即使是不真诚的称赞，也会让接受者感觉更好。因此，他把热敏打印机和一个带 Arduino 的大按钮组装在一起，创造了一台打印赞美的机器。把一台打印赞美的机器放在哪里最好？当地的酒吧，还有哪里？

一个 Arduino Nano 克隆体连接到热敏打印机上运行节目。Nano 克隆不喜欢 9 伏电源，所以使用降压转换器将 Nano 的电压降至 5 伏，而打印机则获得全功率。在最初的试验中，打印机打印速度非常慢，需要[Andrew]一段时间来调整参数——在调整速度和加热时间后，他能够让打印机工作，而不会烧纸或花费太多时间来打印。

一旦机器开始工作，就该加一个按钮了。一个大的发光按钮被连接并粘在打印机的侧面。使用了更多的胶水(在对打印机底盘进行了一些“修改”之后)来固定电源适配器的筒形连接器。

[Andrew]决定既然他经常去他最喜欢的酒吧，他就把它建在那里。顾客可以按下按钮，在消除悲伤的同时接受赞美。他让他的一个做文案的朋友想出一些写得很好的赞美词打印出来。这款打印机大受欢迎，以至于酒保在脸书上给安德鲁发了一条信息，说了这样的话。如果你身边有一台热敏打印机，你可以使用这个[教程将它连接到互联网](https://hackaday.com/2014/01/25/electric-imp-thermal-printer/)，或者，如果你没有一台[，你可以建立自己的](https://hackaday.com/2010/08/30/arduino-based-thermal-printer/)。
