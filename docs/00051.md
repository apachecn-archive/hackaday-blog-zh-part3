# 电话范围构建使用旧光驱

> 原文:[https://hack aday . com/2015/09/10/phone-scope-build-uses-old-optical-drive/](https://hackaday.com/2015/09/10/phone-scope-build-uses-old-optical-drive/)

你可以把你的智能手机当作一个非常糟糕的示波器，这已经不是什么新闻了。你甚至可以用它作为音频信号发生器。也有很多项目允许你缓冲进出手机的信号，使这些应用程序更有用，并在某种程度上保护你的手机电路。在【loboat】的[电话示波器项目](http://www.instructables.com/id/OscilloPhone-Use-your-Smartphone-as-an-Oscilloscop/)中引起我们注意的是它的构造。

使用的附件是一个旧的 CD ROM 驱动器(尽管我们认为任何光驱都可以)。这些非常便宜，而且——如果你的车间壁橱和我们的一样——可能还是免费的。最终的构建不会赢得任何选美比赛，但对于一个自制测试设备来说，它看起来相当不错，并且需要最小的努力。

电路和电话应用程序相当普通。在之前，我们已经[覆盖了手机范围。甚至还有一个使用蓝牙的](http://hackaday.com/2012/07/14/android-oscilloscope-built-from-parts-just-laying-around/)。然而，随着廉价的数字示波器的出现，推动了二手模拟示波器的价格下降，你真的应该有一个真正的示波器。正如[大卫·琼斯]在下面的视频中指出的，你应该能花 50 美元左右买到一台非常耐用的旧示波器。跳过两个星期的星巴克，你就可以做生意了。
