# 从自动售货机上买你的巧克力或巧克力豆

> 原文:[https://hack aday . com/2017/10/12/get-your-smarties-or-MMS-from-a-vending-machine/](https://hackaday.com/2017/10/12/get-your-smarties-or-mms-from-a-vending-machine/)

有些辩论把世界分成了两半。严肃的东西:M&Ms，还是 Smarties*？是的，这两种巧克力豆可能表面上看起来很相似，但你要么是聪明的情人，要么是 M&M 的情人。不妥协。

[Maximusvo]在他的文字中明智地回避了所有关于品牌忠诚度的问题，如果不是在他的图像中的话，尽管很明显他在他的糖果自动售货机中使用的是哪种糖果。硬壳巧克力被装入一个漏斗中，从漏斗中一个彩色的瀑布被释放到一个秤上。当所需的重量已经积累，它被倾倒入一个抽屉，为饥饿的收件人。

在这一切的背后是一个 Arduino，它有一个电机来释放豆子，一个称重传感器来称重，一个液晶显示器来给出状态报告。一个马达振动斜槽，以确保他们沿着它移动，但正如我们在休息时的视频演示中看到的那样，它并没有完全成功。有一个外部蜂鸣器来指示交货，除了机器的木质结构外，秤中还有 3D 打印的部件。
