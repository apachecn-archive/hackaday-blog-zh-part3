# SLJ900 造桥机功能技术

> 原文：<https://hackaday.com/2017/11/22/functioning-technic-slj900-bridge-builder/>

乐高建造者对细节和准确性有着强烈的热情，他们重新创造了可识别的现实世界元素，如特定的汽车模型和著名的建筑。然而，技术建造者将它提升到了常规 AFOLs 所不能达到的水平:他们的模型不仅要看起来像原版，还要有同样的功能。一个典型的例子是，[Wolf Zipp]版本的大型桥梁建筑钻机。中国制造的 SLJ900 沿着桥顶滚动，增加了巨大的混凝土跨度，这种沉着只有在加速的 YouTube 视频中才能找到。尽管如此，它仍然是一个坏蛋机器人，是一个值得技术化的目标。

[Wolf]的模型长 2 米，重 10.5 公斤，由 13 个乐高马达和一个气动钻机组成，全部由一个手持控制箱运行。该设备将乐高连接器插入一个模拟的桥跨中，将其提起，移动到下一个桥墩上，然后放下到位。这个扳手本身重 2.5 公斤——这可不是泡沫聚苯乙烯！项目中有很多很酷的细节。例如，转动车轮进行横向移动的机制由一个[乐高制造的气动压缩机](https://www.youtube.com/watch?v=mSdQdTZmoU8)组成，该压缩机使气动致动器跳闸，使车轮离开地面，并允许它们转动 90 度。
