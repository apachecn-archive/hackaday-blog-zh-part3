# 音乐阴影

> 原文:[https://hackaday.com/2016/02/17/musical-shades/](https://hackaday.com/2016/02/17/musical-shades/)

在你感到厌烦之前，你可以用微控制器创建的闪烁光模式就这么多。[加勒特]显然有这种感觉，并决定在一些 LED 灯罩上制作一个音乐驱动的 LED 显示屏。该系统有三个主要元件:麦克风，前置放大器和 7 波段频谱分析仪芯片。你可以在下面的视频中看到结果。

[Garrett 的]项目跨越了三个连续的帖子，他包含了相当多的设计细节，包括一个 Javascript 计算器和一个电子表格。频谱分析仪芯片 MSGEQ7 是一种廉价的芯片，它可以读取音频信号并输出七个频段中每个频段的信号强度。该芯片没有很多引脚，因此它将输出多路复用到一个引脚。

除了硬件设计，[Garrett]还展示了他在印刷电路板上构建项目的技术。他还有一个有趣的技巧，用喷胶压住焊接模板。自从几年前我们第一次在 Maker Faire 上看到[Garrett]戴着这款眼镜以来，我们就一直是这款眼镜的忠实粉丝。我们在 Hackaday 商店还有几双[的存货。](http://store.hackaday.com/products/rgb-shades-kit-v2)

我们已经看到这些便宜的频谱分析芯片用在自行车上。他们甚至成了一些轻松笑话的[笑柄](http://hackaday.com/2014/01/23/the-butt-lamp-light-from-where-the-sun-dont-shine/)(我们将把不可避免的百威轻松笑话留给评论者)。
