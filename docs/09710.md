# 本周失败:两个转子并不比四个转子好

> 原文:[https://hack aday . com/2018/06/12/每周故障-双转子并不比四转子好/](https://hackaday.com/2018/06/12/fail-of-the-week-two-rotors-are-not-better-than-four/)

公平的警告:[帕韦·斯彼哈尔斯基]的视频主要是他在谈论他的“双直升机”有多糟糕。有几个命运多舛的无人机进行飞行测试的序列，其中大部分似乎以它做了一个合理的钻孔后螺旋钻的印象结束。我们不得不承认这是一架相当糟糕的无人机。但是，如果一个人不能从中获得一些乐趣，那么他就真的失败了，[pawez]获得了相当大的成功，至少从他一次又一次把飞船装箱的喜悦来看是这样的。

整体想法似乎很有意义，同轴道具安装在圆形 3D 打印框架的中间。安装在支柱下面的是由两个伺服系统控制的交叉叶片。叶片位于旋翼洗流中，提供俯仰和滚转控制，而偏航和推力通过改变反转螺旋桨的速度来控制。[pawe]知道这是一个粗略的空气动力学设计，并惊讶于它的表现。但是，由于地面效应限制了接近地面的滚转和俯仰控制，由于转子之间的湍流造成的推力不足，以及质心和重心相互失去平衡的趋势，所有这些都使飞机非常不稳定，难以控制。

尽管表现不佳，[pawez]仍计划制造马克 2 号双旋翼飞行器，这是一种更小的飞行器，根据他的经验做了一些改动。他在推动多旋翼极限方面毫不逊色， [3D 打印赛车四框架](https://hackaday.com/2018/01/15/3d-printing-a-better-quadcopter-frame/)和[使用 LoRa 进行超视距控制](https://hackaday.com/2018/05/05/lora-system-commands-drones-from-a-distance/)。尽管如此，我们相信他会感谢评论中的建设性批评，我们祝他下一次好运。
