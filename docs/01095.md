# 乡巴佬乐高焦点拉具

> 原文：<https://hackaday.com/2016/01/01/hillbilly-lego-focus-puller/>

用一套合适的乐高零件，几乎没有什么是你做不出来的。[Rigjob]建立了一个基于乐高的无线远程跟踪对焦系统,它将使专业系统物有所值。

现在[Rigjob]自称是一个乡巴佬，但他不仅仅是一个拿着相机的乡巴佬。他设置了乐高控制器来记住最小和最大焦点位置以及沿途标记点。控制器只是不会将镜头转到对焦范围之外，一个交互式图形会向您显示您在该范围内的位置。对于聚焦轮，他使用(请击鼓！)一个乐高越野车轮。看起来真的很舒服，很好用，实际上也挺专业的。

乐高控制器和马达中有很多技术，使这个“简单”的黑客变得简单。在引擎盖下，有一个蓝牙连接，一个带位置传感器的齿轮步进电机，一个通信协议，以及乐高控制器中的一大堆编程，使其可以拖放编程。但是对于一个长胡子的乡巴佬摄影师来说，这一切都像是儿戏。这是优秀设计的标志。太棒了，乐高。
