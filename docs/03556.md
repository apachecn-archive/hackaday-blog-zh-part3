# BuildTak、PEI 和早期采用者综合征

> 原文：<https://hackaday.com/2016/10/07/buildtak-pei-and-early-adopter-syndrome/>

我猜 Hackaday 社区的大多数成员都是大多数人认为的早期采用者。当然，我们之间存在差异，但与一般人群相比，我们可能都是合格的。我花了很多年时间来做一个早期采用者。我拥有一台电脑，一台 TiVO，一台数码相机，一台 3D 打印机，一架无人驾驶飞机，以及许多其他小玩意，直到它们变得广为人知。我已经避开了对自动平衡交通工具的狂热(我会坚持使用我的摩托车)。

当然，你知道如果你是一个早期用户，你会多付钱。毕竟，新东西是有溢价的。但是还有另一个代价:你经常有第一个，但不是最优的。我的第一台数码相机拍摄了 3.5 英寸的软盘。我的 TiVO 有一个模拟调谐器。

上周有人提醒我这一点。几年前，我建造了一台 3D 打印机。当时很多打印机没有加热成型板，所以打印 ABS 需要 rafts 和 ABS 果汁和挫折。我确保得到了一张加热床，像当时的大多数人一样，我有一个覆盖着 Kapton 的玻璃印花表面。

这对 ABS 来说效果很好，但并不完美。Aqua Net 发胶使它更好地坚持，但大平面打印仍然需要一点工作。稍加练习，还不错。我最终换成了铝床，不需要经常调平头部，但这并没有让事情变得更好，只是更容易重复。

岁月流逝，其他小玩意也在召唤。我使用打印机就像使用钻床一样。我不是每天都用它，但是当你需要它的时候，它会很方便。我不得不承认，自从 PLA 不变形后，我就越来越喜欢它了。但在休斯顿炎热的阳光下，解放军并不总是一个好的组合，所以我仍然打印了相当数量的腹肌。

前几天，我注意到一款名为 [BuildTak](https://www.buildtak.com/) 的产品。我还听说有人在 PEI 床单上打印。我决定试试 BuildTak。哇！差别真大。

我不想把这变成对 BuildTak 的评论——你可以在互联网上找到它们。我无法将其与 PEI 或任何其他更新的解决方案进行比较。我会这样说:起初我没有调整我的第一层参数-我通常打印有点低，过度挤压 smoosh 对 Kapton。当你这样做的时候，你必须接受一点点破碎的基础，但这有助于翘曲。

使用 BuildTak，打印出来的效果很好，但是很难从打印床上去除。稍微看了一下(是的，我不太会看说明书)就知道这张床太粘了，你不应该把塑料压扁。有史以来第一次，我打印的第一层足够高，看起来像所有其他层，没有过度挤压。印刷品很快就印好了，而且非常平整。

我很尴尬的发现这不是一个新产品。这对我来说是全新的——早期采用者。这些床单并不便宜，但是如果你照顾好它们，它们看起来会很耐用，所以这是一个便宜又简单的升级。顺便说一下，你仍然需要一个加热的 ABS 床，所以它不是一个非加热打印机的廉价升级。

但这不是我的重点。总的来说，小玩意变得越来越好。早期采用是有趣的，但是不要忘记跟踪技术状态，看看你可以或者应该在哪里进行改进。
