# 逆向工程使老式汽车音响的巧妙蓝牙解决方案成为可能

> 原文：<https://hackaday.com/2017/02/21/reverse-engineering-enables-slick-bluetooth-solution-for-old-car-stereo/>

我们这些更喜欢开老爷车的人，往往要在娱乐系统部门做出牺牲，才能体会到没有车贷的好处。最新的汽车拥有所有的铃铛和哨子，而美国的汽车在 iPod 革命之前就已经存在，许多汽车甚至没有辅助输入插孔。同时也是黑客的吝啬鬼经常用转换项目来弥补这一点，比如吉普收音机上的这个非常巧妙的蓝牙转换。

有很多方法可以把你最喜欢的音乐从手机传到一个旧的汽车音响上，但很少有像[帕克·迪尔曼]的项目这样完美的整合在一起。一个比他 1999 年的吉普车上的 OEM 立体声更新的售后收音机是一种选择，但这没有什么运动可言，此外，软顶车很容易买到花哨的立体声音响。[帕克]下定决心要破解原来的立体声系统，他从易贝买了一个复制的单元，这样他就可以在工作台上对它进行逆向工程。真正令人印象深刻的是[Parker]在不改变 OEM 功能的情况下集成蓝牙的方式，这需要一个定制的 PCB 来容纳音频电平转换器和输入开关。他在休息后的视频中非常彻底地记录了他的努力，但在接近结束时发出了警告。

如此多的黑客劫持了[磁带机](https://hackaday.com/2015/08/13/tricking-a-car-stereo-to-think-your-cellphone-is-a-tapedeck/)或 [CD](https://hackaday.com/2014/01/02/un-crapifying-a-car-stereo/) 输入，但由于他的侦查和构建技能，[帕克]增加了功能而没有牺牲任何东西。
