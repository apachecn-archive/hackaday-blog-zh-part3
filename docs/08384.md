# 周五黑客聊天:成为电子人

> 原文:[https://hack aday . com/2018/01/24/Friday-hack-chat-being-cyborg/](https://hackaday.com/2018/01/24/friday-hack-chat-becoming-cyborg/)

成为半机械人是什么感觉？拥有增强的听力、改善的视力和大脑的协处理器来增强你的记忆意味着什么？我们可以问戴着助听器、眼镜、手腕上绑着智能手机的人，但这很无聊。我们正在展望未来和酷型的电子人，这就是本周黑客聊天的全部内容。

我们本周黑客聊天的嘉宾将是呼唤·威尔金斯，他们在这里讨论如何成为半机械人。现在，他们正在运动一个磁性植入物，一个 NFC 植入物，并且很快将有一个北感，一个告诉你的大脑北在哪里的外感装置。

呼唤目前在多伦多，是多伦多大学的博士生，也是 Site 3 实验室的主任。他们利用空闲时间制造机器人，玩激光，思考如何修改身体，以及生物黑客技术和可穿戴技术在不久的将来会在哪里相遇。

在这次聊天中，我们将讨论作为一个电子人意味着什么。仅仅是戴隐形眼镜，换个髋关节或心脏瓣膜，还是更酷的事情？RFID 标签算吗？做胰岛素泵？我们将深入挖掘成为半机械人意味着什么，以及未来的技术将使人体能够做什么。当然，我们鼓励你提出自己的问题；把这些留在[黑客聊天事件页面](https://hackaday.io/event/34316-being-cyborg-hack-chat)上。
