# 装甲喂食器让猫和人类的战争升级

> 原文:[https://hack aday . com/2016/12/23/cat-vs-human-escalates-with-armor-plated-feeder/](https://hackaday.com/2016/12/23/cat-vs-human-escalates-with-armor-plated-feeder/)

战争通常会增加创新，因为敌对双方试图以更有效的方式杀死对方。即使是猫科动物和它们的人类仆人之间每天进行的软战争也会导致创新，也就是说，我们提供了[这种装甲自动喂猫器](http://quinndunki.com/blondihacks/?p=3023)。

[链轮 H.G. Shopcat]和她的人类[Quinn Dunki]之间的冲突始于一个精心提供的自动食物分配器。就像人类自动售货机的顾客看到一袋刚买的奶酪在分配螺旋钻的末端晃来晃去一样，[链轮]了解到当轻推时，进料器会分配一些鱼腥味的鸡块。[Quinn]开始了一个迭代设计过程来控制[Sprocket]的计划外小吃。将喂食器牢牢固定在地板上，并添加障碍物以防止她抓到分配槽——似乎没有什么能阻止这只聪明的猫的袭击。[Quinn]然后拉出所有的挡板，用 1/8″钢板和铜为进料器做了一个[链轮]安全外壳。这好像把猫放回了直狭上，看起来也不差半分。

玩笑归玩笑，[奎因]解决这个问题的方法很有启发性。仔细的观察提供了合理修改的几个周期，直到变得很清楚，只有最极端的解决方案将工作。以前这里有成吨的猫喂食器，从简单的[到复杂的](http://hackaday.com/2015/05/12/super-simple-cat-feeder/)的[，但是我们认为所有的都将成为聪明的【链轮】的猎物，如果没有一点改进的话。](http://hackaday.com/2015/11/05/automatic-cat-feeder-dispenses-noms-wants-cheezburger/)
