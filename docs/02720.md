# 这个 Arduino 控制台有 64 位图形

> 原文：<https://hackaday.com/2016/06/25/this-arduino-console-has-64-bit-graphics/>

当应用于技术规范时，数字是很奇妙的东西。以[Bobricius]' [基于 Arduino 的手持游戏机](https://hackaday.io/project/12310-diy-led-8x8-matrix-game-console-64bit)为例。对于 8×8 LED 矩阵显示器来说，它不会赢得任何奖项，但当他推动可疑规格声明的边界时，他并没有严格地说任何谎言，他半开玩笑地说图形是 64 位的。

玩笑归玩笑，这是一个使用 Arduino MCU 的 DIP 版本和定制 PCB 上的所有通孔元件的整洁构建。电力来自 CR2032 电池，它包括三个按钮和一个小型压电扬声器。他实现了一系列游戏，包括 Pong，Breakout 和 Tetris 的克隆，从下面的视频来看，它的可玩性令人惊讶。

现在你可能看着这个控制台，想知道这有什么大不了的。毕竟，有很多类似的设计可以找到，这不是什么新鲜事。当然，对于任何黑客或制造商来说，这都是一个不错的项目，但我们可以看到，对于你生活中想要尝试构建电子产品的年轻人来说，这将是一个很好的起步项目。所有通孔结构便于焊接，最后是一个整洁的游戏。

他[发布了设计过程的完整报道](http://letsmakerobots.com/node/44115)以及上面链接的 hackaday.io 页面，所以如果你想自己建造一个，没有什么可以阻止你从 Arduino 中挤出 64 位图形的好处。
