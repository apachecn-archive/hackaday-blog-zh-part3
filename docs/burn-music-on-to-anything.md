# 将音乐刻录到任何东西上！

> 原文：<https://hackaday.com/2017/03/11/burn-music-on-to-anything/>

如果一开始你没有成功，尝试，尝试，再尝试。当你的努力涉及到一台回收的电唱机、一台激光切割机和一台 He-Man 时，尤其如此。牢记这个建议，杰出的疯狂制造者[威廉·奥斯曼]设法将音乐刻录到 CD 上。

考虑到激光切割唱片的可行性是可疑的——尤其是在偷工减料的情况下——需要几次令人沮丧的测试才能最终看到结果，同时还要冒着他的激光镜头的风险。最终，[奥斯曼]的毅力得到了回报。镜头由一块 delrin 松散地固定，delrin 本身接触到一个播放音乐的扬声器。扬声器的振动导致透镜将激光的焦点振荡到能够在电唱机上播放的波长。你没有得到太多的高端音频和静态几乎淹没了音乐，但它绝对是一首歌的真正劣质的记录！

此时，黑胶发烧友们肯定要拔头发了。对于我们其他人来说，如果你读了[Jenny 的] [关于唱机的初级读本](http://hackaday.com/2017/03/03/record-players-explained-for-the-streaming-generation/)，你会认识到前置放大器(你的放大器上的“唱机”输入)是这种设置所缺少的，肯定会产生更多的听觉效果。
