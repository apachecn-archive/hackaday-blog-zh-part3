# 树莓 Pi Zero Hub 的新角度

> 原文:[https://hack aday . com/2016/02/06/new-angle-on-raspberry-pi-zero-hub/](https://hackaday.com/2016/02/06/new-angle-on-raspberry-pi-zero-hub/)

总的来说，Hackaday 的读者感叹道，“不要再来一个 Pi 零 hub 了！！!"。但(肖恩·哈金斯的)枢纽是不同的。它有一个新的角度，真的。此外，它还是 [Hackaday 和 Adafruit Pi Zero 大赛](http://hackaday.com/2016/02/01/announcing-the-adafruit-pi-zero-contest/)的参赛作品。

![1514291454445337873](../Images/99ffffcd7bd300f308a46ce1e31096f0.png)[Sean Hodgins]acute 方法与我们见过的大多数其他中心都是正交的。他把轮毂和零点成直角。集线器插入移动 USB 端口和 USB 电源端口。不需要额外的电缆或电线。[Sean]计划在他的[Kickstarter 活动结束](https://www.kickstarter.com/projects/idlehandsdev/piangle-zero-usb-power-and-4-port-hub)后在 GitHub 上发布这个设计。他为那些喜欢锡膏味道的人提供裸板。

这个项目很好地解决了在零点增加一个中心的问题。物理连接是牢固的，板通过 USB 连接器连接。电力通过集线器以 Pi 期望的方式供应，这意味着 Pi 基础构建到板载调节中的所有保护都保留在原位。这也减少了通过集线器反向供电和热插拔 USB 设备时可能出现的电涌问题。另一个简洁的功能是开槽角，便于 HDMI 端口的使用。同样，Pi 的 GPIO 引脚也没有障碍。一个缺点是集线器是 2 安培的保险丝，就像 Pi 一样。如果能为耗电的 USB 设备多留一点空间就好了。考虑到零点的使用，可能还需要 0.5 安培。

[Sean]休息后将两者合二为一。
