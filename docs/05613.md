# Hackaday 奖品入口:HID Dial

> 原文:[https://hack aday . com/2017/04/26/hack aday-prize-entry-hid-dial/](https://hackaday.com/2017/04/26/hackaday-prize-entry-hid-dial/)

微软最近推出了一些有趣的硬件，迄今为止最值得流口水的是微软 Surface Dial。这个神奇的输入设备是什么，它将彻底改变计算机上的创造性工作？基本上，它是一个 Griffin power mate——一个旋转编码器和按钮——一个在十五年里没有改变过的界面。

[K.C. Lee]认为这种简单的设备将成为一个伟大的 Hackaday 奖参赛作品，[所以他制作了一个 USB HID 多媒体拨号盘](https://hackaday.io/project/20431-hid-multimedia-dial)。这是一个旋转编码器和一个按钮。不过，这个灯亮了，使它成为了一个*游戏玩家* USB HID 多媒体拨号盘。

这种构建的电子器件基于 STM8S003，这是一种非常便宜的微控制器，在这种应用中足够好地工作。这个建筑机械部分更有趣一些；[K.C.]说不是每个人都有机会接触到花哨的 CNC 或 3D 打印设备，所以[他用塑料、金属和强力胶的碎片制作了这个模型](https://hackaday.io/project/7700-collection-of-misc-small-project/log/52476-rotary-encoder-knobcase-from-junk)。这个外壳实际上是一个旧的强力胶瓶盖，一个空的牙线线轴和一些金属。它很好用，而且比用 CAD 程序设计花费的时间更少。

现在，[K.C.]正在解决 STM8 微控制器的一些 USB 问题。一旦这些问题都过去了，他的办公桌上就会有一个非常酷、非常有用的输入设备。它可能看起来像等待回收站的零件，但它至少会像花哨的微软版本一样有用。
