# 声控壁炉是最酷的了

> 原文：<https://hackaday.com/2016/12/07/voice-activated-fireplace-is-as-cool-as-it-gets/>

一个壁炉可以给任何一个家增添一种舒适、轻松的氛围，还能增添一丝时尚气息。Redditor [hovee]看到了为他的燃气壁炉增加一些特色的机会，他把它做成声控的。看看下面的视频吧。

Google Home 和 Google Assistant 提供了语音识别组件。一个带家庭助理的覆盆子 Pi 3 做跑腿工作。一个 iTach TCP/IP-to-Contact-Closure 继电器切换壁炉，一个连接到 Google Assistant 的 IFTTT 帐户将所有这些都集中在一起。

[hovee]然后将一些粗 16/2 线从中继网络端口连接到壁炉的远程接收器电路，以实际打开它。一些自定义代码和家庭助理文件的配置是必要的，但[hovee]已经展示了他的工作，此外还有一些提示，如果你想扔在一起一个类似的设置。如果你的壁炉有一个“远程”设置，这是一个帮助，如果有文件可以找到壁炉，将有助于建造过程，这是一个双重奖励。

一旦完成，你所需要做的就是在家里的自动化奢侈品的大腿上用你最喜欢的饮料放松。只是要确保你有一个备份来关闭你的壁炉，以防你的设置像天网一样。当你这样做的时候，你可以设置你的[壁炉来节约能源](http://hackaday.com/2014/03/29/energy-saving-fireplace-thermostat/)。
