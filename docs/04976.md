# 修复水晶耳机

> 原文：<https://hackaday.com/2017/02/07/repairing-crystal-earpieces/>

如果你做水晶收音机，你可能有几个水晶耳机。名字相似有点巧合。晶体收音机中的晶体是整流器(目前最常见的是锗二极管，这是一种晶体)。水晶耳机中的水晶是一种压电声音传感器。

回到 20 世纪 60 年代，这些在廉价的晶体管收音机和助听器中相当普遍。它们的声音保真度不是很好，但它们非常灵敏，阻抗相当高，这就是它们适合晶体收音机的原因。

[Steve1001]有几个这种便宜的耳机要么不能用，要么声音输出很低。他发现根本原因通常是一个简单的问题，并分享了如何毫不费力地解决这些问题。

这些故障的根本原因通常是设备内部的铜片接触不良。[Steve1001]用小刀拆卸该装置，钻一个新孔，并添加一根额外的电线。这个过程并不复杂，但是看看它是如何完成的可能会让你在试图拆卸的时候不去撕毁它。

在当地寻找水晶耳机的来源可能很困难，但压电扬声器在主板或微波炉等发出哔哔声的东西上很常见。Hackaday 自己的[Steven Dufresne]在下面的视频中展示了如何将一个耳机转换成一个可用的水晶耳机。我们也看过他制作其他自制版本。当然，[大时代水晶收音机](https://hackaday.com/2016/09/16/high-performance-crystal-radio/)有放大器，可以使用传统的扬声器和耳机。
