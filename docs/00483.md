# 那不是月球南瓜，那是空间站

> 原文:[https://hack aday . com/2015/10/26/thas-no-moon-pumpkin-its-a-space-station/](https://hackaday.com/2015/10/26/thats-no-moon-pumpkin-its-a-space-station/)

每年，维生素 T 都会为奥斯汀及其周边的创意机构举办#ATXPumpkinChallenge。每个小组都有一个假南瓜和制作 15-60 秒视频的挑战。作为去年的卫冕冠军，[SiteGoals]必须加大赌注。所以他们发射了一个南瓜到太空。

当第一次接受挑战时，团队只需要简单的三个字就可以开始。*南瓜。英寸空间。*接下来是长达一周的疯狂，为南瓜的首次飞行做准备。

南瓜本身很简单。一个塑料南瓜灯被精心绘制成第三颗死星的样子。这使得这个项目的名字有了双重含义:“南瓜归来”。它们甚至包括绕着毁灭之球的赤道飞行的标志性宇宙飞船，毁灭之球无比强大却又脆弱得令人质疑。仅仅把南瓜发射到太空是不够的。他们内置了遥测系统和 GoPro 来记录航行。休息后留下来看看非常有趣的视频制作，设置酒吧乐队的曲调。
