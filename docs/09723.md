# 拆卸式分形天线

> 原文:[https://hack aday . com/2018/06/12/tear down-locates-fractal-antenna/](https://hackaday.com/2018/06/12/teardown-locates-fractal-antenna/)

[IMSAI Guy]拆开了一个带有无线网卡的设备，并决定调查金属罐下面是什么。下面可以看到[他检查](https://www.youtube.com/watch?v=COCSlXrqgc4)的视频。总的来说，它相当不起眼，但有趣的是它在 PCB 上使用了分形设计的天线。

你可能知道分形是“自相似”的，因为它们是由更小的相同图案组成的图案。一个老笑话是，本华·B·曼德博的 b 代表本华·B·曼德博。你可以把它想象成软件中的递归。用分形图案制成的天线具有一些不寻常的有用特性。

这些天线由[Nathan Cohen]于 1988 年开发，并于 1995 年发表，尽管对数周期天线已经在使用中，它使用了未被识别的分形特性。像对数周期天线，但不同于大多数其他天线，分形天线可以非常宽带。原因是分形结构创建了提供许多不同谐振的电容器和电感器的虚拟组合。

当然，这些设计可能具有挑战性，通常通过仿真来完成。也有批评者说[分形天线并没有真正更好地工作](http://www.antenna-theory.com/antennas/fractal.php)。如果你想看一些实际的东西，看看[这篇论文](https://www.researchgate.net/publication/276355483_Fractal_Antenna)来自 Nirma 大学的几个学生，他们使用模拟来设计这样的天线，并观察它的性能。

如果你想了解一点历史，我们之前已经讨论过分形。这看起来有点类似于另一个[分形天线](https://hackaday.com/2016/12/05/body-cardio-weighing-scale-teardown/)在所有地方的规模发现。
