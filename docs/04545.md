# 宜家餐桌 3D 打印机

> 原文:[https://hackaday.com/2016/12/22/ikea-table-3d-printer/](https://hackaday.com/2016/12/22/ikea-table-3d-printer/)

[在这份说明性的](http://www.instructables.com/id/PRINTTABLE-the-Affordable-Attractive-IKEA-Lack-Tab/)中，【Wayne Mason-Drust】分享了如何基于 Ikea LACK 拉克桌子制作一台酷炫、好看的 3D 打印机的分步指南。从[宜家灯笼气象站](http://hackaday.com/2016/08/22/ikea-lantern-houses-full-featured-weather-station/)到基于宜家桌子的全打印 [CNC，几乎可以肯定地说，3D 打印机宜家 hack 早该上市了。](http://hackaday.com/2016/02/02/fully-printed-cnc-on-an-ikea-table/)

[Wayne]第一次想到用宜家桌子作为 3D 打印机的底座，是因为他用这张桌子来支撑他在公司工作的其他 3D 打印机。他意识到，即使用了五年，桌子也没有磨损或变形的迹象。所以他决定在这张精确的桌子的基础上开始 3D 打印机的工作，这张桌子是用来放打印机的。

[Wayne]将两个堆叠在一起，并将其命名为 Printtable(双关语？).这款开源的笛卡尔 rep-rap 3D 打印机看起来非常漂亮。构建面积为 340 毫米 X 320mm 毫米，Z 轴为 300 毫米，零件价格最低为 395 美元，看起来是一台相当不错的 3D 打印机。通过一些采购零件的工作，价格可能会更低。

或者我们可以等到宜家开始销售它们。
