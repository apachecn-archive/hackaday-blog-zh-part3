# 第二个皮肤合成像手套一样合适

> 原文：<https://hackaday.com/2016/01/12/second-skin-synth-fits-like-a-glove/>

加州纺织艺术家和音乐家[push_reset]挑战自己，在不使用柔性感应电阻的情况下，制作出可穿戴的、基于手势的合成器。最终，她使用导电织物、电阻涂料和 3d 打印部件从头开始设计了它的几乎每一部分。

这只手套里有几根手指可以做双重工作。四个指尖都有一个由聚氨酯、导电涂料和导电织物制成的传感器，用小铆钉连接到电线上。这些传感器在爱迪生上触发不同的样本，这些样本由 [Timbre.js](https://mohayonao.github.io/timbre.js/) 生成。食指和中指也有关节驱动器，由 3d 打印的销槽机构制成，可以转动修剪盆。弯曲一个关节改变延迟时间，而另一个操纵三角波。

手套的背面是两个由导电织物制成的传感器。上下触摸一个长度将改变混响。上下滑动另一个会改变正弦波的频率。[push_reset]友好地提供了从手套模型到关节致动器的 STL 文件重新创建该构建所需的一切。休息之后，请观看手套的简短演示。如果你喜欢游行，这里有一款模拟行进乐队的可穿戴合成器。
