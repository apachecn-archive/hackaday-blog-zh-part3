# 最简单的智能眼镜概念

> 原文：<https://hackaday.com/2015/11/10/the-simplest-smart-glasses-concept/>

谷歌眼镜来了又去，给英语留下了一个重要的补充。甚至谷歌自己也用“glasshole”这个词来形容那些以令人毛骨悚然的方式使用该产品的人。我们不能决定戴一副[显然是自制的眼镜，像【Jordan Fung】](http://www.instructables.com/id/Arduino-Based-Smart-Glasses-by-a-13-Year-Old-Jorda/)生产的那种，是更令人毛骨悚然，给你更多黑客信誉，还是只是让你看起来像个博格人。也许是所有这些的组合。虽然为谷歌眼镜开发的成本和复杂性肯定是黑客在该硬件上的障碍，但这个项目只是恳求你建立自己的概念。

(乔丹的)建设，称为 Pedosa 玻璃，真的是非常值得尊敬的一个自建设置。Arduino Nano 有点笨重，三个按钮占用了一些空间，但这并没有扼杀将它们安装在眼镜外形中的能力。FLCoS 显示器可以让你看到[Jordan]仍在开发的软件的输出。现在的功能包括一个定时器和一个使用头戴式白色 LED 的手电筒。我们承认，不多，但足以证明硬件和整个点将添加你想要的软件。

诚然，它并不完全像谷歌眼镜。虽然两者都使用 FLCoS 显示屏，但 Pedosa Glass 使用的是相机取景器的显示屏，所以你不会真的透过它看。尽管如此，在你的视野中安装一个小显示器可能还是有一些实际用途的。该系统将通过更好的 CPU 进行改进，更容易通过加速度计等传感器连接到网络——这个项目有足够的空间进行迭代。不过，如果你想扩展这个系统，你还有一整副耳机可以使用。
