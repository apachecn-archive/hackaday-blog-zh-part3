# 平板显示器解决了钻方孔的问题

> 原文:[https://hack aday . com/2017/03/17/panel-mount-display-solves-the-problem-of-drilling-square-holes/](https://hackaday.com/2017/03/17/panel-mount-display-solves-the-problem-of-drilling-square-holes/)

[绝对自动化]有一个七段显示器的问题。将这些显示器安装在一个外壳中是一件痛苦的事情，因为你不能钻完美的方形孔，而且你总是会因为过分热心的文件工作而弄乱一些外壳。这个问题有一个解决方案——面板安装式电表。

这些面板安装式仪表上的挡板隐藏了外壳中的瑕疵，通常不需要螺钉。然而，它们是专用显示器，通常用于显示温度、转速或其他测量值。

[absolute automation]将其中一个专用面板安装显示器[变成了一个通用设备](http://www.absolutelyautomation.com/articles/2017/03/10/panel-mount-7-segment-display-embedded-arduino)。基本上，它是一个面板安装的 Arduino，带有三个七段显示器。

这个项目建立在 perfboard 上，该板被切割成适合放在通常的供应商处找到的非常便宜的面板仪表[的外壳内。隐藏在这个性能板下面的是一个 ATmega、几个电阻和支持部件，让一切运转起来。这款面板安装式仪表既可以是串行从机，也可以是独立控制器，可通过 Arduino IDE 编程。也很便宜。你可以看看下面的视频。](https://www.aliexpress.com/w/wholesale-panel-meter.html?spm=2114.01010208.0.0.pSp1eG&site=glo&groupsort=1&SortType=price_asc&g=y&SearchText=panel+meter&tc=af&initiative_id=SB_20170314145439&needQuery=n&isFreeShip=y&aff_platform=link-c-tool&cpt=1489605704524&sk=QBYFUFA&aff_trace_key=3afc21404d244325a2fddf1c76622b31-1489605704524-09785-QBYFUFA)
