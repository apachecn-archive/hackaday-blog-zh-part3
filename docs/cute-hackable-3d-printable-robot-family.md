# 可爱、可破解、3D 可打印的机器人家族

> 原文：<https://hackaday.com/2016/02/20/cute-hackable-3d-printable-robot-family/>

几周前，我们第一次在慕尼黑制造展上看到了机器人 Zowi(横幅照片中的第一排)，我们对他们只用四个伺服电机就能做出如此有趣的动作印象深刻。大脚、强壮的踝关节和巧妙的编程相结合，让可爱的小机器人单腿站立，做太空步，甚至单脚跳。(看视频，下方休息。)我们知道这是出售的。我们不知道的是它是完全开源的。

发明家哈维尔·伊莎贝尔(Javier Isabel)非常擅长给予应有的赞扬，这是一件伟大的事情，因为他的“机器人”基本上是一个经过改进的鲍勃机器人。也就是说，你真的需要看到这个东西移动，才能知道 Zowi 明显更强的伺服系统和聪明的编程可以产生什么样的影响。

但这还不是全部！由于 Zowi 的一切都是开放的，在 GitHub 上你不仅可以 3D 打印出你自己的，而且你可以很容易地修改这个漂亮的盒子。一些人接受了[Javier]的提议，[将他们的修改作为拉式请求提交回来](https://github.com/bqlabs/zowi/tree/master/mods/)。因此，如果你想稍微更像人，并愿意增加几个伺服，有一个很好的基础为您的探索做好准备。

我们真的很喜欢合作玩具机器人设计的想法，从我们所看到的来看，基本的 Zowi 平台是一个赢家。看看吧，看看你是否没有灵感在设计中加入你自己的个人风格。如果你这样做，一定要贡献回来给别人看！
