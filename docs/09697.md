# 你盘子里的海藻

> 原文:[https://hackaday.com/2018/06/08/algae-on-your-plate/](https://hackaday.com/2018/06/08/algae-on-your-plate/)

对于我们这些在天然游泳洞周围长大的人来说，藻类是我们泡完澡后必须洗手的原因。游泳者的瘙痒，或者只是被绿色的黏糊糊的东西覆盖，并不是度过一个下午的吸引人的方式。把所有的藻类混为一谈是不公平的，有些是令人讨厌的，但有些是美味的，人类已经吃了好几代了。

如果你认为藻类烹饪并不普遍，藻类听起来并不吸引人，你并不孤单。这是一个艰难的推销，就像说服某人第一次尝试蒲公英一样。它可能还不能保证在杂货店里有一个冰箱区，但是[藻类可以生产富含蛋白质的食物](https://spectrum.ieee.org/energy/environment/new-tech-could-turn-algae-into-the-climates-slimy-savior)而不需要大量的加工。

目前，有很多工作要做，以提高藻类养殖场的效率，Qualitas 已经开始。他们的飞跃表明我们还有很大的改进空间。在视频中可以看到的循环桨轮使用的能量是之前版本的三分之一。他们的收割机用三十分之一！现在，他们最大的成本来自二氧化碳罐，考虑到像发电厂这样的地方花钱来处理这些东西，这看起来不太可能。这应该引起一些思考。

2018 Hackaday 奖可以使用一些藻类提交物，你可以把它们拿到银行去。准备好开始[种植自己的藻类](https://hackaday.com/2016/03/23/growing-algae-for-fun-and-profit/)，自动化过程。它也可能[让你在去杂货店的时候不会绊倒](https://hackaday.com/2018/01/28/biologic-additive-may-lead-to-self-healing-concrete/)，或者[你可以用它打印](https://hackaday.com/2017/10/15/cheap-3d-printers-make-cheaperer-bioprinters/)。
