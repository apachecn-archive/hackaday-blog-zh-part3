# 无线示波器评论

> 原文：<https://hackaday.com/2017/10/27/wireless-oscilloscope-review/>

EDN 的 Martin Rowe 最近测试了一台 200 美元的无线示波器。Aeroscope 100A 是探头主体中的单通道示波器，通过蓝牙 le 与苹果智能手机或平板电脑进行通信。你可以在下面的帖子里看到这个视频。

该设备的原始原型具有很高的带宽，但生产模型只能在每秒 100 兆样本的情况下实现 20 MHz 的带宽:没有什么惊天动地的事情。

我们并不是说要把一个单通道示波器扔进一个袋子里，但是我们过去看到的大多数示波器的价格都远低于 200 美元。无线功能可能是有用的，但事实是它在大多数时候并不那么重要。更令人费解的是，该功能仅适用于苹果智能设备(甚至不是所有设备)。4 kB 的样本内存对于一个玩具乐器来说可能足够了，但是——同样——对于 200 美元，我们可能期望更好的结果。

想想看，多花 50 美元，你就可以得到一个非常好用的双通道仪器，规格更好，再花两倍的价格，你就可以得到一个非常好的四通道示波器，带宽更大，样本存储器更多，功能更多。当然，它们不会超便携和无线，但也不会很差。

如果你真的想要小巧便携的产品，我们去年看到的全太阳和 OWon 设备都不到 100 美元，并且拥有更好的带宽，尽管它们缺乏无线功能。如果你想生活在边缘，你可以在你的烙铁上放一个示波器(T2)。然而，如果你使用苹果设备，你真的想让它成为你的示波器，Aeroscope 100A 似乎可以完成这项工作。
