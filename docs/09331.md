# 抓爪机器的互联网

> 原文：<https://hackaday.com/2018/05/02/the-internet-of-claw-machines/>

机器的远程管理对于各种商业、工业和家庭应用来说都是非常有用的工具。[现在，它也可以用于抓爪机——多亏了【免费编码你的冒险】。](https://www.youtube.com/watch?v=HkA3kzmzxSo&feature=youtu.be)

该项目使用 ESP32 板，包括背面的电池盒，用于标准 18650 锂电池，使小型电池供电项目更容易启动。你可以在邦古德和全球速卖通[找到它们，但我们不能 100%确定它们是符合犹太教规的，因为它们被打上了 WeMos 的商标，但不会出现在](https://www.banggood.com/search/wemos%C2%AE-wifi-+-bluetooth-battery-esp32-development-tool.html) [WeMos 的网站](https://wiki.wemos.cc/doku.php)或他们的[官方网上零售店](https://lolin.aliexpress.com/store/1331105)。无论如何，像这样把脂肪细胞绑在背上是个可爱的主意。如果你知道更多，请在评论中告诉我们。

回爪！然后将现成的拇指棒连接到 ESP32，该 ESP32 被编程为通过网络发送数据包来控制爪形机，爪形机通过其自身的网络连接微控制器进行布线。它被包裹在通常的 3D 打印外壳中。

该项目没有解决的一个问题是交付——无论是在本地网络还是在线，远程玩家如何领取他们的奖品？我们只能假设某种尖端形式的无人机送货是解决方案。这也不是我们见到的第一台遥控抓爪机。休息后的视频。
