# 为什么 100 瓦易趣发光二极管不是你的朋友

> 原文：<https://hackaday.com/2015/09/14/why-100-watt-ebay-leds-are-not-your-friends/>

led 是很神奇的东西。随着时间的推移，它们被用于越来越多的照明应用中——从家用灯泡到汽车前灯。但是，对更小、更便宜、更亮的 led 的推动似乎遇到了一些声誉不佳的制造商的阻碍。

一个典型的例子是，[bigclivedotcom] [一直在测试来自易贝](https://www.youtube.com/watch?v=NjKgPLeJ79Q)的约 100 瓦 led。当这些发光二极管正常工作时，它们会发出一束融化面部的光束，你不敢直视(想象一下《T2》中《夺宝奇兵》的场景)。对于 LED 来说，它们也有一些不同寻常的规格，例如在 30 伏下运行，与大多数 LED 的正向电压(约 2.5 伏，20 毫安)相比，这是一个很大的数字。

那么是什么原因呢？许多这种高瓦数的发光二极管使用一串几个串联的发光二极管。正如[bigclivedotcom]所指出的，当一些 LED 开始出现故障，表现得更像一个低值电阻而不是典型的 LED 时，这可能是一个真正的问题。在休息后的视频中，您可以看到[bigclivedotcom]测试 led 以获得更好的外观，以及发生了什么和为什么。
