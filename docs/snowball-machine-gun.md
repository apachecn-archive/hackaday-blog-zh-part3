# 雪球机枪

> 原文：<https://hackaday.com/2016/03/09/snowball-machine-gun/>

[Mark Rober]是三个侄子和侄女的叔叔，当地面有雪时，他要面对三个精英少年雪球高手的无情攻击。较弱的人会面对这样一个事实，即他们在雪地战斗中已经过了巅峰，但不是[马克]。他带来了技术来帮助他，并在他哥哥的帮助下发明了一种雪球机枪，能够在半秒钟内向这些不守规矩的年轻人发射 13 个雪球。

他创作的动力来自一个叶片鼓风机，枪本身由固定在鼓风机出口的 ABS 管制成。顶部被切掉的管子制成的弹匣通过 45 度连接装置装载，发射速度由操作者用木块推动雪球线的速度决定。他以 PDF 格式提供了完整的构建说明，所以假设你正在下雪的地方阅读这篇文章，你还在等什么呢！我们这些生活在很少下雪的地方，下的雪又湿又粘的人只能羡慕地看着。

下面的视频有完整的故事，完成了水果的无端破坏，年轻人蜷缩在他们的雪堡，最后[马克]收到了他的雪报应。
