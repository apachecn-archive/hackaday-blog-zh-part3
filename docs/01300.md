# 铋晶体使磁铁悬浮了 100 年

> 原文：<https://hackaday.com/2016/01/25/bismuth-crystals-hold-magnet-suspended-for-100-years/>

【夜鹰之光】一直在摆弄铋的[抗磁性。抗磁材料得到了](https://www.youtube.com/watch?v=A5pZZJ23rDM)[很多关注](http://hackaday.com/2015/05/26/levitating-objects-in-paramagnetic-liquids/)，因为它们有一种奇怪的能力，可以产生与穿过它们的磁场相反的磁场。简单来说，像铁这样的金属会被磁铁吸引；铋等金属会排斥它们。

[NightHawkInLight]用一个普通的实验室例子建立了他自己的解释，这个例子被用来演示这个非凡的特性，一个漂浮器。悬浮器是通过在两块反磁性材料板之间夹一块磁铁制成的。其中一个板块被放置在一段距离以外的一个更强的磁体所产生的磁场与其下方的磁体相反。当这样做时，中间的磁铁想要排斥上面的板块，却发现当它越靠近下面的板块时，它也受到同样的排斥，从而形成了一个稳定的系统。

最终上面的磁铁将需要重新排列，但[夜鹰之光]向我们保证这只是每 100 年一次。休息后的视频。
