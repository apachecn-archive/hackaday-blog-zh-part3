# 中继计算机由一个发出噪音的加法器启动

> 原文：<https://hackaday.com/2017/02/17/relay-computer-starts-with-an-adder-that-makes-a-racket/>

使用分立逻辑芯片制造的计算机？看过了。由单个晶体管组成的计算机？令人印象深刻，但已经完成了。[用机电继电器建造的计算机？](https://www.youtube.com/playlist?list=PL_1HsIiuOfg3QA91DUd9kGJjQoOHwlt5Q)带上臭氧！

名副其实的[Clickity Clack]的新 YouTube 频道有望非常有趣，如果他真的可以只用继电器就能运行一台电脑的话。但即使他没有超出播放列表中的三个视频，该频道也绝对值得一试。我们从未见过对二进制逻辑更简单、更清晰的解释，而[Clickity Clack]的基本逻辑门的中继版本是对这些概念的极好介绍。

他使用 DPDT 继电器库的定制 PCB，从基本的 AND 和 XOR 门发展到半加法器和全加器，解释了进位和进位是如何工作的。一切都是模块化的，所以他的四个 4 位加法器卡最终组合成一个 16 位加法器，我们假设它将用于构建一个非常嘈杂但有趣的 ALU。我们期待着这一点，并中继实现的触发器和其他元素，他将需要一个完整的计算机。

不要在意我们之前对非传统计算机项目的否定。值得看看[这台分立的 7400 逻辑计算机](http://hackaday.com/2013/11/04/asap-3-the-almost-simple-as-possible-computer/)和[这台全晶体管制造的](http://hackaday.com/2015/06/23/discrete-transistor-computer-is-not-discreet/)。他们也以自己的方式给人留下深刻印象，尽管比[Clickety Clack]的项目要安静一些。
