# 下周末:初级太阳能研讨会

> 原文：<https://hackaday.com/2018/06/28/next-weekend-beginner-solar-workshop/>

下周，Hackaday 将为所有准备好利用太阳能的黑客们举办一个研讨会。我们正在旧金山 Noisebridge 举办一个初级太阳能研讨会。我们邀请你在 7 月 7 日加入我们，我们将提供烙铁。

本次研讨会的讲师将是[Matt Arcidy]，他是一名狂热的黑客日读者，也是 Noisebridge 的成员。他为令人难以置信的 Noisebridge 游戏档案员 Live Arcade Cabinet 做出了贡献，为 Arduino 生态系统的电子组件做了演讲，现在他正在主持一个关于太阳能充电基础的研讨会。

本次研讨会将涵盖太阳能充电的理论，太阳能电池如何将光能转化为电能，何时何地这种技术是合适的，以及锂离子电池的安全处理。在研讨会结束时，每个与会者都将建立一个从太阳获取能量并为电池充电的系统，准备在未来的任何项目中使用。

这是一件大事。现在，Hackaday 奖正在进行第三次挑战，即[能量收集模块挑战](https://hackaday.io/prize/details#three)。这是该奖项的一大部分，已经有一些迷人的项目从胃酸中获取电力，甚至是地球的重力势能。当然，其中一些比另一些更实际，我们真的很有兴趣看看这个电力收集挑战会走向何方，会创造出什么样的伟大项目。
