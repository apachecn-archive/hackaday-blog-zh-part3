# 量子计算硬件拆卸

> 原文：<https://hackaday.com/2018/01/22/quantum-computing-hardware-teardown/>

虽然量子计算仍处于起步阶段，但已经取得了足够的进展，使它看起来比其他“革命性”技术，如核聚变能源或飞行汽车更有前途。IBM、英特尔和谷歌现在都在运行或生产两位数的量子计算机，并且有计划在未来生产更大的量子计算机。有了这么大的惯性，我们的量子计算革命似乎几乎是必然的。

然而，在这些新设备使我们所有的加密变得毫无意义之前，还有很多工作要做。由于在量子水平上没有什么是容易的(或直观的),进展已经比上个世纪晶体管革命时期慢了很多。这些计算机工作是因为两种现象:叠加和纠缠。量子位，或称量子位，之所以有效，是因为与晶体管不同，它可以同时处于多种状态，而不仅仅是“0”或“1”。这些状态很难确定，因为一般来说，一个量子位是由单个原子构成的。更复杂的是，量子计算机还必须利用量子纠缠，从而将一对粒子联系起来。这是任何硬件“观察”计算机状态而不影响任何量子位本身的唯一方法。事实上，观测结果[本身往往还没有最高的精确度](https://hackaday.com/2016/08/14/ion-trap-makes-programmable-quantum-computer/)。

硬件方面也存在一些其他挑战。今天存在的所有量子计算机都必须冷却到非常接近绝对零度的温度，才能利用超导性。无论这是因为热噪声的减少，如基于离子阱或其他技术的通用量子计算机，还是因为有可能像 D 波计算机一样利用超导的其他有趣特性，所有这些计算机都必须冷却到临界温度。更进一步的挑战是，即使在如此低的温度下，量子位仍然以不可预测的方式相互作用，并且随着量子位数量的增加，这种方式变得更加不可预测。

所以，一旦物理和制冷被整理出来，让我们看看一些量子计算技术*实际上*如何操纵这些量子好奇心来产生工作的、可编程的计算机。

## 线环和约瑟夫逊结

可以说，迄今为止量子计算机最成功的商业应用来自 D-Wave。虽然这些计算机没有“完全可编程”的量子位，但它们在解决某些优化问题方面仍然比传统计算机更有效。由于它们没有与“通用”量子计算机相同的功能，该公司更容易在工作的计算机上获得更多的量子位。

D-Wave 计算机背后的基本原理是一个被称为[量子退火](https://en.wikipedia.org/wiki/Quantum_annealing)的过程。基本上，量子位被设置到某个能量状态，然后被释放回到它们最低的可能能量状态。这可以被想象成一种量子旅行推销员问题，事实上这正是量子计算机解决最优化问题的方式。D-Wave 硬件使用超导线圈工作，每个线圈都有一个弱绝缘的约瑟夫森结，通过小磁场存储数据。通过这种配置，量子位实现了叠加，因为导线环中的电子可以同时向两个方向流动，其中电流产生了磁场。由于电流是两个方向的叠加，所以它产生的磁场也是“上”和“下”的叠加。在芯片上每个量子位的位置都有一个可调耦合元件，这就是磁场与之交互的内容，用于对处理器进行物理编程，并控制量子位如何相互交互。

因为 D-Wave 计算机不被认为是通用量子计算机，每量子位的处理能力不等同于通用量子计算机。目前的 D-Wave 计算机拥有 2048 个量子位，如果它真的是通用的，将会产生令人麻木的影响。此外，D-Wave 计算机[是否表现出真正的量子加速](https://phys.org/news/2014-06-independent-group-d-wave-quantum-speedup.html)还不完全清楚，但如果没有实用性，像洛克希德·马丁这样的公司可能不会(反复)购买它们。

不过，还是有办法建造通用量子计算机的。本质上，我们所需要的是展示量子效应并能被外力操纵的东西。例如，一个已经提出的想法包括使用钻石中的杂质。不过现在，我们将关注科学家们成功建造量子计算机的两种主要方式:离子阱和半导体。

## 离子阱

在离子阱中，量子位是通过电离某种原子产生的。这可以通过许多方式来实现，但牛津大学实施的这种使用钙离子的方法涉及加热样品，向其发射电子，并捕获一些带电离子供计算机使用。从那里，可以使用激光将离子冷却到所需的温度。激光器的波长被特别选择为与离子共振，使得离子减速到其热波动不再影响其磁性的程度。激光也被用来给离子施加特定的磁场，量子位就是这样被“编程”的。一旦操作完成，激光再次用于探测离子并确定其状态。

但是，在这个例子中，可伸缩性的问题很快就出现了。为了拥有大量的量子比特，需要捕获大量的离子，并通过一系列激光同时操纵。量子位元可以互相影响的事实增加了这个问题，虽然这个特性也可以用来帮助从系统中读取信息。由于复杂性的原因，通用量子计算机的未来似乎可以在我们都很熟悉的东西上找到:硅。

## 半导体

自然状态下的硅实际上是一种有效的绝缘体。硅有四个价电子，它们都完全满足于被限制在一个原子核内，这意味着没有电荷流动，因此也没有电流流动。为了从硅中制造一些有用的东西，比如可以以特定方式导电的二极管或晶体管，硅制造商在硅中注入杂质，通常是硼或磷原子。这种引入杂质的过程被称为“掺杂”,它将外层电子的过剩或不足注入硅中，这意味着现在硅晶格中存在电荷。这些费用可以被操纵，用于我们用来创造现代世界的所有奇妙的效果。

但是我们可以把掺杂的过程更进一步。科学家们没有在硅中引入大量杂质，而是找到了一种方法，将一种单一杂质，一个孤独的磷原子，包括其最外层的电子，放入一个类似场效应晶体管的设备中。利用这些晶体管人们熟悉和熟知的行为，单个杂质就变成了量子位。

在这个系统中，施加大的外部磁场以确保电子处于特定的自旋状态。量子位就是这样设定的。从那里，晶体管可以用来读取这个单个电子的状态。如果电子处于“上方”位置，它将有足够的能量移出晶体管，该设备可以记录原子剩余的正电荷。如果它在“下”的位置，它将仍然在晶体管内部，器件将看到来自电子的负电荷。

这些([和其他一些](https://spectrum.ieee.org/semiconductors/materials/will-silicon-save-quantum-computing))方法已经允许研究人员在量子位内实现长内聚时间——本质上是量子位在衰变和不再有用之前处于相关状态的时间量。在离子阱中，这个时间大约是毫微秒或微秒。在这种半导体类型中，时间在秒的数量级上，这在量子计算的世界中是永恒的。如果这种进展持续下去，量子计算机可能会在未来十年内变得司空见惯。我们只需要知道如何使用它们。
