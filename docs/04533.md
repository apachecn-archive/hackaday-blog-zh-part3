# 实用深度学习

> 原文:[https://hackaday.com/2016/12/21/practical-deep-learning/](https://hackaday.com/2016/12/21/practical-deep-learning/)

深度学习——使用神经网络和现代技术来解决从计算机视觉到语音识别和合成的问题——当然是当前的热门词汇。然而，核心是一套强大的组织自学系统的方法。多层神经网络并不新鲜，但主要由于伪装成视频卡的大规模并行计算平台的可用性，人们又开始对它感兴趣了。

问题是如何开始这样的事情。有很多学术论文可能很难读懂。或者你可以从 GitHub 获取一些代码，然后试着破解它。

一个更好的主意是参加一个名为[程序员实用深度学习，第一部分](http://course.fast.ai/)的免费课程。课程是免费的，除非你把你的投资算在时间里。他们警告你要在 7 周内每周投入 10 个小时来完成课程。你可以在下面的视频中看到第一部分。

这门课程起源于旧金山大学。以下是他们的描述:

> 这个为期 7 周的课程是为任何至少有一年编程经验，并对高中数学有一些记忆的人设计的。你将从第一步开始——学习如何让 GPU 服务器在线适合深度学习——并一直到为计算机视觉、自然语言处理和推荐系统创建最先进、高度实用的模型。

第一课包括区分猫和狗。有一个聊天、论坛和其他支持资源的 Slack 频道。您可能会关心第 2 部分在哪里。根据该网站，它应该在 2017 年 5 月上线。

我们最近谈到了简单的神经网络。我们还看了一些 DeepMind 的[语音应用。](http://hackaday.com/2016/12/03/talking-neural-nets/)
