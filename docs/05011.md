# 猎豹光束:更多证据表明猫是你的霸主

> 原文:[https://hack aday . com/2017/02/13/cheetah beam-more-proof-the-cats-that-your-霸王/](https://hackaday.com/2017/02/13/cheetahbeam-more-proof-that-cats-are-your-overlord/)

我们不知道猫看到红色激光束时会看到什么，但我们知道这不是我们看到的。至少对许多猫来说，这种反应是即时和极端的。当然，你的猫期望你辞职，随叫随到地玩它。虽然[fluxaxiom]想遵从，但他也知道没有工作就没有猫粮。为了解决这一困境，他建造了一台自动化猫激光器。除了激光模块，该设备还在 3D 打印的外壳中使用了一些伺服系统和微控制器。你可以看一个视频，在下面。狗显然也喜欢它，但当然它们不是建造它的原因。

如果你没有 3D 打印机，你仍然可以拼凑一些东西。微控制器是一个 Adafruit Pro 饰品，本质上是一个 Arduino Pro Mini，带有一些额外的引脚和一个 USB 端口。

有 12 种不同的模式，该设备随机循环，试图迷惑你的猫。我们不禁想知道这是否应该在互联网上，这样你就可以控制和手动玩你的猫。听起来像是布莱恩克的工作。上一次我们看到猫激光，它是一个小[更机动](https://hackaday.com/2013/03/11/laser-toting-robot-taunts-house-cat/)。
