# 智能减速带只能让超速行驶的汽车减速

> 原文:[https://hack aday . com/2017/08/14/smart-speed-bumps-slow-only-speeding-cars/](https://hackaday.com/2017/08/14/smart-speed-bumps-slow-only-speeding-cars/)

不管你喜不喜欢，减速带是我们道路基础设施的重要组成部分，尤其是在像学校附近这样的建筑物密集的地方【英国商业内幕】报道[非牛顿液体填充减速带](https://www.youtube.com/watch?v=-IQSxsbN7cY)正在西班牙、以色列和德国进行测试。

传统减速带确实有缺点；低矮车辆底部的损坏是常见的。虽然它们的尺寸应该是一致的，但实际上它们可能会有很大差异，这使得在不熟悉的颠簸路面上行驶有点不可预测。这一切都将随着非牛顿颠簸而改变，这种颠簸在低速行驶时很柔软，但对于超速行驶的司机来说，它们会变硬，表现得更像传统的颠簸。这给了遵守法律条文的司机更好的驾驶体验，同时仍然阻止超速驾驶的司机..

非牛顿材料并不是什么新东西，但是我们认为这是一种很好的利用这类材料的方法。不管你喜不喜欢，道路变得越来越智能。是时候拥抱科技，改善我们的通勤状况了。
