# 遥控飞车手罗德完全从零开始建造

> 原文：<https://hackaday.com/2016/07/04/rc-hot-rod-built-completely-from-scratch/>

[ossum]的 R/C hot rod 展示了当天才黑客充分利用所有可用的现代资源时会发生什么。结果令人震惊。

[ossum]在赢得一些比赛后，有一堆亚马逊和 Shapeways 信用点数。他有一段时间梦想造一辆遥控汽车，并决定现在是时候了。在从亚马逊订购了所有需要的零件后，他在 [Fusion 360](http://hackaday.com/2016/02/02/making-parametric-models-in-fusion-360/) 中制作了一个[非常漂亮的汽车](http://a360.co/28Rp1ZK)模型。CAD 模型是一个很好的学习资源。如果您想了解如何使用参考照片、零件等来构建一个详细而有用的 CAD 模型，我们建议您将其作为 Fusion 档案下载，并在时间轴上进行擦洗，看看他是如何做到的。

一些零件被送去进行激光切割。其他的是 3D 打印的。其余的是他自己做的。由于他的模型，他们都走在一起很好。你可以在休息后的视频中看到他的遥控杆赛车。
