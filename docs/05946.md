# 虚拟现实又回来了:x 机器人的故事

> 原文:[https://hack aday . com/2017/05/14/VR-and-back-again-an-x robots-tale/](https://hackaday.com/2017/05/14/vr-and-back-again-an-xrobots-tale/)

我们来自 XRobots 的朋友[James Bruton]通过展示一个人如何无缝地从虚拟世界进入现实世界，然后再回到现实世界，参与了另一种混合现实魔法。走开，绿色屏幕和笨重的灯光！

现在，你所看到的大部分都是真正发生在后期制作中的——就目前而言——但测试镜头是未来更加集成的系统的先驱。正如它现在的工作一样，一个 GoPro 被连接到 HTC Vive 耳机的前面，允许[Bruton]同时在两个现实中记录。在虚拟现实测试区，他设置了一个虚拟绿色房间的入口——只比衣柜小一点——允许他将 GoPro 镜头叠加在他通过门口看到的所有东西上，以及他走过时周围的所有东西上。不幸的是，如果戴上耳机，[Bruton]无法看到他要去哪里，所以他被迫用一只手拿着它，在混合现实空间中移动。同样，这是暂时的。

实际上——当他开始通过网关将数字块“扔进”现实世界时，这有点超现实。
