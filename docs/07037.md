# Hackaday 奖参赛作品:用于 EL 面板的触摸感应电源

> 原文：<https://hackaday.com/2017/09/08/hackaday-prize-entry-touch-sensitive-power-supplies-for-el-panels/>

[fool]在 Hackaday 奖竞赛中的参赛作品是一个模块化和可配置的照明系统，其目的是帮助老年人和其他行动不便的人在家中安全导航。对[傻瓜]来说，这意味着电致发光面板和电线安静稳定的嗡嗡声。众所周知，电致发光设备很难供电，因为它只能在交流电下工作。MoonLITE 项目是解决易于使用的 EL 电源问题的答案。目标是创造一个 5 瓦，安静，可穿戴的 EL 电源，输出 100 伏，100 赫兹。

[傻瓜]对 EL 材料感兴趣的原因之一是它还可以变成触摸传感器。这在照明方面有明显的应用，尤其是在辅助技术方面。MoonLITE 项目基于[fool]的[哇板](http://whoaboard.com/store/)，它不仅可以将 EL 线和面板变成触摸感应灯，还可以控制基本上任何东西的模拟开关。例如，这种照明兼作传感器的独特能力为老年人床边的发光 EL 扶手杆提供了机会。他或她起床时无论如何都要触摸它——为什么不在增加稳定性的同时增加照明呢？

这是一个特别酷的项目，它带来了一些我们不常看到的东西。你可以看看下面这个项目的视频，包括 EL 面板被用作按钮的例子。
