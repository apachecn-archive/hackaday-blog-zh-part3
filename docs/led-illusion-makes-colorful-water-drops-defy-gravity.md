# LED 幻像让彩色水滴挑战重力

> 原文：<https://hackaday.com/2018/03/14/led-illusion-makes-colorful-water-drops-defy-gravity/>

60 年代和 70 年代是俗气照明配件的大好时光。熔岩灯、频闪灯、彩色器官、黑光海报——我们全都有。一个特别棒的装置是人工降雨显示器，一个小泵将矿物油滴到围绕一个小雕像的垂直单丝上，整个装置从上面以引人注目的方式照亮。如果这听起来很可怕的话，的确如此，而且随着灰尘和碎片的积累，油变得粘稠，情况变得更糟。

虽然[这个漂浮的水滴展示](https://github.com/isaac879/RGB-Time-Fountain)看起来有些相似，但它与昔日那盏油腻的灯毫无关系。[isaac879]的“RGB 时间喷泉”实际上要复杂得多，看起来相当迷人。时间喷泉的想法很简单——沿着一条可以被闪烁的发光二极管照亮的路径，将水从水泵喷嘴滴到一个较低的容器中。使闪光与控制泵速的 PWM 同步可以将液滴冻结在原位，或者甚至使它们看起来像是向上滴。[isaac879]通过对 RGB 照明进行实验，将时间喷泉的想法向前推进了一步，他发现各种整洁的效果都是可能的。下面的视频展示了所有的酷，像不同颜色的交替滴看起来像下降或上升的油漆滴，以及合并在一起形成一种新颜色的滴。看哪，神秘的反重力杯子滴起来，却又被装满了！

当然，必须考虑到使用闪光灯的项目视频。这个时间喷泉和我们在之前展示过的[类似喷泉的效果很难捕捉，但这个对我们来说仍然很棒。](https://hackaday.com/2017/05/22/gravity-defying-drips-of-a-bike-pump-controlled-fountain/)
