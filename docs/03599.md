# Hackaday 奖参赛作品:LipSync，四肢瘫痪者的智能手机接入

> 原文:[https://hack aday . com/2016/09/21/hack aday-prize-entry-lip sync-smart phone-access-for-quadriplegic-people/](https://hackaday.com/2016/09/21/hackaday-prize-entry-lipsync-smartphone-access-for-quadriplegic-people/)

对于我们大多数人来说，触摸屏智能手机已经成为不可或缺的配件。我们不假思索地在我们的数字生活中点击和滑动，无处不在的真正便携式计算的承诺终于实现了。

尽管智能手机给一些身体有缺陷的人带来了问题。触摸屏需要我们身体健全的人认为理所当然的手动灵活性，但对于不能使用手臂的人来说，它仍然是一个无用的玻璃板。

LipSync 是一个项目，旨在解决四肢瘫痪人群使用智能手机的问题。这是一个用嘴操作的手机屏幕光标操纵杆，带有模拟动作的吸气和吹气真空控制，如屏幕点击和后退按钮。

对于智能手机本身来说，该设备似乎是一个标准的蓝牙指点设备，而在其业务端，操纵杆和压力传感器都通过 Arduino Micro 与蓝牙模块连接。EAGLE board 和原理图文件可以在上面链接的项目 hackaday.io 页面上获得，并且有一个用于代码的 GitHub 存储库。

如今，技术是我们生活的一部分，很高兴看到这样的项目为每个人弥合了可用性差距。不用说，它是 Hackaday 奖[辅助技术](https://hackaday.io/prize/details#five)轮的完美候选。
