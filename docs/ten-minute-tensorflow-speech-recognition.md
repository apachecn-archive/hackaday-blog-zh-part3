# 十分钟张量流语音识别

> 原文：<https://hackaday.com/2017/03/24/ten-minute-tensorflow-speech-recognition/>

像许多人一样，我们对谷歌神经网络软件 TensorFlow 非常感兴趣。如果你想尝试使用它进行语音识别，你会想看看[硅谷数据科学公司]的[GitHub 知识库](https://github.com/silicon-valley-data-science/RNN-Tutorial)，它向你保证可以快速设置语音识别演示。它甚至涵盖了如果您使用 CUDA GPU 来加速处理或者不使用，您需要安装哪些项目。

另一个有趣的事情是使用 [TensorBoard](https://www.tensorflow.org/get_started/summaries_and_tensorboard) 来可视化产生的神经网络。这个工具在您的浏览器中提供了一个页面，让您可以可视化神经网络内部真正发生的事情。存储库中还有语音数据，因此它实际上是一个一站式入门商店。如果你没有看过 TensorBoard 的运行，你可能会喜欢下面来自谷歌的视频。

在您完成我们之前介绍的[非常简单的教程](https://hackaday.com/2017/03/14/google-machine-learning-made-simpler/)之后，这个演示可能是一个很好的第二步。这不是我们第一次研究[神经网络和语音](https://hackaday.com/2016/12/03/talking-neural-nets/)，但这可能是我们见过的最简单的一个。
