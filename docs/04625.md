# 一次性相机线圈枪！

> 原文:[https://hackaday.com/2017/01/07/disposable-camera-coil-gun/](https://hackaday.com/2017/01/07/disposable-camera-coil-gun/)

一些最令人愉快的项目往往有可怕的缺点，也最有可能造成身体伤害，比如在挖掘一次性相机时被电容器电击。但是通常——如果你小心的话——这种好奇心会得到回报，你最终会学会如何用相机闪光灯的电容制作像这个[线圈枪这样酷的东西。这种手持式发射小钉子，并且被包装在具有光开关触发器的手持式形状因子中。](http://www.instructables.com/id/Camera-Capacitor-Coil-Gun/?ALLSTEPS)

[LabRatMatt]预先消除任何潜在伤害的幻想，然后在他的详细指南中反复敦促谨慎。他分解工作中的物理现象，同时保持轻松的语气。这种线圈枪使用一个从一次性相机上拆下来的电容器和充电电路— [LabRatMatt]决定用他以前项目中手头上的另一个电容器来加倍。线圈是从一个旧门铃改造而来的，但如果你自己做线圈，一定要用几百圈。光开关最终适合作为触发器，因为它能够处理电压尖峰。

组装时，它看起来几乎就像你在后世界末日废墟中看到的东西，但它确实有效！
