# 星期五黑客聊天:BadgeLife

> 原文:[https://hackaday.com/2017/07/12/friday-hack-chat-badgelife/](https://hackaday.com/2017/07/12/friday-hack-chat-badgelife/)

在过去几年的 DEF CON 上，我们已经看到了徽章创作者的整个社区的出现。这些不是官方徽章——好消息，因为官方 DEF CON 徽章今年可能会是一块令人困惑的激光切割亚克力。幸运的是，今年是独立电子徽章销量最大的一年。这就是 BadgeLife，为热切的与会者设计和制作数百个徽章的艺术。

本周五，我们将在举办一次黑客聊天，与一些加入 BadgeLife 的人聊天。[Whitney Merrill]，律师、黑客和 Crypto & Privacy village 的监管人，将加入我们讨论今年的徽章、[她设计的谜题](http://cryptovillage.org/cpvtgb/)，以及 BadgeLife 的真正含义。参加我们的还有[Karl Koscher]，研究科学家和加密与隐私村的共同组织者。

同样在甲板上的还有今年狂欢徽章的创造者。去年，还有！XOR 创造了最近记忆中最流行的电子会议徽章之一[。今年，Bender 徽章升级了爆炸处理，一夸脱龙舌兰酒和两打戊基酒。](http://hackaday.com/2016/07/25/hands-on-the-andxor-unofficial-def-con-badge/)

我们很自豪地注意到，Hackaday.io 已经成为 BadgeLife 社区所有最佳徽章的非官方储存库。猎人 S. Rodriguez 徽章在那，还有 DEF CON 的 [Ides。作为今年](https://hackaday.io/project/14756-the-ides-of-defcon-an-unofficial-electronic-badge)最具创新性和最受欢迎的徽章[的创造者，我也将参加这个黑客聊天，讨论在一个非常紧凑的时间表上开发和运输硬件的考验和磨难。](https://hackaday.io/project/25150-the-official-hackaday-defcon-25-badge)

虽然这个 BadgeLife Hack Chat 的重点是为 DEF CON 开发电子会议徽章，但它也可以很容易地被称为“电子设计”。从设计到生产，每个徽章都包含大量的工作，所有这些工作都与任何硬件开发人员高度相关。

### 以下是参与方式:

[![join-hack-chat](../Images/cff5733f39a173302e1cd102e75ac107.png)](https://hackaday.io/project/5373-hack-chat) 我们的 Hack Chat 是 Hackaday.io 上的实时社区活动 [Hack Chat](https://hackaday.io/project/5373-hacker-channel) 群发消息。这次聊天将在太平洋时间 7 月 14 日星期五中午进行。困惑于‘中午’是在何时何地？这里有一个时间和日期转换器！

登录 Hackaday.io，访问该页面，并寻找“加入这个项目”按钮。一旦你成为这个项目的一部分，这个按钮就会变成“团队信息”，直接带你进入黑客聊天。

你不必等到星期五；随时加入，你可以看到社区在谈论什么。
