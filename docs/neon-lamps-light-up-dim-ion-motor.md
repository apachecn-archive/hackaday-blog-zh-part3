# 霓虹灯点亮昏暗的离子马达

> 原文：<https://hackaday.com/2018/04/14/neon-lamps-light-up-dim-ion-motor/>

小型风车式离子马达属于有趣的科学实验或与高压有关的整洁事物，但 Hackaday 自己的[Manuel Rodriguez-Achach]增加了[一个整洁的扭曲，包括霓虹灯](https://www.youtube.com/watch?v=X-r-p2iHoDA)。

通常情况下，你会拿一根直导线，在两端弯曲 90 度，但指向相反的方向，在一根杆子上保持平衡，然后施加高电压和中等电流。如果金属丝的末端足够锋利或者金属丝的直径足够小，金属丝就会在杆子的顶端开始旋转。如果你的电源有足够的电流，那么在黑暗中你甚至会在电线的尖端看到一种紫色的光，称为电晕。

[Manuel]制造了这样一个离子发动机，但他的电源没有足够的电流来产生足够强的电晕，以至于他的相机无法看到。于是他非常巧妙的将[霓虹灯](https://hackaday.com/2016/12/12/the-many-uses-of-the-neon-lamp/)焊接在电线的两端。每个灯的一条腿伸向电线，灯的另一端作为尖端留在空气中发射离子。

点燃每盏灯所需的电压是高压电源输出和周围空气电势之间的电压。空气最初可能处于地电位，但是他也弯曲电源的另一个输出端子，使得其尖端也在空气中。通过这种方式，它将相反极性的离子喷射到周围的空气中。

无论哪种方式，霓虹灯亮起，电线在电线杆上旋转。现在，即使没有可见的日冕，他的离子发动机也能进行令人敬畏的展示。看看下面的视频吧。
