# 也许你可以用金属印刷

> 原文：<https://hackaday.com/2016/12/09/maybe-you-can-print-in-metal/>

让我们面对现实吧。塑料印刷已经过时了。这很有趣。它是有用的。但是它不再是真正的异国情调了。真正的梦想是用金属打印。有以不同方式处理金属的打印机，但是它们对于传统黑客来说通常不实用。即使是一台“便宜”的金属打印机也要花费 10 万美元以上。但是有一些方法可以让你用一台普通的打印机就能达到目的。

PLA 长丝里不缺掺东西的。如果你有一个金属热端，不介意磨损喷嘴，你可以得到不同百分比的金属粉末的 PLA 长丝。你可以得到 50%到 85%是金属的细丝，生产出看起来像是由金属制成的东西。

[博·杰克逊]最近有机会用一种独特扭曲的金属细丝进行实验。虚拟代工的 Filamet 有 10%左右的 PLA。剩余的材料是铜。不仅要热打印材料，还要慢打印(比标准 PLA 密度大很多)。如果它只有近 90%的金属，那会令人印象深刻，但不会太令人兴奋。真正有趣的部分是打印完成后你能做什么。(如果你不想读，你可以直接跳到下面的视频。)

如果你什么都不做，你仍然会得到一个类似金属的印记。不过，如果你能接触到窑炉，你可以把你的零件放在接近 1000 摄氏度的温度下，和该公司专有的“神奇黑粉”放在一起这除去了几乎所有的 PLA，留下了完全(99.9%)的金属物体。

如果你喜欢，你也可以打磨和抛光材料，使其呈现出金属的外观，这可能比在热窑中劳作更容易。无论哪种方式，结果看起来(和听起来)都像金属。

该公司声称其他金属正在开发中。这并不像拥有一台真正喷出熔融金属的打印机那么棒，但它要便宜得多。请注意，并不是说它很便宜。一卷 750 克的材料是 85 美元。不过，肯定比[电镀](https://hackaday.com/2015/08/23/metal-3d-printing-with-your-printer/)强。
