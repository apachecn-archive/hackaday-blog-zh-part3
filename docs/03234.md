# 无人驾驶飞机、聪明的黑客和 CG 一起出现在《星球大战》影迷电影中

> 原文：<https://hackaday.com/2016/08/11/drones-clever-hacks-and-cg-come-together-for-star-wars-fan-film/>

直到后来我们看到了视频的制作，我们才确定这部《星球大战》粉丝电影是否过时了[。他们想用新的方式拍摄传统场景。我们的想法是找一些非常优秀的四轴飞行器飞行员，给他们一些定制的四轴飞行器，让他们在一个风景优美的地方重演一场战斗，然后用一些电影魔术将这一切融合在一起。](https://www.youtube.com/watch?v=vhR4QEEa5dw)

四轴飞行器本身是一些带有 4K 摄像机的高性能赛车四轴飞行器。这种东西有火箭飞船的重量比。不管视频暗示了什么，不幸的是，它们不是 TIE 战斗机的形状。经过一天的飞行和几次长途跋涉，在不可避免的崩溃后取回昂贵的设备(幸运的是，提供了一些不错的镜头)，下一步是合成。

然而，如何欺骗观众相信他们是在 X 翼四轴飞行器中？一个便宜的方法是花费数不清的时间进行运动跟踪，并在适当的位置渲染驾驶舱。看起来不会很真实。他们提出的解决方案有点愚蠢，也有点聪明。用 GoPro 在 2×4 上挂载 3D 打印驾驶舱。拿着这个精巧的装置，在智能手机上播放飞行片段。尝试将驾驶舱移向与飞行相同的方向。我们不确定这样做时是否需要发出嗖嗖声和皮尤皮尤激光声，但这不会有什么坏处。

最后，这一切汇集在一起，拍出了一部傻乎乎的、却令人信服的好影迷电影。干得好！休息后的视频。
