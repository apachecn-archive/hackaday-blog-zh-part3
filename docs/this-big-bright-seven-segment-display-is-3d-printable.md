# 这个大而明亮的七段显示器是 3D 打印的

> 原文：<https://hackaday.com/2018/03/18/this-big-bright-seven-segment-display-is-3d-printable/>

七段 LED 显示屏似乎已经存在很久了，而且现在设计已经非常优化了。现成的单元有各种尺寸和颜色，但如果你想要一个真正的大显示器，你可能需要自己动手做。尽管如此，扩大尺寸并不一定意味着你必须扩大复杂性，如果[这种无光管巨型七段 LED 显示屏](https://www.youtube.com/watch?v=M3jyMX8EXAQ)是一种指示的话。

很明显，[弗兰布兰奇]有收集和建造古怪数字显示器的癖好，比如这个阴极射线管谢妮仿制品或者她的[阿波罗 DSKY 电致发光显示器](https://hackaday.com/2017/05/29/re-creating-the-apollo-dskys-display/)。她的大尺寸七段显示器远没有这两种显示器复杂，这是设计使然；[Fran]想要 3D 打印的单个零件，而不是光导管和扩散器的组件。为此，显示器只是一对 X 形的分隔物，在显示器的表面后面相互堆叠。他们分隔形成六个三角形隔间和一个菱形隔间，每个隔间通向一个扇形窗口。每个三角形隔间放一个 LED，而双倍大小的菱形空间放两个。就是这样——发光二极管照亮每个隔间的内部，打开相应的部分。请看下面的实际操作。

显示屏仍需要一些调整，但它又大又亮，接受角度也很大。更重要的是，它是可扩展的——想象一个使用 LED 灯泡的胶合板大小的显示器。我们期待着[Fran]的改进和她的下一个展示项目，它似乎使用热熔胶作为光管。
