# 360°全景无窗电梯

> 原文：<https://hackaday.com/2016/08/20/a-windowless-elevator-with-a-360o-panoramic-view/>

葡萄牙里斯本的阿莫里拉斯塔最近增加了一个屋顶观景区，向公众开放。塔顶是该市最高的景点之一，观景区给人以 360°的周边美景。然而，到达顶层的电梯还有很多不尽人意的地方。这是一个内部电梯，本身并没有提供任何意见。

所以，阿蒂卡和辛德勒一起被请来解决这个问题。解决方案是在电梯内部安装显示器，以模拟外面城市的 360°全景。视频与电梯同步，因此当电梯乘客在楼层间上下移动时，视图会发生变化。

负责概念、设计和电子设备安装的 Artica 首先在他们的办公楼里建造了一个原型。这是一个全尺寸的电梯复制品，他们可以用它来测试设计并为安装做好准备。然后，他们与迅达合作，在阿莫里拉斯大厦的电梯中安装了该系统，这使得几乎完全重建电梯成为必要。正如你在视频中看到的，最终的视图和伴随的音乐([肯定不是电梯音乐](https://hackaday.com/2016/05/24/secret-listening-to-elevator-music/))非常棒，甚至是在屋顶观看区公开开放时完成的。

和我们一样，你可能想知道这些视频是从哪里来的。场景以明显的视差移动，所以视频显然是以连续的运动捕捉的，而不是滚动的图像。这是一架携带相机的无人机的作品。
