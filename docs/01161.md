# 非常详细的 BB-8 机器人建造

> 原文:[https://hack aday . com/2016/01/09/very-detailed-b B- 8-robot-build/](https://hackaday.com/2016/01/09/very-detailed-bb-8-robot-build/)

[詹姆斯·布鲁顿]刚刚完成并发布了他令人印象深刻的 [BB-8 机器人模型](http://xrobots.co.uk/BB8-2/)的设计。不久前我们报道了他冒险的开始，当时我们正在对新机器人的秘密进行理论化，但那是针对一个完全不同的机器人设计。[詹姆斯]在追求一个设计，用一个小机器人坐在一个大球上。

这个新版本有一个机器人坐在一个球里，头部与身体磁性耦合。在这种构造的许多事情中，我们认为机器人有一个驱动电机并通过旋转和反转机器人底部的大飞轮来转动是很酷的。这当然不是为机器人背后的秘密提出的顶级理论之一。机器人大部分是用 3d 打印机制造的，偶尔会有真空成型的装饰件。如果你想为自己做一个，[詹姆斯]也将机器人的所有设计和 cad 发布在他的 GitHub 上。周四，他发布了他的 10 集视频系列[的最后一集。休息之后请看第一部分。](https://www.youtube.com/playlist?list=PLpwJoq86vov8Tfr2-JUcma3lO0mLid_FX)
