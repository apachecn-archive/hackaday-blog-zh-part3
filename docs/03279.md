# 构建 DIY 热管

> 原文：<https://hackaday.com/2016/08/21/building-a-diy-heat-pipe/>

热管曾经是航空航天设计师的秘密设计工具，现在由于电脑 CPU 冷却的需求，热管已经成为一种常见的固定装置。热管可以将大量的能量从热端转移到冷端，当你需要冷却某样东西，而热端附近由于某种原因无法安装风扇时，热管非常有用。与主动冷却不同，热管也不需要任何外部电源或泵。

詹姆斯·比格用铜管制作了自己的热管。你可以在下面看到一个正在制作的视频。没什么特别的，只是一个装了些水的铜管。然而，在密封之前，[詹姆斯]让水沸腾以降低管中的压力，这是一个有趣的技巧。

他的技术的一个限制是没有内部灯芯。这意味着管道只能垂直安装。如果你以前没有看过热管，大多数热管都有一个灯芯。这个想法是一些工作流体在管道中。你选择流体，使其在你想要处理的温度或以下沸腾。热蒸汽冲向管道的冷侧(携带热量),那里有一个大型散热器，可能有风扇或主动冷却系统。蒸汽冷凝——在这种情况下——落回到管的底部。然而，如果有一个灯芯，毛细作用将返回到管的热端的流体。

你可能会认为用水作为工作流体会将你限制在 100°C，但请记住，[詹姆斯']技术降低了管中的压力。压力越低，水的沸点越低。

在之前，我们已经见过热管和酒柜[用于冷却电脑。事实上，我们甚至在](https://hackaday.com/2007/06/19/heat-pipe-wine-chiller-pc-cooling/)[的完全无风扇电脑](https://hackaday.com/2008/10/11/passively-cooled-computer/)中见过它们。
