# 让 Alexa 打开你的车库门

> 原文：<https://hackaday.com/2017/09/03/have-alexa-open-your-garage-door/>

[yoyotechKnows]在他的 Liftmaster 停止工作后，制造了一个由 Alexa 控制的车库开门器。现在他所要做的就是对着他的手机大喊大叫，他就可以随意升降他的车库门。

他的项目基于一个光子 WiFi 套件，一对 LCC 120 数字继电器触发两扇门，簧片开关和一个串行配备的 LCD 显示门的状态，Alexa， [IFTTT](https://ifttt.com/) 和 [OpenHab](https://www.openhab.org/) 处理命令。你可以在项目报告中找到他的代码。

目前，他有一个液晶显示器通知他每个门的状态，热粘一个簧片开关来跟踪每个门是否关闭。这可能看起来有点无关紧要，因为他也可以从车库里看到门。然而，他正在考虑把显示器放在他的房子里。但他不能直接问阿利克夏吗？

我们热爱 Hackaday 的家庭自动化，从[游泳池](http://hackaday.com/2016/08/06/home-pool-added-to-home-automation/)到[鸡舍](http://hackaday.com/2016/11/06/iot-coop-door-cares-for-chickens-tests-home-automation/)的一切都配备了应用程序控制和数据记录。
