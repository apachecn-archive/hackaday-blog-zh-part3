# 透视旋转发动机揭示汪克尔魔术

> 原文:[https://hack aday . com/2018/05/01/see-through-rotary-engine-reveals-Wankel-magic/](https://hackaday.com/2018/05/01/see-through-rotary-engine-reveals-wankel-magic/)

汪克尔转子发动机因其在主流汽车行业的麻烦生活、高功率重量比以及全速行驶时发出的醉人嗡嗡声而闻名。这款车深受铁杆发烧友的欢迎，同时也惩罚了临时车主，它可能是最受欢迎的替代内燃机设计。有无数的图表来解释它的运作，但是如果你能看到它的内部呢？

该视频由[扭曲感知]提供，以模型飞机的小型汪克尔旋转发动机为特色。发动机的端板被拆下，换上了一块透明板，使得燃烧过程清晰可见。加上一个高速摄像机，你就有了一个伟大的技术视频的配方。

它从汪克尔旋转动力循环如何运转的基本解释开始，然后切入发动机运转的精彩慢动作镜头。它还强调了制作这种类型的视频有用的几种技术，例如将周围的组件涂成黑色，以便更容易对感兴趣的部分进行成像。视觉效果令人惊叹，非常清晰地展示了发动机中进气、压缩、动力和排气冲程的工作方式。
