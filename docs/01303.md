# 不铲雪；建立一个史诗般的融化系统！

> 原文:[https://hack aday . com/2016/01/24/dont-solve-snow-build-an-epic-melt-system/](https://hackaday.com/2016/01/24/dont-shovel-snow-build-an-epic-melt-system/)

当罗纳德·沃尔特斯(Ronald Walters)在建造新房子时，他决定不再铲雪。所以他在他的新车道下建了一个融雪系统。他知道这个系统运行起来很昂贵，但他认为这比背部手术便宜，比心脏病发作便宜得多。

该系统使用钢筋和绝缘材料上的 PEX 管，所有这些都是在车道上的混凝土之前设置的。一个即开型热水器产生的热量可以融化[罗纳德的]车道上的冰雪，而不需要铲雪。当然，工作流体包括防冻剂，并且有一组泵、流量阀和流量计，以在系统运行时保持一切流动。

你可能想知道为什么你需要这么多的硬件只是为了把一些热水推到混凝土下面。记住，如果你加热混凝土太快，它会开裂。通过控制流速、输入温度和回流流体的温度，[Ronald]可以监控加热速度，以防止出现问题。

该系统在八个独立的环路中有超过 1400 英尺的 PEX。由于很难修复混凝土下面的任何东西，每个环都没有断裂或接缝。这个系统十年来一直保持[罗纳德的]硬盘畅通。

这个项目很难适应现有的车道，但也许不是不可能的。然而，你总是可以选择一个机器人吹雪机。当然，你可以学着喜欢雪，然后[播放高压音乐](http://hackaday.com/2015/01/11/massive-tesla-coil-plays-music-in-the-snow/)，直到它自己放晴。
