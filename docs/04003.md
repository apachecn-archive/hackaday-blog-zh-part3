# 不要用南瓜做你的战斗机器人

> 原文:[https://hack aday . com/2016/10/28/don ' t-make-your-the-battle bot-out-of-a-pumpkin/](https://hackaday.com/2016/10/28/dont-make-your-battlebot-out-of-a-pumpkin/)

又到了一年中的这个时候。夜晚变得越来越长，树叶开始变黄。秋高气爽，让人想到战斗机器人:[南瓜皮战斗机器人](http://www.williamosman.com/2016/10/pumpkin-combat-robot.html)。

[![pumpkin-combat-robot-1azglafagdsmkv-shot0005](../Images/9e16402f646336e3715fbe1867d6b557.png)](https://hackaday.com/wp-content/uploads/2016/10/pumpkin-combat-robot-1azglafagdsmkv-shot0005.jpg)

Kids these days can’t even draw without a computer

如果你问自己，“一个激光切割胶合板机器人，套在南瓜里，能站起来对抗一个全金属怪物吗”，你以前没见过战斗机器人。除了令人捧腹的镜头(见下面嵌入的[视频](https://www.youtube.com/watch?v=1AZgLAFAGDs)),许多构建都被记录在案，从制作南瓜的 CAD 模型到激光切割框架，到比赛前几分钟“测试”机器人。(那个*有*是个好主意！)

南瓜机器人的对手 Chomp 的镜头同样很酷。我们喜欢锤子武器加速如此之快，以至于 Chomp 实际上被举到空中，正如牛顿所预测的那样。我们不确定火武器除了展示和面对胶合板南瓜机器人之外是否还有其他用途，但我们喜欢这种效果。

回想起来，我们实际上已经雕刻了南瓜，这样司机就可以知道哪条路是向前的，它的威胁性眩光会让对手的金属心产生恐惧。但我们怀疑结果不会有太大变化。

但是我们真的很喜欢南瓜机器人。也许[威廉·奥斯曼]会考虑与同重量级的其他机器人战斗？还是[打火机](https://hackaday.com/2016/05/07/chomper-antweight-robot-kraves-combat/)？这都是激光切割，由 CAD/CAM 驱动，所以制作另一个应该是快速和容易的。有机会打开设计吗？
