# 机器人和蟋蟀

> 原文:[https://hackaday.com/2016/01/20/robots-and-crickets/](https://hackaday.com/2016/01/20/robots-and-crickets/)

如果你看科幻电影，未来的机器人看起来像我们。然而，事实是，当机器人看起来不像我们时，许多任务会进行得更好。有时它们是某项特定工作所特有的，或者有时从人类以外的事物中汲取灵感是有用的。约翰·霍普金斯大学的一名教授和一些学生决定将蜘蛛蟋蟀作为一种新型跳跃机器人的灵感来源。

在八个多月的时间里，学生们研究了蟋蟀如何能跳至 60 倍于其身长的高度并落地的运动学。诚然，它们 60 倍的身长只有大约 2.5 英尺，但如果它们有人类那么大，那就相当于跳过了整个美国足球场。

通过使用高速摄像机(每秒高达 400 帧)来确定无翼蟋蟀如何管理这些巨大的跳跃。它们显然在飞行中用四肢和触角来稳定自己。他们也使自己的身体流线型，以最大化他们的跳跃距离。研究人员创造了复制蟋蟀跳跃运动的计算机模型。他们希望利用这一知识和这些模型来设计跳高机器人，以便在崎岖的地形上旅行，而无需花费建造一个完全飞行的机器人的费用。

当然，我们已经见过一些受昆虫启发的机器人。我们甚至回忆起一只[俄罗斯机器蟑螂](http://hackaday.com/2015/09/28/robotic-cockroach-built-by-russian-university/)。虽然我们见过一种叫做沙蚤的跳跃机器人，但我们不认为它跳跃的方式模仿了它的名字。你可以在下面的视频中看到蟋蟀的跳跃。然后找点形状记忆合金，开个 Hackaday.io 项目，开始造。
