# 无尽的煎饼

> 原文：<https://hackaday.com/2016/03/11/endless-pancakes/>

有时会出现一台如此简单却又如此吸引人的机器，它的功能和实现目标的方式让你想要一台。不管它有什么作用，也不管所有权是否是一个实际的命题，你都希望你能拥有一个所有权。

你会问，什么机器会引发这种反应？[Robbie Van Der Walt]和[Christiaan Harmse]有了答案，他们的机器执行简单但重要的任务[烹饪无尽的煎饼](http://endlesspancake.com/)。一个漏斗将一层薄煎饼面糊分配到一个缓慢旋转的加热滚筒上，在它被转移到另一个滚筒上烹饪另一面之前，加热滚筒烹饪薄煎饼的一面。这看起来很简单，但这种简单背后一定隐藏着大量的产品改进，可能还有许多英里长的丢失的煎饼。煎饼似乎是一种传统的南非美食，显然他们必须有超大的胃口来满足。

在下面的视频中(南非荷兰语，英文字幕)，他们试图创造有史以来最长煎饼的世界纪录，尽管遗憾的是，他们似乎没有出现在吉尼斯世界纪录搜索中，所以他们可能没有实现。尽管如此，他们的机器是一件艺术品，我们为之喝彩。
