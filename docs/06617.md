# 3D 打印 Curta 获得升级

> 原文：<https://hackaday.com/2017/08/01/3d-printed-curta-gets-upgrades/>

令人惊奇的是，当制造者们专心做某事时，他们能完成这么多。[Marcus Wu]上传了一个关于如何构建 3D 打印 Curta 机械计算器的令人着迷的视频。经过九次设计迭代后，[Marcus]展示了一个经过打磨的设计，它不仅可以工作，而且看起来像一件杰作。

对于门外汉来说，Curta 是一个在二战时期设计的机械计算器。它仍然经常在时间-速度-距离(TSD)拉力赛中使用，以帮助计算到达检查站的时间、偏离赛道的距离等等。许多这样的集会不允许电子计算器，所以 Curta 是完美的。这个奇妙装置复杂的内部运作是一个关键特征，也是爱好者的兴趣点，这个装置本身也是一个非常受欢迎的收藏品。

至于 3D 打印设计，对细节的关注无可挑剔。目前的版本有大约 80 个需要 3D 打印的零件，还需要一些其他的螺丝和弹簧。像倒车杆和选档杆手柄这样的一些零件已经上漆，并添加了数字以完善视觉细节。组装花了[Marcus Wu]大约 40 分钟完成，是我们见过的最令人满意的构建之一。

更神奇的是，职业为软件工程师的[Markus Wu]在 [Thingiverse](https://www.thingiverse.com/thing:1943171) 上免费分享了包括原始设计文件在内的所有文件。还有一个写有书面说明的[博客](http://wudev.digitaltorque.com/)，以及迭代和原始构建的细节。我们已经在之前的版本上发布了一个[帖子，所以请查看一下更多的背景信息。](http://hackaday.com/2017/07/17/3d-printed-math-grenade/)
