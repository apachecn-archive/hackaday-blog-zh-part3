# 鸡舍门

> 原文:[https://hackaday.com/2017/07/15/irising-chicken-coop-door/](https://hackaday.com/2017/07/15/irising-chicken-coop-door/)

有什么比一扇虹膜开关的门更酷的呢？不多。他们在任何入口都添加了很好的科幻细节。[Zposner]的爸爸想为他的鸡舍装一扇自动门，所以[zposner]花了一些时间为他设计了一扇漂亮的带虹膜装置的门。你需要看视频。

[Zposner]使用激光切割和数控刳刨机的组合来切割木材，然后对木材进行打磨和喷漆。组装后，[zposner]开始研究控制机制。他用 Arduino 和电机护罩控制着门。为了让 Arduino 知道停止马达，[zposner]使用了限位开关，当机械装置旋转时，限位开关会受到撞击。一旦开关处于正确的位置，代码编写完毕，就该完成组装并在鸡笼上安装门了。为了保证 Arduino 的安全，它被安装在一个带螺旋盖的塑料容器中，然后被热粘在虹膜旁边。

不幸的是，鸡不一定在乎某样东西有多酷，在这种情况下，它们没有意识到虹膜是一扇门——它们拒绝通过它离开鸡舍。[Zposner]在决定把鸡放在门边上之前尝试了一些事情——然后鸡会意识到它可以穿过门。

[Zposner]的爸爸现在有一个时髦的门，可以用开关打开。这是一个伟大的项目，让他和他的爸爸去做，即使小鸡们看起来不为所动，他们也做得很好。看看底特律黑客空间在它的门上建造的虹膜舷窗，或者，如果你真的想建造一个虹膜机制，但没有 CNC 路由器，激光切割机，或者，你知道，木头，你可以用你周围的碎片建造 T2 这个 T3。
