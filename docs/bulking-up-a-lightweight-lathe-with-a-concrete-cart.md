# 用混凝土车组装轻型车床

> 原文：<https://hackaday.com/2017/03/05/bulking-up-a-lightweight-lathe-with-a-concrete-cart/>

谈到机床，一个很好的经验法则是越重越好。一个大的南本德车床或布里奇波特轧机可能会以可笑的重量倾斜天平，但所有这些质量都用于阻尼振动和提高性能。所以你会想，一台由汽水罐制成的车床可以利用它所能得到的所有帮助；这台浇铸混凝土机械车应该很符合要求

也许你已经看到了[我们最近的报道](http://hackaday.com/2016/07/07/the-best-gingery-lathe-video-series-to-date/)关于【Makercise】关于他的 [Gingery 车床](http://gingerybooks.com/)建造的冗长而详细的视频。如果没有，你可能想看一下[的 5 分钟压缩视频](https://www.youtube.com/watch?v=LBuUOO3qn30)，它展示了从熔化废铝铸造到第一个芯片的整个过程。我们喜欢这个建筑和视频，但木工作台上的轻型车床从未真正为我们工作过，或者对[Makercise]来说，他指出由于振动，他从未能够将车床开到全速。这辆车试图用老办法解决这个问题——增加质量。

在下面的视频中有几个“测量两次，切割一次”的时刻，以及可能变成真正灾难的高褶皱系数平板提升。我们可能会选择可以染色和抛光的台面级混凝土混合料，但那只是为了美观。当该说的都说了，该做的都做了，cart 完全做了它本来要做的事情，甚至还有空间给构建列表上的下一个整形器。我们期待着那一天。
