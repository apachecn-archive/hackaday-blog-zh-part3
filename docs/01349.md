# 密歇根勇士队又开始了

> 原文：<https://hackaday.com/2016/01/29/the-michigan-mighty-mite-rides-again/>

拥有业余无线电执照的最大好处之一就是它允许你合法建造和操作发射机。如果你想建立一个功能齐全的单边带钻机与数字模式，有它。但是从极简主义建筑中有很多乐趣，也有很多东西可以学习，比如这个密歇根巨型单晶体管 80 米波段发射机。

如果 MMM 这个名字听起来很熟悉，可能是因为[最近的这篇文章](http://hackaday.com/2015/09/29/minimal-mighty-mite/)。事实上，[w2ew]的建造灵感来自同样的 SolderSmoke 博客帖子，这些帖子使[Paul Hodges]开始了他的试验板和啤酒罐建造之路。[w2ew]的构建确实有点时髦，但该视频真正的亮点在于对信号质量的探索和改进。基本的 Mighty-Mite 输出了一个相当脏的信号——[ w2ew]的示波器显示了 5 个主要的谐波尖峰，本应是一个不错的正弦波却充满了草皮和坑洞。一个晶体管、一个彩色脉冲晶体和几个电容只能做这么多，因此视频向我们解释了低通滤波器的设计，该滤波器需要消除谐波，并将输出清理为良好的固态正弦波。

如果你的莫尔斯技能没有发挥出应有的作用，无法利用 may-Mite 的 CW-only 模式，那么你需要考虑其他的调制方式。也许一个微型调频发射机会更适合你的需要？
