# 周五黑客聊天:Ladyada 关于创新和互动机器人

> 原文：<https://hackaday.com/2018/06/20/friday-hack-chat-ladyada-on-creative-and-interactive-robotics/>

在微控制器、开源工具链、创客运动和技术进步的交叉点上，有一个模糊的概念，可以最好地描述为机器人或物理计算。这些项目与外部世界互动，而不是盒子里的计算机或愚蠢的微控制器。无论是通过互联网，点击一堆传感器，还是只是挥舞着伺服臂，都需要一个平台来完成所有这些事情。在本周的 Hack Chat 中，[我们将谈论所有关于创新和互动机器人的话题](https://hackaday.io/event/159006-creative-and-interactive-robotics-chat)，你也受到了邀请。

我们本周黑客聊天的嘉宾是 Limor“lady ada”Fried，Adafruit 的创始人，一个不需要介绍的人*，但是我们还是要做*。Adafruit 最初是一个奇怪的兼职项目，销售 Roland TB-303 的精确复制品，制作手机干扰器，并在 USB 电源银行出现之前制作指南来建立自己的 USB 电源银行。这已经发展成为 Adafruit，一家在纽约市中心拥有 100 多名员工的公司，纽约市中心是学习和制造电子产品的最佳地点之一，也是一个以零贷款或风投资金做一切开源工作的地方。无论用什么客观标准来衡量，Adafruit 都已经成为创客运动中最成功的商业故事，无论这一定义是多么模糊。

本周的黑客聊天将聚焦于 [CRICKIT，这是一个创造性的机器人和交互式建筑工具包。](https://www.adafruit.com/crickit)Crick it 是 Adafruit 的 Circuit Playground 的附加产品，允许你用 CircuitPython、MakeCode 或 Arduino 构建自己的机器人。它支持艺术、工艺、传感器、音频、电子动画、物理计算、动态雕塑、科学实验，以及你能想到的任何东西。需要举例吗？[这里是蓝色奥斯特邪教](https://www.youtube.com/watch?v=sjisA9-YyrM)。这是 NES 的[机器人。一切都很有趣。](https://www.youtube.com/watch?v=ffAuebA5WAo)
