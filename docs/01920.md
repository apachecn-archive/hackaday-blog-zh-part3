# 哦不！是爪子！

> 原文：<https://hackaday.com/2016/04/08/oh-no-its-the-claw/>

有什么比爪机更诱人的？毕竟，从一堆商品中挑出一只 2 美元的泰迪熊能有多难？但是 20 个季度后，你意识到你已经花了 5 美元，你仍然没有任何东西可以展示。

[CreativeGuy88]决定[建造他自己的抓爪机](http://www.instructables.com/id/DIY-CLAW-MACHINE/)(这样，他就可以保留硬币)。这个庞大的建筑是一个木工工程。然而，马达和控制操纵杆需要电线，而且[CreativeGuy88]使用乐高积木来制作车厢的大部分。

他唯一没有从头开始创造的部分是真正的爪子。相反，他买了一个商用机器的替换爪。它使用电磁铁来接合把手。你可以在下面看到一段机器运行的视频。

我们过去已经看过[爪机造](http://hackaday.com/2015/09/14/full-size-custom-claw-machine-built-with-parts-on-hand/)。我们甚至报道过[已经连接到互联网](http://hackaday.com/2015/12/24/here-comes-santa-claws/)。
