# Arduino 动力刀挥舞触手会让你忍俊不禁

> 原文：<https://hackaday.com/2015/11/30/arduino-powered-knife-wielding-tentacle-will-leave-you-in-stitches/>

为 Hackaday 写文章，我们看到有趣的项目，也看到危险的项目。很难找到将两者结合起来的项目。这家伙不知何故成功了。【Outaspaceman】熟悉 LittleBits，但刚开始学习 Arduino 编程。他完成了眨眼教程，但闪烁 LED 并不足以让他的第一个 Arduino 程序获得成功。他将 Arduino Mega 的 LED 输出连接到一对 LittleBits，然后这对 little bits 在两个位置之间切换伺服系统。一个光秃秃的伺服不会很有趣，所以[Outaspaceman] [连接了一个触手和一把小小的瑞士军刀](https://www.youtube.com/watch?v=pQ2dI_B_Ycg)。是的，一把刀。

有问题的触手被设计成一个手指木偶。有一些关于一个触手挥舞着一把刀的事情是如此滑稽和荒谬，以至于我们忍不住笑了。显然，我们并不孤单，因为这个视频已经有超过 100 万的浏览量。这几乎就像是[最没用的机器](http://hackaday.com/2012/10/22/battling-most-useless-machine-gets-an-expressive-upgrade/)的暴力报复。对于技术上的好奇，触手看似随机的运动类似于[双摆](https://en.wikipedia.org/wiki/Double_pendulum)。

我们的读者会很高兴地知道，[Outaspaceman] [已经完成了 Arduino 伺服教程](https://www.youtube.com/watch?v=sYZLyafhBm4)，现在直接控制伺服，不需要一点点。我们只是希望他有一个好方法来关闭他的创作——不需要缝针。
