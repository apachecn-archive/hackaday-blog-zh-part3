# 下一级自制飞行模拟器

> 原文：<https://hackaday.com/2016/11/26/a-next-level-home-built-flight-simulator/>

每个爱好都需要有几个人，他们做得有点过火。特别是航空爱好——无线电控制飞行、FPV 多翼飞机竞赛等——似乎激发了更多的核心建设。为了证明这一点，我们向大家展示这个顶级的自制飞行模拟器。

他的妻子和朋友认为他疯了，我们也同意。但是[XPilotSimPro]是一种特殊的疯狂，它推动了艺术的发展，我们为他鼓掌。作为一个飞行模拟器游戏的长期粉丝，他很幸运地在一个真实的 737 模拟器上记录了一些时间。那似乎是他染上 DIY 癖的地方。休息后的视频是他建造的主要部分的旋风之旅，它不寻求忠实地复制任何特定的驾驶舱，而是创造一个似乎令人敬畏的驾驶舱。驾驶舱建立在带有胶合板面板的 PVC 管架上，布满了液晶面板、飞行仪器和航空电子设备舱，看起来像是从驾驶舱出来的。模拟器面对着一面墙，上面有一个液晶投影仪，可以看到外面的世界。最近增加了一个高架面板，展示了更多的液晶面板和仪器。整个事情由一个运行 X-Plane 的大型游戏平台驱动，允许 XPilotSimPro 接受任何航空挑战，包括 T2 将一架 Embraer 109 降落在尼米兹号航空母舰 T3 的甲板上。

[XPilotSimPro]的模拟器下一步会是什么？[加一点气动的运动控制](https://hackaday.com/2014/02/08/pneumatic-powered-flight-simulator/)怎么样？或者更好的是，用[一个真实的 737 驾驶舱作为模拟器](https://hackaday.com/2012/04/20/a-flight-simulator-made-out-of-a-real-737-cockpit/)怎么样？
