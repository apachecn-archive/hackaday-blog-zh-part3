# 专业烟火表演的幕后

> 原文：<https://hackaday.com/2017/07/14/behind-the-scenes-at-a-professional-fireworks-show/>

你有没有想过大型烟火表演的幕后发生了什么？去年[Kenneth]被要求帮助手动点燃一场烟花表演，这包括他用一个道路照明弹跑下一排炮弹，依次点燃每一个。他显然做得非常好，以至于今年[又出演了另一部剧](http://blog.thelifeofkenneth.com/2017/07/working-professional-fireworks-show.html)，这部剧的布景更加复杂。

在[肯尼斯]的帮助下，这场演出由 950 个 3 英寸的贝壳组成，这些贝壳被串联成小组，另外还有 150 个 25 个贝壳组成的贝壳群用于压轴演出。烟花被放在由 5 个直径 3 英寸的 HDPE 管组成的架子上，用 2x4s 固定在一起。每根管子里都有一个外壳，每个外壳都预先装了一根导火索和一个电触发的雷管。每个引爆管或一系列引爆管连接到 45 通道分线点，这些分线点连接到控制板。

即使在演出结束后，[肯尼斯]还有工作要做，四处走动，查看每个试管，看看是否有任何未发射的炮弹。双重布线是这样的，如果雷管是哑弹，可以用照明弹发射炮弹。在这场表演中，他们发现了六枚炮弹，而[Kenneth]的任务是用道路照明弹引爆最后的炮弹——否则，他们就不得不使用一辆有牌照和标牌的车辆来运输几枚炮弹。

想了解更多烟花的优点，请查看这款漂亮的 [Arduino 烟花控制器](http://hackaday.com/2015/08/17/beautiful-arduino-fireworks-controller/)和这款[网络控制烟花发射器](http://hackaday.com/2013/04/13/network-controlled-fireworks-launcher/)。
