# 成为人类最好的朋友的朋友

> 原文:[https://hack aday . com/2017/06/09/成为人类最好的朋友/](https://hackaday.com/2017/06/09/being-a-friend-to-mans-best-friend/)

当[杰森·多利]向我们透露这一点时，他说，“这勉强称得上是一次黑客攻击。”我们不同意，就像其他爱狗人士看到一个简单的改变情绪的狗床旋转木马如何改善了他的狗的生活一样。

[Jason]的黑客技术并不在于旋转狗床——它只是一个由几个 Arlo 机器人轮子驱动的轴承上的胶合板平台。黑客更多的是搞清楚狗需要什么。你看，[瑟伯]是一只老狗，像许多长寿的好朋友一样，他开始表现出行为变化，包括无休止地在同一条环形路径上踱步，直到筋疲力尽。在老狗身上打圈往往是[犬认知功能障碍](http://www.petmd.com/dog/conditions/neurological/5-signs-dog-dementia)的症状，基本上就是狗版的老年痴呆症。认为旋转本身是令人宽慰的，[杰森]手动将[瑟伯]的狗床放在地板上。[瑟伯]立即平静下来，所以苦乐参半命名为“痴呆症-去-圆”建成。

可悲的是，[瑟伯]实际上患有脑瘤，但他仍然很喜欢旋转，这给他在最后几天带来了一些平静。寻找帮助人类痴呆症的方法？在 T1 之前，我们也经历过很多这样的事情。
