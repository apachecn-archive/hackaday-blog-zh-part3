# 微波炉以外的微波东西！

> 原文：<https://hackaday.com/2015/09/11/microwaving-things-outside-of-the-microwave/>

**免责声明:** **请勿在家尝试！**

有没有想过，如果你把一个磁控管从微波炉里拿出来，绑在一根棍子上，然后把它指向随机的日常物品，会发生什么？好吧，你很幸运，几个年轻的俄罗斯人[在他们可能伤害自己和住院之前记录了他们的研究](http://www.youtube.com/watch?v=BGDOE-zDVZc),或者可能只是不育——或者引起白内障——重点是不要做。

他们从微波炉中取出一个磁控管，并在旁边绑上一个汤罐来聚焦微波。发现这一点后，他们做了下一个合乎逻辑的步骤——将它指向不同的东西，看看会发生什么。

在这个特定的视频(他们有很多…)中，你可以看到他们在灯泡内产生等离子体，融化灯泡，点亮真空管，荧光灯泡，液化金属，甚至瞥见无线电波。
