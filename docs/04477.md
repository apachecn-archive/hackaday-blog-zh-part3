# 智能手机将摧毁你的空气曲棍球

> 原文:[https://hack aday . com/2016/12/15/smart phone-will-destroy-you-at-air-hockey/](https://hackaday.com/2016/12/15/smartphone-will-destroy-you-at-air-hockey/)

我们大多数人都在口袋里带着一台功能强大的电脑，除了浏览网页、社交媒体和偶尔打个电话，我们很少用它。我们的手机是技术奇迹，但它们的潜力有时似乎被浪费了。

看到利用手机驱动一些具体硬件的东西总是令人愉快的。[何塞·胡里奥]的项目就是这样做的，[使用手机作为机器人空气曲棍球桌的大脑](http://jjrobots.com/air-hockey-robot-evo/)。

记性好的读者会记得【Jose】以前的[空气曲棍球桌](https://jjrobots.com/air-hockey-robot-a-3d-printer-hack/)，使用由 Arduino Mega 控制的 3D 打印机组件，在比赛场地上方悬挂一个网络摄像头。这个版本将相机、机器视觉和游戏策略转移到一个 Android 应用程序，让 Arduino 在无线网络的指挥下控制硬件。

你可以在休息时间下面的视频中看到的结果是一个节奏极快的游戏，机器人看起来不可战胜。如果你想自己开发，GitHub 上有[完整的指令](http://jjrobots.com/air-hockey-robot-evo-assembly-guide-v-1-2/)和[代码，或者如果你点击上面链接的页面，他会把这个项目作为一个工具包出售。](https://github.com/jjrobots)
