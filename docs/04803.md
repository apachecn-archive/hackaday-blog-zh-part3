# [玛拉]的新手臂

> 原文:[https://hackaday.com/2017/01/19/marlas-new-arm/](https://hackaday.com/2017/01/19/marlas-new-arm/)

在主流媒体上很少看到涉及 hackspace 的报道，所以昨天当撰写这篇文章的当地电视新闻报道了一个故事，不仅突出了 hackspace 的工作，而且是以非常积极的方式进行报道时，这是一个惊喜。

【Marla Trigwell】是一个来自英国纽伯里的年轻女孩，她天生没有左手。她已经得到了假肢，但它们并不便宜，作为一个成长中的孩子，她很快就离开了他们。她的父母像现代父母一样研究这个问题，并发现 3D 打印假肢的最新进展降低了像[Marla]这样天生没有四肢的人的门槛。上个月[【Marla】收到了她的新 3D 打印手臂](http://www.newburytoday.co.uk/news/home/20361/new-hand-opens-up-newbury-girl-s-world.html)，她这样做是出于[纽伯里和地区黑客空间](http://www.nadhack.org.uk/)的【Andrew Lindsay】的工作。

手臂本身是一个[团队未绑定的手臂版本 2.0 Alfie edition](http://www.teamunlimbited.org/the-unlimbited-arm-20-alfie-edition-current/) ，可以在 Thingiverse 上找到[，其中有完整的尺寸说明，用于在定制器中调整到接受者。正如休息时间下面的视频所示，[玛拉]似乎对它非常满意，很快就掌握了它抓住物体的能力。](http://www.thingiverse.com/thing:1672381)

这个故事很好地展示了 hackspace 作为一个真正的社区组织的力量。我们为[Andrew]、NADHack 和所有参与他们工作的人鼓掌，并希望 3D 打印手臂能在[Marla]成长的过程中不断为她提供舒适和负担得起的假肢。
