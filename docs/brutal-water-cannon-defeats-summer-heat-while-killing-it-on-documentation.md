# 残酷的水炮打败了酷暑；在文档上杀死它

> 原文：<https://hackaday.com/2016/06/21/brutal-water-cannon-defeats-summer-heat-while-killing-it-on-documentation/>

战争正在进行，虽然这种夸张的喷水器肯定是水战军备竞赛的升级，但这不是我们所指的战争。我们说的是文档战。黑客们，你们被通知了。

如果你想知道一个项目应该如何被记录，看看[Tim]在 WaterWar.net 的论坛帖子就知道了。从包含目录号和供应商网站链接的极其详细的 BOM，到带有零件号标注的大量构建照片，再到非常详细的构建说明，[Tim]提高了记录任何类型构建的任何人的风险。

这还没有触及 blaster 本身的优点，它有空气和水箱，每一个可以想象的阀门和配件。甚至还有一个由束状苏打吸管制成的直列式气流矫直器，以尽可能保持层流状态。看起来[蒂姆]和他的同事们痴迷于尽可能远地发射水流，尽管恶劣的天气使官方测量至今无法进行，但从下面的视频来看，他肯定看起来像是用一股几乎完好无损的水流覆盖了一段很长的距离，以将全部爆炸传递到预定目标，而不会丢失一滴。

尽管增强了的水枪看起来很有趣，但我们之前并没有在这些页面上看到太多的优雅。很久以前[我们报道过一个 DIY 超级浸泡器](https://hackaday.com/2010/03/20/diy-super-soaker/)。对于不像[蒂姆]的杰作那么复杂的东西，你可以在一个下午组装好[这支加压水枪](http://hackaday.com/2010/06/11/pressurized-pvc-water-gun/)。
