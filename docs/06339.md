# 最佳参赛产品:开源剂量计互联网

> 原文:[https://hack aday . com/2017/07/08/hack aday-prize-entry-open-source-IOT-剂量计/](https://hackaday.com/2017/07/08/hackaday-prize-entry-open-source-iot-dosimeter/)

[Radu Motisan]将一个很酷的项目加入了今年 Hackaday 奖的最佳产品部分。它被称为[开源物联网剂量计](https://hackaday.io/project/7935-open-source-iot-platform/)。它有一个用于检测辐射水平的盖革管，以及互联网连接和许多其他好东西。

这款物联网剂量计被称为 KIT1，可用作便携式辐射探测器，其诺基亚 5110 LCD 作为输出，或用作以太网监控站。通过内置扬声器，它可以提醒用户辐射超标的区域。KIT1 是一个功能齐全的系统，不需要计算机来获取读数，这使得它非常方便易用。它还具有额外传感器的扩展空间，允许完全定制的系统。该项目包括所有的 Gerbers 和一个 BOM，因此您可以将其发送到您选择的 PCB fab 实验室，焊接一些元件，并拥有一个全功能的物联网剂量计。你甚至不需要液晶显示器或以太网；您可以从两个输出中选择您喜欢的输出，并使用它来节省一些钱。

这是一个伟大的项目，现在谁不需要 IOT 剂量计呢？
