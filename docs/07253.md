# LED 公主裙也照亮了女孩的脸

> 原文:[https://hack aday . com/2017/09/30/led-princess-dress-also-lights-up-girls-face/](https://hackaday.com/2017/09/30/led-princess-dress-also-lights-up-girls-face/)

我们很确定[卢克]在去年万圣节拿了年度叔叔奖，当时他为他的侄女做了一件 RGB LED 公主裙。他最近找到时间用一个全面的操作方法来记录这个构建，正好赶上 2017 年万圣节。

[卢克]使系统模块化，以便他的侄女可以用它来搭配任何衣服。RGB LED 灯条实际上是固定在衬裙的里面——一种穿在裙子下面的蓬松的衬裙。发光二极管面向身体，有助于漫射光线。[Luke]首先用他们自己的粘合剂把带子粘上，然后花很多时间把它们缝起来，这样它们就不会动了。在某种程度上，他发现热胶水也同样有效。

这个项目最酷的部分(当然除了闪光灯)是电源。[Luke]用他已经有的东西:一个 18V 的 Ryobi 电池组。他用 Shapeways 的一个印刷帽给它连接了一个步进转换器，该帽用于将金属夹连接到电池触点。这个上限确实使这些包对许多需要长期可移植性的项目非常有用。

这些电池的额定功率为 240W，考虑到负载，这是大材小用。但这是有原因的:它将热量保持在最低水平，因为电子设备藏在一个可爱的小背包里。说到可爱，休息之后你可以看到他的侄女在做裙子的模特。
