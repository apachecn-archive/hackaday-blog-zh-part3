# 黑客日奖参赛作品:黑客微积分

> 原文：<https://hackaday.com/2017/05/01/hackaday-prize-entry-hacker-calculus/>

学校教授的数学有时不能完成教育学生的使命。这是[琼·霍瓦特]和[里奇·卡梅隆]的观点，特别是关于微积分的教学，他们觉得微积分已经成为一门纯粹的代数学科，在理解背后的概念时，许多学生都被冷落了。

他们的[黑客微积分](https://hackaday.io/project/20621-hacker-calculus)项目旨在通过回到【艾萨克·牛顿】1687 年在这个问题上的开创性工作 *[自然哲学数学原理](https://cudl.lib.cam.ac.uk/view/PR-ADV-B-00039-00001)来解决这个问题。*他们被*原理*在多大程度上是几何而不是代数的作品所震惊，他们正在寻求回归【牛顿】的原理，试图让落后的学生在涉及导数和积分时更容易理解这门学科。他们打算改进几何方法，通过 3D 打印物品和电子设备创造一系列实用物品来解释概念。

我们可以看到，这是一种有相当大价值的方法，因为大多数黑客读者有时会耐着性子听完一堂数学课，离开时会想知道老师到底在说什么，并被通过难以理解的数学语言解释它的进一步尝试所困惑。如果你是在物理课上解释速度和加速度之间的关系时“理解”微积分的孩子，那么你可能会理解。

这两人有一些获奖历史，你可能会记得他们以前的参赛作品，比如他们去年为视障人士项目制作的 3D 照片。
