# 如何从零开始制作定制液晶显示器

> 原文：<https://hackaday.com/2016/06/17/how-to-make-a-custom-lcd-from-scratch/>

如果你曾经想知道从零开始制作自己的定制图形液晶显示器需要什么，[【应用科学】](https://www.youtube.com/watch?v=_zoeeR3geTA)的这段视频值得一看。它简明扼要，同时仍然告诉你你需要知道什么，如果你有兴趣推出自己的。还有一个相关的视频深入[更详细的](https://www.youtube.com/watch?v=d4QFNWBSZYg)关于液晶技术的实验。

【应用科学】使用从网上购买的显微镜载玻片和部件制作了一个 LCD，当激活时，它会显示自定义图形。家庭实验者可能难以遵循的唯一步骤是在载玻片上涂上一层透明导电层，在视频中，这是通过一种名为[溅射](https://en.wikipedia.org/wiki/Sputtering)的过程沉积一层薄膜来完成的。但是，您不需要亲自做这件事。预涂玻璃在网上很容易买到。(搜索氧化铟锡或“ITO”涂层玻璃。)

LCD 由悬浮在两层导电玻璃之间的一层液晶组成。电场用于改变悬浮液中晶体的取向，从而调制通过晶体的光。偏振滤光器产生鲜明的对比，从而形成可视图像。为了显示特定的形状，将一些导电涂层从一个层上去除，形成所需图像的形状。应用科学用来做这件事的过程几乎和蚀刻一个定制的印刷电路板一样。
