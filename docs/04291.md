# 在浴缸里寻找红色十月

> 原文：<https://hackaday.com/2016/11/30/the-hunt-for-red-october-in-a-bathtub/>

任何东西都可以是遥控飞机，而“直升机——无论是不是四轴飞行器——只需让空气屈服。遥控车很容易，遥控坦克甚至可以举枪射击。一种类型的车辆在很大程度上避开了遥控爱好者；建造一艘遥控潜艇很难。你不仅要担心浮力，还需要一种方法来保持干燥的部分干燥，同时还要在无线电工作不太好的环境中工作。

[Ivan]已经制造了遥控飞机，但想要应对新的挑战。他建造了一艘遥控潜水艇，而且他使用了最简单的家用材料。

建造潜艇有三个主要挑战。这些是无线电系统，建造了一个水密的外壳，并有某种压载舱来抽水进出。对于无线电系统，现在无处不在的 2.4 GHz 无线电无法在水下工作。那些在 27、40 或 72 兆赫上工作的老式无线电是有用的，在有人听说了[伊万]的潜艇项目后，一些垃圾抽屉被清理出来，为这艘潜艇安装了无线电系统。

水密性是用一根直径为 75 毫米的 PVC 管实现的，两端都用塞子和 o 型圈盖住。我们在互联网上读到，使用 PVC 管作为压力容器造成的死亡人数超过了数千年的瘟疫和战争；在这种情况下，它几乎起作用了 PVC 接头不是不可避免地爆炸，而是当被放在浴缸中时，有*内爆*的危险。这是一种完全不同的失败模式，即使是最直言不讳的批评家也会消除他们的恐惧。

能下到水下不代表什么，除非你能下潜再浮出水面。为了做到这一点，[伊万]的潜艇正在使用一个连接到蠕动泵的大注射器。为了潜水，[伊万]命令潜艇将水泵入注射器。为了再次浮出水面，接头将注射器中的水抽出。
