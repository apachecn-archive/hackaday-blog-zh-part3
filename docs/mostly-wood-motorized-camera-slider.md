# 主要是木头电动相机滑块

> 原文：<https://hackaday.com/2015/12/26/mostly-wood-motorized-camera-slider/>

相机幻灯片可以在视频中制作有趣的移动镜头，或者给单调的延时镜头增添趣味。当轮到他自己的时候，[鲍勃]做了任何一个黑客都会做的事情，在木材商店里推出了他自己的电动相机滑梯。

我们总是喜欢看到基于黑客自己的现有技术的作品，而[Bob]成功地利用了他的[即兴爪机器构建](http://hackaday.com/2015/09/14/full-size-custom-claw-machine-built-with-parts-on-hand/)中的部件和技术来制作这个滑块。这个项目中的滚轴使用与之前相同的 3/4″角钢和滑板轴承。轴承在胶合板条上滚动，胶合板条上覆盖有相同的角钢，以提高耐用性和降低摩擦。步进电机支架和轴台是 3D 打印的，正时滑轮也是。[Bob]承认整个装备有点吵，并将其归咎于滑轮印刷的粗糙质量。他计划用市场上可以买到的滑轮来代替它们，这应该会有所帮助；我们的另一个建议是在 ATtiny85 中编码一个软启动算法，以消除您在他演示下面视频中的滑块时看到的抖动。

有很多种方法可以沿着一个轴移动相机，其中有相当多的方法使用了滚轴运动的部件。我们之前已经报道过不少了，比如[这个使用滑板卡车的滑梯](http://hackaday.com/2015/04/09/camera-slider-utilizes-skateboard-trucks/)，或者[这个由栅栏柱和直排轮组成的非机动装置](http://hackaday.com/2014/01/27/yet-another-diy-camera-slider/)。
