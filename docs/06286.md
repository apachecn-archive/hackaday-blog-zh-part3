# 铜、黄铜、桃花心木和玻璃相结合的时钟具有复古的外观

> 原文:[https://hack aday . com/2017/06/28/铜-黄铜-桃花心木-玻璃-时钟结合复古风格/](https://hackaday.com/2017/06/28/copper-brass-mahogany-and-glass-combine-in-clock-with-a-vintage-look/)

没有两个词比“谢妮”和“蒸汽朋克”更能让普通黑客读者倒胃口但你不是普通的黑客读者，所以如果你对一款可爱的手工制作的手表感兴趣，那就继续读下去吧，这款手表融合了现代电子产品和复古材料。但是不要把它当成谢妮蒸汽朋克钟。

无论你对蒸汽朋克风格有什么看法，你都不得不佩服[Aeon Junophor]的钟表作品，以及他的执着——他在 2014 年开始制作这款钟表，现在才刚刚完成。我们敢打赌，很多时间都花在了寻找合适的材料上。主体和腿是铜管和一些黄铜灯部件，IN-12A Nixies 的连接器是铜马桶水箱部件和黄铜爱迪生灯泡底座，底座是一块精美的红木。整件事情有一个很好的乔治·帕尔的*时间机器*的氛围，Instructables 写的是一个伪维多利亚风格，我们觉得很迷人。

如果你还没有看够谢妮蒸汽朋克的融合，看看这个谢妮太阳能监控器或者这个黄铜和谢妮钟。不要羞于向我们发送这种风格的构建技巧——我们不做评判。
