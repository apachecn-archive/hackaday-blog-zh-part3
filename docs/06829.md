# 冰箱压缩机到二冲程发动机:JB 焊接赢得胜利

> 原文:[https://hack aday . com/2017/08/16/冰箱-压缩机-二冲程发动机-JB-焊接-赢/](https://hackaday.com/2017/08/16/fridge-compressor-to-2-stroke-engine-jb-weld-for-the-win/)

我们喜欢这款车，因为它有一种真正的垃圾场大战的感觉:[将废弃的冰箱压缩机变成二冲程内燃机](https://www.youtube.com/playlist?list=PLjzL2Xfa7hB0riz1XVFIP3wvLGkyTHKCD)。[Makerj101]使用的工具不比钢锯和手钻复杂。还有 JB Weld——许许多多的 JB Weld。

[Makerj101]的视频系列带我们了解他的整个转换过程。尽管压缩机和发动机之间的外观相似，但有足够多的关键差异，使转换具有挑战性。必须实施控制进气和排气的方案，曲轴箱需要密封，需要制造带火花塞的气缸盖。所有这些步骤在有铣床和车床的机械车间都是微不足道的，但是[Makerj101]选择了艰难的道路。旧的 CPU 散热器充当气缸盖，铜线形成气缸垫和垫片以降低压缩比，旧的电机转子充当飞轮。JB Weld 到处都是，效果很好，正如下面视频中的测试所示。

觉得你认得【Makerj101】？你可能知道，因为我们介绍了[他以前的无机械车间发动机制造](http://hackaday.com/2015/12/22/homemade-internal-combustion-engine-sans-machine-shop/)。这家伙肯定从一管 JB Weld 中得到他的钱的价值。
