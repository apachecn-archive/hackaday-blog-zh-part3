# 中继计算

> 原文：<https://hackaday.com/2017/01/19/relay-computing/>

最近，[Manuel]发表了一篇关于用任何东西制作[逻辑门的文章。他提到了一个关于中继逻辑的网站。虽然确实可以使用开关逻辑构建逻辑门(也就是说，两个串联的开关是与门，两个并联的是或门)，但这不是唯一的方法。如果你要连接一个大电路，使用常规模块会有一些好处。许多基于分立开关元件的计算机是这样工作的:你有一个包含一些基本门(比如说，一个双输入与非门)的 PCB，然后逻辑就在于你如何将它们连接在一起。在这种情况下，SPDT 继电器被用作双输入多路复用器(或 mux)。](http://hackaday.com/2017/01/03/make-logic-gates-out-of-almost-anything/)

## Mux 狂热

如何从双输入多路复用器到任意逻辑门？很简单，如果你注意了横幅图像。(或者尝试一下[互动](http://tinyurl.com/h6ynfds))。多路复用符号将输入显示在左侧，输出显示在右侧，选择输入显示在底部。如果选择为零，则“0”输入成为输出。如果选择是 1，则“1”输入路由到输出。

显然，具有两个电极的继电器可以像两个门一样工作(底部的输入必须与两个门相同)。你也可以算出一个缓冲器(交换非门的输入)，或者 A 或 NOT B 和 A 和 NOT B。

也可以对继电器进行“有线或”运算。例如，假设你有十个与门，如上图所示。如果您想将输出连接在一起，只需连接输出线。任何一个(或多个)与门触发将驱动输出高。或者，你可以让地面为 1，浮动高点。这样做的好处是，可以更好地与 IC 和其他电路配合工作，这些电路吸收的电流可能比产生的电流多。那么继电器线圈总是连接到正电源，你需要接地来完成电路。

## 去吧！

如果你曾经考虑过用继电器制造一台计算机，这应该会给你很多启发。请记住，继电器看似简单:它们是由线圈和弹簧钢条制成的非理想设备。例如，触点之间的电弧是不好的，对吗？看情况。一些接触材料依靠电弧来清除腐蚀。其他人只是坑和失败。继电器有很多微妙之处，它们的很多不可靠性实际上只是被误用了。当然，这并不是说它们像现代半导体器件一样可靠，但制作精良的继电器，如果结构适合其预期应用，可能相当可靠。
