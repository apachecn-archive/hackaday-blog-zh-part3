# Hackaday 奖参赛作品:通过头部手势进行远程控制

> 原文:[https://hack aday . com/2017/09/03/hack aday-prize-entry-remote-control-by-head-gestures/](https://hackaday.com/2017/09/03/hackaday-prize-entry-remote-control-by-head-gestures/)

当找不到电视遥控器时，有些人可能会认为他们今天过得很糟糕。然而，有些人甚至连遥控器都不会拿，更不用说在沙发垫里翻来翻去了，遥控器不可避免地会卷到沙发垫上。2017 年 Hackaday 奖辅助技术阶段的参赛作品旨在帮助这些人，通过头部手势触发的通用遥控器[。](https://hackaday.io/project/26830-tv-remote-control-based-on-head-movements)

行动障碍的范围从精细运动控制问题到四肢瘫痪，患有这些障碍的人经常因为无法操作设备而与技术隔绝。[Cassio Batista]在他的项目中专注于控制一台电视，但很容易看到他的方法如何与其他红外遥控器接口，以实现对从警报系统到窗户和窗帘的控制。他的开源项目使用网络摄像头来观察用户的头部手势，在 SBC 芯片上运行的 OpenCV 寻找俯仰、偏航和滚动轴的运动，以控制音量、频道和功率。Arduino 负责向电视发送红外命令。原型在下面的视频中运行良好；借助 OpenCV 的强大功能，我们可以想象嘴部手势，甚至眨眼都可以增加控制器的功能。

辅助技术阶段将于明天结束，所以请务必提交您的参赛作品。你会遇到一些激烈的竞争，比如这个机器人外骨骼。但不要因此而气馁。
