# 垃圾桶盖上的简易抛物面镜

> 原文：<https://hackaday.com/2016/11/01/easy-parabolic-mirror-from-a-trash-can-lid/>

太阳能应用的抛物面反射器是很好的东西，制作自己的反射器本身就是一个伟大的项目。我们见过的最简单的方法之一是[绿色能源科学]，他只使用垃圾桶盖、聚酯薄膜和胶带。不过，你需要一种方法来制造部分真空。

这个想法非常简单，就像作弊一样。剪一圈比盖子稍大的聚酯薄膜，用胶带把它包起来，在这个过程中要注意拉伸聚酯薄膜。做完这些后，你就得到一面漂亮的平面镜了。这是需要真空来迫使薄膜成抛物线形状的地方。从盖子上预先钻好的一个小孔中抽出空气，并用胶带封住以防止真空丧失。聚酯薄膜上的大气压力会完成这项工作，神奇的是，你会得到一个近乎抛物面的反射镜。

在另一个视频中，你可以看到反射器在燃烧东西。这种技术的一个明显的问题是在一段时间后，大约一个小时，根据作者的说法，真空的损失。这里有[另一种方法](https://hackaday.com/2016/07/26/pressure-formed-parabolic-mirror-from-a-mylar-blanket/)来制作一个更耐用的镜子，也是用聚脂薄膜作为反射元件，但是最终的镜子质量并不好。
