# 丽塔的洋娃娃可能比你活得更好

> 原文：<https://hackaday.com/2016/09/06/ritas-dolls-probably-live-better-than-you-do/>

如果不是因为你可能要永远听下去的古怪的荷兰-挪威电子音乐，[Gianni B.]为他的女儿设计的[玩偶屋，[Rita]让生活在芭比世界看起来是一件值得努力的事情。真正的现代形式，它有 LED 照明。它有很多。它有一个应用程序和一个电梯。它甚至有一台可以工作的微型电视。](http://www.settorezero.com/wordpress/ritas-dolls-house-una-casa-delle-bambole-controllata-via-bluetooth-dal-tablet-o-dal-cellulare/)

这一切都始于一个圣诞愿望。[丽塔]再也无法忍受想象她的芭比娃娃在她的地板上过着无家可归的生活，向路过的玩具乞讨足够的大富翁的钱来买一只袜子睡觉。然而，当[Gianni]拜访通常的嫌疑人购买玩具屋时，他发现它们令人失望且昂贵。

所以，按照传统的与圣诞老人合作的策略，他和他的家人很高兴合作了一个玩具屋开发项目。每个房间都由四个超高亮度发光二极管照明。有一部由 H 桥模块控制的电梯，被改装成有电子制动功能。[丽塔]还没有芭比博士，所以安全是最重要的。

家庭自动化的大脑是一个带蓝牙模块的 PIC 微处理器。他为它写了一些代码，[这里有](https://github.com/Cyb3rn0id/rita-s_dolls_house)。他还更进一步，使用[麻省理工学院的 scratch](http://hackaday.com/2013/02/24/reading-sensors-with-scratch/) 为玩偶之家制作了一个应用程序界面。休息过后，你可以在视频中看到它的效果。最后一个小问题是电视。一个旧的 arduino，一个 SD 卡盾，和一个微小的 2.4 英寸 TFT 结合在一起，构成了一个微小的数字相框。

他的女儿对他们的洋娃娃的经济等级的提升欣喜若狂，一位自豪的父亲甚至在一次制造商博览会上展示了它。非常好！
