# 16 通道采样器测试街机按钮的风格

> 原文：<https://hackaday.com/2016/08/16/16-channel-sampler-tests-arcade-buttons-with-style/>

目标很简单:测试一堆来自不同制造商的街机按钮，以获得具有最佳功能和感觉的按钮。最终的构建一点也不简单:[这是一个精心设计的 16 通道 WAV 采样器和混合器](https://imgur.com/gallery/HwXSs#FbuDb1m)。

对于那些想知道为什么[Atarity]会这么麻烦地测试街机按钮的人，我们怀疑是别有用心的——跳到下面长视频的 21:14 标记，看看真正的设计灵感。无论动机如何，毫无疑问的是对制作的关注——CNC 铣削桦木外壳，极其详细的激光雕刻图形，以及覆盖有麂皮的碳纤维背板，因为麂皮。我们尤其喜欢扬声器护栅上的细节:刺绣面料和膨胀的外观与设计的其他部分非常协调，包括皮革手带。

从帖子中还不完全清楚测试的最终目标是什么，但我们假设它将是某种 MAME 版本。在这种情况下，[Atarity]可能想看看我们最近关于桌面 MAME 橱柜[或便携式 MAME 钻机](http://hackaday.com/2014/08/16/table-top-mame-cabinet-dubbed-the-water-cooler/)的文章。但是无论他想出什么，我们确信他的手艺会在那里。
