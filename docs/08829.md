# 弯曲木 LED 灯不需要花哨的工具

> 原文：<https://hackaday.com/2018/03/20/curved-wood-led-lamp-needs-no-fancy-tools/>

我们这些不熟悉木工的人可能不会想到，这个由[Marija]制作的[弯曲木材和丙烯酸 LED 灯](https://creativityhero.com/diy-projects/wood-acrylic-led-desk-lamp-concrete-base-bending-acrylic/)并不是花式雕刻的产物，只是一些深思熟虑的设计和组装工作。基底是塑料碗中几英寸厚的混凝土，然后用砂纸打磨并涂上一层透明的涂层。木材是在倒置的拼图上切割的四层山毛榉硬木，中间两层有一个额外的凹槽，用于两个 LED 灯条。在粗切的层被粘在一起后，不完美的地方被锉掉并打磨掉。由于木材层为 led 灯的凹槽提供了一致的宽度，因此很容易切割出与之匹配的长条状丙烯酸树脂。锯切割丙烯酸树脂可能是冒险的，因为它可能会破裂或熔化，但用横切刀片的台锯做到了这一点。形成丙烯酸树脂以匹配木材的曲线是一个温和的加热问题，并一点一点地将软化的丙烯酸树脂放松到位。

用喷雾罐中的几层缎面透明涂层对透明丙烯酸进行磨砂处理，这是一种我们以前从未见过的技术。方便，因为它提供了一个顺利和完整的涂层沿整个长度的丙烯酸。这很有效，是一个聪明的想法，但是[Marija]仍然可以看到灯里面的 led 和电线，所以她用一些白色胶带覆盖它们。下面嵌入了整个过程的视频。
