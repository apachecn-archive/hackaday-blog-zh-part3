# 构建 Novena 笔记本电脑

> 原文：<https://hackaday.com/2016/01/30/building-the-novena-laptop/>

来自[Bunnie]的最新硬件项目是 Novena，[一款真正开源的笔记本电脑](http://hackaday.com/2014/04/02/bunnie-launches-the-novena-open-laptop/)，几乎每个部件都有非 NDA 的数据表。这是硬件黑客的理想笔记本电脑——它在主板上有一个 FPGA，大量的引脚接头，以及许多额外的功能，使与外界的接口变得容易。

虽然 Novena 的众筹活动包括一台完全定制的笔记本电脑，但它非常昂贵。没关系；是传家宝笔记本电脑，反正这是 DIY 笔记本。随着 Novena 的上市，是时候让人们打造自己的笔记本电脑了。[Ben Heck]是第一个用他自己制造的 Novena 笔记本电脑参加比赛的人，这太棒了。

第二个视频是关于笔记本电脑最重要的部分:键盘。为了键盘，[本·赫克]全力以赴。这是一个完全机械的键盘，背光 led 围绕着带有 Cherry MX 开关的[幻影 PCB](https://deskthority.net/wiki/Phantom) 构建。因为这是一台 DIY 笔记本电脑，而且是要打开的东西，所以键盘是完全可拆卸的。想象一下类似最初的 Compaq luggable，但变成了看起来相当现代的笔记本电脑。

笔记本电脑外壳由铝和激光切割塑料的夹层构成。这些层被粘在一起，拧在一起，部分是小心翼翼地安装到案件。USB 键盘通过几根飞线和热胶水直接连接到主板上的一个芯片上。

完工后的建筑非常棒，尽管有点厚。这是终极黑客的笔记本电脑，有 FPGA，Linux，开源的一切，甚至还有一个可爱的小秘密隔间，用于存储工具和电缆适配器。来自周围最好的建筑商之一的伟大建筑。
