# 几乎有用的机器

> 原文:[https://hackaday.com/2016/06/11/the-almost-useful-machine/](https://hackaday.com/2016/06/11/the-almost-useful-machine/)

[亚历克斯]对制造效用微不足道的机器并不陌生。几年前，他制造了几乎无用的机器 T1，这是一个太阳能系统，可以在你等待的时候切断 20 毫米的定位销。迷恋互联网上的电动钢锯，他试图建立一个更坚固的版本，有点更有用。大约五个月的空闲时间后，他有了[这个几乎有用的机器](https://tinkerlog.com/2016/06/03/the-almost-useful-machine/)。

它依靠刮水器电机和笔记本电脑的回收电源运行。[Alex]用 ATtiny25 滚动他自己的板来控制马达。[电路](https://github.com/tinkerlog/TheAlmostUsefulMachine)将电位计运动转化为 PWM，PWM 通过 MOSFET 控制电机。切割完成后，终点挡板微动开关会立即切断电机。

底盘的每一部分都是由手工加工的铝制成。没有那种设置？有 3D 打印联动装置的电动钢锯怎么样？跳过去看看它的运行，并留下来观看由两部分组成的延时构建视频。
