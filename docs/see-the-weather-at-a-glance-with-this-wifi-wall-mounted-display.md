# 使用这款 WiFi 壁挂式显示器，一目了然地了解天气情况

> 原文：<https://hackaday.com/2017/03/04/see-the-weather-at-a-glance-with-this-wifi-wall-mounted-display/>

无论你是住在像[Becky Stern]一样看不到天空的公寓里，还是在寻找当前天气的一目了然的报告，你都可以考虑这个[极简主义的天气显示](http://www.instructables.com/id/WiFi-Weather-Display-With-ESP8266/?ALLSTEPS)，而不是每次出门都查看你的电脑或手机。

第一项任务是安装她的 Feather Huzzah ESP8266 模块。[Becky]从眨眼测试开始，以确保它正常工作。一旦解决了这个问题，她就开始安装一些库。IFTTT feed 获取的温度数据显示在七段显示屏上，而其他 feed 分别检索每种基本天气类型的信息:晴天、阴天、下雨、下雪。

创造这种时尚的显示效果所需要的只是阴影盒框架内的几块纸板，一张纸作为漫射体，以及隐藏在内部的 12 个新像素 RGB LEDs。修剪和安全的地方，以及开槽的电源电缆完成组装的框架背面。休息之后，请观看构建视频。
