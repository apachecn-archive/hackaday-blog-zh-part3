# Hackaday 奖参赛作品:一个很好的电子学习工具包

> 原文:[https://hack aday . com/2016/08/12/hack aday-prize-entry-a-good-electronics-learning-toolkit/](https://hackaday.com/2016/08/12/hackaday-prize-entry-a-good-electronics-learning-toolkit/)

创客运动是一个非常流行的东西，即使我们无法定义它是什么。毫无疑问，对 STEM 教育的推动完全不同于一代程序员因为金钱而获得 CS 学位。这意味着让孩子对电子产品感兴趣的套件有市场，而且肯定有很多选择。大多数这些“电子学习平台”实际上看起来并不那么好，其教学实用性也非常值得怀疑。 [Evive 不是这些工具包](https://hackaday.io/project/13091-evive-a-prototyping-platform-for-makers)中的一个。它看起来不错，实际上可能很有用。

Evive 的核心基本上是一个 Arduino Mega，并附带了方便的 Arduino shield 兼容性。然而，并非所有的 Mega 引脚都可以用于插入杜邦电缆——许多逻辑都被分线点、显示器、按钮和模拟输入占用了。Evive 中有 1.8 英寸的 TFT 显示屏，SD 卡插座，XBee，蓝牙或 WiFi 模块的连接器，电机驱动器，快速 DAC，模拟输入，以及大量的按钮，旋钮和开关。所有这些都被装入一个紧凑且看似坚固的塑料外壳中，使 Evive 比试验板和一堆跳线更耐用。
