# ESP32 仓鼠车轮跟踪器推特锻炼统计

> 原文：<https://hackaday.com/2017/05/27/esp32-hamster-wheel-tracker-tweets-workout-stats/>

即使所有的仓鼠滚轮跟踪器都存在(在这个网站上),仍然有改进的空间。[波格丹一世]将他的[仓鼠轮](http://www.bogdanberg.com/2017/05/21/hamster-wheel-iot-tracker/)从 Arduino 和数据记录屏蔽升级到 ESP32，并释放了一些通常与仓鼠轮无关的新功能。

[波格丹一世]的项目记录了以英尺为单位的距离、当前会话的持续时间、每分钟转数、总转数、以英里/小时为单位的速度和会话总数，以及一些系统监控统计数据。它还可以跟踪多个轮子，因为[Piontek](仓鼠)有两个轮子。然而，多亏了 ESP32，[波格丹一世]的车轮追踪器[在推特上发布其统计数据](https://twitter.com/HamsterPiontek)，并用[Piontek]的训练更新[的 ThingSpeak 仪表盘](https://thingspeak.com/channels/270678)。

除了功能性之外，[波格丹一世]还强调要让这个项目看起来和感觉上已经完成。他设计了定制的 3D 部件，包括前板、[挂钩](https://www.thingiverse.com/thing:2334203)用于将控制箱连接到笼子，以及[底座](https://www.thingiverse.com/thing:2334208)用于将传感器连接到车轮。
