# 如何制作一个雨果(基地)

> 原文：<https://hackaday.com/2015/09/05/how-to-make-a-hugo-base/>

很多黑客喜欢科幻小说。如果你不是其中之一，你可能不知道雨果奖是一个享有盛誉的科幻奖，每年在世界科幻大会上颁发。这座雕像看起来像一艘火箭船，但每年火箭船的底座都不一样。运动雕塑家[gfish]意识到大会将在斯波坎(他的家乡，靠近他现在的住所)举行，并决定参加比赛来创建基地。他赢了，所以 2015 年的 Hugos 都有[gfish 的]底座，他分享了他用来制作它们的过程，这很棒。

搅拌机的设计并不是最难的部分。[gfish]不得不进行实验，以找到将他的愿景变为现实的正确方法。在几次水射流切割的错误开始后，他有了 3D 打印的底部，并不得不制作一个特殊的磁性夹具来固定东西进行焊接。

总的来说，结果是值得的。我们不禁想知道你是否会因为设计了雨果而得到一个雨果。假设没有。雕刻似乎是一种流行的黑客活动，我们对科幻小说并不陌生。然而，参加标志性的雨果奖让我们羡慕不已。

当然，很多人(见下面的视频)对今年的提名不满意。但这并不意味着他们没有分发一个很酷的雕像。
