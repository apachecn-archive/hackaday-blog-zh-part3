# 建造你自己的黑洞

> 原文:[https://hackaday.com/2018/05/03/build-your-own-black-hole/](https://hackaday.com/2018/05/03/build-your-own-black-hole/)

好吧，也许这个标题有点夸张，但是这个由[Will Donaldson] 制作的黑洞灯是一种在不破坏地球的情况下创建黑洞模拟的有趣方法。这款灯使用一圈 led 围绕着一块黑色莱卡。灯座上的马达拉动莱卡，代表了奇点对时空的扭曲作用。它还证明了黑洞如何(在理论上)通过发射辐射蒸发，这种现象被称为霍金辐射。这是一种简单但有效的方法，物理学家已经用了一段时间来证明重力，使用弹性织物来模拟时空，并显示重力如何扭曲它。这是三维(或更多)现象的二维版本，但它是有效的。而且，希望它不会像真的那样吞噬地球，毁灭我们所有人。
