# 面向合作伙伴的线索电话

> 原文:[https://hackaday.com/2017/08/26/cluephone-for-partiers/](https://hackaday.com/2017/08/26/cluephone-for-partiers/)

[山姆·霍恩]改装了一部旧的学校座机电话，用来给生日聚会的客人提供线索。当客人找到一个数字线索时，他们把它输入键盘来听下一个线索，包括解码一些莫尔斯电码。

这款手机由一个 Arduino Pro Mini、一个 MP3/WAV 触发器和手机本身组成，其中的耳机和键盘被重复使用。[Sam]必须绘制出键盘和焊接引线，将手机 PCB 的各种触点连接到 Arduino 的数字引脚。他使用数字生成的声音来生成音频文件，并使用键盘和密码 Arduino 库来传递音频线索。

这似乎是一个很好的项目，适合任何年龄的聚会参与者，尽管打字速度很快。希望[山姆]的客人有一个高莫尔斯 WPM 或笔快！更多小键盘项目，请查看这个[自定义快捷键](https://hackaday.com/2017/03/17/the-custom-clicky-shortcut-keypad/)和[打印柔性键盘](https://hackaday.com/2017/08/15/print-a-flexible-keypad/)。
