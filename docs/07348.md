# 这是一个壁挂式 Dremel 工作站！

> 原文:[https://hack aday . com/2017/10/12/its-a-壁挂式-dremel-workstation/](https://hackaday.com/2017/10/12/its-a-wall-mounted-dremel-workstation/)

我们都见过 Dremel 钻床，但[Tuomas Soikkeli]使用他的 Dremel 作为马达创建了一个成熟的(尽管是微型的)工作站。他有一个侏儒大小的砂带机，看起来像是滑板轮子转动皮带，Dremel 的工具头拉紧皮带并转动它。有一个小台锯，小车床，可爱的路由器等。

Dremel 通过 3/4-1/2 螺纹端连接到底座，在该螺纹端上可以连接专门的工具端，以及 DIY 附件(如我们之前发布的[这个光环](https://hackaday.com/2015/01/06/light-ring-allows-precise-dremeling/))典型连接。虽然事实上不同型号的螺纹端的抗拉强度不同——即使是仿制品也有相同的一端，但它是否足够牢固以连接到钻机上？

我们喜欢[图马斯]把他的装置安装在墙上的方式。看起来他附近有几个灵活的轴延伸器，使钻机几乎可以充当商店气动工具的角色。
