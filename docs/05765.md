# 小易趣特斯拉线圈获得升级

> 原文：<https://hackaday.com/2017/04/30/little-ebay-tesla-coil-gets-an-upgrade/>

像我们介绍的许多项目一样，这个项目是从购买廉价的易贝模块开始的。在这种情况下，这是一个小特斯拉线圈，使体面大小的弧，但还不够好。结果是[一个超大的固态特斯拉线圈](https://www.youtube.com/watch?v=LbTyEratSTI)，有更好的结果和发展空间。

作为【GreatScott！]发现，小易贝特斯拉线圈有一个非常整洁的设计。激励器是一个 Slayer 电路，一个超级简单的单晶体管设计。他的逆向工程揭示了初级线圈只是次级线圈下 PCB 上的一个环路。遗憾的是，他试图替换主激励器并复制杀戮者激励器的尝试导致了不良表现。在这种情况下，黑客除了造一个更大的线圈还能做什么？更大——就像“建立自己的缠绕夹具”更大。1200 次二次转弯和一次看起来相当危险的初选之后，结果仍然是——贫血。事实证明，杀戮者无法胜任这项任务。他转向之前用于无线能量传输电路的逆变电路，我们终于看到了一点特斯拉线圈的魔力。但是等等！未来还会有更多，因为未来的视频将调整电路并优化线圈以获得更好的性能。

毫不奇怪，特斯拉线圈在这里是一个受欢迎的项目，尤其是音乐类的，从小型的 T1 到大型的 T3 的 T2。音乐好像没开【GreatScott！]的思想，但是，我们将感兴趣地看着他把这个建设。
