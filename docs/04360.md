# OpenAg 是一台个人食品计算机

> 原文：<https://hackaday.com/2016/12/26/openag-is-a-personal-food-computer/>

当一台自称为个人食品电脑的设备进入你的时间线时，你会想到什么？也许是一种与云相连的减肥产品，在电视购物广告中，看起来不太健康的加州女士们将她们的健康归功于一个手掌大小的装置，只需 199 美元就可以成为你的了。幸运的是，事实证明并非如此，进一步阅读我们的时间线故事，发现是关于一个计算机化的农业设备。

麻省理工学院媒体实验室开放农业倡议的 OpenAg 食品计算机标榜自己为:

> *“一个受控环境农业技术平台，该平台使用机器人系统来控制和监测专门种植室内的气候、能源和植物生长”*

它采用桌面围栏的形式，在其中可以测试所谓的气候配方，以复制植物生长的不同条件。可以公平地说，这种最基本的形式与其说是用于全面食品生产，不如说是一种教育设备，尽管他们在更大的规模上应用了同样的技术。他们所谓的“食物服务器”是货物集装箱中的开放环境库，肯定可以用来提供可行数量的产品。

好消息是该项目是开源的，他们最新的故事是[他们已经发布了设备](http://openag.media.mit.edu/blog/)的 2.0 版本(alpha)。如果你感兴趣，你可以[阅读文档](http://wiki.openag.media.mit.edu/)，并找到在他们的 GitHub 库上构建一个[所需的所有资源。上面链接的页面有一个视频，是非常多的圆滑的公关品种，而不是螺母和螺栓，所以我们已经找到了他们的建设视频为您打破。](https://github.com/openaginitiative/)
