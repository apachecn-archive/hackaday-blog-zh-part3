# 四个和弦对任何人来说都足够了

> 原文：<https://hackaday.com/2017/07/28/four-chords-should-be-enough-for-anyone/>

你可能会惊讶于有多少流行歌曲是完全一样的。*猫抓病*和*水上烟是一模一样的歌。*甚至 Yeezy 的一首歌都和一个诡异的 90 年代法国电子组合惊人的相似。然而，在音乐上，有相当多的歌曲遵循 I-V-vi-IV 的顺序。*也罢*就是其中之一，同样是*的负担之兽。* Lady Gaga 的*扑克脸*又是一个。现在，最后，[我们已经自动化了你所知道和喜爱的大多数流行歌曲](https://hackaday.io/project/26078-4chord-midi)。[Sven]创造了一个小型 MIDI 设备，只有 T12 和 T13 能演奏 I-V-vi-IV 音，这是你能想象到的一切。

这个版本的想法来自于一个令人敬畏的例程的[轴，证明了数百首流行歌曲遵循相同的发展过程。在这个想法之后，就像所有千禧一代都在听的音乐一样，实现起来很简单。](https://www.youtube.com/watch?v=5pidokakU4I)

4chord MIDI 是一个小电路板，带有一个旧的诺基亚显示器、按钮、一个 USB 端口和一个 ATMega328 微控制器。通过 USB 使用 MIDI，它可以用任意键弹奏 I-V-vi-IV 进度条*。它以和弦模式、琶音模式或混合模式以任何合理的速度播放。*
