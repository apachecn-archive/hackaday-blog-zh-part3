# 废鲨鱼旨在清洁我们的港口和海洋

> 原文：<https://hackaday.com/2016/09/16/waste-shark-aims-to-clean-our-harbours-and-oceans/>

无人机正在为我们的日常生活增加功能，无论我们是否准备好，自动化都将帮助人类。RanMarine 公司的理查德·哈迪曼(Richard Hardiman)将两者巧妙结合，开发出了从海洋中收集垃圾的小型无人机——他称之为“[废鲨鱼](http://www.popsci.com/amp/waste-shark-is-garbage-collecting-sea-drone)”

“苗条”和“胖男孩”这两个模特的目标是在水面下的“嘴”中每人收集多达 1100 磅的垃圾。“废鲨鱼”仍然被限制在远程控制，并且只有在航路点之间旅行时才是自主的，但人们可以看到这项技术如何演变成“水墙”

 [https://www.youtube.com/embed/JOUzkQA6QTA?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/JOUzkQA6QTA?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



在着手清理世界海洋——即太平洋大垃圾带——之前,[Hardiman]的目标是首先清理我们城市的港口、水道和运河。因此,“废鲨鱼”目前正在接受测试，直到年底用于清理鹿特丹港口。

如果你认为清洁海洋是一个遥远的目标，那么你如何看待自主环球航行？或者一群[无人驾驶船](http://hackaday.com/2016/02/12/swarm-of-robot-boats-coming-to-an-ocean-near-you-soon/)来执行你的命令？
