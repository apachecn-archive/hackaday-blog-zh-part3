# 这五个 Hackaday.io 成员刚刚赢得了漂亮的新 CircuitPython 板

> 原文:[https://hack aday . com/2017/01/28/these-five-hack aday-io-members-just-won-fancy-new-circuit python-boards/](https://hackaday.com/2017/01/28/these-five-hackaday-io-members-just-won-fancy-new-circuitpython-boards/)

就在几个小时前，我们在 Hackaday.io 上与 Adafruit 讨论了他们对 MicroPython 代码库的新扩展 [CircuitPython](https://blog.adafruit.com/2017/01/09/welcome-to-the-adafruit-circuitpython-beta/) 。在聊天过程中，Adafruit 的人员回答了一些问题，并询问聊天参与者他们希望用一些很酷的新硬件来构建什么。这些 CircuitPlayground M0 快速板是全新的，未发布的硬件。很酷的东西。

这些未发布的主板的获胜者以及他们将使用它们的项目是:[RaidDude8]用于灯光绘画系统，[gelliusslime]用于他女儿的对手势做出反应的“魔杖”，[Neon22]用于使用 Neopixels 的多用户游戏，[turbine Reiter]用于使用 Neopixels 和加速度计的重力演示器，以及[todbot]用于 Powermate knob USB HID 克隆。

在聊天中，Adafruit 的人谈到了他们对 MicroPython 的补充。这是对 API 的重新加工，为更多平台提供了更好的支持，并将整个东西扩展到了微控制器。如果你喜欢 Python，并且想进入微控制器领域，这本书很适合你。

如果你错过了聊天，你仍然可以在这里查看 [Adafruit 的实时流](https://www.youtube.com/watch?v=jTNVra3IENs)，或者在这里查看文字记录。下面，你可以看看休息后阿达女士颁发新的董事会。
