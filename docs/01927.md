# 洋葱欧米茄电脑可以通过无线网络控制

> 原文:[https://hack aday . com/2016/03/28/the-onion-omega-car puter-can-be-controlled-via-wifi/](https://hackaday.com/2016/03/28/the-onion-omega-carputer-can-be-controlled-via-wifi/)

洋葱欧米茄(Onion Omega)是一个奇怪的名字，它是一个基于 linux 的超微型 WiFi 板，是一个有用的小型设备，用于所有与物联网相关的东西。[丹尼尔]决定用它把他的车连上互联网。

如今，大多数新车都内置了远程启动功能，慢慢地，制造商们正在追赶现代技术，并加入应用程序来控制车辆的各种功能。但对于旧车来说，除了售后市场的远程启动套件之类的东西，你没什么可做的。

[Daniel]没有被吓倒，他想通过手动添加额外的钥匙链、远程启动协议和与车载计算机的数据连接，将他的车带入 21 世纪。

他正在使用一个 [Onion Omega](http://hackaday.com/2015/04/22/kickstarting-even-more-router-based-dev-boards/) ，一个基于通用路由器的开发板，具有 18 GIPO 和无线 802.11 b/g/n 连接。使用他为自己的手机编写的应用程序，他现在可以远程启动他的汽车，读取汽车信号，控制室内灯，当然，还可以锁上和打开车门。
