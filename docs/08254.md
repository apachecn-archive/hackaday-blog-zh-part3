# 机器人台锯自动化手指关节

> 原文：<https://hackaday.com/2018/01/13/robotic-table-saw-automates-finger-joints/>

我们都见过手指接头或盒子接头，这些互锁的拼图块使激光切割胶合板外壳成为 DIY 项目的固定设备。但是激光切割机使手指关节看起来比传统木工工具更容易制造，这往往导致令人失望的结果。

但是[这个手指关节切割机器人](https://hackaday.io/project/30163-internet-of-tablesaw)不是传统的木工工具，而且【timschefter】在建造钻机方面投入了大量的工作。我们必须承认，当我们第一次看到下面的视频时，在我们的商店里有一个可以通过手机上的按钮打开的台锯的想法让我们犹豫了。但是仔细分析一下，看起来安全是这个版本的主要问题。有了突出的紧急停止和联锁开关，形成机器人基础的小台锯应该足够安全。台面上是一个带有线性滑轨的滑板，它垂直于刀片移动工件，滑板通过气缸在刀片上来回移动。该接头由一个定制应用程序设置，该应用程序计算针的宽度和间距，这些针可以均匀地分布在整个面板上，或者，为了有点极客的乐趣，可以控制制作一个用莫尔斯电码编码信息的接头。

为此做了很多工作，虽然[这不是我们见过的第一个机器人手指关节切割机](https://hackaday.com/2016/08/31/internet-of-things-woodworking/)，但它给人留下了非常深刻的印象。现在，如果它能自动吻合。
