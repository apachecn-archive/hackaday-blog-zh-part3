# DNA 编码电影是用活细胞记录数据的第一步

> 原文：<https://hackaday.com/2017/07/13/movie-encoded-in-dna-is-the-first-step-toward-datalogging-with-living-cells/>

虽然 DNA 是一种相当好的存储介质，但它不是特别快、便宜或方便读写。

如果活细胞可以通过将有用的数据记录到自己的 DNA 中以供以后分析来简化这一过程，会怎么样呢？在哈佛医学院，科学家们正在努力实现这一目标，他们使用 CRISPR 在细菌细胞中编码和检索一段短视频。

CRISPR 是许多细菌免疫系统的一部分，通过在特定位置存储病毒 DNA 序列来识别和消除病毒感染。作为基因工程的工具，它比以前的技术更便宜，缺点也更少。

除了产生活生生的里克罗和 DMCA 违规，这有什么好处呢？廉价的自我复制传感器。哈佛大学科学家团队的成员 Seth Shipman 在一次采访中解释了一些可能的应用。他的重点是工程细胞作为一种非侵入性的数据采集工具来研究神经生物学，例如通过使用工程神经元来记录它们的发育历史。

有可能看到这项技术如何在学术背景之外得到更广泛的应用。目前，生物传感器通常使用[电子或荧光传感器](http://en.wikipedia.org/wiki/Biosensor#Biotransducer)来中继检测事件。通过在活细胞的 DNA 中记录数据，生物传感器可以变得更便宜，并包含内在的数据记录。可能的应用包括长期代谢物(如葡萄糖)监测器、化学检测器和质量控制。

值得注意的是，这项技术仅处于概念验证阶段。数据由科学家手动记录并检索到细菌基因组中，准确率达到 90%，这表明如果细胞可以被设计成自己记录数据，准确性和容量足以满足实际应用。
