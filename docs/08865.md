# 可见光 CT 扫描仪有双重功能

> 原文:[https://hack aday . com/2018/03/17/visible-light-CT-scanner-does-double-duty/](https://hackaday.com/2018/03/17/visible-light-ct-scanner-does-double-duty/)

如果你曾经经历过在本该无籽的脐橙中找到一粒种子的心碎，你会很高兴地听到，只要做一点点工作，你就可以用光学计算机断层扫描仪来保护自己，在把它放进嘴里之前，先检查一下切片的内部。

我们不得不承认一开始是带着怀疑的眼光来读这一篇的。我们并不是怀疑 DIY CT 扫描仪的可能性；毕竟，[在](http://hackaday.com/2013/01/09/ben-krasnow-builds-a-ct-scanner/)之前，我们至少在[见过几次](http://hackaday.com/2012/08/26/visualizing-water-droplets-and-building-a-ct-scanner/)的例子。安装在扫描室上的突出的 DSLR 泄露了在这台扫描仪中使用可见光而不是 X 射线——但实际上，X 射线只是光的另一种波长。如果选择光学半透明的测试对象，原理都是一样的。因此,[Jbumstead]的光学 CT 扫描仪仅限于窥视像西红柿片或橙子片这样的东西的内部结构，它的分辨率令人印象深刻。

与 X 射线 CT 扫描仪相比，这种扫描仪还有一个明显的优势，即它可以在可见光谱范围内对物体的外部进行成像，这使得它除了可以用于诊断小熊软糖疾病之外，还是一种方便的 3D 扫描仪。在透射或反射模式下，DSLR 装有远心透镜，其快门与步进驱动的样品台同步。扫描图像被发送到 Matlab 进行 CT 扫描重建，或发送到 Photoscan 进行 3D 扫描。

结果令人印象深刻，尽管它作为扫描仪可能更有用。想把 3D 扫描变成 3D 打印吗？摄影测量学是它的强项。
