# 增强现实超声波

> 原文：<https://hackaday.com/2016/01/27/augmented-reality-ultrasound/>

想到虚拟现实，脑海中浮现的大多是乐趣和游戏。但是，有许多有用的、真实的应用将很快在医学等领域开辟令人兴奋的可能性。来自斯图加特 Shackspace hacker space 的[Victor]构建了一个[增强现实超声波](http://shackspace.de/?p=5339)扫描应用程序来演示这种可能性。

但是首先，我们不能理解为什么去翻垃圾箱然后带着一台功能正常的超声波机器回来！这就是成员[阿尔夫]有一天出现了。在最初对它的新奇感到兴奋之后，它被放到了一个积满灰尘的角落里。当[Victor]发现它时，他要求为一个项目借用它。Shackspace 很乐意捐给他，腾出一些空间。一段时间后，[Victor]展示了他用超声波机器做的东西。

一旦超声波扫描仪向 VR 应用程序注册，可能使用粘贴到扫描传感器的图像，扫描仪数据就会虚拟地投影到回声传感器下。关于他是如何做到的没有太多细节，但他是使用 [Vuforia SDK](https://developer.vuforia.com/) 结合 [Unity 5 跨平台游戏引擎](https://unity3d.com/5)完成的，Vuforia SDK 有助于构建移动设备和数字眼镜的应用程序。请观看视频，了解它的实际应用。
