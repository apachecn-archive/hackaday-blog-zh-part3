# 我的天啊！物联网

让我们建立**有用的**物联网，而不仅仅是物联网。 [IuT！物联网](https://hackaday.io/prize/details#two)

这就是我们接下来五周要做的事情。 [2017 Hackaday 奖](https://hackaday.io/prize)第二场挑战今天开始。我们正在寻找有用的互联设备的最佳创意。20 名参赛者将获得 1，000 美元，并进入最后一轮，争夺 5，000 至 50，000 美元的最高奖金。

毫无疑问，未来是相连的。自从电报出现以来，这就是我们的未来，毫无疑问，我们正以更快的速度变得更加紧密。你手中、口袋里或包里的手机将你与人类的大部分知识联系起来。但是它还没有把你和很多“事物”联系起来。这种情况不会持续太久。

我们已经看到摄像头(安全设备、婴儿监视器以及介于两者之间的所有设备)作为一些最早的连接设备出现，它们带来了连接到更广泛的互联网的安全措施不力的所有意想不到的后果。至少远程摄像头是有目的的；已经有足够多的产品发布了。我们要举的反例是联网烤面包机，这也是今天早上乔·金精彩艺术的主题。谁需要远程敬酒？没人。

### 让我们发明物联网

这是我们把事情做好的机会。物联网如何让生活更美好？什么东西在加入网络后变得更有意义，它看起来像什么？我们如何继续连接我们的世界，同时保护隐私和注意安全。找到这些问题的答案会让你做出一些有意义的事情。
