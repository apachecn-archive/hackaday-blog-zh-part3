# 剪式升降鞋可能符合 OSHA 标准

> 原文:[https://hack aday . com/2017/01/07/scissor-lift-shoes-may-be-OSHA-compliant/](https://hackaday.com/2017/01/07/scissor-lift-shoes-may-be-osha-compliant/)

据说需要是所有发明之母。这可能是电视剧《小侦探》背后的基本原则，该剧讲述了一个警察特工的故事，他可以随时掌握任何需要的技术。尽管探长的小玩意经常会让他惹上麻烦(他的侄女 Penny 通常会解决实际的犯罪)，但受探长启发制作的鞋比探长袖子里的任何东西(或裤腿，视情况而定)都更有用。

如果是一项杰作，[Make it Extreme]建造了他们自己的“Go Go Gadget Legs”，这是一套气动控制的高跷，允许穿着者按一下按钮就可以显著增加他们的高度。我们经常看到干墙承包商穿着类似高度的高跷，但没有看到任何能够随意升高和降低穿着者的高跷。该团队从零开始建造腿，用普通金属加工几乎每个部件(包括空气活塞)。在添加了一些控制装置并进行一些测试后，该团队发现一次抬起一只脚是更安全的路线，尽管两只脚都可以抬起来进行更令人印象深刻的演示，这可能会使穿着者失去平衡。

这种构建的质量和最终产品的抛光是难以置信的高。如果你在家里有自己的机械车间，这种项目可能是你力所能及的(双关语)。不过，如果你手头只有一台电焊机，你或许可以组装一台[Make it Extreme]的其他著名产品:[啤酒枪](https://hackaday.com/2016/08/16/beer-gun-shoots-non-lethal-doses-of-bewerages/)。
