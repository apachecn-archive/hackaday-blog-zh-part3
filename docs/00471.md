# 断指不是现代黑客的障碍

> 原文：<https://hackaday.com/2015/10/25/broken-finger-is-no-obstacle-to-modern-hacker/>

[Jim Merullo]和他的儿子正在享受一场精彩的飞盘比赛，一次不幸的跳水导致了受伤。他的儿子弄断了他的小指，医生别无选择，只能将他的整个左手绑在一个不合理的大石膏上。对大多数人来说，这意味着几周内不能使用左手，如果你的儿子有《我的世界》瘾，这就有点问题了。然而[Jim]对黑客社区来说并不陌生，他已经开始研究解决方案。他拿出 2 号十字螺丝刀，点燃烙铁，[开始工作](https://www.flickr.com/photos/jmerullo/sets/72157660295820505/)。

对受伤左手的详细分析显示，中指和无名指的使用受到限制，而拇指则完全使用。因为他的儿子玩游戏时用右手操作鼠标，左手操作键盘，所以他需要找到一种方法，让他在有限的左手使用范围内操作键盘。他拆开了一个旧的 USB 键盘，并焊接了一些触觉开关来模拟所需的按键。在制作了一个适合全体演员的时尚 Altoids 锡制坐骑后，他的儿子能够在有限的干扰下享受他最喜欢的视频游戏。
