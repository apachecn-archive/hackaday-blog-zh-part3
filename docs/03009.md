# 本周你会看到的最昂贵的虚拟现实实验

> 原文:[https://hackaday . com/2016/07/24/最昂贵的虚拟现实实验-你将看到整个星期/](https://hackaday.com/2016/07/24/the-most-expensive-vr-experiment-youll-see-all-week/)

在这个基于机器人的虚拟现实物理反馈的简短演示背后没有太多的细节，但视频(嵌入在下面)很好地展示了这一点。这是一个使用 Baxter 机器人和 HTC Vive 为虚拟对象生成力反馈的实验。当用户在虚拟现实中按下一个木块时，机器人会按回去，模拟虚拟物体的质量。力反馈是正在进行研究的领域之一，并且在各种不同的方向上进行。

就像生活中的许多其他事情一样，对于实际的物理反馈来说，没有什么比真实的东西更好了。此外，让一个 25，000 美元的机器人在虚拟现实中模拟几个简单的木块也很棒，这样你就可以绑上虚拟现实装备，基本上给机器人一个逼真的拳头碰撞。
