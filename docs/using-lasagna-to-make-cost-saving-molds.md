# 用千层面制作节约成本的模具

> 原文：<https://hackaday.com/2018/01/25/using-lasagna-to-make-cost-saving-molds/>

构建一次性原型通常非常简单。找到一些穿孔板并开始焊接，焊接一些废金属，或者打开 3D 打印机。但是，如果你要做一个产品的生产运行，那么事情需要有更多的抛光。在[Eric Strebel]的案例中，这意味着通过[在千层面的帮助下，将实心的模制部件转换成空心的东西](https://www.youtube.com/watch?v=2DDQnM3y2FA)，从而节省重量和材料。

[Eric]在这个视频中向我们展示了如何制作一个哭泣的模具。首先，固体部分是用硅树脂浇铸的。使用铸件，一些“片状粘土”被应用到内部，最终将形成新部分的墙壁的空隙。然而，粘土需要与模具顶部齐平，完成这项任务的一个技巧是冷冻模具(在千层面旁边)，这样可以在不变形的情况下刮去粘土。

从那里，模具的第二部分被倒入，使用特殊的通道，让树脂从模具中“流出”(因此得名)。这两个部分的过程创造了一个更有效的薄壁零件，而不是昂贵的固体原型零件。

[Eric]在这里也不陌生。他是一名工业设计师，有很多这个行业的技巧和诀窍，包括用钻床和老虎钳制作加工工具的方法，以及如何让小批量生产的产品发挥最大效益的一些技巧。
