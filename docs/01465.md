# 荒谬的自动化宿舍

> 原文:[https://hack aday . com/2016/02/13/可笑的自动化宿舍/](https://hackaday.com/2016/02/13/ridiculously-automated-dorm-room/)

拿三个 NRF24L0+收音机，两个 Arduino Nanos，一个树莓 Pi。加一个百无聊赖的学生和莱斯大学的宿舍。你得到的是 [RRAD:赖斯荒谬的自动化宿舍](https://hackaday.io/project/4898-rice-ridiculously-automated-dorm-rrad)。【乔丹·波尔兹】建造了一个模块化系统，灵感来自[布拉德](http://lab.dereklow.co/brad/)(伯克利荒谬的自动化宿舍)。

RRAD 有三种类型的节点:

*   执行节点–允许外部执行机构，如继电器或螺线管
*   传感节点–报告来自传感器的数据(光、温度、运动)
*   Hub 节点–托管控制面板，记录数据，提供外部数据接口

该集线器还允许[乔丹]通过 Tasker 控制他的 Android 手机。他在 GitHub 上有 Arduino 和 Raspberry Pi 代码，如果你想可笑地自动化你自己的东西。不过，你可能想让它适应你的宿舍、房子或房车。
