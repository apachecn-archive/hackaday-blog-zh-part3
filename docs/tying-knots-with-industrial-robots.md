# 用工业机器人打结

> 原文：<https://hackaday.com/2017/02/12/tying-knots-with-industrial-robots/>

我们并不羞于承认我们非常想要一双高端工业机器人手臂来玩耍。我们不知道该把它们放在哪里——也许是客厅？—但我们知道我们会想出办法的。这个演示的目的是让童子军对机器人感兴趣，方法是将野蛮的手臂应用到所有孩子都喜欢的东西上，学习打结。(如果你问我们，他们已经弄反了。)反正下面嵌了两个视频给你偷看。

如果你没有想过打结，这实际上是机器人手臂的一个整洁问题。它们必须以奇怪的角度相互靠近，不能相撞。编写各种各样的程序将是一个非常有趣的挑战，我们只能希望看到这一点的一些孩子，和我们一样热情，能够投入进去，而不是简单地观看表演。

那么今天怎么样？你会用一两个工业机器人做什么？[把它们用于艺术](https://hackaday.com/2015/12/20/madeline-gannon-the-robot-tamer/)？还是做[精准纹身](http://hackaday.com/2016/08/12/tattoos-by-robotic-arm-with-pinpoint-accuracy/)？还是[去 DIY](http://hackaday.com/2015/03/14/open-source-robotic-arm-now-within-reach/) ？
