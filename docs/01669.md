# 海牙 Rev 空间的短程线穹顶建筑

> 原文:[https://hack aday . com/2016/03/07/geodesic-dome-build-at-rev-space-den-haag/](https://hackaday.com/2016/03/07/geodesic-dome-build-at-rev-space-den-haag/)

[Morphje]一直想建造一个短程线穹顶。他对它的形状和设计，以及用基本材料建造它的可能性很感兴趣。因此，在几个朋友的帮助下，他建造了一座直径 9.1 米的建筑，开始实现自己的抱负。

这个行动发生在荷兰海牙的黑客空间 [Rev Space](https://revspace.nl/Main_Page) (荷兰语网站)。[Morphje]首先必须制作大量的木制支柱，每个支柱都有一根管子，每端都有一个扁平的凸耳，支柱的外部有一个凸缘，以防止它裂开。如果你拥有一台配备了正确工具的重型飞压机，那么展平数百根管子的末端是一个相当简单的过程，但[Morphje]没有这种奢侈，必须用手锤平每一根管子。

然后，通过这些扁平的管耳将支柱栓接在一起，形成三角形截面，这些三角形进一步栓接在一起，形成最终的穹顶。或者理论上是这样。在下面的视频中，你可以看到他们中止了从外向内组装穹顶，然后改变策略从屋顶向下组装。

这个项目仍在进行中，[Morphje]只组装了穹顶的框架，还没有覆盖物或门。但它仍然是一个值得效仿的建筑，我们期待在今年夏天的欧洲制造商活动中看到完工的穹顶。
