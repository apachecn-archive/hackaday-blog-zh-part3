# 在你的扶手椅上玩抓爪机

> 原文:[https://hack aday . com/2017/11/19/play-a-claw-machine-from-your-armchair/](https://hackaday.com/2017/11/19/play-a-claw-machine-from-your-armchair/)

你有没有被街机里的抓爪机诱惑过，结果你想要一个可爱玩具的希望破灭了，因为它没有抓住你的选择？那么你很幸运，因为现在你可以在网上尽情地玩了。[Ryan Walmsley]想让你控制他的树莓 Pi 驱动爪形机器。

硬件方面，他用 Pi 和定制的 H 桥 PCB 取代了原来的 8052 微控制器和继电器控制。我们特别强调警告:“高电压”，我们认为它应该更频繁地出现。在他的 GitHub 库中有一些代码[，但是我们怀疑它没有所有的东西。](https://github.com/ryanteck/wmhclaw)

我们挖掘了关于这个的文档，从中获得了很多乐趣。从[他最初的想法](https://ryanteck.net/piclaw-conversion-part-1/)通过[一些原型](https://ryanteck.net/piclaw-conversion-part-2-initial-plan/)和[一个电路板故障](https://ryanteck.net/piclaw-conversion-part-2-2-failure-analysis/)，到[推出网络版](https://ryanteck.net/pi-claw-update-part-3/)和最后一个关于[它是如何工作的](https://ryanteck.net/the-pi-claw-update-3-5-how-it-works/)，他已经完成了。

遗憾的是，这台机器本身并不总是在线，它似乎只在[瑞安]在家的时候才在线，因此，如果你住在地球的另一端，远离他的英国基地，你可能就不走运了。幸运的是，虽然他以前的直播流是在线的，所以你可以在休息时看到它在过去的活动中。

当然，你在像这样的抓爪机里装什么样的赃物？在他的推特上，我们已经看到了《T2》玩具总动员中外星人的测试。在 is 记录的游戏中展示的大部分物品——现在数量超过 2000 个——都是我们心爱的[同伴魔方](https://hackaday.com/2014/11/15/hand-machined-companion-cube-can-be-destroyed-and-rebuilt/)。
