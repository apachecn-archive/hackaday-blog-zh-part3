# 学生 3D 打印眼睛

> 原文:[https://hackaday.com/2018/02/15/student-3d-prints-eyes/](https://hackaday.com/2018/02/15/student-3d-prints-eyes/)

[ondřej·沃西卡]是捷克共和国布尔诺科技大学的学生。此外，23 岁的他左眼失明。在参加一个关于 3D 打印的讲座时，他想知道他是否可以 3D 打印一个眼科假体——一只假眼。原来，[他可以](https://www.novinky.cz/internet-a-pc/462519-brnensky-student-vytiskl-ocni-protezu-na-3d-tiskarne.html)。如果你不会说捷克语，你需要像我们一样求助于[翻译服务](https://translate.google.com/translate?hl=en&sl=auto&tl=en&u=https%3A%2F%2Fwww.novinky.cz%2Finternet-a-pc%2F462519-brnensky-student-vytiskl-ocni-protezu-na-3d-tiskarne.html)。

与传统的玻璃或塑料眼睛不同，3D 打印义肢时，改变颜色等参数是微不足道的。这对于 iris 来说尤其重要，打印成品大约需要 90 分钟。需要额外的时间将产品涂上丙烯酸层以模仿自然眼睛的光泽。

显然，在捷克共和国，一只传统的义眼大约需要 2500 克朗。印刷眼睛的价格大约是它的三分之一。该学生希望创造一个完全 3D 打印的更便宜的版本。

为了制造假眼，他扫描了现有的义眼和自然眼。计算机建模可以生产出具有正确形状、纹理和颜色的眼睛。

当然，印刷的眼睛看不到。但是有一种技术可以帮助某些形式的失明。我们之前已经讨论过眼睛的常规制造(虽然原文已经没有了)，但你可以在下面的视频中看到它是如何完成的。如果你想知道[IEEE 认为人造身体部分](https://hackaday.com/2008/06/05/bionic-senses/)在 2008 年将走向何方，我们也报道了。我们认为 3D 打印和材料的最新进展改变了很多这些假设。
