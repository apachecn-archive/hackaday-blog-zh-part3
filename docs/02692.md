# 成为《公民科学家》的同行评审员

> 原文：<https://hackaday.com/2016/06/20/become-a-peer-reviewer-for-citizen-scientist/>

我们科学界的关键之一是同行评议的概念。当有了重要的发现时，这项工作会由同一领域的其他人来回顾，以检验这些发现。这可以验证工作，但也可以打开新的问题，导致新的发现。

我们在 Hackaday 奖中加入了同行评议。这是一种运用你的技能造福大众的新方式。当前的挑战是公民科学家；呼吁有助于科学研究更广泛应用的项目。一组独立的眼睛给这些条目建设性的反馈可以是一个巨大的成功的终点。毕竟你不知道自己不知道的事。当踏入未知领域时，帮助识别绊脚石，或者只是接受第二种意见，即你在正确的道路上，会有很大的不同。

成为同行评审员很简单。[选择一个你感兴趣的项目](https://hackaday.io/submissions/prize2016_citizen/list),彻底检查它，同时以尊重、积极和建设性的方式做笔记。准备好后，[使用此表格](https://docs.google.com/a/hackaday.com/forms/d/1GEEcFSbNHJiIhlyLAM8Pv2t4H8qLewHMS5HBSc_Ka30/viewform)提交您的同行评审。我们将私下与项目创建者分享您的评论。

Hackaday.io 是世界上最具活力的硬件协作平台。同行评议是另一种有趣的方式，可以让我们社区中更聪明的人参与到有意义的建设中来。
