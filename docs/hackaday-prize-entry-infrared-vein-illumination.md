# Hackaday 奖参赛作品:红外静脉照明

> 原文：<https://hackaday.com/2017/09/23/hackaday-prize-entry-infrared-vein-illumination/>

静脉切开术是一个有趣的词，也是寻找静脉的艺术。虽然护士和医生经过几十年的磨练，已经掌握了在手臂上扎针的技巧，但当然，还是有找到静脉的技术解决方案。一种更令人印象深刻的医疗设备使用近红外成像技术——基本上是用几乎是可见光来观察皮下组织。这些设备价格不菲。

Hackaday Prize 中的一个项目正试图改变这种情况。[是近红外静脉探测仪](https://hackaday.io/project/26158-assistance-system-for-vein-detection)。代替数千美元的专业单位成本，这一个不到一百美元就可以建造。

就这种构造而言，静脉是通过大约 950 纳米的红外线照射的。这种光的反向散射是通过树莓 Pi NoIR 相机捕捉的，常规的旧摄影胶片会阻挡可见光。从那里，它只是一个简单的图像处理和点击增强几次，直到静脉出现在显示器上。

这个项目背后的团队已经[开发了该设备的移动版本](https://hackaday.io/project/26158-assistance-system-for-vein-detection/log/65074-mobile-version)，并配有 3D 打印部件。这是一个方便的设备，也是 Hackaday 奖的绝佳参赛作品。
