# 面向预算机器人专家的激光切割麦克纳姆轮

> 原文:[https://hack aday . com/2018/04/04/laser-cut-me canum-wheel-for-the-budget-robot ist/](https://hackaday.com/2018/04/04/laser-cut-mecanum-wheel-for-the-budget-roboticist/)

对于初露头角的机器人专家来说，omniwheels 可能是你日常“入门”机器人套件设计模式的下一步。这些轮子由位于轮子周边的微小滚轮组成，使轮子能够自由地横向滑动。由于每个轮子都有独立的电机控制，平台可以通过在滚轮上滑动而自由地侧向移动。你可能会想:“轮子做的轮子？听起来很贵……”–你说得对！不过，不要害怕；比利时[Incubhacker]的工作人员已经用[一种激光切割设计覆盖了你的身体，只需点击一下鼠标，它就能落在你的工作台上](https://www.thingiverse.com/thing:1771830)。

对于任何试图以一个角度可靠地匹配平面激光切割零件的人，我们可以告诉你这不是一件容易的事。这里的设计既简单又可靠。他们不仅优雅地解决了这个问题，他们还成功地创造了一种设计，可以承受机器人底盘的负荷。激光切割设计通常也受到材料选择范围狭窄的困扰。这里实际的滚轮需要比胶合板所能提供的更多一点的抓地力。他们也有效地解决了这个问题，依靠热缩管提供传统车轮所期望的牵引力。

在下面的视频中，【Incubhacker】带你一步一步地实现你自己的梦想。在过去的中，我们当然见过一些[令人印象深刻的激光切割全方位轮，但我们喜欢设计的简单性，以及可能已经存在于我们工作台上的零件的组合。](https://hackaday.com/2016/02/20/mdf-omniwheel-uses-no-metal-or-plastic/)
