# 瑞克的传送门枪

> 原文:[https://hackaday.com/2015/11/05/ricks-p-p-portal-gun/](https://hackaday.com/2015/11/05/ricks-p-p-portal-gun/)

为了这个万圣节，[戴夫·道尔顿]全力以赴地穿上了卡通片《瑞克和莫蒂》中[瑞克]的服装。他设计并 3D 打印了一把传送门枪。不，不是从传送门。[瑞克的传送门枪](https://www.youtube.com/watch?v=AVIVrs0SOKM)，设定为现实 C137。

他在堪萨斯城的创客空间建造了它，叫做锤子空间工作室。3D 打印的外壳非常符合传送门枪的卡通描绘，但除了看起来逼真之外，它实际上是可行的！你可以看到[戴夫]真的去把一个微型投影仪塞了进去，这样当你使用它的时候，这把枪实际上会投射出入口——甚至还有声音效果。

但是视频中我们最喜欢的部分可能是[戴夫]对[里克]酒后吐词的出色模仿。

> 我-我-在这个房子里，我们-我们-我们有各种很酷的电子小发明，我们-我们有一些来自苹果的东西…
