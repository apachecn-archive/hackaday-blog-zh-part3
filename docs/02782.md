# Hackaday 奖参赛作品:DIY 足部矫形器

> 原文：<https://hackaday.com/2016/07/01/hackaday-prize-entry-diy-foot-orthotics/>

你的步态对你的脚来说是什么样的？在你步态的哪个部分，你的脚掌承受的压力最大？有什么问题吗？你能通过添加或移除定制鞋垫的材料来修复它吗？所有这些答案都可以通过一个昂贵的系统和去看足病医生得到，但是如果查尔斯·弗里德成功了，你可以在家里建立一个类似的系统。

该设备的工作原理是在鞋内扁平的鞋垫上安装一组压力传感器。当病人行走时，该设备将数据传输到计算机并记录下来。然后，计算机会生成这个人脚步的热图。计算机还产生一种非常有用的可视化效果，叫做步态线。这使得矫形师能够指定或制作正确的矫正器。

[Charles]的版本比专业版本有另一个优势。他的将能够无线传输到数据记录器。这意味着你可以将传感器戴在身上一段时间，获得更真实的步态图像。就像在牙医面前用牙线剔牙一样，许多人在足部医生面前有意识地思考他们的步态；这会影响结果。

他目前有一个原型在工作。他不确定他的压力传感器在目前的建设中能坚持多久，他暂时搁置了无线测井。然而，这个项目很有趣，我们迫不及待地想看看[查尔斯]是否能达到他所有的设计目标。
