# 废木料和金属组合，用于 DIY 麦克纳姆车轮

> 原文：<https://hackaday.com/2017/03/15/scrap-wood-and-metal-combined-for-diy-mecanum-wheels/>

一些碎木头，几片金属片，一套旧的齿轮马达，和一些基本的手动工具。如果你也有一点诀窍的话，这就是制作一个带有麦克纳姆轮子的全方位机器人所需要的一切。

对于外行人来说，[麦克纳姆轮](https://en.wikipedia.org/wiki/Mecanum_wheel)可以向任何方向旋转，这要归功于圆周上的一系列锥形滚子，它们相对于主轴倾斜 45°。[Navin Khambhala]制造麦克纳姆车轮的方法显然是低技术和非常劳动密集型的，但却能制造出工作的车轮和相当敏捷的机器人。滚筒的支架由钢板切割而成，手工弯曲以支撑木制滚筒，每个滚筒都用孔锯切割，并在临时车床上加工成桶形。每个轮子都直接连接到齿轮马达轴上，所有的东西都安装在钢板底盘上。控制和建造方法一样简单，但下面的视频展示了一个麦克纳姆轮式机器人能做什么。

这里有很大的改进空间，但主要是在制造方法上。[例如，整个车轮可以是 3D 打印的](https://hackaday.com/2010/04/28/printable-mecanum%C2%A0wheel/)，甚至是从 MDF 上激光切割下来的[，只需做一些设计上的改变。但是[纳文]赢得了一个奖项，因为他几乎从零开始制造了一个可以工作的轮子和一个可以工作的机器人。](https://hackaday.com/2016/02/20/mdf-omniwheel-uses-no-metal-or-plastic/)
