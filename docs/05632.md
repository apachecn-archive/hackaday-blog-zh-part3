# 构建您自己的电脑——真的

> 原文:[https://hackaday.com/2017/04/14/build-your-own-pc-really/](https://hackaday.com/2017/04/14/build-your-own-pc-really/)

曾经有一段时间，构建自己的计算机意味着大量的焊接或电线缠绕。不过，从某种意义上来说，制造一台个人电脑已经意味着购买一块主板、一个电源，只需把几根电线插在一起。这没什么不好，但(斯科特·贝克)想真正制造一台个人电脑。他[组装了](http://www.smbaker.com/building-an-xi-8088-pc)一个 Xi [8088](http://www.smbaker.com/building-an-xi-8088-pc) ，一个来自[【谢尔盖】](http://www.malinov.com/Home/sergeys-projects)的设计师，他的网站上有很多有趣的项目。[Scott]做了一个很棒的构建日志和一个视频，您可以在下面看到。

顾名思义，这不是一个现代的 i7 发电站。它是一台经典的 8088 PC，带有 16 位背板。从好的方面来说，几乎所有的东西都是传统的通孔部件，除了一个可选的紧凑型闪存插座和 VGA 卡的一部分。[Scott]从[retro brew 论坛的电路板库存](https://www.retrobrewcomputers.org/doku.php?id=boardinventory)中获得电路板，论坛用户在那里为这样的项目提供 PCB。

如果你试图复制[斯科特的]项目，你可以从阅读他的帖子中学到一些技巧。例如，他了解到 CPU 板上的电阻槽比 IC 插座高一点。教训是先安装插座。

斯科特没有使用普通的 8088 芯片，而是使用 NEC V20，这是当时常见的升级。除了背板和 CPU 卡，[Scott]还构建了一个软盘控制器板、一个声卡、一个小型闪存阅读器和一个 VGA 板。

制造一台个人电脑有一定的声望，即使是老式的。我们也可以试着做一个来连接一些我们还在使用的旧打印机端口设备。

虽然 8088 不是一台超级计算机，但这仅仅意味着你必须更加努力地去做。尤其是如果你想以每小时 8088 英里的速度去 T2 的话。
