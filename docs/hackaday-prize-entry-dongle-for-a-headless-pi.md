# 黑客日奖参赛作品:无头 Pi 加密狗

> 原文：<https://hackaday.com/2017/07/18/hackaday-prize-entry-dongle-for-a-headless-pi/>

大规模生产意味着有很多非常便宜的硬件。但这也意味着制造商不会花费数年时间在固件上，以挤出其中的每一个功能。不，这取决于我们。

【德清】拿了一个蓝牙低能耗/ USB 加密狗重新修补了固件[把它变成了一个远程键盘鼠标](https://hackaday.io/project/25829-ble-keyboard-mouse-dongle)，然后写了一个手机 app 来控制它。结果呢？插入 USB 加密狗，电脑认为它看到了键盘和鼠标。通过 BLE 连接电话，你就可以打字了——即使你身边没有你的[可靠的 F 型](http://hackaday.com/2016/03/07/reviving-the-best-keyboard-ever/)。

[Deqing]指出，从长远来看，人体工程学和延迟会让你讨厌使用它，但它只是意味着在你在那个无头单板 Linux 上安装并运行 SSH 之前，它会一直工作。如果你曾经使用过 USB 或 BLE 规范，你会意识到在使一切即插即用的过程中有一些幕后工作，并且[基于网络的界面](https://deqingsun.github.io/BLE-Keyboard-Mouse-Dongle/kbd_access.html)非常光滑。

好样的，[德清]！
