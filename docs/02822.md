# 在 FreeCAD 中将插槽和标签添加到您的盒子中

> 原文:[https://hack aday . com/2016/07/10/add-slots-and-tabs-to-your-box-in-free CAD/](https://hackaday.com/2016/07/10/add-slots-and-tabs-to-your-boxes-in-freecad/)

FreeCAD 是一个相当复杂的开源参数化 3D 建模工具。开源部分意味着你可以随心所欲地修改它。[Alexandre]正在开发一个模块，可以让他轻松地为模型添加标签、手指关节和 t 型槽。

现在这个插件仍然是实验性的，但是看起来是可用的。在视频演示中，[Alexandre]搭建了一个简单的盒子，然后添加了各种物理连接部件。你会注意到标签看起来像是添加到主界面上的部件——那是因为它们确实是！然后，他将轮廓导出到 SVG，并删除从侧面分隔标签的线条，并将这些文件交给他的激光切割机。瞧啊。一个完美的标签和 t 槽盒，只需要一点点手工。([Alexandre]提到这还只是实验性的，你应该在把它送到激光器前检查一下你的设计。)

这显然不是项目的结束，但却是一个伟大的开始。如果你正在使用 FreeCAD，并且需要用激光切割机制作一些盒子，请在 GitHub 上查看[。如果你正在寻找用你的激光切割机将](https://github.com/execuc/LCInterlocking)[连接](http://hackaday.com/2012/04/16/joinery-sure-to-be-useful-on-your-next-sheet-goods-enclosure/)、[弯曲](http://hackaday.com/2012/06/12/bending-laser-cut-wood-without-steam-or-forms/)或[紧固](http://hackaday.com/2013/02/27/acrylic-enclosures-use-integrated-clips-to-do-away-with-fasteners/)的更好的方法，我们可以满足你。
