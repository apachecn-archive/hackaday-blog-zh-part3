# Hackaday 奖参赛作品:控制 E-ZPass

> 原文：<https://hackaday.com/2016/06/11/hackaday-prize-entry-controling-e-zpass/>

你可以从波士顿开车到芝加哥，而不用从收费站拿一张票，或者把钱交给一个收费站工作人员。你能做到这一点是因为 E-ZPass，这是一种安装在美国东北部大多数汽车上的小型塑料砖。E-ZPass 包含一个与您的支票账户关联的 RFID 应答器。是的，这很方便，是的，这是政府在你不知情的情况下远程跟踪你的行动的一种方式。

对于他的 Hackaday 奖参赛作品，[Jordan]正盯着他仪表板上那个可疑的白色盒子[，并在他的电子通行证](https://hackaday.io/project/11938-e-zpass-command-and-control)上添加通知。他已经升级了他的 E-ZPass，增加了一点电路，以便在扫描时通知他，无论是在收费公路广场还是在曼哈顿中城行驶三个街区。

由于[Pukingmonkey]在 DEF CON 上的一次演讲，E-ZPass 砖的通知系统已经存在了几年。因为这个简单的电路，我们知道 [NYPD 正在收集在曼哈顿开车的人的 E-ZPass 数据。为什么？主权公民之类的东西。](http://gothamist.com/2014/01/07/whats_the_government_doing_with_you.php)

[乔丹]是把电子通行证通知系统比以前的版本更进一步，并增加了一个小 GPS 模块的日志功能。当然,[乔丹]的产品仍然会有闪光发光二极管，在电子通行证被读取时通知他，但通过将这些数据记录到 SD 卡，他将能够在电脑上播放一次公路旅行，并做一份适当的费用报告。收集费用数据时的安全性研究；没有比这更好的了。
