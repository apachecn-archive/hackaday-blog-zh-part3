# Toa Mata 乐高乐队真的很棒

> 原文:[https://hack aday . com/2016/05/29/toa-mata-Lego-band-actually-rocks/](https://hackaday.com/2016/05/29/toa-mata-lego-band-actually-rocks/)

[ [Opificio Sonico](http://www.opificiosonico.com/) ]已经在基于乐高的机器人音乐制作业务上有一段时间了，它显示了这一点。他在 YouTube 上发布了四个视频(以下都是内嵌的)，每个视频都展示了他的风格和乐高机器人技术范围的明确演变。

蠢朋克《活着》的第四集显然是最完美的。一个滑动平台可以让一个乐高玩具在某种设备上播放音乐。).演奏 DS 的机器人用唱针反复敲击一个音符，由于蠢朋克作曲的“效率”，这项工作变得更加容易。第三集，德佩什模式的“一切都很重要”是美妙的，部分是因为他使用压电垃圾作为打击乐器(就像 DM 自己一样)，部分是因为滑动[唱针](https://en.wikipedia.org/wiki/Stylophone)。但是看着他们。

除了明显的书呆子之外，这里还有一些严肃的制作工作在进行。哦对了，还有一个 Arduino 埋在那里做 MIDI 到 Lego 的转换。马塔乐队远不如[压缩机头](http://hackaday.com/2015/11/07/compressorhead-best-robot-band-ever/)金属，但同样，我们会给他们优势。

感谢[Johnyy]的提示！我们以前怎么没报道过这些塑料人？
