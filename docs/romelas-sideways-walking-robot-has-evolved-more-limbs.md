# 罗梅拉的侧身行走机器人进化出了更多的肢体

> 原文：<https://hackaday.com/2018/05/21/romelas-sideways-walking-robot-has-evolved-more-limbs/>

尽管波士顿动力公司(Boston Dynamics)等团体的原型取得了成功，但两足行走仍然很难实现。当机器人抬起一条腿时，它必须将重心转移到另一条腿上，以避免侧身摔倒。

具有增强动力的自主腿个人助手机器人(ALPHRED)通过从不同的角度来解决这个问题。 [ALPHRED 侧身行走，抛开胳膊腿的区分](https://www.youtube.com/watch?v=JgeoR5utVUw&feature=youtu.be&t=5m32s)。

这个机器人是加州大学洛杉矶分校的 RoMeLa，是他们解决传统两足动物路障方法的最新进展。当我们谈论[他们之前的版本，只有两条腿的纳比](https://hackaday.com/2016/12/04/this-robot-goes-through-life-sideways/)时，我们谈到了侧身行走。ALPHRED 将其扩展到四肢。正如下面的视频所示，所有的四肢都可以用于行走，或者是一个宽而稳定的伸展，或者四肢可以重新调整为一个更窄的狗或马一样的姿势，以便跑得更快。

除了行走，一个或两个肢体可以用作手来开门或递包裹，这就是为什么他们称它们为肢体而不是腿或手。只有一个动画展示了这种配置，但 RoMeLa 是一个机器人实验室，我们一直在关注，所以我们会让你知道他们是否演示了它。

该视频继续展示了一个整洁的主动顺从致动器，他们称之为熊，可反向驱动的机器人电磁致动器。搜索没有发现进一步的细节，但如果你有任何问题，请在评论中告诉我们。我们还喜欢看他们如何使用扬声器来给出电流量的粗略想法。虽然它既实用又简单，但也增加了一点科幻的感觉。
