# Hackaday 奖品条目:WiFi Game Boy 墨盒

> 原文：<https://hackaday.com/2017/05/14/hackaday-prize-entry-wifi-game-boy-cartridge/>

[DaveDarko]在今年的 Hackaday 奖中加入了一个独特的项目:WiFi 游戏男孩墨盒。如果你在 Hackaday.io 很活跃，我肯定你会在某个时候碰到 Dave，也许我们需要开始向他收取租金。

这个项目的目的是使用 ESP32 创建一个支持 WiFi 的 Game Boy cartridge，然后使用户能够做许多不同的事情。例如，它可以用作便携式[战争驱动](https://en.wikipedia.org/wiki/Wardriving)设备。你可以开车四处扫描当地的 WiFi 网络，这一切都来自于一个经典游戏男孩带来的童年美好回忆。

这个 WiFi Game Boy cartridge 甚至可以进行一些非常简单的网络浏览，或者作为一个独特的控制器来控制你所有的互联网连接。不管怎样，这个项目看起来很有希望，我们期待着在未来几个月看到进展如何。
