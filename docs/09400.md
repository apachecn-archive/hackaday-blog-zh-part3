# 使用 Google Home 进行位置共享

> 原文:[https://hack aday . com/2018/05/16/location-sharing-with-Google-home/](https://hackaday.com/2018/05/16/location-sharing-with-google-home/)

由于谷歌在互联网上近乎垄断的地位，在网络空间里行走很难不接触到这个庞大的数据收集巨头的某些方面。这通常需要共同努力，但在技术上是可能做到的。虽然[Mat]仍在使用一些谷歌产品，但他至少找到了一种方法，让[Google Home 使用位置数据，而无需与谷歌](http://www.notenoughtech.com/home-automation/google-home/google-home-location-sharing/)共享这些数据，这是朝着正确方向迈出的一步。

[Mat]的目标是通过 Google Home 使用谷歌的位置共享功能，但没有谷歌知道他生活中所有事情的可怕因素，也没有必须使用谷歌地图的麻烦。他正在使用一些东西来实现这一点，包括在 Raspberry Pi Zero 上运行的节点服务器，If This Then That (IFTTT)的免费帐户，带有 AutoRemote 插件的 Tasker，以及谷歌地图 API 密钥。将所有这些放在一起，并对 IFTTT 进行一些配置，他可以向他的 Google assistant(或 Google Home)请求位置数据，而无需与 Google 共享这些数据。

这个项目是 Google 工具的一个很好的实现，也是对 IFTTT 的一个强大的使用。此外，它还规避了谷歌在寻求了解所有数据时倾向于加入的一些令人毛骨悚然的因素。
