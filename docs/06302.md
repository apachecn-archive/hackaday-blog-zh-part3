# EV3DEV 乐高 Linux 更新

> 原文:[https://hackaday.com/2017/06/23/ev3dev-lego-linux-updated/](https://hackaday.com/2017/06/23/ev3dev-lego-linux-updated/)

这个月，Linux 发行版得到了更新。发行的目标是乐高 EV3，这是一个 CPU 乐高提供驱动他们的头脑风暴机器人。新版本包括 Debian 8.8 的最新内核和更新。它还包含一些 Wi-Fi 加密狗和其他更新所需的工具。

如果你以前没有见过 ev3dev，那么它就是使用 SD 卡在 EV3 硬件上引导的 Linux。你不必刷新电脑，如果你想回到股票，只需拿出 SD 卡。如果你喜欢，你也可以在树莓皮或比格犬骨上使用 ev3dev。有一个驱动程序框架，用于使用文件系统处理传感器、电机和其他项目。

在 EV3 上安装完整的 Linux 可以让您在 Linux 计算机上用 Java 或 Python 或任何其他工具进行编程。例如，下面这个来自[Juan Antonio brea Moral]的视频是他用 Java 发布的几个视频之一。

去年我们简单地提到了 ev3dev。我们知道当你想到 EV3 的乐高积木时，你会想到移动机器人，但是你也可以用它们[来建造自动化](https://hackaday.com/2017/05/29/chocolate-factory-simulation-makes-bars-with-lego/)。
