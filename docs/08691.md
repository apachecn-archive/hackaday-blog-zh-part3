# 看看这只巨眼的酷似虹膜和瞳孔的机制

> 原文：<https://hackaday.com/2018/03/04/behold-the-giant-eyes-orrery-like-iris-and-pupil-mechanism/>

这是一个更老的项目，但是用来创造这个巨大的、令人瞠目的[的机电解决方案值得一看](http://jarkman.co.uk/catalog/robots/giantstaringeye.htm)。[Richard]和[Anton]需要一个可以向任何方向看的大而不眨的眼球，他们的解决方案甚至提供了可调的瞳孔和虹膜大小。让瞳孔按需扩张或收缩也是一个非常好的功能。

巨大的织物球体从内部照亮，中心是一个灯泡，虹膜和瞳孔机制像一个手术室的一部分一样围绕着灯泡。通过将灯泡保持在中心，并围绕灯泡周围的蓝色凝胶(用于虹膜)和不透明圆盘(用于瞳孔)，眼睛可以看起来向不同的方向凝视。通过调节圆盘与灯泡的距离，可以改变虹膜和瞳孔的大小。

照相机系统挑出物体(如人)并引导眼睛注视它们。系统很聪明，但是执行并不完美。正如你在下面嵌入的短片中看到的，对行人的检测严重滞后。此外，在虹膜和瞳孔的运动中存在振荡。尽管如此，作为一种机制，它是一种美。

在视频中，你可以看到眼睛在检测人的时候有所滞后，但你也可以看到虹膜和瞳孔在移动时会跳动。如果电机太弱或负载过大，就会发生这种情况；当马达努力移动对它们来说太重或平衡差的负载时，它们最终会不断地超调，然后矫枉过正。缓慢而稳定的移动可能是好的，但更快更大的移动会振荡得更多。
