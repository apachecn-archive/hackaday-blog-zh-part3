# 3D 打印你一天内的下一个住所

> 原文：<https://hackaday.com/2017/03/11/3d-print-your-next-dwelling-in-a-day/>

建造一个 400 平方英尺的房子最短需要多长时间？几个星期？花 10，000 多美元，在 24 小时内尝试一个完全打印的结构。

这个放射状的住宅是由[Apis Cor]和六家合作公司在俄罗斯斯图皮诺用混凝土建造的原型。与传统的预打印或预制部件的组装不同(比如它在很大程度上是处于起步阶段的技术),建筑是作为一个整体打印的，在完成建筑的其余部分之前，打印机被起重机移走。它有一个浴室、走廊、客厅和一个紧凑的厨房——单身汉或单身女子需要的一切。

在如此短的时间内建造一座房子的诱人前景有几十种用途。此外，如果这种低成本住房被证明像承诺的那样持久，它可能意味着人们开始自己生活的另一种选择，或者为世界各地的家庭提供一种负担得起的模块化选择，在这些家庭中，更多的传统住所是他们无法企及的。

我们已经展示了[个人努力](http://hackaday.com/2016/06/28/one-mans-journey-to-build-portable-concrete-3d-printer-produces-its-first-tiny-house/)到 3D 印刷厂，以及[酒店规模的运营](http://hackaday.com/2016/04/15/3d-printing-houses-from-concrete/)，但是我们总是很高兴看到这个新兴领域的新创新。
