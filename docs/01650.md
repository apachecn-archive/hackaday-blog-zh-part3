# 2100 个机械装置

> 原文:[https://hackaday.com/2016/02/29/2100-mechanical-mechanisms/](https://hackaday.com/2016/02/29/2100-mechanical-mechanisms/)

[Nguyen Duc Thang]的史诗 [2100 动画机械装置](https://www.youtube.com/user/thang010146/videos)是我们见过的最好的 YouTube 频道之一。作为一名退休的机械工程师，[Nguyen Duc Thang]接受了一个巨大的挑战:在 Autodesk Inventor 中建立几乎所有可以想象的机制的 3D 模型，并制作动画供您娱乐和启发。不，我们没有为你看完所有的视频，但是我们相信，没有我们的帮助，你至少可以浪费几个小时。

如果你真的在寻找某个特定的东西，那么 YouTube 并不是一个完美的查找表。令人欣慰的是，[Nguyen Duc Thang]还制作了几百页的文档(pdf，zipped) 来配合该系列，对每个机制进行了分类、描述，并链接到视频。

这是一个惊人的资源，而且我们没有 3D 文件可能是一件好事；在灯丝成本和通过 2，100 个机制管理我们的 3D 打印机所花费的时间之间，我们已经破产了。还好我们不知道[数字机构和齿轮库](http://hackaday.com/2012/07/18/dmg-lib-digital-mechanism-and-gear-library/)或者 [KMODDL](http://hackaday.com/2012/06/08/kmoddl-a-mechanism-makers-dream-site/) 。
