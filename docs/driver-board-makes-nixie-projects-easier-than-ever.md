# 驱动程序板使谢妮项目比以往任何时候都更容易

> 原文：<https://hackaday.com/2017/12/08/driver-board-makes-nixie-projects-easier-than-ever/>

我们知道，我们知道——又一个谢妮钟。但实际上，这一个有一个巧妙的技巧:[一个易于使用的、功能丰富的 Nixies 驱动程序](https://github.com/dekuNukem/exixe)让好看的项目变得轻而易举。

尽管 Nixies 很酷——我们承认，在一定程度上，熟悉会滋生蔑视——但他们很难整合。[dekuNukem]指出，除了高电压之外，接触像 7441 这样的老式驱动器芯片可能具有挑战性，而且成本高昂。这个问题用价值约 3 美元的零件解决了，包括一个 STM32 微控制器和一些高压晶体管。PCB 有两种类型，一种用于 IN-12，另一种用于 IN-14，SPI 接口和高低压电源的连接均引出至接头引脚。这使得模块很容易插入主板或扩展卡。该驱动器支持过驱动，以适应中毒的阴极，127 个亮度水平的平滑调光，以及一个完全可调的管下 RBG 背光。在下面的视频中可以看到电路板的工作情况，其特点是具有漂亮的风格和高精度的时钟。

从[谢妮的 tachs](https://hackaday.com/2017/07/29/nixie-tachometer-displays-in-style/) 到[谢妮的 IoT clocks](https://hackaday.com/2017/10/13/hackaday-prize-entry-iot-nixie-clocks/),【dekuNukem】的主板应该会让谢妮的创意项目变得更加容易。但是如果你想驾驶[谢妮·达斯·维德](https://hackaday.com/2017/02/01/darth-vader-in-a-nixie-tube/)，你可能只能靠自己了。
