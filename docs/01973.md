# 英国广播公司微:位上的蛇

> 原文:[https://hackaday.com/2016/04/13/snake-on-a-bbc-microbit/](https://hackaday.com/2016/04/13/snake-on-a-bbc-microbit/)

BBC 的第一个微比特正在慢慢地进入黑客的圈子，正如任何廉价的教育小工具所预期的那样(见:Raspberry Pi)。[马丁]得到了一个，并创造了 LED 显示屏的“hello world ”:他创造了一个可以在这个小板子上运行的贪吃蛇游戏。

对于那些新手来说，Micro Bit 是嵌入式 ARM 系统中的最新产品。它有一个用于输入和输出的 23 针连接器，它有蓝牙和 USB 连接，大量的传感器和 25 LED 显示屏。对于完整的显示来说，这是很小的，但对于(马丁的)贪吃蛇游戏来说却绰绰有余。他能够使用[ntoll]中的 [upyed](https://github.com/ntoll/upyed) 工具创建一个十六进制文件，并将其上传到 Micro Bit。一旦他解决了所有的问题，他又进一步将游戏移植到《我的世界》和[树莓派感知帽子](https://www.raspberrypi.org/products/sense-hat/)。

[Martin]已经提供了所有的代码，如果你足够幸运能得到其中的一个。现在，它们似乎主要掌握在一些英国教师和学生手中，但它们变得像树莓派或原版 [BBC Micro](https://hackaday.com/2015/10/03/vintage-bbc-computer-gets-fpga-buddies/) 一样无处不在只是时间问题。它[已经运行 python](http://hackaday.com/2015/10/20/bbcs-microbit-gets-python/) ，所以这些新的主板前途无量。
