# 胶合板从升级的碎玻璃艺术灯中抢尽风头

> 原文:[https://hack aday . com/2017/02/04/胶合板-从回收的碎玻璃艺术灯中偷出表演/](https://hackaday.com/2017/02/04/plywood-steals-the-show-from-upcycled-broken-glass-art-lamps/)

环顾他的工作室，你会发现[保罗·杰克曼]比我们更喜欢胶合板。对于这些灯的底座，他将足够多的材料夹在一起，使其成为作品视觉效果的一个独特部分。一些工作与路由器和一些整理，他们看起来很棒！你可以在下面他的视频中看到他的工作和成果。

胶合板底座还隐藏了电子设备:一个变压器和一些发光二极管。为了给它们腾出空间，他把它们扔进数控镂铣机并挖空。罐子的盖子和底部用了一点环氧树脂。用彩色玻璃和一个透明的管子把罐子装满，让光线一直照射到顶部，这样就完成了。

优质胶合板很棒。它很结实，不会变形，而且从侧面看起来很棒。如果[Paul]的项目不能说服你，也许你想看看[Gerrit]写给材料的[情书，下次你在大盒子里的时候拿起一些。](http://hackaday.com/2016/04/26/materials-to-know-baltic-birch/)
