# 禅与阿杜伊诺艺术

> 原文：<https://hackaday.com/2017/05/16/zen-and-the-art-of-arduino/>

禅宗花园应该是放松和逃离日常生活的一个来源。重点应该是远离——除了别的以外——你的电子产品。除非你是【MakrToolbox】。然后你将制作一个漂亮的 zen garden 茶几，让你可以使用[滚珠轴承和 Arduino](https://www.hackster.io/MakrToolbox/zen-garden-cnc-end-table-b673f4) 在沙子上制作图案。下面可以看到一个视频。

从技术上来说，该设备几乎是一台倒置的 3D 打印机，没有 Z 轴。这个机械装置移动一块磁铁，磁铁控制着钢球并在沙子上画出图案。然而，这个项目真正令人印象深刻的部分是茶几的木工和令人印象深刻的文件，如果你想自己复制这个项目。

我们不禁认为这是一个非常好的成人蚀刻素描。[MakrToolbox]最初使用 3D 打印机控制板来移动一切，但后来决定采取不同的方法。从用户的角度来看，一个操纵杆驱动球。我们无法评论它是否有同样的舒缓效果。

你可能会认为数控禅宗花园是一个新奇的想法，但时间已经证明[它不是](https://hackaday.com/2014/05/31/cnc-zen-garden/)。不是[甚至接近](https://hackaday.com/2012/07/06/cnc-zen-gardening/)。[认真地](https://hackaday.com/2012/04/26/zen-rock-garden-table-uses-magnets-and-sand/)。然而，它可能是我们见过的最具美感的一个。
