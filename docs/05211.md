# 逆向技术:立体声唱片

> 原文：<https://hackaday.com/2017/03/28/retrotechtacular-stereo-records/>

20 世纪见证了一些惊人的技术发展。我们从飞机到月球。我们从计算尺发展到了数字计算机。将电话转换成手机。但那个时代最惊人的两项壮举是非技术人员可能很难想到的。广播和电视从单声道和黑白到立体声和彩色的转变。有趣的是，工程师们设法找到了一种方法，将新的更好的结果转化为与旧版本相同的形式，这是令人惊讶的部分，以旧技术仍然有效的方式进行。也许这是当今新技术发展的速度，但我们今天没有这样做。数字电视需要全新的一切:发射机、接收机、频率和录音设备。祝你在 25 岁的电脑上玩最新的视频游戏好运。

很难记得商店里什么时候充满了各种各样的音频和视频媒体。我们注意到所有形式的媒体都开始消失。如今，一切音频和视频都是流式传输或下载的。唱片、8 轨唱片、磁带，甚至 CD 和 DVD 正在消失。然而，黑胶唱片在过去几年中因其新奇或怀旧的价值而卷土重来。

蜡、箔或乙烯基上的录音或多或少是由[托马斯·爱迪生](或者，也许是为他工作的人)早在 1877 年完善的相同过程，尽管我们认为的平面唱片直到 1890 年左右才出现。

原理很简单。声音产生的气压在记录媒体上切割出一个凹槽。压电唱针(或后来的带有动态元件的唱针)跟踪凹槽并再现相同的声音。放大它，留声机就可以用了。你可能会喜欢下面视频中修复的留声机。

## 立体声

到 20 世纪 50 年代，他们那个时代的黑客正在建造或购买“高保真”设备，这种设备听起来比当时唱机和调幅收音机发出的劣质音频更好。最终，公司会推出立体声录音。但这些唱片看起来没有任何不同，它们仍然可以在标准(单声道)唱机上播放。这怎么可能呢？不，它不是两张独立的唱片，就像在巴黎技术博物馆里发现的这个帖子顶部的老式播放器一样。

## 真实的故事

当时相当令人印象深刻的特效。[鲍勃]确实把一些事情简单化了。首先，凹槽可以具有垂直分量和水平分量。但是垂直方向的分辨率没有那么好，所以这意味着一个频道会处于不利地位。相反，单个凹槽中的两个轨道旋转 45 度，使得每个通道具有一些水平分量和一些垂直分量。

[Bob]想让你认为这是 RCA 发明的，尽管他从来没有直截了当地说过。事实上，百代唱片公司的 Alan Blumlein 早在 1931 年就申请了该方案的专利。第一张商业立体声唱片也不是来自 RCA，直到 1957 年才出现。

由于凹槽的旋转方式，唱针的水平移动是左右轨道的组合，与单声道信号相同。垂直运动带来了差异:左声道减去右声道，或 L-R。

这就是单声道唱片在立体声播放器上正常播放的方式。轨道上的水平运动将在两个声道上再现相同的声音。相反，读取立体声轨道的单声道唱针只会拾取轨道的水平部分，并一起播放两个通道。不幸的是，许多单声道播放器不能很好地上下移动，并且可以佩戴立体声唱片，所以用户学会了不要在单声道播放器上播放立体声唱片，即使它可以工作。当然，前提是你有同样大小的凹槽。旧唱片有更宽的凹槽和更宽的唱针。

## 和睦相处

在一个市场上，埃尔维斯·普雷斯利仍然在销售每分钟 78 转的唱片，因为他的粉丝买不起新的唱机，这种兼容性非常重要。我们想象[Alan Blumlein]看到我们如何例行公事地告诉每个人扔掉他们的磁带换成 CD，扔掉他们的 CD 换成数字音乐，会感到震惊。电视也是一样。制作一个黑白电视机仍能接收的彩色信号是一个奇迹(也是未来技术革新的一个主题)。让每个人扔掉电视机换新的或者购买政府补贴的转换器的想法确实不会被接受。

我们不禁会想，我们是否在兼容性上竭尽所能。我们真的必须每隔几年就扔掉操作系统和 CPU 吗？每次合同到期，我们真的需要把手机内存翻倍吗？或者这是一个聪明的计划淘汰策略？作为创造事物的人，我们在兼容性方面做得怎么样？我们就看历史如何评判了。
