# 小型 1000 瓦感应加热器

> 原文：<https://hackaday.com/2015/10/23/a-small-1000w-induction-heater/>

[Proto G] [建造了一个小型的桌面感应加热器](https://www.youtube.com/watch?v=1wwlbN-9jsU)，它能够制造小型铸件，熔化少量金属，并作为我们见过的最好的焊料锅之一。

感应加热器由定制的零电压开关(ZVS)驱动器构成，由一个 48V、1000W 的小电源供电。虽然这使得一个非常小的感应加热器，它仍然非常有能力。在下面的视频中，只需要几秒钟就可以将螺丝刀加热到熔化焊料的温度。

虽然感应加热机器对烙铁基本上没用，除非你有几个古董的、无动力的、喷灯驱动的烙铁，[但它确实是一个很好的焊料锅](https://www.youtube.com/watch?v=xjs8OknXyYw)。[Proto G]用李兹线替换了他的感应加热器中的工作线圈。实际的焊料锅是由包裹着气凝胶的玻璃纤维绝缘材料的钢管制成的。与他的旧焊料锅相比，这台机器可以瞬间升温，并且能够润湿一些电线接头。

这种感应加热器的未来计划是为不同的金属制造更多的附件，a [Proto G]有一些气凝胶毡，他可以用来制造一些小型金属铸件。
