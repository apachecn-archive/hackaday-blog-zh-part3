# 旧电视借给复古魔镜的情况

> 原文：<https://hackaday.com/2017/12/12/old-tv-lends-case-to-retro-magic-mirror/>

还记得电视是家里最重要的电器的日子吗？黎明时播放早间新闻和天气预报，约翰尼·卡森说晚安时关闭，这是通往更广阔世界的必不可少的门户。广播电视可能已经放弃了对公众思想的控制，转而支持智能手机，但是内置在旧电视中的信息门户可能会带你回到过去。

看起来[MisterM]对复古造型有点兴趣。为了证明这一点，我们可以看看视频中的壁纸，以及几个月前他的谷歌化 Radio Shack intercom 项目。他目前的项目应该很合适，因为它是基于一台 70 年代的 8 英寸黑白电视。当时的电视体积庞大，以适应 CRT 的长颈，所以他决定剪掉大部分外壳，只使用边框。一个 8 英寸的 Pimoroni 显示器位于旧显像管曾经存在的地方，复制了原来 4:3 的纵横比。Chromium 设置在 kiosk 模式下，家人可以使用原来的调谐旋钮从各种新闻和信息“频道”中快速选择，而从回收的鼠标中取出的部件可以将音量控制变成滚轮。

这是对魔镜概念的一个很好的扭曲，与我们见过的其他复古电视项目略有不同，比如复古游戏控制台或智能电视的老式外壳。
