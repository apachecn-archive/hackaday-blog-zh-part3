# 烤一个新鲜的树莓 Pi:再也不用为配置 Pi 而烦恼了

> 原文：<https://hackaday.com/2016/11/04/bake-a-fresh-raspberry-pi-never-struggle-to-configure-a-pi-again/>

大卫·弗格森发明了一个叫做 Pi Bakery 的小工具。一半是 MIT Scratch，一半是 configuration utility，它[在所有不同的启动脚本上放置了一个漂亮的可视化界面，并且组装了](http://www.pibakery.org/)Raspberry Pi 社区用来配置流行的单板计算机的组件。

树莓派有点奇怪。它们大多被塞进微控制器曾经居住的插槽中。微控制器的好处在于它们只是打开并开始运行。没有操作系统可以启动。没有要装载的文件系统。当然，微控制器的缺点通常是没有操作系统可以引导，也没有文件系统可以安装。不管怎样，在 Raspberry Pi 真正开始侵占微控制器的领地之前，你必须[花点时间配置](http://hackaday.com/2016/02/03/code-craft-cross-compiling-for-the-raspberry-pi/)一个 Raspbian 安装。

Pi Bakery 把这些都抽象化了。你可以拖动代表脚本的块，按照你希望它们运行的顺序。如果你想让你的 Pi 启动，连接到 WiFi，然后启动一个 VNC 服务器，这很简单，只需按照正确的顺序拖动方块，然后填空。休息后，您可以在视频中看到它的运行示例。
