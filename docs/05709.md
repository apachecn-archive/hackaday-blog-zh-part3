# 这个黑客在一个 Altoids 罐子里装了一整个 RetroPie

> 原文:[https://hack aday . com/2017/04/23/this-hacker-fit-an-entire-retro pie-in-an-alto ids-tin/](https://hackaday.com/2017/04/23/this-hacker-fit-an-entire-retropie-in-an-altoids-tin/)

几个月前，[wermy]制作了 mintyPi，这是一款基于 Raspberry Pi 的游戏控制台，可以放在 Altoids 罐中。这太神奇了——有一个复合 LCD、一个音频 DAC 和一个切碎的任天堂控制器，所有这些都连接到一个树莓 Pi 上，以便在路上玩老式游戏。现在，有一个新的明蒂皮。[mintyPi 2.0 与上一代基于 mint 的游戏主机](http://www.sudomod.com/mintypi-2-0/)相比有了很大的改进，拥有更好的屏幕、更好的按钮、定制的 3D 打印挡板和更好的音频。的确，我们生活在一个黄金时代。

mintyPi 的第二个版本使用 3D 打印部件，并包括一个真正的铰链，以在 Altoids 罐打开时保持显示器支撑。[wermy]没有使用基于 DAC 的音频解决方案，而是使用 USB 声卡来获得更清晰、更清脆的声音。这个版本还使用了新的无线版本的树莓 Pi Zero。Raspberry Pi Zero W 允许这个 Altoids tin 连接到互联网，或者，让用户能够在这个东西上转储 rom，而不必将其连接到计算机。

对于软件来说，这款复古 Altoids 视频游戏机运行的是 RetroPie，这是一种非常流行的在低功耗 Linux 机器上运行复古视频游戏的方法。一切都在那里，从 NES 到阿姆斯特拉德到世嘉主系统。

目前，还没有很多关于[wermy]如何创建 mintyPi 2.0 的细节，但他承诺很快会有一个指南。在那之前，我们只能对下面嵌入的视频垂涎三尺。
