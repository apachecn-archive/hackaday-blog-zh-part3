# Hackaday 奖:一个自主沙滩艺术机器人

> 原文：<https://hackaday.com/2017/04/29/hackaday-prize-an-autonomous-beach-art-robot/>

有些人发现很难看到一大片平坦的地板或地面而看不到画布。从棒球公园的外场草地到一些贫穷农民的麦田，践踏、修剪或绘制图案都可能带来不可抗拒的诱惑。但是画布越大，构图就越有挑战性，这就是这个自动沙滩梳理艺术机器人发挥作用的地方。

很大程度上仍在进行中，[帕布罗·奥德修斯]的海滩机器人是为了利用他家附近退潮留下的几公里宽的海滩而建造的。那块巨大的画布正在等待整理，而这个机器人就是为此而生的。行走装置本身很简单——一个挤压成型的铝制底盘，由轮椅驱动，增加了光学编码器，并拖着一个可伸缩的耙子——但这个机器人布满了专用于导航的电子设备。一对 Arduinos 运行双里程表、指南针和一个 GNSS 接收器，并提供智能手机界面进行动态更改。该艺术作品被合成为 DXF 文件，转换为经纬度点，并作为 KML 导出到谷歌地球。这意味着机器人可以被带到海滩上，并被允许自主绘画。一个早期的测试运行被认为是下面的突破；更好的“画笔”正在制作中。

看着艺术在海滩上展开会让人放松，就像看着一个禅宗花园被创造出来。我们期待着[巴勃罗]在这件事上的进展。当然，艺术机器人并不是宽阔的海滩吸引的唯一自主机器。
