# Hackaday 奖参赛作品:模块化轨道照明

> 原文:[https://hackaday.com/2017/05/10/modular-rail-lighting/](https://hackaday.com/2017/05/10/modular-rail-lighting/)

在经营任何一种水培农场时，都有许多照明解决方案——很少有便宜的。起初，为了寻找宜家昂贵的水培系统照明的替代方案，Fartsparkle 教授]和他们的同事制作了一个原型[轨道系统，允许夹式 LED 板](https://hackaday.io/project/20653-dirt-cheap-modular-led-lighting-system)用于可变照明选项。

[![](../Images/2c3d77f7566750a040a4df910f0fb589.png)](https://hackaday.com/wp-content/uploads/2017/04/2569791490574180085.jpg) 灵感来自电线和轨道照明系统，关键是安装在 LED 板底部的 5 毫米保险丝座。剪断止动夹，便于安装和从安装导轨上拆除。横杆本身也是 LED 板的电源导管，但使用各种 3D 打印的挂钩[法特斯巴克教授]可以很容易地将它们挡在外面。照明由电源注入板上的电位计控制，以及通过 ESP8266 的任何家庭自动化控制。

Fartsparkle 教授]声称，电路板可以沿着轨道滑动，没有任何明显的闪烁，但它们确实存在散热问题。除此之外，原型工作得足够好，3W 的 led 可以以一半的功率运行。

当[阳光](http://hackaday.com/2016/07/12/solar-powered-hydroponics/)不是一个选项，但是你仍然在寻找一个能够[自动化](http://hackaday.com/2016/08/28/hackaday-prize-entry-automated-hydroponics/)的解决方案时，这是一个巧妙的——并且便宜的——解决方法。
