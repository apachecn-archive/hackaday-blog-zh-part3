# 花 200 美元学习高级 PCB 设计，值得吗？

> 原文：<https://hackaday.com/2017/05/09/learn-advanced-pcb-design-for-200-worth-it/>

[Helentronica]从学生时代就开始使用 Altium Designer 来设计 PC 板。现在作为一名自由职业者，他觉得自己并不完全知道他想知道的一切。请记住，他做过 BGA 和 LVDS 布线的多层板，所以他不是新手。他决定花大约 200 美元参加联邦学院的高级课程。在这个互联网一切免费的时代，花 200 美元看一些视频值得吗？

[Helentronica]可能也在考虑同样的问题。然而，他对课程项目感兴趣，该项目是一个开源计算机模块，配有 i.MX6 处理器、1gb DDR 3 SDRAM 和许多扩展选项。事实上，卖给他的广告文案是:

> *您将在一个真正的高速主板上练习，该主板配有 1.2GHz CPU 和 DDR3、PCIE、SATA、HDMI、LVDS、1Gb 以太网等等*

他完成了课程。值得吗？我们不会破坏这个故事，但你应该看看他的帖子，了解一下。即使你不想降低 200 美元或者你不使用 Altium，你也可能会学到一些关于 PC 板布局的技巧。

尽管有付费的课程视频，但 Fedevel Academy 在 YouTube 上确实有很多免费的内容，包括 Altium 的介绍，你可以在下面观看。不过，整个频道都值得一看。

Altium 不像一些开源工具那样在黑客圈子里很常见。几年前，我们确实对 Allegro 和 Altium 进行了比较，但我们敢打赌，这两款产品从那以后都发生了很大的变化。如果你想调查更常见的黑客工具，如 Eagle、Protel 和 KiCAD，[Brian]已经在[发布这些工具几个月了](https://hackaday.com/2016/09/21/creating-a-pcb-in-everything-introduction/)。
