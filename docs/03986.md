# pi Cart:2400 款游戏合二为一

> 原文：<https://hackaday.com/2016/11/01/pi-cart-2400-games-in-one/>

一场比赛变成 2400 最快的方法是什么？将一个树莓派零跑 RetroPie 塞进一个 NES 墨盒，称之为 [Pi Cart](https://howchoo.com/g/mti0oge5nzk/pi-cart-a-raspberry-pi-retro-gaming-rig-in-an-nes-cartridge) 。

只要你有良好的电缆管理技能和合适的零件，这种优雅的小构造不需要焊接。为此，[Zach]评论说，找到一个足够小的 USB 适配器(另一个主要组件)以适合盒式磁带需要冗长的反复试验，所以他[很有帮助地链接了一个](https://www.amazon.com/gp/product/B003M0NURK/?tag=mti0oge5nzk-20)他保证会工作。人们可以跳过这一步，但沙发合作的潜力可能值得努力。

另一个症结可能是任天堂对安全螺丝的使用；如果你有合适的钻头或螺丝刀，太棒了，否则你可能[不得不临时准备](https://www.youtube.com/watch?v=tqleaD3yYys)。削减一些塑料以扩大卡座开口，从而为 USB 集线器、微型 USB 电源端口和 HDMI 端口创造足够的空间。如果你选择缩短电缆，将它全部安装在里面应该很简单，但你可能需要玩一点俄罗斯方块来确保一切都合适。

使用《回到未来》游戏卡带概括了这个项目的精髓，考虑到它的内容在 20 世纪 80 年代几乎是科幻小说——这是一个很好的接触。我们有很多复古的设置——每个都有自己独特的风格——但如果你正在寻找一个更适合时代的游戏站，你可以简单地为这个目的挖出一个 [NES。](http://hackaday.com/2014/06/01/original-nes-guts-upgraded-with-retropie/)
