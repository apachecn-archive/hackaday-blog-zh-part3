# 对病人阿尔法进行老式的弯曲循环

> 原文：<https://hackaday.com/2016/02/15/good-old-fashion-circuit-bending-with-patient-alpha/>

对于我们中的许多人来说，某种音频电路是我们第一次涉足电子领域。说话和拼写，雅达利朋克游戏机，薄荷罐里的 LM386，听起来熟悉吗？如果没有，你应该帮自己一个忙，完成几个简单的项目。对于我们这些做过一点被孩子们称为*电路弯曲的人来说，*【尼克拉斯·彼得】用他的[病人阿尔法项目](https://hackaday.io/project/9640-patient-alpha)给我们带来了熟悉的黑客技术。休息后，您可以在视频中看到构建过程的延时视频和一个简短的演示。

[Nickolas]对他的 2013 款 Executor 钥匙扣做了一些改装；用于电阻交换的强制性电位计总是很受欢迎。通过 3.5 毫米插孔添加音频是我们中的一些人不会想到的，但它让执行者对着你的严重音频设备尖叫，以获得最大的耳机。值得一提的是，[Nickolas]在这个黑客的成品外观上做得很好，尽管他是从一个外壳中的产品开始的，但他仍然不厌其烦地以一种审美愉悦的方式定制他的所有部分。然后他做了第二个。

我们已经报道了大量的电路弯曲项目:从[一个老派的电路弯曲项目](https://hackaday.com/2015/09/03/the-kraakdoos-musical-abuser-of-an-ancient-opamp/)到[一个在一个适当的音频套件上建立电线的项目](https://hackaday.com/2015/06/24/hackaday-prize-entry-ultimate-circuitbending/)。翻出你的烙铁，开始吃吧。
