# 树莓 Pi-动力回到未来时间电路

> 原文:[https://hack aday . com/2015/11/10/raspberry-pi-powered-back-to-the-future-time-circuits/](https://hackaday.com/2015/11/10/raspberry-pi-powered-back-to-the-future-time-circuits/)

这里有些东西庆祝一下*回到未来*中的所有事件都发生在过去有点晚，但这就是时间机器的作用，对吧？[迪特尔]创造了 [Pi 供电的时间电路](http://www.deater.net/weave/vmwprod/hardware/time_circuit/)和一个通量电容器。他可能没有德罗宁，但他有把德罗宁变成酷车的设备。

在*回到未来*中屏幕上显示的“时间电路”实际上并不复杂；《泰晤士报》只是用灯光和凝胶剪出的剪贴画；不需要真正的电子魔法。当然， *BttF* DeLorean 已经被改造和翻新，配备了看起来像*和*的时间电路，而且【Deater】的时间电路拥有你所期望的一切:显示目的地、当前和最后时间、音效、数字键盘、通量电容器和速度计。

虽然它没有完全模拟电影中的时间电路，但这未必是一件坏事。电影中的时间电路是彩色凝胶，对于基于树莓派的道具来说并不实用。这是一个伟大的建设，一个会看起来很棒的 98 年日产 Altima 或德罗宁
