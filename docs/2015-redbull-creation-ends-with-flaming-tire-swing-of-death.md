# 2015 红牛创作以燃烧的轮胎死亡摇摆结束

> 原文：<https://hackaday.com/2015/10/01/2015-redbull-creation-ends-with-flaming-tire-swing-of-death/>

神圣*#$&。这就是 2015 年红牛创作大赛的总结。太棒了。你还能在哪里乘坐汽油发动机驱动的轮胎旋转死亡，完成火球发射器？嗯…也许燃烧的人…

总之，是黑客马拉松吗？不，这是一次**马拉松**。可能使用了一些 Arduinos 和 Atmel，但大多数项目都是严肃的机械奇迹。参赛队伍有 72 小时的比赛时间，以“严肃的乐趣”为主题。

他们真的很搞笑吗？从人类大小的仓鼠轮子，到玻璃粉碎回收机，甚至是垃圾箱里的街机风格的嘉年华游戏，没有一个团队对严肃的乐趣有类似的想法。太棒了。

虽然[Mike]和[Brian]今年没能来(见他们去年的报道)，但我还是从加拿大开车过来看比赛，并在路上拍了一些视频。看看这个。
