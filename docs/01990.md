# 大学设想飞行甲虫群；但是在你飞之前先爬

> 原文：<https://hackaday.com/2016/04/12/universities-envision-flying-beetle-swarms-but-crawl-before-you-fly/>

南洋理工大学和加州大学伯克利分校的研究人员想要回答这个问题:如何制造一架可以全天飞行的小型无人机？问题是，无人机需要电池或其他能源，但大电池需要大无人机。

他们的回答？拿一只巨大的甲虫，[在上面绑上足够的电子设备](http://www.livescience.com/54233-scientists-turn-beetles-into-cyborg-insects.html)来传递微小的震动来引导昆虫的飞行。这种微小的震动不需要太多的能量，一旦甲虫进入轨道，除非人类飞行员需要纠正方向，否则没有必要进一步震动。最近的工作允许一个类似的控制器来控制甲虫的每条腿，把它变成一个更加多功能的飞行或行走机器人。

这些甲虫大约 3 英寸宽，可以举起控制背包和传感器。一块小电池可以让一架甲虫无人机工作一整天。该团队希望最终能够制造出控制背包，这样它就可以依靠收集的能量运行，而不需要任何电池。

这种背包的材料成本不到 8 美元，并且大部分使用现成的组件。有机蜂蜡对甲虫的甲壳无害，可以将电子设备固定在昆虫身上。六个电极连接到甲虫的视叶和飞行肌肉。无线信号影响甲虫起飞、转向，甚至盘旋。行走版本使用八对电极来控制生物的前腿。最终目标是完全控制所有的腿。

我们已经报道了大量的电子人改造，但通常是针对人类的。然而，我们已经看到了早期版本的[robobeetle 无人机](http://hackaday.com/2009/09/25/radio-controlled-beetle-flight-footage/)。如果你从未见过面包板上的甲虫，你一定想看看下面的视频。
