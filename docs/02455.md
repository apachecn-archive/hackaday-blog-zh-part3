# Hackaday 奖参赛作品:机器人假腿是开源的，可以 3D 打印

> 原文：<https://hackaday.com/2016/05/25/hackaday-prize-entry-robotic-prosthetic-leg-is-open-source-and-3d-printable/>

我们以前一直在为自我复制机器 3D 打印零件，但我们一直在错误的机器上工作。软件和机器人工程师[大卫·桑切斯·法勒罗]将通过他的 Hackaday 奖参赛作品来纠正这一问题，这是一个用于人类的 [3D 可打印、开源的机器人假腿](https://hackaday.io/project/10904-drakkar-printable-robotic-prosthetic-leg)。

[David]在那里找不到合适的、可 3D 打印和定制的假肢，鉴于商业假肢的高昂价格，他开始了自己的假肢项目，名为 Drakkar。他设计的“骨骼”由 M8 钢螺纹杆制成，这有助于保持低成本，但在世界各地也很容易获得。膝盖由 DC 电机主动弯曲，根据[源代码](https://github.com/davidsanfal/drakkar)，电位计将膝盖的位置读回 PID 回路。

[![drako_foot](img/a0b2ecd6584881d384b31abc7fa107c8.png)](https://hackaday.com/wp-content/uploads/2016/05/drako_foot1.gif) 在制作他的第一个原型时，【大卫】很快发现，从 3D 打印的零件复制人类足部的形状和复杂的机械结构会太脆弱。相反，他研究了山羊的蹄子是如何在只有两个较大脚趾的情况下适应不平坦的地形的。所有的结果和学习然后进入第二个版本，现在也适应用户的高度。这项完全在 FreeCAD 中完成的设计看起来确实很有前景，有一天可能会与高价的商业假肢竞争。
