# 指纹扫描仪真的没有那么安全

> 原文：<https://hackaday.com/2016/03/11/finger-print-scanners-really-arent-that-secure/>

也许你已经怀疑这一点，但 MSU 计算机科学的研究人员刚刚发表了一篇论文，解释了用手指的喷墨打印扫描来欺骗指纹扫描仪是多么容易。

我们不是在谈论用强力胶或任何东西铸造一个新手指，而是使用你可以在纸上打印的导电墨水。一个可以解锁你的智能手机的纸质指纹。我们已经告诉过你[指纹对于安全性来说很糟糕](http://hackaday.com/2015/11/10/your-unhashable-fingerprints-secure-nothing/#more-177237)，但是希望这能让你明白这一点。

[曹凯]和[安尼尔·K·贾恩]发布了概述他们技术的论文。使用现有的指纹扫描(可以从手机的扫描仪中获取)，图像被镜像，然后使用常规的喷墨打印机打印，其所有的彩色墨盒都被 AgIC4 银导电墨水取代。
 现在这种墨水和与之配套的纸张相当昂贵——你可能会看到这些墨盒接近 500 美元，但当涉及到假冒某人的身份时，这肯定不足以阻止一个潜在的小偷。
