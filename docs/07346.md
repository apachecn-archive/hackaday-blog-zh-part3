# 周五黑客聊天:不可思议的猎兔犬板

> 原文:[https://hack aday . com/2017/10/11/Friday-hack-chat-the-incredible-beagle board/](https://hackaday.com/2017/10/11/friday-hack-chat-the-incredible-beagleboard/)

在过去的一年左右，BeagleBoard 社区已经看到了一些令人难以置信的硬件。[芯片上的 beagle bone](https://hackaday.com/2016/05/10/new-part-day-a-beaglebone-on-a-chip/)—Octavo OSD 335 x—是一个完整的计算系统，具有 DDR3、数吨 GPIOs、千兆位以太网和那些所有重要的 pru，它们被塞进一片布满焊球的环氧树脂中。这种芯片被制成了[微型 DIY 袖珍骨牌](https://hackaday.com/2017/04/15/an-even-smaller-beaglebone/)，现在[官方袖珍骨牌](https://hackaday.com/2017/09/23/the-tiny-25-pocketbone/)在普通电子元件经销商处大量现货供应。

[![](../Images/9adbdaf6350e84ad9e845bbbd178d32e.png) ](https://hackaday.com/wp-content/uploads/2017/10/allaboutbeagleboardhackchat-01-1.png) [在本周的黑客聊天](https://hackaday.io/event/27601-beagleboard-community-and-hacks)中，我们将讨论猎兔犬板、猎兔犬骨、口袋猎兔犬和口袋骨。beagleBoard 和 beagle wrangler 的联合创始人[Jason Kridner]将现场回答您关于 Beagle 平台的相关性、BeagleBoard 的发展方向以及在 Linux 环境下闪烁 led 的最佳方式的内部工作方式的所有问题。

这次聊天的主题将包括 BeagleBoard 的发展方向，BeagleBoard 涉及的社区，以及如何充分利用这些宝贵的可编程实时单元。一如既往，我们接受来自社区的问题，[提交到这里](https://docs.google.com/spreadsheets/d/17HGpRljQE87t50rX47U-BkkFI61FZJWCDueec7KckKQ/edit#gid=0)。

作为额外的特别奖励，本周我们将赠送一些硬件。Digi-Key 提供了一些 PocketBeagle 板。如果你有一个项目的想法，把它放在讨论页上，我们将挑选最酷的项目，并给某人发送一个袖珍猎犬。
