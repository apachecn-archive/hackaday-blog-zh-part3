# 玩纸工艺品电子活动书

> 原文:[https://hack aday . com/2017/07/27/play-with-a-paper craft-electronics-activity-book/](https://hackaday.com/2017/07/27/play-with-a-papercraft-electronics-activity-book/)

随着导电墨水变得越来越容易获得，种类也越来越多，我们开始看到一些有趣的应用。[Marion Pinaffo]和[raphal Pluviange]创建了一本关于 papercraft 项目的书，这些项目使用银基墨水制作电路导线，碳基墨水制作电阻，以及变色墨水。电子元件的引线滑入纸上切割的缝隙中，与导电墨水迹线相连。

[Marion]和[raphal]使用 555、ATtiny85s、手表电池和其他硬件来使每个活动或项目独一无二。许多项目使用滚动滚珠轴承在压电扬声器中发出嘟嘟声。他们还制作了设计精美的页面来搭配电子产品。

对于新手来说，这看起来是一种玩电子产品的有趣方式，一旦纸质部分被废弃，用户就只剩下硬件了。想象一下，一个初学者在谷歌上搜索 Tiny85 的引脚排列，或者发现步进音调发生器，并用 555 制作一个。

如果你喜欢这个项目，你会喜欢我们之前发布的[工作纸风琴](http://hackaday.com/2017/02/14/god-of-papercraft-builds-working-organ-used-for-own-adulation/)和[纸电阻计算器](http://hackaday.com/2012/04/27/a-papercraft-resistor-calculator-from-adafruit/)。
