# 压力板 LED 杯垫！

> 原文：<https://hackaday.com/2016/09/24/pressure-plate-led-coasters/>

为了更多地使用他的柔性 LED 灯，Hackaday alum 和 Tindie 作家[Jeremy Cook]试图在他的晚餐或咖啡桌上加入一种时髦的口音:[点亮杯垫](https://www.youtube.com/watch?v=9idrDSooZlQ)。

使用他的 CNC 路由器雕刻出两块半透明的塑料来容纳四块 3V CR2032 电池、四块 LED 条和一些电线，[库克]已经创建了一个压力板电路，一旦饮料放在上面，它就会激活。然而，电路的原始布局不起作用，LED 灯条的空间太小。快速的重新设计和更多的时间在他的路由器上产生了一个几乎可以工作的产品。起初，他打算用螺丝来固定杯垫，一旦他将杯垫顶部变薄，让它更容易弯曲并完成回路，热胶就提供了完美的弹力。

 [https://www.youtube.com/embed/9idrDSooZlQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/9idrDSooZlQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



[Jeremy]指出，目前杯垫只有在杯子接近满的时候才会发光，但他建议这在餐厅中可能很有用，当饮料快空了时，一眼就能通知服务员。如果你想造一个杯垫，告诉你你的咖啡或茶何时处于最佳温度，[别再找了](http://hackaday.com/2015/12/04/smart-coaster-informs-you-when-your-drink-is-ready/)。
