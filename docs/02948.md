# 用旧手机制作智能手表

> 原文：<https://hackaday.com/2016/07/19/make-a-smartish-watch-from-an-old-cell-phone/>

寻找一个有趣的垃圾箱黑客？你有一部老式的诺基亚手机吗?(与你现在的智能手机相比),它就是不死心。Tinkernut 为你准备了一个不错的窍门:[用旧手机制作智能手表](http://www.tinkernut.com/portfolio/make-smartwatch-old-cell-phone-part-1/)。具体来说，这个项目详细介绍了如何制作一款智能手表，它可以显示时间、日期、来电和来自诺基亚 1100 手机显示屏的文本以及其他一些信息。

这 3 个视频系列讲述了如何提取显示器，将其连接到 Arduino，并通过蓝牙将其连接到 Android 手机。我们已经见过[几个类似的](http://hackaday.com/2016/01/10/pic32-smart-watch-2/) [智能(ish)手表](http://hackaday.com/2014/07/07/diy-oled-smart-watch/) [构建](http://hackaday.com/2014/02/17/make-your-own-smart-watch/)，但是这个很好地涵盖了整个过程，包括在 [MIT AppInventor](http://appinventor.mit.edu/explore/) 中构建 Android 应用。当然，最终的结果没有苹果手表那么完美，但它便宜得多，也更容易破解…
