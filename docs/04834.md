# 蓝牙自动化远程挂机

> 原文:[https://hack aday . com/2017/01/28/bluetooth-automation-remote-hang-around/](https://hackaday.com/2017/01/28/bluetooth-automation-remote-hangs-around/)

用你的智能手机来控制你的家庭自动化系统给了你很大的灵活性。但是对于像开灯和关灯这样简单的事情来说，经历解锁手机、选择正确的浏览器页面或应用程序，然后最终点击你需要的按钮的整个过程可能是一种痛苦。如果这一切都可以通过触摸一个单独的物理按钮来完成，那就简单多了——但手机已经没有很多这样的功能了。[fall dear]带来了解决方案——[一个可穿戴的智能手机四按钮蓝牙遥控器，启动](http://falldeaf.com/2017/01/four-button-bluetooth-remote-mobile-phone-control/)。

该项目是围绕 [RFDuino](http://www.rfduino.com/) 构建的，这是一个 Arduino 平台，用于快速轻松地构建蓝牙兼容项目。到目前为止，非常简单——四个按钮连接到一个具有无线功能的微控制器上。真正的诀窍是 3D 打印的衣夹式外壳，它允许你将四按钮遥控器夹在衣服上。[fall dear]的第一次尝试是安装在掌上的装置，他们发现这妨碍了常规任务；我们一致认为，可穿戴版本在实用性方面提供了重大升级。

智能手机方面的事情由使用 [PhoneGap](http://phonegap.com/) 编码的定制应用程序【falldeaf】处理。这是可以定制按钮动作的地方，包括使用按钮来导航菜单系统，以使用户能够为每个按钮选择不止一个功能。它增加了高度的灵活性，所以你可以创建各种宏来控制你的整个家庭自动化系统从你的按钮剪辑。

很高兴看到一个项目考虑了人机工程学和可用性，而不仅仅是创建基本功能。遵循这一思路，你会发现自己在最初构建之后的使用阶段很享受你的项目。另一个很好的例子是这个[自充电电加热外套](https://hackaday.com/2015/02/03/self-charging-jacket-eliminates-forgetfulness/)。休息后的视频。
