# 非常有用的自制砂带研磨机

> 原文：<https://hackaday.com/2016/06/20/an-extremely-useful-shop-built-belt-grinder/>

什么东西是绿色和黑色并用在整个商店里？这是[Make It Extreme]的最新产品，[一款临时打造的砂带研磨机](https://youtu.be/T4b3KOHlrv8)。像往常一样，建造视频让我们有心情切割金属。

这里我们要大胆地说，车床，而不是砂带磨床，才是*最重要的金属加工工具。这一点从这款车中可以看得很清楚——传动装置完全是在车床上加工的。但是，正如车床是机械制造的核心一样，像这样的砂带磨床在商店的实用性方面也必须有很高的评价。*

你可以用它们磨利，快速去除原料，清理焊缝，形成倒角，去除铁锈和腐蚀。它们是非常棒的全能工具，并且具有快速释放惰轮功能，可以快速更换皮带以适应不同的工作，使其更加灵活。我们希望看到工作台有更多的可调节性——相对于皮带调节工作台角度的能力非常方便——但总的来说，这是一个很好的构造，也是一个很好的工具。

最重要的是，观看[让它变得极致]的制作——就像这个[喷砂机](http://hackaday.com/2016/05/15/propane-tanks-transformed-into-extreme-sandblaster/)、[点焊机](http://hackaday.com/2016/04/24/diy-spot-welder-doesnt-look-like-it-will-immediately-kill-you/)或[砂带机](http://hackaday.com/2016/04/03/turning-an-angle-grinder-into-a-belt-sander/)——就像一堂高速车间课。有很多东西要学，尽管我们不得不承认穿着短裤和 t 恤焊接让我们心惊肉跳。
