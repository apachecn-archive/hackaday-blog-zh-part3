# 荧光显微镜满足 DIY 流体管理

> 原文:[https://hack aday . com/2017/06/03/fluorescence-microscopy-meets-DIY-fluid-management/](https://hackaday.com/2017/06/03/fluorescence-microscopy-meets-diy-fluid-management/)

荧光显微术是一种结合荧光或磷光(与反射和吸收相反)的光学技术，用于研究有机和无机物质的性质。对于将 DIY 技术带入实验室，菲利普并不陌生，他正在使用 3D 打印资源来推进科学，并让来自世界各地实验室的实习生们感到高兴。

在荧光显微镜中，减少可以从单个样品中收集的数据量的一个巨大限制因素是可以用荧光标签标记的目标的数量。然而，荧光团光谱发射的重叠限制了可以并排使用的荧光团。这意味着在典型的设置中，只有大约四个目标可以用荧光标签标记，如果仔细进行光谱分层，十个是绝对最大值。然而，在单个样本中，可能有数百个组件。显然，我们差了一个数量级(或更多)。

然而，研究人员很聪明。一种当前的解决方案是以连续的方式标记目标，其间有探针信号无效步骤。理想情况下，在不将样品移出显微镜的情况下，按顺序引入探针。成像后，可以移除探针，使得标记目标的数量只限于探针更换的次数。而且，通过巧妙的“条形码”方案，每一轮的回报甚至可以成指数增长，而不是线性增长。

但是，为了完成这一壮举，单个样品必须通过标记和剥离步骤反复处理。手动完成这项工作并不少见，包括许多小时异常繁琐的工作。这就是菲利普的用武之地。通过使用像 Cartesian robot 这样的 3D 打印机，[Philip]正在自动化贴标和剥离步骤，从而产生快乐的实习生，并最终产生更精确的产品。你可以在他的 GitHub repo 中找到所有[Philip 的]设计文件，然后花大约 1000 美元制作一个，而不是花几万美元买一台商用机器。准备好了吗？[我们支持你](https://hackaday.com/tag/microscopy/)。
