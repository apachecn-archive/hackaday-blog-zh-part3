# 阿尔法战士五号

> 原文：<https://hackaday.com/2018/03/13/up-alphagoer-five/>

AlphaGo 是深度学习程序，可以在围棋比赛中击败人类。你可以阅读谷歌在这方面的高度技术性的论文，但你必须涉水通过一些非常学术的语言。[阿曼·阿加瓦尔]帮了我们一个忙。他拿着[原纸，用浅显的英语解剖了 in 的重要部分](https://medium.com/@mngrwl/explained-simply-how-an-ai-program-mastered-the-ancient-game-of-go-62b8940a9080)。如果标题对你没有意义，你需要多读读 [XKCD](https://xkcd.com/1133/) 。

[Aman]说他的治疗将对那些不想成为神经网络专家但仍想了解这一重要突破的人有用。他还认为，母语不是英语的人可能会发现他的分析很有用。顺便说一下，AlphaGo 击败[Sedol]的实际围棋比赛是流式的，你可以在 YouTube 上观看所有的重播(第一场比赛出现在下面)。

有趣的是，这个解释并没有假设你知道如何下围棋，但它假设你对某种双人棋盘游戏有所了解。作为你会在原始论文中找到的语言类型的例子，你可能会看到:

> 使用随机梯度上升在随机采样的状态-动作对(s，a)上训练策略网络，以最大化在状态 s 中选择的人移动 a 的可能性

接下来是一些数学方程式。这篇文章解释了随机梯度上升，甚至将其与另一种反向传播技术随机梯度下降进行了对比。

我们不得不说，我们希望看到更多的学术论文像这样被拆开，给那些对此感兴趣但不是该领域专家的人。我们当时报道了 AlphaGo 的比赛。就我个人而言，我们总是喜欢好的[象棋电脑](https://hackaday.com/2016/05/07/chess-computers-improve-since-90s/)。
