# 天线旋转 Arduino 风格

> 原文：<https://hackaday.com/2016/08/31/antenna-rotation-arduino-style/>

回到你不为你的电视节目付费的日子，在屋顶上有一个八木天线是很常见的。如果你足够幸运，让该地区的所有电视台都指向同一个方向，你只需对准天线就可以了。如果你没有，你需要一个天线旋转器。如今，旋转器更多地出现在像业余无线电波束这样的通信天线上。地面使用时，天线只需左右摆动，不需要改变仰角。然而，它确实需要一个坚固的马达，因为风载荷会对系统施加很大的力。

[SP3TYF]有一个 HyGain AR-303 旋转器，并决定[为其构建一个基于 Arduino 的控制器](http://www.tyfek.republika.pl/Sterownik_rotora_arduino/sterownik_rotora_arduino.html)。成品有一个液晶显示器，能够驱动一个 24 V 的电机。你可以通过旋钮或电脑控制天线的方位角。

[Waldemar Lewandowski]构建了一个转子的变体(从[SQ9OUB]中获取了一些额外的想法)，并制作了一个设备运行的视频(见下文)。视频有点安静，但你会明白其中的意思，你可以看到最初的[SP3TYF]版本的代码和文档。

如果你想工作卫星，你需要一个额外的旋转轴。如果你仔细想想，旋转天线和[移动太阳能电池板](https://hackaday.com/2011/06/17/a-simple-diy-solar-tracker/)，可能有很多共同点——太阳也在太空中漂浮。
