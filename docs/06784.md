# Hackaday 奖参赛作品:检查员机器人的目标是看下面

> 原文：<https://hackaday.com/2017/08/17/hackaday-prize-entry-inspectorbot-aims-to-look-underneath/>

当你可以把一个机器人推进去的时候，为什么要费力地爬进那个狭小的下水道，然后在克苏鲁身上涂上一层呢？更不用说可能会被卡住了。[这就是 InspectorBot 做的事情](https://hackaday.io/project/25581-inspectorbot)。这是[Dennis]的 Hackaday 奖参赛作品，也是我们最佳产品竞赛的决赛选手。

检查员机器人是一个低调的漫游者，旨在检查下水道的黑暗角落，爬行空间和其他人类不适合或不想去的恶心地方。它配备了一台 Raspberry Pi 电脑，上面有一个高清摄像头和一个指向前方的普通网络摄像头，用于导航。它使用点对点 WiFi 进行通信，并通过一对 L293D 电机驱动器控制 rocks 全轮驱动。

这似乎是我们的沃土。管道爬行器、烟囱攀爬器、太空爬行器听起来都很有用，特别是与某种计算机视觉结合起来，即使操作员没有注意到问题，机器人也能注意到。现在，[丹尼斯]已经准备好了，目前大部分工作都集中在软件上。[两个摄像头现在都在工作](https://hackaday.io/project/25581-inspectorbot/log/63935-forward-pointing-and-upward-pointing-cameras-running-at-the-same-time)，允许检查员机器人同时向操作员发送前视和仰视视频。单单这一点，就已经是当前 Raspberry Pi 漫游者的一大进步，并为一个易于构建的平台增加了许多功能。
