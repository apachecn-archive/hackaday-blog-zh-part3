# 如何悬浮 100 磅

> 原文:[https://hackaday.com/2017/02/01/how-to-levitate-100lbs/](https://hackaday.com/2017/02/01/how-to-levitate-100lbs/)

我们的大多数读者已经熟悉了电磁体的工作原理——铁芯中感应出电流(通常由线圈产生),该电流与铁芯中的磁畴对齐。正常情况下，这些磁畴以不产生累积力的方式随机排列。但是，当线圈产生的电场使它们对齐时，就会产生一个净力，核心就变成了磁铁。

正如你所料，这是一个非常有用的概念，电磁体被用于从电动机到粒子加速器，再到 Beats by Dre 耳机的所有东西。你在高中物理课上可能熟悉的另一个用途是悬浮。当两块磁铁以相同的磁极朝向彼此时，它们相互排斥而不是相互吸引。同样的原理也适用于电磁体，所以一个物体可以用好的电悬浮起来。

然而，这并不是使用磁铁悬浮某物的唯一方法。如下图所示，[永磁体可以用来在导电材料](https://www.youtube.com/watch?v=pCON4zfMzjU)中感应出电流，进而产生磁场。永久磁铁仅仅通过移动来感应电流——在这种情况下，是在由电动机旋转的转子上。如果将导电材料放在磁铁下面(就像视频中一样)，它会向后推，你就有了悬浮。

虽然视频中显示的设置只是一个原型，旨在展示这一原理，但类似的系统也有可能用于磁悬浮列车。这个原型能够提升自己(超过 100 磅)，并且似乎能够容纳更多。还有很多其他的悬浮方法，有些比其他的更实用，但这个绝对是最突出的——这个视频很好地解释了它是如何工作的。
