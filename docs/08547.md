# 管道，三通和齿轮导致流畅的视频拍摄

> 原文:[https://hack aday . com/2018/02/13/pipes-tees-and-gears-result-in-smooth-video-shots/](https://hackaday.com/2018/02/13/pipes-tees-and-gears-result-in-smooth-video-shots/)

制作糟糕的视频容易得令人沮丧，但只需要一点点小心就能扭转局面。在充足的灯光和不错的音频之后——而不是纵向拍摄——也许最大的改进来自于在移动时稳定相机。毕竟，让你的观众晕车是一种不好的形式，为了平滑那些美丽的镜头，相机滑块可以是一个很大的帮助。

然而，并非所有的相机滑块都是一样的，我们必须承认，当第一次看到[Rulof Maker]制作的一个平滑、同步的平移和滑动相机架时，我们感到困惑。我们只是不知道这些齿轮将如何投入使用，但随着下面视频的进展，很明显这是一个可调节的受电弓装置，而[Rulof]的易贝齿轮旨在将两组受电弓臂连接在一起。这些臂由螺纹管和三通管件形成，轴承压入其中，这是一种非常聪明的建造技术，似乎高度依赖于幸运地找到与螺纹过盈配合的轴承。但是，[Rulof]仍然让它工作，用一点环氧树脂和相当多的欺骗，他最终得到了一个复杂的链接，产生了预期的效果。以及能够通过对摄像机支架枢轴点的微小调整来配置运动的加分。

几个月前，我们见过类似的[受电弓滑板](http://hackaday.com/2017/09/01/a-compact-portable-pantograph-camera-slider/)。那张是 3D 打印的，用同步带连接，但原理是一样的，两张照片看起来都很棒。
