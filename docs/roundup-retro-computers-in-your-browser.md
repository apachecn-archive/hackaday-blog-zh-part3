# 综述:浏览器中的复古电脑

> 原文：<https://hackaday.com/2015/09/28/roundup-retro-computers-in-your-browser/>

有两个因素阻止我收集更多的旧电脑:成本和存放它们所需的空间。我有我原来的老 TRS-80，和一个旧的串行终端(见下面的视频)。然而，我摆脱了我的数据通用硬件，我失去了我的老 1802 COSMAC Elf 在一些洪水。已经有一些不同保真度的复古计算机复制品，它们通常比原件更便宜、更小。我有一个复制的牛郎星，一个复制的精灵，和一个复制的金-1。

然而，很难证明实物或复制品的花费和成本是合理的。对于真正的大型机器来说，情况甚至更糟，其中一些需要特殊的电力或冷却，很难保持运行。当然，另一个选择是软件模拟。像 SIMH 和赫拉克勒斯这样的选项工作得很好，但它们并不总是图形化的，而且设置一台机器只是为了玩几个小时或向学生展示它在过去的好时光是如何完成的，这是一项很大的工作。

另一方面，有相当多的老式计算机模拟器可以在你的浏览器中运行。不需要安装软件，不需要设置硬件，而且很多都是图形化的。我知道很多人不想把他们的工作发布到“云服务”上，但是我愿意用我的 30 行 BASIC 程序来猜测你的号码。

*   PC/OSI——这实际上是两个不同的模拟器:一个用于 IBM PC(据我估计，它几乎不是一台旧电脑),另一个用于俄亥俄科学挑战者 1P。该网站有一些预配置的演示(包括 Windows 1.0)，您也可以配置自己的演示。或者你可能更愿意玩 [Wolfenstein](https://jamesfriend.com.au/pce-js/ibmpc-games/) 和其他一些游戏。
*   如果你不喜欢 Windows，你可能会喜欢尝试 Mac OS 7。
*   Amiga–Amiga 拥有许多领先于时代的功能，你可以在这里试试 Amiga 500。
*   TRS-80–我还有一台 model III，但你可以在你的浏览器中运行一台虚拟的。
*   COSMAC Elf–COSMAC Elf 是我个人拥有的第一台电脑，这个网站非常忠实地再现了这个系统，还有一些额外的功能。
*   KIM-1–KIM 是一款非常实用的单板计算机，它普及了 6502。这个浏览器内置版本给你一种早期业余爱好电脑的感觉。
*   你需要 Java，但是这个模拟器可以让你用可以说是开创了家用和业余电脑市场的电脑进行实验。
*   PDP/11–在 Linux 之前有 Unix。此页面允许您在虚拟 PDP/11 上运行第六版 Unix。第六版太老了，你可能需要知道 cd 命令当时还不存在(试试 chdir)。使用#键退格，使用@键删除一行(假设您正在使用电传打字机)。如果你厌倦了大写字母，请阅读网站上的 FAQ 链接。
*   威尔克斯博士的 EDSAC 是第一批可以实际运行的现代计算机之一。他在剑桥的团队开发了很多我们今天认为理所当然的技术。这是一台奇怪的编程机器，你可能想浏览一下基于 PC 的仿真器 T2 的手册，以了解更多关于如何为它编程的信息。
*   如果你认为 EDSAC 很难编程，试试这个(但是你需要一个支持 Java 的浏览器)。

我敢肯定还有其他人，但这足以让你开始，我敢肯定评论会拿起更多。如果你想了解加载一个网页和让一些真正的硬件工作之间的区别，你可以把 PDP/11 条目比作[带来真正的东西](http://hackaday.com/2015/04/08/restoring-a-vintage-pdp-1104-computer/)。还有一些网站，你可以连接到一些[真正的旧硬件](http://hackaday.com/2014/12/12/online-altair-8800-clone-lets-you-play-zork/)，这可能是两全其美。对于另一个十字架(一个现代模拟器和一个老式串行终端)，你可以在下面看到我试图从克林贡人手中拯救银河系的视频。
