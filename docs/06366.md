# 最佳参赛产品:带 RDuinoScope 的望远镜控制

> 原文:[https://hack aday . com/2017/07/03/best-product-entry-telescope-control-with-rduinoscope/](https://hackaday.com/2017/07/03/best-product-entry-telescope-control-with-rduinoscope/)

Hackaday 奖不仅仅是给硬件黑客数万美元。这也是为了资助下一批开源硬件*产品*。除了 Hackaday 奖(我们资助将改变世界的硬件的竞赛)，我们还将向最有可能成为产品的项目提供 3 万美元。这几乎就像我们在这里资助硬件初创公司。

[Dessislav Gouzgounov]想要为他的望远镜建造一个小硬件——GoTo。这个手持控制器可以让他使用软件将望远镜对准他正在检查的任何天体。

许多 GoTos 只是与笔记本电脑连接，但[Dessislav] [围绕 Arduino Due 和 240×400 触摸屏建立了一个独立的系统](https://hackaday.io/project/21024-rduinoscope-boiana)，在引擎盖下有 GPS、RTC 和蓝牙。它在两个半球上工作，包含 250 个天体的数据库，以不同的速度对天体、月球和太阳现象进行延时跟踪，并且可以与任何配备步进器的望远镜一起工作。

我们介绍了[Dessislav]以前版本的 RDuinoScope，但他用 2400 多行代码对该项目进行了相当大的改进，包括一个新的菜单系统，并添加了一个星图，显示望远镜当前指向的天空位置，以及其他改进。该项目是开源的，你可以[在【德西斯拉夫】的项目页面上了解更多关于它的信息](http://rduinoscope.byethost24.com/?i=1)，或者[在 GitHub 上查看他的代码](https://github.com/dEskoG/rDUINOScope)。
