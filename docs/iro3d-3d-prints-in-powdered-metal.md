# 粉末金属中的 Iro3d 3D 打印

> 原文：<https://hackaday.com/2018/01/15/iro3d-3d-prints-in-powdered-metal/>

用塑料甚至树脂印刷变得相当普遍。然而，金属印刷仍然属于舶来品。一家名为 [Iro3D](http://iro3d.com/) 的公司正致力于用一台钢制打印机来改变这种状况，你可以花大约 5000 美元在测试版中买到这种打印机。当你能以不到 200 美元的价格买到塑料打印机时，这似乎太贵了，但对于可以用真正的金属打印的东西来说，这绝对是便宜的。

当然，有一个条件。打印机不会马上产生一个固体金属物体。它所做的是用沙子和金属粉末准备一个坩埚。然后你把坩埚放在窑里，出来的就是最终产品。你可以看到[他们原型机](https://www.youtube.com/watch?v=4FkzLs7cLes)的视频评测，下面来自【3D 打印呆子】。该公司的宣传视频显示，一个部分出来的窑也在下面。

在某些方面，这种打印机不像传统的塑料打印机那样复杂。没有热端，风扇，或加热床需要担心。机头拿起一个装有材料的容器，用螺旋钻将其卸下。金属粉末有两种粒度——一种用于可视表面，另一种用于内部填充。同样有两个装沙子的容器。沙子就像支撑材料，支撑着物体直到它凝固。

根据 Iro3D 的网站，这种粉末每磅大约 5 美元，机器可以做 0.3 毫米的层。“灌注器”——我们假设它类似于热端——直径为 1 毫米。如果你想买的话，有一个条件。他们现在只在西雅图地区销售。

看着这台机器，它看起来并不像是很难用非常标准的 3D 打印设备来复制，特别是如果你已经有一个螺旋钻来挤出糊状物。我们确信魔鬼在细节中，但是我们打赌如果这流行起来，我们将会看到一些自制尝试。

我们以前曾感叹过[合理的金属打印机](https://hackaday.com/2016/05/19/its-time-for-direct-metal-3d-printing/)。这个唯一的问题是，如果你无论如何都必须有窑，它可能会很容易做到[失去解放军铸造](https://hackaday.com/2014/08/20/lost-pla-casting-with-a-little-help-from-your-microwave/)。
