# 简单的自制丝锥，用于给木头穿螺纹

> 原文：<https://hackaday.com/2017/07/24/simple-shop-made-taps-for-threading-wood/>

木材可以是多种项目的首选材料，但如果需要螺纹紧固件，它通常不适合金属或塑料。但是，只要有一点点聪明才智，你就可以制作自己的木丝锥，并切割出非常棒的螺纹。

让木材做木材不应该做的事情是马蒂亚斯·万德尔的事情。全世界的黑客都知道并使用他的[木制齿轮设计师](http://woodgears.ca/gear_cutting/template.html)为各种项目设计齿轮，从[音乐弹珠机](http://hackaday.com/2016/03/03/incredible-marble-music-machine/)到[木制安蒂基西拉机械装置](http://hackaday.com/2016/04/27/wooden-antikythera-mechanism-is-geared-for-greatness/)。几个世纪以来，木工一直在给木头穿螺纹，所以让木头穿上像样的螺纹并不是什么新鲜事。但是，以低廉的成本做到这一点，并使螺纹干净、坚固，一直是一件棘手的事情。休息后的视频展示了[马提亚斯]从普通螺纹杆甚至现成的方头螺钉上切下丝锥的方法。他用一个简单的夹具来固定坯料，这样就可以用角磨机切割出凹槽。水龙头在他测试的材料中工作良好，视频结尾的一点非正式压力测试表明螺纹的使用寿命很长。

Wood threads 并不适合每一个项目，但是知道自己可以做到这一点可能会为快速、简单的构建开辟道路。这是一个需要记住的好建议。
