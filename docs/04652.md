# 自主交付:你的冲动购买仍然是安全的

> 原文：<https://hackaday.com/2017/01/16/autonomous-delivery-your-impulse-buys-will-still-be-safe/>

前几天，我在 NPR 听了一个“一年回顾”节目，BBC 世界服务小组讨论了 2017 年的前景。一个预测是无人机递送包裹将在今年变得司空见惯，作为证据，评论员报道称亚马逊已经在英国进行了成功的测试。但他对这在美国是否可能表示怀疑，他说，“第一架飞越某人财产的无人机将被击落，货物将被没收。”

他似乎对自己的评论相当真诚，但我们姑且假定他只是在开玩笑以表明观点，而不是真的对枪支的局限性一无所知或对美国的枪支拥有者出言不逊。不管怎样，他提出了一个很好的观点:当自动包裹递送变得司空见惯时，谁会确保货物到达预定收件人手中？

根据 Beeb 的妙语和我之前关于自动交付的帖子的评论中的讨论语气，似乎主要的担忧是，如果没有人类在交付链中，包裹将会被盗。这种想法显然是，当没有人在回路中时，满载货物的机器人卡车车队或载有包裹的无人机更容易成为犯罪分子的目标。

![automatedfreight](img/49f3c48624449709cd05b9c92b10d061.png)真的吗？在人类驾驶的情况下，仅在美国，每年因货物失窃造成的损失估计就达 300 亿美元。机器人能差很多吗？

如果你深入研究货物运输的统计数据，你首先注意到的是大相径庭的损失记录。这似乎与报道方法有关——当司机被枪指着脸时，这是一起暴力抢劫；当一个码头工人拿着一个装着现金的信封去休息一段时间，而一批货物却被装上了错误的卡车，这更像是一个欺诈案件。随着每年数十亿美元的“销售额”，货物盗窃更像是一个有组织的企业，而不是企业家——想想 MS-13，三合会和拉科萨·诺斯特拉。这导致航运公司倾向于少报货物失窃，这也许反映了人类不希望因尖叫而死的特殊愿望。机器人不太在乎被打中膝盖。

随着越来越多的供应链自动化，越来越少的人会接近所有潜在的战利品，这只能是一件好事。人类远比机器人更容易腐败，机器人不会被诱惑去支持低薪的薪水，或者通过从传送带上偷走一个包裹来与经理算账。诚然，机器人包裹处理系统可以被编程为将货物转移到可疑的普通白色卡车上，但这仍然是人类的罪行，至少在我们的供应链中有人工智能之前是如此。

但是，所有这些来自供应链早期的盗窃行为可能并不是 BBC 评论员或 Hackaday 社区所关心的。这有点太离谱了，对我们的影响仅限于它推高了价格，因为制造商试图弥补他们的损失。对我们来说，真正重要的是发货人的仓库和我们的前门之间的最后几英里。自主交付会对此产生怎样的影响？

同样，我不得不认为机器人会做得更好。这句话出自一个人之口，他通常的 UPS 伙计是一个摇滚明星——这家伙真的从卡车跑到我的门口，轻轻地放下包裹，敲门，按门铃，是你见过的最友好的人之一。尽管如此，如果有人拿枪指着他的脸，让他下车，他还是会走。他不会为了保护一堆 UPS 已经投保的包裹而放弃自己的生命。“这是钥匙。祝你愉快！”

这是一个极端的例子来说明一个问题:在“外出递送”阶段，人类几乎没有采取什么措施来保护您的包裹，而人类的易错性更有可能增加被盗的风险。例如，UPS 或联邦快递的司机可能会在带着包裹上路时忘记锁车。机器人不会。如果有人试图拦截或劫持一辆自动送货车，我认为机器人司机更有可能遵循有效的紧急程序——锁上门，呼叫调度，通知执法部门——而不是突然陷入高度紧张状态并记不起 911 号码的可怜男孩或女孩。

那么在递送的最后阶段，你的包裹就在门口等着你下班回家呢？再说一次，我认为自动交付提供了很多优势。我们已经有了与碳基送货司机一起工作的[智能包裹箱](http://www.smartparcelbox.com/)(市场上有一些像 Pelipod 一样的智能包裹箱，如下所示)，让它们与送货车辆对话不会是一个很大的技巧。

一句话:没有什么能阻止某人得到你的东西，如果他们想要并且有机会得到，或者可以通过使用武力创造机会。但在我看来，很明显，当联邦快递(FedEx)的自动卡车车队穿梭于我们的社区，或者亚马逊(Amazon)的无人机为我们当天送达的商品遮暗天空时，我们可能就没那么担心了。这不会阻止盗贼的尝试，但会让他们有更少的机会摆脱人类。
