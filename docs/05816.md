# 铜箔制造音乐——只需一点点帮助

> 原文：<https://hackaday.com/2017/05/02/copper-foil-makes-music-with-a-little-help/>

工艺品商店通常是古怪灵感的来源。在彩色玻璃部分，我们看到了铜箔，甚至用它来制作鞋盒顶部的一些射频电路。然而，我们永远也找不到一种好的方法来将 IC 连接到相对较厚的箔片上。[布莱恩·塞拉]还是做到了。他的 [paperSynth](http://www.instructables.com/id/PaperSynth-an-8-bit-Synthesizer-Made-Out-of-Paper-/) 使用一些纸和纸板作为基底，铜箔和一个 ATtiny CPU 来制作音乐。你可以在下面的视频中看到这个设备在工作。

铜箔是粘性的，背面不导电，所以在铜箔接触的任何地方，你都需要一滴焊料。我们不相信绝缘材料本身能够跨越电线，但如果在它们之间夹上一点绝缘材料，比如一张纸或一条绝缘胶带，你就可能跨越而不受惩罚。对于 RF 电路，你甚至可以制作这样的低值电容。

即使我们以前使用过铝箔，我们从阅读这个项目中得到了很多。电池夹和开关很巧妙。合成器有五种不同的模式。单音符、双音符、三和弦、四音符和鼓模式。

如果你想要一些声音上更令人印象深刻的东西，就像我们最近看到的这种[基于青少年的聚合成器](https://hackaday.com/2017/04/29/teensy-and-3d-printer-make-beautiful-music-together/)，你可能必须改变不同的构建技术。对于一个快速的项目，使用铜带和纸板可以让你进行得很快，并且仍然有一个整洁的外观。我们喜欢它。如果你认为磁带是低技术含量的，顺便说一句，[再想想](https://hackaday.com/2016/01/04/the-unreasonable-effectiveness-of-adhesive-tape/)。
