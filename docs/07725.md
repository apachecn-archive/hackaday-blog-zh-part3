# 机器人拖车伙伴车轮内部的华丽工程

> 原文:[https://hacka day . com/2017/11/19/华丽-工程-机器人拖车车轮内-巴迪/](https://hackaday.com/2017/11/19/gorgeous-engineering-inside-wheels-of-robotic-trail-buddy/)

总的来说，机器人是伟大的，而[泰勒]目前正在研究一种有点不寻常的东西:[一种 3D 打印的探索机器人，可以自主沿着户外小径前进](http://reboot.love/t/rover-a-robot-you-can-make-at-home/94)，名为 Rover。漫游者仍在开发中，[泰勒]最近完成了驱动系统和车身设计，所有这些都通过 OnShape 共享。

Rover 在每个车轮中都嵌入了 3D 打印的 4.3:1 减速行星齿轮箱，带有现成的轴承和无刷电机。一个树莓派坐在驾驶座上，目标是使用 NVIDA 的 TrailNet 框架的一个版本，用于无 GPS 的路径导航。因此，[泰勒]希望最终能有一个机器人“拖车伙伴”，可以用现成的组件和 3D 打印零件制造。

将电机和变速箱移入轮子本身使机器人的主体变得非常小，看到轮子在轮毂的对面*旋转有点奇怪。查看下面嵌入的展示车轮最新发展的视频。*
