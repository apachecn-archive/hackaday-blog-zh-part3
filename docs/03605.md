# 这是你期待已久的涡轮增压烧烤架

> 原文:[https://hack aday . com/2016/09/21/heres-the-turbo-turbo-BBQ-grill-you waiting-for/](https://hackaday.com/2016/09/21/heres-the-turbocharged-bbq-grill-youve-been-waiting-for/)

我们实际上并不确定这是一个好主意，但是它有很多风格；【摩根】的[烧烤架涡轮增压](http://morgan-wild.com/turbocharged/website/index.php)。字面上。

与汽车主题保持一致，福特野马的伺服电机驱动的节气门作为(自然吸气)进气口，本田思域歧管将它输送到格栅。但是当他真的需要加热的时候，一个 360 瓦的风扇可以强制加热。

然而，这是在 Hackaday 上的原因是风扇和油门都在埋在它下面的 ESP8266 的控制下。[摩根]甚至编写了一个网络应用程序，通过手机来控制这一切，包括预设荒谬的汽车名称或涡轮加速控制。查看以下视频中的功能。

如果是我们，我们会接下来解决烤架自动化的问题，尽管我们不确定如何保持烤架美妙的汽车美学。但是现在的汽车基本上都是机器人，对吧？为什么不给“ECU”增加一些[温度传感器](https://hackaday.com/2015/10/21/bluetooth-thermometer-minds-your-meats/)？之后，这是一个简单的工程问题[调整烧烤和烘烤温度以达到最佳效果。这是](https://hackaday.com/2014/11/26/an-engineers-guide-to-cooking-the-perfect-turkey/)[大众会做的](http://hackaday.com/2015/12/31/32c3-dieselgate-inside-the-vws-ecu/)！
