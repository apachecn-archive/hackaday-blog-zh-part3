# Hackaday 奖参赛作品:智能堆肥系统

> 原文:[https://hack aday . com/2017/08/14/hack aday-prize-entry-smart-堆肥-系统/](https://hackaday.com/2017/08/14/hackaday-prize-entry-smart-composting-system/)

堆肥在我们的社会中有一个重要的目的，重新利用我们的食物残渣和庭院垃圾来给花园施肥，而不是填满垃圾填埋场。知道大多数人不堆肥，[Darian Johnson]着手创建一个由 Arduino 控制的堆肥系统，使其尽可能简单。它会监控垃圾箱的湿度、温度和气体排放，以确保其得到适当的浇水和充气。

[Darian]的项目结合了一个检测可燃气体的 MQ4 气体传感器、一个土壤湿度传感器和一个温度和湿度探头。附近的水库由超声波传感器监控，跟踪水位；由 TIP120 触发的水泵打开水。同时，伺服控制的通风口保持空气流动恰到好处。

智能堆肥系统听起来对家庭园丁很有用；这是 2017 年 Hackaday 奖的最佳产品入围奖。
