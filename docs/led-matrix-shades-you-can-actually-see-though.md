# 你可以看到 LED 矩阵阴影

> 原文：<https://hackaday.com/2016/11/02/led-matrix-shades-you-can-actually-see-though/>

[盖尔·帕芙琳]承认喜欢偶尔的舞蹈音乐表演。对于那些从未去过的人来说，在这些展会上领先他人是一件真实而可怕的事情，以至于围绕它存在着一个完整的市场。为此，[ [Gal 制作了一套漂亮的 LED 眼镜。](http://e.pavlin.si/2016/08/14/led-shades/)

完成最终的设计花了不少功夫。所有的电路和发光二极管完全安装在一副来历不明的太阳镜框架上的透镜外壳内。电池挤在使用者的头部和太阳穴之间。

除了巧妙的包装，还有一系列同样令人印象深刻的功能。每个透镜是 69 个发光二极管的矩阵。它们有一个加速度计、一个麦克风和一个光传感器。甚至还有振动警示电机，我们感觉只是在炫耀。最重要的是，由于巧妙的布局和非常小的 led，你可以透过眼镜看到东西。

该设备需要一个[标签连接](http://hackaday.com/2014/04/27/a-small-replacement-for-large-programming-headers/) t 或焊接在引线上以进行编程。如果你想自己做一个，所有的文件都可以在[Gavin]的网站上找到。休息之后有一段视频。
