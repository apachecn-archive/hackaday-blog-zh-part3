# 机电一体化猫耳朵给我们剩下的人

> 原文：<https://hackaday.com/2016/02/26/mechatronic-cat-ears-for-the-rest-of-us/>

时不时你会看到一个让你会心一笑的项目。它可能不会带来世界和平或让饥饿的人吃饱饭，但当它在早上在你的浏览器中打开时，你会因为这种体验而更开心地去工作。

Radomir Dopieralski 的一套可穿戴的机电猫耳朵就是这样一个项目。随着你的动作而移动的角色扮演配件。非常[卡哇伊](http://www.oxforddictionaries.com/definition/english/kawaii)，但是好玩。

你可能已经见过商用的 [Necomimi](http://www.necomimi.com/) 脑波激活的机电耳朵。[Radomir 的]版本没有分享他们的复杂性，相反，他使用加速度计来检测头部运动，并与 Arduino Pro Mini 耦合，驱动一组伺服系统来操纵耳朵。他提供了源代码，并计划在自己的 PCB 上使用 ATtiny85 开发一个小型化版本。

除了有趣可爱之外，Radomir 还必须注意一些适用于任何头戴式可穿戴计算机的考虑因素。不仅仅是把 Pro Mini 和它的电池放在一个比他的头顶更不显眼和不受天气影响的地方的问题。他还发现，他使用的微型伺服系统没有足够的运动范围来完全弯曲耳朵，他可能会在未来版本的更大伺服系统中解决这个问题。他还没有解决一个特别棘手的问题:安装在你头上的一对伺服系统可能会相当吵。

这些年来，我们已经报道了相当多的角色扮演故事。这甚至不是我们[第一个猫耳](http://hackaday.com/2012/02/21/live-out-your-inner-animist-with-animatronic-cat-ears/)的故事。[比](http://hackaday.com/2015/11/24/pip-boys-as-a-service/)[多](http://hackaday.com/2015/06/20/the-first-pipboy-well-see-this-year/)一个的例子[皮普男孩](http://hackaday.com/2015/11/16/the-most-drool-worthy-pip-boy-to-date-can-be-yours/)，一个[哈尔 9000 服装](http://hackaday.com/2014/11/23/cosplaying-as-hal-9000/)，和一个制作精美的[惠特利木偶](http://hackaday.com/2011/08/11/portal-puppet-probably-wont-kill-us/)已经做出了这些页面，仅举几例。因此，向下滚动，欣赏[rado mir]的耳朵在行动的视频演示。
