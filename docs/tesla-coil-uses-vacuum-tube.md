# 特斯拉线圈使用真空管

> 原文：<https://hackaday.com/2018/04/20/tesla-coil-uses-vacuum-tube/>

当你在当地的电子商店发现一个 5 kW 的发射管时，你会怎么做？如果你是【TannerTech】，你造一个[真空管特斯拉线圈](https://www.youtube.com/watch?v=oV1f7MAX2n0)。这不是通常的小懦弱线圈，而是一个大坏男孩，会在一个老恐怖电影看着家里。

第一次通电有点虎头蛇尾，虽然它在工作，但除了灯管明亮发光之外，它并不十分壮观。一些调整和一些矿物油做的把戏。

老实说，建设细节有点稀疏，但如果你有任何业务建设这样大的线圈应该足够了。此外，你不太可能有和他完全一样的组件，所以你必须随机应变。无意冒犯，但这可能不是你的第一个高压项目。

这不是我们看到的第一个真空管线圈设计。事实上，最近的一个有[一个聪明的浪涌限制器电路](https://hackaday.com/2017/09/14/tesla-coil-uses-vintage-tube/)可能也适用于这个。电子管很擅长处理像这样的大量电力，而不会很难使用，而且我们总是喜欢灯丝发光。

如果你想从小事做起，试着用一个灭虫器。如果你懒得自己上弦，总会有一些[自动帮助](https://hackaday.com/2017/08/06/make-a-tesla-coil-winding-rig-with-knex/)。
