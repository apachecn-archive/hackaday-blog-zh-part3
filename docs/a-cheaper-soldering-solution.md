# Hackaday 奖参赛作品:更便宜的焊接解决方案

> 原文：<https://hackaday.com/2016/05/07/a-cheaper-soldering-solution/>

每个人在探索电子的过程中都要经历几个阶段，而这一点在烙铁的选择上表现得最为明显。My First Soldering Iron 是一种直接插入墙壁的烙铁，没有温度控制功能。它们很便宜，而且电子产品并不适合每一个人，这给了戒烟者把烧柴作为爱好的机会。下一步是温控熨斗，可能是奥悦或哈科。最好的烙铁？你看的是 Metcal 或者 Weller，你的钱包会变轻几百块。

不过，你的“我的第一个烙铁”并不可怕。为了他的 Hackaday 奖项目，[惠普]正在研究一种便宜、精确的烙铁，并且使用非常好的 Weller RT 尖端。不，它不如 Metcal 或适当的 Weller 好，但对于一些精细的焊接工作来说已经足够好了，会让 Aoyues 和 Hakkos 的钱跑起来。

如果价格是衡量烙铁质量的合理标准，那么使用这些更好的烙铁头的烙铁是最好的烙铁。不过，这些建议相当便宜:大约 30 美元，你就能得到一个加热器和热敏电阻，其他就没什么了。人们对这款熨斗进行了无数次逆向工程设计([[1](http://dangerousprototypes.com/2012/02/27/weller-rt-type-tip-teardown-and-reverse-engineering/)和[[2](http://hackaday.com/2015/05/11/hacking-amazing-soldering-features-into-the-already-great-weller-wmrp/))，甚至还有一些基于 Arduino 的电路复制了 Weller 基本单元的功能。

[惠普]正朝着不同的方向加热这些烙铁头。他没有建造一个大盒子来装电子产品，而是把所有东西都装进了烙铁的手柄里。凭借 ATMega168 捐赠的大脑、几个运算放大器、MOSFETS 和一个电源插孔，[惠普]可以在一个紧凑的手持装置中加热这个烙铁头。
