# 从 Craigslist 恢复一个奇怪的铣床

> 原文:[https://hack aday . com/2017/06/01/restoring-a-strange-milling-machine-from-craigslist/](https://hackaday.com/2017/06/01/restoring-a-strange-milling-machine-from-craigslist/)

【diyVT】在 Craigslist 的[这台铣床上发现了一个真正的白象。这只花了他 200 美元，对一个小工厂来说很便宜，所以值得一赌。我们不知道该怎么称呼它——它不完全是龙门磨，也不是卧式磨，更不是膝磨。标签上写着 V-Mill，由 Pierce West 或 Tree Tool and Die Works 制造，这取决于您阅读的 ID 标牌。这棵树有一个三相电机，但它带有一个相位转换器，所以它应该可以用单相 220 伏家用电源运行。](https://www.youtube.com/watch?v=htS9ylr0MqU)

这台机器的物理状态很好，至少在前主人试图把它搬出车库之前是这样。在移动过程中，一个铸铁链传动手轮支架断裂成三段。铸铁焊接起来一点也不好玩。必须预热，用镍棒焊接，慢慢冷却。一些黑客可能会放弃或者制造一个新的部件，但是 diyVT 接受了这个挑战。他把拼图拼在一起，用角磨机把它们磨出凹槽，然后把所有东西焊接起来。结果并不漂亮，但它只需要手轮的力量和 200 磅的大猩猩旋转它。

在对马达和头部做了一些工作后，包括一条新的皮带，这棵树就可以砍了。[diyVT]溜出家庭酒吧，在新机器上切下他的第一批芯片。
