# 做你自己的谷歌地图

> 原文:[https://hackaday.com/2016/12/13/be-your-own-google-mapper/](https://hackaday.com/2016/12/13/be-your-own-google-mapper/)

谷歌地图是世界现代奇迹之一。很难记起过去获得高质量的航空图像有多贵。当然，你不能选择它们何时飞过地球的某个特定区域。如果你像丹尼斯·鲍德温那样，那还不够好。他一直在用他的无人机记录一所高中体育馆的建设。

[Dennis]使用开源 GDAL 工具从无人机图像创建谷歌地图切片。更好的是，他在下面的视频中记录了这个过程。一旦你可以制作自己的地图切片，你就可以控制拍摄图像的时间——如果你像(丹尼斯)一样记录建筑，这一点很重要。

[Dennis]知道有更多的自动化方法来进行映射，但他想了解整个过程。他还计划使用[开放无人机地图](http://hackaday.com/2016/12/11/make-use-of-your-drone-video-with-webodm/)彻底重做工作流程。如果你想看到一些非常昂贵(非开源)的测绘软件在工作，你可能想了解一下[这个测绘无人机](https://hackaday.com/2015/05/28/hackaday-prize-entry-a-3d-mapping-drone/)。
