# 可爱的火柴盒机器人

> 原文：<https://hackaday.com/2016/02/19/adorable-matchbox-robot/>

[wattnignies]一直在玩火柴，反正它们是装在盒子里的。一天，他让 synapses 在无人监督的情况下启动，并想知道他是否能在火柴盒里制造一个机器人。他的第一个原型是一个硬币锂电池和从那些你可以买到的 3 美元的伺服马达。它跑来跑去还不错，但它立即耗尽了电池，有点无聊。

接下来，他蚀刻了一块木板。它有一个小 [PIC micro](http://www.wattnotions.com/matchbox-car-v1/) ，一个迷你 LiPo 的连接器，和一个 H 桥。它启动得很好，尽管它耗尽电池的速度太快，但至少它不再是无脑的了。根据我们的经验，机器人往往会丢弃它们收集的所有有用数据，所以失明并不是太大的问题。

受到鼓舞和鼓励，突触光荣地没有被吓倒，[wattnignies]开始制作第二版。这一次，他从 OSHPark 订购了一块电路板，在 SketchUp 中制作了一个 3D 模型，然后开始将自己锁在自己的芯片之外。没有一个[高压编程器](http://high voltage programming)他就倒霉了。不幸的是，开发被搁置了。

在他进行一次小型机器人冒险时，和[wattn ideas]一起阅读很有趣。我们希望他能完成第三版，让一群小家伙到处跑。
