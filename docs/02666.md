# Hackaday 奖参赛作品:电子可视口琴

> 原文：<https://hackaday.com/2016/06/16/hackaday-prize-entry-electronic-visual-harmonicas/>

[sholnkin]的任务是教一个幼儿园班如何演奏乐器。不，不是那些便宜的塑料录音机。[shlonkin]正在教幼儿园的孩子们如何演奏唯一一种既能吹又能吸的乐器:口琴。

不像教室里有塑料录音机的孩子们，那里的指法要么对要么不对，[shlonkin]需要教孩子们把嘴放在正确的孔上，然后吸气或吹气来发出音符。教室里有一张展示口琴音符的海报，但他们需要更好的东西。[shlonkin]设想了一个以不同颜色发光的大型照明标志，并可以用扬声器播放显示的音符。

该项目的高级设计包括一个 Teensy 3.2，带有驱动小型音频放大器的音频适配器分线点。Teensy 还控制安装在木箱内的一束 led。这些 led 的布局出乎意料地好，很难找到背光面板[被均匀地照亮](https://hackaday.io/project/9657-classroom-music-teaching-aid/log/38417-soldering-lots-of-tiny-leds)。

作为一种课堂音乐教学辅助工具，这种设备已经存在了几十年——在世界各地学校乐队房间的深处，你可以找到旧的 Wurlitzer 钢琴，其设备与这个简单的设备没有太大区别。这是一种当时有效的教学方法，现在也应该有效。
