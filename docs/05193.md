# ESP32 教学课程

> 原文：<https://hackaday.com/2017/03/01/esp32-tutorials/>

ESP8266 已经成为每个人都知道的无处不在的部件之一。然而，新的 ESP32 也有许多伟大的新功能。如果你想带着 ESP32 兜风，你应该看看关于这款设备的视频系列。当我们说系列，我们不是在开玩笑。最后一次统计，有 19 个视频。有些只有几分钟长，但有些长达近 20 分钟，平均长度介于两者之间。

主题包括设置工具和使用 Eclipse 和 GDB。还有关于特定任务的教程，如 PWM、模拟转换、实时操作系统等。

您可以将所有视频[作为一个播放列表](https://www.youtube.com/watch?v=9zxU9uAifMI&list=PLB-czhEQLJbWMOl7Ew4QW1LpoUUE7QMOo)来查看，尽管这样做时顺序似乎不对。[Neil]也有一本 [ESP32 的书](https://leanpub.com/kolban-ESP32)出版，你可以免费下载或者支付少量费用购买。我们没有看过，但我们确实喜欢这些视频。【编者按:我们用了他的 ESP8266 这本书入门。太棒了。]

如果你更喜欢基于文本和简短的东西，我们有自己的 [ESP32 入门](https://hackaday.com/2016/10/04/how-to-get-started-with-the-esp32/)帖子。如果你喜欢这类东西，甚至还有一个[隐藏的 BASIC](https://hackaday.com/2016/10/27/basic-interpreter-hidden-in-esp32-silicon/) 解释器。
