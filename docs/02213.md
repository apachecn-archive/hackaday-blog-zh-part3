# 柯林·福尔泽飞越危险的天空

> 原文:[https://hack aday . com/2016/04/28/Colin-furze-flys-the-dangerous-skies/](https://hackaday.com/2016/04/28/colin-furze-flies-the-dangerous-skies/)

引用我们线人的话:"弗兹是我的英雄……你只需要知道如何焊接，完全不用考虑你的个人幸福。"我们不确定他有没有考虑过，但[柯林·福尔泽]肯定实施了一些危险的黑客行为。这次？双引擎气垫自行车。我们没必要让你看视频吧？

一旦你被这个建筑的大胆所震惊，也可以随意浏览他的“制作”视频。你将了解到气垫自行车“几乎”没有足够的升力让他离开地面，没有多余的重量来做转向之类的事情。所以他通过转移体重来“控制”自行车。

你还会在视频中看到他把自行车拴在秋千上，由一个朋友扶着，让他学着一点点驾驶。所以这不是完全不顾后果。只是大部分。你还会不止一次看到风扇叶片坏掉，他失控旋转。所有学习过程的一部分。我们很高兴他的脚没有被风扇叶片伤到。

我们喜欢他的疯狂视频，即使我们有时会放弃去耦电容。(开玩笑:去耦电容很重要。)但是请注意，即使你是一个著名的优步，如果你[玩火](http://hackaday.com/2014/05/30/furze-sets-fire-to-everything-with-pyro-gloves/)，你将[有时会烧伤](http://hackaday.com/2016/03/05/even-colin-furze-gets-burned/)。

谢谢你的提示。
