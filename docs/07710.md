# 家酿 SNES 迷你旨在历史准确性

> 原文:[https://hack aday . com/2017/11/15/home brew-snes-mini-aims-for-historical-accuracy/](https://hackaday.com/2017/11/15/homebrew-snes-mini-aims-for-historical-accuracy/)

当“正常人”在沃尔玛的过道里战斗，抢购任天堂不时推出的官方“经典迷你”游戏机时，黑客们一直在完善他们自己的经典游戏系统的迷你版本。“经典迷你”系列无疑是利用怀旧人群的一种非常酷的方式，这些人现在发现自己已经到了有可支配收入的年龄，但价值主张有点弱。这些家酿游戏机可以从跨越多代和制造商的系统中播放成千上万的 rom，而不是局限于任天堂打包到官方产品中的少数几代限制游戏。

但是对于那些年纪足够大的人来说，当他们第一次在这些系统中玩的时候，这些现代的化身总是缺少某些东西。T4 总觉得不太对劲。[这种隐约的不舒服感正是[ElBartoME]想要通过他非常光滑的微型 SNES 建筑消除的。他的 3D 打印外壳不仅符合原始(PAL)控制台的美学，而且该系统除了 NFC“墨盒”外，还使用真正的 SNES 控制器来加载不同的 rom。](https://www.thingiverse.com/thing:2492366)

Thingiverse 上的项目页面有所有的接线图和内核配置信息，以使内部的 Raspberry Pi 3 通过 GPIO 引脚读取原始的 SNES 控制器。他还对支持 NFC 的卡盒与 EmulationStation 配合工作以在插入时启动适当的游戏所需的硬件和软件进行了全面概述。尽管他承认这比控制器的设置要复杂得多。

[ElBartoME]在 YouTube 上放了一段视频，显示他插入模拟墨盒，并用一个原装 SNES 控制器浏览菜单。如果不是因为这个控制台只有智能手机那么大，屏幕上的显示超出了 SNES 几代人的能力，你会认为他在玩真的。

我们已经看到了一些令人难以置信的基于 Raspberry Pi 的仿真盒，以及[试图采用原始硬件组件的构建，但这个特定的项目可能代表了两个世界的最佳。](https://hackaday.com/2011/03/28/nintendo-hacked-to-fit-inside-an-nes-cartridge/)
