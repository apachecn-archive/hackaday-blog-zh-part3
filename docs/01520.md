# 由 Pi 零点控制的四轴飞行器

> 原文:[https://hack aday . com/2016/02/16/a-quad copter-controlled-by-a-pi-zero/](https://hackaday.com/2016/02/16/a-quadcopter-controlled-by-a-pi-zero/)

四轴飞行器和其他无人机的飞行控制器是令人难以置信的工程作品。这些板不仅可以保持飞机水平，还可以保持无人机在一个地方，或者读取 GPS 传感器并从一个航点飞到另一个航点。这些最新的飞行控制器[是建立在每个人最喜欢的 5 美元电脑上的，树莓 Pi Zero](http://erlerobotics.com/docs/Artificial_Brains_and_Autopilots/Autopilot_shields/PXFmini/Intro.html) 。

PXFmini 控制器和自动驾驶盾是乐儿机器人公司的最新项目，该项目在 Pi、气压计和 IMU 传感器、电源和所有适配器上放置了 8 个伺服输出，以将 Raspberry Pi Zero 变成一个有能力的飞行控制器。由于 Pi Zero 在保持四轴飞行器水平后会有一些计算能力，因此有可能出现一些非常酷的外设。乐儿机器人公司[已经在许多无人机上使用深度相机](http://erlerobotics.com/docs/Artificial_Brains_and_Autopilots/Additional_Sensors/Xtion_Depth_Sensor.html)和[激光雷达](http://erlerobotics.com/docs/Artificial_Brains_and_Autopilots/Additional_Sensors/Lidar.html)。这使得一些有趣的应用我们现在只能想象。

PXFmini 的原理图在 RC 和无人机社区的最佳传统中是开源的，[将很快可用](http://erlerobotics.com/docs/Artificial_Brains_and_Autopilots/Autopilot_shields/PXFmini/Schematics.html)。你可以看看 FXPmini 在下面的办公室里飞行的视频。
