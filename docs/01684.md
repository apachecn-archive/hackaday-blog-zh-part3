# 微型开源机器人

> 原文:[https://hackaday.com/2016/03/07/tiny-open-source-robot/](https://hackaday.com/2016/03/07/tiny-open-source-robot/)

我们看了这个[小型开源机器人](http://www.wirebeings.com/)的视频介绍，虽然我们不能 100%确定我们希望在睡觉时有一双发光的小眼睛看着我们，但它似乎确实是一个不错的黑客小平台。这个机器人是[马修]的一个兼职项目，他正在攻读信息科学学位。

这个机器人有一个小小的可驱动的抓臂，用来在前面拿手机。当它不拿着手机的时候，它可以用它的两个小小的超声波传感器四处跑而不会撞到东西。我们喜欢机器人上使用的被动平衡。机器人没有复杂的自平衡装置，而是使用小脚轮来提供其他的接触点。

机器人的头部有足够的空间来放置你喜欢的任何口味的 Arduino。你只需要几个小时的 3D 打印和一些维生素就可以让一个小机器人潜伏在你的房间里。休息后的视频。
