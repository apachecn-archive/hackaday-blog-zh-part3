# 快速简易太阳能热气球

> 原文:[https://hack aday . com/2017/10/30/快速简易太阳能热气球/](https://hackaday.com/2017/10/30/quick-and-easy-solar-hot-air-balloon/)

[贝基·斯特恩]喜欢利用太阳能。我们大多数人会立即想到太阳能电池和其他奇异的太阳能技术。但是[贝基]展示了如何只用胶带和垃圾袋制作热气球。

这个想法很简单。你用黑色垃圾袋做成一个大信封，然后充入空气。贝基就是这样做的，带着它跑，把它系好，然后用手吹一下。一旦太阳加热了黑袋子，它就会浮起来。

我们希望她能告诉我们是否有有效载荷的余量，但是她没有。然而，电子产品一直在变得越来越小，越来越轻。当然，电池很重，但也许有机会在气球上安装一些太阳能设备，以进一步利用太阳能。

如果你想试一试，[Becky]有一门免费的太阳能课，内容包括利用太阳能雕刻木头，以及利用太阳能电池板给电池充电或给 Arduino 供电。

我们怀疑这种太阳能齐柏林飞艇的性能会像丙烷动力气球一样，但它仍然是有趣且有教育意义的。看来你也不太可能带着它去[到达太空](https://hackaday.com/2015/01/07/global-space-balloon-challenge/)。
