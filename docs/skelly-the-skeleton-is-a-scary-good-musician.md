# 骷髅斯凯利是一个可怕的好音乐家

> 原文：<https://hackaday.com/2017/11/30/skelly-the-skeleton-is-a-scary-good-musician/>

[BoneConstructor]的 [Skelly 骷髅机器人项目](https://hackaday.io/project/27181-skelly-the-skeleton-robot)有很多让人喜欢的地方。请注意，我们说的是“项目”。这是因为不仅机器人工作良好，制造精良，而且他制造机器人的过程中包含了我们所有人都经历过的步骤。我们可以自信地说，因为这是他的第一次，我们都经历过。

Skelly 在当地车展上作为一具骨架坐在[BoneConstructor]的古董赛车里。它的眼睛亮了起来，发出呻吟的声音，这种声音并不总是正常的。从那里学到了头部和手臂伺服系统的教训，随后是 PS2 遥控器和控制板的问题。当他意识到他必须自己编写代码时，他因缺乏编程技能而陷入困境。但是后来他发现了 [Visuino](https://www.visuino.com/) ，正如你从名字中可以猜到的，这是一种可视化的 Arduinos 编程方式，主要由拖放组成。从那以后，这条路即使不完全是直线的，也更加平坦了。

他没有通过将骨骼直接安装到伺服臂上来快速烧穿伺服系统，而是将轴承安装到骨承窝中，通过这些轴承将四肢放在轴上，并使用连接到伺服臂的推杆来转动这些轴。难怪胳膊这么好用。他对手腕和脖子也采用了这种结实而有弹性的方法。他甚至让它的右脚能够随着音乐节拍轻敲。

并且从那里我们开始了解到他疯狂的一些方法。看看下面的视频，在他的 Hackaday.io 页面上，你会看到 Skelly 是如何随着音乐美妙地移动的。我们甚至过了一会儿才意识到他其实不是在弹钢琴。但最重要的是，我们喜欢他在 AC/DC 的《颤栗》(钢铁侠 2 版本)中的表现。鉴于这是第一个机器人，我们对这些机器人手臂的支撑能力印象深刻。

而你做了第一个骷髅机器人之后做什么？[BoneConstructor]已经开始着手第二个项目，骨骼机器狗 Boney，仍在进行中，但进展良好。如果你已经看过了，再看一遍，这次要多注意他是如何实现关节的。
