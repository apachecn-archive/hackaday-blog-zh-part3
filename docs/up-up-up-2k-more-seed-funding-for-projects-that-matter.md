# 涨，涨，涨:为重要的项目提供 2000 多万美元的种子资金

> 原文：<https://hackaday.com/2017/04/18/up-up-up-2k-more-seed-funding-for-projects-that-matter/>

启动一个项目通常意味着前期的部分投资。Hackaday 正在加大努力，为那些想要建立有意义的东西的人扫清障碍。2017 年[hack aday 奖](https://hackaday.io/prize)的种子资金很简单，输入你的设计计划，广泛分享它，这样很多人就会在项目页面上用“赞”来表达他们的钦佩。你可以从每个赞中获得一美元来帮助启动构建阶段。

今年像火箭一样开始了。上周，我们通过了 4000 美元的种子资金限额，尽管“设计你的概念”轮还有两周时间。我们把奖金总额提高到 6000 美元。这意味着有更多的机会。[现在输入您的项目](https://hackaday.io/project/add?light&tag=2017HackadayPrize)。如果你已经这样做了，润色你的演示文稿，并在社交媒体上展示给你的朋友。最高 200 美元，你将获得 1 美元，或者直到我们再次达到新的限额。不要拖延，是时候建立一些重要的东西了！
