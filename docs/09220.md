# 机器人模块挑战:建造机器人，赢得奖品

> 原文：<https://hackaday.com/2018/04/23/robotics-module-challenge-build-robot-win-prizes/>

全新的今天，我们将全力以赴迎接[机器人模块挑战！](https://hackaday.io/prize/details#two)这是 [2018 黑客日奖](https://hackaday.io/prize)的最新部分，它只有六周的历史，我们已经看到了近 600 个令人难以置信的参赛作品。但是新的挑战意味着一个新的开始，也是你开始参赛的最佳时机。

这是你的呼吁，建立一个模块，可用于机器人项目在世界各地。20 个模块设计将获得 1，000 美元的奖金，并有机会在[获得包括 50，000 美元大奖在内的五个最高奖项](https://hackaday.io/prize#section-prize)！

机器人是电子世界的厨房水槽。你必须处理电机，传感器，旋转激光雷达，计算机视觉，机电一体化，以及意想不到的 3D 打印零件摩擦系数的原型问题。机器人是你展示技能的地方，这是你向世界展示你所拥有的机会。

如果有更多的方法跳过难的部分，那不是很好吗？简而言之，这就是机器人模块的挑战。我们希望看到伟大的模块化开放硬件设计，可以被全世界的机器人专家使用。这可能是一个电机控制器、一个底盘或肢体设计系统、一个传感器网络方案、一个通信系统、数据收集和传输——基本上是任何与机器人相关的东西。制作一个原型，展示你的模块是如何使用的，并记录所有需要的信息，以便在其他机器人制作中整合和重复你的设计。

[现在就开始参赛吧](https://hackaday.io/project/add?light&tag=2018HackadayPrize)向我们展示你对开放硬件的看法。
