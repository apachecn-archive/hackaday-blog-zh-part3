# 改进用于 PCB 制造的廉价激光雕刻机

> 原文：<https://hackaday.com/2017/12/22/improving-cheap-laser-engravers-for-pcb-fabrication/>

几个月前，[Marco]从一家熟悉的中国经销商那里买到了一台便宜、便宜、廉价的激光雕刻机。这是一件简单的事情，由铝型材、二极管激光器和控制板组成，看起来像是五年前设计的 3D 打印机控制器。现在，[Marko] [正在为这位雕刻师](https://www.youtube.com/watch?v=ZzO9skEYqwY)做一些升级，他的 PCB 制作技能已经一飞冲天。

[Marko]拿起的激光雕刻机叫做 EleksMaker，幸运的是在 Thingiverse 上有相当多的升级。他发现了两个 3D 可打印部件，一个保持皮带平行于铝挤压件，另一个在皮带上提供可调的 x 轴松紧度。有了这两个模块的结合，[Marko]实际上有了一个很好的，平滑的运动平台，更精确，雕刻更好。

这些升级并不都是 3D 打印的；[Marko]还接触了一些 Trinamic TMC2130 步进电机驱动器。这些步进驱动器是 3D 打印和其他桌面 CNC 机器的新热点，看看示波器中的波形，很容易明白为什么。这些驱动器通过解释微步进产生一个完美的平滑波形，它们在运行中几乎是无声的。如果你想制作一个 CNC chiptune 播放器，这是很可怕的，但如果你想在一块覆铜板上平滑雕刻，这是很棒的。

自从我们几个月前[最后一次看这个项目](https://hackaday.com/2017/08/22/laser-etching-pcbs/)以来，这个项目已经取得了很大的进展，而且结果越来越好。[Marko]正在用成本不到 200 美元的激光雕刻机制作真正的印刷电路板，他已经投入的升级也没有增加多少。你可以在下面的视频里看看[Marko]的进步。
