# 具有单壳 PLA 母模的金属铸件

> 原文：<https://hackaday.com/2016/07/20/metal-casting-with-single-shelled-pla-masters/>

【3DTOPO】做了很多[金属铸造](https://www.youtube.com/watch?v=tA1znkGbnSQ)(视频链接，下面嵌入)。这是显而易见的完整和适当的安全装置，在 YouTube 上是罕见的。

他们有所有的设备来做正常的方式:工艺或数控出一个主，生产一个拖动和复制，使任何必要的核心，最后；倒模。这是一个漫长而乏味的过程。它的出错率很高，而且有分割线。

另一套方法是失传的。使用这些方法，母版是由泡沫或蜡之类的材料制成的。母模被耐火材料包围，然后熔化、燃烧或烘烤出模具。最后，金属被倒入。从理论上讲，一个完美的复制品无需打开模具就能制作出来。

他们最喜欢的方法是一种叫做失落的 [PLA 方法](http://hackaday.com/2013/05/10/update-lost-pla-metal-casting-the-movie/)的变种。在这种方法中，母版是在 PLA 中 3D 打印的。PLA 是乐于被一个模子烧出来或者烤出来的。然而，这样做确实提供了一个打破模式的机会，额外的步骤需要时间和燃料。

最近，他们发现只需打印一个外壳的模型，就可以将金属直接倒入模具中，无需燃烧步骤。对于大多数部件来说，没有足够的塑料来引起严重的排气事件，尽管他们建议穿戴全套适当的安全装备以防万一。休息后的视频。
