# JumpRoaCH 有点可爱，有点吓人

> 原文：<https://hackaday.com/2016/07/22/jumproach-is-kind-of-cute-kind-of-creepy/>

有一种理论认为对匆忙奔跑的恐惧是遗传的。同样，一个类似的理论也出现了，即人类倾向于觉得无助的东西可爱。毕竟，我们无用的婴儿在没有害虫的环境中表现最好。这可以解释为什么我们发现这种机器人蟑螂既有点可爱又有点令人毛骨悚然。

大学赞助的项目 JumpRoaCH 是南韩 SNU 生物机器人实验室和 T2 柏克莱仿生微系统实验室的合作项目。模仿昆虫已经成为[机器人研究](https://hackaday.com/2016/06/05/robot-moth-is-learning-to-fly-like-a-real-moth/)的一个流行途径，并且经常产生非常有趣的实验。

这个机器人看起来像一只正在经历青春期叛逆阶段的瓢虫。它依靠六个钩状的腿运行，这使得它可以穿越比轮子更广泛的表面，但代价是速度和更高的振动。机器人在测试台的表面上做了一个非常令人信服的动作，虽然摇摇晃晃。

它还有一个秘密攻击，就是位于腹部的单个 [Rockem Sockem 机械臂](http://hackaday.com/2010/08/03/robot-boxing-with-wrist-watches/)。随着有力的爆发，手臂可以将机器人发射到几英尺高的表面。如果机器人轮子着地，研究人员击掌。如果机器人背部着地，它可以用它的“翅膀”将自己翻个身。

[结果论文](http://robotics.eecs.berkeley.edu/~ronf/PAPERS/GJung-ICRA2016.pdf) (PDF 文件)对机器人及其巧妙的跳跃机制有很好的描述。至少如果这些开始像蟑螂一样繁殖，黑客们将永远不会为他们的项目缺少微型马达。休息后的视频。
