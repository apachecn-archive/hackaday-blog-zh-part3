# 你喜欢的混凝土桌子

> 原文：<https://hackaday.com/2016/05/11/concrete-table-swings-to-the-height-needed/>

你需要一张咖啡桌，一张餐桌。你真的需要两张桌子吗？【Shua】[认为答案是“没有”。](https://pumpingstationone.org/2016/05/balance-coffee-table/)这就是为什么他用混凝土和铝建造了这个摆动埋头钻桌子。

他首先制作了一个简单的一半大小的原型。然后是一个更大的。通过这些探索，他了解了桌子是如何制作的，需要什么样的重量，以及如何构造最稳定的桌面。

接下来，他在 Autodesk Revit 中设计了最终的表格。这是传统上用于建筑的软件。由于桌子是由混凝土制成的，Revit 的混凝土工具对这个项目很有用。

大部分建造过程都相当标准。然而，使用 CNC 的粉红色隔热材料作为混凝土的模具是很有趣的。泡沫是闭孔的，所以它工作良好，并给出了一个很好的完成。该组件以玻璃顶部和包含浪涌抑制器和两个插座的地毯底部完成。休息后，可以在视频中看到桌子在两个位置之间摆动。
