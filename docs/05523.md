# 老式电报发声器再次点击

> 原文：<https://hackaday.com/2017/04/06/vintage-telegraph-sounder-clicks-again/>

当你想到这一点时，你会感到悲哀:一位退休的铁路电报员，他可能曾经以每分钟 40 个词的速度敲打出代码，而一堆老式音响静静地在架子上积灰。[kthrace]决定对此做些什么，并建造了这个莫尔斯发送器让那些旧的发声器起死回生。

尽管莫尔斯看起来很古老，但这是一种生活技能，这位 92 岁的前铜管乐演奏者渴望再次练习。当然，有代码练习振荡器，但是 dits 和 dahs 不能代替曾经充满这个老家伙日子的电磁咔哒声。关于电路的信息不多，但看起来[kthrace]选择了一个 RedBoard 来从 SD 卡中读取莫尔斯，并驱动一些继电器来支持最多四个发声器；那会很吵的！表壳是定制的，很好地补充了 J.H. Bunnell 和 Co. sounder 的木材和黄铜，这些年来听起来仍然很棒。

在下面的视频中测试你的莫尔斯技能——从发声器中复制代码比从振荡器中复制代码要难得多。发现自己需要练习？[我们已经为你准备好了](https://hackaday.com/2015/09/24/arduino-teaches-morse-code/)。
