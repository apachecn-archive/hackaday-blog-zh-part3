# 平板相机不使用镜头

> 原文:[https://hackaday.com/2016/01/11/flat-camera-uses-no-lens/](https://hackaday.com/2016/01/11/flat-camera-uses-no-lens/)

早期的相机和现代相机的工作方式非常相似。透镜(或充当透镜的针孔)将图像聚焦到传感器上。当然，在这种情况下，传感器是一个广义的术语，可以包括一片薄膜，毕竟它通过化学反应来感知光线。当然，镜头和传感器变得更好，或者至少有所不同，但自从中国人在公元前 400 年左右建造了暗箱(希腊人在那之后不久)以来，基本设计一直保持不变。

当然，镜头/传感器的安排很好，但它确实限制了你可以把相机做得多薄。手机摄像头非常薄，但有些应用甚至会让它们太厚。这就是为什么莱斯大学的研究人员正在研究一种没有镜头的平板相机的新概念设计。你可以在下面看到一个关于新型相机的视频。

这个想法很简单:拿一个传统的传感器，在上面放一个有网格状排列孔的面具。最终的图像与你看到的不匹配，但它提供了足够的信息，计算机可以重建图像。

在 Hackaday，我们对[自制相机构建](http://hackaday.com/2015/08/20/hack-your-own-analog-camera/)并不陌生(包括围绕[一个 Arduino 和一个单像素](http://hackaday.com/2015/01/31/a-single-pixel-digital-camera-with-arduino/)构建的相机)。然而，这些都没有承诺是超级薄。
