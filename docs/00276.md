# 苹果酒压榨正赶上秋天

> 原文:[https://hack aday . com/2015/10/04/apple-cider-press-is-just-in-time-for-fall/](https://hackaday.com/2015/10/04/apple-cider-press-is-just-in-time-for-fall/)

你喜欢黑客吗？你喜欢苹果酒吗？你喜欢吊扇吗？如果你对这些问题中的任何一个回答“是”，那么男孩，我们有你的项目！[Lou Wozniak]有一个很棒的教程，教你如何使用吊扇马达建造一个苹果酒榨汁机，还有一些在你当地五金店可以买到的东西。

这个构建在概念上非常简单，但在执行上却非常复杂，而[Lou]在他的两个项目视频中非常出色地详细介绍了每一个步骤。该项目有两个主要组成部分:磨碎机将苹果切碎并制成多汁多汁的汤，压榨机提取果汁。研磨机由风扇电机驱动，而印刷机使用连接到电钻的螺旋驱动器，然后使用棘轮挤出最后一滴。

渴望更多的吊扇电机好吗？你很幸运！很明显[Lou]是改造风扇马达的大师，不久前我们展示了一个他制作的陶轮。
