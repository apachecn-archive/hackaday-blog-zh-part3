# 家酿真空管比你想象的要容易

> 原文：<https://hackaday.com/2016/05/04/home-brew-vacuum-tubes-are-easier-than-you-think/>

这一切都始于一台廉价的中国旋片真空泵和学习 DIY 真空管巫术的渴望。它以一连串的成功而告终——工作的真空室、灯泡、辉光管、二极管，最终[这个自制的功率三极管](http://simplifier.neocities.org/power.html)和[围绕它构建的音频放大器](http://simplifier.neocities.org/audio.html)。

[Simplifier]的工作室似乎是一个非常酷的地方。它一定有点 20 世纪早期的感觉，就像约翰·弗莱明早期研究真空管时使用的商店。玻璃制品、金属制品、电子产品——看起来[Simplifier]什么都有。忠于他的风格，一旦[Simplifier]有了一个便宜但有效的真空装置，他就从最简单的项目开始——白炽灯和气体放电灯。他对自己能够制造牢固的电气和物理连接并抽空电子管感到满意，于是转向二极管，并最终转向三极管。管子的质量令人印象深刻——杂散气体通过烘干炉和感应加热钛吸气剂去除。正如下面的视频所揭示的那样，性能相当稳定。

总体来说非常令人印象深刻，这不仅仅是因为他从零开始制造电子管的事实—[我们在](http://hackaday.com/2014/11/21/artisanal-vacuum-tubes-hackaday-shows-you-how/)之前已经看到了这一点。这里的亮点是，不需要专门的设备来制造工作可靠的电子管——只需要一个 MAPP 火炬、简单的手工工具和一个低端真空设备。任何人都可以——或许应该——尝试一下。
