# 自动弹奏的吉他

> 原文：<https://hackaday.com/2018/01/31/strumbot-the-guitar-that-strums-itself/>

[克莱尔]不是最喜欢音乐的人，但她可以乱弹吉他。多亏了一个 Arduino 的小小帮助，[她甚至不需要这么做](https://www.youtube.com/watch?v=BJIYJV4E6cM)。

她制作了扫弦机器人，负责处理弹奏吉他的手部工作。虽然[克莱尔]确实相信她的甲状腺机器人，但她不想在她的吉他上钻孔，所以热熔胶和双面胶带是当天的订单。

妓女机器人的商业终端是一个微型伺服系统。伺服移动两根筷子并拉动拨片穿过琴弦。令人惊讶的是，这个小小的伺服系统在让琴弦鸣响方面做得非常好。唯一的缺点是塑料齿轮在晃动时会发出噪音。

Strumbot 的用户界面是一个 3D 打印的外壳，有三个按钮和三个 led。每个按钮都会激活 Arduino 程序中不同的弹拨模式。led 指示当前活动的模式。一切都由 USB 电源组供电，使这成为一个独立的黑客。

[克莱尔]能够编码一些复杂的扫弦模式，但扫弦机器人仍然有一点限制，因为它只能容纳三种模式。对于她演唱的《也许给我打电话》来说已经足够好了，你可以在休息后的视频中看到。当然，这是一个简单的项目，[不像我们过去见过的一些](https://hackaday.com/2013/08/26/update-vladimirs-robot-guitar/)[机器人吉他模块](https://hackaday.com/2017/08/15/guitarbot-brings-together-art-and-engineering/)那么复杂。尽管如此，它只是一个有趣的晚上或周末项目的门票——特别是如果你正在向年轻的程序员介绍 Arduino。音乐、黑客和改装——你还能要求什么？
