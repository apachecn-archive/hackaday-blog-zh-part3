# Hackaday 奖参赛作品:一个开放的蓝牙开关接口

> 原文：<https://hackaday.com/2016/10/12/hackaday-prize-entry-an-open-bluetooth-switch-interface/>

上一届 Hackaday 奖挑战赛的主题是辅助技术，也许没有什么辅助技术比一种设备更迫切地需要来帮助那些不能使用普通输入设备的人。使用键盘、鼠标或触摸屏可能很难，但这个黑客奖项目将所有这些问题转化为一个简单的蓝牙开关。

BOSI——蓝牙开源开关接口——的核心是一个用 Solidworks 设计的 3D 打印外壳内的大蓝牙按钮。这些外壳内有一个与 Adafruit Bluefruit EZ-Key 相连的按钮。加上电池和充电电路，你就有了一个任何人都可以按的按钮，可以连接到任何设备，可以做任何事情。

像这样的系统的真正诀窍是软件栈，为此，BOSI 可以与 iOS 和 OS X 一起使用[开关控制接口](https://www.youtube.com/watch?v=JuLETwuxSbs)。Android 也能工作，而且整个设备对于任何不能使用普通输入设备的人来说都非常有用。Hackaday 奖的最佳参赛作品。
