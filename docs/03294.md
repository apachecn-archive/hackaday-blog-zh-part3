# 电弧打火机变成等离子烙画笔

> 原文:[https://hack aday . com/2016/08/18/arc-lighter-been-plasma-pyro graphy-pen/](https://hackaday.com/2016/08/18/arc-lighter-become-plasma-pyrography-pen/)

木材燃烧可以是一种非常引人注目的艺术形式，但谁愿意被一种老式的电阻加热元件烧焦呢？当然，你可以用激光雕刻，但那似乎太缺乏人情味了。那么，为什么不试试由花式打火机驱动的迷你等离子笔和喷灯呢？

电弧打火机是一种可充电的电子打火机，看起来像一个微型电击枪，而[NightHawkInLight]已经为它们提供了一些有趣的技术。在这种情况下，他将电极引线伸出，并将其安装在一个木柄上。火花间隙只有约 2 毫米，但产生的电弧足够热，足以相当精确地烧焦木头。不过，你必须快速工作，否则高电压会开始在 char 中寻找有趣的路径，产生 [Lichtenberg 图](http://hackaday.com/2015/07/04/high-voltage-wood-and-resin-result-in-fractal-art/)。如果你需要一个微型喷灯，那么[NightHawkInLight]也能满足你的需求——一个带有夹断式喷嘴的小铜管连接到一个水族箱泵上，为你提供压力。

除了烙画之外，这还有其他应用吗？也许焊接或脱焊？当然是对 ESD 不敏感的元件。
