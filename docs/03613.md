# 人造天窗给任何房间带来阳光

> 原文:[https://hack aday . com/2016/09/24/人造天窗给任何房间带来阳光/](https://hackaday.com/2016/09/24/artificial-skylight-brings-sunlight-to-any-room/)

人类不应该整天关在室内，但是谁想被紫外线、昆虫、过敏原、交通、体育活动和其他人打扰呢？另一方面，一个阴暗的生活空间通常会抑制生产力——如果不是让你很难找到你想要的东西的话。因此，如果你想照亮你家的任何一个房间，并且你有足够的现金和耐心等待它的广泛发布，那么 [CoeLux 就是一个不需要天空或阳光的天窗](http://www.thisiscolossal.com/2015/02/artificial-sunlight-system/)——并不是说你能分辨出其中的区别。

意大利开发商[CoeLux Srl]也许明智地对如何实现这一效果保持沉默，但证实了天窗中的纳米粒子模拟了大气波动的效果，将广阔的深蓝色压缩到几毫米，同时保持了无限深度的感觉。

下雨天走进室内，却发现阳光照耀着你，这会给任何空间增加一种扭曲现实的感觉——尤其是地下室车间或娱乐室；为此，设计者希望这将有助于地下“铲土机”的开发。目前，CoeLux 有三种照明设置——热带、地中海和北欧——但不难想象有一天它可以模拟地球上任何一点的任何天气。

如果与此同时，你被一个更传统的枝形吊灯困住了，[看看这个绞盘](http://hackaday.com/2013/05/15/winch-hack-lets-your-change-chandelier-light-bulbs-at-ground-level/)，它将简化更换 burt 灯泡的过程。如果你正在建造一个筒仓，并且需要进一步欺骗你的~~俘虏~~地窖居住者，试着[建造一个这样的电梯](http://hackaday.com/2016/08/20/a-windowless-elevator-with-a-360o-panoramic-view/)。
