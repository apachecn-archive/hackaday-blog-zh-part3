# 适合各种场合的 LED 效果

> 原文：<https://hackaday.com/2018/03/28/an-led-effect-for-every-occasion/>

高质量的软件开发例子可能很难获得。当然，在谷歌上找到一个包含所有正确关键词的`块是很容易的，但是正确地解释每一件事可能是偶然的。主题越小众，论坛帖子就越少。在标题为 [Arduino LED 条形效果的综合文章中，HansLuijten 提供了一套令人震惊的 LED 条形图案。](https://www.tweaking4all.com/hardware/arduino/adruino-led-strip-effects/)`

不要让朴实无华的标题将你从内容中引入歧途，因为所提供的内容超出了普通初学者如何设置 NeoPixels 的教程。[HansLuijten]彻底到了一个断层；提供了从简单的单一颜色渐变和经典的赛昂人眼睛到看起来像从天而降的[](https://www.tweaking4all.com/hardware/arduino/adruino-led-strip-effects/#LEDStripEffectMeteorRain)*流星的效果的例子。说真的！查看他们网页上的视频。你在剧院标志周围看到的那些追逐灯？检查。色彩闪烁和闪耀？检查。彩色湿巾和彩虹褪色？检查，检查。*
