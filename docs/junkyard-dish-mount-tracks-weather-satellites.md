# 垃圾场碟形天线跟踪气象卫星

> 原文：<https://hackaday.com/2017/01/02/junkyard-dish-mount-tracks-weather-satellites/>

现在在环绕地球的轨道上有一个宏伟的宇宙飞船星座，许多飞船将有用的数据清晰地传回地表，以供利用。问题是，这往往需要专门的设备，这可能是一个真正的预算克星。但是，有了一个库存充足的废物箱，一些战略性的易贝采购，再加上一点点努力，[一个动力驱动的方位角-仰角卫星天线座](http://imgur.com/a/6Lo8M)就变得可以负担得起了。

[devnulling]努力的目标卫星是 NOAA 的[极地轨道运行环境卫星](http://www.ospo.noaa.gov/Operations/POES/) (POES)，一个低地球轨道气象鸟系统。[devnulling]对直接接收来自卫星 L 波段下行链路的高清图像特别感兴趣。他在漫长的下载过程中想出的跟踪卫星的坐骑是垃圾场建造技术的杰作。

方位轴在雪佛兰的后轮轴承上旋转，仰角轴使用廉价的轴台，框架由废角铁和管材焊接而成。带有 15:1 齿轮头的 NEMA-23 步进器旋转方位角，而 36”线性致动器负责仰角。这种坐骑还需要经受风的考验；我们担心由碟形天线提供的帆面积可能会引起问题。这里希望坐骑像它看起来那样结实，我们将期待后续。

这对我们来说是可行的，但是在后院旋转的 4 英尺的盘子可能不是每个人对草坪附属物的口味。如果那是你，而你仍然想直接从数据源获取你的天气数据，试着使用[一个 SDR 加密狗和一大块电线](https://hackaday.com/2015/08/02/downloading-satellite-images-via-fm-radio/)。
