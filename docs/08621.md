# 冰壶运动的奇特物理学

> 原文:[https://hack aday . com/2018/02/20/the-strange-physics-of-curling/](https://hackaday.com/2018/02/20/the-strange-physics-of-curling/)

原来冰壶涉及到一些复杂的物理。《每天都更聪明》的[Destin]已经开始寻找为什么大西洋两岸的科学家在冰壶石为什么会卷曲的问题上存在分歧。

如果你一直在看奥运会，你可能看过冰上溜石游戏，这是一项苏格兰竞技运动。顾名思义，冰壶石不直。投掷者用一点旋转来推动它们，石头沿着它们旋转的方向弯曲。这与人们的预期完全相反——在光滑的桌子上用倒置的酒杯亲自尝试一下。玻璃将向旋转的相反方向卷曲。顺时针旋转将导致向左卷曲，逆时针旋转将导致向右卷曲。

当你想到其中的不对称摩擦时，这个杯子是有意义的。杯子正在减速，这意味着前缘的压力更大。旋转的前缘更用力地推动桌子，导致杯子向旋转的相反方向卷曲。

问题是冰壶石不会这样。瑞典乌普萨拉大学的科学家们认为这是由于划痕理论——石头的前缘在经过时划伤了冰，这给了石头的后缘一个力。

加拿大北不列颠哥伦比亚大学的马克·谢格尔斯基博士不同意这种观点。他的理论是不对称摩擦熔化在起作用。石头经过时会融化冰，形成一层薄薄的水膜，石头就骑在上面。

可能这两种理论都是正确的——而且两种力量都在起作用。困难的是将两者分离开来。两位科学家都做了一些实际实验，但不足以让科学界满意。我们的一个黑客能拿出明确的证据来解释为什么冰壶石会这样卷曲吗？请在评论中告诉我们。
