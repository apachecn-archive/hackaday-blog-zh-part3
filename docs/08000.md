# 自制红宝石激光器具有强大的冲击力

> 原文：<https://hackaday.com/2017/12/19/home-brew-ruby-laser-packs-a-wallop/>

在过去的半个世纪里，激光已经从昂贵的使用红宝石棒的物理实验发展到廉价的切割或雕刻工具，以及用来逗猫的玩具。物理学的进步使这一切成为可能，但事实证明，如果你能在不杀死自己的情况下玩红宝石激光器，玩红宝石激光器仍然很有趣。

通过一个看起来像《疯狂科学家》电影场景的设置，[styropyro]的高能激光器比我们看到的普通激光器更接近科幻小说中的射线枪，尽管很难携带。钻机的商业端是一个巨大的红宝石棒，坐落在一个盘绕的氙闪光灯内，依次包含在一个抛光的反射器内。灯的电源是巨大的——微波炉变压器，一个巨大的电压倍增器，和一组电容器，他说可以储存 20 千焦。当由 555 振荡器和一个旧汽车点火线圈发出的高压脉冲触发时，激光输出一个强大的光脉冲，[styropyro]用它来产生戏剧性的效果，包括破坏他自己的光学系统。我们希望听到更多关于电源设计的信息；由 PVC 管制成的科克罗夫特-沃尔顿倍增器需要一些探索。

无论细节如何，这个建筑都给人留下了深刻的印象，但是我们强烈要求一些简单的安全预防措施。也许看一看[【本·克拉斯诺】的 8 千焦红宝石激光器](https://hackaday.com/2014/05/23/ben-krasnow-and-his-ruby-laser/)会有帮助。
