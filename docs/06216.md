# 老年人用的外护腕

> 原文：<https://hackaday.com/2017/06/29/an-exoarm-for-the-elderly/>

近年来，假肢和辅助技术取得了长足的进步。当不仅有主要的医学研究组织，而且还有个人参与设计工具来改善他人的生活时？这很特别。在本栏位输入一篇自制文章: [ExoArm](https://hackaday.io/project/20663-assistive-exoskeleton-arm-exoarm) 。

ExoArm 设计师[ Kristjan Berce ]通过可用的东西连接到身体上——在这种情况下，是一个以气体为动力的除草机的支撑吊带——它将负荷分配到上半身，还有一个 Arduino 用于大脑，此后，他面临着肌肉传感器的障碍，这些传感器旨在实现更本能的控制。因此——目前——功能仅限于由两个开关控制的简单上下运动。值得注意的是，向下开关目前的安装方式是，当用户向下移动他们的手臂时，外臂也会跟着移动，因此在当前的迭代中使用手臂会有一些自然的感觉。

记住，这个原型是为老年人和其他需要增强体力才能过正常生活的人开发的，它能够卷曲超过自身重量 10 公斤。目前，唯一的马达在肘关节上——允许基本的运动范围——一个适应肩关节的马达即将出现！而且，成本只有 100 美元，这是一个很好的开始。
