# Arduino 的核心内存升级

> 原文：<https://hackaday.com/2018/04/28/core-memory-upgrade-for-arduino/>

当 Linux 程序行为不当时，会产生核心转储。他们之所以有这个名字，是因为磁心存储器是过去计算机的主要存储器，我们许多人仍然把计算机的主存储器称为“核心”如果你曾经想要一台有真正核心内存的电脑，你可以得到一个插入 Arduino 的主板，为它提供 32 位核心存储。当然，Arduino 不能直接运行内存之外的程序，正如设计师[Jussi kilpelinen]提到的，这是“非常不切实际的”该板已经存在了一段时间，但最近的一个视频让人们关注这种复古设计。

不管是否不切实际，在现代 CPU 上拥有真正的磁心内存还是很有魅力的。核心平面不像旧的商业产品那样密集，旧的商业产品只能在一立方英尺中容纳 32 千比特(不是字节)。我们将让您自己来计算 8g 的笔记本电脑需要增加多少内存才能使用核心内存。

老实说，这纯粹是一种新奇的东西，但我们确实有点怀念核心内存。它本质上是不挥发的。你可以关掉电脑，再打开，一切都和你离开时一样。当然，有点奇怪的是，读一点也会破坏它，但许多旧计算机都在 CPU 架构中内置了读后写周期，所以这不是什么大问题。

如果你想看看[是如何修复一个大型核心系统](https://hackaday.com/2015/10/19/repairing-55000-of-vintage-core-memory/)的，我们之前已经看过了。然而令人惊讶的是，这并不是我们看到的第一个 [Arduino 核心内存装备](https://hackaday.com/?s=magnetic+core)。
