# 最后，现代特雷门琴

> 原文：<https://hackaday.com/2016/01/24/finally-a-modern-theremin/>

曾经想拥有自己的特雷门琴，但无法证明花几百美元买一台是合理的吗？现在你可以建立自己的，或购买一个五重哈密尔顿。[开放。Theremin.UNO 项目已经围绕无处不在的 Arduino Uno 板建立了基于天线的振荡器控制](http://www.gaudi.ch/OpenTheremin/index.php?option=com_content&view=article&id=103&Itemid=93)。

那里面的 Arduino 是干什么用的？这是一个数字特雷门琴，但看看下面的视频，你会同意它听起来很棒，而且有很好的响应。用于音量和音高的铝天线附在屏蔽的顶部，但听起来好像它们不包括在套件中。别着急，你可以为此使用多种材料。在底部，你需要连接一根扬声器电缆，如果电缆没有接地，还需要一根地线。

顾名思义，这是开放硬件，我们对他们网站上的文档和 BOM(在[GitHub repo](https://github.com/GaudiLabs/OpenTheremin_Shield)上找到)非常满意。这个设计早在 2013 年就已经展示过了[藏在一包香烟里](https://www.youtube.com/watch?v=PdDYnI0xOrY)。如果你不想自己动手，他们在自己的网站上卖 48 欧元的套装，或者在 Tindie 上卖 55 美元的[。](https://www.tindie.com/products/gaudi/theremin-shield-for-arduino/)

好吧，我们已经搞砸了很多次，我们将尝试在这里得到它:在《星际迷航》第一季的开场和《神秘博士 T2》的开场中都没有听到特雷门琴。它也没有出现在《好的振动》中。据我们所知，它根本不用于流行文化中的任何东西…但是识别声音并知道它是什么仍然是极客的核心知识。

如果你想用你的整个身体弹奏特雷门琴，你需要特雷门琴萜品酮。
