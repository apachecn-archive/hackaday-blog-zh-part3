# Hackaday 奖参赛作品:Bellcycles 是开源的、紧凑的、独特的

> 原文：<https://hackaday.com/2017/10/12/hackaday-prize-entry-bellcycles-are-open-source-compact-and-unique/>

我们想要自行车做什么？它应该可以在家里建造，即使你的家是纽约的公寓。它应该是开源的，所以我们的朋友可以自己制作。它应该很紧凑，这样就不会把我们的小公寓弄得乱七八糟。它应该是独一无二的，而不是另一个我也是。Bellcycles 的 Alex Bell 正在 hackaday.io 上展示他的自行车 T1，它满足了所有要求。

这种不寻常的形状大大减少了传统自行车的尺寸、转弯半径和存储空间。它分享了 [penny farthing](https://en.wikipedia.org/wiki/Penny-farthing) 的大前轮设计。不像巨大的轮式硬币，骑手离地面更近，所以不需要特殊的技术就能骑上去。事实上，从自行车上下来就像站起来一样简单，因为骑车人面前没有任何东西，这对城市通勤来说是个好消息。

如果实用性让位于特殊性，看看这辆 [Strandbeest 自行车](http://hackaday.com/2016/06/30/a-strandbeest-bicycle/)，如果你宁愿呆在你的公寓里，你仍然可以[在虚拟现实](http://hackaday.com/2017/03/12/take-a-bicycle-tour-anywhere-in-the-world/)中进行一次环球自行车旅行。
