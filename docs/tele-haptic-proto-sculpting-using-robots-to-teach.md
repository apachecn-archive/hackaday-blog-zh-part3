# 远程触觉原型雕刻:使用机器人教学

> 原文：<https://hackaday.com/2015/12/18/tele-haptic-proto-sculpting-using-robots-to-teach/>

在一个启发式的远程触觉原型雕刻环境中，有可能有效地传达触觉教学信息吗？让我们试着换个说法。如果你可以用一个机器人来帮助教别人一种创造性的技能会怎么样？想象用画笔引导某人的手。现在想象一下，使用一系列相连的机器人，用画笔引导一群人的手。

来自[Morgan Rauscher]的是[Art-Bot 2.0](https://www.youtube.com/watch?v=ujQ3BaUihjU)——一种创造性的学习工具，它提供了一种全新的方式来教授绘画、雕刻或几乎任何需要技巧或工具的事情。几年前我们报道过 Art-Bot 1.0，但是如果你已经忘记了，它是一个[八英尺高的电锯挥舞机器人](http://hackaday.com/2014/08/05/what-could-possibly-go-wrong-giving-a-robot-a-chainsaw/)在一个围栏里。即使是孩子，使用遥控器，也可以玩电锯。

在六芒星研究所的帮助下，Art-Bot 2.0 由三个坚固的伺服驱动机器人手臂组成。一个是给老师的，用来指导动作，一个在一个工件上做同样的动作，第三个机器人手臂可以让学生感受正在发生的事情。

手臂上的一个小触觉反馈垫甚至可以给学生的手提供触觉。

这基本上是上周[Wolf Tronix]向我们展示的技术的延伸，通过伺服系统感受力。

这是未来人们学习某些动手技能的方式吗？你怎么想呢?
