# 脱焊不一定很烂

> 原文:[https://hacka day . com/2015/12/28/desoldering-needly-suck/](https://hackaday.com/2015/12/28/desoldering-doesnt-necessarily-suck/)

你最喜欢用什么方法来修复焊接错误或从垃圾箱里找到的电路板上取下可用的元件？我一直偏爱拆焊辫子，尽管我已经开始根据情况使用真空泵。[原型 G]发送了一个指令，列出了九种不同的方式来解除组件的焊接，这需要不同的时间和技能。

他从一个经常被忽视的问题开始，如果你没有一个锡罐的话。[Proto G]只在不想留板的时候推荐这个方法。用助焊剂覆盖您想要保留的元件的焊点，并在用钳子拔出元件时将其放在焊料罐上。流量并不重要，但它能使清除更快更容易。

对于需要维修的电路板，[Proto G]使用手动泵或涂有焊剂的铜脱焊编织物。如果你买得起的话，拆焊机似乎是个不错的选择——它结合了烙铁的热量和手动泵的真空。拆焊镊子和热空气返工站看起来是移除表面贴装元件的好方法。

如果你喜欢这个，看看[【Bil Herd】关于元件拆焊的指南](http://hackaday.com/2014/08/21/smt-and-thru-hole-desoldering/)。还有一些[Proto G]没有提到的方法，比如把板子举过[酒精火焰](http://hackaday.com/2013/09/19/desoldering-chips-with-fire/)。请在评论中告诉我们你最喜欢的脱焊方法。
