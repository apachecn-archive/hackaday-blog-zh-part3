# 有时正方形是正方形:基本机械师技能

> 原文：<https://hackaday.com/2016/10/25/sometimes-square-is-square/>

有没有可能制作一个有趣的视频，把一个铝立方体变成一个稍微立体一点的立方体？事实证明，是的，如果你看《这位老托尼》[讲述平仓股票的基本知识](https://www.youtube.com/watch?v=tW8HNAlUXxU)，你甚至可能会从视觉笑话和内部笑话中学到一些东西。

无论你是在做木头还是金属，从平坦、光滑和垂直的表面开始是获得高质量结果的关键。[Tony]主要是一名机械师，所以他用一块很好的铝条工作，并学习一些每个金属工人都需要知道的基本技能。当你工作到千分之一英寸时，很容易出错，在虎钳钳口和枪托之间使用滚珠轴承来防止倾斜等技巧是至关重要的技能。他介绍了如何控制磨机，选择要切割的面和切割的顺序，以及如何检查你在平板上的工作，并在出错时进行纠正。寻找机械师同事[Abom79]和[Stefan Gotteswinter]的客串，包括[Stefan]在一个非常妥协的位置。但是一个老虎钳里的球没有参考？走吧。

[托尼]通过使事情变得轻松，使一个潜在乏味的主题变得非常有趣，我们欣赏这种幽默和对细节的关注。他制作了一些我们之前报道过的很棒的视频，比如[制作自己的弹簧](http://hackaday.com/2016/05/29/making-springs-at-home/)或者[一个自制的无聊头](https://hackaday.com/2016/06/23/machine-tool-build-is-anything-but-boring/)，他的东西真的很值得一看。
