# 无线充电器让我们一窥工业设计流程

> 原文：<https://hackaday.com/2018/05/26/wireless-charger-gives-a-glimpse-into-industrial-design-process/>

几乎市场上的每件产品在其开发的某个阶段都经过工业设计师的手。从你口袋里的电话到你车道上的汽车或你壁橱里的吸尘器，事物的外观和工作方式都是精心设计过程的结果。看看这个过程的内部，就像[这个无线手机充电器概念](https://www.youtube.com/watch?v=H3B6wgV60qg)一样，令人着迷，可以产生真正有价值的设计见解。

我们之前介绍过很多[埃里克·斯特雷贝尔]的作品，主要是他提供的伟大的制作技巧和诀窍，比如[如何获得精美的漆面](https://hackaday.com/2018/04/29/how-to-put-the-pro-in-prototype/)或者[Bondo](https://hackaday.com/2018/04/07/you-can-build-anything-out-of-what-is-holding-your-97-eagle-talon-together/)的多种用途。但这一次，他向我们展示了他设计无线手机充电器和支架的精简版。他的客户有具体的要求，比如能够在横向或纵向模式下举起手机，所以他从纸和笔开始，勾画了一些想法。一个可旋转的圆柱体似乎符合要求，在 PVC 管中快速模拟后，他开始制作全尺寸的聚氨酯泡沫原型。在下面的视频中有一些很棒的制造技巧，主要集中在处理没有车床的问题上。

对我们来说，所有[Eric]的视频，尤其是这一个，是看到了设计过程，从开始到(几乎)结束。他确实让工业设计看起来像一个很酷的工作，一个对在这里闲逛的各行各业的人有吸引力的工作。
