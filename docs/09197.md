# 本特·PETG 填补了谢妮的空白

> 原文:[https://hack aday . com/2018/04/19/bent-petg-fills-a-谢妮-gap/](https://hackaday.com/2018/04/19/bent-petg-fills-a-nixie-gap/)

你有没有想过谢妮电子管很酷，但太难用现代电子设备控制？它们太贵了吗？[david.reid]显然是这样认为的，并决定创造他自己版本的数码管，它不会比这个便宜太多。

![](../Images/4200bf0c64c51967a6c91b544d73f599.png)

PETG Nixie Tube

在和他的儿子一起研究 3D 打印的机车时，[david.reid]使用透明的 PETG(聚对苯二甲酸乙二醇酯)3D 打印机灯丝将 led 发出的光传输到机车的各个部分。他发现这是成功的，但粗糙的灯丝外部，看看会发生什么。你瞧，一股暖流出现在试管的表面！像任何优秀的黑客一样，他的下一个想法是谢妮电子管，正如你已经在 [中](https://hackaday.com/2018/02/19/the-white-rabbit-nixie-clock/)[见过](https://hackaday.com/2018/03/21/neon-lamps-make-for-the-coolest-of-nixie-clocks/) [许多](https://hackaday.com/2018/03/20/mystery-nixie-clock-from-2001-a-space-odyssey-and-pulsar/) [时钟](https://hackaday.com/2017/07/16/slimline-nixie-clocks/)。

他的基本想法是，只要稍微加热，你就可以将灯丝弯曲成你喜欢的任何形状([大卫·里德]使用定制模具)。然后你用一些砂纸将你想要显示光线的地方弄粗糙，并在底部添加一个 LED 来照亮它！

不是第一个使谢妮地铁现代化的人。这些年来，我们见过它们[结合 Wi-Fi 板](https://hackaday.com/2017/11/10/esp-powered-nixie-clock-knows-the-time/)、[单个 LED 段](https://hackaday.com/2018/03/26/its-a-nixie-its-a-vfd-no-its-a-custom-led-display-in-a-tube/)，甚至[激光切割机& WS2812s](https://hackaday.com/2017/01/14/smoothly-modernized-nixie-display/) ！
