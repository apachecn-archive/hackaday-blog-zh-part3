# 修理复杂的转轮

> 原文:[https://hack aday . com/2016/05/26/fixing-a-complicated-scroll wheel/](https://hackaday.com/2016/05/26/fixing-a-complicated-scrollwheel/)

[托马斯]喜欢他的罗技 MX 主鼠标，它有一个非常复杂的滚轮机制。或许过于精细；使用一周后，当他开始习惯这个功能时，它就坏掉了。那么他做了什么？[拆开固定](https://toemat.com/logitech-mx-master-fix/)，自然。作为奖励，我们可以在导游的带领下参观这个有趣的装置。看看下面他的视频吧。

加重滚轮在两种不同的模式之间切换，一种是像你可能习惯的那样带有制动器，另一种是允许滚轮自由旋转进行长途旅行。为了做到这一点，它实际上有一个小马达在里面旋转一个凸轮，把一个杠杆扔进滚轮的一侧，用于制动模式，然后把杠杆拉出，用于自由旋转。它还必须有一些内部逻辑来检测滚轮旋转的速度，因为滚轮一停止，它就会重新啮合。

这是一个甜蜜的机制。但是它有一个缺点。凸轮可以一路向上，错过一个停止点并过度旋转，使滚轮处于半止动半旋转的无人区。简单地把它推回去似乎就能解决问题，至少暂时如此。

我们喜欢修补。我们喜欢窥视机器内部很酷的机制。所以谢谢，[Thomas]在相机运行时打开你的鼠标。
