# 步进电机和密钥卡控制的 Strandbeest

> 原文:[https://hack aday . com/2018/05/12/stepper-motor-and-key-fob-controlled-strand beest/](https://hackaday.com/2018/05/12/stepper-motor-and-key-fob-controlled-strandbeest/)

我们永远不会厌倦观看有着多条腿四处走动的角马，尤其喜欢钢筋混凝土角马。多产的 Strandbeest 制造商杰里米·库克(Jeremy Cook)刚刚通过对一个小型塑料风力驱动套件(T1)进行机动化和增加遥控功能制作了一个。

我们以前见过这样的 Strandbeest 套件转换，比如这个 [DC motor one](https://hackaday.com/2016/02/03/motorized-strandbeest-is-remote-controlled-and-awesome/) ，但看到它如何以不同的方式完成总是很有趣。在[杰里米的]店里，他带走了两个便宜的 2 美元的步进电机。RC 是使用一个带有接收器板的密钥卡发射器完成的，接收器板连接到 Arduino Nano 的模拟引脚。他试图直接用脂肪电池驱动它，但遇到了一些问题，他通过添加一个 5 伏调节器解决了这些问题。在下面的视频中，看看他的体型和经过改造的 Strandbeest。

正如我们所说，[杰里米的]是一个多产的 Strandbeest 制造商。我们很高兴之前在这里展示了他的[更大、清晰的聚碳酸酯 ClearWalker](https://hackaday.com/2017/03/15/strandbeest-not-fooling-anyone-we-see-right-through-it/) 以及他的更小的 [Strandmaus，他在其中为 RC](https://hackaday.com/2016/04/11/strandmaus-small-rc-strandbeest/) 改装了四轴飞行器部件。
