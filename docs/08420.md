# HandHolo:一个自制 ARG

> 原文:[https://hackaday.com/2018/02/01/handholo-a-homebrew-arg/](https://hackaday.com/2018/02/01/handholo-a-homebrew-arg/)

进入虚拟现实或增强现实——曾经是科幻小说中的梦想——不仅对普通消费者来说是可能的，而且精心制作这些体验也是可能的！Hackaday.io 用户[kvtoet]的 HandHolo 是一个自制的方法，可以让你在[窥视虚拟世界](https://hackaday.io/project/41120-handholo)的时候大开眼界。

这个项目需要一部运行 Android Oreo 的智能手机作为主干，一个蓝牙鼠标，一张纸板和一个小镜子或高反射表面。手机被插入纸板外壳——用你现有的东西做原型！—在鼠标上方，当用户探索虚拟场景时，与屏幕成角度的镜子将图像反射回用户。

在 Unity 中，[kvtoet]使用了一些访问电话功能的脚本，即陀螺仪，它与鼠标的移动同步。这种运动被转化为对构建在 Unity 中的虚拟空间的探索，并被投射到门户状的镜子上。看看吧！
