# 使用 PVC 和加热安装显微镜 DSLR

> 原文：<https://hackaday.com/2017/02/10/microscope-dslr-mount-using-pvc-heat/>

显微镜是在更小的尺度下观察宇宙奥秘的好方法。当它们最初被开发出来时，科学家们不得不依靠插图来通过镜头传达他们的发现。谢天谢地，我们现在可以依靠摄影来帮助我们。许多显微镜都带有一个特殊的端口，通常称为三目镜端口，用于安装摄像头。利用这一点，[【Brian】为他的显微镜](https://hacknbuild.github.io/blog/posts/DSLR-Microscope-Mount/index.html)开发了一个 DSLR 底座，使用的是一种黑客主食:PVC 管。

PVC 管的优点多种多样。它有各种形状和尺寸，有各种各样的耦合器、适配器、溶剂和胶水。最重要的是，你可以将它加热到变得柔软易弯的程度，这样就可以在需要的地方定制合身的衣服。[Brian]演示了如何使用热风枪加热减速器，使其与 DSLR 镜头支架摩擦配合。除此之外，支架使用一对来自珠宝商 loupes 的镜头，将图像聚焦在相机的传感器上，传感器整齐地安装在 PVC 连接器内。
