# 为更好的摄影制作一个轻量级的软盒

> 原文：<https://hackaday.com/2018/03/13/building-a-lightweight-softbox-for-better-photography/>

如果你想拍出好照片，你需要好的光线。对我们来说幸运的是，你可以从中国花几便士买到一卷又一卷的 LED 灯，电源无处不在，任何人都可以焊接几个 LED 灯条。拼图中缺少的部分是一个适合所有这些 led 的外壳和一个光扩散器。

[Eric Strebel]最近需要一个柔光箱来拍摄一些产品，[于是想出了这个非常便宜、非常好的照明解决方案](https://www.youtube.com/watch?v=ti0K_hmbMw8&feature=youtu.be)。它由铝制成，所以它应该能够应对摄影的严格要求，而且它绝对是*装载的*led 灯，可以让所有的光线照射到主体上。

这个软盒的金属外壳是由大约 22 号的铝板制成，并在一个压弯机上折叠。这是你能用一个刹车和一片金属做的最简单的项目，用环氧树脂把外壳的标签粘在一起。这个盒子的安装是简单的磁铁超级粘在背面，以连接到轨道照明设备。5000 K 的 LED 灯条用 3M Super 77 喷胶固定在盒子上，剩下唯一要做的就是将所有的 LED 灯条串联起来。

但如果没有某种扩散器，这实际上只是一个金属盒，里面有一些发光二极管。为了让他的主题获得均匀的光线，[Eric]用白色胶水将画纸贴在金属框架上。结果相当惊人，这是一个非常轻和坚固的摄影软盒。
