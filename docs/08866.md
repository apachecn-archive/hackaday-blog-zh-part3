# 用 Arduino 和云为 Linux 设备编程

> 原文:[https://hack aday . com/2018/03/15/programming-Linux-devices-with-arduino-and-the-cloud/](https://hackaday.com/2018/03/15/programming-linux-devices-with-arduino-and-the-cloud/)

回到过去，当电线库还很烂的时候，Arduino 只是一个微控制器。现在，我们有单板电脑和内置 WiFi 的廉价微控制器。一如既往，我们需要让编程和嵌入式开发变得更容易访问，并在当今数百种可用设备中得到更广泛的支持。

在本周的嵌入式 Linux 会议上，[Massimo Banzi]宣布了 Arduino 对云、在线 ide 和一个庞大的互联设备生态系统的回应。这是 Arduino Create ，一个允许任何人开发嵌入式项目并远程管理它们的在线 IDE。

[正如在【Massimo】的主题演讲](https://www.youtube.com/watch?v=SuFUkxxpDrE)中所展示的，Arduino Create 的核心理念是将连接设备放在互联网上，并允许无线更新和开发。由于这是 Arduino，数百个不同平台上可用的库的容量被利用来使这成为可能。目前，支持各种各样的板，包括 Raspberry Pi、BeagleBone 和几个英特尔物联网板。

这种开发的焦点是平台无关的，几乎完全集中在易用性和互操作性上。与五年前的 Arduino 相比，这是一个显著的变化；曾经有一段时间，Arduino *是一个 ATmega328p，仅此而已。几年后，你可以把 Arduino 草图放在 ATtiny85 上。从那以后，很多事情都变了。我们有了 Raspberry Pi，我们有了英特尔涉足物联网设备领域，我们有了一百万块基于智能手机 SOC 的主板，英特尔退出了物联网市场。*

虽然其他公司和组织已经进入了 Raspberry Pis 和其他单板计算机的在线 IDE，即 [Adafruit webIDE](https://learn.adafruit.com/webide/overview) 和 [Codebender](https://codebender.cc/) ，但这是一个受欢迎的变化，已经得到了 Arduino 组织的支持。
