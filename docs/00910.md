# DIY 电致发光线，俄罗斯风格

> 原文:[https://hack aday . com/2015/12/11/DIY-电致发光-电线-俄罗斯风格/](https://hackaday.com/2015/12/11/diy-electroluminescent-wire-russian-style/)

我们最喜欢的俄罗斯疯狂科学家[Kreosan]又一次用另一个非常不明智的超级坏主意**震惊了我们。自制的 EL 电线，用电源电压供电。**

从图片上看它很像 EL wire，不是吗？嗯，它实际上只是一根连接到主交流电源的镍铬线。呃，测试室？看起来他们不像住在那里，所以我们想知道为什么它还有电。不管怎样，是的，他们用电线做了一个不稳定的负载，并直接连接到面板上。所以除了火灾危险，你也可能触电！

对于房屋装饰来说，这是一个温暖房间的好方法，如果你幸运的话，它甚至可以帮助生火！
