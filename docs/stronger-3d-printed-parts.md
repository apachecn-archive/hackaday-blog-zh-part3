# 更坚固的 3D 打印零件

> 原文：<https://hackaday.com/2015/11/20/stronger-3d-printed-parts/>

当[hobbyman]想要一些 3D 打印的零件来给他的自行车装一个袋子时，他担心当袋子装满时，这些零件不够坚固，无法支撑。他决定想办法用玻璃纤维和环氧树脂加固这个部分。他的第一个模型有洞和凹槽，用环氧树脂填充。

然而，在与该部分合作了一段时间后，他决定采取不同的方法。他没有把这个部件做成有环氧树脂空间的几乎实心的塑料，而是把它做成一个外壳，然后用纤维和环氧树脂填充。在它完全固化后，一点点打磨开始去除一些塑料外壳，剩下的大部分是铸造的玻璃纤维部分(尽管一些塑料还留在上面)。

你可以在下面的视频中看到结果。根据[hobbyman]的说法，这些部分就像“花岗岩”这项技术让我们想起了 [OpenHand 项目是如何为聚氨酯](http://hackaday.com/2015/02/25/openhand-combines-3d-printing-with-urethane-casting/)制造模具的。如果你打算尝试制造像[hobbyman]那样的零件，你可能想尝试[给你的环氧树脂着色](http://hackaday.com/2015/09/04/tint-your-epoxy-resin-with-toner-powder/)。
