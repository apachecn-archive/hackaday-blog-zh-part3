# 利用磁力将硬盘射向天空

> 原文:[https://hack aday . com/2016/03/10/shoot-hard-drive-platters-skywards-on-the-power-of-magnetic/](https://hackaday.com/2016/03/10/shoot-hard-drive-platters-skywards-on-the-power-of-magnetism/)

Project Hathor 是一个电磁环形发射器，只需按一下按钮，就可以将铝制硬盘发射到 45 英尺高的空中。这项艰巨的工作是由一组电容器完成的，这些电容器从微波炉变压器充电到 2kV，然后放电到硬盘驱动器所在的线圈中。由此产生的磁场爆发在盘子中感应出巨大的电流，而电流反过来又产生了一个相反的磁场，将戒指发射到空中。

这个发射器是拉斯维加斯的[Syn 商店](https://synshop.org/)hacker space 的[Krux]的作品，他做得很漂亮。电容器组有十个 3900uF 400V 电解电容器，连接成一个 1560uF 2kV 电容器，有两个 225W 2Kohm 线绕放电电阻器，以及一个设计精美的自制高压接触器，采用钨电极。为了安全起见，整个项目都被小心地放入一个丙烯酸外壳中，因为正如[Krux]指出的，微波炉变压器会杀了你。

除了项目网站，还有一个 [YouTube 播放列表](https://www.youtube.com/watch?v=ToYnjJG9PhE&index=2&list=PLNhWAQoSmSm9ohV3jEapeWcU2dwzoJiHP)，一个[图像库](https://krux.org/gallery/index.php/electronics/Ring-Launcher?page=1)，以及一个 [GitHub 库](https://github.com/krux702/Project_Hathor)，包含了项目的所有细节。在下面的视频中，你可以看到发射器正在工作，它正在向内华达之夜发射大盘子。
