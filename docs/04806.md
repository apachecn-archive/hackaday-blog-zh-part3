# 梦想爆破 X2:一个现代的 MIDI 合成器为您的声霸卡

> 原文:[https://hack aday . com/2017/01/21/dream blaster-x2-a-modern-midi-synth-for-your-sound-blaster-card/](https://hackaday.com/2017/01/21/dreamblaster-x2-a-modern-midi-synth-for-your-sound-blaster-card/)

早在 90 年代，游戏玩家就在他们的电脑上安装了 Creative 的 Sound Blaster 系列声卡。那些真正重视音频的人可以连接一个名为 [Creative Wave Blaster](https://en.wikipedia.org/wiki/Creative_Wave_Blaster) 的子板。这张卡使用波表合成来提供比声霸内置的雅马哈 FM 合成芯片更真实的乐器声音。

DreamBlaster X2 是 Sound Blaster 声卡的现代子板。使用声卡上的连接器，它具有立体声音频输入和 MIDI 输入和输出。如果你没有使用声霸，提供 3.5 毫米插孔和 USB MIDI。由于 MIDI 使用 TTL 电压，它可以直接连接到 Arduino 或 Raspberry Pi。

此卡采用 Dream SAM5000 系列 DSP 芯片，最多可进行 81 种复音声部的波表合成。它还执行混响、合唱和均衡器效果。该芯片将音频数据发送到 24 位 DAC，该 DAC 将音频输出到声卡或 3.5 毫米插孔。

DreamBlaster X2 还带有加载波表的软件，以及可供试用的波表。我们相信它将是您 2017 年发布的 486 的最佳升级。如果你感兴趣，你可以[订购一个组装的 DreamBlaster](http://www.serdashop.com/DreamBlasterX2) 。休息之后，用音频演示进行回顾。
