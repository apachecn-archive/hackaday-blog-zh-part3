# 甘草发射器锁定你的声音

> 原文:[https://hack aday . com/2016/09/12/甘草发射器锁定你的声音/](https://hackaday.com/2016/09/12/licorice-launcher-locks-on-to-your-voice/)

随着红色甘草战争的激烈进行，似乎红色藤蔓团队比扭曲者团队有了新的优势——[TV miller]的[语音激活，房间跟踪，弹射发射，弹匣重装 Arduino 甘草发射器](https://hackaday.io/project/12893-arduino-licorice-launcher)。

黑客攻击和吃零食经常是齐头并进的。如果你有一个能根据命令给你扔好吃的东西的机器人，它们会搭配得更好。这就是这个糖果弹射器背后的梦想。我们在过去报道过不少[TVMiller]的项目，所以我们知道他花了很多时间做黑客。那么，这个甘草发射器是如何让他坚持下去的呢？

就像我们说的，它是声控的。机器人回答“雅各布”,并在听到“红葡萄”这个词时采取行动。雅各布锁定了声音的方向，并转过身去面对它。然后他发射有效载荷。因为雅各布正在重新装弹，所以总是有红色的藤蔓在房间里等待[电视米勒]的零食攻击。发射后，堆叠底部的甘草落入接收支架的槽中。转动摇篮，把它装进投石机。休息后观看雅各布杀死渴望。

让你在放弃商品之前正确回答琐事问题的糖果分发机是你能得到的离“雅各布，红藤”最远的东西，但至少你会觉得你应该得到它。
