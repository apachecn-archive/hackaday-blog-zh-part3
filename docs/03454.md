# 碳纳米管晶体管在超车道上

> 原文:[https://hack aday . com/2016/09/04/碳纳米管晶体管正在路上/](https://hackaday.com/2016/09/04/carbon-nanotube-transistors-are-on-the-passing-lane/)

在将碳纳米管转化为有用的东西的道路上有许多障碍。威斯康星大学麦迪逊分校的材料工程师现在已经使碳纳米管(CNTs)更接近实际应用于半导体电子器件。特别是，该团队成功组装了碳纳米管晶体管阵列，其性能优于其硅基前辈。

研究人员必须克服的一个障碍是金属杂质，这些杂质存在于电弧放电产生的碳纳米管中，实际上短路了材料的半导体特性。为了去除这些杂质，研究人员开发了一种基于聚合物溶液的分类机制，该溶液包裹在碳纳米管周围，在特殊条件下，从纯度为 99.98 %的坏碳纳米管中选择好的半导体碳纳米管。

[![cnt-fesa](../Images/aafc0f184436de8eb568d696709c34f6.png)](https://hackaday.com/wp-content/uploads/2016/09/cnt-fesa.jpg)

Floating Evaporation Self-Assembly ([image source](http://pubs.acs.org/doi/abs/10.1021/la500162x))

另一个挑战是碳纳米管在晶片上的排列和间距，在这种情况下，高性能晶体管的生产需要碳纳米管平行排列并具有一定的间距。该团队可扩展的快速[“漂浮蒸发自组装”(FESA)](http://pubs.acs.org/doi/abs/10.1021/la500162x) 过程通过缓慢地将衬底垂直拉出水浴来实现这一点，之前生成的高纯度纳米管溶液漂浮在表面上。这使得当溶剂蒸发时，管在空气-水界面上在基底上自对准。随后烘烤所得的晶片从分类步骤中去除了非导电聚合物包装材料。

为了将沉积的碳纳米管转变成晶体管，首先在晶片上涂上一层 PMMA 抗蚀剂，然后用电子束光刻法形成图案。然后，在用丙酮从剩余的 PMMA 中除去所得的晶体管结构之前，蚀刻掉不需要的区域。最后，沉积钯触点以与晶体管连接。

[![cnt-fet](../Images/34bf091cc5bc90b331ee00f0672563d8.png)](https://hackaday.com/wp-content/uploads/2016/09/cnt-fet.jpg)

Carbon nanotube transistor ([image source](http://advances.sciencemag.org/content/2/9/e1601240.full))

这种新的工艺链导致碳纳米管场效应晶体管(FET)阵列的密度为每微米 50 个晶体管，并具有准弹道传导，这是一种快速通道传导状态，其中电子几乎不受阻碍地穿过纳米管，因此遵守牛顿第二运动定律。在测试中，这些晶体管显示出 1.7 毫秒/微米的电导和 900 μA/微米的饱和电流，研究人员将其描述为相同尺寸和架构的最先进硅晶体管电流的 1.9 倍。

描述该过程及其结果的[研究论文](http://advances.sciencemag.org/content/2/9/e1601240.full)标志着碳纳米管研究的一个重要里程碑。碳纳米管晶体管第一次以可扩展的方式制造出来，其性能至少可以与硅和砷化镓相媲美。由于碳纳米管研究的突破定期发生，*管*-逻辑的回归至少是可以预见的。欣赏下面的视频，研究人员解释了这一发展的意义。
