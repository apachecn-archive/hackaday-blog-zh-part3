# 益智闹钟让情侣在早上起床

> 原文:[https://hack aday . com/2016/02/13/puzzle-alarm-clock-gets-couple-up-in-the-morning/](https://hackaday.com/2016/02/13/puzzle-alarm-clock-gets-couple-up-in-the-morning/)

[BrittLiv]和她的男朋友为了谁设的闹钟吵了很多次架。这似乎是他们唯一重复的论点。所以，出于工程师的本性，她过度设计了。结果就是这个[大拼图闹钟。](http://www.instructables.com/id/Puzzle-Alarm-Clock/?ALLSTEPS)

前面显示的时间不是当前时间。因为争论首先是关于闹铃时间的，所以[BrittLiv]决定最突出的数字应该是下一个闹铃。要听到时间，必须按下时钟前面的一个按钮(冒号中的一个点)。然而，要设置闹钟，你必须手动将磁化段移动到你想要起床的时间。在处理方面，对于一个时钟来说，它带着一些热量。它运行在一个英特尔爱迪生上，用来[合成](http://hackaday.com/2015/12/02/audio-effects-on-the-intel-edison/)时间、新闻、天气和推文的声音。听起来很棒，休息后看看吧。

总的来说，这个钟看起来很棒，也很好用。我们希望它能给[布里特利夫]的家庭带来和平。
