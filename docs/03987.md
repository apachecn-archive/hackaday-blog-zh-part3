# 在阴极射线管的光芒下

> 原文:[https://hackaday.com/2016/10/31/by-the-glow-of-the-crt/](https://hackaday.com/2016/10/31/by-the-glow-of-the-crt/)

如果你到了一定的年龄，你可能还记得在一个旧的 CRT 终端的琥珀色或绿色灯光下编写软件(或玩冒险游戏)。如果你年龄更大，你可能会发现它比打卡好得多，但那是另一回事了。[鸢]想要重温那些日子(嗯，听起来他太年轻了，还没有开始过这些日子)，所以他在他的 Linux 盒子上连接了一个 VT220 终端。

这并不奇怪。Linux 的前身 Unix 期望这种终端(或硬拷贝电传打字机),并且使用玻璃终端的所有功能仍然存在。你必须处理一些[鸢]解决的配置项目。

事实上，他似乎是在 VT220 上使用 vi 编写了他的博客文章，使用基于文本的 Web 浏览器来研究链接。他有很多资源可以将任何类型的终端(甚至是终端模拟器)连接到 Linux 计算机上。

最近人们对旧码头很感兴趣。你看到很多旧的[vt 100 躺在](https://hackaday.com/2013/10/16/vt100-gets-beagleboned/)周围。我个人有一台 ADDS Regent 100，偶尔会连接到我的几台计算机。你可以在下面的视频中看到。
