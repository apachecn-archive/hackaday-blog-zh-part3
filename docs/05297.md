# 机械泡茶法

> 原文:[https://hack aday . com/2017/03/13/机械泡茶/](https://hackaday.com/2017/03/13/tea-making-the-mechanical-way/)

对于一些茶迷来说，制茶是一件严肃的事情。对他们来说，浓茶、红茶、叶茶放在脏兮兮的茶壶里炖了很久，会让你的胸毛脱落(就像应该做的那样)，这是不行的。这些饮料无政府主义者要求仔细调节温度和时间的准备过程，出于某种原因，反复将茶包泡在水中。

对他们来说，[道林·达蒙]有一个自动化的解决方案来让关键的扣篮过程变得正确。他做了一个自动茶袋浸泡器。茶袋安装在由曲柄操纵的滑板上，曲柄通过一对自行车轮毂驱动。动力来自一个主罩极电机，一个不寻常的双向电机，他只使用一侧。他测得他个人的扣篮速度约为每分钟 50 次，所以他只需要 4:1 的减速比来匹配 200 RPM 的马达。

只要机器开着，它就会很乐意以这样的速度泡他的茶包。他上传了几个视频，其中一个我们放在了休息时间下面。
