# 禅与泡沫芯艺术

> 原文:[https://hack aday . com/2018/02/10/Zen-and-the-art-of-foam-core/](https://hackaday.com/2018/02/10/zen-and-the-art-of-foam-core/)

我们的一些消遣是如此深入冥想，以至于我们迷失了自己几个小时。我们的手似乎在执行每一步，就像钢琴家背诵她最喜欢的歌曲一样。如果[埃里克·斯特雷贝尔]的声音和视频有任何暗示的话，与 foam core 合作就能产生这种效果。

泡沫芯是艺术商店、业余爱好商店和办公用品商店的主要商品。它有不同的颜色，但共同的特点是一片泡沫夹在几层纸中间。这种合成材料是一种多用途的材料，[Eric]在他的高级教程中很好地展示了这种材料，制作了一个复合表面以及后来的一个[扬声器模型](https://www.youtube.com/watch?v=rkqZ29PjZ-k)。

休息过后，你可以看几个初学者教程，解释一下草草制作的泡沫芯模型和吸引人的模型之间的区别。适当的工具和深思熟虑的计划可能是前两个视频的最大收获，除非你把禅宗的叙述算进去。上面链接的高级视频展示了一些使用泡沫芯的巧妙方法，如胶印、可调节的超级结构和纸张转移。

每个视频不到十分钟长，所以如果你刚刚开始喝咖啡休息，你现在就可以完成一个视频。或者用 [papercraft strandbeest](https://hackaday.com/2011/05/01/papercraft-strandbeest-is-a-great-rainy-day-project/) 看看另一种 2D 材料变得令人惊叹，然后用[再看看乙烯基切割机](https://hackaday.com/2017/08/01/a-case-for-the-desktop-vinyl-cutter/)来加强你的游戏。
