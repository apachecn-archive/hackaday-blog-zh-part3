# 鼠标中的 PC

> 原文：<https://hackaday.com/2017/04/10/pc-in-a-mouse/>

[Slider2732]让他的橙色 Pi Zero 与一个 3 瓦放大器、无线键盘(带内置鼠标)和倒车监视器一起工作。但他需要一个箱子来装它。他记得他曾经用各种电子产品填充电脑鼠标外壳来制作捉鬼的参数。那么为什么不把橙色的圆周率零点也放在一只老鼠身上呢？翻看他的鼠标收藏，他挑选了一个旧的罗技光学鼠标，然后去工作。

我们喜欢罗技有透明的下半部分，完美地证明给任何人谁可能怀疑 PC 真的是在鼠标里。我们认为一个很大的改进就是让鼠标也成为鼠标！但是似乎没有足够的空间了。还有什么比运行 armbian Linux 发行版、OpenELEC Mediacenter、Kodi 和一堆游戏的 Pi Zero 更小的呢？

他甚至为观看 YouTube 视频设置了无线网络。休息之后，请观看构建和演示视频。

如果我们不能让老鼠成为老鼠，那么在 Hackaday 上搜索就能找到各种可以用废弃的老鼠内脏制造的东西。你可以[用传感器观察电表的旋转表盘](http://hackaday.com/2015/06/30/disassembled-mouse-keeps-track-of-gas-meter/)或者[制作一个里程表传感器来告诉你的机器人已经走了多远](http://hackaday.com/2012/09/15/robotic-odometry-from-an-optical-mouse/)。
