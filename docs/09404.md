# 拯救计数何:重建一个历史性的游艇

> 原文：<https://hackaday.com/2018/05/11/save-the-tally-ho-rebuilding-a-historic-yacht/>

[利奥·桑普森·古尔登]是一名造船和水手。他是以传统方式建造帆船的日益减少的造船工人群体中的一个典型。2017 年，他有机会购买建于 1910 年的游艇 Tally Ho。Tally Ho 曾经是一艘骄傲的船，现在却像一个壳一样坐在收缩包装的防水布下。她的甲板腐烂了，龙骨裂开了。任何理智的人都会继续前进。幸运的是[里奥]还不太清醒，于是[开始寻求让这艘历史之船回到它曾经的荣耀](http://sampsonboat.co.uk/category/tally-ho/)。

Tally Ho 不仅仅是一艘旧船。她是一艘 48 英尺长的斜切游艇，由著名的阿尔伯特·斯特兰奇设计，在英国苏塞克斯郡建造。1927 年，[法斯特耐特帆船赛](https://en.wikipedia.org/wiki/Fastnet_Race)(修正时间)中，当汹涌的海浪使除了两艘船之外的所有船都退出时，塔利·何赢得了冠军。

说[狮子座]有适合他的工作是一种保守的说法。从赛车到捕鱼，Tally Ho 过着艰苦的生活。需要彻底修复。事实上，构建一个副本比恢复原始版本更便宜、更容易。[李奥]想救 Tally Ho，所以他买了一美元的船，并开始把他所有的时间，精力和资金投入到修复她。这项工作包括仔细记录每一件被移除的物品。

一些工具和材料是传统的，如凿子和红丹油灰。但是[Leo]也在使用电动工具，包括一个定制的电锯架来塑造龙骨。他的视频很有趣，展示了许多造船技术。只要有可能，[Leo]会添加文字说明来解释造船术语的含义，以及解释在英国和美国使用的不同术语。在最新的视频中，你可以一起观看[Leo]创造了一个荷兰人来填补龙骨上的一个结。休息后可以在视频中看到。
