# 用于水下推进的腿装啤酒瓶

> 原文:[https://hack aday . com/2016/05/31/支腿式啤酒瓶水下推进/](https://hackaday.com/2016/05/31/leg-mounted-beer-bottles-for-underwater-propulsion/)

一天，坐在沙滩上，喝完一瓶啤酒，[Rulof]意识到，如果他在啤酒瓶里放一个马达，在瓶口处放一个螺旋桨，他就可以[将这个结果连接到他的腿上，用它来推动自己在水中前进](https://www.youtube.com/watch?v=iDGe_XCJGRM)。即使没有他推动自己前进的美丽的地中海水域的额外奖励，这也是我们都希望自己想到的一种方式。

这些特殊的啤酒瓶是铝制的，用他的角向磨光机很容易将它们切开，把马达放进去。而且[Rulof]很好地利用了那台研磨机，因为他不仅用它磨圆了电机安装支架的零件，并切割了一个活塞外壳，他还用研磨机切割了一些旧运动鞋，他在运动鞋上安装了瓶子。

你可能想知道活塞在哪里发挥作用。他实际上并没有使用整个活塞，而只是使用了活塞外壳的一部分和延伸出活塞的轴。这是因为轴从外壳中露出的地方有防水密封。从下面的视频中你可以看到，海豹在它游泳的浅水区工作得很好。

为马达提供动力的电池放在一个独立的防水 PVC 圆筒中，安装在他的上身，从电池圆筒到瓶装马达的电线有防水密封。但是怎么做一个水密的通断开关呢？为此[Rulof]在电池外壳内放置一个簧片开关。簧片开关使继电器通电，继电器将电机与电池电连接。他确保将簧片开关安装在 PVC 圆筒壁附近。为了闭合簧片开关，他将一个磁铁放在圆筒外面，靠近圆筒内簧片开关的位置。为了再次打开簧片开关，他移开了磁铁。因此，簧片开关保持在防水圆筒内部，同时利用圆筒外部磁体的磁场来打开和关闭开关。

[Rulof]不是海里唯一的电鱼。还有这种用鳍推进的机器鱼，它可以用声纳与同类交流。然后是[喷气推进，从一个入口吸水，从另一个出口排出，就像这个 ROV](http://hackaday.com/2012/06/28/diy-rov-explores-the-watery-depths/) 一样。
