# 慢动作帧将成为新的魔镜

> 原文:[https://hack aday . com/2017/01/24/慢动作帧将成为新的魔镜/](https://hackaday.com/2017/01/24/slow-motion-frame-will-be-the-new-magic-mirror/)

[Paul]创建了一个使用 Arduino 和 led 的框架，[创建一个精致项目的慢动作幻觉](http://www.instructables.com/id/Slow-Dance-a-Fusion-of-Art-and-Magic/)(像一朵花或一根羽毛)。正如你在下面的视频中看到的，效果是惊人的。【T2

[Paul]曾看过类似的项目(既有一次性的，也有作为产品出售的)，但他想自己来做。原理很简单:该设备以一种频率振动物体，并以略有不同的频率(在这种情况下，80 和 79.5 赫兹)频闪发光二极管。频率之间的差异(拍频)是你的眼睛感知到的非常慢(这里是 0.5 赫兹)的运动。

一旦你知道了装置背后的秘密，制造起来就不是很复杂了。框架的木工活是大部分工作。Arduino 激发电磁体来振动目标物品。它还脉冲 LED 条，以实现频闪效果。这是简单的，引人注目的，一个展示作品。似乎每个人都在构建自己的魔镜项目，但我们提供这个令人敬畏的概念，作为每个人都应该在自己的工作台上尝试的下一件大事。让我们看看其他几个例子，让你思考一下。

[保罗]的灵感之一是时间框架，它出现在下面的第二个视频中。你可以在 [GitHub](https://github.com/cubic-print/timeframe) 上找到它的代码。它还使用 Arduino 来创建相同的效果。另一个灵感是我们之前提到的[慢舞](https://hackaday.com/2016/11/05/slow-dance-appears-to-make-time-run-in-slow-motion/)。我们也看到了用[水滴](https://hackaday.com/2011/03/11/water-droplet-sculpture-using-leds-and-arduino/)玩的类似把戏。
