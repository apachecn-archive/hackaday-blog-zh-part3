# Quickie USB 键盘设备

> 原文:[https://hack aday . com/2016/06/12/quickie-USB-keyboard-device/](https://hackaday.com/2016/06/12/quickie-usb-keyboard-device/)

我们使用的许多应用程序都可以从键盘快捷键中获益，我们虔诚地使用它们。事实上，有些任务我们做得太频繁了，以至于它们需要自己的物理按钮。唯一比定制键盘更酷的是你自己做的定制键盘。

这让我们想到了[Dan]的四按钮 Cherry MX USB 键盘。它实际上并不超过四个按键开关和一个 AVR ATmega32u4，但加上一些软件就是你真正需要的。他将 Arduino 引导加载程序编程到芯片中，然后他使用 Arduino Leonardo 键盘库。嘭！看看下面的视频。

我们认为这种设计更像是一个演示或积木的集合，而不一定是一个通用的解决方案。你可能需要五个按钮，或者想要一个不同的布局，或者……都是开源的，所以去疯狂吧。而且，您也不局限于击键，鼠标按钮甚至多个脚本操作都触手可及。

构建一个特殊功能的 USB 键盘或游戏设备曾经是一项艰巨的工作。但是今天，在硬件和软件设计可用性之间，这是小菜一碟。无论你是需要一个[脚踏板](http://hackaday.com/2013/02/26/simple-to-build-programmable-foot-switches/)、一个[单手和弦键盘](http://hackaday.com/2015/11/29/strike-a-chord-with-this-pocket-keyboard/)，还是仅仅为了[更新一台旧打字机](http://hackaday.com/2015/08/27/turning-a-typewriter-into-a-mechanical-keyboard/)，控制我们每天使用八小时的输入设备的能力是一种解放。实验！
