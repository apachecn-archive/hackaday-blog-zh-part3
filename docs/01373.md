# 伦敦尝试智能城市

> 原文：<https://hackaday.com/2016/01/31/london-tries-smart-cities/>

什么是智慧城市？根据维基百科，智能城市使用 ICT(信息和通信技术)来提高城市服务的质量、性能和交互性，同时降低成本和资源消耗。多年来，黑客一直在使用技术来增强各种东西。

伦敦正与欧洲各地的城市联手展示智能城市技术，主要是在皇家格林威治区。该项目与[欧盟地平线 2020 项目](https://ec.europa.eu/inea/en/horizon-2020/smart-cities-communities)相结合，该项目仍在征集资金提案。似乎一些黑客读者——尤其是在欧盟——应该有一些值得资助的想法。

已经宣布的举措包括:

*   300 个智能停车位，旨在优化停车位，帮助司机快速方便地找到停车位
*   测试共享电动自行车，看看它们是否支持私家车的转变
*   测试本地交付和汽车共享的电动汽车
*   利用泰晤士河作为可再生能源，为当地家庭提供可负担的热量
*   在家庭安装太阳能电池板，以提供绿色能源和提高能源效率。

让我们看看。[停车，检查](http://hackaday.com/2012/10/15/arduino-parking-lot-attendant/)。黑客[电动自行车](http://hackaday.com/2014/12/06/extending-the-range-of-an-electric-bike/)，没错。电动[车](http://hackaday.com/2015/05/27/hackaday-prize-entry-an-electric-vehicle-from-recycled-parts/)，明白了。我们见过[水电](http://hackaday.com/2010/10/11/bicycle-hub-hydropower/)。当然，太阳能也是黑客项目中的主要部分。所以我们认为，如果一个普通读者不能为一个项目获得一些资助，那将是令人惊讶的。你可以在下面看到一个关于地平线 2020 的视频。当你得到你的资助，一定要让我们知道在黑客日热线。
