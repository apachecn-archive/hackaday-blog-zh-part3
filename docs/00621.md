# 现在就行动，只需 0.00 美元就能获得尖头保护器！

> 原文：<https://hackaday.com/2015/11/12/act-now-and-receive-the-prong-saver-for-only-0-00/>

嗯，其实这个是买不到的。但是对于[TV miller]的最新项目，他决定从视频中获得一些乐趣——所以他为它制作了一个商业信息片。

被称为**插脚保护器**，[的装置](https://hackaday.io/project/8376-prong-saver)夹在任何电器的电线上，以帮助防止你不小心拉得太用力而弯曲电气插脚。这基本上是一个绳索张力报警器。问题是——你能通过吸尘器听到它吗？

因为他可以，它是太阳能的。因为为什么不呢？他用在车间周围找到的废料建造了它。其中包括一个太阳能 LED 钥匙链、一个小型压电扬声器、一个吊环螺栓和一个压缩弹簧。不管怎样，休息之后看看广告。它让我们忍俊不禁。
