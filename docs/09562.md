# 不要吃鱼——自动喂它们

> 原文：<https://hackaday.com/2018/05/25/dont-flake-on-your-fish-feed-them-automatically/>

我们明白了。你爱你的鱼，但它们不会吠叫或轻轻咬你的胫肉，让你知道它们饿了。(而且他们看起来总是有点饿，不是吗？)一天流血到第二天，你发现自己在想今天是否喂饱了它们。还是你想起了昨天？鱼应该得到更好的待遇。为什么不给它们造一个智能喂鱼器呢？

Domovoy 是一款完全开源的自动喂鱼器,可以让你按照时间表、通过蓝牙或手动喂鱼。这种简单而优雅的设计使用一个小型步进电机来驱动 3D 打印的螺旋钻来运送货物。只需打开盖子，装满薄片，并通过 3 按钮和 LCD 界面编程每天最多四次喂食。您甚至可以设置剂量，剂量是通过螺旋钻的完整旋转来测量的。

它是围绕 ATMega328P 构建的，但你必须旋转自己的电路板，并使用他出色的[指令](http://www.instructables.com/id/SMART-FISH-FEEDER-DOMOVOY/)将馈线组装在一起。想看看这个喂食器的运行吗？游过去就好了。

懒得自动喂你的鱼？[训练它们自己吃饭](https://hackaday.com/2015/03/26/training-fish-to-feed-themselves/)。
