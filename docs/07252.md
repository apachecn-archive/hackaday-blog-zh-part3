# 铣削流量传感器以保护激光管

> 原文:[https://hack aday . com/2017/09/29/milling-a-flow-sensor-to-safeguard-a-laser-tube/](https://hackaday.com/2017/09/29/milling-a-flow-sensor-to-safeguard-a-laser-tube/)

强大的激光变得非常热，如果你不保持冷却，你将付出代价。在经历了两次这样的经历后，[NixieGuy]变得聪明起来，[将这种流量传感器作为故障保险](https://electronicmercenary.wordpress.com/2016/05/18/watersports-to-kill-a-laserbird-3-of-x/)。

激光切割机太棒了。但是购买一个会很贵，而且保持它们正常工作也不是一件容易的事。从蒸发切割原料的副产品粘糊糊的东西，到保持光学系统正确聚焦，这是一个迫使你成为激光切割机操作员而不仅仅是用户的游戏。最糟糕的事情之一是不得不更换烧坏的激光管。他们确实有自己的生活，但在这种情况下，水冷却系统的过滤器堵塞，管煮自己。两次。

![](../Images/4971e80da218dfc2b9039023a7cfa2a3.png)

Flow sensor shown in the upper right.

该流量传感器现在充当与激光使能线的互连。从一根丙烯酸棒开始，[NixieGuy]加工出一个磁性塞子的中心孔，然后铣出三个通道，让水绕过它。杆的每一端都在车床上转动，以与水冷系统的塑料管连接，并在外部铣出一个用于簧片开关的槽。

演示视频如下。你可以看到，当水流动时，它将磁性挡块向上推(克服重力),与簧片开关接合，使激光器工作。如果有东西阻碍水的流动(即使水泵仍在运行)，激光器将被禁用，并(希望)防止未来的管损失。
