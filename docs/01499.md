# 新型变形聚合物努力工作，尽情玩耍

> 原文：<https://hackaday.com/2016/02/16/new-shape-shifting-polymer-works-hard-plays-hard/>

罗切斯特大学的一个研究小组开发了一种新的聚合物，它具有一些令人惊奇的特性。它可以被拉伸或处理成新的形状，并且在加热之前会保持这种形状。像这样的变形聚合物已经存在，但这种聚合物很特别:当被人体的热量触发时，它可以回到原来的形状。哦，它还能举起 1000 倍于自身质量的物体。

该小组的领导人，化学工程教授[Mitch Anthamatten]对这项发明的可能性感到兴奋。当材料被拉伸时，会产生应变，使链变形并引发结晶。这种结晶是它保持新形状的原因。一旦加热，晶体就会破裂，聚合物就会恢复到原来的形状。这些特性意味着一些生物医学应用，如缝合线和人造皮肤。它也可以用于定制服装或可穿戴技术。

变形过程在聚合物中产生弹性能量，这意味着它可以在弹回到正常状态时做功。分子连接物的小心应用使得该组能够拨入结晶开始分解的所谓熔点。[Anthamatten]在休息后的一个视频中解释了该材料的特殊属性。另一个视频展示了这种聚合物的一些与工作相关的应用示例——一根拉长的绳子可以将一辆玩具卡车拉上斜坡，或者压碎一个干燥的豆荚。
