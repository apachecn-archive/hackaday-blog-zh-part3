# 一只像章鱼一样温柔抓握的机器人手臂

> 原文：<https://hackaday.com/2017/04/19/a-robot-arm-with-the-tender-grip-of-an-octopus/>

如果你曾经尝试过机器人手爪，你就会知道，虽然模拟人类用拇指和食指抓握的能力很容易，但要利用触摸反馈捕捉抓握的细微差别，以提供仅够抓握和保持物体所需的力是极其困难的。作为一个人，你可以用拿起棒球或板球的同一只手拿起一个精致的蛋壳而不打破它，但让你的机器人做同样的事情是一个工程挑战。

机器人手爪是一件锻炼了 Festo 员工头脑的东西，他们得出的解决方案既漂亮又新颖。他们制造了一种基于章鱼触手动作的抓手，尽管与实物的肌肉不同，他们制造了一种充气时向内弯曲的硅树脂管。它的内表面覆盖着章鱼状的吸盘，其中一些可以通过真空激活。其结果是一个非常有能力和多功能的手爪，由于其柔软的结构，非常适合在机器人和人类交互的环境中使用。

他们上传了一段精美的视频，展示了这款设备的运行情况，我们把它放在了休息时间的下方。像抓取卷起的杂志或塑料瓶这样的任务，比传统的抓取工具更费力，但却能完美完成。
