# 树莓派为未来的桌子提供动力

> 原文：<https://hackaday.com/2015/10/21/raspberry-pi-powers-the-desk-of-the-future/>

最近的科幻电影总是展示人们坐在时尚的半透明办公桌前，里面有各种奇怪而奇妙的小工具。我们的桌子看起来像你祖父坐过的东西。[弗雷德里克·范登堡]一定看过这些电影，所以作为比赛的一部分，他决定建造未来的书桌。

这张桌子既是一个家具项目，也是一个电子项目，但它确实有一个树莓 Pi，一个废弃的笔记本电脑 LCD，嵌入式触摸传感器和 led，一个无线充电器和一个内置的音响系统。此外，它还使用一个 Gertbot 和一些步进电机来提升和降低桌面上的屏幕(观看下面的视频，看看是什么样子)。

[Fredrick]用 Python 编写了桌子的主要功能。我们不禁想到了使用易于编程的桌面可以做的所有事情:显示股票报价(或体育比分)、通知电子邮件或其他事情。虽然对于一个简单的用户来说，添加这些东西看起来并不简单，但是如果你是一个方便的程序员，它们看起来似乎触手可及。

我们以前见过一些桌子，但没有一张像这样的。我们不禁想知道你是否能给[弗雷德里克的]已经令人印象深刻的办公桌增加一些[少数派报告式的优点。](http://hackaday.com/2014/10/24/huddlelamp-turns-multiple-tablets-into-single-desktop/)
