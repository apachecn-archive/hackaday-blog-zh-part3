# 一个叫伯努利的家伙和导管螺旋桨设计

> 原文:[https://hack aday . com/2015/09/28/a-guy-named-Bernoulli-and-ducted-propeller-designs/](https://hackaday.com/2015/09/28/a-guy-named-bernoulli-and-ducted-propeller-designs/)

想了解为什么[导管螺旋桨同时提高推力和效率](http://www.youtube.com/watch?v=Cew5JF8q6eY)？嗯，我们已经为你准备好了课程和老师。[布鲁斯·辛普森]并不是一个家喻户晓的名字，但我们打赌你们大多数人已经知道他是谁了。他是来自新西兰的家伙，在 2003 年开始制作一个 5000 美元的 DIY 巡航导弹——他声称如果新西兰政府没有阻拦，他会成功的。如今，布鲁斯将大部分精力放在了他的两个 YouTube 频道上，即 [RC Model Reviews](http://www.youtube.com/user/RCModelReviews/) 和 [XJet](http://www.youtube.com/user/xjet) 。

回到手头的课程:在休息后的视频中，他出色地解释了管道壁如何阻止螺旋桨的高压区向低压区移动。你会在喷气式飞机的机翼上看到类似的东西。在机翼的末端会有一个小的垂直尾翼，很像一个导管，它的工作是物理上分开两个区域，防止翼尖涡流效应。这反过来又提高了效率。[Bruce]有一些其他很棒的 RC/model 类型的视频，所以如果你喜欢你看到的，你可能会想要订阅。或者你可以了解更多关于 [DIY 巡航导弹传奇](http://www.aardvark.co.nz/pjet/cruise.shtml)的信息。
