# 用于拉直 DIP 芯片和非拉直电阻的 3D 打印工具提示

> 原文：<https://hackaday.com/2018/05/10/3d-printed-tool-tips-to-straighten-dip-chips-and-unstraighten-resistors/>

观看某人组装一个工具包是一个很好的方式，可以看到一些你以前可能没有遇到过的工具，也许还可以学到一些新的技巧。在[Marco Reps]最近[制作 GPS 同步谢妮时钟套件](https://www.youtube.com/watch?v=ge_9CNiZZ_A)期间，我们发现了一些方便的工具，你可以为自己的工作台进行 3D 打印。

刚从工厂出来的双列直插式封装(DIP)芯片的腿稍微分开一点，足以不适合电路板上精心设计的尺寸。你可能习惯于在工作台上用手不精确地弯曲它们。[Marco]更加精致，展示了一个整洁的小弹簧加载工具，只需几次挤压就可以整齐地弯曲蘸料的两侧，使每条腿都保持完美的角度。这里展示的是一个 3D 打印版本，叫做 IC 引脚矫直机，你可以用弹簧和普通紧固件把它扔在一起。

另一个引起我们注意的工具是他用来弯曲金属薄膜电阻引线的工具:“Biegelehre”或引线弯曲工具。您可以看到[Marco's]工具有一个成角度的沟槽，以适应不同的电阻体宽度，并带有阶梯形边缘，以适应标准 PCB 封装间距。我们打赌你经常使用相同的电阻体，所以通过对每种宽度使用单一工具的，3D 打印变得更容易。如果你真的一定要复制[马可]正在使用的，我们确实找到了[另一个型号](https://www.thingiverse.com/thing:168412)更接近他的。

至于新花样，套件组装中有很多值得欣赏的小细节。[Marco]用剪刀剪去支撑材料，然后在平面上用砂纸打磨，清理木板。不是所有的谢妮管都是完全一致的，所以需要一些手动调整。总的来说，他的焊接技术是我们见过的最好的。像往常一样，一路上有很多(马可的)独特的幽默可供欣赏。

我们心中有一个温暖的地方，你可以在“ole 3D 打印机”上制作简单的工具。查看[PCB 虎钳](https://hackaday.com/2018/01/24/printed-it-rubber-band-pcb-vice/)，一套[球窝辅助手](https://hackaday.com/2018/02/08/printed-lockable-ball-and-socket-helping-hands-plus/)，以及[一套工具包必备](https://hackaday.com/2018/05/07/printed-it-toolbag-essentials/)。
