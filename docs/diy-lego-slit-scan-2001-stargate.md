# DIY 乐高狭缝扫描 2001 星际之门

> 原文：<https://hackaday.com/2015/10/03/diy-lego-slit-scan-2001-stargate/>

[电影制作人智商]有一堆关于制作电影的技术方面的很棒的教程，但是这一集关于[用乐高](http://filmmakeriq.com/lessons/slit-scan-recreating-the-stargate-sequence-from-stanley-kubricks-2001-using-legos/)复制斯坦利·库布里克著名的《2001:太空漫游》是一个黑客的乐趣。

2001 年的星际之门是我们的主角戴夫“对不起戴夫”鲍曼穿越时空进入另一个宇宙并重生为星际之子的漫长而迷幻的一幕。(对，剧情变得有点离奇。)但是星际之门的镜头，以及电影的其他视觉效果，为他们赢得了奥斯卡奖。

你会认识的其他狭缝扫描动画的例子包括《神秘博士》的片头字幕和《星际迷航:TNG》中的曲速驱动效果。

 [https://www.youtube.com/embed/KhRo2WbWnKU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/KhRo2WbWnKU?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



非线性视频剪辑之前的日子效果是怎么做的？一次一帧，相机在轨道上，向后面有移动插图的狭缝前进。作为演示，[约翰·赫斯]用乐高制作了他自己的狭缝扫描机。你真的应该看完整集，但是如果你只是想直接进入乐高黑客游戏，请点击这里。

为了以防万一，你想精确地重新制作实际的星际之门，[这里是投影在幕布后面的原始图像](http://seriss.com/people/erco/2001/)，是在一台旧的 SGI 计算机上通过逆向技术提取的。
