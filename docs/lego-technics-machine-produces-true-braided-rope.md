# 乐高工艺机器生产真正的编织绳

> 原文：<https://hackaday.com/2016/11/28/lego-technics-machine-produces-true-braided-rope/>

我们和其他人一样喜欢一个好的乐高积木，但是在创造工作机制方面，Technics 把它带到了一个新的水平。而且没人把工艺做到[Nico71]那么远，他的[超快工艺绳编织机](http://www.nico71.fr/lego-braiding-machine/)就是明证。

上一次我们看到[Nico71]的作品，也是[一台乐高工艺制绳机](https://hackaday.com/2016/02/19/rope-braiding-machine-built-entirely-from-lego-technics/)。当时，我们称之为“编绳机”,并在评论中受到指责，因为绳股只是被拧成了最终产品。[Nico71]必须牢记这一点，因为目前的产品是真正的编织跳线。这种技巧是通过不附在两个反向旋转的三辐轮子上的飞行梭来实现的。梭子通过清扫臂在两个轮子之间转移，每个梭子在转移到另一个轮子之前都与一个轮子一起旋转一整圈。每个梭子的线在两个固定线轴的线周围形成一个缠绕的数字 8，结果是一个五股编织线。整台机器看起来令人着迷，即使在高速下，机械装置也如丝般平滑。这似乎也是一个比之前更简单的设计。

你不得不把它交给像[Nico71]这样的建设者，他们在技术世界的限制下工作，却能制造出令人着迷的机器。那些在他们的构建中利用技术平台的人可以想出非常好的东西，像[这个用于音乐机器的纸带阅读器](https://hackaday.com/2016/07/06/paper-tape-drive-for-a-live-performance-music-box/)。
