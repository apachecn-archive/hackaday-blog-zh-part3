# MIDI 绘画用钢琴键盘绘画

> 原文:[https://hack aday . com/2017/03/25/midi-drawings-paint-with-piano-keyboards/](https://hackaday.com/2017/03/25/midi-drawings-paint-with-piano-keyboards/)

音乐家[Mari Lesteberg]正在创作绘画音乐。或者她在制作描绘音乐的图片。很复杂。查看[视频](https://www.youtube.com/watch?v=hofzp4fbKUU)(嵌入下方)，你会明白我们的意思。结果是一半是中国卷轴画，一半是乐谱，它们很好地结合在一起。

许多 MIDI 录音机/播放器使用[钢琴卷帘窗](https://en.wikipedia.org/wiki/Piano_roll)作为输入模式——时间滚动到一边，几个照亮的像素代表弹奏的音符。她也用这些像素来绘画:卡通河上的波浪发出上下的琶音。那是一个(音乐)黑客。她不是唯一一个画迷笛的人。你会在 reddit 上找到更多[。](https://www.reddit.com/r/MidiDrawings/)

当然，人们也可以对无声像素做同样的事情——只需设置一个音量为零的音符——但这是欺骗，一点也不好玩。据我们所知，你可以听到滚动图像中的每个音符。对于黑色 MIDI 种类的音乐来说，情况就不一样了，这种音乐的目的是在尽可能短的时间内压缩尽可能多的音符。在我们听来，它没有那么美，但味道却是无法解释的。

令人惊讶的是，在过去的几年里，我们看到了古老的钢琴卷帘门技术的巨大变化。当然，由于钢琴卷纸本质上是乐器的穿孔卡，我们不应该对这一切都有可能感到太惊讶。的确，我们有点惊讶新的艺术可能性仍然存在。有人见过打孔卡图纸是可执行代码吗？或者嵌入可播放图像的实体钢琴卷？
