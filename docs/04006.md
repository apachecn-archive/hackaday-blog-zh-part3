# 新的锂电池技术需要勇气

> 原文:[https://hack aday . com/2016/11/01/new-lithium-battery-technology-takes-guts/](https://hackaday.com/2016/11/01/new-lithium-battery-technology-takes-guts/)

研究人员已经建造了一个锂硫电池的原型，一旦完善，其能量密度将是目前锂离子电池的五倍。英国和中国的研究人员从肠道中获得灵感，以克服电池结构中的问题。

在你的肠道中，被称为绒毛的小毛发状结构增加了你的身体用于从食物中吸收营养的表面积。在新的锂硫电池中，研究人员使用细小的氧化锌线形成一层具有绒毛状结构的材料。这些绒毛覆盖着一个电极，可以在活性材料断裂时捕获它们，使它们继续参与产生电流的电化学反应。

锂硫电池并不新鲜(事实上，它们在 2008 年被用于一架打破多项记录的太阳能飞机上)，但是这项新技术可能会使它们更加实用。你可以在下面看到一个关于普通锂硫电池的视频，以及更多关于这项研究如何提高技术水平的信息。

典型的锂离子电池包含石墨和锂钴氧化物。带正电的锂离子从阴极来回移动，通过电解质进入阳极。由于石墨中的碳原子只能携带(最多)一个锂离子，这就为你从电池中获取的能量设定了理论上限。

硫和锂通过[多电子转移机制](https://en.wikipedia.org/wiki/Lithium%E2%80%93sulfur_battery)发生不同的反应。这就是为什么硫可以提供更高的理论容量。然而，随着电池经历几次充放电循环，一些硫会脱离，导致电池逐渐失去活性物质。氧化锌绒毛倾向于捕获这些碎片，这减缓了电池的退化。

villi 提高了原型电池的充放电次数，但仍无法与传统的锂离子电池相媲美。另一方面，新电池不需要经常充电。
