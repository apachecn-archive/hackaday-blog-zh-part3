# 简单听觉放大器

> 原文：<https://hackaday.com/2016/11/08/simple-hearing-amplifier/>

助听器可能比你想象的更高科技。它们很小。他们必须经历很多麻烦来阻止反馈。他们需要较长的电池寿命。好的放大器将它们的放大与你听力损失的倒数相匹配(只放大你听不到的波段)。

【NotionSunday】组织了一个[听觉放大器项目](https://www.youtube.com/watch?v=bAZbHFOIRfg)，该项目可能没有达到许多设计标准。然而，由于 3D 打印的外壳，它看起来相当不错。该设备使用双运算放大器来增强两个麦克风的输出，并将其馈送到传统耳机。

这个设备是点对点连接的，可能只有口袋大小。运算放大器电路很简单。我们可能会考虑使用 [LM386](https://hackaday.com/2011/05/01/lm386-altoids-tin-amp/) 或其他集成音频放大器模块来获得更好的性能，而不会增加器件数量。

从技术上来说，像这样的设备并不是真正的助听器，它们是“个人声音放大器”, FDA 警告人们不要用它们代替助听器。如果你想要一个合适的助听器，你可以[看看这个](https://hackaday.com/2013/12/15/diy-hearing-aid/)，但是你需要一个大耳朵，因为它里面有一个 Arduino。
