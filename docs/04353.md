# Thwomp 在复古游戏上丢砖头

> 原文:[https://hack aday . com/2016/12/03/thwomp-drops-brick-on-retro-gaming/](https://hackaday.com/2016/12/03/thwomp-drops-brick-on-retro-gaming/)

[极客制造]想要应对使用水泥建造 USB 集线器的挑战。作为《马里奥兄弟》的粉丝，一个合适的敬意就是[用水泥](https://www.youtube.com/watch?v=I6Q48aZD7j4)建造一个复古游戏控制台，看起来就像你最喜欢的马里奥粉碎敌人。凭借一个树莓 Pi Zero 和一个嵌入其中的 USB 集线器，[极客制造]将大型水泥块的马里奥宇宙角色——th womp——赋予了生命。

[极客制造]经历了五次迭代才找到一个正常工作的。最初，他尝试使用 3D 打印模具；水泥粘在塑料上，破坏了脸上的水泥。然后他转而使用液体橡胶模具(在打印出 Thwomp 的正面模型以用于创建模具之后)。但是模具的泡沫板框架支撑不住，所以[Geeksmithing]加了一些木头来稳定东西。不幸的是，橡胶粘在泡沫板和 3D 模型上，使得取出模型极其困难。

![Like [Han] in carbonite, that's a Rapsberry Pi Zero being encased in cement](../Images/96534f2e1c3d9842be7e7fcc8b4f136a.png)

就像【韩】在碳化，那是一个覆盆子零被水泥包裹着

接下来就是正规的硅胶模具材料了。他没有足够的硅橡胶来覆盖模型，所以他添加了一些木材作为填充物来提高液体的水平。他还翻转了模型，这样他至少可以得到脸部的细节。他找到了一些其他的硅胶，并用它来填充模具的其余部分。尽管不同的硅胶，这个模具工作。然而，他用来防水覆盆子酱的胶带却没有。他又试了一次，这次他用了热胶水——很多热胶水！–使码头防水。这个演员更好，他能够启动 Pi，但在几场比赛后，他的控制器停止工作。他敲开水泥查看 Pi，意识到热胶上的一个小洞导致泄漏，使 Pi 上的 USB 端口短路。最后一次，他想，这一次他用液体绝缘胶带给 Pi 防水。

最后的铸造工作和油漆后，[极客锻造]有一个成品水泥 Thwomp 控制台，可以玩复古游戏。他错过了 USB Hub 挑战赛的截止日期，但它仍然是一个很棒的控制台，他的视频中有很多关于他在构建过程中出错(和正确)的细节。YouTube 上有一个关于其他参赛作品的[很棒的播放列表](https://www.youtube.com/playlist?list=PLtOpzFLh4wxwkOPFP7-Euqf8ZpEhvOrsd)，看看它们和下面的【极客制造】的视频吧！
