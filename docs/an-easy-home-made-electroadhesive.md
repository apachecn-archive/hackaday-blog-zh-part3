# 一种简易的自制电胶

> 原文：<https://hackaday.com/2016/10/12/an-easy-home-made-electroadhesive/>

曾经有一段时间，如果你想爬上一堵墙，你必须把摄像机转到一边，然后给(亚当·威斯特)或(莱昂内尔·里奇)打电话。除非你是一只壁虎。它们身上的细小刚毛和分子内力使它们能够轻松地爬上墙壁和天花板。

你可能已经看到合成壁虎一样的粘性表面使用相同的效果。一些令人印象深刻的爬壁机器人已经使用了这些材料，尽管它们都有相同的问题。壁虎粘连关不掉。令人高兴的是，对于不愿意花费额外的力量来分离一只脚的机器人爬壁者来说，有一种替代方法。电胶使用静电力将一个保持在高电位的板附着在一个表面上，今天的专题视频是[【卡特·赫德】自制的电胶面板](https://www.youtube.com/watch?v=mbKz46bS_Go) (YouTube)。

他引用了[这篇论文](http://www.mdpi.com/1996-1944/7/7/4963)和[这篇技术描述](http://www.hizook.com/blog/2009/08/06/electroadhesive-robot-climbers)作为对他设计的影响，两个铝箔电极夹在塑料片和胶带之间。他将 Emco 直流-DC 转换器中的 6 千伏电压施加到他的盘子上，就像变魔术一样，它粘在了他的干墙上。当然，事情没那么简单，他试了几个表面，才找到它能粘住的那一个。附着力完全在控制之下，这样一个简单的装置表现得出奇的好。令人遗憾的是，Emco 转换器并不便宜，但这是一种非常高效的产品，他只需要在低压侧安装几个 AA 电池。

他的完整描述在休息下面的视频里。
