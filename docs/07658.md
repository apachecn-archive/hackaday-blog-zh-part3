# 3D 打印螺旋卫星通信馈送

> 原文:[https://hack aday . com/2017/11/09/3d-printed-helical-satcom-feed/](https://hackaday.com/2017/11/09/3d-printed-helical-satcom-feed/)

随着几年前 RTL-SDR 项目流行的廉价软件无线电的出现，卫星通信现在甚至在最普通的黑客的预算之内。花 20 美元，你就可以得到一个 USB SDR 模块，它能够接收来自任何数量的地球同步卫星的信号，但你需要一个比兔子耳朵更强大的东西来接收来自 22，000 英里以外的信号广播。

[![](../Images/04cad667244e9a6e1d2aee98186e7968.png)](https://hackaday.com/wp-content/uploads/2017/11/3dsatcom_detail.jpg) 建造一个支持卫星的天线并不一定很难，但要做好这件事确实需要一些神秘的魔法和数学；一些可以吓跑新爱好者的东西。[但是通过使用 3D 打印的心轴，【Tysonpower】已经开发出一种馈电，你可以在标准的碟形天线](https://www.thingiverse.com/thing:2627680)上制作和安装，而不必参加天线理论的速成班。[Tysonpower]报告说，馈送的中心频率为 1550 MHz，可以很好地接收 Inmarsat、AERO 和 HRPT 信号。

馈送的 3D 打印核心中的通道确保插入的电线具有正确的长度，并且处于最佳接收的完美位置。你需要做的只是打印内核，用导线包裹，然后将末端焊接到一个接地层上的连接器上，这个接地层只不过是一片铝片。[Tysonpower]甚至好心地制作了一个支架，可以让你将这个信号栓接到标准的卫星天线上。

我们之前已经讨论过使用 RTL-SDR 接收 Inmarsat 传输的[、Outernet 项目的](https://hackaday.com/2015/08/21/decoding-satellite-based-text-messages-with-rtl-sdr-and-hacked-gps/)[硬件](https://hackaday.com/2017/06/20/testing-the-outernet-dreamcatcher-sdr/)，这两者都是这种天线的绝佳应用。
