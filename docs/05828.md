# 被动测量步态速度以诊断疾病

> 原文：<https://hackaday.com/2017/05/06/measuring-gait-speed-passively-to-diagnose-diseases/>

你可能没有意识到，一个人走路的速度是整体健康的重要指标。我们都本能地知道，当感冒或流感来袭时，我们会明显滞后，但监测步态速度可以帮助诊断过多的慢性疾病和状况。像 Fitbit 这样的可穿戴设备将是监测步态速度的一种方式，但麻省理工学院的计算机科学和人工智能实验室认为还有更好的方式:[一种被动测量步态速度的无线设备](http://news.mit.edu/2017/dina-katabi-csail-team-develop-wireless-system-to-detect-walking-speeds-0501)。

CSAIL 的传感器被称为 [WiTrack](http://witrack.csail.mit.edu/witrack-paper.pdf) (PDF)，是一个安装在墙上的牌匾，可以很容易地隐藏为一幅画或一面镜子。它发出大约 5-7 GHz 的低功率 RF 信号，以实时执行 3D 运动跟踪。WiTrack 传感器在这些频率下的分辨率约为 8 厘米。通过他们的 [WiGait 算法](https://people.csail.mit.edu/cyhsu/papers/wigait_chi17.pdf) (PDF)，由【陈-许玉】领导的 CSAIL 团队不仅能够测量整体行走速度，还能够测量步幅。事实证明，这对于预测帕金森氏症等疾病的发病至关重要，帕金森氏症在疾病早期有一种非常典型的缓慢步态。ALS 和多发性硬化症等其他疾病引起的运动障碍也可以被识别出来。

WiTrack 建立在[Hsu]之前的[穿墙射频跟踪](http://hackaday.com/2015/10/29/using-rf-to-see-through-walls/)工作的基础上。很高兴看到一种新颖的技术越来越接近一种有用的产品，我们将拭目以待，看看这种技术会走向何方。
