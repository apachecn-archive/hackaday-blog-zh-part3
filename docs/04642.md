# 1/3 比例的增压燃油喷射 V10 发动机

> 原文：<https://hackaday.com/2017/01/06/supercharged-fuel-injected-v10-engine-at-13-scale/>

近三年的制作，看看这个 1/3 比例的 V10 发动机的原始动力和精度。

进来在 125 立方厘米的位移，[基思哈洛]的燃料喷射杰作是不太远的一些摩托车发动机的大小。我们怀疑当地的 Vespa 俱乐部会把它视为合法的 mod，但我们很想看到它。[Keith]的构建日志是一长串论坛帖子，但从我们所看到的来看，除了燃油喷射系统之外，似乎每个部分都是手工制作的。甚至火花塞帽也是在[Keith]的店里定制注塑成型的。而且似乎没有使用 CNC 甚至那些进气总管和增压器的转子也是用手工轧机从铝制成的。单单排气管头就是直接的艺术作品。这里有惊人的工作量，这就引出了一个问题:为什么？这种情况下的答案显然是，“因为他可以。”

很少有建筑能与这里展示的工艺水平相提并论。点击弹簧骨架时钟出现在脑海中，但对于模型引擎的构建，我们必须指出[【基思】早期的 1/4 比例 V8 引擎](https://hackaday.com/2013/10/11/a-48cc-v8-with-injection-and-opensource-ecu/)。我们会赶紧补充说，正如[Keith]花了那么多时间来建造这些机械艺术作品，他可能也花了同样多的时间来记录它们并回馈他的社区。我们都可以从中吸取教训。
