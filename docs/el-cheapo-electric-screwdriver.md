# El Cheapo 电动螺丝刀

> 原文：<https://hackaday.com/2015/10/26/el-cheapo-electric-screwdriver/>

如果你有一些业余爱好伺服躺在周围，这里有一个黑客，让你回收他们，把他们好好利用。【Kedar Nimbalkar】拿了一个[微型伺服，改装成电动螺丝刀](https://www.youtube.com/watch?v=W56Ege6rqBc)。这很简单，值得一个简短的视频显示他是如何做到的。

他首先打开一个 9G 微型伺服系统，并拆除电子设备。所有需要的是 DC 发动机和齿轮。两条电机线通过极性反转开关直接连接到电池，使电机可以双向转动。伺服喇叭被切割成合适的尺寸，这样它就可以紧配合在螺丝刀插座内。大量的胶水被用来确保它固定不动。然后将喇叭连接到修改过的伺服系统上，准备好接受可互换的位。关闭伺服系统前的最后一个方法是通过切断传动齿轮上的制动器将伺服系统转换为连续旋转。

他从零开始构建电源，使用 18650 锂电池、5V USB 充电器、允许方向控制的 DPDT 开关和启动螺丝刀的按钮。一对背靠背连接的发光二极管作为方向指示器和一些局部照明。

有很多的范围来随机应变，做不同的事情，但使用一个方便的电动螺丝刀未使用的伺服系统的基本前提是非常整洁。
