# 伺服系统上的底漆击中所有的基础

> 原文：<https://hackaday.com/2016/10/30/primer-on-servos-hits-all-the-basics/>

对于经验丰富的黑客来说，伺服系统是非常基本的东西。但是每个人都必须从某个地方开始，肯定有人会从这本关于伺服内部的入门书中受益。谁知道呢——也许即使是老手也会从新的角度学到一些东西。

【GreatScott！]在过去几年里已经建立了一个全面的基础电子视频库，涵盖了从使用万用表到 Arduino 编程的所有内容。最后两部分深入机电领域，对步进电机和下面的伺服视频进行处理。他涵盖了现代钢筋混凝土型伺服的本质，在一个清晰和迷人的风格，使新手很容易理解如何 PWM 信号可以转化为 180°扫描的位置变化。他展示了如何用 Arduino 直接控制伺服系统，还包括一个简单的基于 555 的控制器电路。快速浏览一下将任何伺服系统转换为连续旋转所需的模块，就可以结束这段视频。

如果【GreatScott！]的视频激起了你更多的欲望，一定要看看[Richard Baguley]的[深入伺服](http://hackaday.com/2016/03/24/hackaday-dictionary-servo-motors/)。当你准备将新发现的知识付诸实践时，也许一个不错的项目是[将一个业余爱好的伺服系统转换成一个线性致动器](http://hackaday.com/2016/03/31/3d-printed-case-turns-servo-into-quality-linear-actuator/)。
