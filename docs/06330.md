# Hackaday 奖参赛作品:通过生物反馈进行中风康复

> 原文:[https://hack aday . com/2017/07/10/hack aday-prize-entry-stroke-recovery-through-biological feedback/](https://hackaday.com/2017/07/10/hackaday-prize-entry-stroke-rehabilitation-through-biofeedback/)

普渡大学魏尔东生物医学工程学院的学生创造了 [ExoMIND](https://hackaday.io/project/25455-purdue-exomind-glove) ，这是一种 Arduino 供电的手套，通过跟踪患者经历的运动范围来帮助中风患者恢复。

手指、手腕和前臂中的一组 7 个加速度计跟踪患者用那只手经历的运动范围。手背上作为参考的加速计。同时，一个 EMG 传感器与一个导电纤维套一起工作来测量肌肉活动。用户遵循由交互式软件程序发出的一系列指令，允许系统在治疗开始时测试患者的活动范围，并记录在治疗结束时是否有任何改善。物理治疗师使用该数据来个性化治疗计划。互动程序还提高了患者自我指导锻炼的可能性，ExoMIND 告诉他们如何调整自己的运动，以获得最大的体验。

作为该大学 [MIND 生物医学工程俱乐部](http://web.ics.purdue.edu/~mind1234/website/)的一部分，ExoMIND 原型由分别专注于电子、材料和编程的三个跨学科团队设计。
