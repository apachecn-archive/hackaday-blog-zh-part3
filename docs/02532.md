# Hackaday 奖参赛作品:在黑暗的阴谋中发光

> 原文:[https://hack aday . com/2016/06/02/hack aday-prize-entry-glow-in-the-dark-plotting/](https://hackaday.com/2016/06/02/hackaday-prize-entry-glow-in-the-dark-plotting/)

我们喜欢大显示器，无论是巨大的 LED 显示器、数控白板，还是巨大的激光投影仪。为了他的 Hackaday 奖参赛作品，[nilo]想出了一个简单的方法来创建一个巨大的图形显示。它可能只在晚上工作，但 Glowboard 绘图仪仍然非常酷。

[nilo]的显示器只是缠绕在两个滚筒上的黑色薄片中的一个非常大的发光体，由步进电机控制。通过运行马达，黑暗中的空白发光慢慢滚动。

为了给这个显示器增加一些色彩，[nilo]在显示器的一端固定了 64 个紫外发光二极管。通过以特定的间隔和特定的强度点亮这些发光二极管，一个幽灵般的绿色发光板图像出现在显示器上。有用吗？差不多和其他巨型 LED 矩阵一样多。很酷吗？差不多和其他巨型 LED 矩阵一样多。

现在，[nilo]正在 Glowboard 绘图仪的 mk. 2 上工作，升级控制系统，将 UV LEDs 的数量增加到 128 个。它可能看起来没什么用，但它仍然非常酷，并且是今年 Hackaday 奖 everything Goes 部分的优秀参赛作品。
