# 自制不倒翁扩散清晰的发光二极管

> 原文：<https://hackaday.com/2015/11/05/home-made-tumbler-diffuses-clear-leds/>

当你有大量的透明 LED，但是你想要或者需要一个项目的扩散模型，会发生什么？你可以再去买一些，但是[泰勒]说不行！去商店拿些废料，给自己做一个[沙玻璃杯来分散你已经有的](http://discspace.org/how-to-make-a-ton-of-clear-leds-into-diffuse-leds/)LED。

不倒翁机制类似于岩石不倒翁，但它是由一些木头，一些杆，一个意大利面条酱罐和一个 DC 发动机制成的，这是我们已经拆开的机器类型中可用的。一旦建成，干细沙和发光二极管装入罐子，并设置翻滚几个小时。

一旦完成，就很容易找出现在扩散的 LED，它们比直接从末端射出大部分光线的透明对应物发出更均匀的光。虽然这需要时间，但这比手动分散 LED 灯要容易得多，如果你需要的不只是几个，这是一个巨大的劳动力节约。
