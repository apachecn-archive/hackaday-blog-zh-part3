# Hackaday 奖参赛作品:太阳能 WiFi 漫游车夜间漫游

> 原文：<https://hackaday.com/2016/08/08/hackaday-prize-entry-solar-wifi-rover-roves-at-night/>

[TK]对他的遥控汽车项目有一个长远的目标——使它能够在白天用太阳能充电，晚上在远程互联网控制下漫游。它就像是美国宇航局的好奇号漫游车的微型后院版[。](http://www.nasa.gov/mission_pages/msl/index.html)

现在，他有一个树莓派 Zero 和一个摄像头，让他们通过 WiFi 控制机器人。他看起来很享受在家里驾驶它。看看下面的视频(crashy)遥控操作。

我们迫不及待地想看看太阳能发电是否有可能实现(嘻嘻！)作为本车的选装配置。通过 3G 蜂窝调制解调器连接它的最终计划在未来仍然是关闭的，可能会比现在需要更多的树莓 Pi 的智能。但是我们喜欢长期运行的自动驾驶汽车的想法，所以我们支持你，[TK]！
