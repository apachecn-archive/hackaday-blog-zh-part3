# 获取 beaglebone 的精彩音频

> 原文：<https://hackaday.com/2016/02/19/aquire-awesome-audio-for-beaglebone/>

[Henrik Langer]将他的[强大的音频采集和输出板](https://hackaday.io/project/9634-linux-based-low-latency-multichannel-audio-system)放到了 Hackaday.io 上，我们认为应该向您指出这一点。这是几年前曾经是专业音频的项目之一，但今天是可行的(和负担得起的)DIY:双立体声输入和四个(！)立体声输出，采样速率均为 24 位，最高 192 kHz。它被配置成一个 BeagleBone cape，并带有一个定制的 Linux 发行版。

你会用这种东西做什么？它本质上是一个放在你口袋里的录音室，附带一台电脑。视频(链接在广告下方)演示了如何将该设备用作实时立体声延迟效果单元，但这只是利用了一个通道。在效果、录音以及各种比 CD 质量好得多的声音合成和回放可能性之间，这是一个开放式的音频游乐场。

所有这些都来自 ADI 公司一款出色的 DAC/ADC 芯片的分线板:AD1938。我们希望在我们的桌面上有一个这样的东西。查看[【Henrik】的 GitHub](https://github.com/ctag-fh-kiel/ctag-face-2-4) 中的 PCB 和构建说明和 BOM 以及你需要开始的所有东西。非常好的工作！
