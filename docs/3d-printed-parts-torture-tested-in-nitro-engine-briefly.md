# 3D 打印零件在 Nitro Engine 中经过严格测试——简要说明

> 原文：<https://hackaday.com/2018/02/22/3d-printed-parts-torture-tested-in-nitro-engine-briefly/>

增材制造在短时间内取得了长足的进步，你可以用一些高端 3D 打印机生产的零件在耐用性方面可以与机械加工金属相媲美。但消费级技术通常落后于好东西，所以你还没有办法在普通打印机上 3D 打印内燃机零件，对吗？

事实证明，如果引擎和你的期望都得到适当的缩放，你至少可以 3D 打印连杆。[Johnny k90]喜欢他的微型硝基发动机，我们之前已经看到他用它来驱动[旋转工具](http://hackaday.com/2017/05/30/nitro-powered-rotary-tool/)和[手钻](https://hackaday.com/2017/01/16/cordless-drill-uses-no-electricity/)。因此，拆开一台完美的发动机，用 PETG 图案替换铝制连杆有点令人惊讶。这样一个简单的部分，设计过程是非常容易的，当并排放置时，打印看起来像是原件的合理复制。但有明显的差异，如压配合青铜轴承和原始部件的曲柄和手腕端的油口，更不用说沿着塑料部件的均匀厚度，而不是原型中沿着轴的浮雕。

尽管如此，杆被安装到一个带有透明塑料盖的引擎中，让我们观察旋转的比特，直到不可避免的失败时刻，这可以在下面的视频中看到。对我们来说，似乎无法颈缩杆轴可能不是一个好主意，但主要的故障模式是轴承，或缺乏轴承。尽管如此，我们还是惊讶于该部件持续的时间，我们不禁想知道复合连杆的表现如何。
