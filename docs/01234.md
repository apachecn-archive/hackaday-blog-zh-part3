# 瑞士计划利用风筝发电

> 原文：<https://hackaday.com/2016/01/16/swiss-project-looking-to-harness-kite-power/>

瑞士给我们买了很多东西:布谷鸟钟，有洞的奶酪，还有..风筝动力？这是一个瑞士项目的想法，该项目试图利用日内瓦湖和阿尔卑斯山之间的常规风的能量。该组织希望制造在离地 150 米高空飞行的大型风筝，在地面上安装发电机和其他部件。风能转化为电能的方式很有趣:风筝被风拉起，盘旋上升，拉动驱动发电机的电缆。一旦它达到最大高度，风筝就会被调整，这样它就会下沉到一个较低的高度，然后风筝会再次被调整以抓住风并爬升。

这是一个迷人的想法:通过控制风筝，该系统可以按需发电。当然，只要有风，但在瑞士的这个地区，这不是问题，因为风是可以预测的。它不需要像风力涡轮机那样多的永久性基础设施，风筝比涡轮机更有吸引力。这让我们想知道像这样的系统是否能适应更小的规模:你能为多风地区的徒步旅行建立一个便携式或离网系统，这样可以给电池充电吗？

项目网页从 2013 年开始就没有看到任何更新，但是[研究项目似乎仍然活蹦乱跳](http://control.ee.ethz.ch/index.cgi?page=sada;action=details;id=670)。有人知道任何细节或胡乱猜测吗？
