# Hackaday 奖参赛作品:用 Keymu 完成的小游戏

> 原文：<https://hackaday.com/2017/06/30/hackaday-prize-entry-gaming-done-tiny-with-keymu/>

在 2016 年 Hackaday 超级大会上推出的世界上最小的游戏男孩色[是一件艺术品](http://hackaday.com/2016/11/28/tiniest-game-boy-hides-in-your-pocket/)。这个微型游戏控制台启发了[c.invent]去创造自己的游戏手持设备。他在 hackaday.io 上的 Keymu 项目描述了一款[开源、钥匙链大小的游戏手持设备](https://hackaday.io/project/21453-keymu-open-source-keychain-sized-gaming-console)，其开发者声称*真的是世界上最小的*。他是怎么把它变小的？这是一个微型游戏机高级 SP，它可以折叠在一个方便的蛤壳式外壳中。

虽然他是圆周率的粉丝，但[c.invent]觉得圆周率 Zero 对于他心目中的钥匙链大小的手持设备来说太大太笨重了。只有英特尔爱迪生足够紧凑。他首先制作了一个定制的 PCB，上面有一个连接器，用于连接 Edison 脆弱的带状电缆，然后添加了一个 1.5 英寸的有机发光二极管显示屏和一个 11.7 毫米的扬声器，所有这些都由 220 毫安时的锂电池供电。[c.invent]还设计了一个定制的折叠式 3D 打印外壳，可以保护 Keymu 的电子设备免受钥匙和口袋线头的影响。

与迷你游戏男孩颜色不同，[c.invent]旨在创建一个完全成熟的仿真主机。Edison 包含了一个 Linux 发行版，允许它安装 GameBoy Color、GBA、NES 和 SNES 的仿真器。
