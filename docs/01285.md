# 男人射灯

> 原文:[https://hackaday.com/2016/01/24/man-shoots-lamp/](https://hackaday.com/2016/01/24/man-shoots-lamp/)

当你把所有在易贝能买到的便宜东西混合在一起，再加上一点创意和一些 PVC 管，你会得到什么？[Austiwawa]得到一个由玩具枪遥控的台灯，当你射击它时[关闭*并倒下*。你一定要看休息时间下面的视频。](http://www.instructables.com/id/How-to-Make-a-Gun-Controlled-Lamp/)

这不是技术入侵。更确切地说，它是对一堆容易获得的部件的创造性使用，在这里进行一些切割，在那里进行一些剪切，使其工作。例如，[Austiwawa]直接从机架上取下一对遥控发送器和接收器，焊接一些电线以延长 LED 并将其安装在玩具枪内。一个继电器模块控制着灯，并直接插入一切背后的 Arduino。即插即用。

这并不是说这盏灯缺乏技巧。我们特别喜欢用作伺服电机行程终点挡板的螺钉，以及由两个 Ls 制成的制作精美的伺服支架。你无法抗拒摔死效应。[还是可以？](http://hackaday.com/2015/05/12/robo-lamp-is-too-damn-cute/)不过说真的，伟大的项目【Austiwawa】！
