# 中空状态存储器

> 原文:[https://hackaday.com/2017/10/26/hollow-state-memory/](https://hackaday.com/2017/10/26/hollow-state-memory/)

很多人喜欢电子管，要么是因为它们的声音，要么是因为它们的收藏价值。[Uniservo]喜欢古怪的管子。他最近的视频(见下文)展示了一个 rade chon——一个计算机内存管。显然，直到最近，这种电子管才被广泛应用于雷达设备中，与威廉姆斯电子管有些相似之处。

显像管实质上是一个阴极射线管，它照亮一片云母片或另一种电介质，而不是荧光屏。电介质具有与之接触的细网格。通过在云母上沉积电荷，电子管可以存储模拟值。理论上，电子管可以存储大约 16k 比特的信息，但实际上，分辨率要低一些。

电荷将持续几百毫秒，存储系统将不得不刷新试管内容，否则试管将丢失其数据。对于二进制存储，表面得到负电荷，或零电荷。一些电子管使用电容耦合到带电表面，而另一些则从表面的前面收集二次电子发射。

我们在[[nixie bunny 的](http://www.nixiebunny.com/)网站上找到了一张 Radechon 数据表的扫描图。在冷战基础设施网站上，有一张野兽的 [RCA 广告](http://coldwar-c4i.net/SAGE/EE0858-BC.html)的扫描图，显示 SAGE 显然使用了电子管。此外，还提供了一些关于试管的其他网页的链接。

如果你对古老的记忆技术感兴趣，我们去年去过一次记忆之旅。我们不确定你是否可以用这些做同样的调试技巧，你可以用威廉姆斯管。有了这些，你可以把一个阴极射线管和显像管并联起来，得到[一个你的记忆的可视化表示](https://hackaday.com/2016/01/06/babybaby-a-1948-computer-on-an-fpga/)，这是 Baby 和其他几台旧电脑使用的技巧。
