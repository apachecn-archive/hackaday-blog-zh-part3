# 3D 打印发条星体跟踪器

> 原文：<https://hackaday.com/2018/06/03/3d-printed-clockwork-star-tracker/>

天体摄影是你自然会认为一定非常困难的事情之一；当然，如此令人敬畏的东西需要多年的练习和专门的设备，其成本相当于你的汽车。你对着天空挥舞着拳头(因为你已经放弃给它拍照了)，继续你的生活。另一个你会错过的经历。

但在现实中，戏剧性的结果不一定需要贴纸休克。我们之前在 Hackaday 上讨论过廉价的 DIY 追星族，但是这个由[Tinfoil_Haberdashery]发布在 Thingiverse 上的设计可能是我们见过的最简单的。它通过使用廉价的 [24 小时时钟机芯来随着地球自转](https://www.thingiverse.com/thing:2934503)旋转 GoPro，从而使事情变得简单。结果是时间推移，当地平线旋转时，星星看起来是静止的。

使用 24 小时时钟机芯是一种绝对聪明的方法，可以使相机与地球的旋转同步，而不需要通常必须跳过的环。当然，你可以用微控制器、步进电机和一些数学知识。但是，时钟是一种基本上从头开始设计的设备，用于记录地球的旋转，所以为什么不使用它呢？

如果说钟表机芯有什么缺点的话，那就是它没有太大的扭矩。这是为了移动时针，而不是你的相机，所以它不需要太多的拖延。GoPro(和其他“动作”相机)应该足够轻，这没什么大不了的；但是不要期望你的 DSLR 达到 1。即使在休息后的视频中，随着相机的重量开始推动齿轮，时钟看起来可能会在下降的过程中跳过几步。

如果你想要更有力量的东西，我们最近报道了一款非常光滑的 Arduino 驱动的“谷仓门”星际追踪器。但是有更简单的选择[，如果你今晚想拍照的话](https://hackaday.com/2016/12/15/build-this-barn-door-tracker-today-take-stunning-shots-of-the-galaxy-tonight/)。
