# 让不会说话的机器人进化

> 原文:[https://hackaday.com/2016/03/14/making-dumb-robots-evolve/](https://hackaday.com/2016/03/14/making-dumb-robots-evolve/)

进化是生活的事实，除了在堪萨斯州。这是生命本身的定义特征，但这并不意味着一个愚蠢的机器人不能进化。为了参加 Hackaday Pi Zero 竞赛，【diemastermonkey】[正在做的正是](https://hackaday.io/project/9848-iq-zero-evolving-unprogrammed-robots):围绕微控制器和树莓 Pi 构建的机器人进化。

[diemastermonkey]的项目是遗传算法的物理扩展。就像 DNA 和蛋白质不知道它们实际在做什么一样，微控制器也不知道。[diemastermonkey]的项目不是随机交换碱基对和氨基酸，而是根据这些引脚的值随机连接引脚。

这些蹩脚的、随机编程的机器人的潜力只相当于健身功能，迄今为止[diemastermonkey]已经取得了一些令人惊讶的成功。当把这些算法放入一个连接到倾斜工作台机构和 PIR 传感器的微控制器中时，机器人最终选定了一点代码来保持球的运动。你可以看看下面的视频。
