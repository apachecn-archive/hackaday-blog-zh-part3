# 烙铁烧灼

> 原文：<https://hackaday.com/2016/05/03/soldering-iron-cauterization/>

医学技巧不适合肠胃不好的人，所以继续读下去，后果自负。[【托德·哈里森】向你展示如何使用优质烙铁去除顽固的皮肤疣](http://www.toddfun.com/2016/04/03/weller-wart-removal-body-hacking/)，以及对疼痛的忍耐力。毕竟，烧灼是一种众所周知的和有文献记载的医疗程序。如果你对此感兴趣，请继续阅读，或者更好的是，在休息后看看他的 9 分钟视频。如果周围有孩子，在 1:40 到 2:20 之间调低音量。

[Todd]右手食指上有一个复发的疣，尽管尝试了很多次也无法除掉。每次他以为它走了，它又会回来。他甚至尝试了一些昂贵的商业试剂盒，声称可以通过快速冷冻来消除疣，但他没有任何快乐。最后，他认为是时候烧灼吸管了。像任何自尊的工程师一样，他首先用一些外用酒精清洁问题区域。他在 1:48 左右开始使用熨斗，一直持续到 4:17。在这一阶段，他甚至评估了他使用的技术，该技术最终导致疣周围的敏感皮肤因辐射热而起泡，以及如何以不同的角度握住烙铁并在下次尝试该程序时移动它可能会更好。

从那以后，就是等待和观察多日，看它如何发展。第二天显然不太好，有一个肿胀的水泡，这是“不是最好看的事情”。大约在第八天(@5:39)，他在做一些院子里的工作时割破了皮肤，但没有什么是创可贴不能修复的。到了第十四天(@6:41)，部分皮肤变干，准备脱落。他使用他的方便和值得信赖的斜切刀(当然！)剪掉带死疣的皮肤。从那时起，他就完全康复了，仅仅过了 40 天就完全恢复了正常。他说他每次都成功了，这意味着他显然至少尝试过几次。

他在做手术时穿着休闲衫，这给了他加分！谢谢[莫里斯]给我们通风报信。[托德]不是新手，是老校友。我们之前已经介绍了他的很多事迹，比如这个[德瓦尔特无线电修理](http://hackaday.com/2015/07/10/dewalt-radio-repair/)、 [1976 年的乒乓克隆修理](http://hackaday.com/2015/01/07/repairing-and-reviewing-a-1976-pong-clone/)和[史酷比闹钟修理](http://hackaday.com/2014/01/13/scooby-doo-alarm-clock-repair/)。
