# 机器人只需一秒钟就能解开魔方

> 原文：<https://hackaday.com/2016/01/27/robot-solves-rubiks-cube-in-just-one-second/>

一些世界上最快的魔方解答者已经在 5 秒内完成了解答——这对人类来说是一个不可思议的壮举——但是 1 秒怎么样呢？嗯，【杰伊·弗拉特兰】和【保罗·罗斯】刚刚制造了一个机器人，[可以做到这一点。](http://gizmodo.com/this-robot-can-solve-a-rubiks-cube-in-one-second-1754882972)

该机器人使用四个 USB 网络摄像头，六个步进电机和一个 3D 打印框架。对魔方的唯一修改是在中心部分钻了一些孔，以允许步进电机通过 3D 打印附件抓住它们。

该软件在一台 Linux 机器上运行，该机器将数据输入到一个魔方算法中进行求解。大约在一秒钟内，立方体被求解。

就在不久前，人类还能在解魔方方面打败机器。事实上，我们之前分享的世界纪录是一个机器人可以在 10.69 秒内解决一个立方体，比最快的人类解决者慢了整整 5 秒。
