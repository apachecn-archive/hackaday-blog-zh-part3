# 鲁贝·斯洛伯格

> 原文:[https://hackaday.com/2016/01/10/rube-slowberg/](https://hackaday.com/2016/01/10/rube-slowberg/)

这个问题在我们的提示热线中隐藏了几个月，但我们很高兴它最终浮出了水面。[鲍勃·帕廷顿]制造了[“鲁比·斯洛伯格”装置](https://www.youtube.com/watch?v=o-c7tBlnFsw)——它被宣传为世界上最慢的鲁比·戈德堡机器。他开球的高尔夫球花了六个星期才到达其颇具戏剧性的终点。

鲁布·戈德堡机器很迷人，但通常这种乐趣很快就结束了。[Bob]决定放慢速度，花了几个黑客才完成。对我们来说值得庆幸的是，大量使用定格和快进的剪辑视频将这场追逐缩短到了不到三分钟。

看看下面的视频。它开始于高尔夫球在糖蜜上骑着一艘慢船，搭上一只乌龟，穿过一系列融化的冰棒棒，然后被一堆生长的草一次推进一点点。如果你对幕后有兴趣，可以看看另一个视频，他讲述了他是如何做到的。
