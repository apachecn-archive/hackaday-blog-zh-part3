# 3D 打印火箭是一种气体

> 原文：<https://hackaday.com/2017/02/12/3d-printed-rockets-are-a-gas/>

我们小时候可能都做过火柴棍火箭。在这里，任何看起来像火箭的东西都会有一些想象中的飞行时间。但是[austiwawa]正在制造一些非常酷的 3D 打印火箭，它们使用普通的二氧化碳弹药筒作为推进剂。你可以在下面的视频中看到他们的行动。

你可能会认为只是在 3D 打印的夹克上贴一个二氧化碳气瓶没什么大不了的，但[austiwawa]真的做了更多。他阅读了如何使火箭稳定(通过操纵[重心和](https://spaceflightsystems.grc.nasa.gov/education/rocket/rktstab.html)压力中心)的资料，并解释了如何让火箭像你预期的那样飞行。

另外，发射管也挺有意思的。3D 打印零件包含一个尖点和一个弹簧。你锁定弹簧，当它松开时，会在推进剂外壳上打出一个干净的洞。实际的管子是一根长的 PVC 管。从视频上看，这些小火箭飞得相当高。

从视频来看，火箭体和发射架来自 TinkerCAD。austiwawa 穿上鱼鳍的方式既简单又巧妙。
