# 轿车制造推动着坚定的决心

> 原文：<https://hackaday.com/2017/11/30/car-cade-build-drives-unthrottled-determination/>

还记得 80 年代那些仪表板形状的赛车游戏吗，比如 Tomy Turnin ' Turbo？[丹尼尔]一直想把其中一个带入现代。在发现[有人捷足先登](https://circuitbeard.co.uk/2017/08/28/tomy-turnin-turbo-dashboard-outrun-arcade/)后，他转向另一个方向[从头开始建造自己的迷你柜子，专门用于污垢集会](http://projectable.me/my-dirt-rally-tabletop-car-cade-game/)。

这个想法是建立一个尽可能小的计算机，它可以运行 SteamOS，并且可以放在一个印在 Prusa 克隆体上的柜子里。起初，[丹尼尔]试着开着小滑板到处跑。帧速率很糟糕，所以他换了一个华硕的 mini-STX 板，然后从那里开始。

印刷方向盘和油门都是模拟输入，每个都使用一个 10kΩ电位计连接到 Pimoroni PiCade 控制器。我们喜欢[丹尼尔]用橡皮筋让自己居中的低科技方式。我们也喜欢他在方向盘上做的后处理，让它有恰到好处的抓握感(这是塑料浸渍橡胶涂料)，因为它看起来很棒。

可爱的蓝色橱柜是对[丹尼尔]选择的污垢拉力毁灭者，95 年蓝色拉力赛斯巴鲁翼豹的致敬。他为原型制定了一个艰巨的打印/砂/底漆/油漆计划，但最终用不同的颜色打印零件以获得正确的外观。[丹尼尔]独自经历了四种不同的蓝色细丝才感到满意。

在休息时绕着完成的橱柜快速走一圈，然后停下来观看预告视频，该视频通过社交媒体的魔力为[Daniel]赢得了一个来自 Dirt Rally devs 的礼品袋。现在是北半球的寒冷和流感季节，也许你更喜欢玩不碰任何东西的驾驶游戏。
