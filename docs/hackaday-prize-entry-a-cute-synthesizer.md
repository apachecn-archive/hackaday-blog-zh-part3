# Hackaday 奖参赛作品:一个可爱的合成器

> 原文：<https://hackaday.com/2016/05/15/hackaday-prize-entry-a-cute-synthesizer/>

对于电子爱好者来说，很少有比音乐合成器更酷的设备了。第一个合成器是运算放大器和梯形滤波器的巴洛克式组合。在 70 年代和 80 年代，合成器开始不可阻挡地向数字化进军。在 27 系列 EPROMs 上有存储样本的波表合成器。芯片上的合成器，如 MOS 6581“SID 芯片”，今天仍然受到推崇。为了他的 Hackaday 奖参赛作品，[【Tim】正在从零开始建造他自己的合成器](https://hackaday.io/project/11574-nanoegg-music-synthesizer)。它不是一个旧合成器的副本，而是一个带有经典声音的完全现代的合成器。

[Tim]是一名前游戏开发者，已经发布了一款合成器。用于任天堂 DSi 和 3DS 的 Rhythm Core Alpha 2 是一个功能齐全的合成器，但任天堂硬件的限制使[Tim]想从头开始构建自己的合成器。

synth 的规格更多的是一个愿望清单，但是[Tim]已经确定了一些设计特征。这是一个虚拟的模拟合成器，一切都是数字的，由 DSP 算法处理。它具有复音和 MIDI 功能，几乎每个参数都有按钮和转盘。对于一些你不能用旋钮做的事情，[Tim]包括一个触摸屏显示器。

[Tim]已经有了合成模型，从他收集的视频来看，整个事情听起来相当不错。下一步是把一堆电线、试验板和元件变成看起来像仪器的东西。我们迫不及待地想看看这个结果如何！
