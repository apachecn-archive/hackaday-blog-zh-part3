# 德克斯特有能力完成任务

> 原文：<https://hackaday.com/2018/06/30/dexter-has-the-precision-to-get-the-job-done/>

机械臂——它们很有用，是我们现代制造业经济的重要组成部分，在适当的情况下也很迷人。但最重要的是，它们因能够以高度精确的方式重复执行复杂的任务而受到重视。DEXTER 在所有方面都表现出色，它是一款开源的五轴机械臂，具有令人难以置信的精度。

DEXTER 由 3D 打印零件制成，结合现成的碳纤维部件来增加强度。控制是通过五个 NEMA 17 步进电机，这些电机连接到谐波驱动器，以 52:1 的比率逐步降低输出。每个电机都装有一个光学编码器，它提供反馈来控制末端执行器的位置。

与许多更简单的项目不同，DEXTER 没有使用 8 位微处理器，甚至没有 ARM 芯片 FPGA 为 DEXTER 的运营提供了智力支持。这为 DEXTER 提供了广泛的配置和扩展能力。此外，它还为开发训练模式等功能提供了充足的动力，在训练模式中，机器人手动完成动作，并在以后记录下表现。

这是一个高性能和开源的项目，这总是令人高兴的。我们期待看到这一个如何进一步发展！
