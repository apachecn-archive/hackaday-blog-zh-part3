# 完全由乐高工艺制造的绳索商标机

> 原文：<https://hackaday.com/2016/02/19/rope-braiding-machine-built-entirely-from-lego-technics/>

如果你曾经看过一台正在运转的绳索编织机，你就会知道它们是令人惊奇的机器，在那里线轴旋转着，以一种复杂的舞蹈来编织彼此上下的线。制造一台这样的机器一定非常困难；用乐高工艺块做一个几乎是疯狂的。

[Nico71]作为一名技术建造者，他们解决了这个复杂的问题，并使其发挥作用。一个大鼓水平旋转，并携带三组三个行星安装线筒。整个滚筒朝一个方向旋转，而线轴和另一个有三个孔的模具朝另一个方向旋转。然后将得到的编织物送入另一个纺丝模具，得到的 9 股绳由卷筒卷绕。卷筒有一个自动反向进给机构，以保持成品卷筒整洁。然而，该建筑最令人印象深刻的是，它全部由一台电机驱动，一切都通过齿轮、轴、链轮、链条和离合器同步。休息过后，你可以看到这是一次技术绝活。

[Nico71]身材相当惊人。有些是纯艺术，有些是老爷车和摩托车的模型，但像他的[织布机](http://www.nico71.fr/small-mechanical-loom/)和他现在正在工作的[计算器](http://www.nico71.fr/mechanical-calculator-part-1-carry/)这样的东西是非凡的。当然，如果你想看更多令人着迷的绳索编织机芭蕾，看看[这款 16 线轴手摇版](https://hackaday.com/2014/07/29/16-bobbin-rope-braiding-machine-inspired-by-surplus-store-find/)。
