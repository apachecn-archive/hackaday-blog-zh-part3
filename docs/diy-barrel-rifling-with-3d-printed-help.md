# DIY 桶膛线与 3D 打印帮助

> 原文：<https://hackaday.com/2017/03/08/diy-barrel-rifling-with-3d-printed-help/>

[Jeff Rodriguez]一直在忙着测试一种可行的 DIY 方法来膛线枪管，并发现了一些成功的使用盐水、电源、电线和 3D 打印部件来创建膛线凹槽，而不需要任何移动部件或切割工具。盐水在桶的内表面和 3D 打印件之间流动，3D 打印件以精确的模式固定电线。电流在桶和导线(导线实际上不接触桶的内部)之间流动，结果材料被侵蚀掉。10-15 分钟后，由于他的 DIY 过程，测试件上出现了一些看起来很有希望的凹槽。

至少从 19 世纪开始，膛线枪管就已经很普遍了(尽管这肯定是一个密集的过程),而且它仍然是一个最好留给工业环境的工作；如今，任何需要枪管的人通常只需从制造商那里购买一把来复枪枪管坯料。没有人会自己做，除非出于某种原因 T2 想要 T3 做，但这正是杰夫的想法。这个过程看起来很混乱，但是[杰夫]有很大的空间去试验各种不同的方法以获得不同的结果。

[杰夫]收集的结果还没有一个全面的地方，但两个图片库展示了他从[DIY 膛线测试](https://imgur.com/a/diHx7)到使用概念为[枪管切割枪膛](https://imgur.com/a/FuVCP)的过程。[Jeff]在下面嵌入的两个较长的视频中介绍了他的整个过程。
