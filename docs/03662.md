# Hello 3D 打印推车

> 原文：<https://hackaday.com/2016/10/04/hello-3d-printed-dolly/>

[伊凡]喜欢拍摄延时录像。利用他的 3D 打印机和步进电机，他制作了一个装备，允许他控制相机在光滑的地板上向任何方向移动。

推车有一个三脚架兼容的安装板和踏板车车轮。一个 Arduino 运行这个东西，一个手机电池提供能量。一个电位计设定速度，而[Ivan]为他建议的线性电位计和他手头的对数电位计提供代码。你可以在下面看到结果的视频。

由于 3D 打印的机身，该设备看起来很好；[伊万]提供的 STL 文件，如果你想复制他的工作。你必须多次打印几个部分。

如果你比 3D 打印更擅长木工，我们已经看到了由木头制成的好看的[台车](https://hackaday.com/2015/06/21/diy-plywood-camera-dolly-looks-professional/)。我们也看到了一个[看起来不那么完美的使用 Makeblock](https://hackaday.com/2015/10/08/camera-dolly-uses-makeblock/) 设置。
