# 本周失败:Arduino Sand Matrix 打印机

> 原文:[https://hack aday . com/2017/08/31/fail-of-the-week-arduino-sand-matrix-printer/](https://hackaday.com/2017/08/31/fail-of-the-week-arduino-sand-matrix-printer/)

纽约海滩是沉迷于冰毒的热带海滩的死亡之地。[Vije Miller]在为他的 Arduino sand matrix 打印机撰写的文章中如是说。这是一个聪明的想法，五个伺服操作的纸板活塞，随着设备向前移动，在沙子上形成一个点的图案，这就像一个可以在沙子上写信息的点阵打印机。

他把它作为本周的失败提交给我们，因为它没有很好地完成写在沙子上的工作，而且它烧坏了一个伺服系统。但是我们觉得这并不完全公平，因为不管它是否交付了货物，它仍然是一个优秀的构建。纸板不是我们在 Hackaday 这里经常看到的材料，但在这种情况下，他在一个复杂的机制中掌握了它，虽然它可能被证明对于手头的工作来说有点太灵活了，但仍然是一件相当令人印象深刻的作品。

你可以在休息时间下面看到一个简短的视频，展示它的作用。他告诉我们，他对这个项目的动力已经减弱，并表示希望其他人将接过接力棒，生产出更可行的机器。
