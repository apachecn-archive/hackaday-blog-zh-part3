# Hackaday 奖品入口:外汇显示是一个很好的执行黑客

> 原文：<https://hackaday.com/2016/05/12/hackaday-prize-entry-forex-display-is-a-well-executed-hack/>

在[Stefan]工作的地方，了解外汇市场的确切状况对于公司的赚钱计划非常重要。检查应用程序或网站太慢，让他脱离了自己的工作流程。操作系统桌面小工具已经或多或少地暂时离开了这个世界。当时唯一的解决方案是[为他的实际办公桌](https://hackaday.io/project/11081-forex-display)制作一个小部件。

该设备的大脑是一块 ESP8266 板、一些外设和一个小型背光 TFT 显示屏。该设备可以使用电池或壁式电源。[Stefan]甚至添加了一些在这类黑客中通常找不到的漂亮功能，例如光电池，它可以检测光线水平并相应地调暗屏幕。

该软件使用一种有趣的方法来获取最新的时间和时区。他没有使用为这项任务制作的图表或服务，而是使用一个开放的天气 API 来完成这项任务。相当聪明。

该案件是 3D 打印和打磨。为了获得图中所示的漂亮表面，Stefan 随后对外壳进行了喷漆。所有这些放在一起，这个设备看起来很棒，并且给他提供了他想要的桌面小工具。
