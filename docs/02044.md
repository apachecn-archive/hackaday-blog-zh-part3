# Everyman 的涡轮分子泵

> 原文:[https://hack aday . com/2016/04/09/everymans-turbo molecular-pump/](https://hackaday.com/2016/04/09/everymans-turbomolecular-pump/)

一个非常好的真空泵能做什么？你可以造一个电子显微镜，x 射线管，粒子加速器，薄膜，它可以让你的咖啡保温。当然，得到一个好的真空泵需要专家级的搜寻或者很多钱，这使得[DeepSOIC]和[Keegan]成为今年 Hackaday 奖的一个很好的参赛作品。这是普通人的涡轮分子泵，基于尼古拉·特斯拉的一项专利。糟透了，这是好事。

从电子显微镜和真空管中抽出空气的通常方法始于活塞泵或隔膜泵。这就把大部分的大气排出去了，但是还有一点剩余。为了把压力降得更低，可以用油扩散泵(虽然有点脏，但有点便宜)或涡轮分子泵(干净、棒、贵)把大气中最后的几个分子吸出来。

涡轮分子泵[DeepSOIC]和[Keegan]正在使用多个旋转圆盘[建造，就像[Tesla]1909 年的专利](https://www.google.com/patents/US1061142)一样。看起来，问题在于找到一种可以制成圆盘并能承受每分钟数万次旋转的材料。这是一个非常非常困难的建造过程，任何一个零件的制造错误都会导致这个涡轮分子泵惊人的快速拆卸。然而，回报将是巨大的。廉价的涡轮分子泵在任何黑客空间、fab 实验室或车间车库中都是非常有用的设备。
