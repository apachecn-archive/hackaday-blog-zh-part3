# 太空中的胡萝卜

> 原文:[https://hackaday.com/2018/03/22/carrots-in-space/](https://hackaday.com/2018/03/22/carrots-in-space/)

对于今年的 Hackaday 奖，[威尔·史蒂文斯]正在种植自己的产品，现在他正在寻找一种方法来保护他的努力免受英国冬天的危险。为了实现这个目标，他决定在密封的容器中种植蔬菜。受现有技术的启发和研究的支持，他的方法一方面是应用懒惰，另一方面是不计后果的过度工程。这个项目中的密封容器是 PET 瓶，选择它们是因为它们的可用性，产品是胡萝卜，主要是因为它们可以通过瓶口收获。胡萝卜还具有高能量密度，可以为植物性建筑材料提供纤维，因此[威尔]认为它们是理想的太空殖民者食物。

该项目目前正在进行第四次尝试，从胡萝卜种子、泥土和苏打水瓶中的一些水到目前的状态，该设置萌发了人工照明和二氧化碳传感器。完全意识到密封温室是一个经过验证的概念，[will.stevens]提供了一个人在尝试类似事情之前应该阅读的文献的链接，以及他的进展的定期更新。

传感器和发光二极管已经就位，添加一个 [raspi](https://hackaday.com/2017/08/26/raspberry-pi-is-the-brains-behind-automated-greenhouse/) 只是时间问题。或者我们可能会看到土壤的消亡，取而代之的是[水培](https://hackaday.com/2016/08/28/hackaday-prize-entry-automated-hydroponics/)设置。
