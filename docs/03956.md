# 冲浪开始了。实际上

> 原文:[https://hackaday.com/2016/11/01/surfs-up-virtually/](https://hackaday.com/2016/11/01/surfs-up-virtually/)

虽然如果每个人都有一片海洋，他们就会像加利福尼亚一样冲浪，这可能是真的，也可能不是真的，但如果你没有保持平衡，用水泵、一些铜带和一些废料就可以制作一个冲浪模拟器。

你可以在下面的视频中看到这个简单的设备。我们假设[Adi_10]确实在冲浪，但无法评论模拟器的逼真程度。没有电脑。只是一个由底座和平衡板组成的开关。

就项目而言，这相当简单，只不过是一个泵、一个电池和一个开关(当然，开关是关键)。然而，这是一个很好的提醒，不是所有的东西都需要 Arduino 或 Raspberry Pi。这将是一个有趣的项目，可以让孩子们思考构建硬件。

如果你想要更有野心的东西，你可以试试移动的冲浪板。或者你可以全力以赴，虚拟出一个[悬浮滑板](https://hackaday.com/2014/05/22/oculus-rift-and-wii-balance-board-make-hoverboards-a-virtual-reality/)。
