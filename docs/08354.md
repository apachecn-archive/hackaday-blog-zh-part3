# 修复晒伤的游戏机屏幕

> 原文:[https://hack aday . com/2018/01/26/repair-a-sunburned-game-boy-screen/](https://hackaday.com/2018/01/26/repairing-a-sunburned-game-boy-screen/)

游戏男孩原版是经典。当然，它没有背光，但在经典的绿色屏幕上玩游戏有一些特殊之处。不幸的是，这些旧系统中的一些正在遭受可怕的命运——屏幕烧伤。Game Boy 在光线充足的情况下玩得最好——尤其是在阳光下。但是同一个太阳对屏幕做了可怕的事情。液晶显示器中心的黑色斑点是一个烧毁的游戏机的迹象。你可能认为更换屏幕是唯一的选择，但[【复古未来】向我们展示了如何修复这个问题](https://www.youtube.com/watch?v=JJ-yw4YA1OQ)。

反射式 LCD 是一个由偏光镜、两块玻璃和一个反射镜组成的多层蛋糕。在 Game Boy 屏幕上经常看到的烧伤通常发生在偏光镜和将塑料偏光镜贴在玻璃上的光学透明胶上。我们猜测这些烧伤发生在有人把游戏机放在阳光下的时候。在直接照射到顶部偏振镜的太阳光和从屏幕后部的反射镜反射回来的光线之间，那个可怜的偏振镜根本没有机会。

修复烧伤是一个微妙的操作，因为一个错误的举动可能会打破薄液晶玻璃。第一步，小心翼翼地剥掉烧焦的偏光片。这会留下一堆干胶，可以刮掉或用酒精溶解。然后可以在屏幕的前面放置一个新的线性偏振器。[复古的未来]选择不粘偏振器，但我们打赌手机屏幕保护器的一些紫外线固化 LOCA(液体光学透明粘合剂)会达到目的。

如果你喜欢经典游戏男孩的外观，但想玩任何经典游戏，[拿一个树莓派零，并建立一个复古派男孩](https://hackaday.com/2016/04/10/gameboy-case-lives-on-with-a-pi-zero/)。
