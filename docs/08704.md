# 罐头里的太阳能！

> 原文:[https://hackaday.com/2018/03/02/solar-power-in-a-can/](https://hackaday.com/2018/03/02/solar-power-in-a-can/)

野营时，人们通常会带上灯笼、手电筒之类的东西——你甚至可以带上太阳能充电器。Instructables 用户[bennelson]将太阳能装进一个罐子，将所有电力需求结合起来。

为了抵御恶劣天气，[bennelson]使用了 50 口径的手枪。便携式外壳的弹药罐。在内部，他在中央植入了一块 15AH、12V 的铅酸电池，以保持平衡，并为布线和存储留出空间。一个纸板模型之后，他用激光从 [![](../Images/074a7ca146687a745943992bfb0fd706.png)](https://hackaday.com/wp-content/uploads/2018/02/solar-power-in-a-can-internals.jpg) 1/8”胶合板切割出顶部面板，并在其顶面固定了一个 20A 太阳能充电控制器、一个四合一插座面板和两个香蕉插头。

[bennelson]正在使用 12 AWG 导线来匹配太阳能充电控制器的 20A 额定值，包括一个用于安全的保险丝，以及杠杆锁紧螺母连接器来解决一些布线问题。工业 velcro 将顶部面板保持在原位，需要时可轻松拆除。当他外出露营时，他使用 18V 1A 太阳能电池板充电，但仍然可以使用 DC 电源适配器从电网充电。休息后查看完整的构建视频！

[bennelson]在弹药箱里塞了一些方便的东西——一个鹅颈 LED 灯，一个 LED 灯泡和一根挂在树枝上的长电线，以及一个 100 瓦的交流电源逆变器，用于家中或院子里的任何大型电器。
