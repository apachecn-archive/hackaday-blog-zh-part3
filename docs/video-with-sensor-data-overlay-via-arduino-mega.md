# 通过 Arduino Mega 与传感器数据叠加的视频

> 原文：<https://hackaday.com/2015/11/11/video-with-sensor-data-overlay-via-arduino-mega/>

如果你一直没有注意，大车轮三轮车是一件事。有电机驱动的版本，也有 [OG](https://en.wikipedia.org/wiki/Original_gangster) 脚踏推动的类型。[Flux Axiom]是 OG(你只有一个链接，现在轮到你了)风格，他写了一个说明如何在屏幕上获得一些好看的数据的说明书，他将这些数据与视频同步，制作出专业外观的成品，休息后你可以在视频中看到。

[Flux Axiom]在他的设置中使用了 Arduino Mega 以及大量的传感器，所有的数据都记录在 SD 卡上。他的设置中使用的所有代码都可以在他的 GitHub 库中获得[。[Flux Axiom]还很好地包括了他用于传感器的校准过程，这也可以在 GitHub 下载中找到。](https://github.com/fluxaxiom/Stat_Cache)

遗憾的是[Flux Axiom]使用了憎恨自由的软件来组合视频和数据， [Race Render 3](http://racerender.com/RR3/Features.html) 是他目前的解决方案，他对结果很满意。如果你有一个结合视频和数据的开源解决方案，我们可以提供给他作为替代，请在评论中留下。
