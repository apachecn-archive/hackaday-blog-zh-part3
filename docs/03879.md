# Hackaday 奖参赛作品:双侧大脑刺激器

> 原文：<https://hackaday.com/2016/10/27/hackaday-prize-entry-bilateral-brain-stimulator/>

1987 年，一位美国心理学家发现，主动眼球运动可以降低消极思想的强度。这是 EMDR 的基础，或者说眼球运动和再加工去敏感化，如果这听起来太不真实，我向你保证心理学中甚至有更不真实的技术。

[David]的 Hackaday 奖参赛作品是一个帮助心理学家应用 EMDR 治疗创伤后应激障碍的装置。为了做到这一点，心理学家要求病人在眼球运动时描述一个创伤性事件。根据文献记载，这有助于大脑半球之间的联系，并减少情绪负担。

如果在重温你最大的恐惧时简单地来回移动你的眼睛就足够了，那么硬件是怎么回事？[David]的项目只是一些帮助实现眼球运动的发光二极管。通过移位寄存器和 PIC 微控制器控制的 led 线性阵列，该设备足以让患者从左到右再到左反复扫视。

这是一个很好的项目，更好的是，[大卫]的姐姐，一位心理治疗师和 EMDR 从业者，要求[大卫]为这项技术创造一个电子设备。[David]已经在他的设备上制作了一个视频，你可以在下面查看。
