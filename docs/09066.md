# 乐高遇上任天堂 Switch

> 原文:[https://hackaday.com/2018/04/04/lego-meets-nintendo-switch/](https://hackaday.com/2018/04/04/lego-meets-nintendo-switch/)

你可能知道，任天堂 Switch 是目前非常受欢迎的游戏机。你当然也知道，自从有历史记载以来，乐高就一直很受欢迎。因此，有人认为这两个年轻娱乐巨头需要像某种印钞机一样联合起来，这只是时间问题。你知道，为了科学。

[![](../Images/884b3fd18436f2b16298f4ed34cec037.png)](https://hackaday.com/wp-content/uploads/2018/04/legoswitch_detail.jpg) 【维姆·帕特尔】，一位众所周知的与塑料砖相关的所有事情的大师，决定用一些[实验性的乐高配件来接受这个挑战，换成](http://cargocollective.com/vimalpatel/nintendo-switch-experiments)。这些附加组件主要是为了让玩 Switch 更舒服一点，但代表着未来更复杂的硬件修改的有趣的第一步。

这些实验的关键是一套 3D 打印轨道，允许您将标准乐高零件连接到开关上。安装导轨后，[Vimal]展示了一个简单的“支架”,当不在手持模式下使用时，它可以提高系统的稳定性。

还演示了一些不同的方向盘修改，这些修改使用了令人印象深刻的工程技术，通过方向盘上的旋转输入来左右移动控制器的模拟杆。这两种变化都显示在马里奥赛车中使用，并且似乎完成了任务。

本月晚些时候[任天堂 Labo 上线](http://hackaday.com/2018/01/20/nintendo-switch-gets-making-with-labo/)，看看 Switch 和 LEGO 的交集会有什么样的项目成为可能，这将是一件有趣的事情。
