# 可编程圣诞树是一个 JavaScript 解释器

> 原文：<https://hackaday.com/2017/12/24/programmable-christmas-tree-is-a-javascript-interpreter/>

在 Hackaday，我们发现圣诞节非常令人兴奋，因为这意味着大量以节日为主题的黑客涌入，真正帮助我们融入节日气氛。在 [HackMyXmas](http://www.hackmyxmas.com/) 举办的【Andrew 的】可编程圣诞树当然是我们的最爱之一。该项目由一个包裹在典型圣诞树周围的 500 个 RGB LEDs 组成，由一个青少年控制。然而，不满足于 led 的典型、简单和循环模式，[Andrew]决定这棵树当然必须是可编程的！因此，一台运行 Linux 的单板计算机(C.H.I.P)被用来提供 Wifi 连接和网络服务器，以方便对采油树进行编程。

这就是事情变得非常有趣的地方。C.H.I.P 委员会有一个全面的网站，让你可以方便地选择使用 Scratch like draggable blocks(使用 Googles Blockly)甚至纯 JavaScript 来编程 led。一旦构思出完美的模式，你可以在在线模拟器上试运行它，甚至直接把它发送到树上，看着它在提供的直播流中闪烁。

我们为[Andrew]以如此有趣的方式调用编程所做的巨大努力喝彩！你可以看看下面[安德鲁]圣诞树的直播流。
