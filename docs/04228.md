# Mintomat:一台过于复杂的口香糖贩卖机

> 原文：<https://hackaday.com/2016/11/25/mintomat-an-overcomplicated-gumball-machine/>

你如何让青少年对科学、技术和工程感兴趣？[Erich]在卢塞恩应用科技大学的团队让他们[操作三个机器人来得到一个口香糖球](https://mcuoneclipse.com/2016/11/19/mintomat-worlds-most-complicated-bubble-gum-automata/)。整个演示是在几天内完成的，并且已经[实地修复了至少一次](https://mcuoneclipse.com/2016/11/21/failure-with-solder-points-and-mechanical-pressure/)；绿线固定在焊料上有点重，当施加机械力时会短路到相邻的走线。

有一些有趣的细节，包括选择使用 nRF24 2.4 GHz 无线电在主板和[漫游机器人组件](https://mcuoneclipse.com/2016/03/25/zumo-robot-with-magnetic-encoders/)之间进行通信，以及一些免费的 LED 驱动器和一个有趣的飞行时间距离传感器。如果您对构建小型机器人感兴趣，请查看一下。

这个游戏看起来很有趣，如果建造很酷的玩具是让年轻人接受技术教育所需要的，我们都支持。无论是简单的机器人系统，如这些机器人系统，还是由乐高积木制成的 T2 恐怖现实的核反应堆模拟器，我们不得不说我们也受到了启发。
