# 埃菲尔铁塔的壮观景象都是用镜子做的

> 原文:[https://hacka day . com/2015/12/14/壮观的埃菲尔铁塔景观是用镜子完成的/](https://hackaday.com/2015/12/14/spectacular-view-of-the-eiffel-tower-is-all-done-with-mirrors/)

哪个巴黎人不想要一套能看到埃菲尔铁塔的公寓呢？唉，并不是每扇窗户都能面对这个著名的地标，对一些人来说，百万欧元的景观遥不可及。这就是[Lurluberlu]的情况，但是用了一点光学诡计[他就能够在一个角落里窥视埃菲尔铁塔的壮观景色，并把它传送到他的卧室。](http://imgur.com/a/MDYhi)

[Lurluberlu]设计了一个简单的水平潜望镜，使用两个全身镜。休息后的视频显示了建设——作为一个边注，我们非常嫉妒他的手工工具包装车间。通过一些胶合板背衬和简单的旋转，镜子被安装在他的窗台上，以反射标志性塔楼的内部图像。稍加调整后，图像被窗户完美地框住，加上公寓里的灯关了，从床上看到的景色相当壮观。我们敢打赌，这比搬到视野更好的公寓要便宜得多。

当然，用[一点点聪明才智(和一个阳台)](https://hackaday.com/2013/08/27/a-facelift-for-the-view-out-your-livingroom-window/)任何人都可以看到这座光之城。或者其他任何地方。
