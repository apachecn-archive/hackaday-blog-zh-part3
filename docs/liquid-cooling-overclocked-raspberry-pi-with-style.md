# 液冷超频树莓皮与风格

> 原文：<https://hackaday.com/2017/05/15/liquid-cooling-overclocked-raspberry-pi-with-style/>

[HydroGraphix 总部]以此赢得了他的名字。虽然他使用矿物油而不是水，但他确实在图像方面做得很好。问题中的“它”是一个[超频的树莓 Pi 3，在一个充满矿物油的透明容器中，并带有一个循环风扇](https://www.youtube.com/watch?v=JIqp76FK_BI)。

他在运行任天堂 64 模拟器时，在 1.45 GHz 下运行 Pi 没有问题，温度在 40°C 和 50°C 之间。循环风扇是一个五伏计算机 USB 风扇。很难判断石油是否真的在移动，但我们很确定我们在视频的结尾看到一些石油在移动。

矿物油不导电，通常用于[防止高压倍增器板](http://hackaday.com/2016/06/15/wrangling-high-voltage/)上的元件之间产生电弧，但这些元件总是焊接在一起。如果你曾经使用过矿物油，你就会知道它会渗入每个角落，让我们怀疑它是否会在这款树莓 Pi 上的各种 USB 连接器的一些(非焊接)触点之间发挥作用。可能不会，但是我们这些有经验的人可以证明它的隐蔽性。

我们以前见过 Arduino 进行这种浸入式冷却，但通常电路板只是被放入一个顶部开口的容器中。在下面的视频中，[HydroGraphix]展示了他在制作这个盒子时所付出的心血，以展示一些值得展示的东西。led 呈现出漂亮的现代外观。一些塑料水族箱植物，一个宝箱和一些漂浮在周围的假鱼将会很适合。你觉得是不是矫枉过正了？我们认为光滑的外观以及结果使它值得努力。

以这种方式用水冷却树莓酱当然会使零件短路，但我们已经看到它是通过[水在管道中循环，以及机械加工的零件与芯片物理接触](http://hackaday.com/2013/07/20/water-cooled-raspberry-pi/)来实现的。还有一种最简单的方法，[廉价的铜垫片制造出更传统的散热器](http://hackaday.com/2016/12/20/pi-zero-keeps-cool-at-1-5-ghz/)。
