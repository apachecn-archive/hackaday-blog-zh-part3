# 微型真空室臂有助于自制半导体

> 原文：<https://hackaday.com/2018/04/23/tiny-vacuum-chamber-arm-to-help-with-homemade-semiconductors/>

[谢妮]想在国内制造半导体，这需要一些不寻常的工具。其中最主要的是一个进行薄膜沉积的真空室，真正符合黑客信条的是他是自制的，并将很快配备一个带有磁耦合机械控制的微型机械臂。

如果[谢妮]的装置看起来很熟悉，那可能是因为几天前我们介绍了他的等离子体实验。他当时对自己的目标有些谨慎，但他已经坦白了自己的愿望[制作自己的 FETs](https://hackaday.io/project/107598-semiconductors-home) (这是他 2018 年 Hackaday 奖的参赛项目)。这样做不仅需要产生稳定的等离子体，还需要能够在真空室中移动衬底。从外科医生用于腹腔镜手术的细长和可操作的器械中获得灵感，[谢妮]正在研究一种可以在他的真空室中工作的微型手臂。下面的视频是一个 3D 打印的概念验证模型，显示了手臂的各个部分如何由电缆控制。真正有趣的是，控制电缆不会穿过真空室——它们将通过磁铁穿过玻璃墙。

我们渴望看到来自[谢妮]家工厂实验室的芯片，但看起来这里和那里之间会有很多很酷的黑客。我们会密切关注。
