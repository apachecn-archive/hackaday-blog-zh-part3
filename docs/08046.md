# 将圣诞彩灯表演带入室内

> 原文:[https://hack aday . com/2017/12/21/bring-a-Christmas-lights-show-inside/](https://hackaday.com/2017/12/21/bringing-a-christmas-lights-show-inside/)

Instructables 用户[Osprey22]多年来一直在为这个圣诞节做准备。为什么？今年，他带来了一个令人印象深刻的音乐圣诞彩灯展示，而且价格也只是它的一小部分。

圣诞树底部的一个盒子隐藏了电源和控制板——一个 Raspberry Pi 和一个 SanDevices e682 像素控制器，用于 400 ws 2811 RGB led——并添加了一个路由器将它们连接到主网络。Pi 运行的是 Falcon Pi 播放器，一个 100 美元左右的投影仪补充了灯光秀。

至于绘制 led，Xlights 是首选程序，在手机视频记录的帮助下，在空间中定位 led。[Osprey22]必须用 C 写一个快速程序来修复网格中的 LED 重叠。(在这里，电子表格也可以)。哦，圣诞树底部的礼物还可以兼作投影屏幕！
