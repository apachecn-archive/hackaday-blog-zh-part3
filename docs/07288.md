# 一把老式莫尔斯键变成了 USB 键盘

> 原文:[https://hack aday . com/2017/10/03/a-vintage-morse-key-变身 usb-keyboard/](https://hackaday.com/2017/10/03/a-vintage-morse-key-turned-into-usb-keyboard/)

曾几何时，只有很酷的孩子才有新式的 102 键键盘，有数字键盘、箭头键和功能键。它们比陈旧的 86 键布局有了很大的改进，没有人会梦想回去。但是回到单键键盘是非常酷的，在这个例子中[这个莫尔斯键盘到 USB 键盘适配器](https://imgur.com/gallery/lNAtQ)。

为了恢复她父亲的旧直键，一个来自军事或商业机构的 20 世纪中期的坚固野兽，[Nomblr]开始适当拆卸和清洁黄铜和胶木磅。一个十几岁的孩子被选中从事将莫尔斯转换成键盘敲击的工作；仔细考虑 dits 和 dah 的时间以及接触去抖的容差对于完成工作至关重要。一个新的木制底座不仅为钥匙提供了稳定性，还隐藏了细节，打造了新的外观。下面的视频展示了它的作用；我们唯一的抱怨是，当你一次点击一次，敲出下一部伟大的小说时，没有侧音听莫尔斯电码。

精心修复的电报设备在这里是一件小事；不久前，我们推出了这款老式电报机，并配备了莫尔斯电码发送器。
