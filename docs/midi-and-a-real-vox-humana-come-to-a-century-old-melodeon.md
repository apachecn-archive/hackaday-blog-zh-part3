# 迷笛和真正的人类之声来到了一个百年老店

> 原文：<https://hackaday.com/2017/10/15/midi-and-a-real-vox-humana-come-to-a-century-old-melodeon/>

一百年或更久的消费级录制音乐已经将我们带到了一个位置，在这个位置上，我们大多数人都不自觉地认为音乐是一种录制的体验，而不是现场体验。一个多世纪以前，情况并非如此，许多家庭会有某种形式的乐器供自己娱乐，而不是高保真音响或其他设备。更昂贵的钢琴可能成为重要的地位象征，为各地富裕的商店生产钢琴和其他乐器的行业也很繁荣。

这些室内乐器中的一种来自[Alec Smecher]，这是一种泵风琴，也被称为口琴或 melodeon。他小心翼翼地为它添加了 MIDI 功能，并因此用一套真正的人类声音取代了它破碎的“人类之声”颤音效果，该效果旨在模拟 19 世纪的唱诗班。正如你在下面的视频中看到的,[对这个真实的 Vox Humana](https://www.youtube.com/watch?v=jaWHwQM8JBk) 的演示有一种近乎*巨蟒*的品质。

以免你认为他在转化过程中破坏了器官，请放心，这是一件敏感的应用作品。每个按键下都放置了一个微型开关，保持原有的机制完好无损并正常工作。Arduino Leonardo 将微型开关复用到类似键盘的矩阵中，并模仿 USB MIDI 设备。公平地说，它因此缺乏你可能需要模仿钢琴的力的敏感性，但它确实导致了一个相当有吸引力的 MIDI 乐器，也兼作一个真正的器官。
