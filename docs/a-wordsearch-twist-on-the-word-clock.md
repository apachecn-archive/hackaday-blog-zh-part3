# 对单词 Clock 的一个单词搜索扭曲

> 原文：<https://hackaday.com/2017/03/16/a-wordsearch-twist-on-the-word-clock/>

我们喜欢看到新的想法出现在现有的想法上，而[Danny]通过他的[单词搜索钟](http://danny.makesthings.work/wordsearch.html)无疑将单词时钟概念带入了一个不同寻常的方向。丹尼没有点亮单词拼出时间，而是决定接受这样一个事实:钟面上明显杂乱的字母就像一个单词搜索难题。

在单词搜索游戏中，单词可以向前或向后拼写，字母水平、对角或垂直排列。重要的是，正确的字母排成一行，并且顺序相邻。[Danny]的时钟一个接一个地点亮正确的字母和单词，就好像它在解决一个单词搜索难题，寻找恰好显示正确时间的单词。你可以在下面的视频中看到它的运行。

[Danny]在计划阶段做得更多。在使用单词搜索拼图生成器工具辅助设计布局后，他编写了一个[处理](https://processing.org/)草图来模拟时钟的运行。视觉模拟时钟使他能够对布局进行调整，确定要解决的边缘情况，并深入了解整个过程。如果你对自己制作感兴趣，这个项目有一个 [GitHub 资源库](https://github.com/walkerdanny/Wordsearch_Clock)。
