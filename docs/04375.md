# 无蚀刻电路板印刷

> 原文：<https://hackaday.com/2016/12/08/no-etch-circuit-board-printing/>

如果你曾经试图在家里制作一个印刷电路板，你就会知道这有多痛苦。有一桶桶的酸需要搬运，有很多等待和挫折，而且通常情况下，传统上用家庭设备制作的电路质量最终并没有那么好。幸运的是，[Rich]想出了一种消除多重印刷和蚀刻所需酸的方法。

他的过程包括使用激光打印机(而不是传统的喷墨打印机)来获得一层银粘合剂，以粘在一张纸上。银附着在墨粉上，就像闪光粘在 Elmer 的胶水上一样，允许激光打印机一次通过，就可以制作出可靠的电路。从那里，纸可以固定到更坚固的东西上，元件可以回流焊接到上面。

虽然[Rich]发布了一些关于这种方法的警告。银可能是不健康的，所以避免接触它，当它应用于碳粉时，会释放出不确定的棕色烟雾，这也可能是不健康的。不过，先不说警告，这是自制 PCB 的一个很好的方法，尤其是如果你不想家里到处都是酸的话，不管多有用。

感谢[Chris]的提示！
