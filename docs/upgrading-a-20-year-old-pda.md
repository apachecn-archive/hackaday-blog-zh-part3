# 升级 20 年前的 PDA

> 原文：<https://hackaday.com/2016/05/01/upgrading-a-20-year-old-pda/>

在我们有电子设备和机器人之前，甚至在黑莓之前，我们就有了掌上电脑。这些 90 年代中期的计算设备中最著名的是 Apple Newton 和 Palm 产品，但是 90 年代的 PDA 世界明显比这两种设备更加多样化。Palm 的竞争对手 Handspring 发布了一款更便宜、更好的 Palm OS 设备，带有 Visor。1998 年，惠普曾生产硬件，你可以在手持设备上运行 Windows 操作系统，包括 Excel 和 Word。

当时，一家名为 Psion 公司也生产带有翻盖设计和键盘的 PDA。不考虑操作系统，这些小翻盖 PDA 可以被称为昨天的上网本和今天的 Surface 平板电脑的先驱。[RasmusB] [通过更换所有重要部件，同时保持这款 20 岁 PDA 的简洁设计，将他的心灵感应 5 PDA 变成现代的东西](https://hackaday.io/project/4042-psio)。

这个项目的目标是完全取代 Psion 5 的电子设备，同时保留所有的机械装置。这意味着键盘将保持不变，设备将使用两节 AA 电池，所有的开关和端口都将工作。这项工作始于[通过用铅笔和纸逆向工程键盘矩阵，使 Psion 键盘兼容 Arduino】，并将键盘变成 USB 键盘。](https://hackaday.io/project/4042-psio/log/28423-arduino-compatible-keyboard-adapter)

将这种 Psion 转变为现代设备的努力正在进行中，但至少[主板的轮廓现在在 KiCad](https://hackaday.io/project/4042-psio/log/30263-reverse-engineering-mechanical-parts) 中，有一个微控制器来解码键盘，盖子和其他按钮的开关，以及 CompactFlash 卡和电池触点的正确空间。下一步是选择微处理器和设计电路，但是[拉斯姆斯]要把这个古老的 PDA 变成现代的计算设备已经有了一个很好的开始。
