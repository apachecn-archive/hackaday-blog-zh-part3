# 鲍勃的袜子

> 原文：<https://hackaday.com/2017/01/11/socks-by-bob/>

不，这篇文章不是关于 SOCKS4 或 SOCKS5 或代理的。是关于真正的袜子，穿在脚上的那种。认识一下[鲍勃·卢瑟福]，88 岁，住在加拿大萨斯卡通。他和他的伙伴们(92 岁的格林·萨利，85 岁的乔治·斯莱特，和 65 岁的年轻的巴尼·沙利文)已经为社区和全国的避难所制作了 10，000 只袜子。差不多有 8 英里长的袜子。仅去年一年，他喜欢称之为“鲍勃袜子行动”就生产了 2000 只袜子。

那么这四个人是怎么做到的呢？原来[鲍勃]有一点创造者的精神，他实际上为制造袜子制造了一台快速、廉价的针织机。使用污水管作为基础，机器可以每秒钟编织 90 针。

他不久前制作了它，但它并没有太多的用途。可悲的是，七年前他的妻子去世了，留给他一个空虚的人生。按照他儿子的建议“如果你想帮助自己，就帮助别人”，他决定开始这个项目。

> “我们中有很多人，随着年龄的增长，我们坐在家里看着墙，无所事事！鲍勃的袜子给了我一些事情做。”[鲍勃]

如今，该团伙有两台机器稳定工作，每周一次，他们将长管羊毛切割成袜子。一半的纱线是捐赠的，另一半加上运费由[鲍勃的]儿子筹集。针织机在运转时看起来非常棒。自己看下面的视频。
