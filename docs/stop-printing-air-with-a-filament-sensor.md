# 用灯丝传感器停止打印空气

> 原文：<https://hackaday.com/2017/03/04/stop-printing-air-with-a-filament-sensor/>

如果你有一台 3D 打印机已经有一段时间了，你会知道当你进来检查一个 8 小时的打印结果却发现在第 7 个小时你的灯丝用完了(或者灯丝断了)并且你的打印机已经毫无理由地尽职尽责地四处移动时的心碎。[查克·埃莱比克]知道，他决定做一个他在 Thingiverse 上发现的灯丝传感器。

在 Thingiverse 上找到一个零件并打印出来可能不会引起太多关注。但如果你看了下面的视频，这是一个很好的例子，说明 Thingiverse 的东西并不总能满足你的需求。微动开关[Chuck]比使用的设计要大。于是他将 STL 文件加载到 TinkerCAD 中并修复。他向你展示了他是如何做到的。这是一项有用的技能，因为你永远不知道什么时候你需要修改你在互联网上找到的某些部分。

这个想法很简单。灯丝的消失触发了一个蜂鸣器，让你知道你只有几分钟的时间关掉灯丝。一个简单的设备，但它可以让你在路上省去很多麻烦。如果配备的话，你也可以把它连接到打印机的灯丝停止输入端。

当然，基本的想法并不新鲜；今年早些时候，[唐纳德]制作了一个非常相似的。同样，[一个更好的传感器](https://hackaday.com/2016/12/08/this-old-mouse-keeps-track-of-filament-usage/)会让你做的不仅仅是知道它不见了。
