# Hackaday 奖参赛作品:Bloodhound 自主无线电定位无人机

> 原文：<https://hackaday.com/2017/08/24/hackaday-prize-entry-bloodhound-autonomous-radiolocation-drone/>

如果你是第一响应者，比如说，寻找在内陆或雪崩下迷路的人，并且你正在寻找一个有无线电信标的人，找到那个电信标的最快方法是什么？爬到高处将是一个好主意，如果你使用无线电测向，你将希望能够快速覆盖许多地面，即使只是为了使三角测量更容易一点。又高又快— [听起来是无人机](https://hackaday.io/project/25995-bloodhound-autonomous-radiolocation-drone)的绝佳机会，对吧？

[Phil Handley]的 Bloodhound 项目是一种自主无人机，可以扫描广阔的区域，监听紧急信标，同时向搜救人员发出警报。他的测试床 tricopter 使用 DT750 无刷 outrunners，由 18A Turnigy 毛绒 ESCs 控制，由 2200mAh LiPo 提供动力。一个金属齿轮伺服工作偏航机制。他还拥有 Pixhawk 自动驾驶仪、ArduPilot 飞行控制器、NavSpark GPS、软件无线电加密狗和树莓 Pi。他用木钉制作了空气框架，遵循了 RCExplorer 的三翼机设计。

下一个挑战涉及无线电测向，本质上是创造猎犬的猎狐技能。它需要能够通过从多个角度读取数据来自动追踪信号。除了寻找迷路的滑雪者，[菲尔]还设想用猎犬追踪其他信标，当然——比如野生动物转发器或迷路的业余火箭。
