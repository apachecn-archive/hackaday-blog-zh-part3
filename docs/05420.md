# 令人生畏的互动 LED 舞池建设是一个巨大的胜利

> 原文:[https://hack aday . com/2017/03/24/threaking-interactive-led-dance floor-build-is-huge-win/](https://hackaday.com/2017/03/24/daunting-interactive-led-dancefloor-build-is-huge-win/)

如果你曾经想过在一个活动中拥有一个灯光舞池，你可能会被租金震惊。看来，在《周六夜狂热》中扮演年轻的约翰·特拉沃尔塔并不便宜。当在 2017 年 Furnal Equinox 大会之前面临这个问题时，[Av]和朋友们决定改为[建造他们自己的 LED 照明地板](http://www.avbrand.com/projects/dancefloor/)。

他们的设计和建造显示在我们放在休息下面的视频中，虽然每个单独的灯光单元都很简单，但项目的规模和史诗般的建造使它成为一个非常令人印象深刻的成就。有 64 个 4 个光单元的面板，总共有 256 个单元和 7680 个 RGB LEDs，排列成 2560 个像素。每个面板都有一个移位寄存器 PCB 接口 led 到控制地板的 Teensy，还有与 Arduino Mega 对话的微动开关，为地板提供交互性。很难想象如果没有众多参与多伦多黑客实验室组装过程的人，这种构建是可能的。

由此产生的 17 英尺见方的舞池是一件艺术品，定制的编程图形对舞者的动作做出反应，甚至还有一些内置的 *Dance Dance Revolution* 游戏。看完下面的视频，你们有多少人会偷偷想要一个？
