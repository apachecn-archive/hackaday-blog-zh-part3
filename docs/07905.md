# 两年后容易了

> 原文:[https://hackaday.com/2017/12/05/easyeda-two-years-later/](https://hackaday.com/2017/12/05/easyeda-two-years-later/)

有些人想把所有东西都放在云上，而有些人拒绝把哪怕是最微小的数据放在互联网上。我们大多数人都介于两者之间。几年前，我们讨论了一些基于云的 PCB 布局规划，包括一个名为 EasyEDA 的规划。我们印象深刻，因为它是一个完整的包:原理图捕获，仿真和 PCB 布局。它是免费使用的，尽管他们会给你一个生产你的主板的报价，尽管你没有义务去购买它们。当然，两年后事情会发生变化，所以如果你想知道 EasyEDA 做得怎么样，[Yahya Tawil]发布了一篇深入的评论。

一些新功能包括自动布线器和直接从 BOM 订购零件的能力，而不仅仅是 PCB。云方面很方便，不仅因为你在任何地方都不需要安装和更新软件来使用它，而且因为在项目上与他人合作是非常自然的。不过，我们确实注意到，自动路由器可以运行在云中，或者您可以下载并在本地运行它，因为它显然会显著增加服务器的负载。

与一些在线 PCB 布局程序不同，EasyCAD 会生成符合行业标准的 Gerber 文件，因此尽管他们会提供您的电路板，但您可以将您的文件带到任何地方。两年前，我们确实有一个短期运行的董事会，只是看看他们会出来，他们很好，价格非常有竞争力。

如果你错过了我们最初的 EasyEDA 帖子，[我们看了一下，没开玩笑，我发了](https://hackaday.com/2015/08/21/a-tale-of-two-browser-pcb-tools/)。我们也知道[Hari]用它们来做他的[巨型 LED 立方体板](https://hackaday.com/2017/06/02/540-pcbs-make-a-giant-led-cube/)。如果你喜欢短视频中的评论，下面是[MickMake]评论的第一部分。
