# 用于定制电脑机箱的谢妮温度计

> 原文:[https://hack aday . com/2016/02/21/谢妮-温度计-指定用于定制的 pc-case/](https://hackaday.com/2016/02/21/nixie-thermometer-destined-for-custom-pc-case/)

不可否认，一套 Nixies 的温暖光芒具有复古的吸引力，当一位朋友正在为他的新液冷 PC 外壳寻找独特的触感时，[Luca]提供了这个可爱的谢妮温度计。

从[Luca]详细的博客条目来看，他从新年开始就在这个建筑里了。他首先列出了一系列要求，包括对圆形 PC 板奇怪的特殊需求。对于温度计，三个数码管用于显示，两个用于显示温度，一个用于显示单位。在提交 PCB 设计之前，一切都在 perf 板上进行了原型制作，但即使经过精心规划，最终 PCB 上的谢妮插座还是离得太近了。幸运的是，管子仍然适合，即使他们依偎在一起。是的，灯管底座都包括讨厌的 RGB LEDs 嘿，这正是顾客想要的。规格是一按按钮颜色就会改变；我们希望看到与温度相关的颜色渐变——蓝色代表“凉爽宜人”,红色代表“离开房间”你可以在裂口下面看到温度计的成品。

最近运行的谢妮项目继续有增无减，这一个有一个漂亮的外观，肯定会补充完成的情况。我们已经要求[Luca]让我们了解项目的最新进展，因此希望我们能够了解为什么需要圆形 PCB。在我们等待的时候，看看早期的[谢妮温度计，它有一个扭曲的条形图](https://hackaday.com/2011/02/14/nixie-tube-thermometer/)。
