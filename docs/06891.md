# 周五黑客聊天:微控制器上的 JavaScript

> 原文：<https://hackaday.com/2017/08/23/friday-hack-chat-javascript-on-microcontrollers/>

今天的微控制器比过去的 8051 功能更强大。现在，他们有了很棒的外设和 USB 设备接口。是时候用稍微现代一点的语言来给这些小芯片编程了。

[![](img/e441755fc9283026a7f1e02562f103f4.png) ](https://hackaday.com/wp-content/uploads/2017/08/espruinohackchat-01.png) [在本周五的黑客聊天](https://hackaday.io/event/26310-espruino-hack-chat)中，我们将讨论微控制器上的 JavaScript。[Gordon Williams]将加入我们来谈论[Espruino](https://www.espruino.com/)。这是一个运行在小型嵌入式芯片上的小型 JavaScript 解释器，具有调试接口，允许您在任何平台上对您的电路板进行编程，而无需任何外部编程硬件。

[戈登]是 Espruino 的主要开发者，迄今为止，他已经在 Kickstarter 上推出了[全尺寸 Espruino](http://hackaday.com/2013/09/03/the-javascript-of-things/) 和 [pico Espruino](https://www.kickstarter.com/projects/gfw/espruino-pico-javascript-on-a-usb-stick) ，都取得了惊人的成功。软件栈也非常受欢迎——它已经被移植到了[ESP8266](http://hackaday.com/2015/09/27/javascript-for-the-esp8266/)和其他几十个即将出现在物联网中的微控制器上。

在 Hack Chat 期间，我们将讨论微控制器上的解释语言、解释器设计和优化，特别强调使用 Espruino 创建设备，并通过 WiFi、蓝牙和其他疯狂的无线电将 Espruino 板放到互联网上。和往常一样，如果你想问问题，我们为每个人准备了一份电子表格。
