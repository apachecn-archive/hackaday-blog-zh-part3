# Hackaday 奖参赛作品:Dtto 模块化机器人

> 原文:[https://hack aday . com/2016/07/03/hack aday-prize-entry-dtto-modular-robot/](https://hackaday.com/2016/07/03/hackaday-prize-entry-dtto-modular-robot/)

一个机器人去探索未知，并自动完成明天的任务和接下来的任务，需要非常多才多艺。理想情况下，它可以是任何大小，任何形状，任何功能，像水一样没有形状，灵活而聪明。为了他的 Hackaday 奖参赛作品，[Alberto]正在建造这样一个模块化的、自我重新配置的机器人: [Dtto](https://hackaday.io/project/9976-dtto-v10-modular-robot) 。

[![ditto_family](../Images/1d429134834c62364b1ba3666935a996.png)](https://hackaday.com/wp-content/uploads/2016/06/ditto_family.jpg) 为了实现尽可能高的可重构性，[阿尔贝托的]机器人被设计成一个更大的机械有机体的构建模块。受类似的 [MTRAN III](https://unit.aist.go.jp/is/frrg/dsysd/mtran3/index.htm) 的启发，个体机器人具有两个驱动铰链，使它们具有灵活性和自行移动的能力。机器人两端的耦合机制允许小爬虫以各种配置自组装，并一起执行复杂的任务。它们可以链在一起形成一条蛇，变成一个轮子，甚至变成四条(或更多)腿的步行者。每个机器人有六个耦合面，允许四个方向的连接，几乎任何拓扑都是可能的。

[![](../Images/f082cf6e3984a8fc01db9619d0ad153d.png)](https://hackaday.com/2016/07/03/hackaday-prize-entry-dtto-modular-robot/11s/)[![](../Images/6ef30ff7bf2adad52b827af481b557d2.png)](https://hackaday.com/2016/07/03/hackaday-prize-entry-dtto-modular-robot/12s/)[![](../Images/176674a232cc7ee859e4f21f835528fe.png)](https://hackaday.com/2016/07/03/hackaday-prize-entry-dtto-modular-robot/14s/)[![](../Images/70ec83bd7bdeb787ab44d9c5322f3b79.png)](https://hackaday.com/2016/07/03/hackaday-prize-entry-dtto-modular-robot/15s/)

每个机器人包含两个强大的铰链伺服系统和三个较小的耦合机制。对齐磁铁有助于机器人在闩锁将它们锁定到位之前相互索引。这种巧妙的机构兼作顶出器，因此连接可以克服定位磁铁的作用力而松开。大部分电子产品，包括 Arduino Nano、蓝牙和 NRF24L01+模块，都密集地安装在机器人的一端，而另一端可以用来添加额外的功能，如相机模块、加速度计等。下面的视频显示四个 Dtto 机器人以蛇的形态爬行通过一根管子。
