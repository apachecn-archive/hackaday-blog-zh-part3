# 电动火车演示器

> 原文:[https://hack aday . com/2016/09/22/electric-train-demonstrator/](https://hackaday.com/2016/09/22/electric-train-demonstrator/)

如果你想激起孩子对科技的兴趣，最好是简单而又酷的东西。曾经有一段时间，向一个孩子展示一个水晶收音机如何从城市的另一端接收一个广播电台是合适的。现在，这是一个 yawner 作为孩子可能携带一个高科技手机与强大的无线电已经。您最新的 FPGA 项目可能太复杂而难以理解，您的 Arduino 电容计也太无聊而无法满足散热系数标准。

有一个老的学校项目通常被称为“电磁列车”，它工作得很好([俄亥俄州有一个关于它的 PDF 文件](http://www2.ece.ohio-state.edu/~anderson/Outreachfiles/Instructions_ElectricTrain.pdf))。你把一些裸铜线缠绕在一个管状的模板上来做一个隧道。然后，一节 AAA 电池和一些磁铁组成了火车。当你把火车放进隧道时，磁力推动火车穿过隧道。嗯，要么这样，要么它射出去。如果发生这种情况，请将列车调头再试一次。在互联网视频中有一些这样的例子，你可以在下面看到其中一个(来自[BeardedScienceGuy])。

这不是一个新的想法，但俄亥俄州的指令非常适合在课堂上使用，或者只是与一个好奇的孩子一起工作。因为这很简单，你可以和孩子们一起实际制作这个设备，或者让他们分组制作几个。

让孩子们对科技感兴趣可能是终极秘诀。如果你正在寻找其他想法，你可以尝试[一个电磁蛋蛋](https://hackaday.com/2016/01/20/kids-and-hacking-electromagnetic-eggs/)(或[更传统的一个](https://hackaday.com/2015/10/23/hacking-and-kids-the-one-hour-egg-drop/))或一点点[假装机器人练习](https://hackaday.com/2015/09/10/kids-and-hacking-blind-robotics/)。
