# 伸缩笔如何工作

> 原文：<https://hackaday.com/2015/09/22/how-retractable-pens-work/>

[Bill Hammack]，又名[EngineerGuy]又来了，这次他解释了可伸缩圆珠笔是如何工作的。

在这个精彩的视频中，他描述了这种机制的简单(但非常复杂)的工程设计，这种机制允许一支笔弹出圆珠笔机制，然后再放回原处。这是一个很好的例子来说明和解释一个复杂的概念，很像他的视频关于如何[你的相机的 CCD 传感器工作](http://hackaday.com/2012/05/15/engineerguy-explains-how-a-ccd-works/)。

然而，也许视频中最有趣的部分是他的即兴观察。首先开发可伸缩装置的帕克公司担心这种新设计会失败。所以他们直到几年后，这支笔大卖的时候，才把独特的派克箭夹放在笔上。看起来，虽然一些工程问题很容易解决，但目光短浅的会计师是一个更难解决的问题。
