# 向蚂蚁学习编程

> 原文:[https://hack aday . com/2018/03/19/learn-programming-from-ants/](https://hackaday.com/2018/03/19/learn-programming-from-ants/)

人类和昆虫思考的尺度不同，但是昆虫学家研究这些小生物的行为，所以它们不是一个完全的谜。一只蚂蚁或一立方毫米的灰质中没有多少智慧，但当它们都开始一起行动时，你会得到比各部分之和更大的东西。我们很容易陷入将所有智能或编程放在一个盒子里的陷阱，因为这就是我们的工作方式。相比之下，像微控制器和单板计算机这样的小脑容量既便宜又丰富。进入群体心态，新任务成为可能。

[Kevin Hartnett]谈到一篇论文，该论文研究了管理蚂蚁军的简单规则，蚂蚁军在遇到道路上的缺口时，用自己的身体作为桥梁。任何人只要有一把尺子和一张地图，就可以决定两地之间的最短路线，但军蚁在地面上实时执行这一优化，并且只有少数神经元可供其支配。两个简单的规则控制桥梁建造行为，这可能会在一些群体机器人的记忆库中留下一些空间。

一个更简单的群体心理的例子可能是机器人，当它们感觉到来自上方的红外波时，就会向前推进。通过这种方式，任何观察蜂群的人都可以观察到红外光何时出现以及它被导向何处。你可以用便宜的太阳能玩具汽车做同样的事情，但是我们已经可以看到可见光了。

我们并不是说应该招募蚂蚁来控制机器人，但我们也不反对人道地对待机器人。我们已经研究[蜂群机器人很长时间了](https://hackaday.com/2011/06/10/14-swarm-robot-kilobot-is-extremely-cool/)。
