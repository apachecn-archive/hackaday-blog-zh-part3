# 仁慈的我，感谢治愈

> 原文：<https://hackaday.com/2017/11/01/mercy-me-thanks-for-the-heals/>

有时候，你必须请专家来。[CorSec Props]，精细道具、服装等的建造者，被委托复制游戏《守望先锋》中仁慈的治疗杖。听起来很简单，但客户——一如既往地正确——要求它[旋转并点亮，就像最初的](https://www.youtube.com/watch?v=cWRvhIKP9Ao)一样。

为了查看电子设备，旋转头在移除一个螺钉后滑下。在内部，可充电的 18650 锂离子 3.7V 电池——通过直流到 DC 转换器——被提升到 5.5V，以运行 12V，120rpm 的电机。在全电压下，五线谱的头旋转得太快，因此它故意降低功率以获得更适合复制品的速度。

一圈 RGB 发光二极管和一对指向标尺顶端的发光二极管在黄色和蓝色之间切换。为了在这些不同的照明模式之间进行切换，双刀三掷开关被修改为像比我们在商店中拥有的更适合任务的三位置双刀双掷开关一样工作。

在电机轴上，一对螺柱插入五线谱顶端的一块丙烯酸树脂中。这阻止了它滑动，但也允许 LED 光扩散到顶部以及五线谱侧面的舷窗外。休息后请查看构建！
