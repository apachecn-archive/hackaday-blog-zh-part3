# 回流焊的好主意

> 原文：<https://hackaday.com/2017/07/13/a-bright-idea-for-reflow-soldering/>

黑客有多少，对表面贴装电路板进行回流的方法就有多少。今天，我们在列表中添加了另一种方法。[Dasaki] [将一盏卤素泛光灯改造成了一个 SMT 烤箱](http://therandomlab.blogspot.com/2017/07/halogen-floodlight-smt-reflow.html)，并且做了所有的花里胡哨。查看休息下方的视频。

实际上，在 T1 之前，我们已经看到了这种技术的低技术版本，但是我们不想在日常生活中使用它。[Dasaki]需要完成 100 块电路板，所以努力把它做好是值得的。

泛光灯是一个自然的选择——它有自己的内置室，当[Dasaki]测试它时，它以 5℃/秒的速度升至 300℃，没有停止的迹象。相比之下，烤面包机在 250 摄氏度左右就开始抱怨，如果它们真的能做到的话。除了灯本身，其余都是标准的温度控制器，一个固态继电器和一个与微控制器接口的热电偶。代码在他的 GitHub 上。

我们自己[偏爱煎锅](http://hackaday.com/2013/07/28/electric-skillet-reflow-soldering-guide/)，但是专业的卤素/红外线装置是[比尔·赫德](http://hackaday.com/2014/08/21/smt-and-thru-hole-desoldering/)的最爱。事实上，有太多的方法来完成这项工作，从[的粗糙](http://hackaday.com/2012/03/08/toaster-oven-reflow-soldering-roundup/)到[的近乎专业的](http://hackaday.com/2016/06/02/tools-of-the-trade-reflow/)，你一定会找到符合你的预算和宽容的回流焊炉设计。[Dasaki]的卤素灯烤箱看起来很值得加入我们的清单。
