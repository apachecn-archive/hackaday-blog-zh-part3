# 以太坊区块链的物联网

> 原文:[https://hack aday . com/2017/11/09/IOT-with-the-ether eum-区块链/](https://hackaday.com/2017/11/09/iot-with-the-ethereum-blockchain/)

今天，任何关注金融新闻的人肯定都会被比特币和其他加密货币的故事淹没。虽然大多数新闻都是关于这些硬币的潜在膨胀价值，以及它们在短短十年内的价格变化，但幕后还有其他有趣的事情。例如，货币以太坊允许在区块链实现各种分布式编程平台，[GusGorman402]已经在他的最新项目中利用了这一点。

他构建的设备基于一个 ESP8266，它连接到一个运行 Go 以太坊节点实例的路由器。本质上，他使用以太坊区块链来控制连接到 ESP8266 的 LED，使用以太坊的一个功能，称为[智能合约](https://www.coindesk.com/information/ethereum-smart-contracts-work/)。虽然这可能是一个误导性的名称，但智能合约基本上是一个自主程序，可以做程序员编写的几乎任何事情。虽然这是一种迂回的方式来编写“Hello World”程序，但与其他加密货币相比，它确实展示了以太坊平台的力量。

如果你对货币交易、区块链、密码学或计算的未来感兴趣，一定要在休息后看看详细的视频。这是一个奇怪的新工具，看看开发者和黑客如何使用它来完成我们从未想过的事情会很有趣。
