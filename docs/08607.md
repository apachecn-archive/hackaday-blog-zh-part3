# 从百万英里之外拍摄《星人》

> 原文:[https://hack aday . com/2018/02/18/photographing-starman-from a million-miles-away/](https://hackaday.com/2018/02/18/photographing-starman-from-a-million-miles-away/)

不管你喜不喜欢，将一辆跑车发射到太空是一个壮观的场景，并且在聚焦猎鹰重型飞船上做了很好的工作。这让[罗赫略]想知道——有没有可能从地球上抓拍一张《星人》的照片？

[Rogelio]对天体摄影游戏并不陌生，他拥有一个强大的双望远镜装置，具有恒星跟踪功能和冷冻 CCD，用于在弱光条件下减少噪音。多亏了[美国国家航空航天局 JPL 跟踪该物体并提供星历数据，确定 Tesla Roadster 的位置变得更加容易。](https://ssd.jpl.nasa.gov/horizons.cgi)

拍摄跑车需要一些投入——从[Rogelio]选择的拍摄地点来看，只有在凌晨 3 点到 5:30 之间才能看到。最初的尝试并不成功，但是熬了一夜之后，放弃不再是一个选择。几天后的回访也同样无望，并因云层覆盖而泡汤。

经过重要的分析后，问题才变得清楚——在美国宇航局网站上计算该物体的星历表时，[罗赫略]使用了标准坐标，而不是实际的成像位置。这造成了足够的误差，也意味着他们看错了地方。然而，多亏了望远镜的宽视野，在进一步分析后——Starman 被捕捉到了，不仅仅是静止的，[而是视频](https://www.youtube.com/watch?time_continue=6&v=m4r54ODsOM0)！

[Rogelio]的工作是实用天文学的一个很好的例子，如果你热衷于参与其中，[为什么不考虑建造你自己的恒星跟踪装置呢？](https://hackaday.com/2016/04/14/a-star-tracking-telescope-mount/)休息后的视频。
