# 星球大战机器人翻译头盔

> 原文：<https://hackaday.com/2016/02/20/star-wars-droid-translating-helmets/>

这件世博会服装怎么样？[AEInnovations]组装了一套可以用机器人语言互相对话的机器人头盔！

在你变得过于兴奋之前，我们必须让你知道，他们并没有真正翻译任何东西。第一，《星球大战》里没有完整的机器人语言，第二，这太荒谬了。不，他创造的有点简单，但仍然很棒，很聪明。

头盔内置了对讲机，所以两个戴头盔的人可以简单地用简单的古英语——嗯，或者克林贡语——互相交谈。但是当你说话时，头盔中的传感器会注意到你在说话，并开始广播一段随机的机器人讲话给其他人听。

结果是一场令人惊讶的机器人对话。听听吧！
