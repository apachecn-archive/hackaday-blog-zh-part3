# 自制焊台做得更好

> 原文：<https://hackaday.com/2015/10/11/homemade-soldering-station-does-it-better/>

焊台可能是黑客武器库中最重要的工具之一。问题是——好的产品很贵，有时擅长焊接和擅长焊接之间的唯一区别是你使用的工具的质量！这就是为什么[艾伯特]和[马蒂亚斯]决定制作他们自己的自制韦勒克隆体。

由于烙铁最重要的部分是一个好的尖端，他们使用的是 Weller 的针，他们只需要能够控制它。他们设计了一个 3D 打印外壳([源文件，此处为](http://www.thingiverse.com/thing:1064001))，用于一个 1.8 英寸的小液晶屏、一个 Arduino Pro Mini 和一个 MOSFET 屏蔽，以及他们选择的 12v 8A 电源。只有两个控制器——开/关和一个调节温度的电位计。

他们花了一段时间制作原型，最终结果是一个非常漂亮的产品。他们实际上卖了一些，但意识到他们没有赚到钱，所以他们停止提供它——相反，你可以自己从他们的 GitHub 获得所有的源文件。在【马提亚斯】博客上还有[更多信息。](https://debugginglab.wordpress.com/2014/10/30/soldering-station/)
