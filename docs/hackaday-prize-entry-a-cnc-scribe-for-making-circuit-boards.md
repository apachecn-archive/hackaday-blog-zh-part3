# Hackaday 奖参赛作品:制作电路板的 CNC 划线机

> 原文：<https://hackaday.com/2016/08/25/hackaday-prize-entry-a-cnc-scribe-for-making-circuit-boards/>

我们对任何可以用覆铜板制作 PCB 的设备都感兴趣，而这个 Hackaday 奖的参赛作品可能是迄今为止制作 PCB 最简单的机器。它被称为 Projecta，这是一种将 Eagle 和 KiCad 文件变成真正电路板的简单方法。

对于家用 PCB 制造商来说，有两种方法可以将覆铜板变成真正的电路板。首先是数控机床。将一块 FR4 放在切割机下面，你会得到一个电路板和许多玻璃纤维粉末。对于这个来说，Othermill 很棒，但是对于除了最雄心勃勃的周末战士之外的所有人来说，它有点贵。

家用 PCB 制造的第二种方法是化学蚀刻掉铜。[蚀刻抗蚀剂掩模可以用干膜抗蚀剂](http://hackaday.com/2014/01/05/testing-the-limits-of-home-pcb-etching/)或一直流行的激光打印机、杂志和层压机来制作。无论哪种方式，结果都是在你不想去除的铜上覆盖一层耐酸涂层。

虽然 Projecta 看起来和听起来都像一个微型数控机器，但它不会咬穿铜，也不会产生大量的玻璃纤维粉尘。在铜被用记号笔、记号笔或其他墨水基的抗蚀剂遮盖之后，该项目描绘出电路板的图案。当电路板从项目中出来时，电路板上有一个完美的电路图案，为蚀刻槽做好了准备。

这种把覆铜板放入 CNC 机器中，然后进行蚀刻的技术是我们以前没有见过的。这有一个很好的理由——如果你已经在切割机下面放了一块木板，你还不如在放的时候把铜啃掉。

仅仅因为我们以前没有见过这种技术，并不意味着它是一个坏主意。因为 Projecta 只是在板上刻划一点墨水，所以 CNC 机制不需要那么复杂。它不需要到处扔纺锤，而且 Projecta 可以很容易地降低价格。
