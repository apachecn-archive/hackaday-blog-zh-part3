# 使用珠宝套件对电子产品进行树脂封装

> 原文：<https://hackaday.com/2017/03/01/using-a-jewelry-kit-to-resin-encase-electronics/>

我们中的一些人已经在网上看到了透明树脂珠宝铸造套件，并开始有了想法。Hackaday 自己的【纳瓦·怀特福德】决定[冒险一试，分享成果](http://41j.com/blog/2017/02/embedding-electronics-in-transparent-resin/)。在从易贝购买了相当经济的透明树脂珠宝铸造套件后，一个简单的试运行包括将太阳能灯嵌入一些透明树脂中，以观察结果如何。结果很粗糙，但是很有希望。下面嵌入了一个简短的视频概述。

最大的难题是缺少合适的模具。[Nava]试图使用塑料袋和杯子作为合适模具的权宜之计，但这并不是没有缺陷，杯子需要在固化树脂被移除之前被打碎。尽管如此，结果还是不错的；混合需要仔细测量，固化过程也很漫长，但固化后的树脂就像广告承诺的那样诱人。混合会在混合物中引入许多气泡，但大多数气泡似乎会在固化过程中自行消失。这个快速测试的结果可能并不漂亮，但树脂似乎已经完成了它的交易，并提供了预期的平滑，清晰的完成。
