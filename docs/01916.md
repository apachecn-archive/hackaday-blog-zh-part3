# 旋转式低音炮结合了扬声器线圈和风扇

> 原文：<https://hackaday.com/2016/03/30/rotary-subwoofer-combines-a-speaker-coil-w-a-fan/>

当你把一个风扇和一个超重低音扬声器结合在一起会发生什么？显然，你得到的是一个[高效超低赫兹(3-5hz)旋转式低音炮！](http://www.youtube.com/watch?v=poo6zCFXtKM)

首先，信不信由你，这些东西真的存在。[克里斯]在看到旋转式低音炮的商业产品 [TRW-17](http://www.rotarywoofer.com/) 后，有了打造自己的想法。

这个概念非常简单。如果你使用一个巨大的低音炮，你可以得到低频响应，但它使用巨大的能量来移动一个巨大的扬声器线圈。那么如果你在一个更小的扬声器线圈上放点东西来增加气流呢？比如，风扇什么的？

[Chris]搭建的原型只是一个 300W 的放大器(180W RMS)，但正如你在下面的视频中看到的，它真的震动了他的房子。
