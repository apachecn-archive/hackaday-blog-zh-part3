# (合成)音乐的声音

> 原文:[https://hack aday . com/2017/07/15/the-sound-of-synthesized-music/](https://hackaday.com/2017/07/15/the-sound-of-synthesized-music/)

什么是 ADSR 包络发生器？如果你是一个音乐大黑客，你大概知道。如果你像我们其他人一样，你可能需要阅读[【Mich 的】帖子](https://synth.michd.me/module/adsr-eg/)来了解它是一个攻击-衰减-持续-释放(ADSR)包络发生器。还在迷茫？这是一种用于音乐合成的电路。你可以在下面的视频中看到该设备的演示。

在 Altair 出现之前——那是你实际上可以买到的第一台业余爱好者电脑——电子杂志上充斥着音乐合成器项目，这些项目与过去的模拟电脑有许多共同之处。许多人对此非常重视，然后计算机接管了集体意识，我们发现音乐黑客开始使用基于(数字)计算机的合成器。但是旧的模拟合成器设计不会消亡。ADSR 是基于一个古老的设计，他提供的信息量和附加内容值得一读，即使你不喜欢建造一个。

特别是，[Mich]发现原来的电路有一些问题，在建造一次电路后，他做了修复问题的工作。他详细记录了他的发现以及他是如何修复的。他还解释了很多术语，甚至像或非门触发器这样的基本元件是如何工作的。

如果您决定复制该单元，PCB 布局是在线的。还有一些关于为您的特定设置调整电路值的建议。

Hackaday 对[合成器项目](https://hackaday.com/2014/11/26/a-16-voice-homebrew-polyphonic-synth/)并不陌生。从微小的到巨大的，我们什么都见过。在这个范围内，你肯定能找到适合你音乐爱好的东西。
