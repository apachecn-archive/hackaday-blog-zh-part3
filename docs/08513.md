# 猫玩银球来获得奖励

> 原文:[https://hack aday . com/2018/02/08/cat-plays-the-silver-ball-for-treats/](https://hackaday.com/2018/02/08/cat-plays-the-silver-ball-for-treats/)

训练一只狗做一些有趣的事情是很容易的。他们渴望取悦他人。但是一只猫？不管有多少款待，大多数猫都有比学习技巧更好的事情要做。但是如果你把学习一个技巧变成一个自主游戏，他们可能会去玩。

这就是为猫设计的弹球机 Touchy Fishy 背后的想法。这是[金]为他的猫米迪制造的最新一代零食分发机。前一个版本的形状像一个狗的头，鼻子上有一个操纵杆。MIDI 非常擅长将操纵杆拉向自己，因此[Kim]决定尝试使用杠杆的新设计。

人类也喜欢挑战，而[金]这次想做一些纯机械的东西。最终产品多为弹簧和激光切割亚克力。MIDI 向下拉动弹簧杆，以弧形向上发射弹球。在其轨迹的顶端是一个被圈起来的旋转器。当弹球击中旋转器时，它将糖果扫向一个开口，糖果落到 MIDI 可以吃到的地方。最精彩的部分？旋转器还将被捕获的弹球返回到它的起点，这样 MIDI 可以一直玩到[Kim]厌倦了往洞里扔糖果。休息之后看着 MIDI 一路攀升到高分。

我们见过的大多数与猫相关的项目都是为了防止饥饿的猫在凌晨 3 点坐在主人的胸前，要求喂食。这里有一个更进一步的例子，让猫收集威弗球，用来支付少量的粗粮。
