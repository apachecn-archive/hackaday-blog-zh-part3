# 冷却过热的相机

> 原文：<https://hackaday.com/2017/06/29/chilling-a-hot-camera/>

[Eric]的相机有问题。它过热了。虽然如果[Eric]一次拍摄一张照片，这不是问题，但这款相机也有视频模式，可以连续拍摄多张照片，一张接一张。虽然使用时过热的相机可能是差的热工程的证据，但解决方案非常简单:[在背面绑一个巨大的散热器](https://www.youtube.com/watch?v=DsZEHjx0kpQ&feature=youtu.be)。这正是[埃里克]所做的，成品看起来很棒。

为这个应用程序选择的散热器是一个巨大的铝立方体，很可能取自一个旧的奔腾 4 CPU 冷却器。当然,[Eric]几乎不可能找到一个足够大的散热器来精确地安装他的相机背面，这意味着他必须打磨这个巨大散热器的侧面。[Eric]实际上是在他的钻床上用十字滑动虎钳和立铣刀做的。这肯定不是正确、理智或安全的做事方式，但我们将在下面让花生画廊对此进行评论。

散热器是通过一种我们在这里不太常见的技术来固定的——电线弯曲。[Eric]使用了 0.055 英寸(1.3 毫米)的钢琴线，并小心地将其弯曲，以缠绕散热器和相机机身。散热器给相机降温了吗？是的，这款相机的小上翻式屏幕使它成为一款非常方便的视频记录设备。你可以看看下面的视频。
