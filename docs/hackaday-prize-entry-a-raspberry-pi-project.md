# Hackaday 奖参赛作品:一个树莓 Pi 项目

> 原文：<https://hackaday.com/2016/06/23/hackaday-prize-entry-a-raspberry-pi-project/>

没有哪项技术比 Raspberry Pi 更有用，对下一代系统管理员和工程师的影响更大，也更具两极分化。花 35 美元(或仅仅 5 美元)，你就能得到一台完整的单板计算机，能够运行 Linux，功能强大到足以做有用的工作。为了 2016 年的 Hackaday 奖，【Arsenijs】[打造了完美的树莓 Pi 项目](https://hackaday.io/project/12122-raspberry-pi-project)。这就是你所期望的 Pi 驱动项目的一切，甚至更多。

虽然 Raspberry Pi 和围绕 Raspberry Pi 的社区因为大多数项目的相对简单的方法(实际上只是 casemods)而受到了很多批评，但这些项目的批评者忘记了微型个人计算机的历史背景。早在早期，当迷你 ITX 主板刚刚发布的时候，[网站就会弹出](http://www.mini-itx.com/)以迷你 ITX 机箱*为特色，其他什么都没有。*虽然塞进 NES 、[的电脑](http://www.mini-itx.com/2002/09/14/bit-tech-create-project-anenome)[、旧收音机](http://www.mini-itx.com/2002/09/20/the-nespc)、[或 AMD 标志](http://www.mini-itx.com/2003/10/07/the-amd-case)在今天已经是相当平庸的项目了，但我向你保证，它们在 15 年前也一样普通。然而，这些迷你 ITX 机箱的创造者成为了今天的硬件黑客。这一切都始于简单的构建，一个 Dremel，和一些 Bondo。

[Arsenijs]将他的树莓 Pi 项目比简单的 casemod 更进一步，从[一部树莓 Pi 智能手机](https://hackaday.io/project/7540-raspberry-pi-smartphone)、[树莓 Pi 安全系统](https://hackaday.io/project/4542-raspberry-pi-security-system)、[便携式树莓 Pi](https://hackaday.io/project/2287-ultimate-portable-raspberry-pi) 和[一部树莓 Pi 腕式电脑](https://hackaday.io/project/4282-raspberry-pi-wrist-computer)中汲取影响。这些都是他们自己的优秀项目，但是[Arsenijs]在这个项目上有他自己的特别之处:他使用了一个树莓 Pi 和一些树莓 Pi 附件。

虽然这个项目首先是一个 Raspberry Pi 项目，但[Arsenijs]并没有将自己局限于 Broadcom 芯片平台。当香蕉 Pi 问世时，这个 Raspberry Pi 项目背后的团队正忙着将该项目移植到 Odroid。这改变了一切，需要重构，然后宣布了 Orange Pi。跟上技术是很难的，这也是为什么这个树莓 Pi 项目还没有交付的一个重要因素。关于 Raspberry Pi 基金会你可以说很多，但至少他们的板子在向前兼容方面做了很好的尝试。

Raspberry Pi 项目是 Hackaday.io 上最受欢迎的项目之一，也是今年 Hackaday 奖最受欢迎的项目之一。这种受欢迎程度是否会转化为今年 Hackaday 奖的一个小胜利仍有待观察，但对[Arsenijs]来说，这似乎无关紧要；他已经在树莓派项目的前沿了。
