# 最后——大屏幕 Vectrex

> 原文:[https://hack aday . com/2017/11/02/finally-a-big-screen-vectrex/](https://hackaday.com/2017/11/02/finally-a-big-screen-vectrex/)

Vectrex 绝不是有史以来最受欢迎的控制台，但它是更独特的控制台之一。它避开了典型的基于光栅的渲染，而是依赖于基于矢量的显示。因为那个时代的普通家庭电视完全不能显示这种信号，所以 Vectrex 的屏幕是内置的。这让[Arcade Jason]想知道—[是否有可能将 Vectrex 连接到更大的屏幕上？](https://www.youtube.com/watch?v=Gj1tC_QvIKQ)

首先，必须找到一台合适的显示器。19V2000 被证明是一个很好的候选产品——19 英寸大得多，在过去几年的各种街机橱柜中都可以找到。从那时起，这个项目就变成了识别 Vectrex 的信号输出。[Arcade Jason]擅自修改了 Vectrex 板上的信号水平，然后通过调整板上的微调按钮来修复原始屏幕上现在过扫描的图像。Vectrex 的 X 和 Y 信号现在有所增强，它们被连接到更大的拱廊屏幕的输入端。对于 Z 信号，事情变得更糟——通常用于扬声器的沃尔玛“计算机放大器”被压入服务中以放大信号。

在这个视频中有很多电线在地毯上运行，但事实是，它工作得非常出色。未来的计划包括升级到更大的 23 英寸显示器，甚至可能试验彩色矢量显示器。这表明，即使在今天，威格斯仍然拥有一批铁杆粉丝。

也许你想试试这个，但需要先修复你原来的 Vectrex 屏幕？不要害怕，那也是可能的。
