# 一棵讨厌的圣诞树

> 原文：<https://hackaday.com/2015/12/26/a-nerdy-xmas-tree/>

和妻子分居后，他和三岁的女儿一起庆祝第一个圣诞节，[亚历克]决定最终得到一棵树。他不信教，但对孩子们来说这是一个有趣的节日，所以他决定融入其中。但是他不得不自己制作 LED 饰品。

他开始买了一棵塑料圣诞树，但很快就后悔了。所以他把它还了回去，取而代之的是一棵活的诺福克松树——如果你把它养在花盆里，它会作为室内植物全年生长。不必处理许多庆祝圣诞节的人不得不处理的塑料与谋杀的困境。

他的第一步是用 DC 电力给圣诞树布线，让他可以连接他未来的装饰品。目前，他只做了三个——但目的是随着他的女儿长大(希望她喜欢修补),在未来几年里和他一起做新的！

首先，他有一个触摸敏感的冰柱，一个发光的眼球，和一个机器人“嗬嗬嗬”——老实说，最后一个听起来相当令人毛骨悚然。看看下面的视频，也许能为你的下一个古怪的圣诞装饰品找到一些灵感！
