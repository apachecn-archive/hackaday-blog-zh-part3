# ChaiBot:一个实用的茶水机器人

> 原文：<https://hackaday.com/2016/04/06/chaibot-a-tea-robot-steeped-in-utility/>

从表面上看，一杯茶的制作很简单。烧热一些水，放入茶叶，等它泡开。等待时间是一个味道的问题，在为时已晚之前移除袋子或注射器是绝对重要的。否则，你最终会得到一种几乎完全不像茶的液体。

[阿德里安]和他的儿子经常会发现他们在浸泡过程中迷失在谈话中，并让它持续太长时间。但那是在他们制造出 ChaiBot，一个自动茶水看护器之前。这台外观精美的机器使用一个旧的 CD 驱动器来提升和降低茶叶袋，茶叶袋由一片薄的不锈钢网固定。一旦袋子放下，[阿德里安]将热水倒入杯子。水的重量由杯子切口下的电容传感器检测，这触发计时器开始倒计时至完美的杯数。

ChaiBot 最酷的特性之一是内置的循环。每一分钟，袋子都被短暂地取出并重新插入，搅动水，使浸泡更加均匀。因为沏好茶的最后一步是在茶凉之前喝掉它，ChaiBot 向[Adrian]的手机发送了一条推送通知。请务必在休息后观看演示。

这是另一个基于 CD 驱动器的茶机器人,我们之前报道过。它不是很漂亮，但它完成了任务。如果你不是一个喝茶时会走神的人，但也不喜欢看钟，这里有[一个适合你口袋的紧凑计时器](http://hackaday.com/2016/03/17/tiny-tea-timer-for-your-perfect-cuppa/)。
