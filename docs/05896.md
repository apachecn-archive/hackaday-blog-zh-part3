# Hackaday 奖参赛作品:协作净水

> 原文：<https://hackaday.com/2017/05/16/hackaday-prize-entry-collaborative-water-purification/>

查看任何关于水处理厂的描述，你会发现其中包括“混凝罐”这几个字。这些水处理厂在凝结什么？你不会想知道的。他们是怎么做到的？含有化学物质和矿物质。显然，还有别的办法。

为了他们的 Hackaday 奖参赛作品，[Ryan]，[designbybeck]，[Clint]，[Wanda]和[Maker Mark] [正在研究电凝法](https://hackaday.io/project/20812-make-water-collaborative-water-purification)。这是一种用电将污染物从水中分离出来的化学泡沫的替代品。

现在，这些测试的规模比你在水处理厂找到的数万加仑要小得多。事实上，试验台只是一个 16 盎司的梅森罐。虽然这还不足以将污染物从家庭供水中沉淀出来，但它足以证明这个概念。

该团队在这个构建中使用了两个电极，一个是铝，一个是铁。这些电极通过鳄鱼引线连接到他们建造的电子板。这种电子板基本上只是一个 H 桥(用来反转场发射器的极性，防止电极上粘糊糊的东西堆积)和几个电源连接器。结果令人鼓舞；他们有几段延时录像，展示了一个泥瓦匠罐子里的污水在电力的作用下变得清澈。这是一个很棒的项目，有一些很棒的文档。该团队已经有了一堆关于他们项目的更新和如何复制他们硬件的说明。你可以看看下面的视频。
