# 哑盒子？让它变得真正聪明！

> 原文：<https://hackaday.com/2017/11/23/dumb-box-make-it-really-smart/>

[斯蒂芬·哈里森]的[真正的智能盒子](https://www.hackster.io/TinamousSteve/really-smart-box-a3fba8)是一个伟大的概念，它既是一个简单的想法，同时又非常聪明。真正智能的盒子并不是真正的盒子；这是一个嵌入式平台，可以做成任何大小，旨在将任何愚蠢的储物盒变成一个帮助管理和跟踪任何种类的库存或消耗品的水平和使用情况的储物盒。

它通过测量堆在它上面的东西的重量来做到这一点，同时还监控温度和湿度。该平台将这些信息以无线方式传送到后端，从而可以做出关于库存水平、使用和存储条件监控的决策。显然，它最适用于易耗品或其他来来去去的库存。真正智能的盒子平台是电池供电的，但大部分时间都在休眠，以最大限度地延长电池寿命。原型使用 SigFox 物联网框架进行无线数据传输，我们之前已经在[无线游泳池监控器](https://hackaday.com/2016/08/07/jump-in-when-the-water-is-just-right-with-a-wireless-swimming-pool-thermometer/)中见过。

这仍然只是一个原型，还有一些错误要解决，但它很有效，[斯蒂芬]打算将原型放入剑桥制造空间，任务是存储和监控 3D 打印机细丝。下面嵌入了一个简短的演示视频。
