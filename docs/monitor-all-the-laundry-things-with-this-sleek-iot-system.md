# 使用这款时尚的物联网系统监控所有衣物

> 原文：<https://hackaday.com/2017/02/18/monitor-all-the-laundry-things-with-this-sleek-iot-system/>

如果你像我们一样生活在致命的恐惧中，害怕有人在你度假时闯入你的房子，并在洗衣服时点燃烘干机，[这款功能齐全的物联网洗衣房监控器](https://www.instructables.com/id/Laundry-Room-Monitor/)就是为你准备的。还有一辆校车。但是不要问校车的事。

在[seasider1960]所描述的“范围蔓延的经典案例”中，洗衣房发生的事情几乎没有逃过这个执行良好的项目的注意。它最初是一个水传感器，用来防止导致楼下损坏的漏水事件再次发生。但是一旦你开始了，为什么不走得太远呢？[seasider1960]增加了电流感应功能，以了解洗衣机和烘干机何时运行，以及增加用电量。一个温度传感器观察烘干机的通风口，并通过发出令人讨厌的本地警报来警告前面提到的悲剧的可能性——这就是校车来的地方。整个系统还连接到 Blynk 进行物联网监控，你可以在下面的视频中听到同样令人讨厌的警报。哦，还有测试每一个警报的按钮和在网上记录再订购洗衣用品的按钮。

最近我们已经看到了大量的洗衣监控项目，它们都有各自的优点。但是你一定会喜欢[seasider1960]的造型。不锈钢面板和墙内安装造就了时尚、专业的外观，与建筑令人毛骨悚然的本质相得益彰。
