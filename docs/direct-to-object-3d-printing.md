# 直接面向对象的 3D 打印

> 原文：<https://hackaday.com/2016/11/16/direct-to-object-3d-printing/>

随着熔丝 3D 打印机的专利在 2013 年开始到期，全球各地的黑客和制造商开始在他们的车库、车间和黑客空间制作 3D 物体。整个行业和企业从桌面 3D 打印革命中涌现出来，并为 DIY 社区开创了一个新时代。在过去的几年里，黑客们一直在推动这项技术的极限，他们使用越来越多的奇异细丝材料，探索新颖和创新的方法来制作多色 3D 打印。然而，落后于革命的一个领域是将 3D 打印完成为最终产品。我们愿意用一卷 4 米长的 5v 3.5 安培新像素打赌，使用 3D 打印机的艺术家和工匠与传统黑客和制造商一样多。这些勇敢的灵魂目前被迫使用绘画和刷子的穴居人技术，以便将颜色应用到他们的印刷品上。我们 Hackaday 特此声明这是不可接受的。

我们应该能够将墨水直接打印到 3D 打印物体上。继续读下去，润滑你头脑中已经转动的齿轮，让我们来解决这个问题。

## 见见施乐 14.5 万美元的回答

正如您可能已经猜到的，有人已经提供了这个问题的解决方案，但是它不是您所期望的。不到 90 天前，施乐公司推出了其[直接面向物体的打印机](https://www.news.xerox.com/news/Xerox-Direct-to-Object-Inkjet-Printer)，它可以在小到 Nuka 可乐帽、大到橄榄球头盔的物体上打印。它们是为你的产品定制的，价格和房子差不多。最高分辨率为 1,200 DPI，每小时输出 30 个对象，这是一台令人印象深刻的机器。除了试图弄清楚它是如何工作的之外，它对我们的社区也是绝对无用的。据我们所知，它使用的打印头类似于典型的喷墨打印机。打印头在一个双轴系统上，我们称之为 X 和 Y，X 垂直于要打印的物体，Y 平行。有一种扫描仪允许打印头在 X 轴上移入和移出，以符合对象的形状，而 Y 轴允许典型的逐行来回打印。我们不确定最后一个轴…是移动对象以允许打印整个表面，还是打印头在单独的“Z”轴上移动。

这种方法的一个明显的缺点是被打印的物体必须相当光滑和形状规则。像瓶子和玻璃杯之类的东西。

已经有其他尝试创造直接面向物体的打印机，例如欧洲一家名为 Mimaki 的公司生产的 T2 UJF-3042。这些打印机适用于更小的东西，如墨水笔和手机外壳，通常被称为直接成型(DTS)打印机。但是它们也有很多和复印机一样的问题。

## 2D —> 3D

我们之前已经讨论过其中的一些方法，但它们应该被重新审视，因为它们是将颜色添加到 3D 物体上的成熟方法。[计算机热成型](http://hackaday.com/2016/05/04/creating-full-color-images-on-thermoformed-parts/#more-203644)是一种制作铸件的技术，塑料片被软化，然后通过真空在铸件周围成型。那么，如果你在塑料纸上打印艺术呢？那么你就有办法把你的作品放到你的 3D 物体上了！这项技术的伟大之处在于它在我们社区的掌握之中。你需要制作自己的真空机器，[，我们在](http://hackaday.com/2014/10/19/diy-vacuum-former-on-the-cheap/)之前已经介绍过了。你使用一台 3D 打印机，用 PLA 细丝和石膏制作铸件，然后使用热量和真空在物体上形成彩色塑料片。

另一个需要再次提及的技术是[水版印刷](http://hackaday.com/2015/05/13/printing-photorealistic-images-on-3d-objects/)。它类似于热成型技术，因为艺术是应用于 2D 塑料片。使用 hyrographic printing，然后将透明的装饰塑料片放入水中，使其符合 3D 物体。

## 五轴直接目标打印机可能吗？

在上面的段落中，我们概述了两种将艺术添加到 3D 物体上的方法——直接添加到物体上和印刷 2D 艺术，然后将其形成 3D 物体。两者各有利弊，但我们更喜欢直接面向对象的方法。据我们所知，有一种直接面向对象的印刷机还不存在。五轴直接面向对象打印机怎么样？你需要一个五轴数控机床，并用某种类型的喷墨打印头代替工具头。这可能有点棘手，但应该是可能的。这种机器将允许人们真正直接打印到 3D 打印物体上。黑客可以在他或她的车库里制作。
