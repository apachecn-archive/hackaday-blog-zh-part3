# 30 亿台设备和一台世嘉 Genesis 运行 Java

> 原文:[https://hack aday . com/2015/11/23/30 亿-设备-和-a-sega-genesis-run-java/](https://hackaday.com/2015/11/23/3-billion-devices-and-a-sega-genesis-run-java/)

几年前，[Mike]的朋友给了他一台旧的 Sega Genesis，附带了非常酷而且有点罕见的 SegaCD 驱动器。SegaCD 给了他一个想法——虽然在真正的 Genesis 主机上刻录光盘和玩自制游戏并不容易，但每个人都有一个 CD 刻录机。[Mike]开始编写他的演示，然后意识到在 68000 上添加 Java 很容易。结果是 Java 在 30 亿台设备上运行，Sega Genesis 。

这个项目是围绕着 [Java Grinder](http://www.mikekohn.net/micro/java_grinder.php) 这个 Java 字节码编译器构建的，它将把类、工厂和所有可怕的 Java design . design . pattern . pattern . patterns()编译成汇编语言。Java Grinder 已经支持了很多平台，包括 Commodore 64、TI99，多亏了乔·戴维孙的一些工作，[苹果 IIgs](http://www.mikekohn.net/micro/apple_iigs_java.php)

有了字节码编译器、汇编器和用于 Sega 专用硬件的 API，[Mike]开始构建他的演示。由于这是世嘉，它需要在开始的“Sega”的声音。[迈克]最后录下了他说“爪哇！”这通过创世纪上的 Z80 播放。

完整的演示——可在下面的模拟格式中查看——拥有您期望从适当的演示中获得的一切。三分钟长的演示中有星空、跳舞的精灵，甚至还有曼德尔布罗图案。
