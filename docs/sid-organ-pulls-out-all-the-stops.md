# 希德·风琴使出浑身解数

> 原文：<https://hackaday.com/2018/01/02/sid-organ-pulls-out-all-the-stops/>

有人把这个器官留在雨中，但[Tinkartank]拯救了它，并赋予了它作为 SID 控制器的新生命。你会问，SID 是什么？这是 C64 中使用的声音芯片 Commodore，这是一种在复古游戏玩家中备受尊崇的非凡芯片，远远领先于其时代。

除了键盘组件之外，他把所有东西都扔掉了。现在，每次按键都会驱动一个瞬时按钮，这些按钮都通过另一个项目留下的一些 I/O 扩展卡连接到 Arduino Mega。Mega 驱动 MOS6581 SID 芯片，产生这些甜美的 chiptunes。使用 Eurorack 模块，有四种 CV 输出用于扩展风琴的视野。

我们最喜欢的部分是停止旋钮的重复使用——特别是它们的操作方式和以前一样。旋钮仍然在技术上控制声音，但以一种新的方式——现在他们转动锅来改变琶音、频率或他想要他们做的任何事情。

未来的计划围绕着切换到一个十几岁，以帮助解决记忆问题。虽然这是一项正在进行的工作，但这个器官已经有了大量的功能。休息之后一定要去看看。

一旦你潜入 chiptunes 兔子洞，你可能会想把它们带到任何地方。当你到了那一步，[这里有一个便携式 SID 播放器](https://hackaday.com/2011/06/03/portable-sid-plays-chiptunes/)。一个西德曼，如果你愿意的话。
