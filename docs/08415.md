# 所有你希望知道傅立叶变换，但又不敢问的东西

> 原文:[https://hack aday . com/2018/02/10/all-the-stuff-you-wish-you-know-on-Fourier-transforms-but are-fear-to-quest/](https://hackaday.com/2018/02/10/all-the-stuff-you-wished-you-knew-about-fourier-transforms-but-were-afraid-to-ask/)

傅立叶变换支撑着我们如此多的技术生活，在大多数情况下，我们可能没有意识到这一点。将波形以数学方式分解为频率成分以及将频率成分分解为波形的能力是数字信号处理领域的基础，DSP 已经成为许多我们习以为常的电子设备的重要组成部分。

虽然我们大多数人都知道傅立叶变换是什么，但很少有人知道它是如何工作的。它们是从库中调用的函数，而不是在自身中执行的函数。即使在学校或大学课程中教授它们，它们仍然是并非所有学生都能“理解”的东西，如果(正如你的抄写员所做的那样)你有一个不称职的数学讲师，你就会倒霉。

如果傅立叶变换对你来说有点神秘，那么休息时间下面的视频非常值得一看。在书中[Grant Sanderson]通过一系列简单的图形例子解释了这些问题，这种风格也许是那些用粉笔交谈的数学老师应该效仿的。你可能仍然只能通过一个库来使用傅立叶变换，但是在看完这个视频之后，也许它们的一些秘密会被揭示。
