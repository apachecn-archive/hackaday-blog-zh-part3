# 周五聊天:关于超感官知觉的一切

> 原文:[https://hack aday . com/2018/03/07/Friday-hack-chat-everything-about-the-esp/](https://hackaday.com/2018/03/07/friday-hack-chat-everything-about-the-esp/)

当 ESP-8266 第一次到达时，它是一个奇迹。花两美元，你可以买一个简单的模块，作为 WiFi 网络和微控制器项目之间的桥梁。它理解海斯命令集，它不使用太多的权力，而且，如前所述，它只花了两美元。廉价易得的物联网概念就在眼前。

[![](../Images/f9022490591919285527000adbc9ddf2.png)](https://hackaday.com/wp-content/uploads/2018/03/espeverythinghackchat.png) 然后黑客们弄清楚了 ESP-8266 内部到底发生了什么。这是一个成熟的微控制器。你可以在上面放些 Lua 的东西。你可以用 Arduino IDE 对它进行编程。它有无线网络。这是过去十年中最伟大的微控制器发布，它来自一家没人听说过的公司。

从那时起，ESP 生态系统已经开花结果，并且有一个新的 ESP 正在酝酿中。ESP-32 是一个功能更强大的 WiFi 和蓝牙芯片，编程简单，价格为 3 美元。微控制器从来没有这么酷过。

在本周的黑客聊天中，我们将谈论所有关于 ESP 的话题。我们这次黑客聊天的嘉宾应该不需要介绍，但是如果你不熟悉的话，[Sprite_tm] [在他的键盘上玩视频游戏](https://hackaday.com/2014/11/23/sprite_tms-keyboard-plays-snake/)，[在硬盘上安装了 Linux](https://hackaday.com/2013/08/02/sprite_tm-ohm2013-talk-hacking-hard-drive-controller-chips/)。他还在 ESP-8266 和 ESP-32 背后的公司 Espressif 工作，在那里他将自己的技能应用于微型游戏机和微型苹果电脑。

在本周的 Hack Chat 中，我们将讨论有关 ESP 的一切，包括外设、超低功耗、SIP 封装以及 ESP 系列即将推出的产品。一如既往地欢迎您为[Sprite]提交问题；只需在黑客聊天页面上添加这些内容作为评论[。](https://hackaday.io/event/69323-esp-everything-chat-with-sprite-tm)
