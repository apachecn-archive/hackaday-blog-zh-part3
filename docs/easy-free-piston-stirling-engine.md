# 简易自由活塞斯特林发动机

> 原文：<https://hackaday.com/2017/04/07/easy-free-piston-stirling-engine/>

斯特林发动机是非常酷的机器，是由牧师罗伯特·斯特林博士在 1816 年发明的，可以与蒸汽机相媲美，是有史以来最高效的发动机之一。建造一个是非常有益的体验，但它有一定的难度。然而，【阿提拉·刀锋】版本的[自由活塞式斯特林发动机](https://www.youtube.com/watch?v=YiDs0R7E1g0&t=5s)非常简单，几分钟就能造出来。

要制造这台发动机，你只需要一根试管、钢丝绒、一只乳胶手套、一个 O 型圈和一些电线。正如你在视频中看到的那样，构造非常简单。整个引擎在线框上晃动，这也使它不同于你可以在网上看到的大多数其他斯特林发动机。自由活塞只是斯特林发动机几种可能配置中的一种。最常见的一种，是 beta 型，通常是用汽水罐做的，但是比[阿提拉刀锋]的引擎要难造得多。

这绝对是一个有趣的项目，你可能想尝试，也是一个伟大的方式来学习热力学概念。即使你没有建造这个特别的版本，也有许多其他的可能性主要使用家庭用品，或者你也可以查看斯特林发动机背后非常有趣的 T2 历史。
