# Hackaday 奖参赛作品:MCXY–迷你激光切割铝制 3D 打印机

> 原文：<https://hackaday.com/2017/04/24/hackaday-prize-entry-mcxy-mini-laser-cut-aluminum-3d-printer/>

随着廉价的 3D 打印机在常见的中国网站上随处可见，你可能会认为没有多少空间再进行另一个国产 3D 打印机项目了。幸运的是，3D 打印机制作爱好者社区并不这么认为。

[Bobricius]有一个相当不错的 3D 打印机设计正在进行中，我们认为你会喜欢的。它遵循 MakerBot/Ultimaker 的构造风格，因为它是一个盒子而不是一个门架，它由 CNC 切割的铝组装而成，具有坚固和令人愉悦的效果。Whar 使它与众不同的是它的尺寸，只有 190 x 190 x 251 毫米，80 x 80 x 80 毫米的打印量，它是小小的*。你可能想知道为什么这可能是一项资产，但当你考虑到他已经有一个更大的打印机时，很明显，一些小巧便携的快速打印设备可能是一项资产。*

 *对于一台自制的 3D 打印机来说，不同寻常的是，它没有 3D 打印部件，而是完全由激光切割而成。不同寻常的是，所有的 CAD 工作都是在 EAGLE 完成的，EAGLE 以 PCB 工作而闻名。我们今天展示的是一个正在进行中的作品，因为它是一个 Hackaday 奖的参赛作品，但看起来成品将是一个小宝石。

自制的 3D 打印机可能会特别令人印象深刻，例如，[我们已经向您展示了这款出色的 SLA 打印机](http://hackaday.com/2014/03/22/home-made-resin-based-3d-printer-is-incredible/)。
