# PUFFER:智能手机大小的行星探测器

> 原文:[https://hack aday . com/2017/04/08/puffer-a-smart phone-size-planetary-explorer/](https://hackaday.com/2017/04/08/puffer-a-smartphone-sized-planetary-explorer/)

火星和欧罗巴上有可爱机器人的空间吗？[美国宇航局]——与[加州大学伯克利分校]和[遥远焦点公司]合作——有了答案: [PUFFER，一个受折纸启发的机器人](https://www.nasa.gov/feature/jpl/origami-inspired-robot-can-hitch-a-ride-with-a-rover)。

PUFFER 代表弹出式平板折叠探测机器人，它能够感知物体，并通过将自己“折叠”成更小的尺寸来适应角落和缝隙，从而相应地调整自己的轮廓。它的设计使得多架飞机可以驻留在一艘更大的飞船内，然后被部署去侦察否则无法到达的地形。洞穴、熔岩管和遮蔽的岩石突出物可能会隐藏有机物质，是探索的首选。成群的河豚会将收集到的信息发送回母舰，以传达给地球母亲。

我们嵌入了机器人折叠轮子通过低桥的视频。您可以在项目页面上查看[的功能范围。](https://gameon.nasa.gov/projects/puffer/)
