# 保罗:一个机器人和它的球

> 原文:[https://hackaday.com/2017/03/07/paul-a-robot-and-its-ball/](https://hackaday.com/2017/03/07/paul-a-robot-and-its-ball/)

保罗是同类型中最酷的机器人之一，也可能是最酷的机器人之一。性格？检查。全轮？检查。无端的英勇壮举？检查。保罗是一个球平衡机器人。

在引擎盖下，保罗并不那么奇怪。他有两个微控制器，一个负责平衡和运动学，另一个处理发光二极管，语音处理器，扬声器，遥控器和其他装饰。但是数学！保罗是一个站在保龄球顶上的圆柱体，所以它能向前滚动的唯一方法就是向前倾斜。但是当然，它不能倾斜太多，因为它也必须保持平衡。这绝对是最不合理的移动方式。我们喜欢它。

[Jochen]很好，把所有东西都放在了 GitHub 上，所以你可以看到它是怎么做的，尽管它看起来像魔术一样。我们谅你也不敢看下面的视频，至少不会感到一丝同情的痛苦。)他脸朝下摔了下来。他康复了吗？我们很想知道！

保罗只是 2017 年黑客日科幻竞赛中的明星机器人之一，所以如果你还不满足，就去那里吧。
