# Hackaday 奖参赛作品:平衡疗法的 SoleSense

> 原文:[https://hack aday . com/2017/09/16/hack aday-prize-entry-sole sense-for-balance-therapy/](https://hackaday.com/2017/09/16/hackaday-prize-entry-solesense-for-balance-therapy/)

恢复病人平衡感受损的脑损伤不是一件容易的事情。目前的解决方案只有在患者达到某个阈值时才会触发，到那时，可能已经为时过晚，无法顺利恢复。[Simon Merrett]的 [SoleSense](http://hackaday.io/project/27067-solesense) 被设计成能像人类天生的平衡感一样给出持续的反馈。治疗师希望这将有助于通过更接近地模仿我们大多数人的成长环境来帮助康复。

SoleSense 依靠安装在脚下的电容传感器来了解患者的体重。[OSHPark]正在提供第一轮柔性印刷电路板，因此一些幸运的鞋底将获得紫色衬垫。

除了恢复之外，像这样的设备可以教你更好的姿势，或者可能增强平衡感。这可以提高体能。谁知道呢，我们一直在寻找感知世界的新方式。

重新映射感官是一种流行的辅助技术，声音是 SoleSense 搭载的理想选择，因为大脑损伤不太可能影响听力。当然，感官可以被重新映射甚至创造。你可以[获得磁北感](http://hackaday.com/2008/06/13/wearable-haptic-devices-bestow-sixth-senses/)或[扩大你能感知的光的范围](http://hackaday.com/2013/12/09/future-tech-and-upgrading-your-brain/)。
