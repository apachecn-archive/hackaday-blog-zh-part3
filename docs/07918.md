# 垃圾场钢筋混凝土转换看起来像疯狂的麦克斯额外

> 原文:[https://hack aday . com/2017/12/07/junk yard-RC-conversion-looks-like-mad-max-extra/](https://hackaday.com/2017/12/07/junkyard-rc-conversion-looks-like-mad-max-extra/)

这些年来，我们注意到有一部分黑客喜欢把现实生活中的车辆变成遥控汽车。这些交通工具通常会在短时间内被摧毁，要么进行荒谬的跳跃，要么就把它们撞成碎片，直到什么都不剩。事实上，如果我们有一辆全尺寸的遥控车，这可能是我们大多数人都会做的事情，所以没有抱怨。

通常，用于这些改装的捐赠车辆通常较旧且便宜。这才说得通，为什么要花很多钱买一辆你打算毁掉的车呢？尽管如此，最近完成的钢筋混凝土转换(威廉·福斯特)可能会拿走蛋糕。我们不知道他的捐赠车辆有多少是故意“古董”，但总的来说，这东西看起来像是从某个湖底拖上来的。据推测，他得到了很多。

[![](../Images/54c286105d3c3e8883b35060c653658c.png)](https://hackaday.com/wp-content/uploads/2017/12/rcmax_detail.jpg) 发布到 YouTube 上的视频主要是关于(威廉)驾驶着他的作品到处跑(有时甚至是坐在后座上)，但在视频的后半部分，有一个关于用来让这堆铁锈移动的硬件的快速介绍。

一个[标准 RC 发射器和接收器组合](https://hackaday.com/2017/11/20/can-commodity-rc-controllers-stay-relevant/)用于控制安装在中控台上的一对 Arduinos，它们依次连接到外部步进驱动器。轮子通过链条和链轮传动装置转动，踏板由自制装置推动，这些装置看起来像是由用于 3D 打印机的丝杠制成的。

总而言之，看起来[威廉]已经用你能在亚马逊或易贝上买到的商品硬件设计出了一个相当灵敏的控制系统。不确定我们是否会亲自指挥这件事，但每个人都有自己的想法。
