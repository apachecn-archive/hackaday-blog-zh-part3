# 远程饲养的小鸭子

> 原文：<https://hackaday.com/2016/09/08/ducklings-raised-by-remote/>

我们不确定使用遥控电子鸭诱饵养鸭是否绝对必要，但人们有更奇怪的爱好。

[这段来自[想象力丰富的家伙]的 YouTube 视频](https://www.youtube.com/watch?v=6S1ad6b2wmA)(嵌入下文)记录了遥控电子动画的一个令人印象深刻的壮举，不遗余力地让遥控鸭子的“父母”变得现实。视频中有大量的细节，从加工必要的小零件到在必要的地方自由使用热熔胶。

我们尤其喜欢鸭子的声道模拟:粘在汽水瓶瓶颈上的扬声器发出(对我们人类的耳朵来说)非常准确的鸭子声音。切断钢筋混凝土直升机叶片，使舵也是一个很好的黑客。这里有很多东西要学。你知道，当你需要做一只机器鸭的时候。

自然，这不是第一次有人把遥控潜艇绑在鸭子诱饵上。但这绝对是我们见过的最精致的。
