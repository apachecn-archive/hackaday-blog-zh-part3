# 尼龙纤维人造肌肉——由激光驱动！

> 原文:[https://hack aday . com/2016/12/08/nylon-fibre-artificial-muscles-powered-by-lasers/](https://hackaday.com/2016/12/08/nylon-fibre-artificial-muscles-powered-by-lasers/)

如果我们有负担得起的人造肌肉，我们可能会看到假肢、机器人、外骨骼、植入物等方面的快速进步。除了复制我们许多人认为理所当然的肌肉奇迹之外，成本是主要障碍之一，可行的解决方案似乎还有很长的路要走。麻省理工学院的一组研究人员展示了尼龙纤维可以用作合成肌肉，为这些问题提供了一个潜在的答案。

一些聚合物纤维材料具有加热时直径增大而长度减小的奇怪特性。利用这一点，麻省理工学院的团队能够塑造尼龙纤维——使用许多热源，即激光——可以引导它向特定方向弯曲。更复杂的运动需要一系列的热源，这还不现实，但是看到尼龙纤维跳舞会激发想象力。
