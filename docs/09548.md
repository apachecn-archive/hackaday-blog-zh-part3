# 星期五黑客聊天:让编程更容易

> 原文:[https://hack aday . com/2018/05/23/Friday-hack-chat-making-programming-easy/](https://hackaday.com/2018/05/23/friday-hack-chat-making-programming-easier/)

图形编程语言有着悠久的历史。有些人不喜欢编码，对他们来说，图形化编程语言用易于理解的框和线代替了分号和括号。

[![](../Images/a018c12448a6063404fbd9da9b876270.png)](https://hackaday.com/wp-content/uploads/2018/05/gpg_chatv2.png) 本周五，我们将[和【Boian Mitov】一起讨论图形化编程语言](https://hackaday.io/event/158266-making-programming-easier)。他是一名软件开发人员，Mitov Software 的创始人，也是嵌入式领域的图形编程语言 Visuino 的创始人。他专门研究视频、音频、DSP、DAQ、工业自动化、通信、计算机视觉、人工智能以及并行和分布式计算。[Boian]是 OpenWire 开源技术、IGDI+开源库、VideoLab、SignalLab、AudioLab、PlotLab、InstrumentLab、VisionLab、IntelligenceLab、AnimationLab、LogicLab、CommunicationLab 和 ControlLab 库、OpenWire Studio、Visuino 的作者，也是“Visual C++”技术的作者。

在这次聊天中，我们将讨论如何简化微控制器编程。这次讨论的焦点是 Visuino ，一个图形化编程环境。Visuino 允许任何人通过连接电线和选择一些逻辑来编程 Arduino、Teensy 或 ESP。你可以把它看作是在编程环境之上的一个台阶，这个编程环境是伴随着 Lego Mindstorms、Scratch 或者麻省理工学院早期推出的任何东西而来的。
