# 铁砧射击:令人敬畏还是不计后果？

> 原文：<https://hackaday.com/2015/10/11/anvil-firing-awesome-or-reckless/>

英语需要一个词来表示“既可怕又危险”。

我们最近在 YouTube 上偶遇了[铁砧射击](https://www.youtube.com/watch?v=IhQ4dE_RGnQ)这种奇怪的消遣方式(还有哪里？).这个想法是你打包大约一磅(！)的黑火药放在两个铁砧之间，点燃它。火药爆炸，顶部的铁砧被射向空中。如果每个人都离得足够远并且戴上了听力保护装置，欢乐就会随之而来。

 [https://www.youtube.com/embed/IhQ4dE_RGnQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/IhQ4dE_RGnQ?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



在喜庆的场合用铁砧代替大炮可能是传统。这项娱乐活动的现代版[竞技版](http://www.columbiamissourian.com/news/columbia-man-wins-us-championship-anvil-shooting-title-in-farmington/article_26a56637-2c69-521c-9dea-7e1b7d173033.html)相对较新，评分标准包括铁砧飞得多高，以及铁砧落地时离发射台有多近。看起来你把所有东西都做得很平，把粉包得尽可能均匀，往好的方面想。

现在，在你开始做这种傻事之前，你应该知道[它也可能以多种方式出错](https://www.youtube.com/watch?v=q9isw0fKXAE)(当然是 YouTube)。在砧间砸黑粉是要求提前引爆。即使你对这种事情相对有经验，你仍然可能受伤，[就像 2011 年美国全国铁砧射击比赛](http://dailyjournalonline.com/news/local/injured-anvil-shooter-leaves-hospital/article_70c5883a-90f8-11e0-8cff-001cc4c03286.html)中发生的那样。显然，这个问题中的男人[因为这个爱好失去了一个拇指](http://anvilfire.com/iForge/tutor.php?lesson=July4th/anvil-shoot)。最后，除非你采取英勇的预防措施，比如对你的铁砧进行 x 光检查，否则你无法确定铁砧不会破裂。

所以我们不建议你尝试这个。相反，你应该敬畏地盯着 YouTube 上的疯子们，他们是为你冒险的 T2。(请注意上一个视频中使用的电引信，这可能比黑火药引信安全得多。我们*还是*不推荐。)
