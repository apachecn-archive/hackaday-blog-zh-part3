# Arduino 和鱼睡在一起

> 原文：<https://hackaday.com/2016/09/23/the-arduino-sleeps-with-the-fishes/>

[Eric Dirgahayu]想用一些传感器和摄像机探索水下。首先，他需要一个平台来承载它们。这导致了他的 Arduino 控制的游动鱼。这条鱼是由聚氯乙烯和一些防水材料制成的。从视频中(见下文)还不清楚这条鱼有多大的控制力，但它确实像一条真正的鱼一样以起伏的运动游泳。

包含的软件允许红外控制(因此清洁的水是必须的)，并且有一个用于浮力的压载箱。该网站有几个视频和标签，展示了该项目的不同方面。我们发现这个网站有点难以浏览，所以为了帮助你，你可以直接进入的有趣部分。这里还有[电子](https://www.wevolver.com/eric.dirgahayu/robotic-fish/electronics/guide/)。

如果你需要为你的机器鱼找个家，我们已经看到了很多高科技坦克。如果你喜欢你的潜水器更具未来感，你可以尝试一下[《星际迷航》](https://hackaday.com/2009/12/28/star-trek-submarine/)。
