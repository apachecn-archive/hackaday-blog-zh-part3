# 老式模拟仪表告诉你天气

> 原文:[https://hack aday . com/2015/12/07/old-school-analog-meters-tell-you-the-weather/](https://hackaday.com/2015/12/07/old-school-analog-meters-tell-you-the-weather/)

家用气象站是很好的街头信誉。购买一个商业电台会让你得到所有的铃铛和哨子，但看起来更实用。还有一个麻烦是把传感器阵列放在什么地方。为了解决这两个问题，[GradyHillhouse]建造了这个独特的气象站，配备了模拟仪表。

基于[粒子光子](https://www.particle.io/)从 [forecast.io API](https://developer.forecast.io/) 拉取天气数据，温度、压力等数值被发送到模拟 io 引脚。每个引脚都有一个仪表，带有用于校准的微调电位计和定制的打印标签。当发布恶劣天气警报时，还有一个数字输出变高；驱动其中一个仪表面板后面的 LED。所有东西都装在一个胡桃木的铭牌上，这是一个很好的展示。休息后的视频详细介绍了构建。

你可以想象，我们以前报道过很多气象站。一些[在屏幕上显示他们的数据](http://hackaday.com/2015/10/25/raspberry-pi-sense-hat-super-weather-dashboard/)，一些[以更独特的方式](http://hackaday.com/2015/11/08/weather-word-clock-warns-you-of-impending-winter/)。但我们真的很喜欢这个项目的老派外观和简单性。
