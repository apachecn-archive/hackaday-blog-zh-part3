# Box 'o Bangs，一个 2,180J 的电容器组

> 原文:[https://hack aday . com/2016/04/19/box-o-bangs-a-2180j-capacitor-bank/](https://hackaday.com/2016/04/19/box-o-bangs-a-2180j-capacitor-bank/)

当你连接 16 个电容器时会发生什么？准确地说是 16 个 2500V 40uF 电容……[Lemming]称之为 [Box 'O Bangs。](http://www.eevblog.com/forum/projects/in-response-to-dave's-slomo-video-here's-the-box'o'bangs-(warning-lots-of-pics)/)理论上，它在瞬间输出 4000A 的 2500V 电压。

他们从易贝购买电容器，他们似乎是质量良好的博世公司，直接来自德国。他们显然是用于大规模工业闪光灯，但谁知道，因为他们来自易贝。

将它们焊接在一起被证明是一个挑战，因为一旦他们意识到这东西要输出多少安培，他们就需要一些粗电线。它看起来像大约 2ga 线，剧透警告，仍然不够 4000A 但因为它只是一瞬间，它似乎做得很好。

一旦所有东西都造好了，就该进行一些科学测试了——我们可以在引线之间放什么来爆炸？敬请期待一些慢动作的荣耀。
