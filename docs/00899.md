# [科迪]带我们从岩石到环

> 原文：<https://hackaday.com/2015/12/09/cody-takes-us-from-rock-to-ring/>

[科迪里德]有一个问题。他想为他的女朋友【峡谷】做一枚戒指，但是没有足够的黄金。[科迪和峡谷]去年夏天花了一些时间淘那些闪亮的东西。然而他们的收获只有大约 1/3 克。太小了，不适合做任何珠宝。怎么办？如果你是[科迪]，你去你的银矿，拿一些矿石。【科迪】他在犹他州的农场有几个矿。虽然这次他没有下到 75 英尺深的矿坑里，但他确实捡起了一些他的家人几年前带出来的矿石。然而，从矿石到白银是一个漫长的过程。

首先，[科迪]用他自制的碎石机将岩石粉碎成大理石大小。然后他在轮胎炉中烘烤这块石头。这种矿石富含铅和银，以至于一些金属直接掉了出来，在熔炉下面的地面上形成飞溅。[科迪]然后球磨剩下的岩石成细粉，淘出剩下的铅。此时，铅和银混合在一起。[科迪]雇用[帕克斯过程](https://en.wikipedia.org/wiki/Parkes_process)提取银。锌被添加到熔融的铅混合物中。银被锌吸引，而锌不溶于铅。结果是一层锌和银漂浮在熔化的铅上。提取纯银只是去除锌的问题，科迪用了一点酸。

科迪决定用他们的黄金作为石头为[峡谷]制作一枚银戒指。他用失蜡法制作了他的戒指。这包括用蜡制作戒指，然后在模具中浇铸蜡。然后加热模具，烧掉蜡。结果是一个空模具，准备好熔融金属。

铸造戒指需要大量的清理工作才是完美的，但结果看起来肯定是值得所有的工作。
