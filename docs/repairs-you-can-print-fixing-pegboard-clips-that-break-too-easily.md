# 您可以打印的修复:修复太容易损坏的钉板夹

> 原文：<https://hackaday.com/2018/02/14/repairs-you-can-print-fixing-pegboard-clips-that-break-too-easily/>

现在，我们正在举办“修复你能打印”竞赛，其中一名幸运的学生和一个幸运的组织将赢得花哨的 schmancy Prusa i3 MK3，以及 neato 多挤压升级版。[Budiul]是一名学生，所以他想用 3D 打印机修复一些东西。对他来说幸运的是，他的工作室里的钉板是完全可怕的，或者至少钉板钩是。这些钩子是用聚氯乙烯做的，过了一段时间，越来越多的钩子坏了。[解决办法？打印他自己的，并在这个过程中使他们更强大。](https://hackaday.io/project/46931-repairing-a-tool-wall-organiser)

[Budiul]从他的挂壁挂板管理器上剩余的完整挂钩开始修理，并测量相关尺寸。这些在 Creo 4.0 中建模，打印出来，并测试是否合适。在经历了许多错误和失败的模型后，他终于得到了他的塑料钉板挂钩的 3D 打印版本。

当然，用 ABS 挂钩代替 PVC 挂钩并不是一个很好的解决方案。为了彻底解决塑料钉板挂钩的问题，他将挂钩印刷成两半，中间有一条通道。这个通道充满了一些钢丝和丙酮焊接在一起。结果是一个非常坚固的钉板挂钩，可以承受举起一些工具的严酷考验。

虽然打印出钉板挂钩可能看起来不像是最大的时间利用，但这个技巧还是有一些用处的。首先，这些不是我们都知道和喜爱的由钢棒制成的钉板挂钩；这是某种奇怪的专有系统，使用塑料模制挂钩。如果它们是由塑料制成的，你也可以打印出来。其次，能够打印自己的钉板挂钩是一项被严重低估的能力。如果你曾经尝试过整理一个工作台，你会知道你永远也找不到合适的挂钩。显然，在宇宙的某个地方有一个神秘的钉板挂钩的叠加。
