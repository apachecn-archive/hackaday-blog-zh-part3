# PoisonTap 使树莓 Pi 零利用锁定的计算机

> 原文:[https://hack aday . com/2016/11/16/poinsontap-makes-raspberry-pi-zero-exploit-locked-computers/](https://hackaday.com/2016/11/16/poinsontap-makes-raspberry-pi-zero-exploit-locked-computers/)

[Samy Kamkar]，leet haxor extraordinaire，利用漏洞和后门的宝藏，将其变成一个简单的硬件设备，劫持所有网络流量，实现远程访问，并在机器锁定时完成所有这些。这是 PoisonTap，它基于树莓派 Zero，代表我们如此渴望的令人敬畏的科技博客信誉。

PoisonTap 采用了一个 Raspberry Pi Zero，并将其配置为一个 [USB 小工具](http://hackaday.com/2015/12/27/turning-the-pi-zero-into-a-usb-gadget/)，模拟一个网络设备。当这个 Pi-come-USB-to-Ethernet 适配器插入计算机(即使是锁定的计算机)时，计算机发出 DHCP 请求，PoisonTap 通过告诉机器整个 IPv4 空间是 Pi 本地网络的一部分来做出响应。然后，被锁定的计算机上的所有互联网流量都通过 PoisonTap 发送，如果被锁定的计算机上正在运行浏览器，所有请求都会被发送到这个微小的漏洞利用设备。

由于所有网络访问都要通过 PoisonTap，cookies 会被吸走，浏览器缓存会因向外部世界提供 WebSocket 而中毒。即使在拔出 PoisonTap 后，攻击者也可以远程向目标计算机发送命令，并强制浏览器执行 JavaScript。从那以后，一切都差不多结束了。

当然，任何设计为插入 USB 端口并运行一些漏洞的设备都有一些限制。PoisonTap 仅在浏览器运行时有效。PoisonTap 不适用于设置了安全 cookie 标志的 HTTPS cookie。如果您在 USB 端口中填充了环氧树脂，则 PoisonTap 不起作用。PoisonTap 有一千多种限制，如果你把 PoisonTap 带到任何办公室，把它插上电脑，然后走开，所有这些限制可能都不适用。毕竟，这就是这个漏洞的目的。

与所有 ub3r-1337 pen 测试工具一样，我们预计将于明年 8 月在 DEF CON 的供应商区看到 PoisonTap 的一个版本出售。不要购买它。一个 Raspberry Pi Zero 售价 5 美元，一根 USB OTG 线不到 5 美元，[所有代码都可以在 Github](https://github.com/samyk/poisontap) 上获得。如果你*买*像 PoisonTap 这样的设备，你就太不懂技术了*不会用*它。
