# 我们最喜欢的四个硬件讲座

> 原文：<https://hackaday.com/2016/08/30/four-of-our-favorite-hardware-talks/>

黑客日超级会议是这个星球上最大的硬件黑客聚会。去年在 SuperCon，我们看到了关于从零开始构建系统、为技术创造新的有趣用途以及将这些电子比特推向市场的讨论。我们在说什么？以下是去年黑客日超级会议的四个最佳演讲:

 [https://www.youtube.com/embed/tn1sEaw1K2k?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/tn1sEaw1K2k?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



Shanni Prutchi 是 Rowan 大学的 ECE 学生，已经发表了关于射电天文学和计量学的论文。她的演讲概述了如何建立自己的量子纠缠光子源以及如何使用这些光子。量子密钥分发在小规模上是可能的，而且不仅仅是在大学光学实验室领域。

 [https://www.youtube.com/embed/3_-e_cJ1-Gs?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/3_-e_cJ1-Gs?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



我见过的最好的会议发言来自去年的[sprite_tm]。他创造了一个电子鸡的*矩阵*。电子鸡——那些生活在嵌入式系统中的可爱的数字宠物——毕竟只是计算机代码，在对电子鸡本身进行逆向工程后，他在服务器上模拟了几只电子鸡，使它们能够相互交流，甚至生育后代。最精彩的部分？[sprite]的*矩阵*并不是电影公司高管们所要求的怪异的近乎永动机的机器。

 [https://www.youtube.com/embed/kTfieGjMSUg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/kTfieGjMSUg?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



制造一个东西很容易，但是制造一千个就难上一百万倍了。这是制造电子产品的问题，没有人比[【Zach Fredin】和他关于中试规模生产的演讲](http://hackaday.com/2015/12/10/go-from-prototype-to-pilot-production-with-zach-fredin/)更好地涵盖了这个问题。制造的第一步总是最难的，对[扎克]来说，这是中试规模的生产。在演讲中，[Zack]给出了一些建议，比如总是购买模板，在哪里制造电路板，以及 EDA 软件的来龙去脉。

 [https://www.youtube.com/embed/EefHLUz69kI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/EefHLUz69kI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



触觉界面是下一个前沿领域。最终，我们都将戴上我们的电脑，而[【尼尔·莫夫瓦】关于触觉技术的演讲](http://hackaday.com/2015/12/22/neil-movva-adding-wearable-haptic-feedback-to-your-project/)是基于触摸的界面的艺术状态。他在会议上展示了一些硬件，展示了不同的振动感觉起来像不同的纹理。很诡异，而且是科技前沿。
