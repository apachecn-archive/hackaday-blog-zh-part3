# 他们如何使发电站与电网同步？

> 原文:[https://hack aday . com/2017/07/05/how-do-they-synchronize-power-stations-with-the-grid/](https://hackaday.com/2017/07/05/how-do-they-synchronize-power-stations-with-the-grid/)

在每个业余读者的生活中，可能都有这样的时候，你会看到一些东西，并意识到它背后的技术是你一直认为理所当然的，但从未考虑过它是如何工作的。写这篇文章的地方在周末有这样一个时刻，一个熟人在业余无线电场发布了一张照片，照片上是三台便携式燃气发电机连接在一起，同步运行。在这种情况下，有问题的交流发电机是内置自动电子同步的新奇发电机，但它留下了一个问题:它们是如何做到这一点的？他们如何将新电站接入电网，并使其与电网同步？接下来是一个随意的网络搜索，这反过来导致[下面的视频打破了一个台式演示](https://www.youtube.com/watch?v=RGPCIypib5Q)。

如果将两个交流电源连接在一起形成一个电网，它们必须在频率、相位和电压上完全匹配。如果不这样做，就会有电源间电流过大的风险，这会损坏电源和电网基础设施。下面来自[BTCInstrumentation]的视频以最简单的形式演示了如何匹配两台交流发电机的频率，方法是测量它们之间的频率差，并调整它们的速度和频率，直到它们可以连接起来。在视频中，他使用了氖灯，这些氖灯在两台交流发电机之间以不同的频率闪烁，并演示了如何调节其中一台发电机的速度，直到灯泡熄灭。两台交流发电机然后可以连接，然后将一起行动，以保持同步。在更多的视频中，他用闪光灯向我们展示了同样的过程，然后演示了交流发电机保持同步的过程，以及它们之间的相位偏差。

当然，公用事业公司的员工可能不会花时间盯着闪烁的霓虹灯来同步他们的发电站。同样的测量不是通过肉眼进行，而是通过自动控制接触器的机电或电子系统进行，就像前面提到的奇特的电子交流发电机一样。但是我们大多数人可能从来没有想过同步一组交流发电机，所以看到它以这样一种简单的方式演示应该会填补知识空白，即使这只是一个无聊的好奇心。
