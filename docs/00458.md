# 一平方英寸的年糕

> 原文:[https://hackaday.com/2015/11/09/a-square-inch-of-nyan/](https://hackaday.com/2015/11/09/a-square-inch-of-nyan/)

在 Hackaday.io 上，有一个非官方的比赛，将尽可能多的电子产品和令人敬畏的东西塞进一平方英寸的 PCB 中。虽然衡量系统是合乎逻辑的，但互联网不是:它以猫为食，以彩虹为乐。[拉多米尔]的[项目与 Nyan Board](https://hackaday.io/project/7860-nyan-board) 形成了这种奇怪的固定关系，Nyan Board 是一只拉彩虹的流行蛋挞猫，一直在播放哔哔声和嘘声，这很适合任天堂游戏。

【拉多米尔】为[平方英寸项目](https://hackaday.io/project/7813-the-square-inch-project)建造了这一平方英寸的 twee，这是一项非官方竞赛，旨在将最酷的项目安装在 1×1 平方英寸的 PCB 上。奖品是 Hackaday 商店和 OSHPark 的积分。这个条目使用 ATtiny85 微控制器、几个电阻、发光二极管和蜂鸣器来播放奇怪的切分音 Nyan 主题曲。电力通过彩色带状电缆传输，该电缆模仿流行馅饼猫的彩虹尾巴。

这不是平方英寸项目中唯一的项目，甚至不是最闪亮的项目；还有[一个 rrroyyyyygygggbgbpw LED](https://hackaday.io/project/8331-rrroyyyygygggbgbpw-blinktronicator)，一个[平方英寸四轴飞行器](https://hackaday.io/project/7947-quadcopter-in-one-inch)，还有[一个探测世界上最小蝙蝠的装置](https://hackaday.io/project/8353-worlds-smallest-bat-detector)。所有非常有趣的项目，我们迫不及待地想看到所有的参赛作品完成。
