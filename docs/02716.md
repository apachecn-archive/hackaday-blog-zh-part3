# 《我的世界》特洛伊木马教孩子热爱电子和代码

> 原文:[https://hack aday . com/2016/06/24/minecraft-trojan-horse-teaches-kids-to-love-electronics-and-code/](https://hackaday.com/2016/06/24/minecraft-trojan-horse-teaches-kids-to-love-electronics-and-code/)

孩子们喜欢《我的世界》，聪明的教育者可以利用这种爱来教授一些非常实用的技能。蒙大拿州波兹曼儿童博物馆提供的暑期班会让我大吃一惊，如果我们年轻时有这样的事情存在的话。(而不是乞求我的童子军中的一个爸爸为我盗版 Visual Studio，它是以一叠高得惊人的 CD 的形式交付的。)波兹曼的孩子们可以在玩《我的世界》的同时学习硬件、软件及其集成。

《我的世界》是一个身临其境的世界，它已经被证明能够吸引创造性思维。这是吸引孩子们参加暑期班的诱饵，但 Serialcraft 让学习变得同样令人上瘾。这是通过向学生提供与《我的世界》世界联系在一起的实物来实现的，这些实物以我们以前从未见过的有意义的方式联系在一起(至少不是一次性的)。从表面上看，这为游戏增加了物理发光二极管、拨动开关、电位计和操纵杆。但是物理控制邀请对机制本身的理解，它们以令人兴奋的方式交织在一起，通过命令块和其他对学生来说感觉直观的游戏内组件。从他们对游戏机制的理解中，他们理解了物理对象，并立即想用他们在游戏中玩新积木的方式来试验它们。

让这种神奇成为可能的是由[John Allwine]编写的《我的世界》mod，他在 2016 年的 Maker Faire Bay Area 上为我们演示了集成。mod 允许用户从《我的世界》访问 Arduino 的输入和输出，在本例中为 Pololu A-Star 32U4。在课堂上，这一切都以激光切割控制器的形式很好地打包了。它有一些发光二极管，两个操纵杆，按钮，电位计和一个光电传感器。

从休息下面的视频可以看到，真的很酷。孩子们也玩得很开心。例如，[John]向他们展示了如何将他们独特的控制器连接到世界上的一个活塞上。由于这个活塞可以由他们单独控制，他们很快就想出了如何为他们的物品制作秘密安全室。

另一个麻烦的发现是，控制器上的光电晶体管通过改变一天中的时间来设置游戏世界的亮度。孩子们偶尔会起床，通过打开或关闭房间里的灯来改变世界。对任何《我的世界》玩家都有一定吸引力的一个功能是操纵控制器上的一个 led，根据与爬行动物的接近程度来改变亮度。
