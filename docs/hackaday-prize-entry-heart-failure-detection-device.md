# Hackaday 奖参赛作品:心力衰竭检测设备

> 原文：<https://hackaday.com/2017/05/25/hackaday-prize-entry-heart-failure-detection-device/>

早期和低成本的心力衰竭检测是[Jean Pierre Le Rouzic] [为其 2017 年 Hackaday 奖](https://hackaday.io/project/19685-early-and-low-cost-detection-of-heart-failure)的参赛作品提出的建议。他的设备基于一种低成本的多普勒设备，就像那些用于倾听未出生婴儿心脏的胎儿多普勒设备一样，为一种机器学习算法提供信息，该算法可以区分健康和不健康的心脏。

其背后的理论是，正常、健康的心脏组织具有与退化组织不同的声阻抗。基于声阻抗，该设备将组织分类为:正常的、退化的、颗粒状的或纤维状的。每一类都指出了主要在结缔组织中的特殊问题。

拥有一个像[Rouzic]正在研究的这样的工作装置有几个优点。首先，它可以在家里使用，不需要医生或医务人员的干预。在我们看来，这就像使用血压测量设备或胎儿多普勒仪一样简单。它也相对便宜(估计不到 150 美元),而且不需要凝胶。我们讨论了测量不同心脏信号的类似项目，如[开源心电图](http://hackaday.com/2016/04/29/saving-lives-with-open-source-electrocardiography/)，但心电图有一个缺点，它需要将电极附着在身体上。

一个有趣的提议功能是，从单个案例中学到的东西会在下次更新时发送到每个设备，因此设备在使用时会变得“更智能”。当然，有很多方法会出错，但这是一个好主意。
