# 入侵一台工业用 42 英寸多点触控电脑

> 原文：<https://hackaday.com/2017/11/16/hacking-an-industrial-42-multitouch-pc/>

我们正慢慢地朝着这样一个方向发展，即每个人都将拥有一张触摸屏桌子，就像 1982 年的 TRON 电影或 1987 年的《星际迷航:下一代系列》中那样，它具有无处不在的触摸屏星际飞船控件。当地一家公司给他提供了一台他们准备扔掉的工业用 42 英寸多点触控电脑,幸运地迎来了那个未来。几个黑客之后，他让我们都羡慕不已。

不过，在动手之前，他必须处理一些魔法烟雾。这个问题的来源原来是电脑电源上的黄色粘性物质，它通过一个电阻变得导电。清洗它解决了问题。

接下来，他通过使用电位计来控制背光的功率，从而增加了亮度控制。请务必仔细观看下面的视频，在视频中，他将磁铁和电线连接到电位计，并将其全部封装在环氧树脂中。在这一点上，我们非常确定我们看到他旋转了一个硬盘驱动器盘片，上面附有一个砂纸盘，形成了一个台式盘式砂光机，让我们更喜欢这个黑客。

他还用一个更大的扬声器和放大器取代了一个小扬声器，带来了像白天和黑夜一样的音量和音质差异。他还添加了一个带有继电器的分线板，用于电源管理，消除了待机模式下 7 瓦的连续功耗。

请务必观看视频，直到他带我们参观被黑的内部硬件。我们喜欢他把自己所有的作品贴上标签，让未来的黑客可以打开它
