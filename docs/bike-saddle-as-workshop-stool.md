# 作为车间凳子的自行车鞍座

> 原文：<https://hackaday.com/2016/10/12/bike-saddle-as-workshop-stool/>

爱好者将特定爱好或追求的部分重新用作家具并不罕见。举例来说，汽车迷会用一辆轿车的后车身制作沙发，或者用发动机组制作咖啡桌，就像你可能在 *Top Gear* 工作室看到的那样。

一位自行车爱好者要求[Quinn Dunki]用自行车座椅制作一个车间凳子，由此产生的这件家具既优雅又实用，可能只有那些习惯于赛车鞍座的人才会感到舒适。

凳子本身放在一根垂直的管子上，管子底部有一个三脚架，每条腿上都装有一个脚轮。我们经历了制作金属制品的步骤，特别是每条腿所需的相当棘手的 45 度管状接头。我们看到脚轮上出乎意料的高作用力使最初的点焊开裂，并导致更坚固的接缝。最后，我们看到了由链轮猫制作的优雅的凳子，你可以在下面看到。
