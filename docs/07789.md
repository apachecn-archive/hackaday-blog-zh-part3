# 启发灵感的未来派植物

> 原文：<https://hackaday.com/2017/12/04/a-futuristic-plant-to-inspire-bright-ideas/>

一个好的电子游戏道具真的可以美化装饰——如果它是一个发光的、未来的盆栽植物(对不起！)直接出自《杀出重围:人类革命》。

由于种植霓虹灯藤有点困难，这个项目更多的是灯而不是植物。“绿拇指”的制造者是来自“JumperOneTV”的“Phil ”,他正在使用 5 米长的暖白色条状发光二极管，这些发光二极管被切割成不同的藤蔓长度。他还购买了一个商店购买的花盆，方便地反映了游戏中的模型。藤蔓由 16 毫米的聚乙烯管制成，他用热风枪将它定型——通过泵水来设定它们的形状——并用绝缘泡沫固定在花盆中。与他的下一个难题:供电相比，将 LED 灯条穿入并并联起来很简单。

为了延长 LED 灯带的寿命，[Phil]使用可调电源(隐藏在花盆底部)以大约 1.5A 的 9V 电压运行 LED 灯带，强调转动时的安全性。[菲尔]没有提到他是如何打开它的，但是在展示它的时候，[刺激稍微多一点的活植物](https://hackaday.com/2016/09/05/bamboo-plant-becomes-a-stylish-light-switch/)可能是一个巧妙的技巧，前提是你更倾向于[音乐的植物群](https://hackaday.com/2015/04/16/play-robotic-bongos-using-your-household-plants/)不会引起骚动。
