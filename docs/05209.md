# 两全其美:Arduino + 555 应该让评论者感到困惑

> 原文：<https://hackaday.com/2017/03/02/the-best-of-both-worlds-arduino-555-should-confuse-commenters/>

几乎没有一个星期，一些黑客帖子不会引发以下评论之一:

> 那是愚蠢的！当你可以用 555 做同样的事情时，为什么要用 Arduino 呢？

并且:

> 那是愚蠢的！当你可以使用 Arduino 时，为什么要使用一堆零件呢？

然而，我们很少在同一个帖子上看到这两个评论。直到现在。[ZHut]通过展示如何让 Arduino 与 555 定时器一起闪烁 LED，成功地将这两个世界结合在一起。我们知道，我们知道。很难决定如何评论这件事。你可以边看视频边考虑，下面。

从好的方面来看，这可能有一个用例。在 Arduino 完全不干预的情况下，LED 将会闪烁。如果你想的话，你可以让 Arduino 进入深度睡眠状态，LED 灯仍然会闪烁。只需做一点工作，你就可以将这个想法应用到 555 剧本之外的任何数量的电路上，比如一个 [PWM 发生器](https://hackaday.com/2013/09/15/the-easy-or-hard-way-to-build-a-pwm-dimmer/)。

几乎没有什么是 555 做不到的。如果你想看看它毫无表情的脸下面是什么，这个[拆卸](https://hackaday.com/2016/02/23/555-teardown-and-analysis/)是一个有趣的阅读。我们只是希望评论区不会超载，就像柯克船长要求星际迷航计算机计算圆周率的每一位数字一样。
