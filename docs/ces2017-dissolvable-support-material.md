# CES2017:可溶性支持材料

> 原文：<https://hackaday.com/2017/01/07/ces2017-dissolvable-support-material/>

在过去的一年里，3D 打印的下一件大事似乎是双重和多重挤压。这可以让你得到多种颜色的物体，但是如果你有合适的灯丝，它也可以让你打印出其他方式无法打印的物体。本周， [Airwolf 3D 公布了他们的 HydroFill 水溶性支持材料](http://airwolf3d.com/2017/01/05/hydrofill-water-soluble-support/)。他们有一些样本打印和一个鱼缸，是的，这种东西在水中溶解很快。

过去，可溶性支持材料存在一些问题。PVA 是水溶性的，但是打印效果不好。HIPS 可以溶解在柠檬烯中，但是产生的粘性物质是有毒的。Airwolf 的 HydroFill 击败了这两种灯丝，并且打印效果与最近 E3D 展示的支架灯丝相当。
