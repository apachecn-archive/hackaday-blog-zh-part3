# 廉价泡沫滑翔机的 3D 打印升级

> 原文:[https://hack aday . com/2018/07/09/3d-printed-upgrade-for-cheap-foam-glider/](https://hackaday.com/2018/07/09/3d-printed-upgrade-for-cheap-foam-glider/)

我们知道你见过它们:大型泡沫滑翔机是世界上几乎所有大型零售商和一元店的夏季主打产品。它们可能由不同的公司制造，或者有轻微的外观差异，但它们都遵循基本的公式:一个长塑料袋，包含单件机身、两个可拆卸的机翼和一个尾巴。撕开袋子，把机翼塞进机身，看看你能不能把它卡在屋顶上。

但是在你折腾几次之后，事情就开始变得有点不新鲜了。我们这些 Hackaday 集体中仍然保留着童年记忆的人甚至可能记得试图用一些战略上附加的瓶子火箭来增加滑翔机。但是[蒂莫西·赖特]做得比这好得多。随着 3D 打印“背包”的加入，他不仅设法给这些泡沫飞行器添加了一个马达，还添加了一个遥控接收器和伺服系统来移动控制表面。[最终结果是一架便宜且功能惊人的遥控飞机](https://www.thingiverse.com/thing:2948067)，所需工作相对较少。

[蒂莫西]当然不会声称是第一个在泡沫滑翔机上安装马达以从中获得更多乐趣的人，但他的方法非常巧妙，当然还有额外的好处，即其他成年孩子也可以尝试，这要感谢他发布设计时使用的知识共享许可。他提到，不同滑翔机的差异可能会导致一些兼容性问题，但随着一些拉链和胶带的大量应用，应该会很好。

这种特殊的泡沫块[可能不会创造任何高度或距离记录](https://hackaday.com/2018/05/30/autonomous-spaceplane-travels-to-10-km-lands-safely-200-km-away/)，它肯定[不会带你飞上天](https://hackaday.com/2017/08/14/building-an-ultralight-out-of-foam-in-a-basement/)，但如果你有一些遥控装置，这是一个非常容易实现的夏季项目。
