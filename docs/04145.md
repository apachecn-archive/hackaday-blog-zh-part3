# 《星际迷航》的电话基座还不如皮卡德的床头柜上的呢

> 原文：<https://hackaday.com/2016/11/16/star-trek-phone-dock-might-as-well-be-from-picards-night-stand/>

《星际迷航》通常被认为有助于推动我们今天所拥有的技术的发展——手机就是一个很好的例子。当《星际迷航》愚人节的产品启发了一个制造商去制造真实的东西？嗯，这似乎是意料之中的事。【MS3FGX】决定让它这样。3D 打印的以《星际迷航》为主题的电话基座充当了[蓝牙扬声器和白噪声发生器](https://imgur.com/a/oHox9)。结果在下面的视频中展示出来，相当于你期望在银幕上看到的特效。

从它所基于的产品——更大并且有嵌入式屏幕——中获得一些自由，使[MS3FGX]的版本更加实用。两个工业拨动开关控制一个 tech cube 小夜灯和内部蓝牙扬声器。手机基座后面的 NFC 标签启动预装的 LCARS UI 应用程序，并打开手机的蓝牙。尽管对[MS3FGX]的设计来说是一个挑战，但最终产品似乎完全符合预期。
