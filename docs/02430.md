# 树莓派与你同在

> 原文:[https://hack aday . com/2016/05/27/raspberry-pi-levels-with-you/](https://hackaday.com/2016/05/27/raspberry-pi-levels-with-you/)

很容易想象早期人类是如何开始使用石头，然后最终发展出越来越好的工具，直到他们发明了锤子。一些简单的工具需要更长的时间来发明。例如，精神层面，直到 17 世纪下半叶才出现。

这个想法很简单。一根透明的管子盛着一种液体和一个气泡。当气泡在管的中心时，装置在管的方向上是水平的。[马克·威廉姆斯]有一个稍微复杂一些的方法。他用一个内部测量单元(IMU)和一个树莓皮创造了一个现代的水平仪。

该设备使用一些 Python 库在 LCD 上显示虚拟气泡。可以看下面一个视频，在 [GitHub](https://github.com/mwilliams03/BerryIMU) 上找代码。

我们当然已经看到了 IMUs 更令人兴奋的用途。如果你想变得更小，我们甚至研究了一些[微型](http://hackaday.com/2015/05/20/a-very-very-small-imu/)IMU。但这是一个很好的项目，做得很好。我们觉得你应该看看。
