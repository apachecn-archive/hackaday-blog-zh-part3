# 用脚本摆脱视频编辑的苦差事

> 原文：<https://hackaday.com/2016/11/26/script-your-way-out-of-video-editing-drudgery/>

[维克多·弗罗斯特]有一个[低沉的声音和一个花哨的顶级摄像机](https://hackaday.io/project/18447-ingest-a-tron-9000)。虽然人们会认为这对于生活来说是一个非常宽松的环境；它有自己的一系列问题。主要是他的梦幻花式相机使用了流行的 h.264 编码方案的最现代版本，h.265\. Gasp！

虽然这看起来也很专业，但不幸的是，h.265 与他的编辑软件配合得并不好。解决方案看起来很简单，只需转换代码，然后就可以上路了。然而，当你开始谈论从顶级视频源转码 4K 视频并保持质量时。嗯……它能让处理器屈服。因为他宁愿玩监视游戏，而不是在他的主电脑上转码视频，所以他决定把这项苦差事卸下来，自动化到他的备用电脑上。

这就是摄取-a-Tron 9000 发挥作用的方式。它使用许多开源软件，是的，还有 windows 批处理文件，从他的相机中取出文件，在一台计算机上处理，然后转储到另一台计算机上。现在他可以一边等待一边玩游戏(或编辑)。对于那些因为我们最喜欢的软件而与 Linux 疏远的人来说，很高兴知道仍然有办法自动消除痛苦。休息后的视频。
