# Engauge 使图形窃取变得轻而易举

> 原文:[https://hack aday . com/2015/12/22/engauge-makes-graph-steeping-a-cinch/](https://hackaday.com/2015/12/22/engauge-makes-graph-thieving-a-cinch/)

我们以前见过它们:80 年代复印得很差的数据表中的图表和图形，古老的研究论文，甚至我们大学教授的黑板上的鸡肉划痕。可悲的是，这些奇妙的绘制数据被锁在一个呈现糟糕的 png 或教科书图形中。幸运的是，一个程序员团队拯救了我们，给了我们合适的窃取工具来直接从数据源中提取数据，这个工具就是 [Engauge](http://digitizer.sourceforge.net/) 。

Engauge 是一款开源软件工具，能够将绘图图片转换为数据的数字表示。虽然我们中的一些人可能仍然在手工绘制图表，但 Engauge 使我们能够简单地定义图表上的参考点，并且一个聪明的图像处理算法会自动为我们提取曲线！当然，有一点微调来确定什么算作数据，但最终结果是一个一体化的软件工具，可以吃图片和产生数据-不需要中间步骤！

多年来，Engauge 一直在帮助科学家和工程师保存古老的数据日志，但当我们从模拟 o'scope 记录或从教科书中删除那些 *x* 和 *y* 时，它仍然是一个新鲜的工具。在一个日益数字化的世界里，我们要感谢那些积极参与的开发人员，他们为我们提供了合适的工作工具。尽管如此，如果你不喜欢偷图，试试[偷样条](http://hackaday.com/2015/02/01/spline-thieving-makes-hobby-servos-even-more-useful/)从相机到 CAD。
