# 你能重复一遍吗？

> 原文:[https://hackaday.com/2015/10/11/could-you-repeat-that/](https://hackaday.com/2015/10/11/could-you-repeat-that/)

有没有去过一个很大的地方，需要告诉另一端的乔一些事情？如果你幸运的话，莎莉在中间，你可以大声喊莎莉告诉乔你的消息。也许乔会冲莎莉喊着回答。

这就是业余无线电中继器的工作原理。

Hackaday 的朋友肯尼斯·芬尼根(Kenneth Finnegan)厌倦了解释复读机的细节，所以他[制作了一对复读机教程视频](http://blog.thelifeofkenneth.com/2015/10/lectures-on-radio-repeaters.html?m=1)，第一个视频在休息后被发现。

Image source [KV5R](http://kv5r.com/ham-radio/balloon-repeater/)

较高的无线电频率，比如 50 MHz 及以上，通常只在视线范围内传播。加上手持设备有限的功率和天线，通常低于 5 瓦，以及无处不在的“橡胶鸭子”天线，你不能说得很远。装有 50 瓦和更大天线的车辆上的移动设备做得更好，但实际上它们没有那么多帮助。

真正有所提高的是身高来提高射程。高度提供了更长的视线和更少的障碍物。Hams [创造了中继器](https://en.wikipedia.org/wiki/Amateur_radio_repeater)并将其放置在塔、建筑或山顶上，以扩展无线电视野。最终的[中继器是太空卫星](http://hackaday.com/2015/10/10/original-hackers-new-satellite-in-orbit/)。再高不过了。紧随其后的是带有中继器的接近太空高度的气球[将提供多州覆盖](http://kv5r.com/ham-radio/balloon-repeater/)。

除了提供高度，中继器还将具有更高的输出功率和更好的天线，这对于接收远距离手持设备的微弱信号尤为重要。一个信号进来，然后以稍微不同的频率重复出去。这些频率上的所有现代锤齿轮被设置成处理这种*偏移* *频率*操作。

哈姆斯是否提出了这个想法尚有争议，但在早期他们肯定是存在的。
