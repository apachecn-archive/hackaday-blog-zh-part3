# Hackaday 奖参赛作品:鼠标控制显微镜

> 原文:[https://hack aday . com/2016/09/19/hack aday-prize-entry-mouse-controlled-microscope/](https://hackaday.com/2016/09/19/hackaday-prize-entry-mouse-controlled-microscope/)

你可能会认为操作显微镜只需要一双好眼睛。不幸的是，如果你是一个截肢者，情况可能并非如此。例如，兽医实验室工作需要控制焦点，以及在 X 和 Y 两个方向上移动样品的能力，而这些都不是仅用一只手就能轻松同时完成的任务。

[ksk]对这个问题的解决方案是使用齿轮步进电机和 Arduino Mega 来允许通过计算机鼠标或轨迹球控制显微镜的手动功能。电机安装在显微镜控制器上，带有定制的 3D 打印外壳。包含 Arduino 的控制盒上的旋转选择器允许用户选择慢速或快速模式进行微调或粗调。

公平地说，这个项目仍然是一项正在进行中的工作，我们将在我们的系列帖子中介绍它，看看 Hackaday 奖的参赛作品。然而，从目前报道的进展来看，很明显这是一个具有巨大潜力的项目，我们可以看到成品对任何操作显微镜的人都有用。

这些年来，我们已经推出了一两个鼠标控制的项目，尽管不是控制显微镜。这是一个[鼠标控制的机器人手臂](http://hackaday.com/2010/07/22/mouse-controlled-manipulator-arm/)，我们已经用 3D 鼠标覆盖了[另一个手臂。](http://hackaday.com/2015/07/28/3d-mouse-drives-robot-arm/)
