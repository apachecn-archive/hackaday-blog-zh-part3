# 回顾:20 世纪 50 年代的电视是野兽

> 原文：<https://hackaday.com/2017/12/29/retrotechtacular-1950s-televisions-were-beasts/>

电视已经存在很长时间了，但是现在我们所指和所称的电视与消费者最初爱上的完全不同。这段关于 20 世纪 50 年代 RCA 工厂参观的视频让我们清楚地看到，在现代人看来，旧的设计是多么的陌生。

从一开始，电路的明显混乱就令人难以置信，一些元件在电路板上，但许多元件是点对点连接的。叙述者甚至对使用绕线枪的“制造电连接的新技术”发表了评论。据说这比焊接更干净、更快、更整洁。(【比尔牛群】[可能会同意](https://hackaday.com/2014/12/18/wire-wrap-101/)。)然而，并不是所有的方法都在今天的制造业中失传了。PCB 的手工填充和波峰焊仍然用于低成本商品，并且经常用于电源(至少在空间不是很重要的地方)。

当谈到 60 多年前的设计时，毫不奇怪这些是显像管电视。但是这超出了产生图像的阴极射线管(CRT)。他们使用的是真空管，很大一部分视频深入研究了真空管的制造和测试。你会在 3:20 瞥见这一点，但你真正想看到的是 4:30 的自动化测试机。每根管子都沿着一条专门的传送带行进，在传送带上，测试人员甚至可以从致动器末端的软木塞上自动敲击几下。随着管排管的进展，当每个管以 3-4 倍的额定灯丝电压运行时，我们看到了“老化”过程(大约 6:00)。狂野！

还有一个部分详细介绍了 CRT 显像管的制造，尽管这些彩色显像管似乎并不适合在其余电影中使用的电视型号。大约 7 点 07 分，他们称之为“彩色显像管”，这是 RCA CRT 技术的早期名称。

在参观工厂的过程中，我们有一种强烈的感觉，那就是这种制造业与汽车的关系比与现代电子产品的关系更密切。在那个时代，电视(和收音机)更像是家具，看着笨重的机箱被悬挂式传送带运送只是其中的一部分。封闭工厂正在大量生产相同的木制控制台。这从 11:55 开始，显示的自动化与我们今天期望看到的非常相似。看来木工效率在 50 年代就已经是一个解决的问题了。
