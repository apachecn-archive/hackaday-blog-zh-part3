# 1920 年的 Claratone 收音机运行 Windows 10

> 原文:[https://hack aday . com/2018/04/10/1920s-Clara tone-radio-runs-windows-10/](https://hackaday.com/2018/04/10/1920s-claratone-radio-runs-windows-10/)

在过去，我们已经提到过，在如何将一款经典硬件带入 21 世纪的问题上，存在不同的思想流派。你可以走保护主义者的路线，小心翼翼地将原始部件与更现代的部件嫁接，或者你可以选择核能，将所有积满灰尘的旧设备炸出水面。[德里克·特克斯勒] [在他最近将 1920 年代的 Claratone 电子管收音机转换为互联网收音机和播客播放器时，显然决定选择后者。除了旋钮和木箱之外，最初的设备几乎没有留下什么，他甚至设法将一台 Windows 10 电脑塞进了底座。](https://www.youtube.com/watch?v=WIJnUQOP4PQ)

[![](../Images/b9eba9c2867cb6dec8f4b056e88332f8.png)](https://hackaday.com/wp-content/uploads/2018/04/claratone_detail2.jpg) 收音机的核心是 LattePanda，一台极其强大的 Intel 单板电脑。它运行 Windows 操作系统，从一个内置于旧真空管的 USB 拇指驱动器中加载一系列网络广播流和播客进行播放。LattePanda 使用其内置的 Arduino 与收音机原来的前面板旋钮相连接，现在这些旋钮用于在流之间切换。一个特别巧妙的效果是在切换“电台”时人为添加的静态和串扰，听起来好像你真的在一个电台中拨号，而不仅仅是在数字文件之间进行选择。

在音频方面，LattePanda 连接到 SX400 放大器，进而驱动外部扬声器。虽然[Derek]提到它并不完美，但 MSGEQ7 图形均衡器芯片被用来控制安装在原始收音机真空管内的 led。在休息后的视频中，你可以看到灯管随着音乐疯狂闪烁，给最终产品带来互动效果。不幸的是，似乎只有当收音机的“头罩”打开时，你才能看到电子管。

如果这种对历史保护的极度缺乏让你热泪盈眶，不要害怕。我们已经介绍了一些[老式音频设备](https://hackaday.com/2016/11/17/vintage-tube-radio-restorations/)的正确修复工作，这可能会让你变得更好。
