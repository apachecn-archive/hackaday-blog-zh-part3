# 为什么你应该获得业余无线电执照？

> 原文：<https://hackaday.com/2015/10/23/why-should-you-get-a-ham-radio-license/>

你在 Hackaday 上读到的几个作者是业余无线电爱好者，我们经常就“为什么今天要成为业余爱好者”进行黑客聊天毕竟，你可以通过互联网或电话与世界上的任何人交谈，对吗？平局是什么？

英国无线电协会显然也有同样的想法，于是[制作了一个很棒的视频](https://www.youtube.com/watch?v=8x6x_6mDVlQ)来回答这个问题。他们会提到一些平常的事情:了解技术，了解世界其他地方的人，灾难通讯，以及[电台](https://en.wikipedia.org/wiki/Amateur_radio_direction_finding)(在美国以外似乎更受欢迎；人们竞相寻找隐藏的发射器)。

此外，他们还谈论了很多关于哈姆如何参与太空通信的话题，从通过卫星交谈，到与空间站上的人交谈，再到实际建造小型卫星。正如解说员所说，业余无线电有“数百种方式获得技术乐趣”。

我们注意到他们展示了一件事，但没有说太多，那就是教育机会。你可以学到很多东西，和孩子们一起帮助他们学习通常是非常有益的(通常你也会学到一些东西)。只是为了防止评论说这篇文章与黑客无关，我们要注意两件事:有一个树莓 Pi 显示和刚刚过去的两分钟标记，有一个非常聪明的黑在一起的莫尔斯电码键。

我们谈论了很多关于业余无线电的事情，从基于 Arduino 的数字模式到组装便携式电台的 T2(T3)。我们注意到他们没有提到的另一件事是:如今获得执照比以往任何时候都容易。大多数国家(包括美国)已经废除了莫尔斯电码的要求，所以尽管一些火腿仍然享受 CW(火腿的口语操作莫尔斯电码)，它不是一个要求。
