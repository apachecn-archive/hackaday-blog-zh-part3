# 玻璃屋顶的无痛清洁

> 原文:[https://hack aday . com/2018/05/23/graw ler-无痛清洁玻璃屋顶/](https://hackaday.com/2018/05/23/grawler-painless-cleaning-for-glass-roofs/)

[Gelstronic]房子的一部分有玻璃屋顶。虽然他喜欢自然光和温暖，但他不喜欢每次鸟儿沉积或雨水污渍堆积时都爬上梯子清理。他过去曾试图制造一个清洁机器人，但屋顶 25%的坡度让事情变得有点复杂。现在，随着步进电机和防滑坦克履带的加入，[Gelstronic]可以准确地告诉这个版本的 GRawler 要走多远，或者停留在一个地方清洁一个特别脏的地方。

[GRawler 被设计成在爬上屋顶的过程中进行清洁，在爬下屋顶的过程中使用橡胶扫帚](http://www.instructables.com/id/GRawler-Glass-Roof-Cleaner/)。它由 Arduino Pro Micro 驱动，由轻质铝和许多 PLA 打印的零件制成。GRawler 还使用常见的东西，这总是一个额外的好处:刷子是用来清洁电器后面的那种，橡胶刮片来自卡车大小的雨刷。[Gelstronic]可以使用一款名为“操纵杆 BT Commander”的应用程序，通过蓝牙控制 GRawler 的电机、刷子的旋转以及升高/降低雨刮片。勉强通过休息，看看它的行动。

据我们所知，[Gelstronic]仍然需要打开梯子来放置 GRawler 并在面板之间移动他。也许下一个版本可以被拴起来，就像太阳能电池板清洁机器人一样。
