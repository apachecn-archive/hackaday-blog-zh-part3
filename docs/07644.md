# retro techular:Tektronix 方式的电路板

> 原文:[https://hackaday . com/2017/11/30/retro tech tacular-circuit-boards-the-Tektronix-way/](https://hackaday.com/2017/11/30/retrotechtacular-circuit-boards-the-tektronix-way/)

印刷电路板是商业电子设备和我们 Hackaday 专题项目的基本组成部分。我们中的许多人都做过自己的，无论是用一箱蚀刻剂从基本原理开始，还是作为一套 Gerbers 送到 PCB 工厂。

说今天的复古主题是印刷电路板的制造可能看起来很奇怪，因为印刷电路板没有什么过时的，它们仍然和我们在一起。但是休息时间下面的电影从两个角度对这一过程进行了迷人的审视，既告诉我们它们是如何制造的，也告诉我们它们在 1969 年制造时是如何制造的。

![](../Images/18aa26affca0edf976491b609e352907.png)

Board artwork laid out at four-times actual size

泰克公司在当时和现在一样以生产高质量示波器而闻名。20 世纪 60 年代末，泰克的瞄准镜采用了几块带有固态电子元件的印刷电路板，并且按照极高的标准制造。这部电影讲述了从最初的 PCB 布局到组装电路板的制造过程，包括所有生产过程的大量细节。

在 2017 年，你可以用 CAD 软件包开始 PCB 设计，但在 1969 年，那是难以置信的手工操作。所有的东西都是用手从一张纸的示意图转录到透明胶片上的。将比实际尺寸大四倍的元件尺寸纸质模型放在网格上，用铅笔将导体画在一张覆盖的描图纸上。然后，使用黑色转印纸和胶带在描图纸上的胶片上布置焊盘和轨迹图案，电路板的顶部和底部各一个。摄影过程将它们缩小到胶片上的生产尺寸，从胶片上可以曝光和蚀刻，就像 2017 年一样。

![](../Images/8a9463850983ad852616d4d14a9134db.png)

Pantograph drilling machine uses a manually moved styuls on a template to drill six boards at once

自 1969 年以来，制造 PCB 的大部分物理过程没有发生重大变化。我们详细了解了电镀和镀金工艺，然后是蚀刻和丝网印刷工艺，最后是元件安装和波峰焊。

不过，有些机器和现在自动化的零件是在 1969 年由手工完成的，它们已经过时了。印刷电路板的钻孔是手工完成的，小批量的钻孔用的是缩放仪，大批量的钻孔用的是令人着迷的数控钻机，由穿孔带控制，看不到电脑。元件放置都是手工完成的，评论员评论说有一天可能会由机器来完成。

这部电影同时展现了 PCB 生产的有趣一面和 20 世纪 60 年代制造业的迷人一瞥。很可能许多 Tek 的示波器仍然和我们在一起，从我们在无线电集会上的经验来看，它们肯定是熟悉的。
