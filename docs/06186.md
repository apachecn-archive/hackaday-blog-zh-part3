# 太阳能推土机变脏了

> 原文:[https://hackaday.com/2017/06/07/solar-bulldozer-gets-dirty/](https://hackaday.com/2017/06/07/solar-bulldozer-gets-dirty/)

随着气候变化的威胁日益逼近，越来越多的行业开始电气化，而不是使用汽油和柴油等传统燃料。几乎在所有情况下，效率的提高都证明是有益于环境和经济的。显然，我们已经看到更多的电动汽车上路，但这一趋势远远超出了汽车，延伸到草坪设备、自行车、船只，甚至飞机。这种电气化机器趋势的最新消息来自 YouTube 用户[J Mantzel],他已经建造了自己的太阳能推土机。

然而，这台推土机完全由太阳能驱动的事实只是冰山一角。更令人印象深刻的是，这台推土机完全是从零开始建造的。屋顶上的太阳能电池板为一组驱动马达的电池充电，尽管推土机速度很慢，但对于它的小尺寸来说，它的强度令人难以置信。如果阳光充足，它也有可能仅依靠太阳能运行，这几乎完全消除了对电池的需求。它也是由不锈钢和铝制成的，这使得它基本上是防锈的。

这是一个令人印象深刻的建筑，与[J . Mantzel]的其他项目配合得很好，其中大部分都围绕着一种远离电网的生活方式。如果这是你的拿手好戏，那么你可以从他的各种项目中获得很多灵感。一定要看看下面他的推土机的视频。不过，你不必建造一台离网推土机来开始离网生活，而且很容易从一个太阳能电池板和一辆卡车开始做起。
