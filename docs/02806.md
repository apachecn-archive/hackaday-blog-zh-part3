# 城堡是 K'nex 建筑之王。

> 原文：<https://hackaday.com/2016/06/29/the-citadel-is-the-king-of-knex-builds/>

追随自己的激情会带来惊人的结果。有时这会导致技术奇迹；其他时候，人们会惊叹于这项技术的使用。后者的典型展示是城堡。

在三年的时间里，redditor [Shadowman39]拼凑出了这个巨大的 K'nex 结构。有超过 17 种不同的路径(！)，45 种不同的元素，超过 40，000 个零件，你会认为它的建造需要一些精心的规划，但事实并非如此！[Shadowman39]很大程度上是在飞行中组装的，只有几个元件需要绘制草图，只有主电梯被证明是有问题的。三个马达为这个结构提供动力——一个用于主电梯，一个用于底部的小型电梯，一个用于释放门。

这是一个绝对的利维坦爱好项目。为了满足偶然看到这张照片的人明显的好奇心，可以在视频中看到它的复杂性:

 [https://www.youtube.com/embed/zwZAM5G9g_A?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/zwZAM5G9g_A?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



这个令人惊讶的坚固结构的零件花费了[shadow man 39]922.38 美元，还不包括他们已经拥有的那些。无论是为了娱乐还是作为一个白板绘图仪，K'nex 可能是一个制造商工具包中非常不受重视的工具。去想想。
