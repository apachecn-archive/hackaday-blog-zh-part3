# 机器人，出发！

> 原文:[https://hackaday.com/2017/07/26/wire-bots-roll-out/](https://hackaday.com/2017/07/26/wire-bots-roll-out/)

为机器人设计和 3D 打印具有特定用途的零件通常比生产具有一般功能的零件更有效率——即使这样，这仍然需要一些时间。如果你去掉那些笨重的尺寸中的两个，仍然生产出一个功能有限的机器人会怎么样？

哥本哈根 IT 大学机器人、进化和艺术实验室的 Sebastian Risi 和他的研究团队发明了一种生产有线机器人的方法。这一过程与工业折弯机生产产品的方式没有太大区别，而且在机器人生产时，专用喷嘴也能够将电机固定在机器人上，因此可以立即进行测试。

一旦满足测试要求，计算机算法就会不断完善机器人的设计，并能够在一刻钟内生产出下一个版本。浪费也少得多，因为电线可以简单地拉直并回收用于下一次尝试。在这三项测试中，一对马达让机器人在行进中摆动——不管是沿着管道、左右摇晃还是滚动。看那铁丝网！
