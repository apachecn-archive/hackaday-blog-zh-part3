# 快讯:一堆 Arduinos 不是自动驾驶汽车

> 原文：<https://hackaday.com/2016/10/19/newsflash-a-bunch-of-arduinos-is-not-an-autonomous-car/>

没有人是完美的。有时你熬夜写博客，偶然发现一个不可思议的故事。你把它写下来，结果是，嗯，*难以置信的*。 [IEEE Spectrum](http://spectrum.ieee.org/cars-that-think/transportation/self-driving/aussie-roboticizes-a-ford-focus-for-less-than-1000) 上了[这个视频](https://www.youtube.com/watch?v=LI86hWuNrfM)(嵌在下面)的当【Keran McKenzie】声称已经用 Arduinos 制造了一辆不到 1000 澳元的自动驾驶汽车。

这个视频实际上非常有趣，我们不认为它是一个大众媒体的恶作剧，而是 YouTube 上的一个笑话。在让汽车“接管”几秒钟后，它突然转向，[Keran]假装撞到了什么东西。(他在用膝盖顶人！)他在车下拍了很多镜头，指着一根电线，据说这根电线能让整个系统工作。是啊，没错。

 [![home-built-autonomous-ford-focus-for-under-1000-li86hwunrfmmp4-shot0004](img/e7dc1be44911e152da1669955650f749.png)](https://hackaday.com/wp-content/uploads/2016/10/home-built-autonomous-ford-focus-for-under-1000-li86hwunrfmmp4-shot0004.jpg) 虽然我们有点扫兴。我们甚至不认为你能以 1000 美元的价格将传感器系统与方向盘、油门和刹车可靠地连接起来，但我们会非常乐意看到这一切完成。我们不是说运行自动驾驶汽车的软件是容易的部分，但如果硬件负担得起，我们很乐意尝试一下。

无论如何，如果你在寻找真正的自动驾驶体验，我们建议从[黑掉 RC 汽车](http://hackaday.com/2012/04/04/make-your-rc-car-drive-itself-with-this-simple-brain-swap/)开始，给它们[比 Arduino](http://hackaday.com/2016/10/10/self-driving-rc-car-uses-an-intel-nuc/) 大得多的大脑。一旦你完成了这项工作，开发一辆真正的汽车是可行的，但是价格昂贵。这有助于[成为【地球热】](http://hackaday.com/2015/12/17/self-driving-acura-built-in-a-garage/)。

以免你认为我们都比你假仁假义，看看我们有史以来最尴尬的帖子。我们可以蜷缩着死去。感觉很快就好了，IEEE 谱！
