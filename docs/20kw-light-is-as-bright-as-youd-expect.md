# 20 千瓦的灯和你想象的一样亮

> 原文：<https://hackaday.com/2016/07/28/20kw-light-is-as-bright-as-youd-expect/>

【Photonicinduction】购买了一个[非常非常亮的灯](https://www.youtube.com/watch?v=LT5_-A0m8_U)。这种 20，000 瓦半米高的卤素灯几乎可以照亮闭着眼睛的人的后脑勺。这些通常用于照亮电影布景。

大多数人甚至不能打开这样的灯，但[光子感应]是一个疯狂的科学家。在他的阁楼上制造闪电时，很容易在脑海中把他描绘成夏洛克·福尔摩斯小说中的反派角色。对我们来说幸运的是，如果他有任何邪恶的倾向，他们会被引导到 YouTube 视频中。

他很好地描述了光的机械和电学性质。身体就像人们期待的白炽灯一样。灯丝支撑在其中的玻璃灯丝外壳。外壳被抽空并填充适当的气体。这种灯非常危险，必须彻底清除外部的指纹，以防止形成热点，这会导致灯爆炸。

经过一番努力，他设法说服自己，里面的细丝实际上不是车库门弹簧，并演示了它们的特性。例如，当它们被加热时，它们的电阻会上升。为了防止切断电源，这么大的灯丝必须预热。如果做不到这一点，就会有大量电流通过。

下一步是将灯连接到他自制的 20 千瓦电源上。他很好地展示了它有多亮。几秒钟之内，他热得直冒汗，甚至无法睁开眼睛去看，小小的太阳占据了他住所的中心。休息后的视频。
