# 扎染长丝用颜色歌唱

> 原文：<https://hackaday.com/2018/07/03/tie-dyed-filament-sings-with-color/>

制造商在他们的工厂生产线上染色各种 3D 打印机细丝；为什么我们不能？[Richard]将这一想法向前推进了一步，创造了自己定制的多色尼龙卷轴。用这些卷轴印刷会产生一种充满活力的图案，这仅仅需要我们的注意，并让我们不禁要问:到底是怎么回事..？

(理查德的)扎染冒险在博客上有清晰的记录。他只是简单地将一卷尼龙缠绕在一起，并用不同的颜色给线轴的各个部分染色。随着灯丝根据口味进行“调色”,打印机的部件会弹出醒目的彩色罗纹图案。

值得一提的是，尼龙的吸湿性极强，在充满有色液体的浴池中对长丝进行染色，肯定会让它充满水分。话说回来，尼龙的吸水能力可能是它染色如此之好的原因。然而，为了成功印刷，细丝必须被烘干(或等同物)。后期烘干，【Richard】看起来没有任何印刷问题，结果不言自明。

3D 打印机可能是这些页面上的常客，但我们仍然喜欢看到增强视觉吸引力的小修改。更重要的是，这种技巧不需要对打印机本身进行任何修改就能产生惊人的效果。话又说回来，如果这份工作听起来对你来说太多了，我们会建议你用一支记号笔。
