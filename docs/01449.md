# 这辆车让你一碰就解锁

> 原文:[https://hack aday . com/2016/02/09/this-car-let-you-fist bump-to-unlock/](https://hackaday.com/2016/02/09/this-car-lets-you-fistbump-to-unlock/)

在黑暗时代，你必须用一把钥匙来锁上和打开你的车门。现在几乎每辆车的钥匙上都有一个遥控器，你只需按一下按钮就可以开锁或上锁。但是许多现代汽车甚至不需要这个。它们能感应到你身上的钥匙，通常用一个按钮来实现上锁或解锁功能。如果钥匙不在附近，那个按钮就不起作用。

[Pierre Charlier]想要这种简单的锁定和解锁，所以他改装了他的汽车，配备了一个 Keyduino [以允许通过 NFC 环](https://hackaday.io/project/9532-nfc-car-door-control)进入。结果是一个非常酷的拳头碰撞，说服你的车打开车门。

Keyduinio 是[Pierre]支持 NFC 的项目，但你也可以使用更传统的带有 NFC 和中继屏蔽的 Arduino。如果你不喜欢戴 NFC 戒指，这个演示也适用于智能手机。这一轮，他甚至展示了如何使蓝牙低能耗工作(BLE)。

在下面的视频中，你可以看到他如何移除汽车的内部锁开关，并修改线束以连接 Arduino。他还包含了所有的代码。唯一棘手的部分是在你的车里做实际的布线，并找到一个合适的电源。这因车而异，所以给出具体的指示并不容易。

打开这样或那样的门是一个受欢迎的项目主题。虽然[Pierre]的项目可能会为 coupe 打开大门，但我们已经看到了另一个项目[在 coop](http://hackaday.com/2015/04/11/raspberry-pi-controlled-chicken-door/) 上工作。
