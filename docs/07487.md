# 带有清晰说明的干涉扫描仪

> 原文:[https://hack aday . com/2017/10/25/interference-scanner-with-clear-instructions/](https://hackaday.com/2017/10/25/interference-scanner-with-clear-instructions/)

一丝不苟。彻底。苛求。这些都是我们用来描述[BrendaEM]关于她自制的 3D 光学干涉扫描仪的视频的词，在休息后可以看到。该扫描仪使用 3D 打印部件和你可能在备件箱中找到的再利用材料。一个旧的光驱托盘用来移动激光雪橇，而一个被剥去的网络摄像头则是一个光学传感器。相关文件的链接，如 3D 模型和 Arduino 草图将在视频的作者部分找到。

在 2:00 位置的视频中，用塑料容器中的波浪演示了操作原理。通过在光源和传感器之间制造两个小孔，可以测量通过的光波。[BrendaEM]使用一些强大的可视化软件将她的样本转换成 3D 模型，这些模型看起来非常酷，同时展示了光的波动性质。

在她的设备的左侧是控制电子设备，它不需要任何特殊的涂层，因为光不会通过这个区域。对于测量相干光的右侧，借用滚石乐队的一句歌词:不再有颜色，我想让它们变成黑色。即使是有孔的黄铜带也是化学变黑的。

这里的大多数激光黑客*使用*激光而不是测量它们，就像这个[激光时钟](https://hackaday.com/2017/08/29/spell-out-the-time-with-frickin-laser-beams/)和一个[激光投影仪](http://hackaday.com/2017/08/29/boredom-lasers-projector/)。
