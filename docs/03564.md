# 如何钻一个弧形孔

> 原文:[https://hackaday.com/2016/09/22/how-to-drill-a-curved-hole/](https://hackaday.com/2016/09/22/how-to-drill-a-curved-hole/)

下次你在装修时，需要在你的墙壁拐角处铺设一些电缆，建造[izzy swan]的[拐角钻机](http://sploid.gizmodo.com/how-to-drill-a-curved-hole-1786526632)来为自己省点事。这是一种类似于定制隧道掘进机的东西，但规模较小。

从一块钢开始，[izzy]追踪并切割出一个 90 度的曲线，并附有一个臂，允许它从中心块旋转。然后他抓起一个随机的钻头，把它连接到一个弯曲的轴上，这个轴固定在钢曲线的尖端。为了完成方便的设置，整个钻机用螺栓固定在一个将夹在角材上的垫块上。

就目前的情况来看，要钻透这个孔需要一些额外的努力，但这不是一个专门建造的装置。在第二次演示中，柔性轴断裂了，但创意仍然存在。现在，[izzy]建议，当没有干墙阻碍你的钻孔时，这是最容易实现的，但这种钻机的概念仍然可以证明是方便的焊接，研磨，等等沿着任何有角度的曲线。
