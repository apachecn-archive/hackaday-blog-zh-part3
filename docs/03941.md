# 想做 PCB？Pantum 知道…

> 原文:[https://hack aday . com/2016/10/25/want-to-make-a-PCB-the-pantum-knows/](https://hackaday.com/2016/10/25/want-to-make-a-pcb-the-pantum-knows/)

这些年来，我们用墨粉转移法做了很多 PCB。这个想法很简单:用墨粉(只是磨碎的黑色塑料)打印一个图案，然后用熨斗或其他加热和加压设备将墨粉转移到覆铜板上。这很有效，而且效果很好。但是获得热量、压力、离型纸和墨粉的正确组合有时是棘手的。

有些人黑掉了他们的打印机，关掉了定影器的电线(这样墨粉就不会粘在纸上),或者让印刷电路板直接穿过它。但是，如果你有一台昂贵的激光打印机，你可能不想仅仅为了运行 PCB 而把它拆开。你最近看过激光打印机的价格吗？我们不确定是廉价设备充斥市场，还是彩色打印机的压倒性普及，但你可以花大约 25 美元或 30 美元买一台 Pantum P2500，并可能以这个价格获得 WiFi 打印。[Mlermen]拿起其中一个，并向您展示如何[将其转换为 PCB 打印机](http://www.instructables.com/id/Modification-of-the-Pantum-2502W-for-Direct-Laser-/?ALLSTEPS)。

这个模型不适合胆小的人。你必须在打印机上切槽，添加一个微控制器，然后你就可以使用旋转的打印机(为 PCB 提供一个平坦的路径)。然而，结果看起来令人印象深刻。花 30 美元和一些额外的费用，如果你使用墨粉转移制作大量的电路板，这是值得的。

顺便说一下，打印机本身就是一笔了不起的交易。它体积小，速度快，工作出色。当然，它不做自动双工，但它有 Windows，Mac 和 Linux 驱动程序。它还有一个 WiFi 热点，可以用手机打印。开箱即用，我们的不做 Airprint，但有一个固件升级允许它(并有一些其他新功能；见下面视频)。它应该直接支持谷歌云打印，但我们还没有找到固件。碳粉被设置为 700 页后自毁，购买新打印机可能比更换碳粉更便宜，所以要做好准备。

[Elliot]发布了关于[墨粉转移](http://hackaday.com/2016/09/12/take-your-pcbs-from-good-to-great-toner-transfer/)的内容，您可能想了解一下。如果你不想拆打印机，你可以试着做一个[化学墨粉转移](http://hackaday.com/2016/01/12/even-easier-toner-transfer-pcbs/)。我们以前也见过[Mlermen]用[其他打印机](http://hackaday.com/2014/10/12/pcb-toner-transfer-method-now-without-the-transfer/)这样做。
