# Hackaday 奖参赛作品:灾难恢复 WiFi

> 原文：<https://hackaday.com/2017/07/20/hackaday-prize-entry-disaster-recovery-wifi/>

Meshpoint 项目起源于 2015 年叙利亚难民危机期间的克罗地亚，当时[Valent Turkovic]和其他志愿者注意到，包括绿色和平和红十字会等非政府组织在内的第一反应者经常难以在现场建立通信。他们得出的结论是，他们不能依赖正常的通信基础设施，因为它要么被损坏，要么过载。

解决方案是一个由开源、自主的 WiFi 网状路由器组成的网络，可以从一个团队扩展到数千人。没有时间进行艰难登录过程的响应者应该会发现设置就像登录社交媒体网站一样简单。

物理节点将由一个可支持多达 150 个连接的路由器组成，全部由一个 ESP8266 运行，并由一个防风雨外壳保护。它们将具有 6-8 小时的电池寿命，通过太阳能/风能充电，从墙上的电流或发电机或简单的 DC 汽车电池充电。

你可以[了解项目的更多信息](http://www.meshpoint.me/)或者[从 GitHub 下载他们的代码](https://github.com/MeshPointWifi)。
