# 乐高提升他们的机器人产品

> 原文:[https://hack aday . com/2017/01/21/Lego-boosts-their-robotic-offering/](https://hackaday.com/2017/01/21/lego-boosts-their-robotic-offering/)

孩子们经常在学校使用乐高思维风暴工具包第一次接触机器人。现在，乐高正在推出 Boost——一款面向所有 7 岁以上乐高建造者的机器人套件。该套件计划于今年晚些时候上市(它出现在最近的 CES 上)，售价约为 160 美元。

Brick 兄弟]有机会在 CES 上试用该套件(见下面的视频)，你可能会发现[他们的评论很有趣](http://www.brothers-brick.com/2017/01/06/exclusive-hands-lego-boost-robotics-prototype-ces-las-vegas-news-video/)。该套件提供了构建五种不同模型的零件和说明:一只猫、一个机器人、一把吉他、一台 3D 打印机和一辆履带车。你也可以在[官方页面](https://www.lego.com/en-us/boost)上查看。

[![boost](../Images/9bd55aea0cf6760c635984b5e67229e3.png)](https://hackaday.com/wp-content/uploads/2017/01/boost.png) 系统通过蓝牙与苹果或安卓平板电脑通信。评论指出，编程语言有点简单，允许动作因刺激而发生，但没有真正的条件测试和分支。除了电机控制之外，该设备还带有可以感知运动和颜色的模块(见右图)。建造者也可以将积木与其他乐高设计结合起来。

我们已经看到了 Mindstorms 的许多增加和增强(以及许多有趣的 T2 应用)。看看 Boost 会出现什么样的黑客会很有趣。
