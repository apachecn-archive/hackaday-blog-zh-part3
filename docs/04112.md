# Launitor 让你免于意外弄脏衣服

> 原文:[https://hack aday . com/2016/11/10/laun itor-saves-you-from-incidentially-smoked-ways/](https://hackaday.com/2016/11/10/launitor-saves-you-from-accidentally-smelly-clothes/)

[domiflichi]是人，是会犯错的。因此，他不能因为偶尔忘记把要洗的衣服放在某台洗衣机里，结果不太好而受到责备。然而，虽然他会犯错，但他并非无能为力。

如果他的洗衣机/烘干机可以发邮件或短信告诉他要洗的衣服会怎么样？这似乎很简单。在机器的侧面加上一个振动传感器和一些大脑。当负载完成时，它会打扰他，直到他下来按下按钮，否则就会下小雨。

[![img_2437cropped-resized_thumbnail](../Images/5c098aa1728ace2be48f6cd706e5c799.png)](https://hackaday.com/wp-content/uploads/2016/11/img_2437cropped-resized_thumbnail1.png) 他开始使用 Arduino-and-ESP8226 组合和压电传感器。压电传感器有很多缺点，所以他改用加速度计，事情就好办多了。我们真的很喜欢他用印刷电路板的安装螺丝作为角支架将它们安装到洗衣机烘干机侧面的方式。这个箱子是一个标准的项目箱，前面有一些时髦的橙色丙烯酸树脂。

这花了一些时间，但是这些天来，多米弗利奇的衣服更新鲜了，T2 的猫喂饱了，他的电器更有意识了。休息后的视频。
