# 作为持久显示的紫外线敏感灯丝

> 原文：<https://hackaday.com/2017/11/14/uv-sensitive-filament-as-a-persistent-display/>

我们报道的一些黑客攻击是对现有设备的修改，其他的则是全新设备的全新构建。然后是实验，那些值得尝试的事情，因为它们可能会成功。在这最后一个类别中，我们有[Matt]的[作品，用紫外线敏感塑料形成了一个简单的持续显示的基础](https://hackaday.io/project/28177-sort-of-e-ink-kinda)，它创造了一个最好的描述为展示承诺的概念证明的东西，并明确证明了他有一个非常值得尝试的想法。

这个想法利用了一种塑料，当暴露在紫外线下时，它的颜色会从白色变成紫色。他 3D 打印了一个华夫饼干样的结构，位于一个 3×3 的 UV LEDs 网格上，然后他可以在 Arduino Mini Pro 的控制下照明。短暂的光照会改变其上方塑料的颜色，产生一个持续几秒钟的“像素”。在这项研究中，他为一个非常简单的 3×3 矩阵显示器创造了一个可行的概念证明，尽管这个显示器相当笨重。该想法提供的优点是，对于相对短的 LED 照明，可以实现相对长的显示时间，这提供了节能的潜力。

概念验证本身并不是特别有用，但从这个想法出发，实际上有可能制造出更大的显示器。表面贴装 led 阵列也许可以以更高的分辨率照亮更大的塑料阵列，这绝对是一个值得尝试的想法，并且显示了进一步追求的前景。如果你想看他的表演，他上传了一个视频，我们把它放在休息时间的下面。
