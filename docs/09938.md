# 在沙滩上画线:把沙滩涂鸦带到下一个层次

> 原文：<https://hackaday.com/2018/07/03/drawing-lines-in-the-sand-taking-beach-graffiti-to-the-next-level/>

当漫步在沙滩上时，总有一种在沙滩上画画的冲动——似乎有必要在上面刻些什么来留下自己的印记。但是有一个难题:你如何着手绘制它？你可以选择保持站立，并尝试伸出脚趾，但这需要很少人拥有的灵巧程度。唯一的另一个选择是弯下腰，用你的手。最终，没有办法在沙子上画任何东西而不失去你的尊严。

解决办法？当然是机器人—【伊万·米兰达】的创意。这个想法很简单，而且执行起来很优雅:做一个大型线性致动器，把它放在轮子上，并连接一个伺服系统，它可以把蚀刻工具定位在沙子里或上面。整个装置一次向前移动一列，垂直通过，根据需要接合或分离标记。这些列很细，提供相对高分辨率的文本，尽管这意味着需要一些时间。添加另一个伺服系统并同时标记两个相邻的列将是一种快速加倍速度的简单方法。

轮子又大又粗，以确保顶部和底部之间的水平移动距离不会改变。当然，当制造像这样的大部件时，如果你已经建造了一台[巨型定制 3D 打印机](https://www.youtube.com/watch?v=EJyjM9ywYbI)，它总是有帮助的。如果你想阅读更多的【伊万】的大型 3D 打印滑稽动作，请查看[的悬浮坦克](https://hackaday.com/2018/05/03/3d-printed-tank-scores-suspension/)，或者[的加大号七段钟](https://hackaday.com/2016/11/10/really-big-digital-clock-finds-use-for-really-big-3d-printer/)。
