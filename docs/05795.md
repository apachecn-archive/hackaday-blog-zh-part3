# 所有砂带研磨机之母

> 原文:[https://hack aday . com/2017/04/29/万能砂带研磨机之母/](https://hackaday.com/2017/04/29/the-mother-of-all-belt-grinders/)

似乎每个人都在制造砂带研磨机。你可能认为杰里米·施密特只是希望赶时髦，但你错了。他花了整整两年的时间来设计满足他需求的完美砂带研磨机。现在他已经造出了他的完美野兽，我们必须说，这真是令人印象深刻！

[Jeremy]见过可以倾斜的研磨机，但大多数研磨机会倾斜整个机器，包括工作台。他用独立的桌子设计了他的机器。这意味着皮带可以以任何角度放置，而桌子保持水平。他用与工件成 45 度角的粗磨完成了一些非常有趣的抛光。

任何建筑都有它的问题。在[杰里米]的案例中，它是在建造一个盒子，作为机器和桌子的接收器。普通方管的硬度不够，所以棒料是个不错的选择。由于焊接的压力，第一次尝试建造盒子导致了扭曲的管子。[杰里米]第二次更加小心，从四个焊缝的一部分移动到另一部分。这样可以防止热量积聚，并且盒子保持笔直。

最终的结果是一个难以置信的坚硬的机器，它绝对能够承受任何(杰里米)可能扔向它的东西。

如果你想在工作中看到更多的砂带研磨机，请查看[【鲍勃】的跑步机砂带研磨机](http://hackaday.com/2017/02/12/turn-your-new-years-resolution-into-a-belt-grinder/)，或[【迈克】的转换](http://hackaday.com/2016/10/07/treadmill-to-belt-grinder-conversion-worked-out/)。
