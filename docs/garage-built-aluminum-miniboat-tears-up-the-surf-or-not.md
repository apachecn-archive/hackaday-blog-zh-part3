# 车库建造的铝制迷你船撕裂了海浪。或者不是。

> 原文：<https://hackaday.com/2016/08/22/garage-built-aluminum-miniboat-tears-up-the-surf-or-not/>

这相当于用钢管和一台旧割草机引擎建造一辆小型摩托车。除了它是一艘由铝和一把旧电锯制成的 DIY 私人船只外，它有着同样的车库建造感觉——和同样令人失望的结果。

当我们第一次看到下面的视频时，我们希望有一艘可以让你自己滑水的船，或者是一艘跳浪的 rooster tailing DIY 水上摩托。唉，选择为这艘船提供动力的链锯动力不足，这艘船几乎没有足够的能量来唤醒它。[MakeItExtreme]承认结果不尽人意，并提到计划用更强大的发动机和喷水驱动装置来修复船只，而不是他们使用的拖曳马达推进器。尽管如此，无论他们做出什么样的改进，都可能会影响他们在船体上的工作，这是一次令人印象深刻的金属制品展示。我们已经习惯于看到[makitextreme]在钢铁上工作，所以看着铝板被切割、弯曲和焊接成防水的船体是很有趣的。看起来有足够的空间来提供更多的能力，我们期待这个版本的 2.0。

如果你喜欢粗糙和现成的金属加工视频，在[MakeItExtreme]的 YouTube 频道上有很多。我们之前已经讨论过很多，包括这个[全地形悬浮滑板](http://hackaday.com/2016/07/28/all-terrain-hoverboard-junk-pile-build/)和[一个使用起来或多或少安全的点焊机](http://hackaday.com/2016/04/24/diy-spot-welder-doesnt-look-like-it-will-immediately-kill-you/)。
