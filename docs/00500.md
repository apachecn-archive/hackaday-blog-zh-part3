# 利用射频透视墙壁

> 原文:[https://hack aday . com/2015/10/29/using-RF-to-see-through-walls/](https://hackaday.com/2015/10/29/using-rf-to-see-through-walls/)

这是一些非常酷的东西。麻省理工学院的研究人员最近发明了一种可以“透视”墙壁的设备。它实际上可以识别固体物体后面的一个人(或几个人)。

他们称之为射频捕捉，它使用无线电波来识别人。有点像高科技无线电频率声纳。使用一种非常复杂的算法，它可以通过分析传输信号的各种反射来重建人的形象。它非常精确，甚至可以根据大小和姿势区分不同的人，甚至可以在空中描绘一个人的笔迹。

听起来不管他们在做什么，都有可能释放大量的辐射。你会这么想，但不是。

发射功率实际上比手机低一万倍。疯了！
