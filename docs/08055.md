# 咖啡桌模型铁路与所有的铃铛和哨子(和灯和警报器)

> 原文:[https://hack aday . com/2017/12/22/coffee-table-model-railway-with-all-the-bell-and-whistles-and-light-and-series/](https://hackaday.com/2017/12/22/coffee-table-model-railroad-with-all-the-bells-and-whistles-and-lights-and-sirens/)

对一些人来说，你创造和控制的真实的物理世界的诱惑是无法抗拒的。再加上对火车的热爱，你就有了铁路模型。有些非常详细，看起来布局越大越好。然而，并不是每个人都有不动产投入到这样的爱好中，搬到 N 轨距铁路通常是在有限的空间里抓痒的关键。

但是[【Chris Plumley】在咖啡桌上完成的 N 轨距铁路模型](http://www.coffeetabletrains.com/ct3.htm)将这个概念带到了一个新的微小的水平。描述这种布局的最高级词汇有很多，首先是咖啡桌本身，这是一种自由形式的雕塑设计，旨在唤起风景的自然轮廓。揭开盖子，一个错综复杂、细节丰富的世界浮现在电梯上。主线火车和两站电车线在完全计算机控制下穿梭于想象的世界，配有音响效果和动画灯光。液晶屏幕代表了免下车电影院——还记得免下车电影院吗？——房子里的火熊熊燃烧，无法被赶来的消防车扑灭。作为奖励，机车有一个仪表板摄像头，提供工程师的布局视图。对细节的关注是美妙的，但有趣的是[这种布局已经以这样或那样的形式存在了 60 年](http://www.coffeetabletrains.com/history.htm)。谈坚持！

如果你对结合微控制器和模型铁路的世界感兴趣，你真的应该看看这个教程。
