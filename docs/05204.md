# SDDSbot 的魅力

> 原文：<https://hackaday.com/2017/03/01/the-enchanting-power-of-sddsbot/>

谁不爱一个好机器人？如果你不——你怎么敢！——然后这个迷人的小淘气可能会给你带来一丝微笑。

SDDSbot——由旧的索尼动态数字音响系统的卷轴盖制成——除了左转、右转或用四条直流电机控制的腿向前走之外，不能做太多事情，但它使用 Pixy 相机和 Arduino 的力量来实现这些。小精灵从纸张上读取表示停止和前进命令的颜色组合，试图在蹒跚学步时将其保持在视野的中心。一旦机器人足够接近“go”色码，纸张的方向就会指引机器人向左或向右转向——目标是能够在迷宫中导航。虽然还没到那一步，但肯定是屈指可数的。

在一位制造商的关怀下，[Arno Munukka]带我们来到他的机器人引擎盖下，展示他是如何巧妙利用小空间的。他为电机设计了定制 PCB 的阿朵，并将它们贴在机器人顶部附近——你可以看到用于计时的电阻器穿过机器人的外壳，增加了功能性的装饰效果。Arduino 大脑贴在后面，Pixy 贴在前面，电源板贴在底座附近。三个 USB 端口分布在机器人的后部——一个充电端口，一个用于编程 Arduino，第三个用于访问 Pixy 摄像头。

你怎么看——对我们未来的想法改变了——呃，硅基朋友？什么事？这里有一个初学者机器人可以帮你入门。
