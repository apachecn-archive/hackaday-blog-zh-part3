# Key-tar 让你在黑客空间里即兴表演

> 原文:[https://hack aday . com/2017/11/08/key-tar-let-you-jam-at-the-hacker space/](https://hackaday.com/2017/11/08/key-tar-lets-you-jam-at-the-hackerspace/)

我们已经看到步进电机制作音乐，但[坦纳科技]的[key-tar](http://www.instructables.com/id/Stepper-Motor-Key-tar-a-New-Kind-of-Instrument/)将它带到了一个全新的水平。key-tar 包含一个强化步进电机声音的声学鼓和一个为吉他放大器提供信号的前置放大器，是一种完全可演奏的乐器。

通过 Arduino 以不同的速度移动步进器会产生不同的音符。用户界面是一个旧的 PC 键盘。显然，[坦纳]回收了他模型中的大部分零件。步进器来自一台旧打印机，键盘是垃圾箱里的救星。

鼓只是一个冰淇淋桶，让踏步机有更多的声音投射。一个木制支架将步进器固定在鼓的底部。即使是鼓，乐器也相当安静，但连接到压电拾音器的 LM386 前置放大器可以将声音发送到更大的放大器。

[坦纳]说这种乐器很容易演奏，但他也是一位有成就的钢琴家，所以我们不知道非音乐爱好者会怎么样。

我们已经看到了一个[步进电机风琴](https://hackaday.com/2016/08/20/an-organ-made-from-back-driven-steppers/)和另一个使用[步进电机的 keytar，该步进电机仍在惠普扫描仪中](https://hackaday.com/2016/08/04/keytar-made-out-of-a-scanner-to-make-even-the-80s-jealous/)。用不了多久，我们就会看到一支踏步机乐队在你附近的黑客空间巡回演出。
