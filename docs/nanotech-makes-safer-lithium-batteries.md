# 纳米技术制造更安全的锂电池

> 原文：<https://hackaday.com/2016/01/13/nanotech-makes-safer-lithium-batteries/>

锂离子电池通常包含两个电极和一种电解质。电池短路或过度充电会产生热量。如果温度达到大约 300 华氏度(150 摄氏度)，电解质就会着火并爆炸。

人们已经尝试过几次制造更安全的锂离子电池，但这些安全措施往往会使它们在过热后无法使用。斯坦福大学的研究人员有一种新的方法来[保护过热的细胞](https://news.stanford.edu/news/2016/january/safe-battery-toggle-011116.html)，这种方法使用了——还有什么——纳米技术石墨烯。诀窍是一层聚乙烯薄膜，其中包含涂有石墨烯的微小镍钉(见右边的电子显微照片)。

薄膜从一个电极导电，因为镍钉在常温下相互接触。如果电池过热，聚乙烯薄膜会膨胀，尖刺不再相互接触，从而切断电路。一旦电池冷却，薄膜收缩，镍钉再次接触，电池恢复正常运行。

我们最近看到了很多电池研究，从[锂空气电池](http://hackaday.com/2015/11/02/lithium-air-might-be-the-better-battery/)到[钠离子电池](http://hackaday.com/2015/12/02/prototype-sodium-ion-batteries-in-18650-cells/)。我们不能等到一次充电就能为我们的手机供电一年。
