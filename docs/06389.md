# 3D 打印的涡轮旋转工具最高转速为 40，000 转/分

> 原文:[https://hack aday . com/2017/07/03/3d-printed-turbine-rotary-tool-tops-40000-rpm/](https://hackaday.com/2017/07/03/3d-printed-turbine-rotary-tool-tops-40000-rpm/)

对于您的高速度，低扭矩的需求，很少有东西击败像 Dremel 旋转工具。电动马达有它的极限，不过，它们通常在 35，000 rpm 左右达到峰值。此外，无论你在清理什么东西，都会有灰尘和碎屑要处理，所以为什么不为你的吸尘器制造一个涡轮驱动的旋转工具附件来一举两得呢？

[johnnyq90]的 3D 打印涡轮解决了电动 Dremels 的另一个严重缺点，即没有牙医办公室的抱怨。当转速达到 43000 转/分时，他的工具发出的声音足以引发恐牙症的发作。涡轮机的定子和转子是 3D 打印的，机身、进气口和真空管适配器也是 3D 打印的。旧旋转工具的轴可以重复使用，但是新的可以很容易地转动。下面的视频展示了完成的工具的运行情况；毫无疑问，在评论中会有人反对吸入灰尘、碎片和炽热的金属碎片，但我们的感觉是，涡轮机将很好地应对这些挑战。直到它不存在。

我们喜欢[johnnyq90]的设计风格，你可能会想起他的[微型特斯拉涡轮机](http://hackaday.com/2016/06/17/micro-tesla-turbine-is-an-engineering-tour-de-force/)或[硝基动力旋转工具](http://hackaday.com/2017/05/30/nitro-powered-rotary-tool/)。他肯定喜欢快速旋转的东西。
