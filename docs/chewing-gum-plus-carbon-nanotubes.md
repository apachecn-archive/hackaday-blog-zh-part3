# 口香糖加碳纳米管

> 原文：<https://hackaday.com/2015/12/16/chewing-gum-plus-carbon-nanotubes/>

通常，应变传感器的柔性受到底层基板的限制。这使得马尼托巴大学的研究人员找到了一个奇特的解决方案:[将碳纳米管混入口香糖胶基中](http://pubs.acs.org/doi/abs/10.1021/acsami.5b08276)。你可以在休息时间下面观看他们的演示视频。

这一过程有着良好的科学严谨性，是让一名研究生咀嚼几块薄荷口香糖半小时，然后在乙醇中清洗口香糖，并在夜间晾干。然后加入碳纳米管，口香糖被反复拉伸和折叠，就像你处理比萨饼面团一样，以对齐“管”。之后，只要接上电极，弯曲时测量电阻。

口香糖传感器的明显优势是它有点粘，非常有弹性。该团队表示，当拉伸到其静止长度的五倍时，它会起作用。用你的拳套试试。

我们已经看到围绕这些部件的几个不同的 DIY flex 传感器解决方案，一个[基于压缩黑色导电泡沫](http://hackaday.com/2012/02/28/building-a-flex-sensor-from-component-packing-materials/)和[另一个使用防静电袋](http://hackaday.com/2012/06/04/making-flex-sensors-on-the-cheap/)，但纳米管和箭牌的高科技和低科技混合是一个新的。
