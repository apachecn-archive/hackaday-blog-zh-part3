# 平视显示器增强仿生海龟的真实性

> 原文：<https://hackaday.com/2017/05/28/head-up-display-augments-bionic-turtles-reality/>

所有机器人研究背后都有一个残酷的事实:与进化相比，我们在让事物移动方面很差劲。大自然已经有几十亿年的实践来制造可以滑行、跳跃、飞行、游泳和奔跑的东西，那么为什么不利用这些平台呢？这就是这只背上绑着导航机器人的海龟背后的想法。

这让我们想起了 S.M. Stirling 的另一个宇宙科幻故事，叫做*天空人。*在故事中，金星上到处都是恐龙，人族殖民者将它们作为负重兽，植入大脑刺激快感中枢来控制它们。虽然韩国高级科学技术研究所的 Phill Seung-Lee 领导的团队不太可能像故事中的殖民者从他们的仿生恐龙那里获得那么多工作，但从这样的设置中仍然有很多东西可以学习。仿生水龟被训练成能够将食物与闪烁的发光二极管联系起来，它使用的是一条发光二极管形式的抬头显示器，以及一个用于积极强化的食物分配器。然后，当海龟在坦克中的航路点之间导航时，发光二极管被用作提示。可悲的是，[全文](http://www.sciencedirect.com/science/article/pii/S1672652917300136)在付费墙后面，但下面的视频让你领略了扣人心弦的动作。

寻找介于两栖动物和虚构的恐龙之间的东西来玩心理游戏？为什么不[让你最好的朋友仿生](http://hackaday.com/2017/01/21/remote-controlling-a-dog/)？
