# 转盘音序器保持技术节拍

> 原文：<https://hackaday.com/2015/11/25/turntable-sequencer-keeps-the-techno-beat/>

我们不得不承认，我们不知道如何称呼这种黑客。艺术家 Graham Dunning 有点干巴巴地称之为“机械技术方法”，但这并不十分恰当。我们在想“转盘-音序器-合成器-节拍盒-配音-踏步机”之类的东西。用牛铃。”

不管你怎么称呼它，但[格雷厄姆]真的从不起眼的转盘中提取了尽可能多的声音，它更多地被用作同步器而不是声源。尽管它也播放唱片——至少是其中的一部分；[Graham]屏蔽凹槽并固定拾音器臂，以便只播放轨道的一部分。其他唱片用导电膜遮盖，上面放置滑动触点，为各种合成器提供触发器。特别巧妙的是机械打击部分；唱片被径向切割形成凸轮，机械从动件周期性地绊倒，撞击低音喇叭的锥体，或者撞击牛铃，嗯，牛铃。

它可能不会吸引每个人，但你不得不承认，观看这个钻机在行动中有一些迷人的东西。节奏非常朗朗上口，正如你在休息后的现场表演视频中看到的那样，有很大的空间让[格雷厄姆]用这种乐器来表达自己。我们也不介意看看 [Compressorhead](https://hackaday.com/2015/11/07/compressorhead-best-robot-band-ever) 如何在他们的表演中使用这个装置。
