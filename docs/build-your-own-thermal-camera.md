# 打造自己的热感相机

> 原文：<https://hackaday.com/2015/12/25/build-your-own-thermal-camera/>

我们之前已经介绍过[Max Ritter]的热感相机项目，但[Max]刚刚迈出了下一步:他提供了最新版本的[自己动手工具包](http://www.diy-thermocam.net/)。DIY Thermocam 通过捕捉 60×80 像素的热图像，改进了他以前的[设计](http://hackaday.com/2014/06/04/cheap-thermocam-gets-an-impressive-rehaul/)，这些热图像可以与来自附带的 640×480 像素相机的可见光图像结合，产生最终图像。它是围绕 [FLIR 轻子](http://www.flir.com/cores/content/?id=66257&pi_ad_id=58254759025&gclid=Cj0KEQiAqemzBRDh2vGKmMnqoegBEiQAqJPuyBBs0KvZNV96zLqQvbBEuDWb_1Jis_83MxBHiRbyA6IaAsvE8P8HAQ)模块构建的，该模块已经用于我们最近看到的许多商业热感相机中。Max 还增加了电池和显示屏，使整个设备成为一台独立的相机。

运行这一切的固件是开源的，用 C++编写，便于修改，因此用户可以构建自己的热感相机。“这种方法是为人们提供自组装套件，以便他们可以将其作为一个开发平台，做他们想通过热成像实现的任何事情”[Max]告诉我们。该套件运行€429(约 468 美元)，全球免费送货。
