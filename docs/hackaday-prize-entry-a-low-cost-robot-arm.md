# Hackaday 奖参赛作品:一个低成本的机器人手臂

> 原文：<https://hackaday.com/2016/05/13/hackaday-prize-entry-a-low-cost-robot-arm/>

机器人手臂很酷，从世界上每所大学的每个计算机科学系的橱柜里藏着的 SCARA 手臂和旧希斯基特机器人来看，它们仍然非常有教育意义。你可以用机器手臂学到很多关于控制系统的知识，或者你可以复制老式的无线电设备 Armatron 不管怎样，你正在做一件非常酷的事情。现在，教育机器人手臂并不多，你能得到的也很小。对于[乔纳森]的黑客日奖项目，[他正在建造一个一米长的低成本机械臂](https://hackaday.io/project/2055-yarrm-7-dof-low-cost-robot-arm)。

你不会找到许多大型、低成本的机器人手臂是有原因的:[平方立方定律](https://hackaday.io/page/1252-small-robots)。一只蚂蚁可以搬运数倍于自身体重的东西，但是如果你把这只蚂蚁放大到人类的尺寸，它的腿就无法支撑自己了。同样，一个小型的手持机器人手臂也可以很好地配合业余爱好伺服系统，但要扩大规模需要大型的步进电机。

为了减轻一点重量，[乔纳森]在底座上使用了非常大的步进电机——NEMA 34s——较小的步进电机沿着手臂的长度移动。他还在 Hackaday 奖中做了一些几乎闻所未闻的事情:为装配而设计。[Jonathan]有点厌倦了攻丝螺纹，现在正在转向攻丝插件，并在寻找更好的方法将电机固定到框架上。总而言之，这是 Hackaday 奖[的一个很好的参赛作品，并且已经是该奖的决赛选手之一](http://hackaday.com/2016/05/02/these-20-projects-won-1000-in-the-hackaday-prize/)。
