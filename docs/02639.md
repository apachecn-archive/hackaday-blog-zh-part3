# 低成本光学平台项目–现在使用激光切割光学系统

> 原文:[https://hack aday . com/2016/06/14/low-cost-optics-bench-project-now-with-laser cut-optics/](https://hackaday.com/2016/06/14/low-cost-optics-bench-project-now-with-lasercut-optics/)

[PWalsh]有一个学习和实验基本光学的聪明主意:他没有使用实际的透镜，而是使用激光切割的透明丙烯酸片切割成透镜轮廓。它们更容易制造、安装、调整和操作，同时仍然以相同的基本方式弯曲光线。它允许简单的动手实验和大量的视觉反馈，非常适合初学者。

![Optical Group](../Images/513ca1180b77a9b420ed4c568d27b9c1.png)

这个想法是[PWalsh]的低成本光学平台项目的一部分，该项目使用激光切割塑料来创建可调节的光学平台组件。我们在之前已经[报道过这个项目，但是【PWalsh】用这些简单的激光切割光学器件的概念扩展了这个想法，用于基础实验；不需要额外工具和少量材料的添加。几乎零成本的特性和附加值是我们一直希望看到的！](http://hackaday.com/2016/05/06/optical-experiments-using-low-cost-lasercut-parts/)
