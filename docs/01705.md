# 面包板视觉

> 原文:[https://hackaday.com/2016/03/07/breadboard-colecovision/](https://hackaday.com/2016/03/07/breadboard-colecovision/)

早在 1983 年，Colecovision 就是一台最先进的游戏机。基于 Z-80，它几乎是一台个人电脑(通过 Adam 插件，它可以提供这一功能，并配有菊花轮打印机进行输出)。[Kernelcrash]着手在试验板上重建 Colecovision 并记录下这个过程。

他早期的项目是在试验板上建造一个 Funvision(更名为 VTech Creativision ),所以他从那个项目中获得的零件开始。他确实做了一些设计上的改变(例如，生成独立的时钟，而不是使用原始设计的方法来产生所需的不同频率)。

当然，现代的 ROM 比最初的设计者拥有的要大。[kernel crash]的版本最初在一个 ROM 中容纳 8 个游戏，但后来的增强功能进一步增加了这个数字。重新设计是有趣的，但它也是说明性的阅读阶段的发展和故障排除它采取了一切工作。

我们已经讨论了一些之前的[复古控制台版本。我们也有相当多的](http://hackaday.com/2015/09/23/a-single-chip-video-game-console/) [Z-80 项目在试验板上和试验板外](http://hackaday.com/2014/12/01/a-z80-computer-with-switches-and-blinkenlights/)。如果你想了解 Colecovision 的概况，以及一些复古广告，请查看下面的视频。
