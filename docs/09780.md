# terrandome 给了植物和恐龙一个新家

> 原文：<https://hackaday.com/2018/06/22/terradome-gives-plants-and-dinosaurs-a-new-home/>

饲养外来植物或动物提供了一个进入电子自动化世界的绝佳机会。当温度、光线和湿度范围至关重要时，传感器是您最好的朋友。如果木工和其他类型的工艺品是你的最爱，为什么不从头开始做呢？[MagicManu]用 MDF 和透明聚苯乙烯建造的侏罗纪公园主题八角形穹顶做到了这一点。

为了种植一些自己的外来植物，[MagicManu]为穹顶配备了 Arduino 供电的控制系统，该系统可以调节温度和光线，并在 LCD 上显示当前的传感器状态，包括湿度。出于简化布线和隔离的原因，湿度本身暂时没有实现自动化。从旧电脑电源中回收的风扇提供了适当的通风，如果穹顶内的温度过高，一套符合侏罗纪公园主题的伺服控制门将自动打开。

[MagicManu]在一个视频中记录了整个构建过程，您可以在休息后观看——尽管只有法语。今年早些时候我们已经看到了一个类似的 DIY 室内园艺项目，考虑到它简单而实用的应用来了解传感器，加上对室内园艺本身越来越大的兴趣(完全是双关语)，这肯定不会是最后一个。
