# 旧对讲机得到谷歌与树莓皮和 AIY 帽

> 原文：<https://hackaday.com/2017/07/20/old-intercom-gets-googled-with-raspberry-pi-and-aiy-hat/>

旧收音机窝棚对讲机；树莓派的全新谷歌语音界面。这些东西中的一个不像另一个，但它们最终一起出现在[这个复古外观的谷歌语音界面](https://www.instructables.com/id/1986-Google-Pi-Intercom/)中，结果非常漂亮。

接受谷歌 hive-mind 移植的人是三个无线调频对讲机中的一个，得分只有区区 4 分。对讲机外壳看起来很像 80 年代中期你的现代生活方式中必备的办公工具或家居配件，是 Pi 和谷歌 AIY 帽子的完美主人。只使用了外壳——甚至连最初的扬声器都没有做成成品。该案件得到了很好的擦洗，一个新的涂料涂层，以振兴去绿色的塑料，并强调了谷歌标志的颜色超过现在已被否决的电台选择器开关。[MisterM]在扬声器格栅后面提供了一个白色 LED，用于细微的反馈。点击原始的 talk bar 可以获得谷歌对快速问题的回答，集成到家庭现有的家庭自动化平台可以打开和关闭灯。休息之后看看它的实际效果。

[MisterM]很幸运地免费获得了一顶 AIY 帽子，据我们所知，这种帽子仍然很难得到。如果你跃跃欲试，不要害怕——事实证明你可以[推出自己的](http://hackaday.com/2017/05/30/diy-google-aiy/)。
