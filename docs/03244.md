# Hackaday 奖参赛作品:开源视网膜扫描仪

> 原文:[https://hack aday . com/2016/08/10/hack aday-prize-entry-an-open-source-retina-scanner/](https://hackaday.com/2016/08/10/hackaday-prize-entry-an-open-source-retina-scanner/)

检眼镜是一种用来检查眼睛后部的设备。这对于诊断从青光眼、糖尿病视网膜病变到检测脑瘤的各种疾病都很有用。正如你对任何与医学相关的东西所期望的那样，这些设备价格昂贵，使得世界上大多数人都无法使用它们。Hackaday 奖的这个项目是一个成本不到 400 美元的检眼镜。

检眼镜是一种相对简单的设备，实际上只需要临床医生戴上头戴式灯，并在患者的眼睛前面拿着聚光透镜。光线从视网膜反射并进入临床医生的视野。当然，最简单的检眼镜需要一点训练才能正确使用，而且不可能拍摄病人视网膜的照片与其他临床医生分享。

开放式间接检眼镜通过使用 Raspberry Pi 相机模块形式的数码相机来解决这些问题。这款相机在 3 W LED 的帮助下，能够对眼睛后部进行成像，拍摄照片，并将图像发送到世界任何地方。这是一个简单的设备，可以由几面镜子、一个廉价的镜头和几个 3D 打印的部件组成，但对于检测眼科疾病仍然非常有价值。
