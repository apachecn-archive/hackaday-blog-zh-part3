# 树莓酱的加工

> 原文:[https://hack aday . com/2015/11/15/processing-for-raspberry-pi/](https://hackaday.com/2015/11/15/processing-for-raspberry-pi/)

你懂加工吗？它是面向电子艺术、新媒体艺术和视觉设计社区的编程语言和 IDE。[Gottfried Haider]最近对 Raspberry Pi 进行了处理，包括一个硬件输入/输出库来操作 Pi 的 I/O 引脚。

如果您想尝试处理，可以使用下面的命令将其下载到您的 Pi 上:

```
curl https://processing.org/download/install-arm.sh | sudo sh
```

也可以从[下载页面](https://processing.org/download/)下载。有专门的[教程可用](https://www.raspberrypi.org/learning/introduction-to-processing/)或者你可以观看一些关于加工的一般视频(见下文)。

[Ben Fry]和[Casey Reas]在 2001 年春天开始处理，当时他们都是麻省理工学院媒体实验室约翰·梅达美学和计算研究小组的研究生。事实证明，它作为第一编程语言很受欢迎，尤其是对于艺术类课程。

网上有很多地方可以下载[加工草图](http://www.openprocessing.org/)。想必很快就会出现一些树莓 Pi 专用的草图。过去我们已经看到过几次[处理](http://hackaday.com/2012/12/06/dithering-in-processing/) a [的情况](http://hackaday.com/2011/09/26/bluetooth-communications-for-android-devices-via-processing/)，但是如果你想使用处理或者想用它向学生介绍 Pi，那么在 Pi 上提供它会带来很多机会。
