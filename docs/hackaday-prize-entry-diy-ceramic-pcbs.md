# Hackaday 奖参赛作品:DIY 陶瓷印刷电路板

> 原文：<https://hackaday.com/2016/05/14/hackaday-prize-entry-diy-ceramic-pcbs/>

我们已经看到了数百种在家制作 PCB 的方法。如果你有一台激光打印机，你可以在一张覆铜板上留下痕迹。如果你有一些过氧化氢和酸，你可以蚀刻这些痕迹。都没有？建造一个小磨，用锥钻切割铜。如果你的电路板是由 FR4 和铜片制成的，在家里制作你自己的 PCB 是很容易的。

不过，印刷电路板可能比一块 FR4 要酷得多。陶瓷 PCB 是电路板制造技术的最高水平，生产的电路板非常坚硬，具有近乎完美的电气性能、高导热性和类似于矿物变压器油的介电强度。陶瓷印刷电路板是用于太空或核反应堆内部的电子设备。

为了参加今年的 Hackaday 奖，[【Chuck】正在建造这些太空级的 PCB](https://hackaday.io/project/11167-diy-space-grade-pcbs)。他不仅在解决 PCB 制造面临的最大挑战，还在建造一台机器来自动化这一过程。

制造陶瓷 PCB 的基本过程是制造一片氧化铝、玻璃粉和粘合剂。这张纸先被钻出来，然后用银墨水印在上面。这些板材一层层堆叠在一起，整个堆叠在压机中压实，然后在熔炉中烧制。

他没有自己制作未烧制的陶瓷片，而是直接从货架上购买。大约每平方英寸一美元。这种材料被压制在具有真空工作台的激光切割机/喷墨组合机上。这仅仅是一个开始，但是[查克]已经拥有了他所需要的一切来开始他的实验，创造真正的太空级 PCB。
