# 喂猫，重新发明轮子

> 原文：<https://hackaday.com/2016/06/02/feeding-the-cat-reinventing-the-wheel/>

很少有项目像宠物喂食器或植物饮水器一样反复出现。(你们这些人是怎么了？你们都和我们一样健忘吗？)不过，乐趣在于实现的细节。或者至少[Emmaanuel]的喂猫器就是这种情况。

本文分为三部分:[蜗杆传动](http://www.emmaanuel.com/Distributeur-de-croquettes-pour-chat-1-3)、[电子](http://www.emmaanuel.com/DIY-ton-distributeur-de-croquettes-pour-chat-2-3-l-electronique)和[外壳](http://www.emmaanuel.com/DIY-ton-distributeur-de-croquettes-pour-chat-3-3-la-structure)(此处英文为:[蜗杆](https://translate.google.com/translate?sl=auto&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=http://www.emmaanuel.com/Distributeur-de-croquettes-pour-chat-1-3&edit-text=&act=url)、[电子](https://translate.google.com/translate?sl=auto&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=http://www.emmaanuel.com/DIY-ton-distributeur-de-croquettes-pour-chat-2-3-l-electronique&edit-text=&act=url)、[外壳](https://translate.google.com/translate?sl=auto&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=http://www.emmaanuel.com/DIY-ton-distributeur-de-croquettes-pour-chat-3-3-la-structure&edit-text=&act=url))。当然，下面还有一个视频。

螺旋钻和电机外壳大量使用 PVC 管和 3D 打印，分配器单元看起来相当专业。关于电子设备就没什么可说的了——一个 Arduino 克隆体、一个 LCD 和一个便宜的齿轮马达做的正是你所期望的。

然而，带有弹簧扣的 CNC 外壳抢尽了风头。它由中密度纤维板制成，不太容易拧紧或粘合。通过精心设计的带有小弹簧的针脚和标签，[Emmaanuel]能够将零件从轧机工作台上取下，然后将它们合在一起。太棒了。
