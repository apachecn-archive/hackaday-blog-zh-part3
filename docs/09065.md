# 用于薄膜沉积的实用等离子体

> 原文：<https://hackaday.com/2018/04/11/practical-plasma-for-thin-film-deposition/>

[谢妮]想要溅射。我们知道，谁不知道？但是[谢妮]的溅射有一个特定的目的:薄膜沉积，大概是为了支持令人敬畏的科学。但是要达到这一点需要一套工具，这些工具并不是现成的，所以他正在廉价地建造一个 DIY 溅射设备。

如果你不熟悉溅射，这是可以理解的。在这种情况下，溅射是一种通过用某种高能粒子(通常是电子或等离子体)轰击第一种固体来将粒子从一种固体转移到另一种固体的过程。如果控制得当，溅射可以应用于质谱分析和半导体工业，在半导体工业中，溅射可以用来在硅片上沉积薄膜，也可以选择性地刻蚀掉薄膜。

无论何种应用，溅射都需要稳定的等离子流。[谢妮]在他的博客上发表了一系列文章，向我们展示了他的等离子体实验，从抽真空到用微波炉变压器制造高压电源。这个项目需要大量的技能和工具，比如玻璃加工、机械加工和高压电子设备。看看下面视频中的等离子。
