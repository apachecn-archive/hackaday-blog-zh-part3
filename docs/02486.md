# Hackaday 奖参赛作品:树莓派零和无人机

> 原文:[https://hack aday . com/2016/05/29/hack aday-prize-entry-raspberry-pi-zeros-and-drones/](https://hackaday.com/2016/05/29/hackaday-prize-entry-raspberry-pi-zeros-and-drones/)

你如何在博客上吸引眼球？把树莓派零放在标题里。如何获得更多的眼球？把无人机这个词也放进去。幸运的是，在 Hackaday 奖中有一个非常特别的项目结合了这两者。是来自【维克多】的 Pi0drone，和它长得一模一样:[一个会飞的树莓 Pi Zero](https://hackaday.io/project/9811-pi0drone) 。

[Victor]一直在研究 PXFmini，这是一个用于 Raspberry Pi 的“盾牌”或“帽子”，它将气压计、IMU 和一些 PWM 输出集成到一个非常小的外形尺寸中，仅比 Raspberry Pi Zero 本身大一点点。它配有标准的 UART 和 I2C 连接器端口，用于连接 GPS 和 FPV 飞行的屏幕显示。

当然，有几十种无人机和无人机的飞行控制器，但很少运行 Linux，更少的平台像 Raspberry Pi 一样得到良好的支持。为了利用这一点，[Victor]正在 Pi 上运行 [Dronecode](https://www.dronecode.org/dronecode-software-platform) ,用于任务规划、真正的自动驾驶以及将遥控四轴飞行器变成真正的无人机的其他一切。它工作了，而且正在飞，你可以看看下面的视频证明。
