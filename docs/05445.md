# Veni，Vidi，ViciLogic 交互式教你数字逻辑

> 原文：<https://hackaday.com/2017/03/31/veni-vidi-vicilogic-teaches-you-digital-logic-interactively/>

这是一年中你意识到你不会遵守所有新年决心的时候了。然而，如果他们中有一个人要学习 VHDL 和 FPGAs，你可能会很幸运。Vicilogic 有一门免费的数字系统基础课程。你必须注册，但它甚至没有验证我们的电子邮件地址，所以注册应该不会太麻烦。

该培训与爱尔兰国立大学高威分校合作，质量上乘，并在您的数字电路浏览器中提供动画演示。你甚至可以自己控制演示。你可能会认为这项工作是在某个浏览器脚本中进行的，但根据该网站的说法，演示是与真正的 FPGA 板绑定在一起的。你可以在使用视频流时查看它们，但我们从未见过这种情况，所以你的里程数可能会有所不同。如果你想预览它的样子，看看下面的视频。有指导练习，也有测验，你必须与演示互动。

本课程详细介绍多路复用器，并展示逻辑如何映射到 Xilinx FPGA。还涉及到 VHDL。这不是一个五分钟的投资，虽然，课程是相当广泛的，有解码器，寄存器，计数器，移位寄存器等模块。演示做得很好，当你改变输入时，不仅输出会改变，真值表和时序图也会更新。

如果你想要更多基于浏览器的 FPGA 仿真，总有 [EDA 运动场](https://hackaday.com/2015/07/21/learn-fpgas-in-your-browser/)。这是纯粹的模拟，但它工作得很好，我们用它做了很多[教程](https://hackaday.com/2015/09/23/learn-flip-flops-with-simulation/)。
