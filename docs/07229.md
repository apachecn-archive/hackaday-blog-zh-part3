# Hackaday 奖参赛作品:轮椅使用者减压指示器系统

> 原文：<https://hackaday.com/2017/09/30/hackaday-prize-entry-wheelchair-user-pressure-relief-indicator-system/>

很难把自己作为一个身体健全的人融入一个身体残疾的人的经历中。身体健全的人很快就会说出“被限制在轮椅上”这样的话，但不知道这到底是什么意思，他们可能会惊讶地发现，坐轮椅的人会指出，他们的椅子非但不是监狱，事实上可能是他们解放的工具。

身体健全的人也很难理解使用轮椅对身体的一些影响。特别是，一些瘫痪的轮椅使用者可能遭受危险的压疮，而由于他们失去感觉而没有意识到它们。因此，这些人有一套旨在减轻引起疼痛的压力的锻炼方法，这些锻炼必须每半小时完成一次。它们可能不方便，也很难执行，因此为了帮助处于这种状态的人，有一个 Hackaday 奖条目，提供关于锻炼制度执行效果的反馈。

该项目将一组力敏电阻放在椅子垫子下面的床上，并用 Arduino 监控它们，然后通过一组 led 向用户提供反馈。到目前为止，他们已经制造出了第一个原型，正在等待零件和招募用户来测试第二个。

想到这个项目会对它所要帮助的人们的生活产生积极的影响，那就太好了。这不是 Hackaday 奖第一次涉足这一领域，[2015 年的获奖者展示了](https://hackaday.com/2016/11/03/beyond-the-prize-eye-driving-wheelchairs/)。
