# 漂浮死星来得正是时候

> 原文:[https://hack aday . com/2017/12/23/floating-death-star-is-just-in-time/](https://hackaday.com/2017/12/23/floating-death-star-is-just-in-time/)

除非你一直生活在高压变压器下，否则你会知道《星球大战》传奇的最新版本。[约翰]有一个亲戚显然是《星球大战》的超级粉丝，所以他着手为他们打造完美的圣诞礼物——漂浮的死星！[John]没有重新发明轮子，而是决定从地球的磁悬浮模型——地球仪开始。然后他拿起一盏死星情绪灯，优雅地用他可信赖的 Dremel 把它切成两半。

情绪灯的一个巧妙之处在于它由黑客最好的朋友供电——来自 USB 电源的 5 伏电压。这使得它很容易与一个充电器和一些光纤照明线脂电池。一堆猫砂代表一个被炸成碎片的闷烧星球，它将整个建筑联系在一起，只有猫砂能做到这一点。
