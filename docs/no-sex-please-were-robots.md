# 请不要做爱，我们是机器人

> 原文：<https://hackaday.com/2015/10/02/no-sex-please-were-robots/>

曾经有一段时间，技术会进步，并引发关于技术引发的伦理问题的辩论。然而，最近，伦理争论似乎(我希望)领先于实际技术。也许这是件好事。

例子:[德蒙特福特大学在 2015 年伦理大会上发表的一篇论文](https://campaignagainstsexrobots.wordpress.com/the-asymmetrical-relationship-parallels-between-prostitution-and-the-development-of-sex-robots/)警告说，与机器人做爱可能会产生与卖淫同等的负面影响。你可能会认为这是一个孤立的学术概念，但显然有一个名为[关于机器人的爱和性的国际会议](http://loveandsexwithrobots.org/)。甚至有一本 2008 年出版的名为《T4 与机器人的爱和性》的书，既不是科幻小说也不是色情作品。

第二个案例:软银创造了一个名叫[Pepper]的机器人，据称它可以理解人类的情感。你知道你买的每样东西都有许可协议，但你并没有真的去读？下面是[Pepper]附带的部分翻译::[…主人不得进行任何性行为或其他不雅行为。](http://www.japantimes.co.jp/news/2015/09/26/national/softbank-warns-sex-android-pepper/#.Vg3el9qrS00)

没错。如果您与您的[Pepper]发生性行为，您就违反了许可协议。老实说，尽管有第 34 条规则，很难想象谁想和 47 英寸的塑料机器人幽会(见下面的视频)。话又说回来，至少从 18 世纪开始，人们就已经开始和机器发生某种性关系了。

然而，这不是一场关于与机器人做爱的利弊的辩论。我猜如果你要去，我不会阻止你，如果你不去，你甚至不会有一点好奇。我认为有趣的是这个问题和黑客们一直以来的疑问之间的相似之处:你能阻止我用我自己的东西做什么？

有一句老话，如果你不能黑它(或打开它，或修复它)，那么你不拥有它。虽然[Pepper]不是我喜欢的类型，但如果她是你的，你付出了代价，也许这应该是你的事。我可以理解，也许，软银可能不希望[Pepper]与色情联系在一起。毕竟，据说 1920 年的色情作品损害了汉密尔顿海滩专利电动按摩机的销售。但如果你认为与[Pepper]共度一个放松的夜晚是一段美好时光，也许这真的不是软银或德蒙特福特大学可以说的。

我以前说过，如果你把性别排除在外，事情就更容易进行逻辑推理。想象一下，如果微软的许可协议禁止你在运行他们软件的机器上写谋杀小说，因为很明显，谋杀是错误的。这可能是不可执行的，而且无论如何，有点奇怪。如果福特让你同意不开车去快餐店，因为我们都知道快餐对你不好，那会怎么样？

如果[Pepper 的]制造商禁止我对其固件进行逆向工程(可能会；我不知道)，我会刚毛，但我会得到它。尤其是如果它能阻止我传播逆向工程。但是如果我拥有它，那么我如何为我自己使用它应该是我的事。

考虑一下:他们试图强制执行他们的执照会是什么样子，如果他们这样做了，法院会如何回应？

然而，从更广泛的意义上来说，作为事物的设计者，我们应该考虑伦理问题。关于它的讨论和辩论是健康的，但最终很少有伦理情况不需要一些个人的解释。例如，你的职业道德可能会阻止你从事武器系统的工作。显然，不是每个人都同意这一点。虽然有明确的案例，但我怀疑 robosex 是否会属于这一明确的类别。
