# 你是说你用纸板造了一辆雷克萨斯……

> 原文：<https://hackaday.com/2015/11/01/are-you-telling-me-you-built-a-lexus-out-of-cardboard/>

所以，你想要一辆新的雷克萨斯？那就给自己下载一辆免费的车，然后用激光切割出来。加入一些胶水，你就有了一辆漂亮的新车。我们会，不太会。

当然，这个宣传片只是丰田的一个宣传噱头(新闻快讯:你的豪华雷克萨斯实际上是一辆丰田)，但我们必须把它交给他们，它起作用了。它基本上是 1700 个单独成型的激光切割纸板横截面，这些横截面被精心堆叠并粘合在一起。我们喜欢的是这种技术——用 2D 制作一个 3D 物体。

使用 2D 零件创建 3D 形状并不是什么新鲜事。大多数人对这项技术的第一次体验是制作模型飞机。轻木板材被切割成型材，用桁条连接起来，形成飞机的形状，而不是硬纸板。当你只有 2D 材料时，这是一种非常有效的制作 3D 结构的方法。随着 3D 软件的普及，现在是尝试 3D 建筑的最佳时机。一个很好的例子，见这个使用 Autodesks 免费 [123D Make](http://www.123dapp.com/make) 软件制作的[碳纤维吉他](http://www.instructables.com/id/Carbon-fiber-guitar-body-Made-at-Techshop/?ALLSTEPS)。不要局限于平行层，你可以用免费开源的[草图椅](http://www.sketchchair.cc/)软件生成各种形状，包括家具。这将派上用场，因为当你等待你的新纸板车完成打印时，你很可能需要一个地方坐。
