# 你的硬盘作为一个意外的麦克风

> 原文：<https://hackaday.com/2017/10/08/your-hard-disk-as-an-accidental-microphone/>

当我们需要电脑与周围的世界互动时，我们习惯于将外设连接到电脑上。例如，Arduino Uno 需要一个防护罩来开灯。只是有时候，计算机和它周围的真实物理世界之间可能会有意想不到的交互，这是[Alfredo Ortega]在 EKO 党大会上的讲话中发现的其中一个时刻。他演示了传统的旋转生锈的计算机硬盘如何与周围的振动相互作用，[既可以成为一个基本的麦克风，也可以被共振频率为](https://github.com/ortegaalfredo/kscope)的声音破坏。

看起来您可以在读操作期间测量硬盘磁头的响应时间，而不需要任何权限提升。这种定时随着振动而变化，因此可用于重建驱动器所面对的声音。因此，它成为了一个麦克风，虽然不是一个非常好的低音响应。他继续研究声音对驱动器的影响，发现它有一个共振频率，在这个频率下振动会导致它不可读。

遗憾的是，这次谈话本身似乎还没有在网上发布，但鉴于前几年的 EKO 谈话都在 YouTube 上，当尘埃落定时，你可能会看到完整的内容。与此同时，他发布了一个视频演示，我们已经发布在休息。
