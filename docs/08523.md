# 骑自行车，给手机充电

> 原文:[https://hackaday.com/2018/02/08/ride-bike-charge-phone/](https://hackaday.com/2018/02/08/ride-bike-charge-phone/)

北半球的春天来了，很快外面就会变得足够好，可以骑着自行车在城里逛逛了。但是自行车没有电源插座，所以在旅途中给手机充电需要预先考虑和充满电的电池组。不一定非要那样。你在努力让自行车动起来，为什么不让自行车为你工作呢？

如果你曾经使用过马达作为发电机，那么你就能看到它的发展方向。这就是[创意嗡嗡声]的自行车充电手机背后的基本原理。当自行车轮转动时，轮缘会接触到一个连接在 DC 发动机输出轴上的小轮子。转动内部装有永磁体的电机输出轴会感应出一个小电压，在这里，它通过 DC-DC 升压转换器放大，并输出到 USB 插孔。

只要你能找到一种方法将手机固定在自行车架上，或者使用一根长电线和良好的电缆管理，你就成功了。轮滑过去看看[创意嗡嗡声]建立它，并给它一个静态测试运行。当你等待骑自行车的天气时，你仍然可以通过转动曲柄来使用这种充电器[。](https://hackaday.com/2015/01/27/diy-phone-charger-born-from-cyclone-disaster/)
