# 墙上的魔镜，“圆周率和超感哪个更公平？”

> 原文:[https://hack aday . com/2015/12/30/魔镜是最公平的](https://hackaday.com/2015/12/30/magic-mirror-on-the-wall-is-pi-or-esp-fairest-of-all/)

“亲爱的，天气怎么样？”*“不知道。让我照照镜子。”*镜子？

[Dylan Pierce]和[squix]都有显示天气的镜像项目。他们采取了两种不同的方法，这是一个有趣的对比。[迪伦]用的是树莓派，镜子后面有一个真正的监视器。[squix]将一辆[有机发光二极管放在由 ESP8266](http://blog.squix.ch/2015/12/esp8266-weatherstation-with-magic-mirror.html) 驱动的镜子后面。似乎有不止一种方法可以黑一面镜子，或者任何东西，这就是为什么黑客很有趣。

 [![Raspberry Pi Booting](../Images/7c43ee623071515c39b682b3f6713aad.png "pi_booting")](https://hackaday.com/2015/12/30/magic-mirror-on-the-wall-is-pi-or-esp-fairest-of-all/pi_booting/) Raspberry Pi Booting [![Framing Mirror and Monitor](../Images/2886b434293097cb9515bf57c840f37a.png "7_fit_to_case")](https://hackaday.com/2015/12/30/magic-mirror-on-the-wall-is-pi-or-esp-fairest-of-all/7_fit_to_case/) Framing Mirror and Monitor

[squix]从一个相框开始，在玻璃上添加着色膜，这样它就会反射。一小部分色调被删除，让有机发光二极管可以看到。ESP8266 软件连接到地下气象站获取最新信息。

[迪伦]的树莓派版本在镜子后面放了一个 27 英寸的显示器。这要么令人印象深刻，要么有些夸张，但看到 Linux 在镜子后面启动，这种努力是值得的。Pi 会生成一个网页，使其适合作为通用信息亭。
