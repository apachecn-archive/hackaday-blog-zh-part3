# 自动图书扫描仪为埃塞俄比亚学生带来知识

> 原文:[https://hack aday . com/2016/04/15/automatic-book-scanner-to-bring-knowledge-to-Ethiopian-students/](https://hackaday.com/2016/04/15/automatic-book-scanner-to-bring-knowledge-to-ethiopian-students/)

2013 年，埃塞俄比亚一所大学的院长在汉诺威的创客大会上发言，概述了他的一个担忧；发达国家教科书的高昂价格阻碍了发展中国家大学的教育事业，例如他自己的大学。他在那里寻求创客社区的帮助来解决他的问题，他的一群观众[接受了挑战，创造了一个负担得起的和可访问的自动书籍扫描仪](https://hackaday.io/project/10383-automatic-bookscanner)。

他们的扫描仪建立在谷歌工程师[Dany Qumsiyeh]的工作基础上，他的[开源线性书籍扫描仪](http://linearbookscanner.org/)通过在三角形棱镜成型器上遍历打开的书籍来翻页，这样页面在经过表面精心设计的插槽时会被真空翻转。他们的修改用[效应](https://en.wikipedia.org/wiki/Coandă_effect)取代了真空，更温和地打开每一页，希望减少损坏被扫描卷的机会。

整个机器由 Raspberry Pi 控制，扫描由平板扫描仪上的线性扫描光学器件、传感器和电子设备执行。

该项目的一个重要设计目标是确保在没有发展中国家可能难以找到的特殊工具或专业知识的情况下建造扫描仪，并确保它尽可能便宜。机器的框架是现成的挤压铝，主体是丙烯酸板，如果需要，可以用手锯切割成型。据估计，该设备的建造成本将在 500 欧元(约 568 美元)左右。

更多信息可以在该项目的网站(德语，[谷歌翻译链接](http://translate.google.com/translate?sl=auto&tl=en&u=http%3A%2F%2Fwww.bookscanner.de%2F))上找到[，包括一系列视频，比如显示设备运行的视频。](http://www.bookscanner.de/)

从曾经在一家大型出版社从事专业图书扫描工作的角度来看，这台机器的好处是显而易见的。去除装订并自动扫描每一页作为一个单独的纸张产生了非常快速和高质量的结果，但是其本质上损坏了被扫描的体积。这台机器有望为图书扫描问题提供一个侵入性相当小的解决方案。

还值得注意的是，该项目没有解决任何可能因扫描商业出版的教科书而产生的版权问题，尽管这是最终用户更关心的问题，因为他们用它扫描的是什么，而不是制造商。
