# 2017 年最佳会议徽章是 WiFi 草坪

> 原文:[https://hack aday . com/2017/02/17/the-best-conference-badge-of-2017-is-a-wifi-lawn/](https://hackaday.com/2017/02/17/the-best-conference-badge-of-2017-is-a-wifi-lawn/)

现在是二月，会议季节还没有开始，今年的最佳电子徽章已经产生了。对于今年的 MAGfest，[CNLohr]和朋友们[分发了 2000 个基于 ESP8266 的 swag 徽章](https://www.youtube.com/watch?v=8WCIu-3OFEQ)。

这些定制的#badgelife 徽章*不是。*显然，MAGFest 不会允许【CNLohr】将这些设备称为‘徽章’。相反，这些是“swadges”，是赃物和徽章的组合。这些 swadges 上有一个 ESP-12，一个四色 RGB LEDs，以及用于上、下、左、右、A、B、选择和启动的按钮。swadge 由两节 AA 电池供电(从 Costco of all places 采购)，从各方面来看，徽章取得了圆满成功。

[CNLohr]是一位出色的 ESP8266 专家，这个徽章的设计目标之一是让所有这些交换机通过原始 WiFi 帧进行通信。这被证明是一个伟大的想法——使用普通的 WiFi 基础设施，两千个徽章饱和了频谱。的控制系统只是三个徽章，每个 WiFi 频道一个，告诉所有的徽章改变 led 的颜色。

swadge 取得了完全的成功，但随着数百个 blinkey glowey WiFi 设备的出现，你知道[CNLohr]将会推出一些很酷的东西。这一次，他把他的草坪变成了狂欢派对。大约有 175 个开关被布置在草坪上，全部由一个控制器开关控制。院子里每块草地上的 led 灯的颜色会随着 WiFi 信号强度的变化而变化。通过在头上挥舞控制器徽章，[CNLohr]把他的院子变成了一个旋转闪烁的迪斯科舞厅。它看起来很棒，尽管它可能不会像一些【CNLohr】的[和其他 ESP 黑客](http://hackaday.com/2015/02/17/mapping-wifi-signals-in-3-dimensions/)那样可视化 WiFi 信号。

这是一个很棒的建筑，在 MAGFest 上受到了大家的好评。一定要看看下面的视频，它们真正展示了这款非常酷的可穿戴硬件的功能。
