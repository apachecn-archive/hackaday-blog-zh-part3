# 巧克力工厂仿真用乐高制作巧克力棒

> 原文:[https://hack aday . com/2017/05/29/chocolate-factory-simulation-makes-bars-with-Lego/](https://hackaday.com/2017/05/29/chocolate-factory-simulation-makes-bars-with-lego/)

[Michael Brandl]参观了奥地利 Bludenz 的 Milka 巧克力工厂，并受到启发，为哥本哈根 2017 年乐高世界活动制作了这个生产流程模拟图。

这个过程从空模具骑在双排坦克履带上开始。随后的模块似乎用乐高原料填充模具，冷却棒，并将其从模具中取出。最后两步摇滚:[迈克尔]建造了一个自动售货机，将一个小纸箱放到生产线上，大小可以容纳 3 块乐高积木。箱子滚动到生产线的末端，由气动抓具拾取，抓具将箱子放在托盘上。

虽然比我们最近介绍的[乐高液体处理器](http://hackaday.com/2017/04/03/lego-liquid-handler-and-big-biology/)更古怪，但这里有很多有趣的机器人技术可以学习。在[反转角度视频](https://www.youtube.com/watch?v=GORN2jhlapg&t=164s)中，你可以看到更多各种电机和传感器的接线情况。流水线上散落着六块 EV3 砖。砖块控制 15 个小马达、2 个大马达、7 个触摸传感器和 3 个光传感器。[迈克尔]添加了一些漂亮的东西，比如两个颜色传感器的组合，在 1:45 左右的反角度视频中看到，可能用于保持工厂运营同步。

看看[的【阿诺克·威普瑞希】的](https://www.youtube.com/watch?v=fIDgn5oA6DE)头脑风暴发送的饮料机器人礼服。乐高版是为 Robotexotica 打造的。此外，他在自己的网站上还有很多[项目](http://lego.brandls.info)。
