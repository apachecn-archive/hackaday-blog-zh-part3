# 540 块 PCB 组成一个巨大的 LED 立方体

> 原文：<https://hackaday.com/2017/06/02/540-pcbs-make-a-giant-led-cube/>

几乎任何人都可以制作一个简单的 LED 立方体。但是如果你想用 512 个 led 做成一个 1 米的立方体呢？[Hari]想要这样做，所以他[用 EasyEDA 创造了两种不同的 LED 板](https://hackaday.io/project/19115-giant-rgb-led-cube)。每种类型的板有 270 个，总共 540 个(只有 512 个 led，所以我们猜测他有一些备件，因为小板是如何嵌板的)。目标是将这些木板组合成一个每边都超过 3 英尺的立方体。

为了简化布线，电路板被制成菊花链状，就像积木模块一样。然而，为了让东西排成一行，每列 LED 板必须旋转 90 度。你可以在下面看到几个关于该项目的视频。

我们之前已经讨论过 [EasyEDA](https://hackaday.com/2015/08/21/a-tale-of-two-browser-pcb-tools/) 了。它是一个基于浏览器的工具，用于定义 PCB，并包括许多其他功能，如模拟和自动布线。您可以从工具中下载 CAM 文件，也可以非常容易地直接从网站订购电路板。

[Hari]显然喜欢大的或小的 LED 立方体。顺便说一下，EasyEDA[并不是唯一的基于浏览器的解决方案](https://hackaday.com/2017/02/03/creating-a-pcb-in-everything-upverter/)。
