# 巨大的 USB 电源库

> 原文:[https://hackaday.com/2017/09/27/monstrous-usb-power-bank/](https://hackaday.com/2017/09/27/monstrous-usb-power-bank/)

在某些时候，清理备件箱——或者柜子，或者车库——变得很有必要。这是很危险的，因为它可能会诱发更多的项目想法，并完全否定最初的目的。考虑到他在过去十年中收集的一堆电池，他决定不扔掉它们，而是将它们回收到一个[奇形怪状的 USB 电源库中](https://www.youtube.com/watch?v=1GxZyY-mfDo)。

在这个电力银行的大部分内部是令人瞠目的 64 个 18650 锂离子电池，大部分是从笔记本电脑电池中收集的，并以平行的 8×8 模式连接，估计容量超过 100，000mAh(！！).所有这些储存能量的看门人是 Tindie 的双 USB 电源充电器板。

啊——但是如何包装所有这些力量呢？手巧的人的秘密武器:管道胶带！
