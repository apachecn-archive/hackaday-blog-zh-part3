# 旋转等离子体涡流扬声器

> 原文:[https://hack aday . com/2016/04/12/旋转等离子体涡流扬声器/](https://hackaday.com/2016/04/12/rotating-plasma-vortex-speaker/)

[Anthony Garofalo]已经[制造了一个更高级的等离子扬声器](https://www.youtube.com/watch?v=bMkgQshDdcI)。不满足于一个简单的火花，他使用了等离子涡流。为了形成涡流，火花隙被换成放置在环形磁铁中心的电极。电弧受到的[洛伦兹力](https://en.wikipedia.org/wiki/Lorentz_force)使其围绕磁体中心的电弧足够快速地旋转，以呈现为连续的等离子体片。

扬声器从逆变器获得电源，逆变器使用基于 555 的脉宽调制器通过 MOSFET 驱动的反激式变压器。你可以在休息时间下面的视频中看到结果，这非常令人印象深刻，但可能还没有准备好坐在你的高保真堆栈中。由此产生的声音不如固定电弧产生的声音好，但看起来更酷。
