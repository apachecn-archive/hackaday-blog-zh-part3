# 玩火:氧乙炔基础知识

> 原文：<https://hackaday.com/2016/11/14/fun-with-fire-oxy-acetylene-basics/>

如果说几代好莱坞抢劫电影教会了我们什么，那就是抢劫银行金库相当容易。通常是从一个人和一个听诊器开始，但这从来没起作用，所以坏人拿出割炬烧了进去。但是知道如何利用这种原始力量意味着你必须学习氧乙炔的基础知识，而[这个老托尼]的新视频将让你的犯罪生活有一个良好的开端。

在另一个制作精良的视频中，[Tony]详细讲述了氧气和乙炔的秘密，以及如何在不引爆自己的情况下处理它们。他首先参观了设备，包括对乙炔罐内部的有趣观察——原来气体溶解在丙酮中储存在罐内的多孔基质中。沿着软管，他涵盖了所有重要的回火防止器，不同风格的火炬，甚至碳氢化合物燃烧的化学计量，以及如何调整氧气流量导致不同工作的不同火焰类型。他展示了氧乙炔焊接是如何成为穷人的 TIG，并最终通过在不到 6 秒的时间内切开一块 3/8″的钢来满足这种破坏性的冲动。

我们一直想要一个像样的氧乙炔钻机，并且[托尼]已经说服我们，这是商店的另一个必需品。你可以用它们做很多事情，尤其是解开腐蚀的紧固件。但是如果蓝色扳手超出了你的价格范围，而你仍然想把金属粘在一起，你就要学习如何用丙烷焊炬钎焊铝。
