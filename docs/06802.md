# 你可能不想在你的盥洗室里找到这个厕所

> 原文:[https://hackaday . com/2017/08/16/你大概不想在你的洗手间里找到这个厕所/](https://hackaday.com/2017/08/16/you-probably-dont-want-to-find-this-toilet-in-your-washroom/)

好吧，这是一个有点奇怪，但完全符合主题:一个会说话的厕所从裤衩船长儿童书籍的页面上撕下来。Hackaday.io 用户[ hamblin.joe ]的县集市有一个厕所装饰比赛，在邻居儿子的建议下，[hamblin.joe]策划了一个[的计划，按照英雄的敌人](http://www.youtube.com/watch?v=y2o1wsOGDUA)的方式，使用 Arduino 实现厕所自动化。

两个 Arduinos 组成了这个马桶的大脑，一个 Adafruit 波盾赋予了它声音功能，一个声波传感器将在有人靠近时触发马桶的性能例程。挡风玻璃刮水器电机通过一根连接到冲压角铁支架上的扁铁驱动抽水马桶盖。安装马达的底座有点棘手，因为它必须精确切割，这样它就不会在马桶里移动。一个类似的装置可以打开马桶水箱的盖子，但是要让它正常工作要稍微复杂一些。一旦解决了这个问题，就有足够的空间来放置一对 12V 电池和一个扬声器。哦，还有一双幽灵般的眼睛和一些看起来邪恶的牙齿。
