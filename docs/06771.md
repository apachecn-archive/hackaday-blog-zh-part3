# Hackaday 奖参赛作品:除草机项目，现在有了火焰

> 原文：<https://hackaday.com/2017/08/27/hackaday-prize-entry-the-weedinator-project-now-with-flame/>

我们喜欢[除草机项目](https://hackaday.io/project/21744)为今年的 Hackaday 奖大动干戈！这个由[tegwyntmmffat]进行的雄心勃勃的项目是建立在以前的努力之上的，那是一台安装在拖拉机上的除草机(如上图所示)。它无情地撕碎任何杂草；它这样做的方法是翻耕有序生长的韭菜之间的所有东西。这个系统起作用了，但是它确实不够精确。我们怀疑它有残忍地切碎偶尔韭菜的坏习惯。新版本采取了不同的方法。

新的除草机将是一个自主机器人漫游车，结合使用全球定位系统和彩色标记进行导航。通过一个有趣的可调悬挂系统来帮助精确定位，除草机将使用各种附件来帮助植物护理。单个杂草将被光学识别，并通过小型丁烷火炬发出的精确火焰发送到空中的大温室。这是一个雄心勃勃的项目，但[tegwyntmmffat]正在建立从前生获得的经验，我们很高兴看到它的发展。
