# 一种台灯除焊烟器

> 原文:[https://hack aday . com/2016/10/02/a-desk-lamp-solder-fuse-extractor/](https://hackaday.com/2016/10/02/a-desk-lamp-solder-fume-extractor/)

我们这些花了一生时间构建电子项目的人可能吸入了比我们应该吸入的更多的焊料烟雾。这不是一个理想的情况，因为我们可能会因此增加患哮喘和其他疾病的风险。

这些年来，排烟系统和过滤器作为专业焊接环境的一部分变得越来越普遍，这种趋势也开始出现在家用焊接设备中。和往常一样，商业产品走到哪里，硬件黑客就跟到哪里。我们已经看到人们用电脑风扇生产他们自己的焊接烟尘过滤器。

一个特别好的例子来自【无工程师】，他发布了[一个可指示的](http://www.instructables.com/id/Desk-Lamp-Smoke-Extractor/)和 YouTube 视频，显示了安装在台灯上的过滤器的破裂。烤面包机是用来加热一块丙烯酸的。软化的塑料然后被成形以适合灯的轮廓。该灯的铰接臂非常适合在需要的地方放置照明和排烟装置。这不是最复杂的黑客，但我们会毫不犹豫地在我们的工作台上放一个这样的。不过，我们可能会在我们的产品中添加一个活性炭过滤器。
