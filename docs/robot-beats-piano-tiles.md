# 机器人敲打钢琴瓦

> 原文：<https://hackaday.com/2016/04/28/robot-beats-piano-tiles/>

失控的机器是喜剧的主要内容之一。经典表述见卓别林《摩登时代》。因此，虽然[丹佛·芬恩]的[机器人手指可以玩 iPad 钢琴视频游戏](https://www.youtube.com/watch?v=fqOW84ZTL7k)一开始只是令人印象深刻，但它最终实际上非常有趣。查看链接视频，了解我们的意思。

关于这个机器人的细节很少，但不难看出它是如何工作的。四个大型伺服电机在棒的末端有导电泡沫垫，可以触发 iPad 上的点击。电机驱动器和某种微处理器(这是小事吗？)驱赶他们。头顶上的 iPhone 充当相机，必须进行某种图像处理。[丹佛芬]如果你在那里，我们可以使用更多的细节。

我们怀疑，基于叙述者提到手机摄像头的速度，这是他们的主要营销噱头之一，我们正在被病毒式营销。(或者也许我们只是变得太疲倦了。)话虽如此，还是很酷的。

我们以前见过[机器人玩视频游戏](http://hackaday.com/2016/02/10/pi-powered-robot-plays-puzzle-dragons/)，当然[不止几次](http://hackaday.com/2014/04/17/lego-robot-plays-games-for-you-as-you-sleep/)。但这是我们能回忆起的第一个利用机器非凡灵活性的，这很酷。
