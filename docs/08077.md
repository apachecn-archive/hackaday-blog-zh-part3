# 在这个简单的 Stewart 平台上像山羊一样保持平衡

> 原文:[https://hack aday . com/2017/12/27/balance-like-a-mountain-goat-on-this-simple-Stewart-platform/](https://hackaday.com/2017/12/27/balance-like-a-mountain-goat-on-this-simple-stewart-platform/)

在为康奈尔大学[布鲁斯·兰德]的微控制器课程制作这个三自由度斯图尔特平台的过程中，没有山羊受到伤害。

如果“Stewart platform”这个名字听起来没什么印象，下面的视频将会帮助你。[Team Microgoats]建造了一个小型的飞行模拟器中常见的机械系统，选择 3 自由度以简化设计。他们的 PIC32 控制的步进器可以摆动和编织桌子，以响应嵌入 3D 打印山羊底座的 MPU-6050 六轴加速度计的输入。说山羊似乎没有其他作用，但山羊很酷，为什么不呢？如果你曾经见过一只山羊在陡峭的垂直岩壁上嬉戏，就像它走过停车场一样，你就会明白 Stewart 平台提供的平衡和控制之间的联系。

[布鲁斯·兰德]的课程总是有一大堆整洁的项目在每年的这个时候出现在我们的提示栏里，比如[一个 POV 箱风扇](https://hackaday.com/2017/12/23/pov-display-is-fan-tastic/)，[一个硬币电池滚动投掷器](https://hackaday.com/2017/12/22/rickroll-the-masses-with-a-coin-cell-throwie/)，以及[一个小型电动机的测力计](https://hackaday.com/2017/12/18/a-dynamometer-for-measuring-motor-power/)。
