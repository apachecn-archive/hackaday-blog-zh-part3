# 你的无人机很酷，但它不是喷气式战斗机

> 原文：<https://hackaday.com/2017/11/16/your-drone-is-cool-but-its-no-jet-fighter/>

有一些社区，我们的快乐硬件黑客乐队与他们有很多共同之处，但我们并不经常与他们联系。例如，汽车改装或铁路模型场景，或模型飞机兄弟会更面向车间。许多这样的社区更多的是为了活动而不是为了制作而存在，他们中的一些涉足了构建工具包，但是他们中有一批从零开始创建惊人项目的核心人员。

就拿【伊戈尔·内戈达】来说吧。他不满足于制造任何模型飞机，他从零开始，按照自己的设计制造了自己的飞机。如果为自己设计的[相当于缩小版的喷气式战斗机](https://www.youtube.com/watch?v=jlYdP4A46Mo)还不够的话，[他还建造了自己的喷气式发动机为其提供动力](https://www.youtube.com/playlist?list=PLyyfwUFI3XU-CvGsSXnczu4p9AEobpAED)。他的视频都是俄语的，所以如果你不会说俄语，可以使用 YouTube 的字幕功能，但这些视频非常好，如果你无法访问英语翻译，你会想学习语言，只是为了听他的评论。

下面的视频首先向我们展示了使用涵道风扇的快速滑行测试，然后是喷气发动机的全面试飞。在一次令人印象深刻的飞行之前，燃料系统和飞行控制系统有一个解释，这次飞行似乎是从前冷战时期的跑道开始的。有一些有趣的时刻，如在一辆小型掀背车中运输一架大型喷气式飞机模型，但车库车间的工作质量仍然很出色。突然，多旋翼飞机不再合适了，我们想要一架像伊戈尔那样的喷气式飞机！
