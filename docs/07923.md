# 非电子产品的互联网

> 原文:[https://hack aday . com/2017/12/07/the-internet-of-non-electronic-things/](https://hackaday.com/2017/12/07/the-internet-of-non-electronic-things/)

即使是最简单的物联网项目的物料清单也可能包括某种带有某种无线模块的微控制器。但是有一天一个有用的物联网物品的 BOM 会只列出一个物品吗？很有可能，如果[这些无电子设备的 3D 打印物联网设备](https://phys.org/news/2017-12-d-wifi-electronics.html)是任何迹象的话。

虽然你可能认为华盛顿大学学生[Vikram Iyer]和[Justin Chan]在论文中描述的无硅设备没有机会上网，但他们实际上已经建立了一系列有用的物联网东西，包括亚马逊 Dash-like 按钮。他们系统的关键是反向散射，它调制入射的射频波，为接收器编码数据。我们展示的一些反向散射系统包括[一个使用商业调频广播的土壤传感器网络](http://hackaday.com/2017/03/03/using-backscatter-radio-for-a-soil-sensor-network/)和[使用 LoRa 作为载体的混合可印刷传感器](http://hackaday.com/2017/09/19/hybrid-technique-breaks-backscatter-distance-barrier/)。但是这两者都至少需要一些电子设备，因此需要某种能量。[Chan]和[Iyer]使用导电细丝印刷天线，可以通过旋转齿轮进行机械切换。数据可以通过入射 WiFi 信号的交替反射和吸收的速度来编码，或者摄像头可以为按钮和类似的小部件编码数据。

这是一个令人惊讶的简单系统，虽然显示的设备可能需要一些机械调整，但概念验证有很大的潜力。流量计、液位传感器、报警系统——你会打印哪种传感器？下面报数。
