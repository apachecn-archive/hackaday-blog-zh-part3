# 黑客日奖品入口:一个本地定位系统

> 原文：<https://hackaday.com/2016/06/26/hackaday-prize-entry-a-local-positioning-system/>

全球定位系统的使用无处不在。从你汽车上的卫星导航到一个点上方盘旋的四轴飞行器，我们每天都有数百种方式使用全球定位系统。GPS 有几个缺点:获取信号需要一段时间，GPS 在室内工作不太好，因为物联网上的节点会很便宜，他们可能不会有 GPS 接收器。

这些事实为一种新的定位系统打开了大门。使用硬件设备的本地定位系统已经有了，但是仍然能够确定几英尺内的位置。为了他的 Hackaday 奖参赛作品，[Blecky] [正在建造 subps Ranger](https://hackaday.io/project/9242-subpos-ranger)，这是一个基于 802.15.4 无线电的本地定位系统，仍然允许设备确定自己的位置。

SubPos Ranger 基于[Blecky]的 2015 年 Hackaday 奖参赛作品[，subps](https://hackaday.io/project/4872-subpos-positioning-system)使用 WiFi、RSSI 和三边测量来确定接收器相对于三个或更多基站的位置。它工作得非常好，甚至在 GPS 不能工作的地方，比如停车场和地下室。

SubPos Ranger 是基于 802.15.4 的 WiFi 专用 sub PS 的扩展，比 WiFi 专用 sub PS 系统提供更长的范围和更低的功耗。它仍然能够在几英尺内确定接收器的位置，这使得它成为需要知道位置而不依赖 GPS 的设备的理想解决方案。
