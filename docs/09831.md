# TMS9918A 显示芯片中还有 RC2014 生命

> 原文:[https://hack aday . com/2018/06/22/theres-RC 2014-life-in-the-the-the-TMS 9918 a-display-chip-yet/](https://hackaday.com/2018/06/22/theres-rc2014-life-in-the-tms9918a-display-chip-yet/)

20 世纪 80 年代早期家用电脑大战中的一个例外是德克萨斯仪器公司的 TI99/4A。它可能没有竞争对手的游戏库，它的 TMS9900 处理器可能没有以其 RAM 中的寄存器架构轰动世界，但它的支持芯片系列包括一种衍生产品将继续取悦后代。如果你有一台 MSX 或 8 或 16 位世嘉游戏机，TMS9918A 图形芯片提供了索尼克冒险背后的架构。

几十年后，人们对这款经典芯片仍有浓厚的兴趣。[J.B. Langston]有一台 RC2014 retrocomputer，并希望在上面播放 MSX 演示，他已经为 RC2014 总线创建了一个基于 TMS9918A 的显卡。该板的成功取决于显示如何将 9918A 与 SRAM 接口的电路，由于它映射到与其 MSX 对等产品相同的端口，因此理论上它应该与为该平台编写的 Z80 演示兼容。他已经在这个目标上取得了一些成功，这可以从我们放在 RC2014 上运行的大胆的 MSX 演示的休息时间下面的视频中得到证明。

RC2014 在逆向计算机领域获得了大量的追随者，并在这里出现了多次。[我们在 2016 年回顾了一款早期车型](https://hackaday.com/2016/09/08/review-the-rc2014-z80-computer/)。令人惊讶的是，TMS9918A 只在这里出现过一次，作为基于自制 6809 系统的[的一部分。](https://hackaday.com/2010/12/15/update-6809-computing/)
