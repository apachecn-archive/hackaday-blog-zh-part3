# 托尼的弹球向导 3D 打印全尺寸弹球机

> 原文:[https://hackaday . com/2016/11/06/Tony-the pinball-wizard-3d-prints-full-size-pinball-machine/](https://hackaday.com/2016/11/06/tony-the-pinball-wizard-3d-prints-full-sized-pinball-machine/)

[托尼]已经设计并 3D 打印了一个全尺寸的弹球机，这绝对令人难以置信。我们所说的 3D 打印是指 3D 打印！就连柱塞的弹簧也是印刷塑料的。

保险杠设计特别有意思。神奇的事情发生在两个导电细丝环上。底部的一个是固定的，而顶部的一个是带有柔性细丝的多材料印花。当球碰到缓冲器时，顶部的灯丝弯曲，下面的环接触。太棒了。谁想先把这个复制到机器人的操纵杆或碰撞传感器上？给我们发个[提示](http://hackaday.com/submit-a-tip/)！

整个文档可以作为弹球设计的入门读物。[Tony]从描述弹球从法国宫廷到现代的历史开始。然后，他从游戏风格、规则和常见元素逐步发展到他设计的基本原理。太迷人了。

然后，他的指南进入了技术细节。整机是在 OpenSCAD 中设计的。它花费了超过 8.5 公里的 80 种不同的细丝，通过 1200 多个小时的 3D 打印时间(不包括失败的打印)来完成。电子设备被手工布置在一个笔记本上，基于定制的电路板、部件和两个 Arduinos，它们处理所有的螺线管、计分和致动器。这个主题是基于一个最受欢迎的保龄球馆和其他地标。

毫无疑问，这是一项充满爱的工作，也是一项鼓舞人心的成就。休息之后，你可以观看它的运行视频。
