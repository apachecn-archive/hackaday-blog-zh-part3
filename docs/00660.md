# 电动汽车动力不足？自己做。

> 原文：<https://hackaday.com/2015/11/16/ev-motor-not-powerful-enough-make-your-own/>

我们中的许多人都尝试过旧的电动汽车改装技术。猛拉发动机，加上一个工业 DC 或交流电机，你就大功告成了。简单的东西。但是如果你找不到适合你的应用的**电机……为什么[不设计并制造你自己的无刷 DC 电机呢？](http://www.masinaelectrica.com/45kw-bushless-motor-design/)**

这个令人兴奋的建筑是由一位电气工程师设计的，他决定设计一个 45kW 的电机(60HP)。那不是错别字。**设计制造 45kW 电机！**他有一个关于设计过程的视频系列，包括 CAD 和他为使这个东西工作所做的所有计算。一旦他完成了设计并订购了定制零件，他就开始建造它。
