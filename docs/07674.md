# 如果你想在显微镜上花钱

> 原文：<https://hackaday.com/2017/11/14/if-you-want-to-spend-on-a-microscope/>

快速查看一下常用的中国网站，就会发现 USB 显微镜的价格非常低。然而，其中许多只不过是带有廉价光学元件的网络摄像头。这并不是说它们没有用，但它们可能无法与 Dino-Lite 这样昂贵的仪器竞争。[Shahriar]查看了 Dino-Lite 的[最新产品，并展示了它们在检查电子产品时如何有用。你可以看看下面的视频，但是要注意:这些小显微镜可不便宜。入门级型号的起价约为 100 美元，价格会一路上涨。](https://www.youtube.com/watch?v=larmop2B--E)

尽管如此，我们中的许多人还是在必要的设备上花费了同样多甚至更多的钱，而如今，用于检查微小电路的显微镜非常方便。除了光学仪器，[Shahriar]还看了一个步进电机驱动的显微镜载物台，这很有趣。

视频展示了检查和测量 IC 和 PCB 的实际应用，以及照明技巧和其他现实建议。我们意识到，为一台显微镜支付 1000 美元或更多可能对我们大多数人来说都是多余的，但看看这些显微镜的表现是很有趣的，如果你真的需要这种能力，或者如果你正在装修实验室或黑客空间，这并不是遥不可及的。

自然地，对于大多数用途，比如焊接，你可以很容易地用更少的钱来应付。甚至[一个普通的网络摄像头](https://hackaday.com/2012/12/30/usb-microscope-used-for-soldering-very-small-things/)也能完成这项工作。
