# 用合成传感器感知所有的东西

> 原文：<https://hackaday.com/2017/05/19/sense-all-the-things-with-a-synthetic-sensor/>

怎样才能让你的房子比你更聪明？从这些天我们在家庭中心看到的智能电器的价格来看，它需要大量的现金。但是，如果你可以让你的家变得更智能——或者至少更善于观察——用一些便宜的、通用的“超级传感器”来监视你的一举一动，会怎么样呢？

听起来很恐怖，对吧？这就是卡内基梅隆人机交互研究所的[Gierad Laput]和他的团队在设计他们的宽带“合成传感器”时的想法，也是他们故意在设计中省略摄像头的原因。但是，太阳底下几乎所有的传感器都在这个小小的板上:红外阵列、可见光传感器、磁力计、温度、湿度和压力传感器、麦克风、PIR，甚至 EMI 探测器。当然还有一个 WiFi 模块，但它似乎只是用于连接，而不是用于感应，尽管它[显然可能是](https://hackaday.com/2017/05/06/measuring-gait-speed-passively-to-diagnose-diseases/)。使用机器学习和用户培训的结合，所有的原始数据被合成为平台范围内发生的事情的总画面。

休息后的视频显示，传感器从一个中心位置检测典型的家庭事件。这是一个强有力的想法，我们期待看到它如何从原型变成产品。如果敏锐的读者认出了[Gierad]的名字，可能是因为他过去在这些 3D 打印头发的页面上出现过。
