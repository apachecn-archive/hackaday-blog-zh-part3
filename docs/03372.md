# Game Pie Advance 将复古游戏带到您的指尖

> 原文:[https://hack aday . com/2016/09/03/game-pie-advance-bring-retro-gaming-to-your-fingers/](https://hackaday.com/2016/09/03/game-pie-advance-brings-retro-gaming-to-your-fingertips/)

我们喜欢我们在 Hackaday 的 Game Boy 和 RetroPie mods，因为 Raspberry Pi Zero 使我们比以往任何时候都更容易携带一个装满经典游戏的口袋。[ Ed Mandy ]延续了这一伟大传统，将哑光黑色[游戏机进阶变成了复古掌机](http://gamepieadvance.com/)。

关于[Mandy]如何构建他的游戏 Pi Advance 的细节很少，但我们可以从博客帖子和视频中收集一些细节。一个 Raspberry Pi Zero running RetroPie 似乎是搭载在一个定制的 PCB 上，可以整齐地插入 GBA 外壳中。这使得用户可以通过墨盒插槽轻松访问 Pi Zero 的 USB 和微型 HDMI，以连接到外部屏幕，以及第二个控制器来执行一些合作 NES 和 SNES 操作。这里值得注意的是[Mandy]已经放弃在当前版本中添加 X 和 Y 按钮。

按住电源按钮直到绿灯闪烁，即可打开游戏饼前进。它由微型 USB 或两个 2500mAh 14500 锂离子充电电池供电，可持续约 1.5 小时。不幸的是，插上电源后，电池无法充电。随着游戏机的死亡，你可能会发现自己陷入了怀旧的情绪中，你不得不将电池换成一副新的。我们认为这是完成这种紧凑、复古体验的一个特征。
