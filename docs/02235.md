# 世界上最令人不安的无人机的竞争者？

> 原文：<https://hackaday.com/2016/05/02/contender-for-worlds-most-unsettling-drone/>

我们不知道 FESTO 用他们奇怪的飞行沙滩球做什么广告。在鼓舞人心的音乐中，它轻轻地将半透明的附肢放在水瓶上，然后用令人不安的塑料声音将其吞没。随着一声尖锐的哀鸣，它带着猎物盘旋而去，并把它放在一个口渴的商人手里，也许这是一种被误导的养育本能。

尽管发现了一个新的[恐怖谷](https://www.youtube.com/watch?v=m7l-87r4oOY)，这个机器人还是很酷。这似乎是一个小规模的混合飞艇/直升机。气球要么消除机器人的重量，要么提供稍微多一点的升力。剩下的就靠螺旋桨了。

我们喜欢无人机周围的碳纤维桁架。这是一个非常光滑的建筑，几乎没有未驯服的电线。对于室内飞行来说，这似乎比四轴飞行器更安全。如果它的末端效应器不是如此令人毛骨悚然，它会更酷。休息后的视频。
