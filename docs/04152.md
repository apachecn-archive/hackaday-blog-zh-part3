# 用这个 HAL 9000 版本给 Alexa 一个诚实的面孔

> 原文：<https://hackaday.com/2016/11/15/put-an-honest-face-on-alexa-with-this-hal-9000-build/>

亚马逊推出了 Alexa 软件的一个版本，可以在 Raspberry Pi 上运行。阿达果卖了一个大的吓人的红色按钮。对于[Keith Elliott] [来说，未来的项目是一个显而易见的结论](https://keithwelliott.com/projects/hal-9000-replica/)。

Alexa 软件的 Raspberry Pi 版本落后于发布版本。你必须按下一个按钮来输入一个命令，这真的从一个令人毛骨悚然的声控机器人那里偷走了很多快乐，听着你在房子里乱转。现在，它可以根据命令醒来。

由于这说服了他最终将亚马逊的《守望女巫之眼》添加到他的家中，他决定给他穿上合适的邪恶衣服。这些是根据 Adafruit 的指南文件 3D 打印出来的。他最终有了一个相当令人信服的门面。

内心有点忧郁。一个孤独的树莓 Pi 3 由麦克风和音频放大器陪伴。这些都是让你的家庭自动化购物体验梦想成真所需要的。休息后的视频。
