# 制作金属骨牌

> 原文:[https://hackaday.com/2017/04/11/making-metal-dominoes/](https://hackaday.com/2017/04/11/making-metal-dominoes/)

多米诺骨牌几乎就像一副扑克牌一样多才多艺，是任何未雨绸缪的游戏集的绝佳补充。Youtube 频道[carbide3d]的[Apollo]为自己制作了一套定制的多米诺骨牌，上面布满了黄铜小点。

将棒料切割成合适的尺寸，[阿波罗]为黄铜尖头做了一些试刻和孔的尺寸。完成后，他们所要做的就是重复雕刻和铣削过程另外几十次，以及所有必要的干湿砂光和抛光。[阿波罗]选择使用油漆标记为瓷砖添加一点额外的风格，并建议任何其他想要这样做的制造商将雕刻深度设置为 0.01 英寸，这样油漆标记就不会在抛光时被擦掉。

当安装黄铜球时，[阿波罗]将孔缩小了. 001 英寸-.002 英寸，以实现紧密的压配合，并补充说孔的深度略大于球直径的一半。他们用 1/8″的球做小点，用 3/16 的球做瓷砖的中心，这也使得瓷砖可以在玩的时候旋转，带来一点坐立不安的乐趣。休息之后，请观看构建视频。
