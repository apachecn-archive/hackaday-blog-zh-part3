# Hackaday 奖参赛作品:水位站

> 原文：<https://hackaday.com/2017/06/03/hackaday-prize-entry-water-level-station/>

在全世界，特别是在不发达国家，每年都有成千上万的人因为洪水而死亡。水位的突然上升通常是突然发生的，人们在陷入困境之前没有时间做出反应。现代国家通常在有问题的区域周围部署测量设备，但是这些设备对于第三世界国家来说通常是昂贵的。

[Benne]项目设计了一个[低成本、云连接的水位测量站](https://hackaday.io/project/21579-affordable-water-level-measuring-station)，允许地方当局进行远程和中央水位监测。他希望，通过能够更精确和及时地监测水位，当局可以更快地采取行动，警告可能受影响的地区，并增加在自然灾害情况下拯救生命的机会。

目前，该项目仍处于早期阶段，因为他们正在用不同的传感器进行测试，以找出哪种传感器在不同的场景下工作得最好。最新版本主要由 Arduino UNO、超声波距离传感器和 DHT 温度/湿度传感器组成，以提供校准，因为这些特性会影响声速。几年前，我们报道了使用视差 Ping 传感器进行简单的[水位监控，但当时物联网和“云”还不太流行。他们还测试了红外传感器和旋转编码器。](http://hackaday.com/2011/08/30/monitoring-water-levels-with-a-parallax-ping-sensor/)
