# 硬盘拆卸既简单又划算

> 原文：<https://hackaday.com/2016/02/03/hard-drive-disassembly-is-easy-and-rewarding/>

有没有坏掉的硬盘？Hackaday 明矾[Jeremy Cook] [展示了拆卸一个硬盘驱动器](http://www.jcopro.net/2016/01/26/how-to-disassemble-a-hard-drive/)来清理它的好东西是多么容易。最困难的部分是要有耐心和工具去克服那些阻碍你和内心宝藏的螺丝钉。

外壳螺钉通常是梅花型的。任何有自尊的黑客可能都已经有一两个这样的东西了，但是如果你在市场上，[Jeremy] [推荐一套漂亮的](http://www.jcopro.net/2012/01/09/the-wiha-75990-screwdriver-set-review/)，看起来比我们的好得多。一旦箱子打开，你可以找到稀土磁铁，轴承，和一个或多个盘片。

那些异常强大的磁铁对各种项目都有好处。把它们粘在一块有吸引力的木头的背面，安装在厨房的墙上，你就有了自己的刀块。在工作台上放几个来临时磁化工具。使用它们来构建拾音器，以放大雪茄盒吉他或拇指钢琴。或者将拾音器放入一个小的扩音扬声器中，像听诊器一样靠近你的电子产品挥动，听它们发出嗡嗡声。至于释放磁铁，杰里米用老虎钳夹住它，用锤子和凿子把它从致动器硬件上撬下来。

毫无疑问，你已经看到过由旧硬盘制成的时钟，它们大部分都完好无损。包括[杰里米]在内的许多制造商将提取闪亮的盘片作为钟面的基础，并雕刻数字，蚀刻它们，或粘在上面。那些大盘子也是极好的编钟。即使你只是把一个盘子挂在手指上，用指甲轻轻敲打，听起来也很不错。

如果简单的编钟不能真正给你的松饼涂上黄油，那么有各种各样的针对死硬盘的声波项目。把[做成麦克风](http://hackaday.com/2015/07/27/make-a-microphone-out-of-a-hard-drive/)或者[音箱](http://hackaday.com/2015/03/03/hard-drive-speakers/)怎么样？也许一个[硬盘 MIDI 控制器](http://hackaday.com/2015/03/02/the-hard-drive-midi-controller/)或者一个[合成器](http://hackaday.com/2012/01/27/synthesize-with-a-hard-drive/)更适合你。说到合成器，休息之后，看[Jeremy]打开硬盘听一些甜美的声音。
