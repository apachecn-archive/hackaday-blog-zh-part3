# 无接触 MIDI:秘密在手套里

> 原文:[https://hack aday . com/2016/11/06/touchless-midi-the-secrets-in-the-mitten/](https://hackaday.com/2016/11/06/touchless-midi-the-secrets-in-the-mitten/)

MIDI 对几乎所有音乐家来说都是一个伟大的工具。不过，除非你是键盘手，否则很难现场使用。[Evan]最近想出了一个很好的解决方案，适合所有渴望拥有一个和他们的钢琴家兄弟一样有用的 MIDI 界面的吉他演奏者。他创造了一个[无触摸 MIDI 控制器](http://www.allaboutcircuits.com/projects/touchless-midi-controller-with-arduino/)，可以直接与吉他接口。

[Evan]设置一个 Arduino Nano 来处理计算机的 MIDI 接口。以前项目中的检测线圈安装在吉他上，可以识别吉他手的手离吉他主体有多远，让音乐家可以控制他们选择的效果。吉他手只需要戴上一个特殊的连指手套来配合检测线圈使用。[Evan]还增加了三个触觉按钮，这意味着 MIDI 的实用性可以扩展到三种不同的可选效果。

请务必观看下面的视频，观看该界面如何工作的演示。[埃文]还提供了[原理图](http://i.imgur.com/YuPYriA.png)和 [Arduino 代码](https://github.com/evankale/ArduinoTouchlessMIDI)，如果你决定建造自己的话。这也不是[埃文]的第一次迷笛表演。他还从摇滚乐队的鼓组中创建了一个 [MIDI 鼓界面。](http://hackaday.com/2015/07/18/forgotten-rock-band-drum-controller-as-a-midi-instrument/)
