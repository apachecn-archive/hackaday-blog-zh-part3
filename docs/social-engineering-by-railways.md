# 铁路社会工程

> 原文：<https://hackaday.com/2018/06/07/social-engineering-by-railways/>

你每天去哪里旅行？有什么微妙的策略来操纵你的行为吗？你能认出他们吗？还是他们只是这个地方的一部分？社会工程有时会受到不好的指责(或者说它有代表性吗？)在主流中，但这把剑面向公众的刀刃可以[维持秩序，就像它在日本火车站](http://www.citylab.com/transportation/2018/05/the-amazing-psychology-of-japanese-train-stations/560822/)所做的那样。他们采用了一系列心理学方法，让电视台像钟表一样运转。

策略的范围从恶魔般地放置发出高频音调的扬声器以阻止年轻人游荡到体贴地安装蓝光以阻止自杀。并不是每种策略都像预防自杀一样明智，有时，只是将每个车站刺耳的发车蜂鸣器换成独特的曲调，就能大大缓解焦虑。谁愿意站在一个焦虑的、汗流浃背的旅行者旁边？在休息时间下面听听东京地铁的曲调是什么样的。

也许你可以在你住的地方发现这些小技巧，或者类似的东西可以缓解你自己的通勤。也许最近的[地铁有一架钢琴做楼梯](https://hackaday.com/2009/10/07/the-subway-never-sounded-so-good/)或者一个 [3D 打印机器人](https://hackaday.com/2017/10/04/3d-printing-on-the-subway-or-anywhere-else/)。
