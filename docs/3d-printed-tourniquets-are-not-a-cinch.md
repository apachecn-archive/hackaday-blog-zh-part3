# 3D 打印止血带并不容易

> 原文：<https://hackaday.com/2018/06/28/3d-printed-tourniquets-are-not-a-cinch/>

说某事很容易是说它很容易的一种方式。制作一个中间有孔的细手柄似乎是一个简单的任务，一个下午就能完成，这还包括打印一两份拷贝的时间。我们在这里告诉你，这只是为枪击受害者制作止血带的第一项任务。内容警告:有严重外伤的真实图片。下面是加沙 Hayat 中心的止血带训练视频，在人体模型上有一个模拟伤口。

在第一遍中，许多事情都做对了:手柄的长度和直径正确，皮带孔与皮带匹配，零件在压板上[定位良好。与许多第一次迭代一样，它在屏幕上看起来很好，但在现实世界中，我们都生活在墨菲定律下。在实践中，一些表带孔有锋利的边缘切入表带，其中一个印刷带扣意外断裂。](https://hackaday.com/2017/02/26/3d-printed-vise-is-a-mechanical-marvel/)

总的来说，[开源](https://hackaday.com/2018/01/30/making-the-case-for-open-source-medical-devices/)止血带的低成本和可用性超过了不使用止血带的危险。开源医疗设备不仅仅是在现场使用，它们也可以[帮助培训。这个止血带拯救了人们，并且证明了](https://hackaday.com/2017/02/16/rfid-stethoscope-wheezes-and-murmurs-for-medical-training/)[的建模技能](https://hackaday.com/2018/03/27/how-to-reverse-engineer-mechanical-designs-for-3d-modeling/)在现实世界中可以有很大的帮助。
