# 用 Arduino 喂狗

> 原文：<https://hackaday.com/2018/02/27/definitive-dog-feeding-with-arduino/>

有些狗没有自我保护意识。如果有机会，它们会吃到生病。我们人类既要喂养它们，也要记得这么做，这样它们才不会意外地被过量喂养。在一个有小孩的忙碌家庭中，棘手的部分是记忆。

[Bryan]的家人每天早上给他们的狗 Chloe 喂一次。她是一个花了几年时间在街上乞讨食物的救助者，所以她的某个部分总是对寻找食物感兴趣，即使她刚刚吃过。每天早上，整个房子里的忙乱活动都因克洛伊反复索要食物而加剧，所以[【布莱恩】让他的孩子参与进来，建立了一个简单的电路，让每个人一眼就知道——克洛伊是否被喂了](https://www.fractuslearning.com/2018/02/22/dog-food-ardunio-projects-kids-1/)。

克洛伊的食物被保存在一个触摸顶部的废纸篓里，按下按钮就可以打开。[Bryan]的狗食检测器使用簧片开关和 Arduino 克隆来检测盖子何时被打开。当簧片开关变低时，Arduino 会点亮一个 LED。灯亮了两个小时，然后自动关闭，为第二天做准备。你不必乞求演示视频，因为它在休息后等着你。

由于克洛伊在大约两分钟内吞下一碗食物，也许[布莱恩]家庭的下一个项目可以[教她慢一点](https://hackaday.com/2014/09/24/the-thurber-feeder-5000-helps-to-slow-fido-down/)。
