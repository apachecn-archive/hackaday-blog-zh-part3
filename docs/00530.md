# 带 Arduino 的磁悬浮

> 原文：<https://hackaday.com/2015/11/02/magnetic-levitation-with-arduino/>

让一个磁场在另一个磁场上平衡就像让一个保龄球在墨水笔尖上平衡一样简单。然而，在 Arduino mega 的一点帮助下，[EmmaSong]能够在半空中平衡高密度钕磁铁。他用从[淘宝](http://world.taobao.com/)(中国版的易贝)得到的一组四个线圈、一个霍尔效应传感器和一些电流调节集成电路完成了这个棘手的项目。

如果需要，线圈可以在室内制造，每个绕组大约有 800 匝漆包线。电路的其余部分很简单。他似乎使用了一个电位计来粗略调节流向线圈的电流，在代码中进行微调，可以在这里找到(。RAR 直接下载)。

在之前，我们已经在这里看到过[磁悬浮，这个项目增加了完成这个困难项目的成功技术清单。](https://hackaday.com/2011/07/23/frustrating-fun-with-magnetic-levitation/)
