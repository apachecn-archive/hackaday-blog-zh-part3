# Asgard:开源大气数据计算机

> 原文：<https://hackaday.com/2018/01/25/asgard-the-open-source-air-data-computer/>

我们通过举报热线收到了很多很棒的项目。嗯，主要是我们似乎收到了垃圾邮件，但那些不是试图向我们推销东西的电子邮件总是很棒。尽管如此，我们并不经常在它的功能列表中看到包含“确定马赫数”这个神奇短语的提示。所以说我们对[阿斯加德大气数据计算机(ADC)](https://www.basicairdata.eu/projects/airdatacomputer/asgard-adc/) 感兴趣是一种保守的说法。

现在我们必须承认:我们不能 100%确定 Asgard 的目标受众是谁，但是它看起来确实令人印象深刻。团队成员[Erik]给 tip line 写了关于这个令人印象深刻的项目的信息，该项目能够对进入的空气进行许多测量，如真实速度、粘度和温度。该团队表示，它的应用范围从暖通空调到测量自行车的性能。我们不知道谁骑自行车骑得如此之快，以至于他们需要测量空气速度，但当然黑客社区从未停止让我们惊讶。

即使你没有可以从 Asgard 等高性能 ADC 中受益的喷气式战斗机，你也必须对该团队所做的不可思议的工作印象深刻，不仅是设计和制造，还有记录。从设计完美的 3D 打印外壳到堆叠的 PCB 内部，Asgard 的每个方面都是专业硬件。

如果设备要在独立模式下使用，从 Asgard 收集的数据可以存储在内部 micro SD 上，或者借助 HC-05 模块，您可以通过 USB 或蓝牙连接到它。该团队甚至已经将一些脚本整合到一起，将 Asgard 生成的大气数据与 GPS 位置信息合并在一起。

我们都支持把高质量的传感器放到社区的手中，看看他们能想出什么。这个项目的精神和建造质量让我们想起了[【拉杜·莫蒂桑】用他的分布式空气质量传感器](https://hackaday.com/2017/07/16/monitor-your-citys-air-quality/)所做的令人印象深刻的工作。
