# 用塑料碎片劈出一把挂锁钥匙

> 原文:[https://hack aday . com/2015/11/16/hack-a-padlock-key-from-plastic-scraps/](https://hackaday.com/2015/11/16/hack-a-padlock-key-from-plastic-scraps/)

就在几年前，如果你想要一把像样的钥匙，你必须去锁匠店或者最近的五金店，在那里真正的人知道他们的交易。现在，我们通常把钥匙拿到橙色的大盒子里，用半自动机器复制，甚至把它放入越来越多的全自动钥匙复制亭，结果各不相同。但是正如[BlueMacGyver] [向我们展示的那样](https://www.youtube.com/watch?v=jlW8EQ4vxnE)，一把可用的挂锁钥匙可以在家里用除了碎片之外的任何东西很快被制作出来。

下面的视频详细介绍了这一过程——用打火机将钥匙的轮廓熏黑，用透明胶带将碳转移到一些硬塑料上，然后切割出轮廓。只需稍加欺骗，平板电脑就能轻松地进入锁中并打开它。看起来这种方法可以应用于挂锁以外的锁。至于原材料，我们认为我们已经为桌子抽屉里收集的过期信用卡找到了用途。

我们最近对涉及锁的黑客攻击进行了大量报道，包括[从照片中复制钥匙](http://hackaday.com/2015/05/23/pictures-that-defeat-key-locks/)和[用 3D 打印机制作凹凸钥匙](http://hackaday.com/2014/08/29/3d-printed-bump-keys/)。但是我们喜欢这种简单的方式。没错，你需要物理接触密钥才能复制它，这限制了黑客的邪恶可能性。但也许这并不是一件坏事。
