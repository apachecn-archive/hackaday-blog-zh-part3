# 这台无马达拉式割草机是由废品制成的

> 原文：<https://hackaday.com/2016/07/15/this-motorless-pull-behind-mower-is-made-from-junk/>

如果有合适的动力推进器，割草是一个简单而令人满意的过程。一台带有旋转顶部悬挂在三点连接和动力输出装置上的拖拉机将很快完成这项任务。

[Donn DIY]有一辆农用四轮自行车，但在割草时，它没有动力输出装置，这意味着它没有多大用处。当他在网上看到一个自制的四轮自行车割草机时，他不得不亲自尝试一下，并想出了自己用垃圾制成的割草机。

他从一辆俄罗斯造[拉达](https://en.wikipedia.org/wiki/VAZ-2101)的后轮轴和差速器开始，在安装到一个输入轴朝上的木制夹具之前，他对这辆车进行了翻新。然后，他为三个割草机轴做了一个框架，在框架上安装了他定制的转子和他们加工的轴承箱。一些滑轮加工，然后他可以用一系列 V 带和另一根轴将转子轴连接到差速器上，以提高转子速度。

他还没有完成，在转子出现后，一个杠杆机构将刀具抬离地面，一对重量篮确保牵引力保持不变。其结果是一个从轮子上获得驱动力的推进器，当被拖在四轮自行车后面时，可以非常有效地割草。没有防护的刀片可能会让一个农业保险评估员中风，但对于下面的视频来说，至少我们可以看到一切。
