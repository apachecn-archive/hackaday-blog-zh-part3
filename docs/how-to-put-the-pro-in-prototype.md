# 如何将“Pro”放在原型中

> 原文：<https://hackaday.com/2018/04/29/how-to-put-the-pro-in-prototype/>

如果您采取正确的步骤，很容易在您的印刷品和原型上获得专业质量的效果。在他的 Bondo 建筑系列的最后一部分中，产品设计师[Eric Strebel]向我们展示了如何在任何基底下建造。

他是怎么得到这么光滑的表面的？几个关键步骤就能改变一切。首先，他总是使用某种类型的砂块，即使他只是在压舌板周围裹上砂纸。例如，他的手机支架两边都有一个圆形凹痕。我们喜欢[Eric]用更多的 Bondo 和一块 PVC 制作了一个定制的磨砂板(你猜对了)。另一个关键是以集中、扫动的方式喷涂底漆和油漆的薄层，以使涂层堆积起来。

如果你需要一个能和婴儿屁股媲美的表面，不要指望涂一次底漆，刷一次漆，就完事了。你必须寻找并摧毁所有的不完美。[Eric]喜欢用腻子抹平，然后在涂更多底漆之前用湿砂纸打磨光滑。然后冲洗，用更高的粒度重复，直到满意为止。

当然，平滑指纹的方法不止一种。就在几周前，[我们自己的【唐纳德·帕普】深入研究了 UV 树脂的使用](https://hackaday.com/2018/03/08/3d-printering-print-smoothing-tests-with-uv-resin/)。
