# 无线充电器卡车模块保持果汁流动

> 原文：<https://hackaday.com/2017/12/31/wireless-charger-truck-mod-keeps-juice-flowing-on-the-move/>

无线充电是一项伟大的技术，但它的相对新颖性意味着它可能不会出现在你想要的任何地方。当其中一个地方是你的车时，你就像[Braxen McConnell]一样，打开它安装一个无线充电器!

在拆除中控台后，[麦康奈尔]不得不在幕后进行一些削减，为无线充电器腾出空间——以及削减充电器本身。他还把充电器拆开，把电路板和充电线圈翻过来放在盒子里；原因是线圈离手机越近越好。充电器已经隐藏在中控台的塑料后面，所以通过充电器内部的额外距离战斗是没有好处的。充电器是用双面胶带安装的，因为它相对较轻，不会被碰撞。

[McConnell]接入了他卡车上的附属电路，所以它只会在卡车启动时汲取电流——没有人喜欢回到一个没电的电池！电力来自连接到 USB 汽车充电器的香烟插座，然后为无线充电器供电——这有点粗糙，但很有效！插入无线充电器并重新安装中控台后，[McConnell]就设置好了！休息之后，请观看构建视频。
