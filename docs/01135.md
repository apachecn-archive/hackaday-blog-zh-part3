# 用铅球和发光二极管提高音量

> 原文:[https://hack aday . com/2016/01/06/pump-up-the-volume with-lead-shot-and-LEDs/](https://hackaday.com/2016/01/06/pump-up-the-volume-with-lead-shot-and-leds/)

许多现代廉价键盘的可取之处之一是内置音量控制按钮。但是这已经过时了，我们中的许多人(还有你)都有 Ms 型或者新式的机械键盘，它们只有基本的按键。那些多媒体按钮只能调节系统音量。我们敢打赌，我们的许多读者都有甜美的音响系统，但不得不起来改变音量。那么，有什么解决办法呢？[像【Markus】做的那样](http://dangerousprototypes.com/forum/viewtopic.php?f=56&t=7654)做一个变色遥控 USB 音量旋钮。

很像启发他的 Instructable，[Markus]使用 Digispark 板和旋转编码器。这种颜色来自一个 WS2812 LED 环，它非常适合放在一个曾经盛过某种奶油的乳白色塑料盆中。当调节音量时，铃声会在每次增加时闪烁白色，然后慢慢恢复到设定的颜色。按下按钮，音量静音。

这些组件非常轻，而且[Markus]不想让它在桌子上到处滑动。他在这里采用了一种有趣的方法，用猎枪子弹的铅和一些强力胶填充底部。按钮的旋转部分也需要一些重量，所以他增加了几个垫圈来获得满意的感觉。请务必在休息后观看演示。

Digispark 板对你来说不够金属？这里有一个音量旋钮，围绕着一个裸露的 ATtiny85 (反正是一回事)。
