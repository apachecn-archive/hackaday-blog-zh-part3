# 超低科技马里奥

> 原文:[https://hackaday.com/2017/11/16/super-low-tech-mario/](https://hackaday.com/2017/11/16/super-low-tech-mario/)

在互联网的深处浏览时，我们遇到了来自[Sata Productions]的超级马里奥的[超低科技版本。该视频介绍了如何制作著名的超级马里奥游戏的可玩纸板版本的完整教程。如果你是一个球迷，你可能会喜欢这个。](https://www.youtube.com/watch?v=n6gXzqvnc_s)

你基本上需要硬纸板、一把热胶枪、一个滚珠轴承、几台 DC 汽车、一些铁 BBs、一些磁铁、电池、一些电线……这听起来就像《麦盖弗》剧集的购物清单。但它是可行的，而且是可玩的。它有一个有线遥控器，你可以控制马里奥移动，让他在一种转动仪表板的游戏中跳上跳下。它甚至有一个游戏，当马里奥死的时候。是的，马里奥可以死在这个纸板版里。如果你想定制一个版本，你可以打印一个更大的尺寸，并调整纸板盒的大小。

《超级马里奥》也有它的黑客手段，比如这个界面黑客用一个 [Kinect 来控制马里奥](https://hackaday.com/2014/10/25/using-kinect-to-play-super-mario-bros-3-on-nes-ensures-quick-death/)或者这个[超级棘手的越狱黑客](https://hackaday.com/2017/06/03/super-mario-world-jailbreak-requires-no-external-hardware/)，它允许玩家运行他们自己的游戏模块，但是这个只是在另一个层面上:一个低技术的方法。这似乎是一个有趣的周末项目，尤其是如果你有孩子的话。如果你不喜欢超级马里奥，可以打印一个不同的游戏，支持平台非常通用，可以支持几个简化的平台游戏。
