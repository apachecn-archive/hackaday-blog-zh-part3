# 用流化床进行粉末涂覆

> 原文：<https://hackaday.com/2017/12/25/powder-coating-with-a-fluidized-bed/>

高品质粉末涂层零件的美观和耐用性无与伦比。有一些关于外观和感觉的东西远远超出了单纯的绘画，使它值得付出努力和费用。然而，典型的静电喷涂粉末涂覆装置可能很昂贵，而且不一定适合每一个工件。

进入[流化床粉末涂层室](https://hackaday.io/project/28541-fluidized-bed-for-powder-coating)，这是小零件有限运行的完美选择，也是【Andrew Mayhall】的创意。在一家提供铁管家具套件的企业中，[Andrew]需要一种方法来完成法兰和配件，而粉末涂料提供了最好的外观。他制造的流化器是喷涂的绝佳替代品；它将空气吹过一层细小的热塑性颗粒，使它们表现得像流体一样。它类似于我们最近推出的[流化床热水浴缸](http://hackaday.com/2017/12/08/fill-your-hot-tub-with-sand-for-science/)，但规模要小得多，并且根据粉末的超细颗粒尺寸和聚集特性有不同的要求。[Andrew]必须添加机械搅拌来实现均匀的流化床，经过大量实验后，他现在能够将预热的零件浸入流化床中，实现一步粉末涂覆。休息后的视频展示了一些操作细节。

静电粉末喷涂听起来更像你的东西吗？没问题——[DIY 解决方案](https://hackaday.com/2015/10/25/diy-powder-coating/)比比皆是，一个自制烤箱烘烤你的零件可能就在最近的文件柜的[附近。](https://hackaday.com/2014/09/09/diy-powder-coating-oven-gets-things-cooking/)
