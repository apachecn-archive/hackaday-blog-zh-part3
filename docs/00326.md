# USB 黑仔，版本 2.0

> 原文:[https://hackaday.com/2015/10/10/the-usb-killer-version-2-0/](https://hackaday.com/2015/10/10/the-usb-killer-version-2-0/)

你可以用电脑上的端口做很多傻事。最好的例子是 Etherkiller，一种直接连接到电源电缆的 RJ45 插头。不要把它插到路由器上。USB 有点棘手，但只要有足够多的上限，[任何人都可以创建一个 USB 杀手](http://habrahabr.ru/post/268421/)，它会烧坏任何计算机。茹，[谷歌翻译公司](https://translate.google.com/translate?sl=ru&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fhabrahabr.ru%2Fpost%2F268421%2F)

USB 黑仔 v2.0 是[深紫色]的第二个版本的设备。第一个版本只是一个带有 DC/DC 转换器、几个电容和一个 FET 的小电路板。当连接到电脑时，转换器会将电容充电到-110 伏，将该电压注入 USB 信号线，并重复整个过程，直到电脑死机。第二个版本稍微好一点，它现在可以将-220V 直接转储到 USB 信号线。不要在家里尝试这个。

那么，这个装置能用吗？绝对是。在下面的视频中，出于演示的目的，一台劣质的 Thinkpad X60 被 USB 杀手销毁了。这台笔记本电脑最初只是为了测试而购买的，但创造 USB 杀手的怪物却越来越喜欢这台小巧的笔记本电脑。新的主板即将上市，这款笔记本电脑将重获新生。
