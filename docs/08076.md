# 小功率马达可以在一个硬币电池上运行数年

> 原文：<https://hackaday.com/2017/12/25/low-power-motor-can-run-for-years-on-a-coin-cell/>

你能用一个锂硬币电池驱动一个电动机两年吗？[IamWe]想出了如何做到这一点，尽管他的[环形电机](https://www.instructables.com/id/Donut-Motor/)看起来不像我们以前见过的任何电机，但这是低电流设计中非常坚实的一课。

环形电机实际上只是一个无刷 DC 电机，带有一个单极定子和一个多极转子。马达的框架是由聚苯乙烯泡沫甜甜圈制成的，因此得名。转子是一个聚苯乙烯泡沫球体，其赤道周围嵌有钕磁铁。锋利的自行车辐条充当车轴，巧妙的磁性轴承提供近乎零摩擦的旋转。定子线圈来自一个旧的螺线管，由一个非常简单的双晶体管振荡器驱动。[IamWe]的计算表明，单个 CR2032 硬币电池应该可以为该电机供电两年以上。这个看起来很容易制作，对于漫长的冬夜来说可能是一个不错的项目。在下面的视频中观看它的旋转。

这似乎是硬币电池挑战赛的完美参赛作品。当然，它可能不是[你的汽车的硬币电池跳启动器](https://hackaday.com/2017/12/08/coin-cell-challenge-jump-starting-a-car/)，但我们猜测这种电机在 2020 年仍将旋转，这不是一个简单的壮举。
