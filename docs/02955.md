# 您的 ESP8266 需要更多内存

> 原文：<https://hackaday.com/2016/07/16/your-esp8266-needs-more-memory/>

我们刚刚回顾了 ESP8266 上的 MicroPython，其中一个主要观点是我们的 ESP 模块需要更多的闪存。您可能处于同一条船上—市场上最早的(也是最便宜的)模块只有 512 kB 的闪存。对于无线编程，或者为了给你更多的空间播放更好的节目，你需要 1 MB 甚至 4 MB。

解决办法？只需[买一个新的闪存芯片，焊在](http://tech.scargill.net/32mb-esp01/)上。如果您有暴露闪存芯片的 ESP-01、ESP-03 或 ESP-11 模块，这尤其容易。拆焊器，拆焊器，完成。对于那些在芯片周围有一个锡罐的模块来说，这可能有点棘手，但这不是一点热空气不能解决的问题。请参见下面嵌入的视频，以进行一次很好的演练。

订购零件时，不要混淆 4 Mb(兆字节= 512 kB)和 4 MB(兆字节= 32 Mb)。大多数闪存制造商会指定位数，因此您想要的部件的名称中会有 32(代表 Mb)。

闪存的价格甚至比塞满你的垃圾抽屉的 ESP-01 还要便宜，试一试你不会有什么损失。
