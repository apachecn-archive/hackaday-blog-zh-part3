# 乐高火车探索一个闪闪发光的世界

> 原文：<https://hackaday.com/2017/05/17/lego-train-explores-a-world-of-sparkling-light/>

[bananenbuurman]将他的工作室公寓改造成了一个辉煌的四分钟[乐高火车课程](https://www.youtube.com/watch?v=h5ltIXyaLno)，配有灯光、机动效果和创意展示。

火车车厢配有一个 360 度摄像头，让我们可以看到整个过程的缩影:一系列主题“房间”——一个房间贴着看起来像是欧元的纸币，而其他房间则有笔记本电脑、电源、主板和其他旧硬件。当火车经过各种乐高规模的元素时，你会想起它的小尺寸，如迷你人物，仿佛它们有六英尺高。

到处都有灯，从各种设备的 LED 指示灯，到节日灯和令人印象深刻的新奇照明。就细节而言，它几乎就像一个 Katamari Damacy 级别——由软盘驱动器制成的门是杀手。

你可以在[香蕉邻居](http://www.banananeighbor.com/)看到更多【bananenbuurman】的项目。
