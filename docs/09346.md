# 用一罐橡皮泥制作 3D 模型儿童游戏

> 原文：<https://hackaday.com/2018/05/04/make-3d-modeling-childs-play-with-a-can-of-play-doh/>

你需要在 3d 打印机上复制一个小零件，所以你开始准备工具。卡尺、尺子和一个草图板，如果你特别喜欢，也许你可以拿出一个 3D 扫描仪来让工作变得非常简单。但是你也会去搜查你孩子藏的玩具吗？

如果你想跟随[Vladimir Mariano]的领导和[使用 Play-Doh 在要复制的零件中精确地模拟表面特征](https://www.youtube.com/watch?v=GXFg4h_NXjQ)，你可以这样做。Play-Doh 是一种造型化合物，孩子们和过时的孩子们都喜欢玩，尤其是在它变干或与其他颜色混合或被狗毛粘住之前。

对[弗拉基米尔]来说，这种柔软光滑的东西是测量他被要求复制的圆柱体表面小草皮间距问题的完美解决方案。他不是直接在曲面上测量特征，而是简单地在一团扁平的橡皮泥上滚动。goop 拾取了草皮上的压痕，然后很容易测量并转移到 Fusion 360。下面的视频预先显示了 Play-Doh 技巧，但请在整个过程中保持关注，以获得一些关于使用钣金工具包装和打开圆柱体的伟大提示，以及学习如何在 Fusion 360 中导入图像和重新校准图像。

遇到一个 Play-Doh 解决不了的建模问题？放松，我们已经为你准备好了基本的概要。
