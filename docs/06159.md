# DIY 抽水机遵守物理定律

> 原文：<https://hackaday.com/2017/06/07/diy-ram-pump-obeys-the-laws-of-physics/>

尽管下面的视频标题宣称“免费能源”,但这并不是又一个古怪的永动机故事。我们在 Hackaday 完全支持热力学定律，我们认为你也应该这样做。但是你必须承认，一个没有任何明显能量输入的泵乍看起来有点可疑。

正在讨论的设备是一种柱塞泵，这种技术可以追溯到 18 世纪。[垃圾场——创意的起源]建造的这个版本使用了普通可用的材料，如 PVC 管道和配件。BOM 上唯一可能很难找到的东西是黄铜止回阀，它可能是瓣阀，而不是更容易找到的弹簧阀。唯一定制的部分是一个适配器，用于将用作气室的塑料汽水瓶连接到 PVC 上，如果你选择不像[垃圾场]那样砍瓶盖，3D 打印机可以解决这个问题。下面的视频展示了通过挖掘来流的动能而获得的令人印象深刻的升力。

在那里，热力学第二定律保持不变。但是如果你仍然认为你可以不劳而获，看看我们对永动机和过度团结骗术的综述。
