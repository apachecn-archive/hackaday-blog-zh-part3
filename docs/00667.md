# 抛弃 IPad 构建 Commodore 64 平板电脑

> 原文:[https://hack aday . com/2015/11/16/ditch-the-ipad-build-a-commodore-64-table t/](https://hackaday.com/2015/11/16/ditch-the-ipad-build-a-commodore-64-tablet/)

经典的 Commodore 64 已经在操作系统部门实现了现代化。从它的根启动一个基本的提示，到全高分辨率的 GUI 包，如 GEOS，到今天[Jim_64]添加了一个类似[平板电脑的启动器，配有一个触摸屏界面](http://64jim64.blogspot.com/2015/09/cos-has-been-released-for-commodore-64.html)。

GUI 本身利用了 C-64 的高分辨率图形，看起来类似于 iOS，图标是通过光标键或操纵杆选择的(什么？没有光笔？)并启动它们所代表的各种功能。为了增加操作系统的平板感觉，从 digikey 获得了现成的 3m 触摸屏面板及其相应的 RS232 接口板。

面板牢固地连接到监视器上，来自各种手指戳的 XY 数据以 1200 bps 的速度串行发送，程序解释位置。使用可用的演示(下载 [sideA](https://drive.google.com/file/d/0B65J_0YDdJyITUU0QkxlMmJHeTQ/view) 和 [sideB](https://drive.google.com/file/d/0B65J_0YDdJyIZEx6QlJDc0JTb00/view) )和现成的零件，这对于任何拥有经典 C-64 的人来说都应该很容易在自己的家里安装并享受一些乐趣。
