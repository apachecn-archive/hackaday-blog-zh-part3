# Arduino 和编码器形成切割和钻孔的精密夹具

> 原文：<https://hackaday.com/2017/06/04/arduino-and-encoder-form-precision-jig-for-cutting-and-drilling/>

“测量两次，切割一次”在制造的每个方面都是很好的建议，但也许没有什么地方比制造数控机床更重要了。当精度成为游戏的名称时，您需要能够给出可重复结果的测量工具，并且最好不要花费太多。这就是[这款基于 Arduino 的测量夹具](http://homofaciens.de/technics-physical-computing-digital-ruler_en.htm)背后的理念，用于为 CNC 制造零件。

谈到廉价建筑，没有人能与[同性建筑]相比。我们已经看到他的[垃圾桶 CNC 构建](http://hackaday.com/2015/11/21/garbage-can-cnc-machine-build/)和[编码器来自电子垃圾和锡罐](http://hackaday.com/2016/01/08/video-gives-you-the-basics-of-diy-rotary-encoders/)，所有这些都给出了令人惊讶的好结果，尽管加入了碎料板和水管工捆扎的废料等合规材料。为了制造更坚固的机器，他发现自己需要长度一致且精确的零件，所以他制造了这个夹具。碎料板滑板和角铝栅栏定位方形管料，带有纸编码器轮的辊在弹簧压力下压在管上。通过计算来自光学传感器的脉冲，他能够在夹具中精确定位管子，以进行切割和钻孔操作。休息过后，请观看视频中的实际操作。

如果你一直在关注[HomoFaciens]，你无疑会看到他的发展方向——构建一个低端工具，用它来构建一个更好的工具，等等。我们很高兴看到他转向更坚固的材料，但我们会错过纸板和回形针的制作。
