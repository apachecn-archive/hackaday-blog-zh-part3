# 给 C64 一个 WiFi 调制解调器

> 原文:[https://hack aday . com/2015/12/21/giving-the-c64-a-wifi-modem/](https://hackaday.com/2015/12/21/giving-the-c64-a-wifi-modem/)

如果有任何迹象表明 Commodore 64 的寿命，那就是仍在设计和制造的外围设备和附加设备的数量。现在，你可以给 C64 加一个 SD 卡，这项技术是在 Commodore 64 发布 16 年后推出的。多亏了[叶小开·布鲁姆奎斯特]，[你还可以给最珍爱的家用电脑添加无线网络](https://jammingsignal.wordpress.com/commodore-wi-fi-modem/)。

[叶小开]用于 C64 的 WiFi 调制解调器由两个主要部分组成。第一个是[微视图有机发光二极管显示器](https://www.sparkfun.com/products/12923)，允许用户添加 SSIDs、密码，并通过 USB 配置网络。第二个大模块是漫游网络“无线”适配器。这是一个使用熟悉的 Xbee 引脚排列的 WiFi 适配器，这不仅是 C64 的 WiFi 适配器，也是几乎所有无线网络协议的适配器。

本月早些时候,[叶小开]在多伦多的准将世界上介绍了这款用于 C64 的 WiFi 调制解调器。在那里，它获得了 Commodore 爱好者的大量关注，人们可以对硬件进行视频审查。你可以在下面的视频中看到[Alterus]通过 Wifi 上传一个 BBS。
