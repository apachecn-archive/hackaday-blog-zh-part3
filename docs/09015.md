# 禅宗与日本茶机器人艺术

> 原文：<https://hackaday.com/2018/04/02/zen-and-the-art-of-japanese-tea-robots/>

在日本，茶道被视为一种获得对生活和哲学更深刻见解的方式。传统的日本茶道从业者花很长时间来掌握倒茶的错综复杂和细节。成为茶大师的道路是至关重要的，因为它发展了从业者的精神状态和身体技术。

然而，如果你没有时间掌握“茶之道”，那么你可以建立一个机器人，自动化你的禅体验。这正是阿诺实验室的人在建造他们的日本茶道机器人#151A 时所做的。
