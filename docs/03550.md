# Hackaday 大奖参赛作品:Neopixel 怀表

> 原文：<https://hackaday.com/2016/09/15/hackaday-prize-entry-neopixel-pocket-watch/>

在硬件黑客的世界里，时计是一个相当重要的仪式，我们永远不会停止被我们社区的创新所吸引。

今天的例子来自【约书亚·斯奈德】，他做了一块怀表。不仅仅是普通的怀表，[他把一个发条手表的外壳装上了一圈新像素](https://hackaday.io/project/13353-neopixel-pocket-watch)，它由一个 ESP8266 模块驱动。电力来自一个小的脂肪电池，他巧妙地设计了一个小按钮开关，这样它就可以由原来手表上的旋钮来启动。不同颜色的发光二极管穿过戒指，模拟传统时计的指针，整体隐藏在表壳的穿孔盖后面，有点蒸汽朋克的感觉。

他承认目前电池续航时间不是很好，可能是因为目前 WiFi 一直处于开启状态，所以他可以接触到它的网络界面进行调试。遗憾的是，他似乎还没有发布该软件，但他告诉我们，它使用 NTP 来更新时间，并支持新版本的空中更新。他建议，未来的版本可能会取消 ESP，使用 ATtiny 或类似的实时时钟，从而延长电池寿命。

这些年来，我们涵盖了许多 LED 时计，包括相当多的手表。只有一个小选择是这个 [PIC LED 戒指在一个怀表盒](http://hackaday.com/2009/10/14/led-pocket-watch/)，另一个 LED 戒指这次是由 ATMega645 供电的[，还有这个非常](http://hackaday.com/2012/11/30/led-pocket-watch-2/)[时尚的有机发光二极管手表](http://hackaday.com/2013/12/17/stylish-oled-watch-uses-accelerometer-instead-of-buttons/)。
