# 机械建筑让你在现实生活中跳过仙人掌

> 原文：<https://hackaday.com/2017/11/26/mechanical-build-lets-you-jump-cacti-in-real-life/>

学起来简单，掌握起来难，但要戒掉这个习惯需要一辈子。这适用于许多电脑游戏，但 Chrome 的 T-rex Runner 游戏及其各种在线版本尤其阴险。以至于这款游戏促使一对夫妇制作了一款现实版的数字游戏。

对于那些不熟悉这个游戏的人来说，这是一个简单的侧滚游戏，目标是跳跃和躲避一只奔跑的恐龙越过和越过障碍物——想象一下 Flappy Birds，但要快一些。在决定一个周末黑客马拉松项目时，[Uri]认为现实生活版本的游戏会是一个自然的选择，因为他已经是数字版本的粉丝了。在他的女朋友(Ariella)的帮助下，[Uri]能够开发出一个最低限度可玩的游戏版本，一个步进电机提供恐龙跳跃，一个简单的直线传送带移动障碍物。人们非常喜欢它，以至于为 Chrome 开发者峰会设计了 2.0 版本。这个版本更具可玩性，有一个椭圆形的障碍跑道和更好的记分。[Uri]和[Ariella]必须扩展他们的技能来完成构建——PCB 设计、电子纸显示器、激光切割，甚至金属铸造都是必需的。下面的视频展示了最终版本——但是翼龙要闪开哪里呢？

现实世界中跳跃的恐龙并不是数字游戏的第一个物理表现。和网络世界一样，Pong 是第一个——要么是街机版本的 T1，要么是超大型户外游戏 T3 的 T2。
