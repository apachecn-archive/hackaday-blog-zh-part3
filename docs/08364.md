# 不需要分度卡盘

> 原文:[https://hack aday . com/2018/01/25/indexing-chuck-not-required/](https://hackaday.com/2018/01/25/indexing-chuck-not-required/)

学会使用车床是一项强大的技能，但这只是旅程的一半。聪明在后面，这是课程的第二部分。耐心也是存在的，但是让我们把注意力放在变聪明上。[TimNummy]想要一个带有关键参数的[球形螺栓，所以他自己做了一个。休息之后，有一个链接视频的六十秒总结。](https://www.youtube.com/watch?v=W2ed9YpXC1E)

制作库存硬件是初学者的任务，因此定制硬件需要独创性或昂贵的机械。使用分度卡盘可以随意在螺栓头上添加指形缺口，但是没有可用的指形缺口。相反，六角股票成为一个夹具，和平坦的一面是用来保持在六个间歇的角度工件。我们不能否认这个结果，它看起来像是一个要花一大笔钱的零件。

使用在车间里找到的材料是聪明的全部。六边形黄铜原料在侧面和角度上有严格的公差，为什么不利用这一点呢？
