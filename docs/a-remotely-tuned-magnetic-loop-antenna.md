# 远程调谐磁环天线

> 原文：<https://hackaday.com/2017/05/08/a-remotely-tuned-magnetic-loop-antenna/>

如果你是一个无线电爱好者，你可能对磁环天线很熟悉。它不同于大多数传统的线天线，采用带有非常大的单匝线圈和调谐电容器的调谐电路的形式。磁环具有极高的选择性和良好的方向性，但存在调谐电容上感应高电压的危险，而且每次频率改变时都需要重新调谐。

[奥列格·鲍里索夫，RL5D]有一个磁环，很快就厌倦了不断返回。他的解决方案很优雅，[他用步进电机、Arduino 和蓝牙模块](https://rl5d.blogspot.de/2017/05/remote-tuning-for-magnetic-loop.html)(这里翻译为)做了一个远程调音设置。步进器通过一个短的柔性耦合连接到电容器，并在自定义 Android 应用程序的帮助下进行调谐。我们很想知道高射频场对这些组件有什么影响，但他没有报告任何问题，所以它肯定是工作的。

他发布了一个视频的单位在运作中，我们已经张贴在下面的休息，如果你曾经不得不不断地重新调整磁回路，你会欣赏的方便。
