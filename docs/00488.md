# 自行车低头显示器为您指路

> 原文:[https://hack aday . com/2015/10/28/bicycle-heads-down-display-shows-you-the-way/](https://hackaday.com/2015/10/28/bicycle-heads-down-display-shows-you-the-way/)

在自行车上，你通常会向前看，并稍微向下看，以确保你不会骑过任何坑洼。虽然在车把上安装一个速度计很好，但是如果你能把你的速度和其他信息投射到前方的道路上会怎么样呢？嗯，事实证明[如果你有一部智能手机和一点聪明才智](http://streetlight.mobi/)，你就可以做到这一点！

我们以前见过用微型投影仪完成的这种黑客攻击，但这对你的自行车来说是一笔相当大的投资。所以[彼得]有了另一个想法。如果你能用你的手机和一点光学设备会怎么样？原来你可以从中国或亚马逊买到一些非常便宜的“手机投影仪”——它基本上是一个你的手机可以放进去的小纸箱，用一个便宜的塑料镜头来投影你的图像。流明输出是相当悲惨的，但如果你在晚上骑，这是足够的。

一旦他对硬件了如指掌，他就编写了一个开源应用程序，如果你想添加自己的功能，可以在 GitHub 上下载。如果你只是想玩玩，可以在 [Google Play](https://play.google.com/store/apps/details?id=com.ionicframework.streetlight) 上找到。
