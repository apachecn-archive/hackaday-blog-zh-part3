# 最奇怪的真空管上的附加物

> 原文:[https://hack aday . com/2017/10/10/add-on-the-strange-tube/](https://hackaday.com/2017/10/10/addition-on-the-strangest-vacuum-tube/)

[Uniservo]制作了一个他长期以来一直试图获得的电子管视频:一个[罗杰斯 6047 爱迪生](https://www.youtube.com/watch?v=RsYU2f3Q6kI&feature=youtu.be)。没听说过艾迪特龙吗？我们也没有。但它是 1950 年左右加拿大制造的单个真空管中的全二进制加法器。你可以看下面的视频。

这种独特的电子管是为多伦多大学电子计算机(UTEC)制造的。一台普通的电子管计算机需要几个电子管来完成加法运算，但是 additron 是一个电子管，它使用光束转换在一个组件中完成加法运算。[Uniservo]指出了电子管如何彻底改变了电子管计算，但在它真正出现在实际设计中之前，晶体管——以及后来的集成电路——将会取而代之。

电子管确实在 1950 年被用于井字游戏中，但那似乎是有记录以来唯一的实际用途。[数据表其实是在线的](http://www.mif.pg.gda.pl/homepages/frank/sheets/201/6/6047.pdf)。然而，如果你真的想看看它是如何工作的，读读[专利](https://www.google.com/patents/US2784312)吧。

这个电子管比[继电器加法器](https://hackaday.com/2014/04/26/4-bit-adder-built-from-mechanical-relays/)有了很大的改进。如果你想比较一个更现代的加法器，这里有一个在 FPGA 上的 [Verilog。](https://hackaday.com/2015/08/19/learning-verilog-on-a-25-fpga-part-i/)
