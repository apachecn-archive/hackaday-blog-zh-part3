# 用联觉面具闻颜色

> 原文：<https://hackaday.com/2015/11/18/smell-colors-with-a-syntesthesia-mask/>

联觉是感官知觉的混合，其中一种感觉的刺激导致第二种感觉的刺激。在这种情况下，周三可以是蓝色的，你最喜欢的歌曲的最好部分可以是橙色的，六可以在七十三的右边。虽然你无法自学联觉——这是与生俱来的——但[Zachary]决定在他最近的电子项目中模仿颜色来闻联觉。

对于他的联觉面具，[扎克]正在将颜色传感器发现的不同数量的红色、绿色和蓝色变成气味。他正在用现成的颜色传感器、英特尔爱迪生、几个伺服系统和装满精油的试管来做这件事。颜色传感器安装在一个环上，允许[扎克]选择他想闻的颜色，气味头盔包含一个装有风扇的小电子盒，可以将气味吹到他的脸上。

有不止一种类型的联觉，如果你正在寻找更痛苦的东西，[你可以通过一个微型网络摄像头将像素转换成小型振动马达的脉冲，让物体感觉起来很响亮](http://hackaday.com/2011/11/18/syneseizure-makes-objects-feel-loud/)。
