# 丙烷动力等离子步枪

> 原文：<https://hackaday.com/2017/10/12/propane-powered-plasma-rifle/>

它可能不是“40 瓦范围内的相位等离子步枪”，它甚至没有使用严格定义的等离子，但它仍然很酷。是[丙烷动力的瓶子发射步枪](https://www.youtube.com/watch?v=bB01zORIGkA)，看起来很好玩。

[夜鹰之光]当然喜欢流行的东西，比如[他浪费西瓜的气动加农炮](https://hackaday.com/2017/04/07/build-your-own-high-power-air-cannon-out-of-pvc/)和[廉价的流行枪](https://hackaday.com/2016/05/23/feed-the-fun-with-a-semi-automatic-cheesy-poof-rifle/)。这个更有魅力，因为它是由丙烷火炬驱动的。原理很简单:用丙烷装满汽水瓶，点燃气体，乐趣随之而来。不过，细节有点微妙，需要留有余量，以防止背压阻止炮弹装满燃料。[NighthawkInLight]通过对枪管的巧妙加工克服了这一点。下面的视频中的最终生产版本是不必要的，但令人愉快的复杂，一个木制的股票和一个~~线圈的透明乙烯基管~~螺旋血浆收集器在桶前；最后一点只是作秀，我们不得不承认它看起来相当不错。

除非你把使用 CPVC 管制作定制配件的专业技巧也算在内，否则这一条绝对是有趣的。但我们对此没有异议。
