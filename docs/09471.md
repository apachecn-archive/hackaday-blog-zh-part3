# 控制爱好者的倒立摆

> 原文：<https://hackaday.com/2018/05/20/inverted-pendulum-for-the-control-enthusiast/>

一旦你步入控制的世界，你会很快意识到控制即使是简单的系统也不像给伺服系统施加电压那么简单。不过，在你开始研究自己的双足机器人或临时搭建的无人机之前，你可能想在这个复杂的工程领域进行一些实践。这一领域的一个经典问题是倒立摆，[Philip]已经创建了一个很好的模型来帮助说明控制的基础，并在中混合了一些人工智能。

该项目名为 ZIPY，是一个“手推车杆”设计，使用手推车上的可移动手推车来平衡上面的钟摆。钟摆在一点上与手推车相连。通过来回移动手推车，钟摆可以保持在垂直位置。该控件使用 OpenAI Gym 工具包，这是一种在您自己的项目中轻松使用强化学习算法的方法。有了一些 Python，一些 3D 打印部件和工具包，[Philip]能够让他的项目成功地平衡推车上的钟摆。

当然，OpenAI Gym toolkit 对于更多您可能需要某种机器学习帮助的项目非常有用。如果你想在不构建任何东西的情况下玩机器学习，[你也可以在你的浏览器](https://hackaday.com/2018/04/16/tensorflow-in-your-browser/)中探索它。
