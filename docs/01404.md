# 一台由懂行的人制造的家用数控机床

> 原文：<https://hackaday.com/2016/02/07/a-home-cnc-built-by-someone-who-knows-their-stuff/>

[thisoldtony]有一家不错的店[需要一台 CNC](https://www.youtube.com/watch?v=K9UA9ZRFwWU) 。我们不确定他具体做什么，但我们认为他可能是机械师或工程师。不管怎样，他确实造了一台不错的 CNC。许多自制的 CNC 很整洁，但是缺乏。即使是流行的套件也忽略了基本的机器设计原则。这对于他们将[通常用于](http://hackaday.com/2015/11/21/garbage-can-cnc-machine-build/)的工作来说是可以的，但是很高兴看到一个工作做得很好。

大多数自制的机器很难或不可能摆正。也就是说，使每个轴精确地垂直于其他轴移动。他们也忽略了设计机器将会看到的负载，或者调整整个运动的偏差。还有轴承预紧力，背隙，以及更多的担心。[thisoldtony]已经把这些都考虑进去了。

这个系列很长，但看起来很有趣，我们在这个过程中学会了一些技巧。最终的 CNC 非常吸引人，并且在一些调整之后表现良好。在最后一个视频中，他为儿子制作了一把令人惊叹的橡皮筋枪。如果您愿意，也可以下载机器的 [STEP 文件。休息后的视频。](https://drive.google.com/file/d/0BziGqMVEh_qrdlBGTmRKcThVRDg/view?usp=sharing)
