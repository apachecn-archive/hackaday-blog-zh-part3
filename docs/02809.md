# Hackaday 奖参赛作品:模块化、低成本盲文显示器

> 原文：<https://hackaday.com/2016/07/09/hackaday-prize-entry-modular-low-cost-braille-display/>

19 世纪中期，二进制算术的许多工作都是开创性的。布尔代数是由乔治·布尔发明的，但是一个不太明显的二进制发明也在这个时候诞生了:盲文书写系统。使用一个凸点系统(基本上是 1 和 0)，视力受损的人已经能够使用他们的触觉来阅读。然而，在信息高速发展的现代，这就有点困难了。许多人一直在研究可刷新的盲文显示器，包括【Madaeon】他创造了一个[模块化的可刷新盲文显示器](https://hackaday.io/project/12442-a-modular-low-cost-braille-display-system)。

这个想法是用一组微小的螺线管来再造盲文细胞。单元格是一组点，每一个点都可以以特定的排列方式升高或降低，以代表一个字母或其他符号。有了一组螺线管，这可以相当迅速地完成。[Madaeon]已经使用最新的 3D 打印和激光切割方法制作了这些微小可控点的原型，并准备组装他的第一个完整的盲文字符。

虽然这还没有完全准备好进行全面展示，但基本原理看起来像是建立一个坚实的基础。这可能是历史上最文明的关于类似盲文显示器的专利纠纷。希望所有关于盲文显示器的讨论和黑客攻击能够降低成本，让任何需要的人都能够轻松获得并使用它。
