# 苹果 IIc 上的流媒体视频

> 原文:[https://hack aday . com/2015/11/27/streaming-video-on-an-apple-IIC/](https://hackaday.com/2015/11/27/streaming-video-on-an-apple-iic/)

我们介绍的一些项目解决了一个问题。其他人只是证明他们可以做到。我们猜测是后者促使[乔舒亚·贝尔]为苹果 IIc 编写一个 VNC 客户端。为了充分理解这有多疯狂，请看下面的视频。

这种黑客的惊人之处不止一个。不知何故，[约书亚]的 VNC 程序完全运行在苹果 IIc 的内存中，正如他在视频开始时通过串行电缆将所有代码下载到苹果中所演示的那样。在初始引导之后，他运行代码，你会看到(四色辉煌！)一个低分辨率的 Windows XP 出现在 IIc 上。

[![2440964467_decb0daf10_o](../Images/9cfe1201e32124287eaf2b778ed1891b.png)](https://hackaday.com/wp-content/uploads/2015/11/2440964467_decb0daf10_o.jpg) 更不可思议的是，不幸的是，视频中没有展示的是，他似乎不仅仅是在苹果上镜像了 PC 的屏幕，而是实际上成功地获得了每秒一帧的*双向* VNC，工作在 115，200 波特。在这张来自他的 flickr 画廊的快照[中，他似乎正在 IIc 上玩卡拉提卡，并在他的笔记本电脑上观看。](https://www.flickr.com/photos/26000176@N07/sets/72157604727064420/detail/)

如果你有一个 IIc，你想展示更多新的技巧，不要忽视这个为苹果 IIc 编写的浏览器。或者，如果你只有一个苹果 IIc+,并且你完全不喜欢哔哔声与 IIc 的哔哔声不同，你总是可以[进行史诗般的逆向工程探索来“修复”它](http://hackaday.com/2015/10/30/quinn-uses-forsooth-to-win-the-internet-also-fixes-apple-iic-beep/)。
