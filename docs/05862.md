# 任天堂 PlayStation:终于开始工作了

> 原文：<https://hackaday.com/2017/05/06/the-nintendo-playstation-finally-working/>

任天堂 PlayStation 并非用词不当。在 PS1 之前，索尼与任天堂合作生产了一款视频游戏机，该游戏机使用光盘作为分销平台。这些计划落空了，索尼接着设计了 PS1，任天堂设计了 N64，但一些原型“任天堂 PlayStations”却没有成功。其中一台难以置信的稀有游戏机被运送到一家最终破产的公司。这个控制台是在一栋办公楼的物品被拍卖时被发现的，去年[【本·赫克】把它拆开了](http://hackaday.com/2016/07/16/tearing-apart-the-nintendo-playstation/)。

花了一年时间，但现在这个[任天堂 PlayStation 终于开始工作了](https://www.youtube.com/watch?v=gaIfPuziJ-0)。这款游戏机现在可以播放自制软件编写的音频 CD 和游戏。硬件活了，曾经被遗忘的游戏机又活了。

上一次[Ben Heck]看任天堂 PlayStation 的时候，游戏机的 CD-ROM 部分无法正常工作。超级任天堂仍然是功能性的，但对于这个原型来说，CD-ROM 是完全独立的，需要一个“引导盒”来访问 CD 上的任何内容。不知何故——[本]认为这是一个摇摇晃晃的电缆或死帽——光盘复活了。是的，在花费大量时间对控制台进行逆向工程后，晃动电缆是修复的范围。

随着光盘的工作，[本]得到了音频播放，并尝试了这个 PlayStation 原型的几个自制游戏。超级 Boss 外传不太管用，因为这个游戏被设计成批量加载。另一个为这款游戏机编写的游戏，[魔法地板](http://problemkaputt.de/magicflr.htm)，足够小，可以放入整个光盘的缓冲区，并且可以正确加载。这并不意味着游戏*成功了*；任天堂 PlayStation 模拟器和现在存在的实际硬件之间有一些细微的差别。[本]给《魔法地板》的作者发了封电子邮件，现在，在四分之一世纪后，任天堂 PlayStation 开始工作了。

任天堂 PlayStation 的下一步是什么？好了，现在这个系统的模拟器可以正确地反映实际的硬件，希望自制者可以弄清楚如何为这个系统写一个游戏。
