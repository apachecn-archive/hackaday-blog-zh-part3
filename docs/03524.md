# 装有 555 的频谱分析仪

> 原文:[https://hack aday . com/2016/09/18/spectrum-analyzer-with-555-fits-in-a-a-tin/](https://hackaday.com/2016/09/18/spectrum-analyzer-with-555-fits-in-a-tin/)

【VK2ZAY】对 555 芯片情有独钟。在微控制器出现之前，555 是硬件黑客的瑞士军刀。毕竟，尽管芯片应该是一个定时器，但它实际上是一堆可以用来制作定时器的简单部件:一对比较器、几个晶体管和一个触发器。你可以用许多不同的方式使用这些部件，定时器只是其中之一。

[VK2ZAY]在一个简单的频谱分析仪中使用了一个作为关键部件[。555 产生改变振荡器频率的斜坡电压。振荡器与输入信号混合，固定频率的超再生检波器产生与输入信号强度成比例的输出电压。你可以在下面看到一个完整的视频。](https://www.youtube.com/watch?v=eIUR4zuP5Dc)

简单而优雅。不是你能找到的性能最好的频谱分析仪，但它能完成工作，而且它适合一个 Altoid 罐。对于 555 来说，产生一个斜坡并不太难，但我们已经看到[VK2ZAY] [滥用 555](https://hackaday.com/2011/08/05/building-a-computer-out-of-555-chips/)要糟糕得多。

如果你想了解更多关于 555 的信息，你可以通过【Ken Shirriff】获得 [all forensic](https://hackaday.com/2016/02/23/555-teardown-and-analysis/) 。或者，如果你愿意，你可以在[更宏观的层面上](https://hackaday.com/2013/09/22/whats-inside-a-555/)看看它的内脏。
