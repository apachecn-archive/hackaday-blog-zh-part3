# 无支架打印！

> 原文:[https://hackaday.com/2017/12/21/printing-without-supports/](https://hackaday.com/2017/12/21/printing-without-supports/)

如果印刷支撑曾经让你感到悲伤，要知道还有一种替代的印刷方法在工作中。首先:给自己准备一个[桶工业凝胶，在里面打印](http://www.3ders.org/articles/20171215-mits-rapid-liquid-printing-technology-3d-prints-bags-and-lamp-shades-in-minutes.html)。

快速液体打印(RLP)是由密歇根州的公司[Steelcase]和麻省理工学院的[Skylar Tibbits']自组装实验室合作开发的。RLP 正在宣传相对于传统 3D 打印技术的优势，例如减少打印时间、更高质量的打印，以及实现更大规模的打印——所有这些都没有支持！

使用橡胶、塑料或泡沫时，打印材料通过喷嘴注入一盆工业凝胶中。该凝胶在整个过程中悬浮印刷物，而不与印刷物结合，最终产品简单地从凝胶中取出并冲洗掉。本月早些时候在迈阿密设计活动上展示，旁观者可以在几分钟内拿起完成的灯罩和手提袋。
