# 夹子、隔板和发明之母

> 原文:[https://hack aday . com/2015/12/15/clamps-cauls-and-the-mother-of-invention/](https://hackaday.com/2015/12/15/clamps-cauls-and-the-mother-of-invention/)

如果你在木工车间需要一样东西，那就是更多的夹子。有杆夹、管夹、弹簧夹和扳机夹，但是对于车间中的一项任务来说，没有一种夹子能胜任。胶合木板——几块宽木板在边缘接合——要么需要比你更多的夹子，要么需要垫板，即在夹子将木板水平压在一起的同时垂直压在木板上的装置。

[Andrew Klein] [刚刚为这项任务发明了一种新型夹钳](https://www.youtube.com/watch?v=FLtWd0N8QbQ)，再次证明了并非所有问题都得到了解决，仍然有一些地方可以突然出现一项发明。

新夹具是对传统杆夹具的改进，允许两个夹具互锁。在夹钳的每个“工作”端，有两个调节手柄。第一种是水平拧紧夹具，就像任何杆或管夹一样。第二调节手柄上下移动轴承。当该轴承与另一个夹具配合端的竖板啮合时，两个夹具被垂直压在一起。

新的夹具是有效的夹具和隔板，能够将材料从一边推到另一边，从上到下。新的夹子也能工作。在下面的视频中，你可以看到[Andrew]在粘合一块面板。当垂直调节轮松开时，板会垂直分开。当垂直调节轮拧紧时，两块板完全相互对齐，边对边，面对面。
