# 具有多个 pi 的多台显示器

> 原文：<https://hackaday.com/2017/08/17/multiple-monitors-with-multiple-pis/>

Raspberry Pi 在商业环境中最受欢迎的用途之一是电视墙、数字标牌和媒体播放器。很有可能，你已经见过一个显示器或其他发光的矩形显示广告或推文，由树莓派供电。[Florian]一直在为这个用例做一个名为 info-beamer 的项目，现在他有了一些惊人的东西。他可以使用多个 pi 在多个显示器上显示一个视频，配置简单到[用手机](https://www.youtube.com/watch?v=GI00HTJhSMU)拍照。

[Florian]为 Pi 创建了 info-beamer 包，用于视频回放(包括同时播放多个视频)，显示公共交通信息、twitter 墙或会议信息系统。不久前，[Florian]在 reddit 上展示他的作品[时，他得到了一个自动配置多个屏幕的建议。几天后，一切都正常了。](https://www.reddit.com/r/raspberry_pi/comments/6r6vz2/things_you_didnt_know_a_bunch_of_pis_can_do/)

现在，配置屏幕的过程包括在每个显示器上显示基准点，用手机和网络界面拍照，让服务器做一些数字处理。在[Florian]拍摄了所有屏幕的照片后不到一分钟，一部电影就在三个方向怪异的显示器上播放。

下面，您可以查看[Florian]配置三个 pi 和显示器以显示单个视频的视频，随后是一个德语演示，介绍 info-beamer 的亮点。
