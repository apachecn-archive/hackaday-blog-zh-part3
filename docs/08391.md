# 惊人的机械连接和设计它们的软件

> 原文:[https://hack aday . com/2018/01/26/amazing-mechanical-linkages-and-the-software-to-design-them/](https://hackaday.com/2018/01/26/amazing-mechanical-linkages-and-the-software-to-design-them/)

我们大多数人都是些零碎的东西，而不是具体的东西，但是我们对这两者的结合有着最深刻的理解。显然，[rectorsquid]也是如此。看看他的[滚球雕塑](https://www.youtube.com/watch?v=rShllGfOrqM) (YouTube，下面嵌入)的设计和*流程*，看看我们的意思。看到球转移时手臂有一点犹豫了吗？看到上臂是怎么轻轻的用一个微微向下的手势把它放在坡道上的吗？看看用*一个电机*是怎么做到的？[rectorsquid]不可能在纸上设计这个，对吧？

[他当然没有](https://www.youtube.com/watch?v=O5IqIGtqKDk) (YouTube)。相反，他[编写了一个模拟器](http://blog.rectorsquid.com/linkage-mechanism-designer-and-simulator/)，让他实时尝试各种定制的链接。这是一个 Windows 专用的应用程序(唉)，但它是免费使用的，而[视频指南](https://www.youtube.com/watch?v=K9z9JFreHT8)(更多 YouTube)看起来非常全面，并让你快速浏览该工具。特别值得注意的是，[rectorsquid]的软件允许滑动连接，他在这里展示的滚动球雕塑中很好地利用了这一点。

事实上，我们之前已经秘密展示过[rectorsquid]的链接软件，在[这篇文章中，一些令人惊叹的 cosplay 电子动画翅膀](https://hackaday.com/2016/12/17/animatronic-cosplay-wings/)使用该程序进行设计。但是我们真的不希望你错过如果你在做机械设计，需要这样的东西，或者只是想玩玩。

如果你想研究你的螺母和螺栓，请查看我们关于无处不在的四连杆机构的[初级读本，或者浏览 Hackaday 寻找其他伟大的连杆机构驱动的例子，像这个](https://hackaday.com/2017/03/29/marvelous-mechanisms-the-ubiquitous-four-bar-linkage/)[自动钢锯](https://hackaday.com/2016/04/08/3d-printed-linkage-produces-automatic-hacksaw/)或一个[缩放仪 PCB 探针](https://hackaday.com/2017/10/04/precision-pantograph-probes-pcbs/)用于颤抖的手。

有没有人知道一个开源的链接模拟器，也可以输出 STL 文件进行 3D 打印？或者任何可以轻松转换成 OpenSCAD 的格式？求一个“朋友”。
