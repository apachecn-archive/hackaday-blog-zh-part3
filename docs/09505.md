# 观看蜂窝时钟轻轻地跟踪时间

> 原文：<https://hackaday.com/2018/05/26/watch-the-honeycomb-clock-gently-track-time/>

我们在 Hackaday 喜欢钟表，约翰·惠汀顿也是。去年，他通过将一些 RGB 发光二极管与激光切割框架结合起来，创造了一个平滑的时间显示器，利用颜色和位置来显示时间，这是一个简单而有吸引力的系统。

十二个六边形的外环本质上是时针，类似于模拟钟面:十二是向上的，三是直接向右的，六是直接向下的，九是向左的。内环代表每十六进制十分钟。每当内环填满时，外环上的下一个十六进制(小时)就会亮起。整个展示在中午和午夜被一分钟长的彩虹淹没。观看下面嵌入的视频中的实际操作。
