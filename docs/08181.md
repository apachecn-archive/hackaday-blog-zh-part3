# LED 树给圣诞节带来了重力

> 原文：<https://hackaday.com/2018/01/10/led-tree-brings-gravity-to-christmas/>

这里有一个有趣的进入我们的硬币电池挑战。能源是[【弗兰克】的 blinky LED 圣诞树](https://hackaday.io/project/28793-gravity-assisted-blinking-coin-cell-christmas-tree)中的驱动力，它利用硬币电池的物理结构和我们的老朋友重力推出一些节日欢呼。说说正向电压！

我们喜欢这个概念，电路再简单不过了。一个硬币电池在树的顶部被释放，并滚下一系列覆盖有 1/4″铜带的倾斜泡沫板栏杆。当硬币电池移动时，负极端子沿着采油树表面摆动，采油树有相应的接地轨带。这里没有微控制器——闪烁所需要的只是负轨带的断裂。

像这样的项目最具挑战性的部分是执行。在建造之前和建造过程中，让硬币电池在轨道上运行而不脱落需要进行角度实验。现在完成了，保持树向后倾斜靠在墙上是关键。[Frank]探索了几种使用相机马达和旧喷墨打印机的齿轮组件将硬币电池返回顶部的选择，但目前，他六岁的儿子毫无怨言地完成了这项工作。休息之后看看他的职业道德。
