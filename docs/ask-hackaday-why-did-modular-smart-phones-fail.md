# 问问 Hackaday:为什么模块化智能手机会失败？

> 原文：<https://hackaday.com/2017/04/21/ask-hackaday-why-did-modular-smart-phones-fail/>

还记得所有关于模块化智能手机的讨论吗？他们听起来棒极了！你不需要升级你的手机，你只需要升级一些部件，有点像电脑，但是更简单。看起来[模块化手机已经死了](https://www.youtube.com/watch?v=CcQxdjHlKO4)(视频，嵌入在下面)，即使在许多主要的手机制造商都在赶时髦之后。甚至谷歌也加入了随后被取消的谷歌 Ara。LG 发布了 G5，但表现不佳。摩托罗拉的 Moto Z 似乎也同样缺乏兴趣。当这些[模块化手机](https://hackaday.com/2015/10/30/the-key-to-modular-smartphones/)的概念被宣布时，嗡嗡声不绝于耳，人们真的对这些可能性感到兴奋。哪里出了问题？

首先，人们希望他们的手机已经具备一切功能，无论是相机、GPS、WiFi、高容量电池还是高分辨率屏幕。消费者期望这些东西成为标准。为什么他们会出去买一个模块，而市场上的其他手机已经有这些东西了？

当然，你可以得到一些奇怪和美妙的模块，如超大声扬声器或投影仪，但这些项目的需求很小。因为这些额外的功能已经可以作为独立的附件获得，而不是锁定在一个设备上，所以从一开始就不可行。

> 当我们做用户研究时。我们发现大多数用户并不关心核心功能的模块化。他们希望他们都在那里，总是工作，始终如一。Ara 项目首席工程师

至少可以说，如果这些手机成为主流，它们的黑客能力会很有趣。似乎公众想要薄而光滑的铝制手机，他们认为这是一种身份的象征。模块化手机必须体积更大，以容纳模块的电源/数据轨和磁铁，因此它们将失去便携性。尽管如此，我们还是希望这个想法在未来被重新审视，而不是被扔进废弃的垃圾堆。

你会购买模块化智能手机吗？即使它更大或更贵？这真的是他们失败的原因吗？
