# 8 位试验板计算机长达 8 小时

> 原文：<https://hackaday.com/2017/04/10/8-bit-breadboard-computer-is-up-to-8-hours/>

[Ben Eater]发布了几年前他完全在试验板上构建的一台没有 CPU 芯片的 8 位计算机的一些视频。被问及原理图后，他终于承认自己没有。因此，相反，他决定重建它，并对过程中的每一步进行视频记录。你可以在下面看到他的启动视频，但你也可以找到 [30 个更近的视频](https://www.youtube.com/user/eaterbc/videos?sort=dd&view=0&shelf_id=1)，涵盖从 ALU 设计和故障排除到十进制 LED 显示的主题。他甚至使用 Arduino 对 EEPROM 进行编程，用来替换大量逻辑。

你可能想等到你有空的时候，因为到目前为止大约有八个小时的视频。视频从一个简单的 555 定时器开始，然后从那里开始。每一部分都得到一个独立于整体的测试，所以如果幸运的话，你不会有一个不可能的工作去尝试在最后解决整个问题。

像这样的项目显然是不切实际的，但是如果你想真正理解 CPU 是如何工作的，建造一个是开发这种理解的好方法。我们建议学习 Verilog 或 VHDL，并在 FPGA 上构建，但试验板计算机有一定的街头信誉，肯定有一个良好的闪光灯阵列。

CPU 的设计遵循了《数字计算机电子学》一书中的设计，所以如果你真的想重新制作它，你也可以跟着做。这本书已经绝版了，但是现在找到绝版的书并不困难。

我们看到的大多数试验板计算机使用一个 CPU 芯片，所以它们不需要那么多试验板。正如你所料，其中一些比其他的更乱[。](https://hackaday.com/2016/03/07/breadboard-colecovision/)
