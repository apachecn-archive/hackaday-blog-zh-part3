# 私人实验室黑客马拉松将艺术的礼物还给有需要的陌生人

> 原文：<https://hackaday.com/2016/06/05/fablab-hackathon-gives-the-gift-of-art-back-to-a-stranger-in-need/>

如果建造詹姆斯·韦伯太空望远镜和 F/A-18 战斗机还不足以成为为诺斯罗普·格鲁曼公司工作的理由，那么还有私人实验室——该公司赞助的 hackerspace。这是一个一切皆有可能的地方，包括把艺术礼物还给一个有需要的陌生人。

下面的视频讲述了[劳尔·皮萨罗]的故事，他是一个年轻人，也是一个有天赋的艺术家。小时候被诊断出患有肌肉萎缩症，[劳尔]的肌肉逐渐变弱，很难举起手臂。[劳尔]的艺术正在消失——直到[托尼·朗]听到了这个故事的风声。[托尼]管理私人实验室，一旦他向他的同事们发布消息，他就召集了一次黑客马拉松来为[劳尔]研究解决方案。

他们想出的是一个头顶支撑系统，带有工具平衡器和定制的铰接吊索，以减少重力的影响并支撑[劳尔]的手臂。为了补偿他减少的活动范围，他们还建造了一个带有致动器的画架，可以升高和降低他的画布，并将其放置在他需要的地方。看起来 FabLab 团队特别注意让一切尽可能平稳，结果[劳尔]又回到了他最擅长的工作上。哦，如果[托尼]和私人实验室听起来很熟悉，那可能是因为去年我们的[迈克·斯奇斯]访问诺斯罗普时，他是东道主。

我们真的喜欢看到帮助整个人类的黑客，但一群陌生人聚在一起帮助一个人也是很特别的。向[托尼]和他的私人实验室团队致敬，他们完成了一次伟大的黑客行动，把他的艺术还给[劳尔]
