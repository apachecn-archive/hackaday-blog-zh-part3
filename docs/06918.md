# 对不起我们；欧洲人用坟墓听太空

> 原文：<https://hackaday.com/2017/08/27/sorry-us-europeans-listen-to-space-with-graves/>

在欧洲，格雷夫斯雷达站几乎直接在 143.050 兆赫上发射信号，以探测和跟踪卫星和太空垃圾。这意味着你通常听不到电台的任何信号。然而，【DK8OK】展示了你如何——如果你在欧洲——[倾听来自强大雷达](https://dk8ok.org/2017/08/23/graves-reflections-out-of-the-blue/)的反射。反射可能来自飞机、流星或宇宙飞船。你可以从[way1888]上看到一段视频，展示了最近英仙座流星雨的结果。

使用软件定义的无线电接收器，[DK8OK]稍微偏离频率，等待瀑布中出现反射。除了观察信号，还可以处理音频以创建更多细节。

为什么法国中部会有一个巨型垂直雷达发射机？发射器使用相控阵一次在 45 度的天空范围内发送信号。它每 19.2 秒总共走 6 步。几百英里外的接收器监听反射。

甚至[月球在雷达路径上时也会反射](http://www.itr-datanet.com/~pe1itr/pdf/The%20143.050MHz%20Graves%20Radar%20a%20VHF%20Beacon.pdf)信号。如果你对月球的反射感兴趣，你可以建立一个站，不用去欧洲就能听到反射的声音。

当然，如果你能传送自己，你可能想从飞机上反弹自己的信号。如果你想用老办法，你可以效仿[【佐尔坦湾】](http://hackaday.com/2013/11/19/retrotechtacular-zoltan-bays-moon-bounce-coulometer-signal-amplifier/)。
