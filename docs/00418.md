# Arduino.cc 拓展欧洲制造业

> 原文:[https://hack aday . com/2015/10/22/arduino-cc-expands-European-manufacturing/](https://hackaday.com/2015/10/22/arduino-cc-expands-european-manufacturing/)

罗马的创造者大会结束了，这意味着 Arduino 媒体闪电战的时间到了。Arduino 本周已经发布了一项重大公告，推出了由英特尔 Curie 模块驱动的 Arduino / Genuino 101 主板。团队。不过，cc 并没有忘记他们所有的 Atmel 供电板。最新消息是 Arduinos [将在德国由 Watterott Electronics](http://www.golem.de/news/genuino-offizielle-arduinos-jetzt-auch-aus-deutschland-und-italien-1510-116929.html) (。德，[谷歌翻译](https://translate.google.com/translate?sl=de&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fwww.golem.de%2Fnews%2Fgenuino-offizielle-arduinos-jetzt-auch-aus-deutschland-und-italien-1510-116929.html&edit-text=&act=url)。

目前，Arduino.cc 板在中国由 Seeed、[生产，在美国由 Adafruit](https://blog.adafruit.com/2015/05/16/big-news-adafruit-is-manufacturing-arduinos-for-arduino-cc-in-new-york-new-york-usa-arduino-teamarduinocc/) 和 Sparkfun 生产。Watterott Electronics 是德国最大的电子产品分销商之一。

Watterott 制造的板材将带有 Genuino 标志；Arduino.cc 似乎预计会在美国以外的 Arduino 与 Arduino 商标纠纷中败诉。所有在 Arduino.cc 许可下生产并在美国以外销售的主板都将带有 Genuino 商标，而为美国市场生产的主板将带有 Arduino 商标。有趣的是，这场 Arduino 对 Arduino 的分裂[始于一家前制造商](http://hackaday.com/2015/07/23/hackaday-interviews-federico-musto-of-arduino-srl/)，商标纠纷引发了一场鸡毛蒜皮的漩涡。无论如何，Watterott 公司生产的电路板的许可问题现在肯定已经解决了。新的制造合作伙伴保证为所有人提供更多 Arduinos。
