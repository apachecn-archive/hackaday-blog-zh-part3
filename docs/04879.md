# 谜题:什么金属导电，但不导热？

> 原文:[https://hack aday . com/2017/01/27/riddle-what-metal-conducts-electric-but-not-heat/](https://hackaday.com/2017/01/27/riddle-what-metal-conducts-electricity-but-not-heat/)

如果你答不出谜语，不要难过。金属导体通常导电和导热。通常情况下，这是真的，但能源部劳伦斯伯克利国家实验室和加州大学伯克利分校的研究人员发现，二氧化钒可以导电而不导热。

威德曼-弗朗兹定律表明，电的良导体也是热的良导体。二氧化钒不仅在 67 摄氏度(152 华氏度)时从绝缘体转变为导体，而且当它处于导电状态时，似乎也没有该定律预测的那么多热量。

有一些其他材料的导电性能比导热性能好，但这些材料只能在零度以下工作。研究人员发现，可归因于正常导体中电子随机运动的热传递在二氧化钒中减少，因为电子像移动的流体一样以规则的模式移动，而不是像粒子一样随机移动。

通过向二氧化钒中添加不同的材料，研究人员能够降低该材料成为导体的温度，并提高其导热能力。这允许为特定用途调整材料。

除了对材料如何工作提供新的见解，研究人员还希望这种材料能够在回收余热并将其转化为能源方面取得进展。同一个团队一直在寻找这种奇异材料的有趣用途，包括将其用作微型机器人肌肉。
