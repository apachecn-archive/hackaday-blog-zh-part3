# 利用网络摄像头和开源软件进行低成本眼球追踪

> 原文：<https://hackaday.com/2018/05/05/low-cost-eye-tracking-with-webcams-and-open-source-software/>

“你在看什么？”说错了方式，那些都可以是打架的话。但在心理学研究和用户体验测试等不同领域，实时了解人们在看什么是非常宝贵的。眼球追踪软件可以做到这一点，但通常会让家庭游戏玩家买不起。

或者曾经是。通过 20 美元的网络摄像头，[这个开源的眼球跟踪器](https://hackaday.io/project/153293-low-cost-open-source-eye-tracking)将让你看到人们是如何处理他们所看到的东西的。但是[约翰·埃文斯]的黑客日参赛作品不仅仅如此。大部分细节在下面的视频中，其中很大一部分是[John]用来赞美他的眼睛追踪器所用相机的优点，一个罗技 C270。的确如此——这款便宜且容易获得的相机开箱后就具有非凡的微距功能，这是一款将对准几毫米外眼球的相机的关键特性。尽管如此，[John]还是提供了 STL 文件，以便在需要其他焦距的情况下，安装到拆下的相机 PCB 上。

该项目的核心是他的 Jevons 相机浏览器，这是他编写的一个应用程序，可以同时控制和查看两个相机。该软件最初用于拾取和放置，可用于协调两个护目镜式摄像机的视图，一个向外看，一个聚焦在用户的眼睛上。来自相机 LED 的反射被拾取并用于判断眼睛的角度，覆盖应用于另一个相机的视图以显示用户正在看哪里。它看起来很准确，启动速度也很快。

我们认为这是一个伟大的项目，就像 2018 年黑客日奖第一轮[中的许多其他项目一样。你能想到一个很棒的基于眼球追踪的项目吗？这是你省钱的机会。](https://hackaday.com/2018/03/12/2018-hackaday-prize-build-hope-design-the-future/)
