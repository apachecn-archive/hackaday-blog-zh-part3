# 对森林的虚拟一瞥

> 原文：<https://hackaday.com/2016/10/29/a-virtual-glimpse-into-the-forest/>

在秋日漫步林中是一种令人惊叹的视觉体验。然而，它需要一个人住在森林附近。如果你是那些不知道的人之一，[Koen Hufkens]最近推出了[虚拟森林项目](http://virtualforest.io/index.html)——一种带你在落叶林中度过一天的虚拟现实体验。

首先，你不需要虚拟现实设备来观看风景。网络浏览器和大多数智能手机都能够显示 360 度图像。Raspberry Pi 2 控制的理光 Theta S 相机封装在一个玻璃灯罩中，并借助一些 PVC 管安装在标准的栅栏柱上。电力通过 Cat5e 电缆巧妙传输，还包括了电涌保护器以防雷击。根据你浏览网站的时间，你可能会看到一个黑屏，或者万花筒一样的颜色。

 [https://www.youtube.com/embed/j8mpH4STu2M?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/j8mpH4STu2M?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



该项目的目标是追踪东北森林季节变化的时间。如果你有兴趣建立一个类似的系统，构建和部件分解可以在[这里](http://virtualforest.io/faq.html)找到，而代码可以在 [Github](https://github.com/khufkens/photosphere) 上找到。如果 Theta S 相机的成本过高，那么你可以[制作自己的全景相机](http://hackaday.com/2016/10/09/cheap-360-degree-camera/)来保持身材。
