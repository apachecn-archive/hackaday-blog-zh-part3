# 浏览器中的 DOS 教育

> 原文:[https://hack aday . com/2016/02/05/a-dos-education-in-your-browser/](https://hackaday.com/2016/02/05/a-dos-education-in-your-browser/)

在 20 世纪 70 年代和 80 年代，我们中的许多人学会了在 Altairs、Commodores、Apple IIs 等机器上使用优秀的老式 BASIC 编程。在 80 年代的某个时候，运行 MSDOS 的 IBM 个人电脑是因为事实上的标准，但是启动 BASIC 并编写一个简单的小程序还是很容易的。当然，还有其他的程序，一些严肃的如 C 编译器，一些半严肃的如飞行模拟器，还有一些纯粹有趣的如 Wolfenstein 3D。

如果你阅读 Hackaday，你可能已经注意到很多人使用各种技术模拟旧电脑——包括旧的 MSDOS PCs 包括运行 DOSBox 或其他模拟器的 Raspberry PI 板。老实说，运行一些旧软件需要付出很多努力，对吗？你也可以在你的桌面上加载 DOS 模拟器。那就简单一点了，不过还是得找软件。但是如果你和我们一样懒，你可能会想去 archive.org 看看 [MSDOS 收藏。](https://archive.org/details/softwarelibrary_msdos&tab=collection)

该系列有超过 7000 本旧的 MSDOS 书籍，令人印象深刻。但令人着迷的是，它们都将在你的网络浏览器中运行。您只需点击两次鼠标，就可以运行 BASIC、Borland C、飞行模拟器，甚至是 Commander Keen。不过，要小心。某些按键(如 Control+C)可能在浏览器中不起作用。

如果你真的是铁杆，你甚至可以在你的浏览器中启动一些[旧版本的 Windows。或者，如果你想回到过去，试着在你的浏览器中模拟 DOS，然后在 DOS 下模拟 TRS-80。如果你想写老派的博客，甚至还有旧版的微软 Word 和 Wordperfect。](https://archive.org/details/msdos_win3_1)

如果你想做一些硬件黑客，你不必在浏览器中做这些。还有很多旧电脑你可以在你的浏览器中[模仿。甚至还有一个浏览器](http://hackaday.com/2015/09/28/roundup-retro-computers-in-your-browser/)中的 [Windows 95(见下面的视频)。](https://win95.ajf.me/)
