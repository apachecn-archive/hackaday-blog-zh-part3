# Hackaday 奖参赛作品:模拟硬币电池的 PCB

> 原文:[https://hack aday . com/2017/09/18/hack aday-prize-entry-a-PCB-to-emulate-coin-cells/](https://hackaday.com/2017/09/18/hackaday-prize-entry-a-pcb-to-emulate-coin-cells/)

由[bobricius]开发的[硬币电池仿真器 CR2016/CR2032](https://hackaday.io/project/26896) 解决了一些硬件开发人员没有意识到的问题:当在由几乎无处不在的 CR2016 或 CR2032 格式 3V 硬币电池供电的硬件上工作时，电源可能有点麻烦。要么在开发过程中根据需要为设备提供硬币电池，要么开发人员安装一些分线，从更方便的电源提供电力。

[bobricius]的解决方案是一个小型 PCB，它可以像电池一样插入大多数硬币电池座。它集成了一个微型 USB 连接器和一个 3V 调节器，可将 USB 用作外部电源。如果希望方便地测量电流消耗，该板还提供了鳄鱼夹连接点。这是一个有目的的工具，巧妙地利用 PCB 本身的物理形状作为功能的一个组成部分，很像[bobricius]的另一个项目: [Charlieplexed 7 段 LED 显示屏](https://hackaday.com/2016/08/23/charliplexed-7-segment-display-takes-advantage-of-pcb-manufacturers/)。
