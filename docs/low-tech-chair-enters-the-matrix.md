# 低技术含量的椅子进入矩阵

> 原文：<https://hackaday.com/2017/10/08/low-tech-chair-enters-the-matrix/>

这个视频展示了一个非常有趣的实验:将一个 Vive 追踪器粘在一把普通的椅子上，以便与它的虚拟现实对应物完美同步。结果呢？在 VR 中作为虚拟对象可见的椅子，但具有 1:1 的物理世界版本，占据相同的空间。这意味着，与任何其他虚拟物体不同，当用户沉浸在 VR 中时，可以看到、触摸、感受、移动和实际坐入这把椅子。

这个实验的目的似乎是为了虚拟地探索真实世界环境中的座位安排，并由 design studio [Agile Lens]开发了一个[剧院规划工具](http://www.agilelens.com/demos.html)。但我们想知道，将可以触摸和握住(或坐在)的物理对象与虚拟现实中的对应对象连接起来的想法是否有未实现的潜力。下面嵌入了椅子测试的视频演示。
