# 通过克里斯托弗·洛伊德认证的通量电容器支柱

> 原文：<https://hackaday.com/2017/04/05/flux-capacitor-prop-with-christopher-lloyds-stamp-of-approval/>

我们喜欢我们在 Hackaday 的道具，每当我们遇到来自未来粉丝的作品时，都很难抗拒展示它。在这种情况下，[Xyster101]正在展示他建造的 Doc Brown 的[通量电容器](http://imgur.com/a/MtHz4)。

[Xyster101]选择了胶合板箱，这比他花 125 美元买一个合适的电气箱要经济得多。在里面，有一些聪明的变通方法可以让这个看起来尽可能的接近原来的样子。丙烯酸棒和球被成型并粘合在一起，以复制三位一体的玻璃管，由孔锯切割的 3/4”胶合板模仿螺线管，钢棒被打磨成设备中心的三个点，火花塞线和香蕉连接器不起作用，但完成了外观。包括油漆，焊接和大量使用热胶来固定一切，建造阶段花费了大约 30 个小时。

led 有多种模式，由隐藏在盒子侧面管道下的 DIP 开关控制。箱子底部还有一个运动传感器，当你走过时，它会触发 led 闪烁。此外，如果你想进行时间旅行，有一个 9 伏插头可以让你在旅行的任何地方或任何时候炫耀它。休息之后，请观看构建视频。
