# 全 3D 打印吹雪机

> 原文:[https://hack aday . com/2016/12/30/full-3d-printed-snow-blower/](https://hackaday.com/2016/12/30/fully-3d-printed-snow-blower/)

对于任何生活在寒冷气候中的人来说，每年的大雪意味着要花很多时间铲车道和人行道。在一场小雪之后，铲雪可能看起来是浪费时间，而吹雪机可能是多余的。如果有一个快乐的中间地带，需要最少的努力；也许一个带 3D 打印吹雪机的遥控扫雪机会有用。

去年我们展示了这个项目的早期版本。今年的型号具有一个拖鞋离合器-结合了重型钢筋混凝土卡车的差速器-以防止如果你碰巧撞到任何岩石或冰块时损坏附件。在这次迭代中，刀片也更厚并且缺少齿，因为它们会抓住任何坚硬的东西并且经常粉碎刀片。由[ [世爵工作室](http://spykerworkshop.webs.com/) ](又名【The_Great_Moo】)设计的吹雪机安装在 RC 扫雪机的前面——它原本是用来充当犁的。看到吹雪机附件的运行，我们倾向于相信他可能会有所发现。
