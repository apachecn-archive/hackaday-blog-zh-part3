# Hackaday 奖参赛作品:自主皮艇

> 原文:[https://hack aday . com/2016/09/10/hack aday-prize-entry-autonomous-kayaks/](https://hackaday.com/2016/09/10/hackaday-prize-entry-autonomous-kayaks/)

[巴里]有一个探险计划。他将探索海岸线、水湾和其他浅水区，在这些地方，即使是小型遥控潜水器也太大了。[Barry]想要在这次探险中有一辆自动驾驶的车，这意味着他必须自己造一辆。这导致了朱利叶斯计划、重新设计的皮艇以及所有制造合适的海洋电子设备的技术。

[巴里]得到的第一个问题是，“为什么是皮划艇？”在 Craigslist 附近快速巡游就足以回答这个问题了——它们很便宜，而且几乎有无限多种，从大型旅游皮艇到小型游乐船，所有这些都是为不同的条件和探险而建造的。通过几个马达和模块化部件，[Barry]建造了一艘能适应各种条件的自主水上船只。

现在，[巴里]的重点是让推进器工作。这是以几个无刷电机栓在皮艇底部的形式出现的。一次系留测试非常成功，展示了这艘小船可以在一角硬币上转弯。与自动驾驶仪的集成即将到来，但在此之前，[巴里]有一艘整洁的小船在河边闲逛。你可以看看下面的视频。
