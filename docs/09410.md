# 通过数控仿古灯塔镜头

> 原文：<https://hackaday.com/2018/05/11/antique-lighthouse-lens-via-cnc/>

在大功率发光二极管发明之前，甚至在电灯被广泛采用之前，灯塔仍然有义务警告船只危险，同时引导他们进入各种安全的港口。他们用煤气灯和令人印象深刻的玻璃透镜(称为菲涅尔透镜)做到了这一点，菲涅尔透镜有助于将所有可用的光指向正确的方向，同时减少了传统透镜的重量和材料。

现在，佛罗里达州的一家公司正在使用丙烯酸树脂复制古董菲涅尔透镜。乍一看，丙烯酸似乎不是玻璃的最佳替代品，但该公司能够使用数控机器实现极高的精度，然后抛光和烘焙丙烯酸，使其透明，非常适合用于像这样的灯塔镜头。复制镜头由黄铜制成，镜头元件用一种特殊的粘合剂粘合在一起。这是一个令人信服的复制，值得在任何灯塔中使用。

一定要看看下面的视频，看看这些镜头是如何制作的，虽然我们不完全确定在抛光时镜头上到底喷了什么，但也许有人可以在评论区为我们解释一下。当然，菲涅尔透镜除了在灯塔中还有其他用途，我们已经看到了一些很好的例子，它们被用于许多不同的应用中。
