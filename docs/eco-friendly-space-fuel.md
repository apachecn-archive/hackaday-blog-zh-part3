# 生态友好型太空燃料

> 原文：<https://hackaday.com/2016/02/14/eco-friendly-space-fuel/>

如果你想在本周为了一个好的理由而冒险炸掉你的手指，看看[M. Bindhammer]寻找一种[环保火箭燃料](https://hackaday.io/project/5743-green-eutectic-rocket-propellant)就知道了。预测未来固体火箭助推器的使用会越来越多。我们对此感兴趣。目前，火箭发射如此之少，以至于污染没有增加，但当我们每天两次将消费电子产品运送到月球并返回时，我们可能会遇到问题。

最常见的固体火箭燃料在燃烧时会向大气中排放氯气。[Bindhammer]正在探索一种安全的方法来制造一种低共熔平衡且稳定的燃料，这种燃料由糖或糖醇以及硝酸钾组成。如果你像我们一样在周末观看家庭化学视频，那么[Bindhammer]会了解他所有的想法，甚至会详细说明在实验室安全复制他的燃料的过程。

他做了很多工作。得到的燃料是稳定的，可以是液体或固体。它的点火温度很高，但正如你在休息后的视频中看到的。一旦点燃。它就像火箭燃料一样爆炸。
