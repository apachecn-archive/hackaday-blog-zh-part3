# 复古旋转拉斯皮电话铃声 Alexa

> 原文:[https://hack aday . com/2016/04/23/retro-rotary-raspi-phone-rings-Alexa/](https://hackaday.com/2016/04/23/retro-rotary-raspi-phone-rings-alexa/)

是一个符合我们心意的人。他喜欢将老式设备的美感与新技术的实用性结合起来。他的最新项目是 AlexaPhone，它将亚马逊 Alexa 语音服务近乎即时的检索和计算能力与 20 世纪 70 年代旋转式手机的外观和感觉结合在一起。最重要的是，没有必要旋转转盘，然后等着它转回来。一旦拿起手机，AlexaPhone 就准备好回答问题。

问题通过插入 Pi 的废旧 USB VOIP 电话传输。用户必须挂断电话才能触发搜索。一旦 Alexa 有了答案，音频就会通过一个带 USB 充电电池的小型外部放大扬声器回来。由于硬件对 Alexa 来说有点不典型，[MisterM]起初试图用物理按钮查询服务时遇到了一点麻烦，直到他发现了这个 AlexaPi 代码。

这款手机实际上是一款经典 BT Trimphone 的复制品，这解释了表盘上的星号和八叉线。现代的内部机制意味着[MisterM]可以利用接收器挂钩上的带状电缆来触发 Pi 发送查询。休息过后，观看[MisterM]的孩子们测试 Alexa 的能力。
