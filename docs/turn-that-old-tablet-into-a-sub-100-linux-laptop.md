# 将旧平板电脑变成低于 100 美元的 Linux 笔记本电脑

> 原文：<https://hackaday.com/2016/11/05/turn-that-old-tablet-into-a-sub-100-linux-laptop/>

微型笔记本电脑一直以来都是有很多承诺的设备，但不知何故却无法兑现。从雅达利(Atari)的掌上电脑到最近的上网本，它们要么非常昂贵，要么受到尺寸限制的影响，几乎毫无用处。这些年来，我们已经看到了 DOS、EPOC、Windows、WinCE、Palm OS、Linux 发行版和更多的小尺寸产品，但很少有人取得显著的成绩。

然而，你手中的“合适的”电脑的前景还不能放弃。我们现在正在达到这样一个点，上一代更高端的 Android 平板电脑功能强大，数量足够多，价格非常合理。也许这些可以为小型笔记本电脑搜索者提供一些有用的东西的基础。[NODE]当然是这样认为的，因为他的[用二手 Nexus 7 平板电脑和蓝牙键盘外壳生产了一台漂亮的小型 Ubuntu 笔记本电脑](https://n-o-d-e.net/ubuntu_laptop_overview.html)。Android 被替换为 Ubuntu 图像，一个纸板切割的显示器边框用磁条固定在适当的位置。[一个循序渐进的指南](https://n-o-d-e.net/ubuntu_laptop_how.html)已经发布，以帮助其他有兴趣走同样道路的人。

这并不是最令人惊讶的硬件攻击，因为它主要涉及现成的产品和一个软件。然而，它值得一看，因为它确实提供了一条以极其合理的价格买到非常容易接受的小型 Linux 笔记本电脑的途径。一个担心是 Ubuntu 版本似乎不是最近的版本，但是我们确信读者会在评论中指出任何更新的发行版本。如果你想看看完成的笔记本电脑，他已经发布了一个视频，我们已经包括在休息。
