# 将 Pi 转变为 PDP

> 原文:[https://hackaday.com/2015/12/12/turning-a-pi-into-a-pdp/](https://hackaday.com/2015/12/12/turning-a-pi-into-a-pdp/)

学习如何给计算机编程，没有比汇编更好的方法了，而做汇编也没有比用一堆闪光灯和开关更好的方法了。因此，学习编程的最佳方式是使用 PDP-11。很遗憾这些机器被锁在博物馆和非常酷的人的车库里，但是你可以用树莓皮和一些额外的组件来建造你自己的 PDP-11。

[jonatron]用许多 led、大量开关和一些 16 位串行到并行 IC 构建了他自己的 PDP-11 模拟版本。当然，任何 blinkenlight 模拟器最酷的部分是前面板图形，这里[jonatron]没有吝啬。他把这些开关和 led 放在一块激光切割的丙烯酸树脂上，上面有一个漂亮的 PDP11 贴花。该软件带有大量的编译器警告，除了非常简单的机器码程序之外，不运行任何东西。不过，这确实是你用一堆拨动开关和灯所能做的全部事情。

如果这个项目看起来很熟悉，你的记忆没有欺骗你。[PiDP-8/I](https://hackaday.io/project/4434-pidp-8i)是今年 Hackaday 奖的参赛作品，最终成为最佳产品类别的顶级项目之一。今年我们遇到了 PiDP-8 的创造者[奥斯卡]几次。最近的一次是[在 Hackaday SuperConferece，他做了一个演讲](http://hackaday.com/2015/12/05/experiences-in-developing-an-electronics-kit/)。他目前正在制作 PDP 之王 PDP-11/70 的复制品。
