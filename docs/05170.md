# 语音到手语

> 原文:[https://hackaday.com/2017/03/02/speech-to-sign-language/](https://hackaday.com/2017/03/02/speech-to-sign-language/)

根据世界聋人联合会的统计，全世界大约有 7000 万人的第一语言是某种手语。在美国，说美国手语的人数从 50 万到 200 万。但是，如果你去谷歌翻译，没有手语选项。

[亚历克斯·福利]和朋友们决定做些什么。他们参加了 McHack(麦吉尔大学的黑客马拉松)，决定将语音转换成手语。他们以为他们已经准备好了，但结果是他们不得不匆忙地解决一些事情。(不都是这样吗？)但是最后他们还是占了上风，你可以在下面的视频中看到。

该项目的核心是一双 3D 打印的手。起初，他们不小心印出了两个左手。他们很快就打印出了右手，但后来他们发现少了一部分，这是他们用木头雕刻出来的。钓鱼线形成了肌腱，有足够的伺服需要两个 CPU 板来驱动一切。

语音识别来自 Nuance，如果仔细听视频，你会看到他们在打“你好”和“再见”我们必须相信他们的话，签字是正确的，清晰可辨的。

这让我们想起了 Hackaday 奖的手语手套。这双手可能不会推进[假肢的艺术状态](https://hackaday.com/2016/09/05/3d-printed-prosthetic-puts-the-power-in-the-hands-of-those-who-need-it/)，但这不是他们想要的。
