# FPV-罗孚 2.0 拥有 3D 打印胎面和大量拉链

> 原文：<https://hackaday.com/2018/06/17/fpv-rover-2-0-has-3d-printed-treads-and-plenty-of-zip/>

[Markus_p]已经完成了一个非常成功的 3D 打印履带式机器人。现在他已经完成了第二个，使用标准马达，并结合了他从第一个学到的东西。结果非常令人印象深刻你可以在下面看到这个怪兽的视频演示。

大部分机器人是解放军，虽然有一些部分使用 PETG 和柔性塑料。前面有一个红外摄像头，后面有另一个普通摄像头。所有的电子设备都是现成的模块，比如 FPV 发射器和电机的电子控制器。正如你在第二个视频中看到的，有一个伺服系统可以倾斜摄像机。

主体使用螺母和磁铁装配在一起。视频中的机器人受到了很好的打击，似乎没有崩溃，所以它必须足够。吸引我们的是这个东西的大小。看起来，在火星车内部或顶部安装一些处理能力非常容易，它可以为更复杂的机器人提供一个很好的运动基础。

当然，我们已经看到了一些类似的项目。这个[履带式机器人使用精神控制](https://hackaday.com/2017/01/20/brain-controlled-tracked-robot/)。并且[开式轮子](https://hackaday.com/2016/11/28/parametric-3d-printable-wheels-and-treads/)是一个获得踏板和其他运动设计的好地方。
