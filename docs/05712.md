# Hackaday 奖品入口:一个双动键盘

> 原文：<https://hackaday.com/2017/04/28/hackaday-prize-entry-a-double-action-keyboard/>

机械键盘是目前最流行的东西，打造你自己的键盘至少是书呆子信誉的两个额外层次。[这个参赛的项目](https://hackaday.io/project/20665-double-action-keyboard)，是一个你从未见过的 DIY 键盘。这是对电脑键盘工作原理的根本性转变。这是一个双动键盘。轻按一个键，一个字符就会出现在屏幕上。使劲按，屏幕上会出现一个不同的字符。你以前从没见过这样的东西。

[Jaakob]设计了这种键盘，以便每个键帽下面都有两个开关。他采用普通的“ol Cherry MX”开关并对它们进行了修改，这样当开关被完全按下时，“柱塞”就会伸出开关的底部。这些 Cherry 开关安装在一块 perfboard 上，一个小的轻触开关焊接在下面。这是一个类似于触敏 MIDI 控制器或其他类型键盘的想法。这里的区别是，它不是使用两个开关来感应按键的力度，而是映射到两个不同的功能。

一旦[Jaakob]发现如何将两个开关放在一个键帽下，他就连接一个矩阵，附上一个 Teensy，[开始破解固件](https://github.com/jlidauer/DAK)。构建还没有完全完成，但这是我们近年来见过的最具创新性的 DIY 键盘之一。这里有很大的潜力，这种将两个开关结合在一起的方法仍然允许机械开关的奇妙咔嗒声和良好感觉。
