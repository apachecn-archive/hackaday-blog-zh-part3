# 活着走过核磁共振成像

> 原文：<https://hackaday.com/2018/06/24/walking-through-mris-with-a-vive/>

如果你要列出过去 100 年最重要的技术成就，先进的医学成像可能会排在最前面。看到身体内部细节的能力几乎是奇迹，在某些情况下甚至是救命的。

然而，在从磁共振成像仪(MRI)等设备流出的大量数据流所产生的虚拟人体中导航可能是一个挑战。这款直观的 MRI 切片器旨在改变这种情况，让人体的 3D 漫游变得非常简单。[Shachar "Vice" Weis]没有提供关于该系统的大量细节，但从我们可以收集的信息来看，控制器是基于平板电脑和 Vive 追踪器的。Vive 附着在平板电脑的背面，可以检测它在空间中的位置。然后，将平板的平面解释为切片平面，用于正在研究的结构的 3D 重建。下面的视频显示它正在探索一个人类头部扫描；更新速度令人难以置信，没有明显的滞后。[Vice]说这是 0.1 版本，所以我们期待它能带来更多。明显的功能将是通过平板手势放大和缩小的能力，以及在空间中旋转 3D 模型以从其他角度查看模型的方法。

对制作这些图像的机器是如何工作的感兴趣？我们之前已经介绍过核磁共振扫描仪的基础知识。如果你想走得更远，你可以随时[建造自己的](http://hackaday.com/2016/06/30/hackaday-prize-entry-an-mri-machine/)。
