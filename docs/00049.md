# 用于 3D 打印机的气凝胶绝缘材料

> 原文:[https://hack aday . com/2015/09/09/气凝胶-绝缘体-用于 3d 打印机/](https://hackaday.com/2015/09/09/aerogel-insulation-for-3d-printers/)

加热床对于 ABS 印刷几乎是必不可少的。没有它，当塑料冷却时，很难防止零件翘曲。然而，加热大型印刷床既困难又耗时。的确，打印机很容易将热端加热到 200 摄氏度或更高，而床的温度只有一半。然而，热端是一个小的绝缘点，而床是一个大的平面。加热床并保持温度稳定需要大量的能量和时间。

我们用过软木，甚至 Reflectix，效果都很好。然而，[比尔·格茨]从传统材料中得不到他想要的性能，所以他得到了一块气凝胶，并把它用作绝缘材料。气凝胶材料是一种凝胶，其中气体取代了凝胶的液体部分。由于[努森效应](https://en.wikipedia.org/wiki/Aerogel)，气凝胶的绝缘性能可能比它所含的气体更强。

由于测量不准，[Bill]不得不用业余爱好刀加工凝胶。气凝胶是唯一需要的外来材料。一点 Kapton 胶带(这在今天看起来并不稀奇)、剪刀、白板笔和业余爱好刀就搞定了。

我们已经介绍了如何[制作自己的气凝胶](http://hackaday.com/2011/11/16/making-aerogel-at-home/)，尽管它[需要一些奇特的材料和设备](http://hackaday.com/2008/03/23/make-your-own-aerogel/)。互联网和 YouTube 上有很多食谱(见下文)。
