# 一个功能齐全的业余无线电台

> 原文：<https://hackaday.com/2017/09/09/a-ham-radio-go-box-packed-with-functionality/>

"当其他方法都失败时，还有业余无线电."随着飓风哈维刚刚结束，厄玛本周末排队痛击佛罗里达州，哈姆正在准备为那些没有任何通信基础设施的地区提供灾难通信。对于火腿来说，最完美的东西就是这个装在盒子里的火腿小屋。

自从有了 hams 以来，他们所知道的 go-box 就一直是业余无线电现场操作的主要部分。[Fuzz (KC3JGB)]推出的 go-box 绝对是一个完美的 EmComm 平台。下面的视频游览是我们必须继续下去，但我们可以看到一个三频收发器，一个 RTL-SDR 加密狗和一个带 TFT 屏幕的树莓 Pi，用于跟踪卫星。Pi 和 SDR 也可能是 NOAA 卫星接收器的一部分，就像【Fuzz】在[的一个独立视频](https://www.youtube.com/watch?v=ddOgScS0Pwc)中描述的那样；这种装置在应对自然灾害方面非常有价值。一切都由一个 12 伏的电池供电，电池可以通过一个小型太阳能电池板充电。

[Fuzz]准备行动，虽然我们真诚地希望他和其他火腿在佛罗里达州不会被需要，但在这一点上似乎不太可能。你可以阅读更多关于业余无线电台的公共服务形象的文章，或者关于 T2 的更有能力的电台。
