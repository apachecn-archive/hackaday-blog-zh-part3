# 谢妮计时器在厨房的另一边很容易看到

> 原文：<https://hackaday.com/2016/04/22/nixie-timer-is-easy-to-read-across-the-kitchen/>

谢妮钟表。谢妮功率计。谢妮温度计、速度计，甚至是个人装饰品的数码管。有什么没有被谢妮化的吗？谢妮厨房计时器怎么样？除了数码管，这是一个伟大的建设。请看休息时间下面的视频。

正如谢妮爱好者经常发生的那样，Kuroi kou ichi 从电子管开始，并寻找一个项目来使用它们。一个不可靠的厨房定时器为建造提供了几乎不加掩饰的借口——毕竟，任何人都可以花几日元买一个商业替代品，对吗？计时器的特点是四个 12 英寸的电子管和一个大型数字键盘，位于激光切割的丙烯酸外壳上。对于那些对键盘的美学和谢妮厨房环境项目的智慧吹毛求疵的人，[Ko]指出，一个 IP65 键盘的价格将是该产品的两倍多，一点常识就可以大大防止高压侧接触任何潮湿的东西。除了倒计时功能，计时器还可以充当秒表并显示一天中的时间，与七段 LCD 计时器相比，谢妮管提供了更好的可视性。

关于前面提到的谢妮项目，这里有一个我们之前介绍过的[时钟](http://hackaday.com/2015/07/14/unusual-nixie-tubes-lead-to-unique-clock/)、[电表](http://hackaday.com/2016/01/04/nixie-tube-energy-meter-dresses-up-front-hall/)、[温度计](http://hackaday.com/2016/02/21/nixie-thermometer-destined-for-custom-pc-case/)、[速度表](http://hackaday.com/2015/07/16/nixie-tube-speedometer-in-motorcycle-handlebars/)和[项链](http://hackaday.com/2014/01/17/nixie-ify-me-necklace/)。日本人的下一步是什么？我们不知道，但我们很想看看你有什么发现。
