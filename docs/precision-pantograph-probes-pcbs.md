# 精密缩放探针 PCB

> 原文：<https://hackaday.com/2017/10/04/precision-pantograph-probes-pcbs/>

电子元件变得越来越小，对我们大多数人来说，我们的视力越来越差。当[Kurt]开始使用显微镜来更好地观察他的工作时，他意识到他需要另一种工具来赋予他的手同样的精度。那个工具不存在，所以他制造了它。

PantoProbe 是一种缩放机制，用于引导探针到达其 SMT 组件的微小焊盘。他报告说，他不再有任何困难区分 0.5 毫米的针，这是我们最喜欢的自动铅笔中石墨棒的直径。

[Kurt]已经扩展了他的机器的功能，包括一个高频探头的支架，甚至还有用于取放变化的滑轮。没有提到双用途的 PantoProbes 是微型助手，但我们看到的多功能性表明这只是一个时间问题。

四连杆机构能够做出一些令人难以置信的壮举，它们在我们周围随处可见。在他的 [Plexitube 猫头鹰时钟](http://hackaday.com/2016/04/24/plexitube-owl-clock-watches-you-sleep/)中享受【Kurt】的其他定制 PCB，或者让他向你展示用激光雕刻机制作 [3D 物体。](http://hackaday.com/2015/09/02/3d-objects-from-a-laser-cutter/)
