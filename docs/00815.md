# 当你的饮料准备好了，智能杯垫会通知你

> 原文：<https://hackaday.com/2015/12/04/smart-coaster-informs-you-when-your-drink-is-ready/>

有多少次你泡了一杯咖啡或茶，但它太热了，不能喝，然后当你到了它的时候，它变得太冷了？虽然这在很大程度上是一个# firstworldproblem 的问题，但[tonithaex]决定解决它，并同时参加一场比赛。他称之为[杯垫](https://hackaday.io/project/8610-coaster)，本质上，它是一个温度感应闹钟。

杯垫使用 TMP006 红外温度传感器，间接测量你放在杯垫上的任何东西的温度。它也用不了多少电。他围绕这一点设计了 PCB，并创造了一个非常好看的杯垫，由纽扣电池供电——他认为每天使用可以持续 6 个月左右——尽管这取决于你对咖啡因的上瘾程度。

为了设置杯垫，一条 LED 灯用二进制显示温度(因为为什么不呢？)，并且您可以将其设置为您想要的任何值——嗯，8 位 LED 可以显示的最大值。该设备使用电容按钮，这增加了[托尼]追求的相当光滑的设计美感。如果你喜欢这个想法，他已经参加了[硅实验室低功耗竞赛](http://community.silabs.com/t5/Contests/Low-Power-Contest/m-p/153186#U153186)，他会喜欢一些支持。
