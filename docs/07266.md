# Hackaday 奖参赛作品:经济型仿生腿

> 原文:[https://hack aday . com/2017/10/06/hack aday-prize-entry-economic-bionic-leg/](https://hackaday.com/2017/10/06/hackaday-prize-entry-economical-bionic-leg/)

当谈到用于截肢者的高科技仿生腿时，所有酷的东西都是钛、碳纤维或其他更奇特的材料。随着碳纤维“叶片”风靡一时，人们很容易忘记更简单的技术仍然有效，并且可以通过添加一些廉价的电子设备来更好地工作。经济型仿生腿项目就是这个想法的结果。

项目创建者[PremJ20]和[G.Vignesh]并没有开玩笑地降低这些仿生腿的成本。目标是 60 美元一个，不锈钢和硅橡胶作为碳纤维的廉价替代品——橡胶将被模制成适合截肢部位。该项目的核心是一个粒子光子开发板，带有一个 flex 传感器和加速度计来监控假肢并向云提供数据。它本质上是一个内置监控系统的基本假肢。在普通腿上放置一个传感器袖带，假肢的灵活性可以微调，以匹配这两者。

这种廉价的仿生腿会像 2012 年伦敦奥运会上的[奥斯卡·皮斯托瑞斯]一样参加奥运会比赛吗？可能不会——用于假肢的技术与 F1 赛车和涡轮喷气发动机的设计一样，包含了大量的材料科学。尽管如此，这是一种非常廉价的方式，将技术引入迫切需要更便宜的东西，而且这是一个很好的 Hackaday 奖参赛作品。
