# FPGA 制作 ASCII 视频

> 原文:[https://hackaday.com/2018/03/05/fpga-makes-ascii-video/](https://hackaday.com/2018/03/05/fpga-makes-ascii-video/)

人类喜欢图片，这可能就是为什么有一句古老的谚语“一张图片胜过千言万语。”我们现在认为计算机图形输出是理所当然的，但即使在电传打字机和行式打印机的早期，也有从史努比到斯波克等角色制作的艺术品。[文婷 Z]延续了这一传统，创造了一种将 VGA 视频转换为 ASCII 艺术的 FPGA，并通过 DVI 输出。

该设备使用 Xilinx Virtex 设备，并使用大约 500 个 LUT(查找表)，这根本不多。您可以在下面看到该设备的运行视频(包括源视频的叠加)。

事实上，我们认为像这样的艺术是一种计算机现象，但[弗洛拉·斯泰西]在 1898 年在打字机上创造了一只蝴蝶，业余无线电爱好者在二十世纪后半叶用纸带进行艺术创作。甚至在那之前，在 1865 年，《爱丽丝梦游仙境》中有一段文字被排版成老鼠的尾巴。也许顶峰是著名的《星球大战》的 ASCII 版本。

与我们见过的其他一些 ASCII 艺术项目相比，这显然没有那么机械。如果你喜欢更多的文字艺术，看看[其他一些例子](https://hackaday.com/2016/06/28/retrotechtacular-ascii-art-in-the-19th-century/)，包括一个非常古老的使用文字艺术的广告
