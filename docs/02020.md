# 自制笔式绘图仪

> 原文:[https://hackaday.com/2016/04/23/home-made-pen-plotter/](https://hackaday.com/2016/04/23/home-made-pen-plotter/)

作为一个在上个世纪开始使用电脑的人，我发现当前笔式绘图仪的复兴有些怀旧。当然，不同之处在于，本世纪创造自己的世界更容易，这正是[米格尔·桑切斯]正在做的。

受到 Axidraw 的启发，他正在制作自己的笔式绘图仪。到目前为止，他已经取得了很大的进步，创造了一个看起来很容易建造的设计。他的设计是由一个 Arduino Uno 驱动的，它有一个步进屏蔽，几个 NEMA 17 步进电机和一个伺服系统来升降笔。再加上几根杆子、一两条皮带和一些 3D 打印部件，你就有了一个看起来不错的笔式绘图仪。

他最初从激光切割组件开始，但随着设计的发展，他转向了 3D 打印。它没有我年轻时用惠普笔式绘图仪用巨大的字母打印出粗鲁的话那么花哨(我想是一台 [HP7475](http://softsolder.com/2015/04/20/hp-7475a-plotter-rehabilitation/) )，但它是一个整洁的构造。在下面的视频中检查它的运行情况。
