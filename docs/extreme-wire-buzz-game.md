# 极限电线嗡嗡声游戏

> 原文：<https://hackaday.com/2016/01/05/extreme-wire-buzz-game/>

还记得那个古老的巴斯电线游戏吗？有点像操作，你必须在不接触墙壁的情况下完成一项灵巧的任务……这里有一个有趣的转折——如果你把一把[400 万伏的眩晕枪扔进混合物中会怎么样？](http://imgur.com/a/swOP3)

没错，[迈克]得到了一个泰瑟枪手电筒，他想出了一个绝妙的主意，用它来做一个游戏。这款游戏有三个金属线部分，越来越硬，风险也越来越高！使用手柄，您必须沿着电线部分引导吊环螺栓。但是要小心——电路是带电的，如果你接触到金属，你会触电的！

为了确保人们不会受到伤害，或者继续电击自己，你用左手按住一个按钮。这样当你痛得抽搐时，你会切断电路。

第一节取 1.3V，用变压器升压，给你一个电击。第二段 2.6V，第三段— 5V。在第三也是最后一部分，如果你不小心，你会受到高达 400 万伏的电击。
