# GameBoy 原创中的 Raspberry Pi 计算模块 3

> 原文:[https://hack aday . com/2017/11/17/raspberry-pi-compute-module-3-in-a-game boy-original/](https://hackaday.com/2017/11/17/raspberry-pi-compute-module-3-in-a-gameboy-original/)

[Kite]长期以来一直在为 GameBoys 制作定制 PCB。事实上，很长一段时间以来，其他人已经使用他的工作来构建功能更加丰富的 GameBoy 平台。不幸的是，他们的一些工作停滞不前，所以[Kite]捡起来，完成了一个新项目:[一个 GameBoy，它使用了运行在他升级的 GameBoy PCB](http://www.sudomod.com/forum/viewtopic.php?f=38&t=4351) 上的 Raspberry Pi。

构建的核心使用了一个 Raspberry Pi 3，但它已经被缩小为内存模块的形状，称为计算模块 3。(我们之前以[英寸]展示了[的原始版本](https://hackaday.com/2017/06/02/game-boy-mod-uses-raspberry-pi-compute-module-3/)，但[Kite]从那以后接管了它。)升级释放了 GameBoy 外壳中的宝贵空间，以适合[Kite]最初构建的定制 PCB，也消除了切割一个 Raspberry Pi 并将其焊接到他的旧版本 PCB 的需要。建筑很干净，跑起来像个冠军。它还有一些附加功能，例如具有 HDMI 输出。

对于任何人来说，寻找复古游戏机的感觉，但谁想要重要的升级，如背光彩色屏幕，或玩 PSP 游戏的能力，这可能是你的建设。下面的视频详细介绍了这一切是如何组合在一起的。不过，如果你想在你的 GameBoy hacks 中寻找更多的挑战，那么还有一个持续的挑战就是[尽可能制造最小的 game boy](https://hackaday.com/2016/11/28/tiniest-game-boy-hides-in-your-pocket/)。
