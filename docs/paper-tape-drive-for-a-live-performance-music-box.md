# 用于现场演奏音乐盒的纸带驱动器

> 原文：<https://hackaday.com/2016/07/06/paper-tape-drive-for-a-live-performance-music-box/>

音乐对我们有些人来说是个谜。当然，当我们听到它时，我们知道我们喜欢什么，但是实际上能够制作它的想法使我们困惑。能够建造新的乐器来创造它的想法，就像这个纸带可编程音乐盒一样，令人印象深刻。

毫无疑问，你会记得“Wintergatan”乐队的[马丁·莫林]和他令人震惊的[大理石疯狂音乐机](http://hackaday.com/2016/03/03/incredible-marble-music-machine/)。这个乐器的规模要小得多，以一个[现成的纸带音乐盒](https://www.amazon.com/Kikkerland-Make-Your-Own-Music/dp/B000HAUEFY)为中心。但是廉价的塑料驱动齿轮在性能条件下总是失效，所以[马丁]去了似乎是他的洞穴车间，开始研磨。他用 Lego technologies 制作了一个新的纸驱动器的原型，当它工作时，需要有人帮忙拉纸。接下来是一个迭代的设计过程，最终形成了一个塑料和金属技术部件的混合体，可以可靠地驱动纸张，以及一个不仅仅是一个小发条音乐盒的乐器。下面听听另一种新乐器“调音琴”的演奏，它听起来有点像[的特雷门琴](http://hackaday.com/2016/01/24/finally-a-modern-theremin/)，但看起来就像——啊，看看视频吧。

构建视频暗示了更多的细节，我们希望有一个完整的大理石机器系列。我们也很想看到关于 Modulin 的细节——如果曾经有过被黑的乐器，那就是它了。
