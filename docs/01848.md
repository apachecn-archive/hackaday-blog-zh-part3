# 用于等离子切割机的电子罗盘

> 原文：<https://hackaday.com/2016/03/22/electric-protractor-for-a-plasma-cutter/>

如果你是一个创客空间或个人，幸运地拥有一个等离子切割机，这个电动~~量角器~~指南针可能会很方便。[MakeItExtreme]的工作人员制造了这个[圆形切割工具](http://www.makeitextreme.com/en/newsroom/entry/VIDEOS/plasma)来帮助使用他们的等离子切割机在厚金属板上切割圆形和环形。

整个事情是围绕一个电磁铁，所以夹具将只与磁性金属。没有太多的设计细节，但通过他们博客上的视频和照片，可以推断出如何建造一个。一路上有几个不错的黑客。由于电磁铁是固定的，而夹具的其余部分是旋转的，主安装螺栓上钻了一个孔，以帮助电缆布线。旋转~~量角器~~臂由一块铝板制成，将所有其他部件固定在一起——驱动电机、中心轮毂和等离子头。使用的电机似乎是 60rpm 交流同步电机。这些类型通常在两个线圈之间有一个 RC 相移网络，以允许方向反转。摩擦驱动用于旋转夹具，摩擦力来自一对连接到电磁体和电机驱动轮毂的橡胶管带。等离子头支架有一个带有滚柱轴承的杆端，作为脚轮，确保在夹具旋转时保持电弧间隙。启动电磁、电机正转/反转和等离子启动的几个开关完成设置。
