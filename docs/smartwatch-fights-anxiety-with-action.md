# 智能手表用行动对抗焦虑

> 原文：<https://hackaday.com/2018/05/07/smartwatch-fights-anxiety-with-action/>

在我们快节奏的现代世界，难怪这么多人遭受焦虑和恐慌的袭击。有几种陈旧的技术可以处理这些攻击的症状。但正如任何遭受过这种攻击的人可以告诉你的那样，很难感觉到攻击的来临，直到为时已晚。到那时，理性思维已经被侵入性思维所取代。对于今年的 Hackaday 奖，[Austin Marandos]正在尽自己的一份力量，利用技术来帮助我们在担心毁掉自己之前检查自己。

存在类似的智能手表来检测即将到来的攻击，但它们不做任何事情来打击它们。Minder 就像是把一个朋友绑在你的手腕上，他从来不关心自己的问题。无论付出什么代价，它都想提供帮助，这就是为什么它有多种技术可以让你回到平静的状态。

Minder 的大脑是一口大小的 Qduino Mini，它非常适合拥挤的可穿戴设备，因为它内置了充电电路。它使用心率和温度传感器来确定恐慌发作的开始，并使用振动马达来提醒用户。马达也在放松技巧中发挥作用，以保持用户的注意力和控制力。利用即将到来的休息时间放松一下，看看视频。

如果你的焦虑源于不充分的感觉，那可能是骗子综合症。
