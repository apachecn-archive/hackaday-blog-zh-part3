# (机器人)身体电

> 原文:[https://hackaday.com/2017/09/04/the-robot-body-electric/](https://hackaday.com/2017/09/04/the-robot-body-electric/)

如果你处理电子产品，你可能会认为静电是一件坏事。它会炸毁 MOSFETs 和 IC，我们竭尽全力防止这种破坏。但一家名为 Grabit 的初创公司正在利用静电来让机器人操纵现实世界。特别是，耐克正在使用这些机器人[来制造鞋子](https://www.bloomberg.com/news/articles/2017-08-30/these-robots-are-using-static-electricity-to-make-nikes)。你可以在下面看到一个演示视频。

传统机器人使用类似人类的手或爪形抓具来模仿人类处理材料的方式。但 Grabit 拥有多项电粘附专利。最初的重点是爬壁机器人，但真正的回报是在制造机器人方面，因为静电机器人可以做机械手远不能复制的事情。

制鞋机器人和人类伙伴一起工作。人类可以在大约 10 或 20 分钟内安装一只鞋，而机器/人类伙伴可以在大约一分钟内完成。软件向操作者展示如何将材料放在玻璃台上。一个覆盖着电子粘着垫的平台拿起所有东西。当在热压机中处于正确位置时，电荷关闭，释放材料。

因为这项技术获得了专利，所以应该很容易发现它的工作原理是什么。无论如何，基本的操作似乎很明显。我们很有兴趣看看黑客社区能用静电机器人做些什么。
