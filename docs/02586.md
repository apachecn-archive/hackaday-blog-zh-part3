# 每个教室都有空气质量传感器

> 原文：<https://hackaday.com/2016/06/18/air-quality-sensors-in-every-classroom/>

有抱负的爱好者的第一个电子项目是将某种传感器连接到微控制器，然后用新信息做一些有用的事情。[布洛克]已经采取了这种类型的网关项目，并把它变成一种让他的学生参与和熟悉电子学的方法。他对空气质量测量仪的研究实现了这两个目标，并有望帮助他所有的学生成为下一代黑客。

材料清单非常简单。Brock 没有使用 Arduino，而是使用了粒子光子，这种粒子光子具有各种无线连接选项的额外好处。空气质量传感器是 Shinyei PP42ns，它很容易与光子连接。大多数公立高中(至少在美国)可能无法接触到的唯一东西是 3D 打印的外壳，尽管如果你有机会接触到它，[Brock]会将文件放在项目页面上，这样任何人都可以使用它们。

当然，我们是让学生参与标准化考试之外的任何事情的项目的忠实粉丝，而且这个项目在教会学生如何通过考试之外还有很长的路要走。如果你想自己尝试，项目页面上有许多视频和说明，但是如果材料的成本是唯一让你不敢在自己的教室里做这件事的原因，还有一些其他的选择。你可以使用 ATtiny 芯片，或者尝试不同风格的传感器，或者尝试一个完全不同的项目。
