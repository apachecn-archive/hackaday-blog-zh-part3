# 布朗博士的安全公文包需要速度

> 原文：<https://hackaday.com/2015/11/24/doc-browns-security-briefcase/>

如果你只是等待足够长的时间，未来就会成为过去。这就是发生在“回到未来”的未来，你们可能都记得。但是 BttF 主题的项目还是源源不断。

[ossum]给我们发来了他制作的 [Doc Brown 公文包](http://www.instructables.com/id/Doc-Browns-Security-Briefcase/?ALLSTEPS)的链接，该公文包只能在 88 英里/小时以上打开。他的文章非常详细，如果你有兴趣使用 GPS 单元和微控制器，用移位寄存器驱动七段 led，或者只是驾驶一辆旧捷达开得太快，那么值得一看。如果你不是信徒，在休息时间下面有一个视频演示。
