# 欢迎来到树脂铸造公园

> 原文:[https://hackaday.com/2017/03/24/welcome-to-resin-cast-park/](https://hackaday.com/2017/03/24/welcome-to-resin-cast-park/)

从电子恐龙到[杰夫·高布伦]的人造胸毛，*《侏罗纪公园》*以其实用效果和道具而闻名。虽然它不像会呼吸的三角龙那么奇特，但 YouTube 的树脂铸造之神重新创造了这部电影中更可爱的道具之一。[彼得·布朗]和[袖珍 83]【T2]用约翰·哈蒙德携带的琥珀色手杖做了一个复制品，他花了两年时间才完成。

《侏罗纪公园》中的“琥珀中的蚊子”手杖道具正如你所想的那样——一只被困在一亿年前的琥珀中的大蚊子。当然，找到这样一大块带有蚊子的琥珀要花一大笔钱，所以[彼得]转向聚氨酯树脂。这块树脂被浇铸成两半，中间嵌有一只“食蚊者”(或称鹤蝇)。[彼得]花了两年时间铸造这块琥珀，但实际上，除了两个星期，其他时间都在等待一些成年鹤蝇出现。

用树脂包裹一只虫子，这个项目就交给了[Pocket83]，他在他的车床上转动手杖。除了在一块木头的中心钻一个三英尺长的洞之外，这一部分没有太多的制作，尽管最后的处理确实让这根手杖看起来很壮观。

对鹤蝇繁殖季节的漫长等待是值得的。这是《侏罗纪公园》中最好看的功能道具之一。你可以看看下面这个版本的视频。
