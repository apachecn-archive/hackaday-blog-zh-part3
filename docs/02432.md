# FriendlyARM:不同风味的覆盆子

> 原文：<https://hackaday.com/2016/05/27/friendlyarm-a-different-flavor-of-raspberry/>

很多老科幻电影都展示了人们穿着相同或者几乎相同的衣服。我们只能猜测，这是因为只有一个集中的工厂大规模生产紧身连衣裤，还是因为每个人都被命令穿得一样。既然我们生活在过去的未来，看起来科幻小说并不能很好地预测时尚。人们想要多样化。

这让人想起了开发板。我们*需要*多少个不同的？需求并不重要，因为我们有足够的需求。可能会有强有力的领导者:在 8 位世界中，你会想到 Arduino，而在 Linux 方面，可能会想到 Raspberry Pi。但还是有选择的。

[Eric Brown]最近[比较了 FriendlyARM](http://hackerboards.com/open-spec-octa-core-nanopi-m3-sbc-sells-for-35/) 的几款廉价开发板，包括 NanoPi M3、NanoPi M1 和 NanoPC-T3。这些价格从 11 美元到 60 美元不等，M3 售价 35 美元。在下面的视频中，你可以看到 M1 在 HDMI 屏幕上启动。

特别是 35 美元的棋盘(M3)，令人印象深刻:

*   处理器—三星 s5p 6818(8x 28 纳米 Cortex-A53 内核@ 400MHz 至 1.46GHzMali-400MP GPU
*   内存—1GB DDR 3 SDRAM；microSD 插槽(最高 64GB)
*   无线—802.11 b/g/n；蓝牙 4.0 双模；IPX 接口
*   网络— 10/100/1000 以太网端口
*   多媒体 I/O: HDMI 1.4a 输出；LCD 接口；LVDS 接口；DVP 相机接口；3.5mm 音频插孔；I2S 音频接口
*   其他 I/O:2 个 USB 2.0 主机端口；2 个 USB 2.0 主机接头；Micro-USB 2.0 客户端端口；调试串口头；40 引脚、兼容 Raspberry Pi 的 GPIO 连接器，用于 UART、SPI、I2C、PWM 等。
*   其他功能—电源和复位按钮；电源和状态指示灯
*   电源— DC 筒式千斤顶；+5V @ 2A；RTC 电池头；AXP228 电源管理单元
*   尺寸—64 毫米 x 60mm 毫米

该板可以启动几种 Linux 版本和 Android。这看起来是一个强有力的选择。

网格穹顶没有取代传统的住宅，紧身衣也没有取代西装。这些可能也不会取代树莓酱。我们已经报道了过去中的其他“圆周率杀手”，它们要么在价格上竞争，要么在[功能上竞争](http://hackaday.com/2016/03/02/odroid-c2-bests-raspberry-pi-3-in-several-ways/)。虽然它们都不可能取代圆周率，但如果你敢于与众不同，它们确实给了你选择。
