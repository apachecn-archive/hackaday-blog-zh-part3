# 控制音量

> 原文：<https://hackaday.com/2017/07/22/control-the-volume/>

对于任何一个拥有一个音箱或者一个旧的卡带播放器的人来说，数字时代的音量控制令人难以置信的尴尬。不停地按按钮来调节音量会让人很快厌倦。音量旋钮很有意义，在一个简单的项目中，[杰里米·S·库克]为我们带来了[定制电脑音量控制旋钮](https://youtu.be/uRSFS2DFRyA)。

这个建筑采用了一个 Adafruit 饰品板，配有一个旋转编码器和一个按钮，正如设计师们自己描述的那样。我们联系了[杰里米·S·库克]询问这一版本，结果发现他的版本使用了 MDF 外壳和 MDF 旋钮。一个较大的 PCB 上有编码器和按钮焊料，饰品板通过多股电线连接到它们。一块按尺寸切割的丙烯酸板作为顶盖，完成了整个建筑。

该按钮用作播放/暂停按钮，可以派上用场。由于该设备枚举为 HMI 设备，它应该可以与几乎任何操作系统一起工作。它可以很容易地扩展到 Android 平板电脑甚至 iPads 上。看看下面的视频演示，如果你喜欢定制输入设备的想法，看看这个 [DIY 快捷键](http://hackaday.com/2017/06/19/diy-shortcut-keyboard/)。
