# 机械漫威用勇气换巧克力

> 原文：<https://hackaday.com/2017/10/30/mechanical-marvel-trades-courage-for-chocolate/>

当我们看到[杰森·阿莱曼]用乐高做的事情时，我们会想为什么更多的一次性玩具没有用这种方式制作出来。这一次，他制造了一个万圣节机械奇迹，肯定会比他们在巡逻时遇到的任何东西都更让孩子们害怕——以至于许多人甚至会拒绝它分发的巧克力。谁不想得到它呢？你必须伸出一只电子骨骼手，它可能会抓住你，而一只类似的机械化蜘蛛可能会扑向你的手。

巧克力分配、手和蜘蛛都是用四个电机、一个控制它们的乐高思维风暴 EV3 砖和一个触摸传感器制作的。当一个孩子按下一个贴在触摸传感器上的南瓜时，下一颗巧克力糖就会在重力的作用下落到传送带上，并被送到等待的孩子手中。这是自然而然的事。在[Jason]和他的伙伴[Kristal]的判断下，使用红外遥控器和传感器，他们可以在恰当的时候激活骷髅手和扑击蜘蛛。我们只是不确定他们会选择放过谁。毕竟这是万圣节，害怕是乐趣的一部分，所以也许不放过任何人？看看下面的视频，告诉我们你是喜欢只吃零食，还是两者都吃。

我们确实想知道是否有任何项目不能从乐高产品中受益，即使只是在原型阶段或帮助可视化一个想法。作为一个小样本，【杰森】还做了一个[遥控单轮](https://hackaday.com/2016/07/14/lego-monowheel-corners-like-its-on-rails/)和一个[实际工作的打印机以及一个莫尔斯电报机，用来发送东西给它打印](https://hackaday.com/2017/07/13/mindstorms-morse-key-writes-to-drawbot/)。
