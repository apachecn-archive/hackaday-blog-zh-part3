# 打造终极宜家床

> 原文：<https://hackaday.com/2015/09/27/building-the-ultimate-ikea-bed/>

当(埃里克·斯特朗的)儿子长不下他的敞篷婴儿床时，他不想放弃它。为了给下一个最小的孩子买床，[埃里克]和他的儿子做了一笔交易。他要给他造一张终极大男孩床。好家伙[他曾经！](https://www.youtube.com/watch?v=Gj-Ht19_qis)

利用宜家黑客的灵感，[Eric]设计了一张床，不仅有滑梯、球类游戏、藏在书柜后面的密室、隐藏的窗户和后门逃生口，甚至还有一张床。听起来有点像那些好得难以置信的电视广告！

他只用了宜家的三种产品建造了它。两张 Kura 库拉床(一张用于存放建筑材料)、一个 Trofast 多层搁板储物单元和一个 Besta——小型书架。相当足智多谋！因为他建造它的空间有限，整个东西仍然是模块化的。休息后留下来看看它的运行吧！
