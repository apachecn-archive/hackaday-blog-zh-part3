# 刮建径向螺线管引擎是抛光和专业的

> 原文：<https://hackaday.com/2016/05/20/scratch-built-radial-solenoid-engine-is-polished-and-professional/>

星形发动机有一些吸引人的地方。黄蜂、旋风、侏儒——这些经典飞机引擎的机械之美是不可否认的。甚至当星形发动机由螺线管而不是内燃机驱动时，它仍然是一个美丽的东西。

螺线管引擎证明他有一些机械印章。如果你跟随下面的视频，你会看到[泰勒]如何在他的设计中进步，并把他从最早的试验阶段学到的东西融入到接近完成的引擎中。这里有大量令人印象深刻的工作——看起来八角形的外壳在压弯机上弯曲，显然自制的螺线管被封装在铜管和配件中,[泰勒]花时间对其进行了精细的抛光。我们怀疑电动换向发动机的微型开关是否能承受足够多的周期，这是一个有用的发动机，但这不是重点。这是关于学习的，我们认为[泰勒]在这方面做得非常好。

对于更多的径向螺线管发动机的优点，检查出[这种发动机与换向](http://hackaday.com/2015/10/05/radial-solenoid-engine-is-undeniably-cool/)完全不同。或者，如果你需要径向引擎理论的基础，[这个木头模型](http://hackaday.com/2016/02/04/learn-how-a-radial-engine-works-or-gawk-at-amazing-wood-model/)可能就是你要的东西。
