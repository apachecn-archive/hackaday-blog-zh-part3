# Hackaday 奖参赛作品:廉价可见光通信

> 原文:[https://hack aday . com/2016/09/03/hack aday-prize-entry-cheap-visible-light-communication/](https://hackaday.com/2016/09/03/hackaday-prize-entry-cheap-visible-light-communication/)

[Jovan]对可见光通信、或 VLC 带来的可能性感到非常兴奋。令人兴奋又新奇。他的开篇段落充满了如此多的网络缩略语，VLC 可以用于，我们的浏览器搜索历史现在看起来像我们试图学习没有任何元音的英语。

在很多方面，他都有充分的理由感到兴奋。我们都知道，红外可以通信相当多，但当你巧妙地利用频率和颜色，并在一些偏振器中加入一些巧妙的算法来进行良好的测量时，你可以与网站中的任何东西进行非常高的带宽通信。你可以在低功耗的情况下做到这一点，最重要的是，没有讨厌的规定来阻挡你。

他想建立一个可用于个人局域网的系统。要做到这一点，他必须想出一种方法，以不超过一瓦特的功率廉价地建造这个系统。项目页面充满了有趣的实验和相当多的关于 led 主题的论文。

例如，他研究了发光二极管对偏振的反应。他测试了 LED 实际开关的速度，同时仍然能够检测到变化。他还做了大量工作来描述 LED 发出的光的种类。我们还不知道他是否会成功，但我们喜欢他为此所做的有趣的工作。
