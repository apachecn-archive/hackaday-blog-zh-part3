# 为你的黑客节庆典准备的物联网圣诞树

> 原文：<https://hackaday.com/2017/12/31/an-iot-christmas-tree-for-your-hacker-mas-celebrations/>

智能圣诞树可能很快就不仅仅意味着时尚装饰的圣诞树。在[阿克谢·库玛尔]、[Anshul Katta]和[Abhishek Maurya]的帮助下，[Ayan Pahwa]稳步推进这一新定义，将他们办公室的[圣诞树](https://iayanpahwa.github.io/SDIoT-Holiday-Lights-2017/)变成了一个可以现场观看的物联网设备！

作为一种物联网设备，这棵树依赖于一直受欢迎的 ESP8266 node MCU——由 Alexa 激活和控制，也可以通过网页来激活和控制。圣诞树上的发光二极管——以及由 Arduino Pro Mini 控制的离线式树顶——也是同样受欢迎的新像素。

对于那些在网上观看的人来说，这个项目附带了一个树莓派和摄像头，用来检查圣诞树的灯光。为了实现这一点，[Pahwa]不得不使用 ngrok 来使 Pi 的——通常——仅局域网的摄像机服务器可以通过互联网访问。前面提到的网页是用 Javascript/CSS 编写的，并托管在运行 Ubuntu 16.04 实例的服务器上。
