# 越多越好 WS2812FX 库

> 原文:[https://hack aday . com/2016/12/12/more-blinky-more-better-the-ws 2812 FX-library/](https://hackaday.com/2016/12/12/more-blinky-more-better-the-ws2812fx-library/)

WS2812 是一项了不起的技术。30 年前，高亮度 led 甚至还不存在。现在，您可以对 RGB LEDs 进行评分，甚至可以省去控制和寻址它们的所有艰苦工作！但一如既往，我们可以做得更好。

在不断流行的 Adafruit NeoPixel 库的基础上，[Harm]创建了 [WS2812FX 库](https://github.com/kitesurfer1404/WS2812FX#ws2812fx---more-blinken-for-your-leds)。图书馆有一个完整的[效果清单，可以在你的闪光灯上运行](https://github.com/kitesurfer1404/WS2812FX#effects)——从令人兴奋的超级火花到受苹果设备启发的平静呼吸。这个库的奇妙之处在于，它可以大大缩短你的普通闪光灯的开发时间——连接你的 WS2812s，选择你的效果，就大功告成了。

[Harm]已经完成了艰苦的工作，将它移植到一系列平台上——在 Arduino Nano、Uno、Micro 和 ESP8266 上进行测试。作为概念的证明，他们还做了一个很好的软件演示——用木头、铝和拆下来的圣诞灯外壳制作一些可爱时尚的圣诞装饰品。将它与 ESP8266 &应用程序结合起来，可以通过 WiFi 从智能手机控制效果。YouTube 上的[组装视频](https://www.youtube.com/watch?v=DJbWD_HiyVE)展示了制作过程，使用螺丝和钉子，用铝板制作了一个吸引人的框架。

这个项目是图书馆和现代硬件让我们站在巨人肩膀上的一个很好的例子。使用比以往更多的 led 灯，可以比以往任何时候都更快地构建功能强大的项目。多年来，我们已经看到了许多伟大的 WS2812 项目，像这个[日出闹钟](https://hackaday.com/2016/08/11/wake-up-with-a-neopixel-sunrise-alarm-clock/)或这个[便携式锐舞棒](https://hackaday.com/2014/08/28/led-light-staffs-for-the-ultimate-portable-rave/)。
