# 纤细优雅的字钟也能显示天气

> 原文:[https://hackaday . com/2016/04/27/slim-and-classy-word-clock-shows-the-weather-too/](https://hackaday.com/2016/04/27/slim-and-classy-word-clock-shows-the-weather-too/)

单词时钟是对传统时钟用户界面的巧妙改变。用单词和短语拼写时间，而不是用数字，可以很好地想象出一个时钟。如果你把当前的天气和预报添加到显示中，你会得到[这个有吸引力的和方便的基于单词的时间和天气显示](http://chronology.weebly.com/wifi-weather-wordclock----new.html)。

这座钟是[GMG]网站上的众多定制建筑之一，展示了人们对不寻常钟表的某种热情，一个 8×32 的新像素阵列位于激光切割的蒸汽弯曲桦木胶合板后面。每个像素都被字母数字字符或代表天气状况的图标所掩盖。ESP8266 获取时间和天气数据，并串行驱动显示器，控制每个单元的颜色并构建显示器。下面的视频显示时钟做它的事情。

当然，我们以前也推出过很多字钟，甚至[一些带有天气显示，](http://hackaday.com/2015/11/08/weather-word-clock-warns-you-of-impending-winter/)但我们喜欢这种超薄而低调的设计。我们对[GMG]将尽可能多的能力装入 256 像素显示屏的长度印象特别深刻，就像“今天”和“明天”重叠的方式。如果你有观察细节的眼光，你可能会发现当相对湿度超过 80%时会显示什么。
