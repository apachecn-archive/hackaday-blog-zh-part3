# 黑客日奖参赛作品:一个社区网状网络

> 原文：<https://hackaday.com/2017/06/01/hackaday-prize-entry-a-community-mesh-network/>

虽然物联网会一直存在，并会杀死我们所有人，但在机器人起义期间，地球上还有几个地方会毫发无损。这些服务不足的社区仍然需要通信和网络，因此[hlew]创建了一个[社区参与网状网络](https://hackaday.io/project/21641-cemn-community-engagement-mesh-network)作为 Hackaday 奖的参赛作品。

虽然现在有很多很多 DIY 网络解决方案可供选择，但[hlew]还是依赖于[一些[Bruce Land]在康奈尔大学](http://people.ece.cornell.edu/land/courses/ece4760/FinalProjects/f2016/bac239_aw528_rk534/bac239_aw528_rk534/bac239_aw528_rk534/mesh_software.html)的学生所做的一些工作。该项目使用简单、廉价的 nRF24 无线电模块构建真正的网状网络，具有多节点通信、动态路由发现和动态路由重新配置功能。

CEMN 将依靠这一网络向服务不足的社区提供通信。该网络的主要目标是广播信息，如作物报告和健康咨询，但它也可以用于个人之间的点对点通信。
