# 迪斯科灯泡让派对继续

> 原文：<https://hackaday.com/2018/01/27/disco-bulb-keeps-the-party-spinning/>

即使你不喜欢迪斯科，你可能会喜欢这个项目中的滑头动作。[W&M] [在一个标准的白炽灯泡](https://www.youtube.com/watch?v=AExPTzzKL-s)内建造了一个微型电动镜球，结果令人兴奋不已。

除了吹一个玻璃灯泡、造一台发动机和种植木材，这差不多是它所能得到的一切。大部分木工活都是在金属车床上完成的，这包括镜球本身的底座。就像所有瓶子里的好东西一样，球太大了，进不去灯泡，所以[W&M]把它分成四份，钻了几个洞，然后用绳子穿过碎片，这样它们就可以被小心地粘在一起，拉回到一个球体。他甚至切开镜子瓷砖，用镊子小心翼翼地涂抹。

这种迪斯科灯泡应该悬挂在天花板上，像普通的镜子球一样连接到电源上。[M&W]将一个小型 USB 壁式充电器的内脏塞进手工制作的山毛榉底座，为旋转球的齿轮电机和照亮球的微型 LED 提供清洁的电力。穿上你最好的休闲套装(或者运动套装，我们不做评判)，匆匆忙忙地过了休息时间去看健身视频。

我们在这附近看不到很多迪斯科球，但是我们曾经见过一个迪斯科二十面体。
