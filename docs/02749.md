# 艰难地建立一个 Shapeoko

> 原文:[https://hack aday . com/2016/06/23/build-a-shape oko-the-hard-way/](https://hackaday.com/2016/06/23/build-a-shapeoko-the-hard-way/)

[Caleb Peters]查看了 Shapeoko 3 CNC 套件，该套件旨在使任何人都有可能构建入门级 CNC 路由器，该套件消除了等式中的猜测，该套件消除了所有这些困难的技术障碍。他看着这一切，心想:“不。“他想学！所以他决定艰难地建造一个。像早期的美国拓荒者一样，他从零开始建造他的 Shapeoko，一直虔诚地忍受着痛苦。

他的目标是以低于套件的价格构建 Shapeoko 3 的改进版本。第一个问题是龙门架运行的轨道。Inventables 不打算向他出售铁轨，他也不确定所用的 [delrin](http://hackaday.com/2016/03/03/materials-to-know-acetal-and-delrin/) 车轮是否能够承受他更重设计的重量。经过一番争论后，他决定铝硬涂层轨道和钢轮应该足够长，如果铝磨损，更昂贵的钢轨是一个下降的替代品。

每一步都克服了类似的问题。他不能完全复制 Shapeoko 的设计。Shapeoko 的钢件只能在更大的机器上制造，如喷水器或工业激光器。他确实有一个膝磨，并设法巧妙地通过一些轻微的重新设计来避免这种需要。他坚持不懈，做了一些很酷的事情，比如在用作主轴的木刳刨机的外壳上钻一个孔，并在换向器和电刷组件后面放一个霍尔效应传感器，以获得主轴的转速读数。

对我们来说幸运的是，他很好地记录了这一切，并拍摄了一个九集的视频系列；休息后你们将看到最后一场比赛。
