# 通往塞顶码头的简单路线

> 原文：<https://hackaday.com/2017/01/16/a-simple-route-to-a-plug-top-pi/>

实验者需要一个一体化的电源供电的计算平台，而没有全尺寸 PC 或类似设备的烦恼，他们可以使用许多小型的插头顶式计算机。但是在各种各样的型号中，总是缺少一样东西，一个插电式树莓派。为了填补市场上的这一空白，[N-O-D-E]利用官方的 Raspberry Pi PSU 配件和 Raspberry Pi Zero 创造了一个融合 Pi 和插头的产品[，在两者之间夹有一个](https://www.youtube.com/watch?v=dfg4g3YuvnY) [UUGear Zero4U USB 集线器](http://www.uugear.com/product/zero4u/)。

这是一个非常简单明了的构建，PSU 的背面用一点 Sugru 形成了一个平坦的表面，然后电源线被剥回其电线，然后连接到 USB 集线器上的电源引脚。然后将集线器连接到 Sugru 上——他没有说如何连接，但我们怀疑是双面胶带——Pi 安装在集线器的顶部。弹簧针与电脑底部的衬垫进行必要的连接，因此可以随意拆卸和更换。

其结果是对您的 Pi 库的一个有用的补充，可以用于许多小型独立设备。它可以使用封面，但是我们怀疑 3D 打印机所有者可以相对容易地为自己创建一个。完整的描述显示在视频下方的休息。
