# 卡路里意识仓鼠主人计步器

> 原文：<https://hackaday.com/2017/05/20/pedometer-for-calorie-conscious-hamster-owners/>

Arduino 激发了许多有益于人类的创造性项目。另一方面，【约翰·穆勒】设计的 Arduino 仓鼠滚轮计步器是专为可爱毛茸茸的啮齿类宠物设计的。当[约翰·穆勒]的女儿想要跟踪她的仓鼠晚上散步的时候，她的创造者——父亲把这看作是让她参与科技的机会。该项目包括一个带有磁铁的仓鼠轮，当完成一次旋转时，它会触发一个簧片开关。整个组装都是定制的，[约翰·穆勒]用大量清晰的图像出色地记录了建造过程。

轮子固定在一端带有滚珠轴承的轴上，整个装置安装在保持架的侧面，这样就可以很容易地拆卸下来进行维护。簧片开关嵌入木制安装块中，以便连接电缆从组件内部穿过。这可以防止仓鼠接触电缆或以任何方式损坏电缆。液晶显示器和 Arduino Uno 放置在笼子外面，用于显示车轮的转数以及行驶的等效英里数。

Arduino 的代码也提供给任何想要复制这个项目的人，下面的视频展示了这个项目的工作。该项目还可以扩展到计算消耗的卡路里和跑步速度。这个项目是技术如何被用来帮助的一个主要例子，类似于[物联网仓鼠轮子](http://hackaday.com/2016/05/03/not-even-hamsters-are-safe-from-the-internet-of-things/)，它可以在推特上发布仓鼠生活的每个动作。
