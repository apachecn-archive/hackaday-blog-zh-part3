# 中型机器传递微控制器消息

> 原文:[https://hack aday . com/2018/04/16/medium-machine-mediates-micro controller-messages/](https://hackaday.com/2018/04/16/medium-machine-mediates-microcontroller-messages/)

将计算机连接到人脑目前仅限于科幻小说和少数尖端实验室的范围。利用远离我们中枢湿件的一些神经是可能的，[彼得·布茨科夫斯基]向我们展示了他的时髦机器，可以在不主动记忆任何东西的情况下向我们的大脑植入一种模式。

他的中型机器利用一个 TENS 单元按照 Arduino 中编程的模式激活前臂肌肉。用户将他们的前臂放在安装在一个雅致的木制平台上的两个铝电极上，并将单个手指伸到按钮上。电脉冲触发按下按钮的肌肉。仅此而已。重复该模式几次后，用户应该能够按照命令背诵它，即使他们不知道它是什么意思。如果这听起来像一些[约翰尼记忆术]记忆缓存，你是绝对正确的。这个项目的灵感来自于[威廉·吉布森]的小说，这部小说后来被改编成了[山谬·里维]的电影。

用户可以通过莫尔斯电码信息或[秘密敲门程序打开阁楼图书馆](https://hackaday.com/2014/12/25/secret-attic-library-door/)或[演奏一首小曲](https://hackaday.com/2013/10/20/halloween-knock-box/)。[学首钢琴曲](https://hackaday.com/2012/07/22/vibrating-gloves-help-bring-back-finger-sensation-after-injury/)怎么样？
