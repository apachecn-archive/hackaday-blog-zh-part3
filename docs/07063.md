# 用 Tinkercad 的砖块模式制作乐高原型

> 原文:[https://hack aday . com/2017/09/12/Lego-prototyping-with-tinker cads-brick-mode/](https://hackaday.com/2017/09/12/lego-prototyping-with-tinkercads-brick-mode/)

![](../Images/ad00d19cc2931972685eedc8dc367846.png)【Andrew Sink】制作了一个[的简短视频](https://www.youtube.com/watch?v=5vvgZCRyBr8)展示了他是如何导入众所周知的 3D Benchy tuggle 模型的 STL，而不是将其发送到 3D 打印机，而是使用砖块模式功能以令人眼花缭乱的万花筒颜色制作乐高砖块的物理副本。

对于那些最近没有使用 Tinkercad 的人来说，砖块模式允许您以各种比例将模型表示为乐高砖块。你像往常一样建模(或者导入一个模型),然后按下一个按钮，就像用标准砖块一样精确地在乐高中渲染它。

此外，[Andrew]展示了如何将“层”功能用作模型的临时装配指南，尽管他在下面嵌入的视频中解释了一些奇怪的地方。
