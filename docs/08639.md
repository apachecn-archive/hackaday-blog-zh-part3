# 双带图灵机

> 原文:[https://hackaday.com/2018/02/23/a-two-tapes-turing-machine/](https://hackaday.com/2018/02/23/a-two-tapes-turing-machine/)

虽然和许多独立的发明家一样，计算的起源可以说是通过许多人的工作而实现的，但艾伦·图灵无疑是计算机科学中的基础人物之一。他的图灵机是一种思想实验计算设备，其中一个程序对印在无限长的胶带上的符号执行操作，理论上可以计算任何计算机可以计算的任何东西。

在实践中，我们不使用图灵机作为我们的日常计算平台。作为学术抽象练习而设计的机器不是为效率而设计的。但这并不能阻止黑客入侵，为了证明这一点，奥利维尔·贝勒克斯已经使用现成的电子元件做到了这一点。他的双带图灵机出现在一个大的 PCB 上，并在视频中显示在计算斐波那契数列的前几个数字。

原理图以 PDF 格式提供，主要由 74 系列逻辑芯片组成，磁带内容显示为两行 led。该程序被表示为一个可插拔的二极管矩阵，但他以一种特别简洁的方式使用了 led 而不是传统的二极管，使我们可以在访问每条指令时看到它。对于任何希望了解图灵机的人来说，这都是一个有趣的项目，尽管我们希望[Olivier]在他的文章中给我们提供更多的信息。

这种对图灵机的迷恋已经在这些年来的无数构建中表现出来了。一小部分选择是[一个使用 3D 打印](https://hackaday.com/2016/07/12/a-very-modern-turing-machine-build/)，另一个[使用乐高](https://hackaday.com/2012/06/18/a-lego-turing-machine-for-alans-centennial/)，还有[第三个使用滚珠轴承](https://hackaday.com/2011/03/25/mechanical-turing-machine-can-compute-anything-slowly/)。当然，如果你想要即时的满足，看看谷歌在图灵 100 周年纪念日的涂鸦作品中的[。](https://www.google.com/doodles/alan-turings-100th-birthday)
