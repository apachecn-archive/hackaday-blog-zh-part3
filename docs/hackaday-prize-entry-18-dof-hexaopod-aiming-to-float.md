# Hackaday 奖参赛作品:18 自由度六足机器人，旨在漂浮

> 原文：<https://hackaday.com/2017/07/30/hackaday-prize-entry-18-dof-hexaopod-aiming-to-float/>

[肯·康拉德]不喜欢他在互联网上看到的蜘蛛机器人项目:它们的每条腿大多有 2 个自由度——如果不是更少的话。他着手制造一个有 18 个自由度的六足机器人，能够向任何方向移动。测量从尖端到尖端大约 20 英寸，定制的 3D 打印底盘是围绕 18 个 SG90 9g 微伺服系统设计的。每条腿有 3 个伺服系统，一个用于移动尖端，一个用于中间，一个用于来回移动整条腿，像螃蟹一样。

也许这个项目最有趣的概念是巨大的桨状腿。[Ken]希望让机器人通过将其小腿放平来实现某种程度的漂浮，或者由于表面张力，或者在腿上添加一些浮力材料的帮助下保持漂浮。

[Ken]仍然需要为这个怪兽设计一个控制系统，但是我们对他创造性的使用拉链代替传统的扣件感到敬畏。
