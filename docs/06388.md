# Hackaday 奖参赛作品:Etchr 按需印刷电路板

> 原文:[https://hack aday . com/2017/07/11/hack aday-prize-entry-PCBs-on-demand-with-etchr/](https://hackaday.com/2017/07/11/hackaday-prize-entry-pcbs-on-demand-with-etchr/)

雄心勃勃的 etchr-PCB 打印机目前只是一个概念，但我们并不经常看到有人试图以新的方式解决桌面 PCB 生产。创作者[Jonathan Beri]敏锐地意识到，在创作电子产品时，大多数工作流程的瓶颈是 PCB 本身。像 OSH Park 这样的服务使专业制造的 PCB 可以以低成本获得，但部分交易是周转时间通常以周计算。

[Jonathan]的 etchr 概念是一个小型系统，它不仅可以自动蚀刻覆铜板，同时伴随着化学物质的泛滥和排放，还可以应用阻焊膜和丝网层标签。剩下唯一要做的事情就是钻任何需要的孔。

etchr 背后的思路是，先拿一块贴有光阻膜或者喷涂的覆铜板，固定在一个框架里。UV 投影仪负责将走线图案放到电路板上(并在后续步骤中处理 UV 固化阻焊膜)，深框架兼作任何化学处理(如蚀刻和清洗)的容器。这是一个雄心勃勃的项目，但每个步骤背后的过程都很好理解，将它们都放在一台机器上是一种有趣的方法。

多氯联苯的桌面生产可以通过几种方式完成，包括通过墨粉转移法进行蚀刻(我们自己的埃利奥特·威廉姆斯[清楚地解释了如何从优秀走向卓越](http://hackaday.com/2016/09/12/take-your-pcbs-from-good-to-great-toner-transfer/))。另一种方法是直接铣掉多氯联苯，像 [Othermill](http://hackaday.com/2016/08/02/the-othermill-vs-import-a-technical-comparison/) 这样的工具就是专门设计来做这项工作的。有趣的是，有一种方法包括应用阻焊膜。
