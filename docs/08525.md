# 一个我们其他人都能理解的 RISC-V

> 原文：<https://hackaday.com/2018/02/09/a-risc-v-that-the-rest-of-us-can-understand/>

围绕 RISC-V 体系结构，微处理器的世界非常激动人心。这是一种开源的模块化指令集规范，已经在 FPGAs 上实现，并开始在专用芯片上出现。

如果你还不了解这一代最重要的微处理器发展，你应该看看这个视频。正如[Robert Baruch]所展示的，RISC 技术和模块化指令集的结合意味着[符合 RISC-V 规范的最简单的处理器也可以令人惊讶地使用](https://www.youtube.com/watch?v=yLs_NRwu1Y4)。为了证明这一点，他正在用 LSI 和 MSI ~~TTL~~ CMOS 芯片建造一个，这是我们通常更希望在一个更古老的架构的重建中看到的东西。

下面的视频是即将推出的系列中的第一个，在视频中，他介绍了该项目，并在解释他的 RISC-V 实现的寄存器机制之前，给了我们一个易于理解的 RISC-V 概述。这将是他的第一个模块，他已经为它制作了一个 PCB。他贯穿其设计，他选择的指示灯，然后他选择的 PCB 房子。还有一个分线板，有两个 PCI 插座，他将用于他的背板。最后，我们可以看到正在测试的电路板，led 会根据完整寄存器中存储的值点亮。

[Robert]已经在这些页面上出现过很多次，最近一次是带着他基于 TMS9900 的试验板计算机。这个版本远离了他的复古风格，应该很值得在以后的系列中观看。
