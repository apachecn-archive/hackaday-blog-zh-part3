# 2018 最佳新晋 Amiga 称号？

> 原文：<https://hackaday.com/2018/06/21/the-best-new-amiga-title-of-2018/>

仅仅因为一个系统对我们大多数人来说变得过时，并不意味着每个人都停止使用它。看一看这款为 Amiga 500 设计的名为 Worthy 的全新游戏，尽管自 1987 年以来已经有所改进，但它肯定会让我们大多数人后悔升级我们的家用电脑。

开发这款游戏的团队被称为 [Pixelglass](http://pixelglass.org/) ，他们在这个平台上做了很多工作，在过去几年里发布了几款游戏。他们的最新游戏是 Worthy，这是一款动作冒险游戏，看起来类似于 SNES 的自上而下视角塞尔达游戏。对于一个很少有人拥有的系统来说，这是一个令人印象深刻的工作，但是如果你有一个这样的系统(或者即使你有一个好的模拟器)，你可能想试一试。

如果为复古系统开发游戏是你的风格，这并不局限于像 Amiga 这样的个人电脑。我们已经看到了超级任天堂的[开发平台，它将让你运行自己的代码，甚至还有其他方法让](https://hackaday.com/2009/10/23/make-your-own-snes-games-with-developer-cartridge/)[与世嘉土星](https://hackaday.com/2016/07/11/cracking-the-sega-saturn-after-20-years/)一起工作，如果你真的想冒险的话。
