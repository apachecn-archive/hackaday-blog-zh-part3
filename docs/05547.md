# Hackaday 奖品入口:另一个互联网按钮

> 原文：<https://hackaday.com/2017/04/17/hackaday-prize-entry-another-internet-button/>

长期以来，这一直是展望未来的一个主题，即我们将到达一个时刻，按下一个按钮，所有生活的舒适就可以被召唤出来。通过技术的魔力，也就是说，没有富有的维多利亚时代的人为了达到同样的目的而包围自己的人类仆人的军队。

当然，要达到这种按键涅槃，总得有人来做按键。有很多竞争者争夺一键统治一切的奖项，我们可能最常看到的是亚马逊的 Dash。今天我们带给你另一种可能性。【Hendra Kusumah】的 [A.I.B .(又一个物联网按钮)](https://hackaday.io/project/20425-aib-another-iot-button)顾名思义，就是一个连接互联网的按钮。更具体地说，它是一个连接到 IFTTT 的按钮，允许您从那里触发您的操作。

就硬件而言，这再简单不过了。一个按钮、一个粒子光子、一些导线和一个电阻器。然后把代码安装到板上，你就可以走了。只需对代码稍作修改，它也可以与 ESP8266 配合使用。就是这样，再简单不过了。你可能会问这其中的乐趣在哪里，但你没有抓住重点。使用按钮触发的事件才是最重要的，所以为什么要把创建按钮变成一件苦差事呢？

我们已经向您展示了许多物联网按钮，只有几个帖子是[这个 ESP8266 按钮](http://hackaday.com/2017/03/08/point-and-click-to-an-iot-button/)和看一看[第二代亚马逊 Dash](http://hackaday.com/2016/07/26/bending-the-new-amazon-dash-button-to-your-will/) 。
