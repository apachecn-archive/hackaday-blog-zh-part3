# Ham Radio WiFi

> 原文:[https://hackaday.com/2016/07/02/ham-radio-wifi/](https://hackaday.com/2016/07/02/ham-radio-wifi/)

美国许多业余无线电爱好者都参加野外日。这是一项年度活动，鼓励无线电操作员在自然灾害后可能发生的情况下建立电台。通常，这意味着接管一些公园或营地，带上发电机、便携式设备，并在你拆除之前让它们在周末都工作。

没有电的日子并不好过。这就是为什么大多数电站使用发电机，太阳能电池，甚至电池。然而今天，你可能需要一台联网的电脑来做日志记录和其他功能。[HamRadioConcepts]有一个视频(见下文)展示了他们如何从远处抓取互联网[作为他们的场地日网站。](https://www.youtube.com/watch?v=ZoyfldmR3bc)

使之成为可能的关键是，大多数有线电视公司现在从住宅和商业路由器广播公共 WiFi 热点。你通常需要选择退出，大多数人不会。如果您通过该提供商拥有家庭互联网，您通常可以免费使用这些热点。

在这种情况下，范围内没有热点，但湖对面有几个。使用一个碟形天线和一个 1 瓦的 USB 网络适配器，他们能够将计算机接入网络，然后使用连接共享在现场周围传播 WiFi。

如果你喜欢自己卷天线，我们可以帮你，特别是如果你有一个金属网勺。
