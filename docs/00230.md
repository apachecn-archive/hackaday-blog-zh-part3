# HEXXX 游戏机

> 原文:[https://hackaday.com/2015/10/01/the-hexxx-gaming-console/](https://hackaday.com/2015/10/01/the-hexxx-gaming-console/)

担心将项目带到 CCC15(混沌通信营)的演示会上，[Anthony Liekens]必须尽快行动。他本可以带着他的巨大的[飞行意大利面条怪物显示器(简称 FSM)](http://hackaday.com/2013/12/13/flying-spaghetti-monster-display/)，但不幸的是它不适合他的汽车。所以，环顾他的车库，他意识到他有一堆额外的 RGB LEDs，和一个破损的 iCade 核心。他头脑中的齿轮开始转动，他想出了 [HEXXX 游戏机的主意。](https://github.com/aliekens/hexxx/blob/master/README.md)

那是什么呢？这是一个为三人游戏设计的六边形控制台——就像他的三人游戏的 *Tron 和三人游戏*的*Pong——他甚至有一个六人游戏的 Flappy Bird 克隆版！从概念到现实，仅仅花了十个紧张的晚上就完成了…最终的结果非常棒。*

如果你想自己做，他有你需要的所有信息，包括激光切割计划。更不用说他甚至写了一个 HEXXX 模拟器，这样你就可以不用硬件来构建应用程序了！
