# 纸板制成的机器人手臂

> 原文:[https://hackaday.com/2016/09/14/robotic-arm-from-cardboard/](https://hackaday.com/2016/09/14/robotic-arm-from-cardboard/)

谷歌向世界展示了你可以用纸板制作虚拟现实耳机。我们认为这可能是[Uladz]创造同样由纸板制成的机器人手臂的灵感。他说你可以在两小时内复制他的设计。

你需要一个 Arduino 和四个伺服电机。纸板并不太重，所以你可以使用相当小的马达。除了纸板，还有一块硬纸板做底座和几个金属夹子。您可以从 Arduino 程序控制这一切，或者如果您想通过远程控制来运行它，可以添加一个红外接收器。下面有一个名为 CARDBIRD 的手臂的视频。

如果你想建造更强大的东西，我们在之前已经看过一些[严肃的武器。如果你想让](https://hackaday.com/2015/09/23/hackaday-prize-semifinalist-an-affordable-robotic-arm/)[变大](https://hackaday.com/2015/12/26/project-giant-robot-arm/)，我们以前也见过。但是每个项目都必须从某个地方开始，纸板模型是我们最喜欢的东西之一。
