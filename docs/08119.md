# 吃点品客薯片，喂猫

> 原文：<https://hackaday.com/2018/01/01/eat-some-pringles-feed-the-cat/>

你知道这句谚语:“狗有主人，猫有仆人。”尤其是当你的猫主人忘记了时间，坚持要在天黑前喂食的时候。你很想躺在床上装死，但是那只猫坐在你的胸口上，盯着你的灵魂，这是一个很高的要求。

在这样的时刻，自动喂猫器会很好，但是现成的装置很贵。[妈妈会感到骄傲]决定[推出他自己的喂猫器](https://www.youtube.com/watch?v=dqr-AT5HvyM)，结果相当可观，相当于一个垃圾桶。两个旧的食物罐组成了车身——一个品客薯片罐在上面用来装食物，一个坚果罐在下面用来装伺服系统。罐头的金属末端很好地嵌套在一起，每个都有一大部分被移除，漏斗每次旋转都会打开一个孔，将食物从斜槽中落下。一只猎犬黑控制伺服系统，但是任何有 PWM 输出的东西都应该可以。我们倾向于 ESP8266 生态系统，支持 WiFi 远程控制喂食，我们可能会用 PVC 管加强结构，以防止未经授权的访问。但这是一个简单的概念，简单是一个很好的起点。

你应该不希望在这些地方建造宠物喂食器。挑一个吧——时髦的蒸汽朋克、[超便宜的](https://hackaday.com/2016/12/30/cheap-cat-feeder-enhances-sleep/)，或者有阿基米德风格的[。](https://hackaday.com/2016/06/02/feeding-the-cat-reinventing-the-wheel/)
