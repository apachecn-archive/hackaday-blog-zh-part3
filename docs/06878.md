# 用 PVC 管制作你自己的复合弓

> 原文：<https://hackaday.com/2017/08/23/make-your-own-compound-bow-from-pvc-pipe/>

你有没有想过[自己制作复合弓](https://www.youtube.com/watch?v=kcmnG-qjauA)来取乐甚至钓鱼？[新创意 DIY]在他们的 YouTube 视频中向我们展示了这一点。复合弓与他们的长弓祖父母相比非常强大，依靠的是杠杆原理或滑轮。这意味着对于相同的输出消耗更少的功率。

复合弓可以使用滑轮和一些特殊材料进行复杂的设计，但你可以用一些螺母和螺栓、PVC 管、绳子和内胎制作自己的复合弓。PVC 管可以使用热源熔化成型，如便携式炉子甚至喷灯，一旦你成型了你的弓，你会想要用螺母和螺栓在两端放一小段管子。然后，你可以用橡胶来增加你的弓射箭所需的弹性，使用切割成合适尺寸的轮胎内胎。你所需要的就是一根绳子，把它系在你的管子的两端，你就有了一个 DIY 的 PVC 复合弓，可以用来射箭了。或者，你也可以用滑雪板做一个反曲弓。

–更新[Leithoa]在评论中指出，这既不是弓也不是复合弓，他们经常被混淆。这实际上是一种弹弓。–
