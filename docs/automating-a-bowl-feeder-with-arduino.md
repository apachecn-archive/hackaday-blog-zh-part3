# 使用 Arduino 自动化碗进料器

> 原文：<https://hackaday.com/2017/11/11/automating-a-bowl-feeder-with-arduino/>

在 Hackaday 上搜索“碗喂食器”,你会发现除了自动猫狗喂食器之外什么也没有。这是一个耻辱，因为保持你的宠物喂食是很酷的，振动碗喂食器更酷。如果你看过几集“它是如何制造的”，你可能会看到这些惊人而简单的设备，用于自动装配的小零件的进料和对齐。它们看起来很迷人，如果你想知道像顶条上的小针这样的部件是如何处理的，那它很可能是一个碗状喂食器。

纽约 CNC 公司的[John]正在建造一个带 Arduino 控制的碗状进料器，下面的视频带我们参观了建造过程。公平的警告，视频是沉重的数控方面铣削整理出料坡道，这是预计从[约翰]的渠道。我们发现 CNC 很吸引人，但是如果你不喜欢，就跳到最后三分钟[John]讨论控制。他的出料坡道有一个用于光学传感器计数零件的插槽。为了安全起见，Arduino 通过一个外部继电器控制高拉碗进料器，并在分配完所需数量的零件后停止。

我们知道，看着一个人使用价值 2 万美元的数控铣床，对于本来可以 3D 打印出来的东西来说，似乎有点大材小用，但毕竟[约翰]经营着一家工厂，通常会从事大型工业工作。或者是小的，像[这些整洁的彩色填充机器徽章](https://hackaday.com/2017/08/05/ink-filled-machine-badges-score-respect-for-your-gear/)。
