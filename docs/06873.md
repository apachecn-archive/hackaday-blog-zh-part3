# 完成迷你 PS One: SLA Vs 挤压

> 原文：<https://hackaday.com/2017/08/24/finishing-a-mini-ps-one-sla-vs-extruded/>

首次使用 3D 打印机的用户学到的最大教训之一是，不是所有东西都可以复制，打印机是一台机器，而不是奇迹创造者。它在打印内容和输出质量方面都有局限性。对于极小的物体，0.8 毫米的喷嘴是不行的，随着树脂打印机的兴起，问题是，“挤出机打印机过时了吗？”

[Dorison Hugo]用树莓 Pi 做了一个迷你版的 PS One，你可以玩 games one。有趣的是，在他的[视频中，他对 SLA 打印机和更便宜的挤出机打印机](https://www.youtube.com/watch?v=wbx4bDK9pCE)进行了比较。他经历了一系列的步骤，打印、归档、填充、修复、砂漆、砂、油漆等，试图得到一个好的 PS1 模型复制品。然后，他继续用 SLA 打印机打印一个，并完成它以与第一个模型进行比较。对于那些想知道的人来说，贴花印在喷墨打印机上，还有一个从旧电脑中抢救出来的定制切割散热器。

剧透警报！SLA 赢了，但在我们看来，只是一点点。这个想法是，只要有足够的努力和耐心，你就可以用更便宜的机器制作迷你模型。SLA 打印也需要工作，但它相对较少，对于详细的模型，它是一个更好的选择。我们真的很喜欢从头到尾观看这个过程，包括 Dremel 的作品，因为当我们看到 3D 打印时，它会被遗忘。创造美的东西需要时间和努力，这源于一种创造的热情。

看看下面的时间推移视频，对于 SLA 打印机粉丝来说，看看今年 Hackaday 奖的参赛作品 [DIY SLA 打印机](http://hackaday.com/2017/06/07/hackaday-prize-entry-diy-lcd-based-sla-3d-printer/)。
