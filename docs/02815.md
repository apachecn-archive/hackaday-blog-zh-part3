# SpotMini 展示它的东西

> 原文:[https://hackaday.com/2016/07/01/spotmini-struts-its-stuff/](https://hackaday.com/2016/07/01/spotmini-struts-its-stuff/)

波士顿动力公司是一家备受赞誉的机器人公司，以其“大狗”机器人和其他将机械灵活性推向令人印象深刻的极限的机器而闻名，该公司生产了一种小型版本的“斑点”机器人[，称为“斑点迷你”](https://www.youtube.com/watch?v=tf7IEVTDjng)。

这款重量为 55-65 磅的轻型全电动机器人充满电后可持续 90 分钟，并拥有部分自主性——特别是在导航方面，这要归功于四肢的本体感觉传感器。SpotMini 最引人注目的特点是其光滑的新轮廓和机械臂，通过将玻璃放入洗碗机并进行回收来展示这一巨大的升级。

然而，机器人容易失败，所以知道我们未来的霸主和我们人类一样容易被香蕉皮滑倒是件好事。
