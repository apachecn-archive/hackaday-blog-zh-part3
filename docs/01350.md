# 将您的 3D 打印带入下一个维度

> 原文：<https://hackaday.com/2016/02/10/take-your-3d-printing-to-the-next-dimension/>

在被誉为 3D 打印的下一个伟大进步中，科学家已经能够[让 3D 打印的形状在接触水](http://www.technologyreview.com/news/546126/gorgeous-new-4-d-printing-process-makes-more-than-just-eye-candy/) 时改变形状，将 3D 打印直接带入第四维空间。虽然我们目前看到的唯一例子是相对平坦的指纹(可以说从索赔中减去了一个“D”)，但新的程序对该技术来说是开创性的。

该工艺使用纤维素纤维，当其以特定方式排列并暴露于水中时，会膨胀以改变形状。这类似于恒温器中的双金属片的工作原理，但他们真正的灵感来自植物的生物过程，这种过程允许他们根据环境条件改变形状。很难说这种新的打印方法是否会永远改变 3D 打印的前景，但就目前而言，这是一种值得关注的有趣尝试。休息后的视频展示了使用这种技术的快动作打印，随后是浸入水中的打印演示。

我们经常看到新的技术进步利用生物学作为新思想的跳板，这次也不例外。有[受松果](http://hackaday.com/2015/07/10/shapeshifting-material-for-weather-adaptive-structures/)启发的建筑结构和[受鱿鱼](http://hackaday.com/2013/06/20/modeling-squid-cells-in-code-foregoes-connecting-voltage-to-animals/)启发的加工黑客。生物学就在我们身边，任何一种都可以作为你下一个项目的灵感来源！
