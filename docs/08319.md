# 远程 RFID 传单

> 原文：<https://hackaday.com/2018/01/19/long-range-rfid-leaflets/>

选一张牌，任何一张。[Andrew Quitmeyer]和[Madeline Schwartzman]确保你挑选的任何一张卡片都与他们在纽约的艺术装置相匹配。“再植”是一个互动的艺术装置，邀请客人观看来自世界各地的全尺寸叶片~~模具~~铸件。

在这个艺术装置中，带有树叶图像的收据文件被放在了外面。当观众选择一个并将其带到观看区时，RFID 阅读器会告诉 Arduino 检测到了哪个标签。固态继电器控制两个覆盖着透明窗帘的回收衣物传送带。这些简单的单元过去是来回控制的，但通过航位推算，它们可以将任何叶片~~模具~~铸件呈现在前面和中心。

上个世纪的服装传送带以前没有这么智能，它回避了小企业或空间有限的企业的库存自动化问题。
