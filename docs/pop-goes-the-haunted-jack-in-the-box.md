# 流行去闹鬼的杰克在盒子里

> 原文：<https://hackaday.com/2017/10/21/pop-goes-the-haunted-jack-in-the-box/>

万圣节也在悄悄逼近你吗？虽然还有不到两周的时间，但是仍然有足够的时间来做一些让“不给糖就捣蛋”的人和派对客人吓得屁滚尿流的东西。今年，黑客日常客[【肖恩·哈金斯】黑客了他最喜欢的节日，采用了一些带有基本恐怖水平的东西，并使其自主](https://www.instructables.com/id/Haunted-Jack-in-the-Box/)。还有什么比闹鬼的玩具更可怕的呢？

(几十年前的)盒子里的杰克机制很简单。转动曲柄操作一个机械音乐盒，播放传统的“流行歌曲《黄鼠狼去》”。当音乐盒达到高音时，音乐盒桶上的一块突出的塑料会干扰闩锁的另一端，从而释放里面的可怕小丑。[Sean]用 100:1 的 DC 马达从里面转动曲柄，用 Pi 摄像机探测附近的受害者。一旦摄像机锁定一张脸，盒子就会转动曲柄，最终弹出小丑。由于里面的大部分空间已经被弹簧占据了，[Sean]将电子设备放在一个定制的 3D 打印底座中，并为相机的眼睛切出一个洞。

像这样的项目可以进行许多修改。[Sean]现在完全控制了闩锁操作，所以他可以让小丑 pop 立即出现，或随机出现，或有时根本不出现。休息之后，请观看[Sean]的娱乐视频。
