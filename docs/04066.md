# 本周失败:升级失败的 3D 打印

> 原文:[https://hack aday . com/2016/11/08/fail-of-the-week-upcycling-failed-3d-prints/](https://hackaday.com/2016/11/08/fail-of-the-week-upcycling-failed-3d-prints/)

有没有可能[回收失败的 3D 打印品](https://www.youtube.com/watch?v=G-HWrDMr0ks)？事实证明，的确如此——只要你对“回收”的定义有点弹性。毕竟世界只需要这么多杯垫。

公平地说，[Devin]的实验更多的是关于循环等式的向上循环的一面，但它确实值得一试。3D 打印几乎没有被付诸实践，任何花时间打印的人都知道，它很容易搞砸。[Devin]的过程开始于一个装满不合格印刷品的箱子被锤子砸碎。展开放在准备好的烤盘上(永远不要再用来做饼干),在烤箱中低温烹饪，塑料块慢慢融化成一层薄而均匀的薄片。

[Devin]的目标是将它们铸造成可用的物体，所以他尝试做一个碗。他尝试使用倒置的金属碗作为模型来重新加热这种材料的圆盘，但他发现塑料不能均匀软化，导致大理风格的碗有薄点和孔。然后，他翻转碗，试图让材料凹陷到形状中；这稍微好一点，但仍然不是他所期待的胜利。

最终，[Devin]得到的只是一些艺术品和几个漏碗。他还能用塑料做什么？他会不会因为[真空成型](http://hackaday.com/2010/12/02/vacuum-forming-at-home/)碗或者甚至[压力成型](http://hackaday.com/2016/03/22/quick-and-easy-pressure-forming-makes-plexiglas-domes/)碗而变得更好？或者，当你理论上可以自己制造灯丝的时候，升级是没有意义的吗？请在评论中告诉我们你将如何改进这一流程。
