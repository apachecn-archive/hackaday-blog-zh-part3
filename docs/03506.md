# SNES 微是艺术的 Pi Z

> 原文:[https://hack aday . com/2016/09/11/snes-micro-is-a-pi-z-of-art/](https://hackaday.com/2016/09/11/snes-micro-is-a-pi-z-of-art/)

粘土是一种不成形的原材料，它正等待着被你的创造力转化为神奇。树莓派也是。[Dorison Hugo]在他精巧制作的 SNES 微型中把两者结合在一起——一个用粘土雕刻的小型复古游戏控制台。

对于他的构建，[Dorison]用塑料片组装了一个小盒子的两半，两者的大小大约相当于雕塑内部的组件:一个 Raspberry Pi Zero 和一个 USB hub。然后，他在两半的外面塑上粘土，用它们作为实际案例的支架。在更多的建模之后，这个肿块开始呈现出一个小小的 SNES 控制台的形状，原始的比例和微小的细节被仔细地雕刻在粘土上。

[![](../Images/7f187741fc793356887c76641a5136cb.png)](https://hackaday.com/2016/09/11/snes-micro-is-a-pi-z-of-art/snes-micro-case/)[![](../Images/e50486de82d59acc70ab4bdc9958e3e4.png)](https://hackaday.com/2016/09/11/snes-micro-is-a-pi-z-of-art/snes-micro-sculpting/)[![](../Images/1e038592c60361e79e5e972d4163d47e.png)](https://hackaday.com/2016/09/11/snes-micro-is-a-pi-z-of-art/snes-micro-guts/)

一旦粘土变硬，[Dorison]移走塑料支架，打磨模型，用旋转工具雕刻出更多的细节。最后的油漆工作给这个小控制台带来了熟悉的外观。完成最难的部分后，剩下要做的就是将树莓派连接到集线器，用环氧树脂将所有东西固定到位，并启动[仿真站](http://emulationstation.org/)。看看下面的视频，在那里[多里森]带你了解他的巧妙构建！
