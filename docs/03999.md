# 万圣节无人机上的简易 UFO 灯

> 原文:[https://hack aday . com/2016/10/29/easy-UFO-lights-on-your-drone-for-Halloween/](https://hackaday.com/2016/10/29/easy-ufo-lights-on-your-drone-for-halloween/)

有时候，重要的不是你把什么放在一起，而是你如何使用它。Adafruit 公司的人已经提出了一个项目，关于如何在万圣节前夕用“UFO 灯”装扮你的无人机。该项目是一个 RGB LEDs 和一个小型微控制器的环，给任何四轴飞行器一个旋转的“牵引光束”效果。3D 打印夹具处理附件。如果你像他们一样使用 DJI Phantom 4，你可以使用一根短 USB 电缆直接从无人机上为一切供电，这意味着几乎没有任何布线工作，也没有对飞机进行任何类型的永久性改变。否则，你只能靠自己来提供动力，但这可能是任何一个摆弄业余爱好飞机附件的人都有能力做到的。

这个项目展示的一件事是，在零件和工具的可访问性方面，事情已经取得了多大的进展。一个 3D 打印夹具、一个现成的 RGB LED 环和一个小型微控制器的嵌入式软件库使这个项目成为一个下午的项目。视频(嵌入下面)还展示了一些不熟悉的灯光和一些黑暗是如何将熟悉的幻影四轴飞行器变成真正的不明飞行物的。
