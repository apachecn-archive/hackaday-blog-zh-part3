# 将命令行转化为 Web 应用程序

> 原文：<https://hackaday.com/2017/11/19/turn-command-lines-into-web-apps/>

即使你喜欢使用图形用户界面，你也可能同意编写一个图形程序通常比编写一个老式的基于文本的程序更难。把图形用户界面变成在线格式意味着更多的考虑。亚当·考利有这个问题的答案。正如你在下面的视频中看到的，该程序是一个 web 服务器，它将命令行程序作为作业运行。

只需编写一个 YAML 文件来描述程序的输入和输出，Jobson 将为参数创建输入字段，并在网页中显示输出。该程序创建的任何文件都可以下载。基本上，任何命令行程序都可以快速、轻松地进入一个 web 界面来管理它们。

如果一个程序需要很长时间运行，Jobson 会让你切换到别处，然后继续查看输出。您还可以中止作业或查看它收到的参数。Jobson 还可以用几种不同的方法验证用户，以防止任何人执行作业。

如果你真的想写一个图形化的程序，试试 [QTCreator](https://hackaday.com/2016/07/08/join-the-gui-generation-qtcreator/) 。或者，如果你想走这条路，你可以在网络浏览器中给[一个 shell。但这是我们见过的将命令行程序集中到一个地方进行监视和控制的最平稳的方法。整洁！](https://hackaday.com/2017/11/09/linux-fu-system-administration-made-easier/)
