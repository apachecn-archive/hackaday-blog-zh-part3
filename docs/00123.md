# 3D 打印电池表单

> 原文:[https://hackaday.com/2015/09/19/3d-printed-battery-forms/](https://hackaday.com/2015/09/19/3d-printed-battery-forms/)

当你试图展示一个项目时，可能发生的最糟糕的事情是什么？没电可能不是绝对最糟糕的事情，但它非常接近列表的顶部。[ker Martian]每年在世界创客大会上都有这个问题，展示的都是计算器。起初，他试图用没电的电池将电源线插入计算器，以固定电线。然而，在电线拉出之前，并没有太多的磨损。

解决办法？一种 [3D 打印电池形式，接受可连接外部电源的金属硬件](https://www.cemetech.net/news.php?id=760)。AAA 大小的塑料电池插入计算器的电池盒，小机器螺钉和垫圈形成连接点。

话说回来，你也没必要这么花里胡哨。我们已经看到人们[几次用一块木头](http://hackaday.com/2013/06/13/dummy-batteries-let-you-use-an-ac-adapter/)做同样的事情[(虽然我们不建议把 5V 放入 3V 设备)](http://hackaday.com/2010/08/19/replace-batteries-with-usb-power/)。尽管如此，3D 打印形式可能还有一些其他有趣的用途，如嵌入仪器或其他好东西。

如果你想打印工作电池，你需要等待一些不同的打印机技术(除非你算上用金属电极和电解质填充塑料表格)。然而，下面的视频显示了一些哈佛大学关于用特殊打印技术打印微型电池的研究。
