# 有人制造了克莱夫·辛克莱从未制造过的笔记本电脑

> 原文：<https://hackaday.com/2018/01/21/someones-made-the-laptop-clive-sinclair-never-built/>

辛克莱·ZX 光谱是 20 世纪 80 年代 8 位家庭计算领域的大玩家之一，几十年后是所有逆向计算社区中最活跃的运动之一。平台上有欣欣向荣的试玩场景，有新游戏发布，甚至有新的频谱硬件上市。

最有趣的硬件之一是 ZX Spectrum Next，这是一款在 FPGA 上实现了原始硬件和许多增强功能的 Spectrum 主板。它有一系列现代接口，与最常见的 48k 原始内存相比，它有一个兆字节的内存，以及一个允许连接 Raspberry Pi Zero 进行离线处理的端口。再加上最初 Sinclair 模型的设计者提供的一个相当有吸引力的案例，它已经成为人们渴望的对象。但它仍然是一个像原来一样的一体化桌面设备，他们没有制造便携式设备。[丹·伯奇改变了这一切，[凭借他精心设计的 Spectrum Next 笔记本电脑](http://www.dorchester3d.com/printing/blog/2018/01/zx-spectrum-next-laptop)。

他从一个漂亮的 CAD 设计开始，这个设计让人联想起上世纪 90 年代惠普 Omnbook 风格的笔记本电脑，但也有一些 Spectrum Next 风格的线索。这被送到 Shapeways 印刷，回来时看起来特别好。箱子里有一个 LCD 面板和 Next 的 HDMI 端口控制器、一个 Raspberry Pi、一个 USB 集线器、一个 USB 到 PS/2 转换器和一个超薄 USB 键盘。不幸的是，似乎没有电池包括在内，虽然我们相信，与一点独创性的一些空间可以找到一个。

其结果是一款尽可能好的 Spectrum 笔记本电脑，当然也和 Sinclair 或 Amstrad 可能制造的一样好，如果 8 位 micro 以某种方式幸存下来，成为 20 世纪 90 年代的另一个幻想版本，市场条件使其成为高端紧凑型笔记本电脑的外形。这个案例设计会让任何一个国产笔记本电脑设计者引以为豪，作为依据，我们只能敦促他考虑发布一些文件。
