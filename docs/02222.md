# 最小的 MIDI 合成器，再次！

> 原文:[https://hackaday.com/2016/05/08/smallest-midi-synth-again/](https://hackaday.com/2016/05/08/smallest-midi-synth-again/)

不满足于在 MIDI 插头中安装一个小型方波 MIDI 合成器，[Mitxela]继续将一个类似的[noise maker 塞进 USB 插头](https://mitxela.com/projects/silly_synth)本身。

除了体积小，代码也小，预算也少。它使用 [V-USB](https://www.obdev.at/products/vusb/index.html) 作为运行在 ATtiny85 上的 USB 库，以及一些无源部件。他的固件(显然)接收 MIDI 音符并发出方波。
T3】

没有什么太惊人的，直到你想到小型化的努力。为了让这个东西适合 USB 插头的金属外壳，[Mitxela]切掉一些塑料，然后把零件塞进去。我们所说的“塞满”是指把所有不适合的东西都碾碎。通孔齐纳二极管只是有点太高，所以他们去磨床。这件衣服的末端稍微突出了一点。更磨人。

一旦所有东西都装好了，就用一点绝缘胶带把它放回金属盒里，这样就完成了。仅从外形来看，它比他之前的作品要小得多，当然更没用！我们认为，通过削减压电蜂鸣器的情况下，进一步小型化的空间。我们也认为这没有意义。

这个项目似乎是从构建 USB MIDI 控制器的初步研究开始的，所以我们期待着很酷的东西出现。在那之前，请欣赏我们见过的迄今为止最小的单声道 USB MIDI 方波合成器。
