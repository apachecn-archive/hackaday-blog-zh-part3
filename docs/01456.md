# 生日遥控眉毛

> 原文:[https://hack aday . com/2016/02/10/生日遥控眉毛/](https://hackaday.com/2016/02/10/remote-controlled-eyebrows-for-your-birthday/)

我们不确定【亚历克】的爸爸是否真的*要求* [遥控眉毛作为他的生日](http://cassettepunk.com/blog/2016/02/08/remote-control-eyebrows/)，但看起来这就是他*得到的*！正如[亚历克]指出的，他的父亲确实有非常富于表情的眉毛，谁知道呢，他可能厌倦了自己一个人抬高和压低眉毛。所以也许这是件好事？但是对我们来说，它看起来还是有点像发条橙子。但我们在这里不是为了评判或讨论自由意志的问题。继续这个项目。(还有视频，下方休息。)

[![20160208_105209](../Images/de3ed21d185a1cf8ab54d5d29366e77a.png)T2】](https://hackaday.com/wp-content/uploads/2016/02/20160208_105209.jpg)

一台 ATmega328 ( ~~又名廉价克隆机~~亚历克写道，328 来自真正的 Arduino)被训练成响应红外信号运行电机。一个 L293D 和几个齿轮马达负责剩下的部分。缝纫线轴和线连接发动机到眉毛。虽然它在照片中不完全可见，并转向回到不确定我们会在家里做这个，牙签作为线和胶带的锚，固定在眉毛下，以获得最大的牵引力。

我们不得不说，我们最初认为这将是一个[高压肌肉控制黑客](http://hackaday.com/2014/07/18/backyard-brains-controlling-cockroaches-fruit-flys-and-people/)，我们感到欣慰的是，它不是。
