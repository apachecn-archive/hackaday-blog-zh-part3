# 神奇的 Blinky 电容感应镊子

> 原文:[https://hack aday . com/2016/04/23/magic-blinky-capacitive-sensing-镊子/](https://hackaday.com/2016/04/23/magical-blinky-capacitive-sensing-tweezers/)

电子镊子——可以测试两个触点之间的电压、SMD 电阻器的电阻或电路的电容的那种——非常酷，如果有点贵的话，也非常有用。[我们已经看到了这些智能镊子的商业版本](http://hackaday.com/2009/02/13/tools-smart-tweezers/)，[让它们变得更有用](http://hackaday.com/2009/08/31/hacking-the-smart-tweezers/)，以及[自制版本，它们仍然工作得很好](http://hackaday.com/2012/11/07/measuring-smd-parts-with-a-home-brew-version-of-smart-tweezers/)。就镊子而言，所有这些版本都相当大。[kodera2t]为今年的 Hackaday 奖提交的电子镊子版本朝着另一个方向发展:[这是仍然有用的最小的一套电子镊子](https://hackaday.io/project/11159-magical-tweezers)。

[kodera]的电子感应镊子只测量电容器，理由很充分:芯片帽通常不会印上值。这些镊子也不能在显示器上打印出瓶盖的价值。相反，如果电容值高于 0.1uF，这些镊子只会让 LED 闪烁。这很简单，但对大多数焊接工作来说却非常有用。

这对神奇镊子的电路非常简单，所有的智能都包含在一个非常小的附件中。设计的 PCB [kodera]比硬币电池更小，在一些铜带和可能的绝缘体的帮助下，这个设备可以安装到任何一对镊子上。是的，这是一个简单的工具，但这正是它的美妙之处，也是进入 Hackaday 奖的绝佳机会
