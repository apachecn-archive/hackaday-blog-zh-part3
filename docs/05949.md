# 一个为 NES 经典 DIY NES 优势控制器

> 原文：<https://hackaday.com/2017/05/19/a-diy-nes-advantage-controller-for-the-nes-classic/>

如果你是一个 80 年代的孩子，很有可能你年轻时有一台任天堂游戏机，经典的 8 位 NES。如果你是那些 NES 的拥有者之一，那么你渴望的外设很可能是任天堂受街机启发的优势控制器。它用一个全尺寸的街机操纵杆和按钮取代了游戏垫，并在此后的几年里成为一种昂贵和受欢迎的配件。

[Bbtinkerer]有一个 NES 经典，在经历了不止一个无法承受高强度游戏压力的操纵杆之后，[决定自己试着造一个](https://www.hackster.io/bbtinkerer/diy-nes-classic-advantage-controller-b62d1a)。优势是明显的模型可以复制，多亏了 [Wii Retropad 适配器](https://github.com/bootsector/wii-retropad-adapter)项目，他才能够做到这一点。

新的 Advantage clone 在布局上忠实于原版，具有快速射击的涡轮模式，尽管这种模式不是 80 年代的按钮，而是一个拨动开关。使用的操纵杆机构是三和 JLF，按钮是三和 OBSF-30s。他上传了一段视频，展示了成品被测试的过程。
