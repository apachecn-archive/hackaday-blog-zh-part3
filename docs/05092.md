# 宇宙魔方无限书桌

> 原文:[https://hackaday.com/2017/02/20/tesseract-infinity-desk/](https://hackaday.com/2017/02/20/tesseract-infinity-desk/)

我们都见过无限远镜。就连《星际迷航》电影里的斯波克先生也有一个。通常，这些都不是很大，挂在墙上。[QuackMasterDan]决定(在看了另一部电影《星际穿越》之后)尝试用同样的想法做一张桌子。我们不确定这是否会让你更有效率，但是如果你想提升你的办公室酷因子，考虑建造他的[宇宙魔方无限书桌](http://www.instructables.com/id/Tesseract-Infinity-Desk/)。事实上，我们认为这会很分散注意力。不过，肯定会成为话题的引子。

与普通的双板无限远镜不同，[丹的]桌子有六块板。他使用金属作为桌子的结构部分，桌面是一面丙烯酸镜子和一大块半英寸钢化玻璃的三明治(不出所料，可以在亚马逊上买到)。还有一种隐私膜可以把玻璃做成单向镜子。他还包括如何制作木头版本的说明。你可以在下面的视频中看到这张桌子。

这里真正的故事是机械的和艺术的。电子设备只不过是条状 LED 照明和一个通用控制器。我们认为一个定制的控制器可以从电脑上调节和控制灯光可能会很有趣。当一封重要的电子邮件到达时，你的桌子会闪烁。或者让桌子根据你的 CPU 温度改变颜色和亮度。

该视频还展示了一个快速构建的动作序列，当然，除非你在车库里尝试，否则看起来不会太难。我们以前见过一些受电影启发的 LED 建筑，但这张桌子可能是最大的。说到受电影启发的桌子，我们[记得一张受《少数派报告》启发的桌子。](https://hackaday.com/2015/10/21/raspberry-pi-powers-the-desk-of-the-future/)
