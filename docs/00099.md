# 自动泡泡枪只是因为

> 原文:[https://hack aday . com/2015/09/16/automated-bubble-gun-just-because/](https://hackaday.com/2015/09/16/automated-bubble-gun-just-because/)

最近去了一趟迪士尼乐园后，[托马斯]带着一把电动泡泡枪回家了。[托马斯]是一个成熟的男人。但是从什么时候开始我们不再享受吹泡泡的乐趣了？？很明显，一个项目将会使用这个有趣的小玩具。所以他决定让它自动化。

因此，在用他可靠的卡尺进行了一些测量后，[Thomas]打开电脑，开始用 SolidWorks 设计泡泡枪的外壳。这很简单。他设计它来固定泡泡枪，并允许他连接一个小的遥控伺服电机来触发开关。把它连接到 Arduino 微处理器上，他现在可以远程触发它了。

这个项目最酷的部分是他只用了不到两个小时就完成了整个项目。他甚至在等待零件打印出来的时候编写代码。接下来，一队 WiFi 控制的自动泡泡枪！
