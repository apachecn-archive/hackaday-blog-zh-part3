# 国产 8 位 CPU 是一个结实的 Blinky 建设

> 原文：<https://hackaday.com/2018/02/07/home-made-8-bit-cpu-is-a-wiry-blinky-build/>

对一些人来说，这看起来像是随机的一堆电线，但它远非随机的:[保罗·康斯坦丁诺]自己从零开始建造了这个 8 位 CPU。他使用电线和 74HC 移位寄存器芯片，加上一系列 led 来显示各种寄存器，建立了他的非凡创造。

它的最高运行频率为 5MHz，拥有一条 8 位数据和地址总线，尽管后者可以扩展到 16 位。它(还)没有挖掘比特币，但它可以做像[玩马里奥主题](https://www.youtube.com/watch?v=Gp5KDGqQYP0)这样的事情。他最近增加了将数据写入闪存的功能，并且他正在寻找一个键盘来使编程更容易。

目前，他必须通过设置 DIP 跳线对 CPU 进行编程。这是一个令人印象深刻的建筑，虽然有点令人害怕，但[保罗]说他花了几天时间来设计，一周左右来建造。我们已经看到了一些试验板 CPU 构建，([其中一些更整洁](https://hackaday.com/2017/04/10/8-bit-breadboard-computer-is-up-to-8-hours/))和[构建具有类似的移位寄存器芯片](https://hackaday.com/2017/03/28/another-74xx-series-cpu/)，但这一个在 blinky light 和 mad genius 赌注中得分很高。
