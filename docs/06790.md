# 倒 1200 茶:铸造在灭火器里

> 原文:[https://hacka day . com/2017/08/15/purging-1200-tea-foundry-in-a-灭火器/](https://hackaday.com/2017/08/15/pouring-1200-tea-foundry-in-a-fire-extinguisher/)

让我们面对现实吧——大多数家庭铸造厂的设计都有不尽人意的地方。大多数铸造厂都擅长熔化金属，但在浇注金属时，笨拙的操作很容易导致可怕的结果。这就是为什么我们欣赏进入[这个电熔炉铸造厂](https://www.youtube.com/watch?v=ooVUir_V2DM)的想法。

![](../Images/1945155778f2a7bbbd5c7f6e8080b982.png)当然，电动铸造厂缺乏燃气甚至木炭燃料铸造厂的一些性感，但通过避免明火和射击火花，[涡轮征服 Mega Eagle]能够将坩埚集成到铸造体中，并创造出看起来像熔融铝保温瓶的东西。

身体是一个断头的灭火器，而坩埚看起来只是一段钢管。电炉加热元件包裹在坩埚周围，其 PID 控制由外部控制器和固态继电器负责。用珍珠岩隔热并配有把手，现在倒起来就像泡一杯 1200 磅的好茶一样安全。

你可能还记得【涡轮征服巨鹰】对电子铸造厂情有独钟，尽管我们不得不说当前工作的适合度和完成度远远超过了他之前使用旧电炉快速而肮脏的构建。
