# 用这个袖珍键盘弹奏和弦

> 原文:[https://hack aday . com/2015/11/29/strike-a chord-with the-pocket-keyboard/](https://hackaday.com/2015/11/29/strike-a-chord-with-this-pocket-keyboard/)

[Brian]成功抵制了左鲨鱼服装的诱惑，在今年的万圣节扮成了赛博朋克。在他的服装道具中，有一个很小的单手和弦键盘，可以很容易地放进他的口袋里。现在他可以把几个键帽粘到一些小东西上，然后就收工了。相反，[布莱恩]制造了一个功能齐全的模块化和弦键盘，可以通过蓝牙或 USB 进行通信。

你会问，什么是和弦键盘？不再是一次输入一个按键，而是将一个小得多的按键组合成有意义的和弦。一旦你知道你在做什么，它比标准键盘快得多。如果你曾见过法庭记录员在一个小机器上敲敲打打，你就见过和弦键盘在工作。我们自己的【艾略特·威廉姆斯】[在整个夏天](http://hackaday.com/2015/07/06/stenography-yes-with-arduinos/)详细报道了这个话题。

[Brian]的键盘有七个键，每个手指一个，拇指三个。标准 104 键上的任何键都可以通过用手指按下与中间、近或远拇指键相关的组合键来制作。他能够把所有的硬件都装进这么小的 3D 打印包里，这让我们印象深刻。它基于 Arduino micro，使用 Adafruit EZ Key 与手机或平板电脑进行蓝牙通信。

最终的计划是把它做成一个腕戴式和弦键盘，随着你手腕的轻弹，键盘可以伸缩。[Brian]已经在这方面取得了一些进展，开发并打印出了这种机制。但正如你在休息后的视频中看到的，为它添加键盘对他选择移动的爱好伺服来说太多了。尽管如此，如果他能拨进来，这将是可怕的！

该键盘还有一个 ADXL335 加速度计分线点，这意味着它可以用作倾斜鼠标。不过，蓝牙和倾斜鼠标功能都不是必不可少的——如果你想自己开发，把这两个功能都去掉，就没有必要修改代码。
