# 纸做的微型发动机模型

> 原文：<https://hackaday.com/2016/12/11/miniature-engine-model-made-of-paper/>

你可以用纸做很多东西，但是一个不到一英寸宽的单冲程发动机模型？这是一个新的，由俄罗斯黑客【Aliaksei Zholner】提供，他用纸建造了一个[非常出色的单冲程发动机模型(在俄罗斯，](http://only-paper.ru/forum/38-22450-1)[通过谷歌翻译](https://translate.google.com/translate?sl=auto&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fonly-paper.ru%2Fforum%2F38-22450-1&edit-text=&act=url)翻译版本)。它的直径不到一英寸，由压缩空气驱动，精确模拟了单冲程发动机的旋转动作，气缸中的活塞驱动飞轮，从而形成发动机循环。

这位创造者设法让它以每秒 60 转的速度运转，也就是每分钟 3600 转。对于一些纸和胶水来说，这是一个令人印象深刻的速度，甚至还有一个输入限制器，可以控制驱动模型的气流。我们以前展示过一些有趣的纸创作，比如这个 [papercraft 机器人](https://hackaday.com/2012/09/26/astounding-papercraft-skills-result-in-this-working-robot/#more-86198)和一个 [Strandbeest](https://hackaday.com/2011/05/01/papercraft-strandbeest-is-a-great-rainy-day-project/) ，但这个是一个超越。【Aliaksei】也为此制作了[计划和模板](https://drive.google.com/file/d/0B0xqQIZvwhdiRG1UbnkycERiMkE/view)，所以那些手稳的人可以继续尝试制作他们自己的。
