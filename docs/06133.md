# 廉价压力清洗机的喷水切割器

> 原文：<https://hackaday.com/2017/05/31/a-water-jet-cutter-from-a-cheap-pressure-washer/>

我们已经习惯了数控铣床和 3D 打印机成为我们车间的主食，由于国际贸易的奇迹，即使是一个普通的激光切割机也不是大多数实验者无法企及的。但是有一种工具，到目前为止，除了商业运作和极其富有的水切割机之外，其他的都被避开了。这些机器使用高压水射流，通常携带磨料颗粒流，切割下方的材料。从我们的角度来看，它们很有趣，因为它们可以切割金属，这是我们通常无法用激光切割机切割的。

你可能会认为实验者不可能自己制造一个水切割器，但是[应用科学]已经证明了这一点。他拿了一个便宜的压力清洗机，[对其进行了改造，使其为水切割头](https://www.youtube.com/watch?v=Lg__B6Ca3jc)产生更高的水压。

他对修改的详细描述构成了一款非常有趣的手表，我们将视频放在了休息时间的下方。通过使用新加工的套筒和更强的弹簧修改洗衣机的压力开关，可以获得更高的压力。对清洗器开关如何工作的描述本身就很有趣。然后，我们被处理成一个完整的拆卸水切割头，磨料饲料，碳化钨管，和红宝石喷嘴。这最后一个部件出奇的便宜。然后，他给了我们一个其设计的运行，特别是关于选择孔的大小，以匹配泵。最后，我们来看看他的磨料进料系统，以及他用来防止水从漏斗中流出的塑料漏斗。

目前，切割机是静态的，但他的下一步显然是把它带到某种形式的数控工作台。如果这个项目让水切割离群众更近一步，我们等不及了！
