# 黑线追随者:一个现代刚毛机器人

> 原文:[https://hack aday . com/2016/05/25/black-line-follower-a-modern-bristle bot/](https://hackaday.com/2016/05/25/black-line-follower-a-modern-bristlebot/)

我们已经有一段时间没有在刚毛机器人上看到太多的动作了，这太糟糕了。因此，我们很高兴看到[极限电子]对经典入门“机器人”的演绎:黑线跟随器。这些东西的美妙之处在于它们的简单性，所以我们只给你指出[的构建说明](http://www.extremeelectronics.co.uk/blf-bot-black-line-follower-construction-instructions/)，剩下的就交给你了。

最初的 bristlebot 是对电子学的一个极好的介绍，因为它足够简单，你可以很快地拼凑出一个。你只需要一个电池、一个呼叫器马达和一个牙刷头。但它会去它想去的地方，而不是你想让它去的地方。

增加转向系统就像[把两个刚毛机器人](http://hackaday.com/2009/06/16/controllable-bristlebot/)绑在一起，一次启动一个马达完成一个转弯一样简单。黑线跟随器就是这种风格。

当然，任何好的想法都可以走极端，比如这个巨大的重量转移刚毛机器人(T1)，或者这个超微型的红外控制刚毛机器人(T3)。

但那已经是五年前的事了。刚毛机器人创造力的强大引擎发生了什么？b 型机器人经历过它最辉煌的时刻了吗？或者我们只是在等待下一代开始行动？
