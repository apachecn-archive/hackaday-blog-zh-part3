# 纸电路用压缩空气做二进制数学

> 原文：<https://hackaday.com/2017/06/12/paper-circuit-does-binary-math-with-compressed-air/>

我们大多数人都能心算简单的数学，但有些人似乎不使用纸张就无法将 0 和 3 之间的两个数字相加，就像[Aliaksei Zholner]用他的完全由纸张制成的[流体加法器电路](https://www.youtube.com/watch?v=yvANcR4mQ7M)所做的那样。

[![](img/640cc355242f721580c9f7d430e5899c.png)](https://hackaday.com/wp-content/uploads/2017/06/3403905.png)

Pneumatic AND gate

“唯一的纸”论坛上[【Aliaksei】的翻译帖子](https://translate.google.com/translate?hl=en&sl=ru&tl=en&u=http%3A%2F%2Fonly-paper.ru%2Fforum%2F38-23291-1&sandbox=1)中有一些很好的细节，这是一个俄罗斯网站，致力于完全由纸创建的令人难以置信的详细模型。[Aliaksei]从逻辑电路的基本构件开始，即与门和或门。输出由双头活塞在腔室中的位置决定，输出状态由活塞指示，活塞在受压时会发出信号。加法器看起来很复杂，但它实际上只是一个半加法器和全加法器，连接方式与 CMOS 或 TTL 门完全相同。下面的视频展示了它的作用。

如果[Aliaksei]的名字看起来很熟悉，那是因为我们之前已经展示过他的纸上创作，包括这个[工作器官](http://hackaday.com/2017/02/14/god-of-papercraft-builds-working-organ-used-for-own-adulation/)和[一个微小的工作单缸发动机](http://hackaday.com/2016/12/11/miniature-engine-model-made-of-paper/)。我们很高兴他进军数字世界，我们期待着接下来的一切。
