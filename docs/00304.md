# Google OnHub 可以有根

> 原文:[https://hackaday.com/2015/10/07/google-onhub-can-has-root/](https://hackaday.com/2015/10/07/google-onhub-can-has-root/)

深入到设备的根目录总是好的，尤其是如果有问题的设备是您自己的设备。谷歌产品允许访问根目录并不奇怪，但是 OnHub 需要找到隐藏的“开发者模式”开关，而[Maximus64]已经做到了。谷歌的工程师们一直在偷偷摸摸地使用这个按钮，把它放在一个螺纹螺丝孔的底部。之前有人见过这在其他硬件上实现吗？

没有关于这方面的博客帖子，但是[Maximus64]在 YouTube 上分享了一个视频[引导我们通过根和取消根谷歌的 OnHub](https://www.youtube.com/watch?v=UsD7hBjVzr0) 的步骤，它是在休息后嵌入的。他还在视频的描述中提到“wiki 即将推出”，所以我们会继续关注它的更新。

我们在 8 月份报道了[产品发布](http://hackaday.com/2015/08/19/googles-onhub-goes-toe-to-toe-with-amazon-echo/)，并听到了一些关于该设备的评论/意见，但不足以做出固执己见的假设。根植设备似乎不会增加 OnHub 的局域网端口数量，但我们认为这仍然值得努力。
