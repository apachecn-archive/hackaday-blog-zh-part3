# 苹果硬币银行种下储蓄的种子

> 原文：<https://hackaday.com/2018/06/25/apple-coin-bank-plants-the-seed-of-saving/>

想想存钱罐。在这种天真、温顺的表达背后，是一个资本主义的隐喻，它正等着毁掉你为那辆小轮车或其它什么东西存钱的美好记忆，正是这些东西让你明白了一美元的价值。尽管把一枚硬币投入投币口很有趣，但是省下你的便士并学习财务责任的行为可能会更有吸引力。

似乎[朱姆沃尔特]也有同样的感觉。他为他的孙子们设计了一个硬币银行，在存款过程中扮演更积极的角色。将一枚硬币放在平台上，苹果表面的上半部分被一只手臂推开，手臂将硬币拉回到里面。

继续省钱的主题，[gzumwalt]没有使用微型甚至 555。不，这个项目的核心是一对微型杠杆开关，一个小齿轮电机，和 4.5V DC。当硬币击中平台时，第一个开关启动马达。该电机驱动一个模仿 [Hoeckens 连杆](https://en.wikipedia.org/wiki/Hoeckens_linkage)的 3d 打印机构，该机构将旋转运动转换为(几乎)直线运动。第二个开关停止循环。迷茫？休息之后你可以全力以赴。

别担心，去糖果店的时候，孩子们不用把苹果切成片，因为底部有一个旋入式开口。这是因为[gzumwalt]是 3d 打印和设计的奇才。不服气？看看他的[气球动力引擎](https://hackaday.com/2017/03/10/3d-printed-engine-chugs-away-on-balloon-power/)或者他的[空中飞行飞机](https://hackaday.com/2017/12/27/3d-printed-airplane-engine-runs-on-air/)。
