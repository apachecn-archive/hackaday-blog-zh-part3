# 肌肉钢丝笔为杜克核武器而舞

> 原文：<https://hackaday.com/2015/12/22/muscle-wire-pen-dances-to-duke-nukem/>

[serdef]显然只是在这里找点乐子。[没有人会*需要*一支由 MIDI 同步的白板笔，随着《毁灭公爵》的主题起舞。](https://hackaday.io/project/8870-duke-nukem-dancing-marker-midi-muscle-wire)

但是如果你手头上有所有的部件(一个[高流动性 MIDI 驱动的中继板](http://www.highlyliquid.com/)直接连接到声卡，[一些肌肉线](https://en.wikipedia.org/wiki/Nickel_titanium)，磁带，当然还有一支白板笔)，我们很肯定你会的。你可以在休息时间下面的视频中观看跳舞的钢笔。

该项目实际上是记录[serdef]的肌肉丝的属性，他发现，用一小段肌肉丝并不能真正达到预期的效果。所以他加了更多的电线。我们一直想玩肌肉线，在[serdef]的第二个视频中，我们惊讶于它对改变电压的反应如此之快。

现在，跳舞笔并不是我们见过的最复杂的肌肉线项目。这个奖项也没有颁给这个镍钛诺驱动的尺蠖。你知道[微软的 Surface 里面有肌肉线吗？](http://hackaday.com/2015/11/04/microsoft-surface-book-teardown-reveals-muscle-wire-mechanism/)
