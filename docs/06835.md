# 眩晕枪与 220 伏市电

> 原文:[https://hack aday . com/2017/08/19/stun-gun-vs-220v-mains-electricity/](https://hackaday.com/2017/08/19/stun-gun-vs-220v-mains-electricity/)

那些无畏的乌克兰人又来了！这一次，他们给墙壁插座一些高压眩晕枪电击并观察结果，因为[Kreosan]决定看看当你在电源插座上使用眩晕枪时会发生什么。令人惊讶的是，他们仍然活得很好，并创造了更多的视频。。

电击电灯开关会炸坏一些灯泡，而电击插着电视的延长线会炸坏电视。看起来这些家伙从来没有用完油炸的器具，或者完全疯狂的实验去尝试，没有人真的有这个胃口。

虽然他们的实验是极端的，但他们知道自己在做什么，因为他们是电气专业人员，所以除非你也真的知道自己在做什么，并了解其中的风险，否则可能会袖手旁观。视频在下方休息，供大家欣赏。

我们曾经报道过他们一些同样可怕的黑客行为，比如[市电电压电线](http://hackaday.com/2015/12/11/diy-electroluminescent-wire-russian-style/)和[通过高压架空电线给你的手机无线充电。](http://hackaday.com/2017/04/10/wirelessly-charge-your-phone-from-high-voltage-power-lines/)
