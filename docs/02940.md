# 黑客日奖品入口:一种更好的作弊方式

> 原文：<https://hackaday.com/2016/07/17/hackaday-prize-entry-a-better-way-of-cheating/>

信不信由你，一些视频游戏仍然是为个人电脑开发的。随着视频游戏而来的是作弊代码，当它们在个人电脑上时，这意味着使用键盘。您可以轻松地对任何微控制器进行编程，只需按一下按钮，就可以通过 USB 端口发送一串字符。信不信由你，很多人还没有把这两个事实联系起来。danjovic 设计了一个简单便宜的 USB 击键发生器,用于快速输入作弊代码。

[danjovic]是基于 Digispark 构建的，digi spark 是一种廉价的支持 USB 的 ATtiny85 开发板。当然，这意味着没有太多的引脚可供使用——只有四个 I/O 引脚，其中一个通过 LED 接地。这样就只剩下三个 I/O 引脚了，但是[danjovic]成功地在他的项目中加入了七种不同的作弊手段，使用了二极管和一些几乎是 charlieplexing 的东西。

如果你想知道，这是一个非常便宜的项目。[danjovic]使用的是中国的 digispark 克隆产品、一些 1N4148 二极管和一些轻触开关。任何人只要有一个库存充足的零件抽屉，或者在易贝上花 10 英镑，就能造出这个东西。如果你想要这个项目的工作证明，你可以看看下面的演示视频。
