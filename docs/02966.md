# 拆开任天堂游戏机

> 原文:[https://hackaday . com/2016/07/16/撕裂任天堂游戏机/](https://hackaday.com/2016/07/16/tearing-apart-the-nintendo-playstation/)

90 年代中期对视频游戏硬件来说是一个奇怪的时代。有些设备可以播放光盘上的视频。那些从未流行过。虚拟现实是下一件大事。那从未流行过。索尼 PlayStation 最初是超级任天堂的附加产品。那从未流行起来，但是一些原型单位被生产。其中一台“任天堂 Playstations”原型机被运到了一家破产的公司。最终，这家公司的资产被拍卖，这台令人难以置信的稀有游戏机被[特里·迪堡]以 75 美元买下。

[特里]允许[本·赫克]撕毁这段视频游戏历史，[并且他有视频证明](https://www.youtube.com/watch?v=ug-CyGXMabg)这是索尼和任天堂之间的合作。

从外表上看，这款任天堂 PlayStation 或多或少符合你的预期。在该设备的背面有一个 SNES 视频连接器，一个 90 年代索尼 Handicam 上可以找到的附加视频输出，以及一个奇怪的串行端口连接器。当然，还有一个 CD 驱动器和一个 SNES 盒式磁带的插槽

在里面，事情变得有点奇怪。CD 驱动器是完全独立的，许多随机线索表明这肯定是一个原型装置。令人惊讶的是，这个原型单元的 PCB 与标准的超级任天堂没有太大的不同。RAM 芯片的布局是一样的，系统的架构也是一样的，甚至一些部件号也是一样的。这意味着任天堂 PlayStation 没有足够的内存来充分利用光盘。为了解决这个小缺点，该系统的设计者在墨盒上安装了 256k 的内存。是的，使用 CD 驱动器需要一个盒式磁带，而且盒式磁带本身相当昂贵。

随着这次拆卸，我们终于知道这个任天堂 PlayStation 不是最终成为索尼 PlayStation 的原型。它基本上是一个超级任天堂，有一个光盘插件，任天堂和索尼的世嘉光盘的答案。抛开任天堂和索尼之间的政治不谈，即使这款任天堂 PlayStation 上市，它也可能会是一个巨大的失败。

随着拆卸和文档的完成，留给[Ben]唯一要做的就是让它工作起来。现在，有技能的人已经很好地看了这台机器的内部，这是一个非常好的机会，这个经典的控制台将再次复活。这将出现在[本]的下一个视频中，我们迫不及待地想看看会发生什么。
