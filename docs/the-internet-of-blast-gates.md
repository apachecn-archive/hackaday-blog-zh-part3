# 防爆门的互联网

> 原文：<https://hackaday.com/2017/12/11/the-internet-of-blast-gates/>

没有什么比建一个堆满工具的商店更好的了，但是即使是这个令人羡慕的任务也有很多令人厌烦的工作要做。你必须运行电力，你必须建造工作台，你需要建造一个集尘系统。最后一种通常只是将一堆管子和管道装配在一起，并添加几个防爆门，以指导吸尘系统吸入车间内的各种工具。

对于大多数拥有少量工具和集尘口的商店来说，手动打开和关闭每个防爆门是一项令人讨厌的必要任务。但是，如果所有这些都是自动化的，会怎么样呢？这就是[鲍勃]在我喜欢做的事情上所做的。他自动化了他的集尘系统。当工具打开时，真空也随之打开，右边的防爆门自动打开。

这个建筑的第一部分正是你所期望的在商店里安装一个灰尘收集系统。主线是绑在椽子上的 PVC 下水管道。是的，这根管子是接地的，否则就没什么意思了。真正的乐趣来自电子产品。[鲍勃]将标准防爆门改为伺服驱动。每个单独的工具都连接到插头上的电流传感器，所有这些都连接到 Arduino。在集尘系统上安装了一个大的继电器后，唯一阻碍完全自动化的就是一点代码。

这个项目是[Bob]早期对他的集尘系统进行 Arduino 化的延续，其中所有的防爆门都由伺服系统、Arduino 和数字键盘控制。这是一个非常实用的系统，可以绕过整个“靠在机器上开门”的问题，但它仍然不是傻瓜——必须有人按下按钮才能开门。这个新系统在很大程度上是完全自动的，不需要操作者的任何思考。这是一个很好的东西，也是便宜的 Arduinos 的一个很好的应用，让商店生活变得更简单。
