# 问 Hackaday:你能用折纸做什么？

> 原文：<https://hackaday.com/2017/02/26/ask-hackaday-what-can-you-do-with-origami/>

在某种程度上，我们大多数人都学过一点古老的折纸艺术。这是一种迷人的艺术形式，能够通过简单地按照正确的顺序折叠纸张来创建一个可识别的模型会令人非常满意。一旦我们掌握了经典的起重机模型，我们大多数人都会转向其他追求，但折纸背后的数学可以引导一些从业者超越纯粹的艺术，走向更实用的结构，如[这个执法使用的折叠式弹道屏障](https://me.byu.edu/node/11203)。

这个 50 磅重的凯夫拉尔和铝结构来自杨百翰大学的机械工程学院，特别是来自柔顺机构研究项目。柔顺机构通过弯曲或偏转而不是离散部件之间的连接来移动，这种弹道屏蔽就是一个很好的例子。该机制基于吉村折痕模式，可以用一张纸快速建模。将它扩大到全尺寸的结构，足够轻，可以在战场上使用，但又足够强，可以抵挡一发 0.44 口径的马格南子弹，这可不是一个简单的壮举。但正如下面的视频所示，原型有很大的潜力。

现在轮到你了:你见过哪些顺应机制的应用？潜在的应用范围从微注射细胞的 MEMS 连接到在轨道上展开的巨大天线。我们之前展示过一些类似折纸的结构，比如[这个自我组装机器人](http://hackaday.com/2014/08/11/self-assembling-origami-robots/)或者[一个折叠四轴飞行器](http://hackaday.com/2015/05/26/foldable-quadrotor-is-origamilicious/)，但是这些都不符合标准。不过，这种优雅的[抛物面卫星天线](http://hackaday.com/2016/12/09/raincube-spreads-its-umbrella/)更像它。有[设计折纸](http://www.langorigami.com/article/treemaker)的应用和[一个数学基础](http://courses.csail.mit.edu/6.885/fall04/erik_notes/anydpi/L18_paper.pdf)的领域；有人尝试过使用这些工具来设计柔性结构吗？请在下面的评论中发表意见。
