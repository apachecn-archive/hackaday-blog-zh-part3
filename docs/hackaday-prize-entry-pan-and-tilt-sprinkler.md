# Hackaday 奖参赛作品:平移和倾斜洒水喷头

> 原文：<https://hackaday.com/2017/07/26/hackaday-prize-entry-pan-and-tilt-sprinkler/>

有几个非常受欢迎的灌溉系统进入了今年的 Hackaday 奖。事实上，去年该奖项最佳产品部分的获奖者是 Vinduino T1，这是一种用于葡萄园的土壤湿度监测器。这些灌溉系统大多使用滴灌，或者规模相对较小。如果你需要更强大的东西呢？这就是[Patrick]的[pt 洒水喷头的用武之地](https://hackaday.io/project/20572-ptsprinkler)。这是一个巨大的草坪洒水器，与电脑控制的云台相连。把它想象成一个遥控的超级透雨器，或者是水枪的互联网。不管怎样，这都是今年 Hackaday 奖的绝佳参赛作品。

pt 洒水喷头的设计使用了尽可能多的低成本、现成的部件。这是从一个重型室外云台灌溉电磁阀开始的。

这个洒水喷头的想法是首先在草坪上手动定义洒水喷头应该覆盖的形状。从那里，电子设备计算出这个绿色多边形的填充图案。到目前为止，[Patrick]有一个电子板，它可以在树莓 Pi 的帮助下移动云台。你可以看看下面的视频。
