# Hackaday 奖品入口:Dodo 6502 游戏系统

> 原文：<https://hackaday.com/2016/09/22/hackaday-prize-entry-dodo-6502-game-system/>

如果你是一个特定年龄的游戏玩家，你很可能会对你年轻时的 8 位电脑和游戏机情有独钟。有一段时间，令人上瘾的游戏通过最少的图形实现，游戏音频是最刺耳的方波，而不是身临其境的声音景观。

前一段听起来耳熟吗？那我们就可以给你这个装置了。[Dodo 是一款掌上游戏机，它可以追溯到那个时代](https://hackaday.io/project/9325-dodo-6502-game-system),配有 6502 处理器和 128×64 像素有机发光二极管屏幕。游戏从插入式 EEPROM 盒式磁带中加载，声音是合适的周期数字方波音调。这是[彼得·诺伊斯]的创意，他说，当它的运动游戏足够有趣，可以娱乐他 4 岁的孩子时，他就会认为它是完整的。

原型 Dodo 是由两块堆叠的 PCB 制成的手持式设备。上部有显示屏和按钮，而下部有传统的 6502 和相关的通孔 DIP 格式的芯片组。虽然不是，但小型化并不是这些游戏机的游戏名称。最棒的是，所有的控制台资源都可以在 GitHub 库中找到，所以你们都可以玩。

多年来，6502 已经在 Hackaday 上的大量项目中出现。现在它出现在黑客日奖中。
