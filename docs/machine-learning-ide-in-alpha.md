# Alpha 中的机器学习 IDE

> 原文：<https://hackaday.com/2017/06/27/machine-learning-ide-in-alpha/>

Machine 是[使用 TensorFlow 构建机器学习系统](https://machineui.co/)的 IDE。你可以注册阿尔法，但首先，看看下面的视频，看看这是怎么回事。

您将在视频中看到，您可以为模型导入数据，然后进行训练(在本例中，在图像中找到小胡子)。您将看到 IDE 邀请了一种迭代开发方法，因为您可以更改参数、运行实验并查看结果。

IDE 与“云”同步，因此您可以在多台计算机上使用它，并轻松回滚到以前的结果。我们不知道 IDE 何时会离开 alpha 状态(或者 beta，就此而言)，但是团队的目标是发布 Machine 的免费版本，以鼓励广泛采用。

如果你想了解更多关于 TensorFlow 的[，你来对地方了。我们还介绍了](https://hackaday.com/2017/04/11/introduction-to-tensorflow/)[一个基本项目](https://hackaday.com/2017/03/24/ten-minute-tensorflow-speech-recognition/)，如果你愿意这样开始的话。你还可以找到一些[很好的背景材料](https://hackaday.com/2017/06/08/from-50s-perceptrons-to-the-freaky-stuff-were-doing-today/)，可以追溯到早期基于感知器的神经网络。
