# 运动奇迹中的投影映射

> 原文:[https://hack aday . com/2016/10/30/projection-mapping-in-motion-amazes/](https://hackaday.com/2016/10/30/projection-mapping-in-motion-amazes/)

投影贴图相当神奇；做得好的话，当一个建筑的正面开始出现抽象的几何物体，或者在我们眼前倒塌，这绝对是一个奇迹。"[动态投影映射到变形的非刚性表面上](http://www.k2.t.u-tokyo.ac.jp/vision/DPM/)"把它带到下一个层次。(看下面的视频。)

东京大学石川渡边实验室的一个小组有一项技术，他们用一种仅在红外线下可见的墨水在目标上覆盖许多点。一个高速(1000 FPS！)相机和一些非常快速的图像处理然后不仅计算出表面如何变形，而且计算出它是哪个表面。这使他们能够交换纸张，并实时将投影显示在纸上。

一定要看视频。他们将风扇对准投影映射的 t 恤，跳舞的动画斯坦福兔子在微风中扇动，这一点真的很棒。

想要更多 2010 年早期的投影制图热潮吗？查看[这篇关于用工业机器人手臂](http://hackaday.com/2013/10/03/step-into-the-box/)进行投影映射的帖子，或者一个[投影映射沙箱](http://hackaday.com/2015/08/22/augmented-reality-sandbox-using-a-kinect/)。如果这对你来说还不够，看看这段来自 2012 年阿蒙·托宾[现场巡演的片段。](https://www.youtube.com/watch?v=YNlXO4G_xXM)
