# 你的工作不会随着磁力钻压机台钳而移动

> 原文：<https://hackaday.com/2017/09/08/your-work-wont-move-with-a-magnetic-drill-press-vise/>

安装工件通常是任何加工操作中最难的部分。目的是固定工件，使其在加工过程中不会移动，这样就不会妨碍加工。磁性卡盘是安全和灵活固定工件的最佳选择，正如[这个简单的车间制造的电磁虎钳](http://www.makeitextreme.com/en/newsroom/entry/VIDEOS/elebase)所示。

看起来,[Make It Extreme]在为商店建造[物料搬运起重机](http://hackaday.com/2017/01/15/heavy-lift-electromagnet-from-microwave-oven-transformers/)时，学到了一两件关于将微波炉变压器转换成电磁铁的事情。他们的磁性虎钳是为钻床设计的，但可能是铣床、磨床甚至数控刳刨机的最佳选择，它有一个简单而坚固的钢架。两个独立的平台在虎钳的底座上滑动，每个平台包含两个断头的 mot。单独连接到主电源进行选择性控制，并封装在环氧树脂中，磁铁似乎真的可以完成这项工作。下面的视频显示了一块非常厚的钢板悬在一块磁铁上，上面有一个孔；这下压力很大，但工件不动。
