# 电池的快速历史

> 原文：<https://hackaday.com/2016/10/29/a-quick-history-of-the-battery/>

[Colin] [告诉我们这一切都始于[Benjamin Franklin]的电容器电池](https://www.youtube.com/watch?v=4bUxdLtG8PI)。这是一堆莱顿瓶串联在一起，甚至还没有涉及化学，但命名法坚持下来了，不久就演变成我们今天使用的词。

为了改变这个词，[事情变得化学](http://hackaday.com/2013/09/30/diy-soda-can-battery/)。[Alessandro Volta]介绍他的伏打堆。一旦科学家们抓住稳定反应的想法，给他们源源不断的魔法小精灵来玩，没过多久，伟大的头脑就将注意力转向改进这项新技术。

在这个被所有技术人员喜爱的胜人一筹的经典游戏中，我们很快跳到了现代。在这个时代，没有人能够摆脱持续交流的重负。这一切都要归功于一项理论上从第一堆开始就没有改变的技术。休息后的视频。
