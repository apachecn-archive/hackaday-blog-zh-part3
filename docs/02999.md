# 最简单的跳跃袋鼠机器人

> 原文:[https://hack aday . com/2016/08/06/simplest-jumping-kangaroo-bot/](https://hackaday.com/2016/08/06/simplest-jumping-kangaroo-bot/)

我们从 BEAM robotics 获得的一个想法是，机器本身，而不是大量的处理能力，可以做很多事情。你的手可以抓住东西，人类已经通过操纵东西过上了相当不错的生活(他边说边打字)。这些都与大脑无关；都是机制的问题。

这就把我们带到了单马达机器人[“跑步者”机器人](http://www.instructables.com/id/The-Runner-a-Simple-Running-Robot/)。我们承认，看到它不能用两条腿和前轮跑起来，我们有点失望。尽管如此，这仍然是一个令人敬畏的机制，并以一种非常戏剧化的方式完成了运动工作。我们想知道，使用两个马达是否能让它转向，或者只是翻转过来，在它的背上挥舞。波士顿动力 RHex 机器人展示的[六条“腿”设计肯定能完成任务。](https://www.youtube.com/watch?v=ISznqY3kESI) 

不管怎样，你没有理由不做这个。查看下面的视频，获取更多灵感。当你这么做的时候，看看另一个古怪的单电机机构 [Gyroman](https://hackaday.com/2016/05/08/gyroman-walks-with-just-one-motor/) 。你最喜欢什么？
