# 需要力量？你要的是改性木！

> 原文:[https://hack aday . com/2018/02/12/need-strength-its-modified-wood-you-want/](https://hackaday.com/2018/02/12/need-strength-its-modified-wood-you-want/)

木材无疑是目前用途最广的材料之一。根据种植它的树木的种类，它可以有各种各样的颜色和物理属性，并且它已经被制成了几千年来所有可以想象的设备、产品和结构。但是它也有缺点，最明显的一点是它的强度比不上其他材料。为了用一块木头承受巨大的力，这块木头必须做得比相应的一块钢大得多，这在屋顶桁架中不成问题，但在汽车车身中却非常困难。

过去曾有过各种强化木材结构的尝试，最近一项尝试发表在《自然》杂志上。其中描述了首先在化学浴中处理天然木材以除去木质素并仅留下纤维素结构，然后在高温下持续压缩的方法。这使得纤维素纤维相互交错，形成密度更高的木板，其强度被描述为接近钢的强度。他们发布了一个视频，我们把它放在了 break 下面，展示了他们的材料的一些弹道测试。

所有新材料都令人感兴趣，但假设这种材料可以商业化，它会带来一系列特别令人兴奋的可能性。例如，木制机动车辆，木制飞机或船只的新技术，或者作为碳纤维目前可能应用的一些应用中的替代品。

我们在过去研究过一个非常相似的生产透明木材的过程。不过，对于那些把这从深奥的科学论文变成迷人的可能性的普通读者来说，好消息是这可以在家里完成。你们中的任何人能复制这个紧迫的步骤来更上一层楼吗？
