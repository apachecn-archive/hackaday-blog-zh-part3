# “你沉了我的镝！”:周期表战舰

> 原文:[https://hack aday . com/2016/02/05/you-sink-my-镝-周期表-战舰-游戏/](https://hackaday.com/2016/02/05/you-sank-my-dysprosium-periodic-table-battleship-game/)

现在的孩子，他们过得很轻松。回到过去，我们通过反复听汤姆·莱勒的《元素》来学习我们的元素，直到黑胶唱片在 LP 上磨损。现在，多亏了[Karyn]，孩子们可以通过玩“战舰”来学习元素——不需要绕口令的歌词。

对于任何熟悉经典“战舰”游戏的人来说，你会奇怪为什么以前没有人想到这一点。[Karyn]的游戏版本显然是低技术含量的，但是完成了任务。她打印了四份元素周期表，添加了字母来标记各行，并将其层压。一对表格放入一个马尼拉文件夹中，顶部被夹在一起，干擦记号笔被用来标记两到五个元素的块，以代表下层表格上的元素海军的船只。可以通过元素挑战的行和系列标签来猜测敌舰的位置，或者更好的是通过元素名称。命中和未命中在上面的表上用 x 和 o 标记，游戏继续进行，直到隐藏在锕系元素群岛的航母最终被摧毁。

就其简单性和实用性而言，这是纯粹的天才，但当然还有改进的空间。休息后的动作视频揭示了文件夹的一些结构问题，所以这是一个明显的 2.0 版本升级。你可以很容易地看到这是如何用于其他表格材料——乘法表战舰？听起来不错。如果你的孩子们因此对化学感兴趣，一定要和他们一起深入研究元素周期表的结构。

现在，如果你不介意的话:“有锑、砷、铝、硒、氢、氧、氮和铼……”
