# 无焊试验板寄生效应

> 原文：<https://hackaday.com/2016/01/19/solderless-breadboard-parasitics/>

无焊试验板非常方便。当然，你总会听到，高频时需要小心，它们会给电路增加不必要的电容和串扰。这是显而易见的，因为你有相对较长的金属块，它们间隔很近，这正是电容器的定义。

[Ryan Jensen]不仅仅是听取了这些建议。他制作了一个电路，并用示波器研究了一个简单的数字电路能产生多大的耦合。更好的是，他还制作了一个视频(见下文)。测试设置显示了具有正弦波输入的六角施密特触发器反相器的单个门。输出跃迁会振铃，也会耦合回输入。

当然，电路很简单。一些额外的脱钩可能会有所帮助。尽管如此，[Ryan 的]电路并不是你在试验板上看到的典型电路，所以他的观点仍然有效。另一件有帮助的事情是[将试验板安装在坚固的地平面上](http://hackaday.com/2015/10/13/triple-frequency-vfo-on-a-bamboo-breadboard/)。

在视频的结尾，[Ryan]剖开一块面包板展示里面的东西。我们以前见过有人在试验板内部做手术，但如果你以前没见过取出内脏的，你可能会觉得有趣。
