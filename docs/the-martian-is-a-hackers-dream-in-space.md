# “火星人”是黑客的梦想——在太空中

> 原文：<https://hackaday.com/2015/10/10/the-martian-is-a-hackers-dream-in-space/>

你可能已经看过广告，听说过关于电影《火星人》的传言，电影《火星人》讲述了火星宇航员马克·瓦特尼被遗弃在火星上，并被认为已经死亡。你可能已经[读过我们之前关于安迪·威尔的同名书籍](https://hackaday.com/2015/08/04/the-martian-a-landmark-change-in-how-sci-fi-is-written/)的文章。那篇文章怀疑这部电影是否能恰如其分地反映这本书。

确实如此。

总之，沃特尼靠创造一个又一个辉煌而又现实的黑客生存下来。美国国家航空航天局和其他宇航员支持他，一路上想出了一些不可思议的方法。一种是由 [![The_Martian_film_poster](img/286411204587669db3aca789fdfda491.png "20th Century Fox")](https://hackaday.com/wp-content/uploads/2015/10/the_martian_film_poster.jpg) 号飞船的船长梅丽莎·路易斯开发的，包含了幸存宇航员的整个飞船。好吧，这个可能不完全现实，但它令人兴奋。

阅读关于黑客的文章是一回事。在屏幕上看到它们增加了另一个维度。马特·达蒙(饰演沃特尼)将自己的粪便与水混合来给土豆施肥，这是一个你无法在脑海中想象的形象。

好莱坞惯用的伎俩是将次要角色的动作换成主要角色。这就忽略了幕后的“小人物”,他们经常有好主意。通常那是我们。在这里，他们保留了第一个看到瓦特尼在火星上移动设备的女人和天体物理学家，嗯，我不会破坏它，拯救了一天。

对于黑客来说，这部电影应该和[《星际》、](https://en.wikipedia.org/wiki/Interstellar_(film))搭配，因为他们对科学的忠实。“火星人”包含了美国国家航空航天局的实际技术和火星任务的计划。《星际穿越》，嗯，你能说一部源于加州理工学院理论物理学家基普·索恩的想法的电影有什么不好？这部电影中的科学是如此真实，索恩写了一整本书来描述它，并根据准确呈现黑洞所需的研究撰写了几篇科学论文。

这是一个奇妙的趋势，看科幻电影基于真正的科学，而不是被降低到侮辱的程度。你知道，如果 XKCD 拍了一部漫画，那一定很棒。令人惊讶的是，好莱坞并没有对这两部电影做“黑客”工作。
