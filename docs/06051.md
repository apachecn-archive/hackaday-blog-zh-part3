# 这些工程教育项目是我们的黑客

> 原文:[https://hacka day . com/2017/05/25/这些工程项目是我们的黑客/](https://hackaday.com/2017/05/25/these-engineering-ed-projects-are-our-kind-of-hacks/)

用于 STEM 教学的高度抛光的一体化齿轮是解决这个问题的一种方法。但对一些人来说，它们可能会令人望而生畏，而且前期的支出可能会成为你在确定自己想要承诺之前尝试一些事情的障碍。[米兰达]正在采取一种不同的方法，目的是让你家里的垃圾成为工程教育的可能。重点是玩弄工程概念，而不必担心做得完全正确，或者用完全正确的材料。你知道…黑客！

在她位于 Maker Faire Bay Area 的展台上，展示了许多你手头可能有的普通商品。我非常喜欢看到由冰棒棍、胶水、吸管和一些绳子做成的触手。抓住一端并拉动绳子卷起附肢是非常酷的，并且提供了制造和机器人概念的最基本介绍。回想一下当你意识到你不需要像乐高那样专门建造的东西来建造东西时，你最早的一些授权时刻。你可以用*建造任何东西*！

 [![mixing-board](../Images/7b1a9b550e1e3e0964c516e285957a20.png "mixing-board")](https://hackaday.com/2017/05/25/these-engineering-ed-projects-are-our-kind-of-hacks/mixing-board/)  [![DSC_0141](../Images/a3e03136acce3859f76be433d1e135b4.png "DSC_0141")](https://hackaday.com/2017/05/25/these-engineering-ed-projects-are-our-kind-of-hacks/dsc_0141/)  [![DSC_0138](../Images/0ea4387c3509763fbfbbd995b6759038.png "DSC_0138")](https://hackaday.com/2017/05/25/these-engineering-ed-projects-are-our-kind-of-hacks/dsc_0138-2/) 

展台上最受欢迎的是一套电动班卓琴。它们只不过是指板、琴弦和压电元件。连接到一个小的收音机音箱组合，你会得到一个很好的声音。[Miranda]添加了一个 3D 打印品板，她计划为那些不想制作自己的品系的人提供。除此之外，还有一个 DJ 混音板，它使用放在瓶盖上的旧 CD 来旋转，并在下面安装了触觉开关，使它们具有交互性。

她收集的旅游指南有很多值得一看的。她给我演示了她的哈利波特主题魔杖，一个射水，另一个射 BBs。这些将会是完美的暑期项目，让孩子们在校外也能参与其中。有计划让[在未来的某个时候建立并运行一个订阅工具包业务](https://www.simplebrilliance475.com/475-kits/)，但构建说明将永远免费提供。
