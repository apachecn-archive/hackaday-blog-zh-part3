# 机器人脸说话了

> 原文：<https://hackaday.com/2016/05/12/robo-face-speaks/>

如果你在工程学院做一个高级设计项目，制作一个学校校长的机器人复制品需要一些勇气。他或她可能会受宠若惊，或完全被冒犯。我们吗？我们笑出声来。看看下面的视频。剧透:结尾时鼻子/小胡子的扭动每次都会杀死我们。

这个项目使用了各种各样的部件，包括一个塑料面具，一套直立装置，和一个带 MP3 屏蔽的必备 Arduino。有许多关节部分，包括眼睛、鼻子、嘴和摆动的小胡子。面部使用 RC 伺服系统，尽管[gtoombs]说他下次会使用步进电机来实现更平滑的运动。

嘴与音频同步，尽管这是硬编码的，所以要让脸说出任意的话还需要一些工作。尽管如此，只要稍加努力还是有可能的。

人形机器人通常既是技术也是艺术。当然，如果你制造机器人——不管它们看起来多么不人道——你可能不得不担心[规则 34](http://hackaday.com/2015/10/02/no-sex-please-were-robots/) 。
