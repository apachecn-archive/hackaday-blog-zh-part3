# 半固体锂离子电池承诺更好的汽车，太阳能

> 原文:[https://hackaday . com/2016/07/23/半固态-锂离子电池-承诺-更好-汽车-太阳能/](https://hackaday.com/2016/07/23/semisolid-lithium-ion-batteries-promise-better-cars-solar/)

锂离子电池使更小更轻的电子产品成为可能。不幸的是，它们的生产成本也很高。在传统的锂离子电池中，许多薄层形成了成品，就像 baklava 中的 filo 面团一样。一家名为 24M 的初创公司认为他们有制造更便宜的锂离子电池的答案:一种半固体电极，通过混合粉末和液体形成电解液来制造。

不仅电池会更便宜、更快地制造出来，工厂的成本也会更低。目前，2400 万已经有了一条试验性生产线，但是他们希望到 2020 年能够生产出每千瓦时不到 100 美元的电池(目前传统电池的成本在 200 到 250 美元之间)。根据这篇文章，在 100 美元以下，电池将与内燃机的成本竞争。

我们最近报道了传统锂离子电池的内部。也有其他电池技术出现，如[锂空气](https://hackaday.com/2015/11/02/lithium-air-might-be-the-better-battery/)和[固态电池](https://hackaday.com/2015/08/25/could-solid-state-batteries-last-a-lifetime/)。
