# 当你可以给你的孩子造一台游戏机的时候，为什么还要给他们买一台呢？

> 原文：<https://hackaday.com/2016/04/05/why-buy-your-children-a-game-console-when-you-can-build-them-one/>

[buildxyz]不反对他的[孩子玩视频游戏](http://www.buildxyz.xyz/index.php/2015/11/03/rpikids-final-build/)，但他想要一种能提供更多家长控制、更多游戏选择，并且比现代游戏机更包容的东西。

因此，在他的多部分构建日志中，他经历了将树莓派制作成儿童友好型壁挂式游戏机的所有步骤。框架由波罗的海桦木胶合板制成，边缘染色后看起来很酷。显示器是旧的惠普显示器，扬声器是从旧货店买来的简单的米色砖块。控制器连接到前面的 USB 集线器。这不是一个复杂的构建，但它做得非常好。

从父母的角度来看，最酷的功能是前面的密码锁。被新像素包围的旋转编码器提供输入和反馈。根据[buildxyz]输入的代码，他的孩子可以接收不同时期的多巴胺，如果他在生日等场合输入特殊代码，就可以获得无限的游戏时间。

我们希望他准备好让街区里唯一一个能打开保险柜的四岁小孩。这个构建看起来很棒，没有真正的商业产品可以与之匹敌。看视频。
