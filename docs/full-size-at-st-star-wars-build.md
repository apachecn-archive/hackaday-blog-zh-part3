# 全尺寸 AT-ST 星球大战建造

> 原文：<https://hackaday.com/2016/04/04/full-size-at-st-star-wars-build/>

你为你的孩子做了一个很酷的树屋吗？很抱歉打破你的幻想，但我们认为没有什么能超过这个全尺寸 1:1 的星球大战 AT-ST 复制品——配有音效和移动炮塔！我们提到两个座位了吗？

这个 AT-ST 建在他的后院，由木头、塑料和金属片制成。在里面，它装饰有操纵杆、显示面板和其他旋钮和按钮，供您尽情发挥想象力。它甚至在屋顶上安装了太阳能电池板，为两个换气扇提供动力，让你在幻想炸毁霍斯上的叛军败类时保持凉爽。
