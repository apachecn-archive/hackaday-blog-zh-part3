# 注浆树脂原型

> 原文:[https://hack aday . com/2018/05/20/slip casting-resin-prototypes/](https://hackaday.com/2018/05/20/slipcasting-resin-prototypes/)

[埃里克·斯特雷贝尔]不再需要介绍了。如果我们的页面上有一张设计优雅、表面处理专业的零件图片，那么他很有可能参与其中。这一次，他分享了他制作一个看起来像是吹塑成型的零件的方法。吹塑部件有着独特的外观，尤其是用透明材料制成的,[Eric 的]方法当然被认为是这样。如果你需要一些看起来像凝固的肥皂泡的定制部件，这可以升级你的原型游戏。

模具制作不包括在这个视频中，这也可以在下面的休息，但我们可以帮助你与[一个提示](https://hackaday.com/2018/02/25/casting-metal-parts-and-silicone-molds-from-3d-prints/)或[两个](https://hackaday.com/2018/01/25/using-lasagna-to-make-cost-saving-molds/)。出于演示的目的，我们看到创建了一个具有一些不规则表面的医疗部件。树脂被混合和脱气，然后在模具内滚动。然后，大揭露，树脂被允许从模具中流出。重复上述步骤，达到所需的厚度。

这是一种由陶瓷演变而来的技术，叫做滑铸。出于好奇，一个优雅的陶瓷注浆视频演示也可以在下面看到。为了增加画龙点睛之笔，观看如何在成品零件上应用一个 laquer 标志；这种接触会让你的建筑看起来不像一个粗糙的原型。

这位多产的制作人的更多教育可以在他的视频中看到，视频中的[画出了具有专业外观的漆面](https://hackaday.com/2018/04/29/how-to-put-the-pro-in-prototype/)以及他对[使用泡沫芯的提示](https://hackaday.com/2018/02/10/zen-and-the-art-of-foam-core/)。
