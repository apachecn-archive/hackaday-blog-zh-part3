# 黄油传递战斗机器人

> 原文:[https://hackaday.com/2017/05/06/butter-passing-battlebot/](https://hackaday.com/2017/05/06/butter-passing-battlebot/)

对许多有知觉的生物来说，目的的概念是非常重要的；人们只能想象一个专为传递黄油而设计的机器人所经历的哲学恐怖。也许是希望创造一个更有理由存在的机器人，[弥迦“耐嚼”雷保维兹]决定[建造这个装备火焰喷射器的战斗机器人，命名为“火焰战争”。](https://www.youtube.com/watch?v=ifA1191UVOg)

在视频中，我们看到它相当成功地对抗了一个名叫 T800 的机器人，至少在战斗的早期。T800 装备了一种旋转武器，虽然它能够凭借储存的动能发出沉重的重击，但更多时候 T800 似乎是将自己撞倒，而不是对 Flamewar 造成任何严重伤害。Flamewar 能够反复发射它的主要武器，因为它的武器内置了火焰喷射器，远远超过了 T800 的武器。我们不会破坏战斗的结局。休息下的视频。

这个机器人是由与[Team Tiki]竞争的[Micah]建造的，他们已经在网上记录了他们过去的一些建造。尽管如此，我们还是希望看到一些火焰战争实际上传递黄油的镜头。这场比赛是 Robogames 2017 的一部分，我们印象深刻的是[规则中允许使用火焰喷射器之类的东西](http://robogames.net/rules/combat.php)。显然，安全是这些活动的首要关注点，所以看到他们找到了让事情运转起来的方法真是太棒了。

如果您不知道乳制品参考，请[填写此处](https://www.youtube.com/watch?v=3ht-ZyJOV2k)。我们也看到了[其他人对此的看法。](http://hackaday.com/2017/02/24/toast-bot-butters-for-you-sometimes/)
