# 便携包装中的惊人亮度

> 原文：<https://hackaday.com/2016/07/07/incredible-luminosity-in-a-portable-package/>

如果你曾经想把最明亮的一天带进最黑暗的夜晚，这个[手电筒会给你光明](http://imgur.com/a/ch6ku)。凭借高达 32V 的 100W LED 阵列，这款产品非常亮——大约 9000 流明！但最棒的是，构建过程中的每个小细节都被记录下来，这样我们就可以随时跟进。

全铝外壳包含 led 及其散热器、电压调节器和显示器、AD 和 DC 适配器和转换器板及其连接器，以及风扇，以确保足够的通风。它由定制组装的 6400 mAh 11.1V lipo 电池或 DC 20V 10Amp 电源供电，通过 XLR 实现坚固的锁定连接。电池组连接是真空形成的，用于快速交换，如果电池组内的三块电池中的任何一块没电了，电池组本身就会发出警报。一个漂亮的附加功能是检查剩余电量的能力——如果你想在野营旅行或其他短途旅行中带上这种非常强大的手电筒，这种功能尤其有用。
