# 太空中的壁虎脚

> 原文：<https://hackaday.com/2017/07/05/gecko-feet-in-space/>

太空一片混乱，可悲的事实是，我们把它变成了那样。大多数已经发射到地球轨道的卫星都没有退役计划，这些废船，以及各种形式的护罩、整流罩和至少一只宇航员手套，正在成为一个问题。

一项旨在清理太空垃圾的任务将会非常昂贵，但是钱不是唯一的问题。事实证明，在太空中抓取物体真的很难，除非它们是专门为抓取而设计的。吸盘不会在真空的太空中工作，不是所有的东西都是铁磁性的，机械爪必须处理各种各样的形状、大小和质地。

但是现在来自斯坦福大学的消息是一种干粘合剂，其原理与壁虎爬墙的原理相同。壁虎的脚有微小的翼片，由于范德华力而粘在表面上。[Mark Cutkosky]和他的团队的粘合剂工作原理类似，只有在某个方向应用时才能粘附到表面上。这是优于传统压敏粘合剂的一个优点；施加压力会使物体在太空中漂浮。斯坦福抓取器已经在[【呕吐彗星】](https://en.wikipedia.org/wiki/Reduced-gravity_aircraft)和国际空间站上进行了测试。

我们可以想到这种粘合剂在陆地上的大量应用，包括显而易见的壁走机器人。斯坦福大学的研究小组还列出了无人机的着陆平台，可以让无人机停在奇怪的位置，我们觉得这很有趣。
