# 用步进器和八个按钮开始真正的基础

> 原文：<https://hackaday.com/2016/03/03/get-really-basic-with-steppers-and-eight-buttons/>

[Kevin Darrah]制作了一个很好的视频，展示了如何控制步进电机，不是用电机驱动器，而是用我们的手指。采用非常低级的方法来做这类事情让我们对步进驱动器芯片的特性有了更好的理解。例如，为什么半步需要两倍的电流来操作。

[Kevin]从每个步进器教程都会用到的线圈、晶体管和磁铁的标准解释开始。当他用无源器件和按钮连接简单的试验板，然后开始按顺序激活开关时，就是我们“哦”的时刻。起初，他甚至很难记住正确的顺序，但当用触觉开关布置时，步进控制变得直观。

我们做了自己的实验，看看我们是否记得关于这个主题的课。这是复习我们已经知道的东西的一种有趣的方式，并且我们在这个过程中学到了更多。休息后的视频。
