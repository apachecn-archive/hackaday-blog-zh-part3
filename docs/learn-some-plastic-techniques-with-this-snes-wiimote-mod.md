# 用这个 SNES 模型学习一些塑料技术

> 原文：<https://hackaday.com/2016/11/25/learn-some-plastic-techniques-with-this-snes-wiimote-mod/>

并非所有的黑客都必须是深度技术型的。有时候，一场出色的技能展示同样令人印象深刻。[lyberty5]将两大块完全不同的塑料用环氧树脂粘合成一个令人信服的、最令人印象深刻的、可靠的嵌合体。

虽然 WiiMote 的动作控制在首次亮相时确实在互联网上引起了许多冗长的辩论。而每个拥有游戏公司的人和他们的祖母一旦看到任天堂的《龙族黄金宝藏》,就会冲出去复制并超越它。大多数游戏设计师对这个概念有其他想法，主要是它不适合平台玩家。因此，夹在中间的玩家必须将他们的手柄优化矩形旋转 90 度，并在小按钮上起泡来玩游戏。

[lyberty5]记得有一段时间，控制器针对人手进行了优化，以实现精确的时间跳跃。要是有什么方法可以把那段历史粘在 WiiMote 上就好了！这正是他所做的。他切掉了一个 SNES 控制器的顶部，借用了电路板上的按钮部分，焊接，粘合，粘接，然后组装在一起。[绝对可以应用于其他项目的技能](http://hackaday.com/2016/02/09/learn-resin-casting-techniques-duplicating-plastic-parts/)。

经过一些油漆后，新的控制器工作得很好。由于机械部分是从原来的硬件借来的，它保留了良好的老学校的感觉。干得好。休息后的视频。
