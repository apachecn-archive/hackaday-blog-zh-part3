# 努卡可乐铸造简介

> 原文：<https://hackaday.com/2015/11/28/an-introduction-to-casting-with-nuka-cola/>

离下一部《星球大战》上映还有不到一个月的时间，因此还有几个星期，业余道具制作人和角色扮演者就会疯狂地用光剑十字护手和滚动机器人制造他们自己的光剑。在那之前,《辐射》相当酷，而[比尔]在这里给我们介绍如何用这个后世界末日宇宙的定义对象之一制作道具。他制作了一个 Nuka 可乐瓶的真实复制品，并在制作过程中引入了树脂浇铸技术。

与所有适当的零件制作工作一样，这个项目从获得要复制的物体的合理精确的模型开始。在《辐射》中，我们足够幸运，能够在缩放和旋转的同时观察一个特定的物体，给了[比尔]基本的形状。尺寸也很简单:所有的瓶盖都是同样的尺寸，所以[Bill]只是把模型缩小到那个尺寸。

随着模型的创建和零件的打印、组装和完成，是时候创建模具了。[Bill]使用两部分硅胶模具制作基本形状。实际的铸造是通过在模具内部滚动一点树脂来完成的。不需要坚固的瓶状树脂块；反正瓶子是空心的。

[比尔]有一些巧妙的技巧，包括在模具内部涂上铝粉，用乙烯切割器将标签和标识准确无误。成品被证明是伟大的，完美的留在荒地 200 年，直到唯一的幸存者偶然发现它。
