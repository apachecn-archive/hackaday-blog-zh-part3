# Hackaday 奖参赛作品:使用低成本激光切割零件的光学实验

> 原文：<https://hackaday.com/2016/05/06/optical-experiments-using-low-cost-lasercut-parts/>

用光学做实验既有趣又有教育意义。问题是，很多光学元件都很贵。以及其他支持设备，如光具座、试验板和导轨，都会增加成本。[彼得·沃什]和他的团队正在设计一系列低成本、易于制造的[激光切割光学平台组件](https://hackaday.io/project/10707-lasercut-optics-bench)。这些设计是使用普通可用的材料和工具建造的，可以用作高中、家庭实验者和黑客空间的低成本教学工具。

他们设计了几种类型的支架，用于安装激光器、透镜、狭缝、载玻片、比色皿和镜子等部件。支架部分由 1/4 英寸的丙烯酸树脂切割而成，设计成搭扣配合在一起，使装配变得容易。支架由两部分组成。一个是嵌有三块钕磁铁的圆盘，用来固定光学部分。另一个是底座，它有三个调节螺丝，可以让你调整光学部分。磁铁可以让圆盘卡在底座的螺丝上。

这里需要改进的地方是使用[球塞螺钉](http://us.misumi-ec.com/vona2/detail/110300145720/)代替常规螺钉。螺杆末端的球形球和磁体之间的点接触可以提供改进的对准。带有铁质表面的厚重结实的桌子，如厚钢板，可以用作工作台/试验板。激光切割校准棒，内嵌磁铁，让您为您的实验设置各种部件。有一个 [Wiki](https://github.com/OpticsBench/laser-cut-optics-bench/wiki) ，在那里他们将记录可以用这套设备进行的各种实验。构建部件的源文件可以从 [GitHub 库](https://github.com/OpticsBench/laser-cut-optics-bench)获得。
