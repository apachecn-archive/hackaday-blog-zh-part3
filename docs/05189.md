# 你早上第一件事的目标是什么？

> 原文:[https://hacka day . com/2017/03/03/how-good-is-your aim-first-thing of the-morning/](https://hackaday.com/2017/03/03/how-good-is-your-aim-first-thing-in-the-morning/)

对于那些动力不足的人来说——甚至有些人是这样——有时候，醒来是很难做到的，按下贪睡按钮的诱惑是难以抗拒的。如果你想强迫你的大脑立即专注于醒来，这个 [Nerf 目标闹钟](http://sploid.gizmodo.com/how-to-build-an-alarm-clock-that-can-only-be-silenced-b-1792626947)可能会让你准时起床。

不满足于做一个简单的目标，Christopher Guichet 为这个项目建造了一个完整的时钟。传感器的关键是一个记录飞镖冲击的压电晶体，而[Guichet]的信息风格解释了传感器如何在示波器的帮助下工作。一圈 60 个 led 和压电传感器组成了钟面，所有这些都装在一个 3D 打印的外壳中。旋转编码器用于通过 Arduino Uno 控制时钟，尽管即将推出的视频将深入研究事情的代码方面；[Guichet]已经暗示，一旦代码被整理好一点，他将共享这些文件。
