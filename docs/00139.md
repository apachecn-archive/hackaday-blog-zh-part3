# 用木头制作利希滕贝格雕像

> 原文:[https://hack aday . com/2015/09/19/making-lichtenberg-figures-in-wood/](https://hackaday.com/2015/09/19/making-lichtenberg-figures-in-wood/)

听说过李奇登伯格吗？这是你有时可以在绝缘材料上看到的分支放电……没错——当电压足够高时——它会找到方法。使用我们最喜欢的低成本高压微波变压器，【贝克亚德科学家】向我们展示如何[制作我们自己的李奇腾伯格数字！](http://imgur.com/a/PSmcr)

这其实很简单。你所需要的只是一台旧微波炉、一些胶合板和掺有小苏打的水。首先，你需要将变压器从微波炉中取出——这是一个简单的方法，我们在之前已经[讨论过很多次了——你需要以一种允许你获得几千伏电压的方式连接它。](http://hackaday.com/2013/09/16/one-mans-microwave-oven-is-another-mans-hobby-electronics-store/)

然后通过在水中混合小苏打，你可以增加导电性——让木材浸泡一夜，现在你就可以开始了！通过将导线连接到木材的任何一侧，现在它的导电性足以允许电流穿过木材，并在行进中燃烧出令人惊叹的图案——只需看看下面的视频！
