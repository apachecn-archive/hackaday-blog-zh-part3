# 内力药丸

> 原文：<https://hackaday.com/2018/06/12/internal-power-pills/>

可以说植入电子设备的最大障碍是电池。一部现代手机可以在不充电的情况下运行一两天，但这只需要放入口袋，如果电池进入危险状态，它可以很快从口袋中取出。植入式电子设备不是那么容易扔在地上的。如果爆炸或中毒的危险还不够的话，[可植入和可摄入的电池就太大了](http://news.mit.edu/2018/wireless-system-power-devices-inside-body-0604)。

麻省理工学院的研究人员正在研究一种新技术，可以将电源移出体外，并使用无线电力传输系统为体内的东西提供能量。RFID 植入物已经被尝试和测试过了，但它们似乎也是这项技术的先驱。新的植入物从天线阵列接收多个信号，但直到一对天线同时达到峰值，该设备才能获得足够的能量来激活。当天线阵列离病人 1 米远时，用几个天线全部供电，这种情况经常发生，足以给皮肤下 0.1 米的设备供电。多个植入物可以同时使用这些无线电波。

这些设备的局限性将变得显而易见，但它们可以用于在规定的时间释放药物，感知身体化学物质，或向身体发出信号。在这一点上，仅仅是能够让设备在肉体下打开就已经非常令人惊讶了。
