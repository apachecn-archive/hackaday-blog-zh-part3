# 令人印象深刻的绘图机制作如此简单

> 原文:[https://hack aday . com/2017/09/04/impressive-drawing-machine-for-one-made-so-simpley/](https://hackaday.com/2017/09/04/impressive-drawing-machine-for-one-made-so-simply/)

不是所有人都有数控机床、激光切割机和 3D 打印机，我敢打赌，我们大多数人一开始都没有那么好的装备。[jegatheesan]为他女儿制作的低成本绘图机提醒我们，你可以制作原型，然后用非常基本的材料和大部分手工工具制作一个功能性的机械达芬奇。他还编写了自己的绘图软件，其界面非常简单。

[jegatheesan]的项目确实有很多值得喜欢的地方。他首先自己算出了数学，做了一些我们都喜欢的事情，翻出旧的学校三角学和代数书复习。然后他开始制作他的原型，用一个硬纸板管作为主要支撑，吸管和安全别针作为牵引臂。他已经为他的 Arduino 准备了一个马达护罩，但是它只支持 2 个伺服系统，所以他自己做了一个 3 伺服系统护罩。最后，原型告诉他，他必须重做一些计算，让他继续进行最终的机器。

关于最终的机器，我们可以说的一件事是，热熔胶肯定是制造商的全部——你不会在这里找到很多螺丝。甚至伺服系统也用大量的胶水固定住。而且提笔的机制也相当巧妙。整个东西安装在两根垂直的导杆上，这样就可以很容易地上下滑动。为了让它真正上下移动，他把一个玩具车轮偏离中心粘在一个伺服臂上。当伺服系统转动时，偏心轮就像一个凸轮，向下推动木制底座，根据轮子旋转的位置，提升或降低机器。

参见[他的 hackaday.io 页面](https://hackaday.io/project/26872-cute-drawing-robot-servo-and-android)了解完整的逐步开发流程。但首先看看下面的视频，看看这样一个简单的机器是如何令人印象深刻的。
