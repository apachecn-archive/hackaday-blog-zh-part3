# 树莓 Pi Sense HAT 超级天气仪表盘

> 原文：<https://hackaday.com/2015/10/25/raspberry-pi-sense-hat-super-weather-dashboard/>

[InitialState]发布了一个很棒的多部分教程，关于构建他所谓的"[超本地天气仪表板](https://github.com/InitialState/wunderground-sensehat/wiki)"用简单的语言，他创建了一个基于 Raspberry Pi 的网页，融合了地下的天气数据和当地感知的天气数据。

教程有三个部分。第一部分包括使用开发者的 API 从 Wunderground 读取数据(你需要一个 API 键；免费的一天可以查询 500 次)。第二部分包括使用 Pi Sense HAT 测量当地温度、湿度和压力。最后一部分通过制作超本地天气仪表盘(不管它真正的意思是什么)将它们联系在一起。

我们之前讨论过[传感器帽](http://hackaday.com/2015/09/13/sense-hat-lights-up-pi/)(下面的视频中有更多信息)。看起来这些灯可以做一些事情，尽管这不会给你的网络界面带来任何好处。这是一个好看的项目(和教程),而且足够简单，这将是开始
尝试树莓派的好地方。
