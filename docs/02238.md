# 当你拥有 TI 的另一款微控制器 TI-84 时，谁还需要 MSP430 呢？

> 原文:[https://hack aday . com/2016/05/02/who-need-the-MSP 430-when-you-has-tis-other-microcontroller-the-ti-84/](https://hackaday.com/2016/05/02/who-needs-the-msp430-when-you-have-tis-other-microcontroller-the-ti-84/)

我们确信有更多[昂贵的 LED 控制器在那里](https://www.cemetech.net/forum/viewtopic.php?t=11712)，但是 TI-84 必须在那里。除非你手头有一个，那就是免费的。然后，您无疑将需要一个 SPI 库，用于著名的可修改图形计算器。

[Ivoah]正在使用他为 TI 内部的 Z80 处理器汇编编写的库，来控制 Adafruit 的一小条[dot star led。照片中最上面的电路板是一个 ESP8266 电路板，刚好在试验板上。较低的 Arduino 被用作 5V 电源，在这样一个高级计算设备面前被赋予这样的职责。](https://learn.adafruit.com/adafruit-dotstar-leds)

我们中的许多人通过探索 TI BASIC 的特性来娱乐自己，但这无疑是一个进步。你可以在他的 GitHub 上看到他的代码。

在他的概念验证之后，[Ivoah]还制作了一个视频，并开始编写一个图形界面来控制 led。休息后的视频。
