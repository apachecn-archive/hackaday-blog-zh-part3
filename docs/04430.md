# 内置 Raspberry Pi Zero 的智能投影仪

> 原文：<https://hackaday.com/2016/12/18/smart-projector-with-built-in-raspberry-pi-zero/>

你听说过智能手机，但你听说过智能投影仪吗？它们实际上已经存在了几年，有点像电视机顶盒和投影仪的组合，不需要电视。这些功能包括流媒体网飞、Chrome 浏览器和 Skyping 等。然而，它们的价格从几百美元到一千多美元不等。

[Novaspirit]反而[做了自己的廉价智能投影仪](https://www.youtube.com/watch?v=mff-F39fn4o)。他先买了一台 70 美元的便携式投影仪(800×480 的原生分辨率，这个价格还不错)，然后打开了它。他将一个旧的 USB 集线器焊接到一个 Raspberry Pi Zero 上，这样他就可以插入一个 WiFi 加密狗和一个蓝牙键盘加密狗。这些都进入了投影仪。

检查投影仪的电路板时，他发现了即使在投影仪关闭的情况下也可以给 Raspberry Pi Zero 供电的位置。他最后把树莓派做成双可启动的，要么是 OSMC 的 T1，要么是 T2 的 T3。OSMC 是一个可以直接引导到媒体播放器的 Linux 安装程序，RetroPie 是一个类似的安装程序，可以把你的树莓 Pi 变成一个游戏机。休息后，你可以在视频中看到制作过程的延时和演示。

这不是[Novaspirit]用这些 5 美元的树莓派零中的一个做的唯一酷的事情。我们还看到他把一个 u 盘变成了一个笔记本电脑的 u 盘，使用 USB 接口作为以太网接口。
