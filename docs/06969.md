# 你能做到多低的功率？

> 原文:[https://hackaday.com/2017/09/01/how-low-power-can-you-go/](https://hackaday.com/2017/09/01/how-low-power-can-you-go/)

[激光军刀]有一个激情:低功率电机。为了挑战自己，受贝塔伏打电池的启发，他 3D 打印并制造了一个小型[核能](https://www.youtube.com/watch?v=3B5JPuEUCyc)马达！

这种光伏电池使用易碎的玻璃小瓶从钥匙链和一小块太阳能电池板中提取的氚来吸收光线，产生电能。在尝试了许多设计后，[lasersaber]采用了一种 3D 打印的金字塔，其中包含六个线圈和三个磁铁，封装在一个玻璃钟形罩中，并伴有适当的不祥绿色光芒。

你能猜到有多少能量和电流流过这个东西吗？再猜。低一点。低一点。

200mV 和 20nA 以下！

马达的反应如此灵敏，以至于[lasersaber]握住两种不同金属的动作就能让它运转起来；即使一个大的金属球作为周围能量的天线，也足以驱动马达。

去年，我们展示了[【Nur drage】的光电电池](http://hackaday.com/2016/12/01/make-your-own-nuclear-battery/)——正是基于他的【lasersaber】建造的。请注意——这些是适当的放射性元素，因此必须以应有的谨慎和尊重对待它们。绝对不是为了建造[随机数发生器。](http://hackaday.com/2015/08/16/hackaday-prize-entry-nuclear-powered-random-number-generator/)
