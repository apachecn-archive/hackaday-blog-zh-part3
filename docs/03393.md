# 迷你等离子切割机

> 原文:[https://hackaday.com/2016/09/02/mini-plasma-cutter/](https://hackaday.com/2016/09/02/mini-plasma-cutter/)

当你把一个电弧打火机和一些碎玻璃和金属组合在一起时，你会得到什么？[NightHawkInLight]创建了一个简单的[等离子切割机项目](https://www.youtube.com/watch?v=uKmGVNs_y7o)以及一些热熔胶和一些简单的工具。

如果你不吸烟，电弧打火机使用高压火花点燃香烟。他把它拆下来用作电源。任何类似的高压电源应该也能正常工作。他还使用相同的点烟器电源作为弧光笔，我们[在前面](https://hackaday.com/2016/08/18/arc-lighter-become-plasma-pyrography-pen/)讨论过。

切割机使用一个小钢螺钉和一点带手柄的玻璃作为切割头。金属基座充当另一个电极和散热器。对于等离子笔，两个黄铜电极形成火花隙，一个铜管吹动空气将电弧导向工件。如果你正在寻找更大的东西，你可以[拿出大约 8000 美元](https://hackaday.com/2014/06/27/linuxcnc-based-plasma-cutter-router/)。或者——不是真正的等离子切割机——但是你可以走[非常便宜的路线](https://hackaday.com/2015/08/06/build-a-baby-plasma-cutter-right-now/)。
