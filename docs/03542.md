# 高性能晶体收音机

> 原文：<https://hackaday.com/2016/09/16/high-performance-crystal-radio/>

当你想到水晶收音机的时候，你可能会想到一些简单的东西，可能是用家庭垃圾做成的。如果你是克里斯·温德林就不会。他最近[发布了他的高性能水晶套装的视频](https://www.youtube.com/watch?v=Bw4IiTR3Tqw)(见下文)。他没有走任何捷径:他有几百英尺长的天线，使用冷水管道地面系统。在没有放大器的情况下，强信号输入至关重要。

收音机有四个子系统:天线调谐器、带通滤波器、检波器和电动音频输出系统。他在天花板上还有一个真正巨大的扬声器系统——这不是你在童子军里做的水晶收音机。

带通滤波器的一个新颖特征是，使用它时，你可以在天线调谐器线圈和检测器线圈之间滑动线圈。您也可以旋转天线线圈来影响耦合。滤波器只是一个李兹线圈和一个电容，但它增加了系统的选择性。顺便说一下，如果你看了差不多 14 分钟的视频，你会得到这个单元的原理图。

这当然不是我们见过的最小的水晶收音机。但是比你想象的更多的人非常认真地对待[水晶收音机的开发](https://hackaday.com/2016/04/07/getting-serious-about-crystal-radios/)。
