# 床的地面效应照明。

> 原文:[https://hack aday . com/2018/03/23/ground-effect-lighting-for-your bed/](https://hackaday.com/2018/03/23/ground-effect-lighting-for-your-bed/)

如果你曾经因为夜里起床打开浴室的灯而打扰过你的伴侣——或者被什么东西绊倒并在随后的灾难中惊醒了他们——克里斯詹·贝斯)在他女朋友的床上安装动作激活[地效灯](https://hackaday.io/project/74991-underbed-motion-activated-led-light)的想法可能会吸引你的注意。

[ Berce ]正在使用 Arduino Nano 作为该项目的大脑，Adafruit 的 [PIR 传感器](https://hackaday.com/2011/10/11/simple-hack-reuses-an-air-freshener-pir-sensor/)，以及 L7805 电压调节器来处理负载峰值。他没有说明他使用的 LED 灯的类型，但 Neopixels 可能是一个安全的赌注。焊接问题解决后，他将原型板安装在一个 3D 打印的项目箱中。[Berce]没有重新发明 LED，而是从 Adafruit 的 PIR 教程中复制了代码，然后用胶带将该项目贴在床的一侧，这样在见 Sandman 先生之前，开/关开关就在手边。休息之后，请观看构建视频！
