# 相机移动使用 Makeblock

> 原文:[https://hack aday . com/2015/10/08/camera-dolly-uses-make block/](https://hackaday.com/2015/10/08/camera-dolly-uses-makeblock/)

我们称它为机器人，但[埃里克·布伊斯]称它为推车。[Eric]去年买了一个 Makeblock starter 机器人套件，但从未用过。他最近想要一个相机推车来帮助拍摄项目视频，而 [Makeblock 硬件符合要求](http://eribuijs.blogspot.nl/2015/10/homemade-camera-dolly.html)。

[Eric]发现 Makeblock 的一个示例视频展示了一个相机推车，但没有任何构造细节。他打开工具箱，开始复制他所看到的东西。两个 6V 的马达加上一个减速齿轮、一条皮带和一些轮子，小车现在在电脑控制下移动！

说到电脑，Makeblock 套件附带了一个名为 Baseshield 的 Arduino 兼容板。[Eric]不喜欢用于连接主板和外界的电话式连接器，所以他考虑放弃 Baseshield，改用 Arduino Leonardo(但他还没有这样做)。Makeblock 运动底座之后剩下的唯一一件事就是做一个相机底座，这是[Eric]用胶合板制作的。

还有工作要做。小车稍微向左偏移，缺少悬挂有时会使视频不可用。然而，我们相信第二版会更好。

从过去项目的数量来看，T2 已经看到了很多人想要建造的东西。有趣的是看到有多少人用不同的方式处理同一个问题。如果你没看过 Makeblock，下面有一个关于入门套件[Eric]的视频。
