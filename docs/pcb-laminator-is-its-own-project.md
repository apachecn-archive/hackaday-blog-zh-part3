# PCB 层压机是自己的项目

> 原文：<https://hackaday.com/2016/02/29/pcb-laminator-is-its-own-project/>

在家制作 PC 板最简单的方法之一是使用墨粉转移法。想法很简单:用激光打印机打印艺术品，然后用熨斗将墨粉从纸上转移到干净的覆铜板上。碳粉本质上是塑料，所以它会融化并粘在板上，而且它还能抵抗蚀刻剂。

有几件事可以让事情变得更简单。首先是纸张的选择。然而，该过程的另一个高度可变的部分是衣服熨斗。你必须安排适当的热量和压力。如果你没有做很多板，你可能要通过几次才能把它做好，用丙酮和百洁布擦洗不合格的板来再次清洁它们。

[Igor]受够了衣服熨斗，并且知道其他人已经使用层压机器从纸上和空白板上获得墨粉。他从一个商业层压机开始，但是为了温度的 PID 控制，他做了其他的改进。

层压机现在配备了一台 ATmega8、一个 type-K 热电偶(用 IC 读取)和一个 LED 显示屏。成品看起来很棒，我们希望能给伊戈尔更好的印刷电路板。如果你以前没有见过做墨粉转移的层压机方法，你可以看下面的视频(但注意，那不是[Igor 的]层压机)。

在之前，我们已经介绍了[升级层压机。不过，在你改装层压机之前，你可以考虑试试我们几周前讨论过的无热](http://hackaday.com/2014/11/21/upgrading-a-laminator-for-toner-transfer-pcbs/)[转移法](http://hackaday.com/2016/01/12/even-easier-toner-transfer-pcbs/)。
