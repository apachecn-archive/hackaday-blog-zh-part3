# 开源履带式机器人在非洲支持 STEM

> 原文：<https://hackaday.com/2015/10/18/open-source-tracked-robot-supports-stem-in-africa/>

很多黑客项目都是从教育开始的。例如，Raspberry Pi 最初的目标是制造一台负担得起的教室电脑。虾米是英国的一款面向学校的基本 Arduino。我们最近看到了一项旨在非洲 STEM 教育的 3D 打印机器人平台的努力:[azi bot](https://www.thingiverse.com/thing:1040782)。

Azibot 有 3D 打印的胎面，一个简单的夹持臂，并使用结合 Scratch 的 Arduino。他们的网站上有如何组装各部分的说明，并承诺将很快提供软件的定制部分供下载。

无论如何，我们打赌大多数黑客读者不需要这个软件。该机器人显然使用 RC 伺服系统作为驱动和前端的小臂，所以直接从 Arduino 控制它应该足够容易。如果你不想自己动手，总部位于塞内加尔的 Azibot 可以以 99 美元的价格预订套装。当你下令资助其他有需要的学校时，你不能多捐一点，我们有点惊讶。

我们谈到了另一个针对学校的低成本项目，[虾](http://hackaday.com/2012/11/10/the-ultimate-low-cost-dev-board/)，如果你认为贫困的学校不会有 3D 打印机，也许[这台 3D 打印机](http://hackaday.com/2013/10/13/3d-printer-made-from-e-waste-in-africa/)可以拯救你。
