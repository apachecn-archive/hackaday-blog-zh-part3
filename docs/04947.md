# 爆炸性新工艺生产石墨烯克

> 原文：<https://hackaday.com/2017/02/14/explosive-new-process-produces-graphene-by-the-gram/>

你说你需要一些石墨烯，这样你就可以发明下一个大东西，但你不会为产生几毫克这种珍贵物质的过程而烦恼。幸运的是，现在有了一种生产克量石墨烯的新方法。也许不幸的是，这需要制造一个可控的油气炸弹。

石墨烯如今风靡一时，有望彻底改变从电池到超级电容器再到半导体的一切。石墨烯是一种具有独特性质的分子 2D 表面，可以通过诸如用透明胶带从石墨块上撕下薄片等繁琐的方法少量制造。稍微不那么特别的方法包括激光、微波、高温和有害化学物质。但是所有这些方法都是批量生产的方法，产生的东西少得可怜。

堪萨斯州立大学的克里斯·索伦森等人开发的方法，包括在一个带有火花塞的坚固压力容器中引爆乙炔和氧气，可以一次生产出克石墨烯。更重要的是，正如[他们的专利申请所阐明的](http://pdfpiw.uspto.gov/.piw?docid=09440857&PageNum=11&IDKey=B2824241DB2C&HomeUrl=http://patft.uspto.gov/netacgi/nph-Parser?Sect1=PTO1%2526Sect2=HITOFF%2526d=PALL%2526p=1%2526u=%25252Fnetahtml%25252FPTO%25252Fsrchnum.htm%2526r=1%2526f=G%2526l=50%2526s1=9440857.PN.%2526OS=PN/9440857%2526RS=PN/9440857)，这种方法适用于本质上使用乙炔燃料内燃机的连续生产过程。

虽然我们不能鼓励我们的读者在车库里建造一个乙炔瓶，但这个过程非常简单，很容易复制。我们想知道它能在多大程度上安全生产石墨烯。显然，如果你选择重复这个实验，要小心。如果你不喜欢爆炸，可以采购一些大豆油和镍箔，也许[可以试试这个方法，而不是](http://hackaday.com/2017/02/03/graphene-soybean/)。然后你就有东西[和你的橡皮泥](http://hackaday.com/2016/12/18/flexible-sensitive-sensors-from-silly-putty-and-graphene/)混合了。
