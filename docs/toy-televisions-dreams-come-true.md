# 玩具电视的梦想成真

> 原文：<https://hackaday.com/2015/10/21/toy-televisions-dreams-come-true/>

几年前，[亚历克]的老板从墨西哥城给他带了一件纪念品——一台由废木料和纸板制成的小型中世纪落地式电视机。这可能是一个相框，但[亚历克]决心[给它一个更好的生活](http://cassettepunk.com/blog/2015/10/16/tiny-tv/)。

事实证明，[亚历克]的旧三星 I9000 的屏幕非常适合有富余空间的橱柜。一旦亚历克能够找到远程控制它的方法，它就有望成为真正的(YouTube)电视。一个几乎比电视还大的巨大新旧股票遥控器正是我想要的。遥控器内部有足够的空间容纳非 LE Bluefruit 模块，这是 I9000 可以毫无怨言地接受的输入。

问题是，Bluefruit 不支持矩阵键盘，所以[Alec]使用了一个运行在内部时钟上的裸 ATMega328。由于 Bluefruit 板提供电压调节，遥控器能够保持其固有的 9V 电源。[Alec]对结果很满意，尽管他计划改进他的按钮选择，并可能为遥控器做一个新的覆盖。请继续收看一个小小的电视巡演。
