# 做一把喷火吉他来真正的摇滚

> 原文:[https://hack aday . com/2016/05/25/build-a-flame thrower-guitar-to-really-rock-out-with/](https://hackaday.com/2016/05/25/building-a-flamethrower-guitar-to-really-rock-out-with/)

每个人都喜欢的戴安全领带的古怪发明家【柯林·福尔泽】又回来了，这次[做了一把喷火吉他](https://www.youtube.com/watch?v=2seXZcx-0W8)——由英特尔赞助！？

作为一名前管道工，[Furze]是一名制造大师，他为生活带来了许多令人惊叹的机械发明。在这个视频中，也许是第一次，他集成了一个英特尔 Curie Arduino，以进行更精细的控制。

他拆了几个丙烷喷灯，磨和车床加工了自己的配件和歧管，甚至 TIG 焊接了一个燃料压力容器——孩子们，不要在家里尝试这个！

两个喷灯充当第三条气体供应线的指示灯，以制造大爆炸 Arduino 的计划？为了在歌曲中的某些部分开火，添加时间，甚至只是设置一些很酷的模式。
