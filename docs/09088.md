# 社交网络机器人其实是尊重隐私的

> 原文:[https://hack aday . com/2018/04/09/社交-网络-机器人-实际-尊重-隐私/](https://hackaday.com/2018/04/09/social-networking-robot-actually-respects-privacy/)

[Fribo]这个机器人是一个研究项目，它是一个可爱的装置，可以听和说，但不能动。搬家对它的工作来说并不是必须的，它的工作是帮助独居的人感觉和他们的朋友更有联系。更有趣的是(我们敢说，不寻常的是)，它以尊重和维护个人隐私的方式做到了这一点。成为朋友之间的一种“社交连接器和触发器”，每个互动都是可选的，选择加入是[Fribo]背后的设计意图。

该设备通过被动监控一个人的家来工作，并理解像打开冰箱和打开前门之间的区别这样的事情；它能识别语音，但不能录音，而且显然不记得你的活动。每当机器人听到它识别的东西，就会在朋友圈里通知其他单位。例如，[Fribo]可能会突然说“哦，你的一个朋友刚刚打开了他们的冰箱。我想知道他们要吃什么？”人们知道某人做了某事，但不知道是谁。从那以后，有两种完全可选的方式来进一步互动:敲门表示好奇，鼓掌表示感同身受，做任何一种都可以向发起者透露你的身份。所有这些都可以成为以某种方式联系的机会，或者它可以帮助人们感到与他人更紧密地联系在一起。下面嵌入的视频最好地解释了整个事情，它显示了几个用例。
