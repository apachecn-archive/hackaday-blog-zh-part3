# 机器人在结冰的天花板上跳舞

> 原文:[https://hack aday . com/2015/09/30/机器人在冰上跳舞/](https://hackaday.com/2015/09/30/robot-dances-on-the-icy-ceiling/)

美国宇航局喷气推进实验室(JPL)正在研究一种机器人，用于探索欧罗巴的海洋。一个大问题是海洋被永久冰封。JPL 正在用一个在天花板上跳舞的机器人制作天花板。

顾名思义，用于冰下勘探的浮力探测车(BRUIE)是有浮力的，所以它可以漂浮在冰顶上。两个大桨轮让它可以沿着天花板行驶。

![Andy Klesh from JPL with BRUIE](../Images/4ad47e155b646706426aa236a50531a5.png)

Andy Klesh from JPL with BRUIE

2012 年，他们将早期版本带到阿拉斯加的巴罗进行冰下测试。虽然那里遇到的温度可能比不上木卫二冰冻甲烷的温度(木卫二也是水——Rud ),但对人类和机器人来说，这仍然是一个具有挑战性的环境。北极探险队面临的挑战之一是需要测试冰何时薄到足以形成一个洞。他们不得不谨慎行事，以免掉进去。

最近，他们测试了一个新版本的流浪者号[加州科学中心水族馆](http://www.jpl.nasa.gov/news/news.php?feature=4640)，赋予了短语“与鱼共泳”新的含义。安迪·克莱什是 JPL 火星车的首席研究员，也是科学中心的志愿潜水员，他在测试期间陪伴着 BRUIE。在未来的某个时候，他们希望把 BRUIE 放在湖里，让它能够自主探索。

幸运的是，北极团队没有遇到任何北极熊，这是另一个可能的风险。当火星车到达木卫二时，它不太可能遇到外星同类。
