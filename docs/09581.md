# 一把制作精美的 POV 光剑

> 原文：<https://hackaday.com/2018/05/27/a-nicely-crafted-pov-lightsaber/>

我们需要谈一谈。尽管这是一颗难以下咽的药丸，但我们不得不面对这样一个事实:好莱坞的编剧和道具制作人向我们承诺的一些技术是不会实现的。我们永远不会有通量电容器，真正的悬浮滑板不是真实的东西，没有人会用激光束进行剑斗。

但是，仅仅因为我们不能拥有这些设备的真实版本，并不意味着我们不能制作我们自己的带有一些增值功能的道具版本，比如[这把酷的视觉暂留光剑](https://www.electromaker.io/project/view/pov-light-saber)。[鲁尼]在这些地区更为人所知的名字是[比特鲁尼]，他在以他的名字命名的 YouTube 频道上表演魔术，比如[将一台 ESP32 变成软件定义的电视台](https://hackaday.com/2018/02/22/software-defined-television-on-an-esp32/)，这表明他在更多的机械制造方面也不逊色。硬件是标准的 POV fare，有一个陀螺仪来感应光剑刀柄的位置，还有一个 ESP32 来运行刀片中的长 Neopixel 带。还有一个脂肪包和一个较大的 DC-DC 转换器；后者为道具的外观做出了巨大贡献，其大型散热器从铝管柄的末端伸出。还有一个小扬声器和放大器，用于启动和关闭时的必要音效，位置敏感的响声也是一种不错的触摸。查看下面视频中的 POV 操作。

你说什么？你记得以前在这里见过真正的光剑吗？嗯，[有点像](https://hackaday.com/2015/12/22/finally-a-working-lightsaber/)，但这有点推波助澜。或者也许你已经想到了[这个更具破坏性的版本](https://hackaday.com/2013/05/24/3w-handheld-laser-raises-hope-for-a-real-lightsaber-someday/)。
