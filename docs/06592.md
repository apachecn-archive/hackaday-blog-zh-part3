# 用超声波传感器测量空气流量

> 原文：<https://hackaday.com/2017/07/26/measuring-air-flow-with-ultrasonic-sensors/>

测量 HVAC 管道中的空气流量可能是一件棘手的事情。桨轮流量计和涡轮流量计不仅引入阻力，而且由于累积灰尘和碎屑而带来维护问题。能够廉价且非侵入性地测量导管气流，就像使用[这种超声波流量计](https://www.youtube.com/watch?v=BNTSSxzm1GM)，对于 DIY 项目和一般行业来说可能是一件大事。

传感器背后的原理并不新鲜。他发现了 2015 年的一篇论文，该论文描述了测量超声波脉冲穿过管道中运动气流的飞行时间变化的方法。又是那种“我怎么没想到呢？”这些东西在理论上非常有意义，但需要一些工程技术才能变成一个功能传感器。[可能更糟]正在使用现成的 [HC-SR04 传感器板](https://www.sparkfun.com/products/13959)，并且已经完成了概念验证。他得到了真实的数字，并接近一个将用于 HVAC 自动化项目的传感器。下面的视频显示了他迄今为止的进步，并暗示了不久将有更多结果的后续视频。

这里祝他的体格好运。但是如果事情出了岔子，他可能会从我们对故障磁流量计的事后分析中寻找灵感。
