# IPhone 脑外科手术

> 原文：<https://hackaday.com/2017/02/24/iphone-brain-surgery/>

你认为你擅长焊接？你会焊接 CPU 吗？iPhone 内部的 CPU？iPhone 里面的解封 CPU？你能在 iPhone 的内部焊接*一个解封的 CPU 吗？*

如果你做不到，不要害怕——有人可以，我们通过一段视频找到了他或她，这段视频是黄在推特上发布的。视频中没有太多信息，但据我们所知，它来自广州一家名为 G-Lon Technology 的公司。他们的[脸书页面](https://www.facebook.com/notes/g-lon/g-lon-technology/633972846774848)建议他们教手机维修，如果他们修到现在，我们会说学生们的学费是值得的。

修复的原因尚不清楚，尽管标题提到了“CPU to U0301 AP31 AR31 broken repair”，我们认为这是指一个启动错误，可以通过暴露 CPU 内部的几个焊盘并将它们连接到另一个芯片来修复。我们很乐意听到任何熟悉修复的人的评论，但即使没有明确的理由来进行这项工作，这个视频也令人印象深刻。CPU 的环氧树脂盖在显微镜下被小心地磨掉，然后用小工具刮到正确的层。涂上阻焊膜，将头发丝般细的电线固定在焊盘上，并涂上紫外线固化树脂来填充 CPU 的新裂口并稳固电线。拯救一部 iPhone 似乎需要做很多工作，但看起来确实很有趣。

对芯片内部的探索还不够吗？我们有丰富的解套故事:[一个](http://hackaday.com/2016/07/18/russian-decapping-madness/)，[两个](http://hackaday.com/2016/05/11/dirt-cheap-dirty-decapping/)，还有[三个](http://hackaday.com/2017/01/07/yes-you-can-reverse-engineer-this-74181/)，你可能会喜欢。我们之前甚至报道过至少一次 [CPU 内部修复](http://hackaday.com/2015/08/29/co2-laser-decapping-to-fix-soldering-mistake/)。
