# 可视化傅立叶变换

> 原文：<https://hackaday.com/2015/09/17/visualizing-the-fourier-transform/>

如果你从事任何电子工作，尤其是数字信号处理，你可能知道任何信号都可以分解成一束正弦波。相反，你可以通过叠加一串正弦波来产生任何信号。例如，考虑方波。频率为 F 的方波可以由频率为 F 的正弦波及其所有奇次谐波(即 3F、5F、7F 等)组成。).当然，要得到一个完美的方波，你需要无限数量的奇次谐波，但实际上只有少数几个能做到。

像许多抽象概念一样，基本前提很容易理解，你可以查找任何可以对信号进行傅里叶变换的数学算法。但是你能想象出为什么这种转换会这样吗？如果你不能(或者即使你能)，[你应该看看【Mehmet 的】调和圆的 MATLAB 可视化](https://www.mathworks.com/matlabcentral/fileexchange/49533-fourier-series-animation-using-harmonic-circles)。如果你自己没有 MATLAB，你可以随时查看视频(见下文)。

视频的第一部分指出(视觉上)正弦波实际上只是一个在时间中旋转的圆上的一个点。第二部分展示了那些一起运动的圆是如何描绘出带有波纹的方波的。如果你还不能想象正弦波的总和，这个视频将会让你大开眼界。
