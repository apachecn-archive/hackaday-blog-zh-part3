# 3D 打印虎钳是一种机械漫威

> 原文：<https://hackaday.com/2017/02/26/3d-printed-vise-is-a-mechanical-marvel/>

我们经常想知道有多少人有 3D 打印机，结果只是从 Thingiverse 打印小饰品。为了最大限度地利用打印机，你真的需要能够使用 CAD 软件包，并做出自己的设计。然而，就像原理图编辑器不能让你的电子设计工作一样，CAD 程序也不能确保你有一个成功的机械零件。

【TheGoofy】有一个 [100% 3D 打印的](https://www.thingiverse.com/thing:2064269)老虎钳，看起来很有用。然而，真正有趣的是他解释印刷如何影响材料强度和其他进入虎钳的设计考虑因素的视频(见下文)。

视频还展示了如何组装零件。3D 打印的螺丝和齿轮在视频中看起来很棒。老虎钳看起来也很实用。

如果你不想打印，你可以随时拿一个[粘纸](https://hackaday.com/2015/06/13/review-stickvise-needs-a-place-on-your-bench/)。如果你认为越大越好，那就去木材场，[建造这个](https://hackaday.com/2015/05/25/new-cable-based-vise-improves-woodworking-workshop/)。
