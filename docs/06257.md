# 黑客车轴生产定制 90 变速箱

> 原文:[https://hack aday . com/2017/06/15/hacked-car-axle-yields-custom-90-gearbox/](https://hackaday.com/2017/06/15/hacked-car-axle-yields-custom-90-gearbox/)

需要一个坚固的角度变速箱来处理你的下一个大项目的动力传输？为什么不从汽车里取出一辆后轮轴，自己做一辆呢？

当你想到这一点时，后轮驱动车辆的车轴实际上只是内部连接在一起的 90 个变速箱，以及一个非常坚固的组件，可以免费或廉价获得。[Donn DIY]需要一个齿轮箱来驱动割草机，于是他来到一个原料堆放场。下面的视频展示了一些硬件黑客不可或缺的工具——角磨机的出色工作。他不仅用它截掉了半轴中的一个，实际上他在剩下的被缩短的轴上创造了几乎完美的花键。这种工作通常在带分度头和端铣刀的铣床上完成，但是[唐迪]的垃圾场方法非常有效。这正说明了当你别无选择的时候，你能做多少事情。

我们很惊讶以前没有在这里看到[DonnDIY]的任何项目，因为他似乎有相当多的黑客。我们希望很快能看到更多他的作品，但与此同时，你可以随时查看一些汽车差速器的危险和陷阱。
