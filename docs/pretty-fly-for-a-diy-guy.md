# 对一个自己动手的人来说很不错

> 原文：<https://hackaday.com/2017/01/05/pretty-fly-for-a-diy-guy/>

铣床是非常可怕的工作机器，因为它有切割金属的能力。在工厂里安装一台飞刀，风险系数似乎会呈指数级上升。失去平衡的切削刃似乎失去了控制，长的切削冲程，巨大的切屑和烟雾——这可能是可怕的东西。不过，不要担心——比起商业工具，商店自制的飞刀会让你感觉更有控制力。

再次证明拥有一个家庭机械商店的主要原因是为家庭机械商店制造工具，[这个老托尼]在休息后的三部分视频旅程中带我们通过所有的建造细节。它只有三个部分，因为他的磨坊在拍摄期间释放了神奇的烟雾——结果是一个坏的接触器线圈——因为他的崇拜者军团在建造完成后乞求更多的信息。但它们是短视频，如果你想获得一些简洁的技巧，就非常值得一看，比如如何以一个角度面对大量股票，以及如何在主轴中途死亡后恢复那个角度。附录也有很多关于计算飞刀合适速度的很棒的提示，以及面对大表面的飞刀的替代方案，比如使用[镗刀头](http://hackaday.com/2016/06/23/machine-tool-build-is-anything-but-boring/)。

[ThisOldTony]确实在他的店里制造工具以外的东西，但你必须去他的渠道才能找到它们，因为我们在这里没有涉及太多这些项目。不过，我们确实报道了他令人印象深刻的数控机器制造。所有[托尼]的东西都值得一看——有很多东西值得学习。
