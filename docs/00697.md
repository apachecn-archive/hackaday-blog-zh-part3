# 电动野餐桌恐吓珀斯街头

> 原文:[https://hack aday . com/2015/11/19/motorized-野餐桌-bandit-the-perth-streets/](https://hackaday.com/2015/11/19/motorized-picnic-tables-bandit-the-perth-streets/)

澳大利亚珀斯街头有一个公共威胁:两张电动野餐桌。警方正在寻找，呃，车辆的司机，有人看到他们在西海岸高速公路上行驶，车上的乘客漫不经心地坐在桌子上喝酒。这两个野餐桌看起来像是由安装在中央的割草机马达驱动，并由其中一个座位控制。警察对此很感兴趣，因为在澳大利亚的公路上驾驶没有合法执照的车辆是违法的。我敢打赌，他的保险可能也不包括开车在高速公路上兜风。随附的电视新闻报道没有指明负责人，但他们声称已经与建造者交谈过，建造者说这两个桌子可以达到每小时 50 英里的速度。他们声称，他甚至正在研究一种升级的模型，其中包括一个内置的烧烤，这可能会给“得来速”一词带来全新的含义。
