# Hackaday 的奖品是:一块查理牌手表

> 原文:[https://hack aday . com/2016/08/21/hack aday-prize-entry-a-Charlie plexed-watch/](https://hackaday.com/2016/08/21/hackaday-prize-entry-a-charlieplexed-wristwatch/)

如果有一样东西是我们喜欢的，那就是闪闪发光的东西，你不会得到比由微小的 SMD LEDs 制成的显示器更酷的东西了。这就是这款腕表和 Hackaday 大奖参赛作品背后的理念。这是一个很小的板，上面有一个 ATmega，几个按钮，和一大串发光二极管。

这种显示器的最大特点是发光二极管阵列。这是一个 16×5 的 0603 LEDs 阵列，尽可能紧密地组装在一起。这是一个微小的高分辨率 LED 显示器，但即使使用 ATmega88 微控制器为该板供电，所有的 LED 都是可单独寻址的，显示时间或其他任何内容的适当字体已经绘制出来。

LED 矩阵在这些器件中很常见，但用 SMD LEDs 构建定制显示器则完全是另一个层面。我们见过的最好的一个是两年前的这个非官方徽章。这是作弊者用一堆可串行寻址的 LED 驱动器完成的。这个 charlieplexed 版本超越了这个范围，我们急切地等待着电路板文件，这样这个显示就可以很容易地复制。
