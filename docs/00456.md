# 将 MAC 填入 FPGAs

> 原文:[https://hackaday.com/2015/10/24/stuffing-macs-into-fpgas/](https://hackaday.com/2015/10/24/stuffing-macs-into-fpgas/)

几年前，大混乱电线公司的史蒂夫把第一台麦金塔电脑塞进了 FPGA。虽然该项目工作正常，并且能够在虚拟 CPU 上运行 System 6，但存在一些问题:它不太稳定，并且不支持键盘、声音、SCSI 或串行端口。

现在，有一个新的微型 FPGA 板，这个[被完美地设计成适合原来的麦金塔电脑](http://www.bigmessowires.com/2015/10/22/plus-too-mac-replica-new-progress/)。它要稳定得多，而且有一个软盘模拟器即将推出，这样你就不用再处理那些 400k 的 3.5 英寸的磁盘了。

[史蒂夫]的全新 Mac Plus 基于 [MiST 板](https://code.google.com/p/mist-board/)，这是一种 FPGA 板，最初设计用于在 FPGA 和独立的 ARM CPU 上模拟第一款 Amigas 和 Atari SE。已经有很多经典电脑移植到了 MiST，经典的一体机 MAC 是最后剩下的项目。

在下面的视频中，你可以看到在 SVGA 分辨率下运行经典系统 6 的 MiST 板。这意味着 MacPaint 和 Shufflepuck 在一个紧凑的电路板上使用现代硬件。
