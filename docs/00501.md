# 即使我能起飞，我也无法通过牵引光束！

> 原文：<https://hackaday.com/2015/10/29/even-if-i-could-take-off-i-could-never-get-past-the-tractor-beam/>

终于，我们童年的梦想，一个工作拖拉机梁即将实现！这被称为声悬浮，它实际上是利用高度集中的声波，通过基本上创建一个声学力场来使*小物体*漂浮[。](http://www.nature.com/ncomms/2015/151027/ncomms9661/full/ncomms9661.html)

这个概念并不新鲜，事实上我们早在 2014 年就首次报道了它——但从那以后，他们的研究取得了突飞猛进的发展。那时候他们可以让灰尘悬浮起来。现在我们来看看小物体，比如眼睛！可以预见，有了足够强大的扬声器，更大的物体将很快被驾驭…

研究人员发现了三种最佳的声学陷阱，它们允许只通过声音操纵空间中的物体。如果你能看到声音，扬声器阵列一起工作来创建看起来像全息图的东西——这就是声学陷阱。他们发现最有效的形状包括类似镊子的形状，类似龙卷风的漩涡，以及现在的瓶状陷阱——然而只有镊子和龙卷风被实验观察到，请看:
