# 马蒂。你得跟我回去！

> 原文:[https://hack aday . com/2015/09/11/Marty-youve-got-back-with-me/](https://hackaday.com/2015/09/11/marty-youve-gotta-come-back-with-me/)

像任何好的计划一样，你应该总是从建造时间机器的意图开始。这样，如果你未来的自己成功了，你现在的自己就不用担心会不会真的成功了！

不幸的是，这对[[玩具制造商](http://tymkrs.com/)的主机来说并不太成功，但是他们已经设法制造了一个相当真实的[电信时光机！](http://www.instructables.com/id/Telecom-Time-Machine/)

他们创造了一种叫做“哑终端”的东西。早在个人电脑出现之前的日子里，如果你想上网，你必须在一个哑终端上这样做。它本质上是一个单色显示器、一个键盘和一个串行端口。你必须实际连接到主机才能做任何事情，但在当时，你不能只连接一个调制解调器，哦，不，你必须使用声学耦合器来连接。你不得不通过电话播放声音来与主机交流。用你的早晨咖啡来记录一点历史怎么样！？

使用 Raspberry Pi 2，他们构建了一个哑终端仿真器——但不想走捷径，他们实际上使用了一个真正的声学耦合器来与调制解调器通信！这并不容易。

这个项目已经进行了几年——大部分时间用于收集项目所需的所有组件；包括:

*   老式电话(G 型手机)
*   300 波特声耦合器调制解调器
*   双音多频信号发生器
*   专用小交换机
*   绿色荧光阴极射线管监视器
*   M 型(弯曲弹簧，clicky)键盘
*   AT 至 PS2 键盘适配器
*   PS2 到 USB 键盘转换器
*   外部 USB 传真调制解调器
*   Raspberry Pi 拨号服务器
*   树莓 Pi 哑终端
*   女裁缝电脑桌

我们印象深刻。看看这个视频，你也会喜欢的。当然，当人们修复旧硬件并用它来加载 Hackaday 复古版时，我们总是受宠若惊。干得好 tymks！
