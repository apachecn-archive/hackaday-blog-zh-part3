# 新手用线圈枪:学习电磁推进

> 原文:[https://hack aday . com/2016/08/19/coil-gun-for-newbies-learning-电磁学-propulsion/](https://hackaday.com/2016/08/19/coil-gun-for-newbies-learning-electromagnetic-propulsion/)

线圈炮有吸引人的地方，尤其是大的。这可能是危险；在储存在电容器组中的电荷和飞行的射弹之间，大线圈枪对实验来说是致命的。但是，从线圈枪的工作原理中可以学到很多东西，特别是如果你建造了这个 3D 打印的入门级线圈枪。

对于线圈枪新手，[伟大的斯科特]做了一个了不起的工作，解释基本知识。在恰当的时间给线圈施加脉冲会将一个铁磁性的抛射体吸入线圈核心，并让动量将它抛出，正确使用多个线圈可以提高性能。

他的枪是一个简单的手枪设计，有两个线圈，光学传感器告诉弹丸何时在每个线圈的中心，以及一个 Arduino 来协调一切。结果并不惊人——他只使用了适度的电流——但枪仍然工作。[Great Scott]指出了如何使用电容器组来增加电流，但为了简单起见，他将此作为建造者的练习。

这些年来，许多线圈炮和轨道炮已经出现在我们的页面上，包括[他的威力大得离谱的枪](http://hackaday.com/2015/11/09/the-most-powerful-diy-railgun/)，它使用的电容器组如此之大，以至于需要自己的汽车。我们喜欢这个建筑，因为它简单、平易近人，而且对它的功能有很好的解释。
