# 不断创新和无用的鸭子

> 原文:[https://hack aday . com/2016/02/09/constant-innovation-and-used-ducks/](https://hackaday.com/2016/02/09/constant-innovation-and-useless-ducks/)

[迈克]的黑客技术并不复杂，但他们让我们忍俊不禁。无用鸭子公司的首席执行官迈克住在加拿大某处的创新中心，在那里他发明了一些有用的小玩意，比如会自动倾斜的[软呢帽](https://www.youtube.com/watch?v=5dGkVb5kI2U)，或者当你在[隐姓埋名模式](https://www.youtube.com/watch?v=ZmYSHzu7W9Q)下为妻子购买礼物时会上锁的门。

这一切都是从他试图学习 Arduino 开始的，他花了几个小时制作了一个可以从浴室无线发出橡胶洗澡鸭叫声的设备。整个项目让我们想起了我们第一次笨拙地涉足电子世界，用太多的部件来完成一个简单的功能。Arduino 是入门药物，没过多久他就制造了一个调酒机器人。

我们希望他继续制造更多有趣的小玩意。
