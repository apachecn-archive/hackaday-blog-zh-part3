# 大幅面…摄像？

> 原文:[https://hackaday.com/2018/02/08/large-format-videography/](https://hackaday.com/2018/02/08/large-format-videography/)

大画幅摄影赋予了它所产生的图像一种特殊的品质，这是由于它与更现代的手持设备在景深和分辨率上的差异。投影一个餐盘大小的图像，而不是一张邮票，有一些缺点，尽管当涉及到数码摄影时，传感器制造商不会制造那个尺寸的消费品。

[Zev Hoover] [发明了一种大画幅数码相机](https://www.dpreview.com/articles/3423280809/how-i-built-a-large-format-8x10-video-camera)，不仅用于静态图像，也用于视频。这是一个有趣的设备，因为他将一幅巨大的大幅图像转化为现代单反相机中一个相对较小的传感器。他将图像从大画幅镜头和风箱投射到一个由艺术家调色板制成的屏幕上，这是一种方便获得的亮白色塑料，并用安装在大画幅镜头组件下方的单反相机拍摄图像。这通常会导致透视失真，但为了纠正这一点，他将单反镜头偏移安装。

他确实指出，由于到达相机的光线减少，相机的 ISO 设置也有变化，但一旦考虑到这一点，它的表现就令人满意。其结果是，这款相机可以让一些不同寻常的维多利亚风格的大幅图像以视频的形式呈现出来。他在下面的视频中展示了这一点，和朋友们一起穿着合适的老式蒸汽朋克服装。
