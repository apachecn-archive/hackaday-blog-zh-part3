# 给懒人的螺丝刀

> 原文：<https://hackaday.com/2017/10/01/a-screwdriver-for-the-lazy/>

TS100 烙铁是一款时尚的手持设备，配有微型显示屏。现在它背后的人又推出了一款运动控制螺丝刀 [ES120](https://www.youtube.com/watch?v=il86j54dTzc) 。虽然我们喜欢在大型项目中使用大型电动螺丝刀，但我们不确定是否需要 90 美元的螺丝刀来安装小紧固件。然而，如果你看了来自[Marco]的视频评论，你会发现它有一个有趣的用户界面，可能对其他项目有用。[Marco]也是一个小东西，所以你会看到这个小工具能多好地使牛奶起泡沫，提供运输，或者更换轮胎。[马尔科]如果你想看怪兽的内部，它也会撕裂。

引起我们注意的是用户界面。我们以前用过精密电动螺丝刀，特别是我们用过通用工具 500，它的价格大约是 20 美元，有两个位置开关。一个方向使钻头顺时针旋转，另一个方向使刀具逆时针旋转。相比之下，ES120 只有一个按钮。

当你按住按钮时，就像使用普通工具一样转动螺丝刀。ES120 中的加速度计检测到这种旋转，并开始以相同的方向旋转。该工具可以产生四个级别的扭矩，并具有自动设置，以及。

甚至[Marco]也承认 ES120 不会取代他的普通螺丝刀。也许如果你一天要处理数百个紧固件，这是有意义的。话又说回来，我们有很多我们真的不需要的工具和玩具，所以如果你只是想炫耀一个新的闪亮的小工具，ES120 看起来制作精良，似乎功能良好。

我们真正想看到的是有人把 ES120 改造成像绕线机一样酷的东西。当然，如果你有黑客的心情，你也可以随时打造自己的[廉价功率驱动器](https://hackaday.com/2015/10/26/el-cheapo-electric-screwdriver/)。不过，ES120 可能会让一些人更容易[发动他们的汽车](https://hackaday.com/2013/05/02/using-a-screwdriver-to-start-your-car/)。
