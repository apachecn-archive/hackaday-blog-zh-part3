# 老式自动售货机是完美的礼物

> 原文：<https://hackaday.com/2017/04/30/vintage-vending-machine-makes-the-perfect-gift/>

没有什么比一台旧的自动售货机更能表达“我爱你”的了，如果它是一台[修复过并能正常工作的老式 Vendo V-80 可乐自动售货机](https://www.element14.com/community/community/applications/robotics/blog/2014/11/16/repairing-a-vendo-v-80-cola-dispencer-of-the-1950s)，那么你就是赢家了。来自比利时的[Jan Cumps]被一位朋友指派维修有问题的设备。他开始时只有一台能用的冰箱，没有电子设备。在一系列的修复中，他开始用光学和磁性传感器取代机械硬币检测器的开关，以检测硬币的运动。这些传感器依次连接到驱动分配电机的 Arduino。作为修理的一部分，马达本身必须重绕。由于项目有截止日期，整个项目是使用原型板和通孔零件完成的。最后一个系统的工作原理是每次投入一枚硬币就分发一个冰冻的瓶子。

与大多数自动售货机维修相比，这个项目很简单。不是使用现成的硬币检测器，而是一个简单的 LED 和光电二极管对带来了黑客生活。这可以很容易地适用于任何机器，甚至可以用来创建一个廉价的 DIY 自动售货机。

在他的博客中，[Jan Cumps]用视频演示了每个工作步骤，并分享了 Arduino 代码和原理图以及其他有趣的细节。你可以在下面的视频中看到最终的工作版本。
