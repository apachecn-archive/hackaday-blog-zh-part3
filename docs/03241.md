# 赫德伯格是一只由单个克里格制成的仿生手

> 原文：<https://hackaday.com/2016/08/12/hedberg-is-a-bionic-hand-made-from-a-single-keurig/>

发展成为一名现代黑客和修补者需要很多东西:电子学习、编程知识和耐心(以及其他许多东西)。但是，黑客最重要的品质是好奇心。对事物如何工作的渴望是大多数初露头角的黑客拆除家用电器和电子设备的动力。

许多人最终为他们的第一个项目从房子周围的东西中寻找零件。但是，随着资金和更雄心勃勃的建设，需要购买现成的零件。然而，仅仅用拾荒者的零件来建造东西的独创性还是值得一提的，这就是[埃文·布斯]决定要做的，以一种壮观的方式。

[Evan]对从 Keurig 咖啡机中提取零件并不陌生(他很久以前就发现它们是零件的金矿)，他开始只用 Keurig K350 的零件制作一只名为 hed Berg 的仿生手。他花了 200 个小时制作这只手，没有任何计划，仅使用粘合剂和 K350，并拍摄了整个过程。他在 DEFCON 上做了一个关于这个的演讲，但是如果你不能到场，那么看看视频，看看他是怎么做的。

当然，如果你有可用的 3D 打印机，你可以把像这样的构建提升到另一个水平。但是，赫德伯格的魅力在于从卑微的开始获得成功，以及限制自己的内在挑战。
