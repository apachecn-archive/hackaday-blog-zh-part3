# 使用 OpenFrame 的树莓 Pi 艺术框架

> 原文：<https://hackaday.com/2016/03/28/raspberry-pi-art-frame-using-openframe/>

数码相框曾经风靡一时，你仍然可以在当地的大卖场买到。[Ishac Bertran]和[Jonathan Wohl]决定对数字相框进行开源，并创建了 openframe 项目。开源项目使用的是带 WiFi 的 Raspberry Pi，以及 HDMI 显示器或 Pi 可以驱动的显示器(例如，带 HDMI 适配器的 VGA)。

您可能会想:为什么不让 Pi 显示图像呢？openframe 的好处是您可以在 openframe.io 站点远程管理您的框架。你可以把图像、网站(比如 Hackaday.com)或者着色器推到你的任何一帧上。你也可以利用其他用户发布的作品的公共流。

Openframe 不会存储您的任何作品。它只是根据需要从给定的 URL 中检索它。[Ishac]展示了如何将一台旧显示器转换成看起来像高端相框的东西,很容易想到它的用途:学校、办公室，或者给妈妈远程发送孙子孙女的照片。你可以看他的视频，在下面。

[再利用显示器](http://hackaday.com/2016/02/24/picture-frame-mashup-taps-subreddits-for-deep-thoughts/)是[没什么新鲜的](https://hackaday.com/2014/12/11/raspberry-pi-powered-super-digital-photo-frame/)。然而，openframe 是一个可扩展的开源项目，我们很想知道随着时间的推移它会走向何方。
