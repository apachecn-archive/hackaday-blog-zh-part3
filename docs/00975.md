# 网飞和希尔——还有袜子？

> 原文：<https://hackaday.com/2015/12/19/netflix-and-chill-and-socks/>

在《网飞》的第一集疯狂观看环节中睡着后，在最后一集被剧透吵醒，这在第一世界的问题列表中名列前茅。幸运的是，现在有了一个解决方案，那就是[一双网飞袜子](http://makeit.netflix.com/netflix-socks)，这看起来像是一个非常漂亮的可穿戴物联网项目。

可以肯定的是，称这些袜子为网飞使能有点言过其实。除了基于流行的网飞原创系列的袜子设计，没有任何关于流行的流媒体服务的电子产品。这些袜子带有 Arduino Pro 饰品和加速度计，可以检测到你何时停止移动，并发送红外信号来执行你的命令——暂停电影、关闭电视或其他任何操作。构建的电子部分非常容易理解——它只是将几个模块焊接在一起。该项目的纤维艺术方面可能有点超出典型硬件黑客的能力范围，但你可以与编织的人合作——一个有经验的编织者，因为袜子不是初学者的模式——或者只是将毛毡包裹的硬件穿进你最喜欢的舒适的袜子。尽管如此，我们还是有点担心羊毛环境中硬件的 ESD 保护。

《网飞与寒冷》是上世纪“观看潜艇竞赛”的最新版本，因此在这种场合需要特殊的袜子或定制的网飞开关有点令人费解。尽管如此，可穿戴设备的基本想法非常好，有很多扩展和再利用的可能性。
