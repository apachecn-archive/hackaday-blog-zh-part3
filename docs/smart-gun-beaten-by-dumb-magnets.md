# 智能枪被哑磁铁打败

> 原文：<https://hackaday.com/2017/07/25/smart-gun-beaten-by-dumb-magnets/>

对安全破解感兴趣的黑客[Plore]在 2015 年读到了一篇激烈反对智能枪的帖子。“你能想象 DEF CON 的人会用这个做什么吗？”知道他必须做什么:[黑一些智能枪](https://www.wired.com/story/smart-gun-fire-magnets/)。看休息下面的视频。

用 Armatix IP1 武装起来，[Plore]开始使用书中最古老的一个技巧:射频中继攻击。Armatix IP1 的设计是只有当相应的手表在附近时才会开火，这表明一个值得信任的人持有这把枪。然而，通过使用一个定制的 20 美元的放大器来扩大手表的范围，[Plore]能够在十英尺以外开枪，这是足够危险的距离，当然也超过了制造商打算的几英寸。

不止于此，[Plore]走到了另一个极端，创造了他所谓的“电磁兼容性测试器”(换句话说，一个干扰器)，它可以干扰手表的信号，有效地阻止合法的枪支所有者在 10 到 20 英尺的范围内开枪！

没有人放弃，[Plore]意识到这种枪通过一个简单的金属销来防止非法射击，一旦它感应到附近的手表，它就会移开。然而，这种金属恰好是含铁的，你知道这意味着什么:[Plore]，在一些强磁体的帮助下，能够在没有任何电子欺骗的情况下移动针。

现在，我们已经克服了智能枪面临的许多障碍，这种对智能枪技术状态的具体调查并没有使前景看起来更加光明。我们知道后见之明总是 20/20，所以让我们在评论中知道*你*将如何修复 Armatix IP1 的问题。
