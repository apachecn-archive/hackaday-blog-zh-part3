# 从开源 DSKY 开始你的阿波罗收藏

> 原文：<https://hackaday.com/2018/02/02/start-your-apollo-collection-with-an-open-source-dsky/>

考虑到只有六次载人登月，而且几乎所有从发射台上开始的硬件都在途中被丢弃了，获得飞行硬件通常不是普通人的事情。这类文物大多收藏在博物馆或非常富有的私人收藏家手中。爱好者不得不满足于复制品，比如这款开源的阿波罗制导计算机 DSKY T1。

DSKY，或称显示器和键盘，是阿波罗制导计算机的用户界面，阿波罗制导计算机是 20 世纪 60 年代计算机工程的奇迹，专门用于控制指令舱和月球漫游舱的制导和导航。[ST-Geotronics]使用 3D 打印部件制作了一个像样的 DSKY 复制品，用于外壳和表圈。内部有一个定制的 PCB，其中包含用于指示灯面板的新像素矩阵和用于数字显示的七段 led。可悲但可以理解的是，最初的电致发光显示器无法复制，但幸运的是[【Fran Blanche】这些天正致力于那个项目](http://hackaday.com/2017/05/29/re-creating-the-apollo-dskys-display/)。数字显示中的加号和减号的三段显示被证明不可能从商业上获得，因此团队不得不自己制作这种真实的外观。显示屏和键帽采用激光切割和雕刻覆盖，看起来非常逼真，软件甚至实现了一些类似 AGC 的功能。

我们很喜欢这个，虽然我们可以没有声音剪辑，尽管肯尼迪的演讲很鼓舞人心。一切都是开源的，所以你可以自己开发，或者你也可以购买零件，甚至是完整的套件。
