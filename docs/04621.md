# PSA:不要让孩子吃锂电池

> 原文：<https://hackaday.com/2016/12/31/psa-dont-let-kids-eat-lithium-batteries/>

我们在 Hackaday 上收到了很多新闻稿，但这一篇非常恐怖，我们认为值得分享。显然，一些孩子不小心吃了锂硬币电池。当这种情况发生在更大的电池上时，通常大于 20 毫米(CR2032、CR2025 和 CR2016)，就会发生非常糟糕的事情。比如燃烧的食道，甚至死亡。

国家首都毒物中心对此做了一些研究，发现过去两年吞食的电池中有 14%来自如上图的无焰蜡烛。我们知道，我们的一些读者也会在敞开的托盘中处理电池，这对儿童来说显然是相当危险的。

国家首都中毒中心的网站上有一整页的[都是关于电池安全的](http://www.poison.org/battery/tips)，如果你经常和电池和小孩打交道，这可能值得一读。如果发生事故，甚至有热线电话寻求帮助。

所以，请不要吞下电池，或者让孩子放在嘴里。休息过后，一首关于不要把东西放进嘴里的加拿大公益广告歌曲。
