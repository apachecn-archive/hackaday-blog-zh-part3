# 早期显卡的比较

> 原文:[https://hackaday . com/2016/04/09/a-早期显卡对比/](https://hackaday.com/2016/04/09/a-comparison-of-early-graphics-cards/)

我们不得不承认，我们预计会对[8 位家伙]的演讲感到厌烦，但却被他对早期显卡技术的比较吸引住了。

一些演示变得有点儿技术化，这并不坏，但是这个演示的有趣之处在于它清晰地解释了市场是什么样的，以及用户在这段时间里的感受。例如，我们发现真正有趣的一点是提到后来的游戏不支持 CGA 的一些整洁的颜色黑客，因为他们不能在他们开发的 VGA 卡上完全模拟它。同样，有趣的是，通过比较复合文本和 RGBI 更清晰的文本，可以看出为什么像 RGBI 这样的标准会首先存在。

我们学到了很多，一些关于旧游戏中怪异颜色选择的秘密现在变得更有意义了。休息后的视频。
