# DIY KVM 切换器可让您在多台计算机上使用一个键盘和鼠标

> 原文：<https://hackaday.com/2016/11/12/diy-kvm-switch-lets-you-use-one-keyboard-and-mouse-with-multiple-computers/>

如果你在家里或办公室有多台电脑，并且厌倦了摆弄鼠标和键盘，这里有一个快速的 DIY 方法。[Kedar Nimbalkar]——努力寻求解决方案——将[键盘、视频和鼠标切换器](https://www.youtube.com/watch?v=Sll3dYmzRqY)组装在一起，允许一台设备控制两台计算机。

一个 DPDT 开关连接到一个母 USB 端口，两条公 USB 电缆(地线和 5V 电线缠绕在一起并连接到开关)分别连接到一台 PC。[Nimbalkar]建议确保数据线接线正确，并测试 5V 和接地是否正确连接。然后，他用一些热胶水覆盖连接，使其更加坚固，因为它将会有很大的用途。

现在所需要的只是快速按下按钮来改变你正在使用的 PC，简化了繁琐的转换过程——如果你有一个想要一直使用的[定制键盘](http://hackaday.com/2013/05/28/one-mans-adventures-in-custom-keyboard-development/),这将非常有用。
