# 黑客日奖半决赛:超级英雄的力量

> 原文：<https://hackaday.com/2015/09/12/hackaday-prize-semifinalist-superhero-powers/>

[K.C. Lee]的 Hackaday 奖项目的灵感并非来自于看到严重的不公正或不人道的苦难。他正在看网飞电视台的超胆侠。一个关于一个盲人在业余时间打击犯罪的节目。人没有超人的感官，纽约市卡车上掉下来的放射性物质导致忍者神龟，而不是超级英雄。尽管如此，[一个有能力的黑客完全可以获得一种原始形式的回声定位](https://hackaday.io/project/5903-sonar-for-the-visually-impaired)，这对那些法律上失明的人来说非常有用。

[K.C.]对人类回声定位的想法是一种小型可穿戴设备，带有超声波传感器、6 自由度 IMU 以及音频和触觉反馈。通过一点数学知识和大量的练习，你有可能在没有视觉的情况下走过走廊，避开障碍物，找到周围的路。

人类的回声定位[是一件真实的事情](https://en.wikipedia.org/wiki/Human_echolocation)，很高兴看到一种设备使这种人类的微小超能力变得更容易实现。[K.C.]表示，全世界有 4000 万人可以使用这样的设备，对于一个受电视上超级英雄启发的想法来说，这是参加 Hackaday 奖的更有趣的灵感之一。
