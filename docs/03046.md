# 有激光切割机吗？这里有一个很酷的密码锁盒，你可以自己制作！

> 原文:[https://hack aday . com/2016/07/25/have-a-laser-cutter-hes-a-cool-combination-lock-box-you-can-build/](https://hackaday.com/2016/07/25/have-a-laser-cutter-hes-a-cool-combination-lock-box-you-can-build/)

激光切割的胶合板箱很酷。不相信我？看看那些免费的项目，当人们有了激光切割机后，就可以开始做了——这显然是一个受欢迎的项目类型。带有密码锁的激光切割胶合板箱更酷，特别是当密码在同一个箱子的四个侧面的四个独立的数字选择器上输入时。

这正是[Sande24]所做的，最终结果令人着迷。用激光切割机从胶合板上切割出 30-40 个零件，然后将[组装成锁箱](https://www.reddit.com/r/somethingimade/comments/4tvp2c/plywood_safe_with_locks_on_each_side_video_in/)。这种设计可以很容易地重复使用，用丙烯酸，甚至铝或钢来制作盒子，如果你愿意的话。看看下面的视频吧。
