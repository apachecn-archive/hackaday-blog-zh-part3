# 一小时后表面贴装

> 原文：<https://hackaday.com/2017/06/27/an-hour-to-surface-mount/>

我们大多数人已经从通孔器件过渡到表面贴装。有很多零散的教程，但如果你想学习一些技术或者将你的技术与其他人进行比较，你可能会喜欢[Moto Geek]长达一小时的关于他如何利用回流焊接进行表面贴装的视频。你可以看下面的视频。

让这个视频有趣的是，它有一个小时长，涵盖了从哪里可以买到便宜的 PCB，到自制的拾放铅笔。[Moto Geek]使用带有焊膏的模板，他提供了他所使用的材料的链接。

[Moto Geek]使用的不是传统的橡胶扫帚，而是比普通信用卡更灵活的旧驾照。你还可以从视频中获得一些其他的技巧和窍门。

对于回流焊炉，[Moto Geek]使用的是改良的多士炉。然而，你可以买到为此目的制造的便宜的回流焊炉。最后，结果看起来不错。这是一个罕见的视频，你会觉得你在商店里看着他的肩膀，而他在建造他的电路板。

如果你没有烤箱，你可以试试喷灯。如果你更愿意用 [3D 打印你的抓放工具](https://hackaday.com/2014/12/31/3d-printed-smd-tool/)，我们以前也见过。
