# 独特的行星齿轮箱可以为步进机定制打印

> 原文：<https://hackaday.com/2017/03/02/unique-planetary-gearbox-can-be-custom-printed-for-steppers/>

步进电机是各种项目中的主要设备，但通常情况下需要齿轮箱，特别是对于 CNC 机器和 3D 打印机中的线性驱动器等应用。在这些机制中，高扭矩、低背隙的变速箱可能正是合适的东西，而为广受欢迎的 NEMA 17 发动机设计的 3D 可打印分体式行星谐波传动装置甚至会更好。

首先，我们要说的是，我们怀疑任何塑料变速箱能否像[SirekSBurom]声称的那样保持无间隙。但是我们可以看到这个设计的好处，它有一些不错的特点。首先，当然，它完全是 3D 打印的，除了几个螺丝。它与 NEMA 17 发动机完美匹配也是一个非常好的功能，随着 Thingiverse 的设计，相应地放大和缩小它应该不会太难。下面的视频向您展示了这一理论:步进电机驱动一个带有两个行星齿轮的太阳齿轮，每个行星齿轮啮合一个 56 齿的固定环和一个 58 齿的输出环。行星轮围绕固定齿圈每转一圈，输出齿圈就转动一个齿，导致几乎 100:1 的减速比。

我们认为这个齿轮箱上的“谐波”名称有点用词不当，因为谐波传动的定义特征似乎是柔性花键的周期性变形，正如我们在[中看到的这个 3D 打印的应变波齿轮](https://hackaday.com/2017/01/17/3d-printed-strain-wave-gear-needs-your-help/)。但是我们看到了谐波传动的相似之处，我们承认这个小东西有点难以贴上名字标签。不管你叫它什么，它都很酷，可以成为各种构建的便利工具。
