# “诺曼，坐标！”

> 原文：<https://hackaday.com/2017/03/28/norman-coordinate/>

如果说《星际迷航》教会了我们什么的话，很明显我们还没有到达未来。例证:机器人不应该是印刷电路上的小斑点，上面有电线和插座。机器人应该是华丽的女性机器人，穿着聚酯和服，有着蜂巢式的发型，被设计来执行我们的命令，由闪烁、哔哔声、序列号项链控制。

不愿意等到 23 世纪才迎来这辉煌的一天，[彼得·沃什]根据原系列剧集《我，马德》设计并打造了自己的[安卓护身符道具。](https://hackaday.io/project/19996-norman-coordinate)“如果你需要重温这一特别引人注目的 1967 年插曲，下面有一个剪辑，但要点是*企业号*船员被先进但头脑简单的机器人绑架，这些机器人可以被大量不合逻辑和过度行动击败。

机器人的护身符通过闪烁和蜂鸣来指示它们何时 BSOD。[Peter]的护身符是一个忠实的复制品，由激光切割的丙烯酸树脂制成，带有 led 和耳机驱动器。护身符的导线连接到一个带有电池组和令人失望的 Android 的小控制箱，一个掌上微型开关可以让你指示你当前的困惑状态。

尽管如何让烟从你的脑袋里冒出来是留给读者的一个练习，但你肯定会成为任何一个骗子的目标。或者，如果你更喜欢从*到下一代*的更复杂的可穿戴设备，看看[这款抛光的专业通信器徽章](http://hackaday.com/2017/02/21/a-real-star-trek-communicator-badge/)。护身符和通信器都是[黑客日科幻竞赛](http://hackaday.com/2017/03/15/starfish-cat-bowling-ball-bot-and-stargate-all-claim-prizes/)的参赛作品。
