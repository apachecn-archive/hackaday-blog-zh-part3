# 空心南瓜灯的互联网

> 原文：<https://hackaday.com/2017/11/01/the-internet-of-jack-o-lanterns/>

随着糖果热潮的消退，万圣节黑客继续涌入。[杰里米·S·库克]采用了一些基本的概念，并把它们装扮成最聪明的南瓜。

这个南瓜有一个 WEMOS D1 迷你 ESP8266 大脑，LED 眼睛代替蜡烛照明，还有一个小型 USB 电源组供电。【库克】正在使用的代码是 YouTuber【创新汤姆】修改过的草图，它在你的网络上创建了一个服务器——别忘了插入你的网络凭证！—可以通过电脑或智能手机控制 led。

[Cook]已经将 led 连接到 D1 Mini 的相关引脚上，将电池和电路板用拉链绑在一起，并将其放入塑料袋中以保持干燥。把它插到南瓜里，用热胶水把发光二极管粘好，然后测试一下！

[库克]指出，你不必把自己局限在几个发光二极管上——一系列这样的南瓜和一些增加效果的伺服系统是一个完整的鬼屋的开始，就在你的指尖。

[这是另一个](https://hackaday.com/2016/10/27/shocking-halloween-decoration/)快速制作的万圣节装饰，但不要忽视我们之前介绍过的更复杂的[电气化黑客](https://hackaday.com/2013/10/31/spark-up-your-halloween-party-with-this-double-helix-jacobs-ladder/)。
