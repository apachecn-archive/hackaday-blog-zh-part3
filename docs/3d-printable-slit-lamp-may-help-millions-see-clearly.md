# Hackaday 奖参赛作品:帮助数百万人看清事物

> 原文：<https://hackaday.com/2016/08/04/3d-printable-slit-lamp-may-help-millions-see-clearly/>

裂隙灯在印度的第三世界地区极其昂贵，那里最需要裂隙灯。裂隙灯是一项已有一百多年历史的发明，它是一种概念简单的方法来观察和诊断大量的眼部问题。

从技术标准来看，它们相对较老，随着时间的推移，它们背后的原理变得越来越容易理解。虽然配有角膜显微镜的实验室版本当然更好，但制造技术的创新已经大大降低了该设备的理论最低成本，或者至少这是[Kewal Chand Swami]所希望的。

他的设计旨在便于携带和降低成本。它必须能够传送到遥远的地方，而且必须比实验室版本便宜得多。它使用现成的 3D 打印外壳镜头，配有简单的 LED 手电筒，就是你可以在收银台花一美元买到的那种。

该组件滑动到用户的头上，并用带子固定在那里。医生可以调整灯照射的缝隙，也可以通过显微镜来诊断问题。希望像这样的设备将会看到类似于我们已经报道过的义肢项目的[社区支持。](http://hackaday.com/2016/06/08/the-hacker-is-the-future-of-the-prosthetic-hackers-helping-those-in-need/)
