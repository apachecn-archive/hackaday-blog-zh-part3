# 你可能有一个数码管钟，但是你的能悬浮起来吗？

> 原文:[https://hackaday . com/2016/07/31/you-may-have-a-谢妮地铁-clock-but-can-yours-levitate/](https://hackaday.com/2016/07/31/you-may-have-a-nixie-tube-clock-but-can-yours-levitate/)

谢妮管、电磁铁、悬浮和微控制器——这个项目写满了“黑客日”!

[*时光飞逝:悬浮谢妮钟*](http://www.lasermad.com/2016/07/21/about-that-kickstarter-project-i-mentioned-a-while-back/) 来自【托尼·亚当斯】，使用了很多我们以前见过的技术，但是是以一种新的有趣的方式。数码管时钟并不是什么新鲜事，但是使用电磁铁将它悬浮在底座上方，当然还要搭配电感耦合来传输电力，而不使用电线，这使得这个漂浮的谢妮成为一个真正的享受。

 [https://www.youtube.com/embed/1IftoBtdytI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/1IftoBtdytI?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

T2】

为了实现这一壮举，[托尼]使用线圈将电力从基座无线传输到安装管子的浮动平台。这本身就是一个非常酷的想法，会引起我们的注意，但他更进一步，将时间和设置信号从基座无线传输到漂浮的时钟。这是通过基座上的红外发射器向时钟上的红外接收器发送信号来实现的。

这是一个相当巧妙的解决方案，我们以前看到过[托尼的]工作也就不足为奇了。回到 2012 年，我们展示了他的另一个项目，使用了很多相同的技术:[一套数码管象棋](https://hackaday.com/2012/06/27/how-about-a-nice-game-of-nixie-chess/)。有了顶级的工艺和像这样聪明的想法，我们绝对期待看到他还能想出什么！

这个项目还有大约三周的时间，距离目标还有一半多一点。
