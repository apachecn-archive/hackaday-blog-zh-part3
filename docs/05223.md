# “全视派”帮助视力低下的冒险家

> 原文:[https://hack aday . com/2017/03/04/the-all-seeing-pi-AIDS-low-vision-adventurer/](https://hackaday.com/2017/03/04/the-all-seeing-pi-aids-low-vision-adventurer/)

探险旅行可能相当累人，因为它有着典型的旅游目的地所没有的异国情调和潜在的灾难。你可能会发现自己悬在悬崖上，来一张濒死体验的自拍，或者在雨林树冠上滑翔。当然，由于失明，这一切变得非常复杂，所以像[这种树莓派低视力系统](http://www.cyber-omelette.com/2017/03/all-seeing-pi.html)这样的工具将是一个受欢迎的补充，可以添加到几乎失明的冒险家的破旧背包中。

[丹]从小就有视力问题，但是看一眼他的 YouTube 频道就可以看出，他并没有因此而放慢脚步。当[丹]在苏格兰遇到[本]时，[本]注意到他正在用智能手机辅助视力，近距离观看显示屏，并放大以从他剩余的视力中获得尽可能多的细节。[Ben]认为他能帮上忙，所以他用树莓 Pi 和 Pi 相机做了一个抬头显示器。这款相机安装在一个 3D 打印的框架上，配有 5 英寸的 HDMI 显示屏，并佩戴在 GoPro 头戴式支架上，可以提供足够的细节来帮助[Dan]导航，如下面的视频所示。

这个装置现在有点笨拙，但是作为概念的证明(和友谊的证明)，这是一个坚实的开始。我们认为更薄的外形设计可能会有所帮助，在这种情况下，[Ben]可能想看看[这款类似谷歌眼镜的万用表显示器](http://hackaday.com/2016/10/05/hackaday-prize-entry-head-up-for-high-voltage/)以获得 2.0 版本的灵感。
