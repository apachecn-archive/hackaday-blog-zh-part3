# 另一款 74XX 系列 CPU

> 原文:[https://hackaday.com/2017/03/28/another-74xx-series-cpu/](https://hackaday.com/2017/03/28/another-74xx-series-cpu/)

[Jack Eisenmann]对于在广阔的试验板上构建令人印象深刻的 DIY CPU 并不陌生。这一次[Jack]取消了他在最后一台 8 位计算机中使用的 17 个试验板，而是更进一步，为 CPU 设计了一套大量使用的 PCB。结果就是[双雄企业](http://ostracodfiles.com/enterprise/description.html)。

CPU 设计基于 8 位数据总线和 24 位地址总线。像往常一样，一个最小但精心选择的指令集允许[Jack]作为他正在工作的编译器和操作系统的一部分来完成所有繁重的软件工作。目前还没有显示的迹象，取而代之的是计算机通过一个哑终端进行交流。我们爱屏蔽用的铝箔！看看下面的视频，看看我们的意思。

这些年来，我们已经在 Hackaday 上看到了许多[Jack]的其他 CPU 版本。他最初的设计之一是一个可以在 LED 矩阵上玩许多游戏的 4 位 CPU。后来他做了一个更令人印象深刻的 [8 位 CPU 以及模拟视频输出和操作系统](http://hackaday.com/2011/06/17/homebrew-ttl-logic-computer/)。它甚至可以玩乒乓球。他甚至建造了一台[单指令集计算机(SISC)](http://hackaday.com/2016/06/11/designing-a-single-instruction-computer/#more-207810) 。

他在 DUO Enterprise 的最终目标是允许任何人通过提交程序和计算来利用它的计算能力。注意了[杰克]，我们的神经网络需要尽快训练。
