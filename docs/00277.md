# Arduino 生日蛋糕不是谎言

> 原文：<https://hackaday.com/2015/10/07/the-arduino-birthday-cake-is-no-lie/>

给某人做一个生日蛋糕是非常体贴的，但如果他们正在关注自己的体重就不是了。[MrFox]找到了解决办法:[一个 Arduino 驱动的生日蛋糕](http://www.instructables.com/id/Arduino-Birthday-Cake/)。即使你不介意卡路里，Arduino 蛋糕也是一种新奇的东西，肯定会受到比你大一岁的黑客的欢迎。

蛋糕使用了 UTFT LCD 屏蔽，它消耗了大量的引脚和内存，因此该项目使用了 Arduino Mega。一个扬声器播放生日快乐歌(现在[甚至可能是合法的](http://www.rollingstone.com/music/news/happy-birthday-to-you-copyright-deemed-invalid-20150923))，同时一个麦克风检测生日男孩或女孩吹灭虚拟蜡烛。

我们以前见过类似的蛋糕有固定数量的 LED“蜡烛”,但是 LCD 屏幕使它变成了不同颜色的蛋糕。如果你想的话，你可以研究一些真正可以食用的。另一方面，我们自己也有点像[喜欢那个红眼睛、带着烟火的快乐扳手。](http://hackaday.com/2011/09/18/hackaday-birday-cake/)

视频展示了生日蛋糕的运作。鉴于屏幕，我们很惊讶，没有弹出。这不是一个改变生活的项目，但对你的下一次黑客聚会来说仍然很有趣，即使你有一个真正的蛋糕作为备份。
