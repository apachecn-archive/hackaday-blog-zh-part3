# 使用 Verilog 进行 SoC 教学

> 原文:[https://hack aday . com/2017/05/22/an-education-on-SOC-using-verilog/](https://hackaday.com/2017/05/22/an-education-on-soc-using-verilog/)

[布鲁斯·兰德]是少有的拥有自己的黑客日标签的人之一。他和他在康奈尔大学的学生多年来已经完成了许多项目，最近出现在这些页面上的是与 FPGA 相关的项目。如果你只从项目中了解[土地]，你就错过了。他张贴了他的许多课程的讲座，最近增加了一系列新的讲座，内容涉及使用 Verilog 使用 Altera Cyclone FPGA 开发 DE1 片上系统(SoC)。可以在 [YouTube](https://www.youtube.com/watch?v=sKhvMhTiuM4&list=PLKcjQ_UFkrd7UcOVMm39A6VdMbWWq-e_c&index=1) 上看十讲。

2017 年的课程材料有所不同，因此内容新鲜且相关。DE1-SOC 有一个双臂处理器，从 SD 卡启动 Linux。有几个实验室和相当多的背景材料。第一个实验涉及驱动 VGA 显示器。另一个是用于常微分方程的硬件解算器。

我们不得不承认，我们很高兴听到[Land]以一个轶事开始，关于他有一次在 Hackaday 上有一个项目。他显然吸引了一些丑陋的评论，我们发现很难相信，因为我们知道 Hackaday 评论以其善良和礼貌而闻名。

[兰德的]班级过去的一些项目包括一个机器人摄像机小车和[一个测谎仪](https://hackaday.com/2017/01/03/retro-style-diy-polygraph-believe-it-or-not/)。他们也用[计算机视觉](https://hackaday.com/2012/12/20/two-computer-vision-builds-from-cornell/)和其他非常严肃的项目做有趣的事情。
