# 海里奥:自我的互动延伸

> 原文：<https://hackaday.com/2018/03/29/hairio-an-interactive-extension-of-the-self/>

我们在可穿戴技术领域看到的大部分东西都是围绕传统纺织品构建的，比如为更安全的自行车骑行在夹克上添加转向灯，或者用 RGB LEDs 和颜色传感器为围巾布线，使其可以搭配任何服装。虽然我们已经在这里和那里看到了奇怪的发光发饰，[我们从来没有见过像这些由加州大学伯克利分校的学生[Sarah]，[Molly]和[Christine]创造的蓝牙功能，变形，触摸感应的头发延伸物](https://www.instructables.com/id/HairIO-Hair-As-Interactive-Material/)。

HairIO 的理念是头发是自我表达的重要组成部分，它可以成为沙盒可穿戴互动的天然平台。每一根接发都是用镍钛合金丝编织起来的，这种丝在室温下保持一种形状，加热后会变成不同的形状。这个想法是，你可以带着一条直辫子到处走，当你收到短信时，它会卷起来，或者当朋友发送方向时，它会抬起来指路。你甚至可以在工作时用辫子把头发盘成一个发髻，然后在 5 点钟通过发送信号来拉直辫子。广告之后有一段精美的视频展示了各种可能性。

HairIO 由一个 Arduino Nano 和一个定制的 PCB 控制，该 PCB 结合了 Nano、蓝牙模块和驱动编织物的 BJT。每个编织电路也有一个热敏电阻来控制热量。该团队还采用了迪士尼 Touché项目的扫频电容传感技术，使 HairIO 扩展能够对复杂的触摸做出反应。我们最喜欢的部分是，他们用热变色颜料粉涂在一些人造发束上，这样它们就可以受热变色。让我们希望我们仍然有我们的 [Hypercolor t 恤](https://en.wikipedia.org/wiki/Hypercolor)。

镍钛合金丝是很棒的东西。你可以用它来[收回遥控飞机](https://hackaday.com/2017/02/10/a-micro-rc-plane-builder-shares-his-tricks/)上的起落架，或者[为杜克核弹](https://hackaday.com/2015/12/22/muscle-wire-pen-dances-to-duke-nukem/)做一个标记舞。
