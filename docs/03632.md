# 用高压和风筝捕捉闪电

> 原文：<https://hackaday.com/2016/09/25/catching-lightnings-with-high-voltages-and-a-kite/>

在暴风雨天放风筝并不是最明智的做法，当然，除非你是故意想要抓住闪电。来自[kreosan] [的家伙们复制了著名的实验](https://www.youtube.com/watch?v=1naaZvXTqv4)，本杰明·富兰克林曾经用这个实验向[证明闪电的电学性质](https://en.wikipedia.org/wiki/Kite_experiment)。

他们第一次尝试在暴风雨中通过放风筝吸引闪电，风筝系在接地的金属线上，令人惊讶的是[没有](https://www.youtube.com/watch?v=lU8G4o8VfLQ)捕捉到闪电，之后他们想出了一个更可怕的设置:他们将线连接到接地良好的电视显像管的高压轨道上，以几千伏的电压偏置它。[实验成功了](https://www.youtube.com/watch?v=1naaZvXTqv4)，你可以想象，电视没有很好地接受爆炸。闪电蒸发了它的天线，横冲直撞地穿过它的电路，炸开了后盖，并在撞击地面之前在激动人心的点上留下了焊接痕迹。

两人采取了一些安全措施，以确保他们能够活着离开，比如一个自动展开的线圈和第二根不导电的操纵线，以将风筝发射到更高的高度。还是坚果。对下面的视频半信半疑，不要在家里尝试这个。
