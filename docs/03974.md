# 个人指南针指向您的产卵点

> 原文：<https://hackaday.com/2016/11/05/personal-compass-points-to-your-spawn-point/>

传统的指南针指向北方(好吧，反正是磁北)。[Videoschmideo]想做一个指向特定地方的指南针。特别是，结婚礼物指南针指向新婚夫妇订婚的公园。就像在一个全新的《我的世界》世界醒来一样，这里是它们的产卵地，现在它们总能从荒野中找到回来的路。

该设备使用 Arduino、GPS 模块、指南针和伺服电机。作为结婚礼物，它也需要满足一定的审美情趣。该设备是在一个有吸引力的木箱，并使用时尚的黄铜齿轮。齿轮允许伺服电机转动超过 360 度(软件将旋转限制在 360 度)。你可以在下面看到该设备运行的视频。

compass 模块可能很难找到，但是您应该能够对其进行修改，以便与更容易获得的板一起工作。由于你可能无法找到确切的齿轮使用，你的建设可能会有点不同。

黄铜和木头看起来绝对是蒸汽朋克风格。这让我们想起了这个 GPS 项目。如果你有足够的街头信誉来购买现成的 GPS，你可以随时[推出自己的](http://hackaday.com/2013/05/17/homebrew-gps-gets-%C2%B11-meter-resolution-with-a-raspberry-pi/)。
