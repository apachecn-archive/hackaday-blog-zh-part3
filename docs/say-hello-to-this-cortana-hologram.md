# 向这个 Cortana 全息图问好

> 原文：<https://hackaday.com/2017/05/13/say-hello-to-this-cortana-hologram/>

Halo 的 Cortana 就是用这个互联网电器进入现实世界的。[Jarem Archer]为《光晕》和《视窗》中的 Cortana 建造了一个惊人的“全息”家园。该显示器并不是真正的全息图，它使用了古老的 Pepper 的幽灵错觉。一个监视器反射到 3 个倾斜的半镜面面板上。这就产生了令人信服的 3D 效果。Cortana 本身就是一个 3D 模型。[Jarem 的]妻子通过在双 Kinect 深度感应摄像头前行走来给 Cortana 提供她的动作。这种动作捕捉性能驱动了屏幕上的 3D Cortana 模型。

这个黑客背后的大脑是标准的 Windows 10 Cortana 语音助手。说“嘿 Cortana”唤醒设备。为了让整个体验更具互动性，[Jarem]在设备前面添加了一个人脸检测摄像头。当检测到人脸时，Cortana 模型会转向用户。即使几个人都在看这个设备，Cortana 看起来也像是在和观众中的一个人“交谈”。

这个黑客上的樱桃是外壳。[Jarem] 3D 打印了一个黑色塑料舞台。每当 Cortana 被激活时，Arduino 就会驱动 RGB LEDs。发光二极管投射出一种蓝色胶水，这种胶水与胡椒的幽灵错觉很好地结合在一起。结果是一个项目，看起来像是微软可能在他们的一个研究实验室里炮制出来的东西。
