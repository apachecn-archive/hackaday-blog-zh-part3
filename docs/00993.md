# 让我们炸掉一个爆炸避雷针

> 原文：<https://hackaday.com/2015/12/22/exploding-lightning-arrestor/>

闪电是很讨厌的东西。幸运的是，它没有很长的寿命。[BigClive]决定拆除配电系统中使用的 11KV 避雷器。玻璃纤维芯有硅橡胶排水盘，使该装置看起来有点像软盘，但里面是一些严重的硬件。

如你所料，为了保护电路，金属氧化物变阻器将雷击产生的高电压分流到地面。有趣的部分是该设备如何处理故障。如果该设备由于故障导致 11 千伏电力线短接至地，那将是一场灾难。为了防止这个问题，当被闪电击中时，电阻器会发热，并引发爆炸电荷，从而断开接地线并释放一个标志来指示故障。

[BigClive]触发了下面视频中的收费。所以如果你喜欢看东西在一桶水里爆炸，你会喜欢这个视频的。

令人惊讶的是，这不是我们看到的第一次[避雷器被拆除](http://hackaday.com/2012/06/04/whats-inside-a-lightning-arrestor/)(尽管嵌入链接已经失效)。然而，似乎大多数黑客并不想阻止闪电，他们只是[想要](http://hackaday.com/2011/03/26/automatic-trigger-for-lightning-photography/)给[拍](http://hackaday.com/2008/06/11/camera-lightning-capture/)照而已。
