# 3D 打印的人造水晶雕塑将四维空间压缩成三维空间

> 原文：<https://hackaday.com/2016/06/29/3d-printed-zoetrope-sculpture-squashes-4-dimensions-into-3/>

这个引人入胜的项目设法以新的方式做新的事情。艺术家[Akinori Goto]已经使用 3D 打印技术创造了一种无框架的窥镜 T2。它由一个人物的简短动画组成，但该人物随时间的 3D 运动被“涂抹”在一个圆形区域上——该人物的运动不是被捕捉为单个人物或帧，而是被组合成一个单一的对象，以某种方式将 4 维压缩为 3 维。

当被一束细光束照亮时，物体的“切片”揭示了人物在特定时刻的姿态。当物体以这种方式被照亮并旋转时，这个人物看起来像是以一种非常类似于[望远镜](https://en.wikipedia.org/wiki/Zoetrope)的方式被激活。

我们找到了[Akinori Goto]的两个版本。上面展示的是一个正在行走的人形，但是下面有一个更新更宏大的版本，展示了一个舞蹈演员在运动。

由于一束微弱的光线一次只能照亮雕塑的一片，因此也可以使用多个照明点——甚至移动它们——来获得不同的视觉效果。看看下面的视频，看看这些是怎么回事。
