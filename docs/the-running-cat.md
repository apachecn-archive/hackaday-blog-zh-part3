# 奔跑的猫

> 原文：<https://hackaday.com/2015/11/11/the-running-cat/>

有猫在身边很好，但是它们需要锻炼。如果你不能把猫放在户外，当猫咪从 23 个小时的午睡中醒来时，你需要看看别的东西。猫游乐场是有用的娱乐，但这是我们第一次考虑真正的锻炼设备。让我们用一个仓鼠式的猫健身车来给我们的猫朋友们做运动吧。

[bbarlowski]的设计简单但非常巧妙，看起来就像你在宜家找到的平板包装。由数控铣削的桦木胶合板制成，车轮轮辋像拼图一样扣在一起，而地板上有标签可以插入轮辋。弯曲地板的张力将所有东西锁在一起，打造出一个漂亮的车轮。广告之后的视频展示了[缅因浣熊猫 Kuna 在轮子上的动作，并概述了一些扩展计划，包括添加一个 Arduino 来监控 kitty 的活动，并控制一个用于情绪照明的 RGB LED 灯和一个用于积极加强锻炼制度的 cat treat 分配器。

该项目在三月份发起了一场不成功的运动，他们让 T2 可以下载 DXF 的剪辑文件。当然，如果胶合板太多而 Arduino 不够用，就建造 [Arduspider](https://hackaday.com/2011/01/14/arduspider-entertains-children-and-exercises-pets) 来折磨——呃，娱乐你的猫。
