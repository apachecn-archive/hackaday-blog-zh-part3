# NodeConf 欧盟可黑客徽章

> 原文:[https://hackaday.com/2017/11/15/nodeconf-eu-hackable-badge/](https://hackaday.com/2017/11/15/nodeconf-eu-hackable-badge/)

在会议期间，当遇到其他人时，人们首先看的是姓名标签——在精神上试图记住面孔和名字。但是你的名字标签贴在你胳膊上的日子已经一去不复返了。多年来，会议徽章变得越来越有趣和复杂。可破解的电子徽章正在成为常态，而且不仅仅是在硬件公司。为最近在爱尔兰结束的 NodeConfEU 会议，Espruino 著名的[Gordon Williams]设计了一个以 JavaScript 为中心的可黑客徽章。

[NodeConf EU](http://www.nodeconf.eu/) 是欧洲关键的 Node.js 赛事，为 Node.js 社区提供论坛。所以当他们绞尽脑汁想出一个会议徽章的时候，他们显然倾向于一个可以运行 JS 的设计。[戈登]的 [Puck.js](https://www.espruino.com/Puck.js) 完全符合要求，他的任务是基于 Puck.js 创建一个新的设计。功能列表包括蓝牙低能量、低功耗，因此它可以使用 CR2032 电池、高对比度 LCD、一些按钮、NFC 和原型制作区域——所有这些都封装在一个漂亮的六边形 PCB 中(显然)以模仿 Node.js 的标志。徽章上有与会者的名字，但有趣的部分可以通过按下 [Konami](https://en.wikipedia.org/wiki/Konami_Code) 代码序列中的按钮来访问。

易于理解、详细的[文档](http://nodeconfeubadge.org/)帮助黑客快速上手代码示例。通过让他们熟悉徽章，他们还面临一些工作挑战。黑客徽章进入了一个大挑战，有机会赢得明年会议的免费门票。徽章硬件和固件是开源的，源文件存放在 [Github 库](https://github.com/nearform/nceubadge)中。休息之后，请查看视频中徽章的简短概述。

感谢 nearForm 的[Conor]让我们了解这个令人敬畏的徽章。
