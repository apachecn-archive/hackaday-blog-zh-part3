# Battletech Case Mod 展示令人敬畏的木制品，隐藏黑客

> 原文:[https://hack aday . com/2016/05/19/battle tech-case-mod-is-awesome-wood-hides-hacks/](https://hackaday.com/2016/05/19/battletech-case-mod-is-awesome-woodwork-hides-hacks/)

[S.PiC]一直在做一个电脑机箱，看起来像 Battletech 的秃鹫机甲。我们不确定他的严肃脸猫是否同意，但我们同意。

表壳由经过巧妙切割的[胶合板](http://hackaday.com/2016/04/26/materials-to-know-baltic-birch/)制成。我们希望他能保持木头的美感。然而，那将会危险地接近蒸汽朋克。所以也许在最后做一个匹配的油漆工作就可以了。在一些视频中，我们可以看到他是如何巧妙地将电脑组件融入到机箱设计中的。比如正面的黑色网格其实隐藏了电脑的电源进气风扇。

里面的计算机是一个小型微型 itx formfactor one。作为外设添加到它的[S.Pic]已经拿出了黑客电子技巧圣经。从手工焊接的 LED 栅格到重新利用的诺基亚液晶显示屏，他都有。在一个视频中，我们甚至可以看到机甲的炮塔在自己的动力下旋转。

看起来构建在完成之前还有几个步骤，但是它已经足够令人印象深刻，值得任何黑客桌面上消耗的有用的表空间。休息后的视频。
