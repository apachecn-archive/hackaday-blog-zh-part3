# 把一把 VIC-20 从油污的坟墓里带回来

> 原文：<https://hackaday.com/2018/05/25/bringing-a-vic-20-back-from-an-oily-grave/>

无论你进入哪个平台，逆向计算通常都是一项热爱的工作。过时的、不受欢迎的、彻头彻尾的怪异——各种各样的旧电脑被发现，恢复了昔日的辉煌，并获得了新生。在某种程度上，这很感人。但是当一台电脑明显被滥用时，它需要一点额外的努力，特别是在这个被油淹没的 VIC-20 修复的情况下。

在下面的两段视频中，[8 位家伙]讲述了将这位经典准将从坟墓中带回来的血淋淋的细节。第一个视频展示了整容重建，鉴于机器的肮脏状态，这是一个不小的壮举。裂开后，内脏被发现充满了油状残留物；[8 位家伙]把这归因于过去在某种工业环境中的生活，但我们更多地认为这是洪水的破坏。无论这台机器的消亡是在什么样的悲伤环境下，这种情况都需要一个测试来清理，它看起来非常新鲜。内脏也需要相当多的清洁，主要是用刹车清洁剂来切断粘性。

第二部分的重点是让机器重新运转起来，在这里[8 位家伙]也完成了他的工作。通过逻辑探针、信号注入器和一些良好的老式芯片交换，他能够消除大多数潜在问题，然后确定一些 RAM 芯片是他在加电时看到的视频问题的罪魁祸首。最后一切都解决了，机器看起来和运行起来都像新的一样。我们印象深刻。

不过，也许我们不应该质疑[8 位家伙]关于 VIC-20 来自工业环境的呼吁。毕竟，[“小阿米加可以”](https://hackaday.com/2015/07/23/this-little-amiga-still-runs-school-districts-hvac/)运行学校的暖通空调系统超过 30 年。
