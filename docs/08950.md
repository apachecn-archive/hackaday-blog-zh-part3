# 便携式连续性测试仪将多种模式打包到一个小包装中

> 原文:[https://hack aday . com/2018/03/25/handy-continuity-tester-packs-multiple-modes-into-tiny-package/](https://hackaday.com/2018/03/25/handy-continuity-tester-packs-multiple-modes-into-a-tiny-package/)

从 Leatherman multitools 到内置信号发生器和协议分析器的示波器，拥有多重个性的工具似乎永无止境。每个人都喜欢一心多用的人，因为他们让你觉得你的钱花得更值，在大多数情况下，这是真的。但是，一个万事通很少精通任何一门，有时甚至连简陋的万用表也不是这项工作的最佳工具。

考虑到这一点，[sidsingh]开发了我们认为的[一个非常好的专用连续性测试仪](https://hackaday.io/project/96603-continuity-diode-and-led-tester)。他的目标是只使用手头的零件，所以他不得不把所有的东西都装进他的箱子里。所以他开始使用 PIC10LF322 来支持他想要支持的所有类型的连续性测试。除了直接连续性，测试仪还可以处理二极管测试，检测短路或开路二极管，甚至区分常规和[肖特基二极管](https://hackaday.com/2016/09/12/diodes-with-hats-zener-and-schottky/)。它还有一个 LED 测试模式和一个有趣的“不连续”测试模式——只有当连续性中断时，它才会发出蜂鸣声。下面的视频显示了查找间歇性电缆故障的模式，以及所有其他模式。

对于一个表面上单一用途的工具，这个测试器仍然设法将许多测试打包到一个非常紧凑的包中。更简单的连续性测试器也不错——看看[这个廉价的一元店构建的](https://hackaday.com/2014/04/06/creative-continuity-tester-made-for-a-few-bucks/),或者[这个基于 ATtiny85 的稍微复杂一点的单元](https://hackaday.com/2017/11/25/continuity-tester-uses-the-attiny85s-comparator/)。
