# 一个紧凑的便携式缩放仪相机滑块

> 原文：<https://hackaday.com/2017/09/01/a-compact-portable-pantograph-camera-slider/>

呵，哼，又一个相机滑盖，对吧？错了——这是一个带有文字扭曲的相机滑块。

让[Schijvenaars]的 slider 与众不同的是，它不是一个 slider，至少不是通常意义上的 slider。滑块是一种机械装置，可以让相机在拍摄过程中平稳地平移。假设目标是尽可能平稳地将相机从 A 点移动到 B 点，并且滑块通常用于长时间曝光或延时拍摄，那么它们的自然基础是滚珠轴承线性滑块，通常由丝杠上的步进电机驱动。[Schijvenaars]希望他的滑块更紧凑，因此更便携，所以他设计并 3D 打印了一个 3 轴缩放机构。下面的视频显示滑块平移相机通过丝般光滑的 60 厘米；这种安排的一个好处是，它可以从一个方向平移到另一个方向，没有任何抖动。试试线性滑块。

诚然，这个滑块是没有动力的，但鉴于轴与同步带同步，增加一个电机并不困难。我们以前见过很多滑块，从[简单的木制装置](http://hackaday.com/2015/12/26/mostly-wood-motorized-camera-slider/)到[复杂的高架起重机](http://hackaday.com/2016/08/23/time-lapse-rig-puts-gopro-into-orbit-in-your-shop/)，但是这个看起来是一个有很多可能性的伟大设计。
