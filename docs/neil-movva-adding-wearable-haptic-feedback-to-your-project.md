# 尼尔·莫夫瓦:在你的项目中加入(可穿戴的)触觉反馈

> 原文：<https://hackaday.com/2015/12/22/neil-movva-adding-wearable-haptic-feedback-to-your-project/>

尼尔·莫夫瓦不是一般的大学生。他没有为考试而学习或准备论文答辩，而是致力于一个将直接帮助残疾人的项目。这个项目是[探路者，一个为盲人设计的可穿戴触觉导航系统](https://hackaday.io/project/2372)。探路者是一个雄心勃勃的项目，它一直进入了 2015 年 Hackaday 奖的半决赛。触觉技术是一种通过触摸向用户提供反馈的技术，是 Pathfinder 的核心。[Neil]在[黑客日超级会议](http://hackaday.io/superconference)上做了这个演讲。

触觉技术已经从老式的寻呼机电机发展到现在。如今，几乎每部智能手机都有一个触觉反馈电机和驱动芯片。这些传感器可以做任何事情，从快速脉冲，到模拟按钮，到强烈的振动来通知铃声。即便如此，触觉学仍然是一种被广泛使用的资源。[Neil]解释说，任何人都可以在他们的项目中添加触觉，只需要一个振动电机和一个 H 桥电路。更先进的系统可以使用线性共振致动器，它以固定的频率工作，但在它们可以模拟的触摸方面提供了更多的灵活性。

触觉学的大部分研究都是试图与身体现有的系统一起工作。触觉实际上被感知为一系列不同波段的振动。5-15 Hz 用于特征检测，30-50 Hz 用于纹理，200-300 Hz 用于整体灵敏度。在这些波段内混合和匹配振动可以模拟任何东西，从光滑的大理石到金属刷到砂纸。没错，振动可以欺骗你的大脑去感受纹理！

尽管这项技术发展缓慢。苹果在 2015 年初凭借他们的 force touch 触觉反馈系统引起了轰动。Force touch 本质上是一根金属棒上的四圈电线——这是黑客可以在他们的工作室里想出来的东西。这正是[尼尔]希望看到的——黑客们染指触觉技术，并推动这项技术向前发展。
