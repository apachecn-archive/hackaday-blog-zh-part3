# 秘籍灯开关

> 原文：<https://hackaday.com/2018/03/10/secret-book-light-switch/>

你走进书房，看到书架上挂着一个灯泡。你试了房间里所有的开关——没有一个能打开它。想起你在[孤独冲浪者]的家里，你意识到你将不得不开始猛拉眼前的每本书。

虽然经常与蝙蝠洞和其他复杂的隐藏通道联系在一起，但将一本书变成一个秘密开关本身并不复杂。[lonesoulsurfer]基于一个由 [B.Light 设计](http://blightdesign.com/misc_book.html)围绕着一个风扇开关，一些铝条，一个块状终端，钓鱼线，一个铰链，当然，还有一本书，有一个防尘套和一些可以触发的东西。

将铝弯曲成一个角形支架，并钻一个孔来连接风扇开关——确保整体足够小，可以放在后面，不会区分你正在使用的书。将铰链切割成书的大小，用螺丝将铝条固定在上面，然后将杠杆和风扇开关的支架安装在书架上。一旦一段钓鱼绳系在杠杆上，并穿过书页安装到风扇开关上——确保线拉紧——将防尘套滑回到书上就完成了伪装的开关！

[孤独者]正在用这个开关来触发一个电池供电的灯，因为干扰电源是另一种完全不同的事情。因为风扇开关有足够的弹力将书拉回原位，剩下的就是迷惑你的朋友，让他们下次来时留下深刻印象。
