# Venduino 供应小吃，表明售货是一项棘手的业务

> 原文：<https://hackaday.com/2016/07/02/venduino-serves-snacks-shows-vending-is-tricky-business/>

似乎每一个黑客空间最终都以一台旧的自动售货机告终，这台机器被黑客攻击和改装，用来提供零件、工具和消耗品。但是为什么更多的黑客空间不从零开始建立自己的自动售货机呢？因为正如 Ryan Bates 发现的那样，建造一台 DIY 自动售货机并不像看起来那么简单。

[Ryan]的“Venduino”有许多 hackerspace 标准组件——激光切割桦木胶合板外壳、视差连续旋转伺服系统、一部旧诺基亚手机的液晶显示屏，当然还有 Arduino。设计很简单，但细节是关键。该机器没有尝试验证进入其中的硬币，产品螺旋输送器没有完全优化以可靠地分配，并且整个机器可以通过几次快速摇动来清除产品。诚然，[Ryan]并没有试图建立一个可靠的赚钱机器，但他的努力只是强调了现代自动售货机背后的质量工程。当你的奇多被悬挂在螺旋钻的末端时，这可能看起来不像，但是想想在一个有很多变量的环境中，真实的事物处理了多少成功的交易。

当然，每一个失败模式都只是下一个版本中需要改进的地方，但事实上这仍然是一个有着一些伟大想法的整洁的项目。如果你对商业机器的工作方式更感兴趣，可以看看我们在[上的帖子，听听自动售货机的通讯](https://hackaday.com/2014/04/18/sniffing-vending-machine-buses/)或者[发微博的自动售货机](https://hackaday.com/2015/02/10/a-tweeting-vending-machine/)。
