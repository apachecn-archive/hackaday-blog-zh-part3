# 机床制造一点也不无聊

> 原文：<https://hackaday.com/2016/06/23/machine-tool-build-is-anything-but-boring/>

"所以，就像历史上其他伟大的故事一样，我们的故事将从车床开始."更真实的话从来没有说过，因此开始了一个传奇故事[把一根铬钼钢棒变成一个车间制造的镗头](https://youtu.be/7ngNtK9tKME)。

你可能对[ThisOldTony]的努力有一些疑问。首先，除非你熟悉机床，否则你可能会想知道什么是无聊的头。下面的视频说的很明白，但简单的回答是，这是一个打洞的工具。镗刀头用切削刀具旋转镗杆，镗刀头可以偏移以通过可调直径旋转镗杆。它们非常适合制作精确直径的大孔——跳到 25:30 左右，看看它的效果。

另一个问题可能是:为什么他要花这么多时间和精力去做一些他可以直接从货架上买下来的东西？如果你一定要问这个问题，我们认为你可能没有抓住要点。[Tony]似乎主要对构建工具感兴趣；用它们来制造非工具性的东西仅仅是一个快乐的意外。我们完全尊重这一点，此外，只要看看他制造的工具的质量就知道了。我们发现他的视频也很有趣——他很有幽默感，视频制作质量也是一流的。小心香蕉皮和时空连续体问题。

我们喜欢工具，我们真的喜欢用这种工艺水平定制的工具。为了更好的制作工具，看看[这个吉他微动夹具](http://hackaday.com/2016/04/27/engineering-meets-craftsmanship-in-this-guitar-fretting-jig/)或者[这个砂带研磨机](http://hackaday.com/2016/06/20/an-extremely-useful-shop-built-belt-grinder/)。
