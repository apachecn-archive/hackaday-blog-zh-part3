# Hackaday 奖参赛作品:让你的肌肉来工作

> 原文：<https://hackaday.com/2016/07/23/hackaday-prize-entry-let-your-muscles-do-the-work/>

肌电图是一种用于研究和记录肌肉收缩时产生的电信号的技术。它用于医疗诊断、康复、运动学研究，是机器人假肢和外骨骼的首选控制方法。有一些公司有肌电产品，这些产品的用例是在 PowerPoint 演示文稿中翻转幻灯片。幸运的是，Hackaday 奖的这个项目没有受到这些琐事的阻碍。这是一个开放的、可扩展的平台，可以把肌肉收缩变成任何东西。

正如你所料，从肌肉中读取电信号比将电缆插入 Arduino 需要更多的技术知识。这个项目拥有绝对的运算放大器，灵敏度足以作为一个有用的传感器平台。这个项目[已经被用于监测磨牙症](https://hackaday.io/project/10411-mumai/log/41591-citizen-scientist-application-bruxism-alarm-using-emg)——下巴无意中的握紧或摩擦——结果非常好。

虽然这种设备是否会被用于医疗领域还值得怀疑，但它是一个很好的小电路板，可以为机器人手臂增加肌肉控制，或者建造一套非常酷的动力装甲。总而言之，Hackaday 奖的参赛作品非常酷。
