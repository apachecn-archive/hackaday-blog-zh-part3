# 需要拿东西吗？制作定制台钳

> 原文：<https://hackaday.com/2017/06/15/need-to-hold-something-build-a-custom-vise/>

唯一比做一个酷项目更好的事情就是做一个酷项目来帮助做更多的项目！举个例子，[Greg Stephens]和[Alex]想给牛顿摇篮桌面玩具的钢轴承着色。在试用了一个火炬并不喜欢它之后，[Greg]和[Alex]决定制作一个定制的铝虎钳来固定球体，同时将它放在磁感应锻造炉中。

他们的老虎钳——他们称之为制造者的老虎钳——不仅仅是帮助制造摇篮的一次性项目。[Alex]和[Greg]渴望创建一个可用于各种项目的工具。他们希望它是无油的，而且必须是可定制的。理想情况下，它也将有一个可接受的握力，易于使用，并在板凳上看起来很好。

[Greg]和[Alex]已经建立了一个 Hackaday.io 项目，他们的日志显示了两个老虎钳完成迭代和各种 3D 打印和铸造零件的大量进展。最近他们带来了一个 2000 磅的。负载测试和[在他们的老虎钳集合](http://www.solextract.com/vise-collection/)上测试了它，包括两个原型。第一版额定 500 磅。*合理的*夹紧压力——意思是他们没有尽力使压力达到最大。第二版重达 800 磅。，仍然不像台钳，但远比台钳坚固。

他们的磁感应锻造项目也很成功，团队能够快速改变钢珠的颜色。休息之后，请观看一段视频…
