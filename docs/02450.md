# 使用支持 Arduino 的《我的世界》Mod 控制真实世界

> 原文：<https://hackaday.com/2016/06/04/control-the-real-world-with-an-arduino-enabled-minecraft-mod/>

《我的世界》改造几乎和基于方块的游戏本身一样受欢迎，有大量的编辑器和工具可以用来创建新的方块、怪物和武器。现在，有了这个可以与 Arduino 对话的 mod 框架，mod 们可以构建脱离《我的世界》世界的模块来控制现实世界。

虽然从《我的世界》打开一盏灯并不是什么新鲜事，但 MCreator for Arduino 的做法却相当巧妙。MCreator 是一个用于构建 Minecraft mods 的无代码框架，它允许 modders 通过拖放界面来构建新的游戏功能。MCreator Arduino toolkit 允许修改者构建定制的《我的世界》模块，这些模块可以响应游戏中的事件，并通过 USB 与 Arduino 进行通信。Arduino 可以做的任何事情——点亮 LED，感知按钮按压——都可以带入游戏。它是完全开源的，非商业用途免费，非常适合即将到来的基于 STEM 的夏令营季节。我们可以想到一些伟大的项目，当通过《我的世界》界面展示时，它们真的会让年轻的黑客们兴奋起来。
