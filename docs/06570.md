# 金枪鱼三明治铸造厂

> 原文:[https://hack aday . com/2017/07/21/the-tuna-fish-sandwich-foundry/](https://hackaday.com/2017/07/21/the-tuna-fish-sandwich-foundry/)

你能用一条面包和一罐金枪鱼罐头建造一个铸造厂吗？事实证明，是的，你可以。你不仅可以在上述铸造厂熔化铝，还可以用普通的沙滩砂制作模具，铸造出有用的零件。

通过原始森林工程的魔力和加拿大人那种敢做敢为的态度，[AvE]以他独一无二的风格展示了一条热解的酸面包如何可以作为一个完全可以接受的铸造厂，使用金枪鱼罐头作为坩埚。我们之前报道过[【AvE】的碳泡沫制造过程](http://hackaday.com/2016/08/27/amazing-carbon-foam-doesnt-take-much-bread/)，并展示了它的一些令人惊叹的特性，包括铸造服务所需的耐火材料特性。一旦变成碳泡沫，这种面包可以轻松应对丙烷火炬的火焰，并含有足够长的热量来熔化铝。只用海滩沙，AvE 就能消失模铸造出一个球形零件。对于这样一个低端的现场权宜设置来说，结果相当令人印象深刻。

通常情况下，我们会提醒耳朵更敏感的读者注意[AvE]的丰富多彩的语言，以免他们在他说漏嘴时屈服于那些空话。但是他在这件事上表现出了非凡的克制，即使他的切割垫还在燃烧。漂亮的 SFW，所以享受看看你什么也不用做。
