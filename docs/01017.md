# 哈卡戴的编辑愿景

> 原文:[https://hackaday.com/2015/12/23/hackadays-editorial-vision/](https://hackaday.com/2015/12/23/hackadays-editorial-vision/)

我有幸在 11 月的 2015 年 Hackaday 超级大会上就 Hackaday 的编辑愿景这一主题发表了演讲。我们即将结束令人惊叹的一年，在这一年中，我们的写作团队在各方面都有所成长。我们的编辑、作家和社区成员比以往任何时候都多(Hackaday.io [超过了 100，000 名成员](http://hackaday.com/2015/10/29/hackaday-io-just-passed-100000-members/))。凭借这一点，我们已经能够制作大量高质量的原创内容，这些内容对任何对工程感兴趣的人都很重要——其中最好的内容体现在广阔的[综合卷 2](http://store.hackaday.com/products/hackaday-omnibus-2015) 印刷版中。对我们来说，2015 年也标志着一场无与伦比的地面游戏；我们带着 [Hackaday 奖](http://hackaday.io/prize)走遍世界，处处受到你们的热烈欢迎。当然，Hackaday 超级会议(我在那里做了演讲)是一个重要的里程碑:Hackaday 有史以来第一次全面的会议。

这就引出了一个问题，接下来会发生什么？指导 Hackaday 的是什么，我们未来的计划是什么？请欣赏这个视频，它是 Hackaday 的一个真正的“国情咨文”,然后在休息后加入我，了解更多关于我们为什么做我们所做的事情的细节。

要记住的重要一点是，黑客日是由社区驱动的。Hackaday 的价值在于人们，他们在评论区贡献富有成效的讨论，发现优秀的黑客并将其发送到举报热线，在 Hackaday.io 上发布他们的工作并分享他们的技能，并抓住机会随时参与。我们熟练的观众也是我们写作人员的来源——我们从社区中招聘。读者和作者都有对新兴技术的好奇心和对巧妙技巧的热爱，他们之间的反馈和合作是 Hackaday 保持新鲜的原因。

我们正在发展我们文章中所涵盖的内容。这不是一件新鲜事，它一直是我们内容工作的一部分。当然，来自社区的优秀黑客的传统报道将会继续。激发你下一次黑客行动的文章，或者帮助你发展这方面技能的文章，将继续有它们的位置。但严肃工程的原创内容是我们目前的驱动力。

不幸的是，我们生活在点击诱饵的时代，但是 Hackaday 并没有上当。我们可能对我们的标题很感兴趣，但是每次你点击一个帖子，都会有惊人的内容值得一读来支持这个标题。我们不会让流量左右我们对报道内容的选择。我们根据我们认为很酷的东西、值得作为一个社区讨论的东西来做出选择，然后努力推广这些伟大的内容，以便那些还没有听说过 Hackaday 的人能够找到他们的出路。想帮忙吗？当你在 Hackaday 上读到你真正喜欢的东西时，分享它。发布到社交媒体上，提交到 reddit、Hacker News、Slashdot 等网站。让人们知道，伟大的内容是一个崇高的目标，这是一个巨大的帮助。

我正在滔滔不绝讲述的这个伟大内容的一大部分是讲述工程的故事。看看人们讲述伟大的体育传奇及其成就的热情。你可以打赌，一场完美的比赛和后来居上的胜利永远不会被遗忘。为了我们深刻的思考者和我们的科学成就，我们需要这样做。我们关注的一个地方是揭示和讨论人们的故事，以及他们的旅程。目标是传递我们作为一个民族的兴奋感，并防止失去来之不易的知识。

我们也关注时事。我们和您一样对最新的硬件感到兴奋，并且总是试图在它一发布就得到它(Hackaday 荣誉新闻禁运，通过给我们的提示热线发送电子邮件告诉我们您的发布)。当社会运动影响到我们的社区时，Hackaday 将发出警报。FCC 提出的路由器固件法规(T1)和最近 T2 联邦航空局的无人机法规(T3)都是如此。当涉及到蛇油索赔时，我们一直在寻找警察炒作——其中许多在可疑的群众融资活动中露出丑恶的嘴脸。

这就是我们的编辑声音要去的地方。Hackaday 将继续以越来越快的速度发表更长、更深入的文章。我们希望讨论什么是重要的，我们希望与那些有技能和背景的人一起讨论这些问题。Hackaday 一直是，并将继续是这种情况发生的地方。

这就是了。Hackaday 是一个彻头彻尾的全球社区。你是这个社区的一部分，你和这个组织有利害关系。你让 Hackaday 变得伟大。感谢您给我美好的一年，我期待着更美好的一年到来。
