# DIY 脂肪保护器

> 原文:[https://hackaday.com/2017/09/25/diy-lipo-protectors/](https://hackaday.com/2017/09/25/diy-lipo-protectors/)

众所周知，蜘蛛侠的叔叔本说过:“权力越大，责任越大。”电池电源也是如此。[Andreas]想要使用受保护的 18650 细胞，但不想购买现成的细胞。他找到了一个 40 美分的解决方案。你不仅可以在下面的视频中看到，而且他还[解释并演示了电路在做什么以及为什么](https://www.youtube.com/watch?v=1rg3ZWxBNUE)。

保护是重要的脂肪技术。当然，脂肪细胞已经改变了我们使用便携式电子产品的方式，但它们可能是危险的。如果您过度充电或让它们完全没电，然后再充电，它们可能会着火。因为它们的源电阻较低——这通常是可取的——短路它们也可能造成火灾。我们已经深入讨论了化学反应，但是为了防止所有的不良后果，你需要一个充电器电路。

这个小电路安装在标准的 18650 单元上，使用两个芯片(其中一个只是双 MOSFET)和三个分立元件。它确实给电池增加了大约 3 毫米。[Andreas]发现带有螺旋弹簧的电池座可以适应额外的长度，但带有金属板簧的电池座则不行。
