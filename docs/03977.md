# 一个家庭让 NES 来统治他们

> 原文：<https://hackaday.com/2016/10/27/one-home-made-nes-to-rule-them-all/>

任天堂娱乐系统，或 Famicom 取决于你生活在世界的哪个地方，是一个在特定年龄的人心中占据特殊位置的游戏机。如果你生活在一个任天堂在 80 年代末和 90 年代初没有向其出口游戏机的国家，你可能会认为这将是一次错过的经历。例如，东欧人多年没有正式见过马里奥。

[![A Pegasus NES clone. Ktoso the Ryba [Public domain], via Wikimedia Commons.](img/8d2098e95ac36b6d4bbfd03b97af62e9.png)](https://hackaday.com/wp-content/uploads/2016/10/pegasus_consolepadgame.jpg) 

一个飞马 NES 的克隆体。通过 [Wikimedia Commons](https://commons.wikimedia.org/wiki/File:Pegasus_console%2Bpad%2Bgame.jpg) ，ktoso Ryba【公共领域】。对他们来说幸运的是，中国和台湾有一个克隆制造商产业，他们的产品在这些市场上随处可见。对于没有任天堂官方产品的国家来说，是这些游戏机和它们的品牌获得了狂热的游戏地位，而不是真正的东西。

在波兰，[phanick]想通过[建造他自己的克隆控制台](http://www.elektroda.pl/rtvforum/topic3131101.html#15240737)(波兰语，[英语通过谷歌翻译](http://translate.google.com/translate?sl=auto&tl=en&u=http%3A%2F%2Fwww.elektroda.pl%2Frtvforum%2Ftopic3131101.html%2315240737))来重现他的青春。他选择的目标是[飞马](https://en.wikipedia.org/wiki/Pegasus_(console))，这是台湾 NES 的克隆品，是 90 年代初波兰人的必备游戏机。

但是他不仅仅满足于制造一个帕伽索斯克隆体。在此过程中，该项目扩展到包括支持 72 针 NES 墨盒以及 60 针 Pegasus 墨盒，以及播放 PAL 和 NTSC 游戏的能力。为了这种双系统支持，他必须包括两组处理器和图形芯片变体，以及在它们之间切换的逻辑。他详细描述了实现这一转变的艰辛。

结果是一个非常令人印象深刻和执行良好的作品。PAL 游戏具有信箱效果，屏幕顶部和底部有黑条，而 NTSC 游戏的颜色略显褪色。但是如果你是当今的游戏玩家，你会发现这些只是真实体验的一部分。

他发布了一个描述性的视频，我们把它嵌入到了休息时间的下面，但是带有非英语的评论。然而，即使不理解音频，它仍然值得观看，因为它对完整的棋盘和游戏的看法。

这些年来，我们推出了许多游戏机项目，但其中许多都是模拟器，就像这个[受任天堂启发的基于树莓 Pi 的例子](http://hackaday.com/2015/07/09/nin10do-retro-game-console-stands-above-all-others/)。这个项目不同于模拟器，据我们所知，没有人克隆过原版，当然也没有支持这么多变体。
