# 精致的 LED 手提包在野外

> 原文:[https://hacka day . com/2016/05/31/enquisite-led-手袋在野外/](https://hackaday.com/2016/05/31/exquisite-led-handbag-in-the-wild/)

制造者集会上展示了许多奇观。但是说实话，我最喜欢看到的是由热情的黑客完成的执行良好的构建。[黛布拉·安塞尔]就是这种情况。她没有展出，只是像我一样欣赏所有的风景。但是她的包比我那个装满摄影器材的单调灰色背包好多了；她用 LED 矩阵制作了一个手提包，而且做得非常好，你会挠头试图弄清楚她是不是那样买的。

Gerrit 和我走上前去，问她是否愿意给我们看看。我们也不是唯一的。当黛布拉拿出手机，通过蓝牙向 10×15 矩阵发送“黑客日”时，她的包开始吸引人群。看看下面我们的视频采访。

T2】

我认为这真的很特别，因为她掩盖了构成矩阵的 LED 灯条。她布置了 led，切割了皮革条以完美地隔开 APA102c 的每个像素，然后继续编织、缝合和粘合组件。

一个 [Adafruit 羽毛板](https://www.adafruit.com/products/2829)，带一个可选的 Bluefruit LE，驱动显示器。这是一个甜蜜的解决方案，因为它让[黛布拉]手机连接都在一个单一的板。她的网站上还没有这个版本的包，但她正在努力。现在，看看早期版本[吧](http://www.geekmomprojects.com/tweet-my-purse/)将一个 led 网格放在索环后面。一个很酷的想法，但我认为这个新版本更好。

她在自己的网站上还有很多其他的精彩内容，所以一定要多浏览一下。我仍然对我们今年早些时候看过的[黛布拉的]蚀刻机器人感到高兴。它为袖珍蚀刻草图增加了自动化，包括擦除功能，她甚至在 Tindie 上向[出售了一套工具。](https://www.tindie.com/products/geekmomprojects/etchabot/)
