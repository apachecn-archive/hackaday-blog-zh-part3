# 修改 Raspberry Pi 2 以适合小地方

> 原文：<https://hackaday.com/2016/04/22/modifying-a-raspberry-pi-2-to-fit-in-small-places/>

还是找不到圆周率为零的东西吗？我们知道。为什么不从 Raspberry Pi 2 上拆下几个头来适应你的下一个项目呢？使用一对 2.5 英寸硬盘驱动器外壳，[nodenet]用 Raspberry Pi 2 制作了一台[迷你 linux 笔记本电脑——它甚至有一个触摸屏，并配有 1000 毫安时电池！](https://imgur.com/a/J0P5n)

总的来说，所有组件花费了他大约 120 美元，但在你冲出去自己制作之前，你需要相当好的烙铁技能来成功缩小你的树莓派 2。修改需要移除两个 USB 端口、以太网插头、GPIO 引脚、HDMI 端口、A/V 插孔和摄像头连接器。

他用一个迷你钢锯和一个烙铁的组合来移除所有的组件——他的名片大小的计算机只剩下这些了——但真正有趣的部分是用导线重新连接所有的组件。
