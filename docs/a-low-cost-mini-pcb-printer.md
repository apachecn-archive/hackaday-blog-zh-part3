# 一种低成本微型 PCB 打印机

> 原文：<https://hackaday.com/2016/04/10/a-low-cost-mini-pcb-printer/>

家酿电子产品的下一个重大进步是一种将覆铜板转化为功能电路板的简单方法。从 60 年代开始，人们就用蚀刻笔、蚀刻涂层转印纸、一直流行的影印和熨衣架来做这件事，现在又用小型数控铣床来做。这仍然是一个混乱、缓慢和昂贵的过程。[johnowhitaker]和[esot.eric]正试图用一台由 DVD 驱动器制成的微型 PCB 打印机来解决后一个问题。

[john]和[eric]已经做了一段时间的 DVD 驱动器，而且理由充分。DVD 驱动器中有许多有趣的技术，马达、步进器和齿轮能够进行非常、非常精确的运动。大多数印刷电路板也不是很大，所以激光切割机只能穿过几平方英寸的区域，在这种情况下也不是什么大问题。

用一个小型二极管激光器安装在一个由 DVD 驱动器构成的 CNC 机架上，制作 PCB 的过程实际上非常简单。首先，将激光打印机墨粉和酒精的浆液涂在纸板上。接下来，这台印刷电路板打印机上的激光照射在迹线和铜填充物上，熔化墨粉。移除电路板，擦掉多余的墨粉，熔化掉不需要的铜。简单，哪怕有点乱。

当然，这种方法不能像你最喜欢的基于互联网的板房那样做电镀痕迹，但这确实比任何其他传统的自制方法有一些优势。它很便宜，因为 CD 和 DVD 驱动机制在制造商之间是非常标准化的。考虑到阻焊膜可以用光来固化，也很容易将阻焊膜印刷添加到这个构建中。这是一个非常酷的建筑，可以在世界上成千上万的车库和黑客空间中找到自己的家。
