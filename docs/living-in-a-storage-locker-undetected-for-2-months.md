# 住在储物柜里两个月没被发现

> 原文：<https://hackaday.com/2017/08/30/living-in-a-storage-locker-undetected-for-2-months/>

一名温哥华男子[007craft]，又名[Michael]，在 YouTube 上发布了一段视频，讲述了他为了攒钱买公寓而住在储物柜里的经历。狭小的空间意味着他不得不整合一些小工具来使住在那里更舒适。

虽然这可能是非法的，而且几乎肯定违反了储物柜的服务条款，但如果日子不好过，你似乎可以在储物柜里过得很好。[Michael]住在一个 U-haul 储物柜里，每月花费约 160 美元，配有床、酒吧、客厅和厨房，包括自来水。他继续解释他的第一个问题是电力，他必须从离他的单元相当远的插座获得电力，为此他只插入一根大的延长线，并将它绑在墙上，这样看起来不会太不合适，而他的供水系统使用两个水箱，一个用于废水，一个用于淡水。令人惊讶的是，他说他只需要每周从饮水机那里换一次就可以了。他设法在那里住了两个月，尽可能不被发现。

该视频包括一些小技巧，试图充分利用可用的小空间，即使你不打算住在储物单元，也值得一看，所以请在休息时查看。
