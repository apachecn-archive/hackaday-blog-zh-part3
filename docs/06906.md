# 当橡皮筋或 Nerf 飞镖不够用时，可以使用圆盘射击器

> 原文:[https://hacka day . com/2017/08/25/a-disc-shooter-for-when-rubber-bands-or-nerf-darts-arent-AlN-aflow/](https://hackaday.com/2017/08/25/a-disc-shooter-for-when-rubber-bands-or-nerf-darts-arent-enough/)

在每个人的一生中，都有需要以无害的方式射击的时候。对于那些时刻，有橡皮筋和 Nerf 飞镖，但即使这样，他们觉得仅仅是玩具。如果这是你找到自己的点，那么也许[奥斯汀]的[自制电动光盘射手](https://www.youtube.com/watch?v=sgaQ2l9lc0E)可以帮助你。

射手的操作很简单。一叠 3D 打印的塑料光盘被装入一个管状盒中，单个光盘由扳机控制的电机驱动凸轮推动。一旦光盘离开光盘盒，它就会到达由一个更强大的电机驱动的真空吸尘器传送带，从而将光盘加速到弹出速度。

视频下方显示了枪的结构，以及一系列涉及大量气球，汽水罐和食品的破坏。在我们看来，3D 打印弹药似乎是薄弱环节，因为根据我们的经验，这种类型的武器不可避免地会有很高的弹药损失率，但也许[奥斯汀]在一些廉价的细丝上有一条线。不管怎样，他的圆盘枪看起来像是那种可以为许多读者提供娱乐消遣的玩具。
