# 扩大范围:按风格划分区域

> 原文：<https://hackaday.com/2018/06/14/blinging-up-a-scope-scale-your-divisions-in-style/>

当黑客拥有一台示波器时，它就不仅仅是一件物品了。疲惫的夜晚是盯着显示屏度过的，疯狂地转动着刻度盘来哄出重要的信息。随着时间的推移，一种纽带形成了，只有最好的才适合你的范围。既然你可以选择金色，为什么还要满足于普通的塑料表盘呢？好吧，如果你没有注意到，我们在 Hackaday 这里偏爱有点过度工程化，并且【AvE】已经[通过增加金属旋钮](https://www.youtube.com/watch?v=BOabpxRq8io)升级了他的 Rigol 范围。

利用他在商店里的天赋，[AvE]首先从原料中车削出基本的旋钮形状，然后在其上钻孔并以一定角度铣削出外部纹理图案。瞧:六个定制旋钮可实现 100%的扭矩和牵引力控制。不管项目多么琐碎，看他工作总是好的。这个[AvE]视频没有附带通常的果味语言警告；取而代之的是，这种构建被设定为贝多芬的膨胀音调。“少说多做！”如题，但我们不得不说，我们怀念他的妙语。也就是说，他仍然设法通过行动传达他标志性的幽默。

对于一些稍微功能更强的示波器升级，你可以阅读关于[添加混合触摸屏界面](https://hackaday.com/2017/11/03/hybrid-interface-brings-touchscreen-to-rigol-scope/)或[破解 Rigol scope 的软件以解锁更大的带宽、存储深度等内容。](https://hackaday.com/2015/10/22/upgrading-rigols-more-expensive-oscilloscopes/)
