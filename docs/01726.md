# 机器人宠物测试自动宠物门

> 原文：<https://hackaday.com/2016/03/11/robotic-pets-test-an-automatic-pet-door/>

很多人养了一只宠物，然后破解了帮助他们照顾新朋友的解决方案，比如一扇通往户外的自动门。话说回来，有的人[先造宠物门再造宠物来测试](http://www.whatimade.today/geekcon-pets-2016-the-pets-authorization-project/)。

实际上并不像听起来那么奇怪。[Amir Avni]和他的妻子参加了最近的一次 geek con 并参加了 GeekCon Pets 活动。GeekCon 是一个合作性而非竞争性的黑客马拉松，它鼓励无用的构建作为一种培养社区和娱乐的手段。[Amir]和他的妻子想要建造一个全功能的自动宠物门，并且成功了——通过 NFC 和 ESP8266，步进电机驱动的门完全按照计划工作。但是没有任何真正的动物伙伴来测试这个系统，他们不得不招募一些志愿者。他们想出了一个 3D 打印的狗和猫栖息在无线汽车的顶部，随着 NFC 标签从他们的项圈上晃来晃去，这扇门能够区分流浪的替代动物。下面的视频展示了可爱的塑料朋友们的行动。

从所有的[宠物门](https://hackaday.com/2014/02/01/the-k9-curfew-door/)和[自动饮水器](https://hackaday.com/2013/10/21/pet-water-warden/)和[喂食器](https://hackaday.com/2015/11/05/automatic-cat-feeder-dispenses-noms-wants-cheezburger/)中可以清楚地看到，黑客们喜欢他们的宠物，但我们很确定这是第一次宠物本身被机器人取代。这对于测试环境来说很好，但是我们建议将真实的东西用于生产。
