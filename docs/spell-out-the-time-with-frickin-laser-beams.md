# 用该死的激光束拼出时间

> 原文：<https://hackaday.com/2017/08/29/spell-out-the-time-with-frickin-laser-beams/>

对于黑客来说，时钟是无穷无尽的魅力源泉。我们这里有各种各样的钟，从蒸汽朋克谢妮钟到复古酷酷的翻盖钟，再到看起来不像钟的钟。但是这是一个新东西——一个在黑暗中发光的激光追踪时钟。

tuckershannon 的时钟在实用性方面的不足在高端领域得到了弥补。这个想法很简单:用激光在磷光屏幕上描绘出字符。为了实现这一点，[tuckershannon]采用了[这款白板记号笔机器人时钟](https://hackaday.com/2014/02/13/a-clock-that-plots-time/)的设计，用夜光贴纸取代了绘图表面。双伺服缩放绘图仪在表面上跟踪 405 纳米激光二极管模块，为荧光粉充电。他没有提供幽灵般的图像会持续多久的线索，但从它的样子来看，我们敢打赌它会持续一分钟，尤其是在黑暗的房间里。同样，您会希望在下一个写周期到来之前图像完全消失，以防止覆盖前一次。

总而言之，这是一个很好的设计和一个聪明的新时钟显示模式。谁知道呢——也许这种发光的磷光显示屏真的会流行起来。
