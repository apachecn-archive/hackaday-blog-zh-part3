# 在圣马特奥日落之后，LED 接管了 Hackaday 的 BAMF 聚会

> 原文:[https://hackaday . com/2018/06/27/after-the-sun-set-on-San-Mateo-led-takes-over-hackadays-bamf-meetup/](https://hackaday.com/2018/06/27/after-the-sun-set-on-san-mateo-led-takes-over-hackadays-bamf-meetup/)

在今年春天的海湾地区创客大会于周六晚上关闭并赶走所有人后，乐趣转移到了奥尼尔的爱尔兰酒吧，在那里 Hackaday 和 Tindie 为创客大会的与会者举办了第五次年度聚会。我们如何在拥挤的酒吧里找到志同道合的黑客？很简单:找有 led 灯的桌子，打个招呼。不可能看到人们带来的所有东西，但这里有一些有趣的样本。

[![](../Images/3b67c3000b875b957823f6ce9a66dd56.png)](https://hackaday.com/wp-content/uploads/2018/07/benedwardswithmisty1.jpg) 来自 [Misty Robotics](https://www.mistyrobotics.com/) 的团队带来了他们的同名产品，当没有足够的空间让机器人运行时，他们抬着 Misty。这里[本·爱德华兹]正在回答关于米丝蒂能力的问题。该团队希望他们的产品能够吸引广泛的初露头角的机器人专家:从需要温和入口的初学者到对修改 Misty 以满足自己需求感兴趣的经验丰富的老手。

由于那些参加 Hackaday + Tindie meetup 的人倾向于后者，Misty 带来了一个黑客能力的演示:一个 3D 打印的啤酒杯支架取代了标准的左臂，符合 meetup 的位置。看到机器人的个性很有趣，Misty 环顾四周，偶尔对触摸感应头的顶部有所反应。

Hackaday.io 的长期参与者 Joshua Snyder 带来了他的项目 [RGB LED 矩阵](https://hackaday.io/project/13404-rgb-led-matrix-panels-arduino)和 [Neopixel 怀表](https://hackaday.io/project/13353-neopixel-pocket-watch)，给了那些只读过他的项目的人一个亲自玩它们的机会。下图所示的 Neopixel 怀表是一个隐形的变形金刚——当它关闭和不活动时，它会融入另一个时代的衣柜。但当它工作时，它绝对是一款 21 世纪的时计。RGB LED 矩阵也作为即兴酒吧拱廊投入使用。

 [![TinyCircuits made a great bar-top showpiece](../Images/a2f88b84d8c329b36ba2fc273c6a4fe4.png "TinyCircuitsPlayTable")](https://hackaday.com/2018/06/27/after-the-sun-set-on-san-mateo-led-takes-over-hackadays-bamf-meetup/tinycircuitsplaytable/) TinyCircuits made a great bar-top showpiece [![Joshua Snyder with is LED Pocket Watch and RGB matrix](../Images/3eeb6c2a36e4d73b2086aee5f6880a33.png "SnyderNeopixel")](https://hackaday.com/2018/06/27/after-the-sun-set-on-san-mateo-led-takes-over-hackadays-bamf-meetup/snyderneopixel/) Joshua Snyder with is LED Pocket Watch and RGB matrix

一些与会者很难在拥挤的酒吧里找到足够的桌子来布置，但是 TinyCircuits 的工作人员却没有这样的担心。他们能够将整个产品系列放在一个小圆吧台上，同时还能为每个人的啤酒留出足够的空间。如果你只在照片上看过这些作品，很难体会它们实际上有多小。真正理解这个尺寸需要一些实际操作时间，这张桌子上总是挤满了用指甲点击小按钮的人。

![](../Images/feae5a52f05f38bb1e0ef857a5f9e08e.png)

Hackaday 自己的 [Kerry Scharfglass](https://hackaday.com/author/kerryscharfglass/) 是本地人，他带来了他的 sympetrum 板在晚上展示。根据尼尔·斯蒂芬森的书*钻石时代*，这些徽章在同一个房间里会同步它们的颜色。凯瑞[为去年的 DEF CON](https://hackaday.com/2017/07/14/badge-from-diamond-age-comes-to-def-con/) 制作了徽章的第二版，但这里展示的是正在制作的第三版。我们还看到了制作 blinky LED 徽章的幕后场景，因为 Kerry 带来了一款开发工具，该工具采用 3D 打印夹具固定弹簧针，与他的 v3 板接口。

我们邀请了 [Hackaday 奖](https://hackaday.io/prize)的参与者带来他们的参赛作品，迈克尔·格雷厄姆是响应号召的人之一。他的 Mug-O-Matic 是他的[线性致动器模块](https://hackaday.io/project/158014-3d-printed-servo-linear-actuator)的示范作品，是我们机器人模块挑战赛的二十个[获胜者之一！遗憾的是，我们没有机会看到 Mug-O-Matic 的运行，因为迈克尔的饮料是装在一个小塑料杯里，而不是一个合适的杯子里。但是这个杯子足以激发这个夜晚的主要目的:谈论黑客和黑客攻击。](https://hackaday.io/list/159019)

你越往里走，房间就越拥挤，声音也越大，但你的努力会得到回报，你可以把项目插到后墙上的电源插座上。一个引人注目的例子是[贾马尔·戴维斯](https://hackaday.io/Dreadhead541)的[体积视点显示](https://hackaday.io/project/159306-volumetric-pov-display)。他的项目拥有一切——定制的 LED 电路板、为其供电的电线、旋转它们的电机、协调一切的微控制器，所有这些都装在一个 3D 打印的外壳内。
