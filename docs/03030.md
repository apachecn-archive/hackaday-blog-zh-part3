# Liddiard 全向轮

> 原文:[https://hack aday . com/2016/07/23/liddiard-全向轮/](https://hackaday.com/2016/07/23/liddiard-omnidirectional-wheels/)

全方位轮是发明世界中最顽强的多年生植物之一。毫不费力的平行停车的前景似乎让后院发明家的创造性思维活跃起来，多年来的结果是一连串令人印象深刻的工程化方法让汽车侧向移动。

最近的一个来自加拿大发明家[William Liddiard]，它值得再看一眼，因为它没有其他全向轮设计的一些机械复杂性。[Liddiard]的设计使用了一种整体式轮胎，其形式为内部带有一组辊子的柔性圆环，位于一个圆周上装有一组电动辊子的车轮上。整个轮胎可以绕其环形轴线旋转，导致胎面可以相对于车轮侧向移动。

整个过程在视频中演示，视频显示在休息时间的下方，作为演示车辆的小型丰田汽车可以轻松地侧向移动和旋转。在公路汽车上使用这些车轮之前，我们会保持警惕，直到当它们不处于全向模式时，它们能够在侧向稳定性方面与传统轮胎相匹配，但我们可以立即看到，它们将对工业机器(如狭窄空间中的叉车)的操作员提供重要帮助。
