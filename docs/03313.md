# Hackaday 奖参赛作品:另类摄影打印机

> 原文:[https://hackaday . com/2016/08/24/hackaday-prize-entry-a-printer-for-alternative-photography/](https://hackaday.com/2016/08/24/hackaday-prize-entry-a-printer-for-alternative-photography/)

胶片摄影始于汞银汞合金，终于硝化纤维素、碘化银和染料。一路走来，摄影世界中发生了一些非常奇怪的化学反应，从铁盐和银盐到在氰型和蓝图中发现的普鲁士蓝。

金属盐很有趣，为了他的 Hackaday 奖参赛作品，【大卫·布朗】[正在为这些替代的摄影过程制造一台打印机](https://hackaday.io/project/10101-direct-uv-printer-for-alternative-photography)。这不是一个暗室——这是一台激光打印机，旨在用奇怪的化学物质复制图像。

将铁氰化钾和柠檬酸铁铵涂在某种介质上，通常是纸或布，就可以制成氰型。然后通过紫外光(即太阳)曝光，任何未曝光的都被洗掉。[大卫]用普通的紫外激光二极管代替太阳来曝光他的照片。他已经设计好了这台打印机的机械结构，他应该能够达到 750 dpi 分辨率和 8 位单色的目标。

数码摄影永远不会消失，但总会有一些人尝试感光化学物质。我们还没有看到很多人尝试这些奇怪的替代摄影过程，任何让这些非常酷的照片进入世界的东西对我们来说都是好消息。
