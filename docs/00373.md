# 雷神之锤能识别主人的手

> 原文：<https://hackaday.com/2015/10/14/thors-hammer-build-recognizes-its-masters-hand/>

只有那些完全与现代流行文化绝缘的人，才会意识不到雷神的意义。它当然是神话中的雷神之锤，而且只有雷神。这是一个是否值得的问题；这个问题被[这款电磁雷神锤](https://www.youtube.com/watch?v=0_8Xhzt5YQI)完美解决。

用电磁铁很聪明，对吧？只要把东西放在金属物上(金属物本身很重或者固定得很好)，就没有人能把它捡起来。当你添加一个指纹读取器时，它开始变得更加有趣，只允许 Mjolnir 的主人从井盖上检索它。

但对我们来说，真正的天才在于锤子大部分时间并不消耗四节 12V 电池的能量。下面视频中的所有人都可能拿起锤子，如果他们先用脚把它从金属板上推下来的话。该建筑使用电容式触摸传感器来启用和禁用作为电磁铁的微波变压器。像这样的工程技巧真的把神和装腔作势的人区分开来。

我们讨厌承认这一点，但这可能比几年前[Caleb]建造的 Telsa 线圈供电的 Mjolin 更酷。尽管如此，他仍然是多年来最好的，如果你要被取代，这真的是一个值得新头衔的构建:最酷的 Mjolnir hack。
