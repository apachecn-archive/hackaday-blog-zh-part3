# 建造一个 K9 玩具

> 原文：<https://hackaday.com/2017/06/22/building-a-k9-toy/>

(詹姆斯·韦斯特)有一个年轻的医生，他在家里扇风，想造一个可以玩的东西，而不用担心它被碰撞和刮伤。因此，[詹姆斯]没有创造一个复制品，而是为他的年轻粉丝制作了一个简单的遥控 K9 玩具。

K9 是经典《神秘博士》系列中第四个博士(由汤姆·贝克扮演)的同伴。他还出演了几部衍生剧。K9 是一只对 TARDIS 了如指掌的机器狗，也是一台激光器，成为世卫组织粉丝的最爱，尤其是年幼的儿童。[James]希望他的 K9 版本能够通过遥控器控制，并且能够播放电视节目中的声音。

使用一些手工切割的丙烯酸树脂，[詹姆斯]建造了 K9 的身体，然后开始计划运动控制和大脑。[James]为控制板选择了 Raspberry Pi Zero，为音频选择了扬声器 pHat，为移动 K9 选择了几个电机，以及一个电机控制器。K9 由 WiiMote 控制，背部有一个按钮，可以开始与 WiiMote 配对(K9 在配对成功时回答“肯定”。)说到头部，[詹姆斯]有点不知所措，试图用丙烯酸树脂制作头部，所以他找了些泡沫板，用那个代替。头部的红色 LED 通过半透明的红色丙烯酸树脂发光。

这是一个很棒的小项目，[James]已经将 Python 代码放到了 Github 上，供感兴趣的人使用。这些年来，我们在网站上已经有了几个机器狗项目，像[这个](https://hackaday.com/2011/08/16/amazing-60-year-old-robot-dog-is-a-mystery/)和[这个](https://hackaday.com/2012/10/02/professional-looking-dog-robot-was-actually-martins-masters-thesis/)。
