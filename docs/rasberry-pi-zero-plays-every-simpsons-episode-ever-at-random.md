# Rasberry Pi Zero 随机播放《辛普森一家》的每一集

> 原文：<https://hackaday.com/2016/05/08/rasberry-pi-zero-plays-every-simpsons-episode-ever-at-random/>

如果说 Raspberry Pi Zero 比《辛普森一家》(The Simpsons)剧集的洗牌机(shuffler)有更好的用途，我们还没听说过。

创作者[斯蒂芬·科伊尔]从[威尔·史密斯]几年前在测试播客中提到的对这种设备的迫切需求中获得了灵感。这个小工具只是一个带有熟悉的黄色按钮的零——希望它是[Pantone 116 C](https://www.pantone.com/pages/myp_mypantone/mypPaletteDetail.aspx?p_id=117&p_type=p_detail)——从 SD 卡中随机选择一集。[Stephen]对该节目超过一半的作品有明确的看法，认为只有第 2 到第 10 季值得载入卡片。顺便说一句，在看到所有 593 集都可以轻松存储在 128GB 的 SD 卡上后，我们觉得自己很老了——我们开始虔诚地将每一集都记录在 VHS 磁带上，但在几季后，当集合变得太大而无法处理时，我们不得不停止。

如果从 DVD 中翻录剧集不是你的风格，或者你仍然喜欢第一轮的东西，你可能想看看这个令人困惑的名字 [Smart Homer](https://hackaday.com/2014/10/14/homer-robot-ensures-you-dont-miss-an-episode/) ，这样你就不会错过任何一集。
