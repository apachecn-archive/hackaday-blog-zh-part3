# 逆向工程大脑，一次一个神经元

> 原文：<https://hackaday.com/2018/06/30/reverse-engineering-brains-one-neuron-at-a-time/>

这里的大多数帖子都是电气或机械的，也有一些来自其他领域的零星帖子。那些关注生物医学研究进展的人可能已经注意到，某些领域正开始与我们所知的电子学相类似。Rajib Shubert 博士]就是这样一个领域的专家，他也发现了这一共性。他认为通过使用普通观众熟悉的类比来解释他的研究，将两个世界联系起来会很有趣。(视频也嵌在下面。)

他用一点背景知识奠定了基础，确立了我们已经能够通过显微镜载玻片等看到单个静态神经元一段时间，并且我们已经能够通过功能性 MRI 看到整个活体大脑的活动。这些方法逐渐改善了[我们对神经元](https://hackaday.com/2017/06/02/the-neuron-a-hackers-perspective/)的理解，过去几年的进展已经达到了这两点的交汇点:[舒伯特博士]和同事们现在有工具可以窥视功能大脑内部，一次梳理出一个神经元是如何工作的。

[舒伯特博士]的演讲对电子硬件进行了类比，但我们也可以进行软件类比，将大脑视为一段高度优化(和/或混淆)的代码。病毒在这个类比下给单个细胞打上印记，就像是隔离了一个单一的功能，看谁调用它，它调用谁。这与[光遗传学](https://hackaday.com/2017/08/03/optogenetics-for-100-euros/)技术配合得很好，这可以被视为就像修改一个函数来实时观察它如何影响结果。这无疑给“使用实时代码”这个短语赋予了不同的含义！

之后有一个活跃的问答环节和多个小组讨论。不像高能物理学家和他们数十亿美元的实验设备，[舒伯特博士]相信这项技术将在不久的将来进入普通的研究实验室。这种潜力让观众既兴奋又担忧，因为人们引用了各种即将成为现实的科幻小说。人们一致认为公共政策落后于技术发展，但对于如何解决这一问题却没有共识。

最后一个小细节:随着对神经元的更好理解，[舒伯特博士]说，我们现在知道真正的神经网络与它们启发的软件创作截然不同，这推动了对新一代软件的研究。这一点以及视频流中没有的更多精彩见解是亲自参加 [Hackday 洛杉矶聚会](https://www.meetup.com/Hackaday-Los-Angeles)的绝佳理由！
