# 用于机器人的 HID

> 原文：<https://hackaday.com/2018/06/01/a-hid-for-robots/>

无论是这里介绍的项目还是现实世界中的项目，我们都倾向于关注最终产品。汽车，太阳能电池板，甚至机器人。但幕后还有很多事情需要处理，无论是保持汽车运行的燃料基础设施，还是制造硅片的半导体制造商，还是机器人的控制系统。这个项目就是后者之一:[一个完全 DIY 的机器人手臂的人机界面装置](https://hackaday.io/project/158481-circuito-arduino-controller-pad)。

虽然机器人通常是自动化的，但有些仍然需要人工输入。人类的输入可以一直被需要，或者可以被用来最初教导机器人如何执行任务，该任务随后将被自动化。这种由[Ahmed]制造的“键盘”带有操纵杆、电位计和四个开关输入，它们都可以通过 Arduino 完全编程。有了它，你几乎可以用你需要的任何类型的机器人执行任何动作，而且因为它是基于 Arduino 的，所以也很容易扩展。

下面的视频和项目页面有所有的说明和材料清单，如果你想推出自己的。这是一个非常简单的项目，但可能值得一试，因为我们不经常为其他事情配备控制器，[尽管我们有时会看到它们用于控制望远镜](https://hackaday.com/2017/12/05/astro-cat-raspberry-pi-telescope-controller/)而不是机器人。
