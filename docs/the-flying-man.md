# 飞人

> 原文：<https://hackaday.com/2015/11/01/the-flying-man/>

不，那不是火男的错别字。当你拿着一百英尺长的塑料板，焊上一个大坡道，把一辆汽车改装成你自己的高速绞车，并把它们都放在湖边时，你会得到什么？你得到了一些重要的播出时间。

一群法国水上运动爱好者决定建造这个极不明智的超级滑梯。他们在一栋旧农舍旁边建了一个巨大的坡道，通向湖边。底部是一个他们用钢焊接而成的大坡道。现在你只需*沿着这条滑道滑下*就可以了。

但是他们并不满足于此。

他们用千斤顶托起了一辆旧车，并用一个裸露的钢圈替换了其中一个轮子，用作绞盘。他们用一些高强度的电缆将它横跨在湖面上，缠绕在车轴上的另一个钢圈上，作为一个巨大的滑轮。把缆绳牵回湖对岸，把它挂在内胎上，准备好开始你的人生之旅吧！
