# 杰里·埃尔斯沃斯参观无线电和技术博物馆

> 原文：<https://hackaday.com/2017/10/27/jeri-ellsworth-tours-a-radio-and-tech-museum/>

在中断了四年之后，【Jeri Ellsworth】再次做了一些 YouTubing】(是的，这是一个词，就像 YouTuber 一样)。她最近上传了一个非常令人愉快的四集系列[，在西弗吉尼亚州亨廷顿的无线电和技术博物馆巡回演出](https://www.youtube.com/watch?v=NyZ87wNotBg)。

第一部分包含跨越时代的收音机，从火花隙发射机开始，一些美妙的水晶套，从 20 世纪 40 年代的袖珍收音机，从 20 世纪 30 年代到 50 年代的商用业余无线电发射机和接收机，等等。甚至有一个可爱的发射器安装在一个旧冰箱里。第二部分包括教育玩具，第三部分包括电视机和照相机，第四部分是关于所有类型的电唱机和高保真音响。每一个都包含了和第一部分中的火花隙发射器一样古老的设备。

你可能知道[Jeri]是增强现实初创公司[的联合创始人](https://hackaday.com/2017/06/27/castar-shuts-doors/) [castAR](https://en.wikipedia.org/wiki/CastAR) ，该公司最近关闭了，但在此之前，她在黑客中因其众多项目而闻名，从[柔性电致发光显示器](https://hackaday.com/2010/11/16/jeri-makes-flexible-el-displays/)、[使用被黑进的馈电喇叭的厘米波扫描仪](https://hackaday.com/2010/12/03/make-your-own-tsa-naked-scanner/)，到 yours truly 的个人最爱[Commodore 64 bass key tar](https://hackaday.com/2012/07/09/jeri-ellsworth-on-making-her-c64-bass-keytar/)。

所以用微波炉加热一些爆米花，坐下来，享受休息后的旅程。
