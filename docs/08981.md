# 微型电锯获得急需的硝化甘油动力

> 原文:[https://hackaday . com/2018/04/10/微型电锯-获得-急需-硝基-动力-助力/](https://hackaday.com/2018/04/10/micro-chainsaw-gets-a-much-needed-nitro-power-boost/)

当生活递给你世界上最小的电锯时，除了让它变得更可笑，还能做什么？这就是[JohnnyQ90]所做的，当时他[对一台迷你电锯进行了大规模改装，配备了强大的遥控汽车发动机](https://www.youtube.com/watch?v=bCwivPC7YzU)。

有问题的锯子是一款采用“纳米刀片技术”的博世 EasyCut，只能被定义为最宽泛意义上的电锯。这是一种无线工具，用于轻度修剪等，迫切需要[蒂姆·泰勒工具]的治疗。变形开始于动力传动系统的拆卸和 1.44 cc 硝基 RC 汽车发动机的定制离心离合器的添加。引擎需要一个定制的底座来安装在机箱内，原始的 PCB 是完美的模板。原来的箱子被带锯和 Dremel 减轻了很多重量，冷却风扇是 3D 打印的，一个迷人的复杂的油门连杆将所有东西绑在一起。随着燃料箱隐藏在新的 3D 打印手柄中，整个事情看起来就像它一直应该有这个引擎。下面的第三个视频展示了它的作用；不幸的是，由于发动机旋转方向错误，并且没有空转齿轮的空间，[JohnnyQ90]不得不满足于将杆上下翻转来切割。但是对于一些黑客来说，旅程比目的地更让我们感兴趣。

这并不是[JohnnyQ90]的第一次硝基竞技——他以前在无绳电钻[和 Dremel](https://hackaday.com/2017/01/16/cordless-drill-uses-no-electricity/)上做过硝基转化。你也应该看看[的微型特斯拉涡轮机](https://hackaday.com/2016/06/17/micro-tesla-turbine-is-an-engineering-tour-de-force/)，尤其是如果你喜欢精细加工的话。
