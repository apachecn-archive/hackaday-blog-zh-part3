# 家居店 DIY 注塑模具设计

> 原文：<https://hackaday.com/2017/11/11/diy-injection-mold-design-for-the-home-shop/>

3D 打印对于原型制作来说很棒，对于有限的零件运行来说也不错。不幸的是，虽然它真的不能很好地扩展超过几件，所以当你准备好面向大众市场时，你需要考虑注射成型你的零件。但是像这样的事情必须外包出去，对吗？也许不会，如果你对[设计自己的注塑模具](https://www.youtube.com/watch?v=ZYFoWP-3MYE)略知一二的话。

下面的视频来自[戴夫·哈肯斯]通过他的[珍贵塑料](https://preciousplastic.com/)项目，该项目的使命是将塑料回收的手段放到个人手中，而不是依赖于市政项目。[我们在](https://hackaday.com/2016/04/03/think-globally-build-locally-with-these-open-source-recycling-machines/)之前已经报道过他们的工作，看起来他们已经走了很长一段路来实现这个梦想。[Dave]的同事[Jerry]编写的本教程涵盖了注塑模具设计的基本要素，从 Solidworks 中的 3D 建模开始。[Jerry]指出了 DIY 注射成型的局限性，包括零件厚度与注射压力的关系。同样重要的是一些特征，如缓和的曲线以减少加工工作量，在浇口上留有适当的拔模斜度，以及设计零件以方便脱模。[Jerry]和[Dave]外包了这个模具的加工，但是没有理由一个相当复杂的模具不能由家庭游戏玩家生产。

当你学习完模具设计后，你会渴望建造自己的注射成型机。[宝塑的教程](https://www.youtube.com/watch?v=qtZv96ciFIU)看起来死简单，但是[这个机器](https://hackaday.com/2009/08/12/home-injection-molding/)看起来能力强一点。还有[当你可以 3D 打印模具的时候，为什么要数控加工它们](https://hackaday.com/2013/09/15/3d-printed-injection-molds/)？
