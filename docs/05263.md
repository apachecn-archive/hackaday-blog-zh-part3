# 3D 打印变得俗气

> 原文：<https://hackaday.com/2017/03/09/3d-printing-gets-cheesy/>

你有没有想过，你看到的所有待售的东西——无论多么平凡——都是某人的生活激情？或者，至少，他们的工作热情。就在我们说话的时候，某个地方有两三个人在一个房间里，试图找出如何以比以前更低的价格制作一个放屁坐垫。有人在吹捧鸡蛋纸盒最新设计的优点。那些制造管道把你的钱送到银行柜台的人？他们也存在。

我们很自然会想到改进 3D 打印机，但我们大多数人打印的是塑料。我们可能希望我们可以打印金属。但是一些地方的研究人员正在打印奶酪。我们没说有很多黑客，我们说的是研究人员。例如，爱尔兰科克大学有一个彩色幻灯片。他们以两种不同的速度打印奶酪，并使用激光扫描显微镜和流变仪来分析结果。我们在塑料工厂见过流变仪，但从未在厨房见过。与此同时，在黑客方面，显然[喷雾奶酪](https://hackaday.com/2015/03/31/fail-of-the-week-easy-cheese-printer-says-no/)罐可以作为一个简单的冷挤压机工作(见下面的视频)。

如果你觉得这个主题很有趣，食品工程杂志 2 月版有一篇关于 3D 打印如何影响奶酪结构的文章，看起来好像是同一组的。这篇文章不是免费的，尽管你可以在 [ResearchGate](https://www.researchgate.net/publication/313473105_Effect_of_3D_printing_on_the_structure_and_textural_properties_of_processed_cheese) 上申请一份免费的。我们猜测他们真的喜欢不同形状的奶酪。他们的结论？不规则形状的脂肪球使得印刷奶酪比普通奶酪更软。谁知道呢？

我们觉得有必要指出，3D 打印机中的一些东西可能不是食品安全的，所以我们不会吃任何从我们的热端出来的东西。从奶酪罐头挤压机里出来的东西可能不比打印机外面的东西更容易食用，尽管到底有多容易食用还有待讨论。也许你最好打印煎饼。
