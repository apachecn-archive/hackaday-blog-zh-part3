# 迷你复古宠物电脑

> 原文：<https://hackaday.com/2016/12/11/mini-retro-pet-computer/>

曾经有一段时间，准将宠物是北美学校的标准电脑。它集多种功能于一身，坚固的结构使它成为教育市场的理想选择，对于我们中的一些人来说，这只宠物开始了一生对计算机的热爱。Adafruit 的[Ruiz Brothers]已经设计出了一款运行在微控制器上的[微型宠物模型](https://learn.adafruit.com/mini-commodore-pet-with-charlieplexed-led-matrix/overview)，它装载了一个绿色 LED 矩阵，具有真正的复古外观。

虽然不是宠物的工作模型，但该模型运行在 Adafruit Feather M0 基本原型上，该原型是 Atmel ATSAMD21 Cortex M0 微控制器，可以在 Adafruit 的 16×9 charlieplexed led 矩阵上显示图形。ATSAMD21 是在 [Arduino Zero](http://hackaday.com/2014/05/15/introducing-the-arduino-zero/) 中使用的芯片，所以我相信我们在未来会看到更多这种芯片。像 Adafruit 上的所有教程一样，这个教程非常详细，有一步一步的动画图片来帮助你。很明显，你不需要他们正在使用的硬件，但是如果你从 Adafruit 订购，为什么不呢？

3D 打印宠物的计划是免费的，所以即使你不想把他们的 LED 矩阵和微控制器放在里面，你仍然可以自己打印出一个很棒的道具，3D 打印宠物只会使用大约一美元的灯丝。当然，虽然这是一个很酷的复古模型，但如果你身边有一个 Commodore 宠物，你可能会用它做些别的事情。我们没有，所以你听到的声音是我们的 3D 打印机打印过去的声音。
