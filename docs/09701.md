# 一个完全开源的 Raspberry Pi 合成器

> 原文:[https://hack aday . com/2018/06/08/a-完全开源-raspberry-pi-synthesizer/](https://hackaday.com/2018/06/08/a-fully-open-source-raspberry-pi-synthesizer/)

你有没有看到某样东西，立刻就知道那是你想要的东西，尽管几秒钟前你还不知道它的存在？这就是当我们收到一个关于 Zynthian 的提示时的感受，Zynthian 是一个完全开源的(硬件和软件)合成器。您可以直接从开发人员那里在线购买该工具包，或者使用他们的文档和源代码从头构建自己的工具包。拥有众多的滤镜、特效、引擎和本质上无限的升级潜力，他们称之为“合成的瑞士军刀”。我们倾向于同意。

在最基本的层面上，Zynthian 是一个带有触摸屏、几个旋转编码器、专用声卡和 MIDI 支持的 Raspberry Pi 3。从软件角度来看，最大的特点可以说是实时 Linux 内核，以尽可能降低延迟。还有一个自定义的网络界面，所以如果你愿意，你可以从网络上的另一台机器上控制 Zynthian。当然，它还包括一系列预装的音频包来进行实验和创作。

套件以各种价格提供，从 420 美元的顶级型号到几美元的无人居住的 PCB。我们喜欢他们这样分解事物；允许各种技能(和/或耐心)的用户支付他们想要的。如果您只想购买定制板并推出自己的案例和 Pi 解决方案，您可以这样做。

如果你想全力以赴，你也可以从头开始建造一个。从外壳的 CAD 文件到他们定制的旋转编码器库，所有的东西都是完全开放的(大部分在 GPL v3 下许可)，任何人都可以以他们认为合适的方式使用。wiki 中甚至有一个页面列出了项目没有正式支持的硬件，但是对于那些想要开辟自己道路的人来说，仍然是一个选择。

合成器是一个相当受欢迎的黑客项目，从[谷歌的人工智能版本](https://hackaday.com/2018/03/16/google-builds-a-synthesizer-with-neural-nets-and-raspberry-pis/)到[单片机节俭练习](https://hackaday.com/2017/05/09/diy-tiny-single-pcb-synthesizer/)。如果你想了解更多关于数字噪音和音乐之间的细微差别，看看我们自己的【埃利奥特·威廉姆斯】制作的[这个奇妙的系列。](https://hackaday.com/2015/08/07/logic-noise-4046-voltage-controlled-oscillator-part-one/)
