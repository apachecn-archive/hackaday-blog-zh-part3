# 自顶向下的 FPGA 设计

> 原文:[https://hack aday . com/2017/10/24/FPGA-design-with-free-software/](https://hackaday.com/2017/10/24/fpga-design-with-free-software/)

[Roland Lutz]在今年的 MetaRheinMainChaosDays 大会上给了[一个关于 FPGA 设计的演讲](https://www.youtube.com/watch?v=eWo7CINPu4w)。你可以看下面的视频。这是对 FPGAs 的一个很好的介绍，涵盖了最底层的细节和一些更高层次的见解。如果你开始使用这些 FPGAs，这个视频一定要看。

[罗兰]从强制性的介绍材料开始。然后，他跳到一个实际的例子中，然后缩小来看 Lattice FPGA 的内部细节。例如，这种 FPGA 支持多个比特流，所以你可以在不同的“程序”之间动态切换。

如果你想知道瓷砖和细胞是如何联系的，你会从低级的讨论中得到很多。还有一个非常简洁的解释，说明了瓷砖是如何相互连接的。如果你不清楚 span-4 和 span-12 电线的区别，你会想看的。

网上有很多补充的细节，[Roland]在演讲的最后提供了一些参考资料。当 IceStorm 工具首次出现时，我们制作了自己的介绍视频。如果你喜欢图形工具，你可能会对 [Icestudio](https://hackaday.com/2016/02/23/icestudio-an-open-source-graphical-fgpa-tool/) 感兴趣。
