# Hackaday 奖参赛作品:MappyDot，一个微型智能激光雷达传感器

> 原文：<https://hackaday.com/2017/08/21/hackaday-prize-entry-mappydot-a-micro-smart-lidar-sensor/>

[Blecky]的 Hackaday 奖参赛作品是 [MappyDot](https://hackaday.io/project/25571) ，这是一个不到一平方英寸大小的微型板，拥有一个 VL53L0X 飞行时间距离传感器，可以测量高达 2 米的距离。

MappyDot 不仅仅是一个分线板；每个 PCB 上的 ATMega328PB 微控制器提供过滤功能，这是一个易于使用的 I2C 接口，可自动处理多达 112 个总线连接的电路板。这个想法是，一个或几个 MappyDots 可以单独使用，但管理大量 MappyDots 也同样容易。通过在设备上点上多个指向不同方向的地图点，设备可以组合读数，以获得对其物理环境的类似激光雷达的理解。它的大量 MappyDots [Blecky]也正在进行:他刚刚收到了几块裸露的 PCB 板,他很快就会费力地组装起来。好消息是，每块板上没有*那么多组件。*

很高兴看到开源项目和工具，其中很明显有些想法已经使它们变得灵活和易于使用。这意味着他们更容易融入其他工作，并有助于使他们成为 Hackaday 奖的优秀参赛者。
