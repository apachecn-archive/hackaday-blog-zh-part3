# 死亡、税收和洗衣

> 原文:[https://hackaday.com/2016/04/19/death-taxes-and-laundry/](https://hackaday.com/2016/04/19/death-taxes-and-laundry/)

有句老话说，唯一确定的两件事是死亡和税收。然而，除非你住在裸体主义者聚居地，否则这里可能也有洗衣房。[Darpan Bajaj]和一些朋友在一次黑客马拉松上，决定把他们的[洗衣机放到互联网上](http://www.instructables.com/id/Washman-the-Washingmachine-Monitoring-System/)。

我们 Hackaday 的大多数人——从评论来看，还有许多 Hackaday 的读者——都有点怀疑我们到底有多需要连接到互联网上的所有东西。然而，洗衣机可能不是一个坏主意:你经常使用它，你需要知道它什么时候洗好，你可能不想只是坐着看它旋转。此外，计划安装在宿舍，那里有多台机器和许多潜在用户。

该系统有两个主要传感器:超声波传感器用于确定水位，振动传感器用于确定机器是否正在运行。这有利有弊。看起来直接连接到洗衣机的电子设备上会更干净，更不容易出错。然而，使用传感器意味着该设备可能可以用于任何洗衣机。自然，大脑是 Arduino 和 Bolt 物联网平台。

一个网络界面显示了几台机器中哪些是空闲的。你可以在下面看到一个关于这个系统的视频。我们最近收集了很多[洗衣项目](http://hackaday.com/2016/04/02/hacklet-102-laundry-projects/)，如果那是你的东西的话。另外，别忘了洗衣服最糟糕的部分:[折叠](http://hackaday.com/2016/02/24/the-challenges-of-a-laundry-folding-robot/)。
