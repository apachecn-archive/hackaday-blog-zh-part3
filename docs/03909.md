# 3D 宇宙剧场

> 原文：<https://hackaday.com/2016/10/23/3d-universe-theater/>

如果你是一个天文爱好者，你可以在印刷品或网上(甚至在你的智能手机上)找到大量的星图。但如果你是科幻迷(或作家)，你可能会发现那些地图令人沮丧，因为它们是平的。地图上两颗相邻的恒星在页面外的轴上可能相距光年。距离太阳 3.2 光年的恒星在地图上看起来和 100 光年远的恒星一样。

盖亚卫星(欧空局的一个项目)在月球以外的轨道上运行，并仔细绘制它看到的每个光点的三维位置。[Charlie Hoey]获取了大约 200 万颗恒星的数据，并使用 WebGL 在 web 浏览器中给你提供数据的 3D 视图。

我们不确定它有多实用，如果你有一个旧的 CPU 和显卡，放大时会有明显的延迟。但是它看起来很有趣，是基于浏览器的数据可视化的一个很好的例子。你可以阅读更多关于[盖亚](http://sci.esa.int/gaia/)(或观看下面的视频)并注意到他们计划绘制 10 亿颗恒星的地图。值得注意的是，这大约是感兴趣区域内 1%的恒星。Gaia 相机在技术上也很有趣(请看下面的第二个视频)。

如果你想用望远镜看，试试天网。或者你可以[试试无线电天文学](https://hackaday.com/2016/08/30/citizen-scientist-radio-astronomy-and-more-no-hardware-required/)，如果你喜欢的话，不用硬件。
