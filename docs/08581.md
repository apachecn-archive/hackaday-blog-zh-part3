# 周五黑客聊天:创造空间

> 原文:[https://hack aday . com/2018/02/14/Friday-hack-chat-making-a-makers space/](https://hackaday.com/2018/02/14/friday-hack-chat-making-a-makerspace/)

你如何创造一个创客空间？在过去的十年里，有很多讲座和教程给出了一些建议。一周中没有哪一天适合开会，所以会议总是在星期二举行。自行车棚将被漆成橙色，没有例外。你将会是一个营利性的，还是非营利性的？我们是*创客*空间还是*黑客*空间？杰瑞，别耍小聪明了。潘通 021 U。

[![](../Images/4dc277fd21dcdb0756504575517fba5d.png) ](https://hackaday.com/wp-content/uploads/2018/02/makerspacehackchat-011.png) [在本周的黑客聊天](https://hackaday.io/event/49277-making-a-makerspace-hack-chat)中，我们将谈论所有关于创造空间的话题。这些是社区中心，人们聚集在一起，共享资源，把他们的发明变成现实。这并不像看起来那么简单。你需要保险，你需要一栋大楼，你需要一个很酷的房东，还有许许多多可能出错的事情。谁最能带领你度过开黑客空间的风暴？你可以向谁征求建议？

我们本周黑客聊天的嘉宾是 Vaibhav Chhabra，来自波士顿大学的机械师。他花了两年时间研究一种眼睛诊断设备，是麻省理工学院 REDX 医疗保健创新实验室的讲师，也是“不可思议的制造者收容所”的创始人。埃里克·米肖是一名黑客、跑步爱好者和作家，目前正在开发 Rift Recon、Shellcon 和[hackerspaces.org](http://hackerspaces.org/)。他写过关于 Adafruit 的教程，在他去芝加哥创办 PS:One 之前，他是 HacDC 的创始人之一。

本周 Hack Chat 的主题包括开设一个创客空间需要什么，如何为其提供资金，关于盈利和非盈利的组织结构，以及成员们最关心的事情:对于一个成功的创客空间来说，什么设备是最关键的。当然，我们鼓励你在黑客聊天中添加自己的问题；为此，[只需在 Hack Chat 活动页面](https://hackaday.io/event/49277-making-a-makerspace-hack-chat)上发表评论。
