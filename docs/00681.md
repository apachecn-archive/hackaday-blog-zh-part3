# FleaFPGA+Arduino Uno = FleaFPGAUno

> 原文:[https://hack aday . com/2015/11/21/fleafpga-arduino-uno-fleafpgauno/](https://hackaday.com/2015/11/21/fleafpga-arduino-uno-fleafpgauno/)

有些东西在一起更好:我和我的妻子，花生酱和果冻，FPGAs 和 Arduino Unos。资深黑客[Valentin Angelovski]似乎同意这一点:FleaFPGA Uno 是他的最新作品，结合了 FPGA(T2 的 Lattice mach x02 700 HC)和兼容 Arduino 的 CPU。

这是从最初的 FleaFPGA 升级而来的型号。加上一些其他组件(如 HDMI 和复合视频输出以及 WiFi 选项)，您就拥有了一个试验 FPGAs 或构建嵌入式系统的杀手级组合。这是因为 Arduino 部分将 FleaFPGA Uno 从试验板中解放出来:您可以使用 Arduino IDE 通过串行线路或无线链路轻松编程、控制 FPGA 并与之交互。甚至还支持 Arduino 屏蔽(尽管只有 3.3V)，使其更具可扩展性。这将是一个复古游戏系统的可怕起点，因为[许多 8 位控制台](http://hackaday.com/2011/04/14/fpga-based-turbografx-16-clone/)可以很容易地在 FPGA 中模拟。[Valentin]目前正在直接销售这些板，WiFi 版的价格非常合理，为 50 美元或 60 美元。
