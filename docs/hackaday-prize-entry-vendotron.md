# Hackaday 奖参赛作品:Vendotron

> 原文：<https://hackaday.com/2016/10/07/hackaday-prize-entry-vendotron/>

在世界各地的黑客空间中，一个反复出现的想法似乎是零件自动售货机。需要 Arduino、ESP8266 或电机控制器吗？没问题，从机器上买一个就行！

大多数这样的机器都是食品和饮料自动售货机行业的剩余产品，因此从印有流行巧克力棒标志的机器上购买 Arduino 并不罕见。然而，购买二手机器可能会很贵，而且通常需要做一些工作才能投入运行。

自动售货机本质上不是一台复杂的机器，当你背后有黑客空间的资源时，它也不难建造。[迈克·马查多]正在这样做，他建造了 Vendotron，这是一种由激光切割胶合板和中密度纤维板制成的旋转式自动售货机。整个过程由 Arduino 控制，传送带由步进电机驱动。

它没有做任何商业自动售货机多年没有做的事情，除了可能有一个允许电话和比特币支付的软件界面。不过，这个项目的得分在于展示了自动售货机不需要昂贵或难以建造，并为任何想要一台自动售货机的黑客空间拓宽了访问途径。

我们以前在这里有过一些自动售货机，像这个关于商业机器原型制作过程的功能，或者甚至是[这个发微博的](http://hackaday.com/2015/02/10/a-tweeting-vending-machine/)。遗憾的是，很少有人[有发送免费苏打水的秘密按钮](http://hackaday.com/2010/05/17/old-school-vending-machine-learns-new-tricks/)。
