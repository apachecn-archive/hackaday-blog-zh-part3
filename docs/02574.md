# 入侵特斯拉 Model S 后驱动装置

> 原文:[https://hack aday . com/2016/06/06/hacking-the-Tesla-model-s-rear-drive-unit/](https://hackaday.com/2016/06/06/hacking-the-tesla-model-s-rear-drive-unit/)

[Jason Hughes]是特斯拉的忠实粉丝，他花了大量的时间对他们进行黑客攻击，以弄清楚这家汽车制造商在做什么花哨的事情。他最近的冒险是使用特斯拉 Model S 的[后驱动装置。](http://insideevs.com/salvaged-tesla-model-s-rear-drive-unit-spun-bench-videos/)

[Jason]之前在特斯拉社区有些名气；他最广为人知的黑客行为是在固件中找到特斯拉下一版汽车的型号。为了这个项目，他从…某处获得了一个后驱动装置，并在一些帮助下将其放在家中的长凳上。

他的第一步是把它接上电源，开始嗅探 CAN 总线的命令。他花了几个小时，但他能让马达转动起来。他一直在努力，直到掌握了所有的命令。因此，他将循环水连接到该装置上进行冷却，并对其进行测试(该装置一度宣布其速度为每小时 117 英里)。

最后，他能够让所有的功能正常工作，包括生成！他甚至制作了自己的控制板。光是听着马达旋转就很满足了。休息后的视频。
