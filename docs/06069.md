# Hackaday 奖参赛作品:难民团聚者

> 原文：<https://hackaday.com/2017/06/02/hackaday-prize-entry-refugee-reuniter/>

世界正在应对一场严重的难民危机，随之而来的是一个问题:找人。今年 Hackaday 奖的一个项目“难民团聚者”可能会解决这个问题。这是一种允许人们与家人重新联系的设备，无论是在前往目的地国家途中丢失的孩子，还是父母团聚。

难民团聚者试图解决的基本问题是跟踪人。这是一个完整的蜡球，涉及隐私和技术问题。目前提出的想法包括 GPS 追踪器、植入式 RFID 标签以及其他类似的严厉措施。难民团聚者对此做了另一种解释，同时仍然给每个名字分配一个唯一的电子 ID。

难民团聚者的基本硬件只是一个 RFID 腕带或令牌，难民一直随身携带。这个令牌被映射到一个名称，这个名称可以在一个小型终端中被查找到，并被绑定到一个特定的位置。如果难民登录其中一个终端，他们的位置就会被记录下来，他们就可以寻找他们的亲属。这是一个简单的技术解决方案，基本上是一个巨大的死树日志，只备份到一个在线数据库。
