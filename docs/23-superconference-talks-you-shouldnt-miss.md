# 你不应该错过的 23 个超级会议

> 原文：<https://hackaday.com/2015/12/24/23-superconference-talks-you-shouldnt-miss/>

11 月标志着我们的首次[黑客日超级会议](https://hackaday.io/superconference/)，这是我们很久以来一直想做的事情。Hackaday 已经有了一个庞大而活跃的在线社区，但直到现在，我们还没有邀请人们一起参加一个跨越整个周末的硬件会议。超级电脑是黑客时代的化身，数百名非常酷的人出现在几十个讲座、令人惊叹的研讨会以及更多的活动中。

在过去的一个月里，我们一直在整理第一届 Hackaday 超级大会上发生的所有事情。这包括所有会谈的视频，相关的旁白，以及在为期两天的会议中发生的所有事情的帖子。即使你不能参加我们的第一次骗局，这份伟大的材料也应该被所有人分享。

以下是所有演讲的 YouTube 播放列表。如果你想在假期周末消磨八个小时，好吧，现在你有了。休息之后是完整的会议，按日期和发言人进行索引，有演讲的链接和附带的 Hackaday 帖子。

我们要感谢每一个来到第一届 Hackaday Supercon 的人，特别要感谢演讲者、研讨会组织者和志愿者。没有 Hackaday 社区的全力支持，这是不可能发生的。这很好，因为我们明年还会这样做。
