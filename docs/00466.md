# 这只电子手是如此的金属

> 原文：<https://hackaday.com/2015/10/25/this-animatronic-hand-is-so-metal/>

根据他的 Instructables 简介，[bwebby]想在特效行业做一些很酷的东西。根据他制作的电子手，我们认为他有很大的机会。

指状段由铜管制成。它们通过微小的铰链和强力胶相互连接，并连接到金属片手掌上。手指部分里面的东西是环氧树脂油灰。它保持由自行车传动装置电缆制成的肌腱末端牢固地附着在指尖部分，并提供通过手指其余部分的通道。这些电缆穿过 50 毫米铝管，铝管安装在金属薄板前臂上，它们连接到安装在一块 MDF 上的高扭矩伺服系统。[bwebby]使用一个 [Pololu Mini Maestro](https://www.pololu.com/product/1352) 来控制伺服系统，该系统使用主板的原生 USB 接口和控制软件。

观看[bwebby]运行一些动作，并在休息后尝试抓地力。如果你想制造一只电子手，但还没有准备好，你可以从更接近木偶的方法开始。
