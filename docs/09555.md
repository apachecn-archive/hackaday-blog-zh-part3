# 赫塔·艾尔顿如何让泰坦尼克号发出求救信号

> 原文：<https://hackaday.com/2018/05/24/how-hertha-aryton-enabled-the-titanic-to-call-sos/>

[Kathy]最近发布了一个[有趣的视频](https://www.youtube.com/watch?v=hrmx_TKNvyc)，讲述了一位名叫[Hertha Ayrton]的电子先驱与电弧发射器的联系。故事从对弧光灯的观察开始——我们知道这是弧光灯的一个错别字。

[赫塔]出身贫寒，但是——当时很奇怪——接受了科学教育。这本身可能就是一个完整的故事。在她上学期间，她爱上了她的教授(威廉·艾尔顿)并结婚了。

[赫塔]接手了她丈夫对弧的研究，并在理解弧的物理方面取得了令人印象深刻的成果。关键的发现是，在某些情况下，提高电压会降低电流。这意味着电弧具有负电阻。

关于[威廉的]一个研究生和一个叫[鲍尔森]的人的故事还有很多，他使电弧发射器更加可靠。我们将让您观看视频以了解更多信息。

泰坦尼克号上杰克·菲利普斯的故事结局是众所周知的。但是我们不知道 arc 发射器是如何出现的所有背景，也不知道改进的[Poulsen]发射器可能会避免这场灾难。

我们的老朋友[【雷金纳德·费森登】](https://hackaday.com/2018/02/19/you-all-know-reginald-fessenden-who/)在视频的结尾意外出现。小心他。如果你认为这些旧收音机[没有受到黑客攻击](https://hackaday.com/2017/03/02/great-hacks-of-history-the-marconi-radio-hack-1903/)，请再想想。
