# 更文明时代的机械光剑笔

> 原文:[https://hack aday . com/2017/12/18/machined-lightsaber-pen-for-a-more-civilized-age/](https://hackaday.com/2017/12/18/machined-lightsaber-pen-for-a-more-civilized-age/)

在一个日益无纸化的社会中，书写工具对我们许多人来说正在成为一种过时的技术。当然不是那种一般人会想花几块钱买的东西，更不用说用一块实心的铝棒加工出一个了。但显然[鲍勃]不是大多数人。他最近给我们留下了一行关于他上传到他的名副其实的 YouTube 频道“制作材料”的视频，[在那里，他经历了将原材料变成值得绝地武士使用的书写工具所需的步骤](https://www.youtube.com/watch?v=efWOj1FXnNk)。

从一片被扔进车床的铝片开始，[鲍勃]切割出卢克军刀标志性的棱纹轮廓，并在缝隙中填充了一只黑色的锋利无比的东西。然后，他开始制作形状更复杂的发射器，然后在车床上翻转手柄，把它挖空，这样就可以插入一个铜管。

[![](../Images/8e6bfc1e1925c124bc736f79f4c288a1.png)](https://hackaday.com/wp-content/uploads/2017/12/saberpen_detail.jpg)

Turning down the acrylic “blade”

有点令人惊讶的是，似乎更多的努力最终被投入到丙烯酸“刀片”而不是铝手柄本身。一大块丙烯酸被钻孔和攻丝，这样它就可以安装在卡盘上，然后变成一个长圆柱体。然后在末端切一个尖，刀片的长度被挖空，最后它被抛光到一个漂亮的光泽。

通过将一个标准的墨水笔盒插入现在已经完成的军刀的中心来完成构建。当然，钢笔爱好者会感叹，他没有试图建立自己的墨盒，但我们认为他得到了通行证，考虑到它的其余部分是从零开始。

如果像这样一个辉煌的书写工具还不足以让你重新学习如何写你的名字，不用担心。无论你是[制作音乐](https://hackaday.com/2017/01/22/a-violin-bow-lightsaber/)还是[捕捉旗帜](https://hackaday.com/2015/12/15/capture-the-flag-with-lightsabers/)，我们已经在许多其他 saber 项目中扮演过角色；[眼睛安全与否](https://hackaday.com/2013/05/24/3w-handheld-laser-raises-hope-for-a-real-lightsaber-someday/)。
