# 廉价 DIY FPV 微型无人机

> 原文:[https://hackaday.com/2017/01/31/cheap-diy-fpv-micro-drone/](https://hackaday.com/2017/01/31/cheap-diy-fpv-micro-drone/)

对于初学者来说，FPV 无人机是一个有趣但往往昂贵的爱好。选择更小的无人机将减少无人机在不可避免地坠毁时损坏的机会，如果有无辜的旁观者，更小的道具也更安全。YouTuber 和 Instructables 用户建造了想要一架廉价的 FPV 无人机，他们可以舒适地在室内和室外飞行，所以当然[他们建造了自己的](http://www.instructables.com/id/Cheap-and-Easy-Micro-Quadcopter/?ALLSTEPS)。

一旦无人机的框架被 3D 打印出来，将四个小而强大的 8.5 毫米电机焊接到 Micro Scisky 控制板最复杂的部分是确保你以正确的配置连接它们，并对它们进行三重检查。电池连接的快速重组和 FPV 相机的安装几乎完成了硬件部分的构建。

在将你的飞行控制器插入电脑进行编程之前，[construsted]警告说，除非你想烧你的电路板，否则必须断开电池。否则，刷新电路板和编程只需要耐心和大量的工作。一旦完成，你把所有的东西都配对好了，天空——或者天花板——就是极限了！
