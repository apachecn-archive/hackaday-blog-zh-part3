# 电缆切割机可以快速完成一项繁琐的工作

> 原文:[https://hacka day . com/2018/01/14/cable-cutting-machine-makes-fast-work-of-a-diverous-job/](https://hackaday.com/2018/01/14/cable-cutting-machine-makes-fast-work-of-a-tedious-job/)

我们都有过这样的经历:面对一项乏味的工作，只需投入少量的时间就可以手工完成，我们反而选择让任务变得过于复杂，并构建一些东西来为我们做这件事。这就是自动钢丝钳背后的动力，但在这种情况下，目的证明了手段的合理性。

爱德华·卡尔森(Edward Carlson)设法用钢丝钳和卷尺将 20 分钟的会议延长到四天来建造和调整这台机器，这令人印象深刻。构建过程是通过修改现成的电线测量机启动的，这种机器可以在大桔子商店的电气通道中找到。去掉了原来的机械累加器，增加了一个步进器来驱动摩擦轮，该机器现在可以通过计算步数来测量电缆。一个高扭矩伺服系统通过一个漂亮的连杆驱动一把结实的电缆剪刀，或者这台机器可以只测量电缆的长度而不切割。[Edward]寻找问题的解决方案最终带来了额外的价值，所以也许花的时间终究是值得的。

如果整体设计看起来很熟悉，你可能会想到类似于我们不久前推出的另一个电缆切割机器人。那台机器使用的是细丝挤压机，用于比这台机器更细的电线。
