# 魔术 8 的心 ply 你的真爱与俗气的说法

> 原文:[https://hackaday . com/2016/02/14/magic-8-of-hearts-plys-your-true-love with-cheese-saughts/](https://hackaday.com/2016/02/14/magic-8-of-hearts-plies-your-true-love-with-cheesy-sayings/)

正好赶上情人节，路易斯维尔 LVL1 hackerspace 的一个项目应该会温暖你生命中那个特别的人的心。请看红心魔术 8。

这里的隐喻有些混杂，心形盒子、[一个神奇的 8 号球](https://en.wikipedia.org/wiki/Magic_8-Ball)的神秘黑色窗户，以及类似印在[甜心糖果](http://www.necco.com/Candy/Sweethearts/History.aspx)上的一年一度的俗气谚语。[JAC_101]开始手术，在塑料心脏上打了一个洞，用于有机发光二极管的展示。黑底白字的显示屏让人联想到神奇的 8 球外观，尽管添加蓝色滤镜可能会让它看起来更好。一个三轴加速度计检测震动，一个 Arduino Nano 选择显示一条消息。一些白色发光二极管照亮了外壳，增加了一点活力。作为奖励，整个东西都是感应充电的——心脏不需要额外的孔。

如果你的真爱喜欢更华丽的东西，试试这个 LED 情人节心形动画。如果你在浪漫的努力中取得了成功，你可能会发现自己正在制作[这些超级极客的婚礼请柬](http://hackaday.com/2013/01/30/really-really-geeky-wedding-invitations/)。
