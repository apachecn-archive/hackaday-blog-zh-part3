# 让我们回到机器人时代

> 原文：<https://hackaday.com/2018/03/28/lets-bring-back-the-age-of-automatons/>

远在我们今天所知的人工智能概念出现之前，人类就已经开始制造机器，这些机器似乎可以按照自己的意愿移动甚至思考。几十年来，我们一直在制造自动机，即自我操作的机器，旨在模仿人类和动物。导致设计者将人类和动物的运动、行为、甚至言语(通过风箱和气管)分解成预定的顺序动作。

格雷格·朱姆沃尔特(Greg Zumwalt)创造了他所谓的[蜂鸟主题机器人](https://www.instructables.com/id/Hummingbird/)，灵感来自于他妻子喜欢看蜂鸟聚集在他们家附近。他的 3D 打印和组装蜂鸟机器人几乎像它的有机对应物一样流动。这个设计很简单，但却是由令人印象深刻的 97 个印刷零件组成，这些零件是由他在他的说明书中包括的 38 个独特设计印刷而成的。除了精心的装配设计，流体运动还适用于所有印刷零件的试装、修整和打磨过程。再加上加入凡士林作为建筑物活动部件的润滑剂。 除了打印文件，[【Greg Zumwalt】还为您提供了重新创建这种精确造型所需的打印设置，以及一份零件清单，其中列出了每种设计所需的所有多份打印件。

[格雷格]最近运气不错。看看他的[气动引擎](https://hackaday.com/2017/12/27/3d-printed-airplane-engine-runs-on-air/)，或者可能是[有史以来最简单的 3D 打印机器人](https://hackaday.com/2018/01/02/remember-when-scratch-built-robots-were-hard/)。

为了让你的 3D 打印机发挥更好的作用，看看这个 3D 打印的[水滴模拟机器人。](https://hackaday.com/2017/09/21/see-this-mesmerizing-3d-printed-water-droplet-automaton/)如果你想了解更多关于自动机的历史，请到这里来，我们写了一篇关于一个著名的[自动机的文章，这个自动机被称为“绘图员-作家”](https://hackaday.com/2016/12/15/fran-blanche-goes-in-depth-with-the-maillardet-automaton/)，由瑞士机械工和钟表匠 Henri Maillardet 创造。
