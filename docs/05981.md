# 所有东西都是带电子设备的触摸表面

> 原文：<https://hackaday.com/2017/05/18/everythings-a-touch-surface-with-electrick/>

触摸屏很棒，但是大触摸屏很贵，不规则触摸屏根本不好做。 [Electrik](http://yang-zhang.me/research/Electrick/Electrick.html) 是由卡内基梅隆大学的几名研究人员开发的一种方法，使用层析成像技术将几乎任何固体物体都变成触摸表面。问题是导电涂层——以导电片、3D 塑料或油漆的形式——是必要的。您可以在下面的视频中看到演示和许多独特的应用。他们甚至用果冻做了一个触摸感应大脑，用培乐多做了一个可触摸的雪人。

概念很简单。表面周围有多个电极。该系统使用一对电极注入电流，然后检测另一端的输出。手指触摸会改变几个电极的输出。一旦检测到，系统将改变注射电极并重复检测。通过使用多个电极对和断层扫描技术，该系统可以确定触摸的位置，甚至可以像低分辨率触摸板鼠标一样进行粗略的运动跟踪。

除了视频，还有一个演讲视频和一篇论文。有足够多的细节，您应该能够自己试验这种技术。原型使用了一个带有蓝牙的小板子，让物体可以无线运行。激励信号是驱动电压控制电流源的 200 kHz 正弦波，提供恒定的交流电流。多路复用器允许使用多个电极，这是获得合理分辨率的关键。至少，这在某种程度上是真实的。研究人员发现，在一定数量的电极对之后，对于特定大小的物体，准确性实际上变得更差。

一些更有趣的用例包括触摸感应墙、带热点的电脑桌和触摸方向盘。该系统将通过普通的油漆甚至贴纸进行操作。
