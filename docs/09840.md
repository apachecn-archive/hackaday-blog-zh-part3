# 在树莓皮的帮助下第一次拍摄

> 原文:[https://hack aday . com/2018/07/05/shooting-for-first-time-help-from-a-raspberry-pi/](https://hackaday.com/2018/07/05/shooting-for-the-first-time-with-help-from-a-raspberry-pi/)

像许多人一样，[迈克]有一个他一生中想做的事情的清单。其中一个是“用开关开枪”，在一些黑客朋友的帮助下，他在上周末完成了这个项目。

对于那些想知道为什么这个项目的特殊性，背景故事将有助于解释。[迈克]患有脊髓性肌肉萎缩症，这种疾病本应在发病后不久就结束他的生命。37 年后，[迈克]仍然在他的清单上勾项目，但由于他只能随意控制他的右眉毛，他面临着完成其中一些项目的挑战。进入[比尔]和船员在 ATMakers。“AT”代表“辅助技术”，[Bill]承担了建造一个钻机的任务，在[Mike]的命令下安全地发射 Glock 17。

甚至在项目开始之前，[Bill]就做了他的尽职调查，甚至咨询了酒精、烟草和火器局(ATF ),并在当地的室内射击场安排了私人时间。钻机的商业端是一个商业上可用的长凳架，用于控制手枪的后坐力，手枪是由连接到扳机的伺服系统发射的。与[Mike]系统的接口是通过一个定制的 PCB 连接在一起的 Raspberry Pi 和 Crikit。一个 PiCam 允许[迈克]向下看瞄准镜，用他的眉毛开枪。下面的视频展示了开发过程和靶场的一天；说[迈克]很高兴是一种保守的说法。

我们不确定[迈克]的清单上还有什么，但我们在这里看到了很多辅助技术项目——我们甚至有[一整类 2017 年黑客日奖](https://hackaday.com/2017/09/11/these-twenty-assistive-technologies-projects-won-1000-in-the-hackaday-prize/)专门用于它们。也许 Hackaday 社区还能帮他查些别的。
