# 星际迷航 Pi

> 原文：<https://hackaday.com/2016/03/01/star-trek-pi/>

每当我们大喊“好的，谷歌……导航到天鹅绒梅尔文的”时，我们感觉自己就像是《星际迷航》中的角色。毕竟，你从未见过柯克船长(或皮卡德)使用键盘。如果你有同样的感觉，并且你脑子里有一个 Raspberry Pi 项目，你可能会喜欢这个 [Raspberry Pi LCARS 接口](https://github.com/tobykurien/rpi_lcars)。

你可以在下面的视频中看到结果。界面使用 [PyGame，](http://www.pygame.org/hifi.html)如果你不想要星际迷航的外观，你可以用不同的皮肤定制它。

显然，你需要一个触摸屏来使系统有效工作。该库有一些配置变量，允许您通过触摸来移动小部件或显示光标进行调试。

如果你渴望更多的《星际迷航》项目，我们的科幻竞赛中有相当多的参赛作品。如果这些对你来说都不够硬核，你可能会喜欢作为潜艇的企业(这比*星际旅行:进入黑暗*的开场场景早了不少)。
