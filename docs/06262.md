# 开源数字电影

> 原文:[https://hackaday.com/2017/06/17/open-source-digital-cinema/](https://hackaday.com/2017/06/17/open-source-digital-cinema/)

经过多年的努力， [Apertus](https://www.apertus.org/history) 已经为 AXIOM 发布了 25 个测试版开发者套件——他们的开源数字电影摄像机。这不是你的傻瓜数码相机。2013 年的原始概念验证有一个 Zynq 处理器(Zedboard)，一个 super 35 4K 图像传感器和一个尼康 F-Mount。

今天的设备是模块化的，有几种选择。例如，有一个 HDMI 输出模块，但显示端口，4K HDMI 和 USB 3.0 选项正在开发中。下面你可以看到几个用该设备拍摄的视频样本。

目前的设备使用 Cmosis CMV12000 图像模块，尽管柯达 KAC12040 和 Cmosis CMV2000 模块正在开发中。你可以在网上找到[的完整规格](https://www.apertus.org/axiom-beta-tech-specs)，但是请注意其中一些规格是前瞻性的。另一方面，视频清楚地显示它处于工作状态。

如果你一直在寻找开源 FPGA 或微控制器项目，这可能是它。像许多开源公司一样，他们试图商业化以保持项目健康，但是如果你阅读他们的网站，他们似乎致力于开放原则。

我们已经看到[天体物理界](https://hackaday.com/2017/01/07/custom-zynqcmos-camera-unlocks-astrophotography/)在做类似的事情——也许他们应该合作。如果你想复习一下，我们还看了图像传感器中的[技术。](https://hackaday.com/2017/02/07/the-megapixel-race-and-its-clear-winner/)
