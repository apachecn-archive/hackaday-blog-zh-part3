# 火腿为微型发射器疯狂

> 原文:[https://hack aday . com/2017/02/24/ham-goes-nuts-for-tiny-transmitter/](https://hackaday.com/2017/02/24/ham-goes-nuts-for-tiny-transmitter/)

一个正常工作的业余无线电发射机的最小 BOM 是多少？看起来你可以带走七个部分，或者如果你包括核桃的话是八个。你必须有一个核桃。

一些火腿真的很喜欢 QRP 的挑战，或者故意使用低功率发射机来提供长途联系的挑战。我们之前已经报道过 QRP 的世界，并注意到虽然 QRP 的钻机并不提供很多动力，但这并不意味着它们需要简单。有些变得相当复杂，支持许多不同的调制方案，甚至数字模式。由于只有一个 2N3904 晶体管，[Jarno (PA3DMI)]的微型发射器只能使用 CW 调制发送莫尔斯信号，但鉴于它是在核桃壳内进行的，我们没有什么可抱怨的。外壳的两半铰接在一起，为简单的石英晶体振荡器容纳一小块 perfboard。原型在外壳外进行了调整，9 伏电池显然是外置的，但除此之外，它什么也不是，只是坚果。

我们希望看到[雅诺]在两半壳的铰链和触点上加一个弹簧，这样就不需要按键了。谁知道呢？响板风格的键控可能会在此后的火腿中风靡一时。
