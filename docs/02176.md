# 对着手套说话

> 原文:[https://hackaday.com/2016/05/05/talk-to-the-glove/](https://hackaday.com/2016/05/05/talk-to-the-glove/)

两名华盛顿大学的学生在创客空间发挥了他们的创造力，创造了一副手套，为他们赢得了 1 万美元的奖金。显然，它们不仅仅是普通的手套。这些手套可以感知美国手语，并将其转换成语音。

这种手套可以感知手部动作，并通过蓝牙将数据发送到外部电脑。与其他手语翻译系统不同，这种手套方便携带。你可以在下面看到手套的视频。

这副手套赢得了莱梅尔森-麻省理工学院学生奖。计算机使用类似神经网络的算法来检查手势数据。当它找到一个匹配时，相关的单词或短语通过扬声器说出。我们希望能分享更多关于内部运作的细节，但是我们可能还没有看到这两个人的最后一面。

我们在过去实际上已经看到过[几个类似的](http://hackaday.com/2014/12/20/electronic-glove-detects-sign-language/)手套[。我们还看到传感器手套被用于其他事情，如](http://hackaday.com/2010/10/04/from-sign-language-to-spoken-language/)[制作音乐](http://hackaday.com/2014/04/17/interactive-gloves-turn-gestures-into-music/)。
