# 3D 打印线性致动器的发展

> 原文：<https://hackaday.com/2018/06/22/evolving-the-3d-printed-linear-actuator/>

我们的开源社区邀请任何有想法的人在前人工作的基础上再接再厉。我们中的许多人都遇到过控制线性运动的需要，并为此采用了一种廉价的业余伺服系统。[Michael Graham]评估了现有的设计，并相信他有推动艺术发展的想法。我们的 Hackaday 奖评委同意了，将他的 [3D 打印伺服线性致动器](https://hackaday.io/project/158014-3d-printed-servo-linear-actuator)作为我们机器人模块挑战赛的二十名获奖者之一。

[Michael]的致动器跟随了基于齿轮齿条式齿轮的其他设计的脚步，例如这些页面上的[这个](https://hackaday.com/2016/03/31/3d-printed-case-turns-servo-into-quality-linear-actuator/)，但是他从机械工程师的角度处理设计问题。该设计结合了几个兼容的功能，以容忍 3D 打印机(和切片器，和细丝等)之间的差异。)提高成功印刷的几率，从而提高成功项目的几率。初学 3D 打印设计的人(甚至一些老手)会发现他的[设计技巧文档](https://engineerdog.com/2018/05/08/design-for-assembly-tips-building-a-better-3d-printed-linear-actuator/)非常值得花几分钟时间阅读。

[![](img/68efe89e68bd6a7d9716f7ee4e89d109.png)](https://hackaday.com/wp-content/uploads/2018/07/engineerdogmugomatic1.jpg) 他的致动器设计的另一个有用的特点是 20 毫米 x 20 毫米的螺丝安装系统。在输出滑块的任何一端都可以看到，它允许混合和匹配一组用螺栓固定在该执行器上的附件。他已经开始沿着这条路走下去，并面临着[的挑战:有*太多的东西*要分享](https://hackaday.io/project/158014/log/146582-models-complete-testing-in-progress-challenges-encountered)，同时保持它们都被组织起来并可供每个人使用。

灵活的构造系统允许他在模块化系统中实现不同的想法。他带了一件东西(他的 Mug-O-Matic 的变体)到湾区创客大会的 Hackaday + Tindie Meetup，我们确信会有更多。鉴于他的项目的深思熟虑的设计和广泛的文档，我们希望看到他的线性伺服系统被其他人采用，并出现在其他环境中。

这不是我们遇到的唯一的线性致动器。它甚至不是我们机器人模块挑战赛中唯一获胜的线性致动器，但另一个专注于满足不同的约束，如紧凑性。它们是满足不同需求的不同工具，是我们机械解决方案工具箱中的宝贵补充。
