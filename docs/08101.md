# 星期五黑客聊天:KiCad 的状态

> 原文:[https://hack aday . com/2018/01/03/Friday-hack-chat-the-state-of-kicad/](https://hackaday.com/2018/01/03/friday-hack-chat-the-state-of-kicad/)

像大多数 PCB 设计软件一样，KiCad 已经有 25 年的历史了，现在它是最好的开源工具，可以用来布置电路，放下几个电阻，从头开始创建 PCB。在过去的几年里，很多人转向 KiCad 来设计一些令人印象深刻的电路板，这无疑与 KiCad 在啤酒和语音方面都是免费的这一事实有关。

[![](../Images/0f5e2cbb8ca4d032faf79e24538636ae.png)](https://hackaday.com/wp-content/uploads/2018/01/kicadhackchat-01.png) 本周五加入我们的黑客聊天，[我们谈论的都是 KiCad](https://hackaday.io/event/28713-the-state-of-kicad) 。如果你对开发人员有不满或赞扬，这里是表达的地方。我们本周黑客聊天的嘉宾是 Wayne Stambaugh，KiCad 的项目负责人。其中，Wayne 负责领导 KiCad 产品路线图，他也是 [CvPcb 参考手册](https://www.amazon.com/gp/product/9888381881)的作者之一

在这次聊天中，我们将讨论大家最喜欢的开源 EDA 套件的当前和未来特性。这是一个提出建议和愿望清单的好机会。想知道 KiCad 的发音是‘Kai-CAD’还是‘Key-CAD’？这是后者，但不要让这阻止你要求韦恩改变这一点。

供讨论的项目包括:

*   2018 年路线图中的新功能
*   自从上次 KiCad Hack Chat 以来，KiCad [中发生了什么](https://hackaday.io/event/19417-kicad-hackchat/log/52201-edited-transcript-of-the-kicad-chat)
*   引擎盖下发生了什么，为什么你永远不应该相信自动路由器？
*   当你刚开始在 KiCad 的时候，你在哪里转弯？
