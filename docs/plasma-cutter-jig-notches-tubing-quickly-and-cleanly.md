# 等离子切割机夹具快速干净地切割管道

> 原文：<https://hackaday.com/2017/02/23/plasma-cutter-jig-notches-tubing-quickly-and-cleanly/>

这可能是[MakeItExtreme]迄今为止最雄心勃勃的建筑。这个等离子切割机管道切槽机有许多移动部件，但它应该是车间的一个很好的补充，并开辟了许多制造可能性。

我们不得不承认第一次看下面的视频时会有一些困惑。我们通常可以从被切割的第一批原料中看出[MakeItExtreme]的构建进展如何，但带有压入轴承的大管让我们挠头。这个计划很快就变得清晰了——一个装有中空套管的电动水平旋转台，用于等离子炬引线。有一个夹具来固定焊枪，它可以相对于工作台移动。由管段制成的凸轮可以用螺栓固定在固定压板上；一个凸轮从动件骑在凸轮上，随着工作台的旋转将焊炬移进移出。这就形成了将管子正确装配在一起所需的切口，称为鱼嘴切口或鞍形切口。凸轮可被移除用于直切，并且定制的管虎钳可被调整以进行斜切。

总而言之，这是一个坚固而多功能的建筑，应该能够支持大量的新项目，特别是当与[【make it Extreme】最近的滚弯机](http://hackaday.com/2017/01/31/diy-roll-bender-keeps-it-simple-and-sturdy/)合作时。
