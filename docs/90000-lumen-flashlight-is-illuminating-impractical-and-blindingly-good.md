# 90，000 流明手电筒是照明，不切实际和盲目的好

> 原文：<https://hackaday.com/2015/11/27/90000-lumen-flashlight-is-illuminating-impractical-and-blindingly-good/>

也许点一根蜡烛比诅咒黑暗要好，但那是在【RCTestflight】想出这个之前:[一个 1000W 的 LED 手电筒](https://www.youtube.com/watch?v=-JVqRy0sWWY)输出大约 90000 流明的光。很多:最好的袖珍 LED 手电筒输出约 700 流明。

[RCTestflight]用 10 个 100 瓦的发光二极管建造了这个庞然大物，用两节遥控汽车电池供电。每个 led 都连接到一个相当大的电压转换器和一个非常大的散热器，将所有 led 固定在适当的位置。他说，他用这个东西可以获得大约 8 分钟的光照，使用一两分钟后散热器就会变热。我们并不感到惊讶:在将电能转化为光能方面，led 比大多数其他设备更有效，但有些总是以热量的形式损失掉。

休息之后请看视频。令人印象深刻，但这东西作为手持设备并不特别实用。它又大又重，几英里外都能看见。如果你真的想照亮什么东西，它会做得很好(在短时间内)，部分原因是每个 led 都包含一个玻璃透镜。这有效地将光束聚焦在适当分布的区域。我们想知道如果所有的光束都聚焦在一点上会发生什么？只要你不越过小溪…

我们已经报道了一些使用类似 led 的更实际的构建，但这东西确实有某种令人难以置信的魅力，并且可能对高速视频有用，因为光越多越好。
