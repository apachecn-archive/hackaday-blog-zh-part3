# 小型低成本数控机床

> 原文：<https://hackaday.com/2017/02/16/pint-sized-low-cost-cnc-machine/>

一点密度板、一点胶合板、一些螺纹杆——把它们放在一起，你就有了[这个低成本的台式电脑数控制造](http://thimov.com/projects/desktop-cnc/),使用很少的零件，你需要去比当地家庭中心更远的地方采购。

我们以前在这里见过很多电子垃圾和[垃圾箱](http://hackaday.com/2015/11/21/garbage-can-cnc-machine-build/) CNC 建筑；这里吸引人的不仅是该建筑的低廉价格，还有它的平易近人。正如下面的短片所示，[Thimo Voorwinden]在使用手头的工具和材料方面做得非常出色。我们也很欣赏这种结构的模块化——X 轴和 Y 轴托架几乎完全相同，可以互换以改变工作区域的尺寸，如果需要，甚至可以更换为更大的托架。Z 轴与通常的低端 CNC 构建略有不同，因为它不使用 Dremel 或其他小型旋转工具，而是安装软轴旋转工具的机头。让马达离开机器可以获得更大的扭矩、更小的振动和更小的静负载。

最终的结果是一台大约€200 的台式计算机数控系统，其工作区域大到足以制造小型木制和塑料零件，或者铣削泡沫块用作铸模。看起来[Thimo]为他的小型数控机床准备了更多，我们期待看到他能做出什么改进。
