# 单马达让这个机器人做蠕虫

> 原文：<https://hackaday.com/2016/08/07/single-motor-lets-this-robot-do-the-worm/>

随着自主机器人领域越来越多的研究，新的移动方法正在快速出现。忘记轮子和履带，忘记双足、四足、六足和八足，忘记花式滚动的 BB-8 克隆体。如果你想让一个迷你机器人动起来，也许你应该[教它做蠕虫](http://gizmodo.com/this-crawling-climbing-worm-bot-is-a-never-ending-nigh-1784306984)。

Gizmodo 的文章和[David Zarrouk]论文的摘要都没有给出这个蠕虫状机器人构造的太多细节，但从下面的视频中可以收集到一些线索。在 1:41 标记处，我们看到了设计的秘密——3D 打印连杆中心的一个长螺丝锥。


由单个电机驱动，连杆将螺旋运动转化为波动，推动机器人前进或后退。转向由前方的非驱动舵轮提供；我们可以看到这种机器人的双螺旋版本，使用平行轨道来提供方向控制。[Zarrouk] *et al* 也在其他方面修改了基本设计——一个防水版本像鳗鱼一样游泳，一个微型机器人只有钢笔大小。后者有进一步小型化的潜力，目的是建造用于治疗应用的[自主体内机器人](http://hackaday.com/2015/11/30/swallow-the-doctor-the-present-and-future-of-robots-inside-us/)。很棒的东西。
