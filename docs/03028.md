# Hackaday 奖参赛作品:反射转换成像

> 原文:[https://hack aday . com/2016/07/29/hack aday-prize-entry-reflectance-transformation-imaging/](https://hackaday.com/2016/07/29/hackaday-prize-entry-reflectance-transformation-imaging/)

反射变换成像(RTI)或多项式纹理映射是一种非常有趣的成像技术，它允许您捕捉一个对象的所有细节。在考古学中，它被用来拍摄洞穴墙壁上涂鸦的精细照片，为硬币收藏家捕捉硬币的每个细节，并测量艺术作品中非常细微的变化。

RTI 通过以非常特殊的角度照射物体，然后使用图像处理产生最佳图像来实现这一点。尽管只有几个发光二极管和一点软件，RTI 系统贵得离谱。为了他的 Hackaday 奖参赛作品，[leszekmp] [正在建立他自己的 RTI 系统](https://hackaday.io/project/11951-affordable-reflectance-transformation-imaging-dome)。这将花费大约 600 美元，使这成为公民科学家捕捉最佳图像的最佳方式。

RTI 只是将光线照射到一个物体上，并从正上方同步拍摄该物体的照片。正如你所想象的，将 led 放在圆顶中是解决这个问题的显而易见的方法，而且[leszekmp]已经制作了三个系统，可以很好地工作在直径达一米的圆顶上。电子设备就像一个 Arduino 屏蔽和几个 MOSFETS 一样简单，圆顶本身就是一个现成的组件。这是一个伟大的项目，能够实现更好的摄影，也是我们见过的最简单和最好的 Hackaday 奖参赛作品之一。
