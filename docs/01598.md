# 这个沙漏会自己翻转

> 原文：<https://hackaday.com/2016/02/27/this-hourglass-flips-itself/>

从前，[迈克]给他妹妹买了一个沙漏。他打算把它做成一个钟，作为礼物送给她，但是生活和其他项目妨碍了他。时间快进到几十年前，当所有这些都汇集在一起时，[迈克]手头上有了他需要的一切来建造一个漂亮的木制时钟，它能自动翻转沙漏。

每 60 分钟，位于手工枫树环内的灯泡旋转 180 度，重新启动沙子的流动。外轮顶部的数字表示当前小时。下一个小时的数字始终位于相对于当前小时的五点钟位置。这是因为灯泡环外侧的凹槽与外环上的轮齿的比率为 5:6。迷茫？请观看[Mike]的延时视频，该视频展示了它的实际应用。

[迈克]决定只用他手头已经有的东西来造这个钟，比如用来计时的廉价电子表和转动沙漏的车窗电机。他在手表上黑了一个 USB 端口，这样他就可以使用每小时报时功能，通过四路运算放大器来触发马达。马达运行，直到它被触发光学关闭——齿轮上切割的一对狭缝，移动沙漏经过传感器。[Mike]做了一个漂亮的盒子来装一块好核桃的内脏，在设计上不遗余力。

在项目网站上有大量的构建图片和一个深入的视频游览，在休息后嵌入。不管它们是被设计来让 T1 惊奇还是让 T3 困惑，我们都喜欢这里建造的好时钟。如果你喜欢沙漏，我们不久前推出了数字版的。
