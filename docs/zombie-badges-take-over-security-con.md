# 僵尸徽章接管安全骗局

> 原文：<https://hackaday.com/2017/05/22/zombie-badges-take-over-security-con/>

我们永远也看不够黑客徽章。去年 12 月在开普敦举行的 BSides Cape Town，展示了一个配备 IR 的徽章,让与会者在聊天时沉浸在游戏中。

由[ [AndrewMohawk](https://twitter.com/AndrewMohawk) ]和[ [ElasticNinja](https://twitter.com/ElasticNinja) ]领导的一个小组围绕一个 ESP8266 和 128×64 有机发光二极管显示器设计了这个徽章，有八个按钮，一个红外接收器和发射器，五个“水平”led，一个 RGB LED 和一个通过 USB 充电的 600 mAh LiPo。

硬件是专门为玩一个有机游戏而设计的，以便组织者可以实时观看徽章之间的互动。每个徽章都被随机分为一个阵营，红色、蓝色或绿色——可通过徽章上发光的 RGB LED 识别。还有一系列的五个发光二极管表示你在游戏中的水平。当两个或两个以上的徽章彼此接近，足以让 IR 链接时，等级最低的徽章被转换为胜利者的阵营。

当然，徽章上显示了与会者的把手，还包含了一份会议日程安排表。它还向与会者提出了一系列挑战，可以解锁来玩 Pong 或石头/剪刀/布/蜥蜴/斯波克，扫描无线网络，并运行动画。
