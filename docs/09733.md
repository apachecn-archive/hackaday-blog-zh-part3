# 懒惰黑客的自动太阳镜

> 原文:[https://hacka day . com/2018/06/13/automatic-太阳镜给懒惰的黑客/](https://hackaday.com/2018/06/13/automatic-sunglasses-for-the-lazy-hacker/)

[安德烈亚斯]可能已经创造了终极懒惰黑客配件:[自动太阳镜，或“自混合太阳镜”，正如他创造性地为他的视频命名的](https://www.youtube.com/watch?v=Vu_9OK-ql40)。如果从名字上看不出来，这些就是永远不用摘下来的眼镜。如果光线很暗，它们就会远离你的眼睛。回到外面明亮的灯光下？眼镜会移动以保护你的眼睛。

这种眼镜由几个微型伺服系统组成，它们将有色镜片移向或移离用户的眼睛。侧面安装的 Arduino Uno 读取 CdS 电池光传感器并驱动伺服系统。为什么是 Uno 而不是更易穿戴的 Arduino Nano？这就是安德烈亚斯所拥有的。

是的，这个建筑的很大一部分乐趣是喜剧。但最精彩的部分是当他在高速公路上的实际汽车中测试眼镜时。这种眼镜的效果比预期的要好——当他开车穿过隧道时，将镜片移入和移出(安德烈亚斯)的视野。你实际上可以看到(安德烈亚斯)是多么惊讶，它工作得如此之好。

这些不是我们见过的第一副自动太阳镜，也不是 T2 最危险的太阳镜。不过，这是一个有趣的项目，视频给了我们一些笑声。
