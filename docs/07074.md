# 中继计算机:你能听到它思考

> 原文:[https://hack aday . com/2017/09/11/relay-computer-you-can-heat-it-think/](https://hackaday.com/2017/09/11/relay-computer-you-can-hear-it-think/)

现代数字计算机有复杂的指令集，运行在最先进的算术逻辑单元上，而算术逻辑单元又是由微型晶体管构成的小型化逻辑门的结果。这些微小的晶体管本质上是开关。你可以想象用电磁继电器代替，得到所谓的继电器计算机。如果你能想象得到，有人已经做到了。在这种情况下，[jhallenworld]。

Z3 是康拉德·楚泽设计的第一台可编程的全自动数字计算机。该板采用现代半导体器件，如内存和微控制器，但是，CPU 是所有继电器。十六进制键盘允许程序输入，分段显示允许跟踪地址和数据。程序通过管道以串行方式传送到并行解码器，然后传送到 CPU，神奇的事情就发生了。由于内核是机电式的，因此可以将输出连接到外设，如视频末尾演示的铃。

这个项目很好地平衡了复古和现代，对任何对机械计算机感兴趣的人都有用，对极客来说应该很有趣。对计算机理论的学生来说，入侵这台计算机来修改指令集应该是同样有益的，也是一个很好的练习。
