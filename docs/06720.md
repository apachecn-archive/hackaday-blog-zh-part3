# Hackaday 奖参赛作品:给盲人的手镯

> 原文：<https://hackaday.com/2017/08/15/hackaday-prize-entry-a-bracelet-for-the-blind/>

世界卫生组织估计，全球约 2.85 亿视力障碍者中，约有 90%生活在低收入环境中，很少或根本无法获得辅助技术。[Tiendo]为他的 Hackaday 奖参赛作品创造了一种简单且易于复制的盲人寻路设备:[一种能够检测附近物体并提醒佩戴者注意它们的手镯](http://hackaday.io/project/25764-bracelet-for-the-blind)。

它使用超声波距离传感器和 Arduino Pro Mini 来完成工作。手环有两种反馈模式:音频和触觉。在音频模式下，当物体在 2.5 米内时，手环会开始发出蜂鸣声。它的行为方式和你预期的一样——越靠近物体，蜂鸣声越大；后退，它就会减少。触觉模式包括两个微型振动圆盘电机，连接到适合拇指和小指的小型 PVC 袖口。这些马达将根据人与给定物体的接近程度发出不同的嗡嗡声。如果一个物体在 1 到 2.5 米之外，小指马达就会振动。再近一点，它就会切换到拇指马达。

为了增加这个项目的节俭程度，[Tiendo]尽可能地重复使用其他物品。手镯的底部是一个聚氯乙烯制成的袖口。破自行车头盔上的尼龙下巴带和塑料扣使它可以调节以适合任何手腕。为了防止 PVC 袖口发炎，他把一双旧袜子上的小碎片塞到边上。

很容易理解为什么这个项目入围了我们的最佳产品竞赛。这是一个简单、低成本的辅助设备，由容易获得和回收的材料制成，任何对电子学稍有了解的人都可以制造它。再加上它很轻，解放了双手，你就有了一个可以帮助很多人的伟大产品。休息后看它哔哔响。
