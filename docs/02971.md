# 二进制键盘是最纯粹的输入设备

> 原文:[https://hack aday . com/2016/07/20/binary-keyboard-is-the-pure-form-of-input-device/](https://hackaday.com/2016/07/20/binary-keyboard-is-the-purest-form-of-input-device/)

你可能是一个铁杆键盘爱好者，你会从冰冷的手中撬出带扣弹簧的开关，但街上有一种新型号，可以将你的空白键 Das 键盘或你可信赖的 IBM 型键盘归入玩具箱。

新的挑战者来自 Reddit 用户【duckythescientist】，他创造了[一个极简的三键二进制键盘](https://www.reddit.com/r/MechanicalKeyboards/comments/4q0eb1/fully_custom_binary_keyboard/)。它有一个 0 键，一个 1 键，一个回车键，除此之外什么都没有。字符以 ASCII 或 Unicode 的形式输入，设备模拟主机 USB 接口的 QWERTY 或 Dvorak 键盘布局。这是学习起来最简单的布局，尽管我们承认不是每个人都能记住整个二进制 Unicode 表。

按键安装在定制的 3D 打印盒中，电子设备来自创作者自己的基于 ATtiny85 的" [tinydev](https://github.com/duckythescientist/BinaryKeyboard/tree/master/tinyDev) "板。所有的代码都可以在 GitHub 库中找到[，在休息时间下面有一个很短的视频展示它的 Unicode 能力。](https://github.com/duckythescientist/BinaryKeyboard)
