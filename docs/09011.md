# 3D 打印坦克履带

> 原文：<https://hackaday.com/2018/03/31/3d-printed-tank-tracks/>

[Ivan]一直让他的 3D 打印机忙于他正在试验的部件，以便为一辆类似坦克的车辆建造一个[履带式运动装置。他的设计使用了几个互锁的部分，所以如果你想复制它，我们希望你的打印机校准符合标准。他仍在打印更多的零件，并承诺一旦设计被证明是正确的，就发布文件。](https://www.youtube.com/watch?v=GwVgRI7A8gY)

然而，你可以看到他开了一个好头。小片装配在一起，接受一根细丝作为一种铰链。一些针阻止灯丝工作。垫适合主要部分，并用拉链带压住。整个柔性胎面锁定在链轮和驱动轮上的凹槽中。

最初的试印是 PLA，但他打算用耐用的尼龙来印最后的试印。通过第一个视频，可以清楚地看到所有的部分都是合适的(在第一部分有一点点刀工),它们像你预期的那样移动。你会在第二个视频中看到它确实可以工作，只是需要很多部件。一大堆零件。

我们想知道[Ivan]是否见过[open wheel](https://hackaday.com/2016/11/28/parametric-3d-printable-wheels-and-treads/)，它包括一个可以用柔性细丝打印的胎面。当然，[创造踏板](https://hackaday.com/2010/10/26/tanks-treads-for-your-next-robot/)的方式也不止一种。
