# Robo Hobo 迷惑路人

> 原文：<https://hackaday.com/2016/07/27/robo-hobo-bamboozles-passers-by/>

机器人越来越多地看到实验室和工厂之外的世界，我们大多数人认为我们能够相对较快地发现一个。如果你在街上走过一个，你会认出它是什么吗？你要多久才会意识到[无家可归的器官研磨者是一个机器人](http://makezine.com/2016/07/21/dirk-a-life-like-homeless-robot-is-an-interesting-social-experiment/)？

无家可归的机器人 Dirk 是[Fred Ables]的创意，他会在人群中漫步，向路人点头致意，偶尔还会用手一挥，索要零钱，用风琴为那些帮助他的人弹奏几个音符。[Ables]从附近远程控制 Dirk 与其他人的互动，融入蜂拥而至观看栩栩如生的机器人的人群中，销售 Dirk 是真实人类的幻觉。这通常很有效，因为和大多数无家可归的人一样，行人不会多看德克一眼；那些没有忽略他的人的反应从困惑到愤怒，或者在寻找木偶师之前被如此彻底地欺骗而感到开心。

 [https://www.youtube.com/embed/fh2WcTcoLBc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/fh2WcTcoLBc?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



作为工程学的壮举，无家可归者的社会评论，以及互动木偶剧，[Ables]让我们在无家可归和人机互动的问题上暂停下来。对于更异想天开的互动来说，一个围绕着你的机器人球是一种令人愉快的方式来适应机器人在社会中的存在。
