# 将一台老式电视接入示波器

> 原文：<https://hackaday.com/2017/04/06/hacking-a-vintage-tv-into-an-oscilloscope/>

你还有一台老式的模拟阴极射线管电视机吗？随着数字信号的出现，模拟电视正走向垃圾箱或回收中心。但你还是可以好好利用它们，就像【GreatScott！]通过[将电视转换成一个简陋的示波器](https://www.youtube.com/watch?v=aScAZReGQc0)做到了。

诀窍是控制两个偏转线圈，这两个线圈在水平和垂直方向上移动 CRT 内部的电子束。该视频详细描述了识别线圈并使用 Arduino nano 结合 DAC 放大输入信号以在电视屏幕上获得波形的过程。一步一步的解释和伟大的编辑使这个项目愉快地观看。

即使你不跟随[GreatScott！] s 步骤建立一个简单的示波器，不要扔掉那个老式电视！，它是模拟器件的重要来源。反激式变压器可用于制造高压电源，还可以获得一些不错的高压电容(电解电容和聚酯薄膜电容)、高压水平输出晶体管、铁氧体变压器、磁线，以及许多其它小器件。你可能想尝试的旧电视机的其他用途是[将你的电视转换成游戏控制台](http://hackaday.com/2016/10/13/from-crt-tv-to-all-in-one-console/)，或者通过在 CRT 电视上用光敏笔绘图来控制的[音频合成器。](http://hackaday.com/2011/10/29/synthesizing-sound-with-a-light-sensitive-pen-and-crt-television/)
