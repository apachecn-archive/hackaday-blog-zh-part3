# 数控计算器做你做不到的事

> 原文:[https://hack aday . com/2017/12/05/CNC-calculator-does-what-you-cant/](https://hackaday.com/2017/12/05/cnc-calculator-does-what-you-cant/)

黑客社区——以及更大的黑客社区——可以做任何事情。Hackaday 的读者定期驾驶宇宙飞船。元素的转化在这里是一个简单的科学项目，一个高中生就能轻松完成。Hackaday 的读者已经解决了国际危机，登上了珠穆朗玛峰，有一天我们会让读者从火星上的 IP 地址访问 Hackaday。我们社区的能力几乎是无限的。

这个项目做了一件黑客读者完全没有能力做的事情。作为一个很酷的小奖励，这个设备的外壳是一个美丽的铣削铝作品，阳极氧化在一个深刻的，美丽的黑色和刻有严格的精度。

这个构建的核心本质上是一个加载了一些特殊代码的 Arduino，它能做任何人都做不到的事情。此外，还有一个小型锂电池、充电电路、字符显示器和一个小键盘。这里真的没有什么东西不能从您最喜欢的 AliDXExtremeDeal 商店获得。

这里真正的展示是漂亮的铝制外壳。这是在 Fusion360 中设计的，并在装有轻微磨损的立铣刀的 Tormach CNC 上铣削。雕刻是用 Lakeshore carbide 雕刻机完成的。第一个原型是用粉末涂层完成的，因为这是家庭商店里的人在铣削外壳上进行完美涂装的最简单的方法。这款令人惊叹的设备的生产版本([在这里可以买到](https://saundersmachineworks.com/collections/shop-tools/products/the-imperializer-arduino-mm-to-inch-converter?variant=5064662253598)，尽管在撰写本文时已经售罄)经过阳极氧化处理，看起来棒极了。

如果这是那种吸引你的逻辑欲望的项目，只是带有一点反美情绪，一定要看看 Hackaday *上评论最多的帖子*。那里的评论区有很多很棒的观点，即使正在讨论的主题对整个 Hackaday 社区来说是迟钝和怪异的。
