# 自动驾驶卡车自学 Powerslide

> 原文:[https://hack aday . com/2016/05/29/autonomous-truck-teaches-self-to-power slide/](https://hackaday.com/2016/05/29/autonomous-truck-teaches-itself-to-powerslide/)

当你是一个初次体验驾驶的青少年时，“转向打滑”似乎是违反直觉的，但一旦你驾驶了几个冬天，你就会像专业人士一样漂移。我们从经验中学习，事实证明，[这款全自动动力滑行拉力赛卡车](http://spectrum.ieee.org/cars-that-think/transportation/self-driving/autonomous-mini-rally-car-teaches-itself-to-powerslide)也是如此。

弄清楚如何处理摩擦可选的道路完全是佐治亚理工学院 AutoRally 项目的重点，该项目将一辆经过严格技术改造的 1/5 比例拉力卡车放在户外泥路上测试。车载四核 i7 配备了高精度 IMU、高分辨率 GPS、双前置摄像头和每个车轮上的霍尔效应传感器，采样频率为 70 Hz，可以随时准确知道车辆的位置及其与轨道的关系。没有外部传感或计算-运行轨道所需的一切都在 21 公斤的卡车里。下面的视频显示了卡车如何独自在椭圆形轨道上行驶，只有一个简单的目标——尽可能保持每秒 8 米的目标速度。卡车像老板一样处理红乔治亚粘土，不仅处理不同的表面条件，还处理明亮到黑暗的照明过渡。到目前为止，卡车似乎只处理椭圆形轨道，但我们打赌，更复杂的轨道是平台的下一步。

虽然我们真的很喜欢[这种自动追逐车辆](http://hackaday.com/2015/02/01/autonomous-vehicle-following-vehicle/)的乘坐规模，但除此之外，最近这里没有太多非公司自动驾驶车辆黑客。让我们希望 AutoRally 是一个迹象，表明黑客还没有完全放弃谷歌的领域。为什么让他们享受所有的乐趣？
