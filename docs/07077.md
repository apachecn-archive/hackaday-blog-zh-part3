# 你认为你对焊接了如指掌？

> 原文:[https://hack aday . com/2017/09/12/think-you-know-everything-about-soldering/](https://hackaday.com/2017/09/12/think-you-know-everything-about-soldering/)

[Joshua]经常与他的 YouTube 频道的观众讨论他的焊接技术。他最终决定采访 SRA 焊接公司的总裁兰迪·鲁宾斯坦。在将近一个小时的时间里，他们谈论了从焊料合金到适当温度的一切。他们还谈到铅暴露、焊剂清洗和无铅焊料。

他们还讨论了使用无铅焊料和使用特殊焊料移除 SMD 元件的返工策略。老实说，虽然视频的第一帧说“你的焊料很烂”，但我们真的没有发现任何关于每个人都在做错的事情的惊天动地的启示。然而，我们确实找到了很多好的建议和一些有趣的细节，比如不同焊料合金的用途。

采访的重点是用接触铁焊接，尽管热空气在爱好者中使用得更多。然而，使用普通熨斗仍然是一项基本和必要的技能，即使如果你有热风或回流炉，你不会在每项工作中都使用它。

如果你真的想要一个焊接教程，Pace one 是一个老掉牙的好东西。如果你想了解更多，我们之前也讨论过[焊接冶金](https://hackaday.com/2017/02/23/what-the-flux-how-does-solder-work-anyway/)。
