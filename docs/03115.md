# 修复载人多翼飞机没有可怕的坠毁

> 原文:[https://hack aday . com/2016/07/28/repaired-manned-multi copter-flys-without-breaking-crash/](https://hackaday.com/2016/07/28/repaired-manned-multicopter-flies-without-horrifying-crash/)

[amazingdiyprojects]已经在他的疯狂的八螺旋桨汽油动力危险桶中进行了许多[测试飞行。](https://www.youtube.com/watch?v=Uv5RigExyyQ)

我们上一次报道这个项目是在他不幸在一次遥控试飞中毁坏了这个东西的时候。他后来发现发动机的曲轴轴承爆炸了。产生的弹片摧毁了发动机，并使无人机坠毁。他将这种故障模式描述为“令人担忧”。

同样令人担忧的是，一旦所有的螺旋桨都启动了，就要进入座位。他称之为“小心脚步，否则就去死”。不管怎样，他也描述了飞行是如此的令人难以置信的有趣，以至于很难置身事外；就像机械毒品一样。这解释了为什么他的频道最近被他测试多翼机的视频所占据。这些视频是在休息后发现的。

该设备每分钟消耗 0.65-0.7 升汽油，他一直在通过储备解决所有问题。这意味着从弄清楚如何驾驶它到发现地面效应产生的灰尘会堵塞空气过滤器的所有事情；这导致他们运行稀薄，随后烧毁火花塞。危险，但很酷。
