# 一个桌面星球大战为主题的乐高赛车游戏

> 原文：<https://hackaday.com/2018/03/04/a-tabletop-star-wars-themed-lego-racer-game/>

当谈到玩具建造套件世界无可争议的国王的头衔时，丹麦人拥有它。玩具箱里到处都是乐高玩具，世界各地的父母都知道赤脚踩在流浪砖上的快乐。除了面向年轻人和收藏家的主题布景，我们在这些页面上看到了很多乐高项目。有时他们来自硬件黑客，他们选择乐高是因为他们手头有一些或者因为它的实用性，但有时他们来自乐高社区，而不是更广泛的社区。

以【亚历克西斯·多斯桑托斯】的[明星赛车手为例前者。这是一款完全由乐高制作的桌面赛车游戏，由乐高头脑风暴公司提供控制。这是一个真正的滚动公路游戏，轨道由五个连续的灰色乐高部分组成，上面附有障碍物。在用户的控制下，赛车在前面从一边滑到另一边，目的是在它们以不同的速度向你靠近时避开它们。](https://www.flickr.com/photos/tkel86/albums/72157692723961504)

这是一件美丽的作品，除了链接的 Flickr 照片，你还可以在休息时间下面的 YouTube 视频中看到它。贴纸上写着这是一个非常容易上瘾的游戏，我们不会不同意。
