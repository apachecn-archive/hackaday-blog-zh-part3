# 微波炉变压器的重型起重电磁铁

> 原文：<https://hackaday.com/2017/01/15/heavy-lift-electromagnet-from-microwave-oven-transformers/>

没关系，你可以承认——从你第一次在废品场看到那些巨大的电磁起重机开始，你就想拥有一台。虽然报废微波炉捐赠的零件可能不能扔在汽车上，但它可以让你[建造自己的电磁提升装置](http://www.makeitextreme.com/en/newsroom/entry/VIDEOS/electromagnet)并最终实现这个梦想。

我们最近看到[MakeItExtreme]把几个微波炉变压器变成了一个有点不明智的爬壁装备。看起来这可能是这个版本的灵感来源，而成品这次似乎更有用一些。三个 mot 的框架被切开，以移除次级线圈，并留下暴露的铁芯作为未来磁体的磁极。一个浅碟由钢制成，磁铁焊接到位。

初级线圈连接在一起，磁铁用环氧树脂封装，商业端干净利落，整个过程都要经过测试。[makitextreme]在下面的视频中没有进入控制细节，但该网站提到磁体由 24V 15A 电源供电，在市电故障时有备用电池。

到目前为止，他们已经举起了 200 公斤，对于已经挤满了其他产品的商店来说，这看起来是一个非常酷的附加产品，比如他们的 [MOT 点焊机](http://hackaday.com/2016/04/24/diy-spot-welder-doesnt-look-like-it-will-immediately-kill-you/)和[丙烷罐喷砂机](http://hackaday.com/2016/05/15/propane-tanks-transformed-into-extreme-sandblaster/)。
