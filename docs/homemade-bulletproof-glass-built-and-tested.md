# 自制防弹玻璃，建造和测试

> 原文：<https://hackaday.com/2016/01/28/homemade-bulletproof-glass-built-and-tested/>

黑客倾向于隐藏大量看似无用的东西，直到它化险为夷。这不仅包括我们零件箱里的垃圾，还包括商店里充满技巧和窍门的大脑。考虑到这一点，你可能想把[AvE]的视频中的一些技巧归档，这些视频讲述了他是如何以备不时之需制作防弹玻璃的。

据他自己承认，[AvE]的视频有点脱节，防弹玻璃的话题只是在开始时被提及，在结尾时又被简单提及。大部分视频都是关于加工一个坚固的玻璃支架，用于在炉灶上进行测试。不过，从加工过程中可以学到很多东西，而且[AvE]总是很有趣，所以这个视频值得一看。防弹玻璃本身是一个长期项目的一部分，这个项目是[AvE]首先向他的 Patreon 顾客发布的——一个被称为“午夜太阳”的荒谬的过度建造的手电筒。他的前两次层压 Lexan 光盘的尝试都不太理想，因为两种品牌的氰基丙烯酸酯胶都模糊了聚碳酸酯。请继续关注视频的结尾，了解将 Lexan 焊接成光学透明三明治的秘密。

至于在炮火下的测试，[AvE]把装备送到 buddy [TAOFLEDERMAUS]进行热铅处理。广告之后的视频显示，玻璃确实是防弹的，只要问题中的子弹是 0.22 毫米口径的。不过，对于 9 毫米口径的子弹来说就不是那么回事了——那是一个明显的穿洞。尽管如此，自制啤酒的表现还是令人印象深刻。

如果你想要一种可以阻挡箭的东西，有很多材料科学可以向古希腊人学习。
