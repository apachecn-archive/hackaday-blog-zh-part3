# 口袋大小的工作站体育 Pi 零，弹出屏幕

> 原文：<https://hackaday.com/2018/04/11/pocket-sized-workstation-sports-pi-zero-pop-up-screen/>

我们中的许多人都可以使用通用便携式工作站，这种工作站小到足以放入口袋，但仍可用于快速故障诊断会话。智能手机上的终端应用程序通常可以很好地完成工作，但它们缺乏这种便携的便携式树莓派工作站的派头。

看起来[迈克尔·霍恩]对他小巧的 Linux 机器并没有明确的使命，但没关系——我们为艺术而尊重艺术。展览的明星是外壳本身，这是一个用于移动 DVD 播放器或备用相机的仪表板。屏幕是 4.3 英寸的 TFT，分辨率相对较低，所以[迈克尔]没有对它抱太大期望。他面临着一些挑战，比如处理显示器和他打算塞进底座的 Raspberry Pi Zero W 的不同电压需求。幸运的是，显示器内部将 12 伏的电源调节到 3.3 伏，所以他只需接入 Pi 上的 3.3 伏引脚，就可以通过 USB 充电器为一切供电。该显示器还内置了一些智能，在应用复合视频之前一直处于空白状态，这一开始会造成一些混乱。一些带连接器的机箱模块，一个无线键盘，他还有一个漂亮的小机器。

对 GUI 机器不感兴趣？需要一个纯文本的串行终端？我们以前也见过这种情况。这是[的一款，内置了漂亮的滑出式键盘](https://hackaday.com/2017/03/24/pi-zero-w-impersonates-iphone-becomes-terminal/)。
