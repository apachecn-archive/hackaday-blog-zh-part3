# 圆锯+创新围栏=独特的 DIY 台锯

> 原文：<https://hackaday.com/2015/09/17/circular-saw-innovative-fence-unique-diy-table-saw/>

台锯通常是有抱负的木工想要的第一台机器。但即使是轻量级的承包商的锯也不便宜，一个真正好的橱柜锯既昂贵又难以置信的重。任何台锯都是一个可怕的机器，可以在瞬间将你的手指从手中解放出来。为了寻找所有这些问题的解决方案，[Seumas]想出了一种针对圆锯的独特的[台锯转换方法，这种方法提高了安全性，降低了拥有台锯的障碍。](https://www.youtube.com/watch?v=bU-wz0rw5Uw)

把一个低成本的圆锯倒过来放在桌子上是过时的做法——我们以前见过很多这样的例子，包括最近的这篇文章。[Seumas]的想法的闪光点在于栅栏和桌子的结合。典型的栅栏需要保持与刀片完全平行，同时与工作台垂直，但仍需要移动以调整切割宽度。在[Seumas]的设计中，围栏固定在桌子上，整个桌子在高压层压导轨上左右滑动。理论上，栅栏永远不会偏离正确的方向，切割宽度可以比典型的台锯宽得多——令人印象深刻的是刀片右侧 3 英尺。

至于安全性，休息之后，[Seumas]在视频中展示了相当多的 DIY 附件。他建立了自己的 Lexan 刀片保护，反回扣棘爪，和股票压制。再加上一些小细节，比如用于锁定桌子的自制夹子、延伸出料支架和内置灰尘收集装置，你就可以用很少的购买成本让自己成为一台非常强大的机器。
