# DIY 粉末涂料

> 原文：<https://hackaday.com/2015/10/25/diy-powder-coating/>

如果你还没有一个不能用来烘烤食物的烤箱，还有一个原因: [DIY 粉末涂层](https://toemat.com/powder-coating/)。粉末涂层比油漆更坚硬、更耐用——这是因为它是烘烤在零件上的。[Thomas]尝试在一些滑板卡车上喷涂粉末，使用合适的工具，发现这个过程非常简单。

[Thomas]只需要几样东西就可以对零件进行粉末涂装，第一件也是最重要的一件是粉末涂装枪。几年前，工匠公司生产了一种粉末涂料枪，现在亚马逊和易贝上仍然可以买到，价格约为 50 美元。粉末数量多，价格便宜。唯一需要的其他工具是 N95 或更好的呼吸器，一些用于掩盖部分的高温胶带和烤箱。如果你想涂覆大部件，有 [DIY 烤箱可供选择。](http://hackaday.com/2014/09/09/diy-powder-coating-oven-gets-things-cooking/)

零件喷砂至裸露金属后，[Thomas]用聚酰亚胺胶带掩盖了零件的所有孔和螺纹。任何能够耐高温的胶带都可以，而且我们大多数人都有一卷 Kapton 放在 3D 打印机旁边。

零件通过静电荷涂上粉末，这意味着将枪的接地线连接到零件上。在那之后，只是把枪装满粉末，放入 450 华氏度的烤箱，让粉末液化。

在下面的视频中，你可以看到[托马斯]用他的方法对一套铝制滑板卡车进行喷砂、打粉和烘烤。与其他金属零件表面处理方法相比，例如阳极氧化或电镀，粉末喷涂非常简单，任何人都可以在车库里完成。
