# 用 Ping 闪烁灯

> 原文：<https://hackaday.com/2017/07/06/blinking-a-light-with-ping/>

闪光灯奖正在举行，这意味着所有我们喜爱的极客和 YouTubers 都渴望用 555 做些什么。规则很简单:以某种方式打开和关闭灯泡*。[Sprite_tm]正在接听电话，[他正在以光速闪烁灯泡](https://www.youtube.com/watch?v=9t1qx0uWTSg)。*

 *[Sprite]闪烁灯光的方法很简单:使用 ESP32 开发板打开继电器。同时，通过遍布全球的四台服务器向互联网发送数据包。当数据包通过上海、荷兰、香港、德国的服务器，最后到达日本，然后再返回时，灯泡熄灭了。这是光速和海底光纤质量的物理演示。

这条路线相当长，从上海到荷兰到香港到柏林，最后到大阪附近的某个地方的单程大圆路线的合理估计约为 36，000 公里。这个灯泡包的往返行程是 72，000 公里，或者大约 0.2 光秒。当然，不直接穿越喜马拉雅山的光纤和电缆会产生延迟，路由器会产生延迟，真空中的光速和玻璃光纤中的光速之间存在差异。尽管如此，光是快速的，灯以大约 1Hz 的频率闪烁。
