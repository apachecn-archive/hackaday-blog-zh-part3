# 幸运天线

> 原文:[https://hackaday.com/2016/02/27/a-lucky-antenna/](https://hackaday.com/2016/02/27/a-lucky-antenna/)

天线有各种形状和大小，哪种最好完全取决于你用它做什么。从无人机发送视频的一个非常受欢迎的选择是苜蓿叶形天线。它是圆极化的，当你有一辆移动的汽车时，这是一个优势。它还减少了多径干扰。

立体式包含三个以不同角度间隔的闭合环。这种天线发射信号很好，但接收信号不理想。构建后也很难调优。但是，对于合适的工作，它是一个很好的执行者。[Vitalii Tereshchuk]展示了他如何制作适合 WiFi 路由器的苜蓿叶形天线。

这个天线的起点是一个股票鞭状天线。一些烙铁工作和纸板夹具来获得正确的角度可以将鞭子转换成苜蓿叶形。你可以在下面的视频中看到整个过程。

我们已经看到了[微型天线](http://hackaday.com/2012/03/28/ask-hackaday-building-nano-scale-antennas/)，甚至一些由[液态金属](http://hackaday.com/2015/08/31/liquid-metal-changes-shape-to-tune-antenna/)调谐的天线。立体式天线可能更适合周末项目。
