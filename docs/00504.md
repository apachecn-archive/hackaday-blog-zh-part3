# 管状胶带枪“画”出你永远不能坐的家具

> 原文：<https://hackaday.com/2015/10/29/tubular-tape-gun-sketches-furniture-you-can-never-sit-on/>

有时候你只需要一个真人大小的模型。当你这样做时，伸手拿起你的[(高度改进的)胶带枪](https://hpi.de/baudisch/projects/protopiper.html)并开始绘图。

正如 Protopiper 团队所描述的，“枪”是一种计算机辅助的手持制造设备，用于想象大型物体的布局——他们给出的主要例子是家具。想确定沙发是否合适吗？为什么不花 10 分钟建立一个磁带模型呢？

听起来很疯狂？有点，但这个装置本身相当精巧。它采用普通胶带，测量它，并将其卷成管状，这导致了一个惊人的强大结构，让您可以非常容易地建立 3D 形状。从设计的角度来看，它非常出色。

从将胶带卷成结构管的机制到允许您轻松连接到另一个管或结构的翼形末端连接器，整个设计必须经过多次设计迭代才能正确。我们印象深刻。

或者为了打印非常大的形状，为什么不建造一个建筑大小的 delta-bot？
