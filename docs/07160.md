# 周五黑客聊天:关于无人机的一切

> 原文:[https://hack aday . com/2017/09/20/Friday-hack-chat-all-about-drones/](https://hackaday.com/2017/09/20/friday-hack-chat-all-about-drones/)

未来，无人机将会充斥天空。全世界都在议论纷纷(哈！)关于无人机创新用途的新闻。很快，我们的飞行机器人霸主将被用于救援行动、监视、反叛乱任务、恐怖主义、农业以及直接从当地亚马逊机场向你的后院运送冷冻狗食。未来是疯狂的。

[![](../Images/13a24c7d61d0d77a585baf3cacfbafdd.png)](https://hackaday.com/wp-content/uploads/2017/09/8004581505868365473.png) 在本周的黑客聊天中，[我们将谈论无人驾驶飞行器](https://hackaday.io/event/27315-uav-unmanned-aerial-vehicle-chat)。这是一个巨大的主题，包括航空设计、自动飞行机器的法律含义、通过无人机递送包裹的真正效率，以及用数千个移动机器人观察站覆盖一个城市的道德模糊性。简而言之，由于强大的无刷电机和锂电池，未来将被带到我们面前。

我们本周黑客聊天的嘉宾是【Piotr Esden-Tempski】，狗仔队无人机的无人机自动驾驶硬件开发者。狗仔队可以用于多架飞机的自主飞行和控制，我们将讨论可以用于这些应用的嵌入式系统的类型。[Pitor]也是 [1Bitsy ARM 开发平台](https://hackaday.io/project/25632-1bitsy-1up)的开发者，黑魔法探针 JTAG/SWD 程序员/调试器和 [1BitSquared](https://1bitsquared.com/) 的创始人。

在这次黑客聊天中，我们将讨论无人机的开源硬件设计，所有机载机器人，这些飞行机器人中的传感器，消费者和 prosumer 固定翼无人机的停滞发展(唉，另一个双关语)，ARM 嵌入式系统，以及 JTAG 和 SWD 编程和调试。我们也接受观众的提问，这是引导讨论的电子表格。
