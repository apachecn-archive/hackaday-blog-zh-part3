# 激光切割和焊接制造 3D 物体

> 原文:[https://hack aday . com/2015/12/06/laser-cut-and-weld-makes-3d-objects/](https://hackaday.com/2015/12/06/laser-cut-and-weld-makes-3d-objects/)

大家都喜欢 3D 打印吧？但是它比 2D 激光切割要慢。要是有办法将多个 2D 切片组合成一个三维模型就好了。好的，我们知道你已经用胶水和/或关节手工完成了。但是这有什么意思呢？

[激光堆垛机](http://hpi.de/baudisch/projects/laserstacker.html)为您自动化整个程序。他们调整了激光切割机的设置，不仅允许切割，还允许焊接丙烯酸。这使得他们可以在没有人工干预的情况下，通过首先在一个深度切割，然后在另一个深度选择性地重新焊接在一起，用丙烯酸切片构建 3D 物体。他们还开发了一些软件，以及一个功能元素库，使得设计这类部件变得更加容易。

他们的网站上几乎没有关于它如何工作的任何细节，所以你必须观看休息时间下面的视频，并做出一些有根据的猜测。看起来他们向上抬起刀头来进行焊接，可能会将光束分散一点。他们是否也以较低的功耗或较慢的速度运行？我们要求细节！

无论如何，看看 3:30 的演示视频，在那里他们通过他们的步伐运行切片到深度和愈合模式。令人印象深刻。

这个过程会有各种各样的限制，我们还没有完全考虑清楚。例如，突出部分需要切割到一定深度，然后修复顶层。它会变弱(他们在视频中演示了这一点)，但这有多大关系呢？看起来怎么样？无论如何，这个想法是可以随意发挥的。

当然，在激光切割机上制作 3D 物体还有其他方法。你可以[用箱形接头](http://hackaday.com/2015/04/30/3d-printering-laser-cutting-3d-objects/)组装零件，或者，如果你准备迎接挑战，[给你现有的刀具](http://retrotechjournal.com/2015/09/01/cutting-3d-shapes-on-a-laser-cutter/)增加一个旋转轴。但是对于简单的块状 3D 形状，我们认为切割焊接法很有前途。
