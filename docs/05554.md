# Hackaday 奖参赛作品:一整套生物医学传感器

> 原文:[https://hack aday . com/2017/04/22/hack aday-prize-entry-a-complete-suite-of-biomedical-sensors/](https://hackaday.com/2017/04/22/hackaday-prize-entry-a-complete-suite-of-biomedical-sensors/)

如果我们只有听的工具，人体会告诉我们很多东西。不幸的是，从业者使用的大多数诊断设备都很昂贵，如果你只是想在引擎盖下随便看看，那就超出了范围。为了完成这项任务，[这款功能齐全的生物医学传感器套件](https://hackaday.io/project/20912-biomed-shield-for-arduino-101-powered-by-intel)可能会派上用场。

与其说是一个完整的项目，不如说是一个使能平台，[Orlando Hoilett]的防护罩设计结合了许多我们以前见过的传感器。两种主要模式是光电容积描记法，它使用 [MAX30101](https://www.maximintegrated.com/en/products/analog/sensors-and-sensor-interface/MAX30101.html) 通过光的不同吸收和反射来检测血容量和氧饱和度的变化，以及生物电势测量，它使用 AD8227 内置的仪表放大器来提供您可能需要的所有“电图”:心电图、肌电图，甚至是记录眼球运动的眼电图。[奥兰多]甚至还安装了温度和光线传感器用于环境监测。

[Orlando]很快指出，这是一个教育项目，而不是医疗仪器，并且只能在完全脱离电源的情况下使用——请仅使用电池电源和蓝牙。想知道为什么吗？查看[关于无变压器电源的令人震惊的真相](http://hackaday.com/2017/04/04/the-shocking-truth-about-transformerless-power-supplies/)。
