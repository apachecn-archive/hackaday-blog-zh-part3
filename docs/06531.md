# 制作自己的簧片开关

> 原文：<https://hackaday.com/2017/07/26/make-your-own-reed-switches/>

【Lucid Science】向我们展示了如何制作一些简单的簧片开关。簧片开关是检测磁场的简单元件，一旦检测到就可以闭合或断开电路。虽然不是一件真正漂亮的东西，但如果你迫不及待地想订购一些，或者你想亲手从头开始制作一些组件，这些 [DIY 簧片开关](https://www.youtube.com/watch?v=isfYMKtd_Jg)应该可以帮助你。

簧片开关通常有非常小的外形，所以如果你需要小的东西，这可能不适合你，但是视频确实从宏观上向你展示了簧片开关的基本工作原理。要制作你自己的簧片开关，你只需要一些零件:一些铜、漆包线和磁铁。它们真的是简单的设备，但是有时很容易忽略一些东西是多么简单，因为它们太小了，你看不到它们是如何工作的。

从头开始制作您自己的组件可能是理解所述组件内部工作的最佳方式。过去，我们已经看到了一些非常棒的自建元件，从这些漂亮的 [DIY 谢妮管](http://hackaday.com/2016/10/04/the-art-of-making-a-nixie-tube/)到甚至[自己制作发光二极管](http://hackaday.com/2012/06/05/homemade-silicon-carbide-led/)
