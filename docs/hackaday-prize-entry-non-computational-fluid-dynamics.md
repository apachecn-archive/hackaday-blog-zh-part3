# Hackaday 奖参赛作品:非计算流体力学

> 原文：<https://hackaday.com/2016/08/09/hackaday-prize-entry-non-computational-fluid-dynamics/>

计算流体力学，简称 CFD，被应用于从飞机设计到新摩天大楼的机翼有多好的一切领域。当然，建造翼型的科学比 CFD 要古老得多，这就引出了在计算机出现之前翼型设计是如何进行的问题。

答案当然是风洞。逛逛几个非常好的空中博物馆，你会发现风洞，从莱特兄弟建造的长木箱到木工非常感兴趣的奇妙的胶合板装置。

[Joel]作为一名未来的航空工程师，他需要一个最终项目，但他希望他的项目是实物，是一个可以再次使用的工具。他决定建造一个风洞，这也是他参加 Hackaday 奖的原因。

这个风洞不是一个建筑物大小的巨大装置。最早的风洞只是几米长的简单装置。一端有一个风扇，一个稳定风的部分，一个小室，一个空气流动的地方，这也是一个非常简单的装置。仅仅因为有些东西很简单并不意味着最近就有人建造了一个，尽管:[Joel]找不到任何一个一步一步建造风洞的人。这个项目就是——一个开源风洞。

这个风洞的设计非常简单，用相对简单的模具用玻璃纤维建造。该设计可以适用于各种电风扇，并且该建筑最有趣的部分——烟雾机—[已经完成](https://hackaday.io/project/12080-cierzscience-tunnel/log/41659-finished-the-smoke-machine)。
