# 在你的…滑板下面点一把火？

> 原文：<https://hackaday.com/2017/05/15/light-a-fire-under-your-skateboard/>

孩子们，请不要在家里尝试这个。或者至少确保周围没有易燃物。

抛开这一点，我们不得不问——谁不喜欢玩火呢？我们敢打赌，你们中的许多人在生活中的某个时候也喜欢玩滑板。【mikeasaurus】把两个心爱的活动结合起来，做了一个[火焰投掷滑板](https://www.instructables.com/id/Flamethrower-Skateboard/)！零件数量相当少，如果你不能在当地采购所有的物品，看起来一切都可以从亚马逊购买。

[mikeasaurus]给出了一些有用的提示，例如他如何弯曲油箱盖上的两根管子中的一根，以防止燃油流出。此外，他还使用了一个适配器将管子的直径从 1/4”降低到 1/8”，这有助于更好地实现燃料流。

mikeasaurus 没有通过手持遥控器操作额外的电子设备和电线来使这个小脚锅变得更加复杂，而是决定直接在滑板上安装控制器，只需几个脚动开关。这使得他可以自由地向所有看着他飞驰而过的旁观者挥手。或者更好的是，带一个灭火器。

诚然，从视频来看，火焰并没有被“扔”得太远，而且[mikeasaurus]自己也说:

只要火焰被激活时你还在前进，你就可以走了

正因为如此，你可能不想使用你最喜欢的木板，因为它会受到直接的火焰。

你会在休息后观看视频时看到这一点。
