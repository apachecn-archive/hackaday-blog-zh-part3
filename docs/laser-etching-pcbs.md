# 激光蚀刻印刷电路板

> 原文：<https://hackaday.com/2017/08/22/laser-etching-pcbs/>

不久前，[Marco]在一台 CNC 机器上安装了一个强大的激光二极管，试图蚀刻覆铜板并制造一些 PCB。结果不是很好，但是这项技术很有前途。在一项新的实验中，[Marco]从中国购买了一套非常便宜的激光雕刻工具，现在这项技术[看起来可能会成为赢家](https://www.youtube.com/watch?v=1hFNj86L7sk)。

[Marco]他的激光雕刻机是从 Banggood 采购的，这与你对价格低于 200 美元的数控机床的期望非常接近。框架是铝挤压的，马达是现成的步进器，电子设备只是类似 Pololu 的驱动器，软件介于糟糕透顶和糟糕透顶之间。尽管如此，这台机器可以切割木材，皮革，织物，并可以用一个大的蓝色激光二极管清除喷漆。

为了制造他的印刷电路板，[Marco]首先清洁一块覆铜板，用喷漆覆盖它，然后用激光进行爆破。这方面的首选软件是 LaserWeb，对于一台便宜的机器来说，这个结果已经很不错了。

在电路板上涂上油漆并用激光喷砂后，还需要几个额外的步骤来制作 PCB。这种工艺仍然需要在氯化铁或其他酸性物质中进行蚀刻，但效果很好。事实上，如此之好，以至于[Marco]正在用铜箔和 Kapton 做实验来制造柔性电路板。你可以看看下面这些实验的视频。
