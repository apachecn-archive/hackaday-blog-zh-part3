# 过山车大亨 IRL

> 原文：<https://hackaday.com/2017/12/10/roller-coaster-tycoon-irl/>

增材制造已经走了很长一段路，但我们肯定还没有到可以 3D 打印过山车的地步，对吗？[原来你可以](https://3dprinterchat.com/2017/12/roller-coaster-print-my-ride/)，只要 1/25 尺度对你来说就够好了。

有些人建造模型铁路，但[马特·施莫泽]一直对过山车情有独钟。不满足于过山车大亨，[马特]决定建造一个精确的工作模型[Invertigo](https://en.wikipedia.org/wiki/Invertigo_(roller_coaster))，一个在国王公园的飞去来器过山车，俄亥俄州辛辛那提的过山车涅槃。[Matt]的模型覆盖了一层胶合板，高约 3 英尺，从支撑塔和支撑到轨道部分本身，细致入微地再现了原作。看起来他把所有的东西都分成几部分打印出来，就像原件被制造出来一样，各部分用螺栓固定在一起。尽管所有的部分都经过打磨和蒸汽平滑处理，但轨道本身太粗糙，无法使用，所以它们被替换成了塑料管。但是其他的都是打印出来的，都可以用。Arduino Mega 控制电梯电机，打开和关闭轿厢上的安全杆，并操作车站内的乘客门和活动地板。下面的视频展示了它的作用。

想要一个属于自己的杯垫，但是想要更大一点的？[我们完全理解](https://hackaday.com/2014/04/27/backyard-roller-coaster-family-physics-and-fun/)。
