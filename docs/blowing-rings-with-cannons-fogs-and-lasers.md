# 用大炮、雾和激光吹响光环

> 原文：<https://hackaday.com/2018/05/23/blowing-rings-with-cannons-fogs-and-lasers/>

在当今以健康生活方式为导向的世界，吹烟圈不会再给太多人留下深刻印象。当然除非你是【夜鹰之光】[用漩涡加农炮吹它们，加上激光做视觉效果](http://www.instructables.com/id/High-Velocity-Vortex-Cannon/)。虽然，他最初的动机是在他的后院圆盘高尔夫球场建造一个可以将丢失的飞盘从树上射下来的设备，作为一个狂热的[爱好者，他使用丙烷火炬](https://hackaday.com/2017/10/12/propane-powered-plasma-rifle/)在空中射击东西，他选择了涡流炮以避免射弹可能造成的伤害风险。

从一开始就考虑到安全问题，[夜鹰之光]选择了不会让他或跟随他脚步的人暴露在任何有毒气体中的方式来建造大炮。桶是通过固定一卷平台板并简单地将其拉成圆锥形而形成的。一系列聚氯乙烯管和适配器建立了燃烧室，一端适合平台板桶，另一端适合丙烷火炬喷嘴。为了更容易瞄准和稳定，他还增加了一个三脚架。

因为空气漩涡是空气，因此它们本身是不可见的，它们不能提供最大的视觉刺激。[NightHawkInLight]用一个装在枪管上的造雾机解决了这个问题，还有一个激光线模块，你可以在休息后在他的建造视频中自己看到。在之前的漩涡加农炮项目中，我们也可以看到[一种更加户外的方法](https://hackaday.com/2018/04/18/fire-vortex-cannon-need-we-say-more/)来增加它的可见性。
