# 用机器人线切割器自动准备线

> 原文：<https://hackaday.com/2017/10/28/automate-wire-prep-with-a-robot-wire-cutter/>

当你从一次性生产转向大规模生产时，也许是为了满足 Kickstarter 的承诺或者保持你的 Tindie 商店的库存，你将需要装备起来。像制作线束这样的工作可能会很乏味且耗时，所以将它们外包给这个机器人剪线工可能是个好主意。

下面的视频讲述了这个构建的整个故事，尽管事实上[Maclsk]似乎已经从废物箱零件快速组装起来，但它看起来仍然非常专业。机器的商业端是一个 3D 打印机挤出机，当然，减去热端。Nano 控制挤压机的步进器射出正确长度的金属丝，以及挤压剪切机的伺服机构。一个 LCD 显示屏和一些按钮提供了完成构建的 UI。告诉它有多长，有多少，你就可以开始建造了。我们可以看到如何升级，以剥离电线，虽然两端剥离可能是棘手的。

几周前的这个组件胶带切割机器人会不会启发了[Maclsk]的建造？也许吧，但无论如何，两者都很有趣。
