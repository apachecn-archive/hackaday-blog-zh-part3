# 温柔的电鳗

> 原文：<https://hackaday.com/2018/05/16/gentle-electric-eel/>

电鳗被认为是可怕的动物，这并不奇怪。它们是光滑多肉的水蛇，可以召唤闪电。加州大学的生物学家和工程师在设计他们的电鳗 T1 时，心里还有别的想法。这条鱼不捕鱼，而是和它们一起无害地游泳。

传统的遥控飞行器依靠坚硬的外壳和旋转的螺旋桨。对海洋生物来说，这是嘈杂和不自然的。一个沉默的游泳者不会竖起任何眉毛，不是说鱼有眉毛。最具创新性的功能是人造肌肉，虽然细节很少，但它们似乎在内部使用一种介质来传导电荷，在外部，盐水环境传导相反的电荷，导致内外之间的膜收缩。下面可以看到一些游泳动作，也许我们敏锐的读者可以对这位水下冒险家的材料清单有所了解。

我们最喜欢的潜艇之一是 2017 年 Hackaday 奖得主，[开源水下滑翔机](https://hackaday.com/2017/11/11/open-source-underwater-glider-wins-2017-hackaday-prize/)。对于潜水器来说，更具艺术感的是,[号曲线 II](https://hackaday.com/2018/02/23/3d-printing-brings-a-childs-imagination-to-life/) 是我们见过的最优雅的一艘。
