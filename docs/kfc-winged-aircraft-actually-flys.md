# 肯德基有翼飞机居然会飞

> 原文：<https://hackaday.com/2017/04/14/kfc-winged-aircraft-actually-flys/>

[PeterSripol]已经制作了一架遥控飞机模型，但是他没有使用普通的机翼，而是决定尝试使用一些肯德基的鸡桶让它飞起来。两个肯德基桶代替了鸡翅，被连接到一个马达上，马达使桶旋转加速。在[马格努斯效应](https://en.wikipedia.org/wiki/Magnus_effect)的一点帮助下，这产生了升力。

人们尝试了许多不同的配置来让这个装置离开地面。他们最终确定了一个双螺旋桨装置，每个都反向旋转以获得前进的动力。这有助于抵消产生升力的旋转叶片的陀螺效应。经过多次失败的先造后飞的尝试后，他们终于把它发射升空了。它能工作，虽然不是很好，但它确实能飞，而且是可控的。也许再做一些调整和一点点尝试和错误，有人可以用这个概念建造一个真正独特的[遥控飞机](http://hackaday.com/2017/02/10/a-micro-rc-plane-builder-shares-his-tricks/)。
