# 荒野电台大楼

> 原文:[https://hackaday.com/2016/09/17/wilderness-radio-build/](https://hackaday.com/2016/09/17/wilderness-radio-build/)

[AA7EE]对构建无线电并不陌生。他的最新成果是基于流行的 SST 设计，从零开始建造一个 20 米长的 QRP 收发器。虽然 SST 已经作为一个套件提供，但[AA7EE]结合了其他人和他自己的一些设计变化。他甚至增加了一个机载键盘来简化操作。你可以在下面看到收音机的视频。

该建筑使用曼哈顿风格的 PCB 板。尽管文章的结构非常吸引人，但这篇文章的真正价值不仅在于详细解释了事情是如何发生的，还在于解释了为什么事情会是现在这个样子。这不是一个简单的项目，能够看到它一步一步地完成是非常有教育意义的。关于唯一没有充分解释的决定是红色和黄色旋钮改为黑色！你可以在下面的视频中看到这两个版本。

曼哈顿的建筑很整洁，但收音机也有一个吸引人的外壳。尺寸刚好大到可以在上面叠一对桨。

小收音机可能会有更多的改进。我们之前已经报道过[AA7EE]的射频漏洞，包括一个[外形迷人的无线电](https://hackaday.com/2015/09/15/radio-receiver-or-art-why-not-both/)和关于[的细节，同样的建造方法](https://hackaday.com/2011/09/04/a-ham-radio-receiver-manhattan-style/)用在这个无线电上。
