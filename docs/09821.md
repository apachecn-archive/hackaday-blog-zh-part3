# ROPS 将成为 X86 机器人制造商等待的跳板

> 原文：<https://hackaday.com/2018/06/23/rops-will-be-the-board-x86-robot-builders-are-waiting-for/>

如果您的机器人已经超越了 Raspberry Pi，只有 x86 主板的原始计算能力就足够了，那么您很可能会遇到接口问题。ISA 卡的日子已经一去不复返了，一台现代的 PC 并不是为了方便地与嘈杂的机器人硬件对话而设计的。USB 等可访问端口可以连接接口，但在此过程中会有很长的延迟。

一个解决方案来自 [ROPS，或*PCI-e 棒上的机器人*](https://hackaday.io/project/119133-rops) ，一种将 FPGA 放在超高速 PCI-e 卡上的卡，该卡提供有用的现实世界接口，如 CAN 和 RS485，一堆 I/O 线以及 IMU，气压计和 GPS。如果你认为你以前可能见过它，那么你是对的，它是[公开硬件设计挑战赛](https://hackaday.com/2018/05/02/these-twenty-amazing-projects-won-the-open-hardware-design-challenge/)的第一轮获胜者之一。他们仍然处于拥有 FPGA 开发板和开发软件的阶段，因此还没有任何 ROPS 板可供参考，但这是一个正在进行的项目，绝对值得关注。
