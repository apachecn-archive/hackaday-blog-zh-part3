# 用真实的磁带循环制作的磁带循环回声

> 原文：<https://hackaday.com/2015/10/07/tape-loop-echo-made-with-an-actual-tape-loop/>

许多数字流程都是以一种已经被取代的旧模拟设备命名的。例如，现在对一场演出进行“录音”并不罕见，尽管录音工作实际上是由数字录像机完成的。不过，有时候，即使在当今的数字世界，老式的模拟设备也有某种令人向往的特质。[马頔]的[磁带循环回声就是这种情况，它实际上是由一个物理磁带循环](http://www.instructables.com/id/DIY-Cassette-Tape-Echo/)构成的。

构建磁带循环硬件的过程出人意料地非技术性。通过将记录头和回放头彼此紧挨着定位，引入了延迟。回声是通过将原始的现场声音信号与来自磁带的延迟信号混合而产生的。通过改变磁带的速度或其他几个变量，可以获得许多不同的声音效果。

尽管实际上并不像听起来那么简单(该设备需要大量的反复试验)，但最终的效果肯定会让平克·弗洛伊德或贝克感到自豪。模拟并不是唯一的方法，有很多数字效果很容易制作，有些还有有趣的 T2 安装选项。
