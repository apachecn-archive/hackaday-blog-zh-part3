# 美丽的火箭人喷气背包复制品拥有令人印象深刻的金属制品

> 原文：<https://hackaday.com/2017/07/31/beautiful-rocketeer-jetpack-replica-boasts-impressive-metalwork/>

《火箭人》漫画和电影系列的粉丝们将会熟悉主人公的 20 世纪 30 年代风格的火箭背包。这是一个复杂的流线型曲线的复杂结构，多年来激发了许多娱乐活动。

大多数*火箭人*的喷气背包是由塑料、泡沫和其他重量轻的材料制成的，这对于角色扮演者和顾客来说是很熟悉的。但是[【大卫·盖顿】的这个不同，他是用钢板做的](https://www.youtube.com/watch?v=pvD7jzPZZBM)。

在他的视频中，吸引人的不是成品包装，尽管这是一个令人印象深刻的构建。取而代之的是工艺，不，是*工艺*，因为他记录了所涉及的金属制品的每一个阶段。钣金工行业的钣金工具出奇的简单，当你看的时候，很容易想到:“我也能做到！”。但是在短视频剪辑和明显的构建速度背后，隐藏着许多小时的艰苦工作和大量的技能。我们中的一些人会尝试过这种工作，但很少有人能做到这种程度。

视频是在休息，它带我们通过建设的组成部分，包括在年底的一些发动机细节是在树脂铸造。怀着敬畏之心观看！
