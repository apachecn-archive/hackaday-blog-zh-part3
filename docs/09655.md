# 特斯拉 Model 3 电池组拆卸

> 原文:[https://hack aday . com/2018/06/04/Tesla-model-3-battery-pack-tear down/](https://hackaday.com/2018/06/04/tesla-model-3-battery-pack-teardown/)

特斯拉 Model 3 已经上市近一年了，世界各地的黑客和修补者都渴望挖掘埃隆的最新产品，看看它是如何运转的。虽然它比之前的 S 型便宜很多，但新款特斯拉 35，000 多美元的价格仍然有点高，买一辆只是为了拆开它。因此，对于精打细算的“油猴子”来说，唯一能做的就是等到比你有钱的人撞毁一辆车，然后廉价购买残骸。

[![](../Images/7e8e56aaac780c702a0048ccb74fe38c.png)](https://hackaday.com/wp-content/uploads/2018/06/model3_detail.jpg)

Tesla Model 3 battery monitor board

这正是电动汽车鉴赏家杰克·里卡德所做的。他买下了他能拿到的第一辆损坏的 Model 3，然后开始对这台机器的心脏和灵魂进行长时间的拆卸:它的 75 千瓦时电池组。在这个过程中，他有了一些有趣的发现，并对特斯拉如何能够将 Model 3 的成本降低到与之前的车辆相比如此之低有了一些了解。

在特斯拉汽车上，电池组是一个巨大的平板，有效地占据了汽车的整个底部。为了移除它，[杰克]和他的助手在标准升降机上抬起 Model 3 的残骸，然后用一个小升降台放下电池。这里观察到了第一个区别:虽然 Model S 的电池是为快速交换而制造的(这一功能在实践中显然很少使用)，但 Model 3 的电池显然是要成为汽车的永久部件；移除它需要取出内部的大部分。

随着电池从汽车中取出并打开，Model 3 的最大变化变得显而易见。电池组实际上在一个集成单元中包含充电器、DC-DC 转换器和电池管理系统。在 Model S 上，这些组件安装在车辆本身中，而在 Model 3 上，大多数主要电子设备都存储在这个单独的模块中。

这极大地减少了汽车的布线和复杂性，并且[Jack]提到 Model 3 中唯一重要的硬件(除了电机)将是仪表板中的用户界面计算机。当这种电子模块的通信协议被逆向工程时，它可能不仅对电动汽车的转换非常有用，而且对离网储能等事情也非常有用。

一年多前，我们[报道了类似的特斯拉 Model S 电池的拆卸](https://hackaday.com/2017/02/28/tesla-model-s-battery-pack-teardown/)，以及[用多块残骸建造一辆工作车的不可思议的项目](https://hackaday.com/2017/09/20/salvaging-your-way-to-a-working-tesla-model-s-for-6500/)。
