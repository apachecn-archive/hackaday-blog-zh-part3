# 玩具车用气球动力给轮子打气

> 原文：<https://hackaday.com/2017/03/16/toy-car-pumps-the-wheels-with-balloon-power/>

我们已经盯上了格雷格·朱姆沃尔特。他一直在研究一些非常聪明的 3D 打印机械装置，他的最新原型是玩具汽车的空气发动机。你可以用压缩机给单个气缸供气，也可以向气缸吹气，但系上一个充气的气球会使系统自成一体。

上周我们看到了[发动机的原型](http://hackaday.com/2017/03/10/3d-printed-engine-chugs-away-on-balloon-power/)本身，不知道它是否有足够的动力驱动一个小火车发动机。我们几乎是正确的，因为它为一辆小车的前轮提供动力。

这不是[格雷格的]第一次牛仔竞技。他已经研究自给自足运动有一段时间了。这里显示的是[他的弹簧驱动汽车](https://www.youmagine.com/designs/tabletop-tri-mode-spring-motor-rolling-chassis)，你向后拉它来加载弹簧。这是玩具中的一个常见特征，通过隐藏在最宽齿轮内部的 3D 打印弹簧，看起来非常整洁。

这种印刷看起来很壮观，但气球驱动的原型更能激发我们的想象力。休息后观看视频时，请务必打开音响。是楚嘎-楚嘎把这个放在最上面。[Greg]还没有发布文件，所以你可以打印你自己的(它仍然是一个原型)，但在你等待的时候浏览他的其他设计——它们数量众多，会给你带来更大的笑容。还记得几年前马蒂亚斯·万德尔建造的那个多米诺乐高机器人吗？[Greg]有[可打印的模型](https://www.youmagine.com/designs/the-pink-and-green-domino-machine)！
