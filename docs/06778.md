# 超级会议采访:秋叶

> 原文:[https://hack aday . com/2017/08/11/super conference-interview-akiba/](https://hackaday.com/2017/08/11/superconference-interview-akiba/)

秋叶位于一个非常有趣的技术和文化的交汇点。他以在深圳从事制造业的经历而闻名，但他还有其他一些独特的方面，我稍后会谈到。他在中国的制造经验远远超出了你可能会想到的电子产品，涵盖了所有可能制造的东西。他在去年的 Hackaday 超级大会上的演讲是一个关于地区、文化和商业方面的速成班。

现在关于其他维度。在采访中，Akiba 和 Sophi 讨论了他有着令人难以置信的独特观点的另外两个领域。首先是他在东京郊外的农村地区成立了一个黑客集体。黑客农场在过去的三四年里一直在疯狂发展。似乎人们来参观，并意识到在该地区租房是如此便宜，他们不能离开。这导致了营地周围的文化繁荣；一个吸引更多游客(通常是那些为团队提供丰厚食物的主厨)并促进集体发展的自动喂食引擎。

他们正在研究该地区农业技术的新应用。其中一个方面是为该地区的稻农提供的水位传感器，他在 Hackaday 上写了很多。野生动物在这里是一个巨大的挑战——显然蜘蛛会利用任何洞或裂缝来织网，这通常会使传感器变得毫无价值。该小组还开始对园艺的“三姐妹”进行实验:玉米、豆类和南瓜，并计划将其作为各种农业自动化的试验台。

虽然在采访的最后只是简单地提到了一下，但 Akiba 也在极端的水平上与可穿戴技术合作。他为[救援队管弦乐队](http://wreckingcreworchestra.com/en/)在服装中加入了灯光和其他互动元素。听到他处理磨损、通信延迟和舞者自己的用户界面的经验总是一种享受。
