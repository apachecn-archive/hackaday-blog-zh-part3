# 黑客日奖参赛作品:热门逻辑

> 原文：<https://hackaday.com/2017/04/15/hackaday-prize-entry-hot-logic/>

几周前，[Yann]在翻垃圾箱时发现了一些有趣的东西。两个真空管，一个 ECC83S 和一个 EL84。这显然是一个当地吉他手的粪便，但[Yann]想知道他是否能从中制造出一些有用的东西。放大器太普通了，所以他选择了真空管计算机。

电子管放大器中常见的五极管和三极管需要大量支持元件，如输出变压器、电子管插座和高压电源。这对于电子管计算机来说有点太复杂了，但是经过一点点搜索之后，[Yann]为他的 MINIVAC 找到了一个更好的选择——超小型真空管。这些需要较少的支持组件，并且可以通过通常的组件供应商以非常合理的价格找到。他今年的黑客日奖的参赛作品是热门逻辑。这是一台计算机——或者至少是计算机部件——由这些管子制成。

这些电子管分别是几个 1ж29б-в管和 6н21б管，一个真空五极管和一个双三极管。加上几个二极管，就足以满足制造一台计算机的要求了。作为一个整洁的小奖励，这些管有很容易满足的要求。1ж29б-в管上的灯丝只需要 1.2 伏。

这些超小型电子管在音响发烧友和 DIY 电子产品的世界里有点不受重视。这有点可惜；这些电子管是有史以来最先进的真空技术。它们是弹道导弹的心脏和大脑，如果你仔细观察，你可以以非常合理的价格买到数百枚。一台真空管计算机需要很多电子管，如果有人能造出一台真空管计算机，那将是扬和他那堆苏联剩余物资。
