# 直接从辐射 4 打印物品

> 原文：<https://hackaday.com/2015/12/20/printing-objects-directly-from-fallout-4/>

《辐射 4》大约一个月前发布，虽然我们没有像贝塞斯达的上一款游戏那样“一箭射中膝盖”,但有足够的机会进行角色扮演，并在 3D 打印机上打印出死亡之爪和 mirelurks。你如何把藏在游戏文件夹里的文件变成一个真实的、打印出来的物体？这其实很简单，T2 和安格斯会告诉你怎么做。

用[贝塞斯达档案提取器](http://www.nexusmods.com/fallout4/mods/78/?)可以很容易地获取*辐射*敌人和物品的文件，尽管这不会给你 3D 打印机可以理解的文件。你将得到一个. NIF 文件，NifSkope 可以将在*余波*档案中找到的文件转换成一个。OBJ 文件任何三维建模程序都可以理解。接下来的一步是。OBJ 文件进入 [Meshmixer](http://www.123dapp.com/meshmixer) 并用 [Netfabb](https://netfabb.azurewebsites.net/) 修复一切。在那之后，它被送到打印机。

[Angus]在 ABS 上打印了他的死亡之爪模型的多个部分，用一点点丙酮把它们粘在一起。这并没有完全按照计划进行；ABS 中的一些污染物在黑色 ABS 上变成了一层白色薄膜。这个问题最终被 XTC-3D 解决了，这是一种每个人都在试验的 3D 打印涂层。

成品是一个纯黄色但完全平滑的 3D 模型，是《辐射 4》中最强悍的敌人之一。剩下唯一要做的就是给模型上色。在这一点上，最好的方法可能是做模型建造者已经做了几十年的事情——喷枪和数百个小瓶的颜料。[Angus]正在开放他的 YouTube 评论以寻求建议，如果你有更好的主意，他正在寻求帮助。
