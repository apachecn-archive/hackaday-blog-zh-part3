# 邪恶热点服装建立了有价值的联系

> 原文：<https://hackaday.com/2017/11/06/evil-hotspot-costume-makes-valuable-connections/>

今年万圣节，[斯科特]打扮成康卡斯特 xfinity 热点人物。有趣，是的，但这里有更深的含义。[Scott]真的是一个活的 PSA，它说明了仅仅根据它们的广播名称就对 WiFi 网络的相对安全性做出假设的危险。

[Scott]本可以用这种设置变得混乱邪恶，但他没有。实际上没人能通过他上网。在“热点”内部有一个 Wi-Fi 适配器和一个运行强制门户的 Pi Zero。它广播默认的“XFINITY”和“XFINITY wifi”ssid，加上一堆其他常见的网络名称。每当有人试图连接，或者更糟，他们的电话自动连接，他们会听到一个悲伤的大号节奏。这是由 Pi 通过中继板控制的多重音效箱提供的。

同时，标记的设备被重定向到一个内部托管的“xfinity”登录页面。任何真正继续输入他们的登录凭证的人都会得到一个经典的恐怖电影《尖叫》样本，而邪恶的热点会悄悄地存储他们的名字和密码，并在电子墨水显示器上显示给所有人看——一个移动的电子墨水[绵羊墙](https://www.wallofsheep.com/pages/wall-of-sheep)。休息后请欣赏演示。

[斯科特]的邪恶热点是由一个巨大的电池供电，可以运行 24 小时。这是我们几年前报道过的风能和太阳能 WAP。
