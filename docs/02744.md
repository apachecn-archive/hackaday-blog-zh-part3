# 用机器人自动化 3D 打印机

> 原文：<https://hackaday.com/2016/06/27/automating-3d-printers-with-robots/>

3D 打印机从来就不是用来生产的。它们不是制造机器，而是原型机。这并不意味着 3D 打印机不能用于制造业，只是很难——你需要有人管理一队机器，或者某种“自动化构建平台”,而这种平台在整整 14 年内都不会被发明出来。

在没有人付费观看打印机打印的情况下，tend.ai 的[Mark]、[Robert]和[James]创造了一种用机械臂管理打印机车队的方法。这是一个机器人手臂，它可以自动监控装满 3D 打印机的机架上的液晶显示器，从床上取下完成的打印件，将零件放入盒子中，然后开始另一次打印。

Tend.ai 从事云机器人业务，并设计了一个系统，该系统可以使用任何机械臂、任何网络摄像头，并为这个机械臂提供后端，以——等等——照顾其他机器。作为演示，它工作得很好。零件从机器上拿起来，放进盒子里，另一轮印刷开始了。

作为云机器人平台的技术演示，没有比这更好的了。作为一种自动化 3D 打印机车队的方式，我只能想知道这个机器人手臂系统如何处理大型平板打印零件。我猜，机器人抓手总是可以被抹刀取代。
