# 机器人使用钢铁侠技术行走

> 原文:[https://hack aday . com/2018/03/26/robot-uses-iron-man-tech-to-walk/](https://hackaday.com/2018/03/26/robot-uses-iron-man-tech-to-walk/)

考虑到我们现在拥有的一系列工具，设计一个双踏板机器人是一项相对简单的任务。任何人都可以从许多开源例子开始。设计一个不会经常摔倒的……这可不简单。这是因为当我们走路时，我们的平衡中心在不断变化，所以在青春期，我们学会了改变身体的重量来保持稳定的平衡中心。当我们到了十几岁的时候，我们大多数人已经掌握了走路的艺术，甚至可以在激烈的运动中保持稳定，就像在许多运动中看到的那样。

问题当然是，如何将这种学习方式传递给一个双踏板机器人？退一步说也不容易。看看中国广东工业大学自动化学院的机器人团队在做什么。他们把一对管道风扇喷气发动机绑在一个双踏板装置的脚上。这样做的目的是让机器人在很长的距离内保持平衡。一般来说，我们看到双踏板机器人“在蛋壳上踮脚”,因为它们需要尽可能保持平衡中心的稳定。通过施加来自脚的推力；他们能够保持重心，即使机器人已经超出了正常的运动范围。
