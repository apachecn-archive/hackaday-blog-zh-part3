# 音乐接近检测宠物碗

> 原文：<https://hackaday.com/2016/08/29/proximity-detection-musical-pet-bowls/>

对于一个制造者来说，一项重要的技能是即兴创作或重新利用现有材料制作新零件的能力。有时候，人们不需要做很多修改就能创造出新的东西，就像罗宾·斯特林和他的音乐宠物碗一样。

最初，它是一个密封的宠物碗，当接近传感器检测到接近的宠物时就会打开。在帮助设计了这个碗之后，[Sterling]有了一点优势，他决定为公司 BBQ 把它改装成特雷门琴/轻竖琴式的乐器。他将三个接近传感器(三个碗中的每一个)的 PWM 输出路由到一个小吉他放大器，将每个传感器的输出调整到不同的频率。尽管[斯特林]练习的时间很短，但效果相当好！

 [https://www.youtube.com/embed/yYU_UKTcz3Q?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/yYU_UKTcz3Q?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



[Sterling]补充说，碗的固件知道哪个接近传感器被触发，这使得这确实是一个简单的模式。从音乐碗到翻新[纸带音乐盒、](http://hackaday.com/2016/07/06/paper-tape-drive-for-a-live-performance-music-box/)或[一个旋律不断变化的完整音乐合奏](http://hackaday.com/2016/07/05/niklas-roys-music-construction-machine/)，制造者和音乐家的团结从未停止令人惊叹。
