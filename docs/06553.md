# 认识一下视频 DJ 机

> 原文：<https://hackaday.com/2017/07/24/meet-the-video-dj-machine/>

你是否曾经想做一名 DJ，但是发现设备既昂贵又吓人？好吧，你的祈祷已经被设计了 nom 2 的[Dror Ayalon]回应了。这是一个开源的音乐混合项目，使用多达 16 个视频剪辑，让你控制你的下一个热门专辑。

你负责一个有 16 个按钮和 4 个旋钮的物理控制面板。每个按钮可以用来打开或关闭一个特定的剪辑，而旋钮控制重复率，音量，速度和每个轨道的播放长度。Arduino 位于按钮下方，负责将信息发送到运行在您的 web 浏览器中的应用程序。浏览器应用程序使用 [NexusUI 库](https://nexus-js.github.io/ui/)来控制音频剪辑的回放，并为整个体验带来活力。

[Dror Ayalon]一直在忙着完善他的项目，有一些他演示它的精彩视频，请查看下面的视频。代码可从 [GitHub](https://github.com/dodiku/the_video_machine_v2) 下载，BOM 可在 [Hackaday 获得。](https://hackaday.io/project/25975-nomnom-the-video-machine) io 项目页面。web 应用程序可以使用 electron 移植到桌面应用程序，并且可以为未来版本的控制器设计 PCB。

目前，看到硬件和软件以如此和谐的方式结合在一起是不可思议的。这可能是美好事物的开始，但如果你只是想找个方法来惹恼邻居，试试 Midi 音乐警报器吧。
