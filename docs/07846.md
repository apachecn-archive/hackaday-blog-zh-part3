# 用这个红色播放按钮奖励来计算 YouTube 订户

> 原文：<https://hackaday.com/2017/12/02/count-youtube-subscribers-with-this-red-play-button-award/>

专业的 YouTubers 用户的数量决定了他们的生死。对我们来说，这似乎是一种残酷的谋生方式，但对每个人来说都是如此。尽管如此，如果你要做，你最好做对，并且记录你如何使用[这个 Play Button Award 订户计数器](https://hackaday.io/project/28357-led-youtube-live-subscriber-counter)可能是有意义的。或者它会让你发疯。

YouTube[ibuynewstuff]已经达到了号称的 100，000 用户大关，这是获得银播放按钮奖所需的人数。可悲的是，10 万美元是获得 YouTube 关注的最低要求，而等待奖项到来数月的故事并不罕见。[ibuynewstuff]通过 3D 打印他自己的临时播放按钮徽章来解决这个问题。[ibuynewstuff]安装在一个带有 ESP8266 和扩散器后面的 8 x 80 LED 显示器的相框上，可以记录他在 100 万次订阅中获得黄金播放按钮奖的进展。希望他的银牌能在那之前到达。

想复制这一点，但更希望有一个比塑料播放按钮更永久的东西？试试[铸造自己的铜弹按钮](http://hackaday.com/2017/06/14/celebrating-a-subscriber-milestone-with-a-copper-youtube-play-button/)奖励。
