# 为儿童设计的外骨骼

> 原文：<https://hackaday.com/2016/08/25/exoskeleton-designed-for-children/>

外骨骼确实令人敬畏，允许人类完成超出正常能力的力量壮举。这项技术的未来是光明的——不仅仅是工业和军事应用，尤其是在治疗和康复方面。通常，人们会想到四肢失去功能的成年人，但在这个由西班牙国家研究委员会(CSIC)开发的外骨骼案例中，患有脊髓性肌萎缩症的儿童有机会过上积极的生活。

为儿童设计假肢可能很困难，因为他们在不断成长，CSIC 的设计是可伸缩的，以适应 3-14 岁之间的病人。每条腿上的五个电机通过传感器适应患者的个体症状，这些传感器检测儿童的移动意图并模拟他们的自然行走步态。

 [https://www.youtube.com/embed/aPkkjOMB-J4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/aPkkjOMB-J4?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



CSIC 强调了这种独特外骨骼的治疗效果，引用了脊髓性肌萎缩症的心理方面，这对儿童的生活有进一步的退化影响。它目前正在马德里和巴塞罗那进行临床前实验室试验，以测试该设备的功效。[结合 3D 打印的奇迹](http://hackaday.com/2012/08/12/3d-printed-exoskeleton-helps-this-little-girl-develop-more-normal-body-function/)，先进的儿童和成人外骨骼假肢正在迅速接近现实。
