# 用 Pi 打造更好的游戏机

> 原文:[https://hack aday . com/2016/03/24/building-a-better-game-boy-with-a-pi/](https://hackaday.com/2016/03/24/building-a-better-game-boy-with-a-pi/)

到目前为止，最值得收藏的游戏男孩是游戏男孩微。这个小小的游戏机小到可以丢在你的口袋里。它只能玩 Game Boy Advance 游戏，屏幕很小，但看看易贝的价格就知道了:这是少数几个回头看可以被视为投资的消费电子产品之一。

Game Boy Micro 的流行，Raspberry Pi 模仿旧游戏机的能力，以及 Raspberry Pi Zero 的推出只能意味着一件事。这是 PiGrrl Zero ，一款可以玩所有复古游戏的现代掌机。

PiGRRL Zero 的设计目标非常简单:一个 2.2 英寸 320×240 的显示屏，一个 d-pad，四个面部按钮和两个肩部按钮。有一个大电池，音频输出和 3D 打印的外壳。如果不是为 PiGRRL Zero 设计的 PCB，这可能有些不起眼。它被设计成直接焊接到 Raspberry Pi Zero 上，利用了这个微型单板计算机的大部分无组件背面。

有了这个 PCB，Pi Zero 就变成了一个微型电池供电的计算机，运行所有经典作品的仿真。NES，SNES，世嘉，当然还有 Game Boy Advance 游戏都可以在这款设备上玩，而且价格比一款全新的游戏 Boy Micro 要低得多。我们的评委认为成为圆周率为零竞赛的获胜者之一很酷。看看吧！
