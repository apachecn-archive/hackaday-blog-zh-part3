# 俄罗斯大学制造的机器人蟑螂

> 原文：<https://hackaday.com/2015/09/28/robotic-cockroach-built-by-russian-university/>

它的名字叫*黑鲨*，简称死亡之头。这个由伊曼纽尔·康德·波罗的海联邦大学为一个“模糊的”俄罗斯组织建造的曾经秘密的项目现在被曝光了。这个[小家伙](http://www.popsci.com/russia-shows-off-robotic-cockroach)有 20 分钟的电池寿命，可以携带 10 克的有效载荷。当你想把相机偷偷放入难以到达的地方时，这就派上了用场。其他的要求是让它看起来和表现得像一只真正的活昆虫。

考虑到它是在仅仅七个月的时间里从零开始建造的，这是一个令人印象深刻的项目。它复杂的齿轮和其他机械功能需要熟练的制表师来制作。或者，你可以控制活的昆虫，比如控制蟑螂的大脑，或者把一些 T2 的无线电控制器连接到一只活的甲虫上。但是制造机器昆虫仍然很酷。

请务必查看制作精良的视频，其中详细介绍了该项目的一些构建过程。
