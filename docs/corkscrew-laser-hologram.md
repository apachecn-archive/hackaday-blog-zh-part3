# 螺旋激光“全息图”

> 原文：<https://hackaday.com/2015/12/08/corkscrew-laser-hologram/>

如果你看了很多科幻小说，你就会知道在未来，有很多 3D 全息显示器。从莉亚公主的求救电话到《星际迷航》的全息甲板，不缺少可以制作逼真图像的计算机。它可能达不到 holodeck 的标准，但[freedscript] [使用一根筷子、一个马达、一些纸和一束激光为 Arduino](https://www.youtube.com/watch?v=I82BbrUtopU) 创建了一个 3D 显示器。当然，它实际上不是全息图，但也不是你在电视上看到的一半东西(星际迷航的全息人物对于驻波来说是令人不安的固体)。该显示器是一种[立体显示器](https://en.wikipedia.org/wiki/Volumetric_display)。

这个想法看似简单。纸质开瓶器旋转，激光点~~垂直于~~平行于旋转轴，可以创建一个点，该点似乎位于开瓶器外周所包围的体积切片中的任何位置。你可以在下面的视频中看到，这种设备很容易制造，尽管结果并不令人满意。如果开瓶器太大，边缘会移动得太快。此外，你需要很多激光器，包括一个用来感应开瓶器索引点的激光器。

尽管如此，看到 Arduino 驱动 3D 显示器还是令人印象深刻，如果你想尝试，这个系统也足够简单。我们之前已经研究过其他 3D 显示器的光学系统，包括使用激光的[和使用电脑投影仪的](http://hackaday.com/2014/11/06/how-a-real-3d-display-works/)。也许莱娅公主的求救电话很快就会成为现实。
