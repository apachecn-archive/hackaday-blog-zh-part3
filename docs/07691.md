# 开源水下滑翔机赢得 2017 年 Hackaday 奖

> 原文:[https://hackaday . com/2017/11/11/开源-水下-滑翔机-wins-2017-hackaday-prize/](https://hackaday.com/2017/11/11/open-source-underwater-glider-wins-2017-hackaday-prize/)

[开源水下滑翔机](https://hackaday.io/project/20458-open-source-underwater-glider)刚刚被评为 2017 年 Hackaday 奖的大奖得主。作为 Hackaday 奖的最高获奖者，开源水下滑翔机将获得 5 万美元，并在 Hackaday 奖的最后八个月内完成超过 25 万美元的现金奖励。

在 [2017 黑客日奖](https://hackaday.io/prize)期间，一千多件参赛作品响应了*打造有意义的东西*的号召。全球各地的硬件创造者在参赛期间参加了五项挑战:构建你的概念、有用物联网、翅膀-轮子-行走者、辅助技术和任何事情。下面您将看到前五名的完成者，以及 30，000 美元最佳产品奖的获得者。

![](../Images/6f5a4156c83e895c23e90ca23e61cdfc.png)

### [开源水下滑翔机](https://hackaday.io/project/20458-open-source-underwater-glider)

大奖得主(5 万美元):开源水下滑翔机是一种 AUV(自主水下航行器)，能够对海底环境进行长期水下探索。大多数 AUV 在动力和航程上都受到限制，开源水下滑翔机不使用主动推进，如推进器或螺旋桨。这种潜水器滑行，扩展了它正在执行的任何任务的范围和能力。

开源水下滑翔机是由现成的硬件制成的，允许任何人建造自己的这种非常强大的水下无人机。长达一周的延长任务是可能的，之后滑翔机将自动回家。

![](../Images/4c81afbd94190d28c09a68256ef101d1.png)

### [互联健康:开源物联网病人监护仪](https://hackaday.io/project/25380)

Second Place ($20,000): The Connected Health project aims to bring vital sign monitoring to the masses with a simple, inexpensive unit built around commodity hardware. This monitoring system is connected to the Internet, which enables remote patient monitoring.

![](../Images/a771794e68d40ab9d0401eec3b1a8b6b.png)

### [静脉检测辅助系统](https://hackaday.io/project/26158)

第三名(15000 美元):这种用于静脉检测的辅助系统使用现成的组件和近红外成像来检测皮肤下的静脉。该系统使用 Raspberry Pi 和摄像头模块或经过修改的网络摄像头，但与专业解决方案一样可靠，而专业解决方案的成本是该团队原型的几十倍。

![](../Images/7462a760a677df43a6ba54a6251b13d1.png)

### [自适应吉他](https://hackaday.io/project/26088)

Fourth Place ($10,000): The Adaptive Guitar is an electromechanical system designed to allow disabled musicians to play the guitar with one hand (and a foot). This system strums the strings of a guitar while the musician frets each string.

![](../Images/82c839f38c551620fb35eb2e09e04391.png)

### [Tipo:盲文智能手机键盘](https://hackaday.io/project/21175)

Fifth Place ($5,000): Tipo is effectively a Braille USB keyboard designed for smartphones. The advent of touchscreen-only phones has unfortunately left the visually impaired without a modern phone. Tipo allows for physical interaction with modern smartphones.

![](../Images/e676205dffa29b2f727e668276be38ab.png)

### [最佳产品奖:Tipo:盲文智能手机键盘](https://hackaday.io/project/21175-tipo-braille-smartphone-keypad)

最佳产品的获奖者是 Tipo:盲文智能手机键盘。Tipo 是解决现代智能手机越来越没有按钮的问题的解决方案。只有触摸屏的手机不能被视障人士使用，Tipo 在任何手机的背面都添加了盲文键盘。它实际上是一个 USB 键盘，专为盲文输入而设计，可以连接到任何手机的背面。

最佳产品竞赛与五轮挑战赛同时进行，要求参赛者超越原型，展望用户需求、制造和所有进入市场的因素。通过赢得最佳产品竞赛，Tipo 的创造者将改进他们的设计，改善他们的机械构造，开始研究 injecton 成型，并将他们的 3D 打印原型变成有能力改变生活的真正产品。
