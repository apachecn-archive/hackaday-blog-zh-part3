# 为台锯安全添加劈刀

> 原文:[https://hacka day . com/2017/06/22/添加一把用于台锯安全的劈刀/](https://hackaday.com/2017/06/22/adding-a-riving-knife-for-table-saw-safety/)

劈刀到底是什么？这是唯一能让你从糟糕的一天中解脱出来的东西。但是如果你的台锯没有配备，不要担心——只需一点资金，你就可以给几乎任何台锯添加劈刀。

对于那些从未在台锯上体验过[反冲的人，我们可以向你保证，至少它会让你的心怦怦直跳。在最糟糕的情况下，它会把你的手吸进旋转的刀片，让你的手指飞起来，或者也许会把一块木头嵌入你的胸部或前额。劈刀通过防止原料在旋转出工作台时接触刀片来减轻这种灾难。像[Craft Andu]的小牧田(little Makita)这样的承包商台锯经常被剥夺这样的细节，所以他开始增加一个。一把合适的劈刀的基本特征是与刀刃宽度相同，紧紧围绕刀刃，随着刀刃上升和下降，并且不超过刀刃顶部。[Craft Andu]用他的 DIY 刀击中所有这些点，结果是额外的安全，没有任何不便。](https://hackaday.com/2013/10/23/table-saw-kickback-video-ends-badly/)

遭受改变一生的伤害只需要几毫秒，所以在那里要注意安全。即使你正在[建造自己的台锯](https://hackaday.com/2015/09/17/circular-saw-innovative-fence-unique-diy-table-saw/)，这也是你欠自己的。
