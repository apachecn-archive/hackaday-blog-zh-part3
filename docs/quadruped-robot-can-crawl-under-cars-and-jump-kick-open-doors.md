# 四足机器人可以在汽车下爬行，还可以跳起来踢开车门

> 原文：<https://hackaday.com/2017/04/01/quadruped-robot-can-crawl-under-cars-and-jump-kick-open-doors/>

正如他们所说，轮子是一项革命性的发明，但回归基础有时会开辟新的道路。用腿穿越地形的机器人越来越多，最著名的是波士顿动力公司的大狗系列机器人，而[ [Ghost Robotics]的 Minitaur 四足机器人](http://www.popsci.com/minitaur-robot-with-legs)旨在跟上步伐。

[Ghost Robotics]的创始人之一[Gavin Knneally]指出，协调是开发四足机器人时要克服的主要问题之一；Minitaur 的设计是为了爬过特别恶劣的地形，它断断续续的步伐可以爬上陡峭的山坡、楼梯、穿过冰面等等。它的腿还允许它调整自己的高度——视频显示，它快步走向一辆汽车，蹲下，然后开始轻松地在下面摇摇摆摆。

[幽灵机器人]的目标是制造微型机器人，小到八英寸。这些机器人计划作为传感器平台，主要市场是军事，随后不久用于搜索和救援。一个能爬楼梯和踢开办公室门的机器人将会非常有用。

不仅仅是大大小小的公司在制造这些机器人——我们已经展示了[制造商制造的](http://hackaday.com/2014/10/22/baby-quadruped-robot-learning-to-walk/) [四足机器人](http://hackaday.com/2016/01/29/adorable-quadruped-robot-hops-and-walks/)和[同伴](http://hackaday.com/2013/10/18/quadruped-robot-thinks-its-a-puppy/)有一段时间了——看看它们吧！
