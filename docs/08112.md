# 杯垫烤面包机

> 原文:[https://hackaday.com/2017/12/28/coasty-the-coaster-toaster/](https://hackaday.com/2017/12/28/coasty-the-coaster-toaster/)

无论你是否觉得有必要激光切割定制杯垫，你都必须对杯垫的创意留下深刻印象。

他们说正义是盲目的，虽然我们不能在你下次出庭时向你承诺任何事情，但我们至少可以充满信心地说，我们不是那种仅仅因为用橡皮筋和积极的氛围就拒绝好黑客的人。如果它有效，它就有效，不管它看起来像什么。话虽如此，我们还是被这个特殊项目的完成程度震惊了。

由[Bart Dring] 设计和建造的 Coasty 是那些将黑客提升到看起来像商业产品的项目之一。它包括一个普通的纸浆板杯垫，激光切割任何你想要的设计。它大小合适，部件也合适，因为这是 Coasty 的宗旨。它在杯垫上有一个进给槽，并在激光切割过程中以此作为一个轴，激光的左右移动作为另一个轴。这种方法整体占地面积更小，意味着您在正常操作时无需打开保护外壳。

Coasty 最引人注目的元素之一是有多少硬件是 3D 打印的。如果它不是一个马达，光滑的杆，或其他机械部件，它会被打印出来。我们习惯于将 3D 打印零件视为支架或底座，但很少看到像这样打印的整个机箱。这不仅需要大量的预先考虑和设计，而且打印时间本身也非常令人望而却步。

但是通过设计和印刷大部分的 Coasty，它真的给了它一个专业的外观，如果它是一捆铝挤压件，这将很难实现。

Coasty 的背面有一个暴露的 PCB“主板”，上面有令人眼花缭乱的插件板阵列。步进驱动器、蓝牙无线电和激光电源等硬件是独立的模块，便于维护和开发。主板上还集成了一些简洁的硬件功能，如用于检测杯垫边缘的红外传感器。

印刷过滤器是一个特别好的触摸。Coasty 含有商业上可获得的用于家庭空气过滤器的碳布碎片，能够减少用 3W 450nm 激光爆破纸板时不可避免产生的烟雾。

对一些黑客来说，定制杯垫是一项严肃的业务。[这并不是第一次有人](https://hackaday.com/2016/09/24/pressure-plate-led-coasters/)全力以赴[做出他们的标记，同时又阻止标记](https://hackaday.com/2017/05/22/evezor-robotic-arm-engraves-400-coasters/)。
