# 那不是月亮！那是一个虚拟助手

> 原文：<https://hackaday.com/2017/01/24/thats-no-moon-thats-a-virtual-assistant/>

[Wisecracker]喜欢亚马逊 Echo Dot 的工作方式，但他不喜欢它们的声音或它们与冰球的相似程度。不过，通过一点 3D 打印，他把这个点变成了一颗可信的死星。我们猜测，这听起来不太友好，所以他称之为[亚历克斯之星](http://www.instructables.com/id/The-Alex-Star-SpeakerEnclosure-for-the-Echo-Dot/)。

它的工作原理是死星的“超级激光”——由一个控制台操作的武器，看起来可疑地像一些演播室视频设备——恰好与一个两英寸扬声器的大小和形状差不多。[Wisecracker]增加了一个插槽，让声音从第二个扬声器发出。你可以在下面的视频中看到这个东西。

3D 文件在 [Thingiverse](https://www.thingiverse.com/thing:2040264) 上。这其中的电子部分相当琐碎。一对亚马逊音箱用吹风机轻松拆下来软化热胶。[Wisecracker]使用半透明塑料，然后画了大部分。他没有给底座上漆，这样来自回声点的蓝色 LED 就可以透过来了。

如果你对《星球大战》来说太老了，你可能会更喜欢 [HAL 9000 版本](https://hackaday.com/2016/11/15/put-an-honest-face-on-alexa-with-this-hal-9000-build/)。或者，如果你不想让[亚马逊监听](https://hackaday.com/2016/12/28/police-want-alexa-data-people-begin-to-realize-its-listening/)，你可以在不在线的情况下做[更简单的事情](https://hackaday.com/2016/12/11/arduino-clock-is-hal-1000/)。
