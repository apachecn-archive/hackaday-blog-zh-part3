# 黑客攻击中国官方媒体

> 原文：<https://hackaday.com/2015/11/18/hacking-chinese-state-media/>

不久前，几位来自中国的记者参观了位于维也纳的 Metalab hackerspace。他们想做一个关于“私人实验室”和“创客空间”的报道，尽管 Metalab *hackerspace 的居民反对。*显然，在中国中央电视台(是的，它叫 CCTV)提到“黑客”是一大禁忌。

想要向至少几个中国人传递信息，黑客空间的成员必须横向思考。Metalab 成员[amir]想出了一种可以打印在 t 恤上的数据编码方法。在 Metalab 成员的所有采访中，这些明亮、多彩的正方形都带有类似“自由西藏！”、“记住 1989 年天安门”和“质疑政府”。不要相信宣传”
