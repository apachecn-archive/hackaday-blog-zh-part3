# 21 世纪的点唱机融合了树莓派，Sonos，二维码

> 原文:[https://hack aday . com/2018/02/19/a-jukebox-for-the-21 世纪-kit-blends-raspberry-pi-sonos-QR-codes/](https://hackaday.com/2018/02/19/a-jukebox-for-the-21st-century-kit-blends-raspberry-pi-sonos-qr-codes/)

当(克里斯·坎贝尔)的孩子们想在晚餐时播放一张专辑的背景音乐时，切换他家 Sonos 音响系统的输出可能对他们刚刚掌握的技术来说太复杂了。这让他开始考虑使用儿童友好的输入，这样他们就可以探索他的音乐收藏。将二维码、一些乐高积木和一些工艺品混合在一起，一个适合孩子的二维码阅读器媒体控制器问世了！

利用一台 Raspberry Pi 3 Model B 和一台便宜的相机，[Campbell]编写了一些代码来处理二维码的生成和读取——尽管他在另一台计算机上运行媒体服务器以保持快速响应时间。一旦[坎贝尔]有了二维码，他就把它们打印出来，并让他的孩子参与切割和粘贴双面卡片。附加卡可以访问不同的功能，如启动播放列表队列、切换输出频道、播放整张专辑等。提示自发的舞会！
