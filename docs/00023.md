# Hackaday 奖半决赛:节约用水和消除白日梦

> 原文：<https://hackaday.com/2015/09/06/hackaday-prize-semifinalist-conserving-water-and-eliminating-daydreaming/>

我们认为，在淋浴时做白日梦会浪费很多水。我们再次假设，有一天在淋浴时做白日梦时，他想出了在淋浴时继续工作的主意。[淋浴节拍器](https://hackaday.io/project/7199-shower-metronome)就这样诞生了，【mulcmu】的 Hackaday 奖参赛作品。

淋浴节拍器的目标是双重的。首先，它减少了淋浴的用水量。其次，它让用户按时工作。淋浴节拍器通过连接到淋浴框架或浴帘的小型微控制器提供的小型音频嘟嘟声来实现这一点。

该设备的核心是一个 MSP430 微控制器，几个硬币电池和一个霍尔效应传感器，它可以打开设备，就像一个磁性的门或窗报警器。微控制器的选择非常适合该应用；MSP430 功耗极低，在低功耗模式下仅消耗 1uA。这意味着淋浴节拍器将持续一段时间，当每天只使用几分钟。
