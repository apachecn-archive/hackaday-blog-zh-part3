# Hovalin:开源 3D 打印小提琴听起来很棒

> 原文：<https://hackaday.com/2016/01/02/the-hovalin-open-source-3d-printed-violin-sounds-great/>

[Matt 和 Kaitlin Hova]创造了[Hova Lin，一把开源的 3D 打印小提琴](http://www.hovalin.com/)。是的，以前也有 3D 打印的乐器，但[霍瓦斯]创造了革命性的东西——一种 3D 打印的声学乐器，听起来出奇的好。Hovalin 是一把全尺寸的小提琴，可以在桌面大小的 3D 打印机上打印。霍瓦人提到了终极制造者 2、Makerbot 复制器 2(或者许多克隆体中的一个)作为例子。颈部是一体的，而身体是打印在 3 个部分。Hovalin 也是开源的，在知识共享署名非商业共享许可下发布。

纯 PLA 琴颈不足以抵抗琴弦的张力，所以[Hovas]增加了两根碳纤维桁架杆。少数其他组件，如调谐器，当然字符串，也需要购买。总价比亚马逊上 60 美元的小提琴略高，但我们打赌霍瓦林琴比任何便宜的乐器质量都好。

霍瓦林是 10 月份释放的。野外已经有一些建造日志了，比如【乳化油】的[这个。像任何好的工程项目一样，Hovalin 是一项正在进行的工作。【Matt 和 Kaitlin】已经发布了 1.0.1 版本，2.0 版本即将推出。不过，耳听为虚，所以点击休息时间，听听[Kaitlin]演奏她的乐器。](https://imgur.com/a/ddW2S)
