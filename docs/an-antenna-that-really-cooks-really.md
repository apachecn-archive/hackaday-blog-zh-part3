# 一根真正会做饭的天线——真的

> 原文：<https://hackaday.com/2017/06/19/an-antenna-that-really-cooks-really/>

使用 HackRF 板和能够接收 NOAA 19 卫星的 LNA 设置接收器。当然，接收器需要天线，他用一个锅做了一个。天线并不理想——至少在室内——但它确实有用。他希望调整它以获得更好的接收效果。你可以在下面看到天线和接收效果的视频。

该卫星正在发送高分辨率图像传输(HRPT)数据，其图像质量高于自动图像传输(APT)。APT 的频率为 137 MHz，但 HRPT 的频率为 1698 MHz，通常需要更好的接收机和天线系统。

如果你决定自己做一个锅形天线，我们会小心钻不粘涂层。聚合物烟雾热需要聚四氟乙烯加热到近 600 度，所以光加工可能不会造成问题，但话说回来，为什么要冒险呢？在通风条件下工作，防止吸入灰尘。

通常，卫星钻机[要复杂一点](https://hackaday.com/2017/01/02/junkyard-dish-mount-tracks-weather-satellites/)。另一方面，锅状天线也还不能解码图像。在过去，我们已经看到了各种各样的 APT 项目。每个人都见过品客罐头做成的天线。我们甚至见过用勺子和[特百惠塑料](https://hackaday.com/2012/11/27/cheap-biquad-antenna-extends-lan-between-apartments/)做成的[，所以我们猜测一个锅并不那么令人惊讶。](https://hackaday.com/2015/01/19/simple-directional-wifi-antenna/)
