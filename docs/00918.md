# 三周三元二元表

> 原文:[https://hack aday . com/2015/12/10/the-three-week-three-dollar-binary-watch/](https://hackaday.com/2015/12/10/the-three-week-three-dollar-binary-watch/)

三周后有一个制造者集会，你的小组想设计并制作一个二元手表送给与会者。你没有太多时间，你的预算是每块手表 3 美元。你是做什么的？如果你是[Parker@Macrofab]，你想出一个计划，买一些零件，然后[开始原型制作](https://macrofab.com/blog/designing-and-building-the-macrowatch/)。

[Parker]选择 PIC16F527 是因为它有足够的 I/O，而且价格便宜。一个便宜的水晶和一些杂七杂八的分立元件完善了材料清单。一些便宜的静电放电带可以为一个乐队服务。他用 PICDEM 板做了原型，并立即遇到了 PIC 程序员的麻烦:模拟比较器压倒了数字 I/O 引脚。扫清了这一障碍后，[Parker]完成了剩下的设计原型，并在 Eagle 中设计了一块板子。

一些[帕克的]优化(即削减成本的举措)并不完美，所以他不得不在最后一刻改变一些赛道。更糟糕的是，ESD 带在海关被挂起，需要在最后一刻更换到不同的带。这打破了 3 美元的预算，但这是一场战争。

最终，[帕克]有了一块实用的手表送人。它可能并不完美，但正如你在视频中看到的，它表现得很好，尤其是考虑到时间很短。它不会取代自制的智能手表，也不会被 T2 解码秘密信息，但这些都不需要 3 美元。
