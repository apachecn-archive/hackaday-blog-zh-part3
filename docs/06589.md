# 自主船在公海航行

> 原文：<https://hackaday.com/2017/07/30/autonomous-boat-sails-the-high-seas/>

随着人口的持续增长和工业总量的增加，地球上几乎没有一个地方比海洋更能感受到这种活动的负担。无论是温度变化，氧气或二氧化碳含量，还是其他特征，对海洋的研究将继续是一项正在进行的科学努力。然而，一个主要问题是海洋到底有多大。深入研究它们需要机器人，因此[迈克]创造了一艘自主船。

这艘船被设计成部分 3D 打印，使任何人都可以很容易地使用正常尺寸的打印机。这艘船使用 APM 自动驾驶系统和漫游者固件使其完全自主。航路点可以被编入程序，船只将会沿着它的下一个目的地前进，执行它被指示的任何任务。计算机是基于电潜泵模块，该船有一个大尺寸的有效载荷舱。

虽然这艘船的大小可能会限制它在短期内穿越太平洋的能力，但它是其他水体的一个很好的平台，也可能是未来可能有业余爱好者支持的更大型远洋船舶的一个组成部分。事实上，在公海航行的无动力船只已经成为现实。
