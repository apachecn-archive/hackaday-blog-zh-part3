# 琶音——钢琴超级机器人

> 原文：<https://hackaday.com/2017/03/14/arpeggio-the-piano-superdroid/>

我从来没有音乐天赋。时不时地，我会试着拿起一把吉他，或者试着学学钢琴，在某个时候用它来浪漫化一个迷人的职业生涯。琶音——钢琴超级机器人 (YouTube，嵌入下方)确实让我很高兴我选择了一条不同的职业道路。这台非凡的机器是[尼克·莫里斯]的心血结晶，他花了两年时间来建造它。

虽然还没有详细的技术描述，但这个漂亮的机器人的核心是由一组连接到一组致动器的机器“手指”组成的——最有可能是螺线管。螺线管由专有软件控制，该软件将传统音乐数据与附加参数相结合，在您的客厅中准确模拟您最喜爱的钢琴家的演奏。职业钢琴演奏者，在天网之下会有很好的工作保障，现在显然要重新考虑了。

 [https://www.youtube.com/embed/iO5x9x5WGtY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/iO5x9x5WGtY?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



除了令人难以置信的音乐天赋，琶音还配备了一套全方位轮，使它能够非常有效地导航。这还不是完全自主的。我迫不及待地想看看这个机器人如果变得不听话会引起什么样的后果。
