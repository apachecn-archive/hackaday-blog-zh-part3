# 垃圾箱冰箱警报保证牛排安全

> 原文:[https://hack aday . com/2017/10/29/junk box-冷冻室-警报-保持-牛排-安全/](https://hackaday.com/2017/10/29/junkbox-freezer-alarm-keeps-steaks-safe/)

一个装满食物的冰箱可能是一件好事，但也是一场即将发生的灾难。根据你的口味，那里可能有价值数百美元的食物，在它和垃圾填埋场之间唯一的东西是不间断的电力供应。把冰箱放在偏僻的地方，你的食物会面临更大的风险。

降低这种风险是这个垃圾箱电源故障报警器的工作。[德里克]的冰柜在车库里，那里的 GFCI 经销店是由代码授权的。我们之前已经介绍过[电路保护](http://hackaday.com/2016/10/19/tripping-out-a-field-guide-to-circuit-protection/)，包括 GFCIs，虽然它们可以拯救生命，但它们也可能意外跳闸，让你损失牛排。[Derek]根据流向冷冻库的电流制造了一个简单的警报。传感器是一个自制的电流互感器，由一个分裂的铁氧体磁芯和一些电磁线制成，几个运算放大器和一个 555 定时器组成检测和报警部分。这些都是垃圾箱里的东西——看看 1983 年的马洛里·索纳特！

诚然，分支电路断电可能是冷冻机不太可能出现的故障模式之一，但这些原理普遍适用，值得了解。向[德里克]脱帽致敬，因为他避开了微控制器，开创了这个老派。并不是说[物联网冰箱和冰柜报警器](https://hackaday.com/2015/08/23/fight-frost-with-an-internet-of-things-fridge-alarm/)有什么问题。
