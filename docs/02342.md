# 是时候最终弄清楚如何使用 KiCAD 了

> 原文：<https://hackaday.com/2016/06/04/its-time-to-finally-figure-out-how-to-use-kicad/>

KiCAD 最近取得了巨大的进步，尤其是因为 CERN 几乎完全使用它。然而，虽然许多东西是相同的，但它们与我们的常规 CAD 包有足够多的不同，以至于很难开始使用新套件。

[Chris Gammell]经营着 Contextual Electronics，这是一个在线学徒计划，从概念到组装电子产品，涵盖两者之间的所有内容。要上这门课，你只需支付象征性的费用，但[克里斯]上传了一个非常精彩的十集视频系列，是在最后一轮课程中制作的，你可以免费观看。这些视频介绍了 KiCAD 的基础知识，并触及了设计和制造电子产品时需要考虑的要点。

[克里斯]选择的项目是一个简单的电路，用 555 闪烁 LED。第一个视频包括导航 KiCAD 的元件原理图编辑器和库系统。接下来是创建电路原理图和元件示意图。[Chris]介绍了 PCB 布局、Gerber 文件的生成，以及最终从 OSH Park 订购设计，OSH Park 是我们熟悉并喜爱的紫色电路板供应商。该系列以在 [LTSpice 中模拟电路、](http://hackaday.com/2016/02/26/adding-spice-to-your-workbench/)订购器件、最后焊接和调试电路板结束。如果一切正常，你现在应该有一个闪烁的 LED。

如果夏日灿烂的阳光灼伤了你娇嫩的皮肤，而你宁愿被锁在焊料烟雾中，现在就把这一点加入你的观察清单吧！
