# 用触摸屏升级你的 Mac，只需一美元

> 原文：<https://hackaday.com/2018/04/05/upgrade-your-mac-with-a-touchscreen-for-only-a-dollar/>

想象一下给 Mac 笔记本电脑加上触摸屏有多难。你在想又贵又难，对吧？(阿尼什)和他的朋友们*怎么可能*设法[只用一美元](https://www.anishathalye.com/2018/04/03/macbook-touchscreen/)就把他们的 Mac 升级成带触摸屏的呢？那似乎不可能*。*

 *当然，诀窍在于软件。通过在机器的网络摄像头上安装一个小镜子，使用硬卡、热熔胶和门铰链。通过观察屏幕并判断手指的图像是否接触到了屏幕上的图像，一个非常简单的触摸屏就可以被创造出来，而且它只需要花费一美元的承诺也就变成了现实。我们必须向他们致敬，因为他们提出了这样一个优雅的解决方案。

他们有一个视频，我们把它放在休息时间下面，展示了一些简单的界面应用。肯定比更传统的转换少得多。
