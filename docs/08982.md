# 动画中的剪纸和边角

> 原文:[https://hack aday . com/2018/03/29/剪纸与动画中的边角/](https://hackaday.com/2018/03/29/cutting-paper-and-corners-in-animation/)

削减每一个角落可能会导致一些劣质的项目，但是[特瑞·吉列姆]向我们展示了[削减正确的角落](https://www.youtube.com/watch?v=xs7WaL44_Iw)产生令人难忘的动画时，混合了适量的古怪的想象力。巨蟒剧团(Monty Python)的《飞行马戏团》(Flying Circus)的标志性动画风格是[Terry]的艺术工艺和尽可能少的工作的混合。休息之后你可以看。

对[Terry]来说，剪切动画是他所知道的传达一个想法、一个笑话或一个故事的最快、最简单的方式。凭借他的声乐曲目，甚至连音效都可以在地下室的录音室里制作出来。有时，他自己制作艺术品，有时他依靠杂志或印刷媒体。这两种资源都有大量的数字对应物，对动画师有利也有弊。

剪切动画有局限性，如不稳定的运动和签名的背景纸外观，但这并没有阻止南方公园。可以使用纹理和渐变，不像传统动画利用简化的调色板，所以你可以选择你的毒药。

如果你的故事或想法因为无法表达而受阻，也许它需要一个正确方向的剪切动画，说明你的 2018 年[hack aday 奖](https://hackaday.io/contest/47173-the-2018-hackaday-prize)提交物也无妨。在技术光谱的另一端，我们有一个用 3D 打印物体制作的[动画。](https://hackaday.com/2016/09/29/3d-printing-a-stop-motion-animation/)
