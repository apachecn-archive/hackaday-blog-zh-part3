# 我们挖这个乐高挖掘机转换

> 原文:[https://hack aday . com/2017/07/04/we-dig-this-Lego-挖掘机-转换/](https://hackaday.com/2017/07/04/we-dig-this-lego-excavator-conversion/)

[Frank]非常幸运地获得了一套斗轮挖掘机乐高玩具作为生日礼物，我们不会说谎，我们很嫉妒。但是，开箱即用，套件有些受限；只有一个马达来驱动整个机器，而且不能完全遥控。但是不要担心——[Frank][着手改变这种](http://projekte.bummels-welt.de/category/schaufelradbagger?order=asc) ( [谷歌翻译](https://translate.google.com/translate?sl=auto&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fprojekte.bummels-welt.de%2Fcategory%2Fschaufelradbagger%3Forder%3Dasc&edit-text=))。

构建的第一部分是添加马达来控制挖掘机的不同功能。两条履带各增加了一个马达，使机器可以前进、后退和转弯。增加了两个电机来提升和降低挖掘铲斗，并旋转塔。最后，原来的马达留在原地转动传送带。

完成后，[Frank]使用 Raspberry Pi 3 控制所有硬件，确保将新的电子设备放在乐高中，以获得原始外观。Raspberry Pi 可能很难简单地控制几个电机，但它让[Frank]可以快速轻松地通过蓝牙实现 Wiimote 控制器。你可以在他最近的更新中查看几个[演示视频。](http://projekte.bummels-welt.de/zuletzt-es-funktioniert)

这是一个伟大的项目，我们希望看到树莓派通过控制互联网得到很好的利用，这样我们就可以在午休时挖沙子了。我们以前见过一些很棒的乐高黑客，比如[这种修改廉价齿轮马达来与乐高零件一起工作的方法](http://hackaday.com/2017/06/22/converting-a-robotic-motor-for-lego-blocks/)。
