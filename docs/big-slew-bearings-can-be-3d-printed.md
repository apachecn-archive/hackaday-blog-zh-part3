# 大型回转轴承可以 3D 打印

> 原文：<https://hackaday.com/2017/06/12/big-slew-bearings-can-be-3d-printed/>

想想不起眼的滚珠轴承。无处不在的、有用的、目前以烦躁不安的形式困扰着全世界的老师。滚珠轴承不容易 3D 打印。打印一个好的球体很难，但这并不意味着你不能[打印自己的回转轴承](https://www.thingiverse.com/thing:2375124)以获得乐趣和利润。

正如[Christoph Laimer]解释的那样，旋转轴承由一系列圆柱滚子组成，这些滚子围绕内圈和外圈以 90 度角交替排列，因此更易于 3D 打印。回转轴承通常应用于大型、缓慢旋转的应用中，如起重机平台或风力涡轮机机舱和塔架之间的轴承。在下面的视频中，[Christoph]向我们展示了他在 Fusion 360 中的参数化设计；对于我们这些不熟悉这款应用的人来说，它看起来有点像魔术。令人欣慰的是，他提供了 CAD 文件和不同尺寸轴承的 STL 选择。

[Christoph]对复杂的 3D 打印设计并不陌生，比如他最近的[无刷 DC 电机](http://hackaday.com/2017/05/08/powerful-professional-brushless-motor-from-3d-printed-parts/)或更老的[时钟制造](http://hackaday.com/2014/05/24/the-hour-of-the-3d-printed-clock-draws-nigh/)。时钟很酷，但轴承和电机真的让我们着迷——我们需要这样的设计来实现自我复制的机器。
