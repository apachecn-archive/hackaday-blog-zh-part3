# 你的水龙头有多热？水是什么颜色的？

> 原文:[https://hackaday . com/2016/11/06/how-hot-is-your ' s 水龙头-水是什么颜色的/](https://hackaday.com/2016/11/06/how-hot-is-your-faucet-what-color-is-the-water/)

从你的水龙头流出的水有多热？redditor [AEvans28]知道他们公寓里的水变得“疯狂的热”，他选择制作一个[视觉水温显示器](http://imgur.com/gallery/SGkHm)来警告他们当事情变得有点辣的时候。

这个整洁的小装置被藏在一个 Altoids 薄荷罐中——这是一个制造商经常使用的多用途盒子。内部有一个 ATtiny85 微控制器，使用 Arduino UNO 重新校准到更适合家用的温度范围，从深蓝色到闪烁的红色，还有额外的开关空间，而 10k ohm NTC 热敏电阻和 RGB LED 使用电工胶带功能性地捆绑到厨房水龙头上。该设置反应灵敏，清楚地显示了[AEvans28]的水加热有多快。
