# Hackaday 奖参赛作品:做一个可以在幼儿园生存的图书阅读器

> 原文:[https://hackaday . com/2016/10/04/hackaday-prize-entry-making-a-book-reader-that-can-survive-幼儿园/](https://hackaday.com/2016/10/04/hackaday-prize-entry-making-a-book-reader-that-can-survive-kindergarten/)

【atomicthomas】是一位敬业的老师。人们只需要看看他在过去 16 年里为读者所做的工作。随着像 Pi Zero 这样的硬件威胁到廉价电脑的供应链，他开始把目光放得更高。

一切都是从耳机和录音带开始的。对于我们所有使用磁带和学校耳机的人来说，我们知道这个计划的缺陷。没有什么能长时间忍受孩子们黏糊糊的暴力的手。当这本书的录像带出现时，DVD 播放机也遭受了类似的命运。

于是，他开始把磁带和 DVD 拷到电脑上。然而，鼠标上有一个关于小零件的警告是有原因的，也没有持续多久。所以，他造了一台有街机按钮和树莓皮的电脑。这一个运行一个媒体管理器的高度简化的版本并且工作得很好。即使是有特殊需求的孩子也能顺利通过。用 iMac 和任天堂控制器进行的第二次探索效果更好。显然，所有五岁的孩子都本能地理解如何使用任天堂控制器。

利用用户测试数据，在他最近的迭代中，他在任天堂控制器中开发了一台低于 20 美元的阅读电脑。这不是我们报道过的技术上最深入的黑客攻击，但对于恶劣的环境来说，它无疑是排名靠前的。
