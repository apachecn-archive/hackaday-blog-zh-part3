# 塞尔达和万物之笛

> 原文：<https://hackaday.com/2017/03/20/zelda-and-the-ocarina-of-things/>

语音识别是今年家庭自动化的模型，但除了感觉像你在白羊座 1b 上与 HAL 9000 争论，它对我们这些书呆子来说根本不起作用。那么更令人讨厌的是什么呢？在你的口袋里随身携带一个[陶笛怎么样，这样你就可以得到一个树莓皮来为你开门](https://www.youtube.com/watch?v=glZnkpIDWSE)？(YouTube 视频，嵌入下方。)嗯，那就行了。

[足够先进]的视频让我们有 90%的机会复制这个版本。里面有一个装有麦克风和树莓酱的管子。房子里散落着一堆由 ESP8266 供电的小工具，负责开关加热器、给植物浇水，甚至用伺服系统按下(备用)汽车遥控器。

我们很想知道 Raspberry Pi 运行的是什么音高或歌曲识别软件。自从看到[吹口哨的](http://hackaday.com/2013/08/22/update-whistled-platform-upgraded-for-simple-word-recognition/)之后，我们就想实现一个基于口哨的家庭自动化界面。我们可以很好地保持一个曲调，但我们并不总是以相同的音高开始，这是一个自由度，假设它只对一个陶笛做出反应，那么[足够先进的]系统就不必担心。

如果你通过在窗外播放“塞尔达的摇篮曲”来质疑锁上和打开你实际公寓的安全性，你要么高估了普通的小偷，要么你就是没有理解这个笑话。打电话(希望找到)手机的用例足以让我们随身携带一个笨重的陶笛！
