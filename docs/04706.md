# CES2017:目镜中的天体摄影

> 原文:[https://hack aday . com/2017/01/09/ces 2017-目镜中的天文摄影/](https://hackaday.com/2017/01/09/ces2017-astrophotography-in-the-eyepiece/)

如果你从未在你的后院架设过望远镜，你就从未真正失望过。哈勃可以拍摄一些土星、星云和其他天文现象的精彩照片，但即使是昂贵的后院望远镜也只能产生一些污迹。为了更好地研究天文学，你会花时间挤在一台相机和一台电脑前，将图像堆叠起来，制作出几乎符合你预期的东西。

在 CES 上，Unistellar 推出了一种设计用于安装在望远镜目镜上的设备，为您完成所有这些工作。

 [https://www.youtube.com/embed/YDOtde3ZXAs?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/YDOtde3ZXAs?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)

T2】

据 Unistellar 的人说，这个盒子里有一台小型 Linux 电脑、照相机、GPS 和一个液晶显示器。一旦望远镜安装好了，这个模块就会拍摄一些望远镜视野的照片，将这些照片叠加起来，然后将结果叠加在目镜上。把这想象成“现场”天体摄影。

除了让木星看起来不那么像一个巨大的红色污迹，Unistellar 模块还增加了增强现实功能；它知道望远镜指向哪里，如果你在看任何值得注意的天体，它会添加一个标签。

虽然我无法看到这个非常酷的设备，但 Unistellar 的人说他们将在不久的将来发起一场众筹活动。
