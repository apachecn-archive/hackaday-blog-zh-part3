# 周五黑客聊天:来自 Arduino 的 Tenaya Hurst

> 原文：<https://hackaday.com/2017/05/10/friday-hack-chat-tenaya-hurst-from-arduino/>

本周五中午，太平洋时间[加入我们，与 Arduino](https://hackaday.io/event/21770-arduino-hack-chat) 的 Tenaya Hurst 进行一次黑客对话。如果你是过去几年(或无数其他现场活动)的大制造商博览会的一员，并在 Arduino 地区停留，你可能会遇到 Tenaya。她是 Arduino 的教育客户经理，喜欢与可穿戴电子产品打交道。

来讨论创客教育以及 Arduino 在让我们的学生对电子产品和 STEAM 教育感兴趣方面所扮演的角色。Tenaya 还将讨论她正在开发的一款新的可穿戴技术套件。我们希望下周能在湾区创客大会上亲眼看到这些设备。

### 以下是参与方式:

[![join-hack-chat](img/cff5733f39a173302e1cd102e75ac107.png)](https://hackaday.io/project/5373-hack-chat) 我们的 Hack Chat 是 Hackaday.io 上的实时社区活动 [Hack Chat](https://hackaday.io/project/5373-hacker-channel) 群发消息。

登录 Hackaday.io，访问该页面，并寻找“加入这个项目”按钮。一旦你成为这个项目的一部分，这个按钮就会变成“团队信息”，直接带你进入黑客聊天。

你不必等到星期五；随时加入，你可以看到社区在谈论什么。
