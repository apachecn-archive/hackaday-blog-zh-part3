# 北卡罗来纳州黑客空间被大火摧毁，成员发誓要重建

> 原文：<https://hackaday.com/2017/07/28/north-carolina-hackerspace-destroyed-by-fire-members-vow-to-rebuild/>

老工业建筑似乎吸引了黑客空间。这可能是过去大型制造企业所在建筑中典型的开放式平面图，也可能是人们希望重振曾经机器轰鸣、熟练工人频繁工作的地方。无论吸引力如何，这种关系并非没有风险；木地板和框架的老建筑可能是火药桶，悲剧随时可能发生。

上周五，这样的命运降临在北卡罗来纳州花岗岩瀑布的山麓社区研讨会上。[细节仍不清楚](http://wncn.com/2017/07/21/huge-fire-breaks-out-at-nc-mill/)75，000 平方英尺的前 Shuford Mills 纺织厂的残余仍在燃烧，消防队长的调查尚未完成。令人欣慰的是，没有人丧生，来自 16 个县的几名消防队员的受伤仅限于中暑，他们在炎热潮湿的北卡罗来纳州皮埃蒙特与大火搏斗。

这座建筑完全被毁，里面的几乎所有东西都不见了。与 FCW 的 3500 平方英尺的黑客空间一起的还有几家企业、一家仓储公司和一些住宅公寓。[我们在 2012 年 FCW](http://hackaday.com/2012/06/03/hackerspace-intro-foothills-community-workshop-in-granite-falls-nc/) 盛大的开业庆典上介绍过他们，从他们网站的外观来看，从那以后他们的发展突飞猛进。一个大型机械车间，装备精良的木材车间，一个火腿小屋，图书馆，电子车间，fab 实验室，一个带高压灭菌器和通风橱的湿实验室空间，一个巨大的 HO 比例模型铁路，甚至一个大规模艺术和舞台艺术区都被添加，现在都是灰烬。唯一剩下的就是无线电室的一根天线。

FCW 发誓要重建，当然，成员们现在正式进入“乞讨模式”，为重建他们的社区所需的任何事情和一切。经过一周的时间来克服最初的震惊，成员们已经列出了一个坚实的清单，列出了首先更换哪些设备，甚至还计划继续他们定期安排的课程和外联活动。他们已经建立了一个众筹活动来传递虚拟帽子，并且毫无疑问会欢迎设备捐赠，如果你能够加入并帮助一些黑客同伴的话。

可悲的是，除了生活是不可思议的随机，每个人都可能做对每件事，但最终还是会失败之外，没有太多可以从中吸取的教训。这场大火似乎是从大楼里一个与黑客空间完全独立的区域开始的，所以看起来 FCW 没有做错什么。消防队长报告说，火灾前两天的一次检查揭示了喷水系统的一个问题，业主似乎立即跳上了这个问题。零件被订购，24 小时守卫被安排，然而大楼还是被烧毁了。这看起来像是运气不好的例子。

唯一有助于应对厄运的是一份好的保险单，但你也要确保你的黑客空间不会引发火灾。为此，有必要回顾一下商店消防安全的基本知识，也许还有你是如何在 T2 储存易燃物品的。对于那些在旧工厂有办公空间的人来说，尤其是当你与其他租户共用办公空间时，也许定期与消防官员一起走一走是个好主意。如果它能发生在 Foothills 社区工作室，它也能发生在你身上。
