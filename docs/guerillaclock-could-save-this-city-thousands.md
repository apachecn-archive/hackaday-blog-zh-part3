# 游击队时钟可以拯救这个城市成千上万

> 原文：<https://hackaday.com/2018/04/22/guerillaclock-could-save-this-city-thousands/>

他们说需要是发明之母。但是如果你需要的东西已经被发明了，但是非常昂贵，另一个发明之母可能是预算超支。当[克林斯提芬]当地政府决定在公交车站安装倒计时钟时，情况就是如此，每只钟高达 25000 美元。考虑到这有点极端，他决定以更低的价格建造自己的车。

该项目使用 Raspberry Pi Zero W 作为其核心，并使用 16×32 RGB LED 矩阵作为显示器。有些工作已经完成了，因为总线系统有一个随时可用的 API。Pi 通过该 API 接收有关公交车时刻表的信息，并根据其位置，能够确定下一辆公交车的到达时间，并将其显示在 LED 矩阵上。使用定制的 3D 打印外壳和所有其他材料，每个时钟的成本只有 100 美元，便宜两个数量级以上。

希望当地政府从[克林斯提芬]那里得到暗示，决定使用一种更理智的解决方案。与此同时，如果你很难准时到达公交车站，你也许可以[建造自己的公共交通时钟](https://hackaday.com/2016/10/27/train-time-ticker-will-save-your-morning-commute/),可以在自己家里使用，而不是在火车站。
