# 你的 3D 打印品有毒吗？

> 原文:[https://hackaday.com/2015/11/11/are-your-3d-prints-toxic/](https://hackaday.com/2015/11/11/are-your-3d-prints-toxic/)

随着 3D 打印机越来越受欢迎，可用性越来越高，不可避免地有人会开始研究它们带来的潜在环境影响。现在，我们有两名来自加州大学河滨分校的研究人员发出警告，某些塑料对斑马鱼胚胎有毒。付费墙后面的全纸)。

就像科学中经常发生的情况一样，这个发现是偶然发现的。研究生[Shirin Mesbah Oskui]正在使用 3D 打印工具研究斑马鱼胚胎，这是一种在发育生物学中广泛使用的模式生物，但她发现这些工具正在杀死她的动物。她进一步调查发现，Stratasys Dimension Elite FDM 打印机和 Formlabs Form 1+立体平版印刷打印机的打印结果对发育中的斑马鱼胚胎具有“明显的毒性”。对鱼来说，基于树脂的 SLA 打印部分比融合的 ABS 打印更糟糕——暴露于 Form 1+打印的胚胎 100%在 7 天内死亡，少数存活了这么长时间的胚胎在死亡前显示出发育异常。有趣的是，该论文还描述了一种降低 SLA 印刷毒性的紫外线固化工艺，该大学正在申请专利。

当然，对斑马鱼有毒的东西对学校的孩子来说不一定是个问题，就像下面的视频暗示的那样。不过，这是一篇有趣的论文，指出了一个显然需要更多调查的领域。
