# 两用 Arduino 伺服测试仪

> 原文：<https://hackaday.com/2016/11/23/a-dual-purpose-arduino-servo-tester/>

遥控飞机飞行是一种多学科的爱好，可以让你扩展你的技能。你不需要知道太多就可以开始，但是要变好，你需要既是航空工程师，又是试飞员和机械师。但是如果你真的要走得很远，你还需要擅长电子学，这是这个 Arduino 伺服测试器的部分原因。

[Peter Pokojny]决定投身于电子学，以帮助他培养这一爱好，他一头扎进了深水区。他基于 Arduino 构建了一个伺服测试器和演示器，并额外增加了一个良好的用户界面和一系列功能。测试程序可以使用多种轮廓中的任何一种——三角形、正弦形或正方形——在测试中使伺服系统在其整个运动范围内循环。测试周期的速度是可选的，甚至有一个模式来命令伺服手动到一个特定的位置。我们打赌这个构建对[Peter]来说是一个很好的教训，他最终得到了一个有用的工具。

需要比[Peter]更深入的基础知识吗？然后查看[这本关于伺服系统的初级读本](https://hackaday.com/2016/10/30/primer-on-servos-hits-all-the-basics/)和[这本深度指南](https://hackaday.com/2016/03/24/hackaday-dictionary-servo-motors/)。
