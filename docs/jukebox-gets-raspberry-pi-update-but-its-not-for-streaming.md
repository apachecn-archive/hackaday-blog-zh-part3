# Jukebox 获得了 Raspberry Pi 更新，但不是针对流媒体

> 原文：<https://hackaday.com/2016/12/28/jukebox-gets-raspberry-pi-update-but-its-not-for-streaming/>

这是一个扭曲的电子康复中心。通常我们会期待一个自动点唱机 Raspberry Pi 项目用流媒体音乐取代过时的机电设备，但[一个带有 Raspberry Pi 遥控器的完整自动点唱机](https://www.youtube.com/watch?v=6l2vZ3UDlp4)是一个不错的改变。

像[revnhoj]的 1954 年 AMI F120 这样的老式 jukes 是机电奇迹。在一个水平的架子上放着 60 张每分钟 45 转的光盘，一个安装在轨道上的机械臂会取出正确的光盘，并将其放在转盘上播放选定的歌曲。下面视频中的单元是主点唱机，它支持安装在摊位上的“墙盒”，因此顾客可以在不离开桌子的情况下选择歌曲。[revnhoj]模拟了一个墙盒，树莓 Pi 通过继电器连接到原来的墙盒接口。Pi 提供了一个可以通过平板电脑访问的 GUI，正确的联系人会被逗乐，[revnhoj]的 45 集通过原始机制和放大器播放，以其所有的“Sonoramic Sound”威严。这是一个非常巧妙的技巧，在忠于原有平台的同时增加了新的功能。

我们在 reddit 上发现这个黑客的聊天倾向于增加流媒体音频，但我们真的希望 juke 保持完整，只提供黑胶。在之前，我们已经看到过点唱机被掏空，虽然这对一些人来说可能有意义，但我们更喜欢老学校的方法。
