# 快手罗宾！蝙蝠钥匙扣！

> 原文:[https://hackaday . com/2017/08/09/快速循环蝙蝠钥匙串/](https://hackaday.com/2017/08/09/quick-robin-the-bat-keychain/)

我们不知道蝙蝠侠是否有一个钥匙链来存放蝙蝠摩托、蝙蝠直升机和其他所有交通工具的钥匙。但我们猜测，如果他这样做了，它看起来不像是[Krishna 793]廉价收购的那辆。它有一个小按钮，点亮一些发光二极管，播放一首小曲。[Krishnan]认为他可以做得更好。在切碎了一些耳机并添加了一个 LiPo 电池后，他完成了一个[改进的钥匙链](http://www.instructables.com/id/Batman-Keychain-With-ESP8266/)，你可以在下面的视频中看到。第一个视频是之前的视频。二是改装后。当然，这只是对 led 的一个小改进和一个简单的调整，但现在如果你愿意花时间这样做，它可以做更多有趣的事情。

尽管取出现有的电子设备在蝙蝠侠身体内部留下了一些空间，但仍然需要一个 Dremel 工具才能将所有东西塞进去。并没有太多:一个 ESP01 模块，三个表面贴装发光二极管，和一个电阻。电池和充电器占用的空间和其他部分一样多。

[Krishnan]对钥匙链有更多的计划，包括允许它成为一个热点，这样你就可以下载一首新曲子。诚然，如果你有一台彩色 3D 打印机，你可能只需要在外面和里面滚动自己的打印机，但改装现有的廉价玩具也很酷。

也许是时候带着 ESP8266 重游经典的蝙蝠洞秘密开关了。蝙蝠侠以腰带上别着一把名副其实的瑞士军刀而闻名，所以也许[这个 ESP8266 项目](https://hackaday.com/2017/05/21/multipurpose-esp8266-keychain/)会更合适。
