# 为您的触摸屏打印物理按钮

> 原文：<https://hackaday.com/2018/06/20/print-physical-buttons-for-your-touch-screen/>

现代手持游戏硬件非常棒。这些设备是符合人体工程学的发电站，但我们中的许多人都在一个痛苦的矩形智能手机上玩便携式游戏。他们的主要交互方式是食指或拇指，而不是 D-pad 和按钮。肩部触发器只存在于少数手机上。蓝牙游戏垫价格实惠，但它们要么体积庞大，要么你必须找到另一种方式来握持你的手机。可拆卸的肩扣是一个完美的妥协，因为它们可以放在零钱包里，而且很便宜，因为你可以自己制作。

[ASCAS]解释了他的杠杆如何将物理杠杆压力转化为电容触摸响应。基本前提是接触点一直在接触屏幕，但是直到你拉动拉杆，拉杆上覆盖着铝带，屏幕不会感应到那里的任何东西。它非常聪明，整个套件可以用通常储存在五金店和黑客地下室的消耗品来构建，它应该可以在任何电容式触摸屏上工作。

[物理按钮和电话](https://hackaday.com/2016/12/30/smartphone-case-for-the-retro-gamer/)不必被隔离，[成熟的按键开关也不例外](https://hackaday.com/2017/10/16/hackaday-prize-entry-giving-phackaday-prize-entry-giving-phones-their-tactile-buttons-backhones-tactile-buttons-again/)。或者也许许多[电容式触摸开关是你的强项](https://hackaday.com/2015/12/18/touch-piano-hits-all-the-right-notes/)。
