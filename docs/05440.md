# Hackaday 奖参赛作品:一台没有 C

> 原文：<https://hackaday.com/2017/03/30/hackaday-prize-entry-a-cnc-mill-without-the-c/>

这是家用数控建筑的一个主要部分，由可用零件在工作台上建造的 3D 铣床。在管状、胶合板或挤压铝框架上，我们看到了各种能力水平的令人惊讶的轧机阵列。

这样一个工厂的标准是有一台计算机控制它。给它一个 CAD 文件，执行软件魔法，按下按钮，接收成品(理论上是这样)。这是一个惊喜，然后看到一个轧机的输入不是来自一个 CAD 文件，而是所有的控制都是由手通过一个操纵杆的媒介。[【Mark Miller】的 3D 打印自由雕刻机器](https://hackaday.io/project/19624-series-1-3d-printed-3d-freeform-carving-machine)是一个操纵杆控制的研磨机，其手臂上有一个旋转工具，面向一个可旋转的床，它可以在膨胀的泡沫上进行令人印象深刻的雕刻。

你可能会问，当你可以简单地拿起手中的旋转工具开始雕刻时，你究竟为什么要制造这样一台机器。你是对的，从这个角度来看，这台机器有一种光荣的无用感。但是，采取这种观点完全没有抓住要点，这是一个聪明的构建，而且是一个非常好的想法。我们注意到他还没有把文件放出来让其他人试一试，如果其他人喜欢用它制作 CNC 软件，那么我们确信这是可能的。

有一个视频展示了磨坊能做的基本动作，我们把它放在了休息区下面。不过，最好的说法是，这是一款可以启用 YouTube 双倍速度选项的手机。
