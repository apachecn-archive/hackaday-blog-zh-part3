# 制造太阳能电池

> 原文：<https://hackaday.com/2018/01/04/making-solar-cells/>

我们将承认，你的地下室里不太可能有足够的设备来使用这些步骤制造太阳能电池。然而，观察一个裸露的硅晶片如何变成太阳能电池是很有趣的。如果你看过正在制造的 IC，你会发现它们有很多相似之处，但也有一些不同之处。

这个过程需要一个硅晶片、一些烤箱、旋涂机、光刻设备和一个划片锯等等。哦，你可能也需要一个干净的房间。也许你应该买现成的太阳能电池，但看看它们是如何制造的仍然很有趣。

现代太阳能电池有一些额外的结构来提高效率，但这个视频中的电池非常普通。例如，一些实验电池使用多层有源器件，每层吸收不同波长的光。

如果你真的想自己制造，还有另一个过程，你可以从一些铜开始，最后得到一种使用铜基半导体材料的太阳能电池。但是不要被愚弄，认为制造硅品种对黑客来说完全是遥不可及的，[我们已经看到【萨姆·泽洛夫】成功了](https://hackaday.com/2017/02/25/the-fab-lab-next-door-diy-semiconductors/)。
