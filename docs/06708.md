# DIY 风力涡轮机获得免费能源

> 原文：<https://hackaday.com/2017/08/03/diy-wind-turbine-for-free-energy/>

随着电费的上涨和英国天然气价格的上涨，每个人都可以使用一些免费能源。有很多方法可以收获可再生能源，包括太阳能和风能，然而，建立一个风力发电厂的成本可能相当高。Tickles 先生]上传了一个视频，他用一种更便宜的 DIY 方法制作了一个 DIY 风力涡轮机。

他的项目使用商用吊扇作为涡轮机，将风能转化为电能。聚氯乙烯管被用来安装整个东西，使它变得便携。纸板鳍被用来使螺旋桨面向风，但有计划在未来升级它。Tickles 先生]通过点亮一盏灯，然后给手机充电来展示他的项目。

就价格而言，这种方法非常简洁，可以扩展到更大的粉丝群。对于那些正在寻找更简单版本的人来说，看看最简单的风力涡轮机。
