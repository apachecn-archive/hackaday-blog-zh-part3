# 路由器的数控仿真模型

> 原文:[https://hackaday.com/2016/11/27/cnc-dummies-for-routers/](https://hackaday.com/2016/11/27/cnc-dummies-for-routers/)

[这个老托尼]有几个在 Hackaday 上出现过的视频。他的最新作品是[路由器的数控模型](https://www.youtube.com/watch?v=YBGqknN3gGs)(见下文)。副标题，数控基础，是一个诚实的。如果你已经精通 GCode 和 Mach 3，你可能不会看完 14 分钟的视频(尽管 Tony 很有趣，即使你知道他在说什么)。

据他自己承认，这是真正的爱好级数控路由器和轧机数控基础知识。他开始谈论他的定制机器以及一些价格在 500 美元到 5000 美元之间的普通机器。然后，他给出了 GCode 的简单草图。

视频的最后一部分讲述了 CAD 和 CAM 软件。他谈到了 Fusion 360 和 Mach 3，尽管该视频的目的不是为任何特定工具提供教程。当托尼制作猪形砧板时，你确实可以从他的肩膀上看过去。

如果你做过 3D 打印，你不会在这里发现太多新东西。如果你有任何数控经验，你会发现这里没有什么新的东西。但这不是重点。这将是一个很好的视频，用于向学生、新黑客空间成员或任何对 CNC 一无所知的人介绍所需的一般工作流程。

如果你仔细观察，你可能会学到一些技巧，尤其是如果你只习惯 3D 打印的话。例如，在切割完成之前，切割猪得到标签来防止它飞出，并且他使用牺牲的滑板来避免切割到路由器的工作台上。

[托尼]显然一直很忙，因为我们已经看到他建造了，除了别的以外，涡流管。我们还知道他喜欢[玩火](https://hackaday.com/2016/11/14/fun-with-fire-oxy-acetylene-basics/)。
