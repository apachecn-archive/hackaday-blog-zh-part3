# 五金店的镀锡解决方案

> 原文:[https://hack aday . com/2017/10/23/tinning-solution-from-the-hardware-store/](https://hackaday.com/2017/10/23/tinning-solution-from-the-hardware-store/)

在家里制作自己的印刷电路板通常会导致电路板看起来像是自制的。裸露的铜是一个信号。这可能是你的审美，我们不会限制你的风格，但裸露的铜比镀锡铜更难焊接，而且随着时间的推移，它会氧化。在家镀锡可以让你离拥有一个全功能的板更近一步。在广告之后的视频中，著名的化学家【nurdrage】向我们展示[如何在家中制作镀锡溶液](http://youtube.com/watch?v=Hsw3lOnHaas)。

只有三种成分可以制成这种溶液，你可以在街角的五金店找到它们。

*   盐酸。也被称为盐酸。
*   含锡≥ 95%的固体无铅焊料
*   含硫脲的银抛光剂

一切都在视频的前三分钟。[nurdrage]继续解释这个反应背后的化学反应。它不需要电或热，但热会加速反应。有了这种简单，就没有理由再在你的厨房里制作未镀锡的电路板了。如果美学是非常重要的，自己在家镀锡可以让你屏蔽掉某些区域，并在同一块板上暴露铜和锡。

[【Nur drage】对黑客日](http://hackaday.com/?s=nurdrage)并不陌生，他甚至在这里有一篇关于[制作自己的 PCB 蚀刻剂](http://hackaday.com/2012/12/10/10-ways-to-etch-pcbs-at-home/)和[热板](http://hackaday.com/2016/09/28/ask-hackaday-how-do-you-make-a-hotplate/)的文章，让你的 PCB 生产进入高速运转。
