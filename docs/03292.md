# Hackaday 奖参赛作品:蚱蜢神经元

> 原文：<https://hackaday.com/2016/08/16/hackaday-prize-entry-grasshopper-neurons/>

一场蝗灾降临到你的花园，突然你意识到蚱蜢很难捕捉。蚱蜢是自然界完美的防撞系统，这要归功于一种独特的视觉系统，它包括直接从眼睛延伸到动物腿部的神经元。为了这个 Hackaday 奖的参赛作品——作为后院大脑今年夏天的一个研究项目，Dieu My Nguyen 正在研究蚱蜢视觉的神经科学。

大约两年前，我们参观了后院大脑，发现了三个非常有趣的项目。第一个是关于光遗传学的项目，或者说是重组神经元，这样当果蝇暴露在红光下时，它们就能尝到甜味。第二个是遥控蟑螂。第三个数字会让你震惊:一个让我用意念的力量震惊别人来扩大我的自大狂的装置。然而，这并不全是娱乐和游戏。这个蚱蜢神经元探针将使用[后院大脑钉盒](https://backyardbrains.com/products/spikerbox)来调查这些神经元在对刺激做出反应时被激活。

起初，观察普通蚱蜢来了解碰撞和物体回避的效用可能不是很明显。你对神经科学了解得越多，与普通计算机视觉任务的生物学联系就变得越明显。这使得这成为一个伟大的研究项目和进入 Hackaday 奖的绝佳机会。
