# DIY 空气炮从空中拦截无人机

> 原文：<https://hackaday.com/2017/07/20/diy-air-cannon-snags-drones-from-the-sky/>

谁没有过讨厌的无人机在家庭野餐时嗡嗡作响，或者在郊区后院盘旋的经历，穿着比基尼的女儿们正试图安静地晒日光浴？在这种情况下，一把散弹枪就足够了，但是有了这个压缩空气驱动的无人机捕捉器，就不用担心非法开枪来保护隐私了。

在评论栏被愤怒点亮之前，上述场景完全是在开玩笑。我们不容忍在无人机上使用武力，我们也不支持那些使用无人机甚至暗示侵犯隐私的人。我们都可以相处，即使我们希望[让它变得极端]的反无人机枪永远不会在愤怒中使用，它仍然是一个整洁的构建，给了我们很多想法。该装置实际上是四门同轴的窄口径压缩空气炮，每门炮发射一枚子弹，子弹系在一张轻质网的角上。一套相当复杂的联动装置设定了枪管的伸展，一对旧氧气瓶充当了压缩空气的储气罐。安装在复杂的枪托和手柄上的有趣的触发机构使快速动作放泄阀跳闸；我们希望能看到更多关于这一点的捏造。下面的视频显示了一次测试射击，结果一架无人机被干净利落地击落，尽管我们怀疑 quad 的所有者会这样描述它。

这种构建有点偏离[让它变得极端]通常的 DIY 工具，如商店自制的虎钳或 T2 的大型砂带打磨器，或者他们不寻常的车辆，如 T4 的越野悬浮滑板。但是看一个好的虚构视频总是很棒的，不管是什么主题。
