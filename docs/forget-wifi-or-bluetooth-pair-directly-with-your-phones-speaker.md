# 忘记 Wifi 或蓝牙，直接与手机扬声器配对

> 原文：<https://hackaday.com/2016/10/30/forget-wifi-or-bluetooth-pair-directly-with-your-phones-speaker/>

【Kedar Nimbalkar】夸张地[宣传终极手机音箱 dock](https://www.youtube.com/watch?v=zXQQFXG2uHA) 。它值一美元。它不需要你通过蓝牙或 WiFi 与它配对。他解释说，配对速度极快，为 0.00000000001。它也可能看起来像一个坏掉的笔记本电脑扬声器，上面焊着一个被踩坏的壁式电源，但如今谁能跟上工业设计的潮流呢？

他向我们展示了运行中的装置。他开始在手机的扬声器上播放一些音乐。声音不是很大，所以他干脆把手机放在了被告席上。突然间，所有 90 年代戴尔产品所能提供的音频保真度都从设备中爆发出来了！这是怎么做到的？

当然，这没什么大不了的。由于手机[扬声器是一个线圈](http://hackaday.com/2016/08/21/bare-bones-musical-tesla-coil-is-tiny-and-tinny/)，它可以在另一个线圈中感应出一个小电流。由此产生的电压可以被音频放大器拾取并通过扬声器播放。尽管如此，这还是很酷，我们喜欢他的建议，和我们的朋友打赌，我们可以和他们的耳塞无线配对。休息后的视频。
