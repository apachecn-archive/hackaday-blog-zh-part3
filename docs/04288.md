# 一种不同的单词钟

> 原文:[https://hack aday . com/2016/11/27/a-different-sort-of-word-clock/](https://hackaday.com/2016/11/27/a-different-sort-of-word-clock/)

我们这个极富创造力的社区对时钟情有独钟。这些年来，我们看到了如此多的钟表，人们可能会认为没有什么新的东西，没有一件钟表不会让我们打哈欠，因为我们面前摆着一些我们已经见过很多次的东西。

虽然偶尔会出现一个不同的项目。这款时钟吸收了钟表的基本理念，并设法呈现出一些新的东西，证明了这个特殊的项目还没有完全枯竭。

这样一个项目就是由[罗尔德·亨德里克斯]制造的[圆形字钟](https://rhoalt.blogspot.nl/2016/09/a-round-word-clock-sort-of.html)。拿一个传统的圆形挂钟，去掉指针和机械装置，然后在数字后面放上发光二极管。添加“四分之一”、“一半”等词语。放在一个内环里，然后把发光二极管放在它们后面。将所有这些发光二极管连接到一个带有实时时钟的微控制器上，你就可以得到一个令人耳目一新的时计了。

[Roald]的时钟有荷兰语的字样，背后的大脑是一个 Arduino Uno，带有相关的驱动 IC。他提供了一个视频，我们把它放在休息时间下面，展示了时钟在各种演示模式下的运行。
