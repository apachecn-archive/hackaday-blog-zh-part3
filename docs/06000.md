# 定制工作站让等离子切割变得轻而易举

> 原文：<https://hackaday.com/2017/05/23/custom-workstation-makes-plasma-cutting-a-breeze/>

等离子切割机可能是每个金属工人梦寐以求的工具。从手绘曲线到长直切割，没有什么比等离子切割机更能激发创意。不幸的是，还需要处理喷射穿过工件的过热金属，这就是[这个车间建造的等离子切割工作站](https://www.youtube.com/watch?v=RlQzB6Dfvlw)的原因。

[Regalzack]看起来他对他的桌子有几个设计目标。坚实的工作表面对于等离子切割来说并不是一个好主意，所以他将顶部设计成一个可替换的钢板网格。下面是一个收集炉渣的漏斗，这既是为了整洁也是为了防火安全。桌面和料斗安装在定制的轮式钢架上，较低的架子为他的林肯 375 等离子钻机提供了足够的空间。有了电缆挂钩和坚固的接地夹标签，整个东西就是一个很好的独立工作站。下面的视频展示了[Regalzack]使用的建造和一些制造技术；我们尤其被他为桌子板条切槽的巧妙方式所吸引。

等离子是多用途的东西——你可以用它来[制作音乐](http://hackaday.com/2016/04/12/rotating-plasma-vortex-speaker/)、[做汉堡](http://hackaday.com/2017/05/08/cooking-with-the-awesome-power-of-plasma/)，或者[装饰木材](http://hackaday.com/2016/08/18/arc-lighter-become-plasma-pyrography-pen/)。而且[开槽金属管](http://hackaday.com/2017/02/23/plasma-cutter-jig-notches-tubing-quickly-and-cleanly/)也不算寒酸。
