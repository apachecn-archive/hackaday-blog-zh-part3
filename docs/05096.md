# Zork 来到定制 FPGA CPU(再次)

> 原文:[https://hack aday . com/2017/02/21/zork-comes-to-custom-FPGA-CPU-again/](https://hackaday.com/2017/02/21/zork-comes-to-custom-fpga-cpu-again/)

[Robert Baruch]想用 FPGA 解决一个 CPU 项目。你总会遇到的一个问题是，要么模仿有工具和应用程序的东西，要么走自己的路，从头开始构建一切(这要困难得多)。

[Robert]采取了模仿的方法——算是吧。他用运行 Infocom 的 Z-machine 虚拟机的明确想法建造了一个 CPU，这允许它玩 Zork。所以至少当你完成的时候，你不必向你的非技术朋友解释它只是闪烁一个 LED。查看下面的视频，了解更多细节。

如果只是 FPGA 上的 Z-machine，我们以前见过。我们也看到人们走了很长的路，围绕他们的定制 CPU 构建一切。有趣的是，一组非常详细的视频描述了 CPU 以及它是如何诞生的。

我们不是说有一个五分钟的视频描述 CPU。这六个视频中的每一个都很充实(大多数都超过 40 分钟)，[Robert]使用 Xilinx 工具和仿真遍历 Verilog 代码。还有更多的视频，但大约有四个小时的视频，你可能想先开始观看。

如果你曾经想在别人做一个重要的 Verilog FPGA 设计时监视他们，这是你的机会。我们已经看到了在之前完成的最终结果[。如果你想一窥如何构建 CPU 和所有新工具，](https://hackaday.com/2014/11/29/the-zork-virtual-machine-implemented-in-hardware/) [A2Z](https://hackaday.com/2017/01/13/fpga-computer-covers-a-to-z/) 就是一个很好的例子。也许很快就会有人将左克移植到它身上。
