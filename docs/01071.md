# 西蒙妮用马达和伺服系统做奇怪的事情

> 原文:[https://hack aday . com/2015/12/29/simone-does-strange-things with-motors and-servos/](https://hackaday.com/2015/12/29/simone-does-strange-things-with-motors-and-servos/)

如果 DC 汽车是让东西移动的“你好世界”，伺服电机就是下一个合乎逻辑的步骤。[Simone Giertz]正带着[唤醒机](https://www.youtube.com/watch?v=mXLzfAHl4-k)和她的[新发布的剁碎机](https://www.youtube.com/watch?v=ISPnkVMxkcs)沿着这条精确的道路前进。[Simone]发现早上醒来最好的方式就是被机器人反复扇耳光。这种唤醒机器就是专门为这个目的而设计的。谁能在不断被打耳光的情况下睡觉？来自 ServoCity 的一个结实的齿轮马达一圈又一圈地旋转万圣节道具手臂，提供“提神”的拍击。

这个系统是由闹钟触发的。时钟的闹钟输出连接到一个 Arduino Uno。然后 Uno 激活一个继电器，使马达旋转起来。[Simone]意识到她可以在这里跳过 Arduino，但这是这个项目中阻力最小的路径。如果拍打的手不足以让你继续前进，唤醒机器确实有一个秘密武器:它可能会抓住你的头发，把视频拍摄变成一场痛苦的考验。

Simone 的最新项目是[剁碎机](https://www.youtube.com/watch?v=ISPnkVMxkcs)。ServoCity 肯定很喜欢她的第一个视频，因为他们赞助了她这个项目。这台机器由两把刀组成……嗯，剁。两个大功率伺服系统由 Arduino Nano 控制。伺服系统举起装有弹簧的刀，然后刀落下，切蔬菜、手指和其他任何东西。整个机器是由铝槽材和一个巨大的木制砧板构成的。当然，仅仅制造机器是不够的。【西蒙】[拍了一部戏仿的电视购物片，给任何想买切菜机的人看](https://www.youtube.com/watch?v=zBSEGqaK79M)，让任何一个叫查德的人都感到害怕。
