# 塑料模型模拟第一次无缆太空行走

> 原文:[https://hack aday . com/2018/02/11/plastic-model-emulates-the-first-untethered-space walk/](https://hackaday.com/2018/02/11/plastic-model-emulates-the-first-untethered-spacewalk/)

这里有一些非常棒的东西。[戴夫·阿克曼]写下了他尝试使用高空气球试图重现美国宇航局的布鲁斯·麦克坎多斯在太空中以地球为背景自由漂浮的著名图像的结果。[戴夫]这样做是为了庆祝第一次无缆太空行走 34 周年，甚至在 1984 年的同一天发射。他取得了优异的成绩，他的有效载荷记录了大量的视频和图像。

![](../Images/a06f8da3022f138d9aa63974423104e7.png)

80’s “Astronaut with MMU” model kit.

坚持每天进行一次太空行走并不是[戴夫]促成这一幕的唯一障碍。他找到了一个由 Revell USA 公司制造的老式、罕见的“带 MMU(移动操纵单元)的宇航员”塑料模型套件，开始制作这个套件，并安排它留在摄像机的视野之内。覆盆子 Pi Zero Ws 连同相机、LoRA 硬件、动作相机和一个 UBlox GPS 单元都出现在气球的有效载荷中。

令人难过的是，[布鲁斯·麦坎德斯]于 2017 年末去世，但这个项目是对第一次无束缚太空行走的美好提醒。关于建造和有效载荷以及跟踪系统的细节，在[Dave]的博客中有[的报道。下面嵌入了发射和不可避免的气球破裂的视频，但更多信息可在](http://www.daveakerman.com/?p=2236)[摘要报道](http://www.daveakerman.com/?p=2257)中获得。
