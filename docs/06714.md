# 将第二张 SD 卡添加到 Pi 零点

> 原文：<https://hackaday.com/2017/08/05/add-a-second-sd-card-to-the-pi-zero/>

Raspberry Pi Zero 是一个漂亮的硬件，将整个 Linux 计算机装入一包口香糖大小的包装中(不过不要嚼它)。然而，这种大小伴随着有限的 IO 选项，对于某些项目来说，这可能是一个难题。在这种情况下，[Hugatry]想要额外的存储空间，于是设计了一个聪明的方法，在 Pi Zero 上添加第二张 SD 卡。

Pi Zero 的问题是，只有一个 USB 端口，很难在不增加集线器或其他工作的情况下向设备添加任何其他存储空间。此外，当 Pi 运行时，主 SD 卡不能被移除，因此向 Pi Zero 添加易于使用的可移动存储选项是有意义的。

这是一个非常简单的方法——只需要几个电阻、一个 SD 卡连接器和一些跳线。所有的东西都连接好了，一个小小的配置改变就能让操作系统识别新卡。

总的来说，很高兴看到黑客们为一个已经很棒的平台增加了更多的功能。如果你发现它不够强大，你可以尝试超频一个。
