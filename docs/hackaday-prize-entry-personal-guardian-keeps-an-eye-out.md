# Hackaday 奖参赛作品:个人守护者密切关注

> 原文：<https://hackaday.com/2017/10/08/hackaday-prize-entry-personal-guardian-keeps-an-eye-out/>

个人守护者是一种可穿戴的跟踪和监控设备，旨在帮助弱势人群。该项目的目标是让这些患者尽可能地独立和活动，而不需要护理人员在场。戴上传感器包可能会让(例如)记忆障碍患者有更大的自由走动。

该设备由一个带有 GSM 屏蔽的 Arduino 101 开发板组成，用于向护理人员发送短信，例如，如果加速度计显示患者摔倒，或移动到特定的 GPS 坐标之外。此外，护理人员可以监控设备以确定设备的状态，并通过 BLE 传感器带查看患者的心率。

患者还可以按下紧急按钮或切换一系列预设的 SMS 消息。就复杂性而言，该项目的创造者[Ray Lynch]希望界面比智能手机更简单。
