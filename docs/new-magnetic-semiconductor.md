# 新型磁性半导体

> 原文：<https://hackaday.com/2015/11/23/new-magnetic-semiconductor/>

当你想到南达科他州时，你通常会想到拉什莫尔山，也许还有核导弹。然而【Simeon Gilbert】会让你想到半导体。南达科塔州立大学的学生西蒙在一年一度的西格玛 Xi 国家会议上获得了第一名，因为他在一种新型磁性半导体上的工作。

这种材料是与内布拉斯加大学林肯分校纳米磁性小组的研究人员合作开发的，是钴、铁、铬和铝的混合物。然而，一些铝被硅取代。在替换之前，这种材料在高达 450 华氏度的温度下仍能保持其磁性。硅代替了一些铝原子，工作温度接近 1000 华氏度。

虽然许多磁性材料(如磁铁矿)也是半导体，但它们的性质一般无法与硅等传统半导体相比。这导致了更多关于结合磁性材料和半导体来生产具有铁磁性的半导体的研究。

理论上，由磁性半导体制成的控制装置将允许通过电荷载流子(如在传统晶体管中)进行切换，但也提供对自旋状态和极化的控制。实用的“T0”自旋晶体管“T1”可以为非易失性存储器提供更高的存储密度和更低的功率。温度更高的器件还可以降低冷却要求，从而节省大量尺寸和功耗。
