# DPS5005 让数字电源变得轻而易举

> 原文：<https://hackaday.com/2017/08/20/dps5005-makes-digital-power-supply-a-snap/>

对于电子设备来说，没有什么比某种电源更基本的了。可能是一盒电池，也可能是高端实验室用品。[Andreas]用一个带 USB 和蓝牙的 DPS5005 电源模块构建了一个功能强大的开关电源，然后他用它构建了一个源测量单元。

这个小型模块的用户界面非常简单，所以安德烈亚斯很欣赏这种基于电脑的软件，可以远程控制供应。该模块可以输出高达 50V 的电压，但您应该相应地进行规划，因为它确实需要 1.1 倍的输入最大电压输出。它可以在较低的输入电压下工作，但无法输出那么高的电压。

如果你不喜欢 50V，5A，有类似的模块可以处理更多或更少的电压或电流，这取决于你的需要。但是要小心，因为默认模型中没有通信选项，视频中也没有出现好的情况。如果你想要的话，你必须确保你得到这些选择。

如果你想尝试另一种固件，我们之前已经看过了。我们还看了[一个类似模块](https://hackaday.com/2016/12/08/absolute-power/)的早期版本。
