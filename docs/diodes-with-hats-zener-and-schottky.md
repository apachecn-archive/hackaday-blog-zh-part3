# 带帽二极管:齐纳二极管和肖特基二极管

> 原文：<https://hackaday.com/2016/09/12/diodes-with-hats-zener-and-schottky/>

对于初学者来说，二极管类型有时可能是一个心理体操。如果它的作用就像一个神奇的小精灵止回阀，为什么会有这么多种？[当数据手册描述肖特基二极管](https://www.youtube.com/watch?v=bXEyCf1P0UU)时，通常很难在脑海中将其与标准区分开来。齐纳二极管对初学者来说可能是完全令人困惑的，尤其是当错误地将常规的 1N4001 放在电路中时。[Afrotechmods]制作了一个精彩的视频，解释了它们的区别和使用案例。

在这两个视频中，他都出色地描述了利弊，同时设置了实验来展示每一个。对于肖特基二极管，开关速度更快，压降更低。对于齐纳来说，它的缺点更少，更多的是利用其奇怪的电压钳，稳压器配置，并通过音频失真电路使昂贵的吉他听起来很糟糕。

他在两个视频中都给出了作为一名新兴电路设计师选择和使用器件的良好设计技巧。之后，二极管数据手册就不再神秘了。
