# 便携式教室升级:更小、更便宜、更快

> 原文：<https://hackaday.com/2016/11/20/portable-classroom-upgrade-smaller-cheaper-faster/>

[Eric]在 MkMe 实验室有一个梦想:建造一个便宜的便携式系统，提供教育世界任何地方的孩子所需的电子基础设施。他研究这个系统已经有一段时间了，最近他成功地将这个手提箱大小的系统缩小到更便宜、更小的尺寸。

我们上次讨论[Eric]的 EduCase 项目是作为他的 2016 年 [Hackaday 奖参赛作品](https://hackaday.com/2016/05/24/hackaday-prize-entry-worldwide-educational-infrastructure/)的一部分。我们的读者对这个项目的目标有很多怀疑，但是不管你怎么看待[Eric]的动机，事实仍然是这个构建非常酷。早期版本的 EduCase 依靠 Ku 波段下行链路接收来自[外联网](https://outernet.is/)的内容，因此需要在盒子里塞一个大天线。这就决定了行李箱的尺寸范围。目前的 EduCase 是一个精简得多的事物，它依赖于来自 Inmarsat 卫星的 L 波段链路，具有小得多的贴片天线。一个低噪声放大器和 SDR 接收器完成下行链路，一个 Raspberry Pi 提供 UI。[埃里克]的建设只是一个原型，在这一点上，但我们期待着看到一切塞进那个小鹈鹕案件。

是的，Outernet 是经过策划的内容，因此它与网络的体验完全不同。但是对于正确的用例，这个小包可能就能完成工作。BOM 的价格高达 100 美元，这个价格对于实验来说是合适的。
