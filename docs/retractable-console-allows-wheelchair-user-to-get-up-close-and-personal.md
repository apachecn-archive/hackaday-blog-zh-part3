# 伸缩式控制台允许轮椅使用者近距离接触个人

> 原文：<https://hackaday.com/2017/12/15/retractable-console-allows-wheelchair-user-to-get-up-close-and-personal/>

[Rhonda]患有多发性硬化症，这种疾病限制了她的行走和使用手臂的能力。她和波士顿之家的其他居民依靠轮椅行动，波士顿之家是一家为 MS 和其他神经肌肉疾病患者提供长期护理的机构。[Rhonda]的椅子配有一个控制台，它可以向外摆动，使她能够靠近桌子和柜台，但她无法施加足够的力来手动定位它。

遗憾的是，[Rhonda]的保险不包括她问题的商业解决方案。但是波士顿之家有一个设备齐全的商店来扩展和增强居民的轮椅，他们和麻省理工学院辅助技术原理和实践(PPAT)课程的学生一起[破解了一个解决方案](https://www.instructables.com/id/Developing-a-Motorized-Retractable-Joystick/)，这个解决方案不仅对【Rhonda】有用，而且应该普遍适用于其他椅子。学生们分析了问题，测量了所需的力和所需的间隙，并为控制台制作了一个原型受电弓支架。他们使该设备易于复制，并使 BOM 尽可能便宜，因为患者通常需要自掏腰包购买此类增强产品。下面的视频显示了一点关于这个问题和解决方案。

轮椅黑客很常见，比如 2015 年黑客日获奖的 Eyedrivomatic 。我们还覆盖了完全开源的轮椅，包括[手动](https://hackaday.com/2017/09/27/hackaday-prize-entry-toowheels-the-open-source-wheelchair/)和[电动](https://hackaday.com/2016/09/26/the-hackaday-prize-an-open-electric-wheelchair/)。
