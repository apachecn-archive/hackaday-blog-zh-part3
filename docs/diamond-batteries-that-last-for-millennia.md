# 持续数千年的钻石电池

> 原文：<https://hackaday.com/2016/11/29/diamond-batteries-that-last-for-millennia/>

像许多工业化国家一样，在第二次世界大战后，联合王国在核反应堆领域进行了大量投资。英国纳税人为研究、军事和核能反应堆买单。

几十年后，早期的反应堆现在大部分已经退役。太便宜而无法计量的电力变成了数十亿英镑的账单，用于安全应对核反应堆拆除所产生的许多不同类型的放射性废物带来的挑战，随着核工业开始了这一旅程，它反过来催生了一系列基于退役工作产品的研究项目。

布里斯托尔大学的一个团队已经展示了这样一个项目；他们的工作是研究钻石在受到放射性辐射时会产生微弱电流的特性。不幸的是，他们的新闻稿和视频没有解释所涉及的机制，我们的 Google-fu 也没有提供，但如果我们冒险猜测，我们会问他们一些问题，放射性是否会改变从钻石中释放电子所需的功函数，从而通过接触电势差获得电力。也许我们的物理学家读者可以在评论中启发我们。

到目前为止，他们的原型使用镍-63 源，但他们希望从旧反应堆的大量储存石墨块中提取碳-14，并用它来制造放射性钻石，不需要外部来源。由于产生的细胞的输出将与它们的放射性成比例，它们的寿命将与它们的放射性半衰期处于相同的数量级。碳-14 的半容量为 5730 年。

当然，很可能电力的产量不会很高，电压和电流很小，这可能不是一个免费的能源奇迹。但对于科学、航天工业和医疗植入物的超低维护长寿命电子产品的设计者来说，这将是相当有意义的。

我们已经把他们的视频放在休息时间下面了。这是对该项目的一个简单明了的解释，尽管遗憾的是因为它是针对普通大众的，所以在一些技术细节上有点欠缺。尽管如此，这还是值得一看的。
