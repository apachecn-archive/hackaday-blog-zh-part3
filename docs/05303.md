# 像 1998 年一样冲浪，梦之队依然如故！

> 原文:[https://hackaday . com/2017/03/15/surfing-like-its-1998-the-dream casts-still-got-it/](https://hackaday.com/2017/03/15/surfing-like-its-1998-the-dreamcasts-still-got-it/)

如果你在 20 世纪 90 年代末是一个狂热的游戏机玩家，那么你很有可能渴望得到世嘉 Dreamcast。这是一个承诺与众不同的控制台，一个内置 PowerVR 3D 加速的紧凑型机器(当时是沉重的东西！)，能够以某种形式运行 Windows CE，并且首次内置了互联网连接。游戏将不再是以前世嘉游戏机上的塑料盒式磁带，而是一种专有的类似 DVD 的世嘉光盘格式。

遗憾的是，梦之队从未真正成功兑现它的承诺。每个人都在谈论即将到来的索尼 Playstation 2，令人失望的 Dreamcast 销售导致世嘉退出了这款游戏机，并彻底退出了硬件市场。

不过，Dreamcast 的铁杆爱好者仍然存在，他们继续推动这个平台向前发展。Dreamcast 垃圾场的人们决定稍微倒退一下，他们[恢复了游戏机的拨号调制解调器](http://www.thedreamcastjunkyard.co.uk/2017/03/you-can-still-use-dial-up-to-surf.html)，看看近 20 年前的平台是否还能在 2017 年削减它。这并不像你想象的那么简单，因为，嗯，现在谁还用拨号上网呢？当然，在他们的基地英国，这几乎是闻所未闻的。他们找到了一个预付费拨号提供商，用最新的 Dreamkey V3.0 浏览器光盘武装自己，就可以上网了。

正如你所料，结果在很大程度上非常糟糕。依赖 CSS 的现代网站无法呈现，甚至无法加载，但像 Dreamcast 社区中的复古网站应该会出现。我们在休息时间下面放了一个视频，展示了这个相当曲折的过程，尽管遗憾的是他们没有想到加载 [Hackaday 复古版](http://retro.hackaday.com/)。然而，它的特点是罕见的键盘和鼠标配件。
