# PDP-1:开启黑客文化的机器

> 原文:[https://hack aday . com/2017/06/27/the-PDP-1-the-machine-that-started-hacker-culture/](https://hackaday.com/2017/06/27/the-pdp-1-the-machine-that-started-hacker-culture/)

我的遗愿清单中的一个目的地是位于加州山景城的计算机历史博物馆——我知道，我的目标很高。我会懊恼地意识到，我的生活已经跨越了信息时代的相当一部分，但我认为我会从看到旧机器中得到乐趣，其中一些我实际上已经到手了。但是我最想看到的机器是那些在我之前的机器，是那些促成了黑客文化诞生的机器，在这种文化中，我和许多 Hackaday 的常客逐渐成熟。

如果你要追溯黑客文化的起源，很有可能你会发现它的根源是数字设备公司的 PDP-1。对于一台 1959 年推出、仅售出 53 台的机器来说，这是一个很高的要求，相比之下，IBM 同时代的产品卖出了数万台。的确，在 50 年代末和 60 年代初，数字计算爆炸的前沿主要被“大铁”机器占据，大型机为即将到来的所有进步奠定了基础。

当时，计算确实是非常严肃的行业。大铁成本巨大，即使是一个普通的主机系统对一个公司来说也是一笔巨大的资本投资。因此，财大气粗的客户主导了市场，他们的需求决定了硬件的发展。通常情况下，主要的需求是处理大的会计问题，如运行每周工资单或季度利润。当时的计算机主要以批处理方式运行，将一叠叠包含程序和数据的穿孔卡输入机器，几小时甚至几天后输出结果。电脑操作员是移动成堆的卡片并收集输出的人，坐在终端前的想法是不可想象的。但是 DEC 的创始人肯·奥尔森有不同的看法:

> “我们有一个交互式计算的梦想。普通的计算被认为是巨大的、昂贵的、了不起的，超出了普通人的能力。交互式计算令人兴奋和有趣，人们可以直接与计算机互动。”

DEC 是一家非常年轻的公司，专注于向工程师销售其系统构建模块的“数字实验室模块”系列，工程师通常使用它们来构建计算机的测试平台。业务蒸蒸日上，当奥尔森在 1959 年向工程师本·格利提出他的愿景时，公司拥有了快速发展的资源。

在仅仅三个半月的时间里，Gurley 和他的团队将编程数据处理器-1 推向市场，主要是从 DEC 的现有模块中构建的。这种机器比当时的大型机更小更便宜，因此它吸引了较小的公司和学术界。事实上，该原型被捐赠给了麻省理工学院，在那里开发了一些更著名的应用。下面的视频展示了 PDP-1，计算机科学博物馆已经亲切地修复了运行其中一些的 PDP-1，包括 [*太空战！*](http://tvtropes.org/pmwiki/pmwiki.php/VideoGame/SpaceWar) ，一款出奇复杂的游戏，后来的经典之作《小行星》显然从中汲取了灵感。

底线是 PDP-1 真的是第一台鼓励用户坐下来玩的电脑。IBM 的机器在闭门造车的情况下做着无聊但必要的工作，并由一群仆人照看，而 DEC 的机器则进入了实验室和机构的奇怪角落，好奇的人们坐在他们的终端前，手指在键盘上摆好，同时说出一句简单但有力的话:“我想知道会发生什么，如果……”DEC 的机器是第一台允许实时回答这个问题的计算机，这个问题实际上是黑客文化的核心。

所以，如果你哪天碰巧在山景城，去拜访一下 PDP-1，这台机器通过让它可以玩而启动了一切。
