# DIY 印刷电路板的严重黑客打印机

> 原文：<https://hackaday.com/2017/10/24/heavily-hacked-printer-for-diy-pcbs/>

有时我们得到的提示只能让我们猜测一个项目是如何建造的，有时是为什么。[这台 PCB 打印机](https://www.youtube.com/watch?v=JPva4kMxmpg)就是这样；在这种情况下，构建细节是唯一的问题，因为它提出了一些非常令人印象深刻的 PCB。

我们所要做的就是休息后的视频，尽管进行了长达几分钟的彻底搜索，这似乎是[安德洛卡沃]为这个版本所做的唯一记录。标题告诉我们，这台打印机是围绕爱普生针式照片 1390 打印机的内脏建造的。从外面看不出这一点，因为打印机的每一部分都内置在一个定制的外壳中。纸张处理装置已被 A3 尺寸的加热平板取代，可在 Z 轴方向调节，以适应不同的纸板厚度。这张床在看似定制的直线轨道上运行。在引擎盖下，墨盒已被替换为外置墨水瓶，你可以选择任何颜色，只要是黑色。视频显示了一些间距为 0.1 毫米的 0.1 毫米的测试印刷，这些印刷有点不稳定，但在间距为 0.2 毫米的情况下，最精细的印刷效果非常好。木板用通常的方法蚀刻，效果很好；我们想知道这台打印机是否能改装成也能打印抗蚀剂和丝网印刷。

[Androkavo]似乎在他的 YouTube 频道上有很多有趣的项目，其中之一是我们最近特别介绍的[这个木制数字时钟](http://hackaday.com/2017/05/29/digital-clock-goes-with-the-grain/)。不过，我们很想了解更多关于这款打印机的信息。希望[安德洛卡沃]能看到这个并在下面评论。
