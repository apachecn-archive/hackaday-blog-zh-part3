# 用废弃的炉子铸造廉价、清洁的铸件

> 原文:[https://hack aday . com/2016/12/03/foundry-from-screwed-oven-for-cheap-clean-castings/](https://hackaday.com/2016/12/03/foundry-from-scrapped-oven-for-cheap-clean-castings/)

自制铸造厂是一个受欢迎的项目，并且有很好的理由。能够熔化和铸造金属是一个强大的工具，即使它“仅仅”是铝。但是大多数人建造的标准化石燃料铸造厂并不是没有问题，这就是这个快速、清洁的一次性铸造厂发挥作用的地方。

典型的家庭铝铸造厂基本上是一种耐火容器，可以吸收强制通风木炭或煤火的热量。但是正如[涡轮征服大鹰]指出的那样，这种燃料会导致熔融铝的碳污染和铸造金属时的缺陷。对于一个废弃的电动炉灶，[涡轮征服超级鹰]制造了一个铸造厂，以一种极其危险的方式避免了这个问题。烤箱的加热元件包裹在一个旧的不锈钢炖锅周围，炉子上的玻璃纤维球棒使特设的坩埚绝缘，炉灶的电源线直接连接到加热元件上。下面的视频显示，它确实可以熔化铝，铝是用来砂铸一个相当复杂的零件的。

不过，我们看不到这种设置有多种用途，所以它的可持续性取决于你能收集到的范围数量。但对于一次性工作，这是值得记住的。对于更持久的安装，看看这个便携式丙烷动力铸造厂。为了看看你能用它做什么，看看[这个由啤酒罐](https://hackaday.com/2016/06/23/custom-engine-parts-from-a-backyard-foundry/)制成的引擎呼吸器。
