# x 光下的乐高组件

> 原文:[https://hack aday . com/2017/10/01/Lego-components-under-x-ray/](https://hackaday.com/2017/10/01/lego-components-under-x-ray/)

[Nico71]为一家生产工业 CT 扫描仪的公司工作。这些 x 射线检测机可以检查设备内部，允许操作人员验证装配情况并检查材料的完整性。这也让[Nico71]有机会扫描他放在周围的一个乐高伺服系统,这个系统已经坏了。由此产生的图像看起来棒极了，真的让你看到一个封闭的系统，并挑选出它是如何工作的或为什么不工作。在这种情况下，您可以看到其中一根电线已经损坏。

[Nico71]计划扫描一堆乐高组件，比较(例如)官方乐高产品和山寨山寨产品。哪个构造更好？拥有更薄或更便宜的塑料或更低等级的钢材是一回事，但零件是如何设计的呢？

我们已经在 Hackaday 上报道了大量令人惊讶的 CT 优点，包括将 CT 扫描转化为 3D 打印的过程(T1)和关于改进自制 CT 扫描仪的帖子(T3)。
