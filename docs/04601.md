# 顺利现代化的谢妮展示

> 原文:[https://hack aday . com/2017/01/14/smoothly-modernized-谢妮-display/](https://hackaday.com/2017/01/14/smoothly-modernized-nixie-display/)

在围绕旧技术的怀旧情绪中，数码管的复兴已经使它们对于单个项目来说几乎贵得令人望而却步。看到了一个使心爱的设备现代化的机会，[Connor Nishijima]发布了这种新的 LED 边缘发光显示器，他将其命名为[谢利](https://hackaday.io/project/18633-lixie-an-led-alternative-to-the-nixie-tube)。

几年前，我们展示了他的原型。这种设计使用圆点来组成每个角色，但这次升级用圆滑的线条和专业设备的外观来平滑这个设计——或者至少是你在不久的将来会在赛博朋克中看到的东西。变色的 neo pixel led——由设计巧妙的滤镜调节——允许您根据自己的喜好定制内容，激光切割的丙烯酸板允许相对轻松地制作更大的显示器。

上面的图像(和下面的视频)显示了最新的谢利原型的两个版本。右边有一个巨大的改进，因为数字现在是轮廓而不是单笔画，并且是雕刻而不是完全通过丙烯酸切割。差别是显著的，在我们看来，把“回到绘图板”的效果移到“为黄金时间做好准备”。[Connor]和他的团队正致力于此，Tindie 的预购订单已经到位，首批可生产的数字产品即将下线。

考虑到谢妮电子管最初被认为对于钟表等大规模生产的物品来说过于昂贵，具有讽刺意味的是，他们看到的只是出于这一目的的业余爱好者项目的复兴。那么，对于那些看到更便宜的解决方案而又不牺牲结果质量的人来说，谢利可能符合他们的目的。该设计是完全开源的，所以开始黑客攻击吧！

对于一个数码管的适当的赛博朋克应用，检查这个[摩托车速度表](http://hackaday.com/2015/07/16/nixie-tube-speedometer-in-motorcycle-handlebars/)。哦，以免你认为我们在复制自己，就在几周前，这里还有另一个类似谢妮的边缘照明项目[。似乎好主意是一波一波来的。](http://hackaday.com/2016/12/22/light-pipes-and-leds-team-up-for-a-modern-take-on-the-nixie-tube/)
