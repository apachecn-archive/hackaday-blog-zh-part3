# 每次邻居用微波炉烤玉米煎饼时，指纹车库门都不会打开

> 原文：<https://hackaday.com/2016/03/16/fingerprint-garage-door-wont-open-every-time-a-neighbor-microwaves-a-burrito/>

包括他自己在内，有三个孩子，[戴夫]面临着非常真实的可能性，有人心不在焉地开着车库门，被抢劫。他没有安装一些普通的解决方案，而是编制了一个功能列表。这是一个多么好的功能列表啊！

车库门需要通过在房子周围战略性地放置 led 来通知他的状态，并让他完全控制他的设备。他想用他现有的钥匙代码输入系统来打开和关闭它。最后，如果他能给它添加一些生物特征，那将会非常酷；在这种情况下，指纹传感器。

核心硬件是主要的 Arduino，增加了指纹模块、触摸屏、一些维生素和 WiFi 接口。他还在 tinkercad 中制作了一些外壳:一个用于室内硬件，另一个带有翻盖，用于室外指纹扫描仪。

我们认为[戴夫]已经完成了他设定的目标。我们可以想象这个潜在的小偷盯着指纹扫描仪，把他们的行动转移到世界更简单的地方。休息后的视频。
