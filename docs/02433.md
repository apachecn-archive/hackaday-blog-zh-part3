# 对不起戴夫，我只会说 28 个短语

> 原文:[https://hack aday . com/2016/05/23/im-sorry-Dave-I-only-say-28-phrases/](https://hackaday.com/2016/05/23/im-sorry-dave-i-only-say-28-phrases/)

几年前，你可以买一个 IRIS 9000 蓝牙音箱。它之所以出名，是因为它看起来像《2001:太空奇遇》中 HAL 9000 计算机的“眼睛”。让一只 HAL eye 来回答你对 Google Now 或 Siri 的查询，这种想法很有诱惑力。问题是，它听起来仍然像谷歌或 Siri，而不像 HAL。

[Badjer1]也有同样的问题，所以他决定自己造一只眼睛。不过，他的目标并不是与智能手机的虚拟助手互动。他决定让它只是一个带有 USB 端口的延长线。正如你在下面的视频中看到的，这个建筑有 HAL 风格的记忆单元，一把钥匙，可以说电影中的短语(嗯，至少 28 个)。这把钥匙就像电影中戴夫用来让哈尔失去活动能力的那把。

MDF 机箱内有一个 Arduino 和一个处理音频播放的波形屏蔽。记忆卡是丙烯酸和关键是加工黄铜。结果是一个好看的项目。

如果你真的想成为哈尔，你可以[做一套服装](http://hackaday.com/2014/11/23/cosplaying-as-hal-9000/)。我们已经看到了一个类似的建筑[和一个树莓派](https://hackaday.com/2014/07/20/im-sorry-dave-im-afraid-i-cant-do-that/)。
