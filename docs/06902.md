# 便宜简单的磁性 DNA 分离方法需要你的帮助

> 原文：<https://hackaday.com/2017/08/28/cheap-and-easy-magnetic-dna-separation-method-needs-your-help/>

当你考虑到你身体中几乎每一个细胞都有一米多长的 DNA 盘绕在细胞核内时，获取一些进行研究似乎应该是相当容易的。但是由于所有其他细胞粘在一个粗糙的制备物中，DNA 很难分离。这就是[这种便宜又简单的磁性 DNA 分离方法](http://www.thethoughtemporium.com/paramagnetic-silica)的用武之地。如果它能在公民科学界的帮助下得到优化和测试。

商业 DNA 分离方法通常包括将二氧化硅珠混合到粗细胞组分中；DNA 优先与二氧化硅结合，使其有可能从细胞垃圾中机械分离出来。但是，与其使用离心机分离 DNA，来自思想商场的贾斯汀认为磁铁可能会做得更好。这并不是一个新的想法——生物技术公司在商业上提供磁性分离珠，但对于[贾斯汀]的预算来说，价格太高了。他的技术来自于从普通的铁化合物中制造磁铁矿颗粒，如 PCB 腐蚀剂和苔藓杀手，以及家用氨清洁剂。然后在磁铁矿颗粒上涂上硅酸钠溶液，也称为水玻璃。二氧化硅涂层应该允许珠子与 DNA 结合，由磁芯负责分离。

[Justin]在测试他的方法的过程中，他失去了获得所需仪器的途径，所以他呼吁更大的科学界帮助优化他的技术。基于他在从[卫星追踪](http://hackaday.com/2017/03/23/see-satellites-with-a-simple-radio-telescope/)到[石墨烯生产](http://hackaday.com/2017/06/17/graphene-from-graphite-by-electrochemical-exfoliation/)等领域的成功记录，我们打赌他也会搞定这一个。
