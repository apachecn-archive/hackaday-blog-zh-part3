# 带倾斜底座的 3D 打印机

> 原文：<https://hackaday.com/2017/01/19/3d-printer-with-tilted-bed/>

【奥利佛·托拉尔】和【丹尼斯·赫尔曼】两位来自苏黎世应用科学大学(ZHAW)的学生设计并制作了一台 [3D 打印机原型，它有一个可以倾斜的移动打印床](https://www.heise.de/newsticker/meldung/3D-Drucker-mit-beweglicher-Druckplatte-3594506.html)。 通过倾斜，无需额外的支撑材料即可印刷带有关键突出物的物体。打印机有六个轴，三个轴像往常一样控制打印头，另外三个轴控制打印床，允许更大范围的移动。

学生们声称，除了节省支撑材料之外，这台打印机在打印需要大量支撑的物体时还可以节省时间，因为我们认为，倾斜床比打印支撑物本身更快。在普通的 3D 打印机中，印版总是水平的，打印对象在水平层中构建。在这种印刷机中，对于较大的悬臂，印刷台以这样一种方式固定，即印刷物体枢转直到垂直于印刷头。当然，对于圆形，它可能会有所不同，但我们只在一个演示视频中看到了它的作用。也有好的一面，当一个打印完成时，它就完成了。无需 x-acto 刀来移除支架，无需砂纸，无需浪费时间。

他们说，让软件正确控制床比组装打印机更困难。它仍在开发中，例如，它不能同时移动打印头和打印床来产生连续的打印。
