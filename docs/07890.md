# 浏览器中的 PLC

> 原文:[https://hackaday.com/2017/12/06/plcs-in-your-browser/](https://hackaday.com/2017/12/06/plcs-in-your-browser/)

如果你的常用工具是 Arduino 和 Raspberry Pi，你可能会惊讶地发现，工业世界往往运行在可编程逻辑控制器(PLC)上。你可以把 PLC 想象成一个非常坚固的工业 Arduino，但是最好不要做太多的类比。有些 PLC 非常简单，有些相当复杂，但有一点是它们的共同点是通常使用梯形逻辑编程。如果你曾经想学习如何编程 PLC——在某些地方这是一项非常抢手的工作技能— [你现在可以在你的浏览器](http://accautomation.ca/plc-fiddle-online-editor-and-simulator-in-your-browser)中构建和模拟梯形逻辑。[Garry Shortt]有该工具的视频演示，您可以在下面看到。

如果你习惯于常规编程，你可能会发现梯形逻辑有点令人沮丧。最初，它是一个继电器逻辑的文档工具，但是已经发展到可以处理现代案例。实际上，它可以帮助你不要把它想成一种编程语言，而是一种绘制继电器原理图的工具。例如，触点可以是常开或常闭的，并且串联或并联以形成 and 和 or 门，而线圈可以激活触点。

在工业上，梯形逻辑很好理解，PLC 实时实现。它通常也非常清楚梯子实际上在做什么，比试图阅读代码更清楚，特别是对于那些不一定是程序员的人。

尽管如此，您可能更喜欢普通编程。我们最近发布了一篇关于 Arduino 和 PLC 之间摊牌的帖子。如果你读了评论，很多人认为游戏场有利于 PLC，但这仍然是一个有趣的比较。如果你想要真正的硬件，通常的中国来源甚至是[开源 PLC](https://hackaday.com/2016/06/07/openplc-is-ready-for-hacking/) 都有便宜的选择。
