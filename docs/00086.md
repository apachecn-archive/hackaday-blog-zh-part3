# 头部手势跟踪帮助行动不便的学生

> 原文:[https://hack aday . com/2015/09/14/head-gesture-tracking-helps-limited-mobility-schools/](https://hackaday.com/2015/09/14/head-gesture-tracking-helps-limited-mobility-students/)

对于行动不便的人来说，有很多有用的技术。即使是一些可以帮助人们做一些我们大多数人想都不会想的事情，比如开灯或控制电脑，也可以让那些行动不便的人有很大的不同。幸运的是，[Matt]一直致力于使用[网络摄像头和深度摄像头来让人们做到这一点](https://sites.google.com/site/hardwaremonkey/home/headgesture)。

[Matt]发现使用网络摄像头而不是深度摄像头(如 Kinect)往往不太显眼，但在区分个人用户的能力方面受到限制，当然，也不具备相同的 3D 功能。然而，无论使用哪种技术，软件实现都是相似的。摄像头可以检测头部运动，并通过模拟击键来相应地控制软件。不过，深度相机对用户来说更友好一些，允许用户以任何他们觉得舒服的方式移动。

这并不是像 Kinect 这样的东西第一次被用来跟踪运动，但对于 Matt 和他在博蒙特学院的工作来说，这是正在进行的研究的一个重要领域。这尤其有用，因为校园里有许多网络开关(如灯)，所以这个软件可以用来帮助人们更容易地与物理世界互动。这个项目可能对任何对追踪运动感兴趣的人都非常有用，即使他们[因为移动原因](http://hackaday.com/2014/10/26/super-smash-bros-gets-a-revamp-with-the-microsoft-kinect/)不使用它。
