# 在旧路由器上黑进一个 USB 端口

> 原文:[https://hack aday . com/2016/01/26/hacking-a-USB-port-on-a-old-router/](https://hackaday.com/2016/01/26/hacking-a-usb-port-onto-an-old-router/)

有时候，黑客不一定要创新才能令人满意。我们认为【daffy】关于[将一台旧 WRT54G 路由器改造成互联网收音机](https://hackaday.io/project/9317-internet-radio-from-a-wrt54g)的说明和视频(嵌入在插播广告下方)值得一看，即使他[走的是一条我们早在](http://www.dd-wrt.com/phpBB2/viewtopic.php?t=64931)时就报道过的和[走过的路。](http://hackaday.com/2010/02/27/wrt54gl-meet-alice/)

黑客本身很简单。[daffy]找到未使用的 USB 数据线，添加一个 5V 电压调节器来提供 USB 总线电源，然后将其全部连接到 USB 声卡。硬件方面，搞定！虽然他在第一个视频中没有涉及软件方面的内容，但我们知道他的方向。

WRT54G 路由器是第一个被大规模黑客攻击的基于 Linux 的商用路由器，并为其编写了开源固件。如果你在你的任何设备上使用 [OpenWRT](https://openwrt.org/) 或 [dd-wrt](https://secure.dd-wrt.com/site/) ，你应该感谢 WRT54G 的早期可根性。无论如何，很有可能[daffy]会为他的 USB 声卡找到软件支持，但我们仍在悬念中等待细节的最终结果。

我们最喜欢的 WRT54G 黑客仍然是一个老掉牙的东西:[将 WRT54G 变成机器人的大脑](http://hackaday.com/2008/08/28/wifi-robot-a-hacked-wrt54gl-rover/)。但那是八年前的事了，所以肯定有更新更好的东西。你见过被 WRT 路由器入侵的最酷的设备是什么？
