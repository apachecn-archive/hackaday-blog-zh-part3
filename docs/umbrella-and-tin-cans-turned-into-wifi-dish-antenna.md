# 雨伞和铁罐变成了 WiFi 碟形天线

> 原文：<https://hackaday.com/2018/04/28/umbrella-and-tin-cans-turned-into-wifi-dish-antenna/>

碟形天线有一些标志性的东西。当非天线人士想象天线时，他们想到的可能是天线。对于许多应用来说，碟形天线的方向性和增益确实有助于伸出手去触摸某人。因此，如果你想接入一个远距离的 WiFi 网络，[这种伞状碟形天线](https://www.youtube.com/watch?v=E1iv3OpeQ1g)可能正是你想要的。

至少从他的 YouTube 频道来看，拓展 WiFi 连接的极限似乎是(安德鲁·麦克尼尔)建设的一个重点。这个便携式可折叠的盘子旨在提高他的一个食堂的性能，这是一个简单的家庭自制 WiFi 天线，使用食品罐作为定向波导。这个盘子是由一个伞式摄影师的闪光反射器的骨架建造而成的；他选择了这个而不是折扣店的雨伞，因为反射器实际上是抛物线形状。反射材料被剥离并用作模板来切割新的金属窗纱材料。它比反光织物坚硬得多，但它在肋骨之间拉紧，仍然可以折叠，至少在某种程度上。用销钉做成一个臂，把 cantenna 馈电喇叭定位在反射器的焦点上；关于 cantenna 本身没有给出太多的细节，但我们假设它在设计上类似于我们在之前介绍过的[cantenna。](https://hackaday.com/2017/12/28/antenna-alignment-and-hunting-rogue-access-points-with-the-esp8266/)

[andrew]还没有进行严格的测试，但从他的商店内部进行的快速 360 扫描显示了几十个 WiFi 信号，大多数信号非常好。我们很有兴趣看看这个反射器能在多大程度上提高 cantenna 的性能。
