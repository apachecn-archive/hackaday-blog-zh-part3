# 蚀刻时髦黄铜铭牌的三种方法

> 原文:[https://hack aday . com/2018/01/25/three-way-to-etch-snazzy-brass-铭牌/](https://hackaday.com/2018/01/25/three-ways-to-etch-snazzy-brass-nameplates/)

正是这些小细节成就了一个项目，一个漂亮的铭牌真的可以将一个复古建筑联系在一起。这种徽章很容易用数控机床制作，但是如果你没有机床，你可以用这些酸蚀黄铜铭牌让化学为你工作。

[开关和杠杆]用来蚀刻黄铜饰板的蚀刻方法，对于任何以前蚀刻过 PCB 的人来说都是非常熟悉的。氯化铁在黄铜上的效果和在铜上一样好，而且[开关和杠杆]很好地解释了蚀刻过程的化学性质，并提供了一些用粉末氯化铁制作蚀刻溶液的技巧。但是下面视频的肉是三种不同掩蔽方法的头对头测试。

第一种方法是用激光打印机和从杂志上撕下来的光面纸来制作一个面具。使用办公室层压机将墨粉转移到黄铜上，在蚀刻前轻轻摩擦纸张。对于另外两个候选人，他使用激光雕刻机在一个案例中去除纯黑色喷漆的面具，或者在另一个案例中将特殊的激光标记漆转化为面具。

我们不会破坏惊喜，因为哪一个给出了最好的结果，但我们认为你会对制作经典铭牌的简单程度感到满意。你也可以使用[电解方法进行更深的蚀刻](https://hackaday.com/2017/08/05/ink-filled-machine-badges-score-respect-for-your-gear/)，但是我们认为酸蚀刻在偶尔使用时更容易实现。
