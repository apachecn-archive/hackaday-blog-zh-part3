# Hackaday 奖参赛作品:Zappotron 超级测序仪

> 原文：<https://hackaday.com/2017/04/13/hackaday-prize-entry-zappotron-super-sequencer/>

如果你想尝试弯曲电路，你从哪里开始？也许你在旧货市场发现了一个被丢弃的音乐玩具，然后四处乱翻，也许你找到了定时电路，稍微拉一下就产生了一个弯音。加几根线，看看你能做什么有趣的事情把 A 点和 B 点连接起来，那种事情。

我们中的许多人都以这种方式度过了一个愉快的下午，尽管可能很少有人取得了很大的成就。然而[拉塞尔·克莱默]必须坚持不懈地成为一名杰出的电路折弯师*，因为他的最新项目是我们见过的最成功的电路折弯项目之一。*

 *[Zappotron 超级音序器是一种模拟音序器](https://hackaday.io/project/20373)。除了那句话根本没有传达它真正是什么，它是一个模拟音序器，有四个声源:两个磁带走带设备，一个 4046 振荡器和一个电路弯曲的拼写家教玩具，它的音序器组件由任天堂光枪和 CRT 屏幕控制。

你可能会想，你可以在一台现代单板计算机上相对轻松地完成所有这些，但这个项目的特别之处在于，他只用逻辑芯片和二极管逻辑门实现了它，除了拼写玩具中的微处理器之外，没有任何微处理器。构建日志详细介绍了所有电路，我们必须告诉你，这是一件艺术品，展示了他对模拟电路和数字逻辑的精通。

最重要的是，他把它安装在一个华丽的复古控制台上，贴上了复古的浮雕标签。这是一篇高质量的文章，我们建议您花点时间详细阅读。他上传了一个视频演示，如果你想看的话，我们把它贴在休息时间下面。
