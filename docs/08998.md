# 超蓝数控零件夹具

> 原文：<https://hackaday.com/2018/03/29/super-blue-cnc-part-fixturing/>

如果你需要在两个平行的表面上保持压力，简单的夹子是很好的，但是如果你有一个不规则的平面，或者你需要切割它，夹子不是正确的工具。[纽约 CNC]的人制作了一个视频，巧妙借鉴了其他学科。油漆工胶带贴在机器水平安装面的顶部，然后抛光。工件底部也是如此。强力胶被洒在胶带层之间并压在一起，所以现在原料被牢牢地固定在工具架下面。

视频中有部分零件是机械加工的，下面可以看到，附着力保持没有任何问题。他们切割的一个样品很难在不损坏或停止机器的情况下保存。公认的观点是，强力胶可以很好地粘在像胶带这样稍微多孔的表面上，但它不能粘在像金属这样光滑的表面上。在切割结束时移除无残留胶带也比胶水更干净快捷。

如果你还没有开始接触牙齿，你可以观看我们自己的[埃利奥特·威廉姆斯开始接触数控机器](https://hackaday.com/2017/11/27/cnc-milling-is-more-manual-than-you-think/)或[便携式机器](https://hackaday.com/2017/10/25/hackaday-prize-entry-cnc-mill-lets-kids-engrave-on-the-fly/)，甚至一个孩子都可以使用。
