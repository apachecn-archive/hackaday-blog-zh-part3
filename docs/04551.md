# 与明星辛普森一起建造漂亮的木板

> 原文:[https://hack aday . com/2016/12/23/building-beautiful-boards-with-star-Simpson/](https://hackaday.com/2016/12/23/building-beautiful-boards-with-star-simpson/)

在过去十年左右的时间里，生产少量定制印刷电路板的成本大幅下降。现在，您不必使用与某个工厂相关的软件，您只需将 Eagle 或 KiCad 文件放到订单上，然后点击“提交”即可。

有了这种新发现的能力，黑客和 PCB 设计者开始制造漂亮的电路板。一片 FR4 不再只是填充器件的介质，而是覆盖阻焊膜和丝网的画布。

在过去的一年里，明星辛普森一直致力于一个项目，使电子艺术成为现实。[她的电路经典](https://www.crowdsupply.com/star-simpson/circuit-classics)从福里斯特·米姆斯的*电子入门*笔记本中汲取原创艺术，将它们变成功能性 PCB。这是一个工具包，一个教育玩具，一件玻璃纤维艺术品，集所有功能于一身。

在 [2016 Hackaday 超级大会](https://hackaday.io/superconference/)上，Star 给出了她制作漂亮 PCB 的技巧和诀窍。这里发生了很多事情，从可变厚度的阻焊膜，丝网印刷上的矢量艺术，甚至是在美术馆比在电子工作台看起来更自在的多色板。

多氯联苯可以艺术化的想法当然是现代的创造。大多数时候，电子产品都藏在一个塑料外壳里。任何人都没有机会看到这些板，而且在任何情况下，在做出任何美学考虑之前，板首先需要正确地工作。

尽管有这些事实，如果你追溯得足够远，你可以找到大量的例子，其中一些艺术制作成几层蚀刻铜。70 年代和 80 年代的手绘板以俯冲曲线、轮廓样条和只有人类才能绘制的填充为特色。所有这些痕迹都有其用途，但这种介质允许非常有机的电路。在 80 年代和 90 年代，计算机被用来负责绘制迹线，PCB 设计的艺术性迅速演变为一种易于识别的模式:顶层是水平的，底层是垂直的，所有角度都是四十五度。

在过去的几年里，自制者已经可以获得和专业人员一样的工具，随之而来的是对材料和技术的探索。需要考虑的事情很多。PCB 的基底可以是玻璃纤维、聚四氟乙烯、纸板，甚至陶瓷。阻焊和丝印颜色的深度仅次于你钱包的深度。甚至表面可以是美丽的亚光金或闪亮的反光银。

Star 确实向观众中的每个人提供了一些建议。如果你想涉足艺术板的世界，Eagle 是个不错的选择。它不仅做曲线，还有脚本来处理负丝印区域，并且你可以导入位图和矢量文件。

鉴于[星的赛道经典](https://www.crowdsupply.com/star-simpson/circuit-classics)的成功，她有哪些小技巧？迭代，首先。在进入大规模生产之前，她有一张覆盖着紫色原型的桌子。当她到达那里时，董事会所对紧靠赛道的福里斯特·米姆斯的艺术有些困惑。她向 fab 解释说，这不是文件，这是应该印在 PCB 上的艺术。与董事会的沟通至关重要。
