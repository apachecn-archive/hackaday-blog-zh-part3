# 拧上一个灯泡需要多少架无人机？

> 原文：<https://hackaday.com/2016/11/28/how-many-drones-does-it-take-to-screw-in-a-lightbulb/>

想象一下。你在这里，穿着休闲裤很舒服。在你的休息室里闲逛。突然，你听到远处有嗡嗡声。开始很安静，后来声音更大了。你脑袋里的一个灯泡亮了。

你忘了你已经与家得宝和 Dewalt 合作安排了亚马逊无人机维修服务。他们打破窗户，用碎片喷你。他们在墙上画斑点。给你衣柜里的品牌拍照。换灯泡。在你的玻璃伤口上贴上创可贴。拿起碎片离开。在他们出去的路上修理它。

可怕。

当然，在这个黑暗的未来到来之前，第一步是看看它是否可以实现；这就是[马雷克·巴辛斯基]和一个朋友后来完成的许多破碎的灯泡。他们使用一架现成的无人机，顶部粘有三个弹性尖头，一次又一次地尝试拧开和拧上灯泡。他们首先尝试了一种更轻的无人机，但最终切换到一种更坚固的型号。

过了一段时间，他们终于成功了，所以这是可能的。下一步，自动化。休息后的视频。
