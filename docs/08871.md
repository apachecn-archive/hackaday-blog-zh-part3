# 你很想上传你的大脑吗？

> 原文：<https://hackaday.com/2018/03/16/are-you-dying-to-upload-your-brain/>

人体冷冻——冷冻人体以备日后复活——多年来一直是科幻小说的主题。也许你想治愈目前无法治愈的疾病，或者你只是想看到未来。当然，忽略为什么有人想解冻一个 500 岁的人，没有人有成熟的技术来解冻这些尸体。你基本上是在打赌，在你的冰箱坏掉之前，科学会找出答案。由 Y Combinator 投资的一家名为 [Nectome](https://nectome.com/) 的新公司想要改变你对保存的想法。他们不会冷冻你，而是给你注入防腐剂，保护你的大脑，包括科学家目前认为包含你记忆的精细结构。

Nectome 的策略不是像传统人体冷冻那样让你复活。他们认为对你保存的大脑进行高清晰度扫描的技术很快就会出现。这些扫描可能会让未来的科学家在模拟中重建你的大脑。不过，这和复活并不完全一样。至少我们不认为是这样。

该公司将他们的过程称为存档你的大脑，尽管因为这个过程会杀死你，你将需要合法的安乐死资格来利用这个过程。有一种观点认为，被称为连接体的结构保存着你的记忆，这些记忆通过这个过程得以保存。你可以看看下面这个主题的 TED 演讲。

像所有这些保存策略一样，有许多未知因素。我们不确定所有必要的东西都存在，因为我们并不完全了解大脑是如何工作的。我们也不知道是否有人会知道如何使用这些大脑来模拟你的存在。还有一个长期存在的问题，一觉醒来发现自己被一个邪恶的霸主所奴役，或者你的身体是一艘为极权主义政权服务的战舰。

例如，越来越多的证据表明，你的大脑可能真的是一台量子计算机。这可以解释很多，但即使它是错的，也没有办法知道那里有没有其他完全不了解的东西。

那么怎么样呢？你会让他们杀了你来保存你的大脑吗？以后还会有人费心开机复制你吗？如果有，为什么？毕竟，根据所有聪明人的说法，你只会醒来为我们的机器人统治者服务。如果你只是想刺激你的大脑，[试试 DARPA](https://hackaday.com/2017/11/13/shockingly-darpas-brain-stimulator-might-not-be-complete-nonsense/) 。
