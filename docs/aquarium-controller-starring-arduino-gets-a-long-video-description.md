# Arduino 主演的水族馆控制器获得一个长视频描述

> 原文：<https://hackaday.com/2018/07/01/aquarium-controller-starring-arduino-gets-a-long-video-description/>

有句老话说，鞋匠的孩子没有鞋穿。有时我们会有这种感觉，因为我们忙于为别人或演示设计东西，没有时间去做我们想要的东西。[蓝刃鱼]想要建立一个基于 Arduino 的水族馆控制器。到目前为止，他已经在 14 个视频中详细介绍了这个系统，它看起来很可靠。

这也不仅仅是一个简单的控制器。这是一个模块化设计，有一个 Arduino Mega 和许多 I/O，适合一个大鱼缸。有控制加热器，风扇，灯，造波器，甚至关闭阀门。该系统可以在晚上模拟月光，并有一个液晶显示器和按键。还有一个以太网端口和一个 Raspberry Pi 组件，用于创建 web 界面、数据存储和配置系统。甚至故障保险也被设计到系统中，所以你不用煮或冷冻昂贵的鱼。难怪拍了 14 个视频！

我们真的很喜欢 moonlight，它有 32 个带自定义开关和移位寄存器的 led。令人惊讶的是，发光二极管前面是一张月球照片。我们想知道 3D 打印的 lilthophane 会如何，它可能会更多地扩散 led。因为有了移位寄存器，只需几个引脚就能轻松控制所有的灯。

显然，让鱼了解外面的环境光线对养鱼的人来说很重要。因为有安全措施，一个坏的软件错误不会把他的鱼煮熟。尽管看起来如此，显然这个[升级鱼缸](https://hackaday.com/2013/03/11/coffee-pot-aquarium-keeps-fish-warm-without-cooking-them-hopefully/)也不会。
