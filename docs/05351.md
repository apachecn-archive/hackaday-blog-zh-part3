# 破塑料？没问题！

> 原文:[https://hackaday.com/2017/03/17/broken-plastic-no-problem/](https://hackaday.com/2017/03/17/broken-plastic-no-problem/)

当一个电脑机箱经历了几十年的发展，从一个新玩具，到一个不受欢迎的遗物，再到被重新发现并落入一个爱好者的手中，它不可避免地会在这个过程中受到一些损坏。它会磨损，可能会破裂，而且经常会坏掉。如果它在易贝出售后遭遇了国际快递员的考验，那么破产的可能性就会大大增加。

如果你收到一个邮寄的老式电脑，发现它有裂缝或坏掉了，不要害怕。【Drygol】有一个解决方案，[用烙铁](https://www.retrohax.net/broken-plastic-case-no-problem-2/)进行塑料焊接的指南。

彻底清洁后，技术是将裂缝的两侧固定在一起，沿着裂缝运行熨斗以将塑料熔化在一起，并在塑料固化前将溢出的塑料刮回到产生的沟渠中。只要仔细打磨，涂上一点聚酯油灰，再喷上一些油漆，这个破箱子就能恢复原状。

有一个视频显示了这个过程，在这个案例中，修复了 Commodore 64 外壳上的裂缝。
