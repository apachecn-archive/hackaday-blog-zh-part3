# 被砍的钻石制造双原子收音机

> 原文：<https://hackaday.com/2016/12/20/hacked-diamond-makes-two-atom-radio/>

过去，人们热衷于将无线电接收器塞进 Altoid 的罐子里，或者用刀片和铅笔头制作一个微型水晶接收器。但是哈佛大学的研究人员已经远远超越了小型化的成就，他们用一颗经过切割的钻石制成了纳米级的调频接收器。

与所有此类研究一样，在[Marko lon ar]实验室进行的实验远没有新闻稿所说的那么简单。虽然双原子电池确实是探测器的最小材料，但在下面的视频中听到的来自[安迪·威廉姆斯]的季节性最爱的设备使用了数十亿个氮空位(N-V)中心。氮钒中心用氮原子取代金刚石晶体中的碳原子；这会在晶格中产生一个“空位”,并赋予钻石对微波敏感的光致发光特性。当被绿色激光泵浦时，2.8 GHz 范围内的入射 FM 无线电波被转换成 AM 荧光信号，该信号可以被光电二极管和放大器检测到。

[全文](http://nano-optics.seas.harvard.edu/files/nano-optics/files/Diamond_Radio_PRApplied.6.064008.pdf)有所有的细节，表明无线电可以在极端压力和温度条件下生存，并描述了该系统的潜在应用。在这一点上，它远不是一个家庭游戏玩家的黑客，但它是一个巧妙的技巧，值得期待未来的开发。与此同时，这里的[是一个偶然的 FM 收音机](http://hackaday.com/2016/12/04/fail-of-the-week-the-accidental-fm-radio/)，有一个相当小的足迹。
