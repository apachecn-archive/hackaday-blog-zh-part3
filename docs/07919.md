# 令人难以置信的缩小硬币电池包

> 原文：<https://hackaday.com/2017/12/07/the-incredible-shrinking-coin-cell-battery-pack/>

你的硬币电池挑战赛项目进展如何？你只能使用一个，但是 [Hackaday alum【杰瑞米·S·库克】有一个很好的方法将硬币电池包装成一个光滑的小电源包](http://www.jcopro.net/2017/12/05/howto-video-make-coin-cell-battery-pack-shrink-wrap/)无论你需要一个、两个甚至四个。

[Jeremy]正在制作一个无线 Wii 双节棍，所以他需要一个小电池，在控制器主体的范围内不会短路或被刺穿。单个硬币电池座已经有点笨重，他需要串联使用两个。他想，为什么不试试收缩包装呢？这里唯一的缺点是，最大的管来与你的平均热收缩多包可能有点太紧，以适应他们，所以你可能不得不买更多的(啊，呸！).

在尝试了几种方法来实现导线和裸硬币电池表面之间的良好连接后，[Jeremy]决定大量剥离绞合线，并将长股缠绕在末端，形成导电棉签。这很好地滑入硬币电池和预缩管之间。多加热一点会有很好的连接，一些热胶水会固定电线。点击休息时间，观看他的构建视频和他尝试过的其他连接方法。你想到更好的了吗？请在评论中告诉我们。

离开工作台稍远一点，你可能会想到类似于[几个月前我们看到的这个眼睛瞪得大大的电池座](https://hackaday.com/2017/08/15/clever-battery-holder-hack-makes-brilliant-earrings/)。
