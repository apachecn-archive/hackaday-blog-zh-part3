# 拱形反应堆黑客空间正在移动！

> 原文：<https://hackaday.com/2016/01/01/arch-reactor-hackerspace-is-moving/>

当你的黑客空间对于它的建筑来说变得太大的时候会发生什么？嗯——你可以接管你楼里的其他单元，或者一起搬到更大的楼里去！

大约三年前，我们参观了位于密苏里州圣路易斯的 Arch 反应堆。目前的设施是 2400 平方英尺，在过去几年里变得有点拥挤。他们将搬到位于斯科特大道 2215 号的新大楼，该大楼的面积是现有设施的两倍多，高达 5100 平方英尺！

正如你所想象的，将这样规模的黑客空间搬到一个新的建筑里并不是一件容易的事情，但是他们的社区是强大的，他们仍然在继续黑客活动，甚至在搬家的时候！如果你的黑客空间在不久的将来有所发展，你可能会想记下来，在他们的博客上学习一些经验。

他们正在开展一项 Indiegogo 活动，试图帮助支付一些搬家费用。祝你们好运！
