# Tron 的丝网印刷电致发光 t 恤

> 原文：<https://hackaday.com/2016/01/21/screen-printing-electroluminescent-t-shirts-from-tron/>

去参加足够多的安全会议、博览会和节日，你会看到一些疯狂的可穿戴电子产品。大部分都是眨眼的，大部分都是用 led。电致发光面板用于可穿戴设备，但这是一个利基市场——这种面板有点贵，你必须处理高频交流电，而不是基于 LED 的可穿戴设备必须应对的更简单的“插入一个 LiPo here”电路。

尽管如此，电致发光面板是很酷的，而且由于电致发光面板是如何制造的，你可以用丝网印刷法印刷电致发光显示器。这就是 AMBRO Manufacturing 的一些人最近做的事情:[直接在服装上丝网印刷电致发光灯](http://www.ambrolabs.com/wearables/screen-printing-electroluminescent-lights-on-garments)。创的 t 恤变成了现实。

EL 面板和 EL 导线实际上只是三个独立的部分:某种导体、磷光体和另一种导体。让高频交流电流通过导体，磷光体就会发光。EL 线是一根包覆荧光粉的粗铜线，外面包裹着一根非常细的铜线。电致发光显示器由涂有导电 ITO 的玻璃或塑料制成。这是一个相对简单的结构，非常适合丝网印刷。事实上，最早的 EL 显示器之一——阿波罗制导计算机的用户界面 DSKY——使用了丝网印刷的七段 EL 显示器。

AMBRO 的人现在只有一个概念证明，但这是一个完全丝网印刷在织物上的电致发光设计。为了点亮它，t 恤需要一个逆变器，但这是 Tron 恤的开端。
