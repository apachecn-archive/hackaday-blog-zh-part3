# 手势控制的预兆之剑给人超越视力的视力

> 原文:[https://hacka day . com/2017/02/14/手势控制-预兆之剑-超视力/](https://hackaday.com/2017/02/14/gesture-controlled-sword-of-omens-gives-sight-beyond-sight/)

80 年代是科幻卡通的黄金时代。我们有变形金刚，伏特龙，当然，还有雷猫。第三地球上的类人猫科动物的故事一直困扰着一些黑客，包括[胡安·加西亚]。胡安已经将他的粉丝挖掘到了一些令人惊讶的东西——他建造了自己版本的 Lion-O 标志性的预兆之剑。建模和 3D 打印工作都是一流的。虽然这把剑没有雷神之眼作为能源，但它确实有一些非常聪明的电子设备。它也是今年[科幻大赛](https://hackaday.io/project/19623-sword-of-omens-w-sight-beyond-sight-action)中令人惊叹的作品之一。

[Juan]想从电视节目中创造出“视线之外的景象”的效果。他从 MPU6050 六轴陀螺仪+加速度计开始。这些芯片通常位于四轴飞行器控制器的核心。Arduino Pro Mini 读取传感器数据。当剑被垂直举起时，它会打开一排 WS2812 LEDs。发光二极管复制了卡通版剑的脉冲眼睛效果。

[Juan]这真是一个漂亮的建筑。整把剑由 18650 锂离子电池供电。电源控制和充电电路都隐藏在 3D 打印中。当需要给眼睛充电时，可以拧开手柄，露出一个 USB 充电口。点击过去的休息，看看这个黑客的行动！
