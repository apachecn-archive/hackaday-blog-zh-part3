# 电容很简单吧？

> 原文：<https://hackaday.com/2016/07/09/capacitors-are-simple-right/>

人们很容易将电阻和电容等无源元件视为无聊的话题。凯米特电容器公司的[詹姆斯·刘易斯]不会同意。他给[做了一个关于电容器技术的演讲，既平易近人又有深度](http://phy.supplyframe.com/2016/04/17/james-lewis-kemet-theyre-just-capacitors/)。

就像我们使用的其他组件一样，我们总是认为它们是完美的。但是就像导线有我们经常忽略的电阻和电感一样，电容也有不同的不完美特性，你需要注意。

例如，陶瓷电容会随着时间的推移而失去电容。不同陶瓷材料具有不同的温度敏感性。铝电容不会永远存在。施加在电容器上的电压可以改变其 50%的值。

[James]还谈到了聚合物电解质和超级电容器。他的燃眉之急是:你应该将电容降额 50%的旧准则有什么道理吗？想知道他怎么想的吗？看下面的视频。说到燃烧，他谈到了钽电容器这个敏感的话题。顶部的图像是 Kemet 在自己的零件上以远远超出规格的相反极性运行的测试。所有的都被烧坏了，但只有一些看起来烧焦了。这是一个很值得一看的神秘话题。

顺便说一下，这个演示来自硬件开发人员的说教式 Galactic——在 Supplyframe 的旧金山办公室举行的一系列硬件讲座(而且，根据记录，Supplyframe 恰好拥有 Hackaday)。如果你想了解更多关于电容的信息，你可能会喜欢我们最近关于[电容类型](https://hackaday.com/2016/06/21/capacitors-made-easy-the-hackaday-way/)的报道。如果超级电容器引起你的兴趣，你总是可以抓一些[棉绒和狗毛](https://hackaday.com/2016/04/30/lint-and-dog-hair-supercapacitor/)。
