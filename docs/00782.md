# 听好了:IPhone 黑客诊断暖通空调

> 原文:[https://hack aday . com/2015/11/30/listen-up-iphone-hack-diagnostics-HVAC/](https://hackaday.com/2015/11/30/listen-up-iphone-hack-diagnoses-hvac/)

我们都知道这样一个人(或者，在某些情况下，我们就是这样的人)，他可以听到一辆车在跑，然后说:“是的。需要一个升降机调整。”一家名为 Augury [的初创公司旨在用一款 iPhone 应用程序](http://www.technologyreview.com/news/543786/this-gadget-can-tell-whats-wrong-with-your-air-conditioner-by-listening-to-it/)取代这一技能。

针对商业安装，技术人员将一个磁性传感器安装在相关机器的机身上。该传感器连接到一个名为 Auguscope 的定制盒子上，该盒子收集振动和超声波数据，并通过 iPhone 将其转发到后端服务器进行分析。在某些情况下，移动传感器甚至可以让后端确定故障的位置。后端使用的比较数据包括类似机器上的参考数据以及有关该机器的历史数据。

我们不禁想知道为什么黑客社区已经很久没有构建这样的东西了。我们有足够马力的 CPU 来收集音频(或其他传感器数据)并处理它。例如，如果你想诊断汽车，你可以[找到足够容易的声音](http://mycarmakesnoise.com/)用于训练。

有一些关于这个主题的学术论文，但是我们没有发现黑客风格的项目。也许这些评论会充实一些内容。如果没有，这似乎是一个成熟的实验领域。电脑的声学分析可以识别风扇、电源和硬盘问题(只要您没有使用固态硬盘)。任何有活动部件的东西都可能是候选者。

我们在档案中发现的最接近的东西是一个混合磁/音频系统来绘制电机平衡图。我们希望在 Hackaday.io 上看到更多这方面的工作。
