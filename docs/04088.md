# 超大数字时钟可用于超大 3D 打印机

> 原文:[https://hack aday . com/2016/11/10/really-big-digital-clock-finds-use-for-really-big-3d-printer/](https://hackaday.com/2016/11/10/really-big-digital-clock-finds-use-for-really-big-3d-printer/)

怎样才能让[成为一个真正的大数字时钟](https://www.youtube.com/watch?v=viqP5sbzNGM)？如果说(伊万·米兰达)的发明有什么标准的话，那就是它需要一台非常大的 3D 打印机、一大堆 Neopixel 条和一吨热熔胶。

看起来[Ivan]的大尺寸时钟主要是为他最近完成的[大型床定制 3D 打印机](https://www.youtube.com/watch?v=EJyjM9ywYbI)做的练习，这本身就是一个值得检验的项目。但这是一个相当雄心勃勃的项目，并且有一些增强的可能性。四个七段显示器中的每一个都是单独印刷的，黑色背景，半透明白色段，五个 RGB LEDs 凹槽。四位数字和冒号分隔符一起显示，ESP8266 从 NIST 服务器获取时间并驱动这些段。[伊万]的项目真正有趣的是，他限制自己在一周内完成每个项目。这解释了他使用大量热胶的原因，并留下了改进的空间。我们很想看到这款显示器装在一个漂亮的胡桃木盒子里，配有一个巨大的红色漫射镜头。就目前的情况来看，这无疑是一个声明。

我们以前也展示过[其他超大的七段显示屏](https://hackaday.com/2016/09/10/careful-crafting-makes-diy-7-segment-display-shine/)，但很少有像这个这么大的。
