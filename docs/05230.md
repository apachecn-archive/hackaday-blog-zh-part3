# 简单的大理石机器吸引眼球

> 原文：<https://hackaday.com/2017/03/05/simple-marble-machine-captivates-the-eyes/>

大理石机器是那种人人都爱的无用机械。他们的唯一目的是通过不同的路径发送弹珠，以获得观赏乐趣。它们可能是极其复杂的装置，有时这正是关键所在。然而，即使是一个简单的机械装置也能让人赏心悦目。[Denha]刚刚上传了他的最新作品，使用弹簧作为电梯和简单的之字形路径。

结构相对简单，一个适合钢珠大小的弹簧被用作升降机。弹簧由一个小型电动马达通过几个齿轮驱动，弹簧旁边有一个木制的弹珠之字形通道。弹珠随着弹簧上升，然后在无尽的旅程中沿着木头小路返回。

我们认为，一个认真的黑客一生中至少应该造一次大理石机。我们已经贴出了其中的几个，从简单的[到其他更复杂的](http://hackaday.com/2015/03/21/your-marble-machine-doesnt-need-to-change-the-world/)[设计](http://hackaday.com/2012/04/18/multiple-marble-machine-mayhem/)都需要精心的工艺。[Denha]的 Youtube 频道充满了启发你的第一个项目的好主意。无论如何，看着一台大理石机器工作是一种很好的放松体验。
