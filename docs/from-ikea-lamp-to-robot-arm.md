# 从宜家灯具到机器人手臂

> 原文：<https://hackaday.com/2017/01/04/from-ikea-lamp-to-robot-arm/>

我们习惯于将日常家居用品改造或增强成新的、令人兴奋的形式，这是它们的原始设计者从未想过的。这项工作的一个特别主题来自宜家黑客社区，他们从瑞典家具店获得产品，并将其作为工作的基础。

IKEA hacking 的一个特别有创意的项目来自[anastas.car]，[一个基于 Ikea Tertial lamp](https://hackaday.io/project/19235-tertiarm-3d-printed-robot-arm) 的低成本 3D 打印机器人手臂。有问题的灯是[一种相对便宜的弹簧平衡台灯](http://www.ikea.com/gb/en/products/lighting/work-lamps/tertial-work-lamp-silver-colour-art-40370283/)，当从另一个角度看时，当与关节处的五个伺服系统的 3D 打印伺服支架结合时，它已经为一个 5 自由度机器人手臂准备好了所有的金属加工。由此产生的设计[拥有 Thingiverse](http://www.thingiverse.com/thing:1652309) 上所有可用的文件，从我们在下面发布的视频来看，这是一个相当有效的手臂。

快速搜索会发现宜家黑客是这里的一个常见功能，我们为您带来了很多。仅是精选品就有来自 Craigslist finds 的[宜家台式激光切割机](http://hackaday.com/2014/05/25/ikea-desk-laser-cutter-combo/)，被用作 3D 打印机框架的[宜家桌子，以及相当漂亮的](http://hackaday.com/2016/12/22/ikea-table-3d-printer/)[宜家台灯架](http://hackaday.com/2015/05/24/absolute-overkill-ikea-lampan-lamp-hack/)。宜家黑客社区是一群足智多谋的人，我们期待他们有更多的创造。
