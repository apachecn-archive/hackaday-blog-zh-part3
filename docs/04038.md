# 忘了锂电池技术吧，煮个土豆就好

> 原文:[https://hack aday . com/2016/11/03/forget-lithium-battery-technology-just-boil-a-potato/](https://hackaday.com/2016/11/03/forget-lithium-battery-technology-just-boil-a-potato/)

这些研究人员如此认真地对待这一发展，我们不禁产生了怀疑，也许他们都深深卷入了一场赌注，看谁能获得资助并完成最荒谬的技术进步研究。

我们大多数人都有过这样的经历:一位科学老师拼命地试图减轻以标准化考试为中心的科学教育的苦差事，他戏剧性地将一枚铜钉和一枚锌钉放进土豆或柠檬中。“看，我们可以用这个伏特计测量电压。如果你在考试中被问到伏特计是什么，这是一个你们没有足够的实验基础来理解的定义，”老师会说，因为他们成为真正的教育家的梦想被粉碎了一点。

不管怎样，这些研究人员用他们的资助发现，通过煮沸八分钟来降低土豆的内阻，可以提高土豆电池的效率。我们不确定穿着实验服戏剧性地盯着一个沸腾的土豆是否有必要，但它似乎有所帮助。

当然，除了研究人员继续谈论他们如何希望这种“基于土豆的能源技术”将帮助第三世界照亮他们的家园之外，这与 Volta 的原始电池没有什么不同。伏打只是想给他的酷科学家朋友留下深刻印象。除了贫穷的人可能会吃土豆并出售纯铜和锌板来换取更多土豆这一事实之外，它至少会提供一个有趣的课堂演示，说明电池的内阻到底是什么——如果我们能够以某种方式将由此产生的理解转化为多项选择扫描单的话。休息后的视频。
