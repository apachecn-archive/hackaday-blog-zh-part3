# 雷达给了电脑触觉

> 原文:[https://hack aday . com/2016/10/21/radar cat-gives-computers-a-sense-of-touch/](https://hackaday.com/2016/10/21/radarcat-gives-computers-a-sense-of-touch/)

到目前为止，人类在通过触摸识别物体的能力上占有优势。但不会太久。圣安德鲁斯大学的[圣安德鲁斯计算机人机交互小组(SACHI)]使用谷歌的 Soli 项目(一种可以检测最细微手势输入的微型雷达)开发了一个名为 RadarCat 的新平台，该平台使用[芯片来识别材料，就像通过触摸](https://gizmodo.com/students-hacked-a-chip-to-give-your-smartphone-a-sense-1787446499)一样。

意识到不同的材料会向芯片返回独特的雷达信号，【SACHI】团队将其与他们的识别软件和机器学习过程相结合，使 RadarCat 能够实时准确地识别一系列材料！它还可以显示关于对象的附加信息，例如食品的营养信息，或者消费电子产品的产品信息。该视频展示了 RadarCat 是如何学习了一系列令人印象深刻的材料，甚至是特定的身体部位。天网还会远吗？

同样，这可以为机器人提供触觉，在某种程度上，可以与我们人类识别物体的能力相媲美。这在许多机器人领域都有应用:工业机器将能够识别可能需要更多力量才能举起的物体的材料成分，而可穿戴版本将帮助残疾人。这是一项值得关注的技术。

虽然这项技术离广泛使用还有一段距离，但今天你仍然可以通过 [Touché](http://hackaday.com/2012/05/07/turning-anything-into-a-touch-sensor/) 将任何东西变成触摸传感器。
