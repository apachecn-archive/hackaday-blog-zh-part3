# retro tech tacular:1965 年的台式电脑

> 原文:[https://hackaday . com/2015/12/08/retro tech tacular-a-desktop-computer-from-1965/](https://hackaday.com/2015/12/08/retrotechtacular-a-desktop-computer-from-1965/)

大约在施乐公司在 PARC 制造奥拓的八年前，在苹果首次亮相的十多年前，意大利商业设备制造商 Olivetti 生产了真正的台式个人电脑。当 Olivetti 于 1965 年在纽约市的一次商业会议上首次展示这款打字机大小的奇迹时，人们完全惊叹于这款小巧、独立的设备可以执行与当时笨重的房间大小的主机相同的功能。有些人确信它只是藏在窗帘后面的一台更大的机器的一个小输入设备。

但是革命性的 Olivetti Programma 101 不是闹着玩的。它能执行标准的四合一运算，并能处理平方根和绝对值计算。Olivetti 有 16 条跳转指令和 16 条条件跳转指令，这使它牢牢地占据了状态机的领地。程序可以打印在一卷纸上，或者长期存储在长磁卡上。

![Magnetostrictive acoustic delay line memory. Image souce: The Old Computer Hut](../Images/d7d33c58c5053e93c4d0ae35aeb1b486.png)

Magnetostrictive acoustic delay line memory. Image source: [The Old Computer Hut](http://www.oldcomputers.arcula.co.uk/calc3.htm)

为了在内部存储数据，该程序使用了[磁致伸缩声学延迟线存储器](https://www.youtube.com/watch?v=N9cUbYII5RY)，其中一根细线由电磁体内的传感器扭曲。每一个进来的零线都向一个方向扭曲，每个零线都向另一个方向扭曲。这些扭转波沿着电线传到另一端的压电传感器。信号被放大、整形，然后被送回循环，直到被程序调用。

以 3200 美元的入门价格，全世界的人和企业都争相购买这些机器。奥利维蒂无法满足这种荒谬的需求，尽管他们每年能生产大约 10，000 个。

![programma-guts](../Images/fbbc19aea2732f18b65877e750131cc8.png)

下面有几个视频。第一个是使用演示者创建的两个程序的程序演示。第一个程序相当简单，它计算运行时输入的两个极限之间的斐波那契数列。第二种更费力。对于输入的任何数字，如果该数字是质数，程序将返回 1。如果不是，程序计算并返回最小的除数。

希望你能得到这些宏伟的野兽之一？这个我们帮不了你，但是这里有一个模拟器。如果你真的找到了，这里有[手册](http://www.google.com/url?q=http%3A%2F%2Fwww.classiccmp.org%2Fdunfield%2Fcalc%2Folpro101.pdf&sa=D&sntz=1&usg=AFQjCNFZ_pDiOrQJb5idZjKvyEqIAqu4EA) (PDF)。哦，你可以[用那些 101 合 1 电路训练器中的一个制作你自己的延迟线存储器](http://hackaday.com/2014/01/07/acoustic-delay-line-memory/)。
