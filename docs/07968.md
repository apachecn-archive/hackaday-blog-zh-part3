# 外星自主着陆系统登陆火星

> 原文:[https://hack aday . com/2017/12/13/地外自主着陆器系统着陆火星/](https://hackaday.com/2017/12/13/extraterrestrial-autonomous-lander-systems-to-touch-down-on-mars/)

人类的未来在火星上。在 SpaceX、波音、美国国家航空航天局和其他所有国家航天计划之间，我们要去火星。随之而来的一个问题是:飞往火星相对容易，但在另一个星球的表面着陆一个大的有效载荷要困难得多。特别是火星，它很棘手:它有足够的大气层，你需要在它周围设计，但不足以让我们只使用降落伞将几吨重的东西带到表面。最重要的是，我们需要在一个非常小的着陆椭圆内着陆我们的栖息地和特斯拉跑车。登陆火星是艰难的，最聪明的人正在为此努力。

在今年的 Hackaday 超级大会上，我们从美丽的帕萨迪纳喷气推进实验室的工程师 Ara Kourchians(你可能知道他叫[Arko])和 Steve Collins 那里了解到在火星上着陆有多难。在过去的几年里，[他们一直在研究 COBALT](https://www.jpl.nasa.gov/news/news.php?feature=6780) ，这是一个关于如何使用机器视觉、花式 IMU 和一系列传感器在外星世界自主着陆的技术演示器。你可以看看下面他们的超级对话视频。

多年来，有几种方法被用于登陆火星。1976 年的第一次成功着陆——海盗号，只是将着陆器降落在大气层的顶部，希望不要落在一块巨大的巨石上或悬崖边上。好奇号是一辆汽车大小的火星车，已经强劲行驶了五年，但它有点复杂。进入的飞行器有一个偏移质量，当着陆器穿过大气层时，计算机可以绕着它的质心滚动，使它的轨迹有一点偏移。这也是阿波罗模块如何从月球返回，并证明你可以飞砖，只要它没有均匀的密度。

但是现在正在建造火星探测器。火星 2020 探测车目前正在组装中，因此需要新的着陆技术将探测车放在有趣的地质构造旁边。对于火星 2020，这意味着让着陆器在穿过大气层下降时拍摄着陆区域的照片，将这些照片与众多火星轨道飞行器之一创建的地图进行比较，并让着陆器判断它是否会在一堆岩石上着陆。如果着陆器感觉到它将在一个危险的区域着陆，它可以将着陆点转移到几百米以外的安全地带。

COBALT——自主着陆技术的合作混合——是一个改进这项技术的项目。最终，我们会想要在火星甚至是欧罗巴上更危险的地方着陆。这些都是具有挑战性的环境，我们甚至没有木卫二的高分辨率地图。在我们尝试登陆木卫二之前，我们可能不会有木卫二的高分辨率地图。

[![](../Images/12921efda9adcb7b7c28e35a5dfcf6a9.png)](https://hackaday.com/wp-content/uploads/2017/12/cobalthead.png)

The COBALT payload package

为了做到这一点，COBALT 是一个有效载荷包，装载了激光雷达，相机，IMU 和一台强大的计算机，提供对火箭着陆位置的实时传感。去年春天，COBALT 团队实际上有机会在莫哈韦测试他们的有效载荷，乘坐的是 Masten Xodiac 火箭。这枚火箭向上发射，关掉引擎，然后移动到一边，降落在距离地面几百米的发射台上。

这次测试非常成功。你可以从莫哈韦的阿姆斯特朗飞行研究中心查看一些测试视频[，在那里火箭上升，计算出它在哪里，并指引发动机到一个精确的着陆点。](https://www.youtube.com/watch?v=u2Ve6wr9EX4)

我们将有很多方式登陆火星。SpaceX 正在全力以赴提升机身和偏移质心。波音公司很可能会破产。谁知道中国和印度会怎么做。然而，我们最终会到达那里，当涉及到火星或月球以外的世界时，这可能是我们将使用的。
