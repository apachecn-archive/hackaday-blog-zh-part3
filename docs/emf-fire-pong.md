# 电动势火乓

> 原文：<https://hackaday.com/2016/08/11/emf-fire-pong/>

在 2016 年 EMF Camp[上，持续吸引大批观众的装置之一是一个游戏。不过这不是一个传统的电脑游戏，而是一条气体喷射线，两个玩家必须沿着这条线以不断增加的速度在他们之间喷射火焰，直到一个玩家错过了回报。这是由](https://www.emfcamp.org/)[诺丁汉 Hackspace](http://nottinghack.org.uk/) 的成员创建的 Fire Pong 游戏，虽然似乎还没有关于它的在线报道，但他们已经发布了足够多的图片让我们推断出它的构造。

一个由气体管道和喷嘴构成的[网络，所有阀门都连接到一个](https://www.flickr.com/photos/nottinghack/28312604975/)[清晰标注的控制面板](https://www.flickr.com/photos/nottinghack/28417342921/)上，似乎通过连接到[一个继电器板](https://www.flickr.com/photos/nottinghack/28233845051/)的电磁阀来控制气体流量，这个继电器板似乎是由一个 Arduino Pro Mini 驱动的。蝙蝠对于戏剧效果来说是巨大的，但是包含加速计来感知玩家的挥击，并将信息发送回气体控制电路。一对[大得多的火焰发生器](https://www.flickr.com/photos/nottinghack/27671270584/)指示拉力赛结束，分数显示在一个大的 LED 记分牌上。这个系统很可能比我们从这些图片和[各种组件的镜头](https://www.flickr.com/photos/nottinghack/27587254663/)中收集到的更多，但是到目前为止，我们对它们的细节还一无所知。

如果你能原谅手机摄像头在黑暗场地的质量限制，一段比赛视频在休息时间以下。有很多人排队等候击球，这是该活动最受欢迎的夜间景点之一。
