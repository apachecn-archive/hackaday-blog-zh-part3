# 迪士尼正在设计一款智能手表，它能知道你在摸什么

> 原文:[https://hack aday . com/2015/11/10/Disney s-designing-a-smart-watch-that-know-you-touching/](https://hackaday.com/2015/11/10/disneys-designing-a-smart-watch-that-knows-what-youre-touching/)

你知道迪斯尼实际上有一个巨大的 R&D 子公司吗？这叫做沃尔特·迪斯尼幻想工程，他们想出了一些非常有趣的技术。他们目前正在开发一款名为 EM-Sense 的智能手表界面，该界面使用电磁信号来检测和学习用户正在交互的内容。

基本的机器学习允许手表学习不同设备在电磁标尺上的“感觉”。它能够检测你所期望的东西，如电器、电动工具，甚至电子设备——但它显然足够复杂，可以根据结构和电磁反馈判断你何时触摸门把手(以及哪一个)!

他们在下面的视频中更好地解释了这项技术，并展示了一个使用案例，智能手表可以引导您完成各项活动，同时为您提供可能需要的技能教程。听起来像是一个现实生活中的小男孩的开始！
