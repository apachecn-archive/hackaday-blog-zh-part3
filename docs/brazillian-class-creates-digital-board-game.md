# brazilianclasscreatesdigitalboardgame

> 原文：<https://hackaday.com/2016/04/10/brazillian-class-creates-digital-board-game/>

巴西的一个班级被要求制作一个棋盘游戏。大概是[马塞洛]听到他的儿子在哀叹如果木板只是一些画在上面的硬纸板会有多逊，于是[来帮忙了](http://makernews.info/featured/2015/11/the-pyramids-secret-brazilian-high-school-kids-create-arduino-powered-board-game-as-a-class-assignment.html)。

[![ fusion between Operation and one of those disease transmitters at the doctors office](img/e78b03f3f3b92e4bbef88db25fedade8.png)](https://hackaday.com/wp-content/uploads/2016/03/truelove.jpg)

A match meant to be.

他们和全班一起工作，想出了游戏规则。我们不确定这些是什么，但它涉及一个常规的游戏板，一个闪烁的数字光环，以及手术和医生办公室常见的疾病传播器之间的融合。休息过后，你可以试着从视频中把它们猜出来。

该板的大脑是一个 Arduino，带有一个外部 EEPROM，用于所有声音效果和该结构所需的其他数据。一切都布置在一个漂亮的自制蚀刻 PCB 上。可惜板子的另一面看不到。

我们确信孩子们在和(马塞洛)一起工作中学到了很多。如果有一个旅行巫师来到我们学校的一些早期课程，向我们展示如果你懂电子学，你能做多少很酷的东西，那就太好了。
