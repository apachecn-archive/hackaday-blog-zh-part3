# 机器人在石头剪刀布上作弊

> 原文：<https://hackaday.com/2015/10/06/robot-cheats-at-rock-paper-scissors/>

在像国际象棋这样的游戏中打败电脑已经够难的了。现在，日本石川渡边实验室的机器人工程师创造了一个每次都赢的机器人(如果你不知道，janken 是[石头剪刀布的日语名字)](http://jet.wikia.com/wiki/Janken)。它怎么可能每次都赢？简单。它作弊。

janken 机器人经历了三个不同的版本。在第一个版本中，机器人手会用高速摄像机记录下人类玩家的手，然后以大约 20 毫秒的延迟将手移动到获胜的反游戏。在第二个版本中，延迟大大减少。

然而，在第三个版本中，机器人使用扫描技术来捕捉整个视野，并确定人类正在做什么。同样，机器人手会立即产生一个获胜的反游戏。

我们实际上在 2012 年报道过 janken 机器人的早期版本。你可以对比这两个视频，看看它是如何随着时间的推移而演变的。如果你想玩老派游戏，你甚至可以和电脑手套[玩简肯游戏。没那么酷，但可能更容易构建。](http://hackaday.com/2010/03/10/game-glove-learns-your-weakness/)

我们唯一的问题是:如果你要经历所有的麻烦来建造这个机器人，为什么它不玩[石头剪刀布斯波克蜥蜴](http://www.samkass.com/theories/RPSSL.html)？
