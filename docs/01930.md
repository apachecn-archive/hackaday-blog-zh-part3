# 办公用品六足践踏你的借口

> 原文:[https://hack aday . com/2016/04/02/office-supplies-hexapod-tramps-your-excuses/](https://hackaday.com/2016/04/02/office-supplies-hexapod-tramples-your-excuses/)

我们都有不制造酷机器人的理由。“我没有激光切割机。”“我没有 3D 打印机。”[JAC_101]的[六足机器人用它压舌板的身体和橡皮脚在你所有的借口上跳舞](http://wiki.lvl1.org/Office_Supplies_Hexapod)！

有些人喜欢为设计苦恼，在黑板上优化这个，调整那个。其他人只是建立东西，看看什么工作。如果你对后者有兴趣，可以看看这里的一些技巧。压舌板做了一个简单的框架，伺服系统用拉链绑在一起，取代了花哨的伺服系统支架(或热熔胶)。光敏电阻直接焊接到负载电阻上，构成一个简单的光传感器。这一切都很容易接近，非常简约，但它似乎可以走路。(请看下面的视频。)

Arduino 代码自然是[可供你和](http://wiki.lvl1.org/*Office_Supplies_HEXAPOD_Code)一起玩。

还有其他 X-apod(其中`X`是一个小整数)你可以建立在不超过所涉及的爱好伺服成本。脑海中浮现出拉多米尔·多皮耶拉斯基的[手提包](https://hackaday.io/project/9927-tote-had)。如果你更喜欢抓臂，那么[臂](https://hackaday.io/project/181-mearm-your-robot)已经覆盖了你。(为什么没有人把这些结合起来？！？！？)如果两条腿就够了，试试波波头的衍生产品:要么穿[极简的服装](https://hackaday.io/project/3736-bob-biped-robot)要么穿[更花哨的](http://hackaday.com/2016/02/20/cute-hackable-3d-printable-robot-family/)。无论你选择什么口味，只要开始建设。改天再找借口吧。
