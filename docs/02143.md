# DIY 点焊机看着不像会立马要你命

> 原文：<https://hackaday.com/2016/04/24/diy-spot-welder-doesnt-look-like-it-will-immediately-kill-you/>

我们喜欢涉及电源电压的黑客攻击，但大多数时候，为了安全起见，我们私下里希望有一个可怕的评论者详细描述有问题的设计选择将导致死亡的每一种可以想象的方式。这个点焊机可能仍然是危险的，但看起来他们采取了一些预防措施，使其不致命，这很重要。

在他们备受质疑的高速砂带磨光机之后，这一台做得非常好，令人耳目一新。它开始作为一个死标准[微波点焊机](http://hackaday.com/2015/07/04/joining-sheet-metal-together-with-a-diy-spot-welder/)建立:拆开微波，尽量不要死于大电容，删除线圈，修改线圈，并挂钩。

之后，它进入了一些不错的重金属音乐制作。除了稍微令人震惊的 OSHA 可报告的手外伤数量(戴手套！)这种结构很协调。从主动冷却变压器到似乎是用于焊接持续时间的可重置计时器电路，显然进行了大量规划，更不用说它在结束时配合得如此之好。就家用机械而言，有一些不错的想法，我们将在一些项目中使用。

最后，证据就在点焊上。计时器设置好，踏板踩下，测试时，金属片断裂而不是焊缝。休息后的视频。
