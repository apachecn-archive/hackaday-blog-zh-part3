# 新生项目:开源扫描电子显微镜

> 原文：<https://hackaday.com/2016/12/02/nascent-project-open-source-scanning-electron-microscope/>

在我的日常工作中，我曾经可以接触到一些非常好的扫描电子显微镜(SEM)。虽然它们比 3D 打印机稍微复杂一点，但是当你需要它们的时候，它们非常方便。[Adam Guilmet]收购了一个废弃的单元，并开始试图找出如何给它注入活力。他意识到，以今天的标准来看，SEM 并不那么复杂。所以他开始利用他所学到的知识，从废料中造出一辆 T1。

平心而论，他还有很长的路要走，正在寻求帮助。他目前表示，“这是由仙尘、独角兽的屁和让最穷的学生看起来像唐纳德·特朗普的预算驱动的。”尽管如此，他已经收集了许多有趣的数据，我们希望他能建立一个能够成功的团队。

一个问题是产生、聚焦和探测电子可能是最容易的部分。最难的部分是抽真空，需要给绝缘样品镀金(我们曾经把样品放在氩室里，在样品上溅射金；塑料钥匙链上的镀金骰子总是很好的礼物)。你还需要高电压，并且要小心高能电子撞击物体并产生对你不利的 X 射线。

一旦你有了扫描电镜，还有其他你想拥有的东西。想到了 EDS(能量色散 X 射线光谱)和俄歇光谱。另一个有趣的用途是收集运行循环的 CPU 发出的二次电子。如果你在回路中的同一点收集辐射足够长的时间，你就可以确定设备的那个区域是带正电荷还是负电荷(即 1 还是 0)。在 XY 轴上扫描光束，你就可以建立一个设备电气状态的图像。在循环的不同点重复这个过程，你就可以制作一部电影，并观察比特在 CPU 电路中的传输。听起来像是天方夜谭，但它已经实现了——但不是在某人的车库里。至少现在还没有。

在之前，我们已经见过黑客翻新 SEMs [。至少有一个](https://hackaday.com/2015/01/06/a-scanning-electron-microscope-for-the-living-room/)[家酿 SEM](https://hackaday.com/2011/03/23/diy-scanning-electron-microscope/)——当然我们指的是【本·克拉斯诺】的——其视频嵌在下面。
