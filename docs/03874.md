# 超过 140 天 3D 打印的巨大动力装甲套装

> 原文:[https://hacka day . com/2016/10/18/monodably-suit-of-power-armor-3d-printed-over-140-days/](https://hackaday.com/2016/10/18/monstrous-suit-of-power-armor-3d-printed-over-140-days/)

[![fallout-armour-3d-printed-no-helmet](../Images/c90f8a4ea3057cec3a7b8d3ebb92c90f.png)](https://hackaday.com/wp-content/uploads/2016/10/fallout-armour-3d-printed-no-helmet.jpg)【Hiro creations】[在他的独家制造商 Select 3D 打印机上打印了一整套](http://imgur.com/a/1Ivf0)巨大辐射能量装甲，耗时约 140 天，使用了 120 多磅 IC3D PLA 细丝。令人高兴的是，[hirocreations]能够安排 IC3D 赞助建造——谁会疯狂到在这么长的时间里为一套 7 英尺多高的衣服使用这么多细丝，对吗？在这 140 天里，打印机上的皮带需要更换两次，但除此之外，它一直在轧轧作响。

大多数零件以 0.46 mm 的层高度印刷。单个零件被焊接(熔化)在一起，使用的是一个平头烙铁；许多零件太薄，任何接头或固定装置都不实用。零件用干墙腻子抹平，涂上大量填充底漆，然后上漆。一些部件——如胸甲——安装在由 PVC 管制成的框架上。【hirocreations】可能经历了 120 磅的细丝，但最终结果没那么重；西装本身的重量在 85-90 磅，其余的用来支撑材料，裙子和印刷失败。

从一开始就知道重量可能会成为一个严重的问题，所以[hirocreations]选择了非常轻的填充物(10%)和 3-4 个周边层；他还在高温下(约 230 摄氏度)挤出，他说这似乎为他使用的设置和细丝提供了非常强的层结合。他说，到目前为止，已经进行了一些非常猛烈的撞击，没有任何东西破碎或破裂。他有一个记录组装的短片系列，你可以在下面的一个视频中看到一些未加工的装甲部件。

使用的模型包括丹尼尔·莉莉格林的 [T-45b 头盔和 e-NABLE](https://www.myminifactory.com/object/fallout-3-t45-d-power-armour-helmet-15253) 的[猛禽之手(更新设计](http://www.thingiverse.com/thing:476403)[此处](http://www.thingiverse.com/thing:596966))结合[hirocreations]直接从游戏文件中提取并修改供自己使用的盔甲模型。

我们以前见过 Maker Select 系列，比如 [MP Select Mini](http://hackaday.com/2016/06/13/review-monoprice-mp-select-mini-3d-printer/) ，我们最近发布了这个[对垄断制造商 Ultimate](http://hackaday.com/2016/08/30/review-monoprice-maker-ultimate-3d-printer/) 的评论。据我们所知，[hirocreations]是第一个让他们的打印机经历这种 140 天马拉松的人。
