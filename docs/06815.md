# 瑞典火箭刀

> 原文:[https://hackaday.com/2017/08/15/swedish-rocket-knives/](https://hackaday.com/2017/08/15/swedish-rocket-knives/)

在各种视频制作人中，YouTube 视频有其发展趋势。几周前，它还只是关于坐立不安的纺纱工，直到一些利基技术博客把这个迷因打入地下。在那之前，主题是烧红的刀子切东西。设置很简单；只需用喷灯加热一把刀，切开一个西红柿或高尔夫球，按下高速摄像机上的 stop，然后收集那笔甜蜜的 YouTube 钱。

[David]来自 RCExplorer.se 不像大多数 YouTube 明星。他其实知道自己在做什么。当火箭推进刀的最新趋势触及管道时，他知道他可以把它炸出水面。他用一把神奇的火箭推进弯刀成功地切穿了西瓜，并扔进了大卫房子后面的树林里。

不像大多数其他 YouTube 明星尝试使用火箭动力切片器，[大卫]这次做对了。他用的是业余爱好的火箭发动机，是的，但它们是可重装的。[大卫]精心制作了一个发动机外壳，配有一个由不锈钢制成的合适的喷嘴。[火箭雪橇本身](https://www.youtube.com/watch?v=eadKs8zqNGk)是一个铝制支架，用螺栓固定在一块碳纤维板上，借助四个滑板轮子沿着轨道行进。然后用螺栓将一把弯刀固定在板上，以超过 200 公里/小时的速度沿着轨道推进。

说到火箭推进刀，“专业”这个词真的用不上。然而，这是一件令人惊叹的工艺，你可以在下面的视频中看到。
