# 电磁铁给这只舞动的猫带来了和谐

> 原文：<https://hackaday.com/2018/06/01/an-electromagnet-brings-harmony-to-this-waving-cat/>

多年来，我们已经注意到餐馆和商店里挥舞的猫，但即使是它们快乐地摆动手臂也没有真正引起我们的注意。也许[乔希]看到的比我们看到的多，当他想到要把一个拆开来看看它们是如何工作的。它们被设计成在不可靠的光源下在室内运行，似乎永远都在摇摆。无处不在的[巨兽](https://en.wikipedia.org/wiki/Maneki-neko)是如何从一个小小的太阳能电池获得无尽的机械运动的？

鉴于这些设备的普及和成本，答案很简单。关键的相互作用是安装在摆臂/摆锤末端的永久磁铁和附着在身体上的多匝线圈之间的相互作用。当磁铁在线圈上摆动时，它的运动感应出电压。一小团模拟电路通过让电流流过线圈来做出反应。最终效果是，它“感应”到经过的磁铁，并给它一点推力，让东西继续运动。只要有光，电路就会继续推动，钟摆就会永远摆动。如果线圈突然停止晃动，钟摆就会开始摆动，电路的其余部分会再次开始工作。[乔希]指出了一个类似的电路与一个非常漂亮的写在一期坚果和伏特的更多细节。

我们以前报道过【乔希】的玩具拆卸，总是觉得这类设备特别有趣。像 maneki-neko 这样的玩具和小工具通常受到极薄的利润率的控制，因此必须满足极其具有挑战性的产品约束交叉，将简单的设计和制造与足够的可靠性相结合，而不是完全令人失望。
