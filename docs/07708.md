# Face ID 被 3D 打印面具打败(也许)

> 原文：<https://hackaday.com/2017/11/14/face-id-defeated-with-3d-printed-mask-maybe/>

关于这一点的信息仍在不断涌入，所以要持保留态度，但安全公司[【Bkav】声称他们已经击败了苹果 iPhone X](http://www.bkav.com/d/top-news/-/view_content/content/103968/face-id-beaten-by-mask-not-an-effective-security-measure) 【死链，试试[互联网档案](https://web.archive.org/web/20171113113415/https://www.bkav.com/d/top-news/-/view_content/content/103968/face-id-beaten-by-mask-not-an-effective-security-measure)】中的 Face ID 系统。通过结合 2D 图像和所有者面部的 3D 扫描，[Bkav]提出了一个相当噩梦般的创造，显然欺骗了 iPhone，使其相信自己是真正的所有者。到目前为止，还没有公布多少细节，但该公司最近上传的一段 YouTube 视频看起来确实相当有说服力。

对于那些可能不了解这种事情的人来说，Face ID 被宣传为对以前的面部匹配识别系统(如 Android 中的一个)的改进，它使用两个摄像头和一个投影红外模式来对看着屏幕的面部进行快速 3D 扫描。顺便提一下，[这与微软的 Kinect 的工作方式非常相似](https://hackaday.com/tag/kinect/)。虽然 2D 系统可以被高质量的照片欺骗，但基于 3D 的系统会拒绝它，因为人脸没有深度。

[Bkav]当然不是第一个试图欺骗苹果最新产品的组织。《连线》杂志在本月早些时候的尝试中经历了巨大的努力，结果只是将受害者的照片打印出来放在摄像机前。关于[Bkav]如何成功的细节相当简单，本质上归结为他们的声明，他们只是比他们的竞争对手更了解面部识别的细节。在更多细节公布之前，怀疑可能是有根据的。

尽管如此，即使他们的方法在野外被证明是真实有效的，它也有相当大的缺点，需要对受害者的面部进行 3D 扫描。我们不确定攻击者如何在未经同意或不知情的情况下对某人进行干净的扫描，但随着收集和存储的关于普通消费者的信息量的增加，这在未来几年可能不会超出可能的范围。

既然偷脸技术的反面乌托邦式的未来似乎就要降临到我们头上了，你[不妨在主题](https://hackaday.com/2011/03/07/chilling-drinks-with-your-friends-faces/)[上温习一下](https://hackaday.com/2016/02/25/hacked-turntable-rotates-humans-for-3d-scanning/)，这样你就不会落在后面。
