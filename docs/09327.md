# 3D 打印坦克分数悬挂

> 原文:[https://hack aday . com/2018/05/03/3d-printed-tank-scores-suspension/](https://hackaday.com/2018/05/03/3d-printed-tank-scores-suspension/)

坦克是高性能的车辆，它们的履带赋予它们快速穿越各种地形的能力。坦克的一个重要部分是它的悬挂系统，没有它，坦克的履带就没有能力。当[伊万]开始他的 3D 打印坦克项目时，他无法忽视这些评论。他的坦克需要一个悬挂系统。

坦克建造本身令人印象深刻，几乎完全由 3D 打印的组件组成，用现成的轴承和螺纹杆固定在一起。[Ivan]从一开始就改装了坦克，安装了一个旋转悬挂系统，这个系统的设计出奇的简单，但却能操作。特别令人印象深刻的是 3D 打印的弹簧，它[Ivan]调整刚度以适应车辆的重量。

这是一个展示当你掌握 3D 打印和机械设计的基础知识时，你能走多远的构建。设计酷的东西并不需要很多先进的理论，只需要一种学习和实验的意愿以及你身后的一套正确的工具。伊凡的坦克履带也值得一看。休息后的视频。
