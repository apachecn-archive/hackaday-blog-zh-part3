# Hackaday 奖半决赛:玩 PID

> 原文:[https://hack aday . com/2015/09/07/hack aday-prize-semi-finaller-playing-with-pids/](https://hackaday.com/2015/09/07/hackaday-prize-semifinalist-playing-with-pids/)

PID 控制循环无处不在，在无人机的飞行控制器和 3D 打印机的温度控制代码中都可以找到。怎么教 PID 控制回路？[Tim] [为这个](https://hackaday.io/project/6917-sab3t-pid-loop-educational-tool)做了一个很棒的演示，它也是 Hackaday 奖的半决赛选手。

[Tim]的 Sab3t 是一个教育工具，旨在说明 PID 控制环路是如何工作的。这是一个机器人桌子，上面有一个巨大的滚珠轴承，非常平衡。在这张桌子上是一个电阻式触摸屏，显示屏提供滚珠轴承位置的反馈。通过调整 PID 值，滚珠轴承要么固定在桌子上，要么四处摆动，这取决于所使用的 PID 算法中的值。

作为教学工具，很棒；使用 python 脚本显示 PID 值的日志和球在盘子上的位置，任何人都可以轻松地想象振荡是如何发生的，一个调整良好的控制回路是什么样子的，并可以将滚珠移动到不同的位置。
