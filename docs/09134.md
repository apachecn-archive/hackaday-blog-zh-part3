# 一个黑客为了让他的儿子开心而进行的史诗般的探索

> 原文:[https://hack aday . com/2018/04/13/a-hackers-epic-quest-to-keep-his-son-entertainment/](https://hackaday.com/2018/04/13/a-hackers-epic-quest-to-keep-his-son-entertained/)

小人类有一种在他们父母的优先事项上捣乱的本事。任何有过孩子的人都会告诉你，没有什么是你不愿意为他们做的。如果你需要这方面的证据，只要看看这个持续了近一年的传奇故事就知道了，这个故事记录了迈克尔·提乌为他的儿子恩佐建造活动板的过程。

[![](../Images/4a76bd0207b60acd34d0700189ba2713.png)](https://hackaday.com/wp-content/uploads/2018/04/enzo_detail.jpg) 无论是从头开始还是[跳到最后看最终产品](http://michaelteeuw.nl/post/172694607802/activity-board-the-final-touch)，【迈克尔】为这个项目所做的文档确实值得一看。从他仍在决定整体外观和感觉的项目早期，到 Raspberry Pi 支持的用户界面的最终编程，该过程的每一步都经过了一丝不苟的详细描述和拍摄。

这个项目中使用的建造方法涵盖了从用于外部木质框架的基本木工工具到激光切割机，以在设备的透明丙烯酸表面上创建图形覆盖层。[Michael]甚至定制了一个 PCB，通过 MCP23017 I2C I/O 扩展器将所有 led、开关和按钮连接到 Arduino Nano。

即使你不打算制造一个会让一些成年人嫉妒的精致的儿童玩具，也有大量关于将想法转化为最终物理设备的第一手信息。这并不总是容易的，事情也不一定会按计划进行，但正如[迈克尔]清楚地表明的那样:最终产品绝对值得投入精力。

看到有多少黑客在为他们的孩子制作模拟飞船控制面板，我们不禁想知道 T2 是否会收养我们。
