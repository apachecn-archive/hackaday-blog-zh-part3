# DIY 9V 电池

> 原文：<https://hackaday.com/2017/10/19/diy-9v-battery/>

伏打的电池堆——第一个电池——只不过是被浸泡在盐水中的纸隔开的银和锌圆盘。一个经典的课堂实验是用铜币、锡纸、醋或柠檬汁堆成一堆。[Omars2]对这个古老的实验有不同的看法。他用一些锌螺丝、铜线和盐水制作了一个 9V 电池。下面是电池的视频。

注射器活塞充当细胞的基底，每个细胞只是一个用纸包裹的螺钉，然后在其上缠绕 35 圈铜线。电池浸泡在盐水中，尽管我们怀疑醋或柠檬汁会更好。加热电解液也是一个好主意。

理论很简单:盐溶解成带正电荷的钠离子和带负电荷的氯原子，形成电解质。阴极失去了进入溶液的电子，使溶液带正电。另一个电极——阳极——收集电子，所以它有一个净负电荷。两个电极上电荷的差异产生了电位差。当你闭合电路时，电子从阳极流回阴极，产生电流。

如果你更愿意做便士版本的硬币，在美国，自从 1982 年便士变成了铜包锌后，这就容易多了。将每枚硬币一边的铜磨掉，可以让它们充当两个电极。旧币没有锌芯，所以要使用锌芯，你需要另一种金属，比如锡箔。

如果你想要更令人印象深刻的东西，试试可乐罐电池。如果这些对你来说还不够刺激，考虑一下[核电池](https://hackaday.com/2016/12/01/make-your-own-nuclear-battery/)。
