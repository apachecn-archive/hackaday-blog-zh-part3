# 铝热剂创造了一把剑

> 原文:[https://hackaday.com/2017/10/26/thermite-creates-a-sword/](https://hackaday.com/2017/10/26/thermite-creates-a-sword/)

铝热剂能把汽车切成两半，但[回溯科学家]依靠的是它创造而不是摧毁的能力。事实上，铝热剂是他[铸造实心金属剑](http://youtube.com/watch?v=5fYkPljB9Zk)的关键成分。铸造不需要熔炉，因为热量是由铝热剂本身产生的。

以防万一:这个过程不是没有风险的。

[TheBackyardScientist]比较了红色和黑色两种氧化铁，然后根据点火后的质量判断它们的可用性。他的目标是从一次反应中获得最多的金属。他还添加了一些不锈钢珠来提高铸件的质量，并利用一些多余的热量。

在邻居的鼓励下，用耐火砖、水桶和沙子做了几次试验后，[考古学家]成功了。制成的剑经过处理，配有手柄，磨快，然后用普通厨房里的各种东西进行科学测试。

如果你看看[考古学家]工作室的背景，你可能会注意到他的[熔化的锡枪](http://hackaday.com/2017/08/16/wield-the-power-of-molten-metal/)。这把钢剑是几年前他的[回收白蜡剑](http://hackaday.com/2015/05/31/from-scrap-to-sword-casting-pewter/)的升级版。
