# 轻型机床震动的修理

> 原文：<https://hackaday.com/2017/09/16/a-fix-for-the-lightweight-machine-tool-shakes/>

不管你用什么材料工作，总的原则是机床越重越好。有些人买不起或不想要大型工具，尽管他们天生倾向于减少振动。这并不意味着不能做些什么来帮助这些小工具，比如降低承包商级台锯的振动。

这一个可能看起来有点超出了黑客圈的通常范围，但是没有人可以怀疑[马蒂亚斯·万德尔]的黑客能力，他确实用这个展示了他解决问题的技能。多年来，他那台破旧的承包商风格的台锯已经进行了多次特殊改造，其中一些改造让它有了足够的摆振，可以将工件从桌子上振动下来。他用木头为锯子的马达制作了一个摩擦阻尼器，配有球窝接头，允许锯片高度和角度的完全移动。这并不十分奏效，但他的渐进方法最终找到了正确的因素组合，下面的视频显示了一个锯现在足够稳定，可以平衡一个镍币。

如果这个名字听起来很熟悉，但你就是想不起来，看看【马提亚斯】最近的[木制多米诺挤压机](https://hackaday.com/2017/08/12/wooden-domino-laying-machine/)、[他敲木头的捷径](https://hackaday.com/2017/07/24/simple-shop-made-taps-for-threading-wood/)，当然还有[他经典的木制齿轮布局软件](http://woodgears.ca/gear_cutting/template.html)。
