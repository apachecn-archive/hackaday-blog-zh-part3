# 微型 ATtiny85 游戏机

> 原文:[https://hackaday.com/2016/04/10/tiny-attiny85-game-console/](https://hackaday.com/2016/04/10/tiny-attiny85-game-console/)

【伊利亚·蒂托夫】做了一个游戏主机。不仅仅是*任何一款*游戏机，而是[一款极小的基于 ATtiny85 的游戏机，适合戴上钥匙圈](http://webboggles.com/attiny85-game-kit-assembly-instructions/)，用 PCB 和 3D 打印外壳组装成非常专业的产品。这是[自 2014 年](http://webboggles.com/attiny85-breakout-keychain-game/)以来一直在进行的项目，但最近的更新是一个新版本，旨在更紧密、更容易组装。

所有结构都是通孔而不是 SMD，除了 ATtiny85 之外，控制台还使用了有机发光二极管屏幕、压电蜂鸣器、触觉开关和一些无源组件。电力来自屏幕下的一个 CR2032 硬币电池。最棒的是[PCB 设计以 PDF 格式提供](http://webboggles.com/wp-content/uploads/2016/02/tinyarcade_pcb.pdf)和[3D 打印外壳可以在 Thingiverse](http://www.thingiverse.com/thing:473749) 上找到。

有两款游戏机游戏，以及*突围*克隆，其代码在上面链接的 2014 年作品中，他写了 [*UFO 逃生*，一个避障侧滚](http://webboggles.com/attiny85-ufo-escape-keychain-game/)。你必须自己将游戏和 8MHz 内部时钟引导程序烧录到 ATtiny85 上。这款游戏机没有墨盒，但如果处理器安装在 DIP 插座上，游戏可以通过简单地交换用适当的游戏编程的处理器来转换。

他制作了一个完整的组装视频，并加入了一些 UFO 逃脱游戏，显示在下面。
