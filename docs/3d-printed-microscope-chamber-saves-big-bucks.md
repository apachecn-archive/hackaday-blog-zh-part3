# 3D 打印显微镜室节省大笔资金

> 原文：<https://hackaday.com/2016/04/14/3d-printed-microscope-chamber-saves-big-bucks/>

光学显微镜已有 400 多年的历史，在这段时间里，它走过了漫长的道路。显微镜有很多种，包括透镜的选择、照明和其他技巧，以使仪器能够获取更多的样本信息。

提高显微镜分辨率的一个行之有效的方法是油浸。将样品和透镜放在透明且具有高折射率的油中。这可以防止光线在空气-盖玻片界面发生折射，从而提高显微镜的整体性能。

新南威尔士大学有一个实验室使用这种显微镜。他们使用一种特殊的(昂贵的)容器来压住玻璃盖玻片并容纳油。问题？在近 400 美元一个的情况下，更换这些试验箱是一项持续的费用，而且它们不够灵活，无法满足定制尺寸的要求。

大学一年级学生本·古德诺运用他的 3D 打印和激光切割技术设计并建造了一个合适的房间，成本低得多，可以适应不同的项目。除了 [GitHub](https://github.com/PRNicovich/BenlideChambers) 上的所有设计文件，还有一个[文档](https://github.com/PRNicovich/BenlideChambers/blob/master/Chamlide%203D%20Print%20Project.pdf) (PDF)，描述了设计迭代和总成本节约。

主体采用 ABS 塑料，激光切割机生产硅胶垫片。稀土磁体将整个组件固定在一起。

我们预测[本]会有一个光明的未来。虽然你可能不需要油浸显微镜室，但他遵循的过程和他收集的证明数据对于任何想向老板(或者配偶)证明 3D 打印机和激光切割机成本的人来说都是一个很好的模型。
