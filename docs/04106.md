# 建造一个家庭辐射避难所

> 原文：<https://hackaday.com/2016/11/09/retrotechtacular-walt-builds-a-family-fallout-shelter/>

在 20 世纪 50 年代，冷战似乎随时可能恶化，我们都可能被核战争的烈火吞噬。幸运的是，无论是我们这边的领导人还是另一边的领导人，都不是他们对手的宣传所描绘的危险、不可预测的疯子，相反，我们继续赶路，隔着铁幕不安地相互凝视。

对于美国平民来说，政府创造了一系列的宣传活动，让他们为核战争的后果做好准备，并为他们提供生存的手段。他们中的一些人喜欢臭名昭著的“ [*Duck 和 cover*](https://www.youtube.com/watch?v=IKqXu-5jw60) ”电影在几十年后看起来似乎古雅而可悲地不充分，但其他人努力用看起来像是真正的生存手段来装备 20 世纪 50 年代的美国人。

我们今天的电影《休息之下》就是这种努力的一部分。[](https://archive.org/details/familyfalloutshe00unitrich)*家庭沉降物避难所是 1959 年由民防和国防动员办公室制作的一本小册子，它详细描述了一系列不同设计的沉降物避难所的建设。有一个混凝土地下掩体，一个部分埋在地下的填充式双壁掩体，电影中展示的是一个由混凝土块制成的地下室防辐射掩体。我们的解说员和主角是[Walt]，一个穿着格子衬衫，戴着眼镜的中年男人，他带我们参观了避难所的建筑。*

 *我们从他带一些朋友参观完工的庇护所开始，我们看到了它舒适的内部装修，有双层床和所有的现代设施。有人告诉我们，这将成为一个有用的额外备用卧室，或暗室。然后我们闪回到建设，因为[沃尔特]带我们通过所有的步骤，需要建立自己的地下室防空洞。正如他所说，这是一个几乎任何人都可以尝试的项目，接下来是一个非常好的基础砌砖介绍。面对沙皇炸弹，我们不禁担心那些未移动的屋顶块的安全性，但幸运的是，它们从未经受过考验。我们确实觉得这很有趣，这是由国家混凝土砖石协会提出的——还有什么比让大众在每个家庭建造额外的砖墙更能促进销售呢？

这部电影和小册子提供了一个迷人的窗口，让人们了解冷战早期围绕核战争准备的一些文化。人们认为它是可以生存的，在自制的防辐射掩体中呆两个星期就足以确保平民的安全，这与当时政府为自己制定的秘密深层掩体和长期生存计划形成了鲜明对比。很想知道这些家庭避难所有多少是建成的，有多少存活了下来。你有没有在有防爆墙的地下室备用卧室里呆过一晚？

我们将留给你这部电影的闭幕词，来自公民和国防动员办公室主任。

在美国，没有一个家庭没有家庭防辐射掩体是现代的。这是核时代。
