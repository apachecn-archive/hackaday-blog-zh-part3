# 用这个以辐射为主题的迷你盖革计数器漫游荒地

> 原文：<https://hackaday.com/2017/03/16/roam-the-wastelands-with-this-fallout-themed-mini-geiger-counter/>

对于任何与放射性材料打交道的人来说，盖革计数器的随机滴答声有一种奇怪的安慰。那些令人欣慰的咔哒声正是我们喜欢这个简单的袖珍盖革计数器的原因。

另一个喜欢[Tim]作品的理由是这个案例的余波主题。虽然不是游戏中的一个项目，但他对 3D 打印外壳的审美肯定与辐射宇宙相匹配。计数器本身是基于最近在易贝流行的俄罗斯 SBT-11A G-M 电子管。你可能会从这个极简的盖革计数器的报道中回忆起它们，如果你受到启发想买几个电子管，这是一个更好的机会。该案件是塞满了脂肪包，高压电源，和一个小音频放大器来驱动扬声器。下面的视频显示它从校准源愉快地点击。

我们可以看到这个项目可以很容易地扩展——一个可以显示每分钟计数的小显示器将是一个很好的补充。但是这是多么的便携，仅仅点击一下就足够了。
