# Hackaday 奖参赛作品:垃圾互联网

> 原文：<https://hackaday.com/2016/09/13/hackaday-prize-entry-the-internet-of-garbage/>

物联网就是垃圾。虽然物联网最明显的实现是因为负责它们的公司倒闭而停止工作的智能灯，或者因为提供终身支持无利可图而停止工作的智能恒温器，但物联网实际上可能是有用的，尽管在设备中没有智能烤面包机那么迷人。智能电表是个好主意，[智能垃圾也是。这就是[mikrotron]和他的公司进入 Hackaday 奖的东西——智能垃圾桶——它不像花 40 美元买一个灯泡那么愚蠢。](https://hackaday.io/project/13111-internet-of-trash)

垃圾互联网背后的想法是收集垃圾桶有多满的数据，并将这些数据发布到互联网上。这些信息将被城市的垃圾收集者和回收机构用来知道什么时候该收垃圾了。

垃圾互联网的硬件需要知道一个罐子有多满，为此，该团队求助于指向垃圾内部的超声波传感器。垃圾桶里的垃圾量每天被检测一次，信息通过 GSM 网络发送到互联网上。此外，GPS 坐标和唯一的 ID 被传送到服务器，一切最终由太阳能电池板供电。

物联网的未来不是把 Twitter 放在咖啡机里，而是所有的基础设施，无论是电力、太阳能公路还是垃圾。我们很高兴看到十亿个智能事物的有用应用，垃圾互联网是一个伟大的 Hackaday 奖参赛作品。
