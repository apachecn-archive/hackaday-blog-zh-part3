# 树莓 Pi Zero 作为 u 盘

> 原文：<https://hackaday.com/2016/10/24/raspberry-pi-zero-as-a-usb-stick/>

Raspberry Pi Zero 小到几乎可以被误认为是一个 USB 设备，而不是一台独立的计算机。也许这就是促使[Novaspirit]完全“锁定”他的 Zero 的灵感。

如果你有一个零，这是一个非常方便的黑客。通过最少的焊接，他将 Zero 的板载母 USB 插孔转换为公 USB 插头。从那以后，就全是软件了，视频(嵌入在下面)会带你经历 Windows 上的所有步骤。

首先，他将 Raspberry Pi Zero 运行的 Pixel OS 作为 USB 网络设备连接到他的主计算机上，然后将其配置为有用。他在 Pi 上设置 VNC，以便您可以在窗口中登录到它的桌面，在 Pi 上设置网络，以便它可以通过笔记本电脑连接到更广泛的互联网，等等。他安装了 OwnCloud，这样 Zero 就可以作为云存储解决方案，只有“云”被插入到笔记本电脑的 USB 端口。所有这一切的要点是从 Pi Zero 中获得最大的好处，而不必拖着任何电缆:只要插上电源，你就联网了。

当然，[Novaspirit]并不是第一个通过 USB 网络连接 Zero 的人。但他的硬件技术非常简洁，如果你有 Windows 背景，安装软件会让你学到一些东西。看看这个。
