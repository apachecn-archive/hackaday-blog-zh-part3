# 32C3:在 PS4 上运行 Linux

> 原文:[https://hack aday . com/2015/12/31/32 C3-running-Linux-on-the-ps4/](https://hackaday.com/2015/12/31/32c3-running-linux-on-the-ps4/)

在 2010 年的混沌通信大会上，fail0verflow(那是零，不是字母 O)展示了他们的 PS3 越狱。在 2013 年的 CCC 上，fail0verflow 在 Wii U 上演示了主机黑客行为，在最近两年里，这导致了 Wii U 上活跃的自制场景，世界变得更美好。几周前，fail0verflow 取笑了一些关于 Playstation 4 的事情。虽然今年的公告只是在 PS4 上运行 Linux 的演示，但 fail0verflow [可以再次声称自己是地球上最好的主机黑客](https://www.youtube.com/watch?v=2A7V3GLWF6U&feature=youtu.be)。

尽管能够运行 Linux，PS4 仍然有一些事情不能做。当前黑客没有启用 3D 加速；你不会很快在 Linux 下用 PS4 玩视频游戏。USB 还不能用，这意味着 PS4 上的硬盘也不能用。也就是说，将 PS4 变成运行 Linux 的基本计算机的一切——串口、帧缓冲区、HDMI 编码器、以太网、WiFi、蓝牙和 PS4 blinkenlights——都在工作。

虽然五分钟的闪电演讲没有涉及太多细节，但他们的幻灯片上有足够的信息来显示这是一个多么巨大的任务。fail0verflow 在内核中修改了 7443 行代码，并发现 PS4 中负责南桥的工程师“吸了一些真正的好东西”。

这只是 fail0verflow 宣布 PS4 上的 Linux 可以工作，补丁和引导代码“即将推出”。一旦这些信息发布，你将需要“自带漏洞”来实际安装 Linux。
