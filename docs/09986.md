# 本周失败:当好的铸造厂变坏

> 原文：<https://hackaday.com/2018/07/16/fail-of-the-week-when-good-foundries-go-bad/>

像我们中的许多人一样，[托尼]被铸造金属的想法迷住了，并着手制造他需要的工具来熔化用于消失模铸造的铝。他一点也不知道他即将超越他的系统极限，在他的院子里融化一个洞。

[托尼]的悲剧故事开始时非常天真，而且通常是从想要成为金属铸造家的人开始的:从[随机之王]自制的桶中铸造厂开始。它只是一个钢桶，里面有自制的耐火内衬，底部附近有一个孔，作为强制通风的喷嘴。[托尼]的建造非常忠实地按照计划进行，但是缺少最初建造中用作坩埚的用过的灭火器[国王],他临时凑合使用了一个旧丙烷缸的底部。用烧烤木炭试烧有点效果，但显然需要更多的热量。所以[托尼]得到了一些优质的威尔士无烟煤，这就是乐趣的来源。由于额外的热量，铸造厂变成了一个小型高炉，熔化薄钢坩埚，将熔化的铝倒入熊熊的煤火中。下面的视频显示了这场灾难，我们希望一旦[托尼]换了裤子，他会赶紧去买一个便宜的石墨或陶瓷坩埚，以便下次烧制。

玩笑归玩笑，这是一个生动的提醒，当一些意想不到的事情(或完全可预见的事情)出错时，风险有多大，以及有必要做好应对的准备。一桶干沙来灭火可能是个好主意，防护服是必须的。管理好你的工作区域，将潜在的附带损害降至最低也是值得的——我们怀疑 patio 是否还会和以前一样。
