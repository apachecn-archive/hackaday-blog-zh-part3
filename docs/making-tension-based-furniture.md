# 制作基于张力的家具

> 原文：<https://hackaday.com/2017/04/18/making-tension-based-furniture/>

科罗拉多州科林斯堡的艺术家兼设计师罗比·库斯伯特正在创造一种稳定的电缆桌，它既是一项工程壮举，也是一件艺术品

[库斯伯特]的桌子由 1/16 英寸的不锈钢电缆连接在一起，产生相反的张力，从而形成结构稳定、外观迷人的咖啡桌。在他的视频中，[库斯伯特]一步一步地向我们展示了他创建表格的过程。[库斯伯特]开始在他的数控铣床上切割竹腿。然后，他在每条腿上钻了几个电缆孔，并将每条腿安装在他定制的工作台夹具上。然后，他连接不锈钢电缆，注意改变张力方向。电缆穿过支架上的孔，并用铜压接固定。经过多次电缆，他有了一个机械结构，可以支撑他的重量，看起来也很棒。总而言之，[库斯伯特的]艺术是艺术和工程交叉的一个极好的例子。

如果我们吊起了你的胃口，别担心，我们之前已经介绍过许多基于紧张的艺术/工程技巧。你可能会对这些[电脑设计的肖像](http://hackaday.com/2016/07/28/computer-designed-portraits-knit-by-hand/)感兴趣，或者，如果手工编织的想法让你紧张不安，那么[的亲笔签名，一台线绳艺术打印机](http://hackaday.com/2016/04/28/autograph-a-string-art-printer/)可能更符合你的风格。
