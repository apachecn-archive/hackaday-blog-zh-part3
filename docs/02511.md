# FR4 机盾是一个数控铣床从 FR4 电路板

> 原文:[https://hack aday . com/2016/06/04/fr4-machine-shield-is-a-CNC-milling-machine-from-fr4-PCB/](https://hackaday.com/2016/06/04/fr4-machine-shield-is-a-cnc-milling-machine-from-fr4-pcb/)

PocketNC 背后的人听说你喜欢 CNC PCB 工厂，所以他们用 PCB 给你做了一个 PCB 工厂。在今年的湾区制造商博览会上，他们宣布了他们令人惊讶的新开源硬件产品，[一台完全由 FR4 PCB 材料制成的口袋大小的 3 轴数控机床，被恰当地命名为“FR4 Machine Shield”](http://www.pocketnc.com/blog/2016/5/26/may-2016-update-maker-faire-and-the-fr4-machine-shield)。

**更新:**FR4 机盾[现在 Kickstarter 上](https://www.kickstarter.com/projects/1090944145/fr4-machine-shield)

[![fr4_thumb](../Images/078ba422833024dfa94fdbc9d4b3249b.png)](https://hackaday.com/wp-content/uploads/2016/05/fr4_thumb.jpg) 我们从四轴飞行器、小机器人以及通常小到足以利用其 PCB 作为结构组件的事物中了解这个概念。但是，由几十块印刷电路板焊接而成的一台完整的数控机床无疑将它推向了一个新的高度。

毫无疑问，2mm 厚的纤维增强环氧树脂具有惊人的刚性，尽管这种方法的致命弱点可能是焊点。然而，看起来机器的所有承载、机械连接都由紧密互锁的~~“燕尾”~~指形接头支撑，这可能有助于保护所有焊接连接免受连续应力和主轴振动的应变硬化影响。

正如您所料，大多数布线都嵌入到 FR4 框架结构中，为了从 PCB 材料中获得最大价值，电机驱动板通过卡边缘连接器与(目前基于 Arduino 的)控制板连接。除了具有无刷 DC 电机和工具耦合器的铣头，该团队还希望开发用于电路印刷、显微镜、气动拾取和放置、热空气回流和 3D 打印的头。

由于所有这些成本驱动的设计选择，从框架和布线的一步制造工艺到从框架组件中去除螺钉和螺母，“FR4 Machine Shield”确实可以成为市场上最便宜的 CNC 机床套件之一。该团队的目标是在 2016 年 6 月的 Kickstarter 活动中推出 400 美元的价格。他们能送货吗？[Gerrit]在展会上查看了 Pocket NC，最后对他们如何经营业务大加赞赏。
