# 如何升级一台中国数控机床

> 原文：<https://hackaday.com/2015/09/08/how-to-upgrade-a-chinese-cnc-machine/>

想在你的车库或 hackerspace 的工具库中添加一台小型数控机床吗？像任何工具一样——中国为你提供了廉价的选择——但问题是，这值得吗？通常这取决于工具，但是当[你可以升级你的 3040 CNC 路由器](http://www.youtube.com/watch?v=42ew5xjIBWM)来使用 USB 而不是 TinyG 运动控制器的并行端口…最肯定的！

3040 或 3020 CNC 刳刨机是一种受欢迎的中国机器，被许多爱好者使用，这是有充分理由的。坚固的全铝框架，像样的步进电机和相当好的分辨率？大约 1000 美元是一笔不错的交易。在之前我们已经报道过[很多次了。问题是，电子设备有点过时了。尤其是它使用带有并行端口的 Mach3 拜托，现在谁有并行端口？](http://hackaday.com/2014/02/15/chinese-3020-cnc-machine-gets-some-upgrades/)

[约翰·劳尔]着手解决这个问题。TinyG 是一个电机控制器[,我们在](http://hackaday.com/2014/09/24/the-tinyg-motion-controller/)之前已经介绍过几次了——它只是在等待安装到 3040 CNC 中，以便运行一个更好的控制系统，就像 [ChiliPeppr](http://chilipeppr.com/) ！

事实上，[John]订购数控机床就是为了完成这一升级，他正在与大家分享他是如何做到这一点的:
