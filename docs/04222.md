# 开源煎饼

> 原文:[https://hackaday.com/2016/11/24/open-source-pancakes/](https://hackaday.com/2016/11/24/open-source-pancakes/)

这绝对是一个第一世界的问题:当创建一个定制的煎饼需要你把一个设计放在 SD 卡上并插入你的煎饼打印机时，你会怎么做？这就是困扰[博士]的问题。当然，由于他为一家名为“The Stack”的出版物工作，也许煎饼打印机并不令人惊讶。[drtorq]在 Linux 上构建了定制的 PancakeBOT 软件，作为他在烙饼制作机器人上进行黑客攻击的开始。

[drtorq]承诺未来会对打印机进行更多黑客攻击，所以这只是第一步。我们预计 mods 将很像一个典型的 3D 打印机，除了加热床在这个模型上是绝对必要的。该打印机更像是一个数控雕刻机，而不是 3D 打印机，因为它基本上是一个 XY 托架，带有一个喷嘴，用于流动面糊而不是聚合物。

我们见过很多[巧克力打印机](https://hackaday.com/2015/01/14/choctopus-chocolate-printer-x8/)。[巨美熊](https://hackaday.com/2016/05/24/sweet-3d-printer/)也可以印刷。对于更注重饮料的成年人来说，你也可以在[果冻照片](https://hackaday.com/2013/12/26/jello-shot-printer/)中打印设计。我们有点惊讶，我们没有看到传统的 3D 打印机或 CNC 改装来做煎饼，但我们肯定[见过其他](http://hackaday.com/2016/01/16/edible-art-from-a-robot-pancake-printer/)自举[版本](http://hackaday.com/2015/04/23/robottermilk-pancakes/)。
