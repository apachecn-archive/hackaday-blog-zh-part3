# ESP8266 周围的 LED 环

> 原文:[https://hack aday . com/2015/09/12/led-ring-around-the-esp8266/](https://hackaday.com/2015/09/12/led-ring-around-the-esp8266/)

这个世界需要更多闪亮的灯，而[贝尔图斯·克鲁格]发明了一种让灯无线闪烁的巧妙方法。他在电路板中间有一个脚印，用于焊接城堡状的 ESP8266 模块，周围有一个 LED 环，用于创建 [WiFi 像素](https://github.com/Protoneer/WifiPixels)。这是一个可以通过 WiFi 连接控制的 LED 戒指。他的设计基于无处不在的 ESP8266 WiFi 芯片和 AdaFruit 的 NeoPixel ring 的组合，因此已经有了如何编码和控制硬件的很好的例子。该项目仍在进行中，但他已经公布了所有细节，包括开发板的 Gerber 文件和 ESP8266 运行的 Arduino 代码。

这是一个很好的开始:加上电池支持，你可以有一个很棒的方式来拥有便携式 LED 闪光戒指。对于那些想在不构建自己的电路板的情况下尝试一下的人，[贝尔图斯]说可以用 ESP8266 开发板和 [Adafruit NeoPixel ring](http://www.adafruit.com/products/1463) 来构建。目前，他通过 USB 运行设备，但没有理由为什么它不能通过电池供电来实现便携式 USB blinkiness。
