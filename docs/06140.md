# ZeroBot 非常简单

> 原文：<https://hackaday.com/2017/05/31/zerobot-is-as-simple-as-it-gets/>

通常在 Hackaday，我们喜欢发布感兴趣的项目，因为它们很复杂。对于机器人来说尤其如此——马达和传感器越多越好。但是，偶尔我们会遇到一个因简单而美丽的项目。最近在 Hackaday.io 上发布的【麦克斯·克恩】[的 ZeroBot 就是这种情况。](https://hackaday.io/project/25092-zerobot-raspberry-pi-zero-fpv-robot)

ZeroBot 将机器人的本质分解为基本要素:一个树莓 Pi Zero W 用于大脑，一个驱动器和两个电机用于运动，一个电池用于供电，一个摄像头用于观察。底盘完全由易于 3D 打印的零件制成。Zero W 创建了一个用户可以在电脑或智能手机上连接的 WiFi 接入点，并随后提供 FPV 控制。

这个项目让人想起我们许多人开始黑客生活时使用的入门机器人工具包，它是孩子们的一个很好的教学工具。把零件打印出来，你就可以在一个下午就把机器人组装好，而且当你完成后还可以玩得很开心。实体机器人造出来之后，[编程和控制它的可能性是无穷无尽的](http://hackaday.com/2013/03/04/using-opencv-with-the-raspberry-pi/)。
