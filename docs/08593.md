# 现在有一个新的 MIDI 规范，和无人机

> 原文:[https://hack aday . com/2018/02/16/theres-now-a-new-midi-spec-and-drones/](https://hackaday.com/2018/02/16/theres-now-a-new-midi-spec-and-drones/)

乐器数字接口 MIDI 于 1983 年在乐器制造商之间一个真正奇怪的协会中发布。在此之前或之后，不同制造商之间从未有过这样的合作来定义一个标准。从那时起，MIDI 规范已经扩展到 SysEx 消息，通过 MIDI 转储样本的能力，通过 MIDI 重新定义乐器的调音以支持非西方音乐，以及规范的深层内容，卡拉 ok 机。

现在 MIDI 规范有了新的更新 (Gearnews 链接，这是 midi.org 官方的[公告](https://www.midi.org/articles/midi-manufacturers-association-mma-adopts-midi-capability-inquiry-midi-ci-specification)，但他们的网站需要注册，是一个热门的垃圾火)。在今年的 NAMM 上，MIDI 在几十年前首次展示，MIDI 制造商协会宣布了一项对 MIDI 的更新，使乐器和控制器更加智能，几乎可以自学。

MIDI 规范的新更新中有三个新的位。第一个是配置文件配置，这是一种自动配置复杂控制器映射的方式，被描述为“MIDI 学习类固醇”。第二个更新是属性交换，允许 MIDI 设备设置设备属性，如“产品名称、配置设置、控制器名称和补丁数据”。这实际上是在控制器和设备中设置元数据。第三个新功能是协议协商，这是一种通过 DIN-5 连接器自动推动未来下一代协议的方式。

这一切意味着什么？无人机。不，我是认真的。MIDI 协会正在修补一些微小的呜呜声和幻影，并发布了一段由 MIDI 控制器控制的无人机视频。弹个滑音上去，无人机就上去了。你可以看看下面的视频。
