# 你可以从假人身上学到很多东西

> 原文:[https://hack aday . com/2015/12/24/you-can-learn-a-lot-from-a-dummy-load/](https://hackaday.com/2015/12/24/you-can-learn-a-lot-from-a-dummy-load/)

如果你在射频电路上工作——即使你不是业余无线电操作员——你也应该有一个假负载。假负载是一种阻抗已知的非辐射“天线”，可以用来测试 RF 电路而不辐射。对于无线电工作，你通常只需要一个 50 欧姆的无感电阻(至少在你感兴趣的频率下)，它可以消耗你期望它处理的功率量(至少在短时间内)。[VO1PWF]想要一个假货，[自己造了](http://linuxwolfpack.com/dummyload-homebrew.php)。
![Cantenna](../Images/7a7072ca094cfc95d2e1fa8ca96cbe05.png)坎特纳(不是普林格尔的那种；见右)是希斯基特在商业时著名的假负载设计。那是一根浸在油漆罐里的碳棒，里面装满了变压器油(我们现在知道里面装满了[危险的多氯联苯；而且我们不是指印刷电路板](https://en.wikipedia.org/wiki/Polychlorinated_biphenyl)。[VO1PWF 的]设计更实际一点，使用一些并联的电阻(20 个 1K 电阻)，一个塑料管外壳，和矿物油来保持所有的冷却。

并联电阻的原因是为了最大化功率处理能力。电阻为 3W 单位，因此理论上，假负载可以处理 60 瓦。通常，高功率电阻是线绕的，因此具有相当大的寄生电感，使假负载具有电抗(这不是一件好事，因为这会使负载阻抗随频率而变化)。他们确实制造无感线绕电阻器，但这些不是真正的无感。电线朝两个不同的方向缠绕，所以电感会相互抵消。在大功率假负载设计中，我们不会相信它们是纯电阻。

不过，我们注意到这个项目有三点。我们已经匹配了电阻，因为即使是很小的阻值差异(或焊点中的微小差异)也会导致阻值最低的电阻比其它电阻消耗更多的电流。例如，如果除了一个低 5%的电阻(950 欧姆)之外，所有电阻实际上都是 1K，低电阻将消耗大约 3.15 瓦。

如果其余的都是 5%高，情况会变得更糟，这是一个典型的情况，因为许多制造商挑选更高公差的精确值；在一批 5%电阻中，没有一个可能在标称值的 1%以内，因为如果在，它们会被标记为 1%电阻。扩大范围(例如，使用 10%的电阻)，情况会变得更糟。实际上，您可能需要降低电阻的额定值，以便通过设计使其远低于最大设计功率，从而使电路更能容忍容差和布线差异。此外，在短时间内，如果稍微超过额定功率，电阻也不会爆炸。然而，尝试在任何时间长度内将全部 60W 的功率用于这种负载可能并不明智。

焊接工作是整洁和相当短。这是我们注意到的第二件事:在 Cantenna 上，布线会增加寄生电抗，使假负载在某些高频下用处不大。知道频率在哪里会很有趣。总线相当粗，但使用更粗的材料(如下面视频中的 PCB 材料)可能会提高有用频率。

我们想知道的最后一件事是在这个案例中使用 ABS。即使有了 Cantenna，也有关于操作员在 DX(长途)工作时意外挂在假负载上但仍然通过的著名故事。坎特纳装在一个金属盒里。我们预计 PVC 假负载会辐射更多。虽然这仍然比使用天线好，而且从电气上来说这不是问题，但我们希望看到一些辐射功率的测试测量。

例如，我们看到相当多的假负载是 LED 驱动器的可编程负载。然而，我们没有看到许多适合无线电使用的高功率非感应负载。下面的视频显示了一个类似的设计，具有金属罐和更厚的母线，更接近于旧的希斯基特设计，尽管它仍然使用多个电阻。
