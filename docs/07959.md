# 555 架钢琴中最小的一架

> 原文:[https://hackaday.com/2017/12/13/the-tiniest-of-555-pianos/](https://hackaday.com/2017/12/13/the-tiniest-of-555-pianos/)

555 定时器是已经实现硅不朽集成电路的特殊俱乐部之一。尽管它年事已高，其功能几乎在所有方面都被复制和取代，但它仍然在生产，仍然非常受欢迎，因为它非常有用。如果你到了一定的年龄，a 555 很可能是你接触的第一个集成电路，反过来，你的项目也很有可能是一个简单的电子琴。

如果你想重温那个项目，也许[亚历山大·瑞日科夫]可以用他的 555 钢琴找到答案。它是我们[硬币电池挑战赛](https://hackaday.io/submissions/coin-cell-challenge/list)的参赛作品，因此使用了 CMOS 低电压 555，而不是耗电的原件，但它完全是你知道和喜欢的带可切换电阻阶梯的经典 555 振荡器。

从物理上来说，钢琴是一个带有表面贴装元件和物理按钮的微型 PCB，而不是过去的手写笔器官，但正如你在视频中看到的那样，它仍然可以播放。我们说它很小，但有些人可能也会用 tinny。
