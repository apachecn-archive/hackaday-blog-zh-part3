# 本周失败:砂铸铜

> 原文:[https://hack aday . com/2017/06/07/每周失败-砂铸-铜/](https://hackaday.com/2017/06/07/fail-of-the-week-sand-casting-copper/)

随机王国有麻烦了——这个王国的铁匠铺在用砂铸铜方面遇到了麻烦。虽然[金·格兰特]可能没有直接寻求帮助，但我们认为 Hackaday 社区可能对他的努力有很多话要说。

我们以前都见过大量的砂型铸造努力，包括试图制造无法获得的发动机零件。“消失模”铸造是一种相对较新的技术，在这种技术上，最近在[金格里车床上制造的](http://hackaday.com/2016/07/07/the-best-gingery-lathe-video-series-to-date/)产生了良好的效果。但是大多数后院铸造厂生产铝，这显然比[格兰特·汤普森]生产的铜要容易得多。具有讽刺意味的是，[他的第一次倒酒](https://www.youtube.com/watch?v=09Ty96uYKVE)效果最好——虽然不完美，但至少他装饰作品的辐条没有像下面视频中显示的每次倒酒一样断裂并漂走。这让我们认为第二段视频中的绿沙太干了。或者也许铜的密度只是让沙子更容易漂浮。也许上下模是为了保持岛屿的位置，更好地引导铜的流动。

我们知道有很多专业知识，所以请在评论中说出你对这些倾倒的看法。
