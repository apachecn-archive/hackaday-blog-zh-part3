# 通过 WebODM 利用您的无人机视频

> 原文：<https://hackaday.com/2016/12/11/make-use-of-your-drone-video-with-webodm/>

如果你曾经看过原版的《星际迷航》,柯克船长和船员们花了很多时间绘制银河系的新地图。事实上，至少有一集以他们拍摄新空间的图像为中心。这可能并不新鲜，但如果你有一架无人机，你可能已经从你的房子周围(或你飞行的任何地方)积累了大量的航空图像帧。

[WebODM](https://github.com/OpenDroneMap/WebODM) 允许您从无人机镜头中创建地理参考地图、点云和纹理 3D 模型。该软件实际上是[开放式无人机地图](https://github.com/OpenDroneMap/OpenDroneMap)的集成和工作流管理器，它完成了大部分繁重的工作。

开始使用 WebODM 或 Open Drone Map 很简单，因为它们提供了预配置的 [Docker](https://hackaday.com/2016/09/01/how-to-use-docker-to-cross-compile-for-raspberry-pi-and-more/) 图像。您不必担心组装一堆依赖项来使一切正常工作。也有其他的地图应用程序在使用。你可以在下面的视频中看到五个热门选择的对比。WebODM 还没有完成，但他们打算包括任务规划和与移动应用程序的集成。

我们已经看到了其他的测绘无人机，但是它有很多功能和发展潜力。如果你对人们用这样的地图做什么感兴趣，我们谈到了[如何改变农业](https://hackaday.com/2016/05/09/drones-what-exactly-do-they-offer-agriculture/)，当然，这不是唯一的用途。
