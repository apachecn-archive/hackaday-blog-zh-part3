# 巢穴:藏在岩石中的专辑发行

> 原文:[https://hack aday . com/2016/12/11/the-nest-album-release-hidden-in-a-rock/](https://hackaday.com/2016/12/11/the-nest-album-release-hidden-in-a-rock/)

先是黑胶唱片，然后是卡带、CD、那些失败的音频 DVD 格式，然后是下载。引用乐队[Bateleur]的一句话，“如今，你不能付钱让人拿走一张 CD。”那么你如何发行你的新专辑呢？通过[将树莓皮藏在山崖上半透明的假岩石中](https://www.bateleur.xyz/)，并要求*秘密哨子*来启动它，自然。

一旦激活，你就可以插入 USB 端口下载专辑，或者坐在一个偏远的山坡悬崖上俯瞰大海，享受新的曲调。因为石头里有耳机插孔，很自然。除了是一个很酷的黑客，我们认为把人们放在正确的身体和精神空间进行认真的倾听是很棒的。观看下面嵌入的视频，了解该网站的制作理念和观点。

在其他地方，这将是有史以来最酷的专辑发行。但这是 Hackaday，我们认为它与作为激光切割机文件发布的这张 [EP 不相上下，你应该切割自己。你们接下来会想出什么？](http://hackaday.com/2014/11/19/laser-cut-album-released/)
