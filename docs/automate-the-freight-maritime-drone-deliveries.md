# 自动化货运:海上无人机送货

> 原文：<https://hackaday.com/2017/06/06/automate-the-freight-maritime-drone-deliveries/>

海上的船只实际上是它们自己的岛屿。如果你需要的东西不在船上，祝你在太平洋中找到它。因此，大多数船只都配备了备件，甚至原材料和制造大部分无法储存的物品所需的工具，水手们也因能够利用现有资源而闻名。

但是，尽管海上的船只可以自给自足，意外总是会发生。一个重要的系统可能会因为缺少一个简单的备件而失败，往好了说会导致运输公司的延误，往坏了说会使船员处于致命的危险之中。可以派遣另一艘船去协助，或者如果船离岸边足够近，可以安排直升机会合。两者都是昂贵的选择，这就是为什么一些航运公司正在试验无人机在海上运送船只。

### "饼干无人机偏离右舷方向！"

我们已经讨论了用于快速跨洋货运的[重型无人驾驶飞机的计划](http://hackaday.com/2017/04/20/automate-the-freight-drones-across-the-sea/)，但是船到岸和岸到船交付可能是对航运的空中支持的更令人信服的用例。虽然海上船只需要岸上支持的情况确实很少，但无人机可以解决更常见和平凡的运输需求。

冒着说出显而易见的痛苦的风险，船很大。非常大。现代船只是有史以来最大的移动物体之一，一些港口甚至无法容纳它们。有些油轮，如超级油轮，即使能装得下，也会成为糟糕的邻居，许多油轮在离岸几十海里的海上石油码头装卸。停泊在这些仅通过管道与海岸相连的小人工岛上，油轮经常发现自己需要来回运送文件、邮件、小型物资，甚至现金到岸上，同时等待货物作业完成，有时需要几天。

传统上，这样的工作会有一艘发射船，但是花上几千美元来运送一些文件是浪费。这就是为什么马士基航运公司最近在海上进行了一次无人机向油轮送货的实验。原本打算送一个象征性的包裹——一盒饼干——给正在哥本哈根一公里外航行的马士基埃德加 T3 号，恶劣的天气意味着他们不得不作弊。这架 octocopter 是专门为在油轮上方安全操作而建造的，在发生碰撞时不会造成火灾风险，它是从距离油轮 250 米远的一艘小船上驾驶的，并在没有发生事故的情况下放下了有效载荷。

这是一个微不足道的实验，只是为了证明原理，而不是自主交付。但这可能不是近岸海上交货的要求；实际上，大多数船只都可能在岸上远程操作员的范围内。也许有一天，无人机会成为船舶设备的标准组成部分，一名或多名船员接受训练，驾驶无人机从船上飞到岸上，甚至在穿过远海路径的船舶之间飞行。

### 把它变成一门生意

这个有市场吗？很可能，至少有一家公司愿意尝试近岸船舶服务。威廉森船舶服务公司[最近宣布了基于无人机向海上船只送货的计划](http://maritime-executive.com/article/wss-ships-agency-launches-delivery-drone-service)。WSS 在 125 个国家的 2200 个港口开展业务，它很可能准备利用现有的庞大客户群，这些客户愿意为他们的船队节省一些费用。他们计划在 2017 年的某个时候在一个主要港口开始试验。

观察 WSS 及其最终继任者将海上无人机技术带向何方将是一件有趣的事情。海上有很多船只，也有很多固定的海上石油和天然气平台需要服务。无人机可能会改变这个市场的游戏规则，并可能成为自动化货运的另一种方式。
