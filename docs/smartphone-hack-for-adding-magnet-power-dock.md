# 智能手机黑客添加磁铁电源坞站

> 原文：<https://hackaday.com/2016/06/18/smartphone-hack-for-adding-magnet-power-dock/>

这里有一个为手机制作磁性充电支架的巧妙方法。我们知道你在想什么，但这绝对不是传统的非接触式充电系统。那些使用磁铁，但方式不同。这项技术包括在手机外壳上放置几块磁铁，在充电底座上放置几块磁铁。然后，你将这些磁铁连接到 USB 端口的电源输入端，并将 USB 电缆连接到底座上，这样将手机放在底座磁铁上就完成了电路。磁铁本身成为充电触点。

这是一个巧妙的想法，但让我们想知道这将对你的手机或信用卡中的指南针传感器产生什么影响，如果它们就在附近的话。有了这些警告，这是一个整洁的黑客，可以很容易地适应。想做一个垂直的手机支架，或者一种把手机挂在冰箱上(充电)的方法？这可以很容易地适应。
