# 除草机回来了

> 原文：<https://hackaday.com/2018/03/24/the-weedinator-returns/>

我们很高兴看到[除草机](https://hackaday.io/project/53896)入围 2018 年黑客日大奖！农业创新是建设改善我们世界的伟大机会。[tegwyntmmffat]的除草机是一个自主的电动平台，旨在为小型农场提供培育、耕作和苗床除草服务。如果必须由人来做，这种劳动力的成本会使小农场失去可持续性。

传统上，提高农业效率就是增加一个人所能做的工作，通常采用更大更重的设备，在更短的时间内一次做更多的工作。但是有了自主机器人平台，机器人不会感到疲劳或无聊，所以如果较小的平台需要多次通过以覆盖一个领域或完成一项任务，这都没有关系。事实上，对于一个小农场来说，更小往往意味着更易操作、更易管理、更节能。

[最初的除草机](https://hackaday.io/project/53896)是 2017 年 Hackaday 奖的竞争者，我们非常兴奋地看到它带着更新的设计和新人加入他们 2018 年的团队。请记住，已经留出资金来帮助启动有前途的概念，你真正需要开始的只是一个想法、一幅图像和文档。没有比这更好的机会来重温这个想法，看看它是否可行。
