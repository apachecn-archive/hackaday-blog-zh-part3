# 将角磨机改造成砂带机

> 原文:[https://hack aday . com/2016/04/03/将角磨机变成砂带磨光机/](https://hackaday.com/2016/04/03/turning-an-angle-grinder-into-a-belt-sander/)

面对一个需要大量打磨的项目，[George]有两个选择。忍气吞声，买一个砂带磨光机——或者重新使用他已经拥有的工具来做同样的工作。他选择了后者，[把一个角磨机变成了砂带机。](http://www.makeitextreme.com/en/newsroom/entry/VIDEOS/sander)

YouTube 上一个名为 [Make It Extreme](https://www.youtube.com/watch?v=_2jcpNFffsE) 的系列的一部分，[乔治]使用原材料从零开始建造了整个项目。使用车床，他创造了砂带将坐在铝滚动销钉。他将轴承压入其中，然后用废钢焊接一个框架将它们分开。他甚至增加了一个弹簧驱动的张力装置，以确保皮带不会松脱。

至于安装角磨机在适当的位置，它不能更容易。它在两个金属导轨之间滑动，并使用角磨机手柄的螺纹孔连接。
