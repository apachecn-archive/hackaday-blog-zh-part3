# 去大的地方或回家——这辆 Arduino 遥控汽车可以带你去那里

> 原文：<https://hackaday.com/2018/03/16/go-big-or-go-home-this-arduino-rc-car-can-take-you-there/>

不管我们喜不喜欢，最终有一天我们会不得不承认，我们已经长大，不再需要童年的玩具了——当然，除非我们以科学的名义将它们科技化。在某些情况下，我们可能会简单地放大尺寸，使之更适合成年人的尺寸。[kenmacken]通过为自己建造一辆全尺寸 1:1 的 RC 汽车，展示了如何做到这两点。不，我们没有忘记这里的一个数字，[他将一辆真正的本田思域改装成了一辆无线电遥控汽车](http://www.instructables.com/id/Full-Size-RC-Car/)，并记录下了这一过程中的每一步，希望能激励和引导其他人跟随他的脚步。

为了用标准 RC 发射器控制思域，[kenmacken]为它配备了高扭矩伺服系统、一些线性致动器和电子动力转向模块，以处理加速、制动、档位选择和转向的所有机械方面。这一切的中心是一个常规的，现成的 Arduino Uno。他的文章中有大量视频展示了每一个组件，当然，还有他对汽车的控制——休息后你也会看到。

[kenmacken]的最终目标是最终去除无线电控制，建立一个完全自主的自动驾驶汽车，你可以在他的教程结尾看到一些 GPS 航路点驾驶的初步实验。我们以前在普通的遥控汽车中见过[同样的概念，我们也见过](https://hackaday.com/2013/06/16/autonomous-rc-car-navigates-by-waypoints/)[使用神经网络](https://hackaday.com/2017/06/06/self-driving-rc-cars-with-tensorflow-raspberry-pi-or-macbook-onboard/)进一步发展。考虑到他在计算机视觉方面的背景，了解一下[kenmacken]未来会走哪条路会很有趣。
