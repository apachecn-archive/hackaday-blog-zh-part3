# 用 Arduino 和科学揭穿登月否认

> 原文：<https://hackaday.com/2018/02/20/debunking-moon-landing-denial-with-an-arduino-and-science/>

可悲的是，在阿波罗计划取得成就近半个世纪后，我们仍在与某些坚持认为它从未发生的人争论。在历史记录中寻找证明这些任务不可能发生的证据已经成为一个可悲的小家庭工业，揭穿否认者是一个令人厌恶但又必要的持续努力。

一个特别令人绝望的否认理论认为，完全穿着宇航服的宇航员永远不可能走出登月舱(LEM)的小舱口。[AstronomyLive]以一种聪明的方式回击了这种有倾向性的说法——用 DIY 激光雷达扫描仪测量博物馆里的阿波罗文物。硬件很简单，Garmin LIDAR-Lite V3 扫描仪安装在几个伺服系统上，可以快速云台。该钻机有一个果断的符合它的外观，与传感器扑腾左右有点伺服移动。但就目的而言，这似乎完全没问题。

[AstronomyLive]将扫描仪带到了两个独立的博物馆展览，一个扫描了登月舱舱口，另一个扫描了迄今为止最后一个站在月球上的人吉恩·塞尔南在阿波罗 17 号训练时穿的衣服。随着登月舱从椽子上飞下来，扫描仪有点扩展了它的能力，所以他捕捉到的点云有点低分辨率。但最终，一个虚拟的塞尔南能够通过虚拟的登月舱舱口，正如预期的那样。

可悲的是，这样的证据只会让那些不需要说服的人信服；故意无知的人总会找到证明自己立场的方法。所以让我们庆祝阿波罗的成就吧。
