# Hackaday 奖:开放式电动轮椅

> 原文：<https://hackaday.com/2016/09/26/the-hackaday-prize-an-open-electric-wheelchair/>

[艾琳·桑斯]和[阿尔瓦罗·费伦·希福恩特斯] [觉得电动轮椅还是太贵了。](https://hackaday.io/project/13347-openchair)最重要的是，由于每个人的需求略有不同，通常不完全适合轮椅使用者可能面临的问题。为此，他们已经开始了创造一个开放式轮椅设计的过程，他们恰当地称之为 OpenChair。

正如在之前的 Hackaday Prize [中所显示的，在辅助空间中还有许多需要改进的地方。东西一般都贵。这很好，但通常保险不包括它，或者它超出了发展中国家的范围。一如既往，最好的结束方式就是开始，所以这就是艾琳和阿尔瓦罗所做的。](http://hackaday.com/2015/11/17/the-gaze-controlled-wheelchair-that-won-the-hackaday-prize/)

他们最初的设计基于我们都知道的折叠轮椅。对于日常使用来说，它足够强大，并且在世界范围内都是相当标准的。他们设计了一套附件，使轮椅更适合日常使用，并纳入了控制。

下一个问题是运动。事实证明，在不超出预算的情况下找到一款足够强大的现成电机很困难，但他们突然有了灵感。为什么不用大规模生产的玩具废物来为他们服务呢。在刚刚过去的商业假日季节风靡一时的“悬浮滑板”能够让一个人滚来滚去，因此轮椅自然也在这个动力范围之内。

他们取出了两个 350 瓦的轮毂电机、电池和控制板。这需要一点逆向工程，但他们能够将悬浮滑板的轮毂驱动电机与轮椅上的控制集成在一起。

最终，他们能够通过第一次迭代将常规电动轮椅的价格降低一半，并为开放式电动轮椅系统的未来工作奠定了基础。当然，更多的工作可以带来更好的改进。
