# RGB 传感器的新工作:加密货币交易顾问

> 原文：<https://hackaday.com/2018/07/08/rgb-sensors-new-job-cryptocurrency-trade-advisor/>

[XenonJohn]涉足加密货币交易，当他看到一个购买 RGB 颜色传感器的机会时，他的第一个想法——他向我们承认这可能不是大多数正常人的第一个想法——是他可以将它指向他的笔记本电脑屏幕，并让它分析绿色(买入)订单与红色(卖出)订单的比率是为了加密交易。从理论上讲，如果在某个特定的时刻，想买的人比想卖的人多，商品的价值预计会在短期内略有上升。如果相对于买入订单有大量卖出订单，情况正好相反。拥有这些信息并有可能根据这些信息采取行动可能是有用的，但也可能没有用。无论哪种方式，就超出左领域的项目想法而言，向加密货币交易顾问推广 RGB 颜色传感器是一个非常好的想法。

由于 RGB 传感器只能看到正前方的东西，[XenonJohn]组装了一种简单的光导。通过将包含订单的屏幕区域封闭在衬有铝箔的纸板中，传感器可以获得红色(卖出订单)与绿色(买入订单)的大致数量。Arduino 读取数据，进行简单的分析，并在超过阈值时发出警报。他将其命名为**隐眼**，下面嵌入了一段视频演示。

这能完全用软件来完成吗？当然，但 Crypto-Eye 作为一个独立的设备有一定的魅力，它使用简单的视觉输入来进行买卖预测[，就像说话&咒语](https://hackaday.com/2012/12/03/teaching-the-speak-spell-four-and-more-letter-words/)。

创造性的加密交易只是[XenonJohn]的一个副业，他在这些地区更出名的是他对单轮电动汽车的杰出贡献，就像这个 [3000W 电动独轮车](https://hackaday.com/2018/04/23/3000w-unicycles-only-limitation-is-personal-courage/)，它也碰巧有一个带有 80 年代风格语音反馈的 Arduino，就像 Crypto-Eye 一样。
