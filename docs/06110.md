# 汽车雷达和多普勒效应

> 原文：<https://hackaday.com/2017/06/07/automotive-radar-and-the-doppler-effect/>

随着越来越多的汽车自动驾驶，对精确的环境感知传感器的需求也越来越大。从避免碰撞到平稳驾驶，环境意识是任何自动驾驶汽车必须具备的。进入汽车雷达:酷，精确，相对便宜。多亏了一个捐赠的汽车雷达模块，[Shahriar]给了我们一个“[教程、实验和拆卸](https://www.youtube.com/watch?v=8af9YVIqhRk)”

在深入研究 PCB 之前，[Shahriar]解释了这个理论。Shahriar 用足够的数学来解释数学上的神奇倾向，而不是太多的数学上的不利因素，深入到汽车雷达如何不同于普通固定雷达的细节中。

在简要概述了多普勒效应之后，[Shahriar]深入研究了 PCB，揭示了三个板上芯片 ASICs 负责产生和接收耦合到 2D 天线阵列的 77GHz FMCW 信号。此外，[Shahriar]指出了几种微波元件，如“鼠形耦合器”和“支线耦合器”此外，[Shahriar]展示了他来自 SV1AFN 设计实验室的酷炫 [PCB 尺子](https://www.sv1afn.com/pcbruler.html)，他用它作为这些微波元件的参考。最后，用一对 Vivaldi 喇叭天线和一个铜片演示了多普勒效应雷达的物理实施例。

我们真的很喜欢[Shahriar]构建他的视频的方式:理论，随后是拆卸，然后是物理实验，以使他的课程回家。如果他还没有工作，我们会说他可能会考虑教书。如果休息后的视频还不足以引起今天的关注，[我们将为您报道](https://hackaday.com/tag/radar/)。
