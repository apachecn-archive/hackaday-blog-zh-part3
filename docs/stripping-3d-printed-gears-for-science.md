# 为科学剥离 3D 打印齿轮

> 原文：<https://hackaday.com/2017/10/17/stripping-3d-printed-gears-for-science/>

虽然 3D 打印现在正像台锯或车床一样变得“无聊”，但曾经有一段时间，媒体甚至一些早期采用者会告诉你，普通的桌面 3D 打印机可能只比我们在*企业中看到的那种复制器技术落后几十年。但是随着这些机器越来越多，越来越多的人可以近距离观看，现实很快就被消化了。*

许多人认为这项技术只不过是一个新奇的东西，甚至在 3D 打印社区本身也有一种感觉，大多数打印机只不过是用来生产“吸尘器”。有些人会认为这种态度令人沮丧，但黑客们在[Gear Down For What？]把它看作是一个挑战。他们的使命是将打印零件推到越来越荒谬的高度，以展示这项技术的能力，在他们的最新作品中，[开始将一对 3D 打印变速箱推向失败](https://www.youtube.com/watch?v=sKt5I0eQGXQ)。

视频以两个他们自己设计的变速箱之间[的正面交锋开始。当他们用电池驱动的钻头旋转时，两个人中较小的一个很快就放弃了，只剩下 228 磅。第一轮的获胜者然后继续拉一个静态负载，只是最终最大限度地扩大了规模，令人印象深刻的 680 磅。](https://www.thingiverse.com/Gear_Down_For_What/designs)

变速箱可能击败了秤，但实验的目标是让它运行到失败。通过装配一个复合滑轮装置，他们能够将他们的秤所能检测到的力增加一倍。随着容量的增加，变速箱在开始打滑之前达到了惊人的 1000 磅。

但也许最令人印象深刻的是:他们拆开变速箱后，发现十个行星齿轮中只有一个坏了。即便如此，从齿轮的剪切方式来看，这个问题更有可能是由于印刷过程中的层间附着力差，而不是单纯的压力。没有齿轮被剥离，事实上，在机械装置的任何地方都没有可见的损坏。该小组目前无法解释失败的原因，只能说应力可能太大，塑料变形足以使齿轮不再紧密啮合。

这不是我们第一次在[减速做什么？]，就在几个月前，他们[用他们的一种打印机制](https://hackaday.com/2017/07/27/3d-printed-gearbox-lifts-an-anvil-with-ease/)举起了一个铁砧，给我们留下了深刻的印象。他们也[不是唯一好奇](https://hackaday.com/2016/08/06/real-research-towards-an-experimentally-verifiable-model-of-fdm-3d-print-strength/)了解[3D 打印塑料到底能走多远](https://hackaday.com/2017/06/30/using-nanotubes-to-strengthen-3d-prints/)的人。
