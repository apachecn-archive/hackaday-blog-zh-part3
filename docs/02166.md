# 用于自复制的纸板和回形针数控绘图仪

> 原文：<https://hackaday.com/2016/04/27/cardboard-and-paperclip-cnc-plotter-destined-for-self-replication/>

去年 11 月，在[HomoFaciens]' [垃圾筒 CNC 建造完成](https://hackaday.com/2015/11/21/garbage-can-cnc-machine-build/)之后，我们发出了挑战——用硬纸板和回形针建造一个可工作的 CNC。现在， [OP 不仅交付了带有工作数控绘图仪的](http://www.homofaciens.de/technics-machines-cnc-v3-0_en_navion.htm)，他还计划将其开发成一台自我复制的机器。

老实说，我们是带着严肃的态度接受挑战的。毕竟，瓦楞纸板怎么可能为数控机床的框架制造足够坚硬的结构呢？[HomoFaciens]通过使用更不柔顺的刨花板来解决这个问题——可能最接近我们在美国所说的 matboard。他为机器设计的模板是经过深思熟虑的；主框架是一个扭转盒设计，方式和幻灯片是错综复杂的事情。非纸板零件包括丝杠的螺纹杆、为连续旋转而修改的伺服系统、Arduino 和前述的回形针，这些零件在用户界面、限位开关和每个轴的极其智能的编码器中使用。下面的视频展示了构建和结果的亮点。

没错，这台机器只能移动一支笔，精度没什么可夸耀的。但是它是有效的，它完全能够向初学者教授所有的 CNC 构建基础，这是一个关键的设计目标。而且它已经准备好进入下一个阶段，成为一台能够自我复制的机器。我们会密切关注这一次。
