# 周五黑客聊天:人群供应

> 原文：<https://hackaday.com/2017/07/26/friday-hack-chat-crowd-supply/>

众筹充其量是鱼龙混杂。一方面，你有非常成功的产品，如 Pebble、Oculus 和 Kano 个人电脑，它们设法赚钱，生产产品，并成为一家成功的公司。(即使只是一会儿，卵石也很棒。)另一方面，你有明显的骗局，比如由尼日利亚骗子操纵的彩色挑选笔。

[![](img/843ae97f2be7b6fbbb8f838761fae362.png) ](https://hackaday.com/wp-content/uploads/2017/07/2917461501014351373.png) [人群供给不同](https://www.crowdsupply.com/)。与其他众筹平台不同，要获得众筹，你需要一个工作原型。其他平台可以通过成功资助了多少活动来衡量他们的成功，以及这些活动中有多少成功向支持者提供了奖励，但我不知道*有任何*众筹活动完全失败过。

在本周的 Hack Chat 中，我们将与大众供应的首席执行官【Josh Lifton】进行对话。主题将包括确定你的产品是否有市场，履行的细节，运送你的产品。我们接受来自社区的问题，这里有[一份我们将阅读的问题单](https://docs.google.com/spreadsheets/u/1/d/1rQyrgYiTls3NnrgPDwYR1slN2jNU9jA8NsdsROO8LiM/edit?usp=sharing)。

Josh 拥有麻省理工学院媒体实验室的博士学位，并拥有斯沃斯莫尔学院的物理和数学学士学位。在“大众供应”之前，Josh 在各种技术环境中工作过，从为太阳马戏团的表演使用定制的可穿戴计算机为数千名观众配备仪器，到最近担任木偶实验室的工程主管。

### 以下是参与方式:

[![join-hack-chat](img/cff5733f39a173302e1cd102e75ac107.png)](https://hackaday.io/project/5373-hack-chat) 我们的 Hack Chat 是 Hackaday.io 上的实时社区活动 [Hack Chat](https://hackaday.io/project/5373-hacker-channel) 群发消息。这次聊天将在太平洋时间 7 月 28 日星期五中午进行。困惑于‘中午’是在何时何地？这里有一个时间和日期转换器！

登录 Hackaday.io，访问该页面，并寻找“加入这个项目”按钮。一旦你成为这个项目的一部分，这个按钮就会变成“团队信息”，直接带你进入黑客聊天。

你不必等到星期五；随时加入，你可以看到社区在谈论什么。
