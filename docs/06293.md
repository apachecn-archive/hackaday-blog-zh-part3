# 为游戏男孩颜色写一个新游戏

> 原文:[https://hackaday . com/2017/06/20/writing-a-new-game-for-the-game-boy-color/](https://hackaday.com/2017/06/20/writing-a-new-game-for-the-game-boy-color/)

如果你对 Game Boy Color 的产品感到厌倦，这是可以理解的:它已经存在了近 20 年，最近没有太多新的发布。[Antonio nio díaz]花了一年多的时间为 GBC 编写了一款游戏: [City，一款模拟人生城市风格的游戏](https://github.com/AntonioND/ucity)。他设计了图形，甚至写了自己的音乐。

[Antonio]用汇编语言编写了所有的程序，创建了管理交通和电网的模块，创建和破坏建筑物，以及模拟灾难。他在 GitHub 页面[上有大量的注释，详细描述了每个模块](https://github.com/AntonioND/ucity/blob/master/docs/index.rst)并描述了它们是如何协同工作的。他给了它一个 GPLv3+的许可证，所以黑掉。

ROM 可以在模拟器上工作，但是[Antonio]已经验证了它可以在原始硬件上工作；它只是减少了保存城市的数量，以适应掌上电脑较少的统计数据。

Hackaday 喜欢我们的 GBC:我们已经为 GBC 写了 [DIY 协处理器](http://hackaday.com/2016/11/09/diy-coprocessors-for-the-game-boy-color/)，添加了音频放大器，甚至制作了太阳能游戏机。
