# Evezor 机械臂雕刻 400 个杯垫

> 原文：<https://hackaday.com/2017/05/22/evezor-robotic-arm-engraves-400-coasters/>

当你在为一个机械臂运行 Kickstarter 时，你最好准备好证明它的可重复性和准确性。[Andrew]已经用 Evezor、[和他的 SCARA 手臂](https://hackaday.io/project/20416-evezor-robotic-arm)通过激光雕刻了 400 个木制杯垫[，这些杯垫在树莓 Pi 电脑上运行，由光滑的滑板控制步进器。](http://imgur.com/a/kSsjm)

Evezor 是一个相当惊人的项目:一个通用臂，只要有合适的末端执行器，它可以做从布线电路板到焊接的任何事情。如果这听起来很熟悉，那是因为[Andrew]在芝加哥举行的[hack aday 大会上做了一个关于 Evezor 的演讲，](http://hackaday.com/2017/03/23/the-think-tank-at-the-chicago-unconference/)

Evezor Kickstarter 的奖励之一是一个简单的木制杯垫。[Anderw]用台锯切割出每一个木方。然后他做了一堆，并开始编程 Evezor。这 400 个杯垫被分别拿起并放入一个夹具中。然后 Evezor 使用一个小型二极管激光器雕刻自己的标志和一个单独的数字。雕刻好的杯垫然后被整齐地堆放在一起。

编程和设置完成后，[Andrew]点击 go 离开了大楼。不过，他确实在盯着 Evezor。一个婴儿监视器以低分辨率捕捉到了这个动作。两台 DSLR 相机也拍下了每个被雕刻的杯垫的照片。休息之后可以找到由此产生的延时视频。
