# "看，马，没有陀螺！":自动平衡的机械迅猛龙

> 原文:[https://hack aday . com/2017/05/07/look-ma-no-gyros-a-self-balancing-mechanical-velociraptor/](https://hackaday.com/2017/05/07/look-ma-no-gyros-a-self-balancing-mechanical-velociraptor/)

你得先走，然后才能跑，对吗？也许不会，如果这个像恐龙一样奔跑的双足机器人有任何迹象的话。

这个机器人被正式命名为“平面椭圆跑步者”，是人类与机器认知研究所的两足运动测试平台。从鸵鸟的步态中获得灵感——我们认为它看起来更像霸王龙或迅猛龙，但也有相同的区别——IHMC 的[Jerry Pratt]团队已经做出了一些非常了不起的东西。与我们见过的所有两足和四足机器人相反，像 [Boston Dynamics 的大狗](http://hackaday.com/2011/09/30/new-bigdog-video-doesnt-fail-to-impress/)和 [PETMAN](http://hackaday.com/2009/10/27/bigdogs-bipedal-brother/) ，它们都布满了传感器和致动器，PER 非常精简。

![](../Images/5a17ceab2116f594bf3d2d321a13cd85.png)单个马达使用连杆运行整个传动链，这对任何拆开椭圆训练机的人来说都很熟悉，而且车上没有计算机或传感器。PER 通过团队所谓的“反应弹性”保持平衡:驱动链轮和曲柄之间的扭簧自动调节落地腿和摆动腿的动力，以在跑步过程中提供稳定性。下面的视频很好地展示了这一点，如果你从 2:03 开始单帧拍摄；注意机器人在大步行走时曲柄臂的可变角度。

跑步机测试受到几块塑料板的限制，但下一个版本将免费运行。还不清楚方向控制将如何实现，也不清楚 PER 将如何停止运行并保持平衡。但这是运动方面一个有趣的进步，我们期待看到 IHMC 的下一个戏法。
