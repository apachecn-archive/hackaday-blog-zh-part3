# 开始你成为天线大师的道路

> 原文:[https://hack aday . com/2016/11/27/start-your-path-to-be-an-antenna-guru/](https://hackaday.com/2016/11/27/start-your-path-to-becoming-an-antenna-guru/)

这些年来，我们认识了一些对天线有秘密见解的人。然而，对我们大多数人来说，这有点像一门黑色艺术(这解释了所有你能在网上找到的由不可思议的元素制成的准科学天线)。曾经有一段时间，只有火腿和射频书呆子关心天线，但这些天无线无处不在:手机，WiFi，蓝牙，甚至射频遥控器的生死都取决于他们的天线。

你可以找到许多关于天线的高能数学讨论，充满了麦克斯韦方程、球面积分和其他高能微积分，以及许多晦涩难懂的图表。[马克·休斯]最近发布了一个关于天线的两部分介绍，少了一些数学，多了一些动画图像，这对我们来说很好(当你完成了第一部分，请查看[第二部分](http://www.allaboutcircuits.com/technical-articles/antenna-basics-field-radiation-patterns-permittivity-directivity-gain/))。他还附上了一段视频，你可以在下面找到。

第一部分相当简单，讨论了历史和电磁学。不过也讲了叠加，反射，驻波比。第二部分，虽然，进入辐射模式和增益。总的来说，这是一个通向相对神秘的艺术的大门。

我们之前已经讨论过[史密斯图](https://hackaday.com/2015/10/21/nomograms-complex-analog-calculators-simple-for-everyone/)，这可能是天线学徒向导的下一个逻辑步骤。我们还介绍了 [PCB 天线设计](https://hackaday.com/2010/08/05/pcb-trace-antenna/)。
