# 透明 3D 打印？

> 原文:[https://hackaday.com/2017/05/13/transparent-3d-printing/](https://hackaday.com/2017/05/13/transparent-3d-printing/)

透明塑料并不新鲜。然而，3D 打印通常是不透明的，或者充其量是半透明的。[Thomas Sanladerer]想打印一些真正透明的东西。他注意到 Colorfabb 有一篇关于用 HT 丝打印透明件的文章。[Thomas]想用标准的(更便宜的)PETG 做同样的事情，它在化学结构上与 HT 相似。他成功了吗？观看下方的[视频，一探究竟。](https://www.youtube.com/watch?v=QkfQri2B0PY)

你可以得到许多清晰的塑料细丝，但是印刷层的过程使透明度变得模糊，显然主要是由于层之间的小间隙。HT 细丝的想法是在足够高的温度下过度挤压，使得各层可以熔合在一起。

[Thomas]想为灯具打造一些透明部件和漫射器。扩散器使用花瓶模式打印，即使没有透明扩散器，他创建的灯看起来也很棒。

他的第一个实验涉及层的高度和挤压率。他试图确定是什么让事情变得更好和更坏，并在此基础上修改他的技术。他还尝试了一些后期处理步骤。

如果你想看看除了 Colorfabb 之外的其他人制作的 Colorfabb HT 零件是什么样子，请查看【3D 打印教授】下面的第二个视频[。在他做一些后期处理之前，他制作的照片看起来不是很清晰。即使在后期处理之后，它也不会欺骗任何人认为它是透明的。然而，](https://www.youtube.com/watch?v=KesvLciE8rw) [Colorfabb 在他们的博客帖子](http://learn.colorfabb.com/lets-make-something-clear/)上展示的关于材料的部分看起来确实很惊人。在用于防止间隙的过度挤压和后处理步骤之间，[3D 打印教授]警告说，使用这种技术获得精确尺寸的零件并不容易。

如果你预算很大，你可以尝试用真正的玻璃打印。似乎有几种[方法](https://hackaday.com/2017/04/21/3d-printing-glass-using-stereolithography/)可以做到这一点。
