# 业余无线电使断路器跳闸

> 原文：<https://hackaday.com/2017/02/06/ham-radio-trips-circuit-breakers/>

电弧故障断路器是家庭电气安全的福音。普通的家庭电气火灾通常是由故障电线电弧放电产生的热量引起的。但是任何无线电爱好者都知道，火花也会发出宽带无线电噪声。电弧故障断路器(AFCI)是一种特殊的断路器，它监听电力线中的这种噪声，并在听到时跳闸。问题是，他们可能太敏感了，以至于不必要地剪掉了。看看下面这段有趣的视频。

我们的朋友[马丁]搬进了新房子，发现他可以通过 20 米波段的传输来触发断路器。“这个地方所有的灯都熄灭了，我的装备改用电池供电。我觉得这很奇怪，因为我肯定画不到 20 A。我重置断路器，并再次键入。我重新设置了断路器，然后用谷歌搜索了一下。

当然，这是火腿社区中的一个已知问题。特别是，一家制造商在误解有意辐射方面存在严重问题，[向业余无线电爱好者社区寻求帮助](http://www.arrl.org/news/arrl-helps-manufacturer-to-resolve-arc-fault-circuit-interrupter-rfi-problems)开发新版本的原型。当他打电话给制造商并告诉他们时，他得到了免费的抗火腿破碎机，所以一切都很好。
