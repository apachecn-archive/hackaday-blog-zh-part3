# Filmomat 家庭电影处理系统

> 原文：<https://hackaday.com/2015/12/08/the-filmomat-home-film-processing-system/>

电影的死亡已经被广泛报道，但是技术只有在过时后才是完美的。这可能不是即时摄影，但至少有一种机器可以拍摄 35mm 胶片和 5×7″照片[，并自动冲洗它们](http://silver-image.blogspot.com/2015/12/der-filmomat.html)。它被称为 Filmomat，虽然它不会出现在许多摄影师的工作室中，但它是自动化的一个令人难以置信的例子。

Filmomat 是一个由阀门、管道和泵组成的令人难以置信的组合，它可以自动处理任何合理尺寸的胶片，从 35mm 到 5×7 彩色幻灯片。机器的主体是一个丙烯酸立方体，细分成不同的部分，包含照片处理化学品，冲洗水和浴缸。有了微控制器、有机发光二极管显示器和旋转编码器，不同的显影过程可以被编程，化学药品被加热，显影剂被搅动，胶片被处理。Filomat 能够存储使用三种化学品的五十种不同工艺，最多十个步骤。

[这款设备的视频是它的卖点](https://www.youtube.com/watch?v=BhgOWvP1vZA)，尽管还不完全是；如果有足够多的人感兴趣，这部电影有一天可能会被卖掉。这可能是最容易的胶片冲洗了，但是一项技术只有在被淘汰后才是完美的。
