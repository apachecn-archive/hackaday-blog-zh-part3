# 镶嵌式工作灯是漂亮的、模块化的

> 原文：<https://hackaday.com/2018/04/18/tesselated-worklights-are-nifty-modular/>

电灯照明——有什么是它做不到的吗？有各种各样的形式和风格，从紧凑型阅读灯到车库用的强光工作灯，应有尽有。一般来说，不同的灯以不同的方式建造，以适应他们的目的，但这不是唯一的方法。[进入【slisgrinder】和马赛克照明系统。](https://hackaday.io/project/88666-mosaic-ls)

从本质上讲，MOSAIC 是一种用单个模块构建照明装置的方式。有趣的是设计——它们是三角形！电路板上装有各种 led，其布局方式允许相邻电池单元之间通过相邻布局实现电源和数据连接。许多棋盘可以镶嵌在一起，形成更大、更小或不规则形状的阵列。连接是经过深思熟虑的，允许瓷砖沿其 3 条边中的任何一条连接，而不管方向如何。

这个项目开始于在一个不适宜种植秋葵的气候条件下种植秋葵的愿望；为此，既有普通的工作灯模块，也有内置紫外线发光二极管的增光灯模块。模块可以不同方式组合，命令和控制通过 RS-485 完成。

这是一个整洁的项目，展示了一个小小的想法如何通过使用一个不寻常的外形来创造一个多功能的设计。我们以前也见过模块化照明项目，比如去年 Hackaday 奖的参赛作品。
