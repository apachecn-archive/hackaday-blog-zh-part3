# 回到未来闹钟是真实的

> 原文:[https://hack aday . com/2015/10/24/back-to-the-future-alarm-clock-is-as-real-as-it-gets/](https://hackaday.com/2015/10/24/back-to-the-future-alarm-clock-is-as-real-as-it-gets/)

虽然真正的时间旅行显然不会很快发生，但有了这个[回到未来主题闹钟，](http://imgur.com/gallery/KuVQZ)你可以通过睡觉以 7 小时 8 小时为增量前往未来。天哪！

[CrossleyAcoustics]花了几个月的时间来设计和制造这个值得一看的闹钟，它确实展示了。在设计了试验板上的所有东西后，他定制了 PCB，并在 Sketchup 中建模(这是他第一次使用它！)甚至在商店向他开价数千英镑后，他开始尝试金属板材制作。

这个建筑最酷的部分是[CrossleyAcoustics]有一些他 8 岁时自己画的细节图。一起讨论一个计划！
