# 桌面工厂教 PLC 编程

> 原文：<https://hackaday.com/2017/04/16/desktop-factory-teaches-plc-programming/>

如何培养工业自动化方面的年轻工程师是一个棘手的问题。大多数工厂都有大的东西，如果新手导致崩溃，会造成很大的损害和大量的金钱损失。解决方案:[将工厂缩小到桌面大小，让他们在那上面练习](https://shortn0tes.blogspot.com/2017/04/make-interface-board-to-connect.html)。

幸运的是，有一个现成的工厂自动化小型化解决方案:[慧鱼工业培训模型](http://www.fischertechnik.de/en/Home/products/industry.aspx)。这些模型有电机、传送带、气缸和传感器，但控制器并不完全是工业标准的可编程逻辑控制器(PLC)。[Vadim]着手通过在慧鱼模型和西门子 PLC 之间建立接口来解决这个问题。他对他的电路板进行了几次修改，其中一次使用了缝纫店的铆钉来连接慧鱼连接器。最终，他决定采用更坚固的连接器，并设计出一种电路板，让学生在不伤害任何人的情况下钻研 PLC 编程。下面的视频展示了它的步伐；我们只能想象小时候玩这些装备会把我们带到哪里。

虽然[Vadim]的系统对培训工程师很有用，但我们也可以看到它有助于让孩子们对工业自动化的职业感兴趣。我们最近报道了一个类似的尝试，用乐高思维风暴向孩子们展示大科学。这两者都有助于让 STEM 孩子看到更广阔的技术职业世界，或许还能引导他们走向自动化。毕竟，制造机器人的人可能是最后一批被淘汰的人，对吗？
