# 太阳能电池板上的芯片调谐

> 原文:[https://hackaday.com/2018/04/06/chiptunes-on-a-solar-panel/](https://hackaday.com/2018/04/06/chiptunes-on-a-solar-panel/)

凭借其复古的声音，不会弄错 80 年代和 90 年代视频游戏的独特 8 位声音。它变得如此受欢迎，最终引发了它自己的音乐流派，被称为“chiptune ”,音乐家们今天仍在为它作曲。虽然音乐有一些其他的品质，即从数字的角度来看它相对简单。[无处不在的机器人]发现这种简单性使其成为无线电力传输的完美载体。

该项目更像一个无线电发射器和接收器，而不是一个真正的无线电力发射器，但原理是相同的。它使用一个经过修改的扬声器驱动器和放大器连接到光源，而不是扬声器。在接收端，有一个太阳能电池板(本质上是一个大型光电探测器)，直接连接到一对耳塞上。当 chiptune 通过放大器播放时，它通过光发送到太阳能电池板，在那里可以通过耳塞听到。

该项目被限制在每秒 24，000 字节，这比仅仅在你的邻居周围传播随机音频文件有用得多，尽管这仍然有效。你也可以使用类似这样的东西来无线建立长距离串行链接，这可以成为[长距离通信网络](https://hackaday.com/2015/07/02/solar-cell-laser-communication-system/)的基础。
