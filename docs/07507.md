# Hackaday 奖参赛作品:一块试验板上的一堆 VGA

> 原文:[https://hackaday . com/2017/11/06/hackaday-prize-entry-a-mess-of-VGA-on-a-a-bread board/](https://hackaday.com/2017/11/06/hackaday-prize-entry-a-mess-of-vga-on-a-breadboard/)

在我们所有的视频游戏出现在互联网上之前，在它们出现在 CD 上之前，在它们出现在盒式磁带上之前，视频游戏都是离散的逻辑。Pong 是第一个，你可以用几十个逻辑芯片来构建它。伟大的[沃兹]用 44 个简单的筹码建立了一个著名的突破。

对于[马塞尔]进入 Hackaday 奖，他把单板无微处理器计算机带到了一个新的水平。他正在试验板上建造一台多兆赫的 64 色电脑。试验板的电容是多少？就问【马塞尔】。

这种解体计算机的设计几乎拥有你想要的分立 CPU 的一切。没有微控制器或类似 74181 ALU 的复杂芯片，有时每时钟两条指令的流水线，用二极管解码，以及 60 Hz，64 色 VGA 输出和 4 个声道。这块板上只有大约 40 个 TTL 芯片。

这个 Hackaday 奖项的项目日志本身就是一种享受，从主题到 NES 控制器的实现，再到摆脱试验板，将这台计算机变成类似老式游戏系统的东西，但带有定制的 CPU 和指令集。这是一个惊人的构建，也是 Hackaday 奖的一个令人敬畏的项目。
