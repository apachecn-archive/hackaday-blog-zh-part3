# Pi 驱动的机器人玩拼图和龙

> 原文：<https://hackaday.com/2016/02/10/pi-powered-robot-plays-puzzle-dragons/>

你听说过 iPhone 游戏拼图&龙吗？我们知道它在日本很大。谜题部分是所有关于重新安排彩色球和分组在一起，所以他们消失了，给你点。它也是 RPG 的一部分——每种颜色都与一个小家伙相关联，每种组合都会使匹配颜色的家伙变得更强大一点。然后你打怪物，拿战利品。

一位名叫[Junya Sakamoto] [的 YouTube 用户制造了一个机器人来为他玩游戏](http://9to5mac.com/2016/02/09/iphone-game-playing-robot/)。它分析棋盘，想出一系列可能的最佳走法，然后用 Raspberry Pi 执行它们。你可以在连接的笔记本电脑屏幕上看到机器人正在计划什么。由 Gertbot 控制的几个马达使触控笔在屏幕上上下移动，并使手机本身前后移动。

休息之后有两段视频。第一个是一个组合构建和演示视频，证明这个机器人可以做更多的事情，而不仅仅是消磨时间玩拼图&龙。第二个视频展示了玩拼图与龙有多棒。

我们很喜欢这里的游戏机器人。这是一个扮演侏罗纪公园建造者的乐高杜伊诺机器人。另一个机器人是用来测试应用程序的，但它[在业余时间玩愤怒的小鸟](http://hackaday.com/2012/07/24/shakey-robot-plays-angry-birds/)。
