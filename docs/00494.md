# 密码之轮！

> 原文:[https://hackaday.com/2015/10/28/wheel-of-password/](https://hackaday.com/2015/10/28/wheel-of-password/)

在任天堂 Gameboy 兴起之前，Tiger LCD 游戏是手持游戏之王。即使在今天，你仍然经常会发现它们“在野外”或潜伏在你的房子里，价格低廉，对广大观众有吸引力。当[Lee]发现一个“幸运之轮”模型躺在一个盒子里时，[看看里面的情况](http://www.sodnpoo.com/posts.xml/totp_two_factor_auth_on_tiger%27s_wheel_of_fortune.xml)，然后把它变成另一个样子。

基于一个游戏节目，这个特定的型号有一个大多数 Tiger 掌上电脑没有的功能:一个墨盒插槽。最初是为了提供额外的类别和短语，这个插槽是一个通向内部 CPU 的完全开放的总线。没用多长时间，总线盗版者就破解了数据协议。

那么，一个人如何处理被黑客攻击的游戏节目呢？你可以让它说些愚蠢的东西，或者你可以让它变成一个 TOTP 密码生成器。未来的计划是拿掉电脑的脐带，用 AVR 咬开购物车。一旦做了任何人，试图闯入[李的]个人电脑永远不会怀疑无辜的旧玩具是王国的关键。
