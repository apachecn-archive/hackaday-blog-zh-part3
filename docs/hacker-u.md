# 黑客大学。

> 原文：<https://hackaday.com/2017/04/16/hacker-u/>

如果你去南佛罗里达大学，你可以选择“Makecourse”这个为期 15 周的项目承诺教授 CAD 软件、3D 打印、基于 Arduino 的控制系统和 C++。不去南佛罗里达大学？别担心。鲁迪·施拉夫教授和埃里克·特里达斯将整个课程放到了网上。你可以在下面看到几个视频，但是还有更多。学生项目视频也很棒，比如[卡特林·瑞安的]月球项目阶段(见下文)或[达斯汀·杰曼的]漫游者(见上文)。

除了课程计划和项目，还有一整套视频(你可以在下面看到一些)。如果你是一个普通的 Hackaday 读者，你可能不会太关心基本的 Arduino 东西和基本的电子产品——尽管一篇好的评论不会伤害任何人。然而，关于中断、sd 卡、pin 改变中断的更高级的主题可能是正确的。如果您曾经想学习 Autodesk Inventor，也有相应的视频。

如果你不需要所提供的任何指导，这仍然是一个很好的程序，可以在本地黑客空间或任何你想教构建的地方提供。你可以从各种各样的学生项目中看到，它非常平衡，让学生专注于他们最感兴趣的领域。

现在网上有如此多的教育材料，以至于很难找到时间去看其中的一小部分。例如，我们喜欢 EdX，但是谁有时间去上哪怕是一小部分的课程呢？我们总是喜欢看学生的项目——它们给了我们灵感。尤其是布鲁斯·兰德的[课程](https://hackaday.com/2016/06/26/mag-lev-without-the-train-but-with-an-fpga/)，总是鼓舞人心。
