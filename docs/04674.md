# 给你的树莓派一个好锤

> 原文:[https://hack aday . com/2017/01/09/give-your-raspberry-pi-a-good-hamming/](https://hackaday.com/2017/01/09/give-your-raspberry-pi-a-good-hammering/)

Raspberry Pi Zero 的一个特性是它没有安装 GPIO 头引脚。丢失的针脚降低了这台小电脑的价格，也降低了它的装运量。因此，大多数新的圆周率零点拥有者面临的任务是将一组引脚焊接到孔中，事实上，许多供应商会将引脚与新的零点一起出售给你。

英国 Pi 配件供应商 Pimoroni 认为他们可能有解决这个问题的办法，他们有一套无焊针[，用户希望通过用锤子](https://shop.pimoroni.com/collections/raspberry-pi/products/gpio-hammer-header)敲击针和 Pi 来安装。每个引脚都设计为在压力下变形，并夹住 PCB 孔的镀通壁。实际上，它们是[推入配合销](https://www.samtec.com/products/pht)，设计用于安装压力机或特殊工具，但由于一般零购买者既没有也没有，所以他们提供一个小型激光切割夹具，并给出使用销锤或类似工具小心轻敲的指示。他们有一个演示，作为他们定期舱底舱播客的一部分，我们在休息时包括了这一部分。

当用合适的工具安装时，像这样的销是非常可靠的。它们通常用于军事和航空航天系统。不过，在这种情况下，我们预计你们中的一部分人会准备评论说，焊接连接器会好得多，我们不禁同意你的观点。当然，这款产品并没有真正面向黑客读者。相反，像 Zero 这样的董事会的目标市场是儿童。对他们来说，焊接可能太远了。我们不禁想知道锤子的安装是否会提供足够可靠的接触，以及我们是否会看到一群年轻人的圆周率帽由于不可靠的连接器而无法工作。当然，除了那些用锤子打破零分的人之外，还有一些有点热情的人。
