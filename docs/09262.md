# 卷起袖子，使用这款智能手表前臂投影仪观看视频

> 原文：<https://hackaday.com/2018/04/25/roll-up-your-sleeve-watch-a-video-with-this-smart-watch-forearm-projector/>

我们都在慢慢习惯可穿戴技术的概念，尽管有令人毛骨悚然的谷歌眼镜这样的失败案例。但是 tiny tech 的大问题是为用户界面寻找合适的空间。当然，我们可以把它做得很小，但是人类的手指不会变得更小，眼球只能分辨这么多细微的细节。

那么，我们如何让可穿戴设备变得更有用呢？根据卡内基-梅隆大学的研究人员[Chris Harrison]的说法，一种方法是将佩戴者变成显示器和输入设备。更具体地说，他的 LumiWatch 将触摸感应显示屏投射到佩戴者的前臂上。下面的视频非常流畅，一些明显的 CGI“艺术家的再现”显示在前面。但是，即使是视频中显示的有些有限的显示也非常令人印象深刻。这款手表可以占用用户前臂 40 厘米的空间进行显示，即使是在仅略高于手臂的表圈提供的浅投影角度下——考虑到皮肤的不规则表面，这是一个相当大的壮举。它通过一个由红色、蓝色和绿色激光器组成的“微型投影仪”和一对 MEMS 反射镜来实现这一目标。投影仪可以调整显示器的线性度和亮度，以在不平坦的表面上提供一致的图像。10 个飞行时间传感器阵列负责观察触摸输入手势的显示区域。这是一个迷人的项目，有很大的潜力，但我们想知道人体的可变性是如何干扰展示的。更不用说一年四季都需要短袖了。
