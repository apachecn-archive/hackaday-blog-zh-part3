# Hackaday 奖品入口:一把 WiFi 瑞士军刀

> 原文:[https://hack aday . com/2016/08/07/hack aday-prize-entry-a-wifi-Swiss-army-knife/](https://hackaday.com/2016/08/07/hackaday-prize-entry-a-wifi-swiss-army-knife/)

WiFi 就在我们身边，但如果你想使用这种无处不在的网络协议，你需要像穴居人一样拿出笔记本电脑或智能手机。[丹尼尔]有一个更好的主意。这是一个简单、紧凑的工具，用于破解 WiFi 密码或向当地星巴克的每个人发送 deauth 数据包。这是一把 ESP 瑞士军刀，也是 Hackaday 奖的绝佳参赛作品。

正如你所料，这款 WiFI 瑞士军刀由 ESP8266 供电，具有一个微型有机发光二极管显示屏和一系列 UI 按钮。有了这个，[Daniel]就能够在网络上执行 deauth 攻击，将任何人踢出网络，前提是该设备已经拥有受害者的 MAC 地址。

这个微小的无线工具也有一个 SD 卡，可以收集认证帧，供以后在实际上有能力破解网络的设备上解密。有了 LiPo 充电控制器和足够大的电池，这个微小的设备可以被放在办公室的角落里收集认证包几天，直到后来被找回，向任何拥有足够快的计算机的人开放网络。这是一个很好的构建，非常有用，这使得它成为 Hackaday 奖的一个很好的参赛作品。
