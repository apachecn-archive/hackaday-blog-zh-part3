# 收音机和电话扬声器很有风格

> 原文：<https://hackaday.com/2017/02/22/radio-and-phone-speaker-has-style/>

建造一个水晶收音机并不是火箭科学。一些建造它们的人追求在技术上尽可能地推动它们。其他人，比如[Billy Cheung]，则追求时尚。模块化收音机和电话扬声器看起来像是出自电影《巴西》。金属留声机般的喇叭增加了吸引力，也机械地放大了声音。

这个视频(见下文)并不是一个具体的操作方法，但是如果你看完了，有足够的信息让你可以重现至少类似的东西。实际上有几个角。一个是铜做的，一个是纸做的，还有一个是塑料瓶做的。

还有一个插座可以接不同的探测器。收音机似乎有一个很大的频率范围，尽管你必须眯着眼睛看视频才能看到示意图。我们希望[比利]最终也能把它发布到他的博客上。切割和组装喇叭的模板也是如此。但是，如果您冻结视频，您可以看到大部分(如果不是全部)主要组件。

我们以前在这里见过[比利]带着[简单的水晶收音机](https://hackaday.com/2015/09/05/crystal-radio-its-a-match/)。如果你想看看你对水晶收音机的痴迷能走多远，[看看这个](https://hackaday.com/2016/09/16/high-performance-crystal-radio/)就知道了。
