# Hackaday 奖参赛作品:无线充电自热咖啡杯

> 原文:[https://hack aday . com/2016/05/16/hack aday-奖品-参赛-无线充电-自热-咖啡杯/](https://hackaday.com/2016/05/16/hackaday-prize-entry-wirelessly-charged-self-heating-coffee-mug/)

许多高效的黑客流着深赭色的血。在某些评论者群体中流行的理论是，他们充满了它，但它实际上是健康黑客的咖啡流中血液含量低的健康迹象。[Bharath]是那些每天都喜欢喝含咖啡因的豆汁的人之一。然而，他患有一种叫做冷咖啡的可怕疾病。为了解决这一问题，他制作了一个支持应用程序、可无线充电、自加热的咖啡杯。

我们知道大多数黑客不会一开始就计划用荒谬的特性列表来构建对象，它只是发生了。对此有没有替代的墨菲定律？任何可以添加的功能都会？该项目开始时，一些低欧姆电阻连接到一个可充电的电源组。选择具有可拆卸不锈钢内衬的保温烧瓶。电阻器用热环氧树脂固定在外面。

然而，我们如何控制电阻呢？我们不想马上耗尽我们的电池(这可能会比我们希望的更糟)，所以[Bharath]添加了 Seeed Studio 的 [Linkit One](http://hackaday.com/2015/10/02/data-logger-powered-by-linkit-one-board/) 微控制器。有了这些权力，他很自然地给杯子添加了蓝牙、温度传感器和应用程序控制。

在把所有这些都收集起来之后，他意识到，虽然在正常使用的情况下内部与烧瓶中的液体完全隔离，但他必须切开来连接充电电路的孔会提供不可接受的水进入点。为了解决这个问题，他增加了无线充电功能。

有了他的烧瓶在手，我们确信不必费力地喝完一杯冰冷的咖啡残渣所带来的情绪提升将极大地提高工作效率。休息后的视频。
