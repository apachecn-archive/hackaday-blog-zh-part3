# CES2017: Lulzbot 拥有最好的物品名称

> 原文:[https://hack aday . com/2017/01/08/ces 2017-lulzbot-has-the-best-names-for-stuff/](https://hackaday.com/2017/01/08/ces2017-lulzbot-has-the-best-names-for-stuff/)

Lulzbot 是 RepRap 项目的典型代表。他们做的一切都是公开的。在 CES 上，Lulzbot [推出了 MOARstruder](https://www.lulzbot.com/learn/announcements/aleph-objects-inc-announces-new-hardware-ces-2017) ，一种喷嘴直径为 1.2 毫米的工具头。这将挤出大量塑料，从而加快打印速度。这与最大的 [E3D 火山](http://e3d-online.com/blog/volcano_release)的喷嘴直径相同，从展台周围的大照片来看，结果是相似的:你在更大的层高度获得更快的打印，层线成为一种设计特征。

Lulzbot 本周[还宣布了 Cura 2 (Lulzbot edition)](https://www.lulzbot.com/learn/announcements/aleph-objects-inc-announces-new-software-ces-2017) 的发布，这是与 Blender Institute 的合作，旨在开发现有最佳免费 3D 建模软件的简化版本，以及[与 Monkeyprint](http://robotsinthesun.org/aleph-objects-tests-monkeyprint-dlp-printer-software/) 的合作，旨在开发基于树脂的 3D 打印免费软件。
