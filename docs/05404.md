# 使用 SLA 打印任天堂笔记本电脑

> 原文：<https://hackaday.com/2017/03/26/printing-nintendo-portables-with-sla/>

唐宁]对制造便携式游戏机并不陌生，在这个过程中使用了各种各样的技术。然而，当开始这项任务时，[唐宁]决定采取不同的策略——[在这款任天堂 64 便携式产品中使用 Form 2 SLA 打印机。](https://downingsbasement.com/2017/03/15/the-mithzan-64/)

将家用控制台改装成便携式设备通常需要一些技巧，比如 frank encasing——将控制器、外壳和附件等原厂部件组装在一起，使用主体填充物和大量打磨来创建真空成型模板，然后产生无缝的最终产品。有可能得到一些真正令人印象深刻的结果，但它确实限制了制造商对现有零件的依赖。

通过使用 Form 2，[Downing]能够利用 SLA 打印机的能力来创建具有良好表面光洁度的零件，这在使用更常见的 FDM 技术进行 3D 打印时通常需要大量的打印后整理。这特别有用，因为它允许创建自定义按钮和“恰好合适”的小部件——通常这种部件由库存件制成，然后进行修改。

该建筑还具有其他一些很酷的功能——有一个转接盒，允许连接额外的控制器，以及托管 AV 输出以连接到电视。转接盒通过 HDMI 电缆连接至笔记本电脑。这是一个整洁的选择——这是一种标准电缆，有大量的导体可用，所以你不必特别狡猾地通过它来获得 3 个控制器和几个模拟信号。

最终，[唐宁]不会再为外壳本身使用 SLA 印刷——这个过程太慢，太贵了。在这方面，FDM 印刷后可能需要更多的工作，但它仍然在时间和金钱方面领先。但对于像按钮和结构支架这样的小型定制零件，Form 2 是合适的机器。
