# 披萨按钮

> 原文：<https://hackaday.com/2015/09/29/the-pizza-button/>

你有多少次想吃披萨，但是真的不想离开沙发去点餐，或者上帝保佑，真的不想去披萨店拿一个？[布罗迪·贝尔松]为你提供了解决方案！他重新设计了一个亚马逊 Dash 按钮，只需按一下按钮，他就可以为自己订购达美乐披萨。

他最初的灵感来自于我们最近的一篇关于[黑掉亚马逊破折号按钮来做任何你想做的事情](http://hackaday.com/2015/08/10/hacking-the-amazon-dash-button-to-record-whatever-you-want/)——他想要的是一个披萨按钮。他承认，他的灵感来自于过去的单身汉岁月，在那里，这真的很方便，**真的很方便——很明显，他甚至曾经通过邮局订购过麦当劳！**

这也很简单——他所需要做的就是编写一些代码。有人已经做了一个特定的[达美乐披萨 API](https://github.com/RIAEvangelist/node-dominos-pizza-api) ，由于对 Dash 按钮的兴趣，他能够找到[他需要的一切](https://github.com/hortinstein/node-dash-button)来组合一个一键式披萨递送设备。

最棒的部分——现在已经完成了，按照他的 GitHub 目录 [pizzadash](https://github.com/bhberson/pizzadash) 上的说明，你只需要不到 10 分钟就可以自己重新创建一个。这只是等待与[的网飞和冷却按钮](https://hackaday.com/2015/09/27/the-netflix-and-chill-button/)我们昨天看了。
