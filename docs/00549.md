# 用南瓜炮击落无人机

> 原文:[https://hack aday . com/2015/11/04/用南瓜大炮击落无人机/](https://hackaday.com/2015/11/04/shoot-down-drones-with-pumpkin-cannons/)

你担心不可避免的无人机入侵吗？你是否一直在等待一个可以信赖的防御系统？别再看了。这段视频展示了该系统的有效性——没有烟雾和镜子。[只是结果。](http://www.youtube.com/watch?v=HC89KRV7l9M)

忘记射频干扰或 WiFi 黑客。[如果这些设备可以工作的话，](http://hackaday.com/2015/10/24/radio-jamming-rifle-claimed-to-disable-drones/)它们可能只能用来阻止消费类设备。如果你想确保无人机被击落，你需要一门南瓜炮。

等等什么？我们的英雄正在拍摄他的巨型南瓜炮，以大约 200 英里的时速射击南瓜。到目前为止，只是一个普通的十月下午。但是后来，他的女儿建议从树林那边拍一个不错的视频:

> “如果能从树林里得到视频该多好啊！”

听起来很简单。只需将四轴飞行器飞到树的上方，从旁边拍摄南瓜，就能获得戏剧性的效果。这是末日的开始。不过，他不怪她。理论上这是个好主意…
