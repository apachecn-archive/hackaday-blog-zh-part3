# Hackaday 奖参赛作品:IO，纸板电脑

> 原文:[https://hack aday . com/2017/09/10/hack aday-prize-entry-io-the cardboard-computer/](https://hackaday.com/2017/09/10/hackaday-prize-entry-io-the-cardboard-computer/)

[蟑螂博士]的目标是用回收和再利用的垃圾建造一台[四位计算机。由此产生的计算机称为 IO，由一个 555、大约 230 个 PNP 和 NPN 晶体管、230 个二极管和 460 个电阻组成。它采用 RISC 架构，运行速度约为 3 Hz。](https://hackaday.io/project/19048-the-cardboard-computer-io-is-my-name)

他用纸板制作木卫一有一个很好的理由:他没有这个项目的大预算，而且可以从他的工作场所免费获得材料。因为它建造得非常便宜，他也可以建造得非常大，让他能够真正看到每个电路关闭，并在它工作不正常时修复它。当它这么大的时候，你真的可以很好地看到建筑——没有[蟑螂博士]的电线缠结。他使用了 60 多个蓝色发光二极管来帮助监控系统，而且它们看起来也很酷也没有坏处。该项目的一个我们最喜欢的部分是他如何使用铜紧固件来管理纸板和充当接线点。
