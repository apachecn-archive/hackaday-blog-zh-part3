# Cameo Cutter 制作 SMD 模板

> 原文：<https://hackaday.com/2017/05/15/cameo-cutter-makes-smd-stencils/>

你永远不知道你会在工艺品商店里找到什么。相对较新的手工艺是自动切割机，如 Cricut 和 Cameo 切割机。这些通常用于剪贴剪贴，虽然他们会削减或雕刻几乎任何薄的形状。你可以把它们想象成一台用刀片代替打印头的打印机。[mikeselectrictuff]决定尝试使用浮雕切割机来生产 SMD 模板。结果，正如你在下面的视频中看到的[，相当令人印象深刻。](https://www.youtube.com/watch?v=FrxlQ2l_AHM)

如果你曾经想用回流焊炉做 SMD 焊接，模板对于把焊膏快速放到你想要的地方是非常宝贵的。[Mike]的电路板包含一船(超过 2000 个)led，用注射器将焊料滴在每个焊盘上会非常耗时(尽管他确实用注射器进行了一些润色)。

他用的板没有任何非常精细的部分。然而，他做了一些测试模式，并决定他可以得到更好的音高，特别是稍微调整一下。然而，他使用的模板不需要任何改变。他所做的只是将焊膏层导出为 DXF，并将其直接导入 Cameo 软件。

这不是我们第一次看到这种刀具被压进[模板服务](https://hackaday.com/2012/12/27/diy-smd-stencils-made-with-a-craft-cutter/)。你也可以利用你的 [3D 打印机](https://hackaday.com/2016/02/08/solder-stencils-with-a-3d-printer/)。
