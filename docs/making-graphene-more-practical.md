# 让石墨烯更加实用

> 原文：<https://hackaday.com/2016/07/20/making-graphene-more-practical/>

莱斯大学的詹姆斯·图尔(James Tour)和其他人宣布了使用纳米铆钉的石墨烯的改进形式。这种材料将碳纳米管和包裹铁纳米颗粒的碳球结合在一起。纳米管提供了强度和更高的整体导电性，而球体让材料更容易转移。

通常，在转移到另一个位置之前，将石墨烯放置在某物上涉及到在聚合物层上使用化学气相沉积。这种聚合物会降低石墨烯的性能。这种新材料不需要这个中间步骤。此外，球体允许更容易地连接到石墨烯。

这种新材料几乎和石墨烯一样透明，即使在弯曲时也能保持其强度和电性能。研究人员希望这种新材料将加速石墨烯在生产设备中的应用。该材料的生产在化学气相沉积炉中进行。

[制作普通石墨烯](http://hackaday.com/2013/04/09/synthesizing-graphene-in-your-basement-laboratory/)是一般黑客力所能及的。甚至还有一种使用 [DVD 刻录机](http://hackaday.com/2012/12/21/making-graphene-with-a-dvd-burner/)的方法。可能有大量石墨烯的发现等待着人们去发现它们。甚至[最初发现石墨烯的](http://www.graphene.manchester.ac.uk/explore/the-story-of-graphene/discovery-at-manchester/)也没有涉及太多花哨的设备(见下面的视频)。
