# Hackaday 奖参赛作品:一台廉价的机器人显微镜

> 原文:[https://hack aday . com/2016/07/26/hack aday-prize-entry-a-cheap-robotic-microscope/](https://hackaday.com/2016/07/26/hackaday-prize-entry-a-cheap-robotic-microscope/)

显微镜是生物科学最有用的仪器之一，但是它们很贵。幸运的是，中国的一家工厂可以生产网络摄像头和塑料镜头，并以低价出售。这就是 Flypi 背后的想法——这是一种用于科学实验和诊断的廉价显微镜，基于一直很受欢迎的 Raspberry Pi。

Flypi 被设计成一个简单的科学工具和教育设备。随之而来的挑战是价格低廉且功能强大。它基于 Raspberry Pi 和 Pi 相机，以及用于拍摄快照、录制电影和控制几个不同模块的相关软件，这些模块扩展了这台机器的功能。这些模块包括加热或冷却样品的珀耳帖元件、温度传感器、RGB LED、LED 环、LED 矩阵和用于激活样品中荧光分子的特殊蓝色 LED。

Flypi 背后的智囊安德烈·查加斯(Andre Chagas)将 Flypi 设计得很便宜。他当然用一个大部分是 3D 打印的框架和一些出奇便宜的电子设备做到了这一点。Flypi 已经在做真正的科学研究，包括追踪在培养皿中游荡的细菌和斑马鱼心脏的荧光显微术。对于一个相对简单的工具来说还不错，而且是 Hackaday 奖的一个很好的参赛作品。
