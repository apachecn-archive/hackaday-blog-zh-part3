# 自动化 RC 电机效率测试

> 原文：<https://hackaday.com/2016/04/04/automating-rc-motor-efficiency-testing/>

小型无刷电机和脂肪电池是近年来最令人印象深刻的技术之一。就在几年前，遥控飞机要么由贫血的有刷电机驱动，要么由气体驱动。四轴飞行器很少见。现在，有了无刷电机，飞行变得前所未有的容易，建造电动长板变得简单，电动自行车也很常见。

当然，如果你打算用无刷电机让任何东西飞起来，你可能会想知道你的电机和道具设置的效率。这就是[Michal]的[自动化 RC 电机效率测试仪](https://hackaday.io/project/10507-automated-rc-motor-efficiency-tester)背后的想法，他是 2016 年 Hackaday 奖的参赛作品。

[Michal]的项目不是一个测力计，如果你测量一个马达的扭矩或功率，你应该使用的设备。无论如何，如果你正在测试无刷电机和适当的配置，这并不是你真正想要的；同样大小的道具可以有非常不同的推力剖面。[Michal]并没有为无刷马达制造测功器，而是简单地测试马达和螺旋桨组合的推力。

该设备与 Hobby King 出售的设备[非常相似，包括电机支架、微控制器和显示器，以及力传感器，以绘制电机和螺旋桨产生的推力。数据可以保存到 SD 卡，该设备可以连接到计算机上自动生成漂亮的图形。](http://www.hobbyking.com/hobbyking/store/__66623__Turnigy_Thrust_Stand.html)

从遥控飞机和四轴飞行器到机器人和个人运输设备，无刷电机在各方面都有广泛的用途。这些电机的数据手册通常不多，因此任何能够测试这些电机的器件都会非常有用。
