# 与 ESP8266 一起使用

> 原文：<https://hackaday.com/2017/06/17/bbsing-with-the-esp8266/>

调制解调器出现的时间比网络还要长，在我们有脸书之前，我们有 BBS。有点令人惊讶的是，人们仍然在托管 BBS，但这些天很高兴找到一个固定电话。[Blake Patterson]是 retocomputers 的主要爱好者之一，最近他主动评论了一款有趣的新设备。这是 [WiFi232 互联网调制解调器](http://www.bytecellar.com/2017/05/30/the-wonderful-wifi232-bbsing-has-literally-never-been-easier/)，一种将 WiFi 连接转换为带有 25 针 RS-232 连接器的计算机可以理解的设备。

WiFi232 是由[Paul Rickards] 制造的[，考虑到过去几年支持 WiFi 的逆向计算项目，这正是你所期望的。WiFi232 的板载 ESP8266 模块模拟 Hayes AT 命令集。支持从 300 到 115200 的波特率，通过 USB 迷你插孔或焊接端子供电。](http://biosrhythm.com/?page_id=1453)

[【布莱克】的电脑窝是传奇](http://www.bytecellar.com/photo_pano.html)的素材，因此他有足够多的玩具来测试这种通用 WiFi 到串行转换器。测试中使用的设备包括 Apple //c、IIe、Amiga 1000 和 TI-99/4A。简而言之，一切都正常运行。[布莱克]能够调出现存的他收集的古代电脑的公告栏。你可以看看下面[Blake]对 WiFi232 的评论
