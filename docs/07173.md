# 深思熟虑的各种项目和失败

> 原文：<https://hackaday.com/2017/09/24/a-thoughtful-variety-of-projects-and-failures/>

我们在[思想商场]的朋友给我们带来了令人愉快的项目，但并不是所有的项目都需要一个完整的视频。有一群小而有价值的项目的人会怎么做？他们把它们放在一起，就像许多不匹配的乐高积木。 [Grab Bag #1](http://youtube.com/watch?v=AX2QmnlpPJ8) 是一个半月视频系列的开始，该系列展示了在【The think Emporium】通常的视频演示的幕后发生的较小的项目。

日食？有两个原因，因为第一个原因足以吊起(思想商场)的胃口。离子升降机？了解商店周围最受欢迎的变压器，看看高压线靠得太近会发生什么。茶叶激光？用那个变压器用你房子周围的东西做一个合法的激光器。铋铸？宠物用品店可能有你需要的东西来提升你的角色扮演，但这完全是骗人的。失败？我们也找到他们了。

不久前，我们首次报道了 ionocraft(升降机)。[茶激光](http://hackaday.com/2015/07/08/legit-hack-creates-tea-laser-power-by-mr-wimshurst/)之前已经覆盖。[选角](http://hackaday.com/?s=casting)对 hackaday 来说并不陌生，但【The Thought Emporium】用他们的技术打破了传统。
