# 彩色解 2D 数学方程

> 原文:[https://hack aday . com/2018/03/26/solve-2d-math-equations-colorly/](https://hackaday.com/2018/03/26/solve-2d-math-equations-colorfully/)

电子学可以被看作是物理学的一个应用，你也可以反过来认为物理学是数学在现实世界中的应用。不幸的是，我们大多数人学习数学的方式远非直观。幸运的是，互联网上充满了令人惊叹的文本和视频，可以帮助你更好地理解复杂数学主题背后的“为什么”。恰当的例子？[3Blue1Brown]有一段视频展示了如何使用颜色来[解 2D 方程。如果你看得足够多，你会意识到颜色只是一种表示向量的巧妙方法，事实上，这种方法适用于复数。](https://www.youtube.com/watch?v=b7FxPsqfkOY)

老实说，我们不认为你会用手解这样的方程——至少不会用颜色。但是这个视频给你的直观感受是非常有价值的。此外，如果你试图在软件中实现一个算法，这将是为它量身定制的，尽管我们认为你也不会真的在那里使用颜色。

视频有点长，但充满了有趣的见解。除了颜色方法之外，他还展示了几种表示函数的方法。仅仅看动画就能带来一些“啊哈”时刻。如果你本质上是一名程序员，你无疑也会了解算法的二分搜索法方面。

如果你想复习一下复数的内容，我们已经做了。如果你对[软件定义无线电](https://hackaday.com/2015/10/10/sdr-tutorials-from-michael-ossmann/)感兴趣，别忘了这也适用于 I 和 Q 信号。
