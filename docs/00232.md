# 只是不要叫它旧遥控器

> 原文：<https://hackaday.com/2015/10/04/just-dont-call-it-an-old-remote/>

[哈里·威古纳的]父亲已经 90 岁了。他开始难以按下电视遥控器上的按钮，所以[Hari] [决定制作一个定制的遥控器，上面只有他父亲需要的按钮](https://hackaday.io/project/7057-self-learning-tv-remote-for-the-elderly)。哦，按钮很大。

这个项目有一些有趣的地方。[Hari]希望最大限度地延长电池寿命，所以他花了很大力气让处理器保持休眠状态，并最大限度地降低功耗。遥控器是可编程的，但[Hari]无法使用他父亲的遥控器。他的回答很优雅。他用他的 Android 手机模仿所需的遥控器，并为遥控器提供了一种从另一个遥控器(在这种情况下，是手机)学习的方法。
 【Hari】制作了一系列视频，涵盖了从试验板到带有激光切割覆盖层的好看塑料外壳的项目。这是一个经过深思熟虑并记录在案的 Arduino 项目，也是一个很好的模型，可以帮助有特殊需求的人更好地享受生活。

【哈里的】代码[在 Github](https://github.com/hwiguna/g33k/tree/master/ArduinoProjects/2015/_Done/116-Custom_TV_Remote) 上有。我们确信他的父亲会对这个结果感到高兴。有时候更容易想到我们想要什么(比如一个[又酷又复杂的触摸屏遥控器](http://hackaday.com/2014/04/14/a-7-touchscreen-tv-remote-control-from-scratch/)，而不是最终用户会喜欢什么，但是我们认为【哈里】做到了。当然，在过去，[你的遥控器只有七个按钮](http://hackaday.com/2012/05/25/retrotechtacular-7-function-remote-control-color-television/)。
