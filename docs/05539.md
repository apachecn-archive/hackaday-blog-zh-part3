# Hackaday 奖参赛作品:利用人工智能检测掺假食品

> 原文:[https://hack aday . com/2017/04/10/hack aday-prize-entry-detecting-掺假-food-use-ai/](https://hackaday.com/2017/04/10/hackaday-prize-entry-detecting-adulterated-food-using-ai/)

掺假食品是指添加了某种物质以节省制造成本的食品。它可能会有负面影响，可能会降低食物的效力，也可能完全没有影响。在许多情况下，这是非法的。这也是一个普遍存在的问题，[G. Vignesh]决定将这个问题作为他 2017 年 Hackaday 奖的参赛作品，[一种基于人工智能的掺假检测器](https://hackaday.io/project/20565-ai-based-adulteration-detector)。

在他的 hackaday.io 项目详情页面上，他概述了一些现有的食品测试方法，其中一些你可以在家里做:掺假的糖可能添加了白垩，所以把它放在水中，糖会溶解，而白垩不会。他的方法是拍摄食物的高清照片，并在树莓 Pi 上应用滤镜来显示各种属性，如密度、大小、颜色、质地等。他还提到使用深度学习神经网络进行图像分析。这个项目触动了我们所有人，我们将饶有兴趣地关注它。

如果所有这些关于掺假食品的谈论让你对你的食品供应感到紧张，那么考虑种植我们自己的，黑客风格的。我们在 Hackaday 上看到的一个这样的项目是 [Farmbot，一个开源的数控农业机器人](http://hackaday.com/2016/08/19/add-robotic-farming-to-your-backyard-with-farmbot-genesis/)。另一个这样的例子是[麻省理工学院的 OpenAg 食品计算机](http://hackaday.com/2016/12/26/openag-is-a-personal-food-computer/)，一个机器人控制和监控生长室。
