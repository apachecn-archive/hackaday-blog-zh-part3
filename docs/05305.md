# q 对娜奥米·吴一无所知

> 原文:[https://hackaday.com/2017/03/12/q-has-nothing-on-naomi-wu/](https://hackaday.com/2017/03/12/q-has-nothing-on-naomi-wu/)

与其说我们是詹姆斯·邦德的粉丝，不如说我们是为他提供如此美妙东西的黑客 Q 的粉丝。Q 的皇冠有了一个挑战者，[Naomi Wu]——代号[SexyCyborg]——制造了一个名为 [Pi 调色板的史诗级小工具，它将一台 Linux 笔记本电脑藏在一个化妆品盒](https://hackaday.io/project/18999)内。

你可以看到下面 Pi 调板的隐蔽模式。它类似于一个蛤壳式化妆盒，底部有化妆品和涂抹器，翻盖底部有一面镜子。镜子隐藏了一个纵向的 LCD 屏幕，以及一个运行 Kali Linux 的 Raspberry Pi 3。

外壳的底部包括一个位于无线键盘/触摸板下方的便携式电池，当化妆品托盘被移除时，这两个部分都会显露出来。感应充电器连接到电池上，[Naomi]建立了一个基站，Pi 调色板位于其中，用于无线充电。

她设想这是一次秘密渗透测试。为此，Pi 调色板需要能够将 WiFi 加密狗置于混杂模式。她接了一个双拨码开关包，并真的做了额外的工作，把它设计到箱子里。开关安装和完成只是一个微小的细节，说明了整个项目的关注。有了这么漂亮的期末项目，难怪她会上街炫耀。休息之后，请观看视频，了解构建过程。

 [![pi-palette-fit-and-finish-dip-switch](../Images/ba6e871ff94dd3a2307b2b195253fe00.png "pi-palette-fit-and-finish-dip-switch")](https://hackaday.com/2017/03/12/q-has-nothing-on-naomi-wu/pi-palette-fit-and-finish-dip-switch/)  [![pi-palette](../Images/0e7c4837f111a31b31bb0c98e273ada2.png "pi-palette")](https://hackaday.com/2017/03/12/q-has-nothing-on-naomi-wu/pi-palette/) 

我们在 12 月的一篇黑客链接文章中报道过这个问题。昨晚当 Pi 调色板没有一个独立的功能时，[内奥米]看到我们对[丑陋的 Pi 笔记本电脑](http://hackaday.com/2017/03/11/an-ugly-but-functional-pi-laptop/)的报道有点困惑。我们认为她是对的，这是一个绝妙的概念，她抓住了捏造的机会，所以我们很高兴现在花时间来庆祝她的工作。干得好！
