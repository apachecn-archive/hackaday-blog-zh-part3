# 绕道:射频和胶片

> 原文:[https://hackaday.com/2018/04/04/detoured-rf-and-film/](https://hackaday.com/2018/04/04/detoured-rf-and-film/)

捕捉空中视频背后的技术是如此的酷。你不仅有像汽车一样昂贵的照相机，你还有安装在直升机上的遥控照相机和连接地面的惊人的微波链路。这不是你在四轴飞行器上典型的 FPV 设置；这里有喷气涡轮机和 S 波段发射机。

这些令人惊叹的空中镜头背后的一个人是 [RF 电影](https://www.rffilm.com)的【格雷格·约翰逊】在阳光明媚的南加州的怀特曼机场。他为《黑豹》、几部《速度与激情》电影、漫威电影工作过，并且正在制作新的《壮志凌云》。作为供应框架设计实验室*迂回*系列的一部分，首席设计师【Majenta Strongheart】参观了 RF Film 到[看看从空中视频平台](https://www.youtube.com/watch?v=R2T1ZgGsVJg)获得高质量图像的过程。

[Greg]在密执安州的一家直升机公司起步，最终为 ESPN 工作，拍摄海上快艇比赛的视频。除了空运之外，没有其他方法可以做到这一点，但是有一些问题。你需要以某种方式将视频传输到地面，这意味着微波发射器。[Greg]通过简单地做，展示一些黑客信誉，学会了所有这些。事实上，他建造的第一批遥控系统中有一些是围绕喷水灭火系统建造的。

现在，[格雷格]的工作比改良的喷水灭火系统要先进得多。他正在为 1080 台戴在帽子里的赛车相机定制硬件。有围绕轴转动相机的万向节，所有这些都是在[格雷格]的店里定制的。

为航空摄影制作视频需要做很多工作，这比在四轴飞行器上安装一个摄像头要复杂得多。有天线设计，联邦通信委员会的规则，和定制的机器人，指出相机需要的地方。RF Films 是业内最好的公司之一，我们感谢他们让我们得以一窥他们的商店和工作流程。
