# 正好赶上圣诞节！DIY 桌面 LED 圣诞树

> 原文:[https://hackaday . com/2016/10/30/just-in-time-for-Christmas-a-DIY-desktop-led-tree/](https://hackaday.com/2016/10/30/just-in-time-for-christmas-a-diy-desktop-led-tree/)

好吧，我们甚至还没到万圣节，但如果你正在计划某种假日项目，现在是开始订购零件的好时机，尤其是如果你正在设计自己的 PCB。虽然不涉及 PCB，但[designer2k2]使用一些由微控制器控制并由 USB 供电的 WS2812 RGB LEDs 制作了一个[桌面“空心”圣诞树](https://www.youtube.com/watch?v=Fm5TMvk3bUo)。

运行[designer2k2]项目的板是 Digispark，Digistump 的 USB 供电板，包含一个 ATtiny85。led，四个不同大小的 NeoPixel 环，加上顶部的一个像素，使用一些固体线连接在一起，这使得外观非常酷。运行在 ATtiny 上的代码是真正构建这棵树的部分。代码循环显示颜色和一些光线追逐效果，以及一个显示带有白光的绿色树的模式。当你从上到下看这棵树时，整个项目以拼写“XMAS”的例行程序结束。

这些年来，我们已经看到了一些由各种东西控制的[其他](http://hackaday.com/2014/11/09/deck-the-halls-with-a-raspberry-pi-controlled-christmas-tree/)T2 圣诞节[圣诞树](http://hackaday.com/2015/12/26/a-nerdy-xmas-tree/)T6】黑客，但这一个是一个相当简单、酷的设计。[Designer2k2]也发布了圣诞树的代码，我相信我们中的很多人可以想出一些更轻巧的设计。
