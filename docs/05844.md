# 特设的 MIDI 到音乐盒项目显示了黑客社区的力量

> 原文:[https://hack aday . com/2017/05/06/ad-hoc-midi-to-music-box-project-shows-power-of-hacker-community/](https://hackaday.com/2017/05/06/ad-hoc-midi-to-music-box-project-shows-power-of-hacker-community/)

公平的警告:当你发布一个视频，展示你做了一个令人难以置信的繁琐过程，比如手动在纸带上打孔，将 MIDI 文件传输到音乐盒，当[一群黑客在不到一周的时间内自动完成这个过程](https://www.youtube.com/watch?v=XyVyA5ZdWOY)时，不要感到惊讶。

即使是不经意的黑客读者也应该熟悉这个背景故事。[Martin]来自瑞典“Wintergatan”集团，是一家多产的特殊乐器制造商。毫无疑问，你会想起他华丽的大理石音乐机器 T1，这是 T2 的第二个版本，T3 正在制作中。但他也对由纸带编程的[音乐盒情有独钟，最近他发布了一段视频](http://hackaday.com/2016/07/06/paper-tape-drive-for-a-live-performance-music-box/)[，展示了他耗时且完全手动的在纸带上打孔的过程](https://www.youtube.com/watch?v=HjBhO9iqEc0)。由于他的原始素材已经在 MIDI 文件中，他的一群粉丝独立地想出了自动处理的方法。

下面的视频展示了他从他的粉丝那里学到的关于自动化编程的知识，也展示了他从我们工作和玩耍的社区中学到的知识。在没有明确寻求帮助的情况下，由共同兴趣聚集在一起的随机陌生人发现问题，提出解决方案，整理好的和坏的想法，然后[将工作公之于众](https://github.com/Wintergatan/Midi-to-laser-cutter)。对于不到一周的工作来说还不错。
