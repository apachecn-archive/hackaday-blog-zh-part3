# 单板中继计算机

> 原文：<https://hackaday.com/2017/09/18/single-board-relay-computer/>

我们都知道你可以用继电器来建造一台计算机，如果你是 Hackaday 的忠实读者，你可能已经看过一些了。实际上，设计和制造一台基于继电器的计算机完全是另一回事，这一成就将使你与硬件大师齐名。

DIY 微机名人堂的最新入选者是【Jhallen】。他用继电器制造了一台微型电脑“训练器”。它的咔嗒声比梃杆系列还要多，是 DPDT 接力赛中的一件艺术品。

设计中继计算机最大的考虑因素是内存。您可以在几十个继电器中实现一个 CPU，但即使是少量的内存也仍然是数百个附加组件。在这台电脑里，[jhalen]有点像作弊。存储器在微控制器上实现为 256 个 32 位字，旁边是用于前面板的控制器。CPU 仍然是所有继电器，支持自修改代码，一堆条件跳转的指令，以及一个“增量和跳转，如果不等于零”的指令。

下面，你可以查看一个关于中继计算机运行的*非常*深入的视频，从一些令人满意的[欧几里德算法](https://en.wikipedia.org/wiki/Euclidean_algorithm)的咔嗒声和可变时钟速率的演示开始。视频继续演示了中继计算机本身的汇编语言和一点整体架构。这是我们见过的最有教育意义的老式计算演示视频之一。

[jhalen]组装了一些这样的板，他正在出售一些额外的板。如果你有 600 美元，[你可以在 Tindie](https://www.tindie.com/products/jhallen/single-board-relay-computer/) (标准黑客日/ Tindie 披露声明)上拿一个。考虑到组装这块板所需的焊接量，我们猜测这是一个*非常*合理的价格。
