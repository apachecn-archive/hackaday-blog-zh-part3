# 自动气动口琴

> 原文:[https://hack aday . com/2016/04/06/自动-气动-口琴/](https://hackaday.com/2016/04/06/automatic-pneumatic-harmonica/)

一位智者曾经说过“如果你只有一个可爱的台式压缩机和一些电磁阀，一切看起来就像一个机器人口琴。”或者我们在转述。不管怎样，[杨奇煜-舒托] [造了一个气动的自动口琴音乐机](https://hackaday.io/project/10732-harmonica-music-machine)。

这实际上是他另一个项目的分支，一个高速糖果分拣机。在那里，他试图超越更常见的颜色传感器和伺服风格的装置，使用计算机视觉进行颜色检测，并使用一些压缩空气喷嘴将糖果从传送带上吹到适当的垃圾箱中。

但在此之前，电脑控制的气动装置的测试是有序的。这就是口琴进入画面的地方。“建造”实际上只是他把糖果分拣机里的管子放进口琴里。有一个 ARM 处理器旋转它的高功率轮子，和一些 MOSFET 驱动器来打开和关闭阀门。但仅此而已。看它在下面的视频中播放一首“曲子”。
