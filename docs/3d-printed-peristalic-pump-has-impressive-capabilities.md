# 3D 打印的蠕动泵具有令人印象深刻的功能

> 原文：<https://hackaday.com/2015/11/10/3d-printed-peristalic-pump-has-impressive-capabilities/>

[弗兰克·赵的]爷爷得了食道癌。对他来说不幸的是，这意味着他要通过一根管子吃一段时间了。这包括有人用一个大注射器帮助他将粘稠的食物液体推过管子。[弗兰克]知道一定有更好的方法。当[Frank]在中国的医院访问时，他开始[设计 3D 打印蠕动泵](http://www.bocabearings.com/innovation-contest/ContestantDetails.aspx?ProjectID=221)。这就是你所期待的:一个按摩塑料管环的机制，将内容物进一步推向路径。

回到美国后，他进一步完善了自己的设计，并开始 3D 打印。事实证明——它非常有效。在下面的视频中，他展示了如何抽取 mayonaise——因为它是蠕动的，所以不需要启动泵！
