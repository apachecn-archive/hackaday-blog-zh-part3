# 尽可能小尺寸的视频游戏

> 原文:[https://hack aday . com/2016/07/17/尽可能少维度的电子游戏/](https://hackaday.com/2016/07/17/video-games-in-as-few-dimensions-as-possible/)

先是文字游戏，然后是 2D 地牢。当 Wolfenstein 3D 在游戏领域爆发时，它引起了相当大的轰动。但是如果你只有一条 WS2812 LEDs，那就太多了。

[treibair]已经在 Hackaday.io 上启动了一个项目，开始开发可以在单个 LED 条上玩的 1D 视频游戏。虽然最终应用是一个很酷的物理界面，可能由微控制器驱动，但在大型计算机上构建原型要容易得多。不过，他是在处理过程中编写的，所以最终向 Arduino 的过渡会更容易。

看看下面的视频。

在 1D 还有一些其他的游戏，包括 [Line Wobbler](https://www.youtube.com/watch?v=UZ_5ol_kyL4) (YouTube)，当然还有 [Wolfenstein 1D](http://www.wonder-tonic.com/wolf1d/) 。几个月前，我们甚至在慕尼黑电影节上看到了一维的“蛇”复制品。(请作者站起来好吗？)

我们认为这是一个好主意，将每个人的 1D 游戏体验集中在一个地方会有很大的帮助。所以在评论中把代码和评论联系起来吧！
