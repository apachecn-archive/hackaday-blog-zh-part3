# 艺术窃听生活和寻呼机

> 原文：<https://hackaday.com/2017/12/24/art-eavesdrops-on-life-and-pagers/>

在手机出现之前，寻呼机是移动通信的方式。起初，它们几乎是一种身份的象征。最终，它们成了那些不能或不愿携带手机的人的标志。然而，很明显，仍然有一些用户紧紧地抓住他们的寻呼机，包括医疗专业人员。在一个名为 HolyPager 的艺术项目中，[Brannon Dorsey]截获了一个城市中所有的寻呼机消息，并在几台老式的滚筒打印机上打印出来。结果有点令人惊讶。你可以看看下面的视频。

几乎所有的网页都是医学方面的，其中很多都有敏感信息。从技术的角度来看，[Brannon 的]页面并没有透露太多信息，但是一篇关于项目的文章说，它和其他展示隐藏世界或无线电波的艺术项目正在使用我们的老朋友 RTL-SDR 加密狗。

寻呼机使用一种协议——POCSAG——这种协议早于我们对隐私和安全的现代(且有充分根据的)痴迷。这并不令人惊讶，尽管私人医疗数据像这样在空中飞行的想法令人惊讶。解码 POCSAG 并不难。例如， [GNU Radio](https://hackaday.com/2015/11/11/getting-started-with-gnu-radio/) 可以轻松处理任务。

我们过去已经研究过[寻呼机黑客攻击](https://hackaday.com/2017/08/12/flex-pager-protocol-in-depth/)。你甚至可以经营你自己的[传呼机网络](https://hackaday.com/2014/12/30/bringing-a-legacy-pager-network-back-to-life/)，但是如果你被罚款，不要责怪我们。
