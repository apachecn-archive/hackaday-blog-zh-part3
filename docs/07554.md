# 仿生眼试验获得批准

> 原文:[https://hackaday.com/2017/11/07/bionic-eye-trial-approved/](https://hackaday.com/2017/11/07/bionic-eye-trial-approved/)

法国公司 Pixium Vision 已经获得批准，开始进行微型无线视网膜下植入物的人体试验。该设备名为 PRIMA，可以帮助晚期干性年龄相关性黄斑变性患者改善视力。该公司正在谈判也在美国进行试验。

PRIMA 植入物是一个约 2mm 见方、仅 30 微米厚的光伏芯片。这很小，但是这个设备有 378 个电极。患者使用的设备看起来像一副传统的眼镜，但包含一个集成的摄像头，可以将数据无线发送到一个小型的口袋大小的图像处理计算机。然后，这台计算机命令眼镜通过不可见的红外光向植入物发送数据。芯片将光转化为电脉冲，并将其传导至视神经。你可以在下面看到一个关于这个系统如何工作的视频。

据该公司称，PRIMA 已成功用于老鼠和猫。他们计划最终生产带有几千个电极的芯片。

还有其他类似的系统，包括美国食品和药物管理局批准的第二视觉设备，该设备使用电极网格和微型接收器。恢复的视力很少，微型接收器容易受到手机和电器的干扰。其他公司使用类似于相机或纳米线植入物中的光传感器。

我们以前见过黄斑变性，但是我们大多数人都没有足够的勇气去做真正的植入手术。虽然 PRIMA 眼镜不是很时尚，[他们击败了 DLP 投影仪](https://hackaday.com/2016/03/21/correcting-color-blindness-with-dlp-projectors/)。
