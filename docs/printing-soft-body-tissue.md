# 印刷柔软的身体组织

> 原文：<https://hackaday.com/2015/10/28/printing-soft-body-tissue/>

如果你和我们一样，你会倾向于用塑料或者金属进行 3D 打印——如果你足够幸运，可以使用昂贵的打印机。卡内基梅隆大学的亚当·范伯格和他的团队用肉体打印。嗯，算是吧。打印生物材料是一个新兴的研究领域。然而，打印像软组织一样的材料具有挑战性。在最近的一篇论文中，范伯格和他的公司概述了一种叫做 FRESH 的方法。FRESH 使用改良的 MakerBot 或 Printrbot Jr .打印机将水凝胶沉积到明胶浆支持浴中。明胶会保持物体的形状，直到打印完成，打印完成后，明胶可以通过加热去除。如果你不想在实际的论文中费力地通过术语，期刊*科学* [有一个很好的概述](http://news.sciencemag.org/technology/2015/10/3d-printing-soft-body-parts-hard-problem-just-got-easier)(见下面他们的视频)。

明胶与氯化钙混合，并在低温下胶凝 12 小时。然后用现成的消费级搅拌机把它变成浆液。使用离心机除去大部分可溶性明胶。印刷油墨是由胶原蛋白和纤维蛋白等材料制成的。这种新方法实际上使用了液体墨水，这种墨水在明胶中胶凝。

打印机使用的是在 NIH 3D 打印交换网站上找到的开源注射器挤出机(他们从来没有说到底是哪一个，尽管我们很难从图片中找到它)。在真正的黑客时尚中，打印机使用 ABS 和 PLA 塑料的库存打印自己的注射器挤出机。然后，您只需用新打印的挤出机替换标准挤出机(重复使用库存步进电机)。

该论文描述了打印项目，包括一个 5 天大的鸡胚心脏模型、一条动脉和一个微型人脑模型。佛罗里达的另一组研究人员也有类似的系统。

我们之前已经谈过生物打印，甚至提到如何制作自己的基于喷墨的生物打印机。这种新鲜的方法看起来像是黑客的 3D 打印工作室可以达到的。当你终于可以打印身体部位时，我们不敢想象你会打印什么。
