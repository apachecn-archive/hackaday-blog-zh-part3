# 冰箱警报发声，省电省粮

> 原文：<https://hackaday.com/2017/11/04/fridge-alarm-speaks-and-saves-power-food/>

除了空调或加热器，我们家中最耗电的设备之一是冰箱和冰柜。如果门没有完全关闭或磁性密封件没有正确就位，情况尤其如此。[Javier]开始解决他的个人冰箱反复出现的问题，他在门上安装了一个警报器，以确保冰箱不会消耗超过绝对需要的电能。

该设备的核心非常简单。只有当门打开时，微型开关才给小型微控制器供电。如果门开得太久，微控制器就会开始动作。然后，该设备给一个小的无线卡(它看起来像一个非常详细记录的 ESP 模块的变体)供电，该卡与他的微波炉进行通信，这反过来以声音和语音警报提醒他冰箱没有完全关闭。它全部由电池供电，最终需要充电。

虽然肯定有更简单的方法来实现警报，但对于这个项目来说，使用语音警报是一种很好的方式，并且可以实现的电力节省也不小。还有一个额外的好处是，[哈维尔]可以防止他的冰箱结霜。如果你有心情寻找其他好的冰箱窍门，还有其他令人兴奋的、新颖的、独一无二的方法来装饰你的冰箱。
