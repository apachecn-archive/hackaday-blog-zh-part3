# FPGA 驱动旧笔记本电脑屏幕

> 原文：<https://hackaday.com/2016/07/27/fpga-drives-old-laptop-screen/>

每年都有新型号的笔记本电脑上市。这意味着旧笔记本电脑通常会被扔进垃圾填埋场，这并不理想。不过，如果你不想浪费一台旧的或过时的笔记本电脑，有一种方法至少可以重复使用其中的屏幕。[只需从货架上取下 FPGA，然后开始工作](http://www.martinhubacek.cz/fpga/lvds-lcd-hacking)。

[Martin]向我们展示了如何独自完成这一壮举，并详细介绍了所有相关电子设备的工作原理。一旦所有的东西都被拆开，FPGA 被连接起来，他就花了大量的时间来打开显示器。从那以后，一切都走下坡路了:[马丁]现在，在合理的范围内，任何图案都可以出现在屏幕上。他的显示器现在唯一的限制似乎是缺乏外部 RAM。他目前使用该设置来驱动一个令人印象深刻的时钟。

与过去几乎不可能改变笔记本电脑屏幕用途的时代相比，这是一大进步。最终有人发现了驱动这些显示器的方法，现在有来自中国的廉价电子产品通常可以让这样的屏幕运行。不过，看到它从零开始完成令人印象深刻，视频中的大量细节是理解一切如何工作的好方法。
