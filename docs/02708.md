# 真实世界的赛道是真正的黑客

> 原文：<https://hackaday.com/2016/06/22/real-world-race-track-is-real-hack/>

[Rulof]总是给我们留下深刻的印象，他想出了什么，他是如何一起破解的。说真的，他怎么知道他在这个项目中使用的不起眼的伞部件存在，更不用说在制作磁铁底座的时候想到它了？他这次的黑客是为他的弟弟制作的一个[真实世界的桌面赛道](https://www.youtube.com/watch?v=5YN6uIMq_cs)，据他自己说，他的弟弟为此而疯狂。

他的赛道是在一个旋转的桌子上，由以下部件组成:电机，自行车车轮，旅行包的脚轮，橡皮筋(他从哪里得到这么大的？)，他哥哥送的玩具车和方向盘，滑板轮子，前面提到的雨伞部分和硬盘磁铁。在下面的视频中，我们喜欢他用画笔固定轨道表面，让轨道在下面旋转的方式。

从视频中你可以看到赛道已经被钩住了。希望他也让他的兄弟有足够的机会，但我们不太确定。我们可以想象的一些增加将是障碍、照明、声音和一些模拟爆炸效果的机器人(面粉团？).
