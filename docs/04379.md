# 不要让 3D 打印机无人看管，它们可能会着火

> 原文：<https://hackaday.com/2016/12/07/dont-leave-3d-printers-unattended-they-can-catch-fire/>

假期快到了，随之而来的是传统的大众消费品消费和礼物交换。3D 打印机变得非常好，非常便宜，今年不可避免地会有大量 3D 打印机作为礼物赠送。如果你送或接受这些打印机，一定要小心:[它们可能会引起火灾](http://blog.lessdebug.com/2016/12/3d-printer-almost-fire.html)正如【Ben Hencke】在诊断他今年购买的一台打印机的问题时发现的那样。

问题中的打印机是垄断制造商 Select V2，这是一款 Prusa i3 的克隆产品，具有令人印象深刻的 300 美元打印机规格。这台打印机是更名的万豪复印机 i3，[我们对它的评价很好](http://hackaday.com/2016/09/14/wanhao-duplicator-i3-should-put-an-end-to-cheapest-printer-kickstarters/)。这是一台功能强大的打印机，在质量上胜过任何 Kickstarter 打印机(事实上，你现在就可以买到它)。我们很确定今年农神树下会有很多这样的打印机。

几周后，[本]注意到床预热时有一点烟从打印机里冒出来。这不是蓝色的烟雾，不像你在爆炸的电容器上看到的那样。有很多烟。

经过进一步的检查和来自 [embedded.fm](http://embedded.fm/) 的【Elecia White】的帮助，问题被追踪到加热床的电源连接器。这个连接器的绿色溴注入塑料被烧焦了，毫无疑问这可能会引起火灾。

3D 打印是一种神奇的工具，在过去几年里，它已经实现了比我们想象的更多的黑客和建筑。3D 打印机变得非常好，而且非常便宜，当然这最终意味着有人会因为打印机火灾而失去他们的工作室。直到有人发现如何建立一个“热保险丝”或类似的东西，3D 打印机——从高端打印机到仍然非常好的单价和万豪单位——都有可能引发一场火灾。
