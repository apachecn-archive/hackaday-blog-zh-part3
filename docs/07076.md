# 白板上的天线基础

> 原文：<https://hackaday.com/2017/09/11/antenna-basics-by-whiteboard/>

像许多人一样，[布鲁斯]喜欢无线电控制(遥控)车辆。事实上，许多人开始接触电子学是因为他们对 RC 感兴趣。也许这就是为什么[Bruce]做了一个关于天线基础的视频，他花了半个多小时讨论天线。你可以看下面的视频。

[Bruce]避免任何复杂的数学，更专注于对天线的直觉，这是我们喜欢的。天线切割到一定长度有什么关系？[Bruce]用一个秋千和一个老爷钟来解释。为什么有些天线有增益？为什么两极分化很重要？[Bruce]涵盖了所有这些以及更多内容。他甚至演示了一个简单的实验，你可以用一个仪表和一块磁铁来做。

如果你知道什么是史密斯图表，这可能不适合你。如果你不这样做，这个视频将不会涉及任何类似的内容。但是，如果你想更好地了解天线的功能和工作原理，花 30 分钟是很好的。

如果你想知道更多关于[布鲁斯]提到的八木的信息，我们之前已经讨论过了。我们还介绍了一个类似的[直观天线](https://hackaday.com/2016/11/27/start-your-path-to-becoming-an-antenna-guru/)教程，它使用了很多有趣的动画。
