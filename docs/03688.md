# Hackaday 奖参赛作品:高压平视

> 原文：<https://hackaday.com/2016/10/05/hackaday-prize-entry-head-up-for-high-voltage/>

[Alain Mauer]想用一个小的有机发光二极管屏幕建立一个类似谷歌眼镜的东西。0.96 英寸的显示器太大了，但 0.66 英寸的效果很好。结合一个 Arduino、一个蓝牙模块、电池和一些光学元件，他建造了一副能够显示万用表读数的眼镜。

你可能会认为完成这件事很简单，但正如[Alain]发现的那样，由于一些原因，事情并非如此。该设备的价格约为 70 欧元，你可以在下面看到测试结果的视频。

视频显示了一个常见问题及其解决方案。你正在探测一个电源电路，不得不移开视线去读取电压表。有了眼镜，你不用看别处，电压就在你的视野里浮动。

这些让我们想起了之前讨论过的 [Pedosaglass](https://hackaday.com/2015/11/10/the-simplest-smart-glasses-concept/) 。当然，它使用了不同的光学解决方案。我们也看到了[谷歌眼镜仿制品](https://hackaday.com/2014/07/28/a-better-google-glass-for-60-this-one-folds/)作为我们 Hackaday 奖参赛作品的一部分。
