# (机器人)手中的管子

> 原文：<https://hackaday.com/2016/03/25/pipe-in-robot-hand/>

你如何制造一只机器手？如果你是[Robimek]，你从一些[塑料螺旋管，一些伺服电机，一些钓鱼线](http://make.robimek.com/robot-handmade-with-plastic-pipe/)开始。哦，你还需要一只旧手套。

螺旋管道(或管，如果你喜欢)被切割成手状，并用粘合剂融合在一起。关节接头被切除，以允许油管在该点弯曲。钓鱼线将指尖连接到伺服电机。

该项目使用 Arduino 来驱动伺服系统，尽管你可以用任何微控制器来完成这项工作。卷起鱼线会使相关的手指收缩。卷出来让弹性塑料管拉回到原来的位置。手套覆盖了管道，增加了手部的真实感。

当然，作为展示品，这可能比作为工作用手更实用。我们想把它放在下一个万圣节项目中。我们以前见过一些简单的手工制作，但是手套手感很好。出于某种原因，我们的许多机器人手项目喜欢[做出粗鲁的手势](http://hackaday.com/2011/08/01/toy-robot-hand-vastly-improved-now-more-offensive/)。你可以在下面看到[Robimek]的手工作的视频。
