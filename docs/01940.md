# 3D 打印假产品泄露

> 原文:[https://hack aday . com/2016/04/04/3d-printing-a-fake-product-leak/](https://hackaday.com/2016/04/04/3d-printing-a-fake-product-leak/)

几天前，你可能已经看到了下一代任天堂游戏机手柄任天堂 NX 的网络上的[图片。有这么多照片，它必须是真实的！](http://www.neogaf.com/forum/showthread.php?t=1199996)

不对。这只是(弗兰克·桑德奎斯特)在互联网上和每个人捣乱。虽然虚假的产品泄漏通常只是 photoshopped 或 3D 模型渲染，[弗兰克]更进一步，实际上设计了一个完整的控制器，3D 打印它，并拍摄照片。你可能想知道他为什么要这么做。一位 Reddit 用户指出，这背后可能有一个很好的动机:

> 你也可以用它来找一份道具制作的工作。你制作了一个愚弄世界的道具，这在简历上看起来很棒

但是，据[弗兰克]说，这真的只是为了好玩。
