# 电话到电话的窃电

> 原文：<https://hackaday.com/2016/10/30/phone-to-phone-power-theivery/>

再一次，[Rulof]很好地利用了他可观的黑客能力，那是他很好的利用。通过改装一些简单的部件，他组装了一个可以挂在钥匙链上随身携带的东西，这个东西可以让他从朋友的手机里偷电给自己的手机充电。

他首先从一个 iPhone 风扇上切掉电机，以隔离微型 USB 连接器。然后，他从廉价的中国 USB 电源库中取出充电电路板，将微型 USB 连接器的电线焊接到电路板的一侧。最后，他从 Lightning-to-USB 电缆上切下 Lightning 连接器，并将连接器焊接到电路板的另一侧。对于长寿和化妆品，他都放在一个小木块里，连接一个钥匙圈。结果是一个小而整洁的盒子，一边是微型 USB 连接器，另一边是 Lightning 连接器。你可以看到他制作它，然后在休息后用它从他的朋友那里偷电。

只要他的朋友们不知道，他就再也不会没有电源了。但是如果他们真的明白了，他总是可以把 ATX 的电脑电源变成一个 8 USB 接口的充电站，通过给所有的手机充电来赢回他的朋友。如果因为你的城市被飓风袭击而没有电力供应呢？为此，一个非常有能力的黑客为[制造了一个手摇手机充电器](http://hackaday.com/2015/01/27/diy-phone-charger-born-from-cyclone-disaster/)。
