# Hackaday 奖品入口:一个电动长板

> 原文：<https://hackaday.com/2017/04/08/hackaday-prize-entry-an-electric-longboard/>

Hackaday 奖正在如火如荼地进行，这意味着我们开始看到一些精选的人在过去几个月里积攒的所有构建。[yowhwui]已经做了[一段时间的 3D 打印电动长板](https://hackaday.io/project/20462-3d-printed-electric-longboard)，这个构建真的很坚实。他的里程表已经超过了 150 公里，3D 打印的零件仍然保持着。

该电机的电源来自 6374 无刷电机，运行电压为 192 kV。这一点，再加上两个 4S 30C 5000mAh LiPo 电池，推动这个长板的速度高达 42 公里/小时(每刻钟 2.18 撒克逊联盟)，而重量约为 8 公斤。

由于[yowhwui]使用马达来提供动力*和*制动(电动马达很整洁)，所以这个长板需要在设计时考虑到跳带。为此，他设计了一个带惰轮的驱动系统，几乎每个零件都是 3D 打印的。硬件的第一次修订是在 PETG 印刷的。虽然 PETG 足够强大，但它也太脆弱了。[这导致了一些裂缝](https://hackaday.io/project/20462-3d-printed-electric-longboard)。在 ABS 中再次打印出零件后，【yowhwui】在这个长板上多放了几公里，并没有立即磨损的迹象。
