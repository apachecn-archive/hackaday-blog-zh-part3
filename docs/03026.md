# 乐高式化学和生物拼装玩具

> 原文:[https://hack aday . com/2016/08/17/Lego-like-chemistry-and-biology-erector-set/](https://hackaday.com/2016/08/17/lego-like-chemistry-and-biology-erector-set/)

加州大学河滨分校的一组研究人员和学生创造了一个类似乐高积木的系统，使用户能够定制化学和生物研究仪器。3D 打印模块系统可以创造各种科学工具。

这些被称为多流体进化成分(MECs)的模块出现在《PLOS 一号》杂志上。系统中的每个模块执行一项基本的实验室仪器任务(例如，泵送液体、进行测量或与用户交互)。由于这些模块旨在协同工作，用户可以快速有效地构建仪器，如用于制造替代燃料的生物反应器或高中化学课的酸碱滴定工具。这些块尤其适用于资源有限的环境，其中块库可以创建各种不同的研究和诊断工具。

该项目由研究生[道格拉斯·希尔]和助理教授[威廉·格罗夫]领导，这项工作由国家科学基金会资助。你可以在下面看到描述这些积木的视频。在 Hackaday，我们对生物黑客技术并不陌生。化学也出现在许多有趣的电子环境中，包括[太阳能电池研究](https://hackaday.com/2016/07/15/perovskite-solar-coming-soon/)。
