# 领航员无人机用它的螺旋桨来飞行和游泳

> 原文:[https://hack aday . com/2016/01/18/naviator-drone-uses-its-propeller-to-fly-and-swim/](https://hackaday.com/2016/01/18/naviator-drone-uses-its-propellers-to-fly-and-swim/)

罗格斯大学刚刚发布了一段关于“无人驾驶飞机”的视频，这种飞机可以飞行，然后降落到水体中，使用螺旋桨四处移动。这不是我们第一次覆盖一所大学，确保[天网即使在浴缸里也能找到我们](http://hackaday.com/2015/10/04/harvards-microrobotic-lab-sinks-robobees-and-claims-it-was-on-purpose/)，但这一次对家庭实验者来说更容易管理。机器人采用 Y8 电机[组合](https://hackaday.io/project/9066-multicopter-y6-drone-and-gopro-gimbal)。它的四个臂上的每个马达对都以相反的方向旋转，但提供相同方向的推力。通常这为无人机提供了更多的稳定性和更多的冗余。在这种情况下，我们认为它有助于机器人离开水，并在道具变得非常低效时提供更多的水下推力。

我们很期待这个方向的发展。我们已经可以想象出一个人可以用这个永远失去无人机和 GoPro 的新的有趣的方式，即使你的工具箱里有[这个积分](http://hackaday.com/2016/01/07/how-to-find-a-lost-drone-with-the-integral/)。我们还想看看无人机制造社区是否能找出这种无人机的新动力，并为不太喜欢数学的人发布一个库。休息后的视频。
