# 一顶 FRC 2018 的 Gif 播放礼帽！

> 原文：<https://hackaday.com/2018/04/03/a-gif-playing-top-hat-for-frc-2018/>

为了在 2018 年第一届机器人大赛上指导一支团队，redditor [dd0626]希望做一些与今年 8 位游戏主题产生共鸣的酷事情。在几天的时间里，他们将一顶礼帽变成了一个主题封装的跑马灯:一个装满 gif 的 LED 矩阵显示器！

这个展示实际上是一个袖子——由运输泡沫、枕头套和一件旧 t 恤制成——可以套在帽子上，使它保持完整，可以在未来的活动中穿着。Teensy3.6 在四个 WS2812 16×16 RGB LED 矩阵上显示 gif，尽管透明的黑色织物漫射了光线，但从几英尺外观看仍然是最好的。这绝对不是一个隐形的帽子展示。

为了减少电流消耗，[dd0626]使用 5V 30A DC/DC 转换器，并将亮度保持在最低水平，否则，每个面板的电流可高达 15A！为了抵消性能的任何下降，他们捆绑了一个巨大的 22，400 毫安时，24V 电池组，以保持帽子运行一段时间。尽管有所有的硬件，这顶帽子的重量不到两磅——非常适合长时间的比赛。
