# 用录音带制作合成器

> 原文:[https://hack aday . com/2017/07/12/making-synth-out-of-audio-cassettes/](https://hackaday.com/2017/07/12/making-synths-out-of-audio-cassettes/)

8 位混音带是简单的[基于 Arduino 的声音和节拍发生器](https://github.com/8BitMixtape)基于 ATtiny 84s 和 85s，设计为适合旧的盒式录音带，或者至少是那个大小。8bit Mixtapes 由[Dusjagr]、[Ucok]和[Lyok]创建，包括来自世界各地的参与者，是一种小型合成器，可以播放[单线算法交响乐](http://countercomplex.blogspot.com/2011/10/algorithmic-symphonies-from-one-line-of.html)，这是一种简单的声音发生器，只需一行代码即可工作。

该项目已经进行了多年，多年来发布了几个不同的版本——最近的一个是大约一个月前发布的 Mixtape NEO，它具有音频引导加载和一排 NeoPixel LEDs。它有很好的文档记录，完全开源，有代码库和 [wiki](http://wiki.sgmk-ssam.ch/wiki/8bit_Mix_Tape) 。附庸风雅的印刷电路板看起来也很棒！

8 位混音带是电子专业学生需要解决的自然项目。一个有两个锅和两个按钮的 ATtiny85？非常简单，音乐的回报使一天的研讨会变得轻而易举。代码的简单性也使得修改软件变得容易。

古怪的合成器是 Hackaday 的囊中之物，包括我们之前发表的一个合成器，它控制一个六边形的 led 矩阵。
