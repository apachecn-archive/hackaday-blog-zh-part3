# Pi 零动力滑板

> 原文：<https://hackaday.com/2016/11/06/pi-zero-powered-skateboard/>

乘坐你自己的自动个人交通工具在镇上呼啸而过是有好处的。如果是你自己建的就更好了。这正是[树莓派的家伙]所做的——建造了一个由 Wiimote 控制的树莓派零动力滑板,并在剑桥周围呼啸而过以炫耀它。

这是一个相当简单的构建——滑板、电池、电机和支架、控制器、Wiimote 和 Pi Zero。Raspberry Pi 控制电机控制器，进而控制电机速度。Raspberry Pi 的家伙]编写的 Python 代码有大约 100 行，管理电机控制器和与 Wiimote 的蓝牙连接，用于控制电路板的速度，同时用户控制转向。[树莓派的家伙]说他在滑板上已经达到了 30 公里/小时，如果有足够强大的马达和不颠簸的表面[就不难相信](http://hackaday.com/2016/03/10/electric-skateboard-reaches-neck-breaking-speeds/)。

在 Raspberry Pi 上运行一点 Python 来驱动一台电机似乎有点夸张([其他人](http://hackaday.com/2016/06/18/hacklet-112-skateboard-projects/)已经用更简单的方法完成了)，但这仍然是一个有趣的项目。Raspberry Pi 的家伙]描述了他从哪里得到零件来组装滑板，并在他的 GitHub 页面上发布了 Python 代码。
