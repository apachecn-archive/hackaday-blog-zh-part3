# 一边狂看网飞一边减肥

> 原文：<https://hackaday.com/2017/07/13/shed-pounds-and-inches-while-binge-watching-netflix/>

想打破你的流媒体诱导的植物人状态，但似乎无法打破疯狂观看的循环？也许你是这款控制你看多长时间网飞的健身车的候选人。

[Roboro]的反沙发土豆背后的概念很简单——只要继续踩踏板，你就能继续观看。细节也很简单，从 Arduino 监控来自健身自行车制造商精心提供的插孔的信号开始。方波的频率被转换成 PC 上的 Python 脚本通过 USB 读取的速度。一旦网飞流开始，低于用户定义的速度将暂停电影。下面的视频显示它正在做它的事情。

改进很容易浮现在脑海中，比如增加一个速度缓冲器，这样更快的踩踏板可以让你储存一些流动时间并获得休息。也许它可以以某种方式与这些网飞支持的袜子整合，或者甚至与网飞和冷却按钮整合。但这有点违背了初衷。
