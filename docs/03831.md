# 微波炉会把你变成蜘蛛侠

> 原文:[https://hack aday . com/2016/10/14/微波炉-把你变成蜘蛛侠/](https://hackaday.com/2016/10/14/microwave-ovens-turn-you-into-spiderman/)

想像蜘蛛侠一样爬墙？没问题——只要用微波炉变压器做一个攀爬架就可以了。找到一座钢铁建筑。重写正典，让彼得·帕克被电磁蜘蛛咬，而不是放射性蜘蛛。

回到基于现实的世界，如果你使用[让它变得极端]的装备到达地面以上十几英尺，你可能会拿自己的生命冒险。基本的东西听起来很不错，但是问题出在细节上。四个 mot 被切割并剥去次级线圈，连接到脚和手的固定装置上。一个装满凝胶电池的背包为装备提供动力，手柄上简单的常闭开关控制一侧的脚和手磁铁。按一下开关就会释放一侧的磁铁，让攀岩者能够向上攀爬。

这就是我们的安全牛肉:当你犯了一个错误，同时按下两个按钮，会发生什么？看起来这个版本迫切需要一些控制电路来防止这种最明显的故障模式。我们不会在每个问题上都扔一个 Arduino，但在这种情况下，它可能是有意义的，特别是当它可以监控你在充满电池组剩余电量之前的剩余时间。

尽管如此，像大多数危险特技一样，这看起来真的很酷。如果你有任何改进控件的想法，请在下面的评论中留下。如果你想把自己变成一个超级英雄，向一个正在这么做的人学习——我们自己的詹姆斯·霍布森。看看他的一些构造，比如[美国队长盾](https://hackaday.com/2016/05/13/captain-americas-mighty-shield-features-electromagnets/)或者[他的汽车升降外骨骼](https://hackaday.com/2016/01/25/homemade-exoskeleton-lifts-mini-cooper-with-ease/)。
