# 一个非常现代的图灵机模型

> 原文:[https://hack aday . com/2016/07/12/a-very-modern-turing-machine-build/](https://hackaday.com/2016/07/12/a-very-modern-turing-machine-build/)

数学家。如果你让他们使用无限的概念，几乎没有什么是他们无法证明的。典型的例子:图灵机。这个想法是，用无限长的磁带，一个人可以建立一个思想实验机器，只有几条指令，应该能够计算任何可计算的东西。

[伊戈尔]的[图灵机](http://hyperglitch.com/articles/turing-machine)是我们见过建造的最好的机器之一。“胶带”比无限短得多，这限制了他可以实现的计算，3D 打印、电接触和 WS2812 RGB LEDs 在胶带上的使用非常令人满意。

如果指示灯熄灭，磁带上的一个位将被显示为未使用，如果指示灯为红色，则显示为 0，如果指示灯为绿色，则显示为 1。磁带上的每个站都由一组蓝色 led 指示，由写入头的台架观察，写入头使用 3D 打印的手指和电机来改变每个位的状态。程序存储在自制的穿孔卡上，从我们这里获得额外的极客积分。

看着它在“busy beaver”(嵌入在下面)中运行，告诉我们它并不棒，即使它*比*差几个 led。
