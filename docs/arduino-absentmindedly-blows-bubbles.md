# Arduino 心不在焉地吹泡泡

> 原文：<https://hackaday.com/2016/07/30/arduino-absentmindedly-blows-bubbles/>

如果你想用泡泡来营造节日气氛，[Sandeep_UNO]可能会为你准备好这个项目。正如你在下面的视频中看到的(是的，它应该让手机旋转，但它没有)，他的 Arduino 使用伺服电机将泡沫棒浸入肥皂溶液，然后在风扇前拉动它。整个操作一遍又一遍地重复。

我们找不到很多细节和代码，但老实说，如果你知道如何从 Arduino 驱动伺服电机，剩下的就很容易理解了。仔细观察机器人的运动。当间歇地应用时，通常用旋转的泡泡棒和持续的风扇完成的事情变得有趣多了。懒惰的节奏是你期望从人类操作中看到的，这增加了一些效果。

我们见过[更快的吹泡泡机](https://hackaday.com/2015/08/12/blow-14000-bubbles-per-minute-with-this-3d-printed-contraption/)，但它们并不那么简单。我们甚至观察了其他[吹泡泡机器人](https://hackaday.com/2011/07/10/bubble-blowing-bot-blows-big-bouncy-bubbles/)。如果你想了解更多关于伺服电机的信息，我们的【理查德·鲍古利】有[你需要知道的](https://hackaday.com/2016/03/24/hackaday-dictionary-servo-motors/)。
