# 树莓 Pi SDR

> 原文：<https://hackaday.com/2017/01/27/raspberry-pi-sdr/>

[Chris D]注意到优秀的软件定义无线电(SDR)软件 gqrx 现在将在树莓 Pi 上运行。所以他结合了一个 Raspberry Pi 3、一个触摸屏、一个 RTL-SDR 加密狗和一个上变频器，使[成为一个非常好的接收器设置](https://www.youtube.com/watch?v=hreJ6eNAdeU)。你可以在下面看到接收器的动作。

该视频对构建细节的描述不多，但有一个带有标签的设置镜头，您应该能够从那里弄清楚。当然，gqrx 可以与许多不同的 SDR 设备一起工作，所以您可能需要根据您使用的设备进行调整(例如，许多受支持的加密狗不需要[Chris]使用的上变频器)。

对 gqrx 的实验性 Raspberry Pi 支持从去年 10 月才开始，但正如你在视频中看到的，它似乎工作得很好。使用一个更强大的 SDR 模块会更好，但是一定要检查 gqrx 站点上关于什么硬件与 Pi 一起工作的[信息。](http://gqrx.dk/download/gqrx-sdr-for-the-raspberry-pi)

我们已经在许多不同的平台上看到了 gqrx [、](https://hackaday.com/2012/10/08/os-x-port-of-gqrx-is-the-easiest-way-to-get-into-software-defined-radio/)及其特性和广泛支持的 SDR 硬件，确实使它成为一个非常有用的工具。如果[克里斯]正在寻找一个简洁的案例，我们可能会[建议这个](https://hackaday.com/2015/10/04/etch-a-sdr/)。如果你想构建 SDR，但不想使用 gqrx 之类的罐装软件，我们建议 [GNU Radio](https://hackaday.com/2015/11/11/getting-started-with-gnu-radio/) 。
