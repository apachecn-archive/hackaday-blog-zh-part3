# 制作自己的电流钳探头

> 原文：<https://hackaday.com/2018/01/27/make-your-own-current-clamp-probe/>

如果你想用示波器测量交流或 DC 电流，电流钳是一个很好的方法。夹子围绕着电线，所以你不需要断开连接来进行测量。这些曾经很贵，尽管我们见过一些不到 100 美元的，如果你购物的话。我们不知道是成本还是原理促使[电子人]建造自己的电流钳，但他做到了。

这种探针设计只不过是一个 3D 打印的外壳，一个旧的电源环，和一个传统的鳄鱼夹，以使业务结束。该传感器使用铁氧体磁芯和霍尔效应传感器。铁氧体磁环被分成两半，一半在夹钳的每一侧。运算放大器电路提供 100 的增益，以提高霍尔效应传感器的输出。

除了自制探针，视频还展示了 Hantek 电流探针的拆卸过程，并解释了不同类型电流探针背后的理论，包括一些技巧，如使用补偿绕组来防止铁芯磁化。

有用吗？你打赌。校准后，它做得很好。它没有 100 美元的单位漂亮，但情人眼里出西施，我们喜欢自制设备，所以我们会说它肯定更有趣。如果你有一个不错的垃圾箱(和一台 3D 打印机)，这种探测器可能会非常便宜。霍尔效应和 BNC 连接器可能是最昂贵的部件。即使你买了所有的东西，用了非印刷的箱子，我们也很难相信你会花 25 美元以上。

如果你想看看那些大公司是怎么做的，[是德科技去年的表现很好](https://hackaday.com/2017/12/04/current-measurement-with-oscilloscopes/)。我们已经看到其他自制版本的电流探测器，其中一些[非常精确](https://hackaday.com/2017/08/30/a-very-accurate-current-probe/)。
