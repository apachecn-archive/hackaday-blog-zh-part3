# Hackaday 奖参赛作品:一次喝一口对抗脱水

> 原文：<https://hackaday.com/2017/09/02/hackaday-prize-entry-fighting-dehydration-one-sip-at-a-time/>

没有水，人类活不了多久，大多数人即使有足够的饮用水，也会处于慢性轻度脱水状态。很难保持适当的水分，更难的是跟踪你的摄入量，这是这个饮水监测物联网吸管背后的想法。

脱水在老年人中是一个特别严重的问题，因为口渴感会随着年龄的增长而减弱。[jflaschberger]的 Hackaday 奖参赛作品旨在自动完成记录液体摄入的乏味且容易出错的工作，这是护理人员通常不得不通过目测半空的杯子和猜测来完成的事情。水力服务使用一个微型涡轮流量计，可以安装在吸管或水瓶盖上。涡轮机中的霍尔传感器将流量数据发送到赛普拉斯 BLE SoC 模块，该模块累计吸入的容量并记录患者标识符。然后，护理人员可以在一天结束时扫描来自液压服务的数据，以制作图表，并找出是否有人落后于他们的液体。

有些问题需要解决，尤其是涡轮机，它似乎不符合食品安全。但这是一件小事，不应该阻碍像这样好的想法。在 2017 年 Hackaday 奖的辅助技术阶段，我们已经看到了很多好的参赛作品，比如[在楼梯上工作的助行器](https://hackaday.com/2017/08/22/hackaday-prize-entry-staircane-a-walker-that-takes-the-stairs/)或[盲人声波眼镜](https://hackaday.com/2017/08/28/hackaday-prize-entry-sonic-glasses/)。这个阶段只剩下几天了——有什么好主意吗？
