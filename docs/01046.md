# 物联网电源板让你控制所有节日灯

> 原文:[https://hack aday . com/2015/12/27/IOT-power-strip-lets-you-control-all-you-holiday-lights/](https://hackaday.com/2015/12/27/iot-power-strip-lets-you-control-all-your-holiday-lights/)

随着物联网设备在消费者世界中变得越来越普遍，购买物联网设备比制造物联网设备更便宜还需要多久？肯定还没有，这意味着如果你想要自己的物联网电源板，你必须自己制作。还好这并不难！

[开发实验室]想出了这个项目，让他可以用手机控制几个插座。我们真正喜欢的是他设计了一个 3D 打印的外壳，可以安装在电源板的末端。这样可以让所有乱七八糟的电线看不到，看起来就像是被设计在那里一样！

像这样的物联网设备的优点是，除了支持 WiFi 和 HTTP 浏览器的设备之外，它不需要任何基础设施 ESP8266 模块意味着不需要服务器。Arduino 之所以被用在这个项目中，只是因为它简单快捷。但它实际上是一个美化的引脚扩展器。通过从 ESP01 升级到 ESP03 模块，在载板上获得更多 IO，可以很容易地解决这个问题。如果你这样做了，[请告诉我们](http://hackaday.com/submit-a-tip/)！

如果你想自己获取 3D 文件，可以去 [Thingiverse](http://www.thingiverse.com/thing:1211810) ，软件则是 [GitHub。](https://github.com/dev-lab/esp-iot-generic-switch)
