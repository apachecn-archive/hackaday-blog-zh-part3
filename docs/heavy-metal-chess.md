# 重金属象棋

> 原文：<https://hackaday.com/2017/01/21/heavy-metal-chess/>

国际象棋是一个缓慢的游戏，需要谨慎的决策，比棋盘的当前状态超前几步。机械加工也是如此，将两者结合起来是提升你的机械加工水平的绝佳方式。所以我们现在有一个来自[约翰·克瑞希]的项目，他正在用不锈钢加工一副国际象棋。

这不是新奇的计算机数字控制在起作用，这是久经考验的手工加工艺术。像象棋一样，你需要提前计划几个步骤，以确保你有办法为每个渐进的加工过程安装零件。在这个系列的第一个视频中,[John]正在铣削骑士——四个骑士，其中两个最终将进行黑色氧化处理。

磨制马头看起来很有趣，但当作品到达底部时，你会很高兴。[John]在车床上加工底座时，使用一个管件作为夹具来固定已经铣好的马头端。这个过程从去除内螺纹开始，然后非常小心地将管件加工到棋子的直径，以实现完美的压配合。整洁！

最后，[约翰]提到制作四件作品“花了好几个月的周末才达到这一步”。它们看起来很棒，我们迫不及待地想看到布景中的下一件作品变得栩栩如生。你会发现下面嵌入的视频，但如果你无法将这种时间沉入自己的棋局，你可能会[尝试三维激光切割亚克力棋子](https://hackaday.com/2015/06/14/3d-miniature-chess-pieces-made-with-a-laser-cutter/)。
