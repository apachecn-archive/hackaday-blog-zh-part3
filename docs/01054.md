# 使用礼帽导航以时尚的方式出门

> 原文：<https://hackaday.com/2015/12/30/stepping-out-in-style-with-top-hat-navigation/>

可穿戴技术正在成为一件大事。但是我们如何与这种设备进行交互仍然是一项正在进行的工作。为了探索这个空间，[Bruce Land]的微控制器课程学生想出了[一个声学接口来帮助行走时的导航](https://www.youtube.com/watch?v=pugLKdVPpZY)。当然是有格调的。

来自康奈尔大学电气和计算机工程学院的[Bruce]一直在用他的学生们的期末项目烧 Hackaday tips 线。这是[声音导航帽](http://people.ece.cornell.edu/land/courses/ece4760/FinalProjects/f2015/ath68_sjy33_vwh7/ath68_sjy33_vwh7/ath68_sjy33_vwh7/SoundNavigation/SoundNavigation.htm)的概览页面。它使用带 GPS 和指南针的 PIC32。我们花了很多时间来弄清楚如何正确地检索和解析 GPS 数据，但对我们来说，该页面上有趣的部分是方向声音是如何组合在一起的。

音频音调通过相移和振幅传送到耳塞，使声音看起来像是来自你应该行走的方向。导航都是基于预先编程的路线，这些路线是使用一个小的 LCD 屏幕和按钮选择的。有一点是肯定的，从时尚的角度来看，该项目的头饰选择是无可非议的——[工程学与礼帽](http://www.history.co.uk/biographies/isambard-kingdom-brunel)有着悠久的历史，我们认为现在是它回归的时候了。

这是陆地导航的实用解决方案吗？当然不是。但它可以在智能手机音频播放器中实现，用于环境逐路段导航。作为一个学生项目，这是一种展示新颖界面的有趣方式。我们最近报道了[一个为视障人士设计的触觉导航界面](https://hackaday.com/2015/09/04/experimental-theater-helps-field-test-haptic-navigation-device/)，它使用了类似的原理。看看这两个接口中的任何一个会走向何方都是很有趣的。
