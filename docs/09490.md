# 本周失败:不能直线研磨的半导体研磨机

> 原文：<https://hackaday.com/2018/05/21/fail-of-the-week-the-semiconductor-lapping-machine-that-cant-lap-straight/>

从一个旧的硬盘驱动器制造一台半导体研磨机似乎是一个好主意。但是[electronupdate]的构建有点问题，我们认为 Hackaday 社区可能会提供帮助。

对于那些不了解半导体解剖学和生理学的人来说，看看芯片的内部可以揭示逆向工程设备所需的有价值的信息，或者只是满足好奇心。研磨(轻轻磨掉材料)是一种看到位于环氧树脂下面的硅芯片层的方法。设计转速为 7200 转/分或更高的硬盘似乎不太适合轻轻研磨，但[electronupdate]只是想要超光滑、超平坦的表面。

他取下磁头，用齿轮电机和控制器取代原来的电机，使磁盘以低于 5 rpm 的速度旋转。制作了一个用于去盖芯片的小支架，并夹在盘毂和惰轮之间。它轻轻地旋转模具对磨料覆盖的浅盘，因为它慢慢旋转。但是模具磨损不均匀。他尝试了许多不同的模具夹具，但从未达到一层一层看穿模具所需的精度。我们想知道模具夹具的重量是否会使盘片有点偏移？

失败是一种很好的学习方式，如果你真的能找出哪里错了的话。我们期待 Hackaday 社区提供一些见解。看看下面的视频，如果你有任何想法，请在评论中发表。
