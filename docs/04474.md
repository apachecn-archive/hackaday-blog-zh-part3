# 21 世纪的烛台

> 原文：<https://hackaday.com/2016/12/21/a-menorah-for-the-21st-century/>

对于那些新的和有经验的人来说，每年的这个时候是一个很好的机会，让有事业心的制造者运用他们的技能为家人和朋友创造独特的礼物和装饰品。我今天做的*的【迈克·戴蒙德】*造了一个[电话控制的、发光的烛台](http://www.whatimade.today/light-your-menorah-hanukiyya-christmas-tree-remotely-from-your-phone/)。这是一种在假期展示一些家庭自动化知识的迷人方式。

在他之前的项目——一个口袋大小的烛台——的基础上，一个带有 WiFi 转换器、一些 led、电线和茶灯的树莓派 Zero 足以满足这些材料的需求，同时在树莓派上设置 Blynk 和一个控制灯光的手机，在将其安装在一个旧的显示器外壳后将它连接在一起。

 [https://www.youtube.com/embed/Ly-2MeiSrcw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/Ly-2MeiSrcw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



[Diamond]的指南是为初学者编写的，所以这个小事件可以在灵活的预算内适应一系列定制的想法。假设你可以偷偷拿着送礼者的手机安装应用程序，这将是一个巨大的惊喜。光明节从周六开始；如果你现在就下零件订单，你肯定能及时完成！

那些对手机连接不感兴趣的人当然可以建造他们自己的基于微控制器的 LED 烛台。EMSL 有一个带有令人敬畏的 PCB 设计的烛台，Hackaday 自己的(Mike Szczys)提出了一个紧凑的单面设计(T3)，你可以在家里蚀刻。
