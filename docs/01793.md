# R2-D2 让这个大学浴室闻起来很清新

> 原文：<https://hackaday.com/2016/03/20/r2-d2-keeps-this-university-bathroom-smelling-fresh/>

为了一个有趣的大学项目，[Cengizhan Can]和朋友们决定为荷兰乌特勒支大学的一间浴室制作一个自动空气清新剂。碰巧有一个旧的 R2-D2 塑料糖果罐，他们决定试试看能否把它做成《星球大战》主题。

他们在 R2-D2 的外壳上安装了一个 Arduino，一大堆传感器(别担心，没有间谍功能)和一个触发空气清新剂的致动器。在正常操作过程中，LCD 将显示环境温度信息和罐中剩余空气清新剂的大概数量。一个外门传感器检查摊位是否打开，并结合一个光传感器，只在使用后，当没有人在摊位上时喷洒清新剂。

他们还增加了一个运动传感器和一个超声波传感器，也可以用来确定占用情况——我们认为这是未来的升级。
