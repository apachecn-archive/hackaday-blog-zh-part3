# 小提琴弓光剑

> 原文：<https://hackaday.com/2017/01/22/a-violin-bow-lightsaber/>

[Bithead942]十岁的侄女是《星球大战》的超级粉丝，也是小提琴手。这当然让她学会了演奏电影系列中的一些音乐，然后让她的叔叔让她的小提琴弓像光剑一样发光。

乍一看，他的解决方案似乎相当简单，只是简单地将一条 DotStar 可寻址 led 连接到弓上，并从 Arduino Pro Mini 驱动它们，以获得 saber 通电所需的动画。当然，这个项目还有另一个层面。这把弓不仅要完成它的光剑特技，还必须是一把可玩的弓。电子设备不能因为太重或太麻烦而妨碍音乐家，但结果必须有足够的电力储备，以在演出期间保持灯光亮着。

在对 AAA 电池和 CR2032s 进行实验后，通过将一个小型 Li-po 电池连接到带有工业 Velcro 的弓末端的顶部来满足功率要求，并使用正畸医生使用的小型橡皮筋来粘合和进一步固定 LED 带。

休息时间下面是弓的光剑动作的简短演示，我们确信它会给年轻的小提琴家的观众留下深刻印象。
