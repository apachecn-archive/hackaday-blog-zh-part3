# 使用开源硬件构建智能气枪

> 原文:[https://hack aday . com/2017/07/10/building-a-smart-airsoft-gun-with-open-source-hardware/](https://hackaday.com/2017/07/10/building-a-smart-airsoft-gun-with-open-source-hardware/)

[matt8588]为他的气枪猎枪设计了一个[智能枪装备(YouTube，下面嵌入)。他有一个 Rasperry Pi 3 安装在一个 PEQ 盒内，连接到一个红外摄像机，红外 tac 灯有助于照明。一系列按钮控制叠加在摄像机图像上的十字准线图案，该图像显示在平板电脑上。您也可以重新定位十字准线，以拍摄更远的距离。其中一个按钮触发发射器上的信号，用于在战斗中引爆气枪克莱莫地雷。第二个圆周率为零，连接到一个](https://www.youtube.com/watch?v=W88FQqPZpys)[樱桃木](http://ozzmaker.com/product/berryimu-accelerometer-gyroscope-magnetometer-barometricaltitude-sensor/)传感器，该传感器控制一个由 12 毫米 led 组成的“交通灯”装置，当他移动枪支太多而不准确时，该装置会警告他。

如果你想了解[马特]的进步，他已经发布了他展示枪的准确性的视频，其中包括一个休息后他站着低头看平板电脑时准确射击的视频。你可以在他的 GitHub 库上找到两个 Pi 的代码。

我们认为不用说 Hackaday 有太多的气枪项目。我们特别喜欢我们发布的[气枪哨兵](http://hackaday.com/2015/12/06/airsoft-sentry-gun-keeps-your-house-guarded/)、[漫游车挂载气枪](http://hackaday.com/2014/08/24/the-counter-strike-airsoft-robot/)和无人机挂载 [POV 气枪炮塔](http://hackaday.com/2013/10/16/pov-airsoft-turret-on-a-rc-plane/)。
