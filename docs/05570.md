# FPGA 仿真 NES 车；原型如此赛博朋克

> 原文:[https://hack aday . com/2017/04/13/FPGA-emulates-nes-cart-prototype-so-cyber punk/](https://hackaday.com/2017/04/13/fpga-emulates-nes-cart-prototype-so-cyberpunk/)

到目前为止，我们大多数人都有过在新硬件上运行经典视频游戏的经验。无论是在个人电脑上使用键盘作为控制器，还是在定制的橱柜中进行更精细的改造，它都已经成为相对主流的产品。然而，非主流的是构建可以在原始主机 ( [翻译](https://translate.google.com/translate?sl=auto&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fwww.elektroda.pl%2Frtvforum%2Ftopic3326538.html%2316389389&edit-text=))上运行经典视频游戏的[定制硬件。完成的项目看起来令人惊叹，但原型的美丽和复杂让我们惊叹不已。](http://www.elektroda.pl/rtvforum/viewtopic.php?p=16389389)

[![](../Images/2d6d6ef8d2ee97812f6c8861569c3d2c.png)](https://hackaday.com/wp-content/uploads/2017/04/pegasus-famicon-cartridge-loader-pcb-closeup.jpg)【phanick】的项目是一个能够在名为 Pegasus 的波兰 Famicon 克隆体上运行游戏的卡盒。游戏存储在 SD 卡上，但不是在仿真器中运行，而是由 FPGA 加载 rom，并通过控制台盒式插槽中的正常边缘连接器呈现数据。游戏是从复古硬件本身来玩的。装入每个只读存储器需要几秒钟，但之后飞马座就看不出这和原来的弹药筒有什么不同了。

这里展示的原始原型建于 2012 年。从那以后，它经历了几次迭代，缩小了尺寸。多氯联苯是内部设计和制造的，最新的版本还包括一个 3D 打印的外壳，更接近原始 Famicon 墨盒的大小。

即使你对经典电子游戏或仿真不感兴趣，下面的视频也值得一看。(不会波兰语的一定要打开字幕。)[phanick]投入了大量的时间让所有的细节都完全正确，最终产品的抛光水平也体现出来了。事实上，我们之前已经为他制作了自己的 Famicom 克隆版。
