# 机械师魔术:量块绞拧

> 原文：<https://hackaday.com/2017/01/09/machinist-magic-gauge-block-wringing/>

在这个专利骗子和价值数十亿美元的公司对植物基因和白色背景下的拍摄对象提出知识产权要求的时代，你认为在普通钢块上申请专利将是又一个荒谬的想法，这是可以原谅的。但事实并非如此——[ Carl Edvard Johansson]在 1901 年获得了“精密测量量块装置”的专利。正如[AvE]通过视频[向我们展示量块如何“绞拧”在一起](https://www.youtube.com/watch?v=gbsd2OpPOMw)一样，这些小量块并不像看上去那么简单。

对于有经验的机械师来说，量块绞拧可能不是什么新鲜事，但对于我们其他人来说，这是一个非常巧妙的技巧。作为节目的开始，[AvE]向我们简要介绍了“Jo blocks”及其用途。基本上，每个块都是一块经过研磨和抛光达到特定长度的工具钢或陶瓷。这些木块有多种长度可供选择，可以首尾相连地堆叠起来，组成一根非常精确的测量棒。但是砖块不仅仅是彼此相邻放置，它们在拧在一起后，通过重叠的表面物理地粘附在一起。[AvE]演示了绞拧技术，并提供了一些关于这种有点神秘的粘连是如何发生的想法。这是非常令人着迷的东西，让我们有心情得到一个量块来尝试一下。

我们已经有一段时间没有看到[AvE]出现在 Hackaday 周围了——上次他在用一片面包制作碳泡沫。请放心，从那时起，他的频道一直很强大，他独特地融合了笑声和对工具秘密生活的洞察。绝对值得一试，而且仍然像 frig 一样酷。
