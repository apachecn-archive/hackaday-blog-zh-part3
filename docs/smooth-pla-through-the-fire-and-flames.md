# 顺利通过火灾和火焰解放军

> 原文：<https://hackaday.com/2018/05/03/smooth-pla-through-the-fire-and-flames/>

3D 打印使生产复杂的几何形状变得容易，但熔融沉积方法通常会产生表面光洁度差的零件，这主要是因为这些层在成品零件中高度可见。有各种各样的方法来处理这个问题，通常涉及生产后打磨零件，或使用填料和油漆。[【XerotoLabs】有另一种解决方案。](https://www.youtube.com/watch?v=-zWXFQKt0w4) (YouTube，视频下方休息。)

为了使零件光滑，丁烷焊炬被压入使用。火焰温度保持在相当低的水平，焊枪几乎像刷子一样被用来均匀地加热零件表面。当 PLA 达到其熔化温度时，表面张力有助于使零件变得平滑。这与丙烯酸塑料制造中常用的[火焰抛光](https://en.wikipedia.org/wiki/Flame_polishing)非常相似。

这是一项需要一些技巧的技术——过多的热量或集中在一个区域，你很可能最终得到一个熔化的塑料块，而不是一个漂亮闪亮的成品零件。还必须采取预防措施，以避免将你自己或你的车间夷为平地。但是，当您生产希望达到最佳外观的 PLA 零件时，这是一个非常有用的工具。
