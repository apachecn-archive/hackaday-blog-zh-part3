# 制造一个微小的(不稳定的)窃听装置

> 原文:[https://hack aday . com/2016/08/31/build-a-tiny-unstable-bugging-device/](https://hackaday.com/2016/08/31/build-a-tiny-unstable-bugging-device/)

我们不知道[amgworkshop]想监听谁，但他们显然在寻找一个小型调频无线发射机。周围有很多赛道，但他们想要更小的。原始电路有一个可变电容来调谐输出频率。[新设计](http://www.instructables.com/id/1-Transistor-FM-Spybug/)使用固定电容和弹簧作为天线。你可以在下面的视频中看到构建步骤，但不要期望单个晶体管发射器有很高的频率稳定性或保真度。

零件清单很少。除了一个硬币电池座(作为构建基础)，你还需要一个晶体管，两个电阻，三个电容，一个自制的电感器(用一些电线和钻头很容易制作)，和一个驻极体麦克风。当然，你也需要一块电池。整个东西都是用热胶水封装的。

如果你想要一个更好的电路(和更长的电池寿命)，你可以看看[安吉洛的] [类似的建设](https://hackaday.com/2014/07/30/a-dead-simple-well-constructed-fm-transmitter/)。不过，没那么小。这里并不缺少[其他的例子](https://hackaday.com/2013/12/07/super-simple-fm-transmitter/)，很多都使用不同的建造方法。
