# 半自动轨道炮是一台笔记本电脑黑仔

> 原文：<https://hackaday.com/2017/10/14/semi-automatic-rail-gun-is-a-laptop-killer/>

它很大，很笨重，而且要花 45 秒才能打完弹匣里的所有 3 发子弹。但它是[合法的半自动轨道炮](https://imgur.com/a/9OSSa)，相当牛逼。

是的，它有其局限性，但每项新技术都有，尤其是像这样完全自制的。名副其实的[NSA_listbot]一直在他的轨道炮上投入大量工作，这只是一个迭代设计周期的最新产品。

其原理类似于我们在之前介绍过的[其他轨道炮，使用快速脉冲电磁体加速射弹。下面视频中的功能列表读起来像是一个绝密军事项目的规格:场增强圆孔，4.5kJ 电容器组，以及定制的 Arduino Nano，它可以抵抗线圈产生的巨大电磁脉冲(EMP)。但是有趣的部分是在机械设计上，它必须脱离标准的枪械设计来处理无壳的 6 毫米射弹。最终的接收器和杂志完全是 3D 打印的。尽管它具有强大的冲击力，但它的循环射速却慢得令人痛苦。不过，我们预计随着电池和电容器技术的发展，这种情况会有所改善。](http://hackaday.com/2016/08/19/coil-gun-for-newbies-learning-electromagnetic-propulsion/)
