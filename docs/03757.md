# Galaga 橱柜是世界上最好的

> 原文:[https://hack aday . com/2016/10/06/galaga-cabinet-is-out-of-this-world/](https://hackaday.com/2016/10/06/galaga-cabinet-is-out-of-this-world/)

像我们许多人一样，[亚历克斯]童年的大部分时间都是在一个又一个街机游戏中度过的。Galaga 是他一直以来的最爱之一，他早就想打造一个 Galaga 内阁了。一旦他的车间准备就绪，就该把它从清单上划掉了。

这个柜子是按 4/5 的比例制作的。这是一个很大的尺寸，因为他获得了全尺寸机器的稳定性和手感，但更容易移动它。如你所料，内阁里有 Pi。显示器是一台旧电视，是[亚历克斯]在垃圾箱里找到的。虽然它工作得很好，但它会进入待机状态，而不是和其他东西一起关机。为了解决这个问题，[Alex]制造了一个带有红外 LED 和 Arduino Diecimila 的自动遥控器。等待五秒钟后，它会向电视发送开机代码并切换输入。Galaga 的电视应该处于肖像模式，但这被证明是一个挑战。在 Pi 级别更改方向导致性能不佳和声音断断续续，所以他在游戏执行级别进行了更改。

我们一直对[Alex]的多样性和对他们的关心印象深刻。谁会忘记他那台[漂亮的人行道涂鸦机](https://hackaday.com/2015/12/11/beautiful-sidewalk-graffiti-machine/)或者那次他向我们展示[如何拍摄不存在的东西](https://hackaday.com/2011/03/23/photographing-stuff-thats-not-there-by-using-stencils/)？跳转到观看一个简短的演示，然后是一个由两部分组成的构建视频。
