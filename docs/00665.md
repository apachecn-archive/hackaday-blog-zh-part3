# Eyedrivomatic 赢得 2015 年黑客日大奖

> 原文：<https://hackaday.com/2015/11/14/eyedriveomatic-wins-the-2015-hackaday-prize/>

**更新:**我们发表了一篇关于赢得黑客奖的[凝视控制轮椅的深度文章。](http://hackaday.com/2015/11/17/the-gaze-controlled-wheelchair-that-won-the-hackaday-prize/)

[Eyedriveomatic](http://hackaday.io/project/5426) 是 [2015 黑客日大奖](http://hackaday.io/prize)的大奖得主。刚刚在 Hackaday 超级大会的舞台上宣布了获奖者，并由评委颁奖。Eyedriveomatic 是一种为电动轮椅增加眼球控制的非侵入性方法。很多时候，这些轮椅是租来的，无法进行永久性的改造。这种廉价且易于适应的硬件有能力改善那些需要更多选择来控制电动轮椅的人的生活。

我们将在接下来的一周发布更多关于今年获奖者的信息。下面列出了所有的排名。请查看所有的 [2015 Hackaday 奖决赛选手](https://hackaday.io/list/7999-2015-hackaday-prize-finalists)和[最佳产品决赛选手](https://hackaday.io/list/7403-2015-best-product-finalists)。

*   大奖: [Eyedrivomatic](http://hackaday.io/project/5426)
*   最佳产品:葡萄酒
*   第二名: [OpenBionics](http://hackaday.io/project/4745)
*   第三名:[轻型电动多功能车](http://hackaday.io/project/4997)
*   第四名:[急救人员用气体传感器](http://hackaday.io/project/4742)
*   第五名:[多尔皮](http://hackaday.io/project/6958)
