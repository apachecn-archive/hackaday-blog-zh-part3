# 坐在我的魔毯上

> 原文：<https://hackaday.com/2015/11/01/take-a-ride-on-my-magic-carpet/>

为了一个令人敬畏的万圣节版[PrankvsPrank]，他们决定为[阿拉丁]制作一个遥控“魔毯”。这真是太棒了。

[PrankvsPrank]并不以制作我们会在 Hackaday 上刊登的东西而闻名，所以他们从纽约的 YouTube[CaseyNeistat]那里获得了一些项目帮助。你大概可以猜到，这块魔毯的核心其实是一个电动滑板。具体来说，是一个增强版——几年前，它是一个非常成功的 Kickstarter。

他们用塑料管做了一个轻质框架，并用织物包裹起来，让它看起来像漫画中的地毯。一旦一切就绪，他们就开着它在纽约的街道上兜了一圈，沿途捕捉人们的反应。
