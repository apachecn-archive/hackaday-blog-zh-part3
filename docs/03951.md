# 网络门铃全面流行

> 原文：<https://hackaday.com/2016/10/24/internet-doorbell-gone-full-hipster/>

有事情也有事情。安装一个可以“响”压电蜂鸣器或发送短信的联网门铃是可以的，但这并不优雅。在所有的物联网喧嚣中，太多的注意力放在了“互联网”上，这实际上是容易的部分，而对“物”的关注太少。

[Moris Metz]是柏林的一名黑客，他有一个两周一次的国家电台节目。(仅限德国！)本周，他[将无处不在的 ESP8266 连接到一个漂亮的旧(物理)钟](http://dradiowissen.de/beitrag/netzbasteln-das-internet-der-ding-dong-dinge)上，用于他周末的广播。(此处我“翻译”了[。)查看下面嵌入的视频预告。](https://translate.google.com/translate?sl=auto&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=http://dradiowissen.de/beitrag/netzbasteln-das-internet-der-ding-dong-dinge&edit-text=&act=url)

不要管让你的小玩意在广播节目中看起来有多重要，这个东西听起来棒极了。而且声音很大，这让我们质疑在试图直播电台节目时将 IP 地址告诉听众是否明智。但是演出必须继续，尽管在宣布 IP 地址的十分钟内，来自 135 个不同 IP 地址的超过 424 个单独铃声敲响了他的钟。

让我们正式记录在案。作为一个字面上的门铃，这是愚蠢的。你不需要在门铃上安装一个公共互联网上的网页，全球家居商店中数百万个超便宜的 433 MHz 无线电门铃证明了这一点。然而，当你需要被一个在不同城市的朋友触发的响亮的警铃吓得魂不附体时，互联网是你最好的选择，在 ESP8266 上快速创建一个网页很有趣。

但我们喜欢这个项目的是“鼎”本身，正如他们在德国所说的。旧的硬件搭配新的大脑是新的热点。你见过的现代大脑移植中最古老或最不可能的事情是什么？
