# 总是乱放你的钥匙？你可以用一些逻辑芯片解决这个问题

> 原文：<https://hackaday.com/2018/02/04/always-misplacing-your-keys-you-can-fix-that-with-some-logic-chips/>

每次他回家，都是同样的事情。他一跨过门槛，他的钥匙就不见了。对此没有其他合理的解释。每当要离开的时候，他都要把房子翻个底朝天才能找到它们。

有一天，[开箱即用]决定他已经受够了，用他手头的东西——一个十进制计数器、一个廉价的簧片开关门报警器和一些晶体管——建造了一个门启动报警系统。当门关上时，十进制计数器的输出会点亮一个绿色 LED。当他回家开门时，簧片开关关闭，触发十进制计数器将其输出转移到下一个管脚。红色 LED 亮起，NPN 晶体管将压电接地，发出警报。阻止它的唯一方法是将他钥匙圈上的 1/4 英寸短电话插头插入电路板上的插孔，直到他听到令人满意的安全敲击声。

对于那些无法立即将钥匙插入墙壁的时候，或者如果他的钥匙在有机会之前消失了，电路板上会有一个瞬间，可以阻止机器人蝉从压电中发出的交响乐。对于不想一起玩或者还没有挣到 1/4 英寸插头的家庭成员来说，这也是一件好事。

休息之后，请务必查看构建视频，就在那扇门后。看好你的钥匙，好吗？Hackaday 不对丢失或被盗的个人物品负责。如果你丢失了它们，我们只能建议[用备用的](https://hackaday.com/2017/03/29/save-big-by-hacking-your-car-keys/)和[印刷替换任何标准钥匙](https://hackaday.com/2017/12/18/3d-printed-key-saves-the-day/)来制作一把新的汽车钥匙。
