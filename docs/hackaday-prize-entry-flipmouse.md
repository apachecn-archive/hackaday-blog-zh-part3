# Hackaday 奖参赛作品:FLipMouse

> 原文：<https://hackaday.com/2016/10/14/hackaday-prize-entry-flipmouse/>

上一届 Hackaday 奖挑战赛的主题是辅助技术，随之而来的是为有严重运动障碍的人提供技术解决方案。我们见过的最好的设备之一是一种设计用于使用 sip 和 puff 界面和按钮通过 USB 控制光标的设备。对于一个通过手指或嘴唇使用的设备来说，几乎太聪明的名字是 flip mouse T1，现在它正在角逐 Hackaday 奖的决赛。

与其说 FLipMouse 是鼠标，不如说它是一个非常长且非常灵敏的操纵杆。主要的交互方式是一根缠绕着力传感器的长空心管。这些力传感器，就像在任天堂 Power Glove 或[这另一个 Hackaday Prize](http://hackaday.com/2016/10/11/hackaday-prize-entry-handson-gloves-speaks-sign-language/) 参赛作品中看到的那样，将管子变成了一个异常敏感的操纵杆，旨在被用户的嘴唇抓住。这个管子也是中空的，所以一个吸喷接口被用来记录左击和右击。当然，有一些外部按钮可以重新映射到任何东西。

有多大用处？这种基于嘴的鼠标似乎特别能干。在下面的视频中，[Harry htzinger]在舞台上使用步进音序器和鼠标控制的合成器界面现场演奏合成器。这一切都是为特定的音乐片段进行了高度优化，但这是一个令人难以置信的展示，你可以用激光切割机和 Digikey BOM 做什么。
