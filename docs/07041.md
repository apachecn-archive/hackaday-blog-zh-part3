# 自行车上的喷气式发动机。最坏的情况会是什么？

> 原文：<https://hackaday.com/2017/09/18/a-jet-engine-on-a-bike-whats-the-worst-that-could-happen/>

在今天的“不要在家里尝试这个”节目中，我们来到了俄罗斯，去看看【伊戈尔·内戈达】正在工作的[喷气式自行车](https://www.youtube.com/watch?v=_O4mw0OMukE)。

这款标准山地车配备了一个能够产生 18 公斤推力的喷气式发动机，固定在座位下的框架上，带有一个可调支架，可以根据需要改变角度。一部手机被拉链绑在框架上，充当速度计——如果它能工作，那就不傻了——发动机控制器显示推力、转速和温度。LiPo 电池是发动机的电源，带有一个单独的较小的电子电池。这辆自行车几乎长满了给引擎供电的电线和管子，包括一个通常放水瓶的辅助油箱。主油箱在哪里？当然是在[内田]的背包里。

它肯定会扬起一片尘土，发出巨大的噪音，但真正的问题是:它能跑多快？从智能手机的外观来看，72 公里/小时，45 英里/小时，或 18 杆到大桶。

在最大推力下，【Negoda】和他的自行车——总共重约 100 公斤——能够在道路耗尽之前达到 72 公里/小时！

就像我们展示叛逆的后院发明家柯林·福尔泽的[涡轮增压喷气发动机](https://hackaday.com/2013/10/05/please-dont-build-a-jet-engine-from-a-toilet-paper-holder/)一样，我们必须强调，必须非常谨慎地对待喷气发动机。尤其是那些新奇的 [3D 打印的](https://hackaday.com/2016/02/15/a-3d-printed-jet-engine-appears-to-function/)这些天孩子们正在一起破解。
