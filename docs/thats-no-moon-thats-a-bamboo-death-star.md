# 那不是月亮——那是一颗竹制的死星

> 原文：<https://hackaday.com/2016/05/12/thats-no-moon-thats-a-bamboo-death-star/>

乍一看，[弗兰克·豪沃思]的[变成竹子死亡之星](https://youtu.be/7AgD5NUzyOE)似乎是一个直的木工项目。没有 Arduino 控制的灯，没有树莓 Pi 来播放 X 翼攻击或逃离 TIE 战斗机的音频剪辑。换句话说:黑客在哪里？

这是一颗怪异的竹制死星！

如果那对你来说还不够，检查完成的模型表面的图案。那不是画上去的——那是用来创造环的层压竹木材层[Frank],用来形成结构。经过大量的车削、打磨和抛光，竹子特有的维管束创造出明暗相间的面板，呈现出死星表面细节的逼真效果。虽然我们喜欢自然的完成，但我们可以想象更深的污渍可能真的会使细节突出，并产生更接近原始的效果。

还不够黑客吗？然后尽情欣赏[弗兰克]的店铺。这是一个洞穴般的空间，有高高的天花板，大量的自然光，以及人类所知的几乎所有木工机器。虽然车床和台锯为这个构建做了很多工作，但值得流口水的 CNC 路由器在创建构建所需的多个夹具以及为超级激光器制作切口方面看到了重要的职责，这肯定是一个紧张的时刻。

竹子是一种不可思议的材料，无论是像这样有趣的建筑还是更具结构性的用途，比如竹制自行车。所有这些竹子的优点让我们有心情拜访[Gerrit Coetzee]的“你应该知道的材料”系列。
