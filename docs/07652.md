# 饰品冷却你的饮料

> 原文:[https://hackaday.com/2017/11/22/trinket-chills-your-drinks/](https://hackaday.com/2017/11/22/trinket-chills-your-drinks/)

谁想要热饮？嗯，我猜是喝咖啡的人。除了他们，谁还想喝热饮？喝茶的人，当然。室温饮料怎么样？谁也不是，就是谁。买一个冰箱来冷却一杯饮料是愚蠢的，那么你还有什么选择呢？冰块？他们会稀释你的饮料。冰袋和冷却器？当然，他们会让你的饮料保持凉爽，但他们很难凉爽，不是吗？不，如果你想喝冷饮，你可以建一个热电冷却器。如果你想做一个，你很幸运，因为[约翰·派克]有一个教程可以在 AdaFruit 上做这个。

零件清单包括一个阿达果饰品 M0，一个更强大的版本阿达果的饰品线。这个小饰品用来控制这个建筑的主要部分，一个珀尔帖热电冷却器，以及温度显示器和开关。微控制器控制的另一部分是蠕动泵，用于分配液体。控制一切的代码是用 Python 写的，因为默认情况下，小饰品 M0 是随 AdaFruit 的 CircuitPython 一起提供的。教程中还包括支架的文件，如果你想 3D 打印它或用 CNC 或激光切割机切割它。

休息之后，您可以观看[John]检查项目并构建它，或者访问 AdaFruit 网站并按照说明构建您自己的项目。正如[约翰]所说，也许有更好的方法来冷却你的饮料，但这“绝对是更科学、更有趣的方法之一。”对于更多利用帕尔贴效应的项目，试试[这个在真空烹饪中利用该效应的项目，或者](https://hackaday.com/2011/08/03/peltier-powered-sous-vide-rig-gets-it-right-every-time/)[这个帕尔贴制冷微型冰箱](https://hackaday.com/2014/04/03/peltier-mini-fridge-preserves-chip-quik-marriage/)！
