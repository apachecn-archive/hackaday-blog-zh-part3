# 手动 LCD 使信息显示变得单调乏味，富有教育意义

> 原文:[https://hacka day . com/2017/04/28/manual-LCD-makes-information-display-dragonary-educational/](https://hackaday.com/2017/04/28/manual-lcd-makes-information-display-tedious-educational/)

HD44780 是我们小时候了解的第一批芯片之一，您很可能在项目中使用过它，并且几乎可以肯定您在生活中也接触过它。字符液晶显示器无处不在，易于接口，非常稳健。它们的尺寸从 8 x 1 到 20 x 4，甚至更大，但它们几乎都有相同的引脚排列，许多嵌入式环境中都有与它们交互的库。[8 位家伙]决定用一个开关和一个按钮(YouTube，embedded)与一个接口连接，目的是说明如何使用它们，以及它们有多简单。

如果你以前从未使用过字符型液晶显示器，并想对它们有一个很好的介绍，这是给你的视频。事实证明，不用担心时钟，从数据表中很容易看出指令集。[8 位的家伙]甚至对额外的命令很感兴趣。

这些展示在 Hackaday 上经常出现。仅仅有几个是使用 PIC 微控制器的[这种串行驱动破解，以及](http://hackaday.com/2014/02/01/controlling-alphanumeric-lcds-with-two-wires/)[利用自定义字符创建非字符图形](http://hackaday.com/2013/10/16/teach-an-old-lcd-new-tricks/)。

谢谢[emuboy]的提示。
