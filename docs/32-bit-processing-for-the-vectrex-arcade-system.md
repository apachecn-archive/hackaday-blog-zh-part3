# Vectrex Arcade 系统的 32 位处理

> 原文：<https://hackaday.com/2017/04/12/32-bit-processing-for-the-vectrex-arcade-system/>

除了 Commodores、Ataris、Nintendos 和 80 年代的所有其他游戏系统，还有一个独一无二的视频游戏系统脱颖而出。这就是 Vectrex，一个内置 CRT 的控制台，用来显示矢量图形*并且只显示*矢量图形。1983 年的电子游戏崩溃对威格斯来说并不友好，但它仍然伴随着一个相当受欢迎的自制场景而存在。尽管如此，这些自制游戏还是受到硬件本身的限制。[三十年后，威格斯有了升级](http://www.galacticstudios.org/vectrex32smartcart/)。Vectrex32 是一种协处理器，专为 Vectrex 盒式磁带插槽设计，为这种古老的控制台提供了更好的图形和 32 位功能。

有一个网站专门介绍这款 Vectrex 附加产品，硬件与您预期的差不多。这个墨盒上有一个快速的 PIC32 微控制器，一个 USB 端口和一个连接到 Vectrix 本机处理器的双端口存储芯片。

由于这种附加模块实际上就是一台计算机，因此 Vectrex32 可以作为 Vectrex 的基本解释器。这是原始硬件无法做到的，并且使得自制软件的开发更加容易。
