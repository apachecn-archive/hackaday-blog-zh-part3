# 用这个小小的电视装饰品再次制作圣诞广告

> 原文：<https://hackaday.com/2017/12/11/make-christmas-commercial-again-with-this-tiny-tv-ornament/>

一定年龄的读者会记得感恩节后美国圣诞节正式开始的那段时间。从那时起，广告商开始日以继夜地用节日主题的电视广告对公众进行饱和轰炸。广播电视不再像过去那样占据主导地位，广告商现在开始在 9 月份大举进攻，但你可以用这款 90 年代的圣诞广告为你的圣诞树装饰一点复古商业主义。

这个想法是[SeanHodgins]在 YouTube 上偶然发现一组 20 世纪 90 年代的圣诞广告后产生的。确定了内容后，他开始用 Pi Zero W 和 TFT LCD 显示器构建一个值得种树的显示器。一个来自旧平板电脑的音频放大器和微型扬声器以及一个 LiPo 电池和充电器构成了[肖恩]电视的内脏，它们被塞进一个 3D 打印的电视盒中，适当地模仿了*辛普森一家中的电视。*模仿昔日曲面屏的小型菲涅尔透镜手感不错。该软件有一些巧妙的技巧，例如 HTTP 服务器接受 YouTube 视频的片段，获取 MP4，并自动播放。我们更喜欢我们的圣诞树装饰品安静一点，所以音量控制会很好，但除此之外，这看起来很有趣。

当然，这不是[肖恩]第一次涉足装饰精美的饰品；读者可能会想起他上一季的[物联网的测量快乐的圣诞饰品](https://hackaday.com/2016/12/24/these-ornaments-measure-christmas-cheer/)。
