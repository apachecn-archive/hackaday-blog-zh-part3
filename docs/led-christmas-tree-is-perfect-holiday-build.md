# LED 圣诞树是完美的节日建筑

> 原文：<https://hackaday.com/2017/12/24/led-christmas-tree-is-perfect-holiday-build/>

一年中最美好的时光即将来临。全世界的家庭将会聚在一起交换礼物，吃美味的食物，一起享受美好的时光。对许多人来说，这将是自去年假期以来他们第一次见面。对我们黑客来说——这意味着我们有时间谈论自己，展示一下我们的工作。今年过得轻松吗？没什么好谈的吗？现在还不晚！拿出烙铁，制造一个完美的话题引子——一棵 [LED 圣诞树](https://hackaday.io/project/28615-flickering-led-christmas-tree)！

[Gumix]用一把闪烁的发光二极管和一个 DC-DC 转换器来制作他简单而优雅的圣诞树。这里没有微控制器…没有代码运行。一旦通电，闪烁的发光二极管就会产生视觉上的愉悦。

闪烁的 led 一直是一些黑客关注的焦点。它们基本上是设计成像真正的蜡烛一样闪烁的发光二极管。[cpldcpu]将一个示波器与一个挂钩，并猜测一个[线性移位寄存器](https://hackaday.com/2013/12/16/reverse-engineering-a-candle-flicker-led/)负责闪烁背后的随机性，[将在几个月后证实](https://hackaday.com/2014/03/02/reverse-engineering-candle-flicker-leds-again/)。
