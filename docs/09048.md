# 胡椒磨锁住你的门

> 原文:[https://hack aday . com/2018/04/02/pepper-mill-locks-your-door/](https://hackaday.com/2018/04/02/pepper-mill-locks-your-door/)

胡椒！如果你曾经尝试过研磨它，你可能已经注意到它需要一点点的努力。事实上，你可以找电动胡椒磨来为你研磨胡椒。碰巧[MarioM66]手头就有一个，还有一个需要自动化的门锁。

鉴于研磨胡椒需要的扭矩至少相当于转动普通锁中的普通钥匙，电动胡椒研磨器用作锁致动器非常有意义。这个建筑实际上使用电动胡椒研磨器直接转动锁中的钥匙，通过适配器将方形输出轴连接到钥匙。这个适配器是由一种叫做 MultiMorph 的可塑塑料制成的。胡椒碾磨机被用于其高扭矩电机和齿轮箱，这使得它绝对适合这种应用。

项目的其余部分严重依赖于黑客的 go-to、一个 Arduino 和一些现成的手势识别模块。现在，只需按下一个按钮或挥挥手，就可以锁门和开门了！休息后的视频。

很高兴看到普通物品被改造成对新项目有用的部分。同样，看看这辆能让你握拳解锁的汽车。
