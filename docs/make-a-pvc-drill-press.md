# 制作一个 PVC 钻床

> 原文：<https://hackaday.com/2017/03/31/make-a-pvc-drill-press/>

这个世界上有两种人:认为 PVC 只适合做水暖的人，和连你可以用 PVC 来提水都不知道的人。Instructables 用户[amjohnny]显然是后一派。他的 [PVC Dremel 钻床](https://www.instructables.com/id/PVC-Dremel-drill-press/?ALLSTEPS)有点过时，但它仍然是管工艺术的见证。你可以在下面嵌入的视频中观看它的运行。

我们特别喜欢这种构造的东西包括 PVC 平行四边形运动，围绕管子的弹簧将 Dremel 头推回，以及 T 形配件和螺旋塞的巧妙使用，以将印刷机保持在最低位置。我们想知道如何给这个东西加上深度停止。没关系，我们喜欢看它工作。

无论如何，这只是许多黑客中的一个，它强调了钻床在基本上任何人的生活中的重要性，以及 DIY 成一个的容易性。如果你在讨厌聚氯乙烯的阵营中，但周围有一些碎木头和抽屉滑轨或 T2 塑料下脚料，你就拥有了一台基本的印刷机——商店里受欢迎的工具。
