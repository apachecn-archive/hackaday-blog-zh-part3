# 面向游戏玩家的智能桌子

> 原文：<https://hackaday.com/2017/05/05/a-smart-table-for-gamers/>

当制造商开始为自己的家设计家具时，结果是惊人的。对于他们的高级设计项目，[Phillip Murphy]和他的队友们开始着手从头开始建造一张智能桌子。哦，你也可以用它来玩游戏，在下面的视频中演示。

这款桌子在 32 x 16 阵列中使用了 512 WS2812 像素，具有足够的分辨率来玩一系列集成游戏——围棋、双人俄罗斯方块和 Tron light cycle combat——以及一些其他功能，如跳舞的小鸟派对模式——因为如果智能桌子不能兼作狂欢照明，那它还有什么意义呢？

定制板上的 C2000 系列微控制器是大脑，由 Android 应用程序通过蓝牙 RN-42 模块控制。桌子的框架是用 Sketchup 设计的，用激光切割，并精心着色。[Murphy]和他的公司在每个“像素”中都使用了铝管道胶带，桌子的框架实际上形成了像素网格。休息之后，请查看概述和一些正在进行的游戏。
