# 黑客日奖参赛作品:物联网谢妮时钟

> 原文：<https://hackaday.com/2017/10/13/hackaday-prize-entry-iot-nixie-clocks/>

谢妮钟表是现在最流行的东西，至少十年来一直如此。作为他的 Hackaday 奖参赛作品，[姆拉登]正通过[一个 USB 供电的物联网谢妮时钟](https://hackaday.io/project/27349-nixie-tap)将东西带入 21 世纪。它显示时间、温度、当前的菲亚特加密货币价格、你当前的 Twitter 粉丝数量、你最新的 reddit meme 上的更新次数，或者任何可以用四位数表示的东西。

这个谢妮钟使用四个 12 英寸的电子管，带点，这或多或少是小谢妮钟的标准。这些灯管直接安装在 PCB 上，而 PCB 又以 90 度角安装在主板上，为机械加工的木制或铝制外壳提供了一个纤薄的外形。

控制电子设备围绕 ESP8266 构建，通过方便的 USB 连接提供电源和串行连接。BQ3200 实时时钟在超级电容器的帮助下保持时间。这里的杀手锏是一个压电传感器来检测外壳上的水龙头。按一下钟，它就会显示时间。敲两下，你的比特币钱包当前余额就显示出来了。这是一个伟大的项目，[姆拉登]希望将这个项目变成一个产品，并很快在众筹上推出。总而言之，这是获得黑客日大奖的绝佳机会。
