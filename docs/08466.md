# 防止磁化弹珠停止音乐

> 原文:[https://hack aday . com/2018/02/02/keeping-magnetized-marbles-from-stop-the-music/](https://hackaday.com/2018/02/02/keeping-magnetized-marbles-from-stopping-the-music/)

拿几千个钢球，加上一个嵌有钕磁铁的大型木制齿轮，你会得到什么？要么是一个美妙的动感音乐机器的开端，要么是一堆粘在一起的球堵塞了作品。

对于[马丁]来说就是后一种情况，如果他的“大理石机器 X”要重见天日，他需要找到一种在连续过程中使钢球消磁的方法。你可能记得[马丁]是 Wintergatan 乐队的成员，也是最初的弹珠机的发明者，这是一个出色的单人乐队，通过在各种乐器上投掷钢珠来制作音乐。尽管最初的大理石机器是一种神奇的装置，但它严格来说是一种工作室乐器，太脆弱了，不适合巡回演出。

Marble Machine X 是对原版的完全重新想象，旨在足够强大以进行世界巡回演出。[Martin]完全重新设计了提升机构，使用磁铁从回收箱中抓取球，并将其送入复杂的分配器。但是在提升过程中，球变得足够磁化，可以粘在一起，不再滚动到分隔器中。下面的视频展示了[Martin]的解决方案:一个消磁器，它使用极性交替的磁铁，在粘稠的弹珠下缓慢旋转。作为一个旁注，看一个音乐家在调试一个机械问题时拖延时间是很有趣的。

我们迫不及待地想看到 Marble Machine X 的运行，但在它完成之前，我们只会满足于[马丁]的其他音乐黑客，比如[他的纸带编程音乐盒](https://hackaday.com/2016/07/06/paper-tape-drive-for-a-live-performance-music-box/)或[这个合成器和小提琴的混搭](https://hackaday.com/2017/02/13/its-a-synthesizer-its-a-violin-its-a-modulin/)。
