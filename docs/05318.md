# Strandbeest 骗不了任何人——我们一眼就看穿了

> 原文:[https://hack aday . com/2017/03/15/strand beest-not-folling-any one-we-see-right-through-it/](https://hackaday.com/2017/03/15/strandbeest-not-fooling-anyone-we-see-right-through-it/)

这只 Strandbeest 已经准备好迎接一所安全意识很强的高中的警戒线了。像透明背包一样，[它透明的聚碳酸酯部件让你可以看到进入古怪移动机制的一切。尽管有多条腿，但如果你分析一只 Strandbeest 的运动，它实际上像一个轮子一样运动。](http://www.jcopro.net/2017/03/10/clear-polycarbonate-strandbeest-assembly-part-1/)

对我们来说，是下面的叙述制作视频让这个建筑变得非常有趣。多年来，Hackaday 的校友杰里米·库克(Jeremy Cook)一直在打造不同版本的 Strandbeest。 [Strandmaus](http://hackaday.com/2016/04/11/strandmaus-small-rc-strandbeest/) 是一架由微型四轴飞行器控制的小型步行机， [MountainBeest](http://hackaday.com/2014/04/14/mountainbeest-a-theo-jansen-creature-comes-alive-in-my-garage/) 是一项巨大(且沉重)的事业。两者都是木制的。这一次[Jeremy]订购了他的聚碳酸酯零件，这些零件已经被切割以匹配他的设计。但要去总装现场可不容易。

接受五金件的孔不够大，他不得不把它们铰出来，以便把所有的东西组合在一起。我们喜欢看他做夹具来固定扩孔用的垫片。他关于使用偏置滚销将驱动齿轮固定到电机轴上的建议我们会牢记在心。

最后，事情并不顺利。他加工出了一个马达联轴器，但对于驱动腿的扭矩来说，它太弱了。看着(诺姆·亚伯兰)建造家具(和房子)长大，没有一个吹剪或撕裂的末端纹理，这是一剂很好的现实。不是你每一步能做到多完美，而是你有多大能力预见问题并在遇到问题时纠正它们。
