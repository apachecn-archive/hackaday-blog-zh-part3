# 任何人都能造的 Tie 战斗机四轴飞行器

> 原文:[https://hack aday . com/2016/01/23/tie-fighter-quad copters-any-can-build/](https://hackaday.com/2016/01/23/tie-fighter-quadcopters-anyone-can-build/)

这些都是美丽的东西，当飞行时，Tie 战斗机四轴飞行器看起来甚至更好，因为旋转的叶片变得几乎透明。我们见过的大多数以《星球大战》为主题的四轴飞行器都是复杂的构造，我们知道你甚至不会去尝试。但是【拥抱玉米煎饼】[的创作在许多不同的方面适合每一个黑客](http://www.cuddleburrito.com/blog/2016/1/1/tie-fighter-conversion-h107c)。

首先，他从非常小的商用四轴飞行器开始，这些飞行器便宜(而且合法)，任何人都可以拥有和飞行。两者都是 Hubsan X4 的变体；H107C 和 H107L。这些四轴飞行器的储备臂从底盘的中心延伸出来，但这需要为 TFFF (Tie 战斗机外形)而改变。解决方案当然是 3D 打印。两种型号的[设计](http://www.thingiverse.com/thing:1136405)[已经在](http://www.thingiverse.com/thing:1242547)发布，应该很容易印刷。

ABS 被用作打印介质，这使得使用丙酮和 ABS 的浆料将接缝焊接在一起的组装变得容易。电机电线需要延长，并通过印刷臂布线，但除此之外，你不需要任何其他东西。在这个设计中，甚至重新使用了原来的螺钉。休息后看看视频中的试飞，至于我们提到的更多定制版本，还有无人机-enium Falcon 。
