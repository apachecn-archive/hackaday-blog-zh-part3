# 更多自制 PCB 镀锡

> 原文:[https://hackaday.com/2017/11/16/more-homemade-pcb-tinning/](https://hackaday.com/2017/11/16/more-homemade-pcb-tinning/)

[马尔科]自称是疯狂的化学家。他展示多氯联苯快速镀锡溶液的视频(YouTube，见下文)看起来并不疯狂。不过，我们承认，它使用了一些你可能需要搜索的东西。

这个公式需要氯化亚锡——你可以通过将锡溶解在盐酸中来制造氯化亚锡。还有硫脲——Tarn-x 等银清洗液中的主要化学物质。硫酸和去离子水是配方的补充。

不言而喻，如果没有正确的设置和良好的安全意识，你就不应该处理这些令人讨厌的化学物质。你应该知道混合水和酸的正确方法，并有合适的实验室设备和安全设备。[Marko]提到从电池中提取酸，但现在很难提取汽车电池内部的酸。幸运的是，在网上和当地很容易买到化学品——一些你不想运输的东西。

这与我们之前检查的[【nurdrage 的】公式](https://hackaday.com/2017/10/23/tinning-solution-from-the-hardware-store/)相似，但不相同。如果你有时间和技术诀窍，你可以自己制作真正具有专业外观的[板](https://hackaday.com/2015/02/25/diy-through-hole-plating-like-a-boss/)。
