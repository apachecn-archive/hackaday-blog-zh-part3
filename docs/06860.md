# 当你只有一个焊接头盔时，如何让自己黯然失色

> 原文：<https://hackaday.com/2017/08/20/how-to-eclipse-when-all-you-have-is-a-welding-helmet/>

不信任网上便宜的观日食眼镜怎么办？如果所有人都卖光了怎么办？好吧，如果你想看日食，并且你有一个自动变暗的焊接头盔，你可以像[daniel_reetz]那样做，用一个遥控器和你的焊接头盔一起做一些东西，让你看到日食而不会弄瞎你自己。

本质上，黑客欺骗头盔的传感器认为它非常亮。[Daniel_reetz]通过将一个带有红外 LED 的遥控器粘到头盔侧面，并用一个 50 毫米的塑料盖盖住它来实现这一点。[daniel_reetz]的头盔上有两个传感器，所以他用铝带盖住了另一个。这意味着，当他按下遥控器上的一个按钮时，盖着盖子的传感器认为它非常亮，因为另一个传感器被盖住了，它使面具的镜头变暗。

我相信我们的一些读者可以想出一个更复杂的方法，让你用手做其他事情，而不是按下遥控器按钮，但这是一个快速简单的方法，可以让你快速浏览日食——假设你有一个能够遮蔽 13 或 14 级的焊接面罩。如果你希望看一眼日食，看看美国国家航空航天局的安全指南，确保你的眼睛安全。对于观看日食的另一种方法，看看[这个可佩戴的针孔摄像机](https://hackaday.com/2017/08/18/catch-the-eclipse-with-a-wearable-pinhole-camera/)。对于另一个焊接面具黑客，看看[这个增强现实面具](https://hackaday.com/2012/09/11/augmented-reality-welding-mask/)。
