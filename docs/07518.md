# 天网会有山猫

> 原文:[https://hackaday.com/2017/10/26/skynet-will-have-bobcats/](https://hackaday.com/2017/10/26/skynet-will-have-bobcats/)

现在有太多的自主设备可以在 ^((TM)) 内部运行天网，很难保持跟踪。但是仍然缺少一个:多才多艺的山猫。当我们说“山猫”时，我们指的是履带式装载机，它只是名称和品牌紧密联系在一起的事物之一，以至于很难回忆起技术名称。一家名为 Built Robotics 的公司正在押注自动履带式装载机将成为建筑业未来的重要组成部分。

该拖拉机可以使用其激光雷达导航、挖掘和携带 1，000 磅的负载，精度为 1 厘米，专门设计用于高振动、高冲击的建筑挖掘环境。此外，激光还允许机器人测量它铲起的材料量。但是精度不仅仅来自激光雷达。为了定位机器人，Built Robotics 使用增强的 GPS，它结合了现场基站和 GPS 卫星，以产生准确的位置数据。

它应该是完全自主的:给定一个位置和要挖的洞，它可以计划和执行工作。它类似于自动驾驶汽车，但挑战实际上非常不同。汽车意味着在不接触任何东西的情况下四处行驶并到达目的地。就像机器人公司的首席执行官说的:

> “如果一辆车正在改变它周围的环境，那么一定是出了什么问题。”

相比之下，ATL(自动履带式装载机)就是为了做到这一点:改变周围的环境。幸运的是，没有早上的交通使任务复杂化。这是该公司的一个笑话:
