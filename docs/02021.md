# 飞翔的无限可能性驱动

> 原文：<https://hackaday.com/2016/04/11/flying-the-infinite-improbability-drive/>

自 1989 年的冷聚变混乱以来，流行科学媒体行业从未有过像 EmDrive 这样的故事。EmDrive 是一种无推进器的推进器——一种将射频能量转化为动力的设备。如果成功的话，它将彻底改变任何移动的技术。与使用化学物质、冷气、离子或等离子体的火箭发动机不同，配备 EmDrive 的航天器可以仅使用太阳能电池板在太阳系周围巡航。如果行得通，那就违背了已知的物理定律。

在世界各地的几个实验室(包括美国宇航局的先进推进物理实验室 Eagleworks)进行测试后，仅通过电力产生推力的设备的概念仍然没有被推翻、嘲笑和忽视。对于一个违反动量守恒定律的装置来说，这是很了不起的。一些实验的同行评审正在进行中，但[保罗]有一个更轰动的想法:[他正在建造一个 EmDrive，将推动立方体卫星](https://hackaday.io/project/10166-emdrive-flight-version)。

毫无疑问，我们目前对宇宙的理解与 EmDrive 完全不相容。发动机将微波能量注入金属锥体，并以某种方式产生推力，这种想法处于科学的边缘。没有一个理智的学术物理学家会从事这方面的研究，仅仅假设电磁驱动可能有效是不负责任的。在进一步的同行评审实验发表之前，EmDrive 只是一个疯子的异想天开。[也就是说，如果真的有效，我们会得到直升机航母](http://hackaday.com/2015/08/19/the-em-drive-might-not-work-but-we-get-helicarriers-if-it-does/)。安装在一辆特斯拉跑车上的四个电磁驱动装置将构成一辆气垫车。你的孙辈只会把地球的太阳看成夜空中的一个小点。

这并不是[保罗]第一次尝试创造一个可工作的无推进器推进器。为了去年的黑客日大奖，[【保罗】造了一辆小型 EmDrive](http://hackaday.com/2015/05/08/hackaday-prize-entry-an-em-drive/) 。与其他使用 2.4GHz 微波的 EmDrive 实验不同，[Paul]将他的引擎设计为工作在 22 至 26 GHz。这意味着[保罗]的要小得多，可以很容易地装进立方体卫星。如果成功，这个立方体卫星将能够无限期地保持其轨道，飞向月球并返回，或者去太阳系的任何地方，只要太阳能电池板获得足够的光线。

虽然[Paul]创造 EmDrive 的公民科学版本的动机值得称赞，但 Hackaday.io 自己的婴儿 EmDrive 并没有表现出如此规模的项目所必需的科学严谨性。实验设置定义不清，图轴没有标记，也没有足够的信息来正确评论[Paul]的婴儿 EmDrive 实验。

也就是说，我们不能责怪一个尝试的人，EmDrive 仍然是一个活跃的研究领域，有几篇论文正在接受同行评审。[Paul]将 EmDrive 送入轨道的计划是将车放在马的前面几英里，但对于今年的 Hackaday 奖来说，这仍然是一个非常酷的项目。
