# 破解迈克·泰森的打卡密码

> 原文：<https://hackaday.com/2016/11/11/crack-mike-tysons-punch-out-bang-bang-passwords/>

[Bisqwit]对使用感叹号的游戏很有感觉，他在著名的《冲出去》 Bang Bang 中对密码的所有细微差别进行了独特的[演练。](https://www.youtube.com/watch?v=ap1YL_kGGlg)

正如他在他的非常怪异(虽然没有错)的频道介绍中所说，当他不驾驶公共汽车或教以色列舞蹈时，他努力理解他周围的事情。自然，一个神秘的电话号码形状的一组数字在一个喜爱的游戏中是一个值得提取的秘密。

这些数字可以代表游戏中每一种可能的状态。它使用一个非常简单的解码和编码方案，他走过。正如他所说，当你看到源代码时，一切都变得清晰了。

在解决了所有的怪癖之后，他能够在游戏中任意生成任何状态并处理异常(比如任天堂美国的电话号码)。你可以在这里看到他所有的[代码，并亲自尝试一下。休息后的视频。](https://github.com/bisqwit/password_codecs)

我们已经开始尊重[Bisqwit]作为所有主机游戏的解释者。你会喜欢他对如何为 NES CPU 编写代码模拟器的解释。
