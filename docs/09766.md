# 在地下室建造一个超轻型飞机看起来非常漂亮

> 原文:[https://hack aday . com/2018/06/16/building-an-ultra light-in-a-Basel/](https://hackaday.com/2018/06/16/building-an-ultralight-in-a-basement/)

[彼得]又开始了。不满足于成为 YouTube 上最好的遥控飞机专家之一，当然也不满足于他在地下室制造的第一架飞机，[【彼得·斯里波尔】正在他的地下室制造另一架飞机](https://www.youtube.com/watch?v=QIpUNlEWogY)。

他建造的第一架飞机[在 YouTube](https://hackaday.com/2017/11/01/how-to-build-an-airplane-in-a-month-and-a-half/) 上被记录了一个半月。这是一架全电动双翼飞机，由玻璃纤维覆盖的绝缘泡沫制成，由一对通常用于大型遥控飞机的超大电机驱动。这是根据第 103 部分法规建造的超轻型飞机，这意味着实际上没有任何法规。任何人都可以在没有执照的情况下爬进其中一架飞机并驾驶它。飞机飞了起来，但是出现了一些问题。它太快了，电池寿命也不是[彼得]真正想要的。

现在[彼得]开始了他的下一次冒险。与以前的飞机相比，这种飞机的结构更简单、更传统。这是一架铝制框架的高翼单翼飞机。虽然他仍在寻找低压电机，但还是有两台电机。这架飞机还应该飞得更慢，更长，这是你真正想要的超轻型飞机。

至于建造所需的工具，令人惊讶的是组装飞机所需的工具如此之少。当然，有一些过大的流行铆钉枪和一些额外的特殊航空级螺栓，但这架飞机的大部分将由标准铝，绝缘泡沫，一点木头和一些玻璃纤维制成。看着[彼得]用这些简单的零件制造出高端产品是如此令人满意。如果你有一台带十字滑动虎钳的钻床，你也可以在地下室造一架飞机。

这将会是一个非常棒的建筑。[Peter]已经证明了，是的，他确实可以在他的地下室里造一架飞机。不过，这一次，他将驾驶一架能在空中停留几分钟以上的飞机。
