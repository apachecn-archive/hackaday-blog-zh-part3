# 先睹为快的 TS100 烙铁的弟弟妹妹

> 原文：<https://hackaday.com/2018/06/27/a-sneak-peek-at-the-ts100-soldering-irons-younger-sibling/>

许多读者都熟悉 TS100 烙铁，这是一种重量轻、功能强大的工具，手柄中集成了基于 STM32 微控制器的温度控制器。作为一个铁杆，使用它是一种乐趣，它有可破解的代码，并且它已经成为我们社区中的一个坚定的最爱。有一段时间以来一直有 TS100 stablemate 的传闻，型号被吹捧为 TS200，据说它是 USB-C 供电的。但是除了这些花边新闻之外，到目前为止还没有太多的线索。

所以[Marco Reps]' [我们放在断裂处](https://www.youtube.com/watch?v=83STpUXHilk)下面的视频是一个特别有趣的视频，因为它展示了有问题的铁的原型。它被称为 TS80，但在其 PCB 上有证据表明，它过去曾拥有 TS200 的绰号，它是 USB-C 供电的，它具有一个新的集成加热元件和带有韦勒式 3.5 毫米插孔连接器的钻头。

他对它进行了一系列测试，发现它的性能确实非常好，有时甚至比 TS100 还要好，尽管他没有 USB-C 电源能够提供与他的 TS100 通过 DC 插孔获得的电压相同的电压。需要说明的是，TS100 确实仍然是一个非常好的熨斗，这个只是稍微好一点。在一个更坚固的金属桶内是一个 PCB，板上有 STM32 以及一个有机发光二极管显示器，看起来比 TS100 上的略小。较短的元素受到称赞，而 TS100 几乎不是长熨斗，尽可能靠近动作总是好的。

有人担心 DC 插孔的缺乏及其对 USB-C 的依赖，尽管他指出，随着合适的电缆和 USB-C 的日益普及，这应该不会成为长期问题。我们有兴趣确保它可以通过 USB-C 插座从简单的 DC 电源(如电池)供电，因为这种灵活性是 TS100 的一大优势。

那么，TS80 来了，但 TS100 仍然是一个非常好的铁，所以没有必要马上扔掉你的。这是一个铁，我们期待着看到它的到来，毫无疑问，我们会给你我们的裁决。

如果你喜欢，你可以看看[我们的 TS100 评论](https://hackaday.com/2017/07/24/review-ts100-soldering-iron/)，同时，看看[这个社区最棒的 TS100 黑客之一](https://hackaday.com/2017/09/28/part-soldering-iron-part-hand-held-oscilloscope/)。[Marco]思考着人们的 TS80 多久才能通过 3.5 毫米插孔播放音频。
