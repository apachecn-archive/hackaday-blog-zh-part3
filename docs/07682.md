# 少于生产，多于一次性

> 原文:[https://hack aday . com/2017/11/14/少于生产多于一次/](https://hackaday.com/2017/11/14/less-than-production-more-than-one-offs/)

我们建造我们认为很酷的东西。有时候，其他人同意我们的观点，他们想要一份我们的成果。如果你足够幸运，有一个令人羡慕的产品，但你还没有准备好大规模生产，你可能会看到小批量生产。[Eric Strebel]带我们看了一个这样的例子，他为一场秀制作了一些定制色板。休息后的视频。

[Eric Strebel]是一名工业设计师，因此他通过设计样本形状、夹具、工具和衣架来发挥自己的优势。他出租油漆、激光切割和数控加工。这似乎是一个简单的声明，但我们中的一些人很难为我们能够学习的东西付钱给别人。在某些情况下，我们只需要付钱给专业人士，让他们正确地做这件事，并保持我们的注意力。

对于那些在 Tindie 上卖东西的人或者那些想制造更多持续的好零件的人来说，这个视频中的小批量生产的想法是完美的。我们自己的[[勒温·戴](https://hackaday.com/author/lewinday/)最近讲述了他与 200 个被称为 gMix 的[混音器打交道的经历。](https://hackaday.com/2017/10/25/a-passive-mixer-from-concept-to-shelf/)
