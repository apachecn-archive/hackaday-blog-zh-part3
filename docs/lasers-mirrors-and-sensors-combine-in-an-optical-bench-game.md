# 激光、镜子和传感器结合在一个光学工作台游戏中

> 原文：<https://hackaday.com/2018/03/24/lasers-mirrors-and-sensors-combine-in-an-optical-bench-game/>

谁会想到你可以用光学平台做游戏呢？[克里斯·米歇尔]做到了，虽然我们一开始持怀疑态度，但他的[激光弯曲游戏](https://hackaday.io/project/92766-light-bender-a-laser-cut-laser-game-i-made)有一些潜力。小心你的眼睛。

前提很简单:在时间用完之前，将彩色激光光束导向正确的目标。[Chris]在他的操场上使用了激光切割的丙烯酸树脂，有九个正方形的格子图案。红色、绿色和蓝色激光笔排列在网格的底部，光电传感器和 RGB LEDs 排列在网格的其他三侧。游戏开始时，随机点亮三种颜色中的一种，作为目标。相应的彩色激光亮起，玩家必须在网格中插入镜子或直通块，以创建一条通往目标的路径。你越快击中 CdS 细胞，你的分数越高。很简单，但是看起来很吸引人。我们可以想象各种各样的升级，比如同时照亮两个不同的目标，或者添加一个分束器块来用相同的颜色击中两个目标。滤光器和偏振器也可以增加视觉乐趣。

我们喜欢只是为了好玩的构建，尤其是当它们制作精良并且有一点危险的时候。我们最近报道的气球爆破机器人项目让我想起来了。
