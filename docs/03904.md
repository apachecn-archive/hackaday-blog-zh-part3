# 一片 Ubuntu

> 原文:[https://hackaday.com/2016/10/20/a-slice-of-ubuntu/](https://hackaday.com/2016/10/20/a-slice-of-ubuntu/)

Raspberry Pi 操作系统事实上的标准是 Raspbian——一个基于 Debian 的发行版，专门用于小型计算机。当然，你有多种选择，可能没有一个最佳选择适合每种情况。然而，RaspEX 项目[为 Raspberry Pi 2 和 3 发布了一个可行的 Ubuntu 16.10](http://raspex.exton.se/)版本，这的确吸引了我们的眼球。

RaspEX 是一个完整的 Linux 桌面系统，带有 LXDE(一个轻量级桌面环境)和许多其他有用的程序。存在 Firefox、Samba 和 VNC4Server。你可以使用 Ubuntu 库来安装任何你想要的东西。系统使用内核 4.4.21。你可以在下面的视频中看到一个更老版本的 RaspEX 的评论。

如果你习惯使用 Ubuntu，这可能是一个很大的方便。该项目声称该系统比普通操作系统快得多，尽管它没有解释为什么会这样。

话说回来，有些树莓 Pi 板[根本不需要操作系统](https://hackaday.com/2016/10/12/raspberry-pi-boots-cpm/)。或者，你可以试试谷歌的词条[。](https://hackaday.com/2016/08/17/googles-new-os-will-run-on-your-raspberry-pi/)
