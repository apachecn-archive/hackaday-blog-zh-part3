# 这是你爷爷的收音机

> 原文：<https://hackaday.com/2018/02/14/this-is-your-grandfathers-radio/>

电子管收音机有某种魅力。等待它们变暖，黑暗房间里灯丝的光亮。电子管统治了无线电几十年。[Uniservo]发布了一个关于 20 世纪 20 年代背后的历史和技术的视频[克拉普-伊斯特汉姆 C-3 收音机](https://www.youtube.com/watch?v=dmqT-Jt4cx4)。这是一个三管再生接收器，是先进的一天。

如果你担心他不会打开它，不要绝望。在十分钟左右，你的耐心会得到回报。里面是三个装满吸气剂和汇流条而不是电线的大管。再加上家具质量的情况下，这是一个伟大的老收音机。

关于这个接收器的一个有趣的事情是，它使用了一种特殊的变压器，称为可变耦合器，其中一个线圈在另一个线圈内旋转，以调节再生。原来这些电子管比收音机还新，所以[Uniservo]用更适合年龄的电子管替换了它们。

不幸的是，收音机现在是无声的，因为打开了音频变压器。我们希望他能让它正常工作，并制作另一个实际运行的视频。

再生接收机具有相当好的放大性能，而器件数量很少。这是因为放大器工作在振荡附近，所选频率的增益非常高。使用比这些电子管稍新的技术来制造你自己的 T1 是相当容易的。如果你想[深入理论](https://hackaday.com/2017/02/01/everyone-should-build-at-least-one-regenerative-radio-receiver/)，我们也已经做到了。
