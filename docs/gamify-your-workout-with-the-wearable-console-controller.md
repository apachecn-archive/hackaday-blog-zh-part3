# 使用这款可穿戴式控制台控制器，让您的锻炼变得更有趣

> 原文：<https://hackaday.com/2017/11/28/gamify-your-workout-with-the-wearable-console-controller/>

很快就要到了决心动摇的季节，用最好的意图购买的锻炼器材在沮丧中被扔在一边。但是，只要有一点点动力，比如让你的健身机成为游戏控制台控制器，你就可以最大化你的健身装备投资，并且玩一些没有负罪感的游戏。

老实说，没有比上课更好的动机来坚持锻炼了，但是没有多少人有纪律或者钱包来长期坚持去健身房。考虑到这一点，[Jason]寻找一种方法来用他的卧式自行车控制 PS4 游戏，如 Mario Karts 或 TrackMania。为了避免改装自行车，[Jason]决定为他的脚踝安装一个可穿戴式运动传感器。该可穿戴设备由 Uno、MPU9250 加速度计和 433 MHz ISM 频段的发射器组成，每当脚在移动时，它就会向接收器发送信号。这模拟了按下向上箭头控制键来启动游戏。转向和其他游戏动作由常规控制器处理；我们很乐意看到这种扩展包括横卧自行车手柄上的应变仪，以允许通过转移座位上的重量进行左右控制。说说沉浸式的玩法吧！

虽然我们喜欢[Jason]构建的简单性和它提供的积极强化，但这远不是我们看到的第一个健身机黑客。从[制作谷歌街景自行车控制的](https://hackaday.com/2010/10/29/exercise-along-to-google-street-view/)到[自动记录锻炼](https://hackaday.com/2012/03/03/easy-workout-logging-with-ethernet-enabled-exercise-bike/)，锻炼机器被黑客攻击的时机已经成熟。
