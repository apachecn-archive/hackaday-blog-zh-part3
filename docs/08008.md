# Hangprinter 构建视频

> 原文：<https://hackaday.com/2017/12/16/hangprinter-build-videos/>

我们认为，随着 3D 打印的兴起，是时候制定一个新的欺骗法则了:你想打印的任何部分都不适合你的打印床。曾经有一段时间，100 mm x 100 mm 的床对于入门级打印机来说很常见。如今，更多的打印机都有大约(200 mm) ² 的床。悬挂式打印机的工作区域可以更大。大得多。[Thomas Sanladerer]正在建造一个，[有一系列关于建造](https://www.youtube.com/watch?v=vckxUk8xISc)的视频。你可以看到下面的第一个，但有几个张贴，包括约 11 小时的现场会议的建设录音。

如果你没有听说过悬挂式打印机，它本质上是一个 3D 打印头，它悬挂在电缆上，可以将整个房间变成 3D 打印机。当我们看原件时，它正在打印一个五英尺高的巴别塔模型。

hangprinter 忠实于它的 RepRap 根源，有许多自我打印的部分，和一个开放源码的设计。即使你不想建造一个，观看视频也能让你了解一个是如何工作的。

如果你想要更传统的东西，我们几个人已经在网上黑了。有时候你不在乎一个零件有多少体积，但你在乎[有多少体积来制造多个零件](https://hackaday.com/2017/06/06/the-ip-of-the-infinite-build-volume-3d-printer/)，但那是不同的问题。
