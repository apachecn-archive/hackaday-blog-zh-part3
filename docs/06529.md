# 电台司令为 ZX 谱创作的最受欢迎的歌曲

> 原文:[https://hack aday . com/2017/07/16/radioheads-great-hits-for-the-zx-spectrum/](https://hackaday.com/2017/07/16/radioheads-greatest-hits-for-the-zx-spectrum/)

我们承认，在 Hackaday，只有少数人是电台司令的粉丝。然而，我们都忍不住欣赏他们新灌制的 OK 电脑。新的版本包含一些额外的材料。在奖励材料的最后是一个奇怪的噪音，原来是一个 ZX 频谱基本程序。[OooSLAJEREKooO]设法找到它，播放它，[为我们所有人录制它](https://www.youtube.com/watch?v=JDM5YBS6KMc)(见下面的视频)。

两分钟的音调对于现代计算机用户来说可能听起来很陌生，但在过去，音频音调用于通过电话线进行通信，并通过盒式磁带录音机加载和保存程序。你可能会问自己:为什么是 ZX 光谱？电台司令来自英国，但这并不是全部情况。在所有家用电脑中，ZX 频谱在磁带上存储数据时具有更高的有效比特率。基本上，它比 C64 或苹果更节省时间(和磁带)。

电台司令没有发明这个花招。唱片作为音频偷偷放入软件已经有很长的历史了，包括 1983 年[Frank Sidebottom]的一首单曲，当时他还叫[Chris Sievey]，里面有一个辛克莱·ZX 的动画“音乐视频”(见下面的第二个视频)。

你可能会认为这是电台司令第一次出现在 Hackaday 上。毕竟，我们不是真正的音乐网站。但是你错了。如果你身边没有 ZX 频谱，你可以随时去[FPGA 路线](https://hackaday.com/2017/01/12/retro-zx-spectrum-lives-a-spartan-existence/)。
