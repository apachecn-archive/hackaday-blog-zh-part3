# 金属胶会取代焊料吗？

> 原文:[https://hack aday . com/2016/01/17/will-metallic-glue-replace-solder/](https://hackaday.com/2016/01/17/will-metallic-glue-replace-solder/)

一段视频最近浮出水面，兜售一种将材料连接在一起的新方法。它被称为 [MesoGlue](http://motherboard.vice.com/read/will-this-fancy-metallic-glue-kill-soldering) ，显然，它可以在某些情况下取代锡焊甚至焊接。

先公布本月的 *[先进材料和工艺](http://mio.asminternational.org/amp/201601/files/assets/common/downloads/AMP_DigitalEdition_NovDec.pdf)(小心，大文件！)*似乎……合法。基本前提是，它使用纳米棒材料——有点像 velcro——一旦你推在一起，相互缠绕，并成为固体。它们被一层液化的外壳包裹着，这就巩固了结合。这使得它能够承受高温，一旦结合。

这种技术可能会对我们连接焊接电路、管道、将 IC 连接到散热器以及连接具有不同热膨胀系数的不相似材料的方式产生影响。

虽然我们不是材料科学家，但因为它确实出现在 ASM 的《先进材料和工艺》国际刊物上，这可能是真的。
