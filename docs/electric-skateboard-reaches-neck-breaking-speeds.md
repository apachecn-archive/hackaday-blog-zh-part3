# 电动滑板达到惊人的速度

> 原文：<https://hackaday.com/2016/03/10/electric-skateboard-reaches-neck-breaking-speeds/>

加拿大速度狂米肖·板儿刚刚打破了电动滑板的世界纪录。**59.55 迈！**差不多是 100 公里/小时。

这些年来，我们报道了很多电动滑板，以及一些商业版本——比如 Boosted Board，Kickstarter 为数不多的成功案例之一——当然，人们[也黑过它们。](http://hackaday.com/2015/11/10/adding-range-and-bling-to-an-electric-skateboard/)但这款来自[的下一代车辆](http://nextboards.com/) (NGV)的板，似乎已经将速度提升到了一个新的水平。

该板由斯洛文尼亚一家科技初创公司制造，其特点是定制车轮内置直接驱动电机。这篇文章在细节上有点轻，但我们想象它们必须每个几千瓦，以达到这些速度。没有提到范围(我们无法想象在这样的速度下会有多远)，但这只是一个原型。

作为四轮驱动和智能电机控制器，它即使在高速下也是一个非常稳定的系统——尽管如果你看视频，他确实在某一点上摔了一跤。
