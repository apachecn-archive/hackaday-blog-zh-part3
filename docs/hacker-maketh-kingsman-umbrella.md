# 黑客制作金刚伞

> 原文：<https://hackaday.com/2017/09/15/hacker-maketh-kingsman-umbrella/>

是啊！有人做了金曼伞，是的，它可以射击，是的，它里面有显示屏。[James Hobson]刚刚在 YouTube 上为这个优秀的项目发布了一个视频，详细介绍了创建这个现场工作道具的过程，这太棒了。

建造从找到一把结实的伞开始，通过站在伞上测试，同时也测试了一些家用物品。压缩二氧化碳子弹为推进飞镖以及其他非致命形式的弹药提供燃料。该项目的最酷的部分是便携式电脑内的屏幕，它可以让你透过穹顶看到东西。这是通过一个小型摄像机和一个便携式微型投影仪的组合来实现的。简单但很棒。

摄像机安装在枪口附近，而投影仪被切片并集成到手柄中。有问题的手柄本身是 3D 打印的，并在设计中包括一个自定义触发器。查看该项目的演示视频。

电影道具在每个制作者的心中都有着特殊的地位，这个项目是想象力与独创性相遇的绝佳范例。看完这个视频后，安全机构会给伞的主人一些怀疑的眼神，尽管创造自己的伞会是一次非常有益的经历。如果你正在寻找一个更明显的道具，那么看看《辐射》中的 [PiPBoy 终端，它肯定会引起每个人的注意。](https://hackaday.com/2017/03/25/replica-fallout-terminal/)
