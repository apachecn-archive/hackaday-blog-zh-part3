# 最强大的 DIY 轨道炮

> 原文：<https://hackaday.com/2015/11/09/the-most-powerful-diy-railgun/>

美国海军正在进行几个轨道炮项目，最终将取代舰队巡洋舰和驱逐舰上最大的火炮。这些轨道炮将以大约 7 马赫的速度从船上发射一枚射弹，射向一百英里外的目标。这是一门比装有核弹头的枪更令人印象深刻的大炮，总有一天，它会成为现实。

在那之前，我们只能用[【Zebralemur】的 DIY 移动轨道炮](http://imgur.com/a/GrAiE)来解决。他建造了这种轨道炮，能够发射铝射弹穿过南瓜、手机、车门和弹道凝胶块。

所有轨道炮都需要一个储存能量的地方，在所有情况下，这都是一个巨大的电容器组。在这个项目中，[Zebralemur]使用了 56，400 伏特，6000 微法的电容。这些帽子的建议零售价总计约为 50，000 美元，但不知何故——可能是一家剩余商店——[Zebralemur]以 2，400 美元的价格买到了它们。

这些盖子只是轨道炮的电源，并不是这个已经很大的 250 磅炮的结构的一部分。幸运的是，在[Zebralemur]的车里放下座椅，它们可以放在他掀背车的后座上。

这些帽子由一堆首尾相连的 9V 电池充电。当雷管充电时，所有的能量都被注入枪中的两根铜棒，加速铝弹到足以致命的速度。这是一个不可思议的构建，但是任何人都不应该尝试。尽管今年看起来是所有危险寻求者都开始使用电磁投射投掷器的一年。
