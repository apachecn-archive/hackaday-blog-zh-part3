# 激光像魔法一样去除铁锈

> 原文:[https://hack aday . com/2016/03/17/laser-removes-rust-like-magic/](https://hackaday.com/2016/03/17/laser-removes-rust-like-magic/)

如果你曾与钢或铁打交道，你会对铁锈非常熟悉。你将有一个令人印象深刻的钢丝刷和化学品军械库来处理它，你的喷砂家伙可能在你的快速拨号列表中。

最近，有不止一位黑客读者联系我们，向我们展示了一个[看似神奇的手持式激光装置毫不费力地去除锈迹](https://www.youtube.com/watch?v=CLaBFkeHG0A)，并留下几乎完美的表面，几乎没有一点脏乱。他们问，这是真的吗？这是网络骗局吗？毕竟，如果你和可怕的氧化铁打过交道，你就会知道这个问题没有灵丹妙药，但是你如何处理它，传统上需要付出艰苦的努力。

因此，经过一番调查，我们找到了德国公司 CleanLaser，其产品出现在视频中。引用他们网站上的话:“强大、极短、快速和移动的激光脉冲产生微等离子体爆发、冲击波和热压，导致目标材料升华和喷射”。所以是的，看起来它们是真的。

该网站煞费苦心地强调这种设备相对于类似的喷砂或类似技术的环保优势，但关于其安全性的信息却很少。它们的额定功率从 12W 到 1KW 不等，这对于投射来说是一个很大的激光功率，但操作者似乎只戴了护目镜。也许这又回到了上面引用的“强大，非常短，迅速和移动”的一点，难道没有点源灼伤你的视网膜吗？激光专家请在评论中开导我们。

如果你从事金属工作或在金属加工行业长大，这台机器可能会让你垂涎三尺。遗憾的是，对于黑客和制造商来说，它和类似的东西可能会在相当长的一段时间内超出我们的价格范围。尽管如此，在大多数城镇的工业单位中，一个有手机的人出现的前景不会太遥远，这只能是一件好事

录像显示机器在运转。生锈的壁炉，完美闪亮的表面。也许只有那些花了很多时间用钢丝刷的人会明白。
