# Hackaday 奖参赛作品:无线气象站保护农作物免受霜冻

> 原文：<https://hackaday.com/2017/07/07/hackaday-prize-entry-wireless-weather-station-protects-crops-from-frost/>

似乎你每年都会听到这种说法——或早或晚的霜冻威胁着一些作物，迫使农民采取激烈的行动以避免经济损失。但是，即使天气在大范围内合作，小气候仍然可以在足够小的区域引起大问题，直到损害发生后才被注意到。

一如既往，更好的数据可以带来更好的决策，环境数据粒度的增加可以为某些种类的农业创造奇迹。进入 [SLoRa，一个农业无线气象站](https://hackaday.io/project/25336-slora-wireless-weather-station-for-agriculture)。旨在提供一个廉价、低功耗温度传感器网络的[Dorijan]提出的系统将允许农民采取积极措施保护他们的树木免受霜冻——涂抹盆来加热新生的水果，洒水器来覆盖一层保护层，甚至悬停的直升机来将大量温暖的空气转移到寒冷的地方。通过太阳能传感器阵列和连接到山顶网关的 LoRa 链接，每个部署的 SLoRa 传感器将成为农民可以用来确定在哪里部署其有限资源的又一个数据点。

需要了解 LoRa 的最新进展吗？你可以做得比学习如何用 SDR 加密狗监听 LoRa 信号更糟。
