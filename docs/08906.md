# NES 控制器基于滑块的灯

> 原文：<https://hackaday.com/2018/03/20/nes-controller-slider-based-light-theremin/>

从未使用过 555 的[lonesoulsurfer]决定[他对世界上最流行和最通用的 IC 的第一次尝试将是使用 555 来制作美丽的芯片曲调](https://www.instructables.com/id/Slider-Synth-Light-Theremin-555-IC/)。为此，我们赞扬他。他找到了[【迪恩·塞戈维斯】【Slidersynth】基于光的特雷门琴](https://hackaday.com/2011/02/28/fun-slide-whistle-synth-toy/)，并着手构建他自己的版本，并把它塞进一个(山寨！)NES 控制器。

对于外行来说，特雷门琴是一种无接触合成器，它利用人体电容和一对天线来控制振动和振幅。在这种基于光的 Theremin 中，振荡是由白光 led 的光子强度及其与光敏电阻的相互作用控制的，光敏电阻也称为光电池或“弯曲电阻”。

振荡本身是通过将 555 作为一个不稳定的振荡器连接起来而产生的，音高由安装在背面的电位计控制。它有一个小型内置扬声器，但[lonesoulsurfer]用 3.5 毫米音频插孔取代了 B 按钮，这样他就可以将其插入有源扬声器，真正地摇滚起来。休息之后我们已经准备好了他的试听带。

我们喜欢这里的袖珍乐器。如果你喜欢铜管乐器和木管乐器，[这个袖珍木管乐器 MIDI 控制器](https://hackaday.com/2017/11/13/pocket-woodwind-midi-controller-helps-you-carry-a-tune/)可能会把你的嘴唇画成 o 形。
