# 字幕显示使用六打多余的 VFD 管，效果很好

> 原文：<https://hackaday.com/2018/04/19/marquee-display-uses-six-dozen-surplus-vfd-tubes-to-great-effect/>

看来，将多余零件重新用于新的有趣展示的探索永无止境。显示屏越大越好，使用一些真正不起眼的部分可以加分，比如[这些多余的俄罗斯真空荧光灯管变成了一个大显示屏](https://imgur.com/a/w73hh)。

正如[tonyp7]坦率承认的那样，这是一个宠物项目，只是为了好玩，这些天市场上过剩零件的泛滥使之成为可能。VFD 电子管是 IV-25，俄罗斯电子管，通常来源的歌曲都可以买到。显像管中的 7 个小元件本来是用来制作像 VU 仪表那样的条形图显示器的，但是[tonyp7]把 12 个并排放在一起制成了 84 像素的显示器。每个矩阵的定制驱动板需要三个 40 引脚 dip 的旧 SN75518 驱动芯片。一个 3D 打印的支架支撑着每个模块的管子和电路板；看起来目标是一个时钟，六个模块组合在一起。但是下面显示的字幕也很棒，我们期待看到完成的项目。

从用 led 制成的假显示器到由继电器逻辑驱动的翻转显示，我们真的很喜欢独特显示的趋势。你在想什么？
