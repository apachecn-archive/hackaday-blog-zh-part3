# 手掌中的印刷电路

> 原文：<https://hackaday.com/2018/05/05/printed-circuits-in-the-palm-of-your-hand/>

如果你曾经想让自己更全面地融入技术，你可能不得不感谢明尼苏达大学的一个研究团队，在不久的将来，这个团队由[迈克尔·麦卡尔平]领导。他们已经开发了一种技术，可以将电路直接打印到你的皮肤上，该团队认为，一旦这种低成本打印机被改造成紧凑的便携性，它将是“飞行中”电路需求的理想选择。

“但是手并不完全是一个稳定的打印床，”你说。我们听到了您的声音，团队实际上领先了一步——打印机可以通过跟踪放置在手上的标记来补偿打印过程中的细微移动。这种“细丝”由银片制成——类似于[导电墨水](https://hackaday.com/2012/02/24/diy-conductive-ink-lets-you-freehand-circuits-on-the-cheap/)——在室温下打印和固化，可以剥离或洗掉。我们应该希望如此，因为它是要覆盖在人类皮肤上的。

说起来它还能打印细胞！

到目前为止，它只在老鼠身上进行了测试，但允许打印机精确跟踪手的相同技术意味着它可以使用[生物墨水](https://hackaday.com/2013/01/25/build-a-bioprinter-from-very-old-inkjet-cartridges/)直接向伤口或其他表皮疾病添加细胞，以帮助加快愈合过程。

然而，对于电路来说，你仍然需要其他电路元件和一个紧凑的装置来为它们供电——更不用说如果电路是水溶性的，那么一点点汗水就会导致墨水流动。我们很期待这项技术的发展！
