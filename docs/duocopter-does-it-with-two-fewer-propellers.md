# Duocopter 少用了两个螺旋桨

> 原文：<https://hackaday.com/2017/07/25/duocopter-does-it-with-two-fewer-propellers/>

四轴飞行器是一个伟大的直升机设计。成对的叶片抵消彼此的扭矩，改变四个马达的相对速度使其易于转向。但是，如果你可以用更少的叶片，替代一个更好的控制算法，那会怎么样呢？

[Dirk Brunner]的 [DuoCopter 无人机](http://www.research-drone.com/en/DuoCopter.html)使用两个反向旋转的螺旋桨，它通过在一次旋转中增加和降低叶片旋转的速度来转向。一边比另一边旋转得快会使它倾斜。说这是一回事，但让实时控制算法启动并运行是另一回事。从下面嵌入的视频来看，看起来[德克]已经成功了。(他还保持着四轴飞行器最快上升速度 FWIW 的世界纪录。)

当然，你们中的一些人不会满意，直到你们的直升机只有一个螺旋桨。或者也许你[更喜欢第三个道具。](http://hackaday.com/2017/05/26/the-tri-rotor-drone-why-has-it-been-overlooked/)无论您的品味如何，我们很高兴看到人们不断突破直升机设计的界限。
