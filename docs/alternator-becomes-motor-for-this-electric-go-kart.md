# 交流发电机成为电动卡丁车的马达

> 原文：<https://hackaday.com/2016/08/14/alternator-becomes-motor-for-this-electric-go-kart/>

成长于 70 年代和 80 年代的卡丁车是通向酷的捷径，或许仅次于迷你自行车。在这两种情况下，一个焊接钢管框架和一个废弃的割草机引擎都是你和邻里荣耀之间的障碍。看起来几个工程系学生抓住了复古少年犯的错误，为他们的最终项目建造了[这辆电动卡丁车](https://hackaday.io/post/43064)。

虽然[Adrian Georgescu]和[Masoud Johnson]的车架是二手的，但动力系统都是定制的。他们的目标是 3 千瓦的功率输出，但没有发现在这个范围内负担得起的电机。因此，在真正的黑客时尚中，他们用二手斯巴鲁交流发电机制作了自己的马达。三相电机控制器来自电动滑板车，三个 LiPo packs 提供动力，一对 Arduinos 负责油门控制、速度感测和向 Android 手机上的虚拟仪表盘发送数据。一些灯光和漂亮的红黑油漆完成了建筑。虽然下面的视频显示加速并不完全是特斯拉风格的脖子折断，但电子卡丁车可以达到很好的速度——53 公里/小时。不太破旧，没有震耳欲聋的发动机就在你的身后。

如果你有卡丁车的 bug，最好看看我们以前的一些帖子，比如[这款卡丁车由现成的组件](https://hackaday.com/2014/05/22/electric-go-kart-made-from-off-the-shelf-components/)制成，或者[这款四轮驱动迷你卡丁车](https://hackaday.com/2012/05/01/drop-everything-and-build-this-go-kart-right-now/)。不管你用什么方法建造它，你都会统治这条死胡同。
