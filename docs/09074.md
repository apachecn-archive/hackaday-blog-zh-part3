# 拖地机器人让清洁更上一层楼

> 原文：<https://hackaday.com/2018/04/08/floor-mopping-robot-takes-cleanliness-to-the-next-level/>

虽然让机器人帮你打扫地板很好，但吸尘器只能清洁你的房子这么多。对于真正的深度清洁，你还需要在坚硬的地板表面上运行拖把。[Josh]将这一想法进行了合理的总结，并为他参加今年的 Hackaday 奖制作了一个能够真正擦洗地板的机器人。

这个机器人能够用清洁剂喷洒地板，然后开过去，用橡胶扫帚擦洗地板。它的设计允许它进入狭窄的角落而不需要特殊的刷子，当然它拥有其他机器人所拥有的所有功能，例如超声波传感器，碰撞检测，以及允许它导航路线并清洁整个区域的大脑。

在项目网站上有许多机器人运行的视频，显示了它的操作和测试设备的各种功能。这是一个非常有趣的家用机器人，由于它的开放硬件，如果你经常因为不得不拖自己的地板而感到沮丧，你可以制作自己的机器人，或者你可以修改它来做除了清洁地板之外的事情。
