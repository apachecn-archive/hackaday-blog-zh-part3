# 苏联时代的坦克得到了 3D 打印的待遇

> 原文：<https://hackaday.com/2016/07/18/soviet-era-tank-gets-the-3d-printed-treatment/>

3D 打印机因其替换丢失或损坏零件的能力而闻名。一整辆 [T-62 坦克](http://www.gambody.com/blog/gambody-present-3d-printed-t-62-press-release/)怎么样？

等一下，这只是一个模型复制品。然而，这是 3D 打印机无数用途的另一种表达。由 Maya 设计，需要将近三周的时间从 70 米长的 PLA 细丝上打印出所有 62 个零件。组装并不十分复杂，通过打印一些大的部分，如船员部分和船体，而这些部分不会比炮塔舱口小得多，从而使组装变得更容易。尽管如此，当你从头开始设计零件时，最终产品是你能得到的最真实的生活。

坦克的型号[可以在这里](http://www.gambody.com/3d-models/t62a-middle-tank)购买 STL 格式的。

火车模型、轮船模型、人物模型等等为业余爱好者提供了数千种不同的产品，但对于那些有特殊兴趣或品味的人来说，3D 打印机——结合建模程序中的一些技能——将填补你的空缺。而且，如果你有一些严肃的灵感，来自*的*遥控蜘蛛坦克怎么样？
