# 活体 3D 打印机灯丝

> 原文:[https://hackaday.com/2017/12/22/living-3d-printer-filament/](https://hackaday.com/2017/12/22/living-3d-printer-filament/)

这不仅仅是打印细丝黑客——更接近前沿生物工程——但我们不禁被 3D 打印的前景迷住了，这种 3D 打印使用的是在细胞水平上有生命的细丝。

由赵宣和(音译)和 Timothy Lu(音译)领导的麻省理工学院的研究小组已经对细菌细胞进行了编程，使其对特定的化合物做出反应。为了证明这一点，他们打印了一棵树的临时纹身，这种纹身由坚固的细菌和载有营养物质的水凝胶“墨水”组成，当附着在用这些特定刺激擦拭的皮肤上时，这种墨水会在几个小时内发光。

到目前为止，该团队已经能够生产几厘米大的物体，能够在打印时适应活性材料，并集成为可穿戴设备、显示器、传感器等。

使用与这种“活墨水”一起开发的预测模型，设计响应性活材料，其中每个印刷品的不同区域能够在输入和输出信号发光的方式之后进行通信。将这一点发挥到极致，一个遥远的未来目标将是打印“活”电脑。

在那之前，成为廉价的生物黑客已经成为可能。
