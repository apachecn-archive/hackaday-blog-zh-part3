# 高保真贺卡取样器

> 原文：<https://hackaday.com/2016/12/24/lo-fi-greeting-card-sampler/>

我们都熟悉记录自己信息的贺卡。一般来说，它们只不过是朋友生日时的一个可爱的噱头，但是[dgenigt]发现这些[卡片有更大的潜力](http://www.dogenigt.com/2013/02/voice-message-greeting-cards-turned.html)。

从易贝采购了几个便宜的模块后，第一件事就是用 DC 的电源替换手表电池。遵循电路弯曲的艺术，然后他着手探测电路板上的触点。为了控制录制信息的音调，[dgenigt]发现了两个触摸板，当触摸时，会改变播放速度。将这两点连接到电位计的耳上，可以连续改变螺距。还不满意，[dgen IGT]想启用循环播放，并发现一个引脚在消息播放完毕时变为低电平。将它连接到播放按钮上可以让录音连续循环播放。

[dgen IGT]现在手上有一个小巧的采样器，零件价格不到 10 美元。最重要的是，他把这一切都装在一个[甜蜜的 70 年代对讲机外壳](http://www.dogenigt.com/2014/01/greeting-card-sampler-boxed.html)中，使用呼叫按钮来激活记录，甚至用 LDR 使它变得光敏。

这些年来，我们已经看到了一些有趣的电路弯曲——看看这个[数字弯曲的罗兰 TR-626](https://hackaday.com/2015/01/30/digitally-controlled-circuit-bending/) 或这个经典的[黑客 Furby](https://hackaday.com/2016/08/21/hacked-furby-knows-when-youre-near/) 。
