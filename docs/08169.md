# 自动化你的门只需 20 美元

> 原文：<https://hackaday.com/2018/01/11/automating-your-door-for-20/>

我们喜欢星际迷航企业号上的门。我们应该知道它们是由人类操作的，因为它们太聪明了。他们会等人，或者在打斗中有人向他们扑来时不开门。[SieuweE]有一个更加实用的自动门,他称之为 ArduDoor。

你可能会从名字中猜出它使用的是 Arduino。它还使用了挡风玻璃刮水器电机，这是完美的，因为它是高扭矩和低速。如果你经常去废品收购站，你甚至可以花很少的钱或者免费买到一辆。

缺点是在 12V 下运行电机需要 12A 的电流，然而这是高扭矩电机的代价。[SieuweE]正在使用 ATX 的电源来运行它，这不是非常方便，但很容易获得和使用。

Arduino 和超声波传感器是相当标准的东西，但焊接的开门机制是这个项目中真正的工作所在。当然，你的门会有所不同，所以如果你想复制 ArduDoor，你可能需要有点创意。

当然，除了超声波，你还可以做很多事情。[SieuweE]说他很快就会有 Cortana 控制的门。我们在考虑像蓝牙检测这样的事情，甚至使用 [OpenCV 或其他方法来识别你靠近门的脸](https://hackaday.com/2011/05/16/cheap-and-reliable-portable-face-recognition-system/)。
