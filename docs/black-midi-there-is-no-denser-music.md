# 黑迷笛:没有更密集的音乐

> 原文：<https://hackaday.com/2015/11/12/black-midi-there-is-no-denser-music/>

想象一下，如果你同时弹奏钢琴上的所有键。听起来会是什么样的？现在想象一下，你想把音乐转录下来。会是什么样子？笔记太多，你几乎看不到下面的纸。

这就是为什么制作这种“不可能的音乐”的人自称为“黑色 MIDI 乐队”( Black MIDI Crew )( T1 ):如果你把音乐写下来，它看起来就像一个大的黑色斑点。或者至少，这是个笑话。然而令人惊讶的是，这听起来并不像是一场大混乱。看看下面的“圆周率，有 314.15 万个音符的歌”看看我们的意思。

 [https://www.youtube.com/embed/ZQ7ipUQoik8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/ZQ7ipUQoik8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



最终，不可能的琶音和飞溅和弦听起来令人惊讶地像是芯片曲调。具有讽刺意味的是，chiptunes 使用这些快速移动的琶音作为他们一次可以播放的有限数量的声音的感知变通方法——一个接一个地播放和弦的音符，速度之快几乎让你的大脑误以为它们是同时发生的。

我们说“讽刺”是因为在 Black MIDI 中，他们一次演奏尽可能多的音符，而疯狂的琶音似乎是将一些音乐结构融入混乱的一种方式。事实上，黑色 MIDI 乐曲同时演奏如此多的音符，以至于我们无法真正察觉到它们，它们最终听起来像一个单一的合成音色，最终听起来甚至更像一首 chiptune 而不是一架钢琴。这是一个整洁的效果。

如果你认为这些效果与正在使用的合成“钢琴”有关，看看这个项目，它使用了一个[螺线管驱动的真正钢琴作为声码器。](http://hackaday.com/2009/10/05/vocoding-with-a-piano/)(我要求混搭！)

在钢琴上一次弹奏足够多的音符，很难听到“音符”——你失去了基频的概念，你只听到声音。
