# 太糟糕了。特斯拉线圈之死

> 原文：<https://hackaday.com/2017/04/06/that-sucks-death-of-a-tesla-coil/>

【Electroboom】总有些娱乐性的视频。他最近试图在真空中运行他的特斯拉线圈。视频[展示了一些有趣的结果](https://www.youtube.com/watch?v=WM25pUsrODk)，以及他在钻进自己的手时发出的惯常咒骂声，以及以电子产品的名义遭受的其他侮辱。

不幸的是，一点额外的螺栓导致线圈内部电弧，最终导致令人印象深刻的设备拖着它的致命…嗯，好吧，让我们只说它不合时宜的死亡。不过，在这个过程中，你可以看到一些有趣的技术来为真空室制作硅胶密封，以及一些使用封闭注射器的整洁的特斯拉线圈技巧。

当 Electroboom 计算出他的钟罩上的力大约为 6000 磅时，我们有点担心他的安全。毕竟，他并不因为出色的安全记录而出名。然而，他推测是罐子的对称性导致了大部分力的抵消。他甚至测试了这个理论。尽管如此，如果你尝试这样做，一定要小心。他至少应该戴上眼睛保护装置，以防万一。

[Electroboom]并不是第一个将线圈放入真空室的人，但他可能是尝试这种做法的最幽默的受伤者。如果高输入电压让你担心，也许你可以试试这个变化。或者，如果你不想建造这么大的真空室，你可以做总[弄小](https://hackaday.com/2014/12/22/micro-tesla-coil-makes-a-perfect-stocking-stuffer/)或者甚至去[固态](https://hackaday.com/2014/10/20/solid-state-tesla-coil-plays-music/)，两全其美。另一方面，如果你有一个线圈，你可以从中获得很多乐趣，而不用在真空中融化它。
