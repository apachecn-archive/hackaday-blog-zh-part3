# 定制切割风车是一个有用的暖通空调管道流量计

> 原文：<https://hackaday.com/2017/08/25/custom-cut-pinwheel-makes-a-useful-hvac-duct-flow-meter/>

每个人都熟悉风车，很少有人没有用一张纸、一根棍子和一根针制作过一个。从设计的角度来看，风车是非常优化的，而且如此便宜和容易制造，以至于[将风车用作 HVAC 管道流量计](https://www.youtube.com/watch?v=yZzbtLPVkeM)看起来是个好主意。

理论上也许很棒，但正如(可能更糟)发现的那样，自制的风车远不是理想的风速计。之前[钻研超声波流量测量](https://hackaday.com/2017/07/26/measuring-air-flow-with-ultrasonic-sensors/)的风管流量测量实验，让他尝试了机械手段。这需要某种涡轮产生与气流成比例的信号，但当温和的气流无法克服转子磁铁引入的阻力时，使用无刷 DC 电机的计算机风扇的首次尝试失败了。但是一个简单的风车，从一个玩具上按比例缩小的图案上定制下来的，被证明是合适的。当涡轮机在 HVAC 管道中旋转时，反射式光传感器会计算转数，稍加校准，该装置就会产生良好的结果。局限性是显而易见的:管道湍流，脆弱的建设，以及轴承差。但是对于一个又快又脏的测量来说，还不错。

寻找室外风速计，而不是暖通空调流量计？我们有一个由旧电机制成的[或者](https://hackaday.com/2017/01/12/old-motor-donates-rotor-for-coaxial-wind-vane-and-anemometer/)[一个极其精确的超声波装置](https://hackaday.com/2013/08/21/ultrasonic-anemometer-for-an-absurdly-accurate-weather-station/)。
