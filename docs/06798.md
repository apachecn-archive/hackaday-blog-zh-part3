# 从手持带锯到商店带锯

> 原文:[https://hack aday . com/2017/08/14/从手持带锯到商店带锯/](https://hackaday.com/2017/08/14/from-handheld-bandsaw-to-shop-bandsaw/)

如果你在车间周围长大，那么你很可能会对锯子非常尊敬。在成长过程中，父母不断警告他们的风险，这给这个新生的创客留下了不可磨灭的印记，也让他们在这个问题上明显不像其他黑客空间成员那样漫不经心。事实是，电锯为你的车间提供了一些最容易发生危险的机会。但是哪些是最不危险的呢？在撰写本文的地方附近的车间里，当谈到手指固定时，带锯肯定比圆锯好得多。

[制作材料]有一个便携式带锯，包含在一个大型手持电动工具中。他上传了一段视频，详细介绍了他如何将它改装成一种更传统的垂直或水平带锯，增加了一个坚固的焊接管状框架和工作台。

视频从移除扳机和手柄的塑料环绕开始，然后继续进行切割、测量、钻孔和焊接的各个阶段。支点是自行车的曲柄轴承，稍微复杂一点的开关是固态继电器，而不是传统的东西。金属工作做得很好，虽然工程没有特别之处，大多数普通读者也可以做类似的事情，但它具有研讨会视频的引人注目的质量，其中一切都做得很好，结果也很好地呈现出来。你可能不会做这个电锯，但如果你有一个它不会让你失望。完整视频在休息下方。
