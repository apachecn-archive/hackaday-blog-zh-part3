# 为坐立不安的旋转爆炸定制福斯特纳位

> 原文：<https://hackaday.com/2017/07/18/customize-forstner-bits-for-fidget-spinner-explosions/>

[马蒂亚斯·万德尔]是一位杰出的木工。他是所有那些木制齿轮装置背后的家伙，他在台锯上制作了凹口成型*，如果这句话“除非你知道你在做什么，否则不要这样做”适用于任何人，它也适用于(马蒂亚斯)。现在，他开始热衷于坐立不安的旋转器，但车间里有一个问题:[马蒂亚斯]找不到合适尺寸的钻头，所以[他修改了福斯特纳钻头，以包含旋转器的心脏](https://www.youtube.com/watch?v=h7Fc7G4olRo)。*

[马提亚斯]有几个滚轴溜冰鞋轴承，直径 22 毫米。然而，最接近这个尺寸的钻头是 7/8 英寸，或 22.23 毫米。钻头可以研磨，所以钻头被夹在手钻上，并被带到台式研磨机上。和大多数事情一样，除非你知道自己在做什么，否则你不应该这样做。[马提亚斯]有。

随着钻头研磨到 22 毫米，[马蒂亚斯]在一块木头上钻了一个洞，插入轴承，完成了一项史诗般的任务，这就是他的命运。坐立不安纺纱没有用，所以[马提亚斯]决定让这一个爆炸。在这个木制旋转器上切了几个槽口后，[马提亚斯]大量使用车间空气，旋转旋转器，直到它散架。
