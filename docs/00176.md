# 启用 RFID 的机器人为 3 岁儿童播放音乐

> 原文:[https://hack aday . com/2015/10/03/RFID-enabled-robot-plays-music-for-3 岁/](https://hackaday.com/2015/10/03/rfid-enabled-robot-plays-music-for-3-year-old/)

[罗纳德]有一个三岁的女儿，她喜欢音乐，但还没有完全掌握复杂的 MP3 播放器或收音机——怎么了，三岁很老了吗？！受到他看到的一个支持 RFID 的卡带播放器的启发，[Ronald]决定给她做一个可爱且易于使用的东西。

他从可爱的安德鲁家庭入侵者扬声器开始，然后在里面塞了一个树莓派。他想做的是能够将 RFID 标签放在某些物体上，这样他的女儿就可以将其与她最喜欢的音乐联系起来——唯一的问题是，他不知道如何使用 RFID 标签！幸运的是，他找到了[另一篇解释如何用 Python](http://www.instructables.com/id/Attendance-system-using-Raspberry-Pi-and-NFC-Tag-r/) 编写脚本以便轻松使用 RFID 系统的文章。

事实证明，即使他有基本的编程知识，他也能弄清楚如何使用 Pi 的 GPIO 和 RC522 卡，并编写与之配套的软件。结果非常棒！
