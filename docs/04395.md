# VGA 显示器成为绘画玩具

> 原文:[https://hack aday . com/2016/12/08/VGA-monitor-becoms-drawing-toy/](https://hackaday.com/2016/12/08/vga-monitor-becomes-drawing-toy/)

我们不想打断[罗布蔡]，但他已经建立了一个[VGA 绘图玩具](http://www.instructables.com/id/VGA-Etch-a-Sketch-With-Arduino-Uno/)，而不是蚀刻素描。我们怎么知道？简单，蚀刻素描是注册商标。不管怎样，他的项目展示了 Arduino 如何使用 VGAx 库驱动 VGA 显示器。当然，你只能用 120×60 的分辨率做四种颜色，但另一方面，除了 Arduino(你需要四个电阻)，它几乎不需要任何硬件。

硬件包括两个锅和正确的固件，它也可以玩乒乓，如果你不想给本特你的艺术一面。你可以在下面看到艺术玩具和乒乓球游戏的视频。

因为该设备最初是一个 pong 游戏，[Rob 的]版本有两个盒子，每个盒子都有一个锅和一个按钮。当然，如果你真的只是为了绘画玩具而建造它，你可能会把它都放在一个盒子里。甚至可能是一个红盒子。如果我们在建造它，我们会在盒子里放一个倾斜传感器或加速度计，这样你就可以摇动它来擦除图像。随便说说。

如果你想从 Arduino 获得 640×480 的分辨率，这是可以做到的，但它需要更多的硬件。如果你试图让一个孩子对 Arduino 感兴趣，你可以从两个有趣的视频项目开始，使用一些容易获得的部件，并共享硬件。同样，如果你属于“要么做大，要么回家”的阵营，我们会转到这个[乒乓游戏](https://hackaday.com/2016/09/10/giant-scale-physical-pong/)。
