# Hackaday 奖参赛作品:辛普森一家之手

> 原文：<https://hackaday.com/2016/10/18/hackaday-prize-entry-simpsons-hands/>

这个 Hackaday 奖项的创造者称，每个月都会有新的 3D 打印义肢解决方案出现。这并不意味着他们不做一些不同的事情；是的，他们仍在制造一只假手，但他们在上面加入了自己的想法。这个没有使用线/电缆/腱设置，手甚至没有四个手指。[Giovanni]和[Jenny]走自己的路，他们想出的东西很特别。

这种假手最明显的特征是缺少一个手指——辛普森一家的手——但如果你仔细想想，这很有意义。在不久的将来，任何 3D 打印的假手是否会演奏钢琴或触摸键盘都值得怀疑，所以小指是阑尾；一个不必要的虚荣心，只会增加 BOM，使事情更难组合在一起。

除了辛普森一家的手，这只假手或多或少是你所期待的。电路只是一个 Adafruit ATmega 板，机制只是几个伺服系统，机械结构在碳纤维 PLA 中设计得很好。这个假肢手对这个团队来说很特别，因为这是他们的祖国哥伦比亚第一个这样的假肢。即使每隔几周 Thingiverse 上就有一个新的 3D 打印假手，这个项目也使它成为一个真正的全球努力，并且是 Hackaday 奖的一个很好的参赛作品。
