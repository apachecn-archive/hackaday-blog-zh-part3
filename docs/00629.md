# 热导管漫游者探索恶臭，拯救手电筒

> 原文：<https://hackaday.com/2015/11/12/heat-duct-rover-explores-stink-rescues-flashlight/>

一切都是从热记录器发出的难闻气味开始的。[CuddleBurrito]回忆起有一次，他父母家的管道系统里出现了一些恶臭的东西，结果他们花了一大笔钱去探索。黑客思维避开了这些支出，转而使用自己的解决方案来调查恐惧。在这个过程中,[CuddleBurrito]带我们进入了他家的内部。

他的调查工具的 1.0 版本是一个手电筒和连接在卷尺末端的廉价网络摄像头。他被推进管道，在笔记本电脑上监控，得到了很好的视频——直到他的设备被什么东西钩住。几艘拖船给他留下了一个毁坏的相机和一个搁浅的手电筒——他的最爱！任务很快从调查气味转向寻找仍然亮着的手电筒。随着时间的推移，他意识到他的装备需要轮子来避免被绊住，他用压舌板和热熔胶迅速制作了一个雪橇，并用一个结实的纸板管作为滚轴。装备了另一个手电筒、另一个网络摄像头和一个带拉链手指的小伺服系统，[可爱的布里多]再次潜入深海，试图通过绳索找回他最喜欢的手电筒。在下面的视频中，救援在一分钟后开始。

可悲的是，一旦任务转为营救，臭味的来源就再也没有找到。不过，我们怀疑好奇心会战胜[可爱的巴瑞托]，当它战胜好奇心时，我们想看看他会对这个装置做些什么来减轻气味。
