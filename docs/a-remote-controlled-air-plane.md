# 遥控飞机

> 原文：<https://hackaday.com/2017/12/11/a-remote-controlled-air-plane/>

空中大鲨鱼是一种由压缩空气驱动的自由飞行模型飞机。当它在 90 年代末发布时，它是一个相当创新的玩具，具有一个完全由注射成型塑料制成的惊人新颖的压缩空气发动机。这些模型飞机的销售起飞，降落在邻居的屋顶上，再也看不到了。

几周前，[汤姆·斯坦顿]通过创建自己的 3D 打印副本，重新审视了这个新颖的小型气动马达。是的，它成功了，是的，这是一个非常令人印象深刻的 3D 打印。然而，那个建筑只是在工作台上，为了真正测试这个空气马达，[Tom] [用它来推动一架遥控飞机在空中飞行](https://www.youtube.com/watch?v=1ZqFDsq09H8)。

本实验所用的马达是由[【Tom】原来的气动马达](https://hackaday.com/2017/11/20/acetone-smoothing-results-in-working-motor/)稍加改装而成。最初的发动机使用标准的 3 叶片四轴飞行器螺旋桨，但适合飞行的建造使用了更大的螺旋桨，可以摆动更多的空气。这一点，加上马达中的一个新弹簧和一个由塑料瓶制成的更大的空气罐，使得马达不是很重，但仍然可以摆动一个道具几十秒钟。虽然不多，但也不算少。

这项实验的机身是使用[Tom]的 [3D 打印翼肋](https://hackaday.com/2017/09/08/3d-printed-ribs-for-not-3d-printed-planes/)建造的，尾部是碳纤维吊杆，只有方向舵和升降舵控制。在解决了一些 CG 问题后——发动机重量不大，飞机通常在机头有大电池——飞机飞得非常好，尽管时间很短。
