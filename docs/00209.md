# 纸板机器人死亡竞赛

> 原文:[https://hackaday.com/2015/09/29/cardboard-robot-deathmatch/](https://hackaday.com/2015/09/29/cardboard-robot-deathmatch/)

战斗机器人甚至比普通机器人还要牛逼。但我们很难想象把所有的钱(更不用说血、汗和泪)扔进一个机器人，然后看着它被撕碎。位于密苏里州哥伦比亚市的黑客空间哥伦比亚小工具工厂的员工们找到了解决方案:用纸板制作机器人。

用硬纸板和热熔胶制造机器人最酷的一点是它很便宜，但如果它们的规模适中，它们仍然可以相当坚固，可以快速修复，而且在一次残酷的失败后，你很可能能够找到所有的零件。简而言之，这是一个黑客空间活动的好主意。
