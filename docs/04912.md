# 自制焊台，售价 15 美元

> 原文：<https://hackaday.com/2017/02/01/home-made-soldering-station-for-15/>

合适的烙铁是优秀黑客需要的基本工具之一。优选地，其具有温度控制，从而可以处理不同类型的焊料和连接器。

体面的焊接站并不便宜，但[代码和焊接]告诉你如何以大约 15 美元的零件制作一个。这使用了一个便宜的非温控 USB 烙铁，一个 Arduino 和其他一些他们从全球速卖通买的东西。计划是在烙铁上加一个热电偶，让 Arduino 控制温度。旋转转盘和液晶显示屏控制设定点，Arduino 通过 FET 开关加热元件的供给。

这不是世界上最干净的构造，这些 USB 烙铁不适合大接头或长时间的焊接工作，但对于预算有限的制造商来说，这是一个整洁的小黑客。在之前，我们已经看到了这些[相当整洁的小 USB 烙铁的拆卸，但这是一种扩展其功能的有趣方式。](http://hackaday.com/2016/04/25/usb-soldering-iron-is-surprisingly-capable/)
