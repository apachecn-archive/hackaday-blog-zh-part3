# 培养一个机器人，探索你的家园

> 原文：<https://hackaday.com/2016/06/24/foster-a-robot-explore-your-home-planet/>

我们送去探索其他世界的机器人是令人印象深刻的工程壮举。但是像我们这样被困在重力井的底部，它们是 DIY 社区无法企及的昂贵的冒险。尽管如此，在你自己的后院仍有很多东西需要探索，这个机器人需要你的帮助来探索地球。

这个项目叫做 RoboSpatium，是[诺伯特·海因茨]的创意。这个想法有点像 HitchBot，只不过它是通过邮件从一个主机发送到另一个主机。(这是一个真实的机器人，而不仅仅是装在桶里的大脑。)希望每个主持人都能在分配的 24 小时里让机器人做一些有趣的事情，比如探索当地的地标，用机器人的眼睛观察黑客空间里发生的事情，或者只是在一些美丽的地方观看日落。项目参与者将通过网络界面驾驶机器人，对他们可能永远也不会看到的世界进行一点虚拟探索。

我们推断，下面视频中的机器人目前只是一个原型，传感器套件和机械装置还有待整理。Hackaday 的常客们无疑会更了解[Norbert]，因为他有着出色的口音[HomoFaciens]，他发明了[垃圾来源的 CNC 机器](http://hackaday.com/2015/11/21/garbage-can-cnc-machine-build/)，由[锡罐盖](http://hackaday.com/2016/01/08/video-gives-you-the-basics-of-diy-rotary-encoders/)和[电阻轮](http://hackaday.com/2016/06/15/wheel-of-resistors-form-unique-rotary-encoder/)制成的编码器，以及[一个潜在的自我复制 CNC 绘图仪](https://hackaday.com/2016/04/27/cardboard-and-paperclip-cnc-plotter-destined-for-self-replication/)。[诺伯特]有黑客的能力来实现这一点，我们认为这是一个非常好的想法，有可能吸引和教育很多人。我们认为这需要黑客社区的一点支持。
