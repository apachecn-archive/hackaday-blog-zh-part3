# 令人印象深刻的 Z80 电脑，名字却不太好听

> 原文:[https://hacka day . com/2017/01/22/the-impressive-z80-computer-with-the-不幸的名字/](https://hackaday.com/2017/01/22/the-impressive-z80-computer-with-the-unfortunate-name/)

我们已经看到了很多 Z80 的复古造型。没有几个像[dekeNukem]的[FAP 80](https://github.com/dekuNukem/FAP80)项目那样做得如此干净利落或记录如此完整。在你急着发表评论开这个明显的玩笑之前，我们会告诉你，每个人都已经对同一个玩笑做出了自己的变形。我们还会告诉你，这个名字是来自[Steve Ciarcia]的一个名为 ZAP80 的旧设计和该器件中使用的 FPGA 的结合。

[dekeNukem]说他的目标是创造一台 Z80 计算机，摆脱使用周期校正支持芯片的所有包袱。你可以争论这种方法与更纯粹的构建相比的相对优点，但 FAP80 有 5 插槽背板、VGA 输出、PS/2 键盘端口等。您可以在下面的许多视频中看到一个展示该机器的视频。

这台机器开始时是手工布线的，但是由于噪音问题，现在是在印刷电路板上。除了 GitHub 页面之外，还有一组广泛的[博客帖子](https://dekunukem.wordpress.com/category/fap-z80-computer/)，其中有详细介绍该机器设计和构造的视频。并不是所有的博客帖子都有标签，所以你可能需要四处看看，但是有很多关于这个项目的细节。如果你不喜欢博客帖子，也有一种[视觉之旅](http://imgur.com/gallery/hfG6e)。

你可能会认为 Z80 计算机使用 FPGA、CPLD 和两个 ARM 芯片来支持有点奇怪，但它确实是一个强大而简单的系统。然而，我们已经看到了由更现代的设备支持的其他 Z80 系统。不过，我们认为你可以在 FPGA 或更大的 CPU 中[模拟整个过程](http://hackaday.com/2014/12/22/zx81-emulated-on-an-mbed/)。
