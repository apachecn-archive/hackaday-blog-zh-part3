# 激光驱动的飞行器重 100 毫克

> 原文：<https://hackaday.com/2018/05/30/laser-powered-flying-machine-weighs-milligrams/>

我们已经习惯于在飞行器规模的较小一端看到一些美丽的手工创作，包括固定翼和旋翼的小型飞机。重量只有几克的飞机是完全有可能制造出来的，这是部件可用性方面令人难以置信的进步。

但是一架可以工作的飞机能做得小多少呢？给定一个合适的团队和预算，如何进入毫克？Sawyer Fuller 博士]和他在华盛顿大学的团队制造了一架扑翼飞机，这可能是迄今为止最轻的飞机，它使用压电驱动器来拍打灵活的翅膀。这本身并不是全新的，但以前的努力依赖于系绳线供电，最新的创造是通过激光向机载微型太阳能电池供电来自主飞行，太阳能电池通过电线伸出飞行器。

令人沮丧的是，富勒博士在机器上的页面没有我们想要的那么详细，这可能是因为他们把这些有趣的东西留到了一个会议演示上进行大展示。然而，从技术角度来看，这是一个非常有趣的发展，也为飞行器的应用开辟了一个全新的领域。无论发生什么，我们都会随时通知你。

你可以在视频中看到这个飞行器，如果你对更传统的微型机器[感兴趣，可以看看 2.9g 野马型号](https://hackaday.com/2017/02/10/a-micro-rc-plane-builder-shares-his-tricks/)的创造者。
