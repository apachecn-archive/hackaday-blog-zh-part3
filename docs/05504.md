# 星际迷航桌面浏览器在你的手掌！

> 原文:[https://hack aday . com/2017/04/07/star-trek-desktop-viewer-in-the-palm-of-hand/](https://hackaday.com/2017/04/07/star-trek-desktop-viewer-in-the-palm-of-your-hand/)

有制造小型计算机的——比如树莓派——然后有制造小型计算机的——比如《星际迷航》中的桌面浏览器。

[蒙他·埃尔金斯]在这个项目中使用了甲虫；这是一个 Arduino 克隆，托管 ATMega32U4 微控制器，具有一个独特的功能，允许您扭曲连接线以将其固定到板上。相反，[Elkins]做出了焊接它们的合理选择。对于显示器，他使用了 SPI 系列有机发光二极管 128 x 64 单色屏幕，他在许多标志性的《星际迷航》TOS 符号和动画中循环播放。这些图像在最终上传到甲壳虫之前被转换成 PROGMEM，然后加载到闪存中。

在 ABS 塑料中完成一些精细的 3D 打印工作，呈现出桌面观众的情况后，[Elkins]使用丙酮将碎片溶解焊接在一起，并快速涂上一层油漆完成。这个小复制品将成为一个伟大的桌面小工具，因为它需要一个微型 USB 来为设备供电。
