# 通过基因编辑更好的啤酒

> 原文:[https://hack aday . com/2018/03/27/better-beer-through-gene-editing/](https://hackaday.com/2018/03/27/better-beer-through-gene-editing/)

尽管今天的美国啤酒爱好者似乎喜欢啤酒花和其他淡色麦芽酒，但啤酒花的生产和运输成本如此之高实在令人遗憾。你知道吗，种植足够的啤酒花来生产一品脱精酿啤酒需要 50 品脱的水。虽然啤酒花对啤酒酿造并不重要，但它们添加了精油和香料，可以将原本平淡无味的啤酒变成美味的泡沫。

使用加州大学伯克利分校自己的简单实惠的 CRISPR-CaS9 基因编辑系统，[研究人员【Charles Denby】和【Rachel Li】编辑了啤酒酵母的菌株，使其尝起来像啤酒花](http://news.berkeley.edu/2018/03/20/brewing-hoppy-beer-without-the-hops/)。这些改良菌株既能发酵啤酒，又能提供啤酒饮用者渴望的啤酒花风味。这些音符来自薄荷和罗勒基因，研究人员将它们与 CaS9 蛋白和启动子一起拼接到酵母基因中，帮助编辑成功。这尤其具有挑战性，因为啤酒酵母有四组染色体，所以他们必须做四次。否则，酵母可能会排斥供体基因。

那么，味道怎么样？附近一家啤酒厂的一群员工参加了一项盲品测试，他们一致认为转基因啤酒比对照啤酒味道更好。这是值得举杯庆祝的事情。打电话，打出租车，开车穿过休息时间，看一段视频。
