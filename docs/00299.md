# 护士在医疗领域创造

> 原文：<https://hackaday.com/2015/10/21/nurses-create-in-a-medical-makerspace/>

虽然有许多有技能和专门类型的卫生保健专业人员，护士往往是医疗机构和病人之间的主要联系点。你可能会花更多的时间和你的护士在一起——尤其是在医院里——比其他任何医疗服务提供者都要多。每个病人的需求都是不同的，所以护士们有时会临时想出独特的解决方案来帮助他们的病人更舒适或更快康复也就不足为奇了。

这是一个名为[maker course](http://www.makernurse.org/)的创新项目的指导思想，该项目由麻省理工学院和罗伯特·约翰逊基金会支持。这个想法是鼓励护士成为创造者。该项目的联合创始人之一安娜·杨发现，中美洲的护士们利用手头的资源凑合着生活，自然地扮演着创造者的角色。“我们看到一名护士用头顶透明物修理听诊器振动膜，”她说。Young 指出，护士们往往没有意识到她们所做的事情的重要性——这只是她们如何度过一天。

通过 MakerNurse 计划，Young 的小组调查了美国各地的护士，然后去了五家医院，研究护士的收入以及哪些支持可以帮助他们做得更多。这项研究将于明年发表，它确定了来自护士的足智多谋的创客项目，甚至护士、患者及其家庭护理人员之间的合作。在马萨诸塞州发现的一个小工具使得用饲管给病人喂食变得更加容易。杨说，“它很简单，很结实，解决了一个非常明确的问题。”它是由病人及其家人在家得宝购买的材料制成的。

合乎逻辑的结论是:医院里的创客空间。杨已经做到了这一点，得克萨斯州加尔维斯顿的得克萨斯大学医学分院(UTMB)约翰·西利医院拥有第一个这样的空间，她希望将会有许多这样的空间。UTMB MakerHealth 项目为护士(和其他人)提供了一个永久性的设施，以建立定制的解决方案，使他们的工作和病人的生活更轻松。用户可以接受培训和其他帮助，把他们的想法变成现实。

我向杨询问了制造医疗器械的后果。她回答说，“医院已经有了进行调查研究的流程，这些研究就像那些研究一样被对待。”她还提到，他们确保为制造商提供合适的材料。她的信念是，这些创客空间将使护士和其他人能够做那些没有足够容量来保证大型研究的工作(就像上图中心脏病人的左手臂板)。此外，该设施将允许“一次性”件(如定制切割绷带)容易和精确地复制。

麻省理工学院的一个分支机构 [Pop Up Labs](http://www.popup-labs.com/) ，为该计划提供后勤支持和培训。创客空间将有一个专门的经理，不久还将有一个“常驻创客”来帮助设计和建造。杨说，许多护士都很“谦虚”，需要鼓励来提出自己的想法，而像黑客马拉松和移动创客空间这样的活动往往会错过不太敢说话的潜在创客。[创客健康](http://makerhealth.co/)网站是一个在线社区，Young 希望“将创客运动的分享带入医疗保健”，并鼓励更多的医疗保健创造。

下面的视频展示了更多关于 MakerHealth 的信息，包括对该设施的参观。我们已经看到黑客对 STEM 教育产生了巨大影响，由于这些举措，黑客文化可能开始对医院患者产生类似的影响。互联网的先驱们可能无法想象网络会如何彻底改变我们的世界。同样，很难知道随着时间的推移，我们强调创造力、开放性和共享的文化会走向何方，因为它会继续超越我们的传统社区。我很兴奋地看到我们下一步的发展方向，黑客和过滤器将成为主流。
