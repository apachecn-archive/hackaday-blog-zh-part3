# 日本国际空间站补给船兼作系绳实验

> 原文：<https://hackaday.com/2017/01/27/japanese-iss-supply-ship-dual-purposed-as-tether-experiment/>

当火箭向国际空间站发射一个装有补给的太空舱时，他们通常会发射一堆垃圾，所有这些垃圾在重返大气层时都会被烧掉。但是只要你有了那辆(注定要失败的)车，你也可以在路上用它做一些科学研究。这正是日本航天局(JAXA)周五刚刚离开国际空间站的 Kounotori 6 号补给船正在做的事情。

这个实验是用一个[电磁系绳](http://www.ard.jaxa.jp/eng/research/kite/kite.html)进行的，它可以用来将电能转化为动能，反之亦然。当你从地球向外串一根长导线，两端在不同的高度穿过地球磁场从而穿过不同强度的磁场，就产生了电势。在[风筝实验](http://www.ard.jaxa.jp/research/kite/kite-details.html) ( [翻译](https://translate.google.com/translate?sl=auto&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=http://www.ard.jaxa.jp/research/kite/kite-details.html&edit-text=&act=url))中，补给船上的电阻负载和电子发射器被设计用来消耗电能，降低船的动能，并使其轨道降落到地球。

最终的计划是使用像这样的系绳系统从太空垃圾中移除动能，将它拉出轨道并脱离危险，而不需要飞船携带额外的燃料来这样做。为期七天的测试任务将调查 700 米长的铝线产生的推力和电流，以及在它展开时控制它的实验。下面嵌入的视频看起来像是他们要主动控制负载来稳定系绳末端相对于船的位置。有人想插话吗？这不是第一次在太空测试电动绳索，但如果运气好的话，这将是第一次在轨道上成功测试 T2。

这绝对不是黑，科学真的很酷。但是用一艘注定会在大气层中烧毁的货船清理太空垃圾是一个胜利。国际空间站摆脱了其隐喻的脏衣服，主要是耗尽的电池，轨道得到清理启动。我们祝他们好运。
