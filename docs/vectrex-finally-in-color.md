# 威格斯终于上市了

> 原文：<https://hackaday.com/2018/01/03/vectrex-finally-in-color/>

自 20 世纪 80 年代初以来，Vectrex 是每个人最喜欢的矢量控制台。矢量图形确实没有在视频游戏市场上流行起来，但 Vectrex 仍然拥有一批铁杆粉丝，他们直到今天还在继续修补这个平台。[[Arcade Jason]现在刚好领先。](https://www.youtube.com/watch?v=2uC-oMtVSYI)

Vectrex 一直是一台单色机器，只能在其矢量显示器上显示白线。颜色是由粘在屏幕上的塑料覆盖层提供的，然而这从来没有被认为是控制台上一个特别令人兴奋的附加功能。[Jason]决定他可以做得更好，并深入挖掘他收集的向量监视器。

手里拿着一台 36 英寸的彩色矢量显示器，Vectrex 被放在工作台上，准备接受黑客攻击。通往其中一个 DAC 的总线被劫持，并通过一系列“或”和“与”逻辑产生彩色信号，因为原始 Vectrex 硬件无法做到这一点。这然后被馈送到彩色监视器，具有惊人的结果。

[Jason]的设置能够在屏幕上产生 8 种颜色，几乎是出于某种奇怪的巧合，这确实使经典的 Vectrex 游戏以一种前所未有的方式流行起来。这也证明了一个更简单的时代，即有可能在试验板上黑掉这台游戏机的视频信号；现代硬件运行速度太快，无法逃脱这样的厄运。

这是一次史诗般的破解，通过实验和一些意外收获，产生了一些令人兴奋的结果。[【Jason】现在正在进行下一阶段](https://www.youtube.com/watch?v=ll4l2zLZK3A)的工作，尝试添加色彩强度控制和其他功能。

这也不是[Jason]第一次来到这些地方——[就在一个月前，我们看到了他的大屏幕 Vectrex！](https://hackaday.com/2017/11/02/finally-a-big-screen-vectrex/)
