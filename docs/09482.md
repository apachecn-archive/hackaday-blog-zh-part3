# 长 PCB 显示出荒谬的速度效果

> 原文:[https://hack aday . com/2018/05/17/long-PCB-shows-effects-of-ludicrous-speed/](https://hackaday.com/2018/05/17/long-pcb-shows-effects-of-ludicrous-speed/)

输电线路看起来像魔术一样。当你使用它们时，似乎很奇怪，一根电线可以阻止或通过某些频率。使用带脉冲的传输线并不常见，通常电路的传输线行为并不那么重要。也就是说，直到你不得不把一个信号移动一段相对较长的距离。[Robert Baruch]一直在使用一个长 PCB 来测试他正在研究的总线上的脉冲行为。他这个系列其实有几个视频值得一看。

有趣的是[罗伯特]在棋盘上有足够的距离来显示光速效果。通过使用一个非常好的 DPO7104 示波器和一个信号发生器，他展示了信号如何在不同的点反射到线路上，并从中进行加减。测量结果与理论相当吻合。你不应该期望它们完全匹配，因为在整个系统中会随机出现一些小的影响。

在你决定跑出去买同样的示波器之前，你最好检查一下你的银行账户。我们能找到的最便宜的二手电脑大约是 5100 美元，大多数是这个价格的两倍(可能是由于安装的选项；我们失去了寻找的兴趣)。一个新的要多得多。

搭建这样一个电路板，展示一端短路的半波或四分之一波短截线如何像滤波器一样工作，将会非常有趣。事实上，如果你想想电流是如何在天线中流动的，就没那么神奇了。传输线可以像[Robert 的]一样在 PCB 上，但你也可以[用 PCB 走线制作天线](https://hackaday.com/2010/08/05/pcb-trace-antenna/)。
