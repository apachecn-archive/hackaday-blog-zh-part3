# 真空管计算器的氖显示

> 原文：<https://hackaday.com/2018/03/27/neon-display-for-a-vacuum-tube-calculator/>

当谈到古董展示时，每个人都被 Nixies 所吸引。这些电子管看起来很棒，但你用这些复古的数字电子管处理某些美学问题。还有另一个选择。作为他的 Hackaday 奖参赛作品，[castvee8] [正在用老式霓虹灯](https://hackaday.io/project/85369-make-your-own-7-segment-displayswith-neon-light)制作七段显示屏。它看起来很棒，而且是全真空管计算器的基础。

这个建筑核心是几个微小的 NE-2 霓虹灯。这些都是你在旧的指示器中发现的相同类型的灯泡，并且需要大约 100 伏的电压来点火。这些灯泡然后被安装在一个 3D 打印的框架中，给[castvee]一个真正的七段显示器，一个加号或减号，以及一个等号。这是计算器的雏形，就在这里。

这个项目[最近的更新之一是用现代逻辑](https://hackaday.io/project/85369-make-your-own-7-segment-displayswith-neon-light/log/129922-driving-the-display-with-modern-logic)控制这些显示器。这可能有点用词不当，因为[castvee]使用二极管控制和 TTL 芯片来循环数字 1 到 4。不过，完成这项工作的实际代码是在微控制器上运行的，所以可能会通过。不过，这只是一个测试，真正的项目看起来是[一个全真空计算器](https://hackaday.io/project/85369-make-your-own-7-segment-displayswith-neon-light/log/131708-all-vacuum-tube-calculator)。该项目仍处于早期阶段，但离 Hackaday 奖还有几个月的时间，我们迫不及待地想看看这个项目会产生什么结果。
