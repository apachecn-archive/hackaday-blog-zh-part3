# 让激光切割机设计在 3D 打印机上工作

> 原文：<https://hackaday.com/2017/03/03/making-laser-cutter-designs-work-in-a-3d-printer/>

黑客店里的主要机械工具曾经是钻床和车床。如果幸运的话，可能是一个数控铣床。激光切割机在个人商店中仍然是一种罕见的工具，但今天的黑客越来越多地使用 3D 打印机。当你有一个激光切割机(2D 零件)的设计，但只有 3D 打印机时会发生什么？你撑船。

[DIY3DTECH]有一个两部分组成的视频,介绍如何将 2D 的设计(以 SVG 文件的形式)引入 TinkerCad。此时，他在软件中组装零件，并创建一个可打印的对象。你可以看下面的视频。

老实说，从表面上看，这听起来并不复杂。但是像其他事情一样，它并不总是像看起来那样简单。当你试图独立完成时，看着别人完成这个过程会对你有所帮助。请记住，虽然激光切割的零件将适合标签，但 3D 打印的零件不需要它们。他还提到了其他一些问题，比如调整填充物使零件更坚固。

为了证明这一点，[DIY3DTECH]制作了一个视频，展示了如何将 2D 的一个固定装置转换成一个用来放湿巾的装置。第二个视频展示了打印结果，并对打印过程提出了一些建议。例如，激光切割机通常不会弯曲，但并非所有 3D 打印机材料都是如此。

如果你喜欢看某人完成他们的 3D 打印设计过程，你不会错过 3D 打印虎钳的视频。如果你渴望在激光切割机中使用的木材，也许你需要尝试一些[异国情调的细丝](https://hackaday.com/2016/12/10/thirty-days-of-3d-printing-filament/)。
