# Hackaday 奖品入口:一套实验室仪器

> 原文:[https://hack aday . com/2016/07/27/hack aday-prize-entry-a-suite-of-lab-instruments/](https://hackaday.com/2016/07/27/hackaday-prize-entry-a-suite-of-lab-instruments/)

为了他们的 Hackaday 奖参赛作品，[Jithin]、[Praveen]、[varunbuboy]和[Georges] [正在研究 SEELablet，](https://hackaday.io/project/12552-seelablet-instrument-cluster-for-laboratories)一种将为初露头角的公民科学家配备控制和测量设备的设备。

最好的“一体化”实验室设备之一是 [National Instruments 的 VirtualBench](https://www.youtube.com/watch?v=hNksWHJ2rZE) ，这是一种集示波器、逻辑分析仪、函数发生器、万用表和电源于一体的设备。你可以用这样的设备做很多事情，但正如你所料，这个名牌版本不是为中学生准备的。

为了将一体化实验室工具的成本降低到普通人可以承受的价格，SEELablet 背后的团队将单板计算机与示波器、频率计数器、逻辑分析仪、波形发生器和可编程电源的功能相结合。

这是该团队多年的项目，从 Python 驱动的工具的[开始，到后来运行该代码的设备](https://hackaday.io/project/5971-python-powered-scientific-instrumentation-tool)的[，这也是一个多功能的实验室工具。如果项目的最新迭代证明是它所承诺的，我们迫不及待地想看到这个盒子将产生的数据。在一个设备齐全的电子实验室里，你可以测量很多东西，这个项目使得整个设备更容易获得。](https://hackaday.io/project/6490-a-versatile-labtool)
