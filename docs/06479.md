# 水浸渍 101

> 原文:[https://hackaday.com/2017/07/15/hydrodipping-101/](https://hackaday.com/2017/07/15/hydrodipping-101/)

风格很重要，有时让 3D 打印机的产品更有活力所需要的只是一台 2D 打印机和一本水文打印指南。

水纹印刷，有时也称为水渍印刷，是一种通过简单的步骤将图形转移到复杂形状物体上的过程。一个设计被印在一个特殊的薄膜上，然后漂浮在一个水箱的表面。要装饰的物体被小心地浸入水中，穿过薄膜，设计一步就包裹住了所有的角落和缝隙。

下面的视频教程详细介绍了水文印刷的步骤，并概述了随着喷墨打印机水转印膜的出现，该方法变得多么简单。这种薄膜是聚醋酸乙烯酯，本质上是白色胶水，因此非常易溶于水。薄膜溶解，使墨水漂浮在表面，准备浸泡。

该视频列出了一些优化 3D 打印零件过程的技巧，应该可以让你快速轻松地装饰零件。一旦你掌握了基础知识，你可能想看看[数学扭曲你的设计，以水文复杂的表面](https://hackaday.com/2017/02/25/decorate-your-3d-prints-with-detailed-hydrographic-printing/)。
