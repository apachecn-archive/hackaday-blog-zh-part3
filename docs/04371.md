# 建造一个有轮子的机器人

> 原文:[https://hackaday.com/2016/12/06/building-a-whegged-robot/](https://hackaday.com/2016/12/06/building-a-whegged-robot/)

[Jaidyn Edwards]正在制造一个机器人。然而，这不是一个普通的机器人，[他正在建造一个有轮子的机器人](https://www.youtube.com/watch?v=GqLrCPbBlT4&t=253s)，灵感来自波士顿动力公司的 [RHex 设计](https://en.wikipedia.org/wiki/Rhex)。

A [wheg](http://biorobots.case.edu/projects/whegs/) (TM)是一条弯曲的腿，绕着固定的 ~~foxed~~ 旋转(编辑注:固定！)一端点上，由电机驱动。因此得名:一部分是轮子，一部分是腿。通过分别驱动每条腿，你可以保持机器人的平衡，并推动它前进。这是一个复杂的系统。与普通的轮子或驱动系统不同，你需要准确地知道腿在哪里才能正确地使用它，因为腿的位置取决于电机的旋转。

腿本身将由刚性和柔性织物的组合 3D 打印而成，应该能够提供力量和抓地力。在第一个视频中，[Jaidyn]概述了他的设计，并解释了他为什么要尝试这种方法。这是正在进行的系列中的第一部，绝对值得关注。
