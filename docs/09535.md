# 树莓派保持凉爽

> 原文：<https://hackaday.com/2018/05/22/raspberry-pi-keeps-cool/>

总的来说，热量是电子产品的大敌。克里斯托弗·巴纳特(Christopher Barnatt)非常认真地想要打败这个敌人，他用不同的冷却方案为树莓 Pi 3 做了一些实验。你可以在下面的视频中看到结果。

一个简单的测试脚本为每个配置生成了七个温度读数。[Barnett]使用一个裸露的 Pi，一个便宜的粘贴式散热器，然后在散热器上安装两个不同的风扇。他还用一个铜垫片组装了一个大散热器，并将其与两个风扇中较大的一个结合在一起。

我们不确定我们是否会在这些测试中使用他的方法。脚本执行得很快，温度上升是否趋于平稳还不清楚。我们也不确定这会给 CPU 带来多大的负载。然而，结果与您的预期相符，所以脚本和数据生成方法可能是好的。

真正有趣的部分不是结果。我们期望更大的风扇做得更好，更大的风扇和散热器做得最好。然而，观察不同的冷却系统安装在 Pi 上并供电的方式很有趣。最终的解决方案——无论如何都是大材小用——没有以适合部署的方式安装。但是其余的风扇和散热器组合可以很容易地适用于实际项目。

如果你真的想认真，你可以随时[把圆周率投入油中](https://hackaday.com/2017/05/15/liquid-cooling-overclocked-raspberry-pi-with-style/)。或者[安装一个热电热泵](https://hackaday.com/2017/04/18/brazil-wins-the-raspberry-pi-overclocking-olympics/)，将多余的热量倒入一桶水中。但对于我们大多数人来说，这里的任何风扇解决方案都已经足够了。
