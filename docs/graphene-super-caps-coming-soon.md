# 石墨烯超级电容:即将推出？

> 原文：<https://hackaday.com/2015/12/10/graphene-super-caps-coming-soon/>

如果你经常阅读 Hackaday，你可能听说过可以用激光制造石墨烯。已经有一些关于如何使用该技术(称为 LIG 或激光诱导石墨烯)制造实用石墨烯超级电容器的研究。莱斯大学的研究人员一直致力于此，显然他们已经取得了重大成功[在 Kapton 基底上诱导石墨烯电容器](http://news.rice.edu/2015/12/03/scientists-see-the-light-on-microsupercapacitors-2/)。该团队在[高级材料](http://onlinelibrary.wiley.com/doi/10.1002/adma.201503333/abstract)(在付费墙后面)上发表了一篇关于他们工作的论文。

特别是 Rice 声称他们已经轻松制造出能量密度为 3.2 mW/立方厘米的超级电容器(大学网站上是这么报道的；它们可能意味着毫瓦-小时/立方厘米)，电容接近每平方厘米 1 毫法。这种构造方法的一个关键好处是，研究人员将电容器弯曲 10，000 次后，电容器仍能继续工作。在经常弯曲的可穿戴设备中，或者在像手机这样可以在你坐着的时候弯曲的设备中，柔性电容器是很有用的。

虽然用激光生长石墨烯很容易(我们甚至看到用 DVD 刻录机比一次更[地](http://hackaday.com/2012/03/20/print-your-own-supercaps/)[做到了这一点)，但赖斯团队研究的有趣部分似乎是他们使用二氧化锰、羟基氧化铁或聚苯胺的电沉积来创建复合正负电极。你可以在下面看到一个关于这项新技术的短片(当然那不是 DVD 刻录机)。](http://hackaday.com/2015/10/03/laser-etching-graphene-supercapacitors/)
