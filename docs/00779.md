# 适合大孩子的坐式旋转

> 原文:[https://hackaday.com/2015/11/28/sit-n-spin-for-big-kids/](https://hackaday.com/2015/11/28/sit-n-spin-for-big-kids/)

人类似乎对测试自己的极限有一种奇怪的爱好，尤其是在纺纱的时候。也许他们坐在重力加速器上，同时梦见他们在 NASA 的 20 g 测试离心机中。然而，当嘉年华游乐设施还不够时，一些无畏的黑客们就开始使用电焊机，自己动手解决问题。这是一个有很多名字的黑客，尽管“乡下人转椅”会在 YouTube 上带来很多点击量。

这个设计非常简单。从旧车或卡车中取出后差速器和车桥总成。将其旋转 90 度，使差异现在指向上。焊接一把椅子。最后，焊接两个牵引杆。拉整个乱七八糟的东西会导致轮子旋转，轮子通过差速器传递动力，使椅子旋转。乘坐不一定要拉得很快，因为汽车差速器通常有 3:1 到 5:1 的减速比。我们把事情做反了，所以缩减变成了乘数。结果，在下面的视频中可以看到，是一个非常头晕的骑手。

我们能找到的最早的这种坐骑是在明尼苏达州伯川姆的鹰山创造的。我们打赌这种特殊的黑客技术已经存在几十年了。在我们最近的记忆中，最接近的是北街实验室的 Centrifury。你知道一个更早的化身吗？请在评论中告诉我们！
