# 使用这款 Arduino 拼图地理缓存来破解您的徒步旅行

> 原文:[https://hack aday . com/2017/05/03/hack-your-hike-with this-arduino-puzzle-geo cache/](https://hackaday.com/2017/05/03/hack-your-hike-with-this-arduino-puzzle-geocache/)

对于那些喜欢徒步旅行的人来说，进入森林是不需要任何借口的。然而，其他人需要一点哄骗才能进入大自然，这就是地理藏宝的用武之地:在森林里藏东西，在网上发布它的位置线索，他们就会来。这个谜题很吸引人，尤其是对于这个拥有 Arduino 驱动的 Hangman 游戏的地理缓存来说，它需要在解锁缓存之前得到解决。

地理缓存的实际内容很少是重点-毕竟，它是旅程，而不是目的地。但[cliptwings]的目的地很可能是一个真正取悦大众的地方。像许多 geocaches 一样，这是一个防水塑料弹药罐。罐子里面是另一扇门，只有正确解决 Hangman 的一个经典游戏才能解锁。这款游戏本身对于长期的 Hackaday 读者来说可能看起来很熟悉，因为我们在 2009 年就特别介绍过它。正确地解决这个难题打开了内室，揭示了地藏王内心的美好。

聪明的是，[cliptwings]将 Arduino 的伏特电池安装在内门的顶部，这样缓存器就可以更换没电的电池并玩游戏；奇怪的是，Geocaching.com(需要注册)上的[缓存条目并没有指示玩家携带电池。](https://www.geocaching.com/geocache/GC72AFD_hangman)

自从几天前被放置在亚利桑那州图森市北部的一个公园后，看起来这个缓存已经被发现并解决了一次。我们展示的其他小工具缓存包括[支持 GPS 的反向缓存](https://hackaday.com/2013/03/20/putting-the-brains-of-a-reverse-geocache-on-the-outside/)，以及[一个需要 IR-vision](https://hackaday.com/2012/04/12/infrared-hides-code-combination-on-geocache-puzzle/) 才能解锁的拼图缓存。
