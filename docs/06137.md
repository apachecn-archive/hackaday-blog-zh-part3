# 哦不！又是爪子！

> 原文:[https://hackaday.com/2017/06/12/oh-no-its-the-claw-again/](https://hackaday.com/2017/06/12/oh-no-its-the-claw-again/)

[瑞安·贝茨]显然[真的很喜欢建造抓爪机](http://www.retrobuiltgames.com/the-build-page/arduino-claw-machine/)。我们注意到他的最新版本有一个新的 PCB，但后来我们向下滚动，发现这台机器的其他化身可以追溯到 2015 年。

激光切割的爪子看起来很有趣，大脑是一个 Arduino。你可以在下面的视频中看到这个动作，在项目页面上还有很多更老的视频。

没有印刷电路板，机器就是一堆乱七八糟的电线。对于 PCB，一根带状电缆将 Arduino 连接到电路板，电路板采用螺丝端子布局。这使得布局比以前更加整洁。

你可能会认为你不会看到许多自制的爪机。然而，那不是真的。我们已经看到了很多，其中一些甚至使用[商业爪子](https://hackaday.com/2016/04/08/oh-no-its-the-claw/)。自然，他们中的一些人也连接到了[互联网](https://hackaday.com/2015/12/24/here-comes-santa-claws/)。
