# Hackaday 奖参赛作品:Arduino 防溅马桶起泡剂

> 原文：<https://hackaday.com/2017/05/04/hackaday-prize-entry-arduino-splash-resistant-toilet-foamer/>

有些普遍的人类经验我们不太谈论，至少在公共场合不谈论。其中一个你会在你自己的房子里，这就是我们不愿意谈论它，我们把它包围在委婉语和俚语的迷雾中。你的厕所、盥洗室、厕所、厕所、厕所、厕所、泥巴厕所、泥塘，或者不管你叫它什么，都是你日常生活的一部分。

为了他的 Hackaday 奖参赛作品，[VijeMiller]正面解决了他最小的房间。你看，对他来说，这次经历的主要恐惧在于可怕的溅水。是的，一点点抛射能量的释放会让旧的臀部有点潮湿。因此，他用一些制造商的聪明才智解决了这个问题，安装了一个 Arduino 控制的泡沫发生器，注入肥皂和甘油的混合物，用泡沫的飞溅阻尼负载填充碗。避免了向后淹没。

零件清单显示，泡沫是由鱼缸曝气器产生的，由 Arduino Uno 通过功率晶体管驱动的继电器触发。一个电磁阀控制着流量，许多乙烯管把它们连接在一起。有一个 HC/06 蓝牙模块，带有一个应用程序，可以从手机上控制设备，尽管他发布了一些 Arduino 代码，但没有应用程序的链接。有几张照片，包括一个快乐扳手的厚颜无耻的位置，以及一个我们只能猜测是文本的镜头，因为泡沫溢出了整个浴室。他把我们放在广告下方的视频放了出来，幽默地展示了这个设备的运行。
