# 3D 打印电脑机箱让性感回归

> 原文:[https://hack aday . com/2016/01/03/3d-printed-computer-case-brings-sexy-back/](https://hackaday.com/2016/01/03/3d-printed-computer-case-brings-sexy-back/)

我们知道你在想什么——case moding 太过时了。你可以忘记过去看到的所有愚蠢的油漆工作、窗户和发光的旋转风扇。与[Complx] [在 overclock.net 论坛](http://www.overclock.net/t/1537934/scratch-build-node-3d-printed-case)上创造的东西相比，这一切似乎都是小儿科。

使用 3D 打印塑料角和激光切割丙烯酸顶部、底部和侧面的混合物，[Complx]能够创建一个非常令人满意的设计。然而，他没有足够大的 3D 打印机来制造零件，所以他决定外包这项任务。他的第一套零件是在 Makerbot 复制器上打印出来的，但出来的东西太粗糙了，所以他开始寻找更好的打印方法。在得到 2000 美元或更多的报价后，他正准备放弃，这时他发现有人拥有 Stratasys Fotus 250，愿意与他合作降低价格，但仍然提供高质量的打印。

机器的内脏也不算太寒酸。我们知道每个人都喜欢零件清单，所以给你:这是华硕 Z97I-Plus 和 i7-4790K，运行 GTX 970，配备 600 瓦电源，8GB 内存和几个固态硬盘。

我们不得不称赞[Complx]的文档、照片和视频。它真的让这个建筑闪闪发光。休息之后，您可以观看构建的 3d 动画。
