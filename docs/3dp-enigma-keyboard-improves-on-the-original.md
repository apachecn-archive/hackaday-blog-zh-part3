# 3DP Enigma 键盘在原来的基础上有所改进

> 原文：<https://hackaday.com/2017/08/12/3dp-enigma-keyboard-improves-on-the-original/>

之前创造了英格玛机器手表的[Asciimation]决定全力以赴，制造一台 [3D 打印的英格玛机器](http://www.asciimation.co.nz/bb/category/3d-printed-enigma-machine)。不是一个完美的复制品，而是一个*改进的*版本，工作相同，但不关心自己的历史准确性。例如，当前步骤包括构建键盘。他没有试图重新创造原来的弹簧销方法，而是简单地换上了现成的双掷微型开关。

这个项目有大量迷人的细节。他做了研究，结果显示:他下载了原始的蓝图，并使用黑客入侵的数字卡尺来精确测量每个转子的齿，这样它就可以被重新创建用于打印。他甚至重新创造了 Enigma 字体，以确保他印刷的转子轮子看起来正确——尽管在这样做的时候，他发现原来的机器使用一种字体用于键盘，一种用于轮子，一种用于指示灯。

我们之前发表了[Asciimation]的[英格玛机器腕表](http://hackaday.com/2015/03/23/enigma-machine-wristwatch/)项目，他用 Arduino 模拟了英格玛的功能。
