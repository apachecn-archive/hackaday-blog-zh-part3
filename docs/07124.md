# Hackaday 奖品入口:廉价的紧急按钮

> 原文：<https://hackaday.com/2017/09/24/hackaday-prize-entry-inexpensive-emergency-button/>

我摔倒了，爬不起来。我们都记得它，我们都知道我们在谈论什么产品。现在，有了廉价的微控制器、无处不在的无线网络和可穿戴电子设备，这里的*一定是一个更简单的解决方案。[【让·帕瑞德尔】的紧急按钮项目](https://hackaday.io/project/27017-emergency-button-reliable-inexpensive-design)旨在取代那些可穿戴的紧急按钮，这些按钮通常包括昂贵的呼叫中心计划。*

[Jean]的按钮基于 ESP8266 模块，如果按下按钮，该模块会向医护人员发送电子邮件。整个装置由 CR2032 手表电池供电，该设备的外壳是 3D 打印的。界面很简单——毕竟它只是一个可佩戴的按钮——形状因素小到完全不引人注目。

[Jean]用运行项目的简单草图重新绘制了 ESP8266 板。首先，按下按钮将设备连接到 WiFi，然后 LED 闪烁，这样你就知道它已经连接上了。当按下紧急按钮时，会发出一封电子邮件让护理人员知道有问题。
