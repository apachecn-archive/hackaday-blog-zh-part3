# 调查世界上最稀有的游戏机

> 原文:[https://hack aday . com/2018/02/27/investigating-the-world-rare-game-boy/](https://hackaday.com/2018/02/27/investigating-the-worlds-rarest-game-boy/)

去年年初，一种非常独特的游戏本颜色出现在中国购物网站淘宝上。不同于原版 GBC 那种看起来相当苍白的显示器，这个版本用现代 IPS LCD 进行了修改。即使在产品页面上显示的图片中，很明显这款设备的显示屏比任天堂以往任何时候都要先进得多。复古游戏社区疯狂起来，很快网站就被订单淹没了。卖家取消了大部分销售，据信只有 75 个被黑的 gbc 被运出。

[![](../Images/b4268aee0cf78c1b7defee6fb8e89df5.png)](https://hackaday.com/wp-content/uploads/2018/02/raregbc_detail.jpg) 多亏了他的一位观众，[科林]才得以得到这些极其罕见的定制掌上电脑之一。显然，这是一个合我们心意的人，他的第一反应是[把这个东西拆开，看看他们是如何建造它的](https://www.youtube.com/watch?v=evUnemODZBs)。虽然他对他们如何管理这种现代和经典技术的混合有一个相当好的想法，但里面有几个惊喜。

该设备有一个完全股票主板，因此工作和播放像一个正常的 GBC。但是在翻转主板时，[Colin]发现了一窝细电磁线，它将新的显示控制器连接到前面板上的各个按钮。正如他后来在重新组装系统时证实的那样，这允许用户通过按住“选择”并使用方向键来调整显示器的亮度。

至于屏幕本身，最大的惊喜是它显然是从相对较新的智能手机中提取出来的。屏幕实际上比 GBC 前面板上的开口大得多，但是通过一些软件欺骗，图像只显示在玩家可见的区域。[Colin]设法联系了一些“知情人士”,他们向他证实，显示控制器的硬件和软件都是专门为此应用程序开发的，其他人不可能复制。

考虑到我们报道的大多数 Game Boy hacks 都是关于某人[将现代硬件塞入其中](https://hackaday.com/2016/11/12/pi-zero-transforms-to-game-boy/)，看到一个如此坚持保留尽可能多的原始硬件同时仍设法改善用户体验的团体，这是一个有趣的节奏变化。
