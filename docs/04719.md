# 精益思维帮助 STEM 儿童建立一个小型风电场

> 原文:[https://hack aday . com/2017/01/18/lean-thinking-helps-stem-kids-build-a-tiny-wind farm/](https://hackaday.com/2017/01/18/lean-thinking-helps-stem-kids-build-a-tiny-windfarm/)

当我们在高德的车库里看到高德的新作品时，我们永远不知道会发生什么。他似乎无论做什么都很熟练，有很好的设计感和无可挑剔的工艺。你可能会期望他在一个 STEM-outreach 风力涡轮机项目上低调一点，但是当你有机会给 28 个五六年级的学生留下深刻印象时，你可能会努力争取。

[![98j6zp](../Images/be871088d806e3b582c9dc1c8b1c529e.png)](https://hackaday.com/wp-content/uploads/2017/01/98j6zp.gif) 从他女儿的老师提出的每个孩子都可以制作风力涡轮机的想法开始，【高德】应用了一点精益方法，这样孩子们就能在规定的时间内完成制作。设计很简单——几张旧 CD，上面放着 PVC 管的垂直部分，用来捕捉微风，并在四个扁平的电磁线线圈上旋转钕磁铁。这足以点亮一个 LED，或许还能点燃一个孩子的想象力。

尽管涡轮机很简单，但建造它的过程需要尽可能多地去除不必要的工作，而[Gord]在这里大放异彩。他制作了大量的夹具和固定装置，预先制作了一些组件，并为制作的每个步骤设置了组织良好的工作站。每样东西都有清晰的标签，成年志愿者在休息后使用视频进行训练，大家都玩得很开心。

有时候，问题不在于产品，而在于过程，而[高德]成功地解决了潜在的让孩子们失望的灾难。如果领略了[高德]的风格让你想看更多，看看他的[吉他微动夹具](http://hackaday.com/2016/04/27/engineering-meets-craftsmanship-in-this-guitar-fretting-jig/)或他的[制动转子 mancave 时钟](http://hackaday.com/2015/09/19/porsche-themed-mancave-clock-there-is-no-substitute/)。
