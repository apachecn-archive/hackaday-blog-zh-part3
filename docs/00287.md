# 手动转动邪恶的机械南瓜

> 原文：<https://hackaday.com/2015/10/06/hand-cranking-the-malevolent-mechanical-pumpkin/>

见见马蒂。他是一个南瓜，被装上了会动的眼睛、舌头和恶意的喜悦表情。你可能会认为这都是由伺服驱动的，对吗？不:马蒂是由[本·勃兰特]设计和制造的老式曲柄机构驱动的。

他想制造一种可以用手摇曲柄驱动的东西。当然，没有什么可以阻止你在背后扔一个电机来驱动这个机制，但是[Ben]希望内部是防火的，这样他就可以用蜡烛来点燃它。他的装置是由旧的金属线和金属片制成的，不易燃，也不会像电机和电源那样受到热量的不利影响。他取得了令人钦佩的成功，他还出色地记录了创造一个基于南瓜的机械怪物的过程，并提供了方便的技巧。

那些黑客做了一点电子湿工作，你应该现在就开始建造他们的[集成 LED 的杰克灯](http://hackaday.com/2010/10/26/70-led-matrix-in-a-jack-o-lantern/)。这些事情要花很多时间。
