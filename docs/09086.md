# 来自设计师包的 DIY 设计师钱包

> 原文：<https://hackaday.com/2018/04/09/diy-designer-wallet-from-designer-bag/>

为什么人们会花数百美元购买名牌商品？简单的答案是，从理论上讲，这些商品是物有所值的。材料、工艺和对细节的关注都是一流的，是金钱所能买到的最好的服饰。

那么，你会花几百美元买一个名牌钱包吗？如果你有皮革制作技能，附近又有旧货店，[你可以效仿[Corter Leather]的做法](https://imgur.com/gallery/MhgxE)，自己动手制作。他发现了一颗未经雕琢的钻石——一个真正的、备受喜爱的路易威登包，在一家二手商店里日渐凋零。皮革底部和处理是干燥和磨损，但签名 LV 帆布仍然保持良好状态。从未制作皮革？如果你找不到免费的废料来练习，那么解构廉价的二手材料是下一个最好的选择。

为了分离帆布，[Corter]小心翼翼地拆下包的提手、底部、衬里和拉链，然后打开铆钉，从布料上剥离背衬。他在 Illustrator 中绘制了一个图案，向这位杰出设计师的钱包致敬，并使用卡片原料模板从 3 盎司植物鞣制皮革中裁剪出部分。

[科特]带来了他的 A 级游戏的细节。每一个可见的边缘都涂上了意大利红，他用锥子在上面画出了清晰的线条。较大的口袋有隐藏的缝线，防止卡片滑落到底部，脱离形状。没必要打开你的钱包去看他是怎么做的——休息后看看视频就知道了。

虽然从技术上来说，它不是真正的路易威登，但小偷直到后来才知道。也许【科特】应该[加个扒手报警器](https://hackaday.com/2013/03/08/quick-wallet-hack-adds-pickpocket-alarm/)。
