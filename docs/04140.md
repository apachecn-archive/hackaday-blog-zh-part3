# 考虑到安全性而打造的激光效果投影仪

> 原文:[https://hacka day . com/2016/11/17/a-激光效果投影仪-安全制造/](https://hackaday.com/2016/11/17/a-laser-effect-projector-built-with-safety-in-mind/)

在黑暗多雾的夜晚挥舞激光笔有种特别的感觉。看着光束穿过薄雾很有趣——让你感觉有点绝地武士的感觉。如果你不能得到足够的激光和薄雾，你可能想看看这个 DIY“激光天空”效果投影仪。

激光天空效果可能会让你想起其他科幻电影——想想来自*外星人*T3 的[“彩蛋场景”。这种效果是通过激光束在一个平面内扫过旋转的烟雾或薄雾来实现的。激光突出了原本隐藏的气流的横截面，并创造出一些令人神往的画面。[Chris Guichet]的投影仪的工作原理很简单——由一个旧的无刷风扇电机和一个激光指示器旋转的八角形镜子。但在快速验证概念构建后，他添加了额外的东西，将它从原型变成了产品。小小的激光笔被一个 200 兆瓦的激光模块取代，六边形的镜面底座和外壳是 3D 打印的，镜面被精心对齐，因此激光扫描出一个平面。添加了 Arduino 来控制电机并提供安全联锁，以确保只有当镜子达到速度时激光才会发射。深宝石红色激光切割烟雾的效果令人着迷。](https://www.youtube.com/watch?v=m9ebkHoU4Zk)

如果激光天空对你来说有点过于一维，用[这个矢量激光投影仪](https://hackaday.com/2014/09/20/vector-laser-projector-is-a-lesson-in-design-processes/)扩展成二维，如果单色不是你的菜，试试[一个 RGB 矢量投影仪](https://hackaday.com/2011/11/10/rgb-laser-projector-is-a-jaw-dropping-build/)。
