# Hackaday 奖参赛作品:一台手动数控取放机

> 原文：<https://hackaday.com/2017/08/29/hackaday-prize-entry-a-manual-cnc-pick-and-place-machine/>

每个想要 3D 打印机的人可能都已经有一台了，甚至激光切割机和数控机床也正在进入世界各地的车库和地下室。拾取和放置机器是个人制造的下一个伟大前沿，尽管还有很长的路要走，[【TEG wyn】今年 Hackaday 奖的项目](https://hackaday.io/project/20260-manual-smt-pick-and-place-machine)正在使我们更加接近可靠地弹出 0201 LEDs。

这个项目是一个手动拾取和放置机器——也称为“镊子”。这比那要复杂一点，因为[Tegwyn]建造背后的整个想法是将人类的精细运动技能与在板上放置组件的能力分离开来。为了做到这一点，这个项目是使用现成的，蓝光特殊数控机床。没有太多，只是一点铝挤压和一些螺纹杆。然而，通过添加真空泵、空心针和一些手动控制来移动轴，操作员可以非常精确地控制电阻器、帽或 LED 的位置。

“在数控机床上安装真空泵”的想法有一些巧妙的补充。这是一个 4 轴机器，使用户能够围绕一个垫旋转零件。机器上还安装了一个小型监视器，上面连有一台显微镜。如果你要组装数百块电路板，这不是你想要的机器。然而，如果你只需要少数几个，不介意花几个小时放置零件，也不想疯狂地使用小 QFN 封装，这是一个很好的构建，也是 Hackaday 奖的一个很好的参赛作品。
