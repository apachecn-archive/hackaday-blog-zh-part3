# 带状连接器

> 原文：<https://hackaday.com/2016/09/27/ghetto-ribbon-connector/>

[Marcel]试图将一些新部件塞进他值得信赖的 Nexus 5 手机。如果你曾经打开过这些小玩意中的一个，你就会知道引擎盖下已经没有多少空间了。拉出一些不必要的部分(如耳机插孔)可以节省一些空间，但如何将它们连接起来呢？

[Marcel]需要一种尽可能细的多线连接器，但他不想走 order-Kapton-flex 路线。哦不！他自己用胶带和一根绞线做了一个。如果仅仅这样还不够的话，请观看视频“如何做”。

由于电线是不绝缘的，[马塞尔]有点小心翼翼地用胶带分开每一股。虽然[Marcel]在视频中制作了一个直连接器，但我们可以很容易地想象制作一个预制电缆，就像手机附带的大规模生产的柔性电缆一样。总而言之，在空间稀缺的时候，这是一个锦囊妙计。

我们确信你们中的一些绕线专家会忍不住让电线松弛下来，但是你能想象如果只增加一些外围设备，手机内部会是什么样子吗？
