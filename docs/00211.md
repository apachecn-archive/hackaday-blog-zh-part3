# 最小的大螨虫

> 原文：<https://hackaday.com/2015/09/29/minimal-mighty-mite/>

如果你正开始建造自己的业余无线电设备，很难想象有比 Mighty Mite 更低技术含量的发射机，但[Paul Hodges，KA5WPL] [更进一步，推出了自己的可变电容器](https://www.youtube.com/watch?v=24BzwsPC-wU)。(就是你在左边看到的带胶带和鳄鱼夹的啤酒罐。)

一只巨大的螨虫根本算不上是一台收音机。一个晶体管、电容器、晶体和电感器以一束电线的形式缠绕在一个药瓶上，形成了一个极简振荡器，然后通过开关控制这个开关，你就可以发送莫尔斯码。[solder smoke 播客](http://soldersmoke.blogspot.com)的[Bill Meara]一直是 Mighty Mite 的热情拥护者，他建议可以通过从一台旧的模拟电视机中搜寻 3.57954 MHz[color burst 晶体](https://en.wikipedia.org/wiki/Colorburst)来制造这种 Mite，这种晶体可以将收音机调谐到业余无线电爱好者的合法频率。(它也可能与你的废品箱中的其他低 MHz 晶体一起工作，但不一定是合法的。)

如果晶体是“容易”清除的，收音机的其余部分是容易自制的，调谐电容器(可从旧的 AM/FM 收音机中获得)可能成为症结所在。所以[保罗]切开了两个铝“饮料”罐，用绝缘胶带把里面的一个包起来，接上电线，自己做了一个可变电容器。通过将罐子滑入或滑出，使它们或多或少地重叠，他可以将收音机精确地调到水晶的自然频率。

如果你对制造一个强大的螨虫感兴趣，你绝对应该看看关于焊烟的话题。网上有更多的[构建说明](http://www.qsl.net/wb5ude/kc6wdk/transmitter.html)以及可选的[滤波器的计划，如果你有野心的话，可以去掉谐波](http://makerf.com/posts/altoids_tin_low_pass_filter_for_shortwave_transmitters)。

如果你不是一个莫尔斯电码奇才，我们不禁要指出，你可以用一个简单的场效应晶体管(我们会使用 2N7000，但无论如何)来代替钥匙，然后你就可以在微控制器的控制下得到无线电。在 [Hackaday 最近的莫尔斯电码项目](http://hackaday.com/?s=morse+code)中搜寻想法，我们相信你会有好的发现。
