# 一个有俄罗斯方块的 YouTube 用户计数器

> 原文:[https://hack aday . com/2018/06/19/a-YouTube-subscriber-counter-with-a-Tetris-twist/](https://hackaday.com/2018/06/19/a-youtube-subscriber-counter-with-a-tetris-twist/)

当谈到 YouTube 用户计数器时，没有多少创造性的回旋余地。当然，你可以选择 Nixies 或者更奇特的展示，但是最终计数器只是一堆数字。

但是[布莱恩·拉夫]找到了一种方法，用这个玩俄罗斯方块的 YouTube 子计数器来增加趣味。对于那些不熟悉[Brian]频道的人来说，这真的值得一看。他倾向于长时间的实时视频，在这种情况下，他为一个马拉松会议的一个项目工作，从他的虚拟肩膀上偷看可以学到很多东西。这个项目源于一个更早的视频，在休息后发布，它本身是几个会议黑客与 RGB 矩阵的浓缩，将形成这个项目的显示。他迷上了便宜且容易获得的 64×32 像素 RGB 显示器，并借用了 [Mc Lighting](https://hackaday.com/2018/04/14/mc-lighting-takes-the-pain-out-of-blinking/) 作者【托布卢姆】的一个想法，他认为从落下的俄罗斯方块组装数字将是一个不错的转变。[Brian]在让 ESP8266 完成获取 subs 和更新显示的工作之前，必须将俄罗斯方块的代码移植到 Arduino。我们认为展示看起来很棒，事实上这个库是开放的，这意味着你也可以将俄罗斯方块动画添加到你的项目中。

这并不是说更传统的副柜台就不酷。从极简的展示到记录你所有社交媒体的 T2，好的设计无处不在。添加[一个实心铜制播放按钮](https://hackaday.com/2017/06/14/celebrating-a-subscriber-milestone-with-a-copper-youtube-play-button/)也是一个不错的选择。
