# 获奖后:温杜伊诺

> 原文:[https://hackaday.com/2016/11/02/after-the-prize-vinduino/](https://hackaday.com/2016/11/02/after-the-prize-vinduino/)

在我看来，Hackaday 奖中最好的项目是最古怪的。用沙子建造一台计算机是一个令人钦佩的目标，极坐标 3D 打印机非常棒。这些项目显然效用有限，而且无论如何也无法解释品味。一个项目有多成功的真正证明，是它能在多大程度上转化为产品。销售产品需要社会证明，去年我们在 Hackaday 奖中引入了最佳产品竞赛。这个想法很简单:做一些其他人想要的东西，你将赢得在供应框架设计实验室的实习机会，把你的项目变成一个产品。

去年 Hackaday 奖最佳产品竞赛的获胜者是来自[Renier van der Lee]的 [Vinduino](https://hackaday.io/project/6444-vinduino-a-wine-growers-water-saving-project) ，这是一个葡萄园节水灌溉项目。在过去的一年里，这个项目取得了一些惊人的成功，节约了大量的水，并证明了自己是 Hackaday 奖的优秀参赛作品。

Vinduino 背后的想法很简单——你应该只在葡萄藤需要水的时候灌溉它们。这与根据时间表开关的无声灌溉系统形成对比。细节决定成败，那么葡萄酒是如何做到这一点的呢？

Vinduino 背后的基本想法是在一些葡萄藤的根部放置一个水分传感器。价格和耐用性在这里是最重要的，需要一些真正有趣的技术。[雷尼尔]开发了一种非常有趣的石膏土壤湿度传感器,它既便宜又精确，足以测量葡萄藤根部周围的湿度。

这些传感器中有几个连接到 Vinduino 电子模块。由于几个远程射频 LoRa 模块和使用太阳能为电池充电的能力，Vinduino 成为了一个测量葡萄树根球湿度的设备。当根干了，灌溉就开始了。当湿度传感器开始发送良好的读数时，灌溉关闭。这非常简单，而且节约了大量的水。

对于任何一种节省资源的设备，都必须有某种计算方法来比较系统的成本和节省的成本。在过去的一年里，[雷尼尔]的 Vinduino 装置为他的葡萄园节省了 430，000 加仑的水，或节省了 1，925 美元的成本。整个 Vinduino 装置仅花费 635 美元。这是一个立竿见影的投资回报，也是其他一些葡萄园已经注意到的。已经有一些葡萄园建立了自己的葡萄酒网络。[雷尼尔]一直致力于向网络添加额外的仪器[，如风速计](https://hackaday.io/project/6444/log/42144-to-spray-or-not-to-spray-adding-anemometer)和 [LoRa/LAN 网关](https://hackaday.io/project/6444/log/46095-vinduino-lora-lan-gateway)。

Vinduino 的生态系统正在扩大，很多人都注意到了这一点。这是一个节约用水的好方法，巨大的投资回报使它成为一个好产品。这是去年 Hackaday 奖的最佳奖项之一，也是开放硬件和专用社区所能做的一个很好的例子。
