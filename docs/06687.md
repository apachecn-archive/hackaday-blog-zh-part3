# 死虫焊接 LED 环的真棒

> 原文:[https://hack aday . com/2017/08/02/dead-bug-soldered-led-ring-of-awesome/](https://hackaday.com/2017/08/02/dead-bug-soldered-led-ring-of-awesome/)

有时候生活中最简单的东西也是最美的。[The Tweaker]已经在 atmega 328 p 芯片顶部焊接了一个 LED 圈，看起来很棒。

仅仅使用一些焊料，电线，20 个 Pico 0402(1 毫米 x 0.5mm 毫米)蓝色 led 和一个 atmega 328 p(7 毫米 x 7 毫米)，[The Tweaker]成功地将 20 个 led 塞进芯片顶部的一个圆圈中，以死虫方式焊接。该芯片正在运行一些 Arduino 代码，并在 8 MHz 内部~~晶体~~振荡器上运行，因此设法保持部件数量较少。焊接是以螺旋形完成的，因此 LED 端子被连接到正确的引脚上，但这似乎增加了项目的美感，看起来好像需要一只真正稳定的手。一旦你连接电源，它会显示追逐灯和其他灯光模式。

这个项目可能没什么大不了的，但看起来确实很棒。
