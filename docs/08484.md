# 你可以打印的维修:修复老鼠攻击的麦克风线

> 原文：<https://hackaday.com/2018/02/05/repairs-you-can-print-fixing-a-rat-attacked-mic-cord/>

我们都经历过——巴布亚新几内亚雨林中的一个潮湿的夜晚，汗水淋淋的睡眠被明显的啃咬声所打扰。早上，我们发现一只味道异常的啮齿动物将我们收发器的麦克风电缆咬成了两半，使我们与大本营失去了联系。如果每次发生这种事我们都有 5 分钱。

这听起来可能不太可能，但这就是[Marius Taciuc]的 3D 打印麦克风线修复背后的背景故事。即使是更常见的故障模式，麦克风上的伸缩线也是出了名的难以修复。几乎任何常见的可疑物，如[热缩管](http://hackaday.com/2017/04/13/heat-shrink-tubing-and-the-chemistry-behind-its-magic/)或电工胶带，一旦坚硬的外壳破裂，都无法恢复失去的机械稳定性。[Marius]的解决方案是打印尽可能小的外壳，以机械方式支持拼接。配合很紧，但有足够的空间来焊接电线，并将所有东西放回原位。电缆扎带可以消除线缆从拼接处出来时的应力，并在接头处注入大量的热胶水。它并不完美——我们敢打赌，拼接作为一个捕捉点，过一会儿就有点烦人了——但如果它让你又快又便宜地回到广播中，它可能是有意义的。

[Marius]将这种激烈的黑客攻击加入了[修复你能打印的竞赛](http://hackaday.com/2018/01/16/win-big-prizes-with-repairs-you-can-print/)。你有没有通过 3D 打印机实现的史诗般的修复？让全世界都知道这件事，你可能会赢得一个奖项。
