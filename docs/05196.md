# 特斯拉 Model S 电池组拆卸

> 原文：<https://hackaday.com/2017/02/28/tesla-model-s-battery-pack-teardown/>

在过去的几年里，我们听说了很多关于特斯拉 Model S 的事情，这是一款具有新闻价值的车型。作为一款快速豪华电动轿车，根据车型不同，每次充电的里程可超过 300 英里，它的宣传是值得的，这还没有提到~~自动驾驶~~驾驶辅助。即便是目前最优秀的量产电动汽车与之相比也相形见绌。

众所周知，特斯拉用标准的 18650 锂离子电池制造电池组，但可以肯定地说，Model S 中的电池组与你的笔记本电脑电池没有什么共同点。对于那些好奇的人来说，幸运的是，[Jehu Garcia] [发布了一段视频](https://www.youtube.com/watch?v=NpSrHZnCi-A),显示 EV West 的人们从一辆报废汽车上拆下了一个 Model S 包，因此我们可以跟随他们了解它的构造。

关于这个包最明显的事情是它的绝对大小，这是一个大项目，占据了车下的大部分空间。我们展示了上一代特斯拉包进行比较，这是小得多。令人眼花缭乱的性能和范围是有代价的，我们正在眼前看到它。

建造标准看起来确实非常高，这是有道理的，因为这不仅仅是一个性能部分，而是一个安全关键部分。被火灾困扰的手机用户会证明这一点，特斯拉的火灾或电气危险能力也相应更大。机箱和外盖由大量的螺栓和梅花螺钉固定在一起，正如他们所说，每一个都标记为已拧紧到特定的扭矩设置。

在盖子下面是第二个被粘住的盖子，这需要小心地撬开以显示模块和它们的电池。冷却剂被排出，模块被断开。最后一项任务尤其危险，因为电池组以非常低的阻抗提供数百伏的 DC。然后可以小心地取出十六个包装中的每一个。每个电池组包含 444 个电池，电池组电压为 24 V，储存的能量为 5.3 kWh。
