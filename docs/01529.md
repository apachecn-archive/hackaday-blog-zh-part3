# 逆向工程:输电线路

> 原文：<https://hackaday.com/2016/02/21/retrotechtacular-transmission-lines/>

Tektronix 在 20 世纪 60 年代中期制作的这个很棒的老视频(嵌入在断点下方)涵盖了一个似乎让人们感到非常困惑的主题——传输线。我们在 Paul Carbone 的博客上找到了它，这是一个为老模拟示波器爱好者提供的很好的网站。

正如许多这些老视频一样，以今天的标准来看，节奏有点慢，但最终呈现的材料质量超过了值得努力控制你的多动症的程度。对于预览，你可以跳到结尾，在那里他们对所有的材料进行回顾。

他们[从 5:31 开始，一个脉冲沿着导线对](https://www.youtube.com/watch?v=I9m2w4DgeVk&t=5m31s)传输，并采用一种非常真实的方法来计算线路的特征阻抗:如果脉冲是由 9V 电池产生的，那么有多少电流在流动？如果导线的 DC 电阻为零，那么根据欧姆定律，应该有一个无穷大的电流，而这显然没有发生。这激发了标准分析，将导线分解为分布电感和电容。

当然，他们会做这样的实验，将一个脉冲注入一个长长的同轴电缆环路，然后摆弄线路另一端的终端。他们还测量生产线的速度系数。我们唯一的抱怨是，他们没有在不同的地方敲击线路来演示驻波。好消息是[我们已经有了 YouTube(和【w3aew】)来应对这个](https://www.youtube.com/watch?v=M1PgCOTDjvI)。

如果你有 23 分钟的空闲时间，对传输线路感到好奇，或者只是享受一名训练有素的电台播音员读出各种终端电阻值的舒缓声音，这个老宝石就是你的门票。尽情享受吧！
