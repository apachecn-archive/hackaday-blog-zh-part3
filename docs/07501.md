# 用人工智能听幽冥

> 原文:[https://hack aday . com/2017/10/24/用人工智能听阴间/](https://hackaday.com/2017/10/24/listen-to-the-netherworld-with-artificial-intelligence/)

又到了一年中的这个时候，随着万圣节被认为是黑客的完美节日，我们开始看到一个幽灵主题的项目出现了。大多数似乎都是为了让一些原本平淡无奇的万圣节装饰品变得可怕而令人敬畏，但这是不同的——使用人工智能来搜索鬼魂。

看起来似乎[马特·里德]的“深度耳语”项目是为了在幽灵季中获得轻松的乐趣，但他偷听幽灵对话的努力可能有一丝严肃性。这背后的原理是电子语音现象(EVP)，新陈代谢和/或维度上的挑战据称会影响电子系统，导致大量处理的音频剪辑似乎有来自逝者的耳语般的喜爱或来自恶意灵魂的威胁。DeepWhisper 更进一步，使用树莓 Pi 将音频输入到谷歌云语音 API 进行分析。如果任何东西是用谷歌知道的 110 种语言中的一种低声说的，它就会显示在屏幕上。[Matt]计划在纳什维尔名副其实的 Butchertown 区设立 DeepWhisper，并在下周直播结果。

看看谷歌的神经网络从它可能只会听到的随机噪音中做出什么将会很有趣。而且[Matt]正计划发布他的代码给所有人看，所以可能会有一些有价值的云技术可以向 DeepWhisper 学习。但是万一他真的发现了鬼魂，很高兴知道你可以拥有打击他们的工具和天赋。
