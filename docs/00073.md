# 构建 3D 打印激光管

> 原文：<https://hackaday.com/2015/09/13/building-a-3d-printed-laser-tube/>

一个名为[1 creature]的 YouTuber 用户正在做一些出色的研究，以构建他自己的 3D 打印激光管。

这一切都是在他设法找到一个极高电压的脉冲驱动器后开始的。在他的视频中，他向我们保证永远不要买。然后，他为一个透明的丙烯酸管设计了一些 3D 打印配件，允许气动和电气连接。使用手泵在[内部制造真空是最困难的](http://www.youtube.com/watch?v=bm-G2g2mxg8)，所以他很快升级到一个合适的工业真空泵。

施加真空，打开脉冲驱动器…然后…你自己看吧！

现在制造一个自制的二氧化碳激光器已经不是什么新鲜事了，[技术已经存在很久了，](http://hackaday.com/2006/12/24/make-your-own-co2-laser/)我们已经看到了一些[五金店的激光器，它们非常棒，](http://hackaday.com/2014/05/05/building-a-co2-laser-in-a-hardware-store/)，但是我们不得不说，我们对其中大部分是 3D 打印的这一事实印象深刻。

我们可能很快就会看到一些 3D 打印的激光切割机！
