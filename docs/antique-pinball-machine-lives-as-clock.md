# 古董弹球机作为时钟生活

> 原文：<https://hackaday.com/2017/06/03/antique-pinball-machine-lives-as-clock/>

修复旧街机或弹球机的一个大问题是找到原始部件让它们再次运行。不过，这也是乐趣的一部分；经过几周或几个月的努力，事情终于成功了。另一方面，有时候，对于那些再也不会出现在弹球机中的旧零件，唯一的希望就是[兰迪]能碰到它们。其中一个零件是一台旧机器的后玻璃，他决定把它变成一个独特的单词钟。

最初的弹球机建于 1956 年，尽管年代久远，但后玻璃几乎没有磨损或损坏的迹象。这台特殊的机器上有 43 盏灯，足够用 12 小时、10 分钟、10 秒钟和一些额外的时间。ATtiny85 充当控制器，驱动隐藏在显示器中的一群新像素。还有三个按钮控制亮度和设置时间。

一定要看看下面这个独一无二的时钟的视频。更多的东西也投入到了这个构建中，包括给玻璃加框，给它涂上一层油漆和抛光，以及将时钟编程到微控制器中。弹球机的旧玻璃酒杯似乎也相对受欢迎，被重新改造成更传统的时钟，甚至原子钟。
