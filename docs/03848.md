# 升级隔离变压器

> 原文：<https://hackaday.com/2016/10/16/upcycle-an-isolation-transformer/>

有几个原因让你应该有一个隔离变压器。它们可以防止接地环路，也可以防止被测器件通过 DC 路径接地(或者将示波器与 DC 地隔离，这本身就很危险，但这是另一个问题)。[Tanner_tech]注意到，随着越来越多的路灯转向 LED 技术，为钠蒸汽路灯寻找镇流器变压器变得越来越容易。如何处理这些变形金刚？[当然是建一个隔离变压器](http://www.instructables.com/id/Industrial-Isolation-Transformer-From-Trash/)。

当然，你的垃圾箱转换器可能与帖子中显示的略有不同(以及下面的视频)。[Tanner]展示如何找到你需要的线索。一个小木工和一个电脑电源盒完成了这个项目。

从评论来看，有些人认为[坦纳]关于安全的言论是在暗示变压器使市电工作变得安全。并没有。如果你知道自己在做什么，会更安全。在高压下工作不是边做边学的地方。

如果你想要一些实用的建议，[Jenny List]为你准备了一本好书。你可能也应该花一个小时看一下[这个视频](https://hackaday.com/2011/05/02/isolation-transformer-101/)，里面有很多实用的建议。
