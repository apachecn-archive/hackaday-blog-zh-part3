# 用于 SMD 显微镜的无线环形灯

> 原文:[https://hack aday . com/2018/06/30/wireless-ring-light-for-SMD-microscope/](https://hackaday.com/2018/06/30/wireless-ring-light-for-smd-microscope/)

当广受欢迎的 Moteino 电路板(最初是无线 Arduino 兼容机)的制造商费利克斯·鲁苏(Felix Schmidt)说，他已经为他的 SMD 显微镜制作了一个无线环形灯时，我们重定向按键来看看。当然，这有点像他的文字游戏。他所做的是制造一种新的环形灯，这种灯使用电池而不是恼人的电线连接到墙上的电源插座。这对于长时间俯身在显微镜前的人来说很重要。哦，他把环形灯建在了一个非常漂亮的 SMD 板上。

该板提供几种电源配置。通常情况下，他用连接在显微镜后面的 1650 毫安的锂电池供电。电池可以使用 USB 或通过 DC 插孔充电，电路板上有一个位置，尽管他还没有焊接一个。必要时，他可以从 USB 或 DC 插孔为灯供电，但到目前为止，他一次充电的时间超过 6 小时，足以满足 SMD 会话。

下面的视频展示了他的 SMD 板制造过程，从在 Eagle 中绘制电路板，激光切割模板孔，粘贴，填充电路板，进行回流，以及一路上的各种技巧。看看吧，这是令人愉快的观赏。

这是另一个显微镜环形灯，带有[可选的照明模式，用于去除那些讨厌的阴影](https://hackaday.com/2013/04/03/microscope-ring-light-with-a-number-of-different-features/)。哪些功能会让您的 SMD 会议变得更容易？
