# 重温贝尔格莱德会议

> 原文：<https://hackaday.com/2016/05/23/relive-the-hackaday-belgrade-conference/>

贝尔格莱德会议取得了惊人的成功。要证明这一点，你只需要看看我们在过去几周公布的一系列会谈就可以了。每个人都从不同的角度看待硬件世界；创造硬件意味着什么，我们去过哪里，我们要去哪里，下一个伟大设计的灵感来自哪里？

这些谈话现在已经全部发表并被收集到一个视频播放列表中。这是紧张的一天，所有的谈话都陷入了一种疯狂的状态。但是如果你不能一次完成，我也列出了休息后的个人演讲，这样你可以挑选。

然而，今天下午刚刚发表了两篇演讲。这是亚历山大·布拉迪奇的开场白和我的结束语。当我们遇到别人时，我们经常被问到幕后发生了什么。很容易认为没有人关心召开一次会议、进行一次令人惊叹的工程挑战需要什么，或者我们如何决定在展望未来时我们认为重要的事情。Alek 讲述了 Supplyframe 和 Hackaday 如何走到一起的背景故事，以及是什么让我们选择贝尔格莱德作为此次会议的举办地。我讨论了我认为黑客日的核心优点；自由开放地分享信息和想法。这是我相信的一个概念，也是记录你的工作的最高尚的理由，以便其他人可以利用你的知识和技能。

贝尔格莱德超出了我们的想象。它加入了 2016 年超级大会(其[演讲视频也已发布](https://www.youtube.com/playlist?list=PL_tws4AXg7asvl-x6VCKqxJsAuFnEWq-8))，成为我们强大、积极和参与的社区的光辉典范，这些社区希望花时间帮助每个人——黑客、设计师和工程师——取得成功。
