# 黑客日入侵多伦多

> 原文：<https://hackaday.com/2016/04/14/hackaday-invades-toronto/>

下周三，4 月 20 日，[黑客日将入侵多伦多](https://hackaday.io/event/10851-hackaday-prize-worldwide-toronto-2)。

[索菲·克拉维茨]和[迈克尔·吉尔福尔]正前往北部边境，在[黑客实验室](https://hacklab.to/)与我们的朋友会合。他们将举办一个有饮料、小吃和礼品的聚会。

由于这是一个带来一个黑客，与会者被鼓励带来你正在做的任何项目，展示它，给一个闪电谈话，并向社区推销它。[【Sophi】最后一次去多伦多](http://hackaday.com/2015/04/24/hackaday-visits-toronto-canada/)带来了一些疯狂的硬件，包括[为准将 64](http://hackaday.com/2013/11/14/commodore-64-power-glove-is-so-bad/) 准备的游戏手套、[安大略省发电厂的演示](http://steamlabs.ca/2015/03/ontario-power-system-education/)，用 PCR 测试汉堡中任何不是牛肉的东西，以及[模拟合成器](http://www.kilpatrickaudio.com/?p=phenol)。

由于 Hackaday 奖正在如火如荼地进行，这是一个与多伦多人合作获得大奖的绝佳机会，或者只是向人们征求一些想法，看看你的想法是否可行。

Hacklab 的见面会是免费的，但我们要求[您回复活动](https://www.eventbrite.ca/e/hackaday-hacklab-bring-a-hack-meetup-2-tickets-24106579428?aff=ebrowse)。活动[也在 Hackaday.io](https://hackaday.io/event/10851-hackaday-prize-worldwide-toronto-2) 上，以防你想和【索菲】或其他与会者聊天。
