# 当斯特林发动机遇到 3D 打印机

> 原文：<https://hackaday.com/2018/04/14/when-stirling-engines-meet-3d-printers/>

让我们面对它，每个人都想建立一个斯特林发动机。它们很精致，而且一般都很棒。它们也是一个相当复杂的制造项目，这就是为什么你在周围看不到它们的原因。

这并没有消除所有的复杂性，但通过这个例子 [3D 打印一个斯特林发动机只是大约一半的可能性](http://www.doityourselfgadgets.com/2017/08/3d-printed-sterling-motor-holder.html)。这一款使用 3D 打印技术制作框架、安装支架和飞轮。那个轮子的大部分质量来自放置在轮子周围的一组金属螺母。这个简单的使用蜡烛的概念验证在休息后的视频中展示，它也升级到了集成的丁烷火焰。

斯特林发动机依靠热量运行，使得打印塑料部件在某些方面无法使用。但这个设计中的非打印部分是我们见过的最简单的部分，包括一个玻璃注射器、一个玻璃圆筒和连接两者的硅胶管。圆筒和注射器的推拉是由蜡烛火焰产生的热空气和空气通过管道移动时的自然冷却引起的交替运动。

我们会说，这一个刚刚超过这些引擎的优秀等级的中间点(考虑到它是多么平易近人，这很好)。在精英方面，这里有一个 16 缸的艺术品。天平的另一端可能看起来不那么漂亮，但是没有什么比[只用垃圾做的聪明建筑更让我们开心的了。](https://hackaday.com/2013/11/29/sterling-engine-kludged-together-from-whatever/)
