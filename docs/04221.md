# ZX81 连接到网络

> 原文：<https://hackaday.com/2016/11/23/zx81-connects-to-the-network/>

[维克多特鲁科]让我们希望我们说葡萄牙语。他做了很多逆向计算项目，包括将 ZX81 连接到互联网以加载程序。该项目使用 ESP8266 来获得 WiFi 通信。想锻炼高中葡萄牙语可以看下面一个视频。

具有讽刺意味的是，ZX81 的 CPU 一直忙于驱动视频，读取键盘，运行频率仅略高于 3 MHz，在该处理器上甚至没有转化为 3 MIPS。与此同时，“仆人”ESP8266 有一个运行在 80 MHz 的 32 位 Tensilica CPU。时代变了。

该界面提供了一个控制网页，用于通过普通 PC 或其他具有网络浏览器功能的设备进行配置。ZX81 使用 LOAD 命令与远程 FTP 服务器交互。视频中提到的 TK85 显然是[一个巴西 ZX81 克隆](https://en.wikipedia.org/wiki/TK85)。

你可以花很多时间在特鲁科的网站上与谷歌翻译较劲。我们以前见过一些 [TK85 黑客](https://hackaday.com/2015/05/20/colorizer-for-zx81-clone/)。我们也报道了[特鲁科]在[雅达利](https://hackaday.com/2009/12/16/2600-game-jukebox/)上的一些工作(尽管[那篇文章中的链接已经移动了](http://www.victortrucco.com/Atari/AtariJukebox/atari_jukebox))。
