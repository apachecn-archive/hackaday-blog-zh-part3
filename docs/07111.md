# 用由管道零件制成的金刚石钻头切割石头

> 原文：<https://hackaday.com/2017/09/15/cutting-stone-with-a-diamond-bit-built-from-plumbing-parts/>

大家最喜欢的加拿大人又来了。这一次，[AVE]需要在石头台面上凿一个大洞。他们为此制造取芯钻头，但这种尺寸的钻头要花费 400 美元以上。没问题。[AvE]拿出工具[，自己造了一个石头切割钻头](https://www.youtube.com/watch?v=mYphOR0Jegw)。

一切从一个 6 英寸的塑料管盖开始。中心在盖子上钻孔，然后拧上螺纹。一个向下的螺栓为这个新工具做了一个很棒的柄轴。然后在瓶盖的边缘开槽。[AvE]使用一台破旧的 Bridgeport 铣床，但你可以用钢锯或 Dremel 工具完成同样的工作。

秘方是工业钻石。没错，这是钻石切割钻头。【AvE】订购 20 克 20-25 目工业钻石。“网眼”定义了单颗钻石的大小——在这种情况下，大约为 50 微米或以上。现在，钻石和塑料如何粘合？水管工的过渡水泥不起作用——钻石和涂层就像晒伤一样剥落了。解决方案原来是 JB-Weld。在工具表面涂上一层 JB-Weld，撒上一些工业钻石，管帽就可以切割了。

切割过程缓慢而稳定，需要大量的冷却水。[AvE]在不得不重新装修他的部分之前，完成了他的工作台面的大部分。

【AvE】通常是一个话多的人，[从这篇帖子中可以看出关于他的电火花机床](https://hackaday.com/2015/06/24/machine-metal-with-electricity-an-edm-attachment-for-3d-printers/)。然而这一次，他给了我们无声的待遇——一整段没有歌词的视频，配上了古典音乐。很高兴看到 YouTubers 走出他们的舒适区，尝试新事物。
