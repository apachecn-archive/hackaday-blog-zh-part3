# 随着点头声摇滚起来

> 原文:[https://hackaday.com/2017/12/10/rock-out-with-the-nod-bang/](https://hackaday.com/2017/12/10/rock-out-with-the-nod-bang/)

在 Hackaday 的这些年里，我们已经看到了相当多的音乐黑客。他们甚至有自己的类别！(专业提示–您可以在类别部分的下拉菜单中找到它)。但是这一个拿蛋糕。[Andrew Lee]是纽约大学的一名学生，他的任务是为他的物理计算课创建一个项目。在大约 60 天的时间里；他从餐巾纸素描到工作项目。这个项目非常有趣——他制作了一种乐器，当你摇动你的头时[就会播放音乐](https://www.vndrewlee.com/posts/portfolio/nodbang/)。

它如你所料的那样工作。用户耳机中的加速度计将数据传送给 arduino。有四个按钮(当然是 3D 打印的)用于选择播放的音频类型。操作是这样的:

1.  按下按钮。
2.  爆炸头。

[Andrew]谈到听到音乐与头部运动节奏同步播放时的特别满足感。一定要看看下面的视频，看看 Nod Bang 的效果。
