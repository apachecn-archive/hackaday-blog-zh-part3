# 达斯·维德，在数码管里

> 原文：<https://hackaday.com/2017/02/01/darth-vader-in-a-nixie-tube/>

这可能是一个有争议的说法，但谢妮地铁在我们的社区已经有点过时了。随之而来的是另一个时钟项目，哦，看！它有谢妮管，而不是 7 段显示器或液晶显示器。曾经有一段时间，这种重新发现的古老组件很酷，但面对现实吧，伙计们，它已经被淘汰了。还是有？

因此，鉴于对无处不在的 Nixies 的不满，你可能会认为没有一个谢妮项目能重新点燃这种兴奋。这可能是真的，直到下面的视频打破了我们的方式。[托拜厄斯·巴图施]制作了他自己的数码管，代替数字的是[一个[达斯·维德]的 3D 模型，配有移动的光剑](https://www.youtube.com/watch?v=_E4IXxzuVsI)。突然之间，尼西人的世界又变得有趣了。

广告下方的第一个视频向我们展示了地铁的运行。我们从各个角度看到[维德]和他的光剑。下面是第二个视频，这是一个详细的建设故事。不过要注意，这是一个相当长的问题。

这个模型是通过精心塑造并将 Kanthal 金属丝点焊到雕塑上制成的，在这个过程中(正如[Tobias]所说)你需要像氖等离子体一样思考。然后，它被包裹在笼状结构中，形成它的另一个电极。他带领我们完成了玻璃外壳的制作过程，电线组件就放在这个玻璃外壳中。结果是一个稍微有点线框但非常容易辨认的[维德]，和一个独特的管。
