# 可爱的四足机器人跳跃和行走

> 原文：<https://hackaday.com/2016/01/29/adorable-quadruped-robot-hops-and-walks/>

目前，ESP8266 正在各种项目中找到自己的路。诚然，这是一个功能强大的小设备，但我们不得不说，发现它运行[一个能跳能跑的四足机器人](https://hackaday.io/project/9334-kame-esp8266-based-quadruped)有点出乎意料。让它以如此可爱的设计出现也很酷。

从[哈维尔·伊莎贝尔]的建造日志来看，他在[卡米]身上花了很多心思。所有的身体部分和连接都是由 PLA 3D 打印而成，并添加了对比色。这些腿由八个高速 Turnigy 伺服系统提供动力，连杆中使用了高质量的轴承。NodeMCU 使用自定义振荡器算法来控制各种步态，包括跳跃运动。BOM 甚至列出了“直径 12 毫米的粘性眼睛”——也许这是某种俚语，更准确地说是“眼睛”

主要是作为研究不同步态的测试平台而建造的，在[Kame]目前的化身中似乎没有太多的传感器。但是引擎盖下有一个 ESP8266，自主操作的可能性很大。我们期待看到这个项目的下一步走向。我们拿可爱的因素开玩笑，但永远不要怀疑一个吸引人的设计激发创造力的力量。

我们之前已经介绍过很多四足机器人，其中很多似乎都趋向于可爱的一端。看看[这只正在学走路的小狗狗](http://hackaday.com/2014/10/22/baby-quadruped-robot-learning-to-walk/)或者[这只以为自己是小狗的小狗狗](http://hackaday.com/2013/10/18/quadruped-robot-thinks-its-a-puppy/)。
