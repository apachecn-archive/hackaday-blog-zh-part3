# 演讲者科学项目

> 原文：<https://hackaday.com/2015/11/18/speaker-science-project/>

他们可能不会取悦发烧友，但[KJMagnetics]向你展示，可以用一些磁铁、一些塑料杯和一些磁线制作一对扬声器。当然，用垃圾制造扬声器并不是一个新想法。然而，这个版本也有一些令人愉快的地方。也许是杯子的对称性，或者是木工的工作台造型。

我们不禁认为这将成为一个很好的科学展览项目或课堂活动。尤其是因为有一篇关于说话者如何工作的很好的文章，而且很容易做出简单的改变来测试关于说话者的不同假设。例如，线圈中的导线多了或少了会发生什么？什么磁铁效果最好？最好是什么意思？声音大吗？失真少？

如果你想听听他们的声音，下面有一个扬声器的视频。我们忍不住开始画看起来很怪的耳机。然而，如果啤酒乒乓杯对你来说太普通了，你可能更喜欢用硬盘做一个扬声器。或者，如果你想使用现成的扬声器，我们认为[这些将成为一个很好的雨天项目](http://hackaday.com/2012/08/09/sound-o-light-simple-and-interesting-diy-speakers/)。
