# 这种卫生棉条枪不会束缚你的风格

> 原文：<https://hackaday.com/2017/01/17/this-tampon-gun-wont-cramp-your-style/>

最后，有一种方法可以摆脱那些几乎没有人使用的无涂抹器卫生棉条，同时还可以销毁一堆 Axe 身体喷雾。在 3D 打印的气动止血枪中，使用斧头作为推进剂即可。

正如你将在休息后的组装和演示视频中看到的,[ harambelabs]模块化枪支设计中的大部分零件都是 3D 打印的。除此之外，你只需要添加一个 PVC 管作为桶，一个适合身体螺纹的瓶子，以及一对 o 形环来形成一个漂亮的，紧密的密封。从打火机上取下压电装置，在瓶子里装满斧头云，然后把它拧到机身上。如果气体/空气混合物足够接近，压缩的棉花子弹应该会飞起来。这种枪是单发的，但[harambelabs]正在研究一种使它全自动的模式。

我们喜欢这里的好枪法，不管它是最温和的还是最恐怖的 T2。这种构造不一定依赖于卫生棉条，但其尺寸、重量和塑料覆盖(减少摩擦)使其成为这种特殊设计的理想选择。如果你能找到合适的枪管，Nerf 省道可能是另一个选择。
