# Hackaday 奖参赛作品:用于摄影的便携式 LED 闪光灯

> 原文:[https://hack aday . com/2017/04/21/hack aday-prize-entry-portable-led-flash-for-photography/](https://hackaday.com/2017/04/21/hackaday-prize-entry-portable-led-flash-for-photography/)

摄影是关于光的。从字面上看，它就在名字中——来源于希腊语， *photos。这就是为什么摄影师会纠结于一天中的拍摄时间，为什么 Instagrammers 会在日落时聚集在地标周围，为什么闪光灯会帮助你在黑暗中拍照。历史上，闪光灯以各种方式工作——例如使用燃烧的镁光灯或氙灯。[为了这次 Hackaday 奖的参赛作品，【Yann Guidon】正在开发一款使用 led 的便携式闪光灯](https://hackaday.io/project/20910-portable-led-flash)。*

到目前为止，你可能已经很熟悉手机上的 led 闪光灯了。然而，[Yann]将此提升了一个档次。该建筑基于大约 100 瓦的 LED 模块，显然可以发出大量的光。有趣的部分是它的双重性质。led 旨在以两种方式之一工作。第一种是在连续照明模式下，在远低于其额定功率的情况下运行模块，以减少 led 和电源的压力，并使闪光灯能够运行大约一小时。在这种模式下，温度反馈将用于控制 led 来管理电源使用。另一种是脉冲模式，LED 将被过压几毫秒，以产生更强大的闪光。

正是这种双重性使得基于 LED 的闪光灯比基于氙气的闪光灯具有潜在优势，后者仅限于脉冲操作。我们可以看到，连续照明模式对于需要紧凑、廉价的照明解决方案的摄像师特别有用，这种照明解决方案也可以作为脉冲单元使用。随着项目的发展，我们很高兴看到[Yann]如何解决包装、热量和控制问题！
