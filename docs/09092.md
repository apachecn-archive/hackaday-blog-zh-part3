# 装饰艺术风格的收音机接收调频信号

> 原文:[https://hack aday . com/2018/04/07/art-deco-radio-gets-FM-reception/](https://hackaday.com/2018/04/07/art-deco-radio-gets-fm-reception/)

拿着一台老式收音机，把它塞满现代的、与互联网相连的东西，这一直是黑客和制作界的一个主要话题。虽然有些人可能会认为这是一种犯罪，把一段可以说是合法的历史变成一个草率的覆盆子 Pi 案例，但我们不得不承认这种想法有一定的吸引力。将经典设计之美与更现代的功能相结合，可谓两全其美。

但是这个由[Nick Koumaris] 设计的项目在某种程度上是一个独特的概念。他没有牺牲一个真正的老式硬件来容纳电子设备，而是设计了一个 3D 打印的外壳，看起来像一个经典的 1936 年 AWA 收音机。但是对我们来说，*真正有趣的是他在里面放了一个基本的调频收音机。*

没错，这里没有网络电台流或智能手机蓝牙兼容性。这只是一个普通的调频收音机，并不完全不同于你所期望的经典收音机中的那种硬件。当然，它更现代，而且[Nick]实际上是用 TEA5767 FM 无线电模块和 Arduino Pro Mini 自己建造的。

虽然从功能上来说，它可能不是非常令人兴奋，但我们确实很欣赏他不辞辛苦地为收音机制作了一个复古外观的用户界面。虽然物理按钮可能更适合这个时代，但手机诺基亚 5110 LCD 上显示的装饰艺术风格的字体和图形看起来确实很光滑。

纯粹主义者肯定会很高兴看到另一个项目，其中一件老式的音频设备没有被牺牲在 Hack 的祭坛上，但我们也主办了许多项目，这些项目与历史保护几乎没有关系。
