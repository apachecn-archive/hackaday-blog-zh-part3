# 花 6 美元建造一个喷砂设备

> 原文：<https://hackaday.com/2018/01/19/build-a-sandblasting-rig-for-6/>

让我们弄清楚一些事情:是的，这假设您已经拥有或可以使用压缩机。所以*如果你做了*，并且知道你要做什么，[为什么不造一个便宜的喷砂设备](https://www.instructables.com/id/600-Sandblaster-That-Actually-Works/)？这就是[adamf135]在 YouTube 上看到有人这么做后的做法。他严重怀疑这是否可行，但结果令人印象深刻。

这个只需要一个 20 盎司的空瓶子，一个便宜的气枪/喷嘴和一个适配器。这种方法最难的部分似乎是在喷嘴上为爆破材料切割一个凹槽，而不是完全切断它。[Adam]从他身上切下 1/2″的部分，但是这么大的开口真的用光了爆破材料。他建议缩小规模。剪断密封圈后，他将喷嘴穿过瓶子最坚固部分钻的 3/16 英寸的孔，并用热胶将其密封。休息后，观看它通过生锈和碎玻璃油漆供电。

如果你像这样做任何公开喷砂，确保至少戴上口罩。如果你不想把细小的颗粒喷得满店都是，你可以建造一个[湿介质喷砂柜](https://hackaday.com/2015/04/26/diy-wet-media-blast-cabinet/)来代替，或者更低技术的[建造一个钻头驱动的零件翻转机](https://hackaday.com/2014/09/01/a-peach-of-a-homemade-parts-tumbler/)。
