# Chromecast 复古电视很神奇

> 原文：<https://hackaday.com/2016/04/15/chromecast-vintage-tv-is-magic/>

当 Moddnstine 博士]在垃圾桶里看到一台 1978 年的通用电气电视时，他不得不挽救它。事实证明，它仍然有效！一个想法出现了——如果他能把它变成一台老式的 Chromecast 电视会怎么样？

他打开电视，开始在里面四处张望。我们应该注意到，如果你不熟悉里面的组件，打开旧电视是非常危险的——一些电容器上残留的高电压会杀死你。Moddnstine 博士]没有讲太多细节，所以在你打开电视之前做一点额外的研究。

他这个项目的部分目标是让所有东西都独立于电视之外，这样你只需要把它插到墙上就可以使用了。由于电视太旧了，它甚至没有视频输入的模拟 RCA 连接——只有 VHF 输入。因此，他需要使用三个独立的连接适配器将视频信号传输到电视上。

第一步是将 HDMI 适配器连接到复合视频，然后将复合视频连接到 RF 调制器，再将 RF 调制器的同轴输出连接到双引脚 VHF 插头适配器。唷。我们很惊讶他在箱子里找到了足够的空间来装下所有的东西！
