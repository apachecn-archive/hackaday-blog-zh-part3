# 周五黑客聊天:开源芯片

> 原文：<https://hackaday.com/2017/04/12/friday-hack-chat-open-source-silicon/>

本周五，Hackaday.io 将迎来开源芯片供应商。[加入我们的 Hackaday.io Hack Chat](https://hackaday.io/project/5373-hack-chat) 本周五，4 月 14 日中午(美国太平洋时间 19:00)与 si five——一家“开放”的芯片制造商——进行对话。

本周，我们采访了 si five T1，这是一家无晶圆厂半导体公司，也是 HiFive1 的制造商，hi five 1 是一款开放式硬件微控制器，你可以直接去购买。去年年底，SiFive 发布了 HiFive1，[，这是 SiFive 的 FE310 片上系统](http://hackaday.com/2017/01/05/hands-on-with-the-first-open-source-microcontroller/)的一个 Arduinofied 化版本。这个 SoC 是一个 RISC-V 内核，也是第一批完全开源的微处理器之一。这是对 Stallmanism 的冒犯，Stallmanism 是我们对真正开放硬件的最大希望，而且启动速度相当快。

SiFive 不仅仅致力于开放式硬件微控制器，他们的商业计划非常类似于“OSH Park，but for silicon”。如果你有一种新型芯片的设计，他们将与铸造厂合作，把你的设计变成一个可爱的小环氧树脂浸渍的斑点。这是一个令人着迷的商业计划，你将在本周五的黑客聊天中听到所有关于它的内容。

### 以下是参与方式:

[![join-hack-chat](img/cff5733f39a173302e1cd102e75ac107.png)](https://hackaday.io/project/5373-hack-chat) 我们的 Hack Chat 是 Hackaday.io 上的实时社区活动 [Hack Chat](https://hackaday.io/project/5373-hacker-channel) 群发消息。

登录 Hackaday.io，访问该页面，并寻找“加入这个项目”按钮。一旦你成为这个项目的一部分，这个按钮就会变成“团队信息”，直接带你进入黑客聊天。

你不必等到星期五；随时加入，你可以看到社区在谈论什么。
