# 硅片传输机非常昂贵

> 原文：<https://hackaday.com/2016/11/30/silicon-wafer-transfer-machine-is-beautifully-expensive/>

没有什么比成为一名看不到预算的工程师更自由的了。[BrendaEM]向我们展示了一台在如此缺乏约束条件下设计的机器的拆卸过程。它坐在一个大盒子里，这个大盒子的工作是一边接收硅片，另一边吐出集成电路。

[BrendaEM]从未真正透露她是如何得到如此昂贵的东西，以至于工程师可以指定“精密烧结金属柱末端的微型光纤棱镜”作为晶片的中断解决方案。然而，我们很高兴她这么做了。

这台机器有许多你能想到的特点；昂贵的超精密马达、丝般平滑的线性运动系统等等。在某一点上，她打开了一个抓手运动，它运动的声音可以被充分地描述为诗意。

这也给了我们一个意想不到的视角，让我们看到以我们期望的可笑的价格生产我们日常依赖的硅是多么具有挑战性。从陶瓷板和可以处理烤箱中硅的热量的钳口，到这个频繁使用的装置的明显清洁度。

这是对我们大多数农民都没有被邀请到的昂贵世界的一次罕见的审视。休息后的视频。
