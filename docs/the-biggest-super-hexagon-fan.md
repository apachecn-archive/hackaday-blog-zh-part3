# 最大的超六边形风扇

> 原文：<https://hackaday.com/2015/12/01/the-biggest-super-hexagon-fan/>

对于那些还没有沉迷于超级六边形的人来说，退一步说，这是相当…上瘾的。通常这款 80 年代的街机风格游戏会在浏览器中运行，但布宜诺斯艾利斯的 Club de Jaqueo 的一些人决定将所有这些都塞进 Arduino。然而，他们并没有就此止步，而且[认为 POV 显示](http://alecu.protocultura.net/games/electromechanical/2015/11/28/super-ventilagon-premiere/)的效果最好。

为了在将 POV 显示与这样的快节奏游戏相结合的复杂迷宫中导航，该小组转向了值得信赖的 Arduino Micro。在最初的想法受挫后，他们意识到这个游戏非常适合 POV 显示，因为它几乎是圆形的。视点[不应该占用太多 Arduino 的处理能力](http://hackaday.com/2013/11/18/a-pov-spinner-display-with-arduino/)，所以大部分时钟周期可以用于玩游戏。由于缺少六边形形状(可能还有版权和其他法律障碍)，他们不能再保留原来的名字，但原来的风格保留得很好。

该小组在上周末展示了他们的设置，从下面的视频来看，结果令人印象深刻。他们还发布了他们的源代码和原理图，以防你有一个老粉丝([或者甚至是一辆自行车？](https://hackaday.com/2014/12/29/bike-rim-lighting-lets-the-night-crowd-know-when-youre-rollin/))躺在那只求被变成迷你街机游戏。
