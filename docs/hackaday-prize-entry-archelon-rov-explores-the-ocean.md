# Hackaday 奖参赛作品:Archelon ROV 探索海洋

> 原文：<https://hackaday.com/2017/08/18/hackaday-prize-entry-archelon-rov-explores-the-ocean/>

Acendtech Robotics 是一家位于新泽西州弗里霍尔德的 4H 机器人俱乐部，他们的核心项目是 Archelon，这是一款由 PVC 管制成的[水下无人机。这也是一个 Hackaday 奖项，旨在监控海上交通，海底，码头，码头和其他水下建筑。](https://hackaday.io/project/26029-archelon-rov)

Archelon 使用八个由舱底泵构成的推进器，这些舱底泵被黑客攻击后增加了一个螺旋桨，使发动机安全地密封在内部。

ROV 的电机由 Arduino Mega 和两个电机驱动板控制，每个驱动板驱动两对 DC 电机。还有一个由另一个改装的舱底泵旋转的机器人爪，由防水伺服系统打开和关闭。包括 Teensy 3.2 在内的机载电子设备密封在 1/2”丙烯酸管内，该管由橡胶 o 形环和定制的不锈钢端盖密封。连接到 Teensy 的是 ROV 的摄像头和 ATTiny88，它们反过来控制马达。

与 Archelon 一起工作的学生不仅学习建造 ROV 的技术方面，如组装和编程，还学习它的任务，学习如何采集琼脂的测试样本，以研究海洋环境中的污染物。
