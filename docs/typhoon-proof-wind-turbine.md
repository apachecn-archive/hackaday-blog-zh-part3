# 防台风风力涡轮机

> 原文：<https://hackaday.com/2017/10/09/typhoon-proof-wind-turbine/>

虽然风能在世界范围内的市场份额迅速增加，但是风力涡轮机并不能在所有需要的地方建造。日本就是一个很好的例子，那里传统的风力涡轮机会被台风损坏。然而，在福岛灾难之后，一名日本工程师承诺专门为日本建造一台能够在飓风风力下正常运转的涡轮机。(YouTube，下面嵌入。)

众所周知的“台风涡轮机”是通过马格努斯效应(Magnus effect)工作的，在这种情况下，一个旋转的物体一边比另一边更快地引导空气绕过它。这种涡轮机使用三个马格努斯效应驱动的汽缸，每个汽缸上都有一个叶片，这使得涡轮机无论风速多高都可以收获能量。飓风和台风的问题不仅仅是风，还有风吹向哪里。虽然没有提到它的抗冲击性，但它看起来肯定是尽可能坚固的。

希望这种涡轮机能够在日本流行起来，这样他们就可以减少对其他类型能源的依赖。风能最近变得令人难以置信地受欢迎，包括在携带便携式风力发电机的徒步旅行者中，甚至在只有几块废料的人中。
