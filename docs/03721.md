# Hackaday 奖参赛作品:树莓 Pi Zero 智能玻璃

> 原文:[https://hack aday . com/2016/10/08/hack aday-prize-entry-raspberry-pi-zero-smart-glass/](https://hackaday.com/2016/10/08/hackaday-prize-entry-raspberry-pi-zero-smart-glass/)

近年来，一些更有趣的消费类硬件设备是智能眼镜。像谷歌眼镜或 Snapchat Spectacles 这样的设备，这种眼镜结合了显示器和计算能力，可以在一个不显眼的可穿戴平台上传递信息或提供增强现实。

[Raspberry Pi Zero 智能眼镜](https://hackaday.io/project/13332-raspberry-pi-zero-smart-glass)旨在提供进入这个世界的入口，由 Raspberry Pi Zero 提供的一副眼镜具有图像识别和 OCR 文本识别功能。不同寻常的是，它没有采用其他设备的显示选项，即在用户的视野中使用镜子或棱镜，而是用显示器代替用户的整个视野，并通过 Raspberry Pi 摄像头将用户重新连接到世界。

有问题的显示器是一套廉价的“3D 虚拟立体数字视频眼镜”，这种眼镜在你最喜欢的拍卖网站上很容易找到。它们的分辨率不是特别高，但 Pi 可以通过其复合视频输出轻松驱动它们。电子设备和摄像头安装在一个定制的 3D 打印外壳中的头带上。所有文件都可以从项目页面下载。

有一些 Python 软件，但公平地说，在项目页面上没有清晰的演示来显示它的工作。然而，这并不是忽视这个项目的理由，因为即使它的软件尚未实现其全部潜力，在其他地方也有价值。3D 打印的 Raspberry Pi 外壳应该可以用于许多其他类似的可穿戴项目，我们几乎可以说它值得一个独立的项目。
