# 健身轮追踪 Kitty 的健身目标

> 原文:[https://hack aday . com/2016/03/20/exercise-wheel-tracks-kittys-fitness-goals/](https://hackaday.com/2016/03/20/exercise-wheel-tracks-kittys-fitness-goals/)

几周前,[贾斯珀·鲁本]给他的猫造了一个巨型仓鼠健身车。他们似乎很喜欢它，所以他决定加大赌注，升级它。这个轮子现在有一个树莓 Pi，可以跟踪速度，计算行驶距离，并确定[贾斯帕的]猫同伴的等效跑步速度。

为了计算车轮的速度，[Jasper]正在使用一个小线圈传感器(类似于自行车速度计的工作原理)。轮子上的六个钉子触发了传感器。一旦数据载入 Pi，一些简单的计算就可以让[Jasper]提供一些不同的指标来衡量 cat 练习的效果。网络摄像头在线发送车轮的实时流和数据叠加，供您观赏。

真正酷的部分是，这个设置让[Jasper]跟踪猫何时(以及是否)真正使用锻炼轮。到目前为止，他们似乎每天使用六次，平均每次步行一分钟——通常在早上 5 点到 8 点之间。那比我们自己的一些运动习惯还要好！
