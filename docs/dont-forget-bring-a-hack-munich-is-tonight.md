# 别忘了:今晚是慕尼黑

> 原文：<https://hackaday.com/2017/05/05/dont-forget-bring-a-hack-munich-is-tonight/>

如果你这个周末在德国慕尼黑，你有一个可爱的黑客可以炫耀，渴望啤酒和/或好极客公司，那么你很幸运！快来参加 [muCCC](https://muc.ccc.de/Treffen) 的 Hackaday，赢取 Hackaday 大奖。

地点是[schlei heimer 街。41](http://www.openstreetmap.org/search?query=N48.153701%20E11.560801#map=18/48.15283/11.56260) ，从 theresienstrae U-Bahn 沿着 hestrae 向西走一小段路。不需要预订，但如果你能在评论中告诉我们你要来(或者更好的是，点击活动页面右上角[的“加入活动”按钮)，我们会很高兴，这样我们手头就有足够的披萨了。](https://hackaday.io/event/21452-hackaday-prize-munich)

派对在 20:00 开始，并非完全巧合的是，慕尼黑车展的参展商设置在[关闭。所以，如果你正在搭建一个展台，请到城市的另一边，在那里你可以向一群经过挑选的黑客伙伴展示一个小项目。如果你只打算参加慕尼黑车展，这是一个很好的热身。](http://make-munich.de/english/)

如果你有想要展示给世界的便携产品，哈卡戴(埃利奥特·威廉姆斯)会在那里拍照。否则，放松一下，和志趣相投的人一起出去玩。需要一个时间和地点来召集团队参加 [Hackaday 奖](https://hackaday.io/prize)？给你，啤酒！(或者 Spezi，但是没有和 Spezi 押韵的。)

再次感谢慕尼黑 CCC 的东道主。
