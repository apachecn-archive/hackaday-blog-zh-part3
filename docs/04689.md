# CES2017:面向大型构建的真正快速 3D 打印

> 原文：<https://hackaday.com/2017/01/08/ces2017-really-fast-3d-printing-for-large-builds/>

大约一年前，欧特克展示了近年来最具创新性的细丝打印机之一。埃舍尔项目是你的基本笛卡尔细丝打印机，但是有一个变化:它有五个头。这些打印头协同工作，非常快速地构建大型对象。

 [https://www.youtube.com/embed/51xhIV8taqw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/51xhIV8taqw?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



Autodesk 开源了埃舍尔的设计，现在多亏了 Titan Robotics，它已经进入商业生产。与 Project Escher 使用相同软件的克洛诺斯很大！这些台架中的每一个都由闭环伺服电机和花式滚珠丝杠驱动，产生 77”x30”x20”的总构建体积。这个露天版本正在解放军印刷。如果您想将它用于环境温度成为问题的材料，可以选择封闭的构建环境。

这款打印机将于 2 月开始发售，但还不知道价格。他们宣传构建量是可定制的，所以我们希望价格是一样的。
