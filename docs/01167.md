# 强大的弩几乎完全是 3D 打印的

> 原文:[https://hack aday . com/2016/01/12/powerful-crossbow-is-almost-all-3d-printed/](https://hackaday.com/2016/01/12/powerful-crossbow-is-almost-entirely-3d-printed/)

事实证明，打印一整把弩还不可行。但是[【Dan】的 crossbow build](http://diydudes.com/3d-printed-crossbow/) 很好地利用了 3D 打印机的优势。大多数打印部件位于弩的扳机组中，文章中的图表清楚地显示了扳机、扣机和保险是如何相互作用的。特别好的是保险的自动性质，通过拉回绳子来接合。我们还喜欢印刷的弹簧，它可以把缰绳固定在适当的位置，还有 Picatinny 挂杆，用来安装瞄准镜。非印刷部分包括股票中使用的铝管，弓本身，一个 PVC 管内的玻璃纤维棒复合设计。下面的视频展示了十字弓的作用，它看起来非常强大。

事实上，我们将部分收回我们之前对完全 3D 打印弩的否定，但[Dan]的版本比这个模型更实用和有用。对于更传统的弩设计，看看[这款完全手工制作的弩](http://hackaday.com/2015/10/23/making-a-crossbow-for-the-zombie-apocalypse/)。
