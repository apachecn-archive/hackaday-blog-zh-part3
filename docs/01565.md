# 滑雪伙伴夹克使用 Arduino 教年轻人滑雪

> 原文：<https://hackaday.com/2016/02/22/ski-buddy-jacket-uses-arduino-to-teach-youngsters-to-ski/>

滑雪看起来很简单，对吗？你只要站着，重力就会起作用。现实情况是滑雪对初学者来说很难学。[19 karpawich]喜欢滑雪，但他很沮丧地看到哭泣的孩子在滑雪板上，尖叫的父母试图指导他们。受可穿戴电子产品的启发，他拿了一个 Arduino，一件旧夹克，一些 led 灯，[创造了滑雪伙伴](http://www.instructables.com/id/Ski-Buddy-the-Coat-That-Teaches-You-to-Ski/)。

夹克里的大脑由一个 [Adafruit Flora](https://hackaday.com/2012/01/21/flora-a-better-arduino-lilypad/) 、加速度计和电池组组成。导电线连接到 LED 亮片。这件夹克可以帮助教授连接转弯、平行滑雪、曲棍球停止和逐渐停止比萨饼。除了制作细节和一些关于不要放置传感器的注意事项(毫无疑问是通过艰苦的努力学来的)，[19 mmarpawich]还详细解释了软件和如何使用夹克。

你可以在下面看到一个很短的滑雪伙伴的视频演示。最近，我们看到了更多的可穿戴设备，其中一些[非常有创意。如果你还不会缝纫，也许是时候学学了。](https://hackaday.com/2016/01/10/warm-up-your-small-talk-with-a-thermometer-scarf/)
