# PolaPi-Zero 提供非常好的即时照片

> 原文:[https://hackaday.com/2017/02/18/polapi-zero/](https://hackaday.com/2017/02/18/polapi-zero/)

PolaPi 中的“宝丽”揭示了这个 Hackaday.io 项目是什么。由[Muth]创造的这款类似拍立得的相机，是一种黑白的，混合了现代 3D 打印的过去的爆炸。它基于一个带有摄像头模块的 Raspberry-pi Zero，一个用于查看图像的夏普内存 LCD，以及一个用于打印实际照片的纳米热敏打印机。加上一些按钮、一块电池和一个光滑的 3D 打印外壳，你就有了自己的 PolaPi。

现在已经是第二次迭代了，因为 Muth 把第一个原型给了某个幸运的人。由于他必须从头开始重建整个相机，他利用了他在第一个原型中学到的知识，并对其进行了改进。这款相机的液晶屏幕显示速度为“实时”20fps，你可以拍摄照片、查看照片，如果你喜欢，还可以打印出来。打印的照片出奇的好，休息后在视频里看看吧。

目前，该软件正在积极开发中，除其他外，最新版本具有狭缝扫描模式。对于那些不知道的人来说，狭缝扫描摄影是一种可以创造一些疯狂扭曲和迷幻效果的技术(在这种情况下，就像黑白照片一样迷幻)。

我们知道你自己也想要一个。如果你不想花时间安装和配置你的 RPi Zero，[Muth]友好地分享了一个准备好的 SD 卡映像。
