# AvE Builds DRINKO(不隶属于 PLINKO)

> 原文:[https://hack aday . com/2015/09/16/ave-builds-drinko-not-affiliated-with-plinko/](https://hackaday.com/2015/09/16/ave-builds-drinko-not-affiliated-with-plinko/)

[SuperUnknown]又名[AvE]，我们最喜欢的加拿大黑客之一[带着 DRINKO](https://www.youtube.com/watch?v=36tIdgt3cOc)回来了，这是一款成人饮料主题游戏，采用了来自*的经典 PLINKO 游戏，价格合适*。他开发了这个游戏作为送给朋友的曼彻斯特暖化礼物。这不是一个特别复杂的构建，但是看到一个项目中的所有小步骤，直到完成工作，总是很棒的。[SuperUnknown]说木材将是这个项目的绝佳材料，但他反对无谓地杀害和平的树木，所以他用 1/8″钢板建造了这个基座。这些玻璃杯是普通的铅玻璃，经过遮盖和蚀刻，拼出 DRINKO。

制作这种游戏最乏味的部分是切割和安装尖齿。[SuperUnknown]用过的旧焊条，在他的桥港用纵切锯切割。棒被 TIG 焊接到形成游戏背板的金属板中。为了增加趣味，[SuperUnknown]增加了一个 Arduino 和一些通孔 WS2812 LEDs。虽然他手头没有平面贴装 WS2812，但这并没有阻止他。在布里奇波特的一次快速旅行中，那些磨砂 LED 透镜变小了。Arduino 通过几种模式驱动 LEDs 很像视频游戏中的吸引模式，或者拉斯维加斯的标志。如果你自己制作饮料，我们建议在每个槽下增加一些微型开关，这样喝的饮料就会发光。
