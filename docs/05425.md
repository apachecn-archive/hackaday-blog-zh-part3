# 廉价和冒险的 EDM

> 原文：<https://hackaday.com/2017/04/04/edm-for-the-cheap-and-adventurous/>

激光切割机、喷水器、等离子切割机、数控刳刨机——大多数黑客空间，甚至许多专业的家庭游戏玩家似乎都有某种将片状物品切割成复杂形状的奇特工具。但是由于无法使用数控机床，并且需要从金属板上切割出复杂的形状，[炼金术士匕首]发明了这个简单但有些危险的电火花加工设备来完成这项工作。

放电加工已经有几十年的历史了，并被大量用于较硬的金属，如钛和工具钢。这一过程对任何见过触点因反复电弧放电而凹陷和腐蚀的人来说都是有意义的——电弧被用来从工件上去除金属，电介质流体被用来冷却工件并冲走碎片。为了[炼金术士匕首]的目的，许多复杂的改进，如高频电源和精确的工具定位，都被忽略了。他建造了一个简单的线性滑轨来手动控制工具的位置，电源只是一个连接到 120 伏电源的桥式整流器，带有一些滤波电容器和一个作为镇流电阻器的大灯泡。虽然下面的视频显示了电导管被开槽，[炼金术士匕首]还制作了一个黄铜饼干切刀式的工具来切割钢的 Instructables 标志。

显然，混合水电是一个灾难的配方，如果你不小心，但这种低端的电火花加工技术是一个很好的文件，以备不时之需。如果您想在自制的 EDM 设备中寻找更多的复杂功能，[我们也能满足您的需求](https://hackaday.com/2016/08/07/homemade-edm-can-cut-through-difficult-materials-like-magnets-with-eease/)。
