# 受浮游生物启发的流动光艺术

> 原文:[https://hack aday . com/2018/01/01/flow-light-art-灵感来自浮游生物/](https://hackaday.com/2018/01/01/flowing-light-art-inspired-by-plankton/)

有了今天的技术，艺术可以朝着以前不可能的方向发展。利用这一点，来自日本的艺术集体[teamlab]展示了一个艺术装置，将与会者融入到这个奇观中。在这件作品的暗室中，[运动产生漩涡，漩涡产生运动，](https://www.teamlab.art/w/vortices/)“你是描绘流动画面的画笔。

灵感来自海洋浮游生物的运动，这借用你的运动来创建带有镜子墙壁的光挂毯，以增强效果。当与会者在房间里走动时，他们的动作被跟踪并转换成流动的图案投射到地面上。人们移动得越快，产生的流量就越大。即使是那些停下来欣赏风景的人，他们自己也是其中的一部分；它们的闲置形式模仿了河中的巨石——就像漩涡在障碍物周围搅动一样，与会者周围的光流也是如此。
