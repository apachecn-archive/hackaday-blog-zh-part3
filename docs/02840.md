# 200W 激光火箭筒就是傻

> 原文：<https://hackaday.com/2016/06/30/200w-laser-bazooka-is-just-silly/>

我们不打算做这个，因为，嗯，这太荒谬了。但是你们中已经有足够多的人通过给我们发提示来恐吓我们，说我们将要投降。我们不会指名道姓，因为真的，你们应该为自己感到羞耻。但是你知道你是谁！

[Styropyro]在 YouTube 上做了很多疯狂的事情。我们非常喜欢他的“微波炉里的东西”系列。他还痴迷于激光和爆黑气球。因此，他从四台 DLP 电脑投影仪(装有 24 个 1.5W Nichia 二极管的投影仪)中取出激光头，并将它们组合在一起。是的， ~~405~~ 445nm 蓝色的 [200W。](https://www.youtube.com/watch?v=IzUoe-9bKa0)

然后他把它们绑在一起，通过镜头。不是紧束，但是这个东西*真的*亮。尽管光束聚焦非常松散，但它会烧掉一些东西。你能说的大概就这些了。很多激光。男孩你好！

好了，我们运行了。不要在家里做这个。这不需要太多技巧，而且会让某人失明。把你的努力花在更文明的东西上要好得多，比如投影仪。至少这样你就可以在墙上玩矢量游戏了。离我的草坪远点！！！(现在的孩子……)

对于那些确实想烧东西的人，[约书亚·瓦斯奎兹]昨天发表了一篇关于[建造一个安全的激光切割机](http://hackaday.com/2016/06/29/taming-the-beast-pro-tips-for-designing-a-safe-homebrew-laser-cutter/)的文章……比任何宣传为激光火箭筒的东西都更值得你花费精力。
