# 小小的发射器把你体内的间谍带了出来

> 原文：<https://hackaday.com/2018/05/18/tiny-transmitter-brings-out-the-spy-inside-you/>

说到监控，为什么让政府独享所有乐趣？这个小小的间谍发射器正是你启动娱乐间谍活动所需要的东西。

当然，我们是孩子——如果你选择建造[TomTechTod]的小型发射器，你会希望遵守法律。仅仅比 337 纽扣电池大一点，废弃的 PCB 封装了相当数量的表面贴装元件，大多数是 0201 封装。即便如此，发射器的设计也很简单，两个晶体管音频级放大来自 MEMS 麦克风的信号，并馈入一个振荡器，该振荡器使用表面声波(SAW)谐振器来实现稳定性。窃听器针对 433 MHz 低功率设备频段进行了调谐，从下面的视频来看，它似乎具有随机线天线的体面范围——可能是 50 米。[TomTechTod]发布了所有构建文件，包括 Gerbers 和带有 Digikey 零件号的 BOM，因此为您的 fieldcraft 套件创建一个应该很容易。

如果你想深入电子间谍的世界，孩子，我们会帮你搞定。这里有一本关于麦克风窃听器的初级读本，一部关于间谍无线电的 T2 历史，或者 T4 如何使用反向散射来窃听大使馆。
