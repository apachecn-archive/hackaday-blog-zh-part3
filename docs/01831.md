# YAGDO——又一个车库开门器

> 原文：<https://hackaday.com/2016/03/25/yagdo-yet-another-garage-door-opener/>

似乎每隔几个月我们就会报道另一个车库门开启器，这个概念正迅速成为 DIY 家庭自动化的“Hello World”。在这一期中，reader [ray] [制作了自己的车库门开启器](http://rayshobby.net/introducing-opengarage/)，并选择 ESP8266 作为无线接口，但通过一个检测汽车是否在车库中的超声波传感器和一个显示历史的 web 应用程序，以及与 Blynk 的集成来进行远程访问，增加了应用程序的趣味性。为了子孙后代，他[也让这个项目开源](https://github.com/opengarage)。

该视频制作精良，包含大量细节和说明，电路板和组件精致而干净。这可能是一个“你好世界”，但它是正确的。

我们去年报道的其他一些车库门黑客包括[指纹扫描开启器](http://hackaday.com/2016/03/16/fingerprint-garage-door-wont-open-every-time-a-neighbor-microwaves-a-burrito/)、[即时通讯开启器](http://hackaday.com/2015/06/08/hacking-the-im-me-to-open-garages/)、[基于运动的安全开启器](http://hackaday.com/2015/02/23/another-garage-door-opener-this-time-with-security/)、[猫启动开启器](http://hackaday.com/2015/02/17/automatic-garage-door-opener-takes-the-cake/)、 [OpenCV Pi 开启器](http://hackaday.com/2015/02/15/a-raspberry-pi-garage-door-opener/)和[蓝牙低能耗开启器](http://hackaday.com/2015/01/02/a-bluetooth-garage-door-take-three/)。
