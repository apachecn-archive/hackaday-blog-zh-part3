# 用纳米粒子 3D 打印铝

> 原文：<https://hackaday.com/2017/09/25/3d-printing-aluminum-with-nanoparticles/>

我们热爱我们的 3D 打印机。但有时我们真的希望能用金属打印。虽然金属打印对我们大多数人来说仍然遥不可及，但 HRL 实验室[宣布了一种粉末铝打印工艺](http://www.hrl.com/news/2017/0920/)，他们声称这是一项突破，因为它允许打印(和焊接)以前不可打印和不可焊接的高强度铝合金。

关键是用特殊的锆基纳米粒子处理金属。纳米粒子充当成核位置，允许铝形成正确的微结构。关于这一过程的完整论文发表在《自然》杂志上。

除了纳米颗粒，该工艺是传统的激光烧结金属粉末工艺。这并不新鲜。然而，[铝是以合金的形式出现的](https://en.wikipedia.org/wiki/Aluminium_alloy)，并不是所有的合金都适合所有的用途。一些常用于高强度的合金是不可焊接的，一般来说，不能焊接的金属也不能激光烧结。HRL 声称 7075 和 6061 都可以用这种方法印刷。据推测，尽管不同的材料可能需要不同种类的纳米粒子，但其他金属也可以从这种纳米粒子处理中受益。

我们对他们在视频中展示的塑料部件的过度挤压感到畏缩，但希望这只是一个库存视频。当然，这不会让[家庭金属印刷变得负担得起](https://hackaday.com/2016/05/19/its-time-for-direct-metal-3d-printing/)。在那之前，你将不得不满足于[电镀](https://hackaday.com/2015/08/23/metal-3d-printing-with-your-printer/)或者你可以用[低温金属](https://hackaday.com/2013/11/07/3d-printing-with-metal-at-home/)印刷。
