# 像游戏机一样说话，像哔哔声一样刺痛

> 原文:[https://hack aday . com/2017/03/12/talk-like-a-game-boy-sting-like-a-beep/](https://hackaday.com/2017/03/12/talk-like-a-game-boy-sting-like-a-beep/)

你有没有听过一首歌，想知道他们是如何创造出机器人声音的？有很多种方法可以做到这一点。[scyt 1005]决定从摇滚历史中获得灵感，[创造了一个游戏机驱动的对话盒](https://twitter.com/i/moments/836410275272105984)(日语，[谷歌翻译推荐](https://translate.google.com/translate?sl=auto&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=https%3A%2F%2Ftwitter.com%2Fi%2Fmoments%2F836410275272105984&edit-text=)给那些不讲日语的人)。

当来自声带的振动通过嘴、舌头和称为“发音器官”的其他身体部位的运动形成可理解的声音时，就产生了人类语言。talkbox 通过使用发音器官来创建机器人语音，同时用替代来源来替换来自声带的振动。

音箱是电吉他最常用的设备。来自电吉他的信号被放大，并通过扬声器或换能器播放，扬声器或换能器连接到放置在用户嘴里的管子。然后，用户开始用口说出他们想要说的话，由吉他的信号而不是声带提供振动。这方面一个流行的例子是皮特·弗朗普顿在《你觉得我们有同感吗？

[scyth 1005]在他们的设计中使用了相同的基本骨骼，使用游戏机将声音输入基本的音频放大器套件和连接到管道的转换器。这给人声带来了非常 80 年代的合成音。这是一个简单的概念，但我们以前没有见过很多。使用现成的模块，您可以在一个周末内构建类似的东西。视频中还有一个 ArduinoBoy——一种通过 MIDI 控制游戏机的有用方式。它在这里被用来连接键盘和手持控制台。休息下的视频。

正如我们之前所见，Game Boy 是一个非常受欢迎的音乐平台——chip tune 艺术家定期修改该设备以获得更好的声音。
