# 商业温室 DIY 火箭质量加热器建造日志

> 原文：<https://hackaday.com/2016/11/27/diy-rocket-mass-heater-build-log-for-commercial-greenhouse/>

[比奇洛布鲁克农场]有一个凉爽的网格圆顶温室，需要在冬天保持温暖。有很多温室加热的商业解决方案，但如果你是那种研究和开发水产养殖解决方案的人，一个更绿色的解决方案可能更有吸引力。

一个[火箭质量加热器](https://en.wikipedia.org/wiki/Rocket_mass_heater)是一个火箭炉和地板下加热的组合。火箭炉的工作原理是通过烟囱上升的热量产生强大的气流，使火焰无法爬上燃料并在户外燃烧，从而形成一个受控的燃烧区。不幸的是，仅仅用一个普通的火箭炉，大量的热量就不必要地散失到大气中。你只需要足够创建草稿。

质量部分解决了这个问题。它通过地板下的散热器排出废气。这被动地将大量热量保留在待加热的空间内。这是一个平衡系统的小技巧，这样它可以在不熄火的情况下将尽可能多的热量释放到空间中，由于一氧化碳和其他原因，熄火是很危险的。一旦达到平衡，用户就能得到一个非常有效地燃烧燃料的炉子，而且最好是被动燃烧。
