# 为您的完美一杯茶准备的微型煮茶计时器

> 原文：<https://hackaday.com/2016/03/17/tiny-tea-timer-for-your-perfect-cuppa/>

如果你对你的茶很认真，你知道完美的冲泡和过度浸泡的垃圾之间的界限是很细微的。几秒钟就能有所不同，对于修修补补的饮茶者来说，这可能会让你建造一个小小的计时器，拥有实现茶的完美所需的功能。

爱喝茶的[酸波旁]为他的计时器设计的电路很简单。它只是在 PCB 的一侧有一个 ATtiny25、一个 LED、两个按钮开关和一个压电蜂鸣器，另一侧有一个硬币电池。电池座是一个有趣的设计——几排引脚和一点弹性金属。用户界面和电路一样简单——按钮可以增加 1 分钟或 10 分钟的时间。计时器马上开始计时，LED 心跳倒计时，一个明显的英国曲调宣布喝茶的时间到了。

一个可能的改进是让 LED 闪烁剩余的分钟数，而不仅仅是一次脉冲心跳。这将是一个很好的反馈，说明你一开始就输入了正确的时间。除此之外，它足够小，便于携带，只做一项工作，而且做得很好——对我们来说，这听起来是个不错的设计。当然，如果你想让它变得复杂一点，你总是可以自动化泡茶过程。
