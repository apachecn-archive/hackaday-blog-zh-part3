# 美丽的气象站使用丙烯酸，RGB LED，和 ESP8266

> 原文：<https://hackaday.com/2016/03/27/beautiful-weather-station-uses-acrylic-rgb-led-and-and-esp8266/>

每个人都知道有形式和功能。这不公平，但人们确实会根据外表来判断，有时甚至会超越所有其他的考虑。因此，虽然你的创客空间伙伴可能会对你建立在试验板上的气象站印象深刻，但你的另一半可能不会。[Dennisv15]采用了一个 0.96 英寸显示屏的普通气象站设计，变成了一个具有更大显示屏和艺术功能外壳的[吸引人的桌面产品](http://www.instructables.com/id/Acrylic-Sheet-RGB-Glowing-Weather-Station-With-Ind/)。

丙烯酸云由于 RGB LED Neopixel 条带而发光，并可以一眼显示天气趋势:红色表示温暖，蓝色表示寒冷，闪烁表示恶劣天气。该项目是真正的多学科，使用激光切割机来生产机身和支架，3D 打印的显示屏挡板，以及使其易于构建的 PCB。

该站可以读取室内温度和湿度，也可以读取室外温度和压力，以及来自互联网服务的本地预报数据。来自[Daniel Eichhorn]的原始气象站设计提供了所有这些，而 ESP8266 提供了网络连接。你可以在下面看到该项目的视频。

我们已经看到了许多气象站，包括许多无线的 T2 气象站。不过，我们不记得见过更好看的了。
