# Gameboy 案件继续存在，圆周率为零

> 原文:[https://hack aday . com/2016/04/10/game boy-case-lives-on-a-pi-zero/](https://hackaday.com/2016/04/10/gameboy-case-lives-on-with-a-pi-zero/)

在以 10 美元的价格拿下一台崭新的不能正常工作的游戏机后，【Chad】决定[是时候做一个有趣的电子项目了，](http://imgur.com/a/shoci)于是他订购了一台液晶显示器并买了一台 Pi Zero。

他花了大约 25 美元从易贝买了一台 3.5 英寸的液晶显示器，并让它运行在 Pi Zero 上。它只有 320×240 的分辨率，但是，嘿，我们是在再造一个游戏机——而不是智能手机。下一步相当棘手:切割外壳以安装新部件。

利用收集的文件，他削减了外壳的屏幕开口，为 LCD 腾出空间，几个小时后，它看起来出奇地好。

从那时起，他开始布置箱子内部的组件，试图找出所有东西都适合的最佳布局。为了给这个单元供电，他使用了三星 Note 的锂离子电池，这应该会给他一些时间。它正好放在游戏卡应该放的地方。

为了增加一些额外的控制功能，他将 SNES 上的游戏垫按钮添加到了电池门的背面，他还在侧面设置了一个 USB 端口，一个 MicroSD 卡插槽，甚至还有一个新的带电位计的音频前置放大器，用于控制扬声器音量。
