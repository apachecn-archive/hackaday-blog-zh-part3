# 令人震惊的万圣节装饰

> 原文：<https://hackaday.com/2016/10/27/shocking-halloween-decoration/>

当然，你可以使用微控制器、一些随机数发生器和一些发光二极管来制作一些万圣节灯的动画，如果装饰品由交流电供电，你可以使用微控制器的一些继电器。如果你没有那么多时间呢？[小工具爱好者]有一些交流供电的装饰品，他之前用 Arduino 和一些继电器制作了这些装饰品，但今年他想做一些更快更简单的东西。

在另一个视频中，他走过荧光灯启动器的电线，用白炽灯泡创造出闪烁的效果。荧光启动器的工作原理是电流加热气体放电管，使一小块金属弯曲并接触另一块金属，从而闭合电路。荧光灯泡是一个足够大的负载，流动的电流保持启动器发热，因此电路闭合。如果你将启动器与普通白炽灯泡串联，启动器会变热，但负载不足以保持启动器足够热，所以它会冷却下来，电路断开，这导致启动器再次变热。这导致灯泡闪烁。[Gadget Addict]使用两个带荧光启动器的电路，每个电路都连接到装饰中的备用灯泡，以使效果看起来更随机一点。

[Gadget Addict]回顾了在这样的电路中使用交流电源的危险，并提到他不太确定启动器能持续多长时间——他猜测灯泡会在启动器之前烧坏。我们之前已经见过几个[闪烁](http://hackaday.com/2011/10/23/halloween-hacks-flickering-jack-o-lantern/) [南瓜](http://hackaday.com/2013/10/01/flickering-pumpkin-lanterns/) [黑客](http://hackaday.com/2015/08/29/shift-register-powered-realistic-candle-flicker/)了，但是这是一个快速、简单并且有点危险的获得闪烁效果的方法！
