# 施法枪使用 POV 显示，而不是魔法

> 原文:[https://hack aday . com/2018/05/29/spell casting-gun-uses-POV-display-not-magic/](https://hackaday.com/2018/05/29/spellcasting-gun-uses-pov-display-not-magic/)

视觉暂留或 POV 显示器在这些地区非常受欢迎。旋转几个发光二极管，你就可以让图像出现在几乎稀薄的空气中——只是不要用手指挡住它。[FriskP]为这个硬件找到了一个很好的应用——[创造一个动漫风格的施法枪。](https://www.youtube.com/watch?reload=9&v=qw8GOB8wrlo)

基本的枪是围绕 Nerf 爆能枪建造的，这在很多蒸汽朋克和动漫中很常见。然后将一个 [Phantom3D](http://phantom-3d.net/) POV 显示器与一些 3D 打印组件一起固定在前面，以展示其风格。然后，这套服装以一种合适的令人敬畏的方式被涂上颜色。

我们不确定使用的软件，但[FriskP]让枪显示一些惊人的咒语类型的图形，当用户扣动扳机时，这些图形似乎悬浮在空中。该作品令人惊叹，展示了我们在 POV 领域看到的一些最佳图形。

总的来说，这是一个非常美观的建筑，任何角色扮演者都会非常自豪地使用它。它依赖于建设者的强大整理和集成能力，而不是原始的电子技能，但最终结果是真正令人印象深刻。

我们以前在这里见过很多 POV 展示—[你可以像一个电脑爱好者一样简单地开始！](https://hackaday.com/2018/04/21/building-a-pov-display-on-a-pc-fan/)休息后的视频。
