# 基于 Z80 的树莓派外观

> 原文：<https://hackaday.com/2017/06/03/z80-based-raspberry-pi-look-alike/>

自制电脑现在很流行，Zilog Z80 是最受欢迎的自制电脑选择。我们已经看到了一些非常棒的版本，但是[马丁·K]的 Z-berry 是有记录以来最小的。顾名思义，retrocomputer 符合包含 GPIO 头的 Raspberry Pi 外形。

Z-berry 设计有一个运行在 10 MHz(可能 20 MHz)的 Z80 CPU，并配有 32 kB ROM
和 512 kB RAM。除了串行接口，该计算机还拥有一条 I2C 总线、一条 SPI 总线和一个 PS/2 键盘连接器。[马丁 K]有一个视频，其中完成的系统被封装在一个树莓派的情况下，并有一个 I2C 有机发光二极管显示器连接和工作。

[Martin K]发布了许多关于如何制作自己的 Z-berry 的细节，包括 BOM、原理图和初步信息。我们联系了他，以了解更多关于该软件的信息，该软件是稳定的，并可根据要求提供 PCB 和样本代码。此外，这个项目承诺比 Raspberry Pi 汲取更少的电流，应该证明对任何寻求创建一个现代问题的复古解决方案的人有用。

看到现代技术与复古技术相结合的项目是很有趣的。我们见过的最好的 Z80 项目之一是 [FAP80](http://hackaday.com/2017/01/22/the-impressive-z80-computer-with-the-unfortunate-name/) 并且在 Hackaday.io 上有一些很棒的[自制计算机项目供你看看并获得灵感。](https://hackaday.io/list/2402-homebrew-computers)
