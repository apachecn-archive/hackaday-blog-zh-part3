# 令人惊讶的梅卡诺弹球机功能齐全，才满足它的命运

> 原文:[https://hack aday . com/2016/09/07/amazing-meccano-pinball-machine-fully-functional-before-meeting-its-fate/](https://hackaday.com/2016/09/07/amazing-meccano-pinball-machine-fully-functional-before-meeting-its-fate/)

伦敦东南部梅卡诺俱乐部的布莱恩·利奇(Brian Leach)发挥了惊人的创造力，几乎完全用梅卡诺零件制作了他的弹球机。他在 2013 年开始，我们在 2014 年看到了一个[早期版本](https://hackaday.com/2014/01/21/a-meccano-pinball-machine/)，但它最终完成了。它有所有适当的弹球的陷阱:分数计数器，超时分数乘数，站立目标，踢出，锅缓冲器，下降目标，和(当然)脚蹼和柱塞。

当[Brian]简要介绍机器时，视频(嵌入在下面)制作得非常好，对不同的机械装置进行了出色的特写。有些元素相对简单，有些则需要变通方法来获得正确的操作，但这一切都做得很好。比如看下面的分数计数器。Meccano 电磁铁太弱，无法直接驱动数字，因此电机通过摩擦驱动连续转动所有数字，电磁铁用于在特定点停止转动。重置包括让数字自由旋转到 9999，然后做最后一点推动，干净地滚回零。
