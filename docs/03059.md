# 难看的 DIY 便携式烙铁

> 原文:[https://hack aday . com/2016/07/26/ugly-DIY-portable-soldering-iron/](https://hackaday.com/2016/07/26/ugly-diy-portable-soldering-iron/)

如果你曾经想要一个电池供电的烙铁，但你又无法忍受买一个的想法，你可以看看下面的视频(只需 5 分钟)。在里面，他拿了一个糖果管、一些废料、一个锂离子电池、一根镍铬合金线、一个 USB 充电器和一个开关，把它变成了一个显然实用的烙铁。

矛盾的是，[Just5mins]使用了一个烙铁来制作这个，所以它可能不是你唯一的烙铁，尽管我们认为你可以在必要时找到一些东西。也许在 rep-rap 风格，使一个质量差的没有焊接，并用它来焊接下一个。

这是一个有趣的小技巧，但老实说，如果你需要一个烙铁，这可能不是你的首选。我们认为它比我们之前看到的打火机烙铁更实用一点。如果你预算有限，想要一个 USB 烙铁，我们也看到了[对那些](https://hackaday.com/2016/04/25/usb-soldering-iron-is-surprisingly-capable/)的评论。或者，你可以选择[焊笔](https://hackaday.com/2014/07/04/solderdoodle-is-an-open-source-usb-rechargable-soldering-iron/)。
