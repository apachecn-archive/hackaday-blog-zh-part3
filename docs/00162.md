# Arduino 教授莫尔斯电码

> 原文:[https://hackaday.com/2015/09/24/arduino-teaches-morse-code/](https://hackaday.com/2015/09/24/arduino-teaches-morse-code/)

你可能想知道为什么有人想学习莫尔斯电码。你不再需要它来申请火腿执照了。然而，至少有三个原因让你想学英语。首先，有些人实际上喜欢它，要么是因为怀旧，要么是因为挑战。另一个原因是，当其他人类可读的方案失败时，莫尔斯电码往往可以通过。莫尔斯电码可以使用低功率、由简单材料制成的设备发送，甚至可以使用镜子或手电筒。最后，莫尔斯电码是一种非常简单的秘密通信方式。如果你知道莫尔斯电码，你就可以通过两条输入输出线与一台隐蔽的计算机私下交谈。我们会让你想象它的用途。

在过去，你通常从一个有经验的发件人那里，通过听收音机，或者从录音带学习莫尔斯电码。当今的技术水平采用计算机来随机生成练习文本。[M0TGN]想要一个生成练习代码的设备，所以[他围绕 Arduino](https://github.com/M0TGN/Morse-Tutor) 构建了它。该设备的行为就像一个旧的商业模型，[大同 D70](http://www.radiomuseum.org/r/datong_morse_tutor_d_70.html) ，虽然它可以选择性地接受液晶显示屏，这是 D70 所没有的。

你可以在下面的视频中看到这个项目的运作。一旦你学会了如何阅读莫尔斯电码，你可能也想[教你的 Arduino 理解它](http://hackaday.com/2015/02/10/converting-morse-code-to-text-with-arduino/)。或者，你可以看看[其他一些基于 Morse 的项目](http://hackaday.com/2015/08/07/hacklet-69-morse-code-projects/)。
