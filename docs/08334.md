# (不必要的？连接器压接技术

> 原文:[https://hack aday . com/2018/01/20/the-不必要的连接器压接技术/](https://hackaday.com/2018/01/20/the-unnecessary-art-of-connector-crimping/)

“向后完成原则”是一种通过想象最终结果来推理问题的方法，然后从那一点开始向后进行。[Alan Hawse]最近写的关于插头连接器压接电线的复杂性的博客文章是这一原则的完美例子。他工作的最终结果是[意识到你可能不应该麻烦地去压接你自己的连接器](https://iotexpert.com/2018/01/18/jst-connector-crimping-insanity/)，但是看着他从那一点开始向后工作仍然是令人着迷的。这也是一张 80 年代摇滚专辑的名字，作者是 *The Tubes* ，但是这并不是一条关于电线的有用信息。

[![](../Images/489e9d4f60a666aab07af81972e3c196.png)](https://hackaday.com/wp-content/uploads/2018/01/crimp_detail.jpg) 当然，有时候人也会做傻事。即使网上有廉价的预卷曲电线，你可能还是想自己做。考虑到这一点，[Alan]已经整理了一篇特别详细和研究充分的帖子，为您提供了您可能想要了解的关于压接的所有信息，这种压接通常被错误地称为“JST 连接器”。

他首先展示了这种连接器的一些常见例子，如果你曾经打开过一件消费电子产品，那就像在翻阅高中年鉴一样。你可能不知道他们的名字，但你肯定记得他们的样子。

然后，我们看到了一系列微距镜头，展示了相关作品的规模。如果你喜欢近距离接触只有几毫米长的金属零件，那么你一定会喜欢这个部分。

最后，这篇文章以几句话结束，介绍了市场上可以买到的压接工具，然后演示了他个人的压接方法。虽然有些工具会让你同时卷曲两组“翅膀”，但[Alan]告诉我们，他发现单独使用它们会带来更好的效果。

如果这个小小的尝试让你渴望一场真正的超专业知识盛宴，一定要去看看[Bradley Gawthrop] 的超级会议演讲。
