# DIY PCBs 的化学转移

> 原文：<https://hackaday.com/2017/10/22/chemotransfer-for-diy-pcbs/>

自从你可以在 Radio Shack 买到你的痕迹以来，用墨粉转移法制作 PCB 就已经存在了。从玻璃纤维片上去除铜的技术有一百万种，从研磨到使用抗蚀笔，到无处不在的激光打印机墨粉转移。[这里有一个我们之前没有见过的技术](https://www.youtube.com/watch?v=zZRHaZjGWq4)。[达科·沃尔克]称之为“电子转移”。这是*主要是*激光打印机墨粉转移过程，但墨粉是在一种特殊混合溶剂的帮助下从纸张转移到铜上的。

这种化学转移过程几乎与制造墨粉转移 PCB 的通常过程相同。首先，将设计反过来印刷在糊精涂层纸上，将纸放在抛光的铜上，整个组件通过层压机，最后用你选择的化学物质蚀刻电路板。这里的关键区别在于，在设计完成之前，铜上涂有一种溶剂。[Darko Volk]用 25%的“清洁汽油”(显然是苯、石脑油或汽油，或某种轻烃)、5%的亚麻籽油和 70%的异丙醇制成混合物。这显然有助于从纸上释放墨粉，并将其粘附到铜上。

由此，该过程实际上是标准的墨粉转移过程。[Darko Volk]正在使用过硫酸钠溶液进行蚀刻，并在数控机床上安装了一台摄像机进行钻孔。

这一过程可以很容易地扩展到两层纸板，在将纸放在铜上之前，使用一个轻型工作台来对齐纸层。你可以看看下面制作单面和双面板的视频。
