# 黄油枪液压成型定制摩托车零件

> 原文：<https://hackaday.com/2017/11/10/grease-gun-hydroforms-custom-motorcycle-parts/>

永远不要低估高压下不可压缩流体的力量。通过适当的约束和对相关力的充分理解，液压可以被用来在家庭作坊里做一些有趣的事情，比如[将不锈钢液压成型为定制的摩托车零件](https://www.youtube.com/watch?v=A66iw8P5NQ4)。

从[Clarence Elias]下面的视频来看，似乎他的店里正在进行 100%定制的摩托车制造。这意味着制造每一个零件，包括灯的反射器。虽然他肯定可以使用传统的方法，如用压平锤敲打不锈钢板或让它经受可怕的英国车轮，但[Clarence]为这项工作建造了一个简单而坚固的液压成形模具。一个厚钢环将不锈钢板夹在一个带有成型流体入口的基板上，成型流体是普通的油脂。[克拉伦斯]经过数学计算，数字令人印象深刻——在这种情况下，1500 磅/平方英寸的黄油枪非常有说服力。结果是一个完美成形的盘子，没有工具痕迹，只需要一点点抛光就可以投入使用。

无论是通过[压力清洗机](http://hackaday.com/2014/08/13/hydroforming-in-the-garage-with-a-pressure-washer/)、[一股空气](http://hackaday.com/2016/03/22/quick-and-easy-pressure-forming-makes-plexiglas-domes/)，还是[在橡胶垫上施加 20 吨的压力](http://hackaday.com/2017/10/20/low-budget-hydroformer-puts-the-squeeze-on-sheet-metal-parts/)，液压成形都是制造定制零件的绝佳方法。
