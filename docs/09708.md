# PCB 作为线性电机

> 原文：<https://hackaday.com/2018/06/11/pcbs-as-linear-motors/>

PCB 现在非常便宜，这意味着每个人都可以尝试在玻璃纤维基板上小心地应用铜迹线。对于他的 Hackaday 奖参赛作品，[卡尔]是把线圈放在一个印刷电路板上。你能用它做什么？[造电机，明明](https://hackaday.io/project/158017-linear-pcb-motor)。然而，这不是任何马达:这是一个线性马达。如果你曾经想要一个 PCB 上的磁悬浮列车，这是你的项目。

这个项目是[Carl]的另一个 PCB 电机项目的一个小扩展，这个项目被恰当地命名为 [PCB 电机](https://hackaday.io/project/39494-pcb-motor)。为了这个项目，[Carl]快速制作了一个小的圆形 PCB，里面嵌入了一些非常小的线圈。通过添加轴承、一些 3D 打印部件和一些磁铁，[Carl]能够创建一个无刷电机，也是一个 PCB。用在四轴飞行器上够强大吗？可能还没有。

像[Carl]早期的 PCB 电机一样，这种线性 PCB 电机遵循相同的基本思想。如果你愿意的话，这个“轨迹”只是一个装载了 12 个线圈的矩形 PCB，每个线圈使用 5 密耳的空间和走线，总共 140 匝。这比用于(圆形)PCB 电机的线圈大，但这仅意味着它可以处理更多的功率。

至于这台电机的移动部分，[卡尔]正在使用一个 3D 打印的滑块，内部嵌入了 N52 钕磁铁。总的来说，这是一个简单的设备，但这并不涉及驱动电路的复杂性。我们期待着更新，将使这个电机移动，把它变成一个伟大的黑客日奖的参赛作品。
