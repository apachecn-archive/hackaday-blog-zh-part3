# 竹子成为时尚的电灯开关

> 原文：<https://hackaday.com/2016/09/05/bamboo-plant-becomes-a-stylish-light-switch/>

如果打开一个普通的老式电灯开关或按下按钮还不能令人满意地在家里使用电器，那么用[拨弄一株植物的叶子](http://www.instructables.com/id/Turn-a-House-Plant-Into-a-Light-Switch/?ALLSTEPS)来打开你的灯怎么样？[X kitz 提供了如何将你生活空间中的任何导电物体变成一个漂亮的电容式触摸开关的详细说明，为这种日常行为增添一点魅力。

电容式触摸继电器在导电介质周围产生静电场，持续监控该场，并在检测到电容的任何微小变化时触发。[Xkitz]用一株竹子作为他的触发器。轻轻触摸任何一片叶子仍然可以作为一个适当的触发器——作为静电场如何工作的冷静演示。

 [https://www.youtube.com/embed/hFiBmEqel6g?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/hFiBmEqel6g?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



[xkitz]聪明地将继电器伪装成铝板，让任何潜在的制造商在厌倦了将他们的工厂用作电灯开关时，都可以进行切换。它们还提醒读者在目标发生变化时重新校准继电器。重新校准只需要几分钟。考虑到这种构建是在短时间内完成的，并且不到 50 美元，所以花时间为你的家增添一点独特的风格是值得的。当你在做这个的时候，何不组装一个触摸屏时钟来赞美你的新电灯开关呢？
