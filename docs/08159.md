# 看到这个光滑的钢筋混凝土 Strandbeest 拉链左右

Bevel gears used to mount motors vertically.

西奥·詹森的 Strandbeest 设计是最受欢迎的，理由很充分；滑行的步态令人着迷，[这个由【tosjduenfs】](https://www.youtube.com/watch?v=1x77e4XvqZ4)创作的 RC 版本看起来非常棒。早在 2015 年,[项目首次出现在 Thingiverse](https://www.thingiverse.com/thing:719812) 上，并在去年悄悄地更新了一个包含完整装配细节的 zip 文件。

所有 Strandbeest 项目——尤其是可操纵的项目——都值得注意，因为建造一个项目绝不是简单地放大或缩小零件。首先，经典的 Strandbeest 设计没有提供任何转向手段。此外，虽然机动化系统在概念上很简单，但在实践中却不那么简单；在 Strandbeest 中没有明显或方便的位置来安装电机。在这个项目中，锥齿轮用于将电机垂直安装在中心区域，左右两侧像坦克一样独立驱动。接受 RC 信号的电机驱动器允许使用现成的 RC 发射器和接收器来控制该单元。下面嵌入了一个精彩的视频，视频中机器平稳地四处滑行。

Strandbeest 的缺点之一是它需要相当平坦的地面。腿抬得不是很高，这限制了它在平坦地形上的活动。这一限制并没有阻止 13 英尺可骑行的锡蜘蛛，但一个替代方案是由 T2 项目 Octo T3 使用的 Klann 连杆，这是一个用于崎岖地形的机器人平台。
