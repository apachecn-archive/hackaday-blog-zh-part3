# 智能笔

> 原文:[https://hackaday.com/2017/07/11/the-smart-pen/](https://hackaday.com/2017/07/11/the-smart-pen/)

“唉。你的意思是我必须按下笔的按钮才能打开它？恶心。”在这个时代，我们可以在手机上滑动并进行语音识别，似乎没有理由我们必须按下按钮。多么陈旧。所以[Marek Baczynski] [将他的笔现代化，用于滑动和语音控制](https://www.youtube.com/watch?v=nY7i7kj2jO4)。这也肯定会让所有的孩子重新学习他们的书法。

不过说真的，并不是所有的黑客都是认真的。[Marek]和[Ghlargh]添加了一个伺服来激活按钮，然后[Marek]添加了蓝牙来控制伺服。在编写了一个手机应用程序后，他可以向下滑动来打开它，再向下滑动来关闭它。然后，在 Redditers 的提示下，他在笔记本电脑上添加了语音控制。我们认为他可以通过将笔连接到笔记本电脑的方式来完成更专业的工作，也许他可以 3D 打印一些东西而不仅仅是使用胶带，或者也许使用 CNC 或激光切割机制作一些东西。像这样一个重要的黑客应该得到同样的待遇。现在他只需要说“电脑”。开笔。”单调乏味的任务被处理掉了。眼见为实，请看下面的视频。

我们之前已经在这里看到过[马雷克]的创造力，他的[换灯泡无人机](http://hackaday.com/2016/11/28/how-many-drones-does-it-take-to-screw-in-a-lightbulb/)和[自动驾驶土豆](http://hackaday.com/2017/06/22/self-driving-potato-hits-the-road/)。我们迫不及待地想看到他接下来会对社会做出什么样的巨大贡献——我们指的是让我们发笑的东西。
