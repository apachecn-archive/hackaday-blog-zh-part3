# 复古的 ZX 光谱过着简朴的生活

> 原文：<https://hackaday.com/2017/01/12/retro-zx-spectrum-lives-a-spartan-existence/>

FPGAs(像 Xilinx 的 Spartan 系列)是很好的构建模块。它们经常让我们想起我们小时候拥有的 100 合 1 电子设备。你可以混合搭配许多组件，打造几乎任何东西。然而，就像一个裸露的微控制器一样，它们通常没有太多的外围设备。所以秘方就是你可以用什么样的元件来包裹芯片。

如果你对复古计算感兴趣，你应该看看 ZX-欧诺主板。它承载一个 Spartan 6 FPGA。它们是出售的，但设计是开源的，如果你喜欢自己制作或进行修改，所有的信息都是可用的。您可以在下面看到该板的视频(如视频中所述，颜色问题是由于采集卡试图处理非标准同步速率造成的)。

以下是主要规格:

*   FPGA Xilinx Spartan xc 6 slx 9-2 tqg 144 c
*   静态存储器 512Kb，as7c 34096 a-10 英寸
*   50 兆赫振荡器
*   视频输出(复合)
*   PS/2 键盘
*   立体声音频插孔
*   耳机插孔连接器(用于读取盒式磁带)
*   JTAG 和 RGB 连接器
*   SD 卡插槽
*   带有 3 个插针带的扩展端口
*   微型 USB 电源连接器
*   PCB 尺寸:86×56 毫米(与 Raspberry Pi 外壳兼容)

在最近的一篇文章中，我们提到了 Amstrad 的克隆版本，至少有一位评论者指出 ZX 频谱更受欢迎。但是，有了这个板子，就不用选了。在下载部分，有一些位流(FPGA 配置)可以让开发板仿真几种不同的计算机，包括几种 Commodore 系统、一些 Atari 系统、BBC Micro、ZX81 等等。当然，ZX 光谱是最重要的。

如果你决定尝试其中一种，你可能需要温习一下你的西班牙语或者准备一个谷歌翻译的链接。周围的许多社区似乎都说西班牙语。但它看起来像一个很好的板，可以做任何你可以塞进 FPGA 的计算机平台。

如果你想学习 ZX 谱汇编语言(谁不想呢？)，不久前我们讨论了一个很好的教程。它可能需要一个 USB 到 PS/2 适配器，但我们很乐意看到这个板与键盘[一起使用，就像这个](http://hackaday.com/2014/07/31/zx-spectrum-turned-into-a-usb-keyboard/)。
