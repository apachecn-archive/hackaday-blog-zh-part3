# 英雄机器人 BB-8 如何滚动

> 原文：<https://hackaday.com/2018/01/01/how-the-hero-droid-bb-8-rolls/>

到现在为止，我们已经开始期待每部《星球大战》电影之后都会有大量的特许商品。今年的作物包括许多口味的 BB-8，所以每个粉丝都可以找到适合自己口味的东西。在这个食物链的顶端是一个移动互动的“英雄机器人 BB-8”。对于那些想知道它是如何工作的人来说，[themiksenna][打开了他的装置](https://youtu.be/PWPv44e__qk)来满足我们的好奇心。

制造商也称之为“旋转大师 BB-8”，这款玩具的价格令人印象深刻。视频调查了球内部的机械部件。展示机器人如何行走，以及头部如何发音。

 [https://www.youtube.com/embed/PWPv44e__qk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/PWPv44e__qk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



接下来是第二段视频，这次是进入 BB-8 的大脑。除了必须的闪烁 led 阵列之外，它还包含扬声器、用于语音控制互动的麦克风阵列和多个红外发射器+接收器，以实现“跟我来”功能。

 [https://www.youtube.com/embed/kVKpkkic8yM?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/kVKpkkic8yM?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



一些不太明显的实现细节没有包括在内。头部的电路板和身体的电路板之间的分工是什么？似乎有一个小天线可以让这两部分进行交流。他们相互之间说些什么，该链接的安全性如何？在我们找到答案之前，对于那些对破解默认电子设备不太感兴趣，而对完全替换它们更感兴趣的人来说，这个概述就足够了。

过去，你必须从头开始打造自己的 BB-8。完成这种球式机器人的方式不止一种，从[调查移动 BB-8 的不同方式](https://hackaday.com/2016/06/24/driving-bb-8-more-than-one-way-to-move-this-bot/)到他与人的遭遇[对一个小球机器人的情绪反应](https://hackaday.com/2017/10/09/our-reactions-to-the-treatment-of-robots/)开始，【史蒂文】对他的旅程的总结涵盖了大部分内容。
