# 简单的相机滑块为您的照片增加一两个维度

> 原文:[https://hackaday . com/2018/06/21/simple-camera-slider-adds-a dimension-or-two-to-your-shots/](https://hackaday.com/2018/06/21/simple-camera-slider-adds-a-dimension-or-two-to-your-shots/)

相机滑块是一个受欢迎的构建，正确执行它们可以为延时序列或实时动作拍摄出令人印象深刻的照片。但它们似乎最适合长镜头，因为沿直线移动相机只会通过画面移动主体靠近相机。

[这个带有平移和倾斜轴的滑块](https://howtomechatronics.com/tutorials/arduino/diy-motorized-camera-slider-pan-tilt-head-project/)可以让移动特写镜头变得更加容易。在他极其详细的构建日志中，[Dejan Nedalkovski]展示了他是如何用最简单的材料和工具构建他的。线性轨道只是由 MDF 框架支撑的几根铜管。相机小车骑在普通滑板轴承上的轨道上，由 NEMA-17 步进器驱动，平移和倾斜轴也是如此。[Dejan]还提供了一个仓门式枢轴，使相机相对于轨道倾斜，允许相机在斜面上上下滑动，拍摄真正有趣的照片。控制器使用 Arduino 和操纵杆手动驱动摄像机，或者可以对钻机进行编程，使其在预设点之间平滑移动。

这比简单的滑块更进了一步，感觉更像是全面的动作控制。我们有一种感觉，一些相当戏剧性的镜头将有可能与这样的钻机，事实上，这是一个简单的建设只是锦上添花。
