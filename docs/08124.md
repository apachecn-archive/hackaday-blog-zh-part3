# 土星光环的威严照亮了你的土坯房

> 原文:[https://hack aday . com/2018/01/08/the-maurity-of-saturns-rings-lighting-your-doe/](https://hackaday.com/2018/01/08/the-majesty-of-saturns-rings-lighting-your-abode/)

[Modustrial Maker]又推出了另一款非常酷的 LED 可视化工具。这一次，他制作了一对吊灯，灵感来自土星环。

环主要由胡桃木胶合板制成，使用圆形刳刨机夹具使切割更容易。如果你受到启发，想为自己制作这些戒指，那么【定制师】很清楚——你裁剪戒指的顺序是绝对重要的。这些碎片被粘在一起，任何边缘都被打磨光滑，由于染色前表面弯曲，使用热风枪进行封边。复制第二个(或者更多，如果你这样选择！)戒指。预先警告——需要一点几何学来寻找锚点，以保持环的适当平衡。

[Modustrial Maker]建议使用现成的 LED 控制器来处理可视化和灯光效果，但他使用 Arduino Mega clone 作为大脑——代码可在此处获得，一个 MonkeyJack MAX9814 驻极体麦克风和一个四通道 RF 遥控器/收发器来控制不同的模式。随着音乐跳动，这些环确实能产生光滑的灯光。
