# 知道下面有什么:埋在地下的设施位置

> 原文：<https://hackaday.com/2017/09/07/knowing-whats-below-buried-utility-location/>

在过去的几个世纪里，我们人类在基础设施上投入了大量的精力，甚至在掩埋大部分基础设施上投入了更多的精力。理由很充分——地上的电缆和管道不仅难看，而且暴露在自然环境中也容易损坏。一些公用事业，如天然气和污水管道，也是危险的，或者至少被认为是危险的，因此最终被掩埋。眼不见心不烦。

但人类也喜欢挖掘，似乎一个铺路工程刚完成，就有一些拿着手提钻的小丑在那里破坏原始的路面。然而，在施工开始之前，神秘的标记将出现在路面上，这是由当地的地下公用事业定位服务机构提供的，他们将彩虹标记应用到地面上，这样我们脚下往往脆弱的基础设施就不会发生任何糟糕的事情。

### 挖之前先打电话

根据法律，美国的每一家公共事业公司都必须参与“一站式定位服务”。联邦通信委员会设立了 811 作为电话号码来访问一次性呼叫系统，该系统允许挖掘者请求定位和标记掩埋的服务。定位服务由公用事业公司付费，因此客户无需直接付费。

任何进行挖掘的人都需要打电话(或者现在，在线提交请求)来安排定位服务，以标记预期的工作区域。这适用于任何挖掘——虽然具有 40，000 磅挖掘力的反铲挖土机对煤气总管的破坏可能是灾难性的，但为新邮箱配备柱孔挖掘机的房主可能同样危险。如果你的深度超过 15 英寸，你需要打电话，因为如果你对地下服务造成任何损害，你就要为此负责。当然，如果你活着的话。

### 哔哔哔

一旦输入票证，定位技术人员通常会在几天内标记工作现场。要求呼叫者用白色喷漆标出大致位置。技术人员不会盲目到来——他或她会有标明主要公用事业线路大致位置的地图。但是我们脚下的世界是惊人的动态，再加上可能存在从未被正确记录的设施，这使得线路位置的物理验证成为技术工作的一大部分。

定位技术有各种各样的工具来定位那些看不见的东西。鉴于大多数地下设施都有某种金属部件——旧的供水和污水管道、天然气管道以及电线或电信电缆中的铜线——电磁工具完成了大部分工作。

探测器可以是有源的或无源的。无源方法通常用于定位已经有交流信号的设施，如电力线或电信电缆。这可能更棘手，因为信号强度在很大程度上取决于电路中的情况。被动方法还可以包括磁力测量法，以检测公用事业管线产生的弱磁场，甚至包括探地雷达，以探测地表下并建立掩埋结构的图像。

主动检测将电流施加到公用事业线路的地上部分，例如水表或煤气表。发射机将交流信号施加到线路上，使其变成天线。公用事业公司有时甚至将铜线和塑料线放在一起，作为主动探测的辅助手段。带有高度定向天线的手持接收器扫过地面以接收信号，使技术人员能够标记位置。有时，甚至将发射器穿入管道，当探头在管道中前进时，使用接收器跟踪信号。
