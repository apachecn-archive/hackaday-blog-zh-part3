# 你是一个加密无人机视频流的明星吗？

> 原文:[https://hackaday . com/2018/01/15/watching-the-watchers-you-the-star-of-an-encrypted-drone-video-stream/](https://hackaday.com/2018/01/15/watching-the-watchers-are-you-the-star-of-an-encrypted-drone-video-stream/)

不管是好是坏，装有流式摄像机的小型飞机现在随处可见。让天空中的眼睛变得如此触手可及，已经产生了有趣的镜头，这些镜头在几年前可能会因昂贵而难以捕捉，但这一新的创意前沿在用于侵犯隐私时也有其黑暗的一面。那些通过加密视频传输来掩盖行踪的人应该知道[内盖夫本古里安大学的研究人员证明了](https://arxiv.org/abs/1801.03074)这种保护可以被攻破。

BGU 团队证明了侧声道分析可以针对视频压缩算法的常见行为进行，因为视频输入的某些变化会导致输出流的比特率发生可检测的变化。通过控制目标的视觉外观来触发这些变化，带宽消耗的相关变化将揭示目标在加密视频流中的存在。

这里演示的想法是早期的，并且面临着每个旁道攻击的共同挑战:从噪声中提取信号。该信号可能被误解，导致假阳性，或者该技术可以通过添加更多噪声来规避。未来的研究肯定会追求这两个方面，就像一些 FPV 设备制造商肯定会努力逃避检测([小心你使用什么频率](https://hackaday.com/2017/12/20/fcc-fines-drone-fpv-maker-for-using-radio-spectrum/))和一些人将[采取更直接的方法](https://hackaday.com/2017/07/20/diy-air-cannon-snags-drones-from-the-sky/)进行威慑。
