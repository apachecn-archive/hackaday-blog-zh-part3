# 圆周率零转变为游戏男孩

> 原文:[https://hack aday . com/2016/11/12/pi-zero-transforms-to-game-boy/](https://hackaday.com/2016/11/12/pi-zero-transforms-to-game-boy/)

[GreatScott]买了一个游戏机盒。通常，你会认为你想用它来修复一个损坏的游戏机，但在这种情况下，[GreatScott]使用 Pi Zero 和一些 3D 打印在这个小盒子中构建了一个游戏系统。你可以看一些视频，在下面。

该项目的两个有趣部分是 LCD 显示器(后视摄像头屏幕)的来源和电池的选择。锂离子电池风靡一时。但是如果你看新闻，你就会知道使用电池有一些安全问题，尤其是如果你使用不当的话。[GreatScott]决定使用镍金属氢化物电池，它仍然需要一个保护电路，但通常比新技术的电池危险小。

还有一些额外的外壳部件需要 3D 打印。最终的设备看起来不错，除了 Game Boy 之外，还可以运行许多不同的仿真程序。

当然，这并不是什么新的东西。我们最近在一个游戏玩家进阶版的[身上也看到了同样的把戏。但这是一个很好的构建，有足够的信息使其可复制。](https://hackaday.com/2016/11/03/emulating-a-gameboy-advance-inside-of-a-gameboy-advance/)
