# 两栖船屋上路了

> 原文：<https://hackaday.com/2017/08/21/amphibious-houseboat-hits-the-open-road/>

有什么比拥有自己的船屋更好？两栖船屋怎么样？这正是席恩·帕斯吉安在自家车道上建造的东西。这一切都始于一艘废弃的 32 英尺长的船屋。一个 1967 年的模型，有一个生锈的钢壳，[席恩]最初买下它作为客房。

但是不能让船在他的后院生锈。很快决定让它回到开放水域…在路上。两栖船屋。在寻找大型拖拉机轮胎时，[席恩]发现了一整台多年未使用的作物喷雾机。这种作物喷雾器是一个巨大的三轮怪物，有巨大的喷洒臂。

最初的计划是为喷雾器挖一个洞，然后把船放在喷雾器底盘上。然而事情从来没有像计划的那样发展。喷雾器有点太短了，所以它的底盘换成了校车上的。轮轴不够长，不足以清除船的吃水，所以它用定制的钢轮垫片延长。

YouTube 上的一个 7 部分系列记录了这个构建。最新一集详细描述了这艘船第一次依靠自身动力行驶。我们不确定水陆两用船屋在街上的合法性，但[席恩]不会开太远，因为在他位于纽约北部的商店后面有一个大湖。游艇将于 8 月 23 日下水。祝你好运！

如果船屋对你来说太大，桶船怎么样？如果你喜欢小船，[你可以乘坐 42 英寸的船](http://hackaday.com/2017/05/29/crossing-the-atlantic-in-a-42-inch-boat/)穿越大西洋。
