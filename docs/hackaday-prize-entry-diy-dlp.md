# Hackaday 大奖参赛作品:DIY DLP

> 原文：<https://hackaday.com/2017/07/01/hackaday-prize-entry-diy-dlp/>

3D 打印革命即将到来，与这些机器相关的技术每天都在发展。立体平版印刷或 SLA 打印机正在成为无法在基于灯丝的机器上制造的高分辨率打印的首选打印机。 [ADAM DLP 3D 打印机](https://hackaday.io/project/16701-adam-dlp-3d-printer)项目是[adambrx]参加 Hackaday 奖的项目，也是他在 DIY 预算下追求更高质量打印的第一步。

[adambrx]目前的迭代采用了 Raspberry Pi 3 和 UV DLP 投影仪，所有这些都封装在一个定制的框架组件中。日志显示了打印机从 Acer DLP 到当前 UV DLP 光引擎的演变。对于一个 DIY 项目来说，结果相当令人印象深刻，[adambrx]张贴了 50 微米柱子的图像和一些漂亮的其他印刷品，显示了该项目投入的工作量。

可以肯定地说，[adambrax]在大约 3 年内花费了 5000 多€，超过了 Hackaday 奖的平均参赛费用。可以[adambrx]可以保持这一个真正的 DIY 的根源还有待观察，但是，很明显，这个项目有潜力。我们希望看到一种高分辨率的 SLA 打印机，它不需要昂贵的成本。
