# 《预言家日报》是一份神奇的报纸！(有点)

> 原文：<https://hackaday.com/2017/07/03/daily-prophet-is-a-magic-newspaper-kinda/>

房子周围的一些小玩意是很好的展示和谈资，但是当一件来自魔法世界的艺术品进入麻瓜家庭时呢？嗯，[你把它框起来](https://www.raspberrypi.org/blog/harry-potter-and-the-real-life-daily-prophet/)。

好的，实际上这是一个以哈利·波特为主题的动画画框——特别是来自系列小说和电影的神话般的报纸, *The Daily Prophet* 。在参观了奥兰多工作室的“哈利·波特的魔法世界”景点后，他想用一种创新的方式来展示他的假期视频。

这个 7 英寸的显示器被固定在一个海报框架内，周围是定制的巫师纸首页——编织他的旅行故事——并由树莓 Pi 3 控制。当有人靠近时，Adafruit 红外传感器会检测到移动并激活显示屏，五分钟后关闭显示屏，以保护屏幕并省电。它安装在一个隐藏在橱柜里的 USB 电源线上，增加了一本神奇期刊的效果。什么，你认为它也是由魔法驱动的吗？
