# 复古技术:像 1971 年一样的汽车导航

> 原文:[https://hack aday . com/2018/06/17/retro tech tacular-car-navigation-like-its-1971/](https://hackaday.com/2018/06/17/retrotechtacular-car-navigation-like-its-1971/)

任何在 GPS 时代之前就已经开车的老人可能会像我们一样想知道，人们是如何找到任何东西的。那时的导航意味着过时的纸质地图，因为错过转弯而绕远路，以及为了问路的耻辱而过于频繁地在昏暗的加油站停车。有时要花很长时间，虽然我们到达了目的地，但似乎总会有更好的方法。

确实有，但不是等待未来和卫星星座来指引道路，而是一些聪明的人在 20 世纪 70 年代初尝试了用于汽车导航的航位推算系统。下面的视频显示了一个名为[卡带导航](https://www.youtube.com/watch?v=KliWHCzE16c)的应用。它包括一个安装在仪表板下的控制器和一个改装过的卡式播放器。使用了特殊的磁带，上面录有特定路线的语音指示。每一步都被一个音调分开，音调的长度编码了汽车在下一步开始之前要走的距离。控制器挂在速度计的电缆上，当移动的距离与音调的长度一致时，下一个指令就开始播放。这种方法有一长串的问题，尤其是在使用它时没有选择道路曲调，但考虑到当时的限制，这是相当巧妙的。

航位推算总比没有好，但它与 GPS 导航相去甚远。如果你仍然对凌晨 3:00 卫星云如何将你指向最近的华夫饼屋感到困惑，请查看我们的 GPS primer 了解详情。
