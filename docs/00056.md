# 家酿迷你 Chumby 融合了 3D 打印、ESP8266 和触摸屏

> 原文:[https://hackaday . com/2015/09/11/home brew-mini-chum by-blends-3d-printing-esp 8266-and-a-touchbook/](https://hackaday.com/2015/09/11/homebrew-mini-chumby-blends-3d-printing-esp8266-and-a-touchscreen/)

当我们的密友去世时，我们都很沮丧。并计划黑客至少有一个死亡时，设备停止供电。[Spiros Papadimitriou]肯定也错过了他的，因为他在[做他自己的无线、触摸屏、智能时钟](http://bitquill.net/blog/esp8266-light-controller-clock/)已经有了一个好的开始。

老实说，它还不能完全取代查姆比。它有一个时钟，可以控制一些设备。有一些钩子可以添加一个尚未完成的天气显示。尽管如此，这是可行的第一步。当然，任何人都可以拿一个树莓派(或类似 T1 的 T0)、一个无线加密狗和一个触摸屏 T3 做同样的事情，对吗？也许吧，但是要在你的床头柜上做一个你(或者你的另一半)想要的就难多了。[Spiros]花了很多时间设计了一个漂亮的 3D 打印外壳。

[斯皮罗斯]使用的一个技巧是摩擦焊接。这包括将一根细丝放入旋转工具中，像焊条一样使用它，允许细丝旋转的摩擦熔化塑料。你应该预热你正在处理的区域，但是,[斯皮罗斯]发现让细丝轻轻地刷一点期望的接缝会很容易预热。一旦预热，就很容易施加更多的压力，做出好看的接缝。他使用这种技术将螺纹孔隐藏在表壳中，这是我们最近讨论过的话题。应用 XTC-3D——一种为完成 3D 打印物体而配制的环氧树脂涂层——也有助于产生专业的效果。

至于黑客攻击，[斯皮罗斯]不得不逆向工程他的无线灯泡的协议，但这似乎比不上案件。你可以在下面的视频中看到最终的结果。当然，[查姆比](http://www.chumby.com/)算是回来了。但是买自己能造的东西有什么乐趣呢？
