# Hackaday 奖品条目:1337 个 Haxxor 键盘

> 原文：<https://hackaday.com/2016/08/29/hackaday-prize-entry-1337-haxxor-keyboards/>

如果你和我们一样，你大部分时间都在电脑键盘前度过，想知道你的生活哪里出了问题。[AnonymouSmst]对生活的看法稍微积极一点，[这让他们创造了一个真正的 DIY 键盘](https://hackaday.io/project/13210-the-fsociety-keyboard)，有 OLED、蓝牙、NFC、模拟操纵杆、一个“物联网”东西、本地存储和炫目的背光。这是一个 1337 h4x0r 键盘，也是自那个怪异的 GameCube 控制器以来，我们见过的最酷的输入设备之一。

[AnonymouSmst]是一名非常精英、非常有特权的黑客，他参加了 [Hackaday Munich meetup](https://hackaday.io/project/13210/log/44218-the-inspiration) ，在那里[sprite_tm]首次演示了他的固件黑客技术，允许任何人[在键盘上玩 Snake](https://www.youtube.com/watch?v=enzU9u328Fc)。在这里，建造*终极*键盘的想法被植入，【mst】很快开始研究使用哪种按键开关。显然，[mst]讨厌他的邻居，选择了令人讨厌的大声樱桃蓝色。

在标准的 60%键盘布局中，[AnonymouSmst]增加了许多硬件，即使是在最壮观的机械键盘构造中，你也通常看不到这些硬件。几十个 WS2812 RGB LEDs 被添加到该构建中，还有一个 Adafruit Bluefruit 模块，一个 NFC 读取器，一个 LORA 模块和一个用于 WiFi 功能的 ESP8266，一个有机发光二极管显示器*，以及两侧的两个模拟操纵杆，一个用作箭头簇，另一个用作鼠标。*

这些年来，我们已经看到了几十种机械键盘，但这将 DIY 键盘的整个概念提升到了一个新的水平。它明亮、闪亮、华丽，是炫耀性消费和工程技术的庸俗展示。这是完美的键盘，因为它是由最终使用它的人设计和制造的。
