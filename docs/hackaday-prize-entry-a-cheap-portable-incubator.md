# Hackaday 奖参赛作品:一个便宜的便携式孵化器

> 原文：<https://hackaday.com/2016/10/13/hackaday-prize-entry-a-cheap-portable-incubator/>

每年有数以百万计的早产儿出生，其中不少出生在距离 NICU 医院数小时路程的地方。[Manoj]的黑客日奖参赛作品[是一个简单但非常有用的原始孵化器](https://hackaday.io/project/15235-aditiaffordable-diagnostic-thermal-incubator)。它和你在世界一流医院里找到的孵化器一样好吗？不，但这不是重点。这是世界其他地区的孵化器，那里缺乏新生儿护理。

你不会让机械呼吸甚至氧气进入一个用于地球上最偏远地区的设备，所以这个孵化器几乎只专注于监测。在一个特大的睡袋里装有足够的电子设备来测量心率、血氧、体温和呼吸。此外，还有一些电阻织物元素可以将电能转化为热能。

当然，你在任何医院或诊所发现的任何东西都将远远超过这个项目所能提供的。不过，这真的不是重点；这种孵化器价格低廉，可以部署在任何地方，并提供足够的信息，有望使早产儿存活。这对我们来说已经足够好了，也是进入 Hackaday 大奖的绝佳机会。
