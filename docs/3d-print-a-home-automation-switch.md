# 3D 打印家庭自动化开关

> 原文：<https://hackaday.com/2018/01/13/3d-print-a-home-automation-switch/>

如果你是那种不会使用廉价的 Sonoff 模块来控制交流供电设备的人，我们不会责怪你，你现在可能应该停止阅读。然而，如果你不介意一点暴露的交流布线，并且你有一台 3D 打印机，你可能会对第二代[530 Project's] [墙内灯开关](https://www.thingiverse.com/thing:2632463)感兴趣。

3D 打印开关适合一个标准的盒子，并使用 Sonoff 控制器的内脏。这些都与所有流行的生态系统，如 Alexa 和谷歌主页。而且它们很便宜。非常便宜。如果你已经有一台 3D 打印机，即使算上灯丝的成本，这些也只是商用开关成本的一小部分。你可以在下面看到一段关于这款设备的视频。

当然，对于你为 TP-Link 或其他 WiFi 交换机支付的价格，你可以得到商店的退货，保证它不会烧毁你的房子，而且——你知道——如果发生了，有人可以起诉。这些可能都是好事情，但另一方面，周围有大量的这些模块，假设你知道你在做什么，没有理由认为它们会爆炸，导致不育或任何其他不好的事情。

这是 Sonoff 开关的第二个版本，它使用 Sonoff 内置的开关进行本地激活。即使你害怕交流线路电流，你也可能会发现设计中巧妙的拨动开关[530 项目]的用处。

如果这能让你感觉更好，你可以去参观工厂——这不是从某人的车库里出来的。你可能已经注意到，由于这些 Sonoff 设备内部有 ESP8266s，[它们非常容易被黑客攻击](https://hackaday.com/2017/05/12/son-of-sonoff/)。事实上，他有一些与交换机一起使用的[定制固件](https://github.com/530projects/SonoffLightSwitch_v2)。除此之外，它还可以让你通过按住开关来触发重置。
