# Hackaday 奖参赛作品:ESPMetric，一个简单易行的矩阵

> 原文:[https://hack aday . com/2017/08/31/hack aday-prize-entry-espmetric-a-simple-and-easy-matrix/](https://hackaday.com/2017/08/31/hackaday-prize-entry-espmetric-a-simple-and-easy-matrix/)

你可以用一束连接到互联网的发光二极管做很多事情。您可以显示时间、天气、公交车时刻表，或者您生活中的任何一个重要数据点。自定义矩阵的设置是一件痛苦的事情，这也是为什么我们希望看到一个看起来相当完美和干净的矩阵。[Faire-soi-meme]用一些扩散器和网格挡板美化了一个 8×32 新像素矩阵。这是 ESPMetric ，也是今年 Hackaday 奖的参赛作品。

新像素矩阵由 NodeMcu 使用来自[squix]的 [ESP82666 气象站代码](https://github.com/squix78/esp8266-weather-station)以及 Adafruit 的 [NeoMatrix 库](https://learn.adafruit.com/adafruit-neopixel-uberguide/neomatrix-library)的元素进行控制。有一个光敏电阻来控制亮度，还有 3 个按钮来控制它的各种模式。点击按钮，你可以看到各种设置，比如时间、WiFi 状态、股市等等。

如果你会法语，或者喜欢谷歌翻译的体验，那么[Faire-soi-meme]已经在他的博客上详细介绍了构建步骤，不过你也可以[从他的 GitHub 库下载他的代码](https://github.com/fairecasoimeme/ESPMetric)。有一个关于这个建筑的很棒的视频，你可以在下面看看。
