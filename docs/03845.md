# 这是一个时钟！这是一个难题！是那个傻瓜盒子！

> 原文:[https://hack aday . com/2016/10/17/its-a-clock-its-a-puzzle-its-the-gooniebox/](https://hackaday.com/2016/10/17/its-a-clock-its-a-puzzle-its-the-gooniebox/)

[杜伊诺博士]最近完成了他称之为“互动家具”的最新产品——GoonieBox。它花了 800 多个小时的设计和组装工作，结果是迷人的。部分时钟和部分拼图盒，它装有符号，移动部件，灯光，谜语，声音，开关和锁定的隔间。这实际上是在恳求游客们仔细看看。

互动家具的概念使[杜伊诺博士]想要创造一种独特的装饰，让参观者可以与之互动。仅仅这样还不够——他想要的是不需要任何解释其工作原理的东西；本质上吸引注意力、检查和探索的事物。这一探索导致了 GoonieBox T1 的诞生，它以 1985 年电影《T2》的两个灵感命名，分别是《T3》、《七宝奇谋 T4》和游戏《房间》中的谜题。

下面嵌入了两个短视频:第一个演示了盒子的功能，第二个介绍了构建过程。有激光切割的木材，大量的 3D 打印部件，以及大量精心的规划和测试。

谜题盒让人们展示他们在各种不同的执行过程中的创造力，像这些更简单的激光切割谜题盒和在光谱的另一端是[这种定时，多阶段的谜题被操纵来引爆](http://hackaday.com/2015/07/23/puzzle-box-is-rigged-to-blow/)。这不仅是我们见过的最复杂的建筑之一，而且我认为我们从来没有见过一个如此精心设计的拼图盒，它还可以作为一件功能性的装饰品。干得好！
