# 无线电接收机制造日志和更多

> 原文：<https://hackaday.com/2015/12/03/radio-receiver-build-log-and-more/>

在 Hackaday，我们喜欢看到构建日志，在 Hackaday.io 上，你可以找到大量的日志。不过，有时候，一个建筑商真的会在一个项目的很多细节上超越自己，简单接收项目当然属于这一类。项目日志记录了完整性的许多不同阶段，我们为您链接了第一个阶段作为起点，但是您肯定会想读到现在。(上次我们检查时，有 16 个部分，有些跨越了多个帖子)。

不过这绝对值得努力。该项目一开始是一个直接变频接收机，但该设计通过并将其转换为超外差接收机。一路走来，[N6QW]分享了建造技术、设计建议，甚至模拟图(有实际范围测量的支持)。当然，本振使用 Arduino 和 AD9850 频率合成器。

即使你从来没有计划建造这样的接收器(尽管名字很简单，但这不是一个小时的建造)，它仍然值得检查详细的笔记和接收器架构和设计的建议。

如果你在寻找可以一次或两次组装的东西，你可以考虑从再生接收器开始。在之前，我们已经看过[它们，甚至](http://hackaday.com/2015/08/16/design-diary-varactor-tuned-regen-receiver/)[也有一个以其设计者从未想过的方式使用 LM386](http://hackaday.com/2015/07/26/simple-one-chip-regenerative-receiver/) 的。
