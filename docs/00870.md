# NFC 标签为新手机增加了老派功能

> 原文:[https://hack aday . com/2015/12/14/NFC-tags-add-old-school-functionality-to-new-phone/](https://hackaday.com/2015/12/14/nfc-tags-add-old-school-functionality-to-new-phone/)

过去，我们有带物理按键的智能手机。不仅仅是电源，音量，*可能还有*前面的另一个按钮。整个滑出式 QWERTY 键盘位于手机底部。这是一片无法无天的荒地，但对于那些渴望 2000 年代末狂野西部时光的人来说，[利维乌][重新创造了现代智能手机键盘](http://simpledevices.blogspot.ro/2015/11/nfc-shortcut-button.html)顶部曾经存在的快捷按钮。

有许多手机的键盘上都有快捷键，但[Liviu]喜欢使用那些允许他在应用程序(或孩子们现在所说的“应用程序”)之间切换的手机，如日历、浏览器或邮件客户端。为了重现这一点，他用了一些 NFC 标签。这些设备可以通过您选择的应用商店中的许多应用程序轻松编程，并且基本上可以放在任何地方。然而，为了让手机能够随时看到它们，他把标签放在了一个透明的塑料盒里，现在可以随时使用。

如果你从未使用或编程过 NFC 标签，这将是一个很好的项目，让你自己熟悉它们是如何操作的。此外，你可以很容易地升级这个项目，让标签做任何数量的其他事情。你可以把这样的项目进行到你想要的程度。
