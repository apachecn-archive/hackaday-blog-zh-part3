# 紫外激光投影仪闪耀着夜光乙烯

> 原文：<https://hackaday.com/2015/12/01/uv-laser-projector-shines-with-glow-in-the-dark-vinyl/>

镜像电流计最初是在 17 世纪开发的，用于精确测量电流的微小变化。与当时的其他仪器不同，镜式电流计可以通过将镜子的微小运动转化为镜子反射光的大运动来清楚地显示微小的电流变化。在干净的电放大成为可能之前，这是测量电流微小差异的最好方法。真镜检流计是非常灵敏的仪器，但业余伺服系统可以用作低保真度的替代方案，就像[robives]在 Hackaday.io 上创建的这个项目。

使用镜式电流计是迄今为止最常见的激光投影显示技术——这是唯一能够足够快速地移动激光束以实时创建实线视觉错觉的方法。镜式电流计的工作原理是利用线圈吸引附着在镜子上的磁铁，当线圈通电时，镜子的角度就会改变。这种移动非常小，但由于到投影表面的距离而被放大，这意味着激光束在一个实例中可以移动很大的距离。如果你曾经看过一场激光秀，[它几乎肯定使用了这种技术](http://hackaday.com/2011/03/16/laser-light-show-features-full-xy-control-via-homemade-galvanometers/)。但是驱动 galvos 需要一个强大的 DAC，所以我们不能责怪[robives]想要保持它的数字化。

[robives 的]项目通过使用夜光乙烯和紫外激光避开了对检流计的需求。结果是激光束轨迹持续时间更长，这意味着无需高速检流计就可以看到实线。像这样的构建可以让你实验激光投影，而不用处理敏感的镜面检流计，而是使用你工作台上可能已经有的组件。
