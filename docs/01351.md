# 打开 Furby 打开 Furby

> 原文:[https://hackaday.com/2016/01/29/open-furby-opens-the-furby/](https://hackaday.com/2016/01/29/open-furby-opens-the-furby/)

记得弗比吗？几年前，当人们在许多圣诞树下发现这种奇怪的会说话的生物时，可爱的反应机器人风靡一时。自那以后，大多数 Furby 都被可悲地忽视了，但 [Open Furby 项目](https://www.facebook.com/openfurby)旨在赋予这个玩具新的生命，将其转变为一个开源的社交机器人平台。

我们之前已经展示了一些 Furby hacks，例如[精彩的 Furby Gurdy](http://hackaday.com/2008/08/23/furbies-transformed-into-a-furby-gurdy/) 和[互联网连接的 Furby](http://hackaday.com/2009/08/31/internet-enabled-furby/) ，但是 Open Furby 项目旨在创建一个开放平台，而不是创建一个特定的 hack。它的工作原理是用运行[机器人操作系统(ROS)](http://www.ros.org/) 的[闪存控制器](http://www.doc.flash-robotics.com/flash-hardware/specification)取代 Furby 的大脑，使 Furby 更容易编程和控制。他们还用小有机发光二极管屏幕取代了眼睛，这意味着它可以做一些事情，如显示天气预报，facebook 通知等。

它仍处于早期阶段，但它看起来像一个有趣的项目。就我个人而言，我在等着邪恶的弗比，他想用那肮脏的嘴杀死你，吃掉你的肉…
