# 你的 VR(还)不臭

> 原文:[https://hackaday.com/2017/03/21/your-vr-doesnt-stink-yet/](https://hackaday.com/2017/03/21/your-vr-doesnt-stink-yet/)

当你在晚上和周末驾驶的 f1 赛车的车轮变热时，闻起来是什么味道？你不知道，因为让你这样做的虚拟现实体验没有鼻子的成分。还没有。

这里展示的是[一个与 Oculus Rift](https://www.youtube.com/watch?v=LCYe4eul3TA) 和其他头戴式显示器一起工作的嗅觉设备。概念验证是[友川·桥本]、[丸野洋介]和[中本孝道]的工作，并在去年的[IEEE VR 会议](http://ieeevr.org/2016/program/research-demos/)上展示。当佩戴者接近虚拟环境中的一棵树时，它可以让佩戴者闻到橙子的味道。换句话说，它让你的沉浸式体验变得很臭。

目前，这是一个非常酷的小装置，它将气味液滴雾化，同时一个小风扇将它们吹到佩戴者的鼻子里。有一篇论文可能有更多的细节，但它在付费墙后面，所以现在看看下面的简短演示视频。传统上，气味系统的一个问题是物质粘在管线上，这种原型通过从储器直接应用而克服了这一问题。尚未解决的是许多不同香味的可用性。

这个构建通过一篇 UploadVR 文章引起了我们的注意，这篇文章很好地报道了多年来一些基于气味的实验。他们看到了一些和我们一样的障碍:气味挥之不去，可生产的调色板有限。我们假设游戏行业的巨额收入将推动该领域的研究，但我们不会很快就闻到火药味和死尸(或腐烂的僵尸)的味道。

更高尚的努力是在虚拟现实应用中，比如带着老人和行动不便的人再次游览他们一生中再也不能去的地方。添加嗅觉，它有能力打开如此多的记忆，使这个用例更加强大。我们认为这是每个人都可以期待的！
