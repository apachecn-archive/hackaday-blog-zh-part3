# 风铃和干冰是一种不寻常的乐器

> 原文：<https://hackaday.com/2017/09/19/wind-chimes-and-dry-ice-make-an-unusual-musical-instrument/>

当谈到制作音乐时，实际上只有几种方法来创造所需的音调——拨动某物、向某物吹气或击打某物。但是，这个用风铃和固态二氧化碳再现曲调的干冰驱动的器官又在哪里呢？

正如[Leah Edwards]对她的项目所解释的那样，事实证明这是坚定的“击中目标”阵营。当金属风铃管与干冰接触时，温差使固态二氧化碳升华。一股气体将试管轻轻提起，让它落在干冰砖上，发出一个声音。该过程快速重复，在电子管放下时提供颤音效果。[Leah]使用螺线管来提升电子管，最近在国家仪器公司完成了一段时间后，用一堆 NI 齿轮来控制它们。下面的视频展示了一些流行的曲调和一点点关于器官建设。但是什么——没有来自《冰雪奇缘》的歌曲？

我们可以很容易地想象使用 Arduino 或其他微控制器的相同构建。事实上，它让我们想起了最近的一个簧片风琴 MIDI 项目，这个项目有一些想法可以提供，比如让那些螺线管安静下来的方法。
