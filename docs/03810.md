# 盛开的花灯将测试你的 3D 打印机

> 原文:[https://hack aday . com/2016/10/11/blooming-flower-lamp-will-test-your-3d-printer/](https://hackaday.com/2016/10/11/blooming-flower-lamp-will-test-your-3d-printer/)

[奥索姆]有一个孩子要出生了。他承认自己有点忘乎所以，充满了父母的兴奋。结果是一个完全铰接的 LED WiFi 灯[在即将到来的婴儿房间的友好范围内绽放并发出引人注目的光](http://www.instructables.com/id/Blooming-Marvelous-Flower-Lamp/?ALLSTEPS)。

在之前，我们已经介绍过[【ossum】的工作。像往常一样，他开始展示他对 Fusion360 的完全掌握，让我们其他人看起来很糟糕。如果你想学习 360，我们推荐你浏览他的模型，看看是怎么做的。该基地包括一个 ESP8266 和一个业余爱好伺服。一个聪明的机制拉下了一根穿过茎干的钢绞线，还有一些 LEDS 的控制线。这打开和关闭花瓣。所有的 led 都装在一个 3D 打印的框架里，发出均匀的光亮。](http://hackaday.com/2016/07/04/rc-hot-rod-built-completely-from-scratch/)

如果你想自己做一个，休息之后可以观看完整的视频。文件可在 [Thingiverse](http://www.thingiverse.com/thing:1805365) 上获得。只是要确保你先调好你的打印机，这是一个棘手的问题。
