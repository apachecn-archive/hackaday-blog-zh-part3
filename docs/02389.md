# 发展讲故事，将古老的技巧与数字时代结合起来

> 原文：<https://hackaday.com/2016/05/20/evolving-storytelling-to-marry-the-ancient-skills-with-the-digital-age/>

讲故事是一门艺术。它可以追溯到人类诞生之初。它在情感层面上吸引人们，并吸引他们的思维。鲍莉娜·格里塔·斯蒂法诺维奇是一名用户体验研究员和交互设计师，她正站在前沿，将我们的技术与这一悠久传统中最优秀的人性结合起来。

信息时代正在威胁着讲故事——不是让它灭绝，而是减少了讲故事者的数量。我们不再依赖我们亲密社交圈中的人来为我们自己的享受讲优美的故事；我们有奢侈(也许是诅咒？)的大众市场讲故事。

鲍莉娜的作品开启了交互式讲故事的大门。这个想法并不新鲜，因为伟大的故事讲述者总是阅读他们的观众，并发挥他们的参与。数字时代的交互式讲故事试图将这种技能设计到传递故事的技术中。这是被动娱乐的回归。

这分为交互式和响应式。最简单地说，把 responsive 想象成一个有暂停按钮的视频。你可以改变故事的流程，但你不能让故事成为你自己的。令人惊讶的是，这是一个新的发展，因为暂停播放的能力只有几十年的历史。因此，你可以暂停一个响应媒体，但真正的互动体验涉及创作——观众沉浸在故事中，可以在体验过程中对结果做出实质性的改变。

这等同于权力转移。媒体的创造者不再完全控制，让出一些给观众。我们刚刚开始这项技术，看起来天空是我们在算法交互方面所能做的极限。

电子游戏是这种变化的先驱。他们已经有了分支故事，让用户做出极大地影响故事情节的选择。这个行业非常庞大，显然故事消费的活跃是成功的重要因素。更有趣的是“戏剧管理系统”(对我来说是一个新名词，但我喜欢它)，它导致一个故事的结局，直到这个特定的观众到达那里才有人知道。这是什么概念，我迫不及待地想亲眼看看！
