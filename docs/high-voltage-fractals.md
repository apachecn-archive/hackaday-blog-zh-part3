# 高压分形

> 原文：<https://hackaday.com/2016/09/01/high-voltage-fractals/>

Int 1777 年，Georg Lichtenberg 发现，在覆盖有粉末的绝缘表面上释放高压，会出现类似分形的图像，有时被称为闪电树。顺便说一句，这是静电复印术的一种原始形式，是复印机和激光打印机工作的原理。

[PaulGetson]从他的 Jacob's ladder 实验中得到了一个高压电源，并决定看看他是否能创造出 Lichtenberg 图形。原来，[他可以](http://www.instructables.com/id/Making-Fractal-Art-With-High-Voltage/)。

如果你有一个高压源，这个项目的其余部分就相当简单了。一块胶合板作为绝缘体，小苏打晶体使木材的顶层更加导电。然而，胶合板中的粘合剂会阻止溶液渗透到板的核心。

另一个有趣的技巧是使用迷你杯作为高压绝缘体。这些很容易买到，也不贵。这是一个不同的高压设置，但你可以在下面的视频中看到类似的实验。
