# 虚拟现实设备浏览器 Firefox Reality

> 原文:[https://hack aday . com/2018/04/06/Firefox-reality-a-browser-for-VR-devices/](https://hackaday.com/2018/04/06/firefox-reality-a-browser-for-vr-devices/)

你正在阅读的这个网页的浏览器将会是一个非常强大的软件，其功能和 API 是 NCSA Mosaic 等 90 年代早期的开发者做梦也想不到的。尽管如此，它在视觉上很可能是那些早期浏览器的后代，一个显示二维网页的窗口。

这其中的一些可能即将改变，因为认识到虚拟现实设备正在为自己创造的空间，Mozilla 发布了 [Firefox Reality](https://blog.mozvr.com/firefox-reality-bringing-the-immersive-web-to-mixed-reality-headsets/) ，用他们的话说“*一款全新设计的独立虚拟和增强现实耳机*的新网络浏览器。目前，它将作为开发者预览版在 Daydream 和 GearVR 设备上运行，但该软件的预期目标是尚未发布的下一代硬件。

记性好的读者可能还记得 20 世纪 90 年代浏览器中围绕虚拟现实的一些炒作，当时阅读过 [VRML](https://en.wikipedia.org/wiki/VRML) 的水晶球观察者会欢呼它是下一件大事，而不会停下来想想市场上是否有支持它的设备。这一次硬件可能会符合预期，也许有一天你会在黑客时代的扳手空间里漫步，而不是在浏览器里阅读这篇文章。那里见！

他们发布了一个视频预览，令人失望的是它包含了一个虚拟现实环境中的 2D 浏览器窗口。但这是一个开始。
