# 现实生活中的乓，机械乓

> 原文：<https://hackaday.com/2016/05/28/pong-in-real-life-mechanical-pong/>

丹尼尔·佩尔多莫和他的两个朋友在过去的两年里一直致力于 Pong f T1 的机械版本。我们可以有把握地说，最后的结果是美好的。看着 PIXE–cube 在表面来回移动，感觉非常空灵。

[Daniel]从事计算机图形广告工作已经超过 20 年。然而，他指出，他和他的朋友们在开始时都没有任何机械或电子方面的经验。谢天谢地，互联网(大概还有 Hackaday 这样的网站)为他们提供了所需的信息。

乒乓球拍和像素球？)坐在玻璃表面上。运动部件被磁铁限制在机械装置上。下面是一个类似于[蚀刻草图](http://hackaday.com/2015/02/02/automated-etch-a-sketch-re-produces-famous-artwork/)的结构，用于在球拍位于一根带的轨道上时移动球。整个组件由 V 形槽挤压制成。

我们最喜欢的部分是用来来回移动桨的滚轮。对于后面有一些质量的漂亮平滑的运动，什么比硬盘驱动器盘片更好？他们打印出一个编码器轮图案，并将其粘在表面上。电子产品都是手工制作的。大脑似乎是一些较大的 Arduinos。8 位段、彩虹 led 等是用看起来像铜箔带连接总线的胶带粘合在一起制成的。这绝对是一个爱的劳动。
