# 来自传送门 2 的机器人惠特利

> 原文：<https://hackaday.com/2015/11/30/robotic-wheatley-from-portal-2/>

传送门 2 推出已经 4 年多了，但惠特利这个带有英国口音的人工智能角色仍然是一个迷人的角色。[伊维·比]制造了一个完全拥有声音、动作和一只发光眼睛的惠特利复制品。

惠特利的车身由蓝色隔热泡沫制成，也称为 XPS 泡沫，用 UHU 聚氨酯胶层压在一起。这形成了一个球体，然后被切割成一个详细的机构。纸糊粘土被用来加强薄薄的泡沫。

这个建筑的电子设备提供光、运动和声音。眼睛由总共 3 个 Arduino 控制的伺服系统移动:两个用于眼睛的移动，一个允许它打开和关闭。移动由两个操纵杆控制。声音由 [Adafruit 音板](https://www.adafruit.com/products/2133)提供，音板连接到一个扬声器和一个 [Velleman 声光套件](http://www.vellemanusa.com/products/view/?country=us&lang=enu&id=350678)。这个套件控制照亮眼睛的发光二极管，使它对惠特利的声音做出反应。

休息之后你可以看惠特利对你的咆哮。当然，如果你想要一份惠特利，你也需要一份 GLaDOS 土豆。
