# 星球大战飞车的点睛之笔:镜子

> 原文：<https://hackaday.com/2017/11/08/the-star-wars-speeders-finishing-touch-mirrors/>

【超 73】做电动滑板车，他们为万圣节做了一些[星球大战飞车](https://www.youtube.com/watch?v=ikXP1NT_eZA)；在自行车底部添加一些镜子面板可以产生不错的视觉效果，不需要维护或花哨的工作。自行车让每个人都感到惊讶，他们随后发布了一段关于建造过程的[视频](https://www.youtube.com/watch?v=gXub4-LXk40)。

飞车是围绕他们的 Super 73 电动滑板车建造的外壳，底座看起来像中纤板，坐落在锚点上。在基础平台上放上纸板和膨胀的泡沫来创造正确的形状，然后用砂纸打磨，再涂上玻璃纤维和 bondo。然后是油漆、风化和所有需要的各种各样的零件，以使飞车尽可能精确到屏幕上。真正的点睛之笔是隐藏车轮和创造悬浮幻觉的镜面面板。只要镜子的角度能让行人看到反射的路面，它就能很好地工作。

最后，穿上戏服，在镇上兜一圈(当然有很多相机，他们自然想抓住一些眼球)，我们不得不说，最终的结果看起来很棒。展示和制作视频都嵌入在下面。
