# Hackaday 大奖参赛作品:3D 打印迷你车床

> 原文：<https://hackaday.com/2017/05/31/hackaday-prize-entry-3d-printed-mini-lathe/>

车床可能是又大又强又危险的机器。但是有时需要用柔软的材料制作非常小的零件，比如塑料和木头。对于这样的工作，你可以使用像这样的东西 [3D 打印微型车床](https://hackaday.io/project/20089-3d-printed-manual-mini-lathe)。

3D 打印像这样的工具有很多好处。最终用户可以定制和改进设计；[castvee8]注意到，只要增加丝杠和导轨的长度，机器就可以变得更长。这台机器确实依靠一些金属零件和一个马达；但是这里真正强大的地方是，如果您无法获得确切的组件，您可以随时定制文件以适合您手头的内容。

[castvee8]旨在让新手尽可能轻松地完成整个构建，甚至连电机和速度控制器都是现成的模块。这是我们生活的黄金时代的证明，整个车床可以由模块和 3D 打印零件构建而成。该项目构成了 3D 打印工具家族的又一成员[【cast vee 8】正在 Hackaday.io 上炫耀](https://hackaday.io/MarkMiller)
