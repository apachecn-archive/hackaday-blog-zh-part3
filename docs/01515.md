# 教学数控新招

> 原文:[https://hackaday.com/2016/02/16/teaching-a-cnc-new-tricks/](https://hackaday.com/2016/02/16/teaching-a-cnc-new-tricks/)

计算机数字控制技术已经存在很长时间了。它是我们 3D 打印机、激光切割机/蚀刻机和数控铣床的核心。它们都以同样的方式工作——你从 CAD 程序开始，进行某种类型的设计。然后，计算机将文件转换成一组 XYZ 坐标，并相应地移动工具头。现在让我们向自己提出一个最有趣的问题。如果你把过程反过来呢？如果你能把一个 CNC 控制的物体转换成 XYZ 坐标会怎么样？

这正是[戴夫]试图做的事情。他制作了一套基本的数控设备，并在步进器上安装了编码器。然后他手动移动工具头来追踪一个物体。同时，编码器将坐标输入电脑进行记录。这个想法是重放坐标，看看数控系统是否能复制对象。

从下面的视频来看，这个项目是成功的！
