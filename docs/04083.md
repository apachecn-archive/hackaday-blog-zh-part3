# 一款真正经典的金属框架迷你电脑

> 原文:[https://hack aday . com/2016/11/13/a-true-classy-metal-framed-mini-CNC/](https://hackaday.com/2016/11/13/a-truly-classy-metal-framed-mini-cnc/)

我们已经在这里看到了许多基于 DVD 和 CDROM 的小型数控机床，但很少有像 Instructables 上的[julioberaldi]的这个一样简单漂亮的。

[![fbis2gciv0ajsdx](../Images/822e5549d818e9d6478d9351485bbf50.png)](https://hackaday.com/wp-content/uploads/2016/11/fbis2gciv0ajsdx.jpg) 我们马上切入正题；是相框。用钢锯从钢片废料上切割下来，用“棒焊”焊接或锡焊在一起。看起来好像进行了大量的打磨、上漆和抛光。我们会很自豪地将结果放在我们的办公桌上。

目前，它只是用笔画画。但是看视频，嵌在下面，你会发现它运行异常流畅。如果我们正在阅读 Instructable 权利，下一步是把这变成一个数控切割机。我们迫不及待地想知道这个项目将何去何从。
