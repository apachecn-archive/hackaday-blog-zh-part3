# 446 字节的俄罗斯方块

> 原文:[https://hackaday.com/2016/10/06/tetris-in-446-bytes/](https://hackaday.com/2016/10/06/tetris-in-446-bytes/)

你想玩俄罗斯方块。你想在任何操作系统上玩俄罗斯方块。你想在旧的 IBM 电脑上玩，你想在新的 MacBook 上玩俄罗斯方块。你想要一个能放入磁盘主引导记录的俄罗斯方块。你想要俄罗斯方块作为操作系统。[你想要 TetrOS](https://github.com/daniel-e/tetros) 。

也许你不知道，但这是一件了不起的作品，我们喜欢小演示。看看下面的休息。或者通读横幅图片中的源代码[。](https://hackaday.com/wp-content/uploads/2016/10/tetros_featured.png)

它没有分数，或者花哨的图形，或者介绍闪屏。它不会显示下一块砖。它不会随着你继续玩而加速。致命的缺陷是后台不玩俄罗斯方块主题。但是它很小，而且可以玩。

说到没有操作系统的俄罗斯方块，如果你有一个相对宽敞的 Arduino T1，你可以得到一个拥有所有装饰的版本。
