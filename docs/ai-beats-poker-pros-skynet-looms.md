# 人工智能击败扑克职业:天网织机

> 原文：<https://hackaday.com/2017/02/07/ai-beats-poker-pros-skynet-looms/>

最近在人工智能对战人类的游戏中出现了一些“第一次”，计算机现在正在琐事、国际象棋和围棋上击败我们。但从某种意义上说，这些都不是真正有趣的；它们都是事实的游戏。扑克不一样。除了计算手握胜牌的几率(计算机在这方面显然具有优势)之外，在扑克游戏中获胜的关键是虚张声势，并判断出对手何时在虚张声势。直到最近，这才帮助人类战胜了机器。那些日子已经过去了。

国际象棋和围棋是博弈理论家所说的[完美信息](https://en.wikipedia.org/wiki/Perfect_information)博弈:每个人只要看一看棋盘就知道关于博弈状态的一切，这意味着原则上每个可能的位置都有一个最佳策略(一系列的走法)。诚然，很难弄清楚这些，因为这是一个*大*蛮力问题，但它仍然是一个蛮力问题，计算机具有先天优势。象棋和围棋是机器*应该*赢的游戏。

在扑克中，你不知道游戏的状态，因为你看不到对手的牌。当你的对手通过下大注发出有好牌的信号时，你的工作就是解读这个信号并加注或弃牌。一个优秀的人类扑克玩家可以通过这样的防守，但真正伟大的扑克玩家也知道什么时候虚张声势。(以及何时走开，何时逃跑。)这种战略欺骗的元素在过去已经足以让人类获得超越计算机坚如磐石的统计数据的优势。我们骗过了机器。

卡内基梅隆团队除了在这个问题上投入大量的计算能力之外，还专注于提高防御能力。每天晚上，电脑都会问自己最大的三个错误是什么，并修改策略以在第二天避免它们。通过在牌局的早期阶段提高对虚张声势的防御，它存活到了后期阶段，在这一阶段，它的优势是可以准确计算赔率。这可能迫使人类玩家在翻牌前比他们应该冒的风险更大。

我们渴望看到更多的信息和分析，但我们还得等几个月才能看到论文发表。不过现在，你应该把这一天记在你的日历上——电脑称之为我们虚张声势的那一天。
