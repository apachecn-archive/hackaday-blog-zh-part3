# 一个便宜的，基于 555 的盖革计数器

> 原文：<https://hackaday.com/2016/10/16/a-cheap-555-based-geiger-counter/>

每个疯狂科学家的巢穴都需要一个盖革计数器。毕竟，如果不明飞行物坠毁在后院，你可能需要知道它是否热。[Tanner_Tech]向你展示如何[建造一个便宜的](http://www.instructables.com/id/Simplest-Geiger-Counter/)来完成这项工作。

你确实需要一个盖革管，但在一个热门的拍卖网站上快速搜索一下，就会发现俄罗斯有很多剩余的几块钱。你需要的另一件事是高压电源(约 400 伏)，这是使用基于 555 的直流到 DC 转换器的电路的核心。你可以在下面看到该设备工作的视频。

直流-DC 转换器需要一个从闹钟里偷出来的变压器。一个压电传感器(从一个垃圾微波炉里偷来的)给你一种特有的咔哒声。如果你喜欢固态而不是中空状态，有一个[开源项目](https://hackaday.com/2014/10/25/use-a-cheap-pin-diode-as-a-geiger-counter/)使用 PIN 二极管作为传感器。或者你可以添加[一个 Arduino 和一些 led](https://hackaday.com/2015/07/26/pimp-my-geiger-counter/)。
