# Hackaday 奖半决赛:分享污染分析

> 原文:[https://hack aday . com/2015/09/19/hack aday-prize-半决赛选手-分享-污染-分析/](https://hackaday.com/2015/09/19/hackaday-prize-semifinalist-sharing-pollution-analytics/)

不久前，[约书亚·杨]和一位环境科学家进行了一次谈话。德克萨斯州没有多少政府资助的污染监测站，但休斯顿有许多富裕的业主协会拥有收集数据的传感器。空气质量监测很重要，数据越多通常越好，如果没有这些 HOA 免费提供数据，这些环境科学家就不会有数据来完成他们的工作。

[Joshua]的项目是将 HOA 的一些成员的想法扩展到整个世界。为了获得黑客日奖，他正在创建一个系统，与整个互联网共享当地的污染数据。

约书亚正在建造的系统使用一套空气质量传感器来测量二氧化硫、一氧化碳、氮氧化物、臭氧和颗粒物。这些传感器通过 ESP8266 WiFi 模块或 LoRa 无线电模块连接到互联网，将数据推送到云上，并让整个世界知道空气质量如何。

以前已经有人使用成千上万个单独的基站来收集数据；为了获得更好的天气跟踪分辨率，Weather Underground 使用的气象站数量是美国国家气象局的十倍。污染传感器通常不是气象站的一部分，通过[约书亚]的项目，跟踪这些数据的环境科学家将有望获得他们需要的数据。
