# 夏天来了:遥控吹雪机

> 原文：<https://hackaday.com/2017/05/11/just-in-time-for-summer-a-remote-controlled-snowblower/>

现在是五月，你知道这意味着什么:我们正在从一场全球工人庆祝活动中逐渐放松，花粉到处都是，南半球的黑客读者不知何故被冒犯了，在某个地方，有人正在完成一个远程控制的吹雪机。

在这个九集两小时长的视频系列中， [Dave]讲述了最令人垂涎的寒冷天气庭院乐器之一的设计和制作。这是遥控吹雪机。想想看:与其捆起来去吹走车道，[戴夫]可以从他舒适的客厅窗户里把车道上的雪都弄走。当然，现在是鳄鱼&袜子天气，这听起来可能没什么大不了的，但这将是七八个月后的一项伟大发明。

这个吹雪机器人是围绕电动轮椅上的两个电机建造的。大多数雪花莲已经有了轨道，所以对于这个建筑来说非常重要的牵引力已经解决了。线性致动器负责“铲子”的角度，自行车链轮、链条和电机的巧妙组合允许吹雪机的“滑道”指向任何方向。电子设备非常简单——一个普通的现成 RC 发射器和接收器处理无线通信，而 Arduino 接收这些信号，并将其转化为继电器和电机可以理解的信号。

这是我们见过的构建较好的 vlogs 之一。这个版本有九个部分，我们在下面包括了最后的总结视频。
