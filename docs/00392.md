# 动力滑板已经过时了；Skelecs 是新的热点

> 原文：<https://hackaday.com/2015/10/17/powered-skateboards-are-passe-skelecs-the-new-hotness/>

对于这个不可避免的问题，哈里斯给出了一个有趣的答案:他在暑假做了一双电动滚轴溜冰鞋。【Harris】是诺丁汉大学电气工程专业的学生，他在返校前完成了他称之为 Skelecs 的第一个版本。他记录了从[最初的概念和建造他自己的控制板](http://hsel.co.uk/2015/09/06/skelecs-and-a-custom-bldc-controller/)，通过[他在钢底座上正确钻孔的失败](http://hsel.co.uk/2015/09/17/skelecs-part-3-the-build-and-the-failure/)，到他的[第一次开车上路](http://hsel.co.uk/2015/09/24/skelecs-part-6-the-final-frontier/)的过程。

他的作品使用一对 120 瓦的小型轮毂电机连接到一个钢制底盘上，底盘连接到一双拆下来的轮滑靴上。

这有点像弗兰肯斯坦(Frankenstein)的构建(他目前将电池和控制器塞在裤子口袋里，这不是一个真正实用的长期解决方案)，但它确实有效。事实上，有点太好了:[哈里斯]说，速度和崎岖不平的道路结合在一起，使其中一个电池脱落，让他飞了起来。尽管如此，他并没有让一点小伤和一点血阻止他:他已经开始了第二版的工作，该版本将使用更轻的铝制结构和一对[全方位轮](http://hackaday.com/2013/06/08/omniwheel-robot-build-uses-a-bit-of-everything/)，以实现更容易的转向和更多的控制。当我们看到它的时候，我们会相信那个声明。

记住，[电动滑板](http://hackaday.com/2014/08/12/electric-longboard-roundup/)已经结束了——非黑客染指它们，所以它们很常见。潮人黑客需要放弃那个版本，开始使用你自己的 Skelecs。
