# 同步调酒师 Drinkro

> 原文:[https://hack aday . com/2016/07/25/drink ro-the-synchro-barber/](https://hackaday.com/2016/07/25/drinkro-the-synchro-bartender/)

黑客和制造商中有一大群人，对他们来说，重要的不是提供饮料的环境或提供饮料的人的性格，而是饮料制备的质量。对他们来说，这不是一个见识过一切的酒保的智慧，而是电脑控制的精确调制的饮料。他们是调酒机器人的创造者，也许有一天所有潮湿的酒馆都会被他们的作品取代。

Drinkro 是由[Synchro Labs]的团队制造的调酒机器人。它使用一个树莓 Pi 3 和一个定制的电机控制器板来驱动一对 DC 蠕动液体泵。将各种成分的饮料倒入用户的杯子中。有一个多平台的 app，多个口渴的饮者可以通过它下单，所有的[源代码](https://github.com/SynchroLabs/Drynkro)和[硬件文件](https://github.com/SynchroLabs/DrinkroHardware)都可以在 GitHub 仓库中找到。这个机器人拥有相当少的伏特加和三个搅拌器，但也许它会扩展到更多的电机驱动器和泵的组合。

有一个[机器运行的视频](https://www.youtube.com/watch?v=6KpcA1cZvbw)，显示在画面下方。我们不禁注意到它不是最快的调酒师，但也许速度不是一切。
