# 廉价的空中皮卡

> 原文：<https://hackaday.com/2016/04/04/cheap-air-pickup/>

如果您使用表面贴装元件，您可能需要一个空气拾取工具(有时称为空气或真空镊子)。你可以找到便宜的使用灯泡或弹簧装置的(像一个焊钳)。虽然这些很便宜，但它们不是很好用。[Natsfr]有一个这样的廉价工具，并决定[添加一个合适的泵](http://more-magic.org/2016/03/23/stylo-ventouse/)使它像一个昂贵得多的工具一样工作。

帖子是法语的，但是翻译起来很容易。下面还有一个视频你可以看看。计划很简单:将一个泵转换成真空源，并连接到笔中。代替无处不在的[改装水族箱泵](http://hackaday.com/2010/12/09/quick-cheap-and-simple-vacuum-tweezers/)，【Natsfr】改装了一个(甚至更便宜)他从网上拍卖得到的泵。

如果你想买些更别致的东西，试试这个带压电泵的取放工具。
