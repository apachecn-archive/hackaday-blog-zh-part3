# 剪刀是伟大的自动电缆切割机

> 原文：<https://hackaday.com/2016/10/19/scissors-make-great-automatic-cable-cutters/>

[2PrintBeta]的团队需要为他们的客户裁剪一堆电缆、热收缩材料和编织物。他们研究了一种工业电缆切割器，但觉得价格有点太高，所以决定自己制作。他们对切割有很多想法:用刀片？还是带切割轮的 Dremel？他们想出了一个 [DIY 电缆切割器](http://ifeelbeta.de/index.php/art-objects/automatic-cable-cutter)，它使用了一把剪刀、一对步进电机、一对 3D 打印轮和一个 Arduino。

团队要做的第一件事是安装剪刀，这样他们就能可靠地剪切。其中一个步进电机安装在一个驱动轮上，驱动轮上装有一个螺栓。这穿过剪刀的一个手柄，另一个手柄用螺丝固定在机器上。第二个步进电机用于旋转轮子，驱动电缆到达正确的长度。[2PrintBeta]使用 Arduino Mega 上的一个 [BAM &骰子盾](https://www.2printbeta.de/Electronics/Controller/BAM-DICE-Arduino-STK-kit::328.html?language=en)和两个[骰子 STK 步进电机驱动器](https://www.2printbeta.de/Electronics/Modules/DICE-STK-V1::330.html?language=en)来控制切割机。

[2PrintBeta]团队非常擅长自己动手，就像我们之前看到的他们的 DIY 塑料弯曲机一样。再一次，有了这个自动电缆切割机，他们看到了需求，并利用他们可支配的东西和一些 DIY 的独创性解决了这个问题。
