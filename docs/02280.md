# 自动驾驶汽车变得很小

> 原文：<https://hackaday.com/2016/05/16/self-driving-cars-get-tiny/>

现在正在进行一场汽车比赛，但不是在任何类型的赛道上。有很多公司都在争着让他们的原型车先上路。【Anurag】已经完成了任务，然而[除了他的车和路都是功能型](http://anuragmishracse.github.io/)。

虽然他的汽车不像谷歌自动驾驶汽车那样复杂，也不需要处理行人和其他活动障碍物，但它确实使用计算机和各种传感器来决定如何驾驶。一个 Raspberry Pi 2 在这个构建中控制方向盘，从 Pi 相机和超声波距离传感器获取输入。Pi 通过 WiFi 与另一台计算机通信，其中一个神经网络运行以做出如何驾驶汽车的决定。它还根据赛道图片数据库做出决定，因此它有一个参考点。

该车在行动中的视频值得一看。它并不完美，但对于这种类型的项目来说，这已经是相当大的成就了。自动驾驶汽车模型可以像铁路爱好者创建的模型一样在模型集周围行驶的可能性非常有趣。当然，这不是[阿努拉格]绕街区的第一圈。他已经因制造了一辆可以基于手势驾驶的汽车而出名。我们期待着他什么时候能[和模型巴士](http://hackaday.com/2016/04/18/the-predictability-problem-with-self-driving-cars/)相撞。
