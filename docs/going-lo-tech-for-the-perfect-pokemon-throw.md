# 为完美的口袋妖怪 Go 投掷去低技术

> 原文：<https://hackaday.com/2016/07/13/going-lo-tech-for-the-perfect-pokemon-throw/>

我们期待史诗般的 GPS 恶搞来捕捉 Pokemon Go 中的一些传奇，但在那个黑客出现之前，我们真的很喜欢[Brian MCE voy]的[黑客完美的扑克球投掷](http://www.24hourengineer.com/2016/07/2016-07-12-tu-pokemon-go-simple.html)。

[Brian]开始认为机械制造将是最好的方式(我们知道他有伺服电机和控制器来驱动它们，从去年他制造的这个泡茶机器人)。但是对于你得到的回报来说，这种机制太复杂了(更少的浪费球)。

他意识到你的手指是最好的机器，只是需要一些增强。他的大部分投球都没有击中一边或另一边，所以他转向纸艺来指引方向。他用一些纸板包装做了一个托盘，然后用两小叠便利贴创造了一个手指可以滑动的通道。只需用一只手握住手机和纸张，用另一只手顺着纸张通道成功抓拍。纸板不会影响屏幕感应手指的能力。

这是我们肯定要尝试的一个。但是震撼世界的游戏硬件黑客的景象仍然在我们的脑海中跳动。你在忙什么吗？如果是这样，我们很乐意听到(所以[发送一个提示](http://hackaday.com/submit-a-tip/)！).那些还在构思阶段的人可以在下面打电话。我们正在权衡在手机和 GPS 芯片之间做中间人来欺骗定位的可行性。这感觉像是一座很难攀登的高山。
