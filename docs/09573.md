# DIY 潜水器的目标是低成本、易操作

> 原文：<https://hackaday.com/2018/05/26/diy-submersible-aims-for-low-cost-ease-of-operation/>

如果你和我们一样，水体是惊奇和敬畏的源泉。惊奇来自于想象隐藏在表面之下的东西，而敬畏来自于害怕尝试去发现并成为那些永久被淹没的物体之一。因此，如果你想相对舒适和安全地探索深海，DIY 遥控水下机器人可能是你需要建造的东西。

如今大多数 ROV 的建造似乎都遵循或多或少类似的设计，这可能是因为他们都有类似于[dcolemans]的项目目标:建造一些在水下四处看看的东西，使其易于操作，并且不花费大量的金钱。为了实现这一点，他使用 1/2 英寸的 PVC 管和配件来建造框架，并将它漆成黄色以便于查看。用于电子设备的干管由 4″ ABS 管制成。干管提供的正浮力几乎被通过排水孔涌入框架的水和储存在着陆滑橇中的铅弹压载抵消。推进由舱底泵筒提供，舱底泵筒带有 3D 打印的导管螺旋桨。一个很好的接触是一个单独的顶部控制箱，带有一个用于 ROV 摄像头的屏幕，可以与常规 RC 控制器对话，以及简化的控制和自动位置保持。查看下面视频中最近的游泳池测试。

海底发生了很多事情，有很多方法可以探索它。你可以部署形状像蛤蜊的传感器，用激光杀死水下的虱子，或者甚至滑行到黑客的领地。
