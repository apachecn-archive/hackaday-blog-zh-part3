# 非常漂亮的万向架，有很长的特性列表

> 原文:[https://hackaday.com/2016/02/29/pretty-gimbal/](https://hackaday.com/2016/02/29/pretty-gimbal/)

当你有一台很好的数控机床，但要造出像[这个三轴万向节](http://e.pavlin.si/2016/02/04/3-axis-gimbal-for-garmin-virb/)这样漂亮的东西，你能做什么？在之前，我们介绍过[Gal]的一些[的工作，他不同意黑客应该看起来像黑客的观点。如果你打算在某样东西上花上几个小时，为什么不把它做得比你能买到的任何现成的东西都好看呢？](http://hackaday.com/2016/02/19/turn-your-10-dollar-mouse-into-a-fancy-10-dollar-mouse-with-cnc/)

摄像机由三个空心轴万向电机固定，电机的齿槽[低](http://hackaday.com/2016/02/23/anti-cogging-algorithm-brings-out-the-best-in-your-hobby-brushless-motors/)。我们不知道空心轴电机，但可以想到很多传感器安装，这样的电机可以用来制作非常紧凑和光滑的传感器安装，而不是通常的爱好伺服配置。大脑是现成的框架控制器。万向节背面有一个 DB9 端口，用于处理内部 LiPo 电池的充电，并给他一个输入 R/C 信号进行手动控制的地方。

该案件是由数控木材和铝。有很多美好的触摸。例如，他添加了两个按钮，这样他就可以微调万向节的倾斜度。每个按钮都单独刻有一个上/下箭头。

[Gal]逆向设计了他正在使用的 Garmin action camera 上的连接器，这样他就可以保持供电、播放视频或添加外部麦克风。接下来，他基于 Boscam 模块制作了一个定制的 5.8Ghz 视频发射器。变送器连接到万向节上的 DB9 充电端口。

当有人为自己建造一些远远超出他们所能买到的东西时，这是非常酷的。休息后的几个视频。
