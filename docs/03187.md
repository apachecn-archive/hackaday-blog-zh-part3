# Kinect 和 Raspberry Pi 为 DSLR 增加了焦点

> 原文:[https://hack aday . com/2016/08/27/Kinect-and-raspberry-pi-add-focus-pulling-to-dslr/](https://hackaday.com/2016/08/27/kinect-and-raspberry-pi-add-focus-pulling-to-dslr/)

Prosumer 数码单反相机是数字媒体民主化的福音。那些曾经以专业价格出售的装备，现在也能被那些预算不多的人买到。这不仅释放了一股在线内容的洪流，还引发了一波相机黑客和配件的浪潮，比如基于 Kinect 和 Raspberry Pi 的自动对焦器。

对[汤姆·皮森斯]来说，佳能 EOS 5D 一直是一个坚实的平台，但遇到了一个问题。数码单反相机可能的狭窄景深使其难以保持对相对于相机移动的物体的聚焦，使得像《T2》这种经典的这样的跟随聚焦场景难以再现。为了实现比股票自动对焦更好的系统，[Tom]将 Kinect 传感器和步进电机致动器移植到 Raspberry Pi，并使用 Kinect 的深度图来驱动对焦环。零件是激光切割的，包括一个漂亮的 Pi 和显示器外壳，使整个东西相当便携。下面的视频显示焦点仍然锁定在选定的感兴趣区域。似乎只允许沿一个轴运动；我们希望看到这个系统扩展到跟随一个指定的物体，不管它在框架中移动到哪里。

如果你需要一个跟踪对焦装置，但没有齿轮镜头，看看这些 3D 打印的镜头齿轮。他们将是对[这个穷乡僻壤的焦点人物](https://hackaday.com/2016/01/01/hillbilly-lego-focus-puller/)的巨大补充。
