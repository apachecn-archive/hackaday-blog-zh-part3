# 雷公藤龙 3D 打印机的曙光

> 原文:[https://hack aday . com/2016/11/30/the dawn-of-the-tripron-3d-printer/](https://hackaday.com/2016/11/30/dawn-of-the-tripteron-3d-printer/)

笛卡尔 3D 打印机是最初的。然后 delta 打印机出现了，它们也很酷。[现在，您可以将 tripteron 打印机添加到组合中](http://forums.reprap.org/read.php?166,719575)。[雷公藤红素](http://robot.gmc.ulaval.ca/en/research/research-thrusts/parallel-mechanisms/tripteron-and-quadrupteron/)是笛卡尔坐标和德尔塔坐标的奇怪组合。该系统由加拿大魁北克拉瓦尔大学的机器人实验室发明。创造它的团队说，它是“各向同性和完全解耦的，即每个致动器都控制一个笛卡尔自由度，相互独立。”这意味着驱动机器人将几乎像驱动标准的 X/Y/Z 笛卡尔打印机一样简单。随之而来的当然是三角洲机器人[，它们遵循一套完全不同的运动学规则](http://reprap.org/wiki/Linear_Delta_Printer_Kinematics)。

这些年来，已经有一些人对 tripteron 打印机进行了实验，但就我们所见，还没有人展示过工作模型。进入[阿卜苏]，他大约一个月前出现。他开始在 RepRap 论坛上发帖讨论他的特殊设计。[阿卜苏]工作很快，因为他现在已经展示了一个制作印刷品的工作原型。当然，它们只是校准立方体，但这是一个巨大的进步。

[阿卜苏]承认他的研究还有很长的路要走——特别是改进 arm 和联合执行。然而，他很高兴他的创作已经从一堆零件变成了一台新型 3D 打印机。我们也是——我们迫不及待地想看到下一次迭代！
