# 3D 打印机的 Hydra

> 原文:[https://hackaday.com/2016/06/30/a-hydra-of-a-3d-printer/](https://hackaday.com/2016/06/30/a-hydra-of-a-3d-printer/)

3D 打印机非常适合生产一个东西，但是如果你需要多个副本，工作流程很快就开始走下坡路了。解决办法？[建造一台带有多个打印头的 3D 打印机](https://www.youtube.com/watch?v=DeVeyUas5vU)，能够在打印一个物体所需的时间内打印四个物体。

这个构建是对大多数印刷的 CNC /多工具的一个实验。这是一台 CNC 机器，使用印刷零件和 3/4”电气导管作为框架和导轨。最后一点是有趣的部分:电导管便宜，容易获得，随处可得，可以用钢锯切割。就台式数控机床而言，没有比这更简单或更便宜的了，其中一些构建是用基于线性轨道的机器的相同质量来铣削木材。它在最近的博卡轴承大赛中赢得了大奖[，并且是一个便宜且耐用的 2.5 或 3D CNC 的重要基础。](http://www.bocabearings.com/innovation-contest/ContestantDetails.aspx?ProjectID=179)

[allted]已经有了这个廉价的数控铣床[切割铝](https://www.youtube.com/watch?v=GIYMuYgX0rY&feature=autoshare)和[用激光雕刻木头](https://www.youtube.com/watch?v=yUk8ttJmpz4)，展示了一个非常便宜但高度可扩展的数控机床的能力。这是一个奇妙的构建，我们迫不及待地想看到更多这样的机器出现在车库和工作场所。
