# 努力融入音乐

> 原文:[https://hackaday.com/2018/05/28/muscle-your-way-into-music/](https://hackaday.com/2018/05/28/muscle-your-way-into-music/)

受到一个古老的香料广告的启发，[Juliodb96]决定他也想通过展示肌肉来创作音乐。一个 Arduino 和一个 MyoWare 传感器完成了这个任务。然而，他也会告诉你如何制作自己的传感器，如果你愿意的话。你可以在下面的视频中看到这台仪器的运行。

如果你使用现成的 MyoWare 传感器，这是一个非常简单的项目。你只是通过弹奏一些音符来响应传感器输入。如果你决定推出自己的，你会有一些电路建设在你面前。

具体而言，传感器的信号调理包括滤波以消除不在 20 Hz 至 300 Hz 通带内的信号、几个放大器、一个整流器和一个削波器。这需要 3 个 IC 封装和少量分立元件。

与最初的广告不同(见下面的第二个视频)，没有用于驱动实际乐器的移动部件。然而，添加一些伺服电机、气泵等并不难。这可能看起来很无聊，但我们不得不怀疑它是否可以被用来让那些不会演奏乐器的人表达音乐。

这并不是我们第一次看到 [MyoWare 在运行](https://hackaday.com/2016/11/20/emg-tutorial-lets-you-listen-to-your-muscles/)。我们甚至讨论过对这类应用有用的[信号处理](https://hackaday.com/2015/12/29/amplifying-the-bodys-own-electricity/)。
