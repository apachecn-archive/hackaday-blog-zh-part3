# 家酿啤酒不是哈克

> 原文:[https://hackaday.com/2018/04/28/homebrew-not-a-hakko/](https://hackaday.com/2018/04/28/homebrew-not-a-hakko/)

我们不知道[马里乌斯·塔西乌克]是否在思考所有绝地如何把制作自己的光剑作为一种成年礼，但他决定是时候制造自己的烙铁了。他使用了内置热电偶的 Hakko T12 电极头。然而，他发现互联网上关于小费的信息不是不完整就是不正确。自然地，他想通了，你可以在下面的视频中看到完整的铁。

问题源于热电偶类型。他发现的一些网站将其鉴定为 K 型设备。还有人说不是，但没说是哪种。他拿了一个盛油的容器，把它加热到不同的温度，然后用商用烙铁和 T12 烙铁头测量。通过绘制已知热电偶曲线的数据，他得出结论，该设备实际上是 c 型。

定制熨斗有几个设计目标:

*   非常小
*   9 至 30V 的外部电源
*   带背光的小型液晶显示器
*   旋转编码器菜单
*   T12 提示
*   监控移动以管理待机温度
*   过热时关闭 PCB 温度读数
*   音频警告和警报
*   屏幕上可用的统计数据

正如你从下面的视频中看到的，他完成了他的目标，看起来熨斗工作得很好。我们已经注意到一个主题，即采用高端烙铁组件，并将其与较便宜的自制零件配对。我想到了最近的 JBC 建筑。我们甚至看到了其他的[哈科建筑](https://hackaday.com/2016/01/05/diy-hakko-soldering-station/)。
