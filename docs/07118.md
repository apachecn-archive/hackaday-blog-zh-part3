# 高克机器人克朗克

> 原文:[https://hackaday.com/2017/09/20/cronk-the-gonk-droid/](https://hackaday.com/2017/09/20/cronk-the-gonk-droid/)

《星球大战》宇宙中的“高克”机器人很容易被忽视，但是它们有移动发电机的重要功能。在地球上，[bithead942]真人大小的复制机器人实现了[几乎相同的目的](https://bithead942.wordpress.com/2017/08/22/gonk-gnk-power-droid/)。

cronk——功能上是一个超大的 USB 充电集线器，有很多铃铛和哨子——由一个经过修改的 Wii Nunchuck very 控制器远程控制，该控制器类似于用来控制他的 R2-D2 的控制器。在 Adafruit Audio FX Mini、Adafruit D 级 20W 放大器和两个 4 英寸扬声器的帮助下，droid 可以在释放蒸汽时发出一些声音效果(实际上，是一个倒置的二氧化碳除尘器)。Arduino Mega 充当 Cronk 的大脑，而它的身体由可铸造的聚氨酯泡沫雕刻而成，重量轻，刚性强。它还装有 FPV 摄像机、麦克风和 DVR，因此可以有效地远程操作。

而且，它会跳舞！

这些腿是一个机器人平台套件，带有伺服系统，有足够的扭矩来支撑和移动机器人的重量。面对压倒性的平衡问题，[bithead942]安装了一套训练轮，解决了稳定性问题，延长了电池寿命，并使其更符合常规——轻松应对推挤、刺激和不平的地板。

如果当你的移动设备电量不足时，这些机器人中的一个不在，打电话给附近的[神奇宝贝中心](https://hackaday.com/2016/08/20/pokemon-center-charging-station/)。
