# 带显示器的 ESP32 板:概述

> 原文:[https://hack aday . com/2018/05/23/esp32-boards-with-displays-an-overview/](https://hackaday.com/2018/05/23/esp32-boards-with-displays-an-overview/)

ESP8266 实际上已经成为 WiFi 连接微控制器的 555 芯片。传统上，你会在一个小的分线板上买一个，上面有一些引脚和一些连接器，然后连接你需要的任何其他东西。ESP8266 的老大哥 ESP32 并没有完全取代 ESP8266，但它有更多的权力和更多的选择。[Andreas]有一个新视频，展示了[七个带有集成显示器的新 ESP32 板](https://www.youtube.com/watch?v=s12XuR7BJkw)。这些主板可以简化许多需要 WiFi 和用户界面的应用。

在接受检查的电路板中，有六块是有机发光二极管显示器，但有一块是电子纸显示器。为了总结结果，[Andreas]在网上的电子表格中总结了他和其他人在这七个方面的发现。

这些板包括:

*   采用 2.9 英寸电子纸显示屏的 TTGO
*   TTGO TS 1.2 版
*   TTGO·T4
*   TTGO 亲 V2
*   TTGO·劳拉·V2
*   韦莫斯
*   Wifi 套件 32

如果你想测试新的电路板或者自己做测试，有两个软件可以在 [GitHub](https://github.com/SensorsIot/ESP32-Boards-with-displays) 上找到。

这篇评论非常实用，考察了功耗、可用引脚以及在试验板上使用的难易程度。由于[Andreas]带着经验之谈来到这里，他也关注电池开关之类的事情，以及如果你断开 USB 电源，设备是否会崩溃。剧透:他对电子纸展示板不满意。

这些显示承载设备比每对数字使用单独的 ESP32】要容易得多。如果你需要一个更大的显示器，[总有这个](https://hackaday.com/2017/07/03/esp32-display-is-worth-a-thousand-words/)。
