# Flite 测试将一把椅子放在空中

> 原文：<https://hackaday.com/2018/05/14/flite-test-puts-a-chair-in-the-air/>

Flite 测试团队因将一些疯狂的飞行装置放在一起而闻名。他们这次超越了自己，推出了一款会飞的宜家椅子。这种构建始于[乔希]向[斯特凡]发出挑战。拿一把标准的宜家梯子背椅，让它飞起来——不到六个小时。在如此紧张的时间表下，测量两次和切割一次是完全不可能的。这是一个黑客马拉松式的“扔在一起，希望它能工作”的构建。

椅子相当结实，所以它成了机身的核心。[Stefan]从以前的飞机上抓起机翼，放在椅子的座位上。钻入座椅框架的两根碳纤维杆形成了尾梁。尾翼由 Flite 测试泡沫——纸涂层泡沫芯板制成。

随着结构的完成，[Stefan]和他的团队增加了控制伺服系统，一个强大的动力马达和一些大的脂肪电池。电池悬挂在椅子底部，以保持合理的重心。

当第一次飞行的时候，每个人都在期待一次壮观的失败。椅子违反了逻辑，跳到了空中。它飞得很稳，足以让(乔希)把手指从棍子上拿开。当整个 Flite 测试团队高兴地跳起来时，看到一个疯狂的构建工作的纯粹兴奋充分展示出来。[亚历克斯]甚至扔在一个侧手翻。这就是我们在 Hackaday 这里喜欢报道的故事——看着一个完全疯狂的构建走到一起，并且表现得比任何人预期的都好。
