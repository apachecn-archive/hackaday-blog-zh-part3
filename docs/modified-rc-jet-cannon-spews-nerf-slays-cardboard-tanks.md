# 改装的遥控喷射加农炮喷出 Nerf，杀死纸板坦克

> 原文：<https://hackaday.com/2016/08/22/modified-rc-jet-cannon-spews-nerf-slays-cardboard-tanks/>

[ajw61185]制作了一个无线电控制的 A-10 喷气机[的视频概述，它被改装成在一个阳光明媚的日子里喷射出一片无害的 Nerf 球](https://www.youtube.com/watch?v=8nMFvzZKKmk)当它扫射 T-72 坦克无助的纸板剪影时。

喷气机中的点火组件来自 Nerf 的竞争对手宙斯爆能枪，这本身就是一个有趣的装置。它使用两个电动飞轮发射软泡沫球——很像一台投球机。一个飞轮比另一个跑得快一点，轨迹就可以改变。例如，轻微的上旋球会让球有更长更稳定的飞行路线。当然，泡沫球一旦发射就会迅速减速，在高速飞行时，飞机可以追上它刚刚发射的相同的射弹，但它仍然很有趣。

将点火组件塞进飞机需要一些小聪明。喷气机的前部包含飞轮组件，一个包含泡沫球的可拆卸杂志安装在它的后面。控制器上的开关轻轻一按，飞轮就旋转起来，另一个开关控制伺服系统，让球进入发射组件并发射出去。飞机上的弹药容量很低，每次装载只有 12 发子弹，而且它在大约半秒钟内发射了全部 12 发子弹。因为球是在已知区域的地面上发射的，所以很容易找回。

比更大的弹药容量更好的是第一人称视角的驾驶舱，但是在塞进一个 Nerf 爆能枪后，即使在像自由 A-10 那样大的模型中也可能没有多少空间了。在飞机规模的另一端是[模型，如此之小，以至于给它们配备最基本的可控飞行都是一项成就](http://hackaday.com/2014/11/11/tiny-rc-p-51d-mustang-tips-the-scales-at-3-grams/)。
