# 终极电池备用模式

> 原文：<https://hackaday.com/2016/03/05/ultimate-battery-backup-mod/>

除非你的主工作站是一台笔记本电脑，否则你很可能有一台 APC 或类似的备用电池。但是你有没有想过修改它，让它更有用一点？毕竟可以作为[一个有用的 DC 电源……](https://www.youtube.com/watch?v=-Ur-D37-juo)

[8 位家伙]向我们展示了他如何修改他的 APC，以包括电压读数，并直接 DC 输出插孔。事实证明，如果你不使用内置的讨厌的交流/DC 电源逆变器，你可以获得更长的电池寿命！休息后留下来观看一段关于他是如何做到的非常翔实的视频。
