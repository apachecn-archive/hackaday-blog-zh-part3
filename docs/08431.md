# 螺旋激光切割按钮制成超薄 USB MIDI 板

> 原文：<https://hackaday.com/2018/01/29/spiral-laser-cut-buttons-make-a-super-slim-usb-midi-board/>

我们在 Hackaday 看到了各种各样的人机界面设备，其中有一些非常优雅的设计。不过，在那些使用按键开关的电脑中，绝大多数使用的是为商用键盘或类似设备制造的现成组件。这样做是有意义的，有一些非常高质量的。

有时，虽然我们看到的设计从头到尾都在创造他们的按键开关。这样一个例子来自于[Brandon Rice]，[它是一个特别聪明的按钮设计，因为它使用了激光切割](https://www.whilethegluedries.com/blog/2018/1/26/i-made-a-super-thin-midi-controller)来实现超薄的结果。他用胶合板做了一个夹层，在顶层螺旋切割出关键的机械装置。他对下一层的具体细节有点粗略，但下面似乎是一个胶合板垫片，周围是一层硅胶膜，上面有从商用键盘上取下的导电橡胶。在那之下是底层上的铜带，被切割成用于触点的交织指状设计。Adafruit Trinket Pro 提供了大脑和 USB 接口，整个设备构成了一个有吸引力的专业外观的外围设备。

你可以在他发布的视频中看到实际效果，我们在休息时间下面提供了该视频。
