# 电子纸实验板

> 原文:[https://hackaday.com/2017/09/13/breadboarding-with-e-paper/](https://hackaday.com/2017/09/13/breadboarding-with-e-paper/)

[David Watts]拿起了一个便宜的 Waveshare 电子纸显示器。他制作了[一个关于用它搭配面包板](https://www.youtube.com/watch?v=o2UK_lhnObw)的视频，你可以在下面看到。

电子纸或 E-Ink 显示器有几个优点。它们是低功耗的，即使没有电源也能保持显示，并且在直射光下非常明显。缺点是它们不如其他显示技术更新快。

为了驱动这个设备，[大卫]使用了一个威莫斯 D1 来驱动显示器。他选择了 1.54 英寸的显示器，但它们有几个尺寸，最大约 4 英寸。较小的显示屏售价约为 20 美元。如果你找到一个更便宜的，确保它有 PCB，而不仅仅是原始的显示面板。正如你所料，更大的显示器价格稍高。

[David]并没有真正提供教程，他只是展示了如何将其连接到试验板和他的示例项目。然而，至少有一个 [Arduino 库](https://github.com/sabas1080/LibraryEPD)可用。你还可以在 [Waveshare wiki](http://www.waveshare.com/wiki/1.54inch_e-Paper_Module) 上找到很多文档、库和例子。

如果这些变得更便宜，也许我们会看到更多的电子商务名片。或者它们会变得像[便利贴](https://hackaday.com/2017/05/22/hackaday-prize-entry-wifi-epaper/)一样无处不在。
