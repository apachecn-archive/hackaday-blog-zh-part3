# 花 6500 美元买一辆能用的特斯拉 Model S

> 原文:[https://hack aday . com/2017/09/20/salvaging-your-way-to-a-work-Tesla-model-s-for-6500/](https://hackaday.com/2017/09/20/salvaging-your-way-to-a-working-tesla-model-s-for-6500/)

如果你拥有适度的技术能力和几十个僧侣的耐心，通过一些熟练的讨价还价，你可以通过打捞和修理为自己找到一些很好的便宜货。在采购电子测试设备等物品时，这已经是一种众所周知的意识形态，例如，可以从易贝购买一个非工作单元，并以几个无源元件的价格固定下来。

来自汽车大师的[Rich]成功地以 6500 美元的价格挽救了一辆适合道路行驶的特斯拉 Model S，将这一点提升到了一个全新的水平！

采购和改造汽车总是一个令人生畏的项目，在这种情况下，更具挑战性，因为主题中的车辆是相当新的，最先进的电动汽车。旅程从购买一辆黑色特斯拉 Model S 开始，Rich 亲切地称之为 Delorean。这辆车有严重的水渍，导致它的大多数电子设备和机械紧固件不可靠，所以[Rich]的计划是![](../Images/da4497e21dfa703b973fce5b34461919.png)拆除这辆车的所有这些部件，并出售他能收回他最初购买的成本。在卖掉了电池、马达和其他一些华而不实的东西的工作模块后，他最初的资金投入变成了纯粹的时间投入。

拥有了一个基本上免费但空空如也的特斯拉外壳后，[里奇]将注意力转向寻找内部合适的替代品。[Rich]提到 Tesla 拒绝出售此类项目的备件，因此他唯一的选择是再购买几辆受损车辆。这些沉船中最突出的一艘被戏称为“苗条黑幕”。这一个

![](../Images/9b40fb0990a567e33567bb38987f948e.png)

The Donor

有一个不可修复的外壳，但保留了大部分电子设备，并将作为捐赠工具。在煞费苦心地移植了所有需要的电子产品并再次出售了他不需要的产品后，他的净投资还不到一辆新车的 10%!

所有的努力都值得吗？我们当然认为是！这款车被认为是有道路价值的，甚至具有功能性的超级充电能力，据[Rich]称，如果检测到这样的弗兰肯斯坦制造，特斯拉就会禁用该功能。

在这一点上，问问[Rich]他是否会再做一次可能会有启发性，但他已经在做了，这一次是挽救更快的自我驾驶 P86。我们建议你继续关注。
