# 用 Spacehack 拯救宇宙飞船！

> 原文：<https://hackaday.com/2016/04/17/save-a-spaceship-with-spacehack/>

York Hackspace 需要一件展示品来装点他们在 Maker Faires 和类似活动中的展台。他们的解决方案是 [Spacehack，一款基于多人控制台的星际飞船紧急模拟游戏](https://york.hackspace.org.uk/spacehack/)。每个 Spacehack 玩家都有一个控制台，上面有一系列显示器、开关、转盘和控制杆。玩家必须根据游戏提供给他们的一系列有时令人困惑的命令来操作他们的控制器。每一个错误的举动都让这艘容易发生灾难的飞船更接近毁灭，目的是尽可能长时间地保持它的太空价值。结果是对黑客空间的吸引和上瘾。

在设计出色的控制台、银色管道和金字塔形集线器盒背后，游戏依赖于充当服务器的树莓皮和每个玩家的比格犬骨黑。所有的资源都可以在 York Hackspace 的 [GitHub 知识库](https://github.com/yorkhackspace)上找到。hackspace 在 Spacehack 网站上有[的精选视频，休息时间下方的视频展示了游戏以及游戏构造的蒙太奇。](https://york.hackspace.org.uk/spacehack/vid.php)

正如你所料，我们已经在 Hackaday 上报道了很多游戏。但是浏览档案有些事情变得显而易见，虽然我们难以置信的创意社区已经尝试了几乎每一个可以想象的视频游戏，但很少有逃脱那个盒子的。或许是一款扔球游戏，甚至是[一款单像素游戏](http://hackaday.com/2013/08/17/one-pixel-video-game-rises-from-rgb-button-hardware/)。很少有人制作出像 Spacehack 这样的游戏，但当他们这样做时，它绝对值得再看一眼。
