# 自动跟踪哨兵枪给鹿一个超级浸泡

> 原文:[https://hack aday . com/2018/01/26/auto-tracking-sentry-gun-gives-deer-a-super-smoking/](https://hackaday.com/2018/01/26/auto-tracking-sentry-gun-gives-deer-a-super-soaking/)

当人类和野生动物混在一起时，事情很少会进展顺利。这些问题在郊区变得更加严重，熊以喂鸟器和垃圾桶为食，浣熊住在阁楼上，郊狼带着家猫逃走。在郊区，讨厌的野生动物可能是一个棘手的问题，因为处理它的选择非常有限。

[dlf.myyta]在保护他的玫瑰的战斗中没有被劝阻，他建造了[这种动作激活的哨兵枪](https://www.instructables.com/id/Auto-tracking-Water-Blaster/)来对劫掠的鹿进行一些水厌恶疗法。在下面对抗两足共谋者的行动中，哨兵枪几乎拥有你在引擎盖下所期待的东西——树莓派，黑色相机，瞄准伺服系统和控制水的电磁阀。OpenCV 负责定位入侵者并将喷嘴旋转到中心质量；因为鹿在某种程度上受到栅栏的限制，所以没有必要控制喷嘴的高度。为了便于携带和防水，所有东西都很好地装在一个塑料弹药箱里。任何静止不动超过三秒钟的目标都会被喷一次；我们假设这是有效的，但唉，没有鼻烟电影提供。

我们不确定[dlf.myyta]的代码能否辨别敌友，在这个爱打官司的世界里，给邻居的孩子浇水可能会是一场灾难。或许 2.0 版本可以包含[图像识别进行目标验证](https://hackaday.com/2017/06/14/hackaday-prize-entry-automated-wildlife-recognition/)。
