# 带一对小鞋钉的大号手表

> 原文:[https://hack aday . com/2017/01/20/plus-size-watch with-pair-of-tiny-nixies/](https://hackaday.com/2017/01/20/plus-size-watch-with-a-pair-of-tiny-nixies/)

当你把一对谢妮电子管塞进手表时，制成的手表看起来有点像 Flav 的项链。当然，这是不是一件好事取决于你的品味，以及你是否愿意在手腕上佩戴 200 伏的电压。

不过，作为一名建筑师，[prototype_mechanic]手表还是值得一看的。遗憾的是，由于电脑问题破坏了原始图纸和示意图，细节很少，但我们可以从 Instructables 帖子中收集到一些信息。该案件是加工出固体铝和体育石英玻璃晶体。这对 IN-16 灯管位于边框后面，RGB LEDs 照亮了井。板上有一个 400mAh 的 LiPo 电池，还有一个加速度计，只需轻轻一抖手腕就可以打开显示器。

对于日常使用来说，这可能有点不切实际，但这是一款具有蒸汽朋克风格的精美时计。事实上，[prototype_mechanic]展示了其他几件带有四根管子的皮革和谢妮作品，它们确实捕捉到了蒸汽朋克风格的感觉。如果你对黑客更感兴趣，可以看看这款带有 3D 打印表壳的谢妮手表。
