# 用数控机床进行增量板料成形

> 原文：<https://hackaday.com/2018/03/14/incremental-sheet-forming-with-a-cnc-machine/>

如果你想将一片金属板材成形，你可能会考虑使用模具。这当然是一个很好的方法，但它预先假定你可以创建或购买模具，这可能是一个小项目的阻碍。[Dardy-7]已经研究出如何使用一种较少使用的技术——[增量板材成形](http://www.instructables.com/id/Incremental-Sheet-Forming-an-Introduction/)——用数控机床获得类似的结果。这个想法是用一个圆钝的工具在金属板上描绘出形状。

他用一个便宜的 dapping 工具得到了很好的结果，尽管他见过其他人使用加热的钛滚珠轴承。此外，他还研究出了如何调整现有的刀具路径，比如你可以从网上下载的路径，来使用这项技术。你可以在下面看到工作流程的视频。

该技术需要一个夹具来固定工件，使工具能够向下推动金属。在示例中，夹具只是快速胶合板组件，工件是 0.6 毫米的铝。一些夹具在下面有支撑，以帮助创建有突出部分的零件(即，正曲率)。

我们不禁认为，3D 打印机可以很容易地创建一个背板，它可以达到与当前夹具中的支撑相同的目的。[Dardy-7]指出这个过程很慢，而且不如其他一些方法准确。另一方面，如果你有一台数控机床，这应该是一个很容易添加到你的曲目。

如果这让你想跑出去买个数控机床，你可能想[先看这个](https://hackaday.com/2016/08/17/tips-for-buying-your-first-milling-machine/)。如果你想让[进入刀具轨迹](https://hackaday.com/2016/11/11/an-introduction-to-cnc-machine-control/)，我们也可以帮你。
