# 使用宜家高分椅，尽早开始游戏

> 原文：<https://hackaday.com/2015/11/29/start-gaming-early-with-ikea-high-score-chair/>

如果你想让你的孩子在某件事上非常出色，你必须尽早让他们开始。[菲尔·塔克]一定非常希望他的孩子成为一名电子游戏玩家。[Phil]从一把价值 20 美元的宜家高脚椅开始。他喜欢这些椅子，因为在这个价位上，冲进去不会有太大的风险。更重要的是，你可以购买额外的托盘，这样你就可以把它作为一个模块化的项目，不同的托盘服务于不同的目的。

这把椅子有两个操纵杆和两个按钮，看起来可疑地像一个视频游戏控制器。当前版本(见下面的视频)使用 Arduino Uno 通过 MIDI 接口触发 Akai MPC1000 合成器。

当然，你可以控制其他的 MIDI 设备，甚至编程 Arduino 做其他的事情(比如[作为游戏控制器](http://hackaday.com/2011/06/26/digitized-retro-lcd-games-played-using-an-arduino-as-a-controller/))。我们见过很多宜家的黑客，包括[激光切割机](http://hackaday.com/2014/05/25/ikea-desk-laser-cutter-combo/)、[相机滑盖](http://hackaday.com/2014/02/17/macgyver-made-ikea-camera-slider/)，甚至[打印机](http://hackaday.com/2012/12/07/printing-press-made-from-ikea-furniture/)(不是 3D 打印机，尽管[我们也报道过](http://hackaday.com/2013/12/18/custom-rostock-3d-printer-makes-use-of-ikea-components/))。
