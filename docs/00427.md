# 遥控滑翔机通过扭转翅膀飞行

> 原文:[https://hack aday . com/2015/10/20/RC-glider-flys-by-twisting-its-wings/](https://hackaday.com/2015/10/20/rc-glider-flies-by-twisting-its-wings/)

遥控滑翔机通常像全尺寸滑翔机一样飞行。尾巴和翅膀的方向舵控制着飞行的方向——但是如果翅膀本身能像鸟一样扭曲并改变它们的外形会怎么样呢？嗯，遥控滑翔机制造商[Jaro Müller]就是这么做的——而且它[非常酷](http://www.elektromodely.sk/?p=6205)(不过你需要一个[翻译](https://translate.google.fr/translate?hl=fr&sl=sk&tl=en&u=http%3A%2F%2Fwww.elektromodely.sk%2F%3Fp%3D6205)来阅读它)。

这款名为迷你椭圆的遥控滑翔机被设计成能够在缓慢的热气流中飞行，并且比以前的型号机动得更好。整个机翼轮廓可以通过机翼弯曲来控制——机翼本身非常灵活。不幸的是，我们没有任何关于它实际上是如何做到这一点的信息，但它可能是伺服电机牵引线，或者可能是镍钛诺记忆线…但我们只是猜测。不管怎样，看看下面的视频，让我们知道你的想法！
