# 史上最小的机械键盘

> 原文:[https://hack aday . com/2017/05/20/有史以来最小的机械键盘/](https://hackaday.com/2017/05/20/the-tiniest-mechanical-keyboard-ever/)

拥有一个机械键盘让你成为一个更好的人。它让你凌驾于所有人之上。当然，拥有一个机械键盘也有一些缺点。随身携带一个机械键盘来告诉别人你比他们强通常是不切实际的，但是[~~cahbtexhuk~~Joric]提出了一个解决方案。这是[一个微型蓝牙机械键盘](http://imgur.com/a/OkwEb)也是一个钥匙链。

机械键盘钥匙链并不新鲜——它们实际上只是一块注射成型的塑料，用于安装一个 Cherry MX 按键开关。不过，通常这些只是小饰品——把它们想象成打字的人的坐立不安的旋转器。这里真正的诀窍是将电子设备和电池放入这块塑料中，将它变成一个工作键盘。

这个问题的解决方案来自于全球速卖通上的 TI CC2540 模块[~~cahbtexhuk~~Joric】[。这是一款完整的低功耗微控制器，在一个非常小的封装中集成了蓝牙功能。添加一个小硬币电池，你就拥有了你能以不到五美元建造的最小的物联网。](https://www.aliexpress.com/item/MK00283-Bluetooth-4-BLE-from-the-module-serial-communication-direct-drive-mode-RF-BM-S02-CC2540/32658399946.html)

[ ~~cahbtexhuk~~ Joric]用相关的 TI 工具、Arduino 和简洁的编程剪辑对这个微型蓝牙模块进行了编程。之后，只需将钥匙开关焊接到模块上，并连接电池。结果是有史以来最可爱的机械键盘。现在，[cahbtexhuk]已经将这个键盘编程为映射到他的 Windows 键。这是一个很好的演示(下面的视频)，但是这个迷你键盘可以被编程为当它被按下时输入任何东西。
