# 给相机支架一点(磁性)吸引力

> 原文：<https://hackaday.com/2017/06/09/giving-a-camera-mount-a-little-magnetic-attractiveness/>

可以肯定地说，大多数黑客和制作者并不真的想在制作他们作品的视频文档的细节上做文章。他们宁愿把时间和精力花在手头的实际项目上……你知道——有趣的东西。

[Daniel Reetz]一直希望他的相机支架有更多的安装选项，以使安装更加简单快捷。他现有的相机支架的一端是一个夹子。到目前为止，这对[丹尼尔]很有效，但是他想要更多的选择。意识到他的商店周围有很多黑色金属表面，他有了一个想法——为他的相机支架制作一个磁性底座。

在视频中，[Daniel]带领我们从实际的底座开始，完成了这个磁性相机支架附件的制作过程。它被称为可切换磁性底座(或他称之为 mag-base)，看起来像一个方便的小设备。这无疑是最昂贵的部分，但看起来它应该持续很长时间。基本上，它是一个金属盒子，里面有磁铁，外面有一个旋转开关。当开关在一个位置时，盒子的底部是磁性的。将开关旋转到另一个位置，底部不再有磁性。这些可切换的磁性底座顶部有一个螺柱，用于连接其他东西，看起来像[丹尼尔]已经做到了。不过，从那以后，他解释并展示了构建的其余部分。

一些软钢棒被切割和修改，以滑入管道。该杆由一个固定螺钉固定，便于调整安装高度。然后，他将杆焊接到垫圈上，垫圈又焊接到管子上。焊接完成后，他把整件东西拿到去毛刺砂轮上去清理。之后，用一些喷漆和一个定制的 3D 打印帽进行最后的润色。

在整个视频中有一些有用的提示，其中之一是他如何在焊接前用酸剥离垫圈上的锌。这是因为你不想焊接锌，因为它会产生神经毒素。

现在，在新的磁性底座的帮助下,[Daniel]可以在商店的任何地方快速安装相机支架。

我们已经讨论过很多相机支架的问题。这是另一个涉及[磁铁，但也有自动摇摄功能](http://hackaday.com/2013/07/18/magnetic-panning-time-lapse-camera-mount-couldnt-be-easier/)。你需要一个[滑动摄像头支架](http://hackaday.com/2012/04/03/sliding-camera-mount-is-good-enough-for-amateur-photography/)吗？一个[电动滑动相机支架](http://hackaday.com/2011/12/20/motorized-camera-mount-unexpectedly-popular-for-cnc-aimed-hardware/)怎么样——尽情享受吧。
