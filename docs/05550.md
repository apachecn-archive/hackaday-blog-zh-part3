# 立方体赛弗肯徽章

> 原文:[https://hackaday.com/2017/04/09/the-cubic-cyphercon-badge/](https://hackaday.com/2017/04/09/the-cubic-cyphercon-badge/)

上周在密尔沃基举行了威斯康星州首屈一指的黑客大会 Cyphercon。没有电子会议徽章或挂在挂绳上的 45，你就不能进行黑客大会，而 Cyphercon 2017 徽章不会让人失望。这是一个电子立方体，由 tymkrs 的人们精心设计。这也是一个有安全漏洞和无线通信的难题箱。这是一个网状徽章网络，是我们见过的最好的会议徽章之一。

Cyphercon 2.0 徽章最明显的特点就是额外的维度。从一开始，这个徽章的设计就是一个三维立方体，由制作精美的印刷电路板构成，边缘焊接在一起。将多氯联苯带入三维空间的技术其实并不新鲜——在之前我们已经见过 3D 多氯联苯——但从未有过如此大的体积。有超过 400 个徽章是为塞弗肯制造的，每一个接头都是手工焊接的。这是你的组装厂不会做的事情，我讨厌去想那些可怜的焊料猴子，他们将被迫为一个更大的骗局组装 3D 徽章。

3D 并不是 Cyphercon 徽章的唯一亮点。立方体的每一侧都有切口，露出 led、微处理器、总线和一个 USB 端口。这个 USB 端口允许佩戴者给电池充电，是的，但是如果你在笔记本电脑上安装一个终端模拟器并插入徽章，你就会陷入一个神秘、阴谋和窒息的世界。这个徽章是一个文本冒险游戏，游戏的目标是从导弹发射井周围搜罗的零件重新组装一个基于继电器的计算机。一旦中继计算机完成，徽章就变成了老式分时操作系统的模拟器。在这个操作系统中，您可以编写代码并将其部署到其他徽章。这真是令人印象深刻的东西。

在 cubic Cyphercon 徽章、 [Hunter S. Rodriguez 徽章](http://hackaday.com/2017/03/15/hardware-tribes-growing-up-around-artisanal-electronics/)今年夏天前往拉斯维加斯，以及[徽章*是*任天堂模拟器](https://hackaday.io/project/13262-layerone-2017)之间，今年看起来像是电子会议徽章的伟大一年。这里的艺术性和技巧是惊人的，我们迫不及待地想看看社区还会拿出什么。
