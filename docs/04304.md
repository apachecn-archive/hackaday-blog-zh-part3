# 像埃及人一样编码

> 原文:[https://hackaday.com/2016/12/03/code-like-an-egyptian/](https://hackaday.com/2016/12/03/code-like-an-egyptian/)

马塞洛·马克西米亚诺的儿子有一个学校项目。他和一个学生团队制作了“[金字塔的秘密](https://github.com/marcelomaximiano/BoardGame)”——一款使用 Arduino Nano 的电子棋盘游戏。[Marcelo]在电子方面提供了帮助，但结果令人印象深刻，是包装 Arduino 项目的一个很好的例子。你可以在下面看到这个游戏的视频。

除了处理器之外，该游戏还使用 WT5001M02 MP3 播放器(以及音频放大器)来产生音乐和语音。还有一个旋转编码器，一个液晶显示器，一个 EEPROM(保存测验问题和答案)，和一个 LED 驱动器。还有一堆发光二极管、开关和一个电线迷宫，要求玩家在不碰到电线的情况下导航(想想 2D 操作)。

除了代码和硬件图，GitHub 上还有一个 PDF 文件描述了更多关于游戏的内容。不过，它是葡萄牙语的，所以我们大多数人可能需要一点翻译帮助。然而，一个巴西网站确实有一个关于这款游戏的英文帖子，这可能是一个好的开始。

你可能不想复制这个游戏，但它是一个很好的例子，说明了一个 Arduino 可以用一些简单的外部设备和一些对包装的注意做多少事情。

可悲的是，我们的大多数项目看起来更像是这个游戏(无意冒犯那个黑客)。像这样的项目比闪烁的 LED 或电容表更有可能激发年轻人的兴趣。如果你更有心情玩街机，你也可以去看看 [Arduinocade](http://hackaday.com/2015/09/17/retro-games-on-arduinocade-just-shouldnt-be-possible/) 。
