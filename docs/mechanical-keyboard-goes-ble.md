# 机械键盘走向 BLE

> 原文：<https://hackaday.com/2015/09/14/mechanical-keyboard-goes-ble/>

像许多程序员一样，[Daniel Nugent]喜欢他的旧机械键盘(一种 WASD 代码键盘)。他不喜欢的是电线。当然，你可以得到一个现代的无线键盘，但它不会和你花了很多时间使用的键盘一样。配备了蓝牙低能量(BLE)模块、可充电电池和一些编码，[他保留了键盘，但去掉了电线](https://hackaday.io/project/7646-ble-wasdmechanical-wireless-keyboard)。

尽管他对 WASD 有一些特定的处理，但代码很可能处理任何 PS/2 键盘。PS/2 接口是一个简单的同步串行端口，只有一个时钟和一条数据线。用微控制器处理它并不困难。

您可能希望[重温一下 PS/2 如何与](http://hackaday.com/2011/09/23/interfacing-with-a-ps2-keyboard/)通信。或者，也许你更愿意仅仅由[来构建整个键盘](http://hackaday.com/2015/03/29/3d-printed-mechanical-keyboard/)。我们甚至看到一些人真的走了 T4 传统的机械键盘路线。不过，对我们来说，在 IBM 型计算机上进行这样的攻击就足够了。
