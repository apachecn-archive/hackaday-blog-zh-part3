# 自动化星球大战主题的 Nerf 目标

> 原文:[https://hack aday . com/2015/11/20/automated-star-wars-themed-nerf-targets/](https://hackaday.com/2015/11/20/automated-star-wars-themed-nerf-targets/)

当面临只用现有的零件为儿童活动制作游戏的挑战时，[内森·格雷]必须发挥创造力。他最后做的东西非常棒。这是一个星球大战主题的射击馆。

[![icons-390](../Images/89f7ef2b4769a8dfea0a8ae211f57432.png)](https://hackaday.com/wp-content/uploads/2015/11/icons-390.jpg) 使用一个 Teensy 2.0，他控制着九个连接到双面目标的 RC 伺服电机，这些目标每一轮都随机产生——帝国是坏的，反叛者是好的。它们也有红色和绿色的颜色编码，以防图像太难看清。

为了记录得分，有压电元件记录 Nerf 飞镖的冲击。一个可爱的小控制台，有一个大红色的开始按钮和分数显示，可以设置在靶场前，让孩子们知道他们做得怎么样。
