# 用滚动锯的方式制作 3D 物体

> 原文：<https://hackaday.com/2018/04/24/making-3d-objects-the-scroll-saw-way/>

这些天来，大多数人都认为，如果你想制造一个复杂的三维物体，那么 3D 打印机是唯一的出路。许多人甚至忘记了这些东西曾经可以手工雕刻。另一方面，[JEPLANS]是一位只用一把锯子就能制作这些物品的大师，他的最新作品是从一整块枫木上切割下来的微型骆驼。

他的过程与 3D 打印有很多相似之处。他从电脑绘制的设计开始，这次是骆驼的正面和侧面。剪下之后，他剥掉不想要的部分，骆驼就像变魔术一样从积木中出现了。但他不喜欢为了进一步改进而需要的大量手工工作，所以他通过添加俯视图来修改设计，就像 3D 打印一样迭代。但是在切断之后，他发现他损坏了一条腿。所以他裁剪了一个新的，但只是在做了一个新的设计改变之后，这次调整了骆驼的头部位置。他对这个结果很满意。在下面的视频中，看看他煞费苦心又有些熟悉的过程。

他的并不是我们在 Hackaday 上看到的唯一一件精湛的卷轴锯作品。看看这个漂亮的亚克力骨架钟，它的齿轮和连杆是用线锯切割的。
