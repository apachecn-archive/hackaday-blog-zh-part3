# 黑客日奖:大众外骨骼

> 原文：<https://hackaday.com/2017/08/09/the-hackaday-prize-exoskeletons-for-the-masses/>

虽然全世界的医疗设施在不断改善，但很多人仍然无法获得昂贵的治疗。尤其是在假肢方面，很多人买不起如此个性化的东西，尽管对辅助设备的需求非常高。考虑到这一点，[Guillermo Herrera-Arcos]开始研究[ALICE，这是一种机器人外骨骼](https://hackaday.io/project/25105-alice-robotic-exoskeleton)，成本低，易于制造，作为额外的奖励，100%开源。

爱丽丝的创造者设想外骨骼将应用于康复，人体增强，甚至游戏。此外，由于它是开源的，它也可以作为 STEM 学生学习的平台。目前，该团队正在测试外骨骼腿部的电子设备，但他们已经通过控制系统取得了很大进展，并获得了一个可行的原型。展望未来，由于开源硬件的特性，创作者以及任何在这个平台上开发东西的人都将一直在改进它，并在它的基础上进行构建。
