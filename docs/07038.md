# 如果你要做一个模型引擎，你最好做一个梅林

> 原文:[https://hackaday . com/2017/09/11/if-you-going-to-make-a-model-engine-you-may-well-make-it-a-merlin/](https://hackaday.com/2017/09/11/if-youre-going-to-make-a-model-engine-you-might-as-well-make-it-a-merlin/)

以前在不止一个 Hackaday 帖子中提到过，有许多像我们这样的社区是孤立存在的，并且其中包含了惊人水平的硬件和工程能力。例如，在汽车改装领域，我们根本看不到更多由工程驱动、更少由配件驱动的那一端所做的所有工作，因为总的来说，我们不像他们一样在同一个圈子里活动。

模型制作世界就是这样一个社区，在这个社区中，展示令人难以置信的技能水平的项目是常态。我们年轻时可能都曾用胶水粘过喷火或野马的塑料套件，但在表盘的另一端，当谈到模型时，你会发现工艺远远超过许多高端机器商店中的工艺。

一个项目很好地证明了这一点，那就是[mayhugh1]的老式劳斯莱斯梅林 V12 活塞式航空发动机的四分之一比例模型。这是你会在二战时期许多标志性的盟军飞机上发现的动力装置，包括现实生活中的喷火式战斗机和除了最早的野马之外的所有飞机。让四分之一比例的梅林变得更特别的是，它运行着。加油就行了。

这个构建花了几年时间，用了论坛的很多页面，包括了很多详细的描述、照片和视频。它始于一套发动机缸体的商业铸件，但它们的修整和所有其他发动机零件的制造都在车间里完成。在最后一页左右，我们看到了我们放在休息下面的视频，在测试框架中完成的引擎正在长凳上运行，一个有点可怕的无防备的螺旋桨附在它的前面，等待斩首一个粗心的摄影师。坐下来，喝一杯你最喜欢的饮料，从头到尾读一遍。我们认为您不会失望的。
