# 自主跨大西洋航海

> 原文:[https://hack aday . com/2017/06/23/autonomous-trans Atlantic-seaming/](https://hackaday.com/2017/06/23/autonomous-transatlantic-seafaring/)

[Andy Osusky]提交的 Hackaday 奖项目是[建造一艘穿越大西洋的自主帆船](https://hackaday.io/project/15566-opentransat)。[Andy]的船将参加 Microtransat 挑战赛，这是一项跨大西洋的自主船比赛。为了坚持挑战的规则，【安迪】的船最大只能有 2.5 米的长度，而且要在 25 公里内横跨大洋命中目标点。

该船的主要框架是由冲浪板顶部的铝制成，用沉重的龙骨来保持平衡。由于轻质结构，船不会沉下去，如果翻了，沉重的龙骨会使它直立起来。船帆由尼龙织带和厚碳纤维管加固的防撕裂尼龙制成，以抵御强风。

电子设备分为三个部分。一个安全密封的鹈鹕箱包含 LiFePo4 电池、太阳能充电控制器和基于 Arduino 的导航控制器。通信硬件保存在聚碳酸酯外壳中，以便更好地接收信号。一个箱子里装有铱星卫星追踪器、指南针和 GPS，另一个箱子里装有两个 Globalstar 追踪器。铱星模块允许船只通过铱星短脉冲数据服务传输数据。通过这种方式，可以传输 GPS 位置、风速和指南针方向等数据。

[安迪]的船于九月从纽芬兰启航，驶向爱尔兰。然而，事情似乎很快就出了岔子。暴风雨和撞车导致了错误，太阳能充电器似乎没有给电池充电。测试持续了大约 24 天，在此期间，船行驶了近 1000 公里。

[Andy]正在重新设计船只，更换为刚性船帆，并将硬件封装在船内。同时，该项目是开源的，因此在 GitHub 上描述了硬件和软件。一定要去看看 OpenTransat 网站，在那里你可以看到第一次航行的数据。另外，看看[这篇关于自主皮艇的文章](https://hackaday.com/2016/09/10/hackaday-prize-entry-autonomous-kayaks/)，还有[这篇关于一群自主船的文章](https://hackaday.com/2016/06/17/building-a-swarm-of-autonomous-ocean-boats/)。
