# 构建您自己的动画转向灯

> 原文:[https://hack aday . com/2017/03/13/build-your-own-animated-turn-signals/](https://hackaday.com/2017/03/13/build-your-own-animated-turn-signals/)

汽车照明曾经受到严格控制，尤其是在美国——有人记得封闭式头灯吗？这些天，几乎任何事情都会发生。你甚至可以有一个动画转向灯，因为现在一个简单的闪光灯还不够花哨。你可以在你的新款奥迪车上安装一个扫描 LED 转向灯。[Shravan]想在他们的马自达上安装这个，并着手为他们的汽车制作动画转向灯和日间行车灯。

无论如何，这都不是一个复杂的构建；现成的 WS2812B 条带提供了 blinkums，Arduino Nano 提供了 smarts。使用一个修改过的库来驱动 led 使得[Shravan]能够以最少的忙乱让事情运行起来。我们很想看到这款车更多的细节 Nano 是如何从汽车上获得方向信号的，以及它是如何连接和固定的。当你在汽车上安装定制硬件时，细节才是真正的魔鬼。创造出看起来整洁又功能良好的东西极其困难。

想想我们已经走了多远，真是令人惊讶。当高亮度 led 在 20 世纪 90 年代首次进入市场时，您可能已经开始为自己的织机布线以连接 20+led，构建自己的驱动电路，并可能蚀刻定制的 PCB——而与此同时，您在汇编中编程 PIC，因为它悬在并行端口编程器上。不过话说回来，我们的穴居祖先连火柴都没有。时间在流逝。利用今天的技术，尽你所能建造最好的东西。

我们喜欢看到汽车改装，尤其是那些执行良好的。查看[戴夫]对日产 Juke 的内部照明改装——这是一款作者颇有见地的汽车。休息后的视频。
