# 将实时 2D 视频转换为 3D

> 原文:[https://hack aday . com/2015/11/07/converting-live-2d-video-to-3d/](https://hackaday.com/2015/11/07/converting-live-2d-video-to-3d/)

对于那些在 2013 年认为 3D 电视将成为下一个大事件的傻瓜们来说，这里有一些好消息。麻省理工学院的研究人员开发了一个将 2D 视频转换成 3D 的系统。由此产生的 2D 视频可以在 Oculus Rift、谷歌 Cardboard 甚至客厅里的 3D 电视上播放。

目前，该系统仅适用于 2D 的足球转播，但这仅仅是研究人员解决这一问题的产物。这个问题最初是通过查看游戏 FIFA 13 的屏幕截图来解决的。使用一种叫做 PIX 的分析工具，研究人员既存储了显示数据，又提取了球场、球员、球和体育场的相应 3D 地图。为了生成 2D 足球广播的 3D 视频，系统然后查看 2D 广播的每一帧，并搜索与球场上的动作相对应的 3D 数据集。这种深度信息然后被添加到视频馈送中，仅使用传统的 2D 相机产生 3D 广播。
