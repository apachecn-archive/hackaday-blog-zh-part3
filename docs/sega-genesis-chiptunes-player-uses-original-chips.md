# 世嘉 Genesis Chiptunes 播放器使用原装芯片

> 原文：<https://hackaday.com/2017/02/17/sega-genesis-chiptunes-player-uses-original-chips/>

如果你是 20 世纪 80 年代末或 90 年代初的孩子，你可能会在超级任天堂或世嘉创世纪/超级驱动阵营。其他 16 位游戏机也存在，但这些才是最重要的！任天堂的 16 位 6502 衍生产品或世嘉的 68000 的额外功能提供了一种游戏体验，虽然它可能不太可能是你在当时的电子游戏中发现的，但至少足够接近，你可以*假装*它是。

那个时代的控制台独特的声音在 chiptunes 社区中获得了大量的追随者，活跃的场景组成了新鲜的作品，并创建了与它们一起工作的项目。其中一个项目是[jarek319]的[Sega Genesis 原生硬件 chiptune 合成器](https://hackaday.io/project/13361-sega-genesis-native-hardware-chiptune-synthesizer)，其中以 VGM 文件形式存储在 MicroSD 卡上的音乐由 ATSAMD21G18 处理器解析，然后发送到 YM2612 和 SN76489，就像你在原始控制台中发现的那样。音频输出与原始电路匹配，以尽可能接近地复制经典声音，甚至有一些关于为该硬件添加 MIDI 功能的讨论。

软件是提供了，虽然他承认在一些功能上还有一点路要走。MIDI 支持还没有出现，尽管如果有足够的兴趣，他已经准备好了。你真的应该分享这一行动，有一个视频，我们把它放在休息。
