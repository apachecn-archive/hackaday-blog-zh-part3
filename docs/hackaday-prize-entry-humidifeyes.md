# Hackaday 奖参赛作品:湿漉漉的眼睛

> 原文：<https://hackaday.com/2016/10/28/hackaday-prize-entry-humidifeyes/>

Hackaday 奖辅助技术挑战赛的大多数参赛作品都解决了一个显而易见的问题。3D 打印义肢、用于嘴部的电脑鼠标以及文本到语音系统都有明显的用途。对于他的 Hackaday 奖参赛作品，[spandana]决定了一些不太明显的东西。他正带着保湿眼霜去对付干眼症的祸害。

干眼症的问题乍一看似乎有点深奥，但[spandana]是专门为史蒂文约翰逊综合征患者设计的这种设备。有人提到 LASIK 手术的病人会有几个月的干眼症。显然，这里有一个机会。

这个装置的结构非常简单。目前的原型使用现成的安全眼镜，边缘周围有一点泡沫作为底盘。水分从储存器输送到超声波振动器。这是一种非常有效的雾化液体的方法，并且体积小得足以安装在一副眼镜的框架上。

虽然这是一个有点奇怪的项目，并且允许太多的双关语，但这里有一些有用的东西。干眼症是一个问题，除了一瓶 Visine，没有太多的办法。
