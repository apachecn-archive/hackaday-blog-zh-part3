# 一张照片中的四个季节

> 原文:[https://hackaday.com/2015/12/25/four-seasons-in-one-photo/](https://hackaday.com/2015/12/25/four-seasons-in-one-photo/)

这是一种多么有趣的显示一年的方式:挪威黑客[Erikso]创造了一种浓缩的延时摄影，在一张照片中显示一年[。2010 年期间，在寒冷的北方，他每天都从起居室的窗户拍摄一段时间的风景，使用的是一台锁定在适当位置的相机，每 30 分钟拍摄一次图像。然后，在一些黑客朋友的帮助下，他想出了一个脚本，将这些图像切片并组合起来，这样每天都由一个垂直切片表示。其结果是一个华丽的图像，给人一种奇妙的感觉的季节，以及如何影响树木。你可以看到树叶长了又落，地上的雪来了，去了又来。](http://eirikso.com/2011/01/04/one-year-in-one-image/)

他使用的脚本由 Aslak Hellesø编写，是一个简单的 shell 脚本，它计算所需的切片，并使用来自 T2 Imagemagick 的 CLI 工具对图像进行切片和切割，然后将它们一起粘贴到最终图像中。他还用同样的图片制作了一段视频。就我个人而言，我更喜欢单幅图像。
