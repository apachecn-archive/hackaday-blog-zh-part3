# 简易甲虫机器人使用吸烟烙铁

> 原文：<https://hackaday.com/2016/07/20/simple-beetle-robot-uses-smoking-soldering-iron/>

就机器人项目而言，[创意知识管理]不会给许多黑客读者留下深刻印象。尽管如此，作为一个艺术项目或与孩子们有关的事情，这可能会很有趣。但它引起我们兴趣的原因并不是实际的机器人，而是其构造中使用的临时烙铁。

机器人本身并不是真正自主的。它只是一个电池、一个马达和一个开关。马达的震动使机器人在弯曲的铜线腿上快速移动。一些热胶把它们粘在一起，但是电线是焊接的。

如果你看下面的视频，你会看到焊接是用一种不寻常的方法完成的。一次性打火机产生的火焰击中了缠绕着线圈的铜线。线圈起到热交换器的作用，而导线则成为烙铁头。

我们怀疑你会想用这种烙铁做任何严肃的事情。即使是便宜的熨斗也会更方便。另一方面，你可能会发现这个被黑掉的烙铁作为现场权宜之计很方便，特别是当你没有交流电源的时候。

我们怀疑你会改进这种黑客技术，使之成为超声波技术，除非你让机器人骑在你的打火机上。也许你用[会更好，这是一个使用商业技巧的自制熨斗。](http://hackaday.com/2016/05/07/a-cheaper-soldering-solution/)
