# Hackaday 奖半决赛选手:在急诊室帮忙

> 原文:[https://hack aday . com/2015/09/18/hack aday-prize-semi-finaller-help-out-in-er/](https://hackaday.com/2015/09/18/hackaday-prize-semifinalist-helping-out-in-the-er/)

(摩尔多瓦)和(拉杜)在他们的祖国罗马尼亚解决紧急医疗问题。他们正在开发一种非常便宜的手镯，可以跟踪病人的心跳、血氧和体温，不管是在急诊室还是在候诊室。

正如这些人所说，健康伴侣是一个装有红外发光二极管、光电二极管、温度传感器和 WiFi 模块的小手镯。他们已经将所有这些部分连接到一块自制的板上，连接上电池，并开始测量他们的生命体征。

这是一个简单的设备，但它简单是有原因的:心率和血氧饱和度是医生和护士在对病人进行分类时最重要的指标。通过让他们的健康监测器变得便宜和足够好，它最终会被更多的病人戴上手腕，并有望拯救更多的生命
