# 口袋妖怪 Go 鸡蛋孵化器带你的鸡蛋旋转

> 原文：<https://hackaday.com/2016/07/18/pokemon-go-egg-incubator-takes-your-eggs-for-a-spin/>

口袋妖怪 Go 为世界健康做了一件大好事。或者说如果我们不一直黑它的话。该游戏建议，你可以通过四处走动来繁殖口袋妖怪蛋，但[DannyMcMurray]有一个更好的主意:[将你的智能手机绑在风扇的螺旋桨上](http://www.instructables.com/id/Pokemon-GO-Egg-Incubator/?ALLSTEPS)，并以这种方式带着它们旋转。

他展示了他如何只用一根橡皮筋和一个风扇建造了一个口袋妖怪孵化器。这个小技巧让你不用实际离开房子，但这个过程仍然非常缓慢:视频中的延时旋转导致大约 0.01 公里的进展，是所需 2 公里的 0.5 %。你需要旋转你的手机超过两个小时，直到小家伙孵化。

也就是说，我们仍然抓耳挠腮。Pokemon Go 严重依赖 GPS。为什么鸡蛋培养箱不局限于此呢？如果你有答案，或者已经尝试过，并能确认它的工作，我们希望听到你在下面的评论。
