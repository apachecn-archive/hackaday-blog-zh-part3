# 简单的心电图证明你终究不是无情的

> 原文：<https://hackaday.com/2016/11/07/simple-ecg-proves-you-arent-heartless-after-all/>

我们不认为人体是一件电子产品，但是我们身体中惊人的一部分依靠电力工作。心脏肯定是其中之一。当你想到它的时候，你会觉得它非常神奇。拳头大小的泵，预期使用寿命接近 100 年。

所有这些电活动都是你可以监控的，如果你知道要寻找什么，不规则的模式可以告诉你那里是否一切正常。[Ohoilett]是一名生物医学领域的研究生，他分享了一些用于读取心电图(ECG)数据的简单电路。你可以在下面看到结果的视频。

该设计以仪表放大器为中心，仪表放大器具有高输入阻抗和极高的共模噪声抗扰度，因此非常理想。最简单的电路只是一个仪表放大器，仅此而已。缺点是它需要双极性电源。[Ohoilett]还展示了如何再增加几个器件来简化电源。无论哪种情况，示波器都会提供输出显示(如果您坚持，页面上有一个视频显示连接到 Arduino 的 ECG)。

如果你想知道更多关于如何处理数据的信息，你可能会喜欢这篇文章。如果你决定认真对待，你可以去看看非常强大的[ECG](https://hackaday.com/2014/08/13/an-open-hardware-platform-for-ecg-eeg-and-other-measurements/)和其他生物电测量开放硬件项目。
