# 需要帮忙吗？两个怎么样？

> 原文：<https://hackaday.com/2017/12/05/need-a-hand-how-about-two/>

伸出援助之手对完成一项任务大有帮助。有时它以朋友的形式出现，有时它是一双附在你手臂上的机械手。

意大利初创公司[Youbionic]开发了这双 3D 打印手，旨在扩展用户的多任务处理能力。绑在前臂上，伸过用户的自然手，通过弯曲食指或无名指来单独操作。这种运动被一对柔性传感器条检测到——剧烈的运动将使拳头合拢，而较慢的移动将使拳头合拢一半。

目前，手的使用受到限制——它们被固定在安装板上，因此仅限于抓取任务，但经过一点练习，最终可能会非常方便。休息之后，看看他们的行动视频！
