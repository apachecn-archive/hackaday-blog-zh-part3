# Hackaday 奖参赛作品:一个开源窑

> 原文：<https://hackaday.com/2017/08/30/hackaday-prize-entry-an-open-source-kiln/>

作为他的 Hackaday 奖参赛作品，[Matt] [正在建造一个熔化金属和烧制粘土的小窑](https://hackaday.io/project/21642-repkiln)。他用任何人都能获得的材料——泥土和一点镍铬合金线——来建造这座窑。

你在网上找到的大多数砖窑建筑都使用了你后院可能没有的奇特的耐火砖和其他材料。【马特】的项目完全是 DIY，[从一大堆泥土和石头](https://hackaday.io/project/21642-repkiln/log/59489-processing-natural-clay)开始。在抖掉筛下的泥土，冲洗岩石，滤掉砾石，除掉沙子，吸干水之后，[马特]有了一大袋湿粘土。这种粘土与珍珠岩(一种隔热耐火材料)混合，制成砖，然后烧制。结果是一块看起来足够好的砖，可以用来做窑。

[Matt]已经做了大量的工作来计算出这个窑的热传递。这个窑最多需要 14 个小时才能达到温度。这慢得令人难以置信，但话说回来，这个窑将是电动的，只使用 1500 瓦。与商用电窑相比，这算不了什么，但这是马特自己设计的，没有任何外部帮助，只用了他很容易获得的零件。无论如何，这是 Hackaday 奖的一个优秀项目。
