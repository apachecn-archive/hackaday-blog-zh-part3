# 钙钛矿太阳能:即将到来？

> 原文：<https://hackaday.com/2016/07/15/perovskite-solar-coming-soon/>

用硅制造太阳能电池很困难。有大量的制造步骤，其中许多是在非常高的温度下，你需要高真空和洁净室。然而，[钙钛矿太阳能电池](https://en.wikipedia.org/wiki/Perovskite_solar_cell)——由钙钛矿晶体结构的有机-无机混合材料制成的电池——使用涉及溶剂或蒸汽沉积的湿化学相对容易制造。

理论上，硅太阳能电池的效率可以达到 30%，但实际上，25%似乎是一个实际的极限，商业电池通常会达到 20%。钙钛矿电池现在差不多有这么高了，通过堆叠薄层可以更高，每层对不同波长的光敏感。

劳伦斯·伯克利国家实验室最近的一项进展可能会产生更高效的钙钛矿电池。研究人员发现，某些晶体结构的效率比其他结构高得多。现在的问题是弄清楚如何生产晶体来增加这种结构的普遍性。

然而，这不是唯一的问题。钙钛矿电池降解很快。使用的材料往往是水溶性的，在大多数环境下细胞会分解。已经有了制造更持久的电池的工作。单结钙钛矿电池的当前效率记录是 22.1%。

研究人员希望利用这项技术解决稳定性问题，生产高效太阳能薄膜。他们设想轻质、透明、廉价的太阳能电池将像壁纸一样容易应用，覆盖任何有阳光照射的表面。
