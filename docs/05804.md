# 有电锯，会旅行

> 原文:[https://hackaday.com/2017/05/02/have-chainsaw-will-travel/](https://hackaday.com/2017/05/02/have-chainsaw-will-travel/)

如果你把电锯马达绑在三轮车上，可能发生的最糟糕的事情是什么？事实证明，发生在[这位达斯汀]和他的朋友身上的最糟糕的事情是，它变成了令人愉快地令人敬畏的事情。

这款名副其实的“链锯车”并不太舒适，因此必须在前叉上焊接一对脚钉，以及一个链锯马达支架。后轮轴必须更换为 5/8”的键控股票，修剪，以适应三轮车车轮，并用键控轮毂固定。[ThisDustin]和船员还需要一个中间链轮作为减速齿轮。

在一项测试中，链条从链轮上跳下，并解决了一些问题，如转向能力，链锯可以以相当不错的速度拖着它的骑手。休息过后，请观看视频。
