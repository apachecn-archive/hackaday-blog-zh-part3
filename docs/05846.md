# Sudo 谷歌助手

> 原文：<https://hackaday.com/2017/05/16/sudo-google-assistant/>

一个树莓派放在你的工作台上是一个等待发生的项目——如果它们被闲置足够长的时间以至于被认为是“多余的”如果你发现你一直在渴望一个 Alexa 或你自己的个人 j . a . r . v . I . s ,[ nova spirit Tech]也许可以帮助你——前提是你手边有一个 USB 麦克风和扬声器——提供一个在你的 Pi 上设置[谷歌助手的简单教程。](https://www.novaspirit.com/2017/05/02/google-assistant-raspberry-pi/)

在 Google 的云平台上启用一个新的 API 客户端的快速演练之后，[Novaspirit]跳转到 Raspbian 控制台开始更新 Python 和其他一些依赖项。注意:这是在 Raspbian 的最新版本中进行的，所以在开始使用所有的 sudos 之前，请务必更新。

一旦[Novaspirit]解决了这个问题，他就在 Pi 上建立了一个运行 Google Assistant 的环境，验证了这个过程，并在提供了一些故障排除技巧后让它运行起来。[Novaspirit]计划在不久的将来进一步扩展这一点，实现一些家庭自动化，但如果你一直在寻找一种方法来为自己打破一些高科技家庭美食——或一些[更简单的](http://hackaday.com/2016/05/01/minimal-433-mhz-web-home-automation/)——这是一个很好的起点。如果你喜欢看在命令行打字的视频，请在休息后看看视频版本的教程。
