# 这个拳击钟是一次旅行

> 原文:[https://hackaday.com/2018/02/17/this-boxing-bell-is-a-trip/](https://hackaday.com/2018/02/17/this-boxing-bell-is-a-trip/)

[MeasuredWorkshop]想知道拳击钟装置是如何工作的。最好的学习方法是实践，所以他马上就做了一个。拳击钟是一个罕见的幸存的例子。旅行铃在学校和公共建筑中被用作火警。它们已经被现代电力系统所取代。

跳闸钟后面的机械联动装置是一个单向杠杆。这是你拉的手臂。它有一个铰接部分，当手臂被拉下时保持刚性，但当手臂被释放时旋转离开。[测量车间]使用相当基本的工具建造了他的钟的机械装置。这项工作的主要部分由角磨机和钻床完成。

这个拳击钟的发声器来自一个旧的学校钟。工业灰色油漆经过化学剥离，金属经过清理，呈现出漂亮的磨砂效果。金属在用作底座的木板[测量车间]的映衬下非常显眼。

成品看起来和听起来都很像——现在他只需要找到一个需要铃铛的拳击馆就行了！
