# 你的下一个鞭包记忆锁子甲轮胎吗？

> 原文：<https://hackaday.com/2017/12/02/will-your-next-whip-pack-memory-chainmail-tires/>

美国宇航局格伦研究中心正在试验类似锁子甲的镍钛记忆合金轮胎。这是一个有趣的角度——轮胎可以承受更重的负荷和更高的速度。它们不透气，不会被刺穿。据推测，他们不是字面上的锁子甲，但更接近于建筑毛衣。

这种轮胎是一系列引人入胜的研究成果的顶峰。美国国家航空航天局一直在试验张拉整体结构，作为一种在太空中建造的方式，而不需要在重型硬件上花费大量的火箭燃料。这些结构使用张紧的缆索来保持三维结构。轮胎利用钢丝的硬度以及内部加强筋来保持形状，而不需要整个轮辋。

除了结构张拉整体性之外，记忆合金还通过抵抗变形来帮助保持其原始形状——它会弹回其原始形状。当普通材料被拉伸时，你在拉伸原子结构之间的键。美国宇航局的 NiTi 合金在受到压力时会经历“原子重排”，从而减轻对这些结构的压力。因此，这种合金可以承受 10%的变形，而弹簧钢的变形率为 0.3%，这是普通合金在不发生永久变形(基本上是凹痕)的情况下可以承受的变形量的大约 30 倍。美国宇航局的轮胎实际上可以压缩到车轴，然后弹回。

该配置的另一个优势是牵引力和刚度可以修改，以适应特定的环境-只需改变径向加强筋的张力，使其更硬或更软。这一点，再加上合金的抗变形能力，使它们擅长在岩石上滚动而不会受到伤害；这是火星探测器的一个问题。

美国国家航空航天局已经为这项技术申请了专利——他们称之为超级弹性轮胎(T1)——他们声称这项技术可以用于私营部门。记忆合金滑板轮你准备好了吗？
