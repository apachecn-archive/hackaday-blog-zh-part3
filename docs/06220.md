# Hackaday 奖参赛作品:坐立不安纺纱者的互联网

> 原文：<https://hackaday.com/2017/06/13/hackaday-prize-entry-internet-of-fidget-spinners/>

我们刚刚结束了 Hackaday 奖的有用物联网轮，这意味着我们正在深入评审项目，以便在今年秋天进入最后一轮。上周，Hackaday.io 上的每个人都在忙着准备他们的四个项目日志和插图，为这一轮 Hackaday 奖的最后一次呼吁做准备。这些项目是物联网所能提供的最好的东西，因为这是有用的物联网。

我们不确定[马提亚斯]的项目将如何排名。[这是一个物联网坐立不安旋转器](https://hackaday.io/project/25372-iot-pov-fidget-spinner)。是的，我们知道，但是建造一个联网的坐立不安旋转器有一些有趣的工程挑战。

这是一个 PoV 坐立不安旋转器，这意味着这个三角旋转器的前沿是令人眼花缭乱的 APA102 发光二极管。视觉暂留玩具和 Hackaday 一样古老，坐立不安旋转器的整个想法就是旋转，所以这至少是有意义的。

这些 PoV LEDs 由带有板载闪光灯的 ESP8285 或 ESP8266 驱动。这可能是你能找到的最小的无线微控制器，对于如此小的体积来说是一个重要的考虑因素。动力来自一个微小的脂肪，额外的外设包括一个测量摆动的加速度计和一个测量转速的光学开关。

这些电子设备相当标准，在 Hackaday 奖的任何其他项目中都不会显得格格不入。这里的技巧是机械的。[马蒂亚斯]需要安装一个滑板轴承到印刷电路板上，没有人知道他要怎么做。一个坐立不安的旋转者应该保持良好的平衡，而马蒂亚斯又一次遇到了问题。这里有人做过 PCB 和锂电池的质量和密度计算吗？3D 打印保形配重可能吗？科学是否走得太远了？

物联网 PoV 坐立不安 Spinner 会进入 Hackaday 奖的决赛吗？我们需要等一周左右才能知道。不过，有一件事是肯定的:你将在 9 月前在阿里巴巴上看到这一点。
