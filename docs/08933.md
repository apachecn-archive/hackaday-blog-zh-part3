# 多色发光二极管构成了一个美丽的彩色时钟

> 原文:[https://hack aday . com/2018/04/01/multi-colored-LEDs-make-for-a-beautiful-colour-clock/](https://hackaday.com/2018/04/01/multi-coloured-leds-make-for-a-beautiful-colour-clock/)

This project is so pretty in its own right, it doesn’t need a case!

时钟是我们在 Hackaday 上展示的项目中反复出现的一个特征，其中出现了几个共同的主题。我们看到传统的带指针的钟，有各种显示形式的数字钟，包括无处不在的数码管，还有大量的 LED 环形钟。[Matt Evans]的作品是最后一个类别之一，[一个特别漂亮的 LED 环形时钟，使用线端多色 LED](http://axio.ms/projects/2018/03/19/Colourclock.html)。其他时钟产生的效果从房间的另一边看起来很好，但这个在近距离观察时也是一件美丽的作品。

在它的背后是四个互锁的半圆形印刷电路板，一个控制时钟的 STM32F051C6T6 ARM Cortex M0 微控制器和一对驱动芯片。时钟的不同“指针”表现为不同的 LED 颜色，有各种不同的颜色和时钟“指针”效果。丙烯酸环通过覆盖 led 本身来完成效果。他制作了一个时钟运行的视频，你可以在画面下方看到。
