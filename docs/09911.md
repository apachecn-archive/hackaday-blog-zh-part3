# 我们见过的最大的角天线

> 原文：<https://hackaday.com/2018/07/17/the-biggest-corner-antenna-weve-ever-seen/>

无线电波通过天线接收，因此当信号长距离传输时，需要一个大的反射器。当接收距离实际上是天文数字时，反射器必须相当大。[思想商场]想要从遥远的卫星、月球以及有希望的脉冲星那里获得信号。在自制业余无线电的规模上，这将是一个巨大的天线。视频也跟着休息。

按照黑客的方式，项目是建立在预算的基础上的，所以所有的部件都是直接从五金店购买的，而且工具已经在你的工具箱或者黑客空间里了。电线导管、铁丝网、PVC 管、木块和螺丝构成了这个结构的大部分，所以收起你与中国经销商的疯狂联系吧，除非你在 T2 需要特别提款权。天线的形式是至关重要的，形状是三个垂直的面板，如图像和视频所示。视频中的构造只是一个建议，但它不涉及焊接，所以它向更多的业余爱好者开放。

即使你不想接收脉冲星的信号，我们也有大量的无线电设备和 T2 天线。
