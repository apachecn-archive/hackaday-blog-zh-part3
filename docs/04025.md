# 指弹是最简单的多声道混音器

> 原文：<https://hackaday.com/2016/11/01/fingerring-is-simplest-multichannel-mixer/>

很难用比 [FingerRing](https://www.youtube.com/watch?v=klD_suu74wc) (YouTube 视频，嵌在下面)更少的技术做出混音器。我们很确定[谢尔盖·卡西奇]不会在这个项目上获得专利。但他确实得到了我们的赞赏，因为他将一个简单的想法推进到了足够远，以至于它显然是有用的。

基本思想是利用人体作为导体来传输信号。谢尔盖所做的是在桌子上摆放多个声源和接收器，然后像调音台制作的乐器一样演奏它们。用力按压会减小阻力，使声音变大。连接到两个源头将它们混合在一起。观看视频——他从这一招中获益匪浅。

我们可以对这个系统进行一些改进。一块木板上的一堆钉子作为接触点，比绑在桌子上的 1/4 英寸电缆更容易演奏。你可以让它成为永久的工具。如果你从头开始设计系统，你会希望在接收端有高输入阻抗放大器。添加一个[陷波滤波器](http://www.radio-electronics.com/info/circuits/rc_notch_filter/twin_t_notch_filter.php)来消除电源嗡嗡声，或者在你的脚踝上添加一个[仪表放大器](http://hackaday.com/2016/03/18/beyond-measure-instrumentation-amplifiers/)和另一个电极。很快，你就成了 EKG *和*混音台。

他最大的局限是身体只是一个大导体。有人能想到如何在不使整个设置过于复杂的情况下获得多个通道吗？现在请原谅我们在桌子上贴满了仪器电缆。我们很受鼓舞。
