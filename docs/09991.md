# 迪士尼的人形特技机器人毫不费力地抛出多个后空翻

> 原文:[https://hack aday . com/2018/07/08/Disney s-人形-特技-机器人-投掷-多次后空翻-无汗/](https://hackaday.com/2018/07/08/disneys-humanoid-stunt-robots-throw-multiple-backflips-no-sweat/)

现在人形机器人最大的问题是什么？他们倒下了。迪士尼似乎已经解决了这个问题，它制造了可以摔倒并被网住的机器人。迪士尼的研究部门(你可能知道他们是幻想工程师)正在展示一个名为 Stuntronic 的机器人，它可以在空中飞行时进行受控的翻跟头。检查下面的视频，你真的要看几遍才能确定这是机器人而不是人。

跟踪这个机器人的进化真的很有意思。它始于砖块，这是一个没有四肢的矩形机器人，可以在空中移动时改变重心来控制方向。从那时起，[斯蒂克曼](https://www.disneyresearch.com/publication/stickman/)将这些概念应用到一个棒状机器人中，这个机器人有两个铰接部分，当它在空中飞行时，可以控制它的筋斗。特技感觉像是从那个设计的一个大飞跃。

与 Stickman 一样，它可以弯曲来控制半空中的筋斗，但随着铰接臂的增加，Stuntronic 也可以增加杂技包的技巧。在我们看来，这是非常逼真的，并设法完全逃离了恐怖谷。这是一个响亮的认可，因为这项研究的拟议目的之一是迪士尼主题公园的现场表演。

总统大厅是当时的一个奇迹，因为机器人发表了著名的演讲，同时被装饰成最初发表演讲的领导人的样子。但是站着说话不腰疼是几十年前的把戏了。我们希望这是明年的一个小把戏，而不是我们必须在未来几十年醒来才能亲自看到的东西。

哦，对于那些想知道是否特技演员卡住了着陆？可控分娩进入网的温暖怀抱同样令人印象深刻。希望，成功的着陆是司空见惯的，因为他们用一些真正邪恶的力量发射这些机器人！除了你期望在运动感知机器中找到的陀螺仪和加速度计，该设计还使用了三个激光测距仪，对地面位置进行三角测量，以确定最佳着陆位置。我们还没有看到这个机器人的出版物，但请查看 Stickman 信息以了解更多关于这些传感器的信息。
