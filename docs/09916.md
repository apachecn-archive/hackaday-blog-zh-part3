# 这是一件艺术品

> 原文：<https://hackaday.com/2018/07/04/a-pick-and-place-that-is-a-work-of-art/>

这是黑客空间中的圣杯，拥有一台取放机。这些用于将表面贴装元件放置在 PCB 上的机器人助手是电子生产的一个入口，但它们可能会带来可怕的成本。对于资金紧张的潜在电子制造商来说，令人高兴的是，有可能为自己建立一个提货点。[Mcuoneclipse]已经用[展示了这一点，这是一个相当令人印象深刻的构建](https://mcuoneclipse.com/2018/06/26/building-a-diy-smt-pickplace-machine-with-openpnp/)，它可以与免费提供的 [OpenPnP](http://openpnp.org/) 软件一起工作。

从表面上看，它与你可能期望的小型数控铣床有很多相似之处，因为它有一个由挤压铝制成的框架，框架上有轨道，轨道沿着 X 轴和 Y 轴移动，支撑着工具头。但它没有刀片，而是有一个由激光切割板制成的盒子，里面有一台相机和一个真空拾取工具，可以从胶带上收集元件，并将其放置在电路板上的正确位置。机器的心脏是一块光滑的滑板，工作由各种电磁阀和执行器完成。这款产品对细节倾注了大量的关注，用于不同元件尺寸的所有可互换喷嘴的支架、用于所有电动元件的激光切割支架以及用于 SMD 胶带的自动送料器都经过精心设计和制造。设计的几个迭代被提出，特别是围绕头部本身，它已经通过了一个以上的形式，以消除尽可能多的振动。但是不要把它从我们这里拿走，看看我们在休息下面粘贴的视频。

这不是我们在 Hackaday 带给你的第一台拾放机。如果你已经有了一台 3D 打印机，[你会考虑这次升级吗](https://hackaday.com/2018/05/11/pick-and-place-machine-is-mirror-image-of-3d-printer/)？
