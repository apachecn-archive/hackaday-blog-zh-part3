# 在苏维埃俄国，饮料为你服务

> 原文：<https://hackaday.com/2015/11/17/in-soviet-russia-drink-serves-you/>

有一台自动调酒机会发出“我们现在在未来”的声音，这是我们认可的未来。这个项目被称为 [Alkomat](http://geektimes.ru/post/265722/) ( [谷歌翻译](https://translate.google.com/translate?sl=auto&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fgeektimes.ru%2Fpost%2F265722%2F&edit-text=&act=url))，这是一个方便的小机器，能够同时为五个人混合饮料。

受到 RumBot 的启发，[Strn]想在硬件集成和产品开发方面从想法到创作实践他的手——仅仅看一些这方面的照片，他就做得非常好。

在 Alkomat 的内部是一个回收的喷墨 XY 托架，可以索引五种饮料中的每一种。一系列的四根管子输送到蠕动泵，蠕动泵从吧台架提供酒精。一旦他把硬件整理好并投入使用，他就着手让机器看起来尽可能的像样。他混合使用台面和家具的层压木板，将整个东西装箱，让它看起来像是你实际上可以从商店买到的东西——也许是宜家？

我们最喜欢的部分是他如何集成了一个 CD 驱动机制，以便在混合后将饮料向前滑动。看看这个。
