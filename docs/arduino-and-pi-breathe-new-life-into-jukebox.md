# Arduino 和 Pi 为点唱机注入新的活力

> 原文：<https://hackaday.com/2017/09/22/arduino-and-pi-breathe-new-life-into-jukebox/>

当有人送你一台 1969 年的 Wurlitzer 3100 点唱机，却保留了所有的唱片，你会怎么做？如果你像[蒂华纳·里克]一样，你拿起一个 Arduino 和一个 Rasberry Pi，把它变成一个非常棒的数字音乐播放器。

我们不得不承认，用树莓做音乐播放器并不那么尖端，但修复和整合工作确实令人印象深刻。这台机器有许多损坏的开关被匆忙修复，所以[瑞克]不得不学习制造硅胶模具和浇铸树脂来制造替代品。你可以在下面的视频中看到和听到最终的结果。

[Rick]对他能找到的点唱机软件感到沮丧，直到他从[Thomas Sprinkmeier]那里找到了一些[Python 代码](https://github.com/sprinkmeier/jukebox)。[Rick]使用该代码作为基础，并根据他的需要进行定制。

没有太多的“如何”开关铸件的细节，但有很多照片和结果是伟大的。我们想知道他是否考虑过将假 45 放入机器中，这样至少看起来像是在播放黑胶唱片。

当然，你不需要一个旧硬件[来做一个点唱机](https://hackaday.com/2017/06/02/hackerspace-jukebox/)。或者，你可以妥协，[造出一个复制品](https://hackaday.com/2017/02/04/bantam-sized-jukebox-reproduction-tops-the-fabrication-charts/)。
