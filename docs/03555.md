# 气动顶部仅适用于 3D 打印机

> 原文：<https://hackaday.com/2016/09/25/air-powered-top-only-possible-on-a-3d-printer/>

任何人都会转向 3D 打印机的一个主要原因是，有些形状是无法用传统的“减法制造”技术制造的，即使他们可以去机械商店。我们很多人使用 3D 打印机超过传统加工还有一些更明显的原因，如尺寸和成本，但还有另一个主要原因是 3D 打印机变得越来越普遍。[Crumbnumber1]在 Make everything 的 3D 打印频道向我们展示了[3D 打印机在迭代设计中有多么强大，他的气动上衣](https://imgur.com/gallery/F6605)。它们结合了风扇叶片，让你可以通过向下吹气将陀螺旋转到非常高的速度。

[迭代设计](https://en.wikipedia.org/wiki/Iterative_design)是一种快速制造原型的能力，这种原型在先前原型的基础上进行构建和改进，直到你剩下一些能完成你需要的工作的东西。即使有一个机械车间供您使用，为一个零件设置所有的工具也是很昂贵的，只是发现这个零件需要更换，而您现有的工具不再适用。这就是 3D 打印机可以介入的地方。除了所有其他的优点，它们对于快速原型制作非常有用。[Crumbnumber1]制作了一个装满陀螺的盒子，并能够测试许多不同的设计，然后确定了一个性能超出之前所有设计的设计。
