# 显微镜照片中难以想象的巨大焦深

> 原文：<https://hackaday.com/2018/05/11/impossibly-huge-depth-of-focus-in-microscope-photographs/>

有时候，少即是多。有时候，越多越好。有一类人认为，如果拍摄了足够多的同一主题的照片，其中一张会在其他照片上闪耀，成为一个闪闪发光的例子，说明用手机相机和稳定的手可以做到什么。其他人知道如何在按下快门按钮之前给照片加框。在某些情况下，最好的方法可能是抓拍一些照片来获得一张好照片，这不是偶然的，而是有意的。

[The Thought Emporium]的视频也在休息时间之下，是关于[使用聚焦叠加，有时被称为图像叠加，从 DSLR 相机和显微镜](http://youtube.com/watch?v=3wfI_rEGyDw)获得清晰的照片。前提是拍摄一系列的照片，每张照片都有一个不同部分的主题在焦点上。在显微镜下，这个范围很小，但在公园里，可能有几米。当图像被组合时，他使用 Adobe 产品，聚焦的区域被保存，而非聚焦的区域被丢弃，结果是一张具有不可能的聚焦深度的照片。我们不禁会想起那些光场相机，它们不依靠移动镜头来对焦，而是拍摄许多照片，每张照片的焦距都不同。

[The Thought Emporium]之前已经向我们展示了他的摄影热情，以及他对[从植物中取出细胞](http://hackaday.com/2018/04/30/decellularization-apples-to-earlobes/)和从屠夫身上切下不寻常的肉的喜爱，甚至对[战胜乳糖不耐症](http://hackaday.com/2018/02/14/biohacking-lactose-intolerance/)进行了高尚的尝试。
