# 汽水罐艺术

> 原文：<https://hackaday.com/2018/05/02/soda-can-art/>

一罐汽水的价格大约是半美元，一旦你喝完糖浆，大部分汽水都会被扔进垃圾桶回收利用。有些人出于其他目的重复使用它们，但我们猜测没有人像艺术家[诺亚·德勒达]那样循环利用它们。他把它们变成 T2 的汽水罐艺术品，售价在 2000 到 3000 美元之间。

不要被 GIF 中的精彩照片所迷惑。这只是[诺亚]为了给人们留下深刻印象而做的一些特技摄影。如果你先看看没有背景故事的最终产品，你会认为罐头被操纵成扭曲的形状，至少使用了某种机械辅助，或者可能是专门制造的机器。

但是[诺亚·德勒达]赤手空拳地做到了。这是折纸的裸机版本。多年前在一次公路旅行中，他没有电子设备让自己忙碌。他摆弄着一个空汽水罐，开始把薄金属压成一个抽象的形状。这时，他内心的艺术家意识到，他正在玩一种令人兴奋的新媒体。在用一种朱红色牛能量饮料的空罐制作了一些抽象艺术作品后，他认为如果他能去掉罐上的所有油漆，给它们一个光滑、抛光、自然的表面，看起来会更棒。他做了一个旋转罐子的小机器，这样他就可以剥去油漆，把罐子抛光。这项技术很简单，但需要大量的耐心、练习、时间和技巧，更不用说它会导致拇指疼痛。

如果你曾经去过日本，喝过一罐麒麟猪鬃罐头，你会注意到未打开的罐头是光滑的，但一旦打开，马上就会变成锯齿状钻石的 T2 图案。这个设计是由米乌拉京郎公司创造的，该公司以“T4”米乌拉折叠器而闻名，这种折叠器可以让你在一个平稳的动作中折叠和展开大张纸。就像你坐在车里的手套箱里的那张被丢弃的地图，一边玩着一个空汽水罐。

如果你想磨练一些双手通用的技能，这将是一个很好的方法，当你在下一次公路、飞机或火车旅行时。看看下面嵌入的两个视频。在第二个，你可以看到设计过程的快照。
