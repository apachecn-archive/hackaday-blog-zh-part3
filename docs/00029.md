# Hackaday 奖半决赛:智能配药器

> 原文:[https://hack aday . com/2015/09/08/hack aday-prize-semi-finaller-smart-medicine-dispenser/](https://hackaday.com/2015/09/08/hackaday-prize-semifinalist-smart-medication-dispenser/)

药品最大的问题不是专利、行业代表，也不是允许向消费者做广告的事实；这种情况只发生在美国。不，药丸和药物的最大问题是依从性，或者确保被开药的人*服用*他们的药物。为了他的黑客奖参赛作品，[乔] [正在研究一个解决方案](https://hackaday.io/project/7019-smart-desktop-medicine-organizer)。这是一个智能桌面药物管理器，你可以把它想象成一个智能药盒。

[Joe]的组织者的功能列表包括自动药丸组织-每个处方都独立于所有其他处方进行访问。当该吃药的时候，智能配药器会发出一粒药丸。你可以看看用 M 个& M 个糖果制作的演示视频[。](https://www.youtube.com/watch?v=qk7zqA_ISm0)

智能桌面药物管理器还有更多的功能，包括连接到药房 API 以订购补充药物，检查药物相互作用，以及为不同的药物设置计时器(或不设置计时器)；应该每天服用的药物将每天分发，但在最高限额内按需服用的药物将按需分发。

这是一个非常酷的项目，你可以看看下面这个项目的[Joe]视频。
