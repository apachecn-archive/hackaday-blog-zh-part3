# 周五黑客聊天:过时的技术与弗兰布兰奇

> 原文:[https://hack aday . com/2017/05/24/Friday-hack-chat-antiqued-technologies-with-Fran-blanche/](https://hackaday.com/2017/05/24/friday-hack-chat-antiquated-technologies-with-fran-blanche/)

本周五加入我们关于过时技术的[黑客聊天](https://hackaday.io/event/22120-antiquated-technologies-hack-chat)。

每周五，我们召集硬件领域的一些人，让他们坐在键盘前，讨论他们在做什么。本周，我们将与[Fran Blanche]讨论古老的技术、怪异的电子产品以及从头开始建立一家硬件公司的一切。

[![](../Images/e9258983fbea510541a0f4373896e966.png)](https://hackaday.com/wp-content/uploads/2017/05/franblanchehackchatv11.png) 你可能会问【弗兰布兰奇】是谁？她是一名自学成才的电子工程师、艺术家、音乐家、摄影师、机械工程师和 YouTube 视频博客作者。她是第一批精品吉他效果器制造商之一[弗朗通电子公司](http://www.frantone.com/)的创始人。她的 [Peachfuzz](http://www.frantone.com/peachfuzz.htm) 是非常非常少的*原创*失真/模糊电路之一。她在布朗大学发表了关于精品特效产业的演讲，与富兰克林研究所合作开发了 Maillardet 自动装置，参观了最大的乐器，从土星五号上拆卸了飞行硬件，只是为了看看它是如何工作的，她还去过史密森尼航空航天博物馆的仓库。

有了这么好的工作履历，弗兰现在在做什么？她正试图用阿波罗导航计算机重现 DSKY。DSKY 是阿波罗制导计算机的用户界面，这是一块镶嵌着漂亮按钮和电致发光显示器的奇妙铝块。在过去的几年里，她复制现代 DSKY 的尝试——包括定制的分段 EL 显示器——一直被搁置，但现在[Fran]正试图在 GoFundMe 上筹集资金进行复制。我鼓励你们至少看看 GoFundMe 活动。
