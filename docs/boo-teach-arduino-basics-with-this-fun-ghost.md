# 嘘！用这个有趣的幽灵教 Arduino 基础知识

> 原文：<https://hackaday.com/2015/10/09/boo-teach-arduino-basics-with-this-fun-ghost/>

万圣节就要到了，怪异的主题提示也开始出现了。如果你想超越商店里买的基本装饰品，也许在你做的时候教你的孩子 Arduino 的基本知识——为什么不做一个躲猫猫幽灵呢！

使用一个 Arduino，两个伺服电机和一个超声波距离传感器，很容易就可以制作出这个可爱的小幽灵，当没有人在旁边时，它可以遮住眼睛。他们用纸板做幽灵，但是如果你在你的黑客空间有激光切割机，你可以用中密度纤维板或胶合板做得更结实。

当超声波距离传感器感应到有人向它走来时，它会触发手臂移动——尽管添加一个小扬声器元件并播放一些怪异的音乐也很容易！
