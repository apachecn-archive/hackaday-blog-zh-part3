# 忘了情人节？一个快速的情人节

> 原文:[https://hack aday . com/2017/02/14/forgot-about-valentines-day-a-quick-IOT-valentine/](https://hackaday.com/2017/02/14/forgot-about-valentines-day-a-quick-iot-valentine/)

你忘了情人节吗？你需要一个快速的项目来为情人节做准备吗？[Becky Stern]你看了吗？她已经完成了一个简洁的基于互联网的情人节项目，应该很快就能完成。

它的核心(双关语)是一个 ESP8266 微控制器，在这种情况下是一个 Adafruit Feather Huzzah。几层薄纸心形被缝合在一起，剪成心形，然后附在弹簧上。当信号进来时，一个振动寻呼机马达被用来振动它。两个按钮用于发送信息，一个红色 LED 用于点亮心形灯。整个事情都被封闭在一个阴影盒。[Becky]还将另一个具有类似设置的控制器放在一个塑料外壳中。当按下其中一个控制器上的按钮时，另一个控制器会收到信号，心脏会震动并发光。

这些项目发送和接收情人节礼物，但是它们可以被编程来发送你想要的任何信息。如果你正在寻找一个快速的情人节项目，这是一个伟大的，你可能已经在你的组件抽屉里有你需要的一切。拿出烙铁，给你的情人发个信息吧！如果你仍然在寻找一个快速的情人节项目，看看[这个动画心脏](https://hackaday.com/2015/02/13/animated-led-valentine-heart/)或[这个 PCB 情人节](https://hackaday.com/2014/02/14/valentines-pcbs-to-make-your-heart-melt/)。
