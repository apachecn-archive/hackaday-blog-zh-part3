# 3D 打印保存了可收集的鱼饵

> 原文：<https://hackaday.com/2018/05/14/3d-printing-saves-collectible-lures-from-a-fishy-ending/>

给一个人一个鱼饵，他会一直抓鱼，直到失去鱼饵。给一个渔夫一台 3D 打印机，他就可以[打印出他想要的所有鱼饵](http://www.hunterirving.com/blog/2017/gone_fishin/)，尤其是那些珍贵到不能实际使用的复制品。

有些人收集鱼饵而不是使用它们，这可能看起来很奇怪，但是当你看[Hunter]的收集时，就很容易明白为什么了。诱饵可以非常艺术化，他收藏的赫登河矮子是美丽的东西，非常珍贵。它们在说服鱼自杀方面也非常有效，所以与其拿原件冒险，他和他的父亲 3D 打印了复制品。

在搅拌机中制作好鱼饵的模型后，他们在鱼饵上安装了气囊以增加浮力，并在鱼饵上安装了孔以连接高音挂钩和唇勺，鱼饵是由步枪外壳上的黄铜碎片制成的。成品鱼饵缺少原始 River Runt 的绘画细节和一些魅力，但它拥有 Heddon 先生在 1933 年推出它时无法想象的东西——它在黑暗中发光，这要归功于使用的磷光 PLA 灯丝。这对鲈鱼来说似乎是不可抗拒的，它们经常上钩，以至于厌倦了拍照。请观看下面的视频。

[亨特]和他的父亲一直忙于探索 3D 打印可以做什么，复制各种 Heddon 诱饵。他们甚至计划设计和印刷自己的鱼饵。但是也许射箭比钓鱼更符合你的运动员风格，在这种情况下，这个 PVC 管复合弓或[滑雪板反曲弓](https://hackaday.com/2013/04/09/recurve-bow-make-from-wood-and-skis/)是值得一试的。
