# 径向螺线管发动机是不可否认的冷却

> 原文：<https://hackaday.com/2015/10/05/radial-solenoid-engine-is-undeniably-cool/>

星形发动机非常酷——毫无疑问，任何源于早期航空的技术都是天生的棒。但是，当你想在宿舍里造一台径向引擎，而内燃机是不可取的时候，你会怎么做呢？对于华盛顿大学的学生[Jeffrey Weng]和[Connor Lee]来说，答案是用螺线管代替活塞来提供动力。

处理这样一个项目最简单的方法就是使用微控制器。在 Arduino 上运行一个简单的程序就可以很容易地为每个螺线管提供连续的电源切换时间。然而,[Jeffrey Weng]和[Connor Lee]采用了一种更有趣的方法，通过一个简单的分配器来控制时间。这与内燃机上的火花分配器的工作方式相同，只是它实际上是提供动力来驱动螺线管，而不仅仅是提供点火火花。

同样令人印象深刻的是他们用这些基本工具所能完成的事情。我们这些懒惰的人可以使用更昂贵的工具，他们只需要 3D 打印或 CNC 切割大部分零件。[Jeffrey Weng]和[Connor Lee]要么没有机会接触到这些，要么他们想增加他们的加工信誉，因为他们用带锯和钻床等简单的工具制造了所有的零件。在之前，我们已经看过一些漂亮的引擎项目，但是这个版本在客观美感上的不足在独创性上得到了弥补。
