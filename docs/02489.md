# 在家制作弹簧

> 原文:[https://hackaday.com/2016/05/29/making-springs-at-home/](https://hackaday.com/2016/05/29/making-springs-at-home/)

[这个老托尼]教我们如何在车床上制作弹簧,这个视频是按照制作的方式制作的。以斯纳克一贯的风格，夹杂着许多有用的信息。

机械手册当然有设计基本弹簧形状所需的所有信息，但并不总是必要的。[托尼]指出作弊是完全可以接受的。例如，如果您需要一个接近标准弹簧尺寸的弹簧，只需复制标准弹簧的值。他解释了解密你选择的工程书籍中的页面所需的所有术语。

他展示了在心轴上缠绕弹簧的基础知识(或者那个圆形的金属东西，如果你想用行业术语的话)。首先缠绕不活跃的线圈，然后设置你的车床到想要的弹簧间距。像穿线一样将其接合，然后分离并缠绕最后一个非活动线圈。快速走到打磨机前，将标准螺旋弹簧的两端弄成方形。然而，这些工具也可以用来制造扭簧，甚至是奇特的组合弹簧。

为了一个好的…有教育意义的笑声，休息后看完整个视频。
