# 3D 打印气垫船开始飞行

> 原文：<https://hackaday.com/2018/01/11/3d-printed-hovercraft-takes-flight/>

上次我们看到[伊万·米兰达]的时候，他正在互联网上发布一个钻床。最近，他一直在尝试 3D 打印气垫船，并取得了一些成功。正如你在下面的视频中所看到的，他做了四次尝试才找到一个效果相当好的。不过，我们要提醒你，螺丝刀摄像头有点令人不安，我们建议你吃完饭后至少等一个小时再看。

右舷叶轮在测试中途断裂，尽管单个叶轮工作得很好。[Ivan]认为他可以打印更强的叶轮框架，以防止未来的故障。这个设计在 Fusion360 中，有足够的细节，如果你有冲动的话，你可以复制他的作品。船头上有一个前灯支架和一个摄像机。

然而，可能比实际构建更有趣的是他实现它的过程。每一次迭代都有视频，看到设计如何在每次进化中从概念变成现实是很有趣的。

我们认为气垫船和表面效应车辆(这是有区别的)是现代发明，但事实是，在 18 世纪就有关于这种效应的讨论，甚至在 1870 年就有了专利，尽管当时还没有足够强大的发动机。第一辆军用地面效应车可以追溯到 1915 年，在第二次世界大战中也有一辆分类气垫车。那是在实用的现代设计出现之前的 20 世纪 50 年代。

我们顺便看过其他的 [3D 打印气垫船](https://hackaday.com/2017/11/06/3d-printed-hovercraft-takes-to-the-air/)。我们还看了一种新颖的[推力矢量气垫船](https://hackaday.com/2014/02/13/thrust-vectoring-hovercraft-on-a-budget/)，它也使用推力风扇产生向前的运动。
