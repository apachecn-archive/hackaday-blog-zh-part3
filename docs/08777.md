# 成长为机器猫的传感器阵列

> 原文:[https://hack aday . com/2018/03/06/the-sensor-array-that-grow-to-a-robot-cat/](https://hackaday.com/2018/03/06/the-sensor-array-that-grew-into-a-robot-cat/)

人类的大脑进化到对任何类似人脸的东西都格外关注。(科学术语:“面部幻觉”)[李荣忠]建造了一个机器人传感器阵列，多个发射器和接收器增强了中心的树莓 Pi 相机。当他看着他的传感器阵列时，他看到一只猫的脸在看着他。这开启了他长达数年的 Petoi OpenCat 项目，旨在打造一个受猫科动物启发的身体来搭配脸部。

![](../Images/1d991f20f3bde10f3cd71d37f140bb94.png)

虽然这个项目的名字表明了[Rhongzhong]的最终意图，但他还没有向开源社区发布项目细节。但通过阅读他的项目页面和仔细查看他的 YouTube 视频(下面嵌入了一个最近的视频)，我们可以破译一些细节。运动是通过由 Arduino 配合的业余爱好遥控伺服系统来实现的。更高级的功能，如环境感知和 Alexa 集成，由 Raspberry Pi 3 处理。

(目前)秘方是将它们连接在一起的机械部件。从集成到大腿上的减震弹簧到手腕/脚踝的铰接方式。[荣中]认为当前的迭代太难构建，他希望在发布之前简化构建。虽然我们没有太多关于软件的信息，但启动它的传感器阵列暗示了某种程度的传感器融合能力。

我们见过很多机器宠物，出于某种原因，机器狗比猫多得多。灵感可以来自[波士顿动力](https://hackaday.io/project/52956-mikes-robot-dog)，来自[神秘博士](https://hackaday.io/project/2227-k9)，或者来自……[万圣节](https://hackaday.com/2017/11/21/reanimating-boney-the-robot-dog/)？我们认为缺少猫的代表对于机器宠物来说是一个错失的机会。毕竟，如果机器猫的语音识别模块出现故障，某个命令被忽略……那不是 bug，那是作为猫的一个特性。
