# 是角磨机！不，这是地板打磨机！

> 原文:[https://hack aday . com/2017/08/18/its-an-angle-grinder-no-its-a-floor-sander/](https://hackaday.com/2017/08/18/its-an-angle-grinder-no-its-a-floor-sander/)

面对打磨木地板的艰巨任务，你会怎么做？请专业人士？租合适的工具自己动手？甚至可能花大价钱购买专业级工具？或者你会在你的废品堆里翻来翻去，用旧角磨机拼凑出一个又快又脏的地板打磨机？

这就是[唐尼·迪伊]在他的新桑拿室里看着大片新地板时所做的。从来没有人走捷径，显然是因为[对角齿轮箱](http://hackaday.com/2017/06/15/hacked-car-axle-yields-custom-90-gearbox/)的偏爱，【Donn DIY】从一台烧坏的角向磨光机中取出内脏，用于他的即兴地板打磨机。一个附在旧马达转子上的钻头提供旋转，几块碎木头做压盘。砂纸带夹在圆盘之间，正如下面的视频所示，整个装置做了一件令人钦佩的工作。

我们以前见过很多角磨机黑客，[一些有用的](http://hackaday.com/2016/04/03/turning-an-angle-grinder-into-a-belt-sander/)，[一些愚蠢的](http://hackaday.com/2015/07/08/cd-launcher-looks-dangerously-fun/)。这一个完成了任务，并且是一个很好的快速破解，证明了一堆库存充足的垃圾的价值。
