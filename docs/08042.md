# 焦耳小偷偷有利于圣诞节

> 原文：<https://hackaday.com/2017/12/22/joule-thief-steals-in-favor-of-christmas/>

很多东西在节日期间都会变得捉襟见肘，比如购物预算、腰带和耐心。今年，[Chris]用他的焦耳小偷驱动的 LED 迷你圣诞树扩展了迷你试验板和简陋的 1.5 V LR44 硬币电池的极限。

通过微瞬时按钮，joule 小偷电路从 LR44 获得足够的功率来启动 MSP430 微控制器，该微控制器需要 1.8 V 至 3.6 V。启动后，微控制器控制 joule 小偷电路，并在电压降至 3.2 V 以下时利用它。此树可能体积较小，但功能丰富。按下同一个瞬间按钮，四个不同的灯光秀循环播放，最后以四个灯光秀的混合结束。休息后眼花缭乱。

这棵小树的代码出现在 GitHub 上，它有着令人敬畏的 ASCII 试验板布局和原理图。[Chris]它是否被列为 pure C 初学者的几个易于管理的裸机 430 项目之一。如果这听起来像你，为什么不给自己一个学习新语言的礼物呢？

我们已经看到了一些点亮 led 灯的好方法，但是用蜡烛来照明是最棒的。
