# MIDI 口琴

> 原文:[https://hackaday.com/2017/05/21/a-midi-harmonica/](https://hackaday.com/2017/05/21/a-midi-harmonica/)

MIDI，或称**M**usic**I**乐器 **D** 数字 **I** 接口，自 20 世纪 80 年代以来一直是计算机控制乐器的标准。它最常与电子乐器联系在一起，如合成器、鼓机或采样器，但当与适当的硬件结合时，它几乎可以应用于任何乐器。

[phearl 3s 1]通过将 MIDI 添加到最不可能的乐器，将这一点推向了极限。口琴可能看起来是模拟音乐的终极，然而他[创造了一个巧妙的 Arduino 驱动机制，在 MIDI 控制下演奏口琴](https://www.youtube.com/watch?v=SqS-cZ4B-Ic)。

口琴本身安装在一个抽屉滑轨上，该滑轨与一个从泳池清扫车上取下的轮子相连，由一个电机驱动，该电机可以通过一个提供位置反馈的电位计将乐器从一边移动到另一边，从而形成一个简单的伺服系统。空气供应来自一组三个风箱，通过另一个电机的曲柄驱动，通过一根看起来像 PVC 管的东西输送到口琴的商业端。

结果肯定是一个可演奏的 MIDI 口琴，尽管它没有抓住人类演奏乐器的本质。你们自己判断吧，他发布了一个构建视频，我们把它放在了休息区下面。
