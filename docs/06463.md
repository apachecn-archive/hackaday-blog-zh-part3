# Tinkercad 做 Arduino

> 原文：<https://hackaday.com/2017/07/11/tinkercad-does-arduino/>

如果你做过 3D 打印，你可能至少听说过 Tinkercad。这个流行的 CAD 软件包在你的浏览器中运行，几年前被 Autodesk 从遗忘中拯救出来。[Chuck]最近做了一个关于 Tinkercad 新功能的视频:[构建和模拟虚拟 Arduino 电路](https://www.youtube.com/watch?v=WOCRlmnMGRE)。可以在下面看。

有多种组件可以添加到您的设计中。你会发现一个集成的代码编辑器和一个调试器。您甚至可以访问串行监视器，所有这些都在您的浏览器中完成，无需实际的 Arduino 硬件。您也可以构建不使用 Arduino 的简单电路，尽管元件选择有些有限。

这对于在教室里教授 Arduino 或者当你想在酒店房间里做一些开发的时候是很棒的。布局非常直观，所以如果你习惯于阅读原理图，你可能不会欣赏这种风格。此外，组件的选择有些有限(据我们所知，只包括支持 Arduino UNO)。所以从教育的角度来说，这是很棒的。对于你的下一个伟大的 Arduino 驱动的机器人，可能没有那么多。

如果你还记得 Circuits123(或者 circuits.io)，这也是同样的底层技术。他们刚刚把它和 Tinkercad 整合在一起。然而，除了它们现在在同一个网页上之外，这两者之间似乎没有任何真正的集成。也许在未来，他们会让你在电路上放置同样出现在 3D 设计中的元件(或者，至少有这些元件的插座或支架)。

然而，即使你不关心电路模拟，拥有一个带调试器的模拟 Arduino 也会派上用场。如果真的要做电路仿真，用 [LTSpice](https://hackaday.com/2016/02/26/adding-spice-to-your-workbench/) 很难出错。如果你真的希望它出现在你的浏览器中，总会有[法尔斯塔德](https://hackaday.com/2015/07/20/a-breadboard-in-a-browser/)。
