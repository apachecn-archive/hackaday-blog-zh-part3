# 布拉德利·高斯罗普:关于布线你需要知道什么

> 原文:[https://hack aday . com/2018/01/12/bradley-gawthrop-what-you-need-know-on-wiring/](https://hackaday.com/2018/01/12/bradley-gawthrop-what-you-need-to-know-about-wiring/)

布线——就像将电线插在一起和压接连接器，而不是 Arduino IDE 之类的东西——是一个非常深奥的主题。我们都知道线路工人的拼接是将两根电线焊接在一起的最佳方式，NASA 的电缆和连接器指南要求阅读这些部分。然而，关于连接器和布线有很多可以说的，解释这一切的最佳人选之一是 Bradley Gawthrop。他花了十年时间建造管风琴，随之而来的是成千上万的继电器、螺线管、开关和阀门。所有这些部分由数千英里的电线连接，可以说像老式电话交换机一样复杂。如果你需要找人谈谈如何把成千上万的零件连接在一起，布拉德利就是你要找的人。

布拉德利以对现代原型制作过程的讨论开始了他的黑客日超级会议演讲。如今，我们离几十个芯片坐在一块有数据和地址线的试验板上已经很远了，现在任何类型的原型基本上都是一个开发板，周围布满了一系列模块。连接器的最佳解决方案是直角接头，不仅因为电线保持扁平，还因为直角连接器允许您探测从电路板出来的每一根电线。

当然，说到布线，说说电线本身是有帮助的。布拉德利只需要几个选项，而不是在他的车间上方悬挂一整个仓库的各种颜色、规格和绝缘材料的电线。现在，他只处理三种规格的电线——小、中和大，或者 24、18 和 12 AWG。一根线用于小信号，一根线用于小电流，一根线用于大电流。这不仅减少了车间库存，也意味着布拉德利只需要三种尺寸的压接钳和连接器。当谈到股数时，实芯焊丝被严重低估。不仅更容易剥离和卷曲，还可以支撑自身重量。这很重要，因为这意味着连接器不必承受整条电缆的重量。

如果你正在寻找运行电缆和压接连接器所需的最小工具集， [Bradley 在他的网站](https://www.the-accumulator.net/talks/)上有一个很棒的购物清单。他找到的最好的脱衣舞娘来自维哈，但他们已经被制造商停产了。不过，克尼派克斯是很好的脱衣舞娘。你不需要花大价钱买套圈压接钳，BangGood 的一些便宜货就足够了。布拉德利已经标准化了莫仕 SL 和莫仕 KK 互连，如果你有亚马逊 Prime，电线可以很容易地获得。

虽然布拉德利演讲的主题听起来很容易忽略，但在装配中将零件连接在一起本身就是一项关键技能。我们很高兴布拉德利能在 Hackaday 超级大会上与我们分享他的经验。
