# Hackaday 奖半决赛:行走机器人从零开始

> 原文：<https://hackaday.com/2015/09/17/hackaday-prize-semifinalist-walking-robots-from-scratch/>

通常教授机器人技术的方式——就此而言，几乎所有的东西——都很简单。教师得到一个预先构建的模块或工具包，教学生如何使用该工具包，然后下课。这有明显的缺点。[凯文·哈灵顿]参选 Hackaday 奖彻底颠覆了教育学。这是一个机器人开发平台，鼓励每个人从零开始创造自己的机器人，从这个问题开始，“你希望你的机器人有几条腿”。

Bowler Studio 使用 OpenCV 进行图像处理，它是一个运动学引擎，是一个基于 JCSG 的 CAD 和 3D 建模引擎，可以与电机连接，根据运动学模型创建 3D 模型，将成像数据提供给机器人，并为机器人创建图形界面。这是一个完整的机器人创作工作室，当然一切都可以备份到云。

电子中枢是[Kevin]和 Neuron Robotics 的另一个项目， [DyIO](https://hackaday.io/project/3185-dyio-dynamic-input-output-controller) ，这是一个 USB 外设，是一个伟大的机器人平台。DyIO 可以控制多达 24 个伺服系统，足以用于非常非常复杂的机器人，并且还具有控制电机、读取编码器或只是闪烁引脚的能力。

这两个项目一起为学习机器人的来龙去脉提供了一个很好的方法，这些机器人比简单的轮式机器人稍微复杂一点，并且足够扩展以制作一些非常非常酷的项目
