# 钢铁侠，穿铁！

> 原文:[https://hackaday.com/2017/02/09/iron-man-in-iron/](https://hackaday.com/2017/02/09/iron-man-in-iron/)

有时候，我们会遇到一个项目，它包含了太多的信息，让人不知所措，不知道从哪里开始。一个很好的例子是[Barry Armstead]的钢铁侠套装，我们是通过[一个跨越几年的很长的论坛帖子](http://forums.overclockers.com.au/showthread.php?t=1044707)被介绍的。

自制盔甲是 cosplay 世界的一个主要组成部分，粉丝们制作了许多令人惊讶的好作品。让【巴里】的钢铁侠套装鹤立鸡群的是它的构造；他用的不是玻璃纤维或真空成型塑料，而是真正的金属。(是钢的。但是钢是含铁的吧？我们称之为诗意的许可。)

追踪套装进展的最好地方可能是[Barry]的 YouTube feed，到目前为止，他已经积累了 44 条构建日志。我们看到[关节连接](https://www.youtube.com/watch?v=tNqiiUyEZCc)测试、[早期测试行走](https://www.youtube.com/watch?v=V5fQqofiNoY&spfreload=10)、[标志性头盔](https://www.youtube.com/watch?v=QvOYnNTSiBM)成型，以及用纳米喷雾器模拟的反重力装置[。有这么多视频要看，你会在那里呆上一段时间。下面我们将留给你的是相当简单的，第一次看裸露金属的整个外骨骼。](https://www.youtube.com/watch?v=9P1ILq6o6-c)
