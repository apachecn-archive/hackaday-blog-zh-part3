# 通过 Proxxon 的工具系列升级到 CNC

> 原文：<https://hackaday.com/2016/07/31/escalating-to-cnc-through-proxxons-tool-line/>

Proxxon 主要是德国制造商，生产高于平均水平的微型工具。他们确实出售各种口味的微型铣床，从手动到全数控。[Goran maho vli]不相信这一点。然而，他确实[将他们的旋转工具附件目录整合到一台数控铣床上](http://www.lemilica.com/just-another-proxxon-cnc-conversion/)。

拥有工具是危险的。一旦开始，就真的没办法停下来。这一点用戈兰的数控机床看得很清楚。起初，他的快乐是一个小型高速旋转工具。他用它在电路板上钻孔。

可想而知，他发现用手在印刷电路板上钻小孔是一件单调乏味的事情，而且最终会令人厌烦。所以他为他的旋转工具购买了钻床附件。

有段时间生活很美好。他有他需要的所有工具，但是…如果他能更快地定位孔不是更好吗？他大概翻看了一份已经破损的 Proxxon 目录，订购了 XY 工作台。

恍然大悟。拉动控制杆并转动旋钮！为什么！这是机器人的工作，不是人的！因此，他缠着他的同事寻求帮助，他们很快就在 CNC 的控制下完成了这个装置。

我们想说这就是结束了，Goran 终于开心了，但是他最近把他的 frankenmill 改装成了 3D 打印机。我们以前见过这种情况。过不了多久，他就会清理他的车库，开始修复一台[旧膝磨](http://hackaday.com/2011/11/13/rebuilding-a-bridgeport-mill/)并最终[数控转换](http://hackaday.com/2012/01/24/converting-a-mill-to-cnc/)。休息后的视频。
