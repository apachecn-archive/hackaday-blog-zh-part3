# 用 3D 打印机制作电子原型

> 原文：<https://hackaday.com/2017/01/25/electronic-prototyping-with-a-3d-printer/>

如果你的 3D 打印机能吐出 PC 板就好了。已经做了很多工作来实现这一点，主要集中在沉积导电材料上，尽管我们很惊讶没有人知道如何 3D 打印塑料抗蚀剂掩模。

我们最近为[PCBPrints]找到了一个 GitHub 组，它有[个小模块，在原型和试验板](https://github.com/PCBPrints)中会很有用。它们实际上只是为开关、发光二极管等创建插件模块的载体。看起来这是实现这些设计的其他 GitHub 项目的汇总列表。该小组是在西班牙语，但谷歌翻译是你的朋友，一如既往。你可以在下面看到一个按钮模块的视频。

存储库中有一个通用的原型板计算器。当然，在塑料附近焊接时可能要小心，尤其是像 PLA 这样熔点较低的塑料。虽然仓库称这些为 PCB，但没有导电部分。它们实际上只是包装和机械衬底。尽管如此，它们看起来非常整洁，我们可以想象它们在许多快速项目中非常方便。

人们用 3D 打印机制作 PCB 的另一种方法是将它们用作绘图仪，在板上绘制保护层。或者你可以[减去并去掉抗蚀剂](https://hackaday.com/2015/06/29/etching-pcbs-with-a-3d-printer/)。除了[使用 Ninjaflex](http://hackaday.com/?p=239878) 之外，我们还没有见过用打印机沉积抗蚀剂的方法。
