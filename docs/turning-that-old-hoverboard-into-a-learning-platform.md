# 把旧悬浮滑板变成一个学习平台

> 原文：<https://hackaday.com/2018/06/03/turning-that-old-hoverboard-into-a-learning-platform/>

[Isabelle Simova]正在构建一个灵活的机器人平台 [Hoverbot](https://hackaday.io/project/158256-hoverbot) ，它使用宜家塑料托盘、运行在 Raspberry Pi 上的 JavaScript 和从常见悬浮板上收集的零件。

自动平衡滑板车也叫悬浮滑板，是这个项目的一个很好的零件来源。他们的高扭矩，直接驱动无刷电机可以驱动 100 公斤或以上的负载。此外，你还得到一个匹配的电机控制器板，一个可充电电池及其充电电路。大多数悬浮滑板控制器使用 STM32F103，因此使用 ST-link V2 编程器用您自己的固件刷新它们变得很容易。

建造机器人需要的下一组部件是传感器。有些便宜且容易获得，如麦克风、接触开关或激光雷达，而其他如超声波距离传感器或激光雷达的成本可能要高得多。廉价传感器的一个来源是停车辅助传感器。配件市场停车传感器套件通常由四个传感器、一个控制箱、电缆和显示器组成。通过使用逻辑分析仪，[Isabelle]展示了如何在控制盒的输出端口周围摸索，以对数据流进行逆向工程并破译传感器数据。一旦数据结构被解码，您就可以使用一些 SPI 位碰撞和电压转换将其与 Raspberry Pi 接口。使用 Pi 可以很容易地为 Hoverbot 添加廉价的网络摄像头、麦克风和扬声器。

宜家是黑客的最爱，提供各种黑客友好的设备和用品。他们的产品目录提供了一系列优良的瑞典工程产品，可用作建筑机器人的外壳。[Isabelle]瞄准了一个深的圆形塑料托盘，这是一套储物桌，用胶合板加固过。托盘提供了足够的空间来安装两个电机，两个脚轮，电池和其他电子设备。将悬浮滑板的大部分原始硬件组合在一起时会很方便。

将所有这些结合在一起的软件粘合剂是 JavaScript。Node.js 的事件驱动架构使其成为非常适合用于 Hoverbot 的框架。[Isabelle]开发了一个基本的应用程序，可以远程控制机器人。它包括一个显示机器人实时视频和音频流的仪表板，移动控制按钮，用于将文本转换为语音的输入框，超声波传感器可视化，LED 照明控制，消息日志和电机状态显示。这使得仪表板成为一个有用的调试工具，也是构建更有趣的应用程序的起点。查看构建日志，了解所有有趣的细节。宜家《家居指南》中还有哪些产品可以用来制作气垫机器人？机器人椅怎么样？
