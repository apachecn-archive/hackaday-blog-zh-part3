# 随风奏乐

> 原文:[https://hackaday.com/2017/10/21/making-music-with-the-wind/](https://hackaday.com/2017/10/21/making-music-with-the-wind/)

[尼克拉斯·罗伊]为他的后院建了一个风车驱动的音乐盒，它太棒了，所有的邻居都想给它拍张照。甚至有人喜欢到半夜偷了【尼克拉斯】的风车。(我们不怪他们，这是一个非常干净的建筑。)

在过去的几周里，[Niklas]已经为将于 11 月在比利时那慕尔举行的 2017 年 KIKK 节批量生产了 20 台风车。风车将以集群的形式运行，当风吹来的时候，所有的风车都会发出“Für Elise”的声音。然而，每一个都是独立驱动的，所以音乐是异步的。既然他已经造了一堆，他就为他的后院造了一个替代的风车，并记录了如何做。

大部分机械零件是 3D 打印的，轴承等金属五金件是塑料打印的补充，M10 螺纹杆充当桅杆。他提供了切割“旗帜”的 PDF 模板。在他的指导视频中，Niklas 使用了双层的 coroplast(可回收的政治标志),用双面胶粘在一起；他在他的装置中使用了 3 毫米外汇。此外，还有一个由一些更硬的塑料制成的矩形——他随意放在周围的东西——作为音乐盒的共鸣器。

[Niklas]是柏林的装置艺术家和教育家，也是 Hackaday 的常客；他的[音乐施工机](https://hackaday.com/2016/07/05/niklas-roys-music-construction-machine/)
和他的[壁挂式风扇钻机](https://hackaday.com/2017/09/10/winter-is-coming-keeping-heat-where-its-needed/)是任何小作坊的必备。
