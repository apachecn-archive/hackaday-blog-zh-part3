# 定制媒体播放器帮助黑客的自闭症儿子

> 原文：<https://hackaday.com/2016/04/18/custom-media-player-helps-hackers-autistic-son/>

玩技术通常是黑客进行构建的唯一理由。但是，当你的构建帮助了某人，尤其是你自己有特殊需要的孩子时，黑客就变得不仅仅是玩了。这就是为建筑商患有自闭症的儿子定制的这个媒体播放器的背后。

人们普遍知道自闭症的症状涵盖了以社交和交流为中心的广泛的行为和特征。但是自闭症谱系障碍的一个重要组成部分是孩子们通常表现出非常有限的兴趣。虽然[Alain Mauer]没有谈到他儿子[Scott]的症状，但我们猜测这个媒体播放器是一种吸引他兴趣的方式。当[Alain]无法找到一款既能让他的儿子操作简单又能忍受一些虐待的商用媒体播放器时，这款产品应运而生。一个树莓皮来拯救，随着一些自定义压电控制按钮的帮助下，一个彩色的情况下，和 Shin Chan。该界面允许[斯科特]滚动浏览卡通菜单，并在大型演出前获得预览。[Scott]在下面的视频中满面笑容，我们打赌[Alain]也是如此。

基于 Pi 的媒体播放器构建在 Hackaday 上多如牛毛，但帮助自闭症儿童的构建却非常特别。事实上，我们只展示了几个针对自闭症的项目，比如这个 [2015 年黑客日奖参赛作品](http://hackaday.com/2015/09/10/hackaday-prize-semifinalist-a-device-for-autism-and-pomodoros/)，这令人惊讶。也许你可以为 2016 年的 Hackaday 奖设计出类似[Alain]的造型。
