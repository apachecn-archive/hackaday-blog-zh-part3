# 本周失败:没有升级的点焊机

> 原文：<https://hackaday.com/2018/04/22/fail-of-the-week-the-spot-welder-upgrade-that-wasnt/>

即使你做了非常非常好的东西，也总有改进的空间，对吗？事实证明[这次尝试升级为 DIY 点焊机](https://youtu.be/0N3l6XUqUMY)，没有那么多。

毫无疑问，你会记得我们前段时间展示的[马克·普雷斯林]的[非常精致和专业的点焊机制造](http://hackaday.com/2018/03/07/not-just-your-average-diy-spot-welder/)。这是一件漂亮的衣服，不仅在合身性和装饰上，而且在功能上，都花了很多心思和精力。尽管如此，[马克]并不满意；他觉得电焊机有点动力不足，重绕的微波炉变压器噪音太大。他从一个旧的工业点焊机中获得灵感，决定通过将通常用作次级线圈的双回路电池电缆替换为单回路厚铜材来重建变压器。然而，由于缺少合适尺寸的棒，他在形成环之前将多个薄铜片层压在一起。理论上，新的次级线圈的更高的横截面积应该承载更多的电流，但实际上，他在焊接电流或他的结果中看不到任何差异。不过，这也不全是坏消息——焊机现在几乎没有声音了，被替换的次级绕组可能是一个安全问题，因为电缆绝缘已经开始融化。

鉴于[马克]对细节的明显关注，我们毫不怀疑他会再次处理这个问题，并最终解决这个问题。你有什么建议？升级哪里出问题了？是使用了层压次级而不是实心棒料吗？或者这是 MOT 所能做到的最好的了？请在评论区留言。
