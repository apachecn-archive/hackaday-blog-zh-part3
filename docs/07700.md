# 一个有秘密的梦幻 VMU

> 原文：<https://hackaday.com/2017/11/14/a-dreamcast-vmu-with-a-secret/>

自从 Raspberry Pi 系列主板在 2012 年首次出现以来，我们已经看到它们被巧妙地集成到许多创新的外形中。今天，我们为您带来这一领域的最新产品，[Kite]的 [Raspberry Pi Zero W 安装在世嘉 Dreamcast VMU](http://www.sudomod.com/forum/viewtopic.php?f=13&t=4613) 的外壳中。其结果是一个特别好的构建，其中 Pi 的几个更大的组件被移除或替换为低姿态的替代品，位于定制 PCB 的相对侧，远离小型 LCD 显示器。

PCB 包含相关的按钮、音频和电源电路，当安装在 VMU 外壳中时，可以成为真正专业质量的微型手持控制台。在一个特别好的触摸中，Pi 的 USB 连接与 SD 卡一起出现在 Zero 的末端，在原本保护 VMU 连接器的盖子下面。一些最低限度的剥离世嘉塑料是必要的，但令人惊讶的是，情况是未经修改的，并有足够的空间来容纳一个体面的大小的电池。

VMU，或称[视觉记忆单元](https://en.wikipedia.org/wiki/VMU)，是一个有趣的外壳选择，因为它是主机游戏死胡同之一的遗迹。这是世嘉最后一次进军游戏机市场 Dreamcast 的存储卡，与竞争对手不同的是，它形成了一个独立的小型手持游戏机。VMU 平台的小游戏与完整游戏捆绑在一起，有一个简单的多人游戏系统，VMUs 可以连接在一起。可悲的是，Dreamcast 在 20 世纪 90 年代末和 21 世纪初的游戏机大战中输给了索尼的 PlayStation 2，但它仍然是一款值得注意的游戏机。

vmu 不是最常见的游戏幸存者，但我们已经向您展示了一两个使用它们的项目。早在 2010 年就有了 iPod 的改装[，最近](https://hackaday.com/2010/09/29/dreamcast-vmu-meet-ipod/)[又有了一些令人兴奋的 VMU 原始硬件的逆向工程和仿真](https://hackaday.com/2017/07/17/completely-owning-the-dreamcast-add-on-you-never-hadcompletely-owning-the-dreamcast-add-on-you-never-had/)。
