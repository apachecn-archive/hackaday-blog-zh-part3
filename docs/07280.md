# NIST 利用光学共振探测原子

> 原文：<https://hackaday.com/2017/10/02/nist-uses-optical-resonance-to-probe-atoms/>

你有没有站在穹顶下轻声细语，却听到自己声音的回声传回来大得多？NIST 大学的研究人员利用类似的原理改进了原子力显微镜，使他们能够比以往更精确地测量微观物质的快速变化。

原子力显微镜的工作原理是使用一个极小的尖探针。该仪器通常使用压电传感器或激光传感器来检测探针的偏转。通过将探针移向表面并测量传感器的输出，显微镜可以形成表面的轮廓。NIST 研究小组使用了一束通过调谐到特定频率的圆形波导管传播的激光。波导非常接近(150 纳米)一个非常小的探针，重约万亿分之一克。当探针移动一点点时，它会导致波导的特性发生很大程度的变化，而监控通过谐振器的激光的光电探测器可以检测到这一点。

原子力显微镜上的探针尖端——被称为悬臂——是一种特殊的纳米结构。探针的尖端通常只有几个原子宽。显然，悬臂越小越轻，仪器的响应就越灵敏，但通常越难读取其输出。偏转可能来自物理接触或探头敲击表面。原子力显微镜也可以测量原子力。当探测器接近一个表面时，原子尺度的引力会把它拉得更近。但是当它靠得太近时，排斥力会把它推开。这些是 AFM 用来绘制探针下表面的偏转。
