# 机械师用磁铁

> 原文：<https://hackaday.com/2016/11/02/magnets-for-a-machinist/>

我们不确定[Stefan Gotteswinter]是否会在他的店里制造工具以外的东西来制造更多的工具。这套精美的工具制造商磁铁也不例外。

我们可以推断[斯特凡]的职业是职业机械师。像所有为了工作和娱乐做同样事情的专业人士一样，他被工作中更好的工具宠坏了。有一种工具很特别，那就是工具制造商的磁铁，总是很方便。这些强磁体[已经被磨平、磨方、磨平行](http://hackaday.com/2016/10/25/sometimes-square-is-square/)。

他真的只需要一个磁铁，所以他开始做一个 20 x 20 x 100 毫米的。它将由软钢和黄铜交替制成。钢板上会钻一个洞，他会在中间放置一个方向正确的磁铁。它会被夹紧并粘在一起。

当他决定不能只信任胶水时，构建进行得相当顺利。他刚刚开始研磨，但决定转向快速钻孔作业。穿过整个组件的两根黄铜棒足以将它固定在一起。他开始钻孔，然后，突然，他有了两块磁铁。

集会被分成两半。他决定，综合考虑所有因素，两个 20 x 20 x 50 mm 的磁铁也很方便。于是他完成了钻孔，并将新的一组磁铁接地，使它们彼此完美匹配。最终，他有了一个看起来和商业选择一样昂贵的工具。还有一个关于磁铁的视频系列，[第一部分](https://www.youtube.com/watch?v=HBhnRvCnIcs)和[第二部分](https://www.youtube.com/watch?v=Marc0h7sGLQ)，休息后可以观看。
