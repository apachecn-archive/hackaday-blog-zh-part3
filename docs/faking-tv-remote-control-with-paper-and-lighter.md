# 用纸和打火机伪造电视遥控器

> 原文：<https://hackaday.com/2017/02/07/faking-tv-remote-control-with-paper-and-lighter/>

在一张纸上剪出代表红外遥控器比特流的槽，用于将电视置于待机状态。将它插入电视的红外接收器和打火机的火焰之间，拉动插槽以生成编码图案。弄得恰到好处，你就有了一张[纸和一个更轻的遥控器](https://www.youtube.com/watch?v=S5BE6IVeBbE)。这正是[ViralVideoLab]所做的，你可以在下面的视频中看到它的作用。

把这想象成一个想法的萌芽。想象一下，您将如何自动化并扩展它以包含更多的命令。一个圆周上切割有各种比特流的轮子浮现在脑海中。一个伺服系统会把轮子转到想要的指令，当插槽经过时，其他东西会点燃打火机。现在再深入一点。您已经有一个带键盘和红外线灯的遥控器。黑掉它，与控制伺服系统和红外线灯的微控制器对话。给你。一个没用但很好玩的黑客(提示提示)。

在你哭泣无用之前。黑客对制造无用的机器并不陌生，只要这是一个有趣的黑客行为。就拿这个[世界上最大的无用 AI 机器](http://hackaday.com/2016/09/23/worlds-biggest-most-useless-ai-machine/)来说吧。或者这个[隐形眼睛图书扫描仪](http://hackaday.com/2016/08/01/the-most-useless-book-scanner/)怎么样。或者看经典的无用机器[会自动关机，但有八个开关](http://hackaday.com/2012/12/22/an-even-more-useless-machine/)，这是一种无穷无尽的乐趣。
