# 带 ESP32 的劳拉

> 原文：<https://hackaday.com/2018/06/24/lora-with-the-esp32/>

如果您对部署低功耗远程无线技术 LoRa 感兴趣，您可能会喜欢[Rui Santos 的]项目和视频，了解如何使用带有 Arduino IDE 的[ESP32 来实现 LoRa。](https://randomnerdtutorials.com/esp32-lora-rfm95-transceiver-arduino-ide/)可以看下面的视频。他使用带有分线板的 RFM95 收发器，因此即使您想使用不同的处理器，您仍会找到许多有用的信息。

事实上，视频只是 LoRa 上的背景，不管你使用的是什么主机，它都不会改变。一旦你有了所有的部件，让它工作就相当简单了。[Sandeep Mistry]有一个 LoRa 库，它知道如何完成大部分工作。

虽然该项目使用 RFM95，但它也可以与 RFM96W 或 RFM98W 等类似模块一起工作。也有 ESP32 模块具有兼容的板载收发器。

这是其中一个项目，它本身可能没什么用，但是它真的可以帮助你克服当你开始使用新东西时经常遇到的困难。一旦你有了演示设置，它应该很容易变异成你真正需要的。

我们最近一直在谈论劳拉。我们甚至见过[指挥无人机](https://hackaday.com/2018/05/05/lora-system-commands-drones-from-a-distance/)。
