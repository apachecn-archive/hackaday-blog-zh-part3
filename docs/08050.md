# 厨房材料真空成型

> 原文：<https://hackaday.com/2017/12/26/vacuum-molding-with-kitchen-materials/>

真空泵是强大的工具，因为地球表面的大气压力很大。这种压力足以用惊人的内爆力压碎真空容器。在不太极端的压差下，【hopsenrobsen】向我们展示了如何巧妙利用厨房材料进行[真空成型玻璃纤维部件](https://www.youtube.com/watch?v=7_rH7IbPvq8)在休息后的视频中可以看到。同样的技术也适用于碳纤维成型。

我们已经看到了这些技术与商用真空袋和湿/干真空吸尘器一起使用，但在视频中，我们看到了如何将一个普通的垃圾袋制作成一个能够形成专业外观的长板电池盖的容器。如果垃圾袋还不够，可以用一个钢丝球来防止垃圾袋干扰空气软管。我们中的一些人把这些普通的厨房材料放在同一个橱柜里，所以收集它们应该不成问题。

环氧树脂应根据说明混合，尽管视频中没有显示，但一些环氧树脂需要呼吸器。如果你不确定，就穿一件。肺很重要。

玻璃纤维部件不仅仅是功能性的，它们也可以很漂亮。如果塑料是你的障碍，[真空也形成了](https://hackaday.com/2010/12/02/vacuum-forming-at-home/)那些部分。如果你只是为了吸尘器而来，那么 Roomba 上的 [MATLAB 怎么样？](https://hackaday.com/2017/11/23/bringing-matlab-to-a-vacuum-near-you/)
