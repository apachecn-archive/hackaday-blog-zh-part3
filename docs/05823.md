# 机器人钟琴和黑客硬盘制造音乐

> 原文：<https://hackaday.com/2017/05/02/robotic-glockenspiel-and-hacked-hdds-make-music/>

【bd594】喜欢做奇怪的东西。这次是一个[机器人钟琴和被黑掉的硬盘](https://www.youtube.com/watch?v=MN56LjjbB8k)。【BD 594】对 Hackaday 也不陌生，因为我们已经展示了他过去的许多项目，包括[无用的蜡烛](http://hackaday.com/2014/06/19/this-useless-machine-now-plays-with-fire/)或从旧垃圾中重现歌曲[时髦小镇。](http://hackaday.com/2008/07/13/funkytown-performed-by-old-hardware/)

他的最新项目相当令人兴奋。他将他的[机器人钟琴](http://hackaday.com/2015/04/12/robotic-glockenspiel-crunches-popcorn/)与一个被黑掉的硬盘节奏部分结合在一起，通过 PIC 16F84A 微控制器来播放音频。选择的歌曲是 Axel-F。如果你在 21 世纪初有一部手机，你几乎肯定会在某个时候用这首歌作为手机铃声。无论如何，这是音乐发展的方向；我们越早用机器人取代贾斯汀比伯这样的人越好。或者也许[我们已经有了](http://hackaday.com/2015/11/07/compressorhead-best-robot-band-ever/)？
