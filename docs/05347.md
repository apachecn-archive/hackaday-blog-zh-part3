# 自定义 Clicky 快捷键键盘

> 原文：<https://hackaday.com/2017/03/17/the-custom-clicky-shortcut-keypad/>

你不酷，除非你有一个机械键盘。举个例子:如果你想办法买一个和我用来打字的键盘一样的键盘，至少要花掉你 700 美元。没错，是机械(Topre)，没错，我比你酷。当然，你不能像我一样酷，但你可以打造自己的机械键盘。我认为[罗宾]是一个很酷的人，所以他自己做了一个键盘。它是令人惊奇的快捷键盘，它可以图形化编程。

这个键盘的想法是在[罗宾]作为一名工程师学习的时候产生的。我们假设这是在 AutoCAD 上磨损 Escape 键的代码，但是许多其他软件包也有同样的问题。解决[Robin]问题的方法是一个快捷键盘，一个 3×4 的樱桃开关矩阵，可以为任何任务编程。

这个键盘的设计最初是一个 [Adafruit Trellis 矩阵键盘](https://learn.adafruit.com/adafruit-trellis-diy-open-source-led-keypad/overview)。这与为每个按钮分配宏指令的处理软件结合在一起。这是一个足够的解决方案，但是阿达果棚架上的开关*看起来*软绵绵的。对于那些渴望指尖轻点的人来说，这些开关并不合适。它不是那些渴望激光蚀刻 PBT 键帽的微妙厚度的人的键盘。Adafruit 键盘没有一套完整雕刻的键帽的优美线条。哦，我的上帝，这是双镜头。

[Robin]完成的键盘经历了几次修改，但最终，他决定采用 PCB 安装的开关和一个非常聪明的 3D 打印支架系统来固定 Arduino Pro Micro。外壳也是 3D 打印的，最终结果是一个完全定制的键盘，非常适合混合按键组合。
