# 用现代方式喂养你的猫

> 原文:[https://hack aday . com/2017/12/09/feed-your-cat-the-modern-way/](https://hackaday.com/2017/12/09/feed-your-cat-the-modern-way/)

喂猫应该是一个神奇的时刻，当你可爱的宠物挑逗地摩擦你的脚踝时，你和它亲密接触。当你把一碟柔软湿润的大块肉放在地板上时，她跳过了你的厨房，兴奋的表情出现在她的小猫脸上，尾巴翘在空中。

如果 Hackaday 为猫粮做电视广告，我们会搞定的。但是在我们的日常生活中，可爱的哈卡戴猫在决定喂食的时候会变成一只顽固的小害虫。[ThinkSilicon]的朋友们也有同样的问题，他们毛茸茸的朋友喜欢在清晨到来。他的解决方案？一个自动喂猫器 ( [翻译为](https://translate.google.com/translate?sl=auto&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=https%3A%2F%2Fthinksilicon.de%2F83%2FAutomatische-Katzenfuetterung-CatFatter.html&edit-text=&act=url))将狗粮从漏斗分配到幸运老鼠的喂食盘里。

这项努力的机械部分非常简单，一个伺服移动一个滑动的胶合板，上面有一个洞，穿过装满猫粮的料斗底部。移动滑梯，将食物从滑道分配给等待的快乐猫咪。幕后是一个 ESP8266 和一个 NodeMCU web 服务，通过它可以安排喂食时间，也可以随意分配。

快乐的猫意味着快乐的主人，尤其是在清晨。在那之前，这个刚刚满足的生物决定传播爱，跳上主人的床表示感谢，并对着他们的脸呼吸猫粮。你真的必须爱他们！

我们已经向您展示了许多过去与猫粮相关的项目，包括[这个 Arduino 采用相同的想法](https://hackaday.com/2009/06/24/wireless-arduino-cat-food-dispenser/)。但是，当[猫可以为你](https://hackaday.com/2016/09/01/cat-operated-cat-food-dispenser/)触发它的时候，为什么要自己费力去触发它呢？
