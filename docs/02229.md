# 惊人的示波器图形

> 原文：<https://hackaday.com/2016/05/02/amazing-oscilloscope-graphics/>

据我们所知，[ompuco]已经在 Unity 游戏引擎上建立了一个 2D 音频输出，使他能够在 XY 模式下将 X 和 Y 值从他的立体声声卡直接输出到示波器。他的代码只是扫描场景中的所有顶点，并将正确的电压输出到左右音频流中。他用这个创造了一些非常不可思议的动画。请看下面的视频“添加剂”中的例子。(看你能不能搞清楚“添加”了什么。)

作为第一个演示应用，【ompuco】编写了一个示波器绘图应用: [electricanvas](https://ompuco.itch.io/electricanvas) ( [演示视频](https://www.youtube.com/watch?v=AGeHwNEwbZk))。你画一些线，然后电子管把它们转换成声音，然后送到示波器上。看看他的[金字塔演示](https://ompuco.itch.io/vector-pyramid-generator)。

这项工作做得非常好，以至于他启发了另一个论坛用户[Pishtaco]来开发他自己的[磷光矢量显示器模拟工具](https://dl.dropboxusercontent.com/u/29654821/oscilloscope%20demo/oscilloscope.html)，它看起来也非常可爱。如果你身边没有旧的荧光镜，这是下一个最好的选择。至少听音乐挺好玩的；45 度线表示左右声道相似。只有一个频道的 Fuzz 吉他制作了一个漂亮的毛球。试试看。

我们对 Unity 一无所知，我们很想看看[ompuco]的一些线框代码，因为这些东西看起来太棒了！他给我们回信说他会在清理干净后发布。我们会保持联系。
