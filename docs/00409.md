# 1968 年风格的 DIY 电脑

> 原文：<https://hackaday.com/2015/10/19/diy-computer-1968-style/>

“自己造电脑”是什么意思？今天，它很可能意味着你买了一个主板，一个电源，一个盒子，然后把它们放在一起。你甚至可以用一些芯片，包括现成的 CPU，制造一台嵌入式计算机。然而，有些人(像我一样)已经使用 FPGAs 构建了完整的计算机，有些人(不像我一样)已经使用 TTL 芯片、分立元件甚至继电器构建了计算机，我们已经讨论了其中相当多的部分。

这并不总是那么容易。组件现在很容易获得，而且相对便宜(特别是考虑到通货膨胀)。在 20 世纪 60 年代，简单部件的价格比你今天支付的价格还要高，那时你假想的自己赚的钱更少。几乎从所有能想到的方面来看，成本都高得惊人。

那么，如果你是一个在 1968 年从送报路线上攒钱的孩子，你想造一台电脑，你会怎么做？也许你会求助于[](https://archive.org/details/howtobuildaworkingdigitalcomputer_jun67)一本由【Edward Alcosser】、【James Phillips】和【Allen Wolk】于 1968 年出版的书。这本书正如标题所承诺的那样:你可以建造一台工作的数字计算机。不过，这些部件是回形针、锡罐、线轴和其他家用物品。你必须使用的唯一真正的电子元件是灯泡和电池，尽管你也可能在某些地方使用商店购买的开关，而不是书中显示的自制版本。

正如你所料，用这样的东西来建造一台计算机需要一些独创性。程序存储器是一个盖着纸的锡罐。刷子可以感应到纸上有洞的地方，每个洞的位置都是一个编码指令(非常类似于穿孔卡片，但被包裹在一个圆筒里)。编码器使用空线轴，用线和纸面具包裹。回形针形成了刷子，木螺丝形成了指针，这样你就可以告诉编码器你拨了哪个数字(见右图)。

唯一的妥协？控制单元中的一些灯和一些开关之间的数据传输由您决定。也就是说，当某盏灯打开时，你应该在继续之前将灯的图案复制到一组开关中。有 10 条指令和 16 个单词的内存，所以这是一个非常简单的机器，但没有任何活动设备(如果你不包括你自己的话)，主要由废料制成，这令人印象深刻。

你不得不怀疑到底有多少是真的造出来的。有一个商业版本被称为 CT-650(每台 1000 美元，我们怀疑有多少被售出)。此外，还有几个已知的完整版本，包括一个有几个 YouTube 视频的版本(见下文)。一个[高中数学俱乐部在 1975 年建造了一个](http://www.smecc.org/wickenburg_high_school.htm)(图片在本文顶部)，同样，[的两个九年级学生在 1972 年建造了一个](http://www.apparent-wind.com/mbr/emmerack.html)(售价约 150 美元；但别忘了，在 1972 年，你可以花大约 2000 美元买一辆福特平托，150 美元几乎可以支付你平均一个月的租金。

你可以说时代变了，回形针电脑已经过时了。也许吧。但是我不禁想到，如果你建造了其中的一个——尤其是在你还是个孩子的时候——你会比仅仅通过运行一些 Javascript 或者在一个树莓 Pi 上建立一个网络服务器更深入地了解一台计算机实际上是如何工作的。
