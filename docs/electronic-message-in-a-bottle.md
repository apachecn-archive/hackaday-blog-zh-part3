# 瓶子里的电子信息

> 原文：<https://hackaday.com/2016/09/26/electronic-message-in-a-bottle/>

我们记得去过祖父的车库。他就在那里，牙缝里咬着烟斗，当他专注于他的另一个作品时，一缕一缕的烟雾在他周围的空气中飘荡。在一个简单的玻璃瓶里是不可能的。小心翼翼地，非常小心翼翼地，他会用他定制的工具来拧电线。他会小心地放置每一根导线。最终时机成熟时，他会焊接。最后他会把它放在架子上，和其他的放在一起，一个 LED 矩阵放在一个瓶子里。

[![led-message-in-a-bottle-assembly](img/e27095ae455bf67b7e76ed35e81dea07.png)](https://twitter.com/kosamari/status/779459358119882753) 嗯，也许不是，但是【真理子小坂】的父亲【小坂公夫】做到了。为了构建这个矩阵，他需要一些工具，这些工具既能伸入瓶口内部，又不会占用太多空间来进行精确的移动。为了做到这一点，他将钢琴丝弯曲、铜焊、扭曲并锉成工具，这些工具本身就非常漂亮。这些用来小心地弯曲和定位瓶子内的发光二极管、电线和其他组件。

一旦零件准备好了，他就用改良的 Hakko 烙铁做最后的组合。我们想知道他是否不得不小心地快速焊接，以免在瓶子内部堆积残留物？所有的电子设备都装在瓶子里。其中一个瓶子里装着他的另一个令人印象深刻的发明:一个只有电线的完整 Arduino，被称为 Arduino 骨架。电池安装在软木塞上，当电量不足时，可以取出并更换，不会影响创作。

这是一种可笑的爱的劳动，自然地，我们喜欢它。有一个关于操作的视频以及一个他向[展示如何操作的视频](https://www.youtube.com/watch?time_continue=380&v=Z71gvQQO90I)，在休息后可以看到。他在东京创客博览会上展示了它们，在那里它们肯定会大受欢迎。
