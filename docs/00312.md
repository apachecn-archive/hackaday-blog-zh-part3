# 微型 PIC 时钟不是微型炸弹

> 原文:[https://hack aday . com/2015/10/13/tiny-pic-clock-is-a-tiny-bomb/](https://hackaday.com/2015/10/13/tiny-pic-clock-is-not-a-tiny-bomb/)

学生艾哈迈德·穆罕默德(Ahmed Mohamed)的一项发明被学校和警方误认为是炸弹，尽管该装置显然是一个时钟，但这一事件已经过去几周了。我们要求提交您所有的时钟构建，以显示我们对艾哈迈德的支持，[最新的一个是迄今为止最小的](http://dangerousprototypes.com/forum/viewtopic.php?f=56&t=7472)，但仍然具有全尺寸时钟的所有功能(没有一个是爆炸的)。

[Markus]的微型时钟使用 PIC24，这是一种小巧但功能强大的芯片。计时是在 RTCC 外围设备上完成的，当用户按下按钮时，时钟的七段显示会暂时亮起。由于发光二极管并不是一直亮着，PIC 待机时只消耗几微安的电流，所以时钟可以在背面的小型锂离子电池充电一次后运行多年。还有一个光电晶体管可以在黑暗中调暗显示屏，还有一个白色 LED 可以在必要时用作小手电筒。如果这些功能和构建技术看起来很熟悉，那是因为他去年展示的[Markus '][微型 MSP430 时钟](http://hackaday.com/2014/06/30/a-tiny-bubble-display-alarm-clock/)。

他的两个微型时钟在尺寸、功能和功耗方面都令人印象深刻。我们最近推出的其他一些时钟包括[机器人时钟](https://hackaday.com/2015/10/06/robot-clock-writes-time-over-and-over-and-over/)、[社会公益时钟](https://hackaday.com/2015/09/22/clocks-for-social-good/)，以及[不仅仅是时钟](https://hackaday.com/2015/09/19/nixie-tube-clock-isnt-just-a-clock/)(但仍然不会爆炸)。我们是一个好的时钟项目的吸盘在这里，所以继续发送他们！
