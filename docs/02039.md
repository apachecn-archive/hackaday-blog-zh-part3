# 俄罗斯方块无处不在:字符液晶版

> 原文:[https://hack aday . com/2016/04/09/Tetris-everywhere-character-LCD-edition/](https://hackaday.com/2016/04/09/tetris-everywhere-character-lcd-edition/)

廉价的字符液晶显示器比我们想象的更加通用。大多数便宜货都有一个 5×8 字符的显示屏，看起来有些笨重，但如果你有合适的字体，还是可以看清楚的。有趣的是，大多数液晶显示器还允许你上传自定义字符。

将这一点发挥到极致，[numeric] [滥用用户定义的角色来编写一个小型的俄罗斯方块游戏](https://hackaday.io/project/10761-tetris-on-arduino-lcd)，当你将四个角色组合在一起时，它将在 10×16 的框架中运行。它很小，是单色的，不玩三驾马车主题(这可能是好事)，但可玩。看看下面的视频。

[numeric]也勇敢地将自己的代码(ZIP 文件)放到了网上。退一步说，这很粗暴——他在一个周末这样做只是为了好玩。但是在你跳到他的头文件中寻找 C 代码之前，请注意这对于快速破解来说是非常酷的，而且 Arduino 平台在让初学者进入编码方面做得很好，但它并没有教他们如何正确地做事。我们希望进入我们自己的编码的第一步看起来很酷。

这个*是* Hackaday，我们之前报道过几个俄罗斯方块游戏。如果液晶显示器对你来说太高科技，可以考虑 DIY LED 矩阵上的[俄罗斯方块。如果太小了，摩天大楼上的](http://hackaday.com/2015/01/21/arduino-tetris-on-a-multiplexed-led-matrix/)[俄罗斯方块怎么样？](http://hackaday.com/2015/01/01/skyscraper-tetris-lets-the-city-know-how-good-or-bad-you-are/)即使是惠普工程师[也无法抗拒这些小砖块的诱惑](http://hackaday.com/2014/04/26/playing-tetris-on-an-oscilloscope/)。当然，还有我们的贝尔格莱德徽章。这是一个简单的游戏，也是在一个有限的系统上对你技能的一个巨大考验。你最喜欢的俄罗斯方块平台是什么？
