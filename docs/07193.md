# Hackaday 奖参赛作品:Ebers 糖尿病护理，循序渐进

> 原文:[https://hack aday . com/2017/09/29/hack aday-prize-entry-ebers-diabetes-care-step-by-step/](https://hackaday.com/2017/09/29/hackaday-prize-entry-ebers-diabetes-care-step-by-step/)

糖尿病是一种由于神经病变、血管问题和其他因素的组合而对足部产生显著影响的疾病。你可能见过特殊的糖尿病袜，其特点是无弹性袖口以促进血液循环，并且没有接缝以防止水泡的形成。在糖尿病中，照顾好你的脚是防止受伤和感染的关键。Ebers 是一个旨在帮助这一领域的项目。

Ebers 监测鞋底的足底压力、温度和湿度。然后，它将这些数据反馈给智能手机，通过蓝牙进行分析。该项目的大脑是一个 Arduino Pro Mini，它的任务是与各种传感器接口。

该项目依赖于 3D 打印的鞋垫，它可以贴合穿着者的鞋子。这是 3D 打印的一个特别有用的应用，因为这意味着鞋垫可以定制以适应个人，而不是依赖于预先确定尺寸的较小选择。这具有额外的好处，允许鞋垫被设计成首先最小化脚上的压力，进一步减少受伤和感染的可能性。压力感测实际上内置于鞋垫本身，可以测量脚的几个不同区域的压力。

总的来说，这是一个对糖尿病患者有巨大潜在健康益处的项目。我们期待着看到这个项目的未来走向，以及它如何为全世界人民的生活质量带来改善。
