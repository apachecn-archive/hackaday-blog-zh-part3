# 20kWh 的自制 Powerwall

> 原文：<https://hackaday.com/2016/09/29/homebrew-powerwall-sitting-at-20kwh/>

偶尔会有黑客开始一个项目却忘记停止。反正我们从[HBPowerwall]的渠道得到的印象就是这样。他正在为他家的电网添加大量的 18650 锂电池，并发布他一路上的冒险经历。本周，他让我们看了一下[他使用](https://www.youtube.com/watch?v=hrpo_o4z0eE)的平衡过程，让所有这些细胞一起很好地工作。上个月，他给了[一个已安装系统](https://www.youtube.com/watch?v=Nsw8PhDlhSg)的概述。

他的频道开始时很无辜。都是骑着小摩托到处跑，有规律的好时光。然后他用轻质材料做了一些实验，比如在屋顶上安装几块太阳能电池板。然而，好像有一天他正在看一个关于 Powerwall ( [特斯拉的家用电池存储系统](https://www.tesla.com/powerwall))的新闻简报，他说，“嘿，我可以做这个。”

在对这种新物质做了一些初步工作后，不久他就开始乞讨、借钱，并为澳大利亚布里斯班当地的宇宙可以卖给他的每一个废旧锂电池讨价还价。有大量的视频记录了他的疯狂，但他一直在一个部分离网的房子里，有一个 20kWh 的电池组，他有扩展计划。

在生产自己的非电网电力的过程中，有很多营销谎言和一般的技术陷阱。但是对于阳光充足地区的黑客来说，这是一个有趣的蓝图。
