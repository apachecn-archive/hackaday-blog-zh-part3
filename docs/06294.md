# Go Go 相机滑块

> 原文：<https://hackaday.com/2017/07/01/go-go-camera-slider/>

你的手臂是否因为来回推动相机滑块而感到疲劳？[Max Maker]一定是这种情况，这让他[将手动滑块转换成电动滑块](http://www.instructables.com/id/Make-a-Motorized-Camera-Slider/)。

电子设备很少——一个 Arduino Micro，几个拨动开关，一个 4988 步进驱动器，12V 电池组，以及一直流行的 NEMA 17 步进电机。如果你想知道为什么我们说“开关”而不是“开关”，这是因为 4 个开关用于选择时间范围。时间帧是滑块从一端移动到另一端所花费的时间。

下面视频中展示的捏造会让你收获一些新花样。我们最喜欢的是他如何用胶带制作 NEMA 汽车的模板。在用胶带完全覆盖电机表面后，他在电机板的形状上清楚地标记安装孔和颜色，就像他在做装饰一样。然后，将胶带作为一个整体撕下，贴在滑动架上。

不包括滑块本身的成本，零件清单出来是 75 美元左右。即使你还没有自己的滑块，这是一个伟大的第一次冒险建立一个数控机床。这是一个自由度，滑块的制造商已经考虑到了硬部件。习惯使用皮带和步进电机编程，你就可以用 Z 轴的花式皮带方案组装自己的 [3D 打印机。](http://hackaday.com/2013/04/05/h-bot-style-3d-printer-moves-bed-for-z-axis/)

如果你确实想从头开始制作你的滑板，这里有几个选择:[用滑板卡车来移动](http://hackaday.com/2015/04/09/camera-slider-utilizes-skateboard-trucks/)，[用宜家的零件做一个](http://hackaday.com/2014/02/17/macgyver-made-ikea-camera-slider/)，用栅栏柱做[怎么样](http://hackaday.com/2014/01/27/yet-another-diy-camera-slider/)？
