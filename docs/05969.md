# Monoprice 发布他们的迷你 Delta 打印机(在 Indiegogo 上)

> 原文:[https://hack aday . com/2017/05/17/monoprice-releases-their-mini-delta-printer-on-indiegogo/](https://hackaday.com/2017/05/17/monoprice-releases-their-mini-delta-printer-on-indiegogo/)

大约在去年的这个时候，Monoprice 悄悄推出了一款 200 美元的小型 3D 打印机。当时，这个价位的全功能打印机并非闻所未闻。一台好的 3D 打印机在这个价位。事实证明，这台打印机[实际上非常棒](http://hackaday.com/2016/06/13/review-monoprice-mp-select-mini-3d-printer/)，完全改变了桌面 3D 打印机的价值定位。

自 MP Select 迷你打印机发布以来的一年中，Monoprice 一直在努力降低成本，重新设计，并创造一种更便宜的打印机。现在，它出来了。[现在可以在 Indiegogo 上预订](https://www.indiegogo.com/at/mpmini3dprinter)。这还是 150 美元的打印机吗？不完全是这样:“早鸟”价格是 159 美元，免费送货，8 月份送货，普通价格是 169 美元，加上 9 月或 10 月送货的 10 美元运费。还有一个 279 美元的套装，包括打印机、2 公斤灯丝和一个软件包。

我们第一次看到这个微型打印机是在一月份的 CES 上。它看起来是一台功能极其强大的打印机；唯一的问题是，垄断企业是否能生产这种产品，并把它运出去。这将是一项艰巨的任务；这台打印机配备了 NEMA 17 步进电机，加热床，32 位控制器板，并具有 WiFi 功能。

以下是我们所知道的这台打印机的功能。这是一台相当标准的 delta 打印机，配有鲍登挤出机和加热床。支持 PLA 和 ABS。该打印机具有自动底座调平功能，可在打印前通过在大约十几个位置用喷嘴“敲击”底座来测量底座。从我们在 CES 上看到的情况来看，热端似乎类似于 200 美元的 Select Mini 的第一次修订——可能为 E3D 热端安装打开了大门。

这台打印机值吗？众筹平台上发布的每一台 3D 打印机都应该带有标准警告，但 Monoprice 表示这种机器目前正在生产中。这就提出了一个问题:既然买方垄断已经将“在线接受产品订单”收入囊中，为什么还要在 Indiegogo 上发布呢？我怀疑这个众筹活动只是在建立一个缓冲区；一年前，MP Select Mini 一直缺货，供不应求。同样的事情也会发生在更深入冲动购买领域的 3D 打印机上。

无论如何，我们一直在等待的打印机已经“发布”了，对于不同的“发布”值。第一批设备将于今年夏天开始进入台式机市场，我们将挑选一台进行测试。你可以看看下面 Monoprice 关于这台打印机的视频。
