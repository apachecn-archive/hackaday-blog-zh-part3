# Octave Linux 下的信号处理教程

> 原文:[https://hack aday . com/2016/06/30/tutorial-on-signal-processing-in-Linux-with-octave/](https://hackaday.com/2016/06/30/tutorial-on-signal-processing-in-linux-with-octave/)

如果你在学院或大学上过数字信号处理课，你可能已经接触过 MATLAB。然而，如果你想做你自己的工作，你可以考虑 Linux 和许多科学计算应用程序中的一个。

[David Duarte]最近发表了一篇关于使用 Octave 进行科学音频处理的三部分教程。[第一部分](https://www.howtoforge.com/tutorial/how-to-read-and-write-audio-files-with-octave-4-in-ubuntu/)涵盖了音频文件的基本读取、写入和播放。[第二部分](https://www.howtoforge.com/tutorial/octave-audio-signal-processing-ubuntu/)包括信号的合成、绘图和一些基本的变换。调制是[第三部的主题](https://www.howtoforge.com/tutorial/ubuntu-octave-audio-processing-part-3/)。如果你喜欢视频教程，你可以看看下面的视频。

我们之前已经在消息破解的上下文中讨论过 [MATLAB](https://hackaday.com/2015/02/16/using-matlab-and-sdr-to-reverse-engineer-433mhz-messages/) 。话又说回来，一些最好的信号处理是由人类完成的[。如果你不喜欢 Octave，你可以试试 Scilab，另一个类似的 Linux 包。还有 Freemat，Sage，和 Spyder。当然，你也可以在 Linux](https://hackaday.com/2015/12/30/messages-from-hell-human-signal-processing/) 下运行 [MATLAB。](https://help.ubuntu.com/community/MATLAB)
