# 单部分升压转换器挑战(已完成)

> 原文：<https://hackaday.com/2017/06/13/single-part-boost-converter-challenge-completed/>

[乔希]提出了一个有趣的挑战。创建一个升压转换器,它可以使用几乎耗尽的电池和一个零件点亮蓝色 LED。嗯，我们很怀疑，直到我们看到他没有把 ATtiny 处理器算在内。你可以在下面看到挑战的视频。

这个挑战已经解决了，所以如果你看到这个链接，你可能想避免评论，直到你有时间思考你自己的解决方案。我们承认，我们想到的第一个方法可能不可行，[Josh]解释道。最终答案完全符合黑客的标准。

虽然解决方案很棘手(不，我们不会泄露)，但我们可以说这不是“欺骗”。没有其他能源藏在某个地方或类似的东西。最终结果是合法的和令人满意的，并利用了一个文件化的芯片特性。

最后，真正的阅读是在评论中，但首先给自己一些时间去思考。如果你想要一个有不止一个额外部分的例子，我们已经看到了很多这样的例子。如果你想重温一下它是如何工作的，你[可能会喜欢这篇文章](https://hackaday.com/2017/05/05/the-science-behind-boost-converters/)。
