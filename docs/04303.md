# 游戏控制器切割地毯

> 原文:[https://hack aday . com/2016/11/29/game-controller-cuts-the-rug/](https://hackaday.com/2016/11/29/game-controller-cuts-the-rug/)

电影《Big》中有一个标志性的场景，汤姆·汉克斯和罗伯特·劳吉亚在一架巨大的钢琴上跳舞。这是我们看到[jegatheesan . soundarapandian]的 [PC 游戏手柄地毯](http://www.instructables.com/id/Wireless-Joystick-Soft-Rug-for-PC-No-Stitch/)时想到的第一件事。他的鼓演奏(见下面的视频)没有[汉克斯]和[凉廊]那么悦耳，但话说回来，他们可能有一个音乐总监。

这个项目的核心当然是一个 Arduino。HC-05 提供了与 PC 的蓝牙连接。我们认为可能会使用像 Leonardo 那样具有 USB 输入功能的 Arduino，但相反，[jegatheesan]在 PC 上有一个自定义的 Visual Basic 程序，它使用 SendKeys 来完成脏工作。

这些开关更有趣，是用旧 CD、箔片和海绵做成的。海绵将 CD 分开，直到你踩在上面，金属箔使 CD 导电。他在项目中使用了大量的 Fevicol 据我们所知，这只是一种印度品牌的 PVA 胶水，所以 Elmer 的或任何其他白色胶水都可以。

胶水也处理织物部分。当一个项目说“不要缝纫”时，我们意识到一些人对焊接的感受。CD/箔片/海绵开关在其他情况下可能有用。我们对海绵长期使用后的磨损情况很感兴趣。

我们以前见过其他[巨型控制器](https://hackaday.com/2015/05/21/large-nes-controller-made-from-legos/)。当然，如果你真的想要一个大控制器，你打不过[一辆日产](https://hackaday.com/2015/11/20/turning-a-car-into-a-playstation-controller/)(链接死了，不过视频会给欧出主意)。
