# 垃圾筐自动化

> 原文：<https://hackaday.com/2016/10/30/litter-basket-automation/>

有时候项目的技术部分并不是最难的部分。它是对既有用又可行的东西有一个想法。当然，为你打扫卫生和洗衣服的机器人管家会很有用，但对我们大多数人来说可能遥不可及。另一方面，对于另一个闪烁的 LED 来说，只有这么多的使用案例。

[Martinhui]知道如何使用超声波传感器和 Arduino。驾驶汽车也不是很难。问题是:你用它做什么？【马丁的】回答:[自动化一个垃圾桶](http://www.instructables.com/id/Auto-Trash-by-Arduino/)。你可以在下面看到结果的视频。

当然，你可以找到它的商业版本，但是这有什么意思呢？这个罐子有点小，但更大的马达或不同的机械设计可以很容易地放大它。

就机器人垃圾桶而言，这个[并没有那么雄心勃勃](http://hackaday.com/2012/07/20/robot-trash-can-catches-anything-you-throw-near-it/)，但它是高度可行的。要是它[连接到互联网](http://hackaday.com/2016/03/17/presenting-the-internet-of-trash-cans/)就好了。
