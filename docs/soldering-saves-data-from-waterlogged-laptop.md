# 焊接保存水浸笔记本电脑的数据

> 原文：<https://hackaday.com/2018/04/03/soldering-saves-data-from-waterlogged-laptop/>

当你把笔记本电脑掉到游泳池里会发生什么？是的，你买了一台新的笔记本电脑。但是你的数据呢。你有备份，对吧？不，你当然不知道。但是如果你能像[TheRasteri]那样焊接，你可以[连接到主板上的闪存](https://www.youtube.com/watch?v=ny82c3wLOFo)并最后一次读取它。你可以在下面的视频中看到整个漏洞。

实际上有三项任务。首先是找到主板的原理图和电路板布局。显然，这些通常无法从制造商处获得，但可以在互联网上的一些不太好的地方以很少的费用获得。一旦你有了布局，你必须安排将导线焊接到你需要访问的闪存部分。

最后一项任务是连接闪存。事实证明，这很简单。你只需要买一个便宜的 SD 卡读卡器，把它拆开，在闪存连接器原来的地方焊接电线。你经常可以在折扣后免费得到这些，或者只需花几美元就能得到一个。它不需要持续很长时间。

视频展示了一些通过移除电阻或切入主板来隔离芯片的好技巧。还有一些事情你可能一开始没有想到，比如在操作时将芯片的复位引脚保持在高电平。

它完全起作用了吗？我们不会放弃，但我们会说，既然他是为了朋友才这么做的，有一个像[ther astri]这样的朋友真好。令人惊讶的是，我们以前在湿手机上看到过这种情况。虽然这确实需要一点焊接技巧，但这并不像你必须焊接到 BGA 上的每个球[或任何东西。](https://hackaday.com/2011/03/13/reverse-engineering-the-psp/)
