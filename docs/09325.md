# 古怪的水银蒸汽整流器是电子管爱好者的最爱

> 原文：<https://hackaday.com/2018/05/02/oddball-mercury-vapor-rectifier-is-a-tube-geeks-delight/>

即使你不是电子管发烧友，你也会情不自禁地被水银蒸汽整流器工作时发出的蓝光所吸引。它看起来不像 20 世纪早期的技术，更像是属于《星际迷航》布景的东西。[Uniservo]获得了一个 866 整流器，由于其标记而很有趣，他在下面的视频中[详细解释了这一点](https://www.youtube.com/watch?v=JTp4EBHJARs)。然而大多数人可能会想跳到更靠近它的末端去看那独特的蓝光。确切的色调取决于汞蒸气压，通常包含相当数量的紫外线。

这些灯管有一段有趣的历史，可以追溯到 1901 年，那一年[彼得·库珀·休伊特]发明了一种比传统灯泡效率高得多的水银蒸汽灯。它们有两个主要问题，当你打开它们时，它们需要一些特殊的过程来使里面的汞蒸发，但更糟糕的是，这种灯是蓝绿色的，不太适合家庭和办公室照明。然而在 1902 年，[休伊特]意识到电子管可以充当整流器。电子可以很容易地从作为阴极的汞蒸汽中流出，而碳阳极则不容易放出电子。这很重要，因为直到那时，还没有一种简单的方法将交流电转换成 DC。通常的方法是使用一个交流电机耦合到一个 DC 发电机或类似的机械装置称为旋转转换器。

在接下来的几十年里，汞蒸汽灯会涂上一层磷涂层，将紫外光转化为冷白光，成为荧光灯泡，所以尽管整流器大多让位于更有效的方法，但[休伊特的]灯泡已经使用了很多年。
