# PCB 焊盘修复和清理

> 原文:[https://hack aday . com/2017/07/02/PCB-solder-pad-repair-clean up/](https://hackaday.com/2017/07/02/pcb-solder-pad-repair-cleanup/)

当您的主板被电解尘垢覆盖，焊盘和迹线损坏，正在分崩离析时，您会怎么办？你叫[【retro gamemodz】用环氧树脂和焊料](https://www.youtube.com/watch?v=vx50YtEC2S8)施展他们的魔法。

虽然这个视频有点旧，但参与修复的视频永远不会过时。这个视频*真正*的特别之处在于，它打破了“看我在沉默中焊接”(或者它的近亲，“看我在喧闹的音乐中焊接”)的普遍趋势。相反，[RetroGameModz]带你一步一步地了解他们在做什么，修复主板。男孩，他们有他们的工作切断了他们:他们正在工作的主板肯定看到了更好的日子。具体来说，这是更好的*之前*腐蚀从一个泄漏的电解电容器和善意的触摸它的主人。

快速查看损坏情况后，所有组件都从战区移除。然后开始清洁，特别注意不要撕坏垫子。一切都清理干净后，事情变得非常有趣。[RetroGameModz]开始使用旧焊盘作为模板，用原始铜制作自己的焊盘，以替换主板上缺失的焊盘。涂了一点环氧树脂后，很难看出垫子是手工制作的，它们非常合适。

这个环氧的招数也是用来处理一些破损严重的痕迹的，爽！在修复过程中，[RetroGameModz]使用了一种耐热温度高达 315°C、持续 60 秒的环氧树脂。如果你在市场上发现任何一种指定耐热温度超过 315°C 的环氧树脂，如果你能在评论区留下一些信息，[RetroGameModz]会很高兴，因为他们发现高温环氧树脂很难找到。

这表明有些修理工作确实应该由专业人员来做。[RetroGameModz]肯定同意这一点，他说“如果你不是维修技师，而你的主板已经停止工作，那么为了你自己的利益，最好不要尝试你确实无法处理的维修。”好建议。但是，在把事情交给更有经验的人之前，我们总是忍不住试图自己解决问题。称之为恶习，或美德；我们称之为乐趣。
