# 万圣节头罩有可怕的 LED 凝视

> 原文：<https://hackaday.com/2015/10/08/halloween-hood-has-hideous-led-gaze/>

想用你的万圣节服装制造恐惧吗？[Becky Stern]在 Adafruit，她教你如何制作一个有 LED 眼睛的神秘兜帽，非常适合你的下一个贾瓦人、黑法师或 Orko 服装。

这个看起来令人毛骨悚然的创作基于一个驱动两个[新像素宝石](http://www.adafruit.com/product/2226)的 [Gemma](http://www.adafruit.com/product/1222) 控制器，小圆形 RGB LED 板。Gemma 驱动电路板缓慢地渐亮和渐暗，以获得所需的令人毛骨悚然的视觉效果，但创建其他照明模式也很容易。

说到图案，教程还包括一个车篷的缝纫图案，以及一个 3D 打印电池盒的计划，这将使整个东西非常容易携带。如果眼睛还不够，再加一根 [LED 魔法杖](http://hackaday.com/2013/10/29/led-magic-staff-just-in-time-for-halloween/)来完成你令人毛骨悚然的合奏如何？或者也许是一些[发光的恐龙钉板](http://hackaday.com/2012/09/14/dinosaur-hoodie-to-remind-you-halloween-approaches/)？

你有什么好的万圣节服装黑客计划吗？请在评论中告诉我们，我们会整理出一份最接近可怕的一天的名单。
