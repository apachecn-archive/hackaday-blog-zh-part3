# 3D Carver 使磁场可见

> 原文:[https://hack aday . com/2015/11/19/3d-carver-makes-magnetic-fields-visible/](https://hackaday.com/2015/11/19/3d-carver-makes-magnetic-fields-visible/)

科学史上充满了 3D 物理模型导致重大发现的例子。但是模拟一些实际上看不见的东西会很困难。以磁场为例——卡片上的铁屑会给你一个 2D 模型，但磁场的 3D 可视化会更有启发性。对于这项工作来说，[这个磁场跟随 3D 雕刻机](https://www.youtube.com/watch?v=qczLVOGbdtw)正是它所需要的。

一开始是使用伺服系统和热熔胶的快速原型制作，最终却成了[Frits Lyneborg]快速而肮脏的 3D 雕刻设备。该视频展示了他的思维过程，并详细描述了他如何从热胶水和棍子到乐高工艺零件，最终到雕刻师框架的 [Makerbeam](http://www.makerbeam.com/) 挤压件。一个带有霍尔效应传感器的探针与一个旋转钻头的马达相连，钻头切入一块花卉泡沫。微控制器使霍尔传感器与稀土磁体保持或多或少的固定距离，从而产生泡沫中磁场的 3D 模型，以及一堆泡沫块。尽管由于钻机在飞行中的调整出现了一些人为现象，但该场在区块中清楚地呈现为两个大的波瓣。

当然，雕刻泡沫并不是三维可视化磁场的唯一方式。如果你更喜欢基于当地磁场的灯光秀，试试我们之前介绍过的这个 [3D 指南针](https://hackaday.com/2015/06/21/visualizing-magnetic-fields-in-3d-space)模型。
