# 掌握滚珠丝杠

> 原文：<https://hackaday.com/2016/11/02/mastering-ball-screws/>

大多数廉价的 3D 打印机使用一种丝杠在垂直方向上移动打印机的某个部分。马达转动螺杆，导致螺母上下移动。打印机部分骑在螺母上。这种方式效果很好，但比其他驱动机制要慢(这也是你不常在打印机的水平部分看到它们的原因)。一些便宜的打印机使用普通螺纹杆，这很方便，但容易出现不良行为，因为杆并不总是直的，螺纹会受到反冲，并且公差也不总是最佳的。

更复杂的打印机使用梯形螺纹杆或梯形螺纹杆。这些都是为这种类型的服务，并有螺纹设计，尽量减少像反冲的事情。它们通常也是按照更严格的标准制造的。使螺母比杆更软(例如黄铜或 Delrin)是另一种常见的优化方法。

然而，当丝杠不够好时，机械设计者转向滚珠丝杠。原则上，这与丝杠非常相似，但不是螺母，而是一个包含滚珠轴承的座圈，滚珠轴承在丝杠上上下移动。滚珠轴承减少了摩擦。

Misumi 最近发布了几篇关于[滚珠丝杠](http://blog.misumiusa.com/ball-screw-steel-ball-recirculation/)的博客文章。有些信息是基本的，但也涵盖了[预压](http://blog.misumiusa.com/ball-screw-preloads/)和[摩擦](http://blog.misumiusa.com/mastering-ball-screws-pt-3-ball-screw-operating-performances/)。此外，他们还承诺在未来的文章中对该主题进行扩展。如果你喜欢看视频，你可能会喜欢下面这个。

总的来说，

*   丝杠比滚珠丝杠便宜
*   与滚珠丝杠不同，丝杠是自锁的，不需要制动系统
*   滚珠丝杠在垂直操作时会有困难
*   滚珠丝杠比导螺杆更有效，允许更小的扭矩和更小的电机，导螺杆也会产生更多的热量
*   丝杠比滚珠丝杠需要更频繁的更换

顺便说一下，Misumi 目录是机械工程玩具的奇迹。如果你关注 Hackaday 有一段时间了，你可能知道所有的滚珠丝杠[都没有螺纹](https://hackaday.com/2016/04/28/threadless-ballscrew-for-3d-printer/)。虽然滚珠丝杠在 3D 打印机中并不常见，但它们在[数控机床设计](https://hackaday.com/2013/10/24/a-very-professional-homemade-cnc-router/)中经常出现。
