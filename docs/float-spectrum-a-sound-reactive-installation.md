# 浮动频谱，声音反应装置

> 原文：<https://hackaday.com/2017/05/29/float-spectrum-a-sound-reactive-installation/>

作为利兹(英国)数字节的一部分，山姆·肯特和朋友们制作了一个声音感应 LED 显示屏，并在海德公园图书俱乐部展出。该装置由 25 根管子组成的网格组成，每根管子由四个回收的 2 升瓶子制成，配备有一串由中央 Arduino 控制的十几个 WS2812B LEDs。

通过 USB 连接到 Arduino，运行处理应用程序的计算机分析音频输入，并告诉 Arduino 点亮哪个 led 以及何时点亮。中间的红色灯管响应低音，黄色 led 环响应中频，外环响应高频发出蓝色光。

令人惊讶的是，仅仅一个简单的 2 升就能制造出一个相当有效的光导管来放大每一次色彩爆发的效果。我们认为这个装置将是对我们最近从多伦多的朋友那里看到的宏伟的 LED 舞池的一个很好的补充。如果你在寻找一个更容易移动的 LED 艺术品，你要找的是[一个狂欢购物车](http://hackaday.com/2015/12/05/a-sound-and-led-tastic-tricycle-shopping-cart/)。
