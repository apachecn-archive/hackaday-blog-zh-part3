# 黑弗比知道你何时靠近

> 原文：<https://hackaday.com/2016/08/21/hacked-furby-knows-when-youre-near/>

电路弯曲的经典之一是在简单的制造噪音的玩具中摆弄驱动 CPU 的时钟芯片。[戈兰]用他的 Furby hack 更进一步。如果你只是想看结果，可以跳到下面的[视频](https://www.youtube.com/watch?v=Q5nbIsD0_Vc)。

在第一次实验用一串电阻 (YouTube)修改了 Furby 的时钟[之后，【Goran】决定选择更多的控制，用 Arduino 发出的方波完全控制时钟。然后，世界成了他的牡蛎。](https://www.youtube.com/watch?v=XM-ecABZVGs)

Furby 的眼睛被超声波距离传感器取代，看起来像扬声器的东西被热粘在它的嘴里。因为这只特别的 Furby 只有在你拉它的尾巴时才会“说话”,所以它很自然地连接了尾巴开关控制器。正如[戈兰]所建议的，灯光秀是显而易见的下一步。

如果你没有拆开一个电子玩具，摆弄它，你不知道你错过了什么。我们有一个经典的电路弯曲介绍，以及从简单的 T2 到复杂的 T4 的项目。这对年轻人来说也是一个有趣的电子入门。抓住一个玩具发声器，开始黑客攻击。
