# 做一个更好的，弹簧加载的 SMT 胶带座

> 原文:[https://hack aday . com/2018/01/19/make-a-better-spring-loaded-SMT-tape-strip-holder/](https://hackaday.com/2018/01/19/make-a-better-spring-loaded-smt-tape-strip-holder/)

有时，一个项目值得做一些额外的工作，看看这个想法是否能走得更远。[JohnSL]正忙着用他的[弹簧加载 SMT 胶带座](http://trains.socha.com/2017/02/making-smt-tape-holders-part-i.html)项目来做这件事。在用 3D 打印完成原件后，他一直致力于注射成型设计。这不是一个电动进料器，它仍然是一个手动工具，但它是对通常车间权宜之计的方法的改进，只是将胶带粘贴到桌面上。胶带从一端送入支架，弹簧张力使胶带保持牢固，同时一个小槽允许在剥离后将盖带向后引导。任何使用胶带切割片段手动处理 SMT 零件的人都知道，微小的振动——如剥离透明盖时产生的振动——会导致较小的元件从口袋中跳出，任何长度的剥离盖都会很快变得笨拙。

![](../Images/5051da15bf9b7759dce5a621d1ee1ad1.png)

The design allows for multiple holders to mount side-by-side.

在[JohnSL]的设计中，无论尺寸或胶带厚度如何，所有的 SMT 胶带都位于同一高度。一个中心支撑从底部向上推，张力来自一个侧向拉动的弹簧；中央支撑件被凸轮向上推动，并压在胶带的底面上。结果，SMT 带以均匀的张力从下面得到支撑，并且整个组件保持适合于并排堆叠多个固定器的窄轮廓。[CAD 文件可在线获得](https://www.youmagine.com/designs/smd-tape-strip-holder-8mm-12mm-16mm-24mm)以及他使用的特定弹簧的 McMaster-Carr 零件号。

在解决了 3D 打印原型的问题后，[JohnSL]决定看看设计一个注射成型版本是否可行，并制作了一个视频来概述这个过程，嵌入在下面。
