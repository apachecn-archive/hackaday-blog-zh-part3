# 机动 Strandbeest 是遥控的，很棒

> 原文：<https://hackaday.com/2016/02/03/motorized-strandbeest-is-remote-controlled-and-awesome/>

如果你以前从未见过 Strandbeest，你会想在休息后观看视频。由[Theo Jansen]发明的 Strandbeest 是一件运动学艺术品。它是一个八条腿的结构，可以在风力的作用下行走，或者如果你聪明的话，一个 Arduino 和一些马达。

在一个周末项目中，[Remet0n]决定将一个玩具版本的 Strandbeest 机动化，并对其进行远程控制。这种玩具通常由风力驱动的螺旋桨提供动力——因此很容易用马达代替。你可以在易贝花不到 10 美元买到它们。

使用一个 Arduino Nano、两个小型 3V 电机、一个无线芯片(NRF24L01)和一个 L9110 H 桥，他能够创建这个令人敬畏的小型遥控设备:
