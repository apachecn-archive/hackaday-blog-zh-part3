# 便携式 RetroPie 建立在巨人的肩膀上

> 原文:[https://hack aday . com/2017/01/12/portable-retro pie-builds-on-the-shoulders-the-shoulders-the-shoulders of-the-giants/](https://hackaday.com/2017/01/12/portable-retropie-builds-on-the-shoulders-of-giants/)

对于任何想拍怀旧照片而又不想找 NES 经典的人来说，Retropie 项目是一个很好的起点。当然，抓起一个树莓 Pi，在上面扔一个预建的发行版，插上一个 SNES 转 USB 的转换器，也没什么太值得注意的。然而，值得注意的是建造一个便携的[复古，它具有来自](http://imgur.com/a/RgTot)[山茱萸爱好者](https://www.reddit.com/r/raspberry_pi/comments/5n5jqo/my_newest_portable_retropie_console/)的最新建造的质量和抛光。

首先，激光切割的木箱是定制的。从那里开始，安装所有的 PCB，包括处理每组按钮(全套 D-pad、肩按钮和操纵杆)的特定 PCB 和用于 5”HDMI 屏幕的 PCB。它有立体声扬声器和自己的耳机插孔(让所有新 iPhone 用户羡慕不已)，由运行 Retropie 4.1 的 Raspberry Pi 2 供电。电池组也不应该让你束手无策，尤其是如果你是玩着世嘉游戏装备长大的。

这里的构建质量是杰出的，它的创作者做出了设计选择，使其易于复制，所以如果你想在旅途中玩 N64 或 PS1 游戏，这可能是你一直在等待的。有很多其他的选择可以从翻新中获得一些乐趣，从[把它做成咖啡桌](http://hackaday.com/2016/09/20/classing-up-a-retropie-arcade-with-a-wine-barrel/)到重新利用那个[臭名昭著的游戏装备](http://hackaday.com/2015/03/02/retropie-turned-game-gear/)。

休息之后，你会发现这个便携式游戏的必读片段。
