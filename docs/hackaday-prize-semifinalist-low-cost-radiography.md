# Hackaday 奖半决赛:低成本放射摄影

> 原文：<https://hackaday.com/2015/09/02/hackaday-prize-semifinalist-low-cost-radiography/>

在过去的一年里，[Adam]全职致力于为发展中国家开发一种低成本的 x 射线系统。他已经在这个项目上投入了 3500 多个小时。几个月前，我们宣布了 2015 年黑客日奖，主题是“创造有意义的东西”对于世界上三分之二没有医疗放射技术的人来说，低成本的 x 射线肯定很重要，[这使得这个项目成为 Hackaday 奖的重要参赛作品](https://hackaday.io/project/7225-low-cost-x-ray-systems-for-developing-nations)。

[Adam]的便携式 x 射线系统由一个 x 射线管组成，该管封装在一个填充了绝缘油的环氧树脂 3D 打印外壳中。这个电子管藏在一个漂亮的盒子里，只有一个 12v 直流输入和一个易于理解的用户手册。这只是非常高的电压和 x 射线，没有什么[亚当]没有(安全地)处理过。真正的诀窍在于成像，为此，[亚当]使用荧光屏将 x 射线曝光转化为可见的东西，[一个现成的 x 射线传感器](http://www.gendex.com/gxs-700)，[一个棱镜使传感器](https://hackaday.io/project/7225-low-cost-x-ray-systems-for-developing-nations/log/22932-april-may-2015-designing-a-low-cost-x-ray-sensor-second-attempt)适应荧光屏。

结果令人难以置信。在对手头的东西拍了几张照片后，[Adam]可以看到计算器微处理器内部的连接线。这对于该项目的目标——医学成像来说绰绰有余，而且足够便宜，可以将其发送到地球上遥远的地方。
