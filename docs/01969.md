# 专业数控真空工作台轻松夹持工件

> 原文：<https://hackaday.com/2016/04/12/professional-cnc-vacuum-table-holds-workpieces-with-ease/>

如果你在数控机床上做了很多一次性零件，你就会知道设置是这个过程中最糟糕的部分。通常你使用的是废料，你必须弄清楚如何夹紧它，确保这块料足够大可以使用，等等。直接把材料扔到床上开始加工不是很好吗？嗯，有了像这个这么好的[真空表，你几乎可以！](https://hackaday.io/project/10451-building-a-high-quality-vacuum-table-tl-video)

[杰克·布莱克]有一台很棒的数控机床。由于他一直在扩展他的原型制作能力，他决定需要一种更好的方法来固定工件进行加工，所以他加工了一个两件式铝制真空工作台。

它具有四个独立的真空袋，可以单独使用或一次全部使用。顶面是具有方形通道的网格图案。使用小橡胶塞或螺丝，你可以堵住你不用的通道，以最大限度地提高你身上的真空力。

当固定一个零件时，工件和工作台之间需要另一个垫圈，但正如你在下面的视频中看到的，一旦就位，它就不会移动。
