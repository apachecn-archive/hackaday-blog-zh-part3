# 2015 红牛创作大赛正在进行中！

> 原文：<https://hackaday.com/2015/09/25/2015-redbull-creation-competition-is-underway/>

我作为评委来到底特律的 2015 年红牛创作大赛现场——这是一场超级激烈的 72 小时创作大赛，制造商、工程师和艺术家可以向我们展示他们的作品。今年的主题相当广泛:**严肃的乐趣**。

该活动在 [Recycle Here](http://www.recyclehere.net/) 举行，并对公众开放，所以如果你在该地区，就来看看吧！一个巨大的回收仓库被临时改造成了一个巨大的车间。团队可以使用一些重要的工具，包括重型焊接设备、工业成型工具，如弯管机、折弯机、标准钻床、切割设备，当然还有激光切割机和 3D 打印机。

有两个竞赛类别——公开类，任何人都可以自由组队参加，以及邀请类。六个只接受邀请的团队在 Recycle Here 工作室工作，参加公开组比赛的团队分散在城市各处，使用他们自己的工作室或黑客空间。这个主题昨天才刚刚宣布，从那以后 72 小时倒计时就开始了。我迫不及待地想看看这个周末能及时构建出哪些令人敬畏的交互项目来向公众展示。
