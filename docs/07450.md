# 关于洛克威尔所有值得知道的事情

> 原文：<https://hackaday.com/2017/10/20/everything-worth-knowing-about-lockwire/>

我们得到了一个由[AgentJayZ]制作的旧视频的提示，该视频演示了锁线的正确使用，也称为“安全线”在高振动操作中，如喷气发动机、街头赛车、机枪和你想变成漂移三轮车的摇摇晃晃的旧轮椅，一个松动的螺栓可能会导致灾难。尼龙受热会失效，机械锁紧垫圈依靠摩擦力，但摩擦力有其局限性。只要安全线完好无损，它就能承受高温并防止松动。

我们的许多读者已经熟悉了安全锁线，因为它算不上是一项前沿技术——除非你说的是安全锁线的切割端，如果你不小心，[AgentJayZ]警告说它会割破你的手指。我们中的一些万事通，拥有一英寸深一英里宽的知识，可能没有意识到所有的事情。在开始的八分钟里，我们打赌你已经深入这个主题至少两英寸了。

【编者按:一英寸正好是 25.4 毫米，如果前面的比喻在翻译中迷失的话。一英里大约是 2，933.333 亚述肘。总之，比一英寸大得多。]

现在，那些耗费我们时间和精力的烦人的松脱螺栓有了明确的解决方案。对于老手来说，你可以在休息后通过观看视频的剩余部分来复习《安全线》。

感谢[Keith Olson]的提示，我们将继续关注[AgentJayZ]，迄今为止，他已经发布了超过 450 个关于喷气发动机的视频。

如果安全不是你最优先考虑的事情，考虑一下这个[自行车上的喷气发动机](http://hackaday.com/2017/09/18/a-jet-engine-on-a-bike-whats-the-worst-that-could-happen/)或者惊叹于[可打印喷气发动机](http://hackaday.com/2017/06/21/3d-printed-jet-engine/)的复杂性。
