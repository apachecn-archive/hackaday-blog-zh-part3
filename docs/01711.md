# 室温焊接

> 原文:[https://hackaday.com/2016/03/09/solder-at-room-temperature/](https://hackaday.com/2016/03/09/solder-at-room-temperature/)

你见过科学实验(或魔术)吗？)你把水过冷到不结冰的地方，但当你接触它、倒它或以其他方式扰动它时，它就会结冰？显然，冰晶在液体中的杂质或扰动周围形成，然后“扩散”类似的效应也会发生在金属中，当熔融金属冷却时，即使温度低于通常会使其熔化的温度，它也会保持熔化状态。

衣阿华大学的 Martin Thuo 利用这一特性在室温下使用菲尔德的金属(铋、铟和锡的混合物)制作焊接接头。关键是在熔融金属上涂上几层纳米层，保护它不凝固，直到有东西扰乱了保护层。

研究人员希望利用这种技术创造室温焊接。到目前为止，最大的问题是磁场的金属将在 62 摄氏度时再次熔化，这使得许多可以承受更高温度的电子电路变得不切实际。
