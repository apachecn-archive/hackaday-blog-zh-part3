# 视频控制系统基础

> 原文：<https://hackaday.com/2017/10/17/control-system-fundamentals-by-video/>

如果你接受过经典的工程教育，你可能会模糊地记得有人谈论过控制理论。如果你没有，你可能至少听说过 PID 控制器和开环与闭环控制。如果你不了解控制理论，或者只是想复习一下，[布莱恩·道格拉斯]有一套精彩的近[50 个视频讲座](https://www.youtube.com/playlist?list=PLUMWjy5jgHK1NC52DXXrriwihVrYZKqjk)，会给你一个很好的主题介绍。可以看第一讲，下面。

你可能会认为，当你试图控制一个像化工厂或温度这样的过程时，控制系统只在电子学中有用。然而，从滤波器到振荡器，再到接收机中的自动增益控制，控制理论出现在了数量惊人的地方。你会发现控制理论中许多熟悉的结果背后的背景。有点像当你学微积分的时候，你会发现他们是如何得出你在几何中记忆的所有公式的。

这些视频的呈现风格不错。第一组主题必然是抽象的，但该系列的后续视频显示了使用轨道和 Arduino 的机器人车辆、MEMS 陀螺仪以及如何在行星上着陆。如果你不认为控制系统无处不在，考虑一个我们在和学生交谈时使用的例子:厕所。一个非常简单且非常常见的闭环控制系统。甚至[【比尔盖茨】和加州理工也同意](http://www.contractormag.com/kitchen-bath-industry-show/kohler-designing-next-generation-closed-loop-toilet-flush-system)。虽然，老实说，我们不知道什么是“渴望如厕体验”

我们已经就这个话题制作了自己的[系列，但没有使用视频。我们已经涵盖了从](https://hackaday.com/2015/12/02/beyond-control-the-basics-of-control-systems/)[无人机](https://hackaday.com/2014/07/07/droning-on-pid-controllers-and-bullet-connectors/)到[烙铁](https://hackaday.com/2015/04/28/adding-pid-control-to-a-non-adjustable-iron/)的 PID 控制项目。
