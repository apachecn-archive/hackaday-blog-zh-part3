# 包里的蓝牙音箱

> 原文:[https://hackaday.com/2017/12/08/bluetooth-speaker-in-a-bag/](https://hackaday.com/2017/12/08/bluetooth-speaker-in-a-bag/)

[VanTourist]被他所看到的复杂的项目视频所困扰，他证明了你可以在三个小时内构建一个高质量、多功能的蓝牙扬声器[。](http://vantourist.com/bluetoothspeaker)

他使用简单的手工工具——主要是压线钳、剥线钳、剃刀和一些胶水——将这个重新利用的 GoPro 配件包包装上了相当多的技术。主要组件是一个带漂亮旋钮的蓝牙放大器，一对 15W 扬声器，但他还增加了一个 1W 的 LED 手电筒，1A 和 2.1A 充电端口，电池充电监控显示器，以及时尚点的飞行员盖拨动开关。尽管所有的东西都塞进了包里，还是有一点空间来装一些东西！你可以在这里查看建造图片[，或者休息后的视频。](https://imgur.com/gallery/vgcYY)

由于有 4400 毫安时的电池，这款扬声器在播放音乐时的电池寿命为 20-30 小时，但如果你给设备充电并使用手电筒，里程数显然会有所不同。五个小时的充电时间可能是一天一夜音乐的合理价格。

我们最近推出了一款[声音感应扬声器](https://hackaday.com/2017/05/01/portable-bluetooth-speaker-reacts-to-sound/),可以保证派对上的适当灯光。或者，戴上[这些护目镜](https://hackaday.com/2016/11/20/make-your-eyes-louder-with-bluetooth-speaker-goggles/)来真正见识音乐的力量。
