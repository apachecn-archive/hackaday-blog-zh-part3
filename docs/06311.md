# 两用 DIY 点焊机，以安全为理念

> 原文：<https://hackaday.com/2017/06/27/dual-purpose-diy-spot-welder-built-with-safety-in-mind/>

呵呵，又一个微波炉变压器点焊机吧？不，不是这个——[【Kerry Wong】参加 MOT 点焊机军备竞赛](http://www.kerrywong.com/2017/06/18/dual-purpose-spot-welder-with-pulse-duration-control/)是出于安全考虑，并具有增值功能。

正如[Kerry]指出的，大多数 MOT 点焊机使用某种瞬时开关来给变压器的初级侧供电。考虑到这意味着电源电压危险地靠近你的手指，[凯瑞]选择远离愤怒的小精灵，用三端双向可控硅开关开关初级。不仅如此，他还将三端双向可控硅开关的触发器光学耦合到一个围绕古老的 555 芯片构建的小型单次定时器上。脉冲持续时间控制导致焊接不同厚度的不同材料的能力，而不是烧坏薄的原料并在较厚的材料上得到弱的焊缝。一个很好的补充是一个专门为电池极耳焊接设计的独立探针——在 18650 年代推出。

[凯瑞]建造了一些安全设施，这是值得称赞的，但是当他在高电流源周围工作时，他可能要考虑摘下或盖住那个环。如果你的安全意识不够，这个点焊机可能会也可能不会杀死你。
