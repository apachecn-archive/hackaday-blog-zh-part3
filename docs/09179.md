# 骑 Roomba 的啤酒管家会为我们所有人服务

> 原文：<https://hackaday.com/2018/04/21/roomba-riding-beer-butlers-will-serve-us-all/>

[Josh]不是一个拒绝挑战的人，尤其是当涉及到机器人的时候。来自朋友和家人的最新挑战？[造一个啤酒机器人，可以随叫随到的带饮料](https://hackaday.io/project/138959-beer-butler-robot-refrigerator-on-a-roomba)。

该建筑由两个主要部分组成:冷藏冷却器和管家部分，这是一位机器人专家同事在 Roomba 上发现的。[乔希]是基于双壁和绝缘餐厅冷却器的设计。他用两个直径相差一英寸左右的不锈钢垃圾桶建造了冷藏饮料舱，并用膨胀泡沫绝缘材料填充了空隙。然后，他从内胆底部切掉几英寸，为冷却装置腾出空间，重新安装滴水盘，并通过在抗菌塑料菜板上钻几个孔来制作一个[允许气流的平台]。

起初，他尝试了冰屋冰箱中的帕尔贴装置，但效果不如[乔希]希望的那样好，所以他正在重新设计罐子，使用迷你冰箱压缩机。这意味着用铜管定制蒸发器和冷凝器盘管，以匹配压缩机的负载规格。浏览木卫一上的[Josh]的[构建日志](https://hackaday.io/project/138959/logs?sort=oldest&page=1)，你会得到一个关于膨胀泡沫和制冷的免费迷你课程。

[Josh]目前正在为这个机器人开发一些不同的管家模式。这些服务涵盖了从简单地坐在附近拿着冷饮，挥挥手就可以打开，到在每个人的召唤下进行语音触发的饮料管家服务。我们赞赏他迄今为止所做的努力，并将怀着极大的兴趣关注这一次，看看他如何处理导航和语音控制。
