# 注射成型的奇迹:它是如何工作的？

> 原文：<https://hackaday.com/2017/12/17/the-miracle-of-injection-molding-how-does-it-work/>

如今，几乎任何一件家居用品都有一个复杂的、经过深思熟虑的制造方法，无论是一张纸、一个电源插座、一罐金枪鱼，甚至是房子本身。然而，一些关于这些物体是如何形成的故事是令人信服的，正如[这个老托尼]最近的一个视频显示的那样，[他深入研究了一个 5 美元的球阀](https://www.youtube.com/watch?v=IiGTkl4fSx4)，并用它来谈论你可以用注射成型做的所有很酷的事情。

注射成型是将熔融塑料铸造成更有用的塑料件的过程。在这种情况下，它是一个管道阀门，表面上看起来很简单，但实际上要复杂得多。这些球阀非常可靠，但价格很低，这意味着它们的设计中一定包含了大量的工程设计。视频中挖掘出的是注射成型允许零件被铸造到其他零件的模具中，以及那些零件不全部熔化在一起的手段，以及如何在零件本身内部创建密封。所有这些都是在最少的零件数量下发生的，与人或任何不是注塑模具本身的机器人没有任何互动。

该视频对这些阀门进行了特别详细的介绍，同时也阐述了注塑成型中的各种技术。类似于[认可看似普通的铝罐值得](https://hackaday.com/2015/04/17/you-betta-recognize-the-aluminum-beverage-can/)，注塑球阀也值得同样的尊重。虽然[这个老托尼]通常专注于金属加工，但他也经常处理像这样的其他有趣的话题，这个视频绝对值得一看。
