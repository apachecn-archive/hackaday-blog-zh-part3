# 微型特斯拉涡轮机是一项工程杰作

> 原文:[https://hack aday . com/2016/06/17/micro-Tesla-turbine-is-an-engineering-tour-de-force/](https://hackaday.com/2016/06/17/micro-tesla-turbine-is-an-engineering-tour-de-force/)

戈德温定律的一个推论应该是，任何提到尼古拉·特斯拉的黑客帖子都会有一个长长的、丰富多彩的评论帖子。我们希望这一个也是，但是如果幸运的话，它将专注于[这个小型定制的 Telsa 涡轮机](https://youtu.be/IQmlXJVo9LE)背后的工程。

对于那些不熟悉特斯拉最喜欢的发明的人来说，这种涡轮机是一种超高效的设计，没有叶片，而是依靠光滑、紧密间隔的圆盘，通过运动流体的摩擦来拖动。[Johnny 90]的微型涡轮是一项非常成功的机械加工技术。虽然起初这个建筑看起来有点滑稽，但随着它的发展，我们看到了一些真正的工艺——如果你怀疑苏打罐铝可以被翻转，请观看下面的视频。精密的 25 毫米转子进入数控加工铝外壳；翻盖扣在外壳上的方式非常令人满意。看起来唯一现成的零件是转子轴承；其他的都是刮擦做的。第二个视频以一个听起来相当不错的测试卷轴结束。我们迫不及待地想通过第 3 部分来了解这台涡轮机能转多快。

大小很重要，在这种情况下，小是相当可观的。对于特斯拉涡轮机的更大处理，见[这个由旧硬盘驱动器盘片制成](http://hackaday.com/2011/09/09/engine-hacks-tesla-turbines/)。
