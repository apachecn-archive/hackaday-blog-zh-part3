# 自制的电火花加工机可以轻松切割像磁铁这样的困难材料

> 原文：<https://hackaday.com/2016/08/07/homemade-edm-can-cut-through-difficult-materials-like-magnets-with-eease/>

许多年前[ScorchWorks] [建造了一个放电加工工具(EDM)](http://www.scorchworks.com/Blog/homemade-edm-electric-discharge-machining-machine/) ，最近决定写一篇关于它的文章。休息之后还有一段视频。

这种构造是基于罗伯特·兰格洛斯的《制造一个 EDM》一书中所描述的设计。EDM 的工作原理是在所需形状的电极和电介质溶剂浴下面的材料之间产生大量微小的放电。这可以在操作者希望材料溶解的地方准确地溶解材料。这是最精确和最温和的加工操作之一。

他的 EDM 主要是用找到的零件组装的。电源是一个微波炉变压器，用 18 号线重新布线，将电压降至 60 伏，而不是微波炉原来的 1.5 千伏。电源电阻来自从路边的一个单元抢来的干燥元件。控制板是用记号笔在铜上用手描出的示意图蚀刻出来的。

线性运动元件是两个方形铜管，一个在另一个内部滑动。步进电机缓慢地将电极驱动到零件中。冷却剂通过由一个小小的 3D 打印部件支撑的电极泵出。

EDM 工作得很好，他有几个例子来展示其执行高难度切割的能力。比如刀片上的洞。一个穿过非常小的厚钢板的小孔，甚至一个穿过磁铁的孔。
