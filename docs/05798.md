# 机电月球着陆器

> 原文：<https://hackaday.com/2017/05/03/electromechanical-lunar-lander/>

20 世纪 70 年代街机的热门产品之一是雅达利的*月球着陆器*。一艘在月球轨道上的登陆艇将缓慢下降到月球表面，通过姿态和推力控制，玩家的目标是让它安全着陆。有四分之一的人会被渴望证明自己适合宇航员服务的游戏玩家投入到这个游戏中。

当[Chris Fenton]为 2016 年的 NYCResistor 互动秀寻找灵感时，他转向了这款游戏，结果是一款与众不同的*月球着陆器*游戏，其中[游戏通过物理着陆器和月球表面](http://www.chrisfenton.com/electromechanical-lunar-lander/)进行。在这种情况下，讨论中的月球是一个纸糊的充气球，而着陆器是一个在丝杠末端的 3D 打印模型。控制由 Arduino 提供，它是原始控制面板的粗略复制品，模型上有一组微型开关，用于检测坠机或安全着陆。

结果是一个令人惊讶的可玩游戏，从休息下面的视频可以看出。
