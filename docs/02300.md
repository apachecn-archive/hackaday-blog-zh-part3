# 制作苹果的复制品///

> 原文:[https://hackaday.com/2016/05/21/build-a-replica-apple/](https://hackaday.com/2016/05/21/build-a-replica-apple/)

[需要姓名的先生]给我们看了一个很棒的视频，是关于由[Charles Mangin]制作的一个[苹果///](https://www.youtube.com/watch?v=ITsdbaqbObY) 复制品的建模，该复制品的尺寸小到可以容纳一个树莓派。

[姓名先生]指出，该视频是使用[参考照片为](http://hackaday.com/2016/05/06/up-your-cad-game-with-good-reference-photos/)建模的绝佳范例。[Charles]从寻找模型所需的参考资料开始。谷歌图片搜索和一些苹果历史网站为他提供了所需的图片。

他在 Autodesk 123 中为 Apple ///建模。它有草图工具，但他选择在 iDraw 中绘制路径，并将其导入软件。这很可能是因为矢量编辑软件对布尔组合工具的更好支持。否则他必须花几个小时摆弄修剪工具。

在视频的后面，他展示了如何改变照片中的视角，以获得一个物体的更正交的视图。然后是一些重建模的时候了。他真的把 123 逼到极限了。

模型被送去进行专业的 3D 打印，以捕捉所有细节。然后是一些收尾工作，他的微型苹果///就完成了。休息后的视频。
