# Hackaday 奖参赛作品:探索 M3 ARM Cortex M3 开发板

> 原文：<https://hackaday.com/2016/09/29/hackaday-prize-entry-explore-m3-arm-cortex-m3-development-board/>

即使浏览一下这样的网站，你也会发现目前市场上有多少微控制器板。似乎每一个可能的细分市场都被覆盖了，然后是一些，那么究竟为什么会有人想把另一个产品带入这个拥挤的环境呢？

这是一个你可能想问新的 ARM Cortex M3 开发板 [Explore M3](https://hackaday.io/project/10711-explore-m3) 背后的团队的问题。它基于 LPC1768 ARM Cortex M3，具有 64k RAM 和 512k 闪存，运行速度为 100MHz，并具有常见的大量 GPIOs 和内置外设。

该板的设计者最初的目标是使其能够作为裸机臂使用，或者与 Arduino 和 Mbed 工具配合使用。如果 Mbed 对他们询问的回应导致他们放弃了这种支持。他们指出，他们的[综合教程集](http://exploreembedded.com/wiki/Main_Page)是他们的主板从竞争中脱颖而出的原因，反过来，他们否认试图仅仅制作另一个 Arduino 或 Mbed。他们选择的物理格式是一个紧凑的双列直插式电路板，便于进行试验，与 Arduino Micro 或 Teensy 没有什么不同。

如果你阅读了该项目的日志，你会发现几个解释该项目的视频，并带你通过一个教程。然而，嵌入到 Hackaday 片段中有点长，所以如果你感兴趣，我们会让你继续前进。

我们已经讨论了很多微控制器开发板。如果你想看看这些年我们走了多远，看看 2011 年的[我们的一轮](http://hackaday.com/2011/02/01/what-development-board-to-use/)和[的第二部分](http://hackaday.com/2011/02/14/what-development-board-to-use-part-two/)。
