# 乒乓频谱分析仪

> 原文：<https://hackaday.com/2016/04/28/ping-pong-spectrum-analyzer/>

频谱分析仪是处理信号的非常有用的工具，其中频率成分的大小很重要。通常，显示器是一个屏幕。有时候，你会看到它是用 led 完成的。[Mag Laboratories]用乒乓球做了[。](https://hackaday.io/project/4761-ping-pong-ball-audio-spectrum-analyzer-ppbasa)

该设备使用处理器来计算傅立叶变换，将音频信号切割成 16 个频带。处理器将这些值转换成驱动小型风扇的 PWM 输出。风扇将乒乓球吹到管子里，与风扇速度成正比。你可以在下面的视频中看到结果。

当然，正如你所料，乒乓球的响应时间不会与液晶显示器的响应时间相匹配，但这并不意味着它是一个实用的工具，而是一件电子艺术品。如果能在音乐会或舞池中使用一些照明设备，在更大的规模上看到这一点会很有趣。

上个月，我们看了一个很酷的使用 led 的 3D 分析仪。当然，对于音频，你总是可以[使用声卡](http://hackaday.com/2015/11/11/getting-started-with-gnu-radio/)或者[插上一个](http://hackaday.com/2014/11/19/rtl-sdr-as-a-spectrum-analyzer/) RTL SDR [设备用于射频](http://hackaday.com/2014/11/19/rtl-sdr-as-a-spectrum-analyzer/)。
