# NES·扎珀:用激光改进

> 原文:[https://hackaday.com/2016/09/09/nes-zapper-now-more-useful/](https://hackaday.com/2016/09/09/nes-zapper-now-more-useful/)

最初任天堂的 Zapper 枪走在了时代的前面。不过，那是大约 30 年前的事了，标志性的控制器甚至无法与大多数现代电视兼容。稍加修改，它们就能工作，但如果你想朝不同的方向发展，它们也能做各种各样的事情。比如[这个可以发射绿色激光，可以当鼠标用](http://shaun.cc/post/146339530314/laserzapper)。

使用一组 3D 打印环将激光笔安装在枪中，以确保对准正确。它由一个 Sparkfun 电池组和控制板供电，所有这些都适合枪的情况。然而，激光并不是枪真正发光的地方。那里还有一个 Wiimote，当它与投影仪一起使用时，可以将枪用作鼠标指针。一定要看看下面的视频，看看它的实际效果。没有什么比把现代任天堂和经典结合起来更好的了！

Wiimote 是一个与电脑互动的绝佳平台。自从 Wii 发布以来，通过蓝牙与他们交流变得相对容易。Wiimote 的经典技术之一是使用红外笔和投影仪以很低的价格创建一个智能板。它们也被用于一些[非常有趣的虚拟现实显示器](https://www.youtube.com/watch?v=Jd3-eiid-Uw)。
