# 为了让你尽情享受观看的乐趣:点击弹簧时钟终于完成了

> 原文:[https://hackaday . com/2016/12/06/for-you-binge-watching-pleasure-the-click spring-clock-is-finally-complete/](https://hackaday.com/2016/12/06/for-your-binge-watching-pleasure-the-clickspring-clock-is-finally-complete/)

制造时间和孕育一个人的时间一样长，但是 Clickspring 开放式机械钟终于完成了。结果是惊人的。

如果你对机械加工感兴趣，你应该看完整部 23 集的播放列表。[克里斯]在每一集展示的工艺水平，无论是在时钟构建还是他的视频制作价值方面，都是值得一看的。时钟开始时，CAD 打印粘在黄铜板上作为模板的滚动锯工作，粗糙的框架和齿轮。车削棒料，给零件刻螺纹和滚花，并切割齿轮齿。时钟中的每个螺丝都是定制的，并经过热处理，变成了浓郁的蓝色，与黄铜零件上的镜面形成了美丽的对比。每一集都有一些精密加工的小花絮，即使你对钟表不感兴趣，也值得一看。对我们来说，最好的时刻出现在第十集，当边框和章环一起发出令人满意的咔哒声。

我们这里有很多计时项目，但没有一个能与 Clickspring 时钟相比。如果你仍然不相信，看看我们早期的一些报道，比如当我们第一次注意到[克里斯]频道时的[，或者当](https://hackaday.com/2015/04/03/machining-a-skeleton-clock-in-10000-easy-steps/)[捏造并弄蓝了时钟的指针](https://hackaday.com/2016/03/23/metal-magic-heat-bluing-steel-clock-hands/)时的[。我们已经等不及下一个 Clickspring 项目了，我们知道今晚我们在看什么。](https://hackaday.com/2015/04/03/machining-a-skeleton-clock-in-10000-easy-steps/)
