# Hackaday 奖参赛作品:UAProsthetics，一只动力之手

> 原文：<https://hackaday.com/2017/10/10/hackaday-prize-entry-uaprosthetics-a-powered-hand/>

桌面 3D 打印机的巨大成功之一是定制假肢和矫形器。只需一只义肢价格的一小部分，你就可以买到一台能够生产数百种完全可定制义肢的机器。[塔兰·拉文德兰]角逐 2017 年 Hackaday 奖的项目遵循了制造定制假肢的悠久传统。他的假手设计得比传统的假肢更简单、更便宜，同时在这只手的运动方式上也给了我们一些创新。

[塔兰]手上的数字是由拉动一系列波顿线缆的线性伺服系统控制的。一个伺服系统驱动食指，通过双差动关闭三个不太重要的数字——中指、无名指和小指不需要食指和拇指的关节。这三者一起启动，节省了成本和复杂性——它们基本上是为了支持索引和拇指而运行，而不是独立可控的。拇指本身有两个自由度，可以发挥最大的效用。

[塔兰]关注的另一个重要领域是易用性。如果假体太复杂，太难，或者用起来不舒服，不管它有多棒的功能，它都不会被使用。知道这一点后，他专注于尽可能简化这只手。现在，这个项目已经在 CAD 中建模了，而[塔兰]只是在等待 SLS 零件的到来，然后组装整个东西。这是一个伟大的项目，也是今年 Hackaday 奖的一个很好的参赛作品。
