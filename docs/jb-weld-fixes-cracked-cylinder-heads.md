# JB 焊接修复开裂的气缸盖

> 原文：<https://hackaday.com/2017/05/16/jb-weld-fixes-cracked-cylinder-heads/>

一直有传言说 JB Weld 的主要成分是魔法。这种两部分环氧树脂，你通常会在架子上找到，旁边是基本的 5 分钟环氧树脂、Titebond、各种氰基丙烯酸酯和 Gorilla 胶水，但有些不同。育空地区“某个家伙”在开裂的发动机缸体上使用 JB 焊接的故事比比皆是。这些故事当然遭到了质疑。

现在，最后，[我们有证据表明你可以用 JB Weld 来修理发动机](https://www.youtube.com/watch?v=aVLoJe3JbFI)。YouTube 上的[Project Farm]对它进行了终极测试:他从割草机上取下气缸盖，用研磨机研磨气缸盖，并用 JB Weld 修补了这个洞。头部具有良好的压缩性，发动机实际上在测试结束前运行了 20 分钟。

如果这是一个现场维修的测试，这将是一个非常蹩脚的现场维修的测试。[Project Farm]没有试图确保活塞不会与 JB 焊缝的斑点接触，事实上，活塞敲击环氧树脂的斑点时会产生轻微的敲击声。尽管如此，这种修复工作。

虽然这证明了用 JB Weld 修复发动机缸体的可行性，但 JB Weld 环氧树脂还有一个终极测试:用它造一台发动机。几年来，我一直把我剩下的 JB 焊放在一个小的方形塑料容器里。再过几年，我就会有一块 JB Weld“库存”，大到足以加工一个小型(. 049 cc)辉光引擎的零件，就像你在老式的 tymie 模型飞机和汽车上看到的那样。行得通吗？我不知道，但现在我迫不及待地想知道。
