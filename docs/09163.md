# 定制真空管盒式放音机

> 原文:[https://hack aday . com/2018/04/16/定制-真空管-卡带-播放器/](https://hackaday.com/2018/04/16/custom-built-vacuum-tube-cassette-player/)

正如我们在 Hackaday 上多次说过的，我们没有资格质疑*为什么*人们制造他们制造的东西。毫无疑问，我们涵盖的许多项目都有合理的需求或用途。但也有很多是如此复杂，以至于近乎荒谬。这并不是说我们不喜欢那些疯狂的，事实上，我们通常最喜欢那些疯狂的。

[![](../Images/4182eba71b90f7c9fec588567604df68.png)](https://hackaday.com/wp-content/uploads/2018/04/tubetape_detail.jpg) 所以当我们看到松下 RN-404 微型卡式录音机这一令人难以置信的改装，用一个[定制的真空管放大器](http://vintage-technics.ru/Eng-Tube_Microcassette_recorder.htm)取代了音频硬件，我们没有费事问这是什么意思。也许这是一个尝试，让最不切实际的方法来记录和播放音频，或者可能只是为了看看是否有可能。不管它为什么被做，它现在就在这里，它绝对*光荣*。

如果硬件的外观没有提示你这个项目使用了旧的苏联时代的组件，休息后的视频肯定会。具体来说，它使用的是最初用于军事用途的 1ZH25R 和 1S38A 管。就像所有酷的老苏联技术一样。不管你对冷战有什么看法，它确实让工程学的汁液流动起来。

有相当多的信息是关于这些古老的电子管是如何通过这种华丽的家庭蚀刻 PCB 复活的。可以说，一开始使用管道是一门艺术，但使用如此小而独特的管道是一个全新的水平。

这不是我们第一次看到一些微小的电子管进入消费类音频设备，但这一次在专业最终效果方面无疑占据了首位。
