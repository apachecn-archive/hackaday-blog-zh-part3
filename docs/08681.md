# 电源选择

> 原文:[https://hackaday.com/2018/02/25/power-supply-choices/](https://hackaday.com/2018/02/25/power-supply-choices/)

除非你正在建造一个水晶收音机，或者你终于发明了那个无限能量的机器，否则你做的任何项目都需要某种电源。曾经有一段时间，一个电池就足够了，但现在你可能需要某种监管。但是有很多种可以选择。线性、切换、SEPIC、LDO……你怎么挑？[Andreas Spiess]在他最近的视频中有一些实用的建议，你可以在下面看到。

[Andreas]将该视频称为“电压调节器备忘单”，这是一个恰当的名称。他涵盖了主要的架构，甚至指出了为什么你不能总是相信供应商的某些类型的供应信息。

尽管它被宣传为备忘单，但该视频也涵盖了关于不同调节器如何工作及其效率和热特性的大量理论。和往常一样，他用实践证明来加强他的理论。他甚至以解释事物的名义释放一点魔法烟雾。

如果你想看看线性调节器的内部，我们看到了古老的 7805 的一个很好的拆解。我们不建议这样做，但从电阻器[Andreas]开始的下一步是[齐纳二极管](https://hackaday.com/2017/10/11/a-lesson-on-zener-regulators/)，这是我们去年讨论过的话题。
