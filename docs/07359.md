# 再现流言终结者火箭雪佛兰——按比例

> 原文：<https://hackaday.com/2017/10/13/recreating-the-mythbusters-rocket-chevy-at-scale/>

如果你收看了《流言终结者》的第一集，你会记得一辆火箭动力的雪佛兰黑斑羚。[[David Windestal]决定以 1:10 的比例重新制作。](https://www.youtube.com/watch?v=geFfphwuCrI)

问题中的汽车是一辆越野沙漠之狐遥控汽车——通常是一辆四轮驱动卡车，随时可以运行，这是一种快速启动项目的好方法。火箭动力由我们之前报道过的[瑞典火箭刀中使用的相同类型的马达提供。](https://hackaday.com/2017/08/15/swedish-rocket-knives/)

最初的测试证明是不成功的——当火箭发射时，汽车完全失去了控制。它需要一次打击，在这个过程中失去一个轮子。在此基础上，决定制造一个燃烧速度较慢的火箭发动机，并改用沥青路面进行测试。这是非常成功的，赛车开始看到一些适当的高速度，在这个过程中几乎剥离轮胎的边缘！

这是一个有趣的概念，也可以用现成的火箭发动机来复制。在下面的评论中告诉我们你对更好的火箭动力交通工具的想法。
