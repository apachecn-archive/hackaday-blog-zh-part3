# 错误:LP0 开启？？？？

> 原文:[https://hackaday.com/2017/04/25/error-lp0-on-%f0%9f%94%a5/](https://hackaday.com/2017/04/25/error-lp0-on-%f0%9f%94%a5/)

你不需要花里胡哨的集成电路和数模转换器来为个人电脑制作声卡。正如[serdef]在 hackaday.io 上的构建所示，你真正需要的是一堆电阻。[serdef]复制了 80 年代发布的 PC 声卡，但做了一些改进。这种混乱的电阻具有最佳的 8 位声音，您可以通过低通滤波器、音量分配器和方便的 DB-25 连接器获得。

这款 LPT0 声卡的设计与它作为 [Covox Speech Thing](https://en.wikipedia.org/wiki/Covox_Speech_Thing) 向世界推出时非常相似。这个“声卡”被设计成可以夹在计算机的并行端口上，并通过一个电阻梯发送该端口的 8 位 I/O 信号。把一对扬声器插在这个东西上，你就有了一个完全由电阻构成的声卡。它很便宜，而且在大众中很受欢迎。
