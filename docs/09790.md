# 三极管真空管的历史和物理学

> 原文：<https://hackaday.com/2018/06/22/the-history-and-physics-of-triode-vacuum-tubes/>

三极管真空管今天可能几乎过时了，但它是 100 多年前使无线电实用化的关键技术。[凯西]已经[制作了一个视频](https://www.youtube.com/watch?v=0Smj2nHo9zA)，讲述了这个故事并解释了这个设备的物理原理。

第一个无线电接收器使用一种称为相干器的设备作为检测器，依靠两个微小的灯丝，当施加射频时，它们会粘在一起，允许电流通过。这是一个有效的装置，但不可靠。正是在 1906 年，德·福雷斯特发明了一种无线电探测装置，它使用一个真空管，里面有一个板和一根加热灯丝。这个装置与约翰·弗莱明一年前申请专利的弗莱明阀门非常相似，以至于弗莱明起诉德·福里斯特侵犯专利。

经过一系列绕过专利的尝试后，德·福里斯特决定在电子管中加入第三个元素:网格。栅极是位于灯丝和极板之间的一块金属。施加在栅极上的信号将控制电子的流动，使这种装置可以像放大器一样工作。这一改进创造了三极管，并绕过了弗莱明的专利。
