# 智能手机里的转速表

> 原文:[https://hack aday . com/2018/02/08/智能手机内的转速计/](https://hackaday.com/2018/02/08/the-tachometer-inside-your-smartphone/)

这是这家设备齐全的商店的最新仪器——声学耦合快速傅立叶变换转速表。听起来很贵，但实际上只是[使用智能手机频谱分析仪应用程序间接测量工具速度](https://www.youtube.com/watch?v=xutm6oMTBw8)。它看起来非常方便。

通常，非接触式转速计是光学耦合的，使用光感受器来测量轴或工具的闪光。但是这需要对机器有一个清晰的视野，经常把手放在离危险区域太近的地方。[马蒂亚斯·万德尔]的方法不需要视线，因为它依赖于一个廉价的频谱分析仪应用程序来听机器的声音。该软件显示不同频率下的峰值，通过一点分析和一些简单的数学运算，就可以确定机器的轴速。[Matthias]解释了如何排除谐波，在哪里可以找到电力线嗡嗡声，隔离换向器伪像，以及如何进行所有的计算。你需要对你的工具了解一点，以获得正确的转速，显然你会受到你的手机或平板电脑的音频响应的限制。但是我们认为这是一个很好的建议。

[Matthias]对商店创新和以简单而优雅的方式运用技术并不陌生。我们想知道频谱分析能否用于发现谐波，并帮助[为承包商的台锯](https://hackaday.com/2017/09/16/a-fix-for-the-lightweight-machine-tool-shakes/)提供减振解决方案。
