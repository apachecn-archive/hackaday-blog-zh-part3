# 由机器人在白板上书写

> 原文：<https://hackaday.com/2018/04/26/writing-on-a-whiteboard-performed-by-a-robot/>

对我们中的一些人来说，学校只是一个非常遥远的记忆。我们宁愿忘记所有青少年的尴尬，synth pop 和 8 位计算机第一次很酷，我们更新的教室不再有黑板。白板的未来已经到来，它带有充满溶剂的笔，我们更叛逆的同学发誓，如果你长时间闻它们，它们会让你兴奋。纯真年代。现在的孩子可能从 iPads 上学习功课，但是白板还没有完成。[f4hdk]用 [Scribot 更新了他的白板，Scribot 是一个由几个步进电机和一组精心设计的皮带](https://hackaday.io/project/153194-scribing-robot)驱动的白板书写机器人手臂，它可以用定制设计的矢量字体从 ASCII 文件中书写文本。

在手臂的末端是一个白板标记，在它的后端有一个干净的橡皮擦。一个快速翻转伺服举行标记，它可以擦掉它的任何工作。在这一切的背后是一个基于 LPC1789 Cortex M3 的 Mbed 板和适当的伺服驱动板，对于那些足够好奇的人来说，有一个完整的代码库。正如你在休息后的视频中看到的，结果是一个非常好的白板写手。你 80 年代的老师可能会抱怨新技术，但肯定不会指责它做得不好！
