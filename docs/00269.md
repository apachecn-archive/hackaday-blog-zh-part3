# 带上你的机器人导管

> 原文：<https://hackaday.com/2015/10/06/take-your-robots-tubing/>

当你想到机器人时，你可能会想到有轮子、履带甚至腿的机器人。但今天的机器人很可能有翅膀、道具，甚至是某种推动它在水中前进的方式。如果你厌倦了四轴飞行器，想建造一个水上机器人，你应该看看开源机器人/船 [Jalcboat。](https://www.thingiverse.com/thing:470370)虽然该项目仍在开发中，但视频(见下文)显示他们已经取得了不错的进展。

正如你所料，3D 打印是一个关键因素，文件可以在上面链接的 Thingiverse 上找到。此外，机器人船上有一个树莓 Pi 来控制[无刷电机](https://www.thingiverse.com/thing:707341)。主网站是[更像是一个论坛](http://www.jalcboat.com/)，其中一些是西班牙语的，所以你可能想随身携带谷歌翻译。

另一个简单的方法来创造一个能在水中工作的机器人是从一个遥控玩具开始。然而，Jalcboat 已经对耐海藻螺丝等东西进行了修改，这在被黑客攻击的玩具上更难实现。我们喜欢戴着机器人内胎的样子，但是 3D 打印也可以生产出[更炫的](http://hackaday.com/2014/04/25/3d-printed-hydrofoil-boat-rc-flies/)。

冬天来了。但将会很好地利用室内时间为明年夏天设计一个水上机器人。或者你可以恐吓你当地的室内水上乐园，如果它很快完工的话。
