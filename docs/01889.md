# 用柠檬生火

> 原文:[https://hackaday.com/2016/03/26/lighting-fires-with-lemons/](https://hackaday.com/2016/03/26/lighting-fires-with-lemons/)

我们都知道，你可以把铜和锌粘在酸里，制成电池。典型的例子是用一个柠檬。YouTuber [NorthSurvival]将这一点发挥到了极致——通过将他的柠檬电池与一些钢丝绒短路来点火。(视频嵌入下方。)

现在称之为“生存技巧”是在推动它。很多。你上次带着一堆锌和铜钉去露营是什么时候，更不用说新鲜柠檬了？把一些火柴放在一个防水的罐子里，或者带一个打火机可能会更容易些。但是当僵尸启示录来了，所有的打火机都用完了，有柠檬树的人就成了百万富翁。

不过，说真的，这个演示让我们质疑了一些假设。首先，当人们做土豆或柠檬电池实验时，他们经常使用多个柠檬。为什么？像[NorthSurvival]在系列中所做的那样将引脚连接起来看起来是一件很容易的事情。

而且柠檬似乎正在产出相当多的果汁(安培，也就是说)。我们不禁要问——柠檬电池的短路电流是多少？为什么我们没有在任何地方看到规格？这到底是什么“科学教育”实验，没有测量？

所以我们要找一些铜和锌，自己测量电流。如果需要的话，再放一些柠檬。制造一台[点焊机](http://hackaday.com/2016/03/22/arduino-nano-runs-battery-spot-welder/)需要多少柠檬？

[编者按(完全红着脸):所以我们呼吁进行测量，人们在评论中提出了这些测量。柠檬电池出来的电流是毫安，也就是说这个视频是假的。除非柠檬中有比我们知道的更多的结构，否则共享电解质应该会使它短路，而不是产生更多的电压，当然是在任何有用的电流水平下。这是个骗局，我们被骗了。]
