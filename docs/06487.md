# 激光切割或系列

> 原文:[https://hackaday.com/2017/07/12/laser-cutting-orreries/](https://hackaday.com/2017/07/12/laser-cutting-orreries/)

奥瑞里是一个太阳系的钟表模型，展示了行星以一种崇高的本轮模式围绕太阳运行的机制。地球仪是地球仪的一个子集，显示地球围绕太阳的旋转，以及月球围绕地球的轨道。他用激光切割的零件和一些铜管及轴承制造了自己的地球仪。

这个项目最初是受 holzmechanik 的启发，这是一个由胶合板齿轮和铜管制成的地球仪。[huidont]在这个项目中看到了几个缺点:地球没有自转，月球也没有以其自然的五度倾角绕轨道运行。[惠东]的地球仪将具有这些特征，包括被照亮的太阳，展示季节的变化，并显示月食和日蚀。

虽然在计算齿轮传动时涉及到一点数学，但并不多:地球每 365.25 天绕太阳一周，月球每 27.32 天绕地球一周，恒星时间和太阳时间之间有差异。一个快速的脚本使数学快速工作，任何人都可以很容易地找到工具来创建给定直径和齿数的齿轮。

这种碲酸盐的制造是用丙烯酸在激光切割机上用少量 3D 打印部件制成的。电子设备足够简单——只需要一个马达和几个发光二极管，完成的项目运转良好。你可以看看下面的碲视频。
