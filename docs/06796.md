# Filaween 2.0 即将推出

> 原文:[https://hackaday.com/2017/08/14/filaween-2-0-is-go/](https://hackaday.com/2017/08/14/filaween-2-0-is-go/)

[Thomas Sanladerer]又来了:测试所有适合打印的 3D 打印机细丝。今年，他有一个[新的和改进的测试方法](https://www.youtube.com/watch?v=nlIFMy4vIeM)——视频嵌入下面。并搜索“filaween2”以查看他目前为止评论了什么。里面有一些性感的细丝。

我们真的很喜欢全新的冲击强度测试，在这个测试中，一个锤子在一个枢轴上摆动(3D 打印，当然)，穿透测试中的零件，然后摆动回到可测量的高度。摆动高度的差异反映了破坏试件所需的能量。甜蜜的物理。

[Thomas]去年运行了一个类似的几个月的系列[，我们很高兴看到它带着所有的改进回来。为看着古怪的塑料融化干杯！](http://hackaday.com/2016/12/10/thirty-days-of-3d-printing-filament/)
