# 雄心勃勃的 Hackerboat 项目仍然志在必得

> 原文:[https://hack aday . com/2017/06/28/ambition-hacker boat-project-still-aiming-high/](https://hackaday.com/2017/06/28/ambitious-hackerboat-project-still-aiming-high/)

去年，我们写了关于 Hackerbot 实验室的自主船，项目成员希望有一天能够环游世界。现在被称为“拉冬计划”的项目进展迅速，最近他们改装的 18 英尺皮艇“T2”号 TSV 号争议通行权“T3”进行了试航。皮划艇的内部空间包含一对由自制控制系统控制的铅酸卡车电池，该控制系统使用继电器控制船只的拖曳电机，引擎盖下有一个比格犬骨和 Arduino Mega。

测试并不完全成功，船实际上避开了航路点，而不是坚持它们。幸运的是，这支队伍在一艘追踪船上，所以他们能够密切关注这艘船。不像四轴飞行器那样会掉下来，一艘停泊的船只可能再也见不到了。

进入 [2016 Hackaday 奖](http://hackaday.com/2016/06/17/building-a-swarm-of-autonomous-ocean-boats/)后，该项目持续升温，在 [Toorcamp](https://hackaday.io/project/8522-project-ladon/log/40067-hackerboat-at-toorcamp-2016) 和 [Maker Faire Bay Area](http://makerfaire.com/maker/entry/60458/) 都有展示。此外，他们还在维护他们的 [Hackaday.io 项目网站](https://hackaday.io/project/8522-project-ladon)以及 [Patreon 页面](https://www.patreon.com/hackerboat)。
