# 站起来，小鸡，站起来！

> 原文：<https://hackaday.com/2017/05/23/arise-chicken-arise/>

几个月前，[迈克]开始从他吃的所有炸鸡中收集骨头。如果这是开场白，你就知道会很精彩。

这个电子鸡项目源于【迈克】对战斗机器人的热爱，以及极度缺乏战斗*的博格人*。差别虽然很小，但却很明显:机器人只是一台自动或通过遥控执行指令的机器。另一方面，电子人包含有机和生物机械的身体部分。因为[迈克]在保存鸡骨头，他偶然想到用垃圾、几个伺服系统、一个 MSP430 和他的垃圾抽屉里的一些其他部件来创造一个电子人。

作为早期遥控食品项目的延续，这些鸡战堡的能力和你想象的差不多:它们在轮子上打滚，疯狂地挥舞着它们的鼓槌。[迈克]已经制造了至少两个这样的装置，结果被准确地描述为摇滚袜子博格。看看下面的视频。

在硬件方面，[Mike]拿起一台 MSP430，用 Java 编写了一些代码。三十亿个企业计算系统，现在，两只电子鸡运行 Java。电机和驱动器来自 Pololu，控制是由一对 Atari 操纵杆通过红外提供的。

你可以看看下面这些电子鸡的视频。如果非要问为什么，答案永远是‘因为’。
