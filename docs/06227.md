# 意想不到的一连串失败导致微波的消亡

> 原文:[https://hacka day . com/2017/06/16/occupatible-cascade-of-failures-leads-to-microwave-stever/](https://hackaday.com/2017/06/16/unlikely-cascade-of-failures-leads-to-microwaves-demise/)

一个吹坏的灯泡肯定杀不死一个微波炉吧？您可能不希望如此，但这确实是[mikeselecticstuff]最近调查的一个问题的根本原因；至少可以说，一连串的失败是有教育意义的。

虽然微波炉来到[迈克]的工作台上并不是为了失败而设计的，但它肯定不是为了成功而设计的。我们不会破坏这个惊喜，但可以说的是，在车主报告它死之前的一声巨响后，他对快速修复的希望被内部灯泡的电弧打破了，这种电弧将电源电压脉冲放在了它不属于的地方。一连串的故障导致了这款设备的死亡，这证明了针对某个价位的设计如何限制了设备在数以百万计的产品被装进集装箱运往海外市场之前的测试程度。

尽管[mike]尽了最大努力坚持维修宣言，但最终结果还是一个报废的微波炉。鉴于里面有趣的部分，这并不是一个完全的损失，但仍然令人失望，除非它迫使我们在设计中记住边缘情况下的故障模式。
