# 用定制的钢零件修理损坏的带锯

> 原文：<https://hackaday.com/2016/06/04/fixing-a-broken-bandsaw-with-a-custom-steel-part/>

当一个大型带锯由于铸铁部件断裂成两半而发生故障时，[Amr]利用这个机会记录了使用(非 CNC)轧机和车床为断裂部件设计和制造实心钢替代物的整个过程。

对于我们这些不熟悉机械师完成这样一件事的过程的人来说，这个视频极具教育意义；看到在任何东西通电之前发生了多少设计工作，以及有多少时间和工作用于切割和成型一些钢成为乍看起来相对简单的部件，这可能是令人清醒的。
