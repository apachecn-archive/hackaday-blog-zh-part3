# 3D 打印的应变波齿轮需要你的帮助

> 原文：<https://hackaday.com/2017/01/17/3d-printed-strain-wave-gear-needs-your-help/>

在大多数机械系统中，弯曲的金属齿轮是一件坏事。但对于应变波传动装置来说却不是这样，这种传动装置设计成利用金属齿轮的挠曲来实现类似行星齿轮的动作。然而，乐趣不再局限于金属，如果你像这样 3D 打印一个应变波齿轮。

应变波传动装置并不新鲜——它于 1957 年被发明，并已乘坐月球车前往月球。你可能还记得[【Kristine Panos】最近的一篇关于乐高应变波齿轮](https://hackaday.com/2016/12/09/lego-strain-wave-gear-is-easy-on-the-eyes/)的文章，这篇文章让你很容易想象它们是如何工作的。她还很好地描述了 flex spline、wave generator 和 circular spline 是如何交互的，因此我们将在此省略这些细节。[Simon Merret]对应变波齿轮的解释非常简单，与其他 3D 打印版本相似，只是他使用了一条由内向外的正时皮带作为柔性花键。波浪发生器只是一个两端带有滚柱轴承的手臂，尽管需要一些调整，齿轮做得很好。

Simon 正在寻求帮助，准备将这种设备用于工业版本经常应用的地方——机械臂的第一和第二自由度。如果你有任何想法，请访问他在 Hackaday.io 上的项目页面并参与进来。
