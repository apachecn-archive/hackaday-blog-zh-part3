# 彩虹猫宣布订婚

> 原文:[https://hack aday . com/2016/06/08/rainbow-cats-announce-engagement/](https://hackaday.com/2016/06/08/rainbow-cats-announce-engagement/)

[安泰人寿]将于 2017 年的某个时候结婚。他不想发送纸质公告或更新他的脸书状态，而是想给他们的家人一些持久难忘的东西，比如一个小饰品和一对发光的猫。

这个项目理论上很简单。一对 RGB 发光二极管在为每个发光二极管精心选择的电阻和 ATtiny25 的帮助下，在彩虹的颜色中循环。但是这里有几个挑战。[antalive]希望设计一种非常小的东西，在一个 CR2032 硬币电池上至少可以使用一天。这个项目是他首次涉足 SMD/SMT 设计和建造。我们认为这值得祝贺，特别是因为看起来他至少做了一打这样的东西。

[antalive]购买了一个廉价的热风返工站，并使用芯片夹对“tiny”进行编程，让事情变得简单多了。这些猫是 Thingiverse 的一个设计，他修改后把它们变成了新娘和新郎。休息之后看他们一整行发光。我们真诚地希望这些猫的更大版本最终出现在婚礼蛋糕上。

对于任何有着永恒爱情的人来说，不要忘记[灯光邀请](http://hackaday.com/2013/01/30/really-really-geeky-wedding-invitations/)、[婚纱](http://hackaday.com/2013/08/16/electronic-wedding-attire-for-a-geeky-wedding/)和[中心装饰品](http://hackaday.com/2013/03/28/centerpieces-for-a-geeky-wedding/)。
