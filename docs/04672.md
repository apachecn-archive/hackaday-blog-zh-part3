# 机动摄影小车随着变化而滚动

> 原文：<https://hackaday.com/2017/01/14/motorized-camera-dolly-rolls-with-the-changes/>

在最后一个学期，康奈尔大学的学生[Ope Oladipo]有机会结合他的两个爱好:工程和摄影。他和他的队友[萨塞斯·海格德]和[张杰]利用在[布鲁斯·兰德]课上的时间建造了一个用于拍摄延时镜头的机动摄影车。

在这种情况下，相机来自 iPhone 6，它安装在现成的机器人底盘上，由一对 DC 电机驱动。相机支架使用步进电机来拍摄合适的照片。机器人上的 PIC32 从团队开发的 iOS 应用程序中获取蓝牙命令。推车有两种工作方式:可以在自由模式下手动控制，也可以在编程模式下以设定的速度在指定的时间内遵循预定的路径。

我们最喜欢的建筑部分？相机的视角被传送到一个智能手表，在那里[Ope]和他的团队可以使用手表端的界面拍摄静态照片。休息后请查看，并留下来观看一个简短的延时演示。这些年来，我们已经推出了几个多莉造型。这里有一个更传统的推车，骑在一对可延展的管子上。
