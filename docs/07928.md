# DIY 木制积木

> 原文:[https://hackaday.com/2017/12/07/diy-wooden-building-blocks/](https://hackaday.com/2017/12/07/diy-wooden-building-blocks/)

如果你有钻床、电锯和打磨机，并且正在为这个假期的小孩寻找一个很棒的礼物，[Jonny]可能会满足你。他想出了一个很好的方法来制作一些类似于修补玩具积木 [s](https://imgur.com/gallery/Trt4o) 的简单积木和榫钉积木。

如果你有车间工具，这是一个相当简单的构建，如果你只有手工工具可用，这仍然是可行的。这些木块由方形木块组成，木块上钻有孔，还有一串切割成一定尺寸的木榫。[Jonny]添加了一个带有铰链盖的木箱，用于存储积木，作为建筑的附加功能。

没有发光二极管亮起，没有 Arduino 驱动的微控制器参与，它们没有连接到互联网，但这并没有使它们成为一个伟大的玩具。即使没有车间工具，即使没有木工经验的人也能很快制作出来。如果你对积木玩具感兴趣，可以看看这篇关于如何[将不同类型的积木组合在一起的文章，](https://hackaday.com/2012/03/20/connecting-toy-blocks-with-a-universal-construction-set/)或者看看这篇关于用金属组装套件创建 DIY 数控铣床的[框架的文章。](https://hackaday.com/2017/08/10/cnc-mill-out-of-a-building-set/)
