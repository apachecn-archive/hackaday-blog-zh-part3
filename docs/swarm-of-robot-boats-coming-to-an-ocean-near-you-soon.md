# 一群机器人船很快就会来到你附近的海洋

> 原文：<https://hackaday.com/2016/02/12/swarm-of-robot-boats-coming-to-an-ocean-near-you-soon/>

计划恶意收购你当地的游泳池？这可能会有所帮助:Anders Lyhne Christensen 博士]给我们发来一份关于他在葡萄牙电信研究所生物机器实验室工作的说明。他们已经建造了一个机器人船群来试验自主群体，并取得了一些很好的结果。

在一个自主的群体中，每个机器人都做出自己的决定，并与邻居交谈，群体的组合行为产生了一个整体行为，就像窝里的蚂蚁一样。他们已经创造了能够自主导航、巡逻某个区域或监控某个区域的温度并返回基地报告结果的蜂群。在一个精彩的视频中，[Anders]概述了他们如何使用计算进化来创造这些行为，随机变异神经网络以找到最佳方法，然后将其发送到真正的船只。

也许最酷的是:整个项目是开源的，每艘船的[大脑都在树莓皮](https://github.com/BioMachinesLab/drones)上运行，数控铣削泡沫船体带有 3D 打印组件支架。每艘船的成本约为 300 欧元(约 340 美元)，但你可以通过回收部件来降低成本，一旦更便宜的 Pi Zero 成为可能。这个项目无疑会对许多讨厌在当地游泳池被恶棍泼溅的邪恶天才有用:一群杀手机器人围绕在他们周围是阻止他们的绝佳方式。
