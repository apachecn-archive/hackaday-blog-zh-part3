# 令人瞠目结舌的示波器上的无 IC 乒乓

> 原文：<https://hackaday.com/2017/03/31/jaw-dropping-ic-free-pong-on-an-oscilloscope/>

Pong 可能已经不算什么了，但它是所有视频游戏的鼻祖，通过研究它的内脏，还有很多东西要学。还有什么比在你玩的时候把一切都摆在你面前更好的方法呢？只需要 200 个分立晶体管和两大把无源器件粘在一块覆铜板上，就可以得到一个没有单芯片的 Pong 版本,可以在示波器上播放。

很明显，如果不是一时的疯狂行为，这也是一种爱的努力，[GK]对 Pong 的认识是值得一看的。它的每一个碎片都是他自己设计的电路，以死臭虫的方式执行，显然是因为[GK]喜欢硬模式的生活。游戏本身的可玩性令人惊讶，你甚至可以和机器对战。下面的视频有点难以观看，示波器 CRT 上有些眩光，但我们会让[GK]在这个视频上放松很多；毕竟，看起来整个项目是在一个马拉松式的周末构建会议中完成的。

我们仍然忙于钻研手绘的 [Forrest Mims 风格的](http://hackaday.com/2017/01/18/forrest-mims-radio-shack-and-the-notebooks-that-launched-a-thousand-careers/)原理图，这些原理图本身几乎就是模拟设计的一门完整课程。很多赛道让我们想起了 GK 的 T2 弹跳球模拟，我们之前已经讨论过了。
