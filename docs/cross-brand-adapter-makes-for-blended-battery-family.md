# 跨品牌适配器有助于混合电池系列

> 原文：<https://hackaday.com/2018/02/20/cross-brand-adapter-makes-for-blended-battery-family/>

尽管[理查德·戴]是一个忠实的 DeWalt 无绳工具爱好者，但他承认在工具通道里有一只游荡的眼睛，带着不纯洁的想法看着 Ryobi 的产品。他会忠于自己的品牌，继续使用他的大量黄色工具和电池吗？或者他会屈服于诱惑，添加另一套电池和充电器，这样他就可以使用一些特殊的石灰绿色工具了？

幸运的是，我们生活在未来，所以有第三种方法——制造一个跨品牌的电池适配器，让他用他的 DeWalt 电池为 Ryobi 工具供电。[理查德]的解决方案是一个纯粹的黑客，就像在物理上入侵电池组，并迫使它们一起工作和娱乐。从机械上来说，这很容易——家得宝回收站里的一个报废的 Ryobi 包被拆下来放在盒子里，盒子粘在 Dewalt 20-v 到 18-v 的电池适配器上。棘手的部分来自处理电池控制电子设备。幸运的是，捐赠者 DeWalt line 在适配器中有这个电路，而 Ryobi 将其放在电池中。这意味着简单地将 PCB 从适配器移植到 Ryobi 电池外壳就足够了。下面的视频展示了过程和结果——Ryobi tools 愉快地点击 DeWalt 电池。

虽然[理查德]在这里采取了一种有点蛮力的方法，但我们想象 3D 打印零件可能会成为一种更优雅的解决方案，并提供其他品牌组合。毕竟，打印一个适配器应该比重新组装一个无线电池组更容易。
