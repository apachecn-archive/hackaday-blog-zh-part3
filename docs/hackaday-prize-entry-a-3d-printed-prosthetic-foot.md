# Hackaday 奖参赛作品:3D 打印假脚

> 原文：<https://hackaday.com/2016/10/15/hackaday-prize-entry-a-3d-printed-prosthetic-foot/>

在过去几年的 Hackaday 奖中，出现了不少假肢装置。几乎毫无例外，这些项目的目标都是假手。这是一个值得称赞的目标，但至少在机械方面，脚更有趣。一只人类的脚必须承受比它所依附的人更大的重量，当它由塑料和金属制成时，这意味着一些疯狂的力学。

这个 Hackaday 奖的参赛作品完全颠覆了我们以前见过的所有假肢。[这是一只假脚](https://hackaday.io/project/13312-mechanized-prosthetic-foot)，在容易制造和容易修改的假肢的传统中，这只假脚大多是 3D 打印的。

一只脚将比一只手承受更多的虐待和重量，由于这种 3D 打印，所有部分可能看起来不是最好的主意。然而，奇异的细丝是存在的，这个项目背后的团队确实可以在材料工程实验室中使用一些测试设备。有了正确的几何形状，一切似乎都支持所需的负载。

这款 3D 打印假脚有一些相对较新的变化，包括电子控制、微型水力发电厂和测量和调整用户步态的传感器。这一切都很酷，值得比最复杂的 3D 打印假手更多的工程设计。
