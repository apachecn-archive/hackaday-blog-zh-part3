# 用这款 LED 时钟点亮你的一天

> 原文：<https://hackaday.com/2015/12/05/light-up-your-day-with-this-led-clock/>

我们喜欢时钟，而【克里斯】用[互联网激活的光钟](https://hackaday.io/project/6925-the-light-clock)吸引了我们的注意力。时间通过一个 3D 打印的白色圆盘周围的 RGB LED 条以多种不同的方式显示。所有模式都基于两种可选颜色来指示小时和分钟，以渐变方式或硬停方式显示。

光线由一个 144 LED 新像素条提供，由一个结实的 4 安培 5 伏电源供电，该电源也为控制器供电。大脑由 ESP8266 供电的 NodeMCU-12E 板提供，软件使用用于 Arduino 内核的 ESP8266 编写。

作为一个支持 WiFi 的微控制器，使用 WiFi 连接到时钟并使用嵌入式网页选择您的本地时区、调色板和显示模式是一件简单的事情。正确的时间是由网络设定的，永远不会错。虽然有一个 [Kickstarter](https://www.kickstarter.com/projects/1054187792/the-light-clock) 用于出售完成的项目，但如果你愿意，也可以提供[指令和软件](https://hackaday.io/project/6925/instructions)来制作自己的项目。
