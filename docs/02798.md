# Hackaday 奖参赛作品:自拍机器人让你解放双手

> 原文:[https://hack aday . com/2016/07/08/hack aday-prize-entry-self ie-bot-lets-you-vlog-hands-free/](https://hackaday.com/2016/07/08/hackaday-prize-entry-selfie-bot-lets-you-vlog-hands-free/)

谢尔盖·米罗诺夫提交了他的自拍机器人项目。他的公司耐力机器人(Endurance Robots)出售这种机器人的商业版本，这让我们相信，他决定以一种奇怪、也许是聪明的方式，将产品开发的原型阶段作为一个套件出售。因为他也给出了固件，STLs，BOM，并制作了一个指南，所以任何人都可以建立它，我们没有抱怨。

这个机器人很简单。漂亮的 3D 打印盒中的业余爱好伺服系统负责相机的平移和倾斜。机器人的底部封装了电子设备，包括 Arduino nano、蓝牙模块以及用于电源和电机驱动的支持电子设备。

为了执行面部跟踪，构建假设您有第二部手机。这很傻，但也不是不合理。大多数拥有智能手机几年的人都会在抽屉里放一部备用手机作为备份。一部手机运行面部跟踪软件，通过蓝牙将机器人指向用户。另一部手机录制视频。

在示例视频中，这个机器人相当跳动，但这可以用更好的[马达](http://hackaday.com/2016/05/02/simple-robot-arm-with-steppers-has-pleasingly-smooth-motion/)来解决。对于概念验证来说，这是可行的。休息后的视频。
