# 复活 Amiga CD32

> 原文:[https://hackaday.com/2018/02/16/resurrecting-an-amiga-cd32/](https://hackaday.com/2018/02/16/resurrecting-an-amiga-cd32/)

作为前世 Amiga 杂志的编辑，这有点苦乐参半。[RetroManCave]被捐赠了一个 Amiga CD32 游戏系统，它正在[试图复活它](https://youtu.be/CkVzXi7StKk)。如果你没有听说过，CD32 是 1993 年基于 Amiga 家用电脑系统的游戏机。这是 Commodore 的最后一搏，它是 Amiga 背后的困境公司。在这个系列的第一个视频中，他们把系统拆开，带你看里面是什么，然后启动它。系统启动了，但是视频同步出现了一些问题，接下来他们会仔细研究解决这个问题。我们之前已经展示了几个来自[RetroManCave]的类似项目，例如他们在大 Trak 玩具上的[大脑移植](https://hackaday.com/2018/01/18/big-trak-gets-a-new-brain/)和[Commodore 64 fix](https://hackaday.com/2017/12/21/can-this-commodore-64-be-saved/)。如果你对这样的旧系统感到好奇，想要一些关于复活旧硬件的提示，或者只是想在你浪费的青春在你眼前被撕裂时流泪，那么这个视频(休息后)值得一看。
