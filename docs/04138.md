# 75 美元的 DIY 真空吸尘工具

> 原文：<https://hackaday.com/2016/11/17/a-diy-vacuum-pickup-tool-for-75/>

如果你自己组装 SMD 板的原型，精确地放置零件可能是一件痛苦的事。当然，如果有一台完整的拾放机就好了，但是安装起来非常昂贵和耗时，尤其是对于小批量的电路板。相反，真空拾取工具可以帮助您用手快速准确地放置零件。

Ohmnilabs 的人们花了大约 75 美元组装了他们自己的 [DIY 拾音工具](https://ohmnilabs.com/diy-vacuum-pickup-smt-pick-and-place/)，这已经成为他们内部原型制作过程的一部分。他们厌倦了用镊子放置元件，这需要你在提起它们之前从胶带上取下部件，并且有在最糟糕的时候翻转部件的倾向。

这个构建由几个部分组成，可以从亚马逊上买到。电动真空泵进行抽吸，真空水平由可调降压转换器调节。坚固的脚踏开关让您的双手保持自由，注射器尖端用于拾取零件。

这看起来像是一个简单的下午构建，但是如果你正在构建原型，它可以节省你大量的时间。要查看它的运行情况，请在休息后观看视频。
