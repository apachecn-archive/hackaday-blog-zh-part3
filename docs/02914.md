# BST-1 汽车短波接收机

> 原文:[https://hack aday . com/2016/07/24/BST-1-car-短波-接收机/](https://hackaday.com/2016/07/24/bst-1-car-shortwave-receiver/)

通勤是一种痛苦。幸运的是，几乎每辆车都有某种收音机或其他音频播放器来消磨交通堵塞的时间。然而，大多数收音机都有 AM 和 FM 波段，还有一个天气波段，也许还有一个长波波段。如果你更喜欢短波呢？

[Thomas]发布了一篇关于 BST-1 的评论，这是一款适合汽车使用的短波接收器。该设备安装在看不见的地方，大概是在外部天线附近。它向汽车的调频收音机发射短波信号。控制是一个小的钥匙链，即使你对收音机本身不感兴趣，用户界面的设计也很有趣。

按下两个遥控钥匙按钮中的一个，就会产生一个共同的动作。但是，如果您按住按钮，您会听到一声哔哔声。一直按住它，你会听到两声哔哔声。这种情况一直持续到您听到四声蜂鸣音。当您在一系列哔哔声后松手时，您会执行不同的功能。例如，在两声蜂鸣声后松开底部按钮，在预设和调谐模式之间切换，而在三声蜂鸣声后松开按钮，在 3 kHz 和 5 kHz 之间切换接收器带宽。

你可以在下面看到一个关于 BST-1 的视频。审查得出的结论是，BST-1 不是一个弱信号工作的严重接收器，但就其预期目的而言，它是一个很大的乐趣。

接收器不处理边带；这是 [AM 专用的](https://hackaday.com/2016/06/07/am-the-original-speech-transmission-mode/)。如果你觉得短波很无聊，也许你会发现一些[间谍活动](https://hackaday.com/2015/11/23/swans-pigs-and-the-cia-an-unlikely-radio-story/)潜伏在周围。
