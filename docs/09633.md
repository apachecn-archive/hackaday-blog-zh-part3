# 磁性球体排队旋转编码器的职责

> 原文:[https://hack aday . com/2018/06/05/magnetic-spheres-line-up-for-rotary-encoder-duty/](https://hackaday.com/2018/06/05/magnetic-spheres-line-up-for-rotary-encoder-duty/)

谈到旋转编码器，有很多选择。不过，大多数方法都需要将信用卡号码放入在线供应商的网站，而这有时并不在卡片上。在这种情况下，构建自己的编码器，比如使用磁性球体的编码器，也是一种非常酷的方式。

如果他没有那么多时间，我们想象[Antonio Ospite]会选择一个商业解决方案，而不是从头开始构建一个编码器。然后，他说他的应用程序有噪声的考虑，所以这可能是最好的解决方案。他有一些锁存霍尔效应传感器，但缺少磁性编码器中通常与此类传感器一起使用的环形磁铁。但幸运的是，他有一堆直径为 5 毫米的磁性球体。这些球体围绕一个由 CD 轴制成的旋钮排成一圈，它们以交替的磁极定向，这正是霍尔传感器想要看到的。传感器排列成脉冲间隔 90 °,可以分辨 4.29 步。看看下面的视频，看看它是如何工作的。

小、便宜、有效永远是好东西。但是磁铁并不是家酿旋转编码器背后的唯一东西。在紧要关头，几个微型开关可能就够了，或者甚至[报废硬盘](http://hackaday.com/2018/01/07/scrap-a-hard-drive-build-a-rotary-encoder/)就够了。
