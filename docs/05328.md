# 安妮号旁边的卡车下面有一个新的

> 原文:[https://hack aday . com/2017/03/17/theres-a-mew-under-the-truck-next-the-ss-Anne/](https://hackaday.com/2017/03/17/theres-a-mew-underneath-the-truck-next-to-the-ss-anne/)

在我们深入探讨这个问题之前，我需要花一两个段落来传达 1996 年一个十二岁孩子的知识。当然，大多数 Hackaday 的读者至少有一次是 12 岁，但我们无论如何都要这样做。回报是什么？这是一种针对口袋妖怪的任意代码执行病毒，可能是有史以来最令人惊讶的 Game Boy hack。

在第一代口袋妖怪游戏中，有一个非常罕见的口袋妖怪。第 151 只口袋妖怪 Mew 可以学会游戏中的每一个动作。这是一种精神类型，在第一代中被压制了。你不能获得一个新的，除非带你的游戏机去参加一个特殊的活动(或者去玩具反斗城一次)。如果有人在操场上有一个新的，他们真的只有一个游戏。

围绕着新有一个神话。传说如果你去圣安妮，用力量移动一个在游戏中没有出现的卡车精灵，一个新的精灵就会出现。由于游戏的故事情节，你没有能力在第一次经过这辆卡车的时候到达它。然而，如果你开始了一个新游戏——因此失去了你所有的进度和你的整个口袋妖怪名单——你可以测试这个理论。别担心，你可以把你所有的好口袋妖怪都给我。一旦你有了新房子，我会还给你的。去你的，迪伦。去你的。

现在新卡车的把戏是真的了。你可以在原版游戏机上复制红色或蓝色。如果这种黑客在 1998 年存在，孩子们会失去他们该死的头脑。

这个黑客[的基础来自【Mr cheeze】](https://www.youtube.com/watch?v=h5Igc18hc2Q)，他为第一代口袋妖怪游戏创造了一种“病毒”。基本上，给定手动编辑保存文件的能力，通过游戏链接电缆复制这个保存文件是可能的。结果是一片混乱，但是每个口袋妖怪游戏完成后都有相同的保存文件。

将这种病毒与任意代码执行结合起来，您会发现一些不同寻常的东西。[MrCheeze]创建了一个保存文件，允许您将卡车移到 SS Anne 旁边。当卡车移动时，会出现一个 Mew。这正是每个人在他们姐妹的*后街男孩*马拉松比赛中谈论的*。*

新的“新卡车病毒”不像第一次尝试自我复制保存文件那样容易出错。事实上，除了几秒钟的音乐故障，这个口袋妖怪病毒没有任何异常。只有当新卡车的把戏被尝试的时候，一些事情才显得奇怪，而奇怪只是因为我们知道这不应该发生。结合这种病毒的自我复制特性，你就有了本来会引起大 n 注意的东西，这是一个基于口袋妖怪的任意代码执行的杰作，也是一个可能永远无法匹敌的黑客。
