# 台球机器人注定要在河城闯祸

> 原文：<https://hackaday.com/2017/05/10/pool-playing-robot-destined-for-trouble-in-river-city/>

你会认为台球应该是机器人玩的简单游戏，对吗？这都是数学——计算角度的几何学和处理移动球需要多大力量的基础物理学。最重要的是，它被限制在二维空间，所以应该很容易。

任何一个台球玩家都会告诉你，在现实生活中，这个游戏还有很多很多，但是，和一个机器人打台球仍然是一个巧妙的把戏。作为实现这一目标的一步，[BVarv]明智地决定制作一个台球机器人的微型模型，并在这个过程中重新发明了台球桌本身。意识到履带式或轮式机器人将很难在传统台球桌的底座周围移动，他的模型台球桌是无腿设计，看起来像宜家的东西。但是基座支撑允许机器人附着在桌子上，并绕着整个圆圈摆动，这允许他通过运动学工作，如下面迷人的停止动作视频所示。

[BVarv]已经在挥杆轴以及最终将握杆的手臂上实现了运动控制。他计划通过头顶上的图像分析来识别镜头，当然还有制作全尺寸的东西要做。我们很想和机器人打一两场比赛，所以我们希望他能到达那里。与此同时，来点机器人空气曲棍球怎么样？
