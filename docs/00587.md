# 摩擦焊…木头？

> 原文:[https://hackaday.com/2015/11/08/friction-welding-wood/](https://hackaday.com/2015/11/08/friction-welding-wood/)

你可能会认为为 Hackaday 写作意味着我们可能不会经常对项目感到惊讶……但后来我们看到了一些我们从未想过可能的事情——在这种情况下，*木材*的[线性摩擦焊接加入其中。](https://www.youtube.com/watch?v=X0k04hjdYuQ)

摩擦焊接(也称为搅拌焊接)是这样一种过程，即拿起两块材料(通常是金属或塑料)，在推动同一材料的静止块的同时，超快速振动其中一块材料，由此产生的摩擦会产生大量热量，然后可以将两块材料焊接在一起。

使用 dremel 和一些 3D 打印细丝将塑料部件粘合在一起是一种简单的方法，虽然用金属做这件事要困难得多，[在家里也可以做到。](http://hackaday.com/2015/07/03/getting-mixed-up-with-home-stir-welding/)

但是根据休息后的视频——用木头做这个实际上是可能的。

他们使用的是线性摩擦焊接机，它以大约 200 赫兹的频率振动样品。他们最有可能的猜测是木材中的木质素细胞正在融化、互锁并重新结合——产生了惊人的强结合。
