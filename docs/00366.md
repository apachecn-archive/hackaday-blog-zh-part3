# 手摇回旋加速器

> 原文:[https://hackaday.com/2015/10/14/hand-cranked-cyclotron/](https://hackaday.com/2015/10/14/hand-cranked-cyclotron/)

好吧，实际上不是回旋加速器，但这个球形回旋加速器是回旋加速器的一个很好的模型，它背后的概念让人感觉古怪而神奇。一对[乒乓球在玻璃碗周围尖叫](http://rimstar.org/science_electronics_projects/ball_cyclotron_electrostatic_accelerator.htm)感谢静电的排斥力。

毫不奇怪，这来自 Rimstar，我们已经将它等同于迷人的家庭实验室实验，如[离子风驱动的星际旅行企业](http://hackaday.com/2013/12/09/ion-wind-propelled-star-trek-enterprise/)。那些密切关注的人会知道大多数(史蒂文·杜弗兰的)实验都涉及高压，这次也不例外。他在[Tea 激光演示](https://hackaday.com/2015/07/08/legit-hack-creates-tea-laser-power-by-mr-wimshurst/)中使用的同一台 Wimshurst 机器将再次用于这次演示。

玻璃碗因其形状和作为绝缘体的特性而被使用。以铝带的形式添加一组电极。使用 Wimshurst 机器，它们被赋予相反的电荷。涂有导电涂料的乒乓球很轻，可以通过静电场移动，一个好的曲柄可以让它们在碗周围以非常快的速度运动。

当你移动一个曲柄时，用链条连接到某个东西的想法会突然出现在你的脑海里。这感觉非常相似，但是在球的运动和你在曲柄上的手之间没有直观的联系。有人需要万圣节派对的道具吗？

如果你不想购买或建立一个 Wimshurst 机器，你可以使用范德格拉夫发电机。有没有人能推荐其他在这里工作良好的高压电源？
