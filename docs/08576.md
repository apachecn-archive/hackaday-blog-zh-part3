# 一朵完美的玫瑰，一台步进机，一台 x 光机，还有你

> 原文:[https://hack aday . com/2018/02/14/a-perfect-rose-a-stepper-an-x-ray-machine-and-the/](https://hackaday.com/2018/02/14/a-perfect-rose-a-stepper-an-x-ray-machine-and-thee/)

强制浪漫的日子又一次来到了，为了认可这一事实，我们向你提交了浪漫爱情象征之一的另一种观点:[玫瑰的 x 光片](https://hackaday.io/project/52104-x-raying-rose-in-30kev-in-high-resolution)。

通常，诊断 X 射线是组成各种器官和器官系统的组织的不同密度的温和表示，通常以灰色阴影呈现。但是[Linas K]，一个有时间可以消磨和使用一些很酷的工具的物理学家，在下面的视频中捕捉到的图像不仅色彩鲜明，而且像电影一样。该项目的成像部分包括一个通常用于非临床应用的低功率 X 射线管和一个 CMOS 传感器面板。第二个视频显示[Linas]对 X 射线控制器进行了令人印象深刻的升级，从 FPGA 开始滚动他自己的。这使得他可以安装一个步进电机来旋转玫瑰，同时拍摄他在后期拼接的图像。

对于某些浪漫的子集来说，观察花朵旋转时的内部结构本身就是迷人而浪漫的。说真的，谁会不感激为此付出的努力呢？但是如果你没有 x 光设备，不用担心——一份可爱的情人节礼物只是一瓶氯化铁。
