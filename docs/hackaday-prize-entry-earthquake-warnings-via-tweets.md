# 黑客日奖参赛作品:通过推特发布地震预警

> 原文：<https://hackaday.com/2017/06/10/hackaday-prize-entry-earthquake-warnings-via-tweets/>

地震波以大约每秒 4 千米的速度穿过地壳。光以每秒 20 万公里的速度穿过光纤。考虑到网络延迟，在地震开始前几秒阅读一条关于地震的推特是可能的。这就是 XKCD 漫画和黑客大奖项目背后的概念。

[Zalmotek]的地震验证小工具是一个联网的盒子，设计用于在问自己“这是地震吗”和说“是的，这完全是地震”之间的几秒钟。在这个壁挂式盒子里，有一个灵敏的振动传感器和一个连接到互联网的微控制器。如果振动传感器响起，它会检查互联网——顺便说一下，美国地质调查局的网站是一个很好的开始——任何大的本地地震。如果晃动有可能是地震，警灯和警笛就会响起，告诉你要躲避。

“地震预警装置”的想法并不新鲜。美国地质勘探局已经为这类事情建立了一个系统。不过，很高兴看到独立的研究人员在这方面开展工作，这是获得 Hackaday 奖的一个很好的机会。
