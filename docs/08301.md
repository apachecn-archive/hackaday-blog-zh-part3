# 交互式 LED 桌

> 原文:[https://hackaday.com/2018/01/21/interactive-led-table/](https://hackaday.com/2018/01/21/interactive-led-table/)

一些黑客制造一些功能性的东西，你不能让它出现在有礼貌的公司里。其他人制作的漂亮东西可能来自高端商店。[Marija]属于第二类，她的[交互式 LED 咖啡桌](http://www.creativityhero.com/diy-projects/diy-interactive-led-coffee-table/)可能会在零售市场上卖得很好。你可以在下面看到这个看起来很棒的桌子的视频。

不仅仅是玻璃、中密度纤维板和松木结构。还有一个蓝牙接口连接到[一个来自【Dejan】的定制 Android 应用](http://howtomechatronics.com/tutorials/arduino/how-to-control-ws2812b-individually-addressable-leds-using-arduino/)，他参与了这个项目。然而，如果你对木工不满意，[Marija 的]说明非常详细，有很好的图片，所以这可能是一个很好的开始项目。

在电子设备方面，有可寻址的 led(ws 2812 b)、蓝牙模块、红外接近传感器和 Arduino。接近传感器需要一点黑客，所以传感器可以安装在一种方式，它可以通过玻璃顶部检测东西。

这是一个真正指出一些相对简单的组件如何与软件和机械结构结合起来，真正创造出令人瞠目结舌的结果的项目。文档也给我们留下了深刻的印象，如果你不确定如何做木工或电子，你会发现一个很好的指南，里面有有用的图片。

现在，如果你不和有礼貌的人交往，而只和其他黑客交往，你可能会选择 EPROM 表。如果你迷上了点亮桌子，在完成这个项目后，你可以继续做[巨型 LED 书桌](https://hackaday.com/2017/03/02/making-a-mega-led-desk/)，尽管这更像是一个金属项目。
