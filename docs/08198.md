# 拆除暗室遗迹寻找埋藏的宝藏

> 原文:[https://hack aday . com/2018/01/06/拆毁暗房-埋藏宝藏的遗物/](https://hackaday.com/2018/01/06/tearing-down-a-darkroom-relic-for-buried-treasure/)

如果你的目标是从废弃的设备中获得独特的部件，你走的时间越久远，收获就越多。至少这是[Kerry Wong]在拆除 20 世纪 80 年代早期的暗室色彩分析仪时发现的。

对于只有数码摄影经验的读者来说，你需要明白，曾经有一段时间，图像是用真正的相机在真正的胶片上拍摄的，严肃的业余爱好者和专业人士有暗室来处理胶片。黑白处理在化学上非常简单——只需要显影、停止和修复。彩色处理要复杂得多，当要将胶片放大到彩色相纸上时，事情会变得非常复杂。[Kerry]的易贝 find 是一款 Besler PM1A 色彩分析仪，旨在通过平衡放大机中青色、蓝色和黄色成分的混合来帮助色彩实验室。

这台仪器在当时无疑需要一大笔钱，但实际上非常简单，只有一个 PM1A 光电倍增管及其驱动器是真正的发现。尽管如此，这是一次有趣的拆解，我们急切地想知道[凯瑞]如何看待这颗宝石。也许是一个μ介子探测器？[一台 X 射线背散射机](https://hackaday.com/2016/12/01/quickly-prototyping-x-ray-backscatter-machines/)？或者也许重复[他以前的光速实验](https://hackaday.com/2015/10/06/light-speed-its-not-just-the-law/)正在进行中。
