# 拉皮条我的范围

> 原文:[https://hackaday.com/2018/05/12/pimp-my-scope/](https://hackaday.com/2018/05/12/pimp-my-scope/)

我们大多数人都听说过某种形式的格言，“你可以买得更便宜，但你永远不会少付。”这意味着更便宜的产品最终无法满足更好的同类产品的需求。黑客们喜欢通过对廉价工具进行廉价升级来填充功能丰富的工具包，从而证明这句格言是错误的。以[思想商场]为例，它已经将一台入门级显微镜升级为能够进行偏振和暗场显微镜。休息之后也可以看视频。

从功能上来说，偏振图像可以揭示隐藏的特征，如晶体中的条纹或热熔胶线中的应力线。暗场显微镜就像是用黑色背景取代了通常耀眼的白色背景，我们 Hackaday 赞同这种装饰选择。偏振滤光镜片不贵，安装也很快，这取决于你的观察范围。增加一个暗场滤镜可能要花费一毛钱。

像大多数 mod 一样，最大的投资将是你的时间。通过让您熟悉您的工具及其工作方式，这种投资将会立即得到回报。从长远来看，你将拥有一个更强大的工具。

简单的[模块如光源](http://hackaday.com/2018/01/14/not-so-simple-led-upgrade-for-microscope/)可能很有价值，但是升级不仅限于光学显微镜，一台[电子显微镜通过 Arduino 起死回生](http://hackaday.com/2018/04/07/reviving-an-electron-microscope-with-arduino/)
