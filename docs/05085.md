# Nespresso 胶囊检测器

> 原文：<https://hackaday.com/2017/02/19/nespresso-capsule-detector/>

Nespresso 粉丝欢欣鼓舞！如果你喜欢咖啡(你当然喜欢),并且是 Nespresso 的粉丝，那么你很有可能是两种人之一:选择一种胶囊并坚持使用的人，或者有一罐混合胶囊却分不清哪种是哪种咖啡的人。当然，还有第三种，更罕见的，强迫症。那种把每个胶囊按照颜色整齐地放在一个合适的盒子里，充满了风格。在任何情况下，如果你因为扔掉盒子而忘记了哪种颜色是哪种咖啡，那么这里有一个有趣的周末项目给你:Nespresso 胶囊探测器。

[circuit.io 团队]开发了一个基于 Arduino 的项目，可以使用 RGB 颜色检测器检测哪个胶囊，并在 LCD 显示器上显示相关信息。这是一个非常简单的项目。如果你有 3D 打印机，你可以打印外壳，如果没有，为电子设备和胶囊制作一个工作外壳是相当容易的。

操作很简单，只需将胶囊放入孔中，Nespresso 胶囊检测器就会告诉您它的类型、浓度、味道以及问题咖啡的最佳杯数。我们只是不确定它是否能正确检测到 [Nespresso weddingbots](http://hackaday.com/2016/07/01/interactive-weddingbots-built-into-nespresso-capsules/) ，但是谁知道呢？
