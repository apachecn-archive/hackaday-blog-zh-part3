# 关于生物信号的一切

> 原文：<https://hackaday.com/2016/08/23/all-about-biosignals/>

DIY 医学是有趣的东西。人们可以找出许多电信号，这些电信号使身体以令人惊讶的容易获得的部件和简单的构造运行。虽然医学界可以预见地详细讨论这些信息的医疗用途，但黑客可以随心所欲地做任何他或她想做的事情。

一个良好的开端是观察来自心脏和其他肌肉的相对较强的电信号。[Bernd Porr]组装了一个简单的生物放大电路，他的学生制作了一个[系列视频](https://www.youtube.com/user/bratwursthaggisgeek/featured)来解释它的用途，如果你对这些感兴趣的话，这非常值得你花时间。

你们当中喜欢电学的人可能想从“[从设计到测量](https://www.youtube.com/playlist?list=PLWnTQ-kqjDMX0w_UwOabfXIMTHsiU3gsn)”播放列表开始，它详细介绍了放大器本身的结构。但是真正的好东西藏在“[脑电图要点:如何测量它和它的人工制品](https://www.youtube.com/playlist?list=PLWnTQ-kqjDMX6CSlnBvsTt0gHQ7J4L0S7)”列表中；获得信号只是第一步——解释它们才是有趣的地方。举例来说，如今出售的许多“精神控制”设备更有可能是简单的肌肉控制，而[这个视频](https://www.youtube.com/watch?v=_AqTwAoT6dw)告诉你为什么:小信号被大信号掩盖了。(嵌入下方)。

我们对生物信号的把戏并不陌生。MobilECG 项目人员[在 Hackaday 的 Belgrade 2015 大会上做了一个很棒的演讲](http://hackaday.com/2016/04/29/saving-lives-with-open-source-electrocardiography/)，并制作了这张[甜美的 ECG 名片](http://hackaday.com/2016/03/13/ekg-business-card-warms-our-hearts/)作为演示。 [OpenHardwareExG](http://hackaday.com/2014/08/13/an-open-hardware-platform-for-ecg-eeg-and-other-measurements/) 是这里展示的生物放大器的一个更复杂的版本。离开心脏，我们已经看到了一系列由精神控制的应用。

但是这篇文章的重点是，制造一个生物放大器不需要倾家荡产或者耗费脑力。如果你已经有了零件，这种事情只需在周末就能完成。[你会用自己身体的电信号控制什么](https://hackaday.com/2016/05/26/hackaday-prize-entry-open-source-myoelectric-hand-prosthesis/)？
