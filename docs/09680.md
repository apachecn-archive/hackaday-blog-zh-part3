# 回到弯曲子弹

> 原文：<https://hackaday.com/2018/06/10/coming-back-to-curving-bullets/>

当你有时间，有价值数千美元的磁铁，并且你喜欢《流言终结者》时，你会做什么？科学。至少，科学有戏剧化的天赋。用当今的技术制造的磁性手表可以阻挡子弹，甚至改变子弹方向的神话被彻底打破了。[K&J Magnetics]的工作人员想自己尝试一下这个神话，他们采取了自由的态度。

![](img/79a32d4e44adaba7e625570397bfb617.png)

尽管展览的结果，一个单一的磁铁能够改变抛射体的路径。这不会演变成什么救生技术，因为枪换成了打钢 BB 的动力不足的 BB 枪。最初的流言假设枪支全速射击。这并不奇怪，但它确实告诉我们，为了用磁力控制子弹，这些参数需要被扭曲到什么程度。博客回顾了他们为了磁性地弯曲子弹而不得不忍受的所有必要的妥协，他们的结果视频可以在休息时看到。

在这里，我们谈论射击飞机炮，这样它们就不会在离开炮筒后被误导，还有一些更有趣的 T2 武器，由邱吉尔决定。
