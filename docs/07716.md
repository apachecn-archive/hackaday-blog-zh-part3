# 伺服控制眼球让麻瓜变得喜怒无常

> 原文：<https://hackaday.com/2017/11/16/servo-controlled-eyeball-makes-a-muggle-moody/>

即使你与哈利·波特宇宙中偏执的傲罗有些许相似之处，你仍然真的需要那只[摇摆不定的假眼](https://www.instructables.com/id/Mad-Eye-Moody-Moving-Eyeball-Prop/)来推销那个疯眼汉穆迪角色扮演，而这一个相当令人印象深刻。

当然，除了易贝娃娃的眼睛之外,[daronjay]的假体窥视镜还有更多功能。有一个旋转球体的微型伺服系统，还有一个发送 PWM 信号的小饰品和一个装满电池的口袋。然而，合身和完美真的将它联系在一起，特别是考虑到它是由垃圾制成的——金属食品罐盖、酸奶杯和滚动止汗剂管。一些黄铜螺丝和一条皮革表带让人想起必要的哈利波特式外观，再加上我们认为是修复的伤疤，[daronjay]真的让这个角色栩栩如生。我们认为让伺服眼以某种方式受制于真实眼睛的运动会很酷，并加入一点随机性以使其看起来更好。

活点地图、[魔杖决斗](https://hackaday.com/2017/07/04/a-shocking-wizard-duel/)、[韦斯莱钟](https://hackaday.com/2013/02/11/harry-potter-location-clock-spies-on-your-smart-phone/)——魔法世界对于创造性黑客和道具制作来说时机已经成熟。接下来是什么——光轮 2000 四轴飞行器？拜托了。
