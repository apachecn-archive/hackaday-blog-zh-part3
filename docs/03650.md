# 搞笑诺贝尔奖:高特曼、大众汽车和石头的个性

每年，《不可思议研究杂志》都会为最疯狂的(已发表的)科学研究颁发奖项:搞笑诺贝尔奖。仪式在几天前举行，如果你想看看你错过了什么，我们在下面嵌入了(长)视频。(触发警告:真正的诺贝尔奖获得者是愚蠢的。)

[![stinker-250](../Images/847b12bdc4db3a65fccd602625552524.png)](https://hackaday.com/wp-content/uploads/2016/09/stinker-250.gif)

The Stinker

很难挑选出最好的怪异研究，委员会今年做了出色的工作。诀窍在于，他们不会把这个奖颁给庸医——例如，你的永动机永远不会让你获奖。不，搞笑诺贝尔奖颁给了最古怪的、最终可能会有用的科学。因此，我们得到了一些项目，如《生殖》中穿着聚酯纤维对啮齿动物性活动的影响，以及《经济学》中出于营销目的对不同岩石的感知个性的研究。

头条照片是托马斯·思韦茨，一位后人道主义者，他决定去卡普拉而不是电子人。在造好山羊假肢后，他和一群山羊在瑞士阿尔卑斯山高处生活了一周。他与查尔斯·福斯特分享了该奖项，因为他写的这本书，他以獾、水獭、鹿、狐狸和鸟的身份生活。

最具讽刺意味的化学奖颁给了大众汽车公司，因为他们在减少 T2 柴油发动机排放方面的工作。Zing！
