# 鹳看起来和我们想象的不一样

> 原文：<https://hackaday.com/2016/01/14/the-stork-looks-different-than-we-thought/>

物联网真正需要的是更多的东西，越可笑越好。至少，这是[Eric]的观点，他创造了一个半开玩笑的小工具，添加到不断增长的联网设备列表中。这是一种支持蓝牙的验孕测试，可以自动向全世界发布结果。觉得幸运吗？

工作原理相当简单。一个蓝牙低能耗模块被集成到一个数字验孕棒的末端。这些测试有一套光电探测器，在测试完成后读取化学试纸。如果测试结果为阳性，该模块会向 Raspberry Pi 发送信号，后者会将结果发布给全世界。它还有一个选项，可以马上给你妈妈发短信！

[Eric]在推特上直播验孕测试的项目也导致了一个数字验孕测试的详细拆卸，所以如果你需要验孕测试的任何技术规范(无论出于什么原因)，他的项目网站有丰富的信息。他指出，他的设备也可以用在其他类似的直接驱动液晶显示屏的设备上。然而，乐趣并没有就此结束！一旦怀孕稍晚一点，你也可以在推特上发布宝宝的消息。
