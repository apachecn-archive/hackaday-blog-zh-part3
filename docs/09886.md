# 台球回球系统归功于独创性

> 原文:[https://hack aday . com/2018/06/27/pool-ball-return-system-chalked-up-to-intelligence/](https://hackaday.com/2018/06/27/pool-ball-return-system-chalked-up-to-ingenuity/)

你打台球吗？如果是这样，你可能会认为酒吧和台球厅桌子上的自动回球系统是理所当然的。[Roger Makes]厌倦了每次想玩的时候绕着他的家桌走来走去收集球，[所以他设计了一个省时的回球系统](https://www.thingiverse.com/thing:2978314)。

现在，球不再落入球台附带的小网篮中，而是落入 3D 打印的口袋中，并沿着定位销杆进入中央收集箱，该收集箱由球台上方的带子悬挂。轨道本身用 ABS 筋加固，防止球掉落。

泳池是关于几何的，当[Roger]在设计阶段试图将口袋的漏斗部分与出口斜槽合并时，这真的击中了要害。他用模块化的设计覆盖了所有的角度，让滑槽自由旋转，这从传力杆上带走了很多压力。休息后我们会播放录像，所以不用麻烦拿出你装满硬币的胶卷盒。

我们迫不及待地想看看[罗杰]下一步会做什么。也许会是类似于这个 OpenCV 计分系统的东西。
