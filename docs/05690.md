# 巨型无焊乐高 NES 控制器给每个人小手

> 原文：<https://hackaday.com/2017/04/20/giant-solderless-lego-nes-controller-gives-everyone-tiny-hands/>

如果你在想“我应该花 130 美元买乐高积木，造一个巨大的 USB NES 控制器，看看那会是什么样子”，但你害怕花那么多钱，[BrownDogGadgets]会帮你搞定。他用乐高建造了一个巨大的 NES [控制器。该控制器是在 LEGO Digital Designer 中设计的，它可以让你创建一个虚拟模型，然后获得可以在线订购的完整零件列表。](https://hackaday.io/project/21303-giant-working-lego-nes-controller)

电子设备基于一个被编程为 USB 键盘的微型 LC，按钮是标准按钮。内部用尼龙导电带连接在一起。乐高是一个合适的选择，因为 Teensy 和开关是建立在乐高兼容的印刷电路板之上，所以组件只是扣在适当的位置。这个系统被称为[疯狂电路](https://www.crazycircuits.com/)，是将电子设备转化为通用和可重复使用系统的一个非常好的方法。

如果那个控制器太大，他们也为你自己的控制器使用同样的电路和一些激光切割部件。如果你真的想变得更大，看看[巴伦·冯·布伦克的] [乐高 NES 控制器](http://hackaday.com/2015/05/21/large-nes-controller-made-from-legos/)，它使用了真实控制器中的电子设备。
