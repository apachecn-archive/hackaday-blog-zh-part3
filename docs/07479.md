# 唉，可怜的约里克！他没有亚马逊总理

> 原文:[https://hack aday . com/2017/10/25/alas-poor-yo Rick-he-hath-not-Amazon-prime/](https://hackaday.com/2017/10/25/alas-poor-yorick-he-hath-not-amazon-prime/)

如果你正在寻找一个万圣节项目，你可能会考虑来自维也纳的 Yorick 项目。正如你在下面的视频中所看到的，它将一个树莓 Pi 与一个具有电子动画头骨的亚马逊 Alexa 结合在一起。

这不是技术要求最高的项目，但它有很大的潜力进行进一步的黑客攻击。该项目包括一个 USB 麦克风、一个伺服控制器和一个音频伺服驱动板。看起来音频伺服板正在控制下颚的运动，根据视频，我们想知道你是否可以完全在软件中运行它。

这是一个有点令人不安的调用头骨 Alexa 和听到熟悉的声音出来。看起来像一些模拟变声硬件可能在输出上是有序的。至于输入，将声音文件输入音频输入以响应传感器而不是只接受语音命令会很有趣。换句话说，除了语音命令，你可以告诉它说一些特定的事情，或者当有人打断光束或其他东西时启动一项技能。

如果 Yorick 在剩下的短时间内难以胜任，你可以选择零件较少的产品。或者，如果头骨太紧张，尝试[南瓜](https://hackaday.com/2012/10/31/a-scary-pumpkin-pi/)或[低音](https://hackaday.com/2016/11/08/alexa-brings-back-singing-fish-this-time-its-a-good-thing/)。
