# 电子烟上的 Flappy Bird？

> 原文：<https://hackaday.com/2016/04/15/flappy-bird-on-an-e-cigarette/>

好了，现在我们都看到了。有人把努力放在端口 Flappy Bird…在电子烟上运行。准确地说是一辆 eVic-VTC Mini。所以现在，在抽烟的间隙，你可以玩一个最令人沮丧的游戏。

电子香烟的 SDK 可以在 GitHub 上获得，GitHub 是去年由一群 Redditors 在 T2 提供的。如果你对游戏感兴趣，并且碰巧有这款电子香烟，【银行】在 YouTube 视频的描述中提供了下载和闪烁说明。

正如一位 [Reddit 用户指出的:](https://www.reddit.com/r/electronic_cigarette/comments/4e6mk9/flappy_bird_clone_on_evicvtc_mini/)

> “你们的科学家太专注于他们能不能做到，他们没有停下来想一想他们是否应该。”

创建者[Balázs Bank]回复了下载链接，自己安装。科学是否走得太远了？

撇开所有的笑话不谈，他还为同一款电子烟写了 snake，并承诺他的下一个项目将是设备的实际固件。
