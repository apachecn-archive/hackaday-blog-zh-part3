# 危险，威尔·罗宾逊:甜蜜的 B9 建造

> 原文：<https://hackaday.com/2017/02/10/danger-will-robinson-sweet-b9-build/>

[![3](img/7d8a0a8593f9887550fc0632a7e803b4.png)](https://hackaday.com/wp-content/uploads/2017/02/3.jpg) 如果你看过葡萄牙语版的《迷失太空》，你一定会认出[埃维拉尔多]的 B9 机器人从它肚子里的 SD 卡上读出的短语。如果没有，你可以看看下面的视频，学习一些重要的短语，比如“警告！外星人正在接近。”或者“地球正在分裂”(我们推测)。或者访问[埃维拉尔多]的网站，查看[伟大的模型建造日志](http://erl4ever.blogspot.de/2013/03/lost-in-space-robot-b-9.html)。当你在那里的时候，也可以看看他的[型号 tr s80](http://erl4ever.blogspot.de/2017/01/meu-cp500-prologica-na-versao-nanico.html)。)

这里有很多坚实的模型构建，但隐藏在漂亮的外表下的是一些优秀的老式黑客技术。一旦音频存储在 SD 卡上，[埃维拉尔多]简单地将它直接焊接到项目中。还有一个驱动机器人的红外子板，而闪亮的灯和伺服电机使它变得栩栩如生。我们想要一个放在办公桌上！

如果你还没有做一个基于红外遥控的项目，你真的应该做一个。它仍然是最容易被破解的方法之一，在利用你在家里到处乱放的多余的红外遥控器的同时，向微控制器传输数据。如果你缺乏灵感，又不是模型制作者，看看[这个致力于 IR 的小工具](https://hackaday.com/2016/07/30/hacklet-118-infrared-and-universal-remote-controls/)，或者我们[最喜欢的智能家居设备](https://hackaday.com/2014/10/14/homer-robot-ensures-you-dont-miss-an-episode/)。

你和我们想的一样吗？这将成为黑客日科幻竞赛的优秀参赛作品，该竞赛接受参赛作品，截止日期为 3 月 6 日。
