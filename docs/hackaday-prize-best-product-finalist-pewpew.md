# Hackaday 奖最佳产品入围奖:PewPew

> 原文：<https://hackaday.com/2017/08/06/hackaday-prize-best-product-finalist-pewpew/>

今年的黑客日奖，我们做了一些非常非常酷的事情。我们鼓励硬件企业家想出下一个大的电子产品。我们将给予 Hackaday 大奖中的最佳产品 30，000 美元，以及一个在充满工具的实验室中工作的机会，以将原型转化为可销售的现实。

上周，我们宣布了 [Hackaday 奖最佳产品竞赛](http://hackaday.com/2017/08/01/these-are-the-twenty-finalists-for-the-hackaday-prize-best-product/)的 20 名决赛选手。在最后一轮评审之前，这些黑客和修补者还有很多工作要做，但在那之前，我们可以开始看看今年 Hackaday 奖中已经出现的一些最优秀的产品。

为了进入最佳产品决赛，[Radomir]正在开发一款游戏机。把这当成一个教育玩具。游戏编程很辛苦，从主循环到处理按钮再到推像素都需要一些天赋。[这个项目是最小的游戏机](https://hackaday.io/project/21578-pewpew-featherwing)。这是 Adafruit 的微型开发板家族的羽翼，旨在教授 PyGame 编程。

这块板上有一个 8×8 的双色 led 矩阵、几个开关、电阻和一个芯片，该芯片可以将这些 led 转换成可以进行内存映射的东西。很简单，但这就是重点:这是教授游戏编程的最低可行产品。

目前，商业计划是为这种附加板开发游戏和示例，建立一个社区，编写一些教程，并在 Tindie 上销售一些这种板。从那里，它只是一个增长的问题，已经有计划为 PewPew 配备 TFT 屏幕、STM32 处理器和内置的 tile 和 sprite 引擎。这很可能是一个非常酷的教育玩具的开始，我们很高兴它能入围 Hackaday 奖的最佳产品竞赛。
