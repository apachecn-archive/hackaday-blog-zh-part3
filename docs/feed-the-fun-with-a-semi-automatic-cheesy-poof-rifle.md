# 用一把半自动的劣质步枪来找乐子

> 原文：<https://hackaday.com/2016/05/23/feed-the-fun-with-a-semi-automatic-cheesy-poof-rifle/>

在 Hackaday，我们认为黑客有能力通过他们的构建让世界变得更美好。这种空气动力的廉价步枪不属于这种类型，除非让这个世界多一点乐趣，多一点混乱，让它变得更美好。

[NightHawkInLight]的设计原理很简单——一个电动吹叶机提供动力，一个大罐子提供弹药。将两者结合在一起并提供一个桶是一件简单的事情，用 1”PVC 管和配件。但是等等——以免你认为这个黑客，就像弹药一样，只是一点美味的绒毛，这里有一些关于流体动力学的知识。使用普通的三通配件，吹叶机只会给弹匣加压，使其难以装弹。但通过对进入的气流进行小小的限制，文丘里效应实际上会将弹药吸入枪膛，并沿着枪管向下，这让周围几码远的饥饿野生动物感到高兴。科学！

这种设计还有很大的改进空间——这种[气体动力、管饲雪球枪](http://hackaday.com/2016/03/09/snowball-machine-gun/)将会很受欢迎。就目前的情况来看，这种廉价的枪似乎是一种好的、不干净的乐趣。
