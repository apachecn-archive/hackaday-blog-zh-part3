# ESP 时钟需要更多电源

> 原文：<https://hackaday.com/2016/11/10/esp-clock-needs-more-power/>

[Victor-Chew]厌倦了设置时钟。毕竟，我们现在是在 21 世纪，为什么我们必须调整时钟(我们刚刚为夏令时做的事情)？这就是为什么[Victor]想出了 [ESPClock](https://hackaday.io/project/16742-espclock) 。

基于一个 2 美元的宜家模拟时钟，[Victor]为这个项目设定了几个设计目标:

*   从网络上自动设置时间
*   自动调整夏令时
*   不比普通的钟贵多少
*   靠电池运行一年

最后一个目标是唯一没有实现的目标。即使有一个大电池组，[维克多的]时钟也会在一周左右耗尽电量。你可以在下面看到一些时钟与网络时间同步的视频。

纸上谈兵很容易，但我们认为[Victor]应该研究如何让处理器在大部分时间处于深度睡眠模式。这可能意味着您需要一个按钮来唤醒它进行配置，并且可能需要一些其他的修改。

如果你想浪费更多的 ESP 模块，你可以试试这个时钟。
