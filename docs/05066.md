# 定制零件将宜家窗帘安装在物联网上

> 原文:[https://hack aday . com/2017/02/19/custom-parts-put-IKEA-window-shades-on-IOT/](https://hackaday.com/2017/02/19/custom-parts-put-ikea-window-shades-on-iot/)

不管项目是关于什么的，我们总是对具有良好的适合度和完成度的完美集成的构建感兴趣。对试验板上的鼠窝布线有一定的吸引力，这样的项目是有价值的，因为它们推动了极限。但最终你需要从原型到产品，这就是宜家窗帘自动化项目的亮点。

集成不仅仅是把所有东西放进一个漂亮的盒子里，尤其是家庭自动化设备——它真的需要融合。[ehsmaes]卷帘机动化项目通过 3D 打印的电子外壳以及齿轮步进电机驱动窗帘的定制外壳很好地实现了这一点。驱动装置取代了 IKEA [Tupplur](http://www.ikea.com/us/en/catalog/products/40349258/) 灯罩末端的标准弹簧帽，两种外壳的中性色与灯罩和周围环境完美融合。控制电子设备包括节点 MCU 和电机护罩；[eshmaes]警告说，窄小的灯罩在 USB 电源下工作良好，但较宽的窗户将需要电源提升。物联网方面的事情由 MQTT 和 OpenHab 负责，允许灯罩上升和下降到任何位置。下面的短片展示了遮光罩的校准程序。
