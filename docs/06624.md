# 黑客日奖参赛作品:圆周率驱动的谷歌眼镜

> 原文:[https://hack aday . com/2017/08/03/hack aday-prize-entry-pi-driven-Google-glass/](https://hackaday.com/2017/08/03/hackaday-prize-entry-pi-driven-google-glass/)

(里卡多·费罗)不想买谷歌眼镜，所以他自己做了一个。

[Raspberry Pi Zero Prism](https://hackaday.io/project/18033-raspberry-pi-zero-prism) 由一个 3D 打印的耳机组成，耳机的侧边装有各种电子组件，包括 Pi Zero running Raspbian Jessie、Pi Noir IR 摄像头、WiFi/蓝牙模块和一大堆 SMD 触觉按钮。视频输出由 Kopin 922K 显示模块提供。该模块通常用于智能护目镜，并使用棱镜将信息反射到佩戴者的视野中。

里卡多为这款开源谷歌眼镜设想的一个应用是，将它与面部识别软件和 YouTube 上最受欢迎的红外相机看穿衣服的技巧结合使用。不，他不是用它来实现那个想法，你应该把你的思想从俗套中解放出来。[Ricardo]想要识别蒙面罪犯。撇开让这项技术发挥作用的技术挑战不谈，我们认为，带着 x 光眼镜到处走很可能会让那些为了谦虚而穿衣服的人把眼镜从你脸上撕下来。尽管如此，这仍然是一个迷人的项目，我们喜欢棱镜和视频组件结合在一起的方式。
