# Hackaday 奖品入口:USB GSM GPS 9DOF SD TinyTracker 拥有所有缩写

> 原文:[https://hack aday . com/2017/06/11/hack aday-prize-entry-USB-GSM-GPS-9 DOF-SD-tiny tracker-has-the-all-the-the-all-the-scholympus/](https://hackaday.com/2017/06/11/hackaday-prize-entry-usb-gsm-gps-9dof-sd-tinytracker-has-all-the-acronyms/)

[Paul]组装了一个小得惊人但功能强大的追踪器来监控所有的东西。USB [TinyTracker](https://hackaday.io/project/25211-usb-tinytracker) 是一种封装了 48MHz 处理器、2G 调制解调器、GPS 接收器、9DOF 运动传感器、气压计、麦克风和用于数据存储的 micro-SD 插槽的设备。他成功地将所有这些都装进了一个 USB 拇指驱动器外壳，这意味着你可以在 Arduino IDE 中随心所欲地对它进行编程，然后将其插入任何 USB 端口并让它运行。这使得远程监控、资产跟踪和各种类似间谍的活动成为可能。

然而，他的项目中最不寻常的一个方面是这样一句话:“一切都非常好地结合在一起，零件和 PCB 的高度完全符合我的计划。”[Paul]挑选了一个只能安装一个 PCB 的外壳，但经过仔细计算和挑剔的元件选择，他成功地将所有元件安装到两个 2 层电路板上，这两个电路板通过一个连接器咬合在一起，并安装在外壳内。

我们跟踪了[Paul]在这个项目上的进展，开发了他的 GSM GPS 追踪器的早期版本[,它使用了一个小尺寸，可以紧贴车把，但这个功能更加多样。](http://hackaday.com/2016/11/19/barely-there-gsm-gps-tracker/)
