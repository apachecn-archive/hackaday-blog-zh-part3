# 使用这款模拟乘法器初级读本实现数学运算

> 原文:[https://hack aday . com/2016/06/19/make-math-real-with this-analog-multiplier-primer/](https://hackaday.com/2016/06/19/make-math-real-with-this-analog-multiplier-primer/)

还记得在代数中学习所有关于函数的知识吗？我们也不知道。当然，我们大多数人第一次记得线性情节和理解的魔力 *y* = *mx* + *b* 。但是我们中的许多人仅仅对更复杂的函数如指数和圆锥曲线有一点点的了解。幸运的是，我们当中的功能障碍者可以通过使用模拟乘法器和运算放大器的[演示加深理解](https://youtu.be/m6MqNZ7H99w)。

[devttys0]的视频教程是关于模拟乘法器及其多种用途的绝佳入门读物。从两个输入电压相乘的简单例子开始，他继续展示输出输入电压的平方和立方的电路。看到斜坡输入电压的立方的输出波形，让我们牢牢记住了这个概念，并把我们带回了许多年前代数课上那些看似浪费的时光。通过增加运算放大器进行进一步改进，产生一个输出输入电压平方根的电路，并最终产生一个压控电阻，它可以根据电压衰减输入信号。只需几个芯片就能获得非常强大的功能。

[devttys0]的基础芯片是 ADI 公司的 AD633，这是一款非常方便的芯片。关于这款芯片的更多信息，请查看[【Bil Herd】在模拟计算](http://hackaday.com/2014/07/25/bil-herd-computing-with-analog/)上的帖子。
