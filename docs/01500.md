# 突然，4G 感觉慢了

> 原文:[https://hackaday.com/2016/02/14/suddenly-4g-feels-slow/](https://hackaday.com/2016/02/14/suddenly-4g-feels-slow/)

伦敦大学学院的研究人员成功通过光传输系统[以 1.125 Tb/s](http://www.nature.com/articles/srep21278) 的速率传输数据。这比典型的商业光学系统快十倍以上，比标准宽带连接快数千倍。这项研究发表在*科学报告*上，利用了无线系统中常见的编码技术。

原型系统使用不同波长的 15 个通道。每个频道都使用 256QAM 编码(和你在有线调制解调器上看到的一样)。单个接收器一起恢复所有频道。这项技术还没有商业化。值得注意的是，该实验使用了彼此非常接近的发射器和接收器。未来的测试将考察当它们之间有数百或数千英尺的光纤时，系统的表现如何。

本月早些时候，我们已经讨论过信息理论，同样的原理允许更多的数据在一根光纤上传输。我们刚刚在[用光纤照明](http://hackaday.com/2013/11/24/building-an-led-source-for-a-fiber-optic-ring-light/)。如果你想了解光纤是如何工作的(有一个很酷的防冻演示)，你可能会对下面来自[engineerguy]的视频感兴趣。
