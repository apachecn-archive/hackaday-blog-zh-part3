# Hackaday 奖参赛作品:真实生活

> 原文：<https://hackaday.com/2016/08/27/hackaday-prize-entry-real-life-xeyes/>

电子动画、角色扮演和服装融入了大量的技术。对于他们的 Hackaday 奖参赛作品，[Dasaki]和[Dylan]将服装或万圣节道具中的眼睛提升到了一个新的水平，用电子动物的眼睛看着这个疯狂虚构的穿着者正在看的地方。这是现实生活中的 x eyes，它有望成为一些非常非常酷的服装的一部分。

这个系统的机制实际上非常简单——它只是几个伺服系统连接在一起，使一双机器人眼睛上下移动，左右移动。整个装置安装在一个框架上，框架上连接着一个非常小的摄像头，直接对准用户(真实的)眼睛。软件是事情变得有趣的地方。这是一个基本的眼球追踪设置，用红外光照亮瞳孔，还有一个计算单元可以计算用户在看哪里。

对于该软件，[Dasaki]和[Dylan]已经收集了[一堆链接](http://wiki.cogain.org/index.php/Eye_Trackers)，但目前最好的解决方案是去年 Hackaday 奖的[OpenMV](https://github.com/openmv/openmv)和[Horus 之眼](https://hackaday.io/project/6638-eye-of-horus-open-source-eye-tracking-assistance)项目。这是一个伟大的项目，也是今年 Hackaday 奖自动化部分的一个非常有趣的参赛作品。
