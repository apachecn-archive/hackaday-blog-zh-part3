# 杂耍机器人灵巧地处理球

> 原文：<https://hackaday.com/2015/10/05/juggling-robot-deftly-handles-balls/>

一个设计良好的机器人可以做人类能做的任何动作。无论这是一场杂技表演，还是只是用笔写字，都有一个机器人来完成人类可以完成的任何一个动作。这包括杂耍，但人类杂耍的动作从未如此大规模地被复制。[【内森】造了一个可以同时玩七个球的机器人](https://hackaday.io/project/7727-juggling-robot)。这比其他任何杂耍机器人都要多。

虽然最初的计划是建造一个低成本的版本，可以通过将球抛向空中来变戏法，但这被证明是非常困难的。[Nathan]没有放弃，而是通过将球滚上斜坡来简化问题。整个建筑[被记录在一个 imgur 画廊](http://imgur.com/a/Ytscx)里，这里有一些有趣的技术。3D 打印的手臂由运行在 60V 电压下的结实的步进电机控制。为了防止球在手臂中弹跳，[内森]包括了一个电磁体，在每个循环中的几分之一秒内将球固定在适当的位置。

杂耍七个球很神奇，*但是八个呢？*这是每一个杂耍机器人的建造者都会遇到的问题，并不像再加一个球那么简单。玩偶数个球和玩奇数个球是完全不同的。也就是说，[内森]的机器人确实有四个球。[它大概应该得到那只看着的](https://twitter.com/MrPrezident0/status/646887975079448576)。

这不是[内森]第一个令人惊叹的 3D 打印机器人，也可能不会是最后一个:他最近为下次范·海伦来镇上做了一个[彩虹糖分拣机](http://hackaday.com/2015/09/08/only-eat-red-skittles-weve-got-you-covered/)。他的所有项目都有惊人的技能，他无疑是整个 hackaday.io 社区的资产。
