# 智能吸尘器的新大脑

> 原文：<https://hackaday.com/2017/06/12/new-brain-for-smart-vacuum/>

现在，ESP8266 已经进入了几乎所有领域。凭借其小巧的尺寸、低廉的价格和易于使用的程序员，它非常适合几乎任何需要 WiFi 的应用。[HawtDogFlvrWtr]决定将所有的平台津贴，一个 ESP8266 几乎是乞求被塞进他的自动吸尘器。不过，这不是 Roomba，这是一个现在有自定义 WiFi 接口的 Neato。

新的 WiFi 改造也带来了一些额外的功能。首先，它抛弃了设计糟糕的默认用户界面(通常是任何消费产品中最令人讨厌的专有组件)。此外，真空吸尘器现在可以完全自定义时间表，也可以按下按钮进行部署。现在它有了一个自定义界面，可以通过网络向电话或其他计算机报告自己的状态。

[HawtDogFlvrWtr]仍在开发他的项目，并寻求一些帮助来测试他的新平台。如果你正在拆你自己的东西，他的项目页面上也有 how to 视频。还有许多其他方法来修改真空吸尘器以增加其他有用的功能。
