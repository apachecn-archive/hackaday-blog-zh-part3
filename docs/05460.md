# 太阳能假肢皮肤

> 原文：<https://hackaday.com/2017/04/01/solar-powered-prosthetic-skin/>

假肢使用者最大的问题之一就是手感。如果你曾经试着用麻木的手握笔写字，你就会意识到反馈对电机控制方程是多么重要。研究正在进行，以找到从假肢提供反馈的方法，即使是一种基本的形式。人类的神经系统比普通的串行 UART 接口要复杂一些。许多反馈系统的要求之一是电力，这通常需要大容量电池或某种形式的超级电容器，但一个英国团队已经开发出一种将太阳能电池嵌入触敏假体皮肤的方法。

皮肤依赖于每个人最喜欢的材料，石墨烯。一层薄薄的石墨烯使假肢能够将温度和接触压力的信号反馈给用户。诀窍在于石墨烯皮肤是令人难以置信的透明，据报道可以让其表面 98%的光通过。接下来，在皮肤下安装太阳能电池板就很简单了，收集的能量可以用来为传感器系统供电。

该团队承认，以后将需要一些能量存储，因为如果他们走进一个黑暗的房间时，他们的四肢失去了所有的反馈，这对任何假肢用户来说都是困难的。一个人的手臂在上床后失去所有感觉的想法并不特别吸引人。点击这里查看这篇论文。视频下方休息。
