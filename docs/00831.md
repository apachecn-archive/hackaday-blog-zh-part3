# 这个圣诞节不要成为一个无所事事的人

> 原文:[https://hack aday . com/2015/12/03/don ' t-be-a-drone-noob-this-Christmas/](https://hackaday.com/2015/12/03/dont-be-a-drone-noob-this-christmas/)

传统上，培养飞行模型飞机的爱好需要花很多钱，更不用说花很多时间去制作和学习了。所有这些导致了一次令人毛骨悚然的首次飞行。如果你非常幸运，你会带着一架轻微受损的飞机回家——但我们中的许多人都做了一个漂亮的死亡螺旋直入地面——所有这些都是为了我们可以回家，然后重新开始。也许我们最近看到这么多(负面)无人机相关新闻的原因之一是，加入飞行器俱乐部的门票价格从未如此之低。这一点，似乎班上总有一个孩子想毁了我们其余的人。

今年，美国联邦航空局预计大约有 100 万人在圣诞节早晨醒来时，树下会有一架无人机。人们有很多机会把事情搞砸。因此，如果你计划今年使用无人机，你可能想在休息后观看视频。或者直接转发给你认为需要看的人。如果你对任何类型的飞行模型感兴趣，你应该已经在你的 YouTube 订阅了[flite test]——他们有一些非常有用的视频，特别是对于想要成为这个爱好的初学者。

这个视频指南只是简单介绍了不要做什么。显然它没有涵盖一切。如果我们不说你们当地的法律可能会有所不同，我们就不会为我们的读者着想——所以做好你的功课，保持安全，不要成为一个无所事事的人。
