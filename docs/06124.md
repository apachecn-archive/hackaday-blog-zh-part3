# 无限远处的桌子

> 原文：<https://hackaday.com/2017/06/26/a-table-from-beyond-infinity/>

无限镜子是一些很棒的桌子模型，也是很棒的中心装饰品。Instructables 用户[bongoboy23]在设计这个为我们的现代社会量身定制的令人难以置信的桌子时，采取了超越无限的几个步骤。

白杨和松木构成了框架，红橡木——染色和雕刻——构成了别致的外观。桌面由 Arduino 编程，在 Teensy 3.1 上运行，有 40 个区域的 960 个 led。有四个隐藏在滑动面板后面的 USB 端口，以及一个双端口交流插座和一个感应充电板和电路。隐藏的 Adafruit TFT 触摸屏允许用户控制桌子的功能。控制仅限于改变照明功能，但吃豆人，蛇，文本功能仍然是未来！

它的重量为 850 美元，价格不菲，但看起来令人惊叹。
