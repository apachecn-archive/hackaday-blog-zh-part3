# 用声波悬浮器悬浮蚂蚁和液滴

> 原文:[https://hack aday . com/2017/08/24/带有声学漂浮器的漂浮蚂蚁和液体滴/](https://hackaday.com/2017/08/24/floating-ants-and-drops-of-liquid-with-an-acoustic-levitator/)

逗你的朋友开心，让你的敌人吃惊，让蚂蚁和其他昆虫困惑，只要它们能够困惑。用这个方便的自制声学悬浮器完成所有这些甚至更多。

在任何人开始考虑使用这种技术来建造一个真正悬浮的悬浮滑板之前，你最好降低你的期望。尽管如此，能够漂浮液滴和小生命形式并不是一个简单的壮举，而且看起来非常有趣。[Asier Marzo]Instructable 的帖子实现了他在第一次发表当时流行媒体称之为“牵引波束”的结果时所做的承诺，我们在一月份报道了这个结果。这个漂浮器显然源于早期的作品，3D 打印的半球形部分布满了同相连接的超声波传感器。第二个部分被添加来在空间中间创建声波，正如下面的视频所示，几乎任何足够轻并且至少像蚂蚁一样合作的东西都可以在 Z 轴上被操纵。

[Asier]的设计有很大的扩展空间，可能比烦人的 bug 有更多的实际应用。表面贴装器件非常小——也许声学拾放是可能的？
