# 我们现在处于二级戒备状态

> 原文：<https://hackaday.com/2017/09/16/we-are-now-at-defcon-2/>

如果你有一个工作中的 [DEFCON 仪表，报告真实数据](http://www.instructables.com/id/Working-Wargames-Defcon-Sign/)，会很酷还是很痛苦？

在我们超越自己之前:不，不是那个 [DEF CON](https://hackaday.com/2017/08/04/all-the-hardware-badges-of-def-con-25/) 。Instructables 的用户[ArthurGuy]是 1983 年电影《T2》战争游戏的粉丝，在最近的一次观看之后——hacker 感觉到一阵刺痛——他开始着手建立自己的实时显示器。

利用一些多余的木头，[ArthurGuy]把一个 10x10x50cm 的盒子粘在一起，钉在了招牌上。在某些地方已经被涂成白色，这种油漆出色地充当了每个部分内部灯光的反射器。五个 DEF CON 级别的面板由 3 毫米厚的彩色丙烯酸材料切割而成，经过乙烯基切割机的一点加工后，上面贴上了数字。

[ArthurGuy]做了一点小手脚，使用了 WS2812 新像素 LED 灯条——每层 12 个——并使用粒子光子来控制它们，从而偏离了一个正确的、屏幕精确的复制品。一段[快速代码](https://gist.github.com/ArthurGuy/784ce4b95a895752cb07df6a1320c026)轮询 MI5 恐怖主义 RSS 源并显示其当前级别——遗憾的是，它目前处于 2 级防御状态。

然而，这并不完全是悲观的，因为你可以重新配置这样的显示器来跟踪你可能喜欢的任何东西！
