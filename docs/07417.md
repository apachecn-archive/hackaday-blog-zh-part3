# 用示波器(或仪表)测试无刷电机

> 原文：<https://hackaday.com/2017/10/18/testing-brushless-motors-with-a-scope-or-a-meter/>

与传统的有刷电机相比，无刷电机有很多优点。然而，测试它们可能有点痛苦。因为电机线圈的电阻通常很低，标准的电阻检查不太可能有用。有些人使用 LC 电表，但它们不像万用表或示波器那样常见。[Nils Rohwer]发布了两个视频——一个是两年前的一个是最近的——展示了如何用万用表或示波器测试无刷电机。哦，你还需要一样东西:一个钻头。

你不必在马达上钻孔，而是用钻头来旋转马达的轴。由于电动机和发电机差不多，你可以读取旋转的电动机产生的电压，并确定它是好是坏。第一个视频展示了这种技术，第二个更近的视频展示了一个显示坏电机的示波器。你可以在下面看到这两段视频。

在第二个视频中，你会看到一对线圈短路了——可能是因为绝缘失效，也可能是因为过热——产生的电压只有一个好线圈的 1/10。最后，他打开了故障电机，查看了可疑的线圈，尽管视觉上看不到太多东西。

我们已经看到无刷电机[被用作编码器](https://hackaday.com/2017/01/18/use-a-brushless-motor-as-a-rotary-encoder/)。你甚至可以建造自己的[无刷电机演示](https://hackaday.com/2016/07/13/build-your-own-brushless-motor/)，或者选择一个[更实用的设计](https://hackaday.com/2017/05/08/powerful-professional-brushless-motor-from-3d-printed-parts/)。
