# 用尼康镜头黑进 Pi 相机

> 原文:[https://hack aday . com/2015/09/19/hacking-a-pi-camera-with-a-Nikon-lens/](https://hackaday.com/2015/09/19/hacking-a-pi-camera-with-a-nikon-lens/)

手机扼杀了很多行业。越来越难证明购买普通手表、计算器或日历是正确的，因为你的手机做所有这些事情至少和原版一样好。手机也有相机，所以因为没带相机而错过一个镜头的日子已经过去了(虽然我们总是想知道大脚怪和 UFO 图片的洪流在哪里)。然而，你可能仍然有一个专门的相机藏在某个地方，因为，让我们面对它，大多数手机相机只是没有那么好。

Raspberry Pi 相机与廉价手机相机不相上下。[Martijn Braam]有一台尼康相机，他注意到他可以买到一台带有 C 型镜头支架的树莓 Pi 相机。他拿起一个 C 转 F 适配器，开始在树莓派相机上试验尼康 DSLR 镜头。

你会认为这些照片会很棒，对吗？它们很好，但[Martijn]发现 Pi 的传感器实际上补偿了通常在小廉价镜头中发现的颜色效应。当使用昂贵的 DSLR 镜头时，会影响照片的颜色。当然，有很多色彩校正软件。

这些镜头通常也能碰到更大的传感器。这意味着 Pi 相机得到了一个 DSLR 用同样的镜头会得到的裁剪视图。

实用吗？我们不知道我们会用它做什么。但这确实是一个骗局，我们喜欢这样。如果你想要更实际的东西，也许可以用圆周率来控制整个 DSLR。当然，有些人想要不止一个镜头，我们知道有些人会发现马丁选择的镜头 T2 太现代了。
