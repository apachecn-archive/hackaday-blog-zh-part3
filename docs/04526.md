# 这些装饰品衡量圣诞节的欢乐

> 原文:[https://hack aday . com/2016/12/24/these-hearts-measure-Christmas-cheer/](https://hackaday.com/2016/12/24/these-ornaments-measure-christmas-cheer/)

我们在这里发布的装饰项目往往是简单的独立项目。然而，我们已经进入了物联网时代(不管你喜不喜欢)，节日装饰品不一定是单一的、互不关联的闪烁物体。对于今年的圣诞节，[肖恩·哈金斯]想出了一些[相互关联的 DIY 饰品，可以回应圣诞欢呼](https://hackaday.io/project/19051-esp8266-christmas-ornaments)。

(Sean Hodgins)将一些漂亮的印刷电路板做成节日的形状，然后用手工糊和烤箱焊接两面的 SMD 元件。每一个都由电池供电，由 ESP8266 控制。每个饰品正面的 led 和按钮组成了用户界面。当按下按钮时，数据被发送到 [Phant](http://phant.io/) 服务器，并且“圣诞欢呼”计数器递增。其他饰品，只要能连接到 Phant 服务器，都会定期检查计数器。如果圣诞节的欢乐增加了，装饰品将会演奏一首曲子并闪烁一些灯光。

这些装饰品是开源的——[Sean Hodgins]在 GitHub 上发布了代码和 PCB 设计。它们看起来很棒，这是让人们知道你在假期想念它们的好方法。看看这个[发光烛台](http://hackaday.com/2016/12/21/a-menorah-for-the-21st-century/)或这些[发光亚克力饰品](http://hackaday.com/2011/12/14/lighted-acrylic-christmas-ornaments/)可以获得更多的节日乐趣！
