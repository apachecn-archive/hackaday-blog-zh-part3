# 黑客 IBeacons 用于自动化例程

> 原文:[https://hack aday . com/2017/07/17/hacking-I beacons-for-automating-routines/](https://hackaday.com/2017/07/17/hacking-ibeacons-for-automating-routines/)

每个有自尊的黑客在他/她的锦囊妙计里都有一个自动化黑客。有许多现代技术促进了功能，如 GPS、脚本应用程序，甚至 IFTTT。在一次有趣的黑客行动中，[尼克·李]将 iBeacons 和一个逆向工程的星巴克 API 结合起来，创建了一个自动化的晨间例行程序。

通过创建一个扫描 iBeacons 的移动应用程序，[尼克·李]能够减少每天早上去办公室时所做的努力。当应用程序遇到相关信标时，位于云中的 NodeJS 应用程序就会被触发。这就导致了想要的行为，比如订购优步之旅和订购冰拿铁。

[尼克·李]在 GitHub 上分享了 [Starbucks 应用程序的代码，供任何想自动点他们最喜欢的一杯咖啡的人使用。这个项目可以很容易地扩展到与 GPS 甚至 RFID 标签一起工作，如果你想在咖啡机](https://github.com/TENDIGI/Starbucks)上添加物联网[，你可以一次性自动化所有的饮料需求。](http://hackaday.com/2016/10/11/reverse-engineering-the-internet-of-coffee/)
