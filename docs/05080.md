# 触摸敏感的水泥，只有少量的霓虹灯

> 原文:[https://hack aday . com/2017/02/22/touch-sensitive-cement-with just-a-dash-of-neon/](https://hackaday.com/2017/02/22/touch-sensitive-cement-with-just-a-dash-of-neon/)

很长一段时间以来，我们看到人们用水泥浇铸他们自己的工作台面和其他表面。这是一种模具制作和表面加工的结合，如果不考虑安装时对背部的损伤，它可以以相当低的成本产生光滑耐用的表面。

这个产品有点不同。[Elliott Spelman]制作了自己的[触摸感应水泥桌面](http://www.instructables.com/id/Touch-Sensitive-Concrete-Neon-Table/?ALLSTEPS)。当你把你的脏手放在光滑的表面上时，一圈霓虹灯打开了。这要归功于 4:1 的速凝水泥和氧化铁粉末混合物。裸露的铜线被放置在表面的边缘，被水泥包裹，用于以后的连接。

当(埃利奥特)从模具中取出铸件表面时，有一些悲伤的时刻。他最终打破了它，并建议其他人在铸造前在模具上使用蜡，然后耐心地移除水泥。我们也可以建议使用增强剂，如纤维增强剂。边缘和表面可以打磨成所需的光洁度，在这种情况下，安装桌腿很容易，因为模具的木质底面仍保留在水泥底部。

霓虹灯给这座建筑增添了一丝复古气息。看到这种技术逐渐消失令人难过，所以在我们看来，手工霓虹灯的复兴是件好事。[Elliott]发现了一个叫做[the Crucible](http://thecrucible.org/)的湾区艺术集体，它做了很多艺术玻璃教育，帮助他制作两个玻璃管环，并填充适当的气体。AT42QT2120 ( [数据表](http://www.atmel.com/images/doc9634.pdf))的电容式触摸传感器(曾经是 Atmel，现在是微芯片部分)监控来自平板的电线，并使用继电器板和 Arduino Uno 的组合来切换电子管的电源。

我们发现位置传感在掺杂水泥中的前景令人着迷。有没有人有办法改变这种技术，让一个更长更窄的平板可以在几英寸之内有位置意识？请在评论中告诉我们。
