# Hackaday 奖品:价值 50 美元的树莓派智能手机

> 原文：<https://hackaday.com/2017/09/19/hackaday-prize-entry-the-50-raspberry-pi-smartphone/>

Hackaday 奖是一项创造硬件的挑战，而 ZeroPhone 很可能是 Hackaday 奖中最受欢迎的项目。这是什么？这是一款围绕 Raspberry Pi Zero 打造的手机，可以以大约 50 美元的价格组装成零件。已经有[入围了 Hackaday 奖最佳产品](https://hackaday.com/2017/08/01/these-are-the-twenty-finalists-for-the-hackaday-prize-best-product/)竞赛，[入围了 5 万美元大奖](https://hackaday.com/2017/06/19/twenty-iot-builds-that-just-won-1000-in-the-hackaday-prize/)的决赛，并且是 Hackaday.io *上有史以来最受欢迎的项目之一。*

今年早些时候，我们看了一下 zero phone，虽然这个项目已经取得了重大进展，但其理念仍然大同小异。这是一部带有数字键盘和 128 x 64 像素有机发光二极管显示屏的手机——基本上与诺基亚砖块一样的用户界面。这款手机的大脑是一个包裹在 PCB 三明治中的 Raspberry Pi Zero，有 WiFi，蓝牙，HDMI 和音频输出的选项，一个 USB 端口，电池充电，以及大量的 GPIOs，包括 ISM 波段无线电，红外接收器和发射器，更多的闪存，以及你能想象到的任何其他东西。基本上，我们正在寻找一种模块化、可重新配置的智能手机创意，使用树莓派作为大脑。科技记者*应该*在这个问题上打败自己。我们期待《T4》杂志的封面故事。

与任何开源/ DIY 手机一样，围绕 ZeroPhone 的大问题是蜂窝无线电。2G 无线电既便宜又丰富，但基础设施要么很快就会崩溃，要么已经崩溃。3G 无线电是最低可行产品的必备，Arsenijs 说，有条款规定用 3G 模块替换 2G 无线电。当然，3G 模块不像他们的技术前辈那样开放，但这是另一个时间的讨论。

ZeroPhone 已经取得了巨大的成功。有一个真正的团队在这个项目上工作，[zero phone subreddit](https://www.reddit.com/r/ZeroPhone/)比 Hackaday subreddit 更大，有[简讯](https://zerophone.github.io/newsletter/)，有[维基](http://wiki.zerophone.org/index.php/Main_Page)，并且“很快”会有一个众筹活动。这是一个值得期待的项目，也是 2017 年 Hackaday 奖的一个非常有价值的项目。
