# 制作联网的 32X32 LED 面板外壳

> 原文:[https://hack aday . com/2016/07/06/making-a-networked-32x 32-led-panel-case/](https://hackaday.com/2016/07/06/making-a-networked-32x32-led-panel-case/)

[mania Labs]的[Adam Haile]又来了，掀起了一些 LED 的怪异。不过，这个项目比他的大部分作品都要小:他做了一个漂亮的盒子，里面有一个 32X32 的 LED 矩阵屏幕、控制器和一个树莓派。查看下面视频中的构建和简短演示。

这种漂亮的 3D 可打印设计被称为 Jumbo1K，如果你想用这些屏幕制作一些东西，这将是一个很好的起点，因为它可以轻松访问 Pi 上的所有端口，用于编程、调试和联网设备，而不会破坏外观。它用一个巧妙的技巧做到了这一点，当你给房子重新布线时，使用你放在墙上的梯形插孔。

[Adam]通常在更大的规模上工作，用数千个 led 创造出[显示器。](http://hackaday.com/2016/03/12/how-many-leds-are-too-many/)
