# 建立你自己的大脑

> 原文：<https://hackaday.com/2017/01/26/build-your-own-p-brain/>

我不认为我们会称虚拟助手*完成*直到我们可以说，“给我做个三明治”(不加“sudo”)，并且当我们坐在电视机前时，有人给我们做了个三明治。然而，它们并不是像现在这样完全没有用——它们可以让你知道时间、天气和交通，安排或提醒你开会，它们还可以用来从亚马逊订购东西。[Pat AI]对构建一个开源的、可扩展的虚拟助手感兴趣，所以[他构建了 P-Brain](https://github.com/patrickjquinn/P-Brain.ai/)。

把 P-Brain 想象成一个更复杂的虚拟助手的基础。它从一开始就被设计为增加更多的技能，以增加它的复杂性，它可以做的事情的数量。P-Brain 是用 Node.js 编写的，使用一个名为 Natural 的节点包，P-Brain 解析你的请求，并将其与一项“技能”匹配。目前，P-Brain 可以获得时间、日期和天气，它可以从互联网上获得事实，找到并播放音乐，还可以为你抛一枚虚拟硬币。目前，P-Brain 只在 Chrome 上运行，但[Pat AI]计划消除这种依赖性。休息之后，[Pat AI]将介绍 P-Brain 的一些细节，并展示它的功能。在接下来的视频中，[Pat AI]将详细介绍如何添加新技能。

[Pat AI]对 P-Brain 的想法似乎类似于 Jasper 项目，即。，建立一个运行在廉价硬件上的开源虚拟助手，与亚马逊、苹果和谷歌等公司开发的虚拟助手竞争。P-Brain 的发展还处于早期，但 P-Brain 已经可以做很多事情了。希望随着(Pat AI)和其他人开发 P-Brain 的新技能，它会给其他人带来一些激烈的竞争。
