# 大型游戏和手表:真正的多人游戏

> 原文：<https://hackaday.com/2017/11/16/mega-game-watch-true-multiplayer-game/>

今天，我们已经习惯了像任天堂 Switch 这样的掌上游戏机，它让你在 3D 世界中漫游，其中不仅包括 3D 玩家，还包括比许多人在现实生活中一周所走的还要多的地形。但早在 20 世纪 80 年代早期，任天堂的手持产品是 Game & Watch，它使用分段式液晶显示屏。可以使用整个片段来表示玩家，玩家片段遍布整个显示器。为了移动播放器，前一个播放器段将被关闭，而另一个相邻的播放器段将被打开。这也意味着一台游戏机只能玩一种游戏。尽管有这些限制，它们在当时还是很受欢迎的。

[托马斯·蒂利]决定改进旧游戏&用不同的方式观看，[让它变得更大，更大](http://www.tomtilley.net/projects/game-and-watch/)。事实上，它太大了，以至于许多青少年玩家都不能及时够到向左移动的按钮和向右移动的按钮，这使它变成了一个高度合作的双人游戏。从下面的视频来看，这让玩游戏变得更加有趣。他选择的游戏是&观看章鱼，或者英国的海洋之谜和深海之谜。

他使用的屏幕仍然是液晶电视，但是是现代的 46 英寸液晶电视，像素尺寸小得多。这个放在胶合板后面。托马斯没有给胶合板涂漆，而是选择用他岳母提供的金色窗帘覆盖。这意味着在控制台的前面为字母、数字和各种线条定制一些烫印转印。大部分他可以用熨斗来敷，但是有些地方熨斗不合适。对于这些区域，他将一把宜家六角扳手插入烙铁，用热的六角扳手加热转印。按钮也是胶合板，装有弹簧，以便它们可以被推入。按下时，它们会启动长臂微型开关。这些反过来又被连接到一个通过 USB 连接到笔记本电脑的黑客游戏手柄。

如果你正在寻找一个更容易管理的手持游戏&观看黑客，那么看看由[Andrew]设计的这款[，它的大小与最初的](https://hackaday.com/2015/06/15/give-in-to-nostalgia-with-a-diy-game-and-watch/)差不多，但有自己独特的外观。
