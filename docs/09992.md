# 有缺陷的人造钻石可能是量子计算的关键

> 原文:[https://hack aday . com/2018/07/10/有缺陷的合成钻石可能是量子计算的关键/](https://hackaday.com/2018/07/10/flawed-synthetic-diamonds-may-be-key-for-quantum-computing/)

如果你关注过我们关于量子计算的报道，你可能知道最大的挑战是让量子态持续很长时间，尤其是在移动它们的时候。普林斯顿大学的研究人员可能已经解决了这个问题，他们演示了在实验室制造的钻石中存储量子位。如果你想了解更多，实际的[出版物就在付费墙后面](http://science.sciencemag.org/content/361/6397/60)。

一般来说，量子位被当作光子处理，在光纤中移动。然而，它们在这种状态下不会持续很长时间，并且很难用正确的量子信息存储光子。然而，钻石中的杂质可能具有将光子转换成电子的能力。

有趣的是，这项工作有点类似于我们报道的利用钻石中的氮空位存储数据的努力。这种类型的缺陷——钻石晶格中的一个氮原子取代了一个碳原子——确实可以存储数据，但没有正确的光学属性。硅空位在光学上更好，但是在长时间内不稳定。

理论上，钻石中的硅缺陷是电中性的，但附近的其他缺陷可以改变这一点，并导致硅空位带有电荷。研究人员想知道这种虚假收费是否破坏了数据存储的稳定性。

他们与一家钻石制造公司合作，开发了一种清除额外杂质的方法，以形成真正中性的硅空位。这些空位似乎对储存量子位元以及使用和制造光子都很有效。

他们仍在研究一种方法，将量子计算机中发现的光子电路连接到晶体上，因此这项技术的实际应用不会很快。制造钻石的公司——Element Six——有一段他们用来制造工程钻石的一些技术的视频，你可以在下面看到。
