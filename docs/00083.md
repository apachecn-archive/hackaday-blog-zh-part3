# 逆向工程:韦斯顿电气仪器公司

> 原文:[https://hack aday . com/2015/09/22/retro tech tacular-Weston-electrical-instruments/](https://hackaday.com/2015/09/22/retrotechtacular-weston-electrical-instruments/)

电表是任何电子工作台上最重要的工具之一。当你用尽了你的五种感官试图弄清楚电路中发生了什么之后，下一步通常是启动旧的“电表”。如今，电表基本上都是数字化的，但它们的模拟祖先仍然随处可见。我们要感谢化学家兼发明家[爱德华·韦斯顿]为 DC 测量设备的便携性和普及性做出的贡献。

带着父母希望他获得的医学学位从英国移民到美国后，[爱德华·韦斯顿]声称他对化学更感兴趣。他的职业生涯始于电镀，在那里他很快意识到他需要一个可靠的、恒定的电流源来进行高质量的电镀。对发电的强烈兴趣使他开发了一种饱和镉电池，这种电池被称为韦斯顿电池。其化学性质产生足够稳定的电压，可用于仪表校准。韦斯顿电池也适用于确定电动势。

几年后，他与人共同创立了韦斯顿电气仪器公司。该公司生产了几种类型的仪表以及变压器和传感器，以其便携性和准确性而闻名。1920 年，[韦斯顿公司]与美国海军合作，制作了这部 1920 年的教育电影，作为电学原理系列的一部分。

观众被邀请思考测量对文明的重要性，最值得注意的是长度、质量和时间的基本测量。[Weston]将他的电子测量仪器定位于这一水平，将其吹捧为国际上最受欢迎的产品。我们参观了韦斯顿流量计，从经过永久处理的磁铁到专门设计的能够正确分配磁力线的极片。没有铁屑演示，哪部关于电磁学的教育片是完整的？这一款绝对能送货上门。
