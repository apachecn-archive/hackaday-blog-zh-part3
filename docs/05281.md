# 问问 Hackaday:帮助之手

> 原文：<https://hackaday.com/2017/03/12/ask-hackaday-helping-hands/>

[ProtoG]在[发给我们的这个视频](https://www.youtube.com/watch?v=wVtT6xP_yjo)(也在下面)中，他演示了如何使用机械师的千分表指示臂作为助手。我承认我嫉妒得订购了一双。我不会说我*需要*更多的工具来固定东西，但我肯定*想要*它们。快速粗置结合微调看起来好甜。将它们用作示波器探头支架是明智之举。

我们自己的援助之手，从一家旧货商店花 5 美元买来的，现在已经用了近 20 年了。大约十年前，我对钳夹进行了热收缩和浸塑处理，从那时起，钳夹对电缆绝缘的损害就减少了。夹子不断松脱，但用一点环氧树脂固定住了。我从来没有使用过放大镜，通过移除它，我为下巴买了更多的滑动空间，这是一个轻松的胜利。底座上有一层“防滑”的鞋胶涂层，可以将它固定在桌子上。软木可能更高级。

对于更大的夹持，总是有台钳，虽然我承认我主要用它来夹持 PCB，而焊接，这是一个更好的解决方案，这一特殊的任务不会伤害。[Mike Szczys]告诉我，这里看到的老虎钳是长凳上的一个方便的东西。它始于 Hackaday.io，我们仍然在商店里出售它。

对于抓取精细的小东西，没有什么比一对止血钳和一系列镊子更好的了。台钳里的止血钳是一个很好的临时支架。好的锋利的镊子通过第一次移除碎片或放置 SMT 零件来支付费用。

那么，哈卡代，你用什么来拿东西？焊接时，你用什么拿着你的印刷电路板？你用什么来[压住 SMD 零件](http://hackaday.com/2015/02/05/diy-smd-twofer-manual-pick-and-place-and-the-beak/)？你的第三只手是什么，还是第二十三只？
