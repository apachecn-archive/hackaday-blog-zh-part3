# 驾驶 IRL 的第三人

> 原文：<https://hackaday.com/2016/03/19/third-person-driving-irl/>

梦想成真了:遥控真车。除了有趣之外，真人大小的遥控车辆还有一些实际应用，比如执行救援行动或向危险地区运送物资。对于[卡特]、[戴夫]、[瑞安]和[肖恩]，[来说，在俄亥俄州立大学的黑客马拉松中，这个梦想在 24 小时内变成了现实。他们将这个系统命名为 MagiKarpet，因为它取代了地垫，在精灵身上运行。](http://www.youtube.com/watch?v=2Ok_ntlTiLM)

该计划是使用 PlayStation 控制器来控制雪佛兰 Cobalt 的油门、刹车和转向。为了增加乐趣，安装在后保险杠上方的摄像机将提供第三人称视角，这种反馈将显示在后座的显示器上。一切都由 Arduino Mega 控制。一个结实的线性致动器控制刹车，并临时连接一个绕着踏板滑动的 Shapelock 带。油门是由附在车窗电机上的杠杆推动的。另一个电机通过电缆连接到方向盘上，可以左右转动 90 度。虽然构建是成功的，但是他们遇到了一些问题。但是没有一些问题的黑客马拉松体验是什么样的呢？

在一些早期测试后，线性致动器被卡住了大约一个小时，但他们把它弄开了。PS 控制器坏了，所以他们不得不滚动自己的操纵杆。出于安全考虑，学校不会让他们真的开着它到处跑(扫兴，但我们知道)，所以他们把它放在千斤顶上[向评委们展示](http://youtu.be/2Ok_ntlTiLM?t=5m3s)。他们获得了第二名，尽管我们无法想象还有什么能打败他们。休息之后，请查看完整的构建视频。

你可能还记得去年这个时候的这些人。他们在同一个黑客马拉松中获得了第一名，这是一款由应用程序控制的煎饼机器。既然你已经饿了，那就尽情享受[这个没完没了的](http://hackaday.com/2016/03/11/endless-pancakes/)吧。
