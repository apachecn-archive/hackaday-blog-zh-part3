# 脑性瘫痪摄影师用刮擦式照相机万向架

> 原文:[https://hack aday . com/2017/03/21/scratch-built-camera-gimbal-for-photography-with-脑瘫摄影师/](https://hackaday.com/2017/03/21/scratch-built-camera-gimbal-for-photographer-with-cerebral-palsy/)

我们经常为了黑客而黑客，仅仅为了挑战而独自从事项目。所以很高兴看到黑客技术得到了很好的利用，并帮助了一些人。这个低成本的双轴手持相机万向架就是这种情况，它旨在帮助一个有运动障碍的初露头角的摄影师。

当[Tadej Strah]加入他的学校摄影俱乐部时，一位碰巧患有脑瘫的成员需要帮助稳定相机以拍摄清晰的照片。所以[Tadej]决定为他的朋友做一个，而不是花很多钱买一个商业万向架。几片铝棒被弯成万向架和相机支架。两个业余伺服系统负责俯仰和横滚轴，由 Arduino 与 MPU-6050 通信控制。安装在角磨机的手柄上，电池和电子设备安装在下面，万向节看起来很平衡，很好地保持了相机的水平。

向[Tadej]致敬，他用自己的技能解决了一个现实世界的问题。我们喜欢看到人们直接帮助他人，无论是为帕金森氏症患者制造陀螺勺子还是为几乎失明的冒险家增强视力。
