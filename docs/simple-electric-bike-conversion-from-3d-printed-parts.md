# 从 3D 打印零件转换成简单的电动自行车

> 原文：<https://hackaday.com/2017/06/23/simple-electric-bike-conversion-from-3d-printed-parts/>

挑战:在自行车上进行电动转换。问题:没有重要的金属加工技能或设备。解决方案: [3D 打印电动自行车所需的一切](https://www.youtube.com/watch?v=9rIIJbDkpN8)。

至少这是[汤姆·斯坦顿]制造电动自行车的方法。在最近的长板制造中发现了电动运动的缺陷后，[Tom]着手升级一款廉价的“fixie”，或固定齿轮自行车。他的 delta 打印机足够大，可以打印电机支架和耐候 ESC 外壳，但他需要打印四个象限的驱动滑轮，然后将它们粘在一起。我们不能说我们对通过后轮辐条传递所有扭矩的拉链带有多大信心，但作为概念的证明，它似乎足够坚固。有了电动滑板车的油门和鞍囊里的电池，这辆自行车的性能相当不错——至少在一次小的变速后。一切都融入或强调自行车的黑色框架，所以这是一个很好看的建立启动。

想抓住便宜的电动个人交通工具吗？看看[这个电动长板](http://hackaday.com/2017/01/04/another-electric-longboard-goes-the-distance/)，或者[这个全地形悬浮滑板](http://hackaday.com/2016/07/28/all-terrain-hoverboard-junk-pile-build/)。
