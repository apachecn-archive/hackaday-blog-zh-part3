# Hackaday 奖参赛作品:橱柜下 LED 照明控制器

> 原文：<https://hackaday.com/2016/09/27/hackaday-prize-entry-under-cabinet-led-lighting-controller/>

[Matt Meerian]的工作台似乎永远处于阴影中，因此他已经能够熟练地在所有的架子和橱柜下安装 LED 灯带。这解决了任何在黑暗中寻找东西的问题，但也带来了一个新问题，那就是他冒着让很多 LED 灯带亮着的风险，而且去把它们都关掉是很乏味的。

他的解决方案是让[成为他所有家用 LED 灯](https://hackaday.io/project/12797-under-cabinet-led-lighting-controller)的无线控制器，由他的安卓平板电脑上的一个网络应用程序控制。一个 ESP8266 和一组 MOSFETs 提供内部工作，整个呈现在一个非常紧凑和设计良好的紫色 OSH Park PCB 回流焊上，焊接在一个 20 美元的沃尔玛电炉上，并设置在一个塑料外壳中。web 界面仍在开发中，但对于 ESP8266 代码，它有一个相当简单的 CSS 前端。所有软件、原理图和 BoM 都可以从上面链接的 Hackaday.io 页面下载。

这个项目不会结束世界饥饿或停止战争，但它做得很好，有据可查，它让[马特]的生活变得容易多了。这使得它成为了 Hackaday 奖的一个很好的参赛作品。
