# Hackaday 奖参赛作品:打印骨头

> 原文：<https://hackaday.com/2017/06/08/hackaday-prize-entry-printing-bones/>

你可能会认为 3D 打印只是一卷卷的细丝和一桶桶的树脂。事实是，还有更多 3D 打印技术。从粉末到纸张的一切都可以用来制造 3D 模型。[【Jure】的 Hackaday 奖参赛作品](https://hackaday.io/project/21447-printing-bones-on-a-diy-powder-bed-3d-printer)意在探索那些更古怪的 3D 制造技术。这是一台打印机，它将粘合剂放在粉末容器上，慢慢地建造出矿物制成的物体。

粉末打印机的关键问题是这台打印机到底要用什么材料。为了这个项目，[Jure]计划用羟基磷灰石打印，这是一种占骨骼重量 70%的矿物质。打印骨头——是的，他们确实是这样做的——非常昂贵，而且有多种应用。

这台打印机的设计和你所期望的差不多。这是一个笛卡尔设计，有一个滚筒来分配粉末，一个活塞来将零件放入框架中，还有一个为宽幅打印机设计的工业喷墨打印头。这是一件了不起的作品，也是我们见过的最好的粉末打印机之一，我们迫不及待地想看看[Jure]能用它打印出什么。
