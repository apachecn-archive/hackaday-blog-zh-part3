# 在有谢妮管之前，就有侧光显示器了？

> 原文：<https://hackaday.com/2017/03/23/before-there-were-nixie-tubes-there-were-edge-lit-displays/>

我们已经看到了一堆使用 led 和边缘照明丙烯酸的数字谢妮管的替代品。但最早的数字电压表之一使用边缘发光的有机玻璃板来显示数字，并使用许多白炽灯来照明。

[stevenjohnson]有一个非线性系统 481 型数字电压表，[他已经把它拆了](http://www.stevenjohnson.com/nls/481-display.htm)，所以我们可以一窥其内部。同样，任何看过现代版侧光数字显示器的人都知道它们是什么:一系列刻有数字(或字符)的透明塑料板，每个板下面都有一个光源。你打开一盏灯来照亮一个盘子，打开另一盏灯来照亮另一个，以此类推。这里有趣的一点是使用白炽灯泡和使用顺序继电器来循环照明。继电器会发出很大的噪音，尤其是在外壳打开的情况下。

[stevenjohnson]还指出，他可能在打开存放印版的机器部件时犯了一个错误，因为他花了一些时间将印版放回原位并放回装置中。我们可以想象，如果你用这台机器进行大量的测量，它会发出很大的声音，尽管它的内部看起来很棒，而且很明显，这是一个很好的想法。看看这个[侧光数码管](https://hackaday.com/2012/02/18/edge-lit-nixie-tube-is-sheer-brilliance/)显示器或者这些[侧光数字模块](https://hackaday.com/2016/12/22/light-pipes-and-leds-team-up-for-a-modern-take-on-the-nixie-tube/)。
