# PogoPlug Hacking:拥有设备的逐步指南

> 原文:[https://hack aday . com/2017/04/24/pogoplug-hacking-a-step-by-step-guide-to-owning-the-device/](https://hackaday.com/2017/04/24/pogoplug-hacking-a-step-by-step-guide-to-owning-the-device/)

Kris Hardware 出品的电影]在 YouTube 上开始了一个非常有趣的系列，内容是[黑客攻击和拥有 PogoPlug](https://www.youtube.com/playlist?list=PL2GALWZIiXOCgRfWSQpIFc4QfINjOvsYr) 移动 v4。虽然这在过去已经做了很多次，但他给出了一个很好的循序渐进的教程。到目前为止，该系列令人印象深刻，详细介绍了如何通过 serial a 串行连接获得对设备的 root 访问权限。

PogoPlugs 是运行在 800 MHz 的 ARM 处理器的远程访问设备，它受 Linux 内核支持。有问题的版本(PogoPlug Mobile v4)在过去已经被重新用于类似于[一个廉价的 PBX](https://hackaday.com/2014/09/15/hacking-a-pogoplug-into-a-20-pbx/) ，一个 OpenWrt 路由器，甚至是一个 [squeezebox 的替代品](https://hackaday.com/2013/01/11/building-an-inexpensive-squeezebox-client-replacement/)。即使您没有 PogoPlug，这也是入侵任何基于 Linux 的消费设备的绝佳入门。

到目前为止，我们已经完成了这个八部分系列的第三部分，所以如果您继续学习的话，将会学到更多的东西。他的视频已经涵盖了如何通过串行端口连接到设备，如何发送命令，设置设备，以及阻止它呼叫总部。这将使初露头角的黑客能够让 PogoPlug 执行他们的命令。在这个廉价单板 Linux 计算机的时代，破解这种类型的设备可能会过时，但你在这里学到的技能可能不会很快过时。
