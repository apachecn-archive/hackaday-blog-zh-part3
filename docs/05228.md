# 用于 7 段显示器的苏打碳酸氢盐二极管控制电路

> 原文：<https://hackaday.com/2017/03/06/soda-bicarb-diode-steering-circuit-for-7-segment-display/>

[黑尔斯]一直致力于制造自己的二极管并投入使用，现在他成功地用碳酸氢钠和水、铝带和焊接铜制成了二极管。通过组合其中的 49 个，他组装了一个苏打碳酸氢盐二极管控制电路，用于 7 段显示器，能够显示数字 0 到 9。

他从电解电容器中获得了二极管的想法。一个简单的 DIY 电解电容器有一个浸在液体电解质中的铝片。铝和导电电解质是两个电容器极板。电介质是氧化铝层，当施加正确的极性时，它形成在铝上，阻止电流流动。但是如果你颠倒极性，氧化层就会分解，电流就会流动。对[黑尔斯]来说，这听起来像是它也可以充当二极管，所以他开始工作，做了大量的实验和改进，直到他确信他有一些工作得相当好的东西。

最后，他想出了一种二极管，这种二极管从覆盖着焊料的铜基底开始，以保护铜不受碳酸氢钠和水电解质的影响。一块铝带放在它上面，但与它电绝缘。然后将电解液涂抹在焊料上，一部分涂在铝带上。氧化物在电解质和铝之间形成，提供二极管的结。连接到焊接的铜和铝。

为了真正尝试一下，他为一个七段显示器组装了一个转向电路。为此，他制作了一个二极管矩阵。该矩阵有七列，显示器上的每一段一列。然后有十行，从 0 到 9 的每个数字一行。例如，数字 1 只需要两个片段来点亮，因此对于代表 1 的行，只有两个二极管，即两个 dab 的电解质，其中行与列重叠用于期望的片段。这些列被永久地连线到它们的段，因此最终的连接只需要通过激励适当的二极管行来进行。你可以在休息时间下面的视频中看到[黑尔斯]演示了这一点。

Hales 发现的一个问题是，由于其电容性质，二极管有点慢，但他对此无能为力。然而，他试图解决的另一个问题是电解质变干，需要小心地重新润湿。因此，虽然这还不是永久二极管的方法，但我们钦佩[黑尔斯]的毅力和创造力，如果他坚持下去，他无疑会找到解决办法。
