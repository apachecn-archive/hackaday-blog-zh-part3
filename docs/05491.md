# 最小的波爆卡

> 原文：<https://hackaday.com/2017/03/29/the-smallest-wave-blaster-card/>

在 90 年代早期，创意声霸是*声卡。它不是市场上绝对最好的声音卡，但它很快成为事实上的标准，并以合适的价格提供良好的声音。它主要依赖于雅马哈 OPL-3 FM 合成芯片，但如果你觉得很贵，你可以用 Wave Blaster 附加卡来升级它，它本质上是将一个样本合成引擎固定在卡上。这给了卡一个广泛的样本乐器调色板，用它来演奏 MIDI 音乐更加甜美，这样你就可以给来玩 DOOM 的你的小学同学留下深刻印象。*

现在是 2017 年，如果你没有查看日历的话，去年的声霸只会进一步涨价。不言而喻，附加子板和附件更为罕见，并将相应定价。因此，如果你正在建造一个老式的游戏装备，并且迫切需要这种样本合成器的好处， [[Serdashop]在这里帮助他们的最新产品，Dreamblaster S2。](https://hackaday.io/project/20662-s2-the-smallest-general-midi-board-in-the-world)

据报道，这是目前最小的 Wave Blaster 附加板，非常棒。如果你把它放在声霸 16 的顶部，是的，这是没有意义的——你并不缺少空间。但是如果你想[将这个与一个紧凑的微控制器项目](http://hackaday.com/2014/07/31/ps2-synth-will-knock-you-off-your-broom/)集成呢？尺寸很重要。是的，你可以给它输入 MIDI 信号，它就会为你歌唱。给门外汉的一个热门提示:MIDI 说串行，就像任何事物和任何人一样。你奶奶在战争中学会了说英语，你知道的。

你可以选择把它插在 Wave Blaster 兼容的卡上，或者购买一个载板，除了把它定制到你自己的硬件上之外，还可以把它和游戏端口一起使用。我们很乐意看到这是树莓派零度的帽子。做吧，把它送进来，我们会写下来。

我们以前在这里见过[Serdashop]的硬件——也就是早期的[dream blaster X2](http://hackaday.com/2017/01/21/dreamblaster-x2-a-modern-midi-synth-for-your-sound-blaster-card/)。休息下的视频。
