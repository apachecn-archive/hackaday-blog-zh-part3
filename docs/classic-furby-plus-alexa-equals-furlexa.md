# 经典的 Furby 加 Alexa 等于… Furlexa

> 原文：<https://hackaday.com/2017/11/30/classic-furby-plus-alexa-equals-furlexa/>

[扎克·莱文]写来分享一个刚刚完成的项目:一个经典的 Furby 包一个运行 Alexa 的 Raspberry Pi:他称之为 Furlexa。

最初的 Furby 产品让 90 年代的消费者惊叹不已。除了电子动画运动，它还装有模拟语音学习技术，似乎可以让 Furby 学会说话。它不像市场上的其他任何东西，甚至让这个玩具被禁止进入国家安全局的设施，以防它可以监视他们。优雅地说，机器人只使用一个马达来移动它的所有部件，使用各种塑料齿轮、杠杆和凸轮来控制机器人的所有身体部位，并使其跳舞。

在过去的二十年里，Furby 赢得了有史以来最容易被黑客攻击的玩具之一的声誉——尽管它有神秘的微控制器，它被密封在塑料中以保持制造商的知识产权秘密。[Zach]把控制板换成了 Pi。他还用 Pimoroni 扬声器 pHat 和更好的麦克风取代了 toy 附带的蹩脚麦克风和 pizeo 扬声器。

虽然经典的 Furby 以善于黑客攻击而闻名，但新的 Furby 也不能幸免:这款渗透 Furby 的[基于这款玩具的最新型号。](https://hackaday.com/2017/11/26/mission-impossible-infiltrating-furby/)
