# 业余无线电爱好者(和其他人)的 LTSpice

> 原文：<https://hackaday.com/2017/02/15/ltspice-for-radio-amateurs-and-others/>

我们不认为[VK4FFAB]把他的七部分 LTSpice 教程称为业余无线电爱好者 lt spice 是帮了他自己一个忙。当然，这些帖子确实专注于无线电频率分析，但如今许多人都参与了不一定是业余无线电爱好者的无线电工作。

不管怎样，如果你对模拟 RF 放大器和滤波器感兴趣，你应该看看这些帖子。当然，前几个包括简单的事情，如分压器，只是为了让你的脚湿。最后一部分甚至包括一个双平衡混音器和一些变压器，所以有相当多的材料。

我们喜欢 LTSpice。功能强大，简单易用，运行 Linux(用 WINE)，价格合适。既然是通俗的，也有大量的例子和教程，包括这个。

我们很高兴看到[VK4FFAB]解决了变形金刚，因为如果没有一些例子，它们是不容易解决的。在我们的 LTSpice 教程第三部分中，我们也谈到了变压器问题。事实上，在阅读[VK4FFAB]的前两篇文章时，你可能会发现我们的视频很有帮助。我们已经看到其他无线电项目使用 LTSpice，包括一个[再生接收器](https://hackaday.com/2016/03/02/a-spicy-regenerative-reciever/)。如果安装 LTSpice 对你来说太难了，你也可以在你的浏览器中进行模拟[。Falstad 模拟器没有那么强大，但它确实有一些自己非常简洁的功能。](https://hackaday.com/2015/07/20/a-breadboard-in-a-browser/)
