# 一群伺服机构演奏这个机器人钟琴

> 原文：<https://hackaday.com/2017/11/23/swarm-of-servos-plays-this-robotic-glockenspiel/>

它是行进乐队中最快乐的发声乐器，而且它有最好的名字。这是钟琴，如果这个机器人钟琴有什么要说的，人类钟琴演奏家的日子屈指可数了。

在其目前的原型形式中，[Averton Engineering]的“Spielatron”看起来有点像旋转木马 calliope 或电子披萨餐厅乐队。Spielatron 使用学校音乐室的废弃钟琴作为基础，使用四个木槌来演奏所有的音符。每个键都由固定在由两个伺服电机组成的底座上的木槌敲击。由于缺乏更具描述性的木槌术语，这些伺服系统提供平移和倾斜，因此木槌可以敲击正确的键。下面的视频展示了 Spielatron 的首次独奏会。

Arduino 运行伺服系统和 MIDI 接口；不幸的是，这个版本不能演奏和弦，并且在音符长度上有点限制，但升级正在进行中。我们以前见过类似设计的机器人钟琴，它可能有一些提高性能的想法。但是如果你在寻找更美妙的声音，看看[这个干冰驱动的风铃](https://hackaday.com/2017/09/19/wind-chimes-and-dry-ice-make-an-unusual-musical-instrument/)。
