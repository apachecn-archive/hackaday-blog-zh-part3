# 美丽的 DIY 点焊机提醒我们热爱 3D 打印

> 原文：<https://hackaday.com/2017/07/18/beautiful-diy-spot-welder-reminds-us-we-love-3d-printing/>

[Jim Conner]的 [DIY tab 点焊机](https://www.thingiverse.com/thing:2429361)是我们见过最甜的点焊机。我们并不羞于承认[我们在](http://hackaday.com/2015/07/05/the-nicest-home-made-spot-welder-weve-ever-seen/)之前已经说过了。

点焊机的本质只不过是一个微波炉变压器重新缠绕，以产生低电压和高电流，而不是相反。有些人赤手空拳控制焊接过程中的脉冲长度，而另一些人认为最好用 555 定时器电路来实现。[Jim]的版本使用了 NodeMCU 板，这是非常多余的，但当时它就在他的桌子上。他在 GitHub 中关于 Lua 编码的评论[太熟悉了——数组又是如何工作的呢？](https://github.com/jimconner/NodeMCU-Dual-Pulse-Weld-Timer)

使用更好的微控制器意味着他可以做一些新奇的事情，比如双脉冲焊接等等。他甚至不碰无线功能，但无论如何。有机发光二极管和旋转编码器系统很可爱，但这里的明星是 [3D 打印外壳](http://a360.co/2sEro9Y)，配有柔软的部件，当[Jim]使用焊接机时，他的手就放在那里。看起来他可能买了这个东西。


这些视频不遗余力地记录了这个项目。如果你有时间，去看看。亮点包括第 1 部分中的概述，第 2 部分中的 SSR 和机箱设计，第 3 部分中的烟熏木材和大型电池组延时。有一个额外的视频详细介绍了电极。
