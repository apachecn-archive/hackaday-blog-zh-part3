# 将宋飞低音即兴重复添加到任何门口

> 原文:[https://hack aday . com/2016/01/08/add-Seinfeld-bass-riffs-to-any-doorway/](https://hackaday.com/2016/01/08/add-seinfeld-bass-riffs-to-any-doorway/)

这不是我们本周看到的最复杂的黑客，但它绝对有创意，让我们发笑。[Grant]一些朋友开玩笑说，每当有人进门时就播放情景喜剧风格的入门音乐是多么酷，但在现实生活中。你知道，就像每个人都想有自己的主题音乐？这是同样的想法，[但是宋飞等同于](http://hotdamntv.blogspot.de/2016/01/how-to-do-science-with-grant.html)。(视频嵌入休息下方。)

[![20160104_213816](../Images/e60c6db683ec0b176a2e93a9344ca5bf.png)T2】](https://hackaday.com/wp-content/uploads/2016/01/20160104_213816.jpg)

几个简单的部件(Arduino，MP3 屏蔽，磁性门簧片开关)和一点点随机播放低音重复片段的代码，就完成了。就像我们说的，这不是[火箭科学](http://hackaday.com/2015/12/31/32c3-so-you-want-to-build-a-satellite-2/)，但它是一个看起来像库存充足的游戏室的幽默补充。(那是我们看到的 Joust 控制台吗？)

为了更好的喜剧效果，为什么不在你的滑板坡道上添加宋飞的贝斯片段呢？
