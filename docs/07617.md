# 从树脂施法者到树脂掌控者

> 原文:[https://hack aday . com/2017/11/07/go-from-resin-caster-to-resin-master/](https://hackaday.com/2017/11/07/go-from-resin-caster-to-resin-master/)

说到树脂铸造，时间是至关重要的。这有助于在开始前收集你需要的所有东西并放在手边。但是如果你不知道自己不知道什么，就很难预测需求。幸运的是， [[Botzen Design]有一些锦囊妙计，可以为新手和老手节省时间、材料和理智。](https://www.youtube.com/watch?v=c2HBuM85EA4)

在一次性或可重复使用的塑料杯中混合树脂似乎有些显而易见。但并不是所有的杯子都是一样的。聚丙烯杯子不会向你的树脂中放气，但是聚苯乙烯会。如果您使用硅胶杯或任何标有#5/PP 的聚丙烯食品容器，固化树脂将从杯壁上干净地剥离。

出于某种原因，树脂[博岑设计]使用的大罐子没有配备泵。他们怎么能指望有人从加仑的罐子里倒出树脂和硬化剂的准确数量呢？餐馆供应店的体育场式调料泵使事情变得简单多了，同时避免了昂贵的溢出。

我们最喜欢的技巧(似乎也是[Botzen Design]最喜欢的)是滴水锤。当气泡成熟形成凹坑时，可以用一两滴湿树脂轻松而精确地填满它们。移液管可能会堵塞，但挂在棍子上的固化树脂冰柱是完美的滴涂器。
