# 一个微型钻头完成了它的目标

> 原文：<https://hackaday.com/2017/03/04/one-micro-bit-accomplishes-its-goal/>

像树莓派一样，BBC Micro Bit 的目标是首先成为一种教育设备。这种廉价的计算机很好地适应了当前尽可能削减公立学校预算的趋势，同时仍然能够让孩子们对编码和计算机感兴趣。虽然这两款电脑都被黑客用于各种项目(特别是 Pi)，但[大卫]的最新版本至少让他的孙子们对电脑感兴趣，因为[使用微型芯片给旧玩具](https://www.kitronik.co.uk/blog/microbit-lap-counter-scalextric/)添加了一些很酷的功能。

问题中的玩具是一辆旧的 Scalextric 赛马场赛车——英国的另一个知名产品。但是如果你不能记录圈数或圈速，比赛有什么意思呢？有了 BBC 的微型钻头和一些硬件，新改进的赛道可以做所有这些事情。它还实现了一个赛车风格的灯光系统来启动比赛，并可以判断汽车是否错误启动。直观地了解微型钻头在其 LED 阵列上显示的所有信息可能有点困难，但这不需要太多的练习。

项目页面详细介绍了项目是如何构建的。一定要看看下面的一些激动人心的比赛视频！这个建筑肯定会让[大卫]的孙子们开心一段时间，也能帮助他们参与编程和建造他们能想到的任何东西。也许他们甚至会抽出时间来制造一两个机器人。
