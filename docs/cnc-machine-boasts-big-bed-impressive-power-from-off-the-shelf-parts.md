# 数控机床拥有大的床，来自现成零件的令人印象深刻的力量

> 原文：<https://hackaday.com/2017/05/15/cnc-machine-boasts-big-bed-impressive-power-from-off-the-shelf-parts/>

许多自制的 CNC 机器最终被美化为附有路由器的绘图仪，除了铣削木材和塑料等软材料外，没有什么用处。因此，如果你迫切需要快速、安静地铣削铝和软钢等较硬的材料，那就把眼光放高一点，用现成的部件制造一台大型数控机床。

凭借 2000 欧元的预算,[ s rens 7]不像我们见过的许多低端 CNC 构建那样受到限制，这些构建几乎总是依赖于 [3D 打印部件](http://hackaday.com/2016/01/25/sub-300-cnc-if-you-have-a-3d-printer/)甚至来自垃圾桶的[材料。虽然我们肯定称赞每一个数控建设，这表明，负担得起的和容易获得的机电一体化可以导致相当大的能力螺栓建设。[s rens 7]的这台机器的 BOM 是 100%目录购物，从铝挤压床和龙门到线性轴承和循环滚珠丝杠。工作区是一个宽大的 900 x 400 x 120mm 毫米，步进器是结实的 NEMA23s，主轴是一个 3 千瓦的 VFD 单元，功率充足。下面的视频显示了机器令人印象深刻的干切铝性能。](http://hackaday.com/2015/11/21/garbage-can-cnc-machine-build/)

总的来说，[索伦 7]低于预算 500 €，对于这样一台大而强大的机器来说，这是一个诱人的价格点。
