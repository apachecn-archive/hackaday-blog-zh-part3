# 用石器时代的黑客来玩你的小房子游戏

> 原文：<https://hackaday.com/2016/01/18/up-your-tiny-house-game-with-stone-age-hacks/>

光着脚，光着手，光着胸——如果不是因为短裤和头顶上飞机的短暂声音，我们会发誓下面的视频是穿越时空的片段。没有 Arduinos，没有 CNC 或 3D 任何东西，但如果你怀疑我们石器时代的祖先是黑客，看看[原始技术]在没有现代工具的情况下建造瓦顶小屋时经历了什么。

我们要指出的第一件事是，[原始技术]并不试图成为(前)历史上的准确。他借用了人类历史上不同时代的技术来建造——瓦片屋顶直到大约 5000 年前才出现，到那时他的石凯尔特斧应该已经过时了。但是原始技术爱好的重点是不使用任何现代技术来建造一些东西。如果你需要火，你用火弓；如果你需要一把斧子，那就把一块石头塑造成。他的 102 天建造日志详细记录了每一步。看着原木、泥土、树苗、岩石和粘土组合成一个令人惊讶的舒适结构，真是令人着迷。如果有点不合时宜的话，特别棒的是地下中央供暖系统，它可以把小屋变成一个可爱的桑拿浴室。

原始技术看起来像一个迷人的爱好，有很多东西可以教我们如何走到现在。但是如果你不喜欢在泥地里挖掘，你总是可以 3D 打印一个粘土小屋。不过，我们不确定建造一个巨大的三角洲机器人是否更容易。
