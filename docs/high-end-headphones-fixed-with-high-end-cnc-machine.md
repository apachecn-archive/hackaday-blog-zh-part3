# 高端耳机用高端数控机床固定

> 原文：<https://hackaday.com/2016/08/30/high-end-headphones-fixed-with-high-end-cnc-machine/>

保修？我们不需要什么臭保修！我们是黑客，如果你有价值数百万美元的 CNC 机器和 3D CAM 软件，你就可以自己加工耳机替换零件，而不是接受制造商的免费施舍。

有问题的耳机 Grado SR325s 是手工制作的高端高保真耳机，但[Huibert van Egmond]发现将杯子固定在头带上的万向节松动并脱落。他在 CAM 中复制了原始万向架的设计，生成了数字代码，并让他巨大的布里奇波特铣床在一块大铝块上松脱。在小型膝磨机上对零件进行钻孔和攻丝，在车床上将零件从背衬材料上切下，并进行喷丸处理以去除铣削痕迹。快速喷涂一层油漆——我们更喜欢粉末涂层或阳极氧化——零件就可以放回耳机上了。

当然，这有点过了，但是当你有工具的时候，为什么不呢？甚至一台 DIY CNC 刳刨机也可能生产出这样的零件——当然，速度要慢得多，但还是有可能的。
