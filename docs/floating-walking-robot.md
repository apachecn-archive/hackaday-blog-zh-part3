# 漂浮行走机器人

> 原文：<https://hackaday.com/2016/11/23/floating-walking-robot/>

众所周知，我们喜欢奇怪的机器人运动，所以我们天生喜欢浮力辅助的轻型腿单元 [BALLU](https://www.youtube.com/watch?v=EdSoUbXirVI) (YouTube 链接，也嵌入在下面)。该项目从一个简单的观察开始——行走机器人受到必须支撑自己的限制——消除这种限制会使成功变得容易得多。BALLU 几乎不用走路，而是漂浮着，用*和*仅有的一点净重推动地面。

腿实际上只是中间弯曲的棍子，用电机和电缆控制弯曲。抬起一条腿，重心慢慢落向那只脚。最终将它向外拉伸提供了向前的力。这就是噱头，挑战在于驾驶这个东西，目前它看起来是远程控制的。肯定不会是“右转——马达，bot 左转”那么简单。正如视频最后几分钟展示的那样，你不会想让它顶着强风。

无论如何，如果你不能获得足够多的奇怪机器人机构，这里有一个[最近的蛇机器人](http://hackaday.com/2016/07/20/ourobot-what-happens-when-a-snake-bot-swallows-its-own-tail/)，一个[游动的鱼](http://hackaday.com/2016/09/23/the-arduino-sleeps-with-the-fishes/)，一个[摇摆的伺服果冻堆](http://hackaday.com/2016/08/17/struggling-robot-made-with-diy-soft-limbs/)，还有一个[单电机波浪虫](http://hackaday.com/2016/08/07/single-motor-lets-this-robot-do-the-worm/)。如我们所说，我们有 3 个奇怪的机器人。
