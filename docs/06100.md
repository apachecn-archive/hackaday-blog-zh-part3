# 3D 打印辐射图案

> 原文:[https://hack aday . com/2017/06/11/3d-printed-radiation-patterns/](https://hackaday.com/2017/06/11/3d-printed-radiation-patterns/)

天线的辐射模式可能完全令人困惑，尤其是在二维空间中，因为它们通常是二维的。别担心，[亨特]有 3D 打印和彩色编码的辐射图案做你的后盾。

在天线设计领域，辐射模式表示从天线发射的无线电波的相对强度和接收器/发射器在 3D 空间中的位置之间的关系。在实践中，探针可以用来在记录信号强度的同时从天线周围有记录的位置进行发射/接收，使研究人员和工程师能够确定神秘天线的特性。这些测量通常表示为三维平面的二维切片。自然地，这种有时(经常)复杂的几何图形对于所有人来说都是困难的，除了最有空间倾向的人，仅借助于 2D 的画在精神上概念化。随着计算机的出现，3D 模型出现了，但[亨特]并不满足于屏幕上的模型:他们想要可以拿在手中的东西。

为此，【猎人】模拟了几种不同的天线结构，清理模型进行 3D 打印，用彩色砂岩 3D 打印出来，最终效果很漂亮。通过 Shapeways 在彩色砂岩上打印，[Hunter]现在有了核桃大小的彩色编码辐射图案，他们可以拿在手中。为了节省其他人的工作，[亨特]把他的设计贴在了 Shapeways 的旧工程专柜。到目前为止，他有一个喇叭天线，偶极天线，插入馈电[贴片天线](http://www.antenna-theory.com/antennas/patches/antenna.php)和一个更高阶模式的贴片天线，所有这些都在 15.00 美元以下。难道看不到你梦想中的天线辐射模式吗？别担心，[Hunter]正在寻找请求，所以请在评论中发表你的想法吧！

此外，除了眼前的酷因素，我们还可以看到[亨特]模型的许多合法用途，尤其是在教育领域。随着越来越多的研究促进结构性学习，而不是程序性学习，[亨特的]设计可以很容易地成为未来天线理论课的教学支柱。[亨特]并不是唯一一个让无形变得可见的人，[【查尔斯】正在绘制三维 WiFi 信号图](http://hackaday.com/2015/02/17/mapping-wifi-signals-in-3-dimensions/)。
