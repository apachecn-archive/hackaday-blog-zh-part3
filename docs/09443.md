# 鹰眼，3D 打印陀飞轮机芯

> 原文：<https://hackaday.com/2018/05/15/hawkeye-the-3d-printed-tourbillon-movement/>

似乎建造由几十个运动部件组成的微型机械装置还不够，这些部件需要完美地啮合在一起才能工作，一些钟表制造商喜欢用陀飞轮机芯将他们的钟表设置为硬模式。陀飞轮在典型的齿轮系中增加了多个轴，试图消除重力影响造成的误差——机芯本质上是在万向节上旋转，同时滴答作响。

感觉陀飞轮太酷了，不能锁在专为超级富豪设计的钟表里。[Alduinien]同意这一观点，并通过这款 3D 打印陀飞轮将其机制民主化。被称为“鹰眼”的[Alduinien]陀飞轮是 3D 打印的杰作。该机械装置由 70 多个零件组成，看起来令人着迷，几乎像一个三轴机械陀螺仪。

陀飞轮设计为由 3D 打印的卡簧或小型电机驱动。[Alduinien]的 Thingiverse 页面主要用于演示，仍然只有组装好的机械装置的文件，但他承诺很快会发布各个部件的文件。业余钟表师，预热你的 3D 打印机。

当然，陀飞轮对这些页面并不陌生。我们已经对手表用的陀飞轮做了深入的研究，我们之前甚至向 T2 展示了一款 3D 打印的陀飞轮钟。我们喜欢这个设计的原因是它鼓励人们探索这些非凡的乐器，我们期待看到人们如何利用这个设计。对于那些想了解钟表擒纵机构更多背景知识的人来说，[Manuel]写了一篇关于[我们如何将重复运动转化为计时](https://hackaday.com/2016/11/15/the-little-mechanism-that-made-precise-time-keeping-possible/)的精彩文章。
