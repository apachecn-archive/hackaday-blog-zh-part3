# 屏幕保护程序的回归

> 原文：<https://hackaday.com/2015/12/22/the-return-of-the-screen-savers/>

如果你有幸在 1998 年拥有有线电视，你可能还记得一个名为“TechTV”的新兴频道该电视网的皇冠上的明珠是一个名为“屏保”的节目。嗯，最近 TWiT 网络重新推出了这部剧(没有旧制片人的合作或许可)，名为[“新屏保”](https://twit.tv/shows/new-screen-savers)——这几乎和我们记忆中的一模一样。该节目以科技新闻、技巧和窍门为特色，由可爱的[[]利奥·拉波特]](https://en.wikipedia.org/wiki/Leo_Laporte) 担任主要主持人。

我们假设大多数 Hackaday 读者的知识水平高于节目中的一般介绍，但我们必须承认，我们几乎总是会发现一些我们不知道的小技术提示或软件评论。如果你知道有人开始对所有技术感兴趣，这可能是他们快速开始学习的好方法——并接触各种各样的主题。

如果你想寻找一点怀旧的感觉，许多老节目的共同主持人会定期回来。像[ [【凯文·罗斯】](https://en.wikipedia.org/wiki/Kevin_Rose)，[ [帕特里克·诺顿】](https://en.wikipedia.org/wiki/Patrick_Norton)，[ [莎拉·莱恩】](http://www.sarahlane.com/about/)这样的人。你可以在 [TWiT](https://twit.tv/shows/new-screen-savers) 网站或者 [YouTube](https://www.youtube.com/channel/UCFP9Euhwi3GqbQmWS-M-0hA) 上观看该节目。休息过后，我们已经收录了第一集。
