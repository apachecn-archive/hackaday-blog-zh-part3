# 介绍任天堂吉他男孩

> 原文:[https://hackaday . com/2015/09/18/introducing-the-Nintendo-guitar-boy/](https://hackaday.com/2015/09/18/introducing-the-nintendo-guitar-boy/)

需要研究出一些邪恶的 8 位即兴重复？只有一种吉他是你应该用的，而且是一个吉他男孩！

bitfix gaming boards 的管理员[Fibbef]建造了这个作为他 2015 年 Game Boy Classic 系列的展示作品。他三个月前才开始建造，我们不得不说我们印象深刻。这是一个全功能的 Raspberry Pi 游戏机模拟器，也是一把成熟的电吉他。A 和 B 按钮兼作吉他的音量和音调旋钮，同时也是游戏机的按钮！

引擎盖下是一个 Raspberry Pi b+ running retro pie v 2.3，配有 5〃LCD 显示屏，定制的木制按钮，整个机身是手工制作的，有机玻璃外壳覆盖了整个东西。

你可以在这里跟随他的[整个构建日志，或者只是欣赏下面的视频演示:](http://bitfixgaming.boards.net/thread/190/guitar-boy-admin-build-off)
