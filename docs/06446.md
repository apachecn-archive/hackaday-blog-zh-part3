# 有人需要一点发烟硝酸吗？

> 原文：<https://hackaday.com/2017/07/07/anyone-need-a-little-fuming-nitric-acid/>

如果有一种化学物质的名字比“发烟硝酸”更酷，我们想不出来。几乎纯的硝酸是有用的东西，尤其是如果你从事制造火箭燃料和炸药的行业。但是通常可获得的低端硝酸纯度最高约为 68%，所以如果你想要好东西，你必须自己合成发烟硝酸。(所谓“好东西”，我们的意思是[对最终产品要非常小心](http://wp.me/pk3lN-15zY)。)

发烟硝酸有两种颜色——红色发烟硝酸(RFNA)，纯度约为 90%，含有一些溶解的氮氧化物，使其呈红棕色。白色发烟硝酸(WFNA)是好东西——纯度超过 99%。这两种手套都很难使用——你不会想在使用时戴上[乳胶或腈手套](https://www.youtube.com/watch?v=aBVdGGml6bU)。目前还不清楚[巴斯蒙斯特]需要 WFNA 做什么，尽管他提到蚀刻一些集成电路。合成非常简单，尽管有点危险。向硝酸钾中加入过量的硫酸，从生成的硫酸钾中蒸馏出或多或少纯的硝酸。小心的温度控制很重要，尽管冰快用完了,[巴斯蒙斯特]似乎还是获得了不错的产量。

我们这里没有太多直截了当的化学黑客，但这一个似乎足够粗糙有趣。不久前，我们确实有一个 Hackaday 奖项，是关于生产氨的 Haber 工艺的改进，奇怪的是，氨是商业硝酸生产工艺的原料。
