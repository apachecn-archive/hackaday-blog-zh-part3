# Horten Fyr 在挪威语中是 Blinkie 的意思

> 原文：<https://hackaday.com/2017/01/31/horten-fyr-is-blinkie-in-norwegian/>

我们的挪威语相当弱，所以我们在挪威霍顿的灯塔 ( [翻译](https://translate.google.com/translate?sl=auto&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=https%3A%2F%2Fhortenfyr.no%2Fblog%2Fen%2F2016%2F12%2F20%2Fhei-verden%2F&edit-text=))中为一个[大型公共艺术项目的文档费了点劲。但是我们*确实*会说 blinkies 的通用语言，这个项目已经得到了它们:3008 个 WS2812b 发光二极管环绕着灯塔顶部的窗户，并根据海浪高度和停靠在那里的渡船的远近来创建反应模式。](https://hortenfyr.no/blog/en/)

这似乎是一个不断发展的项目，随着时间的推移，会慢慢增加更多的功能。我们喜欢在渡轮上搜索 WiFi 接入点以告知它何时进港的想法，波高传感器也应该证明是有趣的数据，具有低频潮汐率以及每隔几秒钟出现的高频单一波的趋势。还有哪些输入可用？多少才算多？

这太酷了，一群有技术头脑的艺术黑客可以进入像这样的大建筑。干得好，[Jan][Rasmus]和[其他人]！
