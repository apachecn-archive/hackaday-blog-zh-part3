# 振荡气动机构不需要目的

> 原文：<https://hackaday.com/2016/07/27/oscillating-pneumatic-mechanism-doesnt-need-a-purpose/>

的确，我们在这里介绍的许多项目(以及我们自己构建的项目)都是为了实现某种目标而创建的。但是，很多时候项目本身就是目标。[Proto_G's] [自振荡气动机器](https://www.youtube.com/watch?v=534CzKUF7oM&feature=youtu.be)就是这种情况，他建造这种机器并没有特别的用途。

[Proto_G]开始这个项目只是作为一个工程练习；他只是想看看他是否能成功。我们发现这种精神非常令人钦佩——不是每个项目都需要成为你一生工作的顶点。有时候，做一件事就足够了。

像任何优秀的工程师一样，[Proto_G]认为机制和联系是简洁的。而且，甚至好莱坞也同意[振荡机制不是一个微不足道的设计](http://www.imdb.com/title/tt1054588/)。将气动技术加入其中只会增加兴趣。基本思想是当滑架到达每一侧时，打开和关闭相对的阀门，这是通过气动致动器实现的，该致动器触发“磁耦合无杆气缸”中的空气压力。

磁耦合无杆气缸是一个非常有趣的组件，它利用气缸内的磁性活塞来移动气缸外部的磁耦合托架。这使得活塞在两个方向上都有压力，因为两边都是密封的。这能通过其他方式实现吗？当然可以。但是，这种方式非常酷，你可以把它添加到你的[机制列表](https://hackaday.com/2016/02/29/2100-mechanical-mechanisms/)中，以便在未来的项目中使用。
