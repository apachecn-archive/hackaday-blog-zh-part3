# 智能吸汗带

> 原文：<https://hackaday.com/2016/02/21/smart-sweatbands/>

如果你认识任何必须监测血糖水平的人，你就会知道每次测量都必须用刺血针刺破手指来采血是很烦人的。一种包含柔性电子设备的新型吸汗带可以测量你汗液中的葡萄糖——以及钠、钾和乳酸盐——而不会有针刺般的疼痛。

加州大学伯克利分校和斯坦福大学的研究人员在本月早些时候宣布了这个乐队。它在一个柔性基板上包含五个传感器，并使用电特性来分析你的汗水。例如，汗液中的葡萄糖或乳酸盐越多，产生的电流就越多，而钠和钾的增加会产生更高的电压。一些测量结果取决于其他因素，如温度，这是波段考虑的因素。

这种可穿戴设备与蓝牙通信，可以应用于下一代智能手表和健身追踪器。还有一个一次性传感器阵列，就像一个临时纹身。

我们之前并不是没有见过手表上的[葡萄糖追踪，但那只是一个展示。希望一些](http://hackaday.com/2013/05/27/blood-glucose-monitor-data-pushed-to-smart-watch/)[开源葡萄糖监测项目](http://hackaday.com/2014/07/13/open-source-glucose-monitoring-an-the-front-lines-of-innovation/)能够采用这种或类似的技术。你可以在下面看到一段关于新款吸汗带监听音箱的视频。
