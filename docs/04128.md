# 透明火箭发动机

> 原文:[https://hackaday.com/2016/11/12/transparent-rocket-engine/](https://hackaday.com/2016/11/12/transparent-rocket-engine/)

不可否认，火箭发动机很酷。体验轰鸣声，看到火光，看着火箭直冲云霄……你还能要求什么？嗯，对于[夜鹰之光]，一个透明的火箭体就是答案。

基于【应用科学】的[之前的作品](https://www.youtube.com/watch?v=TLPWqCMb7DE)，他用一根丙烯酸棒作为火箭主体*和*作为燃料。将火焰放入丙烯酸树脂中，从身体另一端的罐子中注入氧气，瞧！火箭发动机启动良好，更好的是，燃烧的强度可以通过提供的氧气量来控制。

构造很简单，只需在丙烯酸杆上钻一个孔，在一端攻一个螺纹，然后连接一条通向氧气瓶的刹车线。他解释说，在固体火箭发动机中，燃料和氧化剂是混合在一起的。业余火箭学中的一个典型配方，被称为[火箭糖](https://en.wikipedia.org/wiki/Rocket_candy)，是糖(作为燃料)和硝酸钾(作为氧化剂)的混合物。在像这样的混合火箭发动机中，燃料和氧化剂是相互分离的，它们只在燃烧室中混合。在这种情况下，燃烧室就是燃料本身！

虽然这种火箭发动机可能效率不高，也不会产生很大的推力，但它绝对是一个非常引人注目的东西。[NightHawkInLight]在这个视频上做得很好，因为它对那些对火箭学感兴趣的人有很大的教育价值。其他在家建造火箭发动机的方式包括这个 [PVC 管版本](https://hackaday.com/2013/03/01/high-powered-rocket-engines-made-from-pvc-pipe/)，甚至是一个 [3D 打印的](https://hackaday.com/2012/10/04/3d-printing-a-rocket-engine/)。
