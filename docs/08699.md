# 物联网家务提醒系列健忘

> 原文:[https://hack aday . com/2018/03/10/IOT-Jerry-reminder-for-the-serial-健忘/](https://hackaday.com/2018/03/10/iot-chore-reminder-for-the-serially-forgetful/)

家庭幸福的秘诀往往在于关注细节，在这方面我们都可以做得更好。但是，如果纸质笔记和智能手机提醒不足以帮助你记住把工作从你的清单上划掉，也许[这个物联网任务提醒](https://www.youtube.com/watch?v=sm1-l5-z3ag)会给你保持家里平静所需的优势。

正如安德烈亚斯·斯皮斯(Andreas Spiess)指出的，他在谷歌日历中安排重复任务的初衷是好的，但这不足以让他掌控自己的家务。他发现，手机上弹出的通知太容易被其他分心的事物扫走，所以他开始建立一个现实世界的提醒。他的解决方案使用了一个亮蓝色 3D 打印盒子中的 WeMOS D1 迷你，前面有一到四个 LED 开关。每个盒子都与他的谷歌日历相关联，当一项任务到期时，它的灯就会亮起。撒在任务附近的房子里，像洗衣房或者回收附近，[Andreas]会忍不住看到提醒，只有当他按下任务按钮取消它时，提醒才会熄灭。简单而有效，也充满了其他用途的潜力。

当然，同样的事情也可以用[一个魔镜建造](http://hackaday.com/2017/12/12/old-tv-lends-case-to-retro-magic-mirror/)来完成，这几年我们已经见过很多了。但是这些设备的简单性和它们与任务的接近性是有意义的——有点像[亚马逊 Dash](http://hackaday.com/2016/07/26/bending-the-new-amazon-dash-button-to-your-will/) 的概念。我们也可以造几个这样的。
