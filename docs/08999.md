# 看这个小圆顶自动打开和关闭成一个螺旋桨

> 原文：<https://hackaday.com/2018/03/30/watch-this-tiny-dome-auto-open-and-close-into-a-propeller/>

[![](img/cae2edfa9f29597e980cc3b07d6f0637.png)](https://hackaday.com/wp-content/uploads/2018/03/spherical-folding-propeller-square.jpg) 仔细的计划和模拟是无价的，但是直接投入到原型制作中也是值得的。这是[Carl Bugeja]带着他的[球形折叠推进器](https://hackaday.io/project/93648)设计采取的方法，他已经进入了 [2018 Hackaday 奖](https://hackaday.io/prize)的开放硬件设计挑战类别。静止时，折叠螺旋桨看起来像一个附在马达顶部的小圆顶。当发动机启动时，离心力导致圆顶的两个主要部分向外展开，在那里它们充当螺旋桨叶片。当电机停止时，组件再次关闭。

[卡尔]已经做了一些初步测试，他的第一个原型连接到一个数字秤，作为一种测量推力的方法。测试单元并不大——圆顶折叠后的直径只有 1.6 厘米——但考虑到道具的尺寸很小，而且最初设计时没有进行模拟工作，他觉得结果很有希望。[卡尔]正在寻求优化实际推力，现在已经表明，他的折叠圆顶的想法如想象的那样工作。

带着一个想法直接进行物理原型制作可能是早期开发的有效方法，尤其是现在高质量的组件和技术甚至对业余爱好者来说都很容易获得。此外，它可以是非常有趣的！你可以在下面嵌入的短视频中看到和听到[Carl]的原型。
