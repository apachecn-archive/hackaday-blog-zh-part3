# 涨，涨，涨:2000 多美元的种子基金用于 Hackaday 奖的参赛作品

> 原文:[https://hackaday . com/2018/04/13/up-up-up-2000-more-seed-funding-for-hackaday-prize-entries/](https://hackaday.com/2018/04/13/up-up-up-2000-more-seed-funding-for-hackaday-prize-entries/)

启动一个项目通常意味着前期的部分投资。Hackaday 正在加大努力，为那些想要建立有意义的东西的人扫清障碍。2018 年[hack aday 奖](https://hackaday.io/prize)的种子资金很简单，进入你的开放式硬件设计，广泛分享它，这样很多人就会在项目页面上用“喜欢”来表达他们的钦佩。如果你参加了有奖竞赛，你可以从每个赞中获得一美元，以帮助启动构建阶段。如果你还没有参加，你可以鼓励和奖励那些最能激励你的项目。

今年像火箭一样开始了。我们已经超过了 4，000 美元的种子基金限额，你还有一周的时间来参与种子基金。由于第一次挑战如此令人兴奋，Hackaday 的母公司 Supplyframe 将奖金总额提高到 6000 美元。这意味着有更多的机会。[现在进入你的项目](https://hackaday.io/project/add?light&tag=2018HackadayPrize)。如果你已经这样做了，润色你的演示文稿，并在社交媒体上展示给你的朋友。拥有最多赞的条目将获得一美元，最高可达 200 美元，或者直到我们无疑再次达到新的限制。不要拖延，是时候建立一些重要的东西了！

正如我们与 2018 年大奖得主亚历克斯·威廉姆斯(Alex Williams)一起发现的那样，种子资金是一件大事。[他提到这笔钱确实帮助了他](https://hackaday.com/2017/11/20/an-interview-with-alex-williams-grand-prize-winner/)降低了早期的构建成本，来自社区的兴趣激励他在整个竞赛中继续开发。通过[评论和投票支持优秀的参赛作品](https://hackaday.io/submissions/prize2018/list)，帮助我们送出这笔额外的资金并激励下一代的决赛选手！
