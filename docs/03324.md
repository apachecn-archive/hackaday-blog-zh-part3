# 让人类远离迪士尼的机器人

> 原文：<https://hackaday.com/2016/08/24/keeping-humanity-safe-from-robots-at-disney/>

几乎每个大公司都有一个研发机构，所以当我们在 Hackaday Tip Line 上发现一个关于迪士尼研究的提示时，一点也不奇怪。这个有争议的项目最终被证明涉及[人类安全的触觉远程呈现机器人](https://www.disneyresearch.com/publication/haptic-telepresence-robot/)非常有意义，尤其是当你的业务保持地球上最快乐的地方平稳运行的时候。

迪士尼希望确保他们的电子动画是安全的，这是一个好消息，但迪士尼项目不仅仅是让游客保持健康。休息后的视频和[附带的论文](https://s3-us-west-1.amazonaws.com/disneyresearch/wp-content/uploads/20160503162533/A-Hybrid-Hydrostatic-Transmission-and-Human-Safe-Haptic-Telepresence-Robot-Paper.pdf) (PDF 链接)描述了一个远程呈现机器人，它通过一种独特的静液压传动装置耦合到操作者。致动器基于限制液压的滚动隔膜设计。在一个人类安全的系统中，这正是你想要的。

该系统是液压-气动混合设计；两个致动器，一个由水压驱动，另一个由空气驱动，在每个关节中彼此相对。充气致动器的行为就像一个质量有效的弹簧，对液压致动器进行预加载。这通过允许系统通过排空空气管线而立即断电来增加安全性。更重要的是，整个系统呈现非常低的机械阻抗，允许通过系统流体向操作者提供触觉反馈。这提供了足够的灵敏度来处理一个鸡蛋，穿针引线，甚至不受惩罚地拍打一个孩子的脸。

这里有一些机器人黑客的伟大想法，你不得不佩服这些致动器的工程设计。更多来自鼠标之家的研究，请查看[这款有点令人毛骨悚然的触摸感应智能手表](https://hackaday.com/2015/11/10/disneys-designing-a-smart-watch-that-knows-what-youre-touching/)，或者[这款空气炮触觉反馈发生器](https://hackaday.com/2013/07/29/disney-prototype-adds-haptic-feedback-to-gestural-interface/)。
