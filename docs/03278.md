# 水冷 led 灯

> 原文:[https://hackaday.com/2016/09/07/water-cooled-led-light/](https://hackaday.com/2016/09/07/water-cooled-led-light/)

[Matt]想要驱动一个 [Yuji LED 阵列](https://www.youtube.com/watch?v=kh9j5mDU_hA)。LED 需要 30 伏电压，100 瓦时会产生大量热量。他使用了为 CPU 冷却器制造的海盗船水冷系统来带走热量。零件清单包括一个麦克风鹅颈管、一个升压转换器、一个降压转换器(用于水冷却器)和定制支架(由 MDF 制成)。还有一个与 LED 阵列配套的透镜和反射镜。

这个单个 LED 可能不需要水冷。另一方面，增加一个风扇会增加发光部分的体积，鹅颈管和水冷管看起来很酷。这个项目是一个很好的提醒，如果你需要带走没有风扇的东西的热量，由于 PC 市场，独立的水冷系统现在相当便宜。

我们可能会在 LED 电源线中放置一个短路插孔来进行电流测量。在视频中，他剪断电线，对其进行首次校准。另一种方法是在盒子里装一个数字电流表。

零件列表中的链接看起来好像被移动了。LED 是 [400HS](http://store.yujiintl.com/products/anniversary-bc-series-high-cri-cob-led-400hs-100w-unit-1pcs?variant=26629621447) ，它是一个 100 瓦的阵列，有暖白色和日光白色两种版本。还有一个循环版本，目前价格约为 75 美元。如果你想要便宜得多的东西，易贝有一个类似设备的链接，价格约为 7 美元。但是，零件清单注意到色彩再现不如 Yuji 单元好(并且在下面的视频中，您可以在 13:30 的一张照片中看到色差)。

这种建造并不便宜，但肯定比我们见过的以前的水冷灯项目简单。另一方面，[一个定制系统](https://hackaday.com/2010/02/26/cooling-leds-by-heating-the-water-saves-on-electricity/)可以让你利用自己生产的热水。
