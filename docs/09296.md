亚马逊发布了 Echo Dot 儿童版，这是一款永远在线的互联网连接麦克风。根据亚马逊儿童隐私披露，亚马逊 Echo Dot 儿童版收集的数据包括，'姓名，生日，联系信息(包括电话号码和电子邮件地址)，语音，照片，视频，位置，以及某些活动和设备信息。亚马逊 Echo Dot 儿童版能够在睡前阅读有声读物，并教你的孩子生活在奥威尔式的反乌托邦圆形监狱中。它有绿色、红色和蓝色。
