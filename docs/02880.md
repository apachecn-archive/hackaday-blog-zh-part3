# 将剪草机引擎转换成用压缩空气运转

> 原文：<https://hackaday.com/2016/07/08/from-a-lawnmower-engine-to-a-compressed-air-engine/>

如今，我们最有可能接触到的蒸汽机可能是铁路机车。一台拥有巨大能量的机器，它的构造可能非常复杂，需要大量的工程技术，这是我们大多数人只能通过二手设备才能看到的。但是有一段时间蒸汽机更容易获得，在内燃机和电动机承担起自动化繁重工作的任务之前，你会在工业的各个角落找到小型固定式蒸汽机。

这些发动机更容易被黑客和制造商接受，尽管目前老式固定式发动机很少，但仍有相当多的人通过将现代汽油和柴油发动机转换为更老式的发动机来建造它们。

[Lindsay Wilson]有一台割草机发动机，几年前他改装了这台发动机，增加了一个套筒阀，以压缩空气为动力。它不是蒸汽机，因为制造一个安全合法的蒸汽锅炉是一个昂贵的过程，但尽管如此，这也是一回事。问题中的发动机是小型侧阀单缸 Suffolk Punch 割草机发动机，他从该发动机中移除并阻塞了阀装置，并添加了由来自曲轴的连杆驱动的套筒阀，并使用火花塞孔作为入口和出口。他提供了许多关于套筒阀结构的细节，这确实是一个令人惊讶的简单的安排。我们可以寻找一种比铜管更硬的金属作为导管，让它穿过。

中断下方的视频显示发动机在储存一段时间后开始运转。这是一个有效的装置，很容易就能吸入比他的压缩机所能提供的更多的空气。
