# 热视觉平视显示器不只是战斗机！

> 原文：<https://hackaday.com/2015/09/15/thermal-vision-hud-isnt-only-for-fighter-jets/>

如果你不知道的话，热视觉设备最近已经经历了大幅度的降价。市场上有一些不到 300 美元的！虽然这看起来还是很贵，但请记住，这是热感应视觉。[蒂姆]在 Seek Thermal 一上市就买了一台，最近刚刚完成了他的第一个项目——为他的车[安装了一个热能 HUD，便于夜间驾驶。](https://hackaday.io/project/7648-thermal-hud)

Seek Thermal 是一款小型热成像设备，配有微型 USB 手机附件。只需插入它，你的手机就变成了预览窗口。但要安装在你的车上，你不能把它放在窗户后面，因为大多数玻璃对热波长是不透明的，所以[蒂姆]必须发挥创意。

他设计并 3D 打印了一个磁性底座，放在他汽车的引擎盖上。但为了防止路上的碎片撞到它，他想保护镜头。所以他开始寻找热透明材料——结果发现它们非常昂贵。

工厂设备上用于隔热窗的最典型材料是 GaAs(砷化镓)或硒化锌(硒化锌)——这两种材料的价格都相当昂贵。所以[蒂姆]开始自己做一些研究。事实证明，一些塑料购物袋实际上是非常热透明的——诀窍是找到一个光学透明的。他测试了他能拿到的所有东西，最终找到了一个塑料照片套——不幸的是，它不能提供太多的保护…
