# RooBee One，一款开源的 SLA/DLP 3D 打印机

> 原文：<https://hackaday.com/2017/01/01/roobee-one-an-open-source-sladlp-3d-printer/>

[Aldric Negrier]对 3D 打印世界并不陌生。他已经制造了一些，他设计并制造了一台 [SLA/DLP 3D 打印机，命名为 RooBee One，](http://www.instructables.com/id/RooBee-One-SLA-DLP-Aluminum-Frame-3D-Printer/)分享了 Instructables 上的计划。他还发表了大量其他东西，比如 [3D 打印注射器泵架](http://hackaday.com/2015/05/29/pump-up-the-volume-with-the-3d-printed-syringe-pump-rack/)和 [3D 扫描装置和 DIY 转盘](http://hackaday.com/2015/05/23/3d-scanning-rig-and-diy-turntable/)。浏览他的整个指令库真的很值得。

这款开源 3D 打印机的灵感来自 criste lia–SLA/LCD 3D 打印机和 Vulcanus MAX 3D 打印机(由他设计)。RooBee One 拥有铝制框架和 80x60x200 mm 的可调打印区域，使用 ACER DLP 投影仪可实现高达 150x105x200mm 的构建体积。此外，还在打印机顶部安装了一个风扇，用于将有毒蒸汽抽离打印机操作人员。电子设备基于 Arduino MEGA，带有 RAMPS 1.4 屏蔽和一个 NEMA 17 步进电机。至于 Arduino Mega 固件，[Aldric]选择使用 Repetier，他通常在他的其他打印机中使用。

他使用的 SLA 树脂是来自 Fun to Do Resins 的标准混合树脂。这些树脂往往会释放有毒的悬浮颗粒，因此在打印时应特别注意通风，并在打印后进行适当的清洁。
