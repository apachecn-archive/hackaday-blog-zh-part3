# 垃圾桶数控机床制造

> 原文：<https://hackaday.com/2015/11/21/garbage-can-cnc-machine-build/>

忘记为你的下一个项目从一些高档的五金店采购零件。如果你真的想展示你的黑客能力，就像[HomoFaciens]一样，试着从[垃圾箱里翻找数控机器的零件。](http://www.homofaciens.de/technics-machines-cnc-v0-6_en_navion.htm)

好吧，我们夸大了一点——但只是一点点。除了控制电子设备，几乎所有[HomoFacien]的产品都可以在垃圾收集日的路边找到。一件废弃的平板包装家具的碎料板，一台旧打印机的电机和齿轮，以及一些钢带都是一台可维修的数控机器的框架所需要的。这台机器甚至比[HomoFacien]的早期版本更垃圾，后者有更多商店购买的零件。但是下面的视频显示了令人印象深刻的表现。

当然，对于 DIY CNC 构建的艺术水平来说，这是一个巨大的倒退。但这才是重点——展示几乎什么都不用就能完成的事情，以及想象力和毅力对于可接受的结果比昂贵的 BOM 更重要。

考虑到这一点，我们发出挑战:有人能用硬纸板和回形针制造数控机床吗？
