# 将电动工具电池的容量加倍

> 原文：<https://hackaday.com/2017/05/19/doubling-the-capacity-of-power-tool-batteries/>

YouTube 用户[Vuaeco]提出了一个新颖的想法，[将电动工具电池组](https://www.youtube.com/watch?v=m1JU4IQ_ayw)结合起来，使容量翻倍。

从两个 2.0Ah 的小型电池组开始，[Vuaeco]想要一个更大的 4.0Ah [重建的钻头电池组](https://hackaday.io/project/21052-18v-li-ion-drill-battery-pack-rebuild)。这些电池组大小不同，所以这不仅仅是在空槽中添加更多电池的问题，相反，他继续向我们展示如何使用一些薄镍带并联电池。一旦完成，他修改电池外壳，使其适合另一堆电池。他通过用长螺丝将顶部和底部固定在一起，并用绝缘胶带将暴露在外的电池端子绝缘。最终产品不像真正的电池组那样美观，但看起来足够好。

有几件事情我们可能会做得不同，例如在绝缘材料周围提供一些硬塑料，这样如果电池在尴尬的位置受到撞击，它仍然有一个硬壳保护它。此外，我们可能会采取相反的方法，将电池完全耗尽，避免不必要的风险，而不是像视频中建议的那样将电池完全充电。如果你试试这个，给它一个 [3D 打印的表壳](http://hackaday.com/2016/03/26/handy-power-supply-with-3d-printed-case/)怎么样？
