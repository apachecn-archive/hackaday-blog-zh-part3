# 将基础科学实验武器化

> 原文：<https://hackaday.com/2016/11/22/weaponizing-elementary-science-experiments/>

[austiwawa]正在摆弄一个简单的线性马达，人们把它作为友好的小型科学实验。在一组磁铁中间有一节 AA 电池。当你把它放入弹簧中时，它会在弹簧中滑动，直到你用完弹簧或电池中的魔法小精灵。

当然，自然的问题出现了，“我如何让它跑得快！？喜欢快！”在制造了一点爆炸和嗖嗖的噪音之后(就像任何优秀的黑客会做的那样)，他静下心来问了一个更具体的问题。如果我把线圈做成气枪的枪管，然后把电池射出去…会不会快一点？

所以，他造了一个空气炮。这需要一些智慧和胶带，但他设法用一个铜线圈内衬桶。之后，他建立了一个实验装置，因为制造危险的东西只有在科学的前提下才是可行的。这就是懂事的大人和小孩的区别。

他通过加农炮打了三发“死”弹，得到了一个基线结果。这些死穴是通过将磁铁放置在不正确的极性上以放弃推进运动的特性而造成的。然后他射穿了三个活的。它跑得更快了！整洁！
