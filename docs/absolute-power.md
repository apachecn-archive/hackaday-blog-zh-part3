# 绝对权力

> 原文：<https://hackaday.com/2016/12/08/absolute-power/>

我们最近在一些中国交易网站上注意到一系列看起来非常酷的电源模块。根据型号，它们提供带计量的数字控制电压。你需要提供至少超过最大期望输出电压 1 伏左右的电压。下面可以看到一个来自【iforce2d】的视频。视频中的模块额定为 5A，最大 50V，但也有其他尺寸可用。对于那些对图形和数字感兴趣的人，[lgyte]对这些模块做了大量的[表征。](http://lygte-info.dk/review/Power%20DPS5015%20UK.html)

曾几何时，从很远的地方进口商品在某种程度上是一门艺术。寻找供应商、计算付款、运输和海关意味着你必须知道自己在做什么。今天，你只需在网上冲浪，找到你想要的东西，用贝宝支付，东西就会从世界各地出现在你的门口。

不过，有一个问题。我们看到很多来自中国的很酷的东西，其中一些很棒，尤其是价格。不过，坦白地说，其中一些是垃圾。很难区分哪个是哪个。更重要的是，即使理论上你可以退货，但通常运费会让这变得不切实际。因此，当你得到一个哑弹，你很可能只是吃了它，并把它记在经验上。那么问题来了:这些电源模块有多好(或多坏)？

视频是一个很好的方式来看看你得到了什么。当然，您可以更好地了解模块的大小。但是，电学上是什么样子的呢？[lgyte]的测试非常全面，不仅包括关键参数的图表，还包括电路板的红外照片，以识别热点。下面还有一个来自[neutronstorm]的类似模块的更长的视频。

由于价格低，这些用品的特点看起来不错。您可以设定电源在超过电压、电流或功率限制时关闭。与 DP 系列相比，DPS 系列模块的用户界面略有改进。

当然，我们在之前已经见过[数字电源(包括一些](https://hackaday.com/2013/12/11/digitally-controlled-dual-power-supply/)[包装精美的](https://hackaday.com/2011/06/06/diy-digital-bench-power-supply/))。然而，这些都很便宜，看起来很好，而且使用起来非常简单。变压器、桥式整流器和滤波电容在其中一个的输入端，这样就设置好了。对于许多应用而言，旧的 18.5V 笔记本电脑电源模块是为其供电的绝佳方式。
