# 这个机器人横着生活

> 原文:[https://hack aday . com/2016/12/04/this-robot-through-life-sideways/](https://hackaday.com/2016/12/04/this-robot-goes-through-life-sideways/)

我们人类走路很滑稽。在臀部向前转动一条腿会产生一个偏移，使我们失去平衡。我们每走一步都要补偿这一点。许多机器人通过采取小的、计算好的步骤来处理这个问题。进入 [NABiRoS，非拟人化的两足机器人系统](https://www.youtube.com/watch?v=Y5UoQsHJskw)(链接到下面的视频)。“非拟人化”意味着它不像人一样走路，而“两足动物”意味着它仍然用两条腿走路。不同的是它是横着走的。

![NABiRoS leg configuration](../Images/0246f1b87040299102912349353a1deb.png)

NABiRoS leg configuration (from [video](https://www.youtube.com/watch?v=Y5UoQsHJskw))

下面是来自加州大学洛杉矶分校的机器人&机械实验室的人们是如何做到的。想象你将双腿旋转 90 度，使它们面向相反的方向。然后你将上半身旋转 90 度，面向你的一条腿。现在你可以移动你的腿向你面对的方向行走，每次你走一步都不会再向旁边倾斜。关节也更简单，因为每个膝关节和髋关节只需要一个自由度。脚踝和脚由一个柔顺的或者有弹性的关节组成，就像你看到的许多假腿一样[。正如你在下面的视频中看到的，除了走路，它们还能做一些令人惊讶的活跃的事情，比如上蹿下跳，以及我们只能称之为跳绳的动作。其实结果有时候很人性。](http://hackaday.com/2016/05/25/hackaday-prize-entry-robotic-prosthetic-leg-is-open-source-and-3d-printable/)

到目前为止，他们已经实现了每小时 0.46 英里(0.75 公里/小时)的行走速度。我们还不确定他们计划如何改变，但他们正在努力，希望我们能在他们的下一个版本中看到。在视频的最后，他们简要介绍了他们的后续工作，看起来非常有趣。这种不倾斜也意味着他们将避免很多我们在过去的 DARPA 比赛中看到的[摔倒。](http://hackaday.com/2015/08/23/hilarious-darpa-robots-falling-video/)

对于一个更古怪的罗梅拉机器人，看看这个[走路时漂浮的机器人](http://hackaday.com/2016/11/23/floating-walking-robot/)！
