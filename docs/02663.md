# Hackaday 奖参赛作品:BLE 灯塔图书馆

> 原文:[https://hack aday . com/2016/06/21/hack aday-prize-entry-ble-beacon-library/](https://hackaday.com/2016/06/21/hackaday-prize-entry-ble-beacon-library/)

虽然使用 nRF24L01+模块伪造 BLE 广告信标并不新鲜，但现在[Pranav Gulati]已经为此编写了一些库代码和几个示例，这已经变得简单多了。

[Pranav]的工作是基于[Dmitry Grinberg]史诗般的[钻头撞击 BLE](http://hackaday.com/2013/09/21/sending-data-over-bluetooth-low-energy-with-a-cheap-nrf24l01-module/)的研究，我们早在 2013 年就报道过。虽然 BLE 的广告频道能够发送的数据量有限，但对于需要长时间偶尔发送少量数据的电池供电设备来说，1 美元的 nRF24 模块和省电的微控制器将是很好的选择。

我们不是 100%确定[Pranav]会把这个项目带到哪里。老实说，这个库看起来已经可以使用了。如果你已经推迟制作你自己的 [BLE 使能的鸟群](http://hackaday.com/2014/08/15/bluetooth-low-energy-beacons-in-a-flock-of-birds/)，或者即使你只是想[乱搞协议](http://hackaday.com/2014/01/22/sniffing-and-decoding-bluetooth-le-advertising-packets-and-nrf24l01-comms-for-under-30/)，你的生活已经变得容易多了。
