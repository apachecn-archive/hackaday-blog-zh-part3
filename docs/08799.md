# 无接触商店门过度设计到一个幸福的水平

> 原文：<https://hackaday.com/2018/03/11/touchless-shop-doors-over-engineered-to-a-blissful-level/>

当[约翰·桑德斯]想要为他的商店安装一扇自动门，而不是满足于一个商业单元时，他设计并制造了一个接近感应开启器来方便他的通过。听起来很简单，对吧？

对我们来说幸运的是，由于多台 Tormach 机床以及知道如何使用它们的人员，Saunders Machine Works 没有折中的办法。下面的视频给我们留下了深刻的印象；第一部分主要是加工开瓶器的许多零件，所以如果你对控制电子和编程更感兴趣，请跳到 8:33。

开启器使用飞行时间距离传感器和 Arduino 来检测有人接近，并使用气压缸来分离塑料带帘。[John]承认这一点不仅仅是一点点范围蠕变，当您可以轻松获得随意创建专用部件所需的工具时，这是可以理解的。

不过，最后，它对每个人都很好，除了商店里的狗(Judd)，而且它看起来确实是一个有趣的构建。[约翰]对混合加工和电子的热情是有感染力的；检查[他的自动碗进料器](https://hackaday.com/2017/11/11/automating-a-bowl-feeder-with-arduino/)用于装配线。
