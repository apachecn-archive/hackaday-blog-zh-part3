# 乔治亚理工学院通过硅泵送水用于芯片冷却

> 原文：<https://hackaday.com/2015/10/05/georgia-tech-pumps-water-through-silicon-for-chip-cooling/>

阻碍电子设备运行速度的因素之一是热量。这就是为什么狂热分子甚至使用液氮来冷却 CPU 芯片，以最大限度地发挥其超频潜力。乔治亚理工学院的研究人员一直致力于在商用硅芯片(确切地说是 Altera FPGA)的背面直接切割流体通道。这些微小的通道大约 100 微米，用另一层硅重新密封。水被泵入通道以有效地冷却设备。

类似的空气冷却设备可以在 60 摄氏度左右工作。随着水冷却通道切入模具，20 度的水以 147 毫升/分钟的速度泵出，研究人员保持芯片在低于 24 摄氏度的温度下工作。

尽管水冷并不是一个新想法，但传统的水冷在封装外部使用冷板，因此它们不如佐治亚理工学院的方法有效。通过将水直接放在热源处，封装可以做得更小，甚至可以堆叠多个芯片而不会引起热管理问题。

更高的速度只是故事的一部分。较冷的电子设备具有更少的电流泄漏和更长的设备寿命。尽管研究人员使用的是 FPGA，但这项技术适用于任何需要散热的集成电路。这项研究发表在 2015 年 IEEE 定制集成电路大会上的一篇论文中。

尽管移除引线框的后部可能是个问题，但封装 IC 并不难。我们不得不怀疑一个有事业心的黑客是否会想出在车库里复制这一壮举的方法，或者至少是类似的方法。毕竟，我们已经见过[水冷树莓 Pi](http://hackaday.com/2013/07/20/water-cooled-raspberry-pi/) 和 [LN2 冷却 Arduinos](http://hackaday.com/2013/08/18/liquid-nitrogen-finally-makes-an-arduino-project-cool/) 。
