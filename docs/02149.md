# 环氧花岗岩精密 CNC

> 原文：<https://hackaday.com/2016/05/01/precision-cnc-with-epoxy-granite/>

环氧花岗岩是一种被忽视的材料，当谈到[制作家庭数控构建](https://www.youtube.com/watch?v=-0Cn0WRXPKs)时。就时间和金钱而言，当你把所有的设备都加进去时，它会比一个铝铸造装置更便宜。[环氧花岗岩](https://en.wikipedia.org/wiki/Epoxy_granite)比铸铁有更好的机械性能，更高的尺寸稳定性，更好的振动阻尼，如果处理得当，看起来棒极了。此外，你可以在设计中铸造精密的表面和螺纹孔，这非常酷。

在[Jr gen he gner]的这两个视频中，我们看到他和一个朋友制作了一套精密数控机床。它的建造风格类似于我们在中展示的[其他好的建造风格。这种制造方法需要比龙门磨更大的占地面积，而且不能造得那么大。然而，它解决了许多机械问题，并与机架设计相协调，同时不像箱式或膝磨机那样难以获得正确的结果。](http://hackaday.com/2016/02/07/a-home-cnc-built-by-someone-who-knows-their-stuff/)

铸造后，他们加工嵌入花岗岩中的材料来安装导轨。方法是直线轴承和滚珠丝杠。很贵，但正如录像显示的，非常准确。机器的其余部分被组装和调整。然后将它安装在自制的 80/20 外壳中。我们非常喜欢集成在机器前罩中的 LCD 面板。他们真的全力以赴用 CNC 控制面板。看起来他们可以做任何事情，从轻推轴到监测和控制主轴的水冷。

似乎所有的精密工作都得到了很好的利用，因为在视频二的结尾有一些镜头是这台机器上的一个漂亮的数控时钟。休息后的视频。
