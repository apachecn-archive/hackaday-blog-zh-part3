# MacGyvering 测试引线夹

> 原文:[https://hack aday . com/2017/02/23/macgyvering-test-lead-clips/](https://hackaday.com/2017/02/23/macgyvering-test-lead-clips/)

好吧，各位制造商，当你穿过外围电源跳线、回形针、弹簧和一些 3D 打印外壳时，你会得到什么？ [DIY 测试引线夹](https://www.youtube.com/watch?v=cGHGMCx4kXY&t=15s)。

测试剪辑很容易获得，但这有什么意思呢？[notionSuday]首先从跳线上取下导线连接器，将它们焊接到回形针的剥离长度上，将连接器上的标签弯曲作为止动器，并将弹簧滑到顶部。四次快速打印外壳后，回形针组件正好放在里面，尖端弯曲并被夹住，作为临时的夹子。一旦套在他们的万用表探针上，他们就能像魔法一样工作。

在视频中，你还可以瞥见一个使用 3D 打印机和一些回形针制作的[电池座](https://www.youtube.com/watch?v=XJE4G_F_V6U)【notion Sunday】。请注意——如果你的工作区没有回形针，你可能会错过重要的黑客材料。
