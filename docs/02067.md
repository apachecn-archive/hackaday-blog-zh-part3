# 一个星体跟踪望远镜架

> 原文：<https://hackaday.com/2016/04/14/a-star-tracking-telescope-mount/>

[克里斯]最近他得到了一个旧望远镜。虽然这种带有高度-方位支架的小型折射镜足以观察我们太阳系中的大型物体，但高端望远镜可能要酷得多。大型反射望远镜可以跟踪夜空几个小时，通常带有一个计算机界面和一个 go 按钮。结合开源的星图 Stellarium，你可以在你的后院拥有一个完整的天文台。

为了获得 2016 年黑客日奖，克里斯正在升级他的旧望远镜。用一个树莓派，一些 3D 打印适配器，和一个新的望远镜支架[来创建一个自制的望远镜计算机](https://hackaday.io/project/10181-starpi)。

alt-az 坐骑真的不是天文工作的合适工具。地球在倾斜的轴上旋转，如果你想在夜空中静止不动，它必须在两个轴上转动。赤道座更适合天球。现在，[克里斯]正在观察德国的赤道架，这是一架仅使用时钟驱动马达就能在夜空中追踪单个恒星的望远镜。

为了给这个望远镜一个大脑，他将使用树莓 Pi，GPS，磁力计，表面上是一个实时时钟，以确保建筑知道星星在哪里。在那之后，很简单的事情就是通过电脑对准望远镜，用一个树莓派相机用一个非常非常小的图像传感器窥视天空。

虽然任何一个有三四百美元的人都可以简单地*购买*一个具有类似功能的望远镜，但这对【克里斯】或业余天文学来说真的不是重点。业余天文学家建造自己的镜子、透镜和支架已经有很长很长的历史了。[Chris]正在延续这一悠久的传统，并在此过程中为 2016 年 Hackaday 奖打造一个伟大的参赛作品
