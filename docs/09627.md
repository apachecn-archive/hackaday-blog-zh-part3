# 机器人随着新 YouTube 的节奏跳舞

> 原文:[https://hack aday . com/2018/06/04/robot-dances-to-the-beat-of-new-YouTube-subs/](https://hackaday.com/2018/06/04/robot-dances-to-the-beat-of-new-youtube-subs/)

当然，你可以建立某种数字计数器来跟踪 YouTube 的新用户。但是越来越多的数字显示真的传达了这样一个事件的重要性吗？当然不是。你需要的是认可这一成就的东西，并和你一起庆祝它。类似于交互式 YouTube 订阅机器人 Subby。

每当[布莱恩·布罗金]有了新订户，Subby 的小电视屏幕就会亮起来，他或者跳舞，敬礼，或者在他令人印象深刻的动作范围内做其他动作。[brian]编写了一个 Visual Basic 应用程序，该应用程序在他的频道页面中搜索订户数，并每一千毫秒通过串行将其发送到 Nano 的 COM 端口。[brian]通过 Dropbox 获得了 VB 应用程序和 IO 上所有可用的 STL 文件。太空步走过休息时间看 Subby 下来。

我们喜欢 Subby 太专注于庆祝每一个新订户，而不关心总数本身。也许他可以被编程为每当频道达到一个里程碑时做一些额外的特殊动作。
