# 测量 HVAC 系统中的气流

> 原文:[https://hack aday . com/2017/11/04/measuring-air flow-in-an-HVAC-system/](https://hackaday.com/2017/11/04/measuring-airflow-in-an-hvac-system/)

[numbian]来信分享了他的实验，用[测量 HVAC 系统](https://youtu.be/BNTSSxzm1GM)中的气流。他的第一个视频是关于超声波传感器的使用。他发现了一份有趣的白皮书，描述了用[单通道声波时差流量计](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC4481905/)测量气流。问题是，他能用现成的组件得到同样的效果吗？

[努比亚]用一对典型的超声波距离传感器制造了一个装备。他把两个传感器从印刷电路板的前面拆下来。然后，传感器在电线上延伸，“发送”胶囊一起指向“接收”胶囊。[努比亚]把传感器安装在 PVC 管里，用风扇把空气吹进去。

在他的第二个视频中，[Nubmian]建立了一个由连接到气流传感器的万用表组成的测试装置。在 AD623ARZ 放大器的帮助下，Arduino 很快取代了万用表。那个装置的电线需要一些屏蔽。连接到地面的铝箔起了作用。

更多暖通空调项目，请查看这个[风车式流量计](https://hackaday.com/2017/08/25/custom-cut-pinwheel-makes-a-useful-hvac-duct-flow-meter/)和【亚当】在[暖通空调技术](https://hackaday.com/2016/03/28/hvac-techs-hackers-who-make-house-calls/)上的文章，这些黑客上门服务。
