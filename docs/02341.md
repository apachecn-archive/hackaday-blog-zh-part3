# 一个更好的捕鼠器，至少对老鼠来说是这样

> 原文：<https://hackaday.com/2016/05/20/a-better-mousetrap-at-least-for-the-mouse/>

在制作这种非致命汽水瓶捕鼠器的过程中，没有老鼠受到伤害。

根据你对这些小动物的看法，这可能是好事也可能是坏事。我们不否认老鼠很可爱，但是当它们破坏了你花了一整个夏天的精力来种植、收获和保存的食物时，可爱就不值钱了。

我们对解决啮齿动物问题的偏好要么是生物修复，要么是 T2 快速颈椎脱臼，但是如果你是那种抓了就放的类型，这个陷阱很适合你。它只是一个装在金属丝枢轴上的 2 升汽水瓶，安装在一个废木架上；当这个讨厌的小动物不明智地进入瓶颈时，它的重量会把瓶子掀翻并堵住出口。释放就像从框架上取下瓶子，让叮当先生自由摆动一样简单。在哪里释放以及你将持续捕捉同一个鼠标多少次的问题留给读者作为练习。

记住——只有定期检查，活捉陷阱才是人道的。为此，也许可以在这个陷阱中加入类似于 [das Katzetelegraf](http://hackaday.com/2015/08/23/hackaday-prize-entry-feral-cats-phone-home-with-das-katzetelegraf/) 的东西。
