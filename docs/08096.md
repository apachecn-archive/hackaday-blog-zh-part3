# 一棵干净的圣诞树意味着更多的灯！

> 原文：<https://hackaday.com/2017/12/30/a-clear-christmas-tree-means-more-lights/>

尽管节日期间熙熙攘攘，人们仍然会想方设法为自己的激情腾出时间。在圣诞节前夕，[Edwin Mol]和几名员工为自己打造了一棵 [LED 圣诞树](https://www.youtube.com/watch?v=fAYeSko__yo)，为任何节日装饰增添了一抹制造者的色彩。

在走得更远之前，他们用硬纸板做了一个树的模型。这是一个容易跳过的步骤，但它可以省去以后的麻烦！一旦对原型感到满意，他们就打印出设计模板，并使用电动工具切割大块的透明丙烯酸树脂——你不需要激光切割机来生产好东西——并在塑料上钻几十个孔来安装 led 和布线。

树莓 Pi 3 和 Arduino Uno 与一些非常聪明的圣诞树联合在一起。MAX6968 5.5V 恒流 LED 驱动芯片和 MOFSETs 完善了控制电路。在构建过程中，中央 LED 柱提出了一个重大挑战——您多久构建一次定制夹具来焊接 LED？完成后，是时候来一个老式的组装蒙太奇了！最终产品可以在彩虹般的颜色中循环播放几种不同的灯光动画，非常适合节日建筑。
