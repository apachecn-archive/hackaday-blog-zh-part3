# 廉价校准隐形激光

> 原文:[https://hack aday . com/2016/03/10/aligning-invisible-lasers-on-the-cheap/](https://hackaday.com/2016/03/10/aligning-invisible-lasers-on-the-cheap/)

排列自制(或改装)激光切割机的光束并非没有挑战。例如，我如何用我剩下的眼睛对准一束看不见的光束，它有足够的能量烧穿它路径上的一些物体？我们中的一些人将经历额外的麻烦和费用，混合在一个可视的引导，跟踪 CO2 激光的路径。不过，对于那些吝啬鬼来说，[斯蒂芬]已经用一种便宜的技术把我们覆盖住了，这种技术只需要几条胶带就能把你砍倒。

斯蒂芬的技巧简单而优雅。他在每面镜子上都贴上胶带，发射激光，留下一个烧伤的痕迹，从激光击中的最后一面镜子开始，一直到第一面镜子。每面镜子上都有一个烧痕，一个穿过一张胶合板制成的导轨，[斯蒂芬]很清楚光束的自然方向。然后，他用红点激光对准烧伤痕迹，然后用安全的可见光对准镜子。唷！现在，这比我们每次想调整镜子的校准时反复发射光束并更换镜子上的胶带要容易得多。

有了这些烧焦的胶带，这个过程会变得有点臭。尽管如此，我们还是把这个归档了，以后当我们开始有那种发痒、火辣辣的感觉，让我们开始建造自己的[自制激光切割机](http://hackaday.com/2015/06/04/lasersaur-that-cuts-12-plywood-and-10mm-acrylic-pew-pew/)。
