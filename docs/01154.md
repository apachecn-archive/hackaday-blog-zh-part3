# 导电硅树脂制造柔性电路

> 原文：<https://hackaday.com/2016/01/07/conductive-silicone-makes-flexible-circuits/>

柔性电路和可穿戴设备最近似乎风靡一时。我们有导电涂料，胶水，甚至线。那么导电硅胶怎么样呢？嗯，事实证明——制作起来并不困难。

[Andrew Quitmeyer]研究柔性电路已经有一段时间了，最近偶然发现了一项过期的柔性点火电缆专利，该专利使用碳纤维混合导电硅树脂。他开始摆弄它，并发现通过将碳纤维碎片溶解在外用酒精中，让其干燥，然后将其混合到双组分硅树脂中，可以获得很好的导电性。事实上，在 40-150 欧姆的范围内，这确实令人印象深刻！
