# 研究人员推出了一种新的机器人移动方式

> 原文：<https://hackaday.com/2017/09/30/researchers-squeeze-out-a-new-breed-of-robot-locomotion/>

研究人员一直在研究各种奇怪的机器人运动形式；当然，我们都看过了，不是吗？不是这样的！幸运的是，[斯坦福大学的研究人员]正在向我们展示一种新的方法，让机器人从 A 点挤出到 B 点。

这种机器人的特殊运动形成机制包括从里到外展开。从一个固定的基座上，一个卷轴将机器人未充气的聚乙烯身体缓存起来，通过加压展开。研究人员可以通过改变外部控制室中的充气量来实现完整的 3D 转弯。更重要的是，它们可以将末端效应器甚至有效载荷放置在生长末端的尖端，并通过电缆固定它们的位置。

正如我们可以想象的那样，任何可以挤到 72 米长的机器人都可以有几十种应用，斯坦福大学的人们已经探索了这个领域的许多角落和缝隙。一路上，他们在空中部署复杂的天线形状，运送小型有效载荷，灭火，并挤过各种不吸引人的地方，如捕蝇草甚至一床钉子。我们在休息时间下面放了一个视频，但看看 Ars Technica 的完整视频套件，就能感受到他们在新作品中赋予的各种应用。

仿生学往往会让我们不假思索地喊出“壁虎脚”或“蛇形机器人”。但是想到的这些运动形式似乎都来自动物王国。这种软体机器人的一个关键因素是，它的固定底座和藤蔓般的运动都源于植物界。这证明了这一领域可能是多么未经探索，研究人员和机器人将在未来几年继续开发人工“出行”的新方法。
