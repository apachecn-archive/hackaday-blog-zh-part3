# 数控时钟本身会显示时间

> 原文：<https://hackaday.com/2016/05/28/cnc-clock-mills-itself-displays-the-time/>

[Christian]编写并销售一些 CAM/CNC 控制器软件。我们有点坚持开源，而这个软件似乎不是，所以“嗯”。但是我们*喜欢的*是附带的复活节彩蛋:磨出时钟底座的路径，然后[移动钢珠轴承显示时间的代码](http://www.estlcam.com/clock.php)。

当然，我们希望看到更多的信息(更多，更多，MOAR！)但是看起来很容易再造。例如，我们可以用弹珠和真空系统来重新设计它。滚珠轴承的座甚至不需要铣出球体。你可以用钻床完成这个部分。谁会为他们的 3D 打印机重建这个呢？你只需要确保机器足够快，能在一分钟内把球移动过来。

现在，我们通常以由数控机床制造*的[时钟为特色，所以令人耳目一新的是，这里的数控机床*是*时钟。](http://hackaday.com/2014/06/10/mechanical-clock-designed-for-a-cnc-router-gets-new-life-using-a-3d-printer/)*

谢谢[Kevin]的提示！
