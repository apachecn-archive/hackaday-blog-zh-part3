# 超薄显示屏让您的皮肤成为您的屏幕

> 原文:[https://hack aday . com/2016/04/19/超薄显示器让你的皮肤变成你的屏幕/](https://hackaday.com/2016/04/19/super-thin-display-makes-your-skin-your-screen/)

日本的研究人员创造了一种 3 微米的显示器，它看起来像塑料包装，可以将你皮肤的任何部分变成电子显示器。这个想法并不新鲜，但这种显示器比以前的设备更薄，更耐用。它还能持续更长时间(几天)并增加亮度。

该显示器使用聚合物发光二极管形成七段数字，因此你不会很快将网飞流传输到你的手背。然而，该团队希望建立更先进的显示器，有朝一日可以取代智能手表或智能手机屏幕。

聚合物 led，顾名思义，使用大分子(聚合物)作为半导体。它们于 1990 年推出，对于像这样的应用，它们有许多有用的特性，如灵活性。

如果你感兴趣，你可以[制作自己的聚合物发光二极管](http://hackaday.com/2010/11/29/diy-oleds/)，[这是我们不止一次提到过的](http://hackaday.com/2014/03/02/making-oleds-in-the-kitchen-sink/)。你可以在下面看到一个超薄显示器的视频。
