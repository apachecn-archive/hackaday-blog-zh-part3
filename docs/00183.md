# 电缆和绞盘成为一个可怕的模拟器

> 原文:[https://hack aday . com/2015/09/24/cables-and-wombers-become-an-awesome-simulator/](https://hackaday.com/2015/09/24/cables-and-winches-become-an-awesome-simulator/)

直接来自马克斯·普朗克生物控制论研究所，并在今年的驾驶模拟会议和展览上展示的[是我们见过的最酷的模拟平台](https://www.youtube.com/watch?v=cJCsomGwdk0)。这是一个球形(或二十面体)防滚架，通过电缆连接到建筑物的角落。有了正确的运动学和一些非常重型的硬件，这个模拟平台有三个平移度，三个旋转度，成千上万的人想用这个巨大的机器人驾驶虚拟汽车或驾驶虚拟飞机。

电缆机器人模拟器使用连接到一个巨大房间角落的电动绞车来推动一个加速度为 1.5g 的平台。该平台可以前后、上下和前后移动，模拟赛车手在赛道上行驶的感觉，或者战斗机飞行员在莫哈韦峡谷中飞行的感觉。真正的虚拟现实系统只需要一个 Oculus Rift，该团队已经用驾驶和飞行模拟程序进行了测试

同一个研究小组的早期项目在 2013 年完成了类似的壮举，但这个全运动机器人模拟器不是由基于电缆的机器人制成的。CyberMotion 模拟器使用了一个机械臂，机械臂末端连接着一个类似驾驶舱的东西。在驾驶舱内，立体投影仪显示广角视图，就像虚拟现实显示器一样。在模拟不同环境的能力和能力方面，CyberMotion 模拟器可能更先进一点；电缆机器人模拟器不能旋转超过大约六十度，而 CyberMotion 模拟器可以让你颠倒过来。

电缆机器人模拟器占据了一个非常大的空间，并且需要一些严肃的工程设计——电缆很大，绞车非常强大。然而，这些事实并不排除这项技术在未来的应用，希望这种技术能进入一些更大的商场。

我们经常看到概念一波一波地出现。本周早些时候，我们展示了用于在仓库中移动托盘的缆索机器人。
