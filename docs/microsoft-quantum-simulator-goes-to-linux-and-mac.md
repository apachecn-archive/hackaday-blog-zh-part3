# 微软量子模拟器走向 Linux 和 Mac

> 原文：<https://hackaday.com/2018/02/28/microsoft-quantum-simulator-goes-to-linux-and-mac/>

每个人似乎都在为争夺量子计算机之王而摩拳擦掌。最新的一波攻击是微软的，他们宣布他们的[量子模拟器](https://www.microsoft.com/en-us/quantum/development-kit)现在可以在 macOS 和 Linux 上运行，相关的库和示例现在完全开源。他们制作了一个关于新版本的视频，你可以在下面看到。

微软还声称，他们的模拟器比以前快得多，尤其是在大型模拟上。当然，真正大型的模拟会遇到内存问题，而不是速度问题。你可以在本地或 Azure 云上运行他们的模拟器。

微软推出了 Q#语言，将传统语言的思想与量子环境融合在一起。还有对 Python 的实验性支持。当然，目前所有这些模拟器都不是非常有用，除了学习这些新计算机一旦实用后将如何工作。

如果你想了解 quantum，你可以查看我们的教程,它使用了基于浏览器的模拟器，或者你可以试试 IBM real hardware。微软正在研究一些非常不同的硬件实现，但如果你想了解更多关于真正的量子硬件的[，我们已经在过去报道过了](https://hackaday.com/2018/01/22/quantum-computing-hardware-teardown/)。
