# JOLED——3D 翻转点显示器

> 原文：<https://hackaday.com/2016/12/02/joled-a-3d-flip-dot-display/>

翻转点显示器是如此的棒，以至于它们正在卷土重来。但是，当你有一个疯狂的三维翻转点显示，点漂浮在半空中时，牛逼就不算什么了。苏塞克斯大学和布里斯托尔大学的研究人员建造了他们称之为 [JOLED](http://www.sussex.ac.uk/newsandevents/pressrelease/id/37420) 的浮动像素阵列，可以通过超声波驻波和静电场的组合来控制。这些“体素”可以通过超声波在空间中单独移动，也可以通过静电场单独翻转或旋转任何角度。

整件事的关键是他们称之为 Janus 对象的东西——因此 JOLED。Janus 粒子在相反的两面有不同的特征或化学性质。每个体素的一部分都有少量二氧化钛纳米粉末的斑点。这给了它一个双极电荷，使它能够响应可变的静电场，从而能够轴向旋转。然后，可以用对比色(红、蓝、黑)覆盖每个白色体素的一半，以实现翻转点效果。每个体素的直径似乎只有几毫米。超声波致动器似乎是每个黑客的零件箱中都有的常规压电发射器。相对两侧的透明玻璃板施加可变静电场。

虽然这仍然是实验性的，并且仅限于研究实验室，但未来的应用将会很有趣。这就像打破电子墨水显示器的平面玻璃界限，给他们一个三维空间。休息后两分钟的短视频很好地解释了正在发生的事情，所以来看看吧。现在，谁想第一个造出 JOLED 钟？
