# 苏联便携式示波器拆卸

> 原文：<https://hackaday.com/2016/11/25/soviet-portable-scopemeter-teardown/>

浏览 YouTube 可能被证明是你在黑客日之外最大的生产时间破坏者，一旦你开始看各种各样的林肯郡水管工或年轻的乌克兰人做危险的特技表演，很容易失去一个小时而没有什么可展示的。有太多的东西可以转移我们的注意力，我们中的任何一个人能够创造出任何东西都是一个奇迹！

因此，为了确保你今天再损失 15 分钟，我们将为你带来[Jesper Broe]的演示和拆卸他最新的示波器。当我们告诉你这是一个带宽为 10MHz 的单通道模型时，这可能看起来没有什么希望，但不要放弃。[这是一个里梅达·C1-112](http://www.eevblog.com/forum/testgear/c1-112a-oscilloscopemultimeter-restoration/)，一个在立陶宛还是苏联的一部分时制造的便携式仪器，它的聚会作品是它包含一个数字万用表，使用示波器 CRT 进行矢量显示。

我们看到这个小巧的设备被拆开包装，然后作为示波器进行测试。它在 10MHz 以上提供了有用的结果，但它明显失去了振幅，最终随着频率的增加，它很难触发。有趣的是，所有的控制都与你习惯的方向相反，逆时针旋转增加而不是减少。然后我们看到了万用表功能，它与现代的数字多用表进行了比较，发现在近三十年后仍然非常准确。

然后示波器的盖子被移开，我们看到一些产生数字显示的逻辑板。一系列苏联 K155 系列逻辑 ic 是它的核心，在视频的结尾，我们看到了俄语的一段时间回顾，其中一瞥他们产生的波形，以矢量绘制数字。

看看休息时间下面的视频，我们相信你会同意这是一种我们许多人今天仍然会发现有用的工具。
