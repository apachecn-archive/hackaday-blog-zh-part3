# 灵活的四轴飞行器几乎坚不可摧

> 原文:[https://hack aday . com/2017/03/18/flexible-qaudcopter-is-nearly-infantible/](https://hackaday.com/2017/03/18/flexible-qaudcopter-is-nearly-indestructible/)

我们都撞过四轴飞行器。几乎是不可避免的。一切都进行得很好，直到大自然张开她的大嘴，向你吹来一阵令人讨厌的风，把你的院子推到墙边。这堵墙恰好由比你的四轴飞行器更坚硬的材料构成。“如果……”你一边从地上捡起闪闪发光的新四轴飞行器的碎片，一边问自己，“他们用柔性材料做了这些东西？”

好吧，看来有人已经这么做了。[防撞四轴飞行器](http://spectrum.ieee.org/automaton/robotics/drones/epfl-bioinspired-hard-to-destroy-drone)由一个柔性框架(显然)组成，该框架用磁铁保持刚性。所以这个框架就像普通四轴飞行器的框架一样。当然，直到你撞毁它。然后就变得灵活了。

这个想法来自黄蜂的翅膀，你显然可以弄皱它而不损坏它。一定要看看下面展示其柔性框架的无人机视频，如果你在野外见过其他类型的柔性框架无人机，请告诉我们。
