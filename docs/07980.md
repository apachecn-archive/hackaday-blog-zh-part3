# 这个硬币电池可以移动火车！

> 原文：<https://hackaday.com/2017/12/14/this-coin-cell-can-move-that-train/>

[迈克·瑞斯比] [用硬币电池](https://hackaday.io/project/28470-coin-cell-train)移动了一列火车。确切地说，是 CR2477 细胞，也就是说，是一个稍微有点粗的例子，而正在讨论的火车不是全尺寸的，而是一个围绕着圣诞树的模型铁路，但尽管如此，火车还是动了。

一个硬币电池本身并不能驱动一个设计为 12 伏电压运行的模型机车。所以[马克]用升压转换器把三伏变成十二伏。不过，硬币电池具有较高的内阻，因此首先将硬币电池放电到两个超级电容中，为升压转换器供电。当他的超级电容器充电时，他仔细记录了电压随时间的变化，发现第一个充电需要 18 个小时，而第二个需要 51 个小时。

对于我们的硬币电池挑战赛的参赛者来说，这是重要而有用的数据，其中一些人还打算采用超级电容的方法来提供一次性的功率提升。不过我们怀疑，如果他选择了专用的超级电容充电器电路，他可能会从电池中提取更多的电量。
