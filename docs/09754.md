# 印刷应变波齿轮

> 原文:[https://hackaday.com/2018/06/19/printing-strain-wave-gears/](https://hackaday.com/2018/06/19/printing-strain-wave-gears/)

我们刚刚结束了 Hackaday 奖的机器人模块挑战部分，如果机器人需要做一件事，那就是移动。这通常意味着某种电机，但你可能也想要一个齿轮系统。要有那种扭力，你知道的。

![](../Images/c87d512657ab0a9e5a6dd1d1148a6f77.png)

为了他的 Hackaday 奖参赛作品，[Johannes]正在建造一个 3D 打印的应变波齿轮。一个应变波齿轮有一个柔性的中间件，当被椭圆形中心转子推动时，该中间件接触外部齿条。柔性轴环和外齿条上的齿数差决定了传动比。

这种齿轮几乎完全是 3D 打印的，零件不需要由柔性细丝制成，也不需要有怪异的支撑结构。它是从 PETG 打印出来的，[Johannes]说这对于谐波驱动来说足够滑，NEMA 17 步进器完全包含在齿轮本身的外壳内。

打印一个齿轮系统当然很好，但是你用它做什么呢？作为一项实验，[Johannes]用一个奇怪的骨状适配器将两个这样的电机[放在一起，为一台相机创建一个云台。是的，如果你不看那奇怪的粉红色和蓝色的骨头，它只是一个带万向节的三脚架上的 DSLR。这个装置的角度分辨率是 0.03 度，所以它应该可以用于天体摄影。令人印象深刻，尽管这个特定的实现看起来有点奇怪。](https://hackaday.io/project/120377-3d-printed-strain-wave-gear/log/145248-the-gear-in-action)
