# Arduino Uno 应变消除

> 原文:[https://hackaday.com/2017/10/07/arduino-uno-strain-relief/](https://hackaday.com/2017/10/07/arduino-uno-strain-relief/)

你的头发被跳线拔出来了吗？对松脱跳线进行故障诊断是否让你失去理智？您的项目是否因为您在 Arduino Uno 中失去了保护跳线安全的所有时间而出现备份？然后你需要全新的 Ardunio 应变消除外壳！

[Jeremy Cook]已经受够了从他的 Uno 中拉出的松散跳线，所以他设计了一个外壳，不仅保护了 Arduino 它还可以防止那些卑鄙的跳线在最不方便的时候拔出来。

Arduino 应变消除外壳由 3/4 英寸厚的 MDF 和 1/8 英寸厚的聚碳酸酯组成，肯定会成为每个黑客工作台的热门产品。[Jeremy]用数控切割机切割外壳和顶部。塑料顶部通过四个 4-40 螺钉固定在 MDF 底座上。有趣的是，他在敲打 MDF 上的螺丝孔之前涂了强力胶。我们将不得不在下一个项目中尝试这个技巧！
