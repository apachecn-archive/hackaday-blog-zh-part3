# 你的电动车需要多大的电池组？

> 原文:[https://hackaday . com/2017/12/25/你的电动汽车需要多少电池组/](https://hackaday.com/2017/12/25/how-much-of-a-battery-pack-does-your-electric-car-need/)

[Elon Musk]最近举行了一次他特有的高调产品发布会，他在会上推出了一款新的特斯拉电动半卡车。它的承诺很长，但电池组重量数据却很少，因此当然[Real Engineering]感到了不妙。他调查这个问题的视频在下面，但它不是这篇文章吸引我们眼球的链接。作为调查的一部分，他还创建了一个在线计算器来[估计任何电动汽车给定性能所需的电池容量](https://battery.real.engineering/)。

这并不完全直观，例如，它使用的是国际单位制而不是真实世界的单位，所以为了与通常的汽车数据进行比较，如果你是公制用户，需要进行一点心理转换，从公里和小时转换为米和秒，如果你使用英制单位，则转换为英里。但如果你对设计电动汽车或改装感兴趣，这仍然是一个有趣的工具，因为你可以无限地调整你选择的车辆的数字，以找到电池组成本的坏消息。

从技术角度来看，看到电动卡车的可信尝试非常有趣，我们希望现有的卡车制造商将向我们展示他们自己更现实的原型。但我们不禁会想，如果能制造出一辆能同时拖多辆拖车的卡车，电动长途卡车运输的整体效率将会大大提高。任何安全问题都可以通过让这些超级卡车拥有自己的高速公路来抵消，有了这样的专用基础设施，电力可以由路边的电缆而不是重型电池提供。在这种情况下，这些长长的电力牵引集装箱列车可能会相当成功，或许我们可以称之为*铁路*。
