# 小心高草:口袋妖怪 Go 上的 Gameboy 口袋

> 原文：<https://hackaday.com/2016/07/25/beware-of-tall-grass-pokemon-go-on-the-gameboy-pocket/>

[Pepijn de Vos]兴奋地与世界上最受欢迎的增强现实计步器 Pokemon Go 进行了互动，并极其失望地发现他的黑莓手机无法运行它。尽管如此，就他在过时的技术墙后面所能告诉我的，口袋妖怪 Go 就是走神，怀疑，偶尔抓一只口袋妖怪。那应该是可能的。

在 Gameboy 上破解[口袋妖怪并不陌生，【Pepijn】整理了一个计划。使用他的 TCPoke 模块，他更进了一步。他不只是在互联网上模仿最初的 gameboy 交易信号，而是用一些定制的 Z80 组件黑了一个口袋妖怪红色 ROM，为游戏中的有线电视俱乐部添加了一些功能。](http://hackaday.com/2015/04/01/use-the-internet-to-get-your-kadabra-to-evolve/)

在等待送货员带来一个可擦写的盒子和一些 Arduino 代码后，他现在可以将他在游戏中采取的步骤翻译成他在现实世界中的步骤。嗯，大部分是。他可以选择他想抓口袋妖怪的地点。角色站在那里。在大约 1 亿英镑的时候，这个游戏将会引发一场随机的口袋妖怪战斗。

[Pepijn]现在不再是一个社会弃儿，你可以在休息后的视频中看到。在一次简单的杂货店之旅中，他抓到了两只口袋妖怪！
