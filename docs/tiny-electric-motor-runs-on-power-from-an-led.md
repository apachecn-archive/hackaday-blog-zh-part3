# 微型电动马达依靠 LED 供电

> 原文：<https://hackaday.com/2017/03/21/tiny-electric-motor-runs-on-power-from-an-led/>

如果你不知道，发光二极管也可以反过来工作:当发光时，它们产生微安范围内的微小电流。如果你在 YouTube 上看，你可以找到几个用 LED 阵列建造太阳能电池板的视频，但用一个 3 毫米的 LED 驱动一个电动马达是我们从未见过的。[Slider2732]制造了一个小型电动马达，[在阳光下愉快地通过绿色 LED 驱动](https://www.youtube.com/watch?v=xGcRCk8SDHw)。

马达使用四个线圈，每个线圈 1000 欧姆。使用多匝非常细的线圈有助于减少电流，同时保持电机运行所需的适当磁场。为了保持摩擦力最小，转子使用一根悬挂在磁铁上的针。转子周围的四个钕磁铁被线圈周期性地推动，产生旋转。一个简单的双晶体管电路负责同步，是的，电机确实依靠 LED 提供的 4 微安电流运行，并且运行得相当好。

制造马达绝对是一项令人愉快的活动，这些小型脉冲马达可以在几个小时内制造出来。你可以使用几十匝的线圈，这更容易制作，但当然你需要四微安以上的东西！制造这样一个超低电流电机的好处是，它可以在一个小电池甚至一个电容器上运行很长时间，我们邀请你尝试建造一个。
