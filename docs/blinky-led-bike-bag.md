# 闪烁 led 自行车包

> 原文：<https://hackaday.com/2016/03/13/blinky-led-bike-bag/>

骑自行车的人永远不能太显眼:你越显眼，有人撞到你的可能性就越小。这是来自 Michal D ' auria 和 Stéphane De Graeve 的开源项目 Arduibag T1 背后的想法。该项目将安装在车把上的操纵杆与背包中的点阵 LED 显示屏结合在一起。通过移动操纵杆，用户可以指示一些事情，例如他们正在转弯、停车、说谢谢或者显示危险三角形来警告事故。

整个项目由简单的组件组成，如一个 [Adafruit LED 矩阵](https://www.adafruit.com/product/420)和一个 [Bluno](http://www.dfrobot.com/index.php?route=product/product&product_id=1369#.VuA1WCgrJaQ) (一个内置蓝牙 4.0 的 Arduino 兼容板)结合一个驱动 LED 矩阵的大电池。这连接到操纵杆，操纵杆在一个 3D 打印的盒子里，夹在车把上，方便使用。它看起来像是一个相当简单的构建，较大的组件安装在一个适合背包的板上，并将所有东西固定在适当的位置。然后，你在 LED 矩阵上方的背包部分添加一个透明的塑料盖，你就可以上路了，希望不会真的上路。

像任何好的项目一样，[Michal]和[Stéphane]还没有完成:他们还在寻找改进的方法。特别是，他们希望减少电池的数量，因为目前有一个大电池驱动显示器，另一个小电池驱动 Arduino。
