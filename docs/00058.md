# Hackaday 奖半决赛:更好的 DIY 水产养殖

> 原文：<https://hackaday.com/2015/09/11/hackaday-prize-semifinalist-better-diy-aquaculture/>

今年 Hackaday 奖的主题是“创造有意义的东西”。对于许多参与项目的团队来说，这意味着解决世界饥饿问题，特别是通过农业。谷物很棒，但蛋白质通常味道更好，[迈克尔·拉特克利夫] [将他的项目集中在水产养殖上](https://hackaday.io/project/7008-fly-wars-a-hackers-solution-to-world-hunger)，或者养殖鱼类和其他水生生物。

[迈克尔]决定解决的问题是根据水温、鱼的年龄和水箱中已经漂浮的食物量定期喂鱼。这其实是一个很难解决的问题；鱼一天喂一次以上会长得更好。目前，大多数水产养殖设施一天喂一次鱼，因为太费时间了。

[Michael]正在使用 Pis、Arduinos、USB 摄像头以及在自动化和控制系统方面的丰富经验，以最高效的方式喂鱼。这个项目的可能性很有趣；最好的研究表明，更有效的喂养计划可以转化为 20%的产量增加，这对世界来说是一个额外的食物。
