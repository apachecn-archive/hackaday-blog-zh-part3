# 改装锁以防止盗窃

> 原文：<https://hackaday.com/2016/12/12/modify-locks-to-baffle-burglars/>

虽然它经常被认为是一种犯罪活动，但实际上有一个活跃的爱好社区围绕着撬锁艺术。就像白帽黑客试图闯入信息系统来学习如何让它们变得更强大一样，locksport 领域的人也试图评估各种锁的弱点。对于业余爱好者来说，体验开锁的轻松可能会令人兴奋(也有一点紧张)，如果你足够担心，你可以走得更远一点，修改你的锁，使它们更难被打败。

[Rallock67]用[Rallock67]用普通工具安装的设备将有问题的锁送到[bosnianbill]。被称为墨菲球，一个比普通弹簧大的弹簧被插入其中一个销中，并由滚珠轴承固定住。这使得锁几乎完全不受[撞击](http://homesecurity.about.com/od/threats/a/The-Truth-About-Lock-Bumping.htm)的影响，也使得【波什尼亚比尔】，一个熟练的锁匠，由于弹簧施加在锁芯上的力的大小，撬开锁变得更加困难。令人惊讶的是，这种修改似乎相对容易做到，只需攻出一些螺纹，并插入一个固定螺钉来固定弹簧。

洛克斯波特和开锁是一个很好的爱好。大多数人开始挑选[小挂锁](http://hackaday.com/2015/11/16/hack-a-padlock-key-from-plastic-scraps/)，因为它们简单方便。甚至可以用一套发夹撬开一些锁[。而且，如果你真的想看看破解某些锁有多容易和/或美国运输安全管理局对你的整体安全有多大好处，](http://hackaday.com/2016/03/15/pick-locks-like-in-the-movies-using-a-bobby-pin/)[你也会想看看这个](http://hackaday.com/2015/09/18/dear-tsa-this-is-why-you-shouldnt-post-pictures-of-your-keys-online/)。
