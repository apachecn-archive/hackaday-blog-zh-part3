# 掏空 USB 电源组运行您的电视

> 原文：<https://hackaday.com/2015/12/17/gutted-usb-power-packs-run-your-tv/>

每个口袋里都有一台电脑，被拴在大型市电供电的电器上有点过时了。当你可以在手机上轻松观看 YouTube 时，你再也不用被困在电视机前了。但是你可能会在停电的时候渴望大屏幕体验，在这种情况下，学习用廉价的手机电源组建造一个简单的电池组可能是一个很好的生活技能。

与其从他的电池组中寻找长期离网可用性，不如寻找概念证明，[Stephen]用 18650 个锂离子电池和一个小型 300W 逆变器拼凑了一个快速电池组。所有的硬件都是从一家名为 Cd-r King 的公司廉价购得，这是一家我们希望在美国看到的基于菲律宾的打折小玩意。他拿了一把 USB 电源包，从每个电源包中取出一节 18650 电池，用 1/2 英寸的 PVC 管和一些螺栓快速制作了一个电池盒来连接逆变器。有了串联的四节电池，他可以轻松地运行一台平板电视，以及一台大型落地扇——比方说，背景中[斯蒂芬]架子上的那台[mool Titus](http://hackaday.com/2014/12/08/the-last-week-of-the-mooltipass-approacheth/)？被掏空的 USB 电源包的好处是它们仍然可以用来给电池充电。

正如[斯蒂芬]所承认的，这是一个简单的项目，有足够的空间进行实验。更多电池并联运行更长时间显然是第一步。他可能会从[这个笔记本电脑电池库项目](http://hackaday.com/2015/06/30/hackaday-prize-entry-laptop-batteries-for-a-power-bank/)中获得一些想法，甚至会升级到[特斯拉锂离子电池黑客](http://hackaday.com/2014/11/11/draft-an-intverview-with-tesla-battery-hacker-wk057/)——尽管我们怀疑 Cd-r King 对后者会有多大帮助。
