# 发现巨型梅森素数

> 原文：<https://hackaday.com/2016/01/21/giant-mersenne-prime-found/>

听说过梅森素数吗？这些是比 2 的幂小 1 的质数。以 17 世纪早期研究梅森素数的法国迷你修士马林·梅森的名字命名，互联网上有一个寻找梅森素数的分布式计算项目，名为 [GIMPS](http://www.mersenne.org/) (大互联网梅森素数搜索)。该项目最近宣布他们发现了已知最大的质数。

这个有争议的数字是 2 74207281 T1-1，有 22338618 位，比之前的记录保持者多了近 500 万位。这个数字对于现代密码学来说太大了。Prime95 是搜索这些独特素数的软件，经常被超频者用作 CPU 压力测试(该软件有一个压力模式，以确保素数测试是准确的)。它还发现了[英特尔 Skylake CPU](http://arstechnica.com/gadgets/2016/01/intel-skylake-bug-causes-pcs-to-freeze-during-complex-workloads/) 中的一个漏洞。

我们最近在《我的世界》讨论了[素数算法。[Mike Szczys]甚至使用了相对较小的 ARM 处理器](http://hackaday.com/2016/01/19/basically-its-minecraft/)[来完成这项工作](http://hackaday.com/2014/03/24/embeded-sieve-of-eratosthenes-hunting-primes-on-arm/)。你可以在下面的视频中了解更多关于新梅森素数的信息。
