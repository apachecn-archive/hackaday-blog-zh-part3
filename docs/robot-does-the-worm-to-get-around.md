# 机器人做蠕虫来四处走动

> 原文：<https://hackaday.com/2016/02/11/robot-does-the-worm-to-get-around/>

行走、跳跃、滚动、飞行、游泳——机器人的运动只受到发明者想象力的限制。[兔子罗杰]显然有非常生动的想象力，因为他正在建造像蠕虫一样移动的机器人。

[罗杰]的第一版机器人只有半蠕虫状，更像是[爬管机器人](https://www.youtube.com/watch?v=UwQYJPD676o)。它有一对 3D 打印的受电弓，可以随着伺服系统伸缩，并在步进驱动的丝杠上沿着机器人的轴移动。Arduino 读取传感器并协调受电弓的扩张，以抓住管道的内径并推动蠕虫机器人前进。这是在管道的有限范围内走动的一种缓慢但有效的方式。

下一个迭代，被称为【Wolly】，更像蠕虫，不局限于管道运行。它有四个可扩展的三角形框架，通过齿轮齿条式主干相互连接。第一个框架收缩，齿条推动它向前，它扩张，下一个框架收缩，很快它在地板上蠕动。仍然很慢，但看起来很整洁，你可以看到它是如何被操纵的。它甚至可以绕着它的长轴滚动，这也是一个不错的爬管机。

这个令人毛骨悚然的自主蠕虫机器人似乎与【Wolly】非常相似，但除此之外，我们还没有报道过太多这样的机器人。这些蠕虫机器人有很多想法和努力，我们很想知道[罗杰]会把这个独特的机器人身体计划带到哪里。
