# 自行车打气筒控制的喷泉克服重力滴水

> 原文:[https://hack aday . com/2017/05/22/挑战重力的自行车泵控喷泉/](https://hackaday.com/2017/05/22/gravity-defying-drips-of-a-bike-pump-controlled-fountain/)

人们喜欢看愚弄他们感官的把戏。这个不言自明的道理在本周末的 Crash Space 展台得到了体现，史蒂夫·戈尔茨坦和凯文·乔丹展示了一个由自行车打气筒控制的滴水喷泉。

这些视错觉滴水喷泉使用频闪灯似乎冻结在半空中滴水。这种情况我们已经见过好几次了(想到了 hack aday alum[Mathieu Stephan][的作品](https://hackaday.com/2015/06/25/blinking-leds-for-a-timeless-fountain/))，但从来没有像自行车打气筒这样令人愉快的用户输入。它连接到一个气压传感器，这个传感器由 Arduino 监控，Arduino 会使灯闪烁。当有人操作泵时，下落的液滴似乎变慢、停止，然后开始逆着重力流动。

遗憾的是，这种现象很难在视频中记录下来，因为我们视觉的延迟是魔术成功的关键。上面视频的帧速率和频闪不太协调，但是仔细看，你有时仍然会看到错觉。

就个人而言，效果是如此完美，以至于整个周末都吸引了一大群人。孩子们被邀请用手指划过滴水的溪流，以确认水确实是真的。即使你把手伸进幻觉中，也不会破坏效果。

喷泉的装修值得称赞。深色丙烯酸树脂组成了一个黑客空间标志形状的三角形外壳。水滴由振荡泵产生，从三角形的顶点落下。底部的马提尼酒杯用钢丝绒接住水滴，以防溅出。水循环系统完全隐藏在视野之外。这是一件艺术品，真的是整体体验的巅峰。这是一个 Maker Faire 湾区的打击，理应如此！

到目前为止，还没有公布建造的细节，但[史蒂夫]和[凯文]听起来好像他们计划记录他们的工作，所以请关注[崩溃空间页面](http://blog.crashspace.org/)。[凯文]写信告诉我们他们在漂浮喷泉上发布了[的细节。](http://blog.crashspace.org/2017/05/levitating-fountain/)
