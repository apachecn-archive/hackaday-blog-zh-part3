# 黑客日的奖品是:一个 400 兆赫的调制解调器

> 原文：<https://hackaday.com/2016/06/24/hackaday-prize-entry-a-400mhz-modem/>

在过去十年的大部分时间里，物联网一直被视为消费电子的未来。尽管没有人真正知道物联网将会做什么，但已经投入了数十亿美元。这些数十亿美元需要用在某个地方，就德州仪器而言，它直接进入了集成亚 GHz 无线电的下一代微控制器。[M.daSilva]参加 2016 年 Hackaday 奖将这些小型廉价的收音机[变成了便携式通讯器](https://hackaday.io/project/11056-waffle-a-pocket-sized-digital-radio-system)。

这个“400 MHz 频段的调制解调器”简单地由一个 ATmega 微控制器、TI 的 CC1101 亚 GHz 收发器、一个有机发光二极管显示器和一个超高频功率放大器组成。就收音机而言，这是最基本的，但是加上一个 USB 串行芯片和一个小程序，这种收音机可以向范围内的任何人或任何东西发送信息。这是一个自己动手做的寻呼机，有几个芯片和一些固件，系统已经可以工作了。

[M.daSilva]对此设备有两个使用案例。第一种是业余无线电寻呼系统，一个带有大功率放大器的基站向许多小模块发送信息。第二个用途是一个灵活的模块，使用业余无线电的数据模式将电脑连接在一起。由于有如此多的可能性，这是我们在今年的 Hackaday 奖中看到的最好的无线电构建之一。
