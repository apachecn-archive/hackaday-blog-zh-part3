# Sudo 给我做个三明治

> 原文：<https://hackaday.com/2017/03/30/sudo-make-me-a-sandwich/>

你喜欢你的火腿和奶酪三明治吗？如果你回答“我喜欢牛肉”，看看[威廉·奥斯曼]的[范·迪塞尔火腿和奶酪三明治](http://www.williamosman.com/2017/03/vin-diesel-ham-and-cheese-sandwich.html)就知道了！[奥斯曼]的博客口号是“有科学可做”，但他是第一个承认这是科学走得太远了。当他的一个追随者[洗手间的声音]评论说“请用激光切割火腿切片的激光切片来雕刻一个[范·迪塞尔]的半身像”时，他不得不这么做。

他的朋友[CameraManJohn]用 Maya 制作了半身像，[Osman]还提供了下载文件的链接，以防万一有人想尝试一下。他们挑选了超市里能买到的最便宜的火腿切片——所以从技术上讲，他们实际上并没有用激光切片火腿。为了帮助生成切片轮廓，他们找到了用于 Autodesk Fusion 360 的[切片器应用程序，该应用程序确实完成了需要完成的工作。该应用程序将 3D 模型转换为单独的横截面，类似于核磁共振成像。它有助于测量各种原材料样品的厚度，以便切片机输出不会过度拉伸(或挤压)。结果是一套编号的 2D 图纸，可以发送到您的激光切割机。](https://apps.autodesk.com/FUSION/en/Detail/Index?id=8699194120463301363&os=Mac&appLang=en)

视频的其余部分在 gross-o-meter 上得分相当高，因为[Osman]开始用激光切割火腿片(和几片奶酪)，品尝激光切割的火腿(当然是为了科学)，并试图防止他的计算机出错。最后，三明治实际上看起来相当不错，尽管我们不会评论它的味道。一双炯炯有神的眼睛为胸部增添了个性。

一个问题是，Slicer 应用程序没有优化其有效打包的结果。最小的部分占据与最大部分相同的边界框。这导致许多被浪费的火腿片被扔掉。[Bill]仍然不知道如何处理他的美味三明治，所以如果你有建议，请在看过下面链接的视频后发表评论。如果你认识[范·迪塞尔]，让他知道。

这并不是[奥斯曼]第一次冒险尝试疯狂的食物黑客——这里有几个美味的例子:一个[为你涂黄油的烤面包机器人](http://hackaday.com/2017/02/24/toast-bot-butters-for-you-sometimes/)，一个[激光切割姜饼拖车屋](http://hackaday.com/2017/01/03/laser-cut-gingerbread-trailer-home/)，以及一个[南瓜皮战斗机器人](http://hackaday.com/2016/10/28/dont-make-your-battlebot-out-of-a-pumpkin/)。
