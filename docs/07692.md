# 车床转过拐角，做一个立方体

> 原文：<https://hackaday.com/2017/11/13/lathe-turns-the-corner-makes-a-cube/>

[Tim]厌倦了用他的车床转动东西。他决定用圆柱形的废料做一个游戏骰子——看起来像是正方形的东西。事实证明，这是有可能做到的车床与三爪卡盘。[Tim]发现当他将圆形侧面切成面时，颚上的斜面将从侧面抓住一个圆柱形的碎片。

在车床上转动一个立方体看起来相当复杂，所以我们更加赞赏[Tim]可爱的手工。正如你将在下面的视频中看到的，事情进展顺利，直到他进行最后一次正面切割。也许刀具没有对齐，或者在夹紧过程中有什么东西掉了，但是第一次通过在枪托上造成了一点凿槽。不过，看起来很容易修好。经过四次 90 度旋转和表面切割，他有了一个好看的粗糙立方体。

这是一个标准尺寸的模具，所以下一步是将其削减到 16 毫米。然后是打磨、抛光和添加圆点的时候了。为了设计它们，[蒂姆]在立方体上喷洒了设计液，并在每个面上划出了独特的线条图案。然后，他钻了缺口，并用铝黑填充。

我们看到的大多数骰子都是电子的，像[这种极其随机的一对](https://hackaday.com/2014/05/24/the-most-random-electronic-dice-yet/)和[这些 PIC 驱动的 LED 骰子](https://hackaday.com/2012/10/25/a-pic-powered-pair-of-electronic-dice/)。我们希望看到[蒂姆]成为第二个 D6，这样他就有一对了。然后做一个 D20。拜托了。
