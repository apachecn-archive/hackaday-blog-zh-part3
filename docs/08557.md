# STM8 入门

> 原文：<https://hackaday.com/2018/02/12/getting-started-with-stm8/>

如今有如此多不同的 CPU，通常使用它们中的任何一个最困难的事情是开始使用和收集正确的软件工具。如果你曾经关注过非常便宜的 STM8 处理器，你会想看看[Shane Burrell 的]视频(见下文)关于如何开始使用 STM 8。

STM8 不是 32 位处理器——你可能从名字就能猜到。[Shane]使用 SDCC(小型设备 C 编译器)瞄准小芯片。他还展示了他如何管理一段相当重要的代码，以及他如何控制构建过程。

好消息是，这不是一个闪烁的 LED 例子。他展示的代码使用中断并读取模拟值。他没有涉及很多细节，但他确实谈到了一些代码的动机。您还会学到一些技巧，比如在 Git 中存储固件和硬件设计。

我们认为这个例子与你在视频中看到的车辆有关。最近，我们已经看到 STM8 在烙铁内部执行任务[。如果你想挑选一款不那么显而易见的处理器，看看[Jay Carlson]对价格低于 1 美元](https://hackaday.com/2018/02/09/review-cxg-e90w-temperature-controlled-soldering-iron)的[21 款微控制器的比较。](https://hackaday.com/2017/11/07/which-microcontroller-is-best-microcontroller/)
