# 电子人艺术家——纹身机器手臂假体

> 原文：<https://hackaday.com/2017/01/04/the-cyborg-artist-tattoo-machine-arm-prosthesis/>

[JC Sheitan Tenet]10 岁时失去了右臂。和我们大多数人一样，他也是右撇子，所以他因为没有手臂而不得不面对的挑战变得更加困难。

> 你曾经尝试过用你的非惯用手做一些平凡的事情吗？如果你是右撇子，你试过用左手吃饭吗？或者如果你是左撇子，你能用右手写字吗？对于一些人来说，使用双手是天生的，但是如果你像我一样，你的非惯用手就没什么用了。

问题是，他想成为一名纹身艺术家。他没有放弃。即使面临找不到愿意收他为徒的纹身师的额外困难，他也没有放弃。所以他成为了一名纹身师，只用他的左臂。直到几个月前，他在一次纹身大会上遇到了工程师背景的生物机械艺术家 Jean-Louis Gonzal。看到【Gonzal】的作品后，他只是问有没有可能改装一个假肢，在上面贴上纹身机。

电子人艺术家诞生了。假体里的纹身机可以 360 度移动，动作范围很广。[JC Sheitan Tenet]通常使用它来帮助处理颜色、阴影和抽象形式。这是一个糟糕的蒸汽朋克假肢，它不只是为了展示，他实际上与它一起工作(尽管不完全)。这似乎只是开始，因为原型的第一个版本工作得如此之好，第二个版本已经由[JC]和[Gonzal]在计划中。我们迫不及待地想看看他们会推出什么，也许是当前版本和纹身机械臂或大脑控制针的混合？
