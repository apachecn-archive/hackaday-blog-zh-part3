# 一个真正的悬浮滑板

> 原文：<https://hackaday.com/2015/12/27/an-actual-working-hoverboard/>

2015 年显然是“悬浮滑板年”，在年底之前，我们有了最后一个竞争者。它被称为私人太空公司 ArcaSpace 的 ArcaBoard。它不使用磁铁，或超导体，或任何烟雾和镜子——只是一大堆管道风扇。

准确地说是 36 个。ArcaBoard 使用 36 个电动马达，每个马达明显有 7.55 马力，由大量锂离子电池供电。它们共同产生 430 磅的推力，这使得大多数骑手可以很容易地漂浮。即使有巨大的电力消耗，它显然持续了整整 20 分钟，考虑到它的大小，这是非常令人印象深刻的。

显然，[它将于今年春天](http://www.arcaspace.com/en/arcaboard.htm)上市。只需要 19900 美元。我们不知道他们是不是认真的。它肯定是可行的，但我们希望先看到一些人们实际上在上面去某个地方的演示——下面视频中的大多数剪辑只是显示他们在原地漂浮……如果你想这样做，就给自己做一个[吹叶机悬浮滑板吧！](http://hackaday.com/2014/12/12/a-simple-hoverboard-everyone-can-understand/)

也就是说，我们完全被他们的想法所打动，并且实际上建立了一个工作产品。显然，制作悬浮滑板的想法是由首席运营官的儿子提出的。首席执行官听到这个，说是的，我们完全可以做到这一点！他们确实这么做了。

它比[雷克萨斯悬浮滑板](http://hackaday.com/2015/08/05/secrets-of-the-lexus-hoverboard-revealed/)大一点，但至少你不需要一条特殊的轨道来骑它。
