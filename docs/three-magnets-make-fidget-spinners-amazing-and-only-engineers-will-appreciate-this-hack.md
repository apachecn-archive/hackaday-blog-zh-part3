# 三个磁铁使坐立不安纺纱惊人，只有工程师会欣赏这个黑客！

> 原文：<https://hackaday.com/2017/06/29/three-magnets-make-fidget-spinners-amazing-and-only-engineers-will-appreciate-this-hack/>

坐立不安旋转贴将继续下去，直到士气提高。这一次，我们正在看[TannerTech]的[电磁加速器，用于一个坐立不安的旋转器](http://www.instructables.com/id/Electromagnetic-Fidget-Spinner-Accelerator/)。[坦纳]可以用他闲置的零件和磁铁与继电器的巧妙应用，电子地旋转他的烦躁旋转器*！工程师讨厌他！*

 *[坦纳]的构造由三块磁铁组成，安装在坐立不安的旋转者的手臂顶端，北极朝外。“驱动电路”由一个电磁体——一个在旧电视机中发现的电感器——一个簧片开关和一个 MOSFET 组成。当电路被放置在小动作旋转器旁边时，簧片开关闭合，给电磁铁供电，向前推动小动作旋转器的尖端，重新开始循环。你可以把它想象成粒子加速器或磁悬浮列车中的相同技术。或者无刷 DC 电机。
