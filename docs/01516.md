# 制造一把金属手枪

> 原文：<https://hackaday.com/2016/02/17/building-a-sheet-metal-pistol/>

互联网上流传着一种由金属片制成的半自动手枪的计划。就像许多 3D 打印枪支的计划一样，似乎没有人真的制造过这种手枪。它只是作为一种技术构造而存在，你可以复印图表，在几片金属上描摹，大概还可以组装成一把枪。这些部件可以变成枪状物体的唯一证据是两年前的一些随机博客帖子，显示了一个非常丑陋的喷涂哑光黑色的手枪。

[克林顿·韦斯特伍德]决定接受挑战，把这些计划变成一把真枪。他在 YouTube 上记录了他的努力，并上传了一堆整个建造过程的照片。这把枪还不能完全工作，但是*几乎*可以了，而且他完全是在一家汽车修理店用任何人都能从家得宝买到的工具做这件事。

这种枪的大部分结构都很简单——毕竟它只是一片金属。该杂志是通过在一块金属上描绘图案，将其缠绕在心轴上，然后焊接在一起而制成的。枪的侧板也是用拼图拼成的。给枪管装膛线——使这把枪既准确又合法的东西——需要制造一些有趣的工具。膛线工具只是一根穿过枪管的圆棒。一小片钢锯刀片被切割成适合这个圆棒的形状，桶被一个自制的工具非常缓慢地切割。

最终的结果看起来像是出自后启示录时代最优秀的工匠之手。一把看起来很酷的枪如果没用就没用了，在这里 DIY 手枪功亏一篑。用过的弹壳不会弹出来。这仍然是一个进步，从第一次建立这种枪只是传言发射空白。

最近，枪械制造领域充斥着无法使用的 3D 打印手枪和可以使用的 3D 打印枪支，但不知何故，这些 3D 打印枪支比最先进的技术落后了 200 年。我们很高兴看到一些人仍然在用他们的双手建造东西，并希望[克林顿]最终能让这把枪发挥作用。
