# 吝啬鬼黑客标签打印机，在自己的游戏中击败制造商

> 原文：<https://hackaday.com/2017/05/24/tightwad-hacks-label-printer-beats-manufacturer-at-own-game/>

有时我们为了创造新事物的刺激而努力，有时我们努力推开无知的黑暗面纱，让问题有了新的曙光。有时候，就像当[把一台二手标签打印机变成一台销售点收据打印机](https://www.youtube.com/watch?v=A7fYWOQu6VI)时，我们攻击是因为我们是吝啬鬼。

我们带着最大的尊重和感情说这些——当你的动机完全是为了金钱时，没什么好羞愧的。在[Dan Herlihy]的案例中，让廉价的兄弟标签打印机使用热敏纸意味着在专用收据打印机上节省 300 美元。但这也意味着在兄弟的“剃刀和刀片”商业模式上击败他们，这种模式让你不断购买他们昂贵的专有品牌。塑料标签卷上的孔图案告诉打印机装载了什么尺寸的标签，因此[Dan]通过撕下一块塑料并将其粘在传感器上来克服这一点。为了让打印机相信普通热敏纸是标签原料，他打印了一个小纸条，上面有与标签背面相同的黑色注册条纹图案。非常聪明的东西，它让他以非常低的成本为他的电子商店打印高分辨率的收据。

[Dan]的破解很简单，但可能会受到纸质编码条的磨损。也许[这种使用齿轮作为编码器的兄弟黑客](http://hackaday.com/2012/07/13/hacking-a-brother-thermal-printer-to-use-non-oem-continuous-rolls/)将为长期修复提供一些灵感。
