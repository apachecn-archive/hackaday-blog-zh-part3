# Hackaday 奖参赛作品:消防员的增强现实

> 原文:[https://hack aday . com/2016/07/24/hack aday-prize-entry-augmented-reality-for-fireworks/](https://hackaday.com/2016/07/24/hackaday-prize-entry-augmented-reality-for-firefighters/)

增强现实现在风靡一时，这都是因为口袋妖怪。当然，这意味着增强现实的整个想法现在都被包裹在拍摄非自然环境中的小家伙的照片中。增强现实还有更多有用的应用，正如[vijayvictory]的 [Hackaday 奖参赛作品向我们展示的](https://hackaday.io/project/12499-code-red)。他为消防员建造了一个增强现实头盔，可以检测温度、气体、烟雾和用户自己的生命体征，并在抬头显示器上显示读数。

该建筑的核心是一个粒子光子，一个支持 WiFi 的微控制器，它也让这个头盔能够将数据中继回基站，表面上是一个没有着火的基站。为此，[vijayvictory]增加了一个加速度计、气体传感器和一个安装在棱镜后面的漂亮的有机发光二极管显示器。该显示器将相关数据叠加到消防队员身上，而不会妨碍他们的视野。

现在，这个系统相当基础，但是[vijayvictory]还有一些锦囊妙计。通过扩展这一系统，使其包括前视红外热成像传感器，这种增强现实头盔将具有看穿烟雾的能力。通过将该系统集成到现有网络中，并添加一些很酷的 WiFi 技巧，该系统将能够使用信号三边测量来定位倒下的消防员。这是一个非常酷的设备，应该非常有用，这使它成为 Hackaday 奖的一个很好的参赛作品。
