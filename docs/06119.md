# 微动开关内部

> 原文:[https://hackaday.com/2017/05/30/inside-a-microswitch/](https://hackaday.com/2017/05/30/inside-a-microswitch/)

我们已经拆开了几个微型开关，主要是为了修复那些讨厌的出现双击综合症的罗技鼠标，但我们从未制作过视频。幸运的是，[Julian]做到了，如果你想[了解这些组件的内部机制](https://www.youtube.com/watch?v=yPNJv-0F15w)，这是值得一看的。

[Julian]谈到了接触的方式。他还讨论了系统中固有的机械滞后，因为金属移动触点具有类似弹簧的性质

我们总是很难把小活塞固定住，所以我们喜欢[朱利安]用一点油灰把它固定住的想法。如果你想知道他一开始摆弄开关的原因，请看第二段视频。他的 8 位试验板计算机使用这样的开关来驱动单步时钟。

不久前，我们对微动开关 a 做了一个[详细介绍，所以如果你想深入了解更多细节，这是一个很好的起点。至于](https://hackaday.com/2017/04/17/microswitches-past-the-tipping-point/)[试验板电脑](https://hackaday.com/2017/04/10/8-bit-breadboard-computer-is-up-to-8-hours/)，我们也喜欢它们。
