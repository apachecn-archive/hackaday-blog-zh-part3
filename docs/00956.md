# 遥控迷你火焰喷射器带来燃烧

> 原文:[https://hack aday . com/2015/12/16/RC-mini-flame-thrower-bring-the-burn/](https://hackaday.com/2015/12/16/rc-mini-flame-thrower-brings-the-burn/)

不言而喻，无线电控制的迷你火焰喷射器只会是个坏主意，你永远也不应该建造它。但是一旦你看了下面的视频，你就会忍不住去尝试。但是不要这样做，已经有人警告过你了。

也就是说，下面的视频显示[Make-log]非常紧凑的结构充满了安全联锁装置，并配有一个考虑周到、信息丰富的用户界面。它由一小罐喷雾除臭剂提供燃料，其阀门由伺服系统驱动，由火花隙点火器点燃。唉，SparkFun 不再提供这最后一个关键组件，所以如果你选择推出自己的产品——你不应该这样做——你需要找到一个替代品。

我们之前展示过数量不合理的火焰喷射器项目，包括一个[吨](https://hackaday.com/2009/11/03/improved-arm-mounted-flame-thrower/)的[腕](https://hackaday.com/2014/09/25/a-wrist-mounted-flamethrower-sure-why-not/)—[挂载](https://hackaday.com/2013/08/18/wrist-mounted-flamethrower-on-the-cheap/) [单元](https://hackaday.com/2008/04/22/diy-x-men-flame-throwing-wrist-guard/)。当然，如果你是一个音乐爱好者，你也会想看看这个[迷你战士装置](https://hackaday.com/2015/06/07/mad-max-inspired-flamethrower-ukulele/)。
