# 本周失败:磁流量测量出错

> 原文:[https://hack aday . com/2016/07/29/每周故障-磁流测量-出错/](https://hackaday.com/2016/07/29/fail-of-the-week-magnetic-flow-measurement-gone-wrong/)

物理学给了我们了解宇宙所需的基本工具，但将理论转化为有用的东西是工程师们的谋生之道。推进这一界限是本周《本周失败》的主题，我们将跟随[制作一个可用的磁流量计](https://www.youtube.com/watch?v=TR0baWuB6v4) (YouTube，嵌入下文)。

理论表明，测量流体流量应该很简单。毕竟，将磁性桨轮插入液体流中，用簧片开关或霍尔传感器计数脉冲是非常简单的，对吗？不过，在这种情况下，[实际工程](http://practical.engineering/)的【Grady】从一种复杂得多的流量测量方式开始——电磁检测。他在解释法拉第感应定律以及流体如何成为穿过磁场并在其中感应出可测量电流的导体方面做了大量工作。电流应该与流体的速度成正比，所以自制一个电磁流量计应该很容易，对吗？ 没有——尽管很努力，【格雷迪】还是没能从他系统的噪音中得到有用的信号。

理论是合理的，他的测试平台看起来可行，而且他有一些相当不错的仪器。那么[格雷迪]哪里出了问题？他能用更好的 [仪表放大器](http://hackaday.com/2016/03/18/beyond-measure-instrumentation-amplifiers/) 来清理信号吗？如果他把工艺流体换成更导电的东西，比如盐水，会发生什么？他自己承认，电气工程不是他的强项——他的职业是土木工程师。你能清除那个信号吗？请在评论区告诉我们。
