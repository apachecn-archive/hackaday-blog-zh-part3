# “那不是刀…”

> 原文:[https://hackaday.com/2016/04/21/thats-not-a-knife/](https://hackaday.com/2016/04/21/thats-not-a-knife/)

喜欢玩危险的游戏。他的最新作品[刀见你](https://www.youtube.com/watch?v=HEOyRuuyNzA)切中了合作游戏的痛处。3 人类一起对抗机器。游戏的目标是尽可能长时间按住按钮。这款游戏通过在游戏场地上手指水平的位置挥刀来增加难度。(视频如下。)

刀遇见你是由一个食肉的 Arduino 控制的。除了读取控制和驱动伺服系统移动刀具，Arduino 还在 2×20 字符的 LCD 上显示鼓励信息。

这个想法是为了吓唬人们，而不是真的把他们切成片。为此，刀实际上是一个电容传感器。当游戏检测到刀接触到柔软的人肉时，它会在血液开始流动之前停止刀的动作。这个游戏表明一个玩家被打败了，他向失败者做了几个砍的动作。如果输的玩家仍然有他们所有的数字，新一轮开始。

该项目是作为一个 24 小时游戏果酱的一部分。最终的产品非常漂亮，装在一个木质的箱子里，便于旅行时关闭。它甚至有一个手提把手，所以你可以带着它去聚会，找到新的受害者玩家。

我们不确定刀和 Arduinos 是什么关系。就在几个月前，我们看到一个 Arduino 人开着一把[刀挥舞着触手](http://hackaday.com/2015/11/30/arduino-powered-knife-wielding-tentacle-will-leave-you-in-stitches/)。世界上最友好的微控制器板会背叛我们吗？
