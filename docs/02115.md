# 桌面攻城武器:火球大炮

> 原文：<https://hackaday.com/2016/04/20/desktop-siege-weapon-fireball-cannon/>

正在寻找一种~~无害的~~方式来真正提升你的办公室战争游戏？为什么不[造一个硝化纤维素桌面炮！？](https://www.youtube.com/watch?v=4fErEShd8xM)

在我们最喜欢的科学 DIY YouTube 频道上，[NightHawkInLight]向我们展示了他是如何用可互换的卡农弹壳制作出这种令人敬畏的卡农的！它甚至有一点蒸汽朋克的感觉。

硝化纤维，或者更普遍的说法是闪光棉，被魔术师用来表演火球魔术。与闪光纸类似，它燃烧起来非常快，几乎不会留下任何灰烬或残渣。创造火球效果就像在管子里点燃它一样简单——膨胀的气体会把它猛烈地发射出去。

所有的动作都在 3/4 英寸的铜管筒里，配有自制的电热塞，电热塞是用从一个坏掉的吹风机上收集来的镍铬合金丝制成的。这些可互换的弹夹让[夜鹰之光]可以提前上膛并快速连续发射。
