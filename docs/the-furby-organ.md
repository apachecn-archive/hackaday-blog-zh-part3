# 弗比风琴

> 原文：<https://hackaday.com/2018/02/12/the-furby-organ/>

有时候，你有一个非常聪明和疯狂的想法，你只需要把它变成现实。2011 年，[看，妈妈，没有电脑]在他的笔记本上画了一个 Furby 风琴的计划，这是一个由键盘和 Furbies 合唱团组成的风琴。对于那些不知道 Furby 是什么的人来说，它是一个小巧可爱、毛茸茸的机器人玩具，会说 Furby 语和大量人类语言。从 1998 年到 2000 年，在最初的生产过程中售出了 4000 万台，此后还卖出了更多。然而生活介入了，而且，[LMNK]放弃了 Furby 风琴，只是最近又拿起了它。

他无法从未经修改的 Furbies 中获得稳定的音符，所以他提出了他所谓的 Furby Forman 融合合成。每个 Furby 由一对 Ardunios 控制。一个 Arduino 给 Furby 内部的部分排序，另一个产生[共振峰音符](https://en.wikipedia.org/wiki/Formant)，让 Furby 唱元音。

我们喜欢他给本来是权力开关的东西贴上的标签，即集体觉醒开关。轻弹它会让所有 44 只(我们数是 45 只，但他说是 44 只)毛皮兽大声说话，同时移动它们的耳朵、眼睛和喙。按下循环开关可以让他们保持他们发出的任何声音。元音转盘改变元音。但是你只能在下面的视频中自己去看和听了。第二个视频也有施工细节。

想让 Furby 为所欲为吗？看看[Jeija]是如何对一个机器人进行逆向工程来控制它的。
