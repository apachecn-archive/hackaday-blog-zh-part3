# 潜水机器人用激光捕捉虱子

> 原文:[https://hack aday . com/2017/04/10/潜水机器人-激光猎虱/](https://hackaday.com/2017/04/10/submersible-robots-hunt-lice-with-lasers/)

去毛是一个艰难的农业过程。这成为世界上最大的三文鱼出口国挪威人养殖的数十万条三文鱼的养殖圈里的一个主要问题，这种环境让这种寄生虫得以繁衍。为了解决这个问题，由[黄貂鱼海洋解决方案]开发的黄貂鱼是一种自主无人机，能够用激光每天消灭数万只虱子。

黄貂鱼于 2014 年引入挪威，2016 年引入苏格兰的一些地区，黄貂鱼漂浮在鲑鱼围栏中，处于警戒和等待状态。如果虱子识别软件(没想到你会听到这个术语，嗯？)在视频中探测到超过两帧的寄生虫，它会立即从两米远的地方用 530 纳米宽、100 毫秒的激光脉冲消灭它。不要担心——鲑鱼的鳞片有足够的反射性，不会伤害它，而害虫会被炸得很脆。实际上，它让人想起了宇宙飞船上的点防御激光器。
