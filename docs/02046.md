# hack let 103–压电项目

> 原文:[https://hackaday.com/2016/04/09/hacklet-103-piezo-projects/](https://hackaday.com/2016/04/09/hacklet-103-piezo-projects/)

压电效应的规则很简单:对材料施加机械应力，你就产生了电荷。反之亦然:给材料施加电压，它就会改变形状。然而，这并不适用于所有情况。只有晶体、某些陶瓷和骨头等材料具有压电特性。压电效应在电子学中使用得相当多，所以许多黑客项目探索这种物理现象也就不足为奇了。本周的 Hacklet 是关于利用压电效应的一些最好的项目。

[![strum](../Images/e6409bf3c4a0d7e45dae40b119ce1d8b.png)](https://hackaday.io/project/9465) 我们先从【miro2424】和[扫弦](https://hackaday.io/project/9465)说起。“扫弦键盘”让您通过扫弦来弹奏 MIDI 乐器，就像弹奏吉他一样。音乐键盘在这里充当吉他指板——可以按下按键来选取音符，但不会产生声音。当拨片被弹奏时，六个铜条充当电容传感器。触摸这些条可以确定将要播放的音符。隐藏在电路板下的压电盘检测音符被敲击的力度。运行扫弦板的 ATmega328 然后将力度和音符开 MIDI 信息传递给合成器。

[![stm](../Images/07600dbac4cc63162ec2233b4727b350.png)](https://hackaday.io/project/4986) 接下来是【丹·贝拉尔】带着[扫描隧道显微镜](https://hackaday.io/project/4986)。受[约翰·亚历山大]一个项目的启发，[丹]发明了自己的扫描隧道显微镜(STM)。像这样的仪器的关键是精确的运动。[丹]通过使用一个普通的压电圆盘来实现。从烟雾探测器到贺卡，这些磁盘被用作扬声器和蜂鸣器，所以它们很常见，也很便宜。[丹]把他的压电圆盘电极切成象限。仔细控制施加在象限上的电压，使得[Dan]能够在 X、Y 和 z 方向上移动他的 STM 针尖。令人难以置信的是，这台显微镜能够在原子尺度上创建图像。

[![touchboard](../Images/8144ea6bb2ba269424b74d2f5a3895f5.png)](https://hackaday.io/project/1990) 【撒切尔·张伯伦】接下来是[低成本触摸屏无处不在](https://hackaday.io/project/1990)。[Thatcher]使用三个压电圆盘使任何平面都变得触摸敏感。三个传感器放置在矩形的三个角上。触摸矩形将在表面产生振动，振动被传递到压电传感器。通过测量振动到达的时间，应该可以确定表面被触摸的位置。这种测量需要一个像样的处理器，所以[Thatcher]在恩智浦的 LPC1114FN28 中使用了 ARM Cortex-M0。最初的测试是有希望的，但是我们还没有听到太多关于这个项目的消息。如果你在网上看到他，告诉他快点！我们希望把我们的停车场变成一个巨大的电子棋盘！

[![cont](../Images/e89ae90017a2e8c78f160ebbf348bf68.png)](https://hackaday.io/project/8467) 最后，我们有【何塞·伊格纳西奥·罗梅洛】与[低功率连续性测试仪](https://hackaday.io/project/8467)。[Jose]以一种稍微平凡的方式使用压电元件——作为蜂鸣器。当你只是想检查几个电路的连续性时，谁还需要一整个万用表呢？该连续性测试仪使用 PIC12LF1571 处理器来查找开路和短路。PIC 中的 ~~5~~ 10 位 ADC 对于这种测试器来说分辨率足够了。事实上，[Jose]甚至包括了一个二极管测试，如果将导线放在一个正常工作的二极管上，就会发出短暂的嘟嘟声。PIC 处理器使用如此之少的能量，以至于这个测试仪应该在 CR2032 手表电池上运行大约 800 小时。
