# 最佳产品条目:模拟内存

> 原文:[https://hack aday . com/2017/07/17/best-product-entry-emulating-memory/](https://hackaday.com/2017/07/17/best-product-entry-emulating-memory/)

对于今年的 Hackaday 奖，我们给每个人都提供了一个硬件创业的机会。这是 Hackaday 奖的[最佳产品部分，该竞赛将向同时也是产品的最佳硬件项目授予 30，000 美元的奖金和我们设计实验室的常驻资格。](https://hackaday.io/prize/bestproduct)

想象一下世界上所有垃圾填埋场里的所有内存芯片。如果我们可以很容易地恢复那些被冲洗过的主板，并替换掉被恶意软件破坏的芯片上的 ROM 文件，那会怎么样呢？这是[Blecky]在 Hackaday.io 上的[EEPROM/闪存仿真器项目](https://hackaday.io/project/20657-eeemu-spi)的承诺。该项目旨在成为终极存储器接口，轻松仿真 SPI 接口 EEPROM 或闪存芯片组。它还可以用作安全设备，检查 BIOS 中的异常变化。

EEEmu 封装了一个基于 Atmel SAM4S Cortex-M4 处理器的微控制器、一个 SD 读卡器、一个用于重新编程的微型 USB、升压转换器、稳压器，并包括额外的 3.3V GPIO/I2C 端口，所有这些都在一个 51.5x20mm 的小电路板上。版本 2 预计将包括更多功能，以方便用作普通微控制器:更多 GPIO 引脚、USB 电压监控和 SPI 输出的高 Z 控制。

EEEmu 是完全开源的，[Blecky] [在 GitHub 上分享了他的代码](https://github.com/EEEmu/)，甚至创建了一个 EEEmu Fritzing 部件，也可以在他的库中找到。
