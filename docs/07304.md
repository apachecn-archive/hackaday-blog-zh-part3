# 很快你就会坐在机器人的脑袋里工作

> 原文：<https://hackaday.com/2017/10/10/soon-youll-sit-inside-a-robots-head-at-work/>

麻省理工学院的计算机科学和人工智能实验室 CSAIL 已经创建了一个使用 Oculus Rift VR 头戴设备[远程操作 Baxter 人形机器人的过程。这个项目的部分目标是让制造业工作~~变得非常有趣~~远程办公。这甚至可能是一种从远处监督机器人工人的方法。](https://www.youtube.com/watch?v=4a-W3Od5-t8)

简而言之，用户在虚拟现实环境中远程控制机器人。用户在一个模拟的虚拟现实环境中这样做，就像一个有多个传感器显示器的控制室，让他们感觉好像坐在机器人的脑袋里。通过使用手控制器，用户可以将他们的动作与机器人的动作相匹配，以完成各种任务。如果你已经看过《环太平洋》,你可能现在正在想象一个杰加——除了精神链接。

[白皮书](https://arxiv.org/pdf/1703.01270.pdf) (PDF)介绍了团队在开发该系统时所做的选择。在过去，遥操作通常采用直接模型或半物理模型。

在“直接”模型中，用户的视觉直接与机器人的状态相耦合，但延迟的信号可能会导致恶心和头痛，因为缺乏视觉运动表现的前庭刺激。在“网络物理”模型中，用户与机器人分离，并与机器人和环境的虚拟副本进行交互。虽然这种方法对驱动程序来说更好，但它需要更多的数据和专用空间。

像任何成功的大杂烩一样，CSAIL 团队的系统将前两种方法结合成了更先进的东西。它解决了延迟问题，因为用户不断地从虚拟世界接收视觉反馈。此外，它还解决了感觉与机器人不同的网络物理问题:一旦用户戴上耳机并登录系统，他们就好像在巴克斯特的脑袋里一样。这个系统的辉煌之处在于它模仿了“侏儒思维模型”——即有一个微小的人类居住在我们的大脑中，控制我们的行动，观看我们看到的图像，并为我们理解它们。
