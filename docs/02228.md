# 完善 DIY BB-8

> 原文:[https://hackaday.com/2016/05/09/perfecting-the-diy-bb-8/](https://hackaday.com/2016/05/09/perfecting-the-diy-bb-8/)

直到大约一年前，机器人建造者俱乐部已经想出了建造《星球大战》中任何一种机器人的一切办法。制造一个 R2D2 克隆人很容易，甚至 R5 和 R6 机器人也很常见。甚至有人试图克隆 IG-88。然后迪士尼发生了，*原力觉醒*上映，向世界介绍了第三部曲的主人公，BB-8。一些人已经不可思议地将 BB-8 复制成一个独特的自制机器人，但没有人比[詹姆斯·布鲁顿]付出更多的努力。[他正在完成他的第三个 DIY 版本的 BB-8](https://www.youtube.com/watch?v=rZVsAi1Izxs) 。

[詹姆斯]b b-8 机器人的第三个版本[有两个我们之前见过的哥哥](http://hackaday.com/2016/01/09/very-detailed-bb-8-robot-build/)。在*《原力觉醒》、*的预告片发布后不久，而且在我们知道 Sphero 机器人玩具[的制造商并不是这个英雄木偶](http://hackaday.com/2015/09/04/sphero-wasnt-actually-behind-the-bb-8/)的幕后黑手之前，詹姆斯就开始制造他最早的 BB-8。从那以后，驱动系统进行了许多改进，使第三版[詹姆斯]b b-8 能够像屏幕上的一样转动。

现在，[詹姆斯]已经完成了他最新机器人的 80%,只需要一点额外的电路就可以拥有“真正的”舞台机器人的所有功能。像外面的大多数 R2D2 建筑一样，这个机器人内部可能有足够的空间来容纳一些额外的功能。其中一个车身面板后面似乎有足够的空间来放置一只伸出的手臂，这使得火焰喷射器竖起大拇指的可能性非常真实。

[James]也是今年 Hackaday 奖的评委之一，并将(有希望)出席今年在旧金山举行的 Hackaday 奖颁奖典礼和 Hackaday 超级大会。如果一组极有可能的概率证明，[詹姆斯]b b-8 也将出现在展会上，我们将看到它沿着伦巴第街的一个奇怪街区倾斜。太棒了。
