# 令人印象深刻的 NFC 控制的 Infinity 镜台没有偷工减料

> 原文:[https://hack aday . com/2015/12/09/impressive-NFC-controlled-infinity-mirror-table-cuts-no-corners/](https://hackaday.com/2015/12/09/impressive-nfc-controlled-infinity-mirror-table-cuts-no-corners/)

如果你想给你的客厅增加一点未来感，你可能想看看这个[教程，来制作一个非常专业的 infinity 镜台。](https://hackaday.io/project/8651-nfc-infinity-table)

这是一款宜家 RAMVIK 咖啡桌，经过改造，加入了 RGB LEDs 和一面单向镜子，营造出令人惊叹的无限效果。从技术上来说，你只需要在桌子上开一个洞。

通过在玻璃下面放置一面大镜子，用一条 RGB LEDs 包裹内边缘，并在原始玻璃顶部涂上反光汽车色，这是一个非常简单的技巧，可以产生非常抛光的产品——这不是我们大多数项目可以做到的！但是更好的是，[Pierre]在桌子下面增加了一个 NFC 芯片，让你只需轻轻一点就可以控制颜色。

他使用的通过 NFC 控制 led 的设备叫做 KeyDuino，这是一个支持 NFC 的 Arduino，目前在 [Kickstarter 上运行良好。](https://www.kickstarter.com/projects/1535978353/keyduino)
