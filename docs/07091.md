# 抽屉柜储存音频记忆

> 原文:[https://hack aday . com/2017/09/18/五斗柜-商店-音频-记忆/](https://hackaday.com/2017/09/18/chest-of-drawers-stores-audio-memories/)

有些人收集邮票，有些人收集铁丝网，有些人甚至收集硅和塑料的小碎片。但是这个名叫[videoschmideo]的人收集记忆，大部分是他和妻子环游世界的记忆。小饰品和珍宝很容易跟踪，但他发现存储他收集的音频剪辑有点困难。直到他建了[这个音频记忆箱](https://www.instructables.com/id/Build-an-Audio-Memory-Chest/)，也就是。

当然，你可能不会收集像音频文件这样无形的东西，即使你是，Google Drive 或 Dropbox 似乎是更明智的存储它们的地方。但是明智的方法并不总是最好的方法，我们真的很喜欢这个想法。从一个看起来像旧卡片目录的文件开始——如果你曾经在图书馆贪婪地看过一个废弃的卡片目录，想知道它是否适合你的商店作为零件存储，请举手——[videoschmideo]配备了 16 个带有传感器的抽屉，当它们被打开时可以检测到。其中两个抽屉被扬声器格栅取代，一个 SD 卡存储所有的音频文件。当一个抽屉被打开时，当你在贝壳、明信片和其他东西中寻找的时候，记忆中的一个随机片段会被播放出来。使用老式打字机做抽屉标签和使用旧卡片目录卡做播放列表的额外加分。

这是一个简单的想法，但却是一个强大的想法，我们真的很喜欢这里的执行。这部电影设法同时让我们有心情进行世界旅行和去真正的图书馆旅行。
