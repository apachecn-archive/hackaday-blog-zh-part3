# 蛋糕机器人不是谎言

> 原文:[https://hackaday.com/2018/04/28/the-cake-robot-is-no-lie/](https://hackaday.com/2018/04/28/the-cake-robot-is-no-lie/)

[52 Skillz]对制造机器人一无所知。所以他决定不只是阅读或者制造一个简单的机器人。他一头扎进去，想要造一个会做蛋糕的机器人。这花了大约一年半的时间，但是现在，正如你在下面的视频中看到的，大部分时间都是有效的。

当然，它并不完美，也不太实际。但作为一次学习练习，这无疑是雄心勃勃的，也是成功的。显然，你仍然需要用手稍微刮一下碗，以便从碗壁上弄掉一些面粉。此外，加载配料可能比手工制作要多得多，但这真的不是重点。

据我们所知，没有任何复制机器人的具体计划。但是我们不认为这是一件坏事。你不会做一个真正的蛋糕。而如果只是照搬已有的设计，就会错过学习的过程，视频里有详细描述。如果你真的想走上同样的道路，请注意[52 Skillz]估计他投入了大约 300 个小时的努力。

然而，观看所有用来做这项工作的机制是很有趣的。这几乎就像一个鲁布·戈德堡机器，但科技含量更高一点。

如果我们有一个蛋糕机器人，我们认为在成品中加入电子产品是公平的。然后，你可以跳过蛋糕，去[全虚拟现实](https://hackaday.com/2015/10/07/the-arduino-birthday-cake-is-no-lie/)。好吧。也许不饱，但肯定热量低。
