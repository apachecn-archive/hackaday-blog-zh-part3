# 从相机中抢救出一个带有电池的专有电池组

> 原文:[https://hack aday . com/2017/07/11/rescuiting-a-proprietary-a-battery-pack-with-a-cell-from-a-camera/](https://hackaday.com/2017/07/11/rescuing-a-proprietary-battery-pack-with-a-cell-from-a-camera/)

如果你有一个旧的电池供电的手持设备，你可能会面临电池容量不断减少的问题，因为它的锂离子电池达到了使用寿命。如果你像[Foxx D'Gamma]一样，他的设备是 Alinco DJ-C7 手持收发器，你会面临完全缺乏替换电池组的情况。尽管如此，一切还没有结束，因为[正如他在休息](https://www.youtube.com/watch?v=6SCvLbkcbOQ)下面的视频中解释的，他注意到数码相机电池使用了非常相似大小的电池，并且能够将相机电池移植到 Alinco 包的外壳中。

拆开 Alinco 包装后，他得到了一个长方形的锂离子电池和两块印刷电路板，一块用于连接器，另一块用于电池管理电路。相比之下，相机电池有一个小得多的电池管理 PCB，它正好适合 Alinco 电池在这些盖子被移除后腾出的空间。连接连接器 PCB 是一项复杂的焊接工作，当他发现收发器是双频型号时，他得到了一个可用的 Alinco 包和一个意外的奖励。

一路上，他不厌其烦地指出处理锂离子电池的安全问题，并确保电池的极性正确。同样值得我们提醒读者的是，这些电池组必须始终伴随着它们的电池管理电路。尽管结果令人满意:一个被专利电池淘汰的多余设备获得了新生。
