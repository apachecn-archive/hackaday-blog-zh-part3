# 混凝土 3D 打印房屋

> 原文：<https://hackaday.com/2016/04/15/3d-printing-houses-from-concrete/>

我们以前见过 3D 打印的房子，但大多数都使用预制块。这座菲律宾的抗飓风和龙卷风酒店套房是一次性打印出来的。

听起来熟悉吗？这是安德烈·鲁登科(Andrey Rudenko)的作品，他从两年前在车库里建造混凝土 3D 打印机(T1)开始，到那年晚些时候在他的后院为他的孩子们 3D 打印混凝土城堡(T3)，现在似乎已经在 T4 拥有了一家成熟的公司(T5)，提供商业 3D 打印房屋。好样的[安德烈]！

建筑是在 Sketchup 中设计的，打印机使用 Pronterface 作为控制软件。看到这个全尺寸的建筑绝对令人着迷。我们想要一个。[](https://www.youtube.com/watch?v=zb3zuk1qNDk)

这个项目是由一个古怪的企业家促成的，他叫[Lewis Yakich]，曾经是英特尔的工程师，现在是菲律宾的房地产巨头。他得到了安德烈的帮助，在他原有的刘易斯大酒店(T1)上建造了这个新的 3D 打印扩建项目。但是他们的目标更高——制造更多的打印机，革新菲律宾的低收入住房市场。

 [https://www.youtube.com/embed/zb3zuk1qNDk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/zb3zuk1qNDk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



总而言之，他们希望能够在不到一周的时间内打印出这样的房子，与传统方法相比，可以节省大量成本。
