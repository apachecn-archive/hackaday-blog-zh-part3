# 模块化机器人可以把自己变成任何东西

> 原文：<https://hackaday.com/2018/06/24/modular-robotics-that-can-make-themselves-into-anything/>

机器人最大的挑战是自主性。通常，这意味着可以自动驾驶的汽车，不会从楼梯上开下来的机器人吸尘器，或者可以在火星上驾驶的火星漫游车。这个项目和那个项目完全不同。这个机器人不是建造一个单一形状的机器人，而是由几个模块组成，这些模块可以自我组装成不同的结构。这是一个有组织的机器人舰队，所有人都互相帮助，就像一个蚁群，或者我们的未来是灰色的粘性物质。

如果自我组装模块化机器人的想法听起来很熟悉，你是对的。[Dtto](https://hackaday.io/project/9976-dtto-explorer-modular-robot)获得了 2016 年 Hackaday 奖的[大奖，是一个项目的猛兽。这是一个机器人的大毒蛇，它可以把自己组装成一条蛇，一个轮子，或者一只手臂。这是*怪异的*，但是如果你想要一个能做任何事情的机器人，这就是你需要的那种模块化。至少离灰色粘性物质更近了一步。](https://hackaday.com/2016/11/05/dtto-explorer-modular-robot-wins-2016-hackaday-prize/)

像 Dtto 一样，游牧人可以通过将每个单独的部分组装成一个巨大结构的一个组件，将自己变成桥梁、手臂、蛇和轮子。这是我们很少看到的，这是一个困难的计算和工程问题。尽管如此，noMad 背后的团队已经取得了显著的进展，我们迫不及待地想看到完成的项目。
