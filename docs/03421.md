# 猫操作的猫粮分配器

> 原文：<https://hackaday.com/2016/09/01/cat-operated-cat-food-dispenser/>

厌倦了被你的猫骚扰？[MomWillBeProud]制造了一种便宜、有效——更重要的是由猫操作的——[猫粮分配器](https://www.youtube.com/watch?v=7DwUOdvJp-8)。

喂食器是一个高效的结构——一个双层猫粮盘，一个储存电子产品的容器，一个品客薯片罐作为漏斗。一个简单的伺服旋转料斗 30 度，并在每次按钮按下回来；利用重力使食物通过由于这种运动而出现的开口落下。这个按钮本身是一个旧的宜家定时器，也是一块大到足以让一只饥饿的猫拍打的塑料。

Arduino 控制着伺服系统，虽然[MomWillBeProud]跳过了他的代码的细节，但你可以在这里查看。

 [https://www.youtube.com/embed/7DwUOdvJp-8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/7DwUOdvJp-8?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



[MomWillBeProud]考虑过定时喂食的想法，但认为他的猫足够聪明，可以自己喂食；的确——训练猫是一项艰苦的工作，花了几个星期的时间才理解这些课程。他还建议使用 Arduino Pro Mini，因为它耗电很少，一组电池可以工作几个月。因为[MomWillBeProud]的猫一周只需要再添一次奶，所以没有那么多时间和猫作伴了。
