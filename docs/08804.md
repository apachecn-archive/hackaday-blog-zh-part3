# 为你的孩子找一个更快的掘墓人

> 原文：<https://hackaday.com/2018/03/10/a-faster-grave-digger-for-your-child/>

孩子们喜欢速度，但很少有电动玩具能提供这种速度。一个孩子能做什么？好吧，如果[PoppaFixit]是你爸爸，你很幸运。

这个项目从一个非常酷的电动轮玩具开始，基于著名的掘墓人怪物卡车。在修改过程中，人们很快意识到原来的电机控制器是不会切芥末。只有基本的开/关控制，它给了一个非常不平稳的乘坐，对传输组件也很苛刻。[PoppaFixit]决定升级到现成的 24 V 电机控制器，使汽车更精细，速度更快。控制器配备了一套替换踏板，包括油门和刹车，以取代股票单位。在发动机方面，几个更强大的 Traxxas 单位取代了杂草原始。

加速度现在有了很大的提高，不仅仅是因为增加了动力，还因为可变油门使驾驶员能够避免在艰难起步时打滑。由于增加了可控性，它还使汽车驾驶起来更加舒适和安全。另一种表明项目成功的方式是新主人脸上纯粹喜悦的表情！

这是一个相当基础的安装，新手很容易上手。这种电动汽车充电站非常普遍，有各种各样的供应商出售易于使用的工具来完成这种工作。出于这个原因，我们已经看到了很多这种类型的黑客——[像这种改装的滑板车](https://hackaday.com/2016/03/02/converting-an-electric-scooter-to-lithium-batteries-and-disabling-safeties/)，或者这些[为赛车设置的动力轮](https://hackaday.com/2015/09/06/better-racing-with-power-wheels/)。
