# 热电风扇收获木材炉热垃圾场风格

> 原文：<https://hackaday.com/2018/04/27/thermoelectric-fan-harvests-wood-stove-heat-junkyard-style/>

用柴炉取暖的人都知道，这种体验和典型的集中供暖完全不同。然而，它并不适合所有人，当然也不是没有利弊。一个主要的抱怨是热量从炉子散发到房子的其他地方，许多业主打开电风扇循环热空气。

然而，这几乎不符合木材加热的绿色本质，而且风扇可能会很吵。所以像这种热能驱动的炉顶风扇可以派上用场。这种使用珀耳帖效应装置驱动小型电动机的风扇在市场上很容易买到。[bongodrummer]认为这听起来一点也不好玩，他主要用垃圾创作了自己的作品。珀耳帖模块是从一个旧的旅行冰箱中抢救出来的，并安装在一台计算机的散热器上，以收集炉子的热量。珀耳帖的另一面需要有一个散热器，以保持它比热的一面冷，而[bongodrummer]选择了一个非常规的补救措施——链锯引擎的气缸。火花塞孔为风扇电机安装了支架，散热片有助于保持珀耳帖冷却。为了防止设备过热，他增加了一个惊喜——一个汽车冷却系统恒温器，当太热时，它会从炉子上物理地提起设备。天才！

下面的视频展示了构建，这不是微不足道的。但我们认为最终结果是值得的，这让我们想起了不久前我们推出的柴炉发电机。
