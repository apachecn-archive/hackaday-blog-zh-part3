# 有了这个鞭炮发射器，你的手指永远不会离开你的手

> 原文：<https://hackaday.com/2016/01/05/your-fingers-will-never-leave-your-hands-with-this-firecracker-launcher/>

谁不喜欢用炸药来敲响新年的钟声？但是从个人经验来说——我既不能确认也不能否认有一次差点被一枚小型干冰手榴弹炸掉我的手——不用手的方式来发射烟花可能是一个优势，在这种情况下，你可能想试试这个自动鞭炮发射器。

[瓦伦丁]的构建具有一个灵感迸发的下午仓促入侵的所有特征。大部分由木头和热胶水建造，有一个新弹药的进料坡道和一个坡道上的弹性动力雪橇。通过一个小齿轮马达的一转，烟花被计量到雪橇上，保险丝被丁烷火炬点燃，另一个齿轮马达将雪橇拉回并发射鞭炮。这次发射有点乏力——也许一些结实的橡皮筋或乳胶管会提供更多的活力。但它仍然是一个有趣的构建，有很大的改进潜力——也许是沿着[这个自动化啤酒弹射器](http://hackaday.com/2011/01/10/beer-catapulting-fridge/)的路线？
