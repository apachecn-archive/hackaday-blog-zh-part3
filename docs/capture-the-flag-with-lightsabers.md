# 用光剑夺取旗帜

> 原文：<https://hackaday.com/2015/12/15/capture-the-flag-with-lightsabers/>

HITCON 每年都会举办一场精彩的夺旗游戏。不过，这可不是你童年时邻居在森林里用真正的旗子夺旗。在这个游戏中，旗帜在安全的服务器上，另一队的任务是以任何方式闯入服务器并夺取旗帜。然而今年，游戏的创造者们设计了一个新的记分牌来记录游戏进度:一把光剑。

在这个特殊的游戏中，每个队都有一个他们必须保卫的服务器。与此同时，每个小组都试图访问对方的服务器。这个项目使用了一个光剑支架，可以将光剑变成 2015 年[台湾黑客大会](http://ctf.hitcon.org)比赛的记分牌。它使用廉价的 OpenWRT Linux Wi-Fi/以太网开发板 LinkIt Smart 7688 与服务器通信。每当得分时，光剑就会亮起，并播放音效。光剑本身来自台湾的光剑匠，本身就是令人印象深刻的技术。作为奖励，各队将把它们带回家。

虽然我们怀疑这是迪士尼更强迫的产品整合广告，但它确实符合游戏的主题。像这样的夺旗比赛是了解网络安全以及如何保护自己的设备免受真实攻击的好方法。如果你想参与其中，世界各地还有其他游戏在进行。
