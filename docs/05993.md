# 人类可能意外地在地球周围制造了一个辐射屏蔽

> 原文：<https://hackaday.com/2017/05/21/humans-may-have-accidentally-created-a-radiation-shield-around-earth/>

美国宇航局花费大量时间研究地球及其周围的空间环境。一个特别有趣的特征是范艾伦带，以至于美国宇航局建造了特殊的探测器来研究它们！他们现在发现了一个保护性气泡，他们认为这是由人类在甚低频范围内的传播产生的。

[VLF 传输](https://en.wikipedia.org/wiki/Very_low_frequency)覆盖 3-30 kHz 范围，因此带宽非常有限。甚低频硬件主要用于与潜艇通信，经常提醒他们，是的，一切都很好，还没有必要发射核武器。它也用于导航和广播时间信号。

似乎这种人类传播在大气中创造了一种屏障，保护它免受来自太空的辐射。有趣的是，这个“甚低频泡”的向外边缘似乎与地球磁场引起的范艾伦带的最内边缘非常接近。此外，与 20 世纪 60 年代相比，范艾伦辐射带的内部界限现在似乎离地球表面更远了，这表明人造甚低频传输可能是将边界向外推的原因。

总体而言，这似乎是人类活动的一个偶然但潜在的积极影响——屏障保护地球免受潜在的有害辐射。美国宇航局在 YouTube 上发布的主题为的视频表明，更好地理解这种机制可以使我们保护我们的卫星和航天器免受太空环境的一些有害影响。

美国国家航空航天局做了很多高端研究，比如现在让很多人非常困惑的 EM 驱动器。
