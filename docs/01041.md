# 小助手:开源硬件黑客 Multitool

> 原文:[https://hack aday . com/2015/12/25/little-helper-开源-硬件-黑客-多工具/](https://hackaday.com/2015/12/25/little-helper-open-source-hardware-hacker-multitool/)

我们喜欢好的多功能工具。知道如果出于某种原因，你需要马上锯倒一棵树，你的口袋里会有一把小锯子，这很有诱惑力。我们也喜欢电子版的多功能工具:在你开发和调试硬件的时候，这些小工具有很多用途。我们见过的最漂亮的一部是菲利普·舒斯特的《小助手》。

开源小工具看起来像一个 iPod(如果一个 iPod 有突出的插头的话)。它具有基本的模拟 I/O 能力，可以产生 PWM 脉冲，嗅探 I2C 流量，并做许多其他功能。它是开源的，所以如果你需要的话，你可以随时添加更多的功能。

即使你不关心小助手的功能，它也是一个很好的研究，可以让你的设备看起来像一个具有相似界面的 iPod。长久以来，类似的设计层出不穷。例如，[super probe](http://hackaday.com/2010/06/04/superprobe/)使用一个 PIC 和一个 LED 显示屏来制作一个多功能探头(你可以在下面的视频中看到一个构建在原型板上的探头)。[TIQ](http://hackaday.com/2014/05/29/tiq-probe-is-more-logical-than-most/)是另一个类似的设备(这导致了这里很多淘气的产卵双关语)。Microchip 曾经发布过一篇应用笔记(AN689 ),内容是一款类似的设备，名为“工程师助手”,不过今天你必须为它寻找一个替代来源，并且可能必须使用更流行的元件。
