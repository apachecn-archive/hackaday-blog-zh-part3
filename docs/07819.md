# 箱子，在扶手椅后面排成整齐的队列！

> 原文:[https://hack aday . com/2017/12/03/boxes-form-an-ordered-queue-behind-the-armchair/](https://hackaday.com/2017/12/03/boxes-form-an-orderly-queue-behind-the-armchair/)

如果你曾经去过黑客训练营，你会知道把你所有的东西运送到你的黑客村，或者运送到你在此期间的任何地方的问题。停车场总是太远，无论你带来的手推车都不够大，你不得不拖着所有东西过去的地形感觉更像奇尔库特小径，而不是城市人行道。

[Jana Marie Hemsing]和[Lucy Fauth]有一个有效的解决方案来解决你所有的黑客营地运输困境，其形式是[一个旨在携带存储箱](https://wiki.warpzone.ms/projekte:transpotter:start)的电动平台。平台下面是一对悬浮板电机，它们的控制板用定制固件刷新。

你现在可能会看着它，心想“那又怎样？”，因为单个平台很方便，但很难成为全面的运输解决方案。令人印象深刻的是，这不是一个单一的董事会，而是有一大群人，他们似乎已经实现了某种形式的光学跟踪系统，这是通过我们放在休息时间下方的视频和这条推文来取笑的，但在维基页面上还没有详细说明。一列整齐的站台跟在前面，以最小的忙乱运送一切。除了“我们也想要一个！”。如果您有兴趣亲自尝试，可以在 GitHub 资源库中找到一些代码。
