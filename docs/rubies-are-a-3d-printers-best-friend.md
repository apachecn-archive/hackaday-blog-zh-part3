# 红宝石是 3D 打印机最好的朋友

> 原文：<https://hackaday.com/2017/11/30/rubies-are-a-3d-printers-best-friend/>

观看 3D 打印机工作总是让我们想起观看面包师装饰蛋糕。粘稠的糖霜从喷嘴中挤出，形成有趣的形状和设计。虽然热塑料的味道不如糖霜，但它确实很容易通过打印机的喷嘴。嗯……反正是普通塑料。如今，先进的 3D 打印机正在使用含有木材、金属、碳纤维和其他添加剂的细丝。这些可以提供令人印象深刻的结果，但其中的硬物质往往会磨损金属喷嘴。如果这是你的问题，你厌倦了更换喷嘴，你应该检查一下[奥尔森红宝石喷嘴](http://olssonruby.com/the-olsson-ruby/)。

在这种情况下，Ruby 不仅仅是一个名字。喷嘴有一小块红宝石，中间有一个 0.4 毫米的孔——或者他们有一些其他尺寸。我们认为钻石会更好，但红宝石更实惠。我们自己没有尝试过这些，但是[3D 打印呆子]有一个有趣的视频评论，你可以在下面看到。

我们假设红宝石——只是氧化铝——是实验室制造的，而不是天然的石头。另一个常见的选择是硬化钢喷嘴，但奥尔森声称黄铜喷嘴导热更好，这使他们的产品具有优势。[Nerd]展示了红宝石喷嘴如何在用碳纤维丝打印后打印 PLA，而没有不良影响。这种喷嘴就像普通的 E3D 喷嘴一样合适，所以它们可以用在很多打印机上。

大约 100 美元，这些都不便宜。正如[书呆子]所指出的，这可能是你买的最后一个喷嘴。硬钢喷嘴大约 25 美元，但如果你用硬丝打印，你会买不止一个。如果你买四个，你还不如得到红宝石喷嘴，享受更好的性能，以及。另一方面，如果喷嘴的价格是你购买打印机价格的 80%,而且你没有打印特殊的塑料，这可能是大材小用。你可以花 100 美元在易贝买到很多便宜的黄铜喷嘴。

如果你想知道用红宝石喷嘴做什么，我们之前讨论过很多奇异灯丝的比较。当然，喷嘴只是等式的一部分。当我们谈到碳纤维丝时，我们注意到它也能[切入你的挤压机齿轮](https://hackaday.com/2016/09/07/3d-printering-xt-cf20-carbon-fiber-filament-review/)。
