# Hackaday 奖参赛作品:发明新的逻辑家族

> 原文：<https://hackaday.com/2016/05/30/hackaday-prize-entries-inventing-new-logic-families/>

电子爱好者最喜欢的消遣之一是制作时钟。时钟是一个有明确目标的简单概念，但重要的是实现。如果你想建造一个仅由电子管和电源供电的时钟，这是一个很好的技能测试。基于继电器的时钟同样很酷，每个人一生都应该建造一个数码管时钟。

为了[Ted]的 Hackaday 奖参赛作品，他正在建造一个时钟。通常，这没什么值得庆祝的，但这不像你见过的任何时钟。[Ted]正在建造这个只用二极管的时钟，[他正在发明新的逻辑系列来做这件事](https://hackaday.io/project/11677-the-diode-clock)。

从第一台计算机开始，使用二极管作为逻辑元件就已经存在了，但是这些计算机内置了一些晶体管。虽然只使用二极管就能制造“与”和“或”门，但通用逻辑门“与非”和“或非”是不可能的。对于 20 世纪 50 年代的计算机来说，这意味着电子管或晶体管以及 DTL 逻辑。

在过去的几年里，[Ted]一直在研究一个只有二极管的逻辑系列，看起来他已经解决了这个问题。新的逻辑系列包括一个仅使用二极管、电阻和电感构建的或非门。这些门的主要设计特点是用一个二极管来开关 RF 电源。它依赖于二极管的一个未记录的属性，但它确实有效。

虽然[Ted]可以创造一个没有晶体管的或非门——这是电子历史上从未记载过的壮举——但这并不意味着这是晶体管逻辑的有用替代品。门的扇出很糟糕，时钟使用大约 60 瓦，门需要交流电源。虽然从理论上讲，用这些门来建造一台计算机是可能的，但是否有人有耐心这样做却令人怀疑。这更多的是一种好奇，但它是我们见过的最令人难以置信的项目之一的展示。
