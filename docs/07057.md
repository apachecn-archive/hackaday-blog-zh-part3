# Hackaday 奖的参赛作品:追踪房间的红藤蔓弗林格

> 原文:[https://hack aday . com/2017/09/13/hack aday-prize-entry-room-tracking-red-vines-flinger/](https://hackaday.com/2017/09/13/hackaday-prize-entry-room-tracking-red-vines-flinger/)

[Vije Miller]的 [Arduino 甘草发射器](https://hackaday.io/project/12893-arduino-licorice-launcher)是基于一个简单而符合逻辑的前提，即一个人必须在车间里始终有一个声控红藤弹射器。当他呼唤机器人时，它会转向他并朝他的头扔一块甘草糖。

底盘由四分之一英寸的 MDF 制成，弹簧弹射器臂由两个伺服系统管理，一个用于张紧臂，一个用于固定臂，直到它被触发。第三和第四伺服瞄准弹射器，并从弹匣中分发另一片甘草。他的机器人采用了一种从 RoboWarner 那里学来的[无线电导航技术](http://robowarner.com/portfolio/radio-homing-robot-diy/)【Vije】，这种技术允许机器人追踪移动的射频信号。

[Vije]的第一个原型使用了一个连接到 PC 上串行端口的 Arduino Uno，但他希望获得一个 MKR1000 WiFi 模块，该模块将 Arduino Zero 与 WiFi 结合在一起。这个红色藤蔓发射器已经取得了圆满成功；Red Vines 的营销团队送给他一大堆礼品和免费甘草，以感谢他的努力。你可以在下面查看[Vije]的项目宣传片。
