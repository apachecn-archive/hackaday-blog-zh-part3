# 个人对抗现代笔记本电脑

> 原文:[https://hack aday . com/2017/01/26/a-personal-fight-the-modern-laptop/](https://hackaday.com/2017/01/26/a-personal-fight-against-the-modern-laptop/)

如果你最近没有去买过笔记本电脑，当你去的时候，你会大吃一惊。虽然当前一代的 MacBook Pros 被认为是一台价格过高的机器，配有一个愚蠢的触摸栏，但整个行业的笔记本电脑都存在问题。没有人知道如何使用高分辨率的 iPad 屏幕并添加键盘，大多数显示器小于 13 英寸的笔记本电脑的分辨率上限为 720，新功能的引入是以旧功能为代价的，二进制 blobs 被投射到 BIOS 白名单和缺失驱动程序的网络中，不，微软 Surface 不算，因为尽管它是一台不错的机器，但它是一台带键盘的平板电脑，而不是笔记本电脑。

经过几个月的搜索，哈米什·科尔曼找到了最接近完美笔记本电脑的东西。这是很久以前的 Thinkpad X230，或者是 2012 年的，看你怎么算了。不过，它接近完美:除了旧的 CPU 和 GPU，唯一真正的阻碍是键盘。更换键盘是[Hamish]与现代笔记本电脑的个人斗争(YouTube，嵌入下面)，他也让我们更容易与当前的垃圾电脑作斗争。

自第一台 Thinkpad 推出以来，这些机器的键盘一直保持相对不变，直到 2011 年。对于完美的 Thinkpad 运动专业人士来说，这是二十年的肌肉记忆，被奇怪的键、删除的键和完全错误位置的 Page Up 和 Page Down 按钮所取代。X220 的键盘适合 Thinkpad X230，便于机械更换，但固件就是不工作。

经过一些逆向工程，[哈米什]设法让这个旧键盘工作。当然，这并不新鲜；去年的这个时候，Zmatt [为他的 Thinkpad 键盘解锁了控制器。[Hamish]更进一步，开发了自己的工具来解锁比 Thinkpad X230 更新的笔记本电脑。](http://zmatt.net/unlocking-my-lenovo-laptop-part-1/)[GitHub repo](https://github.com/hamishcoleman/thinkpad-ec)，现在解锁整个 Thinkpad xx30 系列，xx50 和 xx60 系列的一些工作正在进行中。

在过去的一年里，我们已经看到了许多涉及 Thinkpad X220 和 X230 笔记本电脑的黑客攻击。芯片电池[现在没有芯片](http://hackaday.com/2016/02/11/unlocking-thinkpad-batteries/)，标准的 1366×768 分辨率[可以提升到 1080](http://hackaday.com/2016/08/26/making-a-thinkpad-great-again/) ，我们可以在这些机器上摆脱[可怕的英特尔管理引擎](http://hackaday.com/2016/12/16/installing-libreboot/)。对于一台使用了六年的笔记本电脑来说，这些旧的 Thinkpads 正在成为任何关心免费硬件的人的完美机器。啤酒和演讲都是免费的——你可以很便宜地买到一台这样的机器。
