# 本周失败:廉价芯片引发混乱

> 原文：<https://hackaday.com/2017/11/22/fail-of-the-week-cheap-chips-cause-chaos/>

我们都知道一句老话:如果好得不像真的，那很可能就是真的。但是这条规则似乎没有像我们订购零件时那样经常被打破。邦古德、全球速卖通和易贝充斥着随时可以神奇地从全球各地运到我们家门口的零部件，所有这些零部件的价格似乎都违背了经济规律。

这些交易中的大部分都进行得很顺利，我们得到了完成下一件很酷的事情所需要的东西。但是事情并不总是那么顺利，正如[Kerry Wong]最近发现的，易贝的一个订单导致了一些可疑的芯片。【Kerry】订购了 AD633 模拟乘法器芯片，作为[最近的 Lorenz 吸引子 X-Y 记录器项目](http://hackaday.com/2017/05/11/vintage-plotter-handles-chaos-with-ease/)的后续项目，在该项目中，他使用 Arduino 生成混沌蝴蝶的数据集，作为老式仪器的演示。在评论中被质疑用模拟再做一次时，[Kerry]做了功课，找到了一个电路来实现它。DigiKey 需要的乘数是 10 美元一个，所以他从易贝采购更便宜的芯片。2 美元的芯片似乎是合法的，有 ADI 公司的标志和一切，但电路不工作。[Kerry]下面视频中的诊断很有意思，很明显芯片是假货。顾客小心上当。

希望[Kerry]能尽快找到好的筹码，用一次成功的构建来款待我们。在那之前，你有哪些廉价芯片的体验？之前有没有被海外或者国内的供应商烫过？对你来说，是否有任何一家供应商看起来更好，或者这一切都是偶然的？请在下面的评论中发表意见。
