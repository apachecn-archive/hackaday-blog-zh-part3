# 游戏机版游戏装备

> 原文：<https://hackaday.com/2017/06/02/game-gear-console-edition/>

如果游戏装备是一个控制台系统会怎么样？[Bentika]通过制作这款经典掌上电脑的舒适版来回答这个问题。对于那些不了解 20 世纪 80 年代世嘉游戏机的人来说，游戏设备在技术上与主系统非常相似。事实上，游戏装备甚至可以通过第三方适配器玩主系统游戏。然而，相反的情况并非如此，因为屏幕纵横比不同，游戏装备有更大的调色板，这意味着主系统与游戏装备不兼容。

世嘉决定省略 AV 连接，这意味着游戏装备游戏永远被锁定在一个小小的液晶屏幕上。[EvilTim]用他的 AV 板改变了这一点，所以[Bentika]决定通过建立一个适当的游戏设备控制台版本来自然地结束事情。

他开始抛弃屏幕，在[EvilTim 的]视频适配器板上布线。然后将墨盒插槽移除并重新连接到 PCB 顶部。这使得系统变成了顶部装载机。[本提卡]然后开始着手这个案子。他使用 Bondo 来填充 d-pad 和按钮的孔。一次喷漆失败后，[Bentika]又回到了绘图板。他在一家家用油漆店买到了与原版游戏装备灰色相匹配的油漆颜色。仔细的打底、打磨和上漆使得这个经典的造型更加完美。休息之后看看[Bentika 的]视频！
