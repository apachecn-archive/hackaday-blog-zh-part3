# 黑客蜡烛熄灭

> 原文：<https://hackaday.com/2016/02/15/hacking-candle-extinguishing/>

任何人都可以吹灭蜡烛。根据[物理女孩]的说法，那种方法已经过时了。她制作了一个教育视频，展示了用物理学熄灭蜡烛的五种不同方法。

你可能不需要其他熄灭蜡烛的方法，但如果你想让学生参与 STEM(科学、技术、工程和数学)，这个视频和[物理女孩]的其他视频可能会引发兴趣。

该视频展示了有趣的演示，涵盖热力学、流体动力学和其他科学主题。剧透提示:最后一种方法是用液氮熄灭火焰。但在那之后继续看一个小插曲，她制造了一点混乱，几乎烧毁了她的工作室。

一旦你开始用蜡烛做科学研究，你可能会转向热电发电机，或者你可以做一些 T2 的更高级的研究。
