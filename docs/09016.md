# 逆向工程瓶螺纹的乐趣和利润

> 原文：<https://hackaday.com/2018/04/01/reverse-engineering-bottle-threads-for-fun-and-profit/>

最近，[Eric]的一个客户请他设计一个瓶子。对于产品设计师来说，这很简单，除了客户需要它穿进特定类型的盖子。不，他们不知道规格。

但这没问题，[埃里克]边想边打开排气扇，伸手去拿秘密成分，这将使[轻而易举地铸造出丝线](https://www.youtube.com/watch?v=tVoTXDnZZ3o)的负像。他将恶臭的身体填充物与必要的硬化剂和一些可爱的青色碳粉混合，并用压舌板将其装入瓶盖。然后他在石膏上加了一个小的聚氯乙烯项圈来加长石膏，这样当他要取出石膏时就有东西可以抓住了。

Bondo 看起来确实是铸造螺纹的好选择。你需要一种足够好用的东西，可以从那里拧出来而不会断裂，但又足够坚硬，不会丢失螺纹的小细节。对于脱模剂，[Eric]使用了 Johnson 的膏状蜡。他从经验中注意到，它对 Bondo 特别有效，甚至似乎有助于治愈。

一旦 Bondo 硬化，[Eric]确保它旋入旋出瓶盖，然后继续进行 CAD 建模和 3D 打印瓶子原型，直到他满意为止。休息之后我们会把视频放进去。

你知道吗，你还可以用墨粉给你的环氧树脂上色。记住是颗粒物就行了，做好防范。
