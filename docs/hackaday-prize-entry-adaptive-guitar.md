# Hackaday 奖参赛作品:适应性吉他

> 原文：<https://hackaday.com/2017/09/26/hackaday-prize-entry-adaptive-guitar/>

由于一次滑雪事故，[乔]的新朋友切断了控制她左臂的运动神经。可悲的是，她是一个狂热的音乐家，喜欢弹吉他——当然，吉他需要两只手。是吗？用手指按压琴弦来弹奏复杂的和弦更容易，但是弹奏琴弦可以在脚踏板的控制下机电地完成。至少，当他朋友的家人伸出援手时，这就是[Joe] [实施得如此漂亮的解决方案。](https://hackaday.io/project/26088-adaptive-guitar)

在阅读[Joe]的 hackaday.io 页面上的[项目日志时，有太多的事情可以享受，他已经进入了 hackaday 奖。他从研究别人如何解决这个问题开始。然后他带我们经历了他的第一次尝试和实验。例如，一个早期的发现是按压指板上的琴弦如何将琴弦拉下拨片所在的位置，这导致他重新思考他最初的拨片设计。他对选择致动器的标准引导他自己制造。他制作的致动器非常漂亮:安静、紧凑，致动器本体甚至可以兼作定制控制板的散热器。在他的拨片设计迭代中，他使用弹簧钢获得了很好的结果，使拨片更加灵活，但考虑到有一天会投入生产，他提出了自己定制设计的激光切割板簧，每根琴弦都不同。脚踏板需要](https://hackaday.io/project/26088/logs?sort=oldest)[力敏电阻(FCR)](https://hackaday.com/2016/09/16/resistance-in-motion-everything-you-should-know-about-variable-resistors/)，他迭代制作自己的，在 PCB 上铺设所需的互连迹线(使用 Eagle script)并在上面放一块导电橡胶。这只是他带领我们冒险的一个例子。

就实用性而言，他尽了最大努力使其紧凑并易于安装。脚踏板甚至可以与吉他上的控制板无线对话。非破坏性的粘合剂将磁铁和 velcro 附着在吉他上，这样控制板和琴桥就可以精确而容易地单手固定。结果是，无论是设置还是实际演奏，只需一只手就能轻松操作。下面的视频自己看吧。
