# 激光旋转适配器让你滚动

> 原文:[https://hack aday . com/2018/03/01/laser-rotary-adapter-gets-you-rolling/](https://hackaday.com/2018/03/01/laser-rotary-adapter-gets-you-rolling/)

随着海外进口激光切割机变得越来越容易在车库里买到，但我们中的许多人仍然需要到大学校园或空间来获得我们的切割。现场有一台激光器是一种不错的体验，但有一个旋转轴几乎是闻所未闻的。这些漂亮的附件使你的激光切割和雕刻径向对称的部分。它们的价格标签通常在数百到数千美元的价格范围内，所以尽管这可能会阻止我们，但没有什么能阻止我们建造自己的产品！

这正是[ [塞萨尔](https://www.youtube.com/watch?v=4QUU8ll3pMA) ]和[ [鲁斯](https://www.youtube.com/watch?v=-uafUAX04G4) ]对两个从废料中构建的自制设计所做的，结果看起来与专业默认相当。这个设计本身很简单，但是非常聪明。托架直接固定在 x 轴上，因此其运动与 x 轴紧密相连。底部的轮子起着双重作用。首先，他们让托架随着 y 轴运动平稳滑动。它们还支撑待雕刻的物体，并将轮子的旋转从 y 轴运动转换成物体的旋转。这里有一个缺点，即待雕刻物体的直径会影响旋转角度，但我们从未羞于用θ = s/r 做一点工作。

[塞萨尔]因将这一黑客技术公之于众而受到称赞，但[罗斯]也因发布了一份他的马车的可下载文件而受到了极大的感谢。这证明了分享一个想法如何能够激励我们迭代出更好的设计，让全世界都能欣赏。

滚动第四轴在这些页面上不是什么新鲜事，但它们确实很少见！如果你渴望更多滚动轴的好处，看看[佩里的] [路由器修改](https://hackaday.com/2017/09/03/rolling-around-a-4th-axis/)。
