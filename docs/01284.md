# Link Trucker 是一家小型网络巨头

> 原文:[https://hack aday . com/2016/01/22/link-trucker-is-a-tiny-networking-giant/](https://hackaday.com/2016/01/22/link-trucker-is-a-tiny-networking-giant/)

如果你是一个网络专家，有专业的工具可以验证以太网电缆的商业端是否一切正常。这些专业工具通常带有专业的价格标签。如果你只是想连接一个办公室，专业设备可能会有些过头。除非你自己做的便宜！现在你可以了。

[Kristopher Marciniak] [设计并制造了一种廉价的设备](https://hackaday.io/project/9229-linktrucker)来验证基本原理:

*   链接打开了吗？这根电缆接好了吗？
*   它能得到 DHCP 地址吗？
*   它能执行 DNS 查找吗？
*   它能打开网页吗？

引擎盖下到底发生了什么？你可能会认为是覆盆子酱。猎兔犬板？看到一个更文明时代的回归，我们的心被温暖了:一个 ENC28J60 分线板和一个 Arduino Uno。没错，[克里斯多佛]用几杯拿铁的价格复制了一个几百美元的网络测试器。通过使用预制的外壳，[克里斯托弗]的版本看起来也很棒。在休息时间下方的视频中观察它的工作情况。

构建一个嵌入式网络设备过去需要做更多的工作，但现在已经可以完成了。我们最喜欢的产品之一仍然是 2008 年的名片上的网络服务器，它也使用了 ENC28J60 的以太网芯片。
