# 终极拼图桌——MYST 吃了你的心

> 原文：<https://hackaday.com/2015/12/25/the-ultimate-puzzle-desk-myst-eat-your-heart-out/>

这个项目绝对令人难以置信。这张桌子由【Kagen Sound】设计，将会非常适合 MYST 奥运会。它里面有一个管风琴，你可以通过演奏特定的曲子来打开一个秘密的隔间。

除了是绝对的木工奇迹之外，这张桌子内部隐藏的机制让我们头疼。管风琴方面通过推动小抽屉来工作——这迫使空气进入桌子前面的管风琴管。但是一些空气被重新导入一个气动记忆板——它实际上可以跟踪你弹奏的音符。当正确的曲调响起时，它会触发一个钟摆，释放出一个秘密的隔间。你所需要的只是深渊上方的一个活板门，你已经重现了《七宝奇谋》中的*。*

 *当然，用 Arduino 之类的很容易做到…但是气动记忆板是木头做的。完全由实木制成。[Kagen]说，他花了无数个小时来设计，至少五个不同的版本，才找到一个有效的版本。

是的，它可以被重新编程…但是要花一整天的时间。除了管风琴，这张桌子几乎每个抽屉里都有字谜。老实说，它是为一个非常非常有钱的人定制的——我们对这种东西的价格非常好奇！
