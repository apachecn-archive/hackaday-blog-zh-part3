# 升级版的下一代可穿戴设备

> 原文：<https://hackaday.com/2017/04/10/souped-up-next-gen-wearables/>

可穿戴技术取得巨大进步的最大障碍是人体本身。首先，这上面没有一条直线。加上所有的调整和努力，你就有了一个相当困难的 [![](img/c4410dff70edc981f015d95ad0bf6f9a.png)](https://hackaday.com/wp-content/uploads/2017/04/cream-of-conductivity.gif) 创新平台。对于可穿戴设备来说，时代在变。虽然没有老汤答案，[老汤](https://www.engadget.com/2017/03/13/scientists-make-stretchy-electronics-using-a-soup-ingredient/)里有一些答案。

斯坦福大学鲍实验室的一组科学家创造了一种具有良好导电性的耳语薄共聚物。没错，他们把两种不同的绝缘体放在一起，创造了导体。唯一的问题是制成的材料非常坚硬。在一些花哨的 x 射线设备的帮助下，他们发现，加入一种在标准工业浓汤增稠剂中发现的分子可以阻止聚合物的结晶过程，使它们变得柔韧和有弹性。听着:这种材料拉伸后导电性能更好。

科学家们已经使用这种材料制作了简单的透明电极，以及带有喷墨打印机的整个柔性晶体管阵列。他们希望影响从智能服装到医疗设备的下一代可穿戴技术。谁知道呢，也许他们可以与罗切斯特大学合作，创造出一种可以变形的导电共聚物。休息之后，请观看简短的演示。
