# Hackaday 奖参赛作品:Raspberry Pi Zero

> 原文：<https://hackaday.com/2016/10/02/hackaday-prize-entry-fpgas-for-the-raspberry-pi-zero/>

树莓 Pi 是 2016 年的 Arduino，这意味着盾牌、帽子、附件和其他有趣的玩具可以直接插入 Pi 的 GPIO 引脚。对于今年的 Hackaday 奖，[Valentin]将圆周率与下一个自制计算时代结合起来。[他开发了跳蚤欧姆](https://hackaday.io/project/13048-flea-ohm-fpga-project)，这是一个为圆周率零点设计的 FPGA 背包或帽子。

跳蚤欧姆基于 [Lattice 的 ECP5 FPGA](http://www.latticesemi.com/ecp5) ，具有 24k LUTs 和 112kB BRAM。对于一些相对有趣的应用程序来说，这已经足够了，但真正的乐趣来自于添加的 32MB 或 128 MB SDRAM，一个 micro SD 卡插槽，USB + PS/2 主机端口和一个 LVDS 输出。

Raspberry Pis 和 FPGA 的结合非常有趣，似乎是任何人都可以想象的最好的 FPGA 学习平台之一。另一个 Hackaday 奖项的参赛作品是 ZinqBerry 做了一个类似的把戏，但 ZinqBerry 没有戴 Pi 帽子，而是将一个带有 FPGA 和 ARM Cortex A9 内核的 Xilinx Zynq 放在一个带有以太网、HDMI 和 USB 的板上。

如果是跳蚤或 Zinq，FPGA 的覆盆子 Pis 时代正在迅速到来，希望我们能看到它们入围 Hackaday 奖。你可以看看下面跳蚤欧姆的视频。
