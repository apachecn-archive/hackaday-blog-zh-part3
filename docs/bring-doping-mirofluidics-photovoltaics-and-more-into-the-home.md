# 把兴奋剂、微流体、光伏等等带回家

> 原文：<https://hackaday.com/2016/09/10/bring-doping-mirofluidics-photovoltaics-and-more-into-the-home/>

你能用你手头所有的材料为你的家庭实验室制作一个分光计吗？我们可能不相信这个不太可信的来源，但是这个麻省理工学院的课程确实用泡沫板建造了一个分光计，用两个刀片作为淤泥覆盖层，用一个可写 CD 作为衍射光栅。最酷的部分是移除 CD 的金属背衬。

Hackaday reader [gratian]向我们透露了麻省理工学院名为“纳米制造商”的课件中的课程。它将一些相当复杂的实验归结为一种可以在家庭实验室中进行的实验，而不涉及数千美元的实验室设备。整个重点是揭开我们认为的复杂器件和围绕光伏、有机光伏、压电和热电的主题的神秘面纱。

分光计用于分析光源的波长。现在你已经有了一个测量工具，是时候建造和试验一些你自己的光源了。这里您可以看到一个 LED，这是其中一个课程实验的主题。

如果你有一点化学背景，这是一个很好的分步指南，让你在家里进行这类实验。这让我们想起了杰里·埃尔斯沃斯在她的车库实验室里做的一些非常酷的事情，比如制作自己的 EL 面板。
