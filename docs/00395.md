# 将 3D 打印机从 3 毫米转换为 1.75 毫米

> 原文：<https://hackaday.com/2015/10/17/converting-a-3d-printer-from-3mm-to-1-75mm/>

几周前，我发表了一篇帖子[讨论 3d 打印中常见的细丝直径](http://hackaday.com/2015/09/29/3d-printing-has-evolved-two-filament-standards)。不知何故，消费级 3D 打印机选择了两种不同尺寸的灯丝。是的，存在差异，但这些差异只是工程权衡和历史选择的函数。YouTube 的 3D 打印大师托马斯(Thomas)将这篇文章视为一个挑战:如何让[转换打印机以接受不同尺寸的灯丝](https://www.youtube.com/watch?v=UcMw-3RizIw)？实际上，不多。

打印机[托马斯]正在改变，接受 1.75 毫米是 [Lulzbot 迷你](https://www.lulzbot.com/products/lulzbot-mini-3d-printer)，最受欢迎的打印机之一，需要这种修改。唯一需要的材料是一个适用于 1.75 毫米灯丝的新热端，一个 4 毫米的钻头，以及一些扳手和内六角扳手。这将是一个聪明的想法，得到一个热端，使用相同的热敏电阻作为旧的，但这不是一个交易断路器，因为这个问题可以在固件中修复。

拆卸非常容易，在安装 PTFE 管、切断旧电线、焊接新的热端、热敏电阻和风扇之后，[Thomas]已经做好了一切准备。

需要注意的是，把一个 3mm 的热端换成 1.75mm 其实没什么作用。几乎每种灯丝都有两种尺寸，尽管在当地购买 3 毫米灯丝可能不太方便。更换热端是一个好主意，这样就可以在单一直径的灯丝上标准化你的工作室或黑客空间。
