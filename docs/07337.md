# 对蜜蜂进行简单的经典调理

> 原文：<https://hackaday.com/2017/10/18/hassle-free-classical-conditioning-for-honey-bees/>

当你生病或头痛时，你看事情的角度会有所不同。一个感觉不好的人会表现出认知偏见，并期望世界进一步惩罚他们。蜜蜂也是如此。它们是智能生物，表现出多种生活技能，如决策和学习。

早在 2011 年，就有证据表明，蜜蜂在受到震动后会做出更悲观的决定。这些蜜蜂被训练将糖水的奖励与一种特殊的气味联系起来，并将恶臭的惩罚水与另一种气味联系起来——蚁酸的气味，这是一种对付瓦螨的常用方法。当混合两种气味产生的第三种刺激出现时，实验者发现，被激怒的蜜蜂更有可能预料到不好的气味。果然，当他们闻到第三种气味时，他们把舌头放在嘴里。所有没有摇晃的蜜蜂都期待着吮吸一点糖水。

那么，如何判断蜜蜂的反应呢？每当它们的触角接触到令人胃口大开的东西时，它们就会不由自主地伸出它们的喙来尝一尝。这被称为长鼻延伸反射(PER)，这是根深蒂固的，第一天的行为，导致它们从花朵中吸取花蜜，并将其吐出来制造蜂蜜。

[LJohann]是一位行为生物学家，他想在不惊动蜜蜂的情况下，单独测试瓦螨治疗对蜜蜂行为的影响。他建造了一个测试装置，向蜜蜂散发气味，并判断它们的反应,休息后会有几个简短的演示视频。这个装置使[LJohann]能够控制一只蜜蜂，用蔗糖刺激它的触角，并在 LED 和压电蜂鸣器的提示下向它的脸上释放刺激气味。安装在蜜蜂后面的风扇有助于清除先前气味的空气。我们特别喜欢使用伺服系统在测试之间将试管在蜜蜂的脸上来回摆动。

[LJohann]和他的同事得出结论，瓦螨治疗本身不会使蜜蜂悲观。这对于关心的养蜂人来说是一个好消息，他们可能对使用蚁酸对抗蜜蜂最大的捕食者持怀疑态度。休息之后，请观看简短的演示视频。
