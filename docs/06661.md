# 担心打印中途灯丝用完？加入吧！

> 原文:[https://hack aday . com/2017/07/31/担心-耗尽灯丝-mid-print-join-it/](https://hackaday.com/2017/07/31/worried-about-running-out-of-filament-mid-print-join-it/)

如果你曾经害怕扔掉任何你知道无法覆盖下一个小零件的打印机灯丝——更不用说通宵打印了——你可能会欣赏[starlino]将两个[线轴连接在一起](https://www.youtube.com/watch?v=eJeU9lXiQPE)的方法。

虽然还有其他方法来[追踪你使用了多少灯丝](http://hackaday.com/2016/12/08/this-old-mouse-keeps-track-of-filament-usage/)，但这种方法消除了一些猜测。首先，在对角线上剪下细丝的末端——尽可能接近相同的角度。用收缩包装管覆盖两端——例如 1.75 毫米灯丝用 2 毫米的管——确保两端在包装内重叠。用 Kapton 胶带将细丝缠绕在耐热垫上，露出两根细丝之间的接头。温度传感器可以帮助你找到灯丝的熔点，或者你可以根据需要进行实验来感受一下。

用热空气焊接站或热风枪熔化管道内的灯丝，并用空气喷粉机吹几下，迅速冷却下来。剩下的就是切断胶带和收缩包装上的细丝，刮掉任何多余的部分，以防止打印机堵塞。搞定了。现在，回到印刷！休息之后，请观看教程视频
