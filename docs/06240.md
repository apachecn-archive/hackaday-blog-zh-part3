# 饥饿物联网

> 原文:[https://hack aday . com/2017/06/15/internet-of-hungry-hungry-things/](https://hackaday.com/2017/06/15/internet-of-hungry-hungry-things/)

河马是非洲最危险的大型动物。物联网会害死我们所有人。两者结合，你得到了什么？[饿了饿了](http://www.mikekohn.net/micro/hungry_hungry_hippos.php)。[迈克]将经典游戏*饥饿的饥饿的河马*变成了一个神奇而有趣的物联网设备，具有语音识别和机器视觉。

*饿饿河马*是小孩子的(板？)游戏旨在教导孩子们贪吃的美德。棋盘被四个杠杆驱动的塑料河马包围着，游戏的目标是捣碎杠杆，并在这些贪吃的厚皮动物的嘴里收集弹珠。[迈克]用连接到这些杠杆的四个伺服系统自动化了这个游戏，每个伺服系统由一个 [W65C265SXB](http://wdc65xx.com/boards/w65c265sxb-engineering-development-system/) 单板计算机控制。是的，这个项目有用 6502 汇编写的代码。

更进一步，[Mike]使用连接到上网本的 Playstation 3 摄像头进行图像处理。当摄像机探测到某只河马前面有一颗弹珠时，这只河马就会变得饥饿。饥饿的河马。活着是多么美好的时光。

物联网连接？[Mike]还在电动 Imp 的帮助下，通过亚马逊的 Alexa 控制了这些河马。为了激活蓝色河马，所有[迈克]需要说的是，“Alexa，告诉游戏，蓝色河马需要吃东西。”这是一个物联网计算机视觉人工智能河马。我们都知道技术在把我们生吞活剥，但我们从来没想过技术饿不饿。
