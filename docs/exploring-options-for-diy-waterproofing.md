# 探索 DIY 防水方案

> 原文：<https://hackaday.com/2017/12/26/exploring-options-for-diy-waterproofing/>

TL；DR——不要用硅胶封装电子产品。

结论就是这样【GreatScott！]是在尝试了[几种电子设备防水方法](https://www.youtube.com/watch?v=h8ycmroFQSs)后得出的结论。他的努力源于[最近的一个视频](https://www.youtube.com/watch?v=pGLUsQozT94)，在视频中他发现水和电有时确实是混合的，只要水是蒸馏过的并且饮料中的电子设备相对简单。不出所料，他发现主要问题是电解腐蚀，于是他开始尝试各种防水涂料。在一系列仔细的实验中，他分析了保形涂层和灌封化合物的优缺点。保形试验在 ESC 板上使用简单的透明指甲油；这工作得很好，但有点难以触及所有的角落和缝隙。他还尝试用一种厚的黑色硅树脂化合物灌封，但最终没有真正固化。最后一次尝试用合法的两部分环氧灌封化合物密封了电子稳定控制系统，尽管我们怀疑最终的砖块在四轴飞行器上能否表现良好。

如果你想进一步探索盆栽，请查看[这篇基础知识介绍](https://hackaday.com/2016/10/27/a-short-introduction-to-staking-and-potting/)。
