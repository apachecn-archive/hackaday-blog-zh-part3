# 用简易动力送料器平稳切割

> 原文：<https://hackaday.com/2017/09/26/smooth-and-steady-cuts-with-an-improvised-power-feeder/>

一些木工作业要求原料以平稳的速度进给，为此通常使用动力进给器。然而，它们是昂贵的设备，其成本通常只能由高产量的生产车间来承担。但是当你需要一个的时候，你就需要一个，从一个钻孔机和一个滑轮上拆下一个供电装置是一个可行的选择。

毫不奇怪，这个木工作坊的黑客来自于(马蒂亚斯·万德尔)，他似乎从来没有错过木工挑战。他的前两个版本的权宜电力馈线的任务是在他的新房子里制作大量的踢脚线造型。第三个版本，在下面的视频中展示，允许他将原料斜着穿过他的台锯，产生定制的凹口成型。完成的电力馈线可能看起来很简单——它只是一个在木制夹具中驱动滑轮的无刷钻头——但[马提亚斯]带领我们经历的迭代设计过程非常迷人。我们也欣赏黑客中的黑客，他们总能找到进入他的视频的方法。没有车床？没问题！用电钻和带锯即兴制作。

惊讶于【马蒂亚斯】没有在这个版本中使用一些[他著名的木制齿轮](http://woodgears.ca/gear_cutting/template.html)？我们没有。无刷电机非常适合这种应用，在低速时具有恒定的扭矩。想了解更多关于 BLDC 汽车公司的信息吗？用[一个巨大的演示无刷电机](https://hackaday.com/2015/08/20/giant-stepper-motor-gets-you-up-to-speed-on-theory/)获得基本知识。
