# 印制电路板电机的最新进展

> 原文：<https://hackaday.com/2018/05/12/the-current-advances-of-pcb-motors/>

在过去的十年里，印刷电路板的成本不断下降，这是值得一提的。是的，它打开了 PCB 艺术的世界，但它也允许在玻璃纤维和环氧树脂的层压板内放置细铜线的一些实验。我们可以设计自己的电容式触摸传感器。如果你真的很聪明，你可以把线圈放在四层 PCB 里面。如果你非常聪明，你可以添加一些磁铁[，用一块印刷电路板](https://hackaday.io/project/39494-pcb-motor)制造一个无刷电机。

我们第一次看到[Carl]的 PCB 马达[是在年初](https://hackaday.com/2018/01/24/this-tiny-motor-is-built-into-a-pcb/)，但从那时起我们开始了 Hackaday 奖，[Carl]参加了这个项目的奖项，这个项目已经进入了最后一轮。真的有那么牛逼。自上次更新以来，[Carl]一直致力于提高这种微型 PCB 电机的效率和成本。其中一部分来自新的磁铁。[Carl]找到了一些将转子分成四等份的磁铁，而不是四等份的圆形磁铁。这使得转子在其整个区域有更均匀的磁场，并有望获得更大的功率。

这种 3D 印刷电路板电机的第一个版本使用了压配合衬套和金属轴。虽然这种方法可行，但多一块金属只会提高整个发动机的成本。[Carl]重新设计了转子的轴，去掉了金属轴，代之以设计巧妙的 3D 打印轴。这是一些非常好的 3D 打印技术，会让这个马达非常非常便宜。

现在，[Carl]有一个可以在任何能做四层印刷电路板的板房制造的电机，他有一个可以很容易用注射成型制造的转子。[下一步是该电机的闭环控制](https://hackaday.io/project/39494-pcb-motor/log/132541-closed-loop-control)。这是一个挑战，因为四层绕组产生的反电动势有点弱。这也可以通过霍尔传感器来实现，但现在，[卡尔]有一个工作的 PCB 电机。现在真正要做的只有一件事——提高功率输出，这样我们就可以拥有真正的[四轴飞行器徽章](https://hackaday.com/2017/07/18/def-con-badgelife-someone-finally-did-it/)，而不用摆弄微型有刷电机。
