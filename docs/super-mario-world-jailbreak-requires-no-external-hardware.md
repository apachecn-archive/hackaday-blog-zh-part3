# 超级马里奥世界越狱不需要外部硬件

> 原文：<https://hackaday.com/2017/06/03/super-mario-world-jailbreak-requires-no-external-hardware/>

[SethBling]已经[发布了一个超级马里奥世界越狱，允许玩家安装一个十六进制编辑器](https://www.youtube.com/watch?v=Ixu8tn__91E&t=0s)，然后编写、安装和运行自己的游戏 mod。更重要的是，这一切工作在未经修改的墨盒和 SNES 硬件。不需要硬件黑客。

[Seth]很快就说这一切不是他一个人做的。这个 mod 的出现要感谢[Cooper Harasyn]的帮助，他发现了一个保存文件的错误，[MrCheese]优化了十六进制编辑器，[p4plus2]写了一些很棒的 mod。

虽然不需要焊接和编程，但安装这个模块仍然需要相当多的硬件。除了 SNES 和墨盒，你还需要两个多点，三个控制器和夹子来按住控制器上的按钮。即使这样，手术也需要大约一个小时的精细的屏幕体操。一旦越狱被安装，它会保存在 savegame C 中，所以你只需要做一次。

十六进制编辑器允许你做什么？任何你想要的。马里奥的开机状态可以编辑，一个内存位置可以修改，以完成一个水平在任何时候你想。这不仅仅是修改内存位置——你可以编写运行的代码，例如[p4plus 2]甜蜜的心灵遥控模式，它允许马里奥抓住并绕过屏幕上的任何敌人。

看到老的视频游戏硬件被新一代黑客入侵总是很棒的。我们已经在《超级马里奥兄弟 3》中看到了类似的工作，还有一个原创的[游戏机用来驾驶无人机](http://hackaday.com/2016/08/21/fly-with-a-game-boy-classic/)，这只是举几个例子。
