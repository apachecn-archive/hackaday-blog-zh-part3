# 红外返工站

> 原文:[https://hackaday.com/2016/04/05/ir-rework-station/](https://hackaday.com/2016/04/05/ir-rework-station/)

现代表面贴装元件经常需要特殊工具进行返工。然而，这些工具可能很贵。[Michael Skrepsky]想要一个红外线返工站，但不喜欢它的价格。所以他建造了自己的 T1。

据[迈克尔]说，他在建筑中使用了大量的废料。。他使用了 K 型热电偶、光三端双向可控硅开关、三端双向可控硅开关、20×4 显示屏，当然还有 Arduino。一个旧的浴室取暖器，和一个 600 瓦和 100 瓦的卤素灯泡一起作为取暖器。

Arduino 运行两个 PID。一个是预热的慢速调节器。另一个是用于顶部加热器的中断驱动相位调节器。[Michael]在原型板上构建了电路，但提供了布局图以及 Arduino 软件和关键原理图。你可以在下面看到这台机器运转的视频。
