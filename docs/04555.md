# 远程电动长板比骑手耐用

> 原文：<https://hackaday.com/2016/12/27/long-range-electric-longboard-outlasts-rider/>

有什么比在地中海度假小镇的棕榈树和碧蓝的海水中度假更好的呢？也许乘坐你自己设计的远程电动长板会让这块蛋糕变得更大。

当我们说远程时，我们指的是它-估计 25 英里。[overcloker _ Kris]在下面的试驾中无法得出确切数字的唯一原因是，他在 20 英里后太累了，无法继续。在 64 芯电池组(从旧笔记本电脑中收集的 18650 年代制造的)还剩一点电量的情况下，主板肯定还能再跑 5 英里。定制的模制悬挂碳纤维外壳容纳电池组和电子设备，包括手持遥控器的接收器和两个电机的 ESCs。马达支架由铝制成，并焊接到卡车上，通过正时皮带将动力传输到 3D 打印的滑轮上。这是一个好看的建筑，最高时速 22 英里也不算太差。

我们之前已经报道过电动长板的舰队，从那些带有完全 3D 打印甲板的到带有柔性电池组的。但我们怀疑是否有人有这种板的耐力和性能。
