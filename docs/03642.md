# Hackaday 奖品:价值 50 美元的脚控鼠标

> 原文:[https://hack aday . com/2016/10/06/hack aday-prize-entry-50 英尺控制的鼠标/](https://hackaday.com/2016/10/06/hackaday-prize-entry-50-foot-controlled-mouse/)

ALS 剥夺了[C. Niggel]的一个亲戚使用上半身的权利。这实际上是把他们囚禁在自己的房子里；肌萎缩性侧索硬化症不是什么好东西。同样不幸的是，失去上半身的灵活性意味着他们甚至不能使用电脑与人和外界交流。然而，有一天[C. Niggel]注意到这位亲戚的新电动轮椅是脚控的。这能适用于电脑鼠标吗？

他查阅了商业解决方案，发现它们不仅昂贵得令人望而却步，而且充斥着专有驱动程序和各种糟糕的设计废话。有了今天所有的工具，没有理由不能快速原型化并发送给有需要的亲戚。

他使用了导电线、氯丁橡胶和 velostat 的组合来制作垫子。焊盘与一些串联的调节电阻保持平衡。这些信号被发送到 Adafruit 羽毛板，解释这些信号并将其转换为 PS/2 标准。

第一个版本的鼠标使用单独的垫，用接触水泥粘在中密度纤维板上。然而，这一点，加上其他一些初始设计缺陷，导致鼠标过早失效。[C. Niggel]很快回到实验室，制造了一个结构更坚固的新版本，并寄出去。到目前为止一切顺利！
