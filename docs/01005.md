# 3D 打印 RC 伺服到线性致动器的转换

> 原文:[https://hack aday . com/2015/12/24/3d-printed-RC-servo-to-linear-actuator-conversion/](https://hackaday.com/2015/12/24/3d-printed-rc-servo-to-linear-actuator-conversion/)

遥控伺服是方便的，当你需要旋转的东西。如果你需要的话，你甚至可以修改它们来连续旋转。然而，[兔子罗杰]需要线性运动，但希望由遥控伺服提供简单的控制。解决办法？一个 [3D 打印外壳，将伺服系统的旋转转换为线性运动](https://www.youtube.com/watch?v=Sn-BklfEho0)。

该致动器使用五个不同的部分，几个螺钉，和一个共同的钢筋混凝土伺服。视频显示了执行器在 6V 电源下推拉 200g 负载的情况。有一些调整的空间，所以不同的伺服系统应该工作。

我们想知道 STL 文件；[兔子罗杰]提议如果你发邮件给他们。我们问了一下，然后[他们被发布到了一个 GitHub 回购](https://github.com/tscha70/3DPrinterSTLFiles)上。如果你想尝试一套不同的部件，Thingiverse 上也有一些[类似的设计，尽管其中许多使用步进电机。](https://www.thingiverse.com/thing:896781)

我们之前已经看到过一些自制的线性致动器，但是吸引人的是用脉冲宽度控制 RC 伺服系统是多么容易，而不需要 H 桥或步进电机控制器。正好是[在度假时喂猫](https://hackaday.com/2012/04/21/an-easy-to-build-cat-feeder-driven-by-a-diy-linear-actuator/)的东西。
