# Gimbal SDI 照相机 Mod

> 原文：<https://hackaday.com/2017/06/04/gimbal-sdi-camera-mod/>

有时当你需要某样东西时，有一种便宜又容易买到的产品几乎符合你的要求。关键词:差不多。[Micah Elizabeth Scott]，也被称为[scanlime]，正在创建一个悬停相机来跟随她的猫，她的飞宇 Mini3D 3 轴无刷万向架*几乎*做了她需要的一切。[经过一些修改](https://www.youtube.com/watch?v=iMwg1A4yFd4)，【弥迦】现在有了一个小型廉价的三轴云台，带有 Crazyfire HZ-100P SDI 相机和 LIDAR-Lite 距离传感器。

在三十分钟长的视频中，充满了学习的瞬间。我们以前说过，现在再说一遍:“你就看着吧，以后再感谢我们。”[Micah Elizabeth Scott]有一种方式来处理复杂的概念和过程，并以一种方式解释事情*才有意义*(典型的例子:[旁道故障](http://hackaday.com/2016/10/04/glitching-usb-firmware-for-fun/))。而且，虽然这个技巧并不是最具抽象挑战性的，但作为一名教师，弥迦的天赋仍然得到了体现。她会带你经历什么是对的，什么是错的，确保解释为什么事情是错的，以及她如何制定解决方案。

在她的整个视频中，[Micah]分享了从第一手经验中获得的点滴智慧。从黑色热熔胶到 t-glase(一种 3D 打印细丝)，我们了解到一些可能非常有用的材料。

我们对[米卡·伊丽莎白·斯科特]的作品并不陌生，她已经出现了一段时间。她是 2015 年和 2016 年 Hackaday 奖的评委，总是制作我们喜欢报道的东西。她是我们最喜欢的三个黑客之一，她有一个美丽的 T2 网站，展示她过去的工作。
