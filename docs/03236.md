# 技术上来说是黑客。仍然值得怀疑。遥控食物。

> 原文：<https://hackaday.com/2016/08/12/technically-a-hack-still-questionable-remote-control-food/>

我们以为我们会读到一篇文章，也许是关于一架能拿啤酒或油炸圈饼的四轴飞行器。我们得到的更多的是一个油炸圈饼在地板上拖来拖去，使它变得可怜和难以消化。

有时人们开玩笑说不想记起一个疯子。我们理解。虽然我们当然可以遵循(迈克尔科恩)的逻辑，但动机是陌生的。不管怎样，在一次罕见的事件转折中，没有一个 Arduino 被看到；只是逆向工程，独特的解决方案，甚至定制板。这就是你们中的一些人一直在要求的…我们认为。

这个有问题的装置的大脑是一个 TI MSP430G2231 和一个微小的正向电机驱动电路。MSP 等待来自一辆廉价遥控汽车的被黑红外遥控器的信号。然后，它将这些信号转化为适当的电机控制信号，进入一些漂亮的微型金属齿轮箱。

当然，在将电子设备安装到食物上的过程中有很多技术问题，嗯……它们不需要解决，但它们已经解决了。例如，遮蔽胶带显然不能很好地粘在青椒上，所以必须用牙签将胶带固定住。希望像这样的知识能在我们睡觉的时候被安排在晚上擦拭，但是我们可能会坚持到死，不像昂贵的钢琴课。

最后，我们开怀大笑，这个想法是如此愚蠢，它可能会成为下周的教育 Kickstarter。休息后的视频。
