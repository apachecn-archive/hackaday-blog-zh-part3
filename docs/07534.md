# 感应线圈车辆检测器变得现代化

> 原文:[https://hack aday . com/2017/11/04/inductive-loop-vehicle-detector-gets-modernized/](https://hackaday.com/2017/11/04/inductive-loop-vehicle-detector-gets-modernized/)

就像乔治·卢卡斯和原版《星球大战》电影一样，我们许多人可能会发现我们的激情项目永远不会*完全*完成，尤其是当新技术出现时，或者我们只是为了自己的利益想做一些改进。[穆里斯]曾因一个车辆探测电路而出名，但[带着他的项目](http://www.elektronika.ba/868/dual-channel-inductive-loop-vehicle-detector/)的一些重要升级回来了。(幸运的是，其中不包括任何可怕的 CGI 外星人。)

首先，整个项目从头开始重新设计。对于任何不熟悉[原始项目](https://hackaday.com/2014/04/04/building-an-inductive-loop-vehicle-detector/)的人来说，该电路通过感应回路检测车辆，并能够执行像开门这样的任务。现在，它有两个独立的通道，分别进行轮询，但零件数量减少，这将使结构更简单。固件也被重新编程，除了感应车辆的存在，现在还可以测量任何路过车辆的速度。

完整的改进列表可在项目页面上找到，如果您想尝试推出自己的感应线圈车辆检测器，可获得大量相关文档。当然，如果你不喜欢感应线圈，这并不是检测车辆存在的唯一方法。
