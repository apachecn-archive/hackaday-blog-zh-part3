# 史上最吵的七段表演

> 原文:[https://hack aday . com/2018/01/29/the-noise-seven-segment-display-ever/](https://hackaday.com/2018/01/29/the-noisiest-seven-segment-display-ever/)

很少有机械钟是无声的，许多人觉得它们发出的声音很悦耳。但是，一个古老的落地钟庄严的滴答声或床头柜上发条闹钟的舒缓声音都比不上等待[ProtoG]完成时钟的刺耳噪音，这个机电十进制到二进制到十六进制转换器和显示器将成为时钟的一部分。

在承诺一个完整的六位时钟构建之前，作为概念验证，我们可以说[ProtoG]达到了目标。是的，声音很大，但是声音很棒。下面的视频显示了显示器正在进行测试，当时钟频率上升时，驱动七段翻转显示器的继电器的节奏脉动令人着迷。Alfa Zeta 翻转显示器的每个部分都有一个继电器，其 DPDT 触点通过反转极性来翻转一个部分。作为一项正在进行的工作，[ProtoG]还没有分享更多的细节，但他承诺让我们了解电路转换器方面的最新情况。现在它看起来只是一个简单但嘈杂的司机。我们会饶有兴趣地关注这一事件。

如果你喜欢你的时钟更安静，但仍然喜欢时髦的显示，看看这个混合媒体马戏团主题的时钟。
