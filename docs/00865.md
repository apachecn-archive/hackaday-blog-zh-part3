# 第一机器人公司给了我们下一代黑客的希望

> 原文:[https://hack aday . com/2015/12/13/first-robotics-gives-us-hope-in-the-next-generation-of-hackers/](https://hackaday.com/2015/12/13/first-robotics-gives-us-hope-in-the-next-generation-of-hackers/)

第一机器人公司的顶级得分团队展示了一些高中生的能力。他们的 2015 年机器人被称为 Simbot SideSwipe，是一个机电一体化的天才作品，据我们的线人说，学生们只用了六周时间就完成了。

该机器人本质上是一个遥控码垛叉车，能够收集和堆叠六个回收箱和一个绿色垃圾箱。这是机械控制和制造的令人印象深刻的结合——尽管值得注意的是，这些机器人是远程控制的——而不是自主的。

为了鼓励学习，团队在网上发布了他们的工程报告，甚至是 [CAD 模型。](http://www.simbotics.org/resources/cad)从他们的组件选择来看，他们显然有相当多的资金，但不管怎样，我们对制造他们机器人的设计和执行留下了深刻的印象——特别是如果它真的在短短六周内建成。看看下面的视频就知道了:

第一机器人公司( **F** 或 **I** 灵感和 **R** 科学和 **T** 技术的认知)是一个世界范围的基金会，成立于 1989 年，致力于激励我们的年轻人追求科学和技术。这是全球领先的 STEM 儿童参与项目。
