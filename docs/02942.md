# 巨大的蜘蛛在街上游荡

> 原文:[https://hack aday . com/2016/07/16/giant-spider-roams-the-streets-of-French-city/](https://hackaday.com/2016/07/16/giant-spider-roams-the-streets-of-french-city/)

一只房子大小的大蜘蛛在法国城市南特游行时伸展着巨大而纤细的腿。节肢动物的末日来临了吗？幸运的是，这个蜘蛛是法国戏剧公司 [La Machine](http://www.lamachine.fr/en/) 一系列表演中的最新作品。

像 La Machine 的其他产品一样，这个蜘蛛是一个大型的液压控制模型，不是由单个操作员的计算机驱动，而是由一组位于机械装置内部和下方的操作员驱动，他们将蜘蛛腿的操作变成了一段复杂的舞蹈。反过来，街道上的一个团队会协助他们，确保任何操作都是安全的。蜘蛛只有走路的样子，因为它由一辆载有动力装置的轮式车辆的液压臂支撑，所以不需要腿的支撑，它可以极其优雅地移动。
