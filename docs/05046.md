# 建造自制垂直起落火箭

> 原文：<https://hackaday.com/2017/02/16/building-homebrew-vtol-rockets/>

没有人能否认 SpaceX 和蓝色起源正在做的是技术奇迹的壮举。建造一枚垂直起飞、进入太空、然后返回发射台的火箭是一项惊人的技术成就，是真正的火箭科学。然而，SpaceX 和蓝色起源都有一些有利因素。首先，他们有钱。他们正在建造大型火箭，所以有一个很好的质量推力立方定律的效率冲击。他们使用可以节流的液体燃料引擎。

[乔·巴纳德] [没有像 SpaceX 和蓝色起源](http://www.bps.space)一样受到限制。他仍在建造一枚可以垂直起飞和降落的火箭，但他正在艰难地进行。他在建造垂直起降模型火箭。大部分零件都是 3D 打印的。他用的是固体马达，你可以在业余爱好商店买到。这是一种艰难的做事方式，乔的设计取得了一些有限的成功。

虽然从巴纳德推进系统出来的火箭看起来像 SpaceX 的测试车辆模型，但这里有更多的东西。[Joe]正在使用推力矢量系统——基本上是将 Estes 发动机安装在一个万向节上，万向节与一对伺服系统相连。这使得火箭可以在飞行的最初几毫秒内不用尾翼甚至不用发射杆就能直线上升，发射杆是用来让火箭加速的。这是一个模型火箭的主动稳定，随后不可避免的评论 ITAR 违规。

垂直起飞是一回事，但[乔]也试图让他的火箭垂直着陆。他制造的每一枚火箭都有第二个 Estes 发动机，只用于着陆。在下降过程中，机载微控制器计算速度、高度，并确定尝试垂直着陆是否安全。如果第二个发动机有足够的冲力使速度和高度同时为零，着陆腿展开，火箭有希望在草地上软着陆。

虽然[乔]还没有完全成功地用黑火药发动机实现垂直起飞和着陆，但他记录并直播了他所有的尝试。你可以在下面查看一周前的最新版本。
