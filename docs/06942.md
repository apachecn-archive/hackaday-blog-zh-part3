# 任何其他名称的二极管

> 原文:[https://hackaday.com/2017/08/29/a-diode-by-any-other-name/](https://hackaday.com/2017/08/29/a-diode-by-any-other-name/)

作为有源器件，没有比二极管更简单的了。两个终端。电流朝一个方向流动，而不是朝另一个方向。简单吧？嗯，还有一些有用的副作用的例子，比如发光二极管。[GreatScott]指出还有其他有用的二极管，特别是，他发布了一个视频，介绍了[肖特基二极管和齐纳二极管](https://www.youtube.com/watch?v=GtH8lAzQf2A)。

这些特殊的二极管有特殊的用途。肖特基二极管具有非常低的压降和快速的开关速度。齐纳二极管可用于简单的电压调节。

如果你想知道，这些奇特的二极管是因为它们的发明者而命名的。[Walter Schottky]是德国人，虽然有人称之为热载流子二极管，但大多数人都使用发明者的名字。我们不知道克拉伦斯·梅尔文·齐纳发明的另一个名字。

如果你想知道为什么你关心二极管中的高速，[GreatScott]有一个用常规二极管和肖特基二极管整流信号的很好的演示。在某一频率下，正常的二极管开始导通，而在某一频率下它应该是关断的。肖特基二极管能够更快地关断，因此可以处理更高的频率。

还有其他奇特的二极管，包括 PIN 二极管、耿氏二极管和更多的二极管。启示录之后，你可能想试试[用碳酸氢钠](https://hackaday.com/2017/03/06/soda-bicarb-diode-steering-circuit-for-7-segment-display/)自制。奇怪的是，我们去年报道过[【一名女记者】的视频](https://hackaday.com/2016/09/12/diodes-with-hats-zener-and-schottky/)，也报道过类似的话题，如果你想听第二种观点的话。
