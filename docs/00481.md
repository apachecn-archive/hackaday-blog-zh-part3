# 令人惊叹的超级工厂

> 原文：<https://hackaday.com/2015/10/27/amazing-workshops-of-the-supercon/>

所有为 [Hackaday 超级大会](https://hackaday.io/superconference/)提交研讨会提案的人才都让我们感到惊讶。随着提议的提出和邀请的接受，现在是时候宣布您将在这一史诗般的事件中发现的所有研讨会了。

对于那些刚刚收听的人来说，Hackaday 将于 11 月 14 日和 15 日在旧金山的 DogPatch 工作室举办你们期待已久的硬件大会。华丽的会场将挤满了了不起的人，无论是演讲者还是与会者。一个单独的讲座将持续整个周末，而多个研讨会将在不同的楼层进行。

把你自己放在这一切的中间，对我们所拥有的一切感到惊讶。[立即申请您的机票](https://docs.google.com/forms/d/1Y3wFEfdWQC9lWL4VK5JBti0CfIC4wt64dyY0JZsN6Bs/viewform)！

**高级单片机音频**

在 Teensy 的创作者 Paul Stoffregen 主持的这个实践研讨会中，您将在试验板上连接各个部件，让它们实时播放、合成和分析声音。带上一台能够运行 Arduino 软件的笔记本电脑。建议有一些使用 Arduino 的经验。

**简单射频电路设计**

Michael Ossmann 将展示他自己设计的具体例子，包括 Ubertooth One、HackRF One 和 YARD Stick One。之前有非 RF 电路设计经验的参与者应该能够在研讨会结束后设计使用 RF 收发器 IC 的电路板。

**介绍如何从您的处理器中获取内存和性能**

Jen Costillo 将通过实验室与您合作，重点关注四种常见技术:创建分析工具来分析您的代码；提高绩效；降低内存使用量；和降低 ROM 的使用。这些实验可以在 github 上下载。关于这个工作室的更多信息可以在 Rebelbot 或 Hackaday.io 的[上找到。](https://hackaday.io/project/8037-squeezing-blood-from-a-stone)

**众包控制用的 ESP8266 东西**

Toni Klopfenstein 将对 ESP8266 进行概述，并展示创建可通过 WiFi 控制的电路的基础知识。将有一个互动演示，所有观众都可以参与控制，以展示物联网设备的众包能力。

**Kicad 101**

一个 4 小时版本的 Anool Mahidharia 的流行介绍，使用 Kicad 设计自己的 100%定制 PCB。本次研讨会专为一直在进行电路试验，但从未使用过电子设计自动化(EDA) /计算机辅助设计(CAD)程序的电子发烧友量身定制。

**USSSSSB:来自 Python 的会说话的 USB**

2014 年 Hackaday 奖决赛选手 Colin O'Flynn 将带领您使用 Atmel USB 板实施一个简单的 USB 固件项目。

**用锑设计用于定制骨骼-物理模拟**

使用 3D 设计软件 Atimony，Erin Kennedy 将向您展示如何使用锑进行设计，然后创建一个物理模拟来观察力如何影响您的对象。您还将学习如何为 Atimony 编写自己的脚本。你需要一台装有 Atimony 的笔记本电脑[，并使用 traer.physics 库进行处理。](https://github.com/mkeeter/antimony)

**利用 SPICE 学习电路仿真**

使用 LTspice (Windows)或 NG Spice (Linux/ OSX)，汤姆·安德森将简要介绍什么是仿真以及常见的类型(时域、频域和噪声)。他还会解释它们是如何工作的，以及当你运行它们时会发生什么。然后，您将有机会模拟 RC 滤波器，并了解如何模拟时域和频域来显示波特图。您还将使用压控电压源构建运算放大器的模拟，以及其他模拟。请携带您的笔记本电脑并安装 LTSpice 或 NG Spice。

在周六晚上的超级联赛上，我们将宣布 2015 年黑客日奖决赛前五名的顺序，其中包括将赢得太空之旅的大奖得主。我们还将揭晓最佳产品奖的获胜者，该奖项将带来声望、10 万美元的现金以及 Supplyframe 设计实验室的常驻资格。
