# 用 DLP 投影仪矫正色盲

> 原文：<https://hackaday.com/2016/03/21/correcting-color-blindness-with-dlp-projectors/>

大约百分之五的人口是某种程度的色盲，对他们来说，看到从 Roy 到 Biv 的整个光谱是根本不可能的。他们的眼睛没有视锥细胞来识别特定的颜色。然而，大脑是这个星球上最奇怪的机器，通过正确的光线技巧，即使是色盲也可以看到比他们习惯的更多的颜色。这就是[【pointy ointment】的 2016 年黑客日奖](https://hackaday.io/project/10230-color-blindness-corrective-goggles)参赛作品背后的理念:色盲矫正护目镜。

任何声称可以纠正色盲的设备都有一些说明，并且对“色盲纠正”实际上是什么有一个稍微宽松的解释。出于同样的原因，你看不到深红外线，色盲的人不能区分两种颜色；眼睛只是没有传感器来看到特定颜色的光。然而，这并不意味着色盲个体辨别颜色的能力不能提高。[隐形眼镜](http://enchroma.com/technology/)使用光学陷波滤波器来阻挡蓝色和绿色之间以及绿色和红色之间的所有颜色。这是可行的，因为人类的大脑足够怪异，几乎可以适应任何事情。

[PointyOintmen]不搭配光学陷波滤波器。他使用 DLP 投影仪的旋转色盘和 3D“快门”眼镜在一秒钟内多次以不同的色调呈现世界。这很奇怪，未经测试，需要几个小时来适应，但这是一个非常有趣的想法。会不会让色盲看到更多的颜色？这是一个语义问题，但如果你把“看颜色”定义为能够区分两种不同的颜色，是的，它会。
