# 新像素点亮压敏地板

> 原文：<https://hackaday.com/2016/11/07/neopixels-light-the-way-in-pressure-sensitive-floor/>

它有一点“周六夜狂热”的氛围，但[这种压敏 LED 地板](https://www.reddit.com/r/arduino/comments/59hv55/i_think_you_guys_will_appreciate_my_latest/)是为游戏而生，不是为跳舞而生。

不管怎样，[creed_bratton_]的体型看起来相当不错。地板是在 2×4 木材框架上竖起的 5×6 厚 HDPE 切菜板网格。每个单元都有一个新像素环和一个力敏电阻来检测垫上的压力。需要两个 16 通道多路复用器来整合运行节目的 Arduino 的输入，一大堆 wall warts 为一切供电。下面的视频显示了一点建设和瓷砖下的外观。目前还不清楚这一层到底是为了什么游戏，但你可以很容易地想象出一个迷宫或其他一些需要用脚步解决的谜题。

有了这个游泳池舞池，发光地板在这里已经不是什么新鲜事了。但是[这个互动舞池](https://hackaday.com/2011/06/18/disco-isnt-dead-diy-dance-floor-spotted-at-student-parties/)更接近于【creed_bratton_】的游戏方面。
