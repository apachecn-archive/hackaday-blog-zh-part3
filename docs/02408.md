# Hackaday 奖参赛作品:全球教育基础设施

> 原文:[https://hack aday . com/2016/05/24/hack aday-prize-entry-world wide-education-infra structure/](https://hackaday.com/2016/05/24/hackaday-prize-entry-worldwide-educational-infrastructure/)

教育的未来是 STEM，为了让下一代更健康、更快乐、更有效率，世界各地的教室必须开始向小学生教授编程、计算机工程、科学、数学和电子。在工业化国家，这不是问题:他们有足够的钱购买 iPads、Chromebooks 和高速互联网连接。对于发展中经济体？那个问题有点难解决。这些国家的孩子去上学，但是没有一架子的 iPads，没有电脑，甚至连电都不是给定的。为了解决这个问题， [[Eric]为他参加今年的 Hackaday 奖创造了一个移动教室](https://hackaday.io/project/11010-educase-portable-classroom)。

教室不需要太多，但是最好的教育总是需要电脑和互联网。仅仅凭借维基百科，一个互联网连接就能让任何老师的努力倍增，这也许是任何人对孩子教育的最佳投资。这是十年前“每个孩子一台笔记本电脑”项目背后的想法，但从那以后，运行 Linux 的 ARM 板变得非常便宜，我们正在达到一个点，即到处都有廉价互联网是一种真正的可能性。

为了建造这个便携式教室，[Eric]正在依靠树莓派。是的，有更便宜的选择，但是 Pi 已经足够好了。需要连接到在线资源，为此[Eric]正在求助于外部网络。这是一个从轨道上广播教育材料的系统，使用由廉价便携式 KU 波段卫星天线和廉价接收器制成的地面站。

当涉及到偏远农村社区的教育资源时，选择是有限的。有了[Eric]的项目，教育学生在现代世界中生活的基础变得更加容易，并成为今年 Hackaday 奖的一大亮点。
