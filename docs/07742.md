# 钻床基本加工技巧

> 原文：<https://hackaday.com/2017/11/19/tips-for-basic-machining-on-a-drill-press/>

可以肯定地说，大多数 Hackaday 读者都希望家里有一台磨机或一台漂亮的车床，但这样的设备对于业余爱好者来说并不总是实用的。除非你经常在它们上面建造东西，否则它们所占的费用和空间是很难推销的，所以我们经常被迫临时凑合。在他的最新视频中， [[Eric Strebel]给出了一些关于使用标准钻床来执行任务的实用建议，这些任务通常需要用铣床或车床来完成](https://www.youtube.com/watch?v=HHtfcjwHERA)；虽然他的建议可能不会让老手感到惊讶，但它们可能会帮助一些新手最大限度地利用他们所拥有的东西。

[Eric]解释十字滑动虎钳的概念，这是一种使钻床加工成为可能的设备。本质上，这是一个标准的老虎钳，但有螺钉，允许你在 X 和 Y 方向移动夹紧的工件，而钻头已经可以在 Z 方向移动了。对于那些在家计算的人来说，这让我们达到了完整的三维；换句话说，您不仅可以进行不同深度的切削，还可以沿工件表面向任何方向移动切削。

[![](img/0ef93186edb65a59325886a6ce421c3b.png)](https://hackaday.com/wp-content/uploads/2017/11/drillpress_detail.jpg) 你甚至可以把一块(小)圆木放在钻床的卡盘上，然后把一把好的凿子放在十字滑动虎钳上，把它翻下来。然后，凿子就可以靠着旋转的工件向上移动，进行切割。我们不建议做太重的东西，但是如果你需要把像一块塑料或木头这样柔软的东西压到一定的直径，在必要的时候也可以。

[埃里克·斯特雷贝尔]在这一带很快成为人们的最爱。他制作精良的视频向观众展示了产品设计和内部制造的实用一面。[我们最近报道了他的小规模制作视频](https://hackaday.com/2017/11/14/less-than-production-more-than-one-offs/)，浏览他以前的视频可以获得更多宝贵的信息。

对于 Hackaday 来说，寻求在没有实际机械车间的情况下进行加工并不新鲜。[有许多不同的方法来解决这个问题](https://hackaday.com/2017/07/04/drill-press-to-mill-conversion/)，但是[拿起一个像样的钻床和十字滑块](https://hackaday.com/2016/09/01/likely-everything-you-need-to-know-before-adopting-a-drill-press/)对大多数人来说是第一步。
