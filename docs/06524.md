# 邦古德最便宜的仪表

> 原文：<https://hackaday.com/2017/07/17/the-cheapest-meter-on-banggood/>

根据[pileofstuff]的说法，他其实并不需要另一个数字万用表。然而，当他在 Banggood 上看到 4.99 美元的 DT-832 电表时，他想知道这个价格能买到什么样的电表。你可以在他的[近期视频](https://www.youtube.com/watch?v=3j8JqTh8RoQ)(下图)中看到他的结论。顺便说一句，他确实明确表示，他没有得到审查的报酬，也没有得到计价器。他决定看看 5 美元能在一米内买到什么。

根据你对廉价亚洲电子产品的偏好，你可能会也可能不会感到惊讶。毕竟，你不能指望花 5 美元就能买到顶级的实验室仪器。该设备测量交流和 DC 电压，DC 安培数，欧姆，晶体管β，并有一个二极管测试仪和连续性蜂鸣器。它还具有一些频率测量能力。不过，你不会对它不能自动行驶感到太惊讶。公平地说，尽管他提到 Banggood 是电表的来源，但快速的谷歌搜索显示，你可以从所有常见的来源获得它们，只要你让他们从加拿大发货，价格就可以降到 3.73 美元。

除了测试乐器的性能，他还展示了乐器的内部，但并不多。PCB 有一个保险丝、一个运算放大器、一个运行一切的大块 IC 和一些分立元件。

当然，真正的测试是它在电力方面的表现。原来[pileofstuff]手头有几个 Fluke 电表，一个旧的无线电棚屋电表，和一个 Hioki 的袖珍电表。有了所有这些仪表，他能够对它们进行测量，并发现便宜的 DT-832 比福禄克更好，并且比旧的 Radio Shack 仪表要好。

虽然读数符合非常昂贵的 Fluke 电表，但[pileofstuff]很快指出，电表可能不会像专业电表那样不堪一击，但对于坐在你的工作台上，他对 Banggood 上最便宜的电表印象深刻。

我们偶尔会从港口货运公司拿起免费的计价器，但总是很遗憾他们通常没有连续蜂鸣器。所以，也许 5 美元并不坏。当然，多花一点钱就能得到更多的功能，比如条形图、读数保持按钮、真实 RMS 读数、自动测距和更好的显示。但是只有你自己知道你的特性平衡点的价格到底是多少。

当然，如果你买了 Fluke 或其他品牌的电表，你可以期待保修服务、校准服务(如果你需要的话)，最好的部分是——许多使用电表的已知黑客。如果你对这类事情感兴趣，你甚至可以用 WiFi 连接制作一个[万用表网络](https://hackaday.com/2016/06/07/hacking-a-fluke-multimeter-to-serve-readings-over-wifi/)。
