# SST 是一个非常整洁的 ESP8266 智能恒温器

> 原文：<https://hackaday.com/2017/01/01/sst-is-a-very-tidy-esp8266-smart-thermostat/>

智能恒温器在某种程度上已经成为物联网的公众形象。这表明，普通大众对技术的接受不是由技术有多聪明来驱动的，而是由他们能从中看到多少用途来驱动的。一个提供你的食谱或订购更多鸡蛋的冰箱可能是一个非常好的想法，但在街上，一个让你在下班前打开家里暖气的设备要酷得多。像 Nest 或 Hive 这样的产品已经开始成为普通郊区生活的一部分。

然而，物联网恒温器没有理由像上面提到的两个那样成为商业产品。我们今天的主题证明了这一点；SST 是一个使用 ESP8266 的 Wi-Fi 智能恒温器，可以通过应用程序控制，这要归功于它使用了开源的 [Souliss 物联网框架](http://souliss.net/)。

建筑完成得非常好，PCB、彩色显示器和其他组件都在一个整洁的 3D 打印的盒子里。这是一个你可以放在最终用户面前的项目，它以如此高的标准完成。物理实体文件可从上面链接的 hackaday.io 页面获得，而其固件可从 GitHub 存储库获得[。有一个视频展示了这款设备的一些功能，我们把它放在了广告下方。](https://github.com/mcbittech/souliss-smart-thermostat-WiFi)
