# 丙酮平滑导致工作电机

> 原文：<https://hackaday.com/2017/11/20/acetone-smoothing-results-in-working-motor/>

这是只有 90 年代的孩子才会记得的事。1998 年，以压缩空气为动力的自由飞行模型飞机“空中鲨鱼”问世。这架飞机的特点是泡沫稳定器，机翼，一个充当蓄水池的模制机身，以及一个由压缩空气驱动的新型发动机。完整的 Sky Shark 装置包括一个气泵。你所要做的就是把飞机插到泵里，试着打破压力计，然后让飞机飞到树上或者邻居的屋顶上。这仍然是一个相对有趣的机制，尽管我们不会很快看到压缩空气无人机，但它仍然是一个很酷的玩具。

由于[汤姆·斯坦顿]正致力于小规模航空学和 3D 打印的交叉领域，他想他会尝试建造自己的 3D 打印空气马达。这是一个有趣的挑战——发动机需要密封，需要产生某种可用的动力。标准打印机能胜任这项任务吗？有点令人惊讶，是的。

[汤姆]的发动机的设计与 20 年前发现的气动马达大致相同。活塞连接到曲柄，曲柄连接到飞轮，在这种情况下是螺旋桨。在气缸上方，一个球阀防止空气涌入。弹簧安装在活塞的顶部，将球推开，让空气进入气缸。在冲程的底部，球关闭阀门，空气从气缸底部逸出。很简单的东西，真的，但是它能被打印出来吗？

他没有使用[Tom]常用的打印机，而是拿出了一台稍加改装的老式 delta 打印机，以获得更高质量的打印效果。真的，这只是一个 0.2 毫米的喷嘴和对打印设置的一些调整，但[汤姆]设计的空气马达出来很好，并用丙酮进行了平滑处理。

组装好马达后，[汤姆]将它连接到一个充当压缩空气储器的汽水瓶上。发动机工作正常，尽管用这种发动机驱动的飞机能否飞很长时间还是个疑问。你可以看看下面[汤姆]的视频。
