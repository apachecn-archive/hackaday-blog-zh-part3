# Bartop Arcade 以 4:3 IPad 屏幕向 Aspect 致敬

> 原文:[https://hack aday . com/2015/12/05/bartop-arcade-uses-43-ipad-screen/](https://hackaday.com/2015/12/05/bartop-arcade-uses-43-ipad-screen/)

让我们面对现实吧，我们都喜欢拱廊，但不是所有人都能在家里放一个全尺寸的单口相声。[Bentika]知道解决方案是 bartop 风格的橱柜，[但它必须按照他的规格设计和制造](http://imgur.com/a/quyCl)。你看，他是个有点长高比的书呆子。只有合适的 4:3 屏幕才能模拟专为这种显示器设计的游戏。现代的 4:3 显示器很难得到，除非你手边有一台 iPad。早期型号的 iPads 上使用的 1024 x 768 屏幕非常适合这项任务。

驱动这些屏幕曾经是一件苦差事，但由于黑客逆向工程和海外制造，这些天来，控制器只需点击几下鼠标。[Bentika]从易贝订购了一个 iPad 1 屏幕控制器。他得到的是一个只适用于 iPad 2 屏幕的控制器。谢天谢地，他有一堆旧 iPads 可以玩，所以这不是问题。

[Bentika]根据[Joshendy] 提供的[基本轮廓，使用 AutoDesk 123D 设计了他的橱柜。他的最终裁剪图案是用 Adobe Illustrator 创作的。他花了大约 160 美元就能得到整个橱柜的激光切割，包括材料。由于大量的方形角撑板用于对齐各种部件，橱柜组装很容易。](https://www.joshendyblog.net/diy-mini-arcade-machine/)

这个街机的控制器当然是运行 [RetroPie](http://blog.petrockblock.com/retropie/) 的树莓 Pi 2。[Bentika]使用一个[控制块](http://blog.petrockblock.com/2014/12/29/controlblock-power-switch-and-io-for-the-raspberry-pi/)将操纵杆和按钮连接到 Pi 本身。RetroPie 适合“无键盘”操作，他不必将任何 Pi 的 USB 端口带到机箱外。

我们不得不说最终的结果非常好。该系统具有基于 [CRT 的 bartop](http://hackaday.com/2014/08/16/table-top-mame-cabinet-dubbed-the-water-cooler/) 设置的所有便携性，但没有重量。你可以在 Reddit 上查看[更多关于这个黑客的讨论，或者点击视频。](https://www.reddit.com/r/DIY/comments/3um0ec/i_made_a_proper_43_bartop_arcade_cabinet_with_an/)
