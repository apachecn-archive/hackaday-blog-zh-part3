# 来自现成部件的定制液压缸

> 原文：<https://hackaday.com/2017/05/20/custom-hydraulic-cylinders-from-off-the-shelf-components/>

当你的项目需要动力时，你可能需要转向液压。流体动力有很多神秘之处，但也有一个巨大的供应链，专门为你提供你的项目所需的部件。虽然现成的组件可能不适合你的应用，在这种情况下，知道如何[建立自己的定制液压缸](http://www.makeitextreme.com/en/newsroom/entry/VIDEOS/hydraulic)可能会很方便。

虽然定制柱面构建确实很常见，但看到使用[makeitexreme]的过程仍然很有趣。从现成的活塞和压盖开始，这种双作用气缸的制造在加工中非常简单。气缸杆端有螺纹，活塞端有一个盖。配件的螺纹凸台被焊接上，杆的商业端被刻上螺纹，所有的东西都被组装起来。正如下面的视频所示，这个圆柱体非常强大。

作为[makitextreme]多产团队的产品，我们可以说这个圆柱体注定会有另一个更有趣的版本。很难猜测这个最终会在哪里，但我们打赌它最终会在他们商店的另一个工具中。也许它最终会为他们最近的滚弯机 T1 提供动力。
