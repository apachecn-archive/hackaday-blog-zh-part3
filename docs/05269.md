# 你可以自己动手制作导电玻璃

> 原文：<https://hackaday.com/2017/03/20/diy-conductive-glass-you-could-actually-make/>

透明导电玻璃是一种很酷的材料，可以制作液晶面板等等。但商业方法涉及溅射氧化铟锡，这意味着高真空和一些高电压，这是可行的，但不完全是黑客友好的。[Simplifier]已经[记录了一个替代程序](https://simplifier.neocities.org/optglass.html),该程序只使用一个野营炉加热板和喷枪。和一些化学反应。

毫无疑问，这绝对是在外面做的化学反应。[Simplifier]确定的混合物包括溶液中的氯化亚锡和氟化氢铵。它被均匀地喷在加热的玻璃上(350-400°C ),蒸发后，有一层薄而坚固的透明掺氟氧化锡层。[Simplifier]报告电阻下降到单位数欧姆每平方，这非常棒。当然,[Simplifier]在第一遍中没有完全混合，所以阅读中间步骤也很有趣。

我们的想法会立即转向遮蔽部分玻璃，自己动手制作透明电路和面板，但我们怀疑自己走在了前面。尽管如此，这仍然是一个令人难以置信的早期结果，我们希望它为疯狂的透明导电应用开辟了道路。如果你能制作玻璃电路，你会做什么？现在你可以了，而且看起来也不太难。
