# Hackaday 奖品入口:一个树莓 Pi 密码管理器

> 原文:[https://hack aday . com/2016/08/19/hack aday-prize-entry-a-raspberry-pi-password-manager/](https://hackaday.com/2016/08/19/hackaday-prize-entry-a-raspberry-pi-password-manager/)

每周都会有一个新的网站被攻破，几十万个账户的密码被泄露到一个贴吧里。为了保护您自己，您可以经常更改您的密码，*不重复使用密码*，并使用长的编译字符串；所有这些技术都远远超出了人类的记忆能力，甚至是便利贴。电子密码保管人的时代就这样开始了。

电子密码保管器是保存密码的简单设备，可以通过 USB 连接找回密码。Raspberry Pi Zero 作为 USB 设备的功能非常好，导致[gir]为 Hackaday 奖制作了[Raspi Zero WiFi 启用硬件密码管理器](https://hackaday.io/project/9564-rpi-zero-wifi-enabled-hardware-password-manager)。

这个 USB 小工具使用 [pass](https://www.passwordstore.org/) ，即“标准 unix 密码管理器”来存储所有密码。一切都由几个按钮控制，一个小有机发光二极管显示器，当然还有 Raspi 成为 USB HID 设备的能力。这使得 Pi Zero 可以像使用 USB 键盘一样输入密码。

这是一个伟大的项目，因为 Pi Zero 现在确实存在，这让许多批评者感到惊讶，这是 Hackaday 奖的完美参赛作品。
