# 安全移除钻夹头；接收电机、变速箱和离合器

> 原文：<https://hackaday.com/2016/09/08/safely-remove-drill-chuck-receive-motor-gearbox-and-clutch/>

电钻里面有一个有用的宝库。黑客作家和 Rimstar.org 背后的疯狂科学家 Steven Dufresne 友好地记录了如何安全可靠地从电钻马达上取下卡盘。你可能认为这很容易，但偶尔你会遇到一个决心抓住所有钻头的钻头。在下面的视频中，史蒂文为了让一辆黑色的德克尔放弃它的查克所做的努力让我们很开心。

了解卡盘和变速箱是如何连接的，结合正确的工具和一点力，你就能得到一个电机、齿轮、变速箱和离合器。在那之后，演习中剩下的不多，你可以将这些组件中的一些或全部用于新的用途——比如将它们用于 BB-8 机器人的[驱动系统。](http://rimstar.org/science_electronics_projects/bb-8_v2_hamster_motors_batteries_wheels_rollers.htm)

许多项目(比如这个[步行滑板车](http://hackaday.com/2015/07/10/drill-powered-scooter-walks-with-legs/))都使用无绳电钻作为电机电源。能够跳过卡盘以便直接连接到轴上对于那些钻机至少是建筑的半永久性部分的项目是有用的。问问你的朋友、邻居和工作单位。廉价的无绳钻和螺丝枪已经存在很长时间了。通常是电池坏了，很多人都有电钻，知道你要用它们做一些很棒的事情，他们会很高兴地放弃它们。
