# USB 烙铁能力惊人

> 原文：<https://hackaday.com/2016/04/25/usb-soldering-iron-is-surprisingly-capable/>

我们知道你在想什么。一个 8 瓦的 USB 供电的烙铁不可能在易贝卖 5 美元。[大克莱夫]也是这么想的，所以他买了一个，对熨斗进行了测试和拆卸，然后改变了主意。他也能说服你吗？

首先，[BigClive]发现熨斗可能不适合某些工作。除了它明显不适合需要大量热量的连接之外，当与壁式 USB 电源一起使用时，还有漏电流的问题。熨斗的商业端最终通过电源的电容器获得足够的交流泄漏，从而潜在地损坏 MOSFETs 等。话又说回来，如果你手边有交流电源插座，你不就用 Hakko 吗？看起来熨斗最好由 USB 电池组供电，并且[BigClive]能够通过这种方式焊接一些令人惊讶的坚固连接。拆卸和分析揭示了一个电路，它看起来像是从一本[福里斯特 m .米姆斯三]书中出来的。我们不会破坏给你的惊喜，只要看下面的视频。

虽然不像这款 USB 充电熨斗那样真正无线，但我们可以说，就价格而言，这是一款非常适合某些使用情况的熨斗。有人试过这些吗？加入评论，让我们知道你的想法。
