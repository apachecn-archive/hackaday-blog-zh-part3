# 你可以用你的 97 鹰爪做任何东西

> 原文:[https://hackaday . com/2018/04/07/you-can-build-any-out-of-what-holding-your-97-eagle-talon-together/](https://hackaday.com/2018/04/07/you-can-build-anything-out-of-what-is-holding-your-97-eagle-talon-together/)

我们都知道它，我们都喜欢它，那个停在 7-11 外面的家伙把他的车盖在里面。这是什么？聚酯身体填料，更好地了解几乎通用的商标，邦多。除了整流罩之外，Bondo 还能为你带来更多的好处，兄弟，而[Eric Strebel] [将向你展示你能在多大程度上提升聚酯车身填充物的机械性能](https://www.youtube.com/watch?v=p-BqVqAw5GM)。

我们并不总是有聚酯身体填充物。在职业安全与健康管理局成立之前，汽车车身工人会使用手电筒、铅砖和研磨机。你可以在这里查看 OSHA 之前的视频。不用说，在你的车间里蒸发和研磨铅不是最好的主意，肯定有更好的方法。这使得罗伯特·邦多·鲍勃·斯平克发明了一种毒性更小的汽车车身填充物，我们现在称之为邦多。

在演示开始时，[Eric]将一杯聚酯填充剂与一些特殊的添加剂混合在一起:他使用打印机墨水将混合物变成某种东西，而不是我们都知道的那种粉红色。虽然 Bondo 有点太厚，无法铸造，但他确实设法将一点点放入方形模具、PVC 管中，并在泡沫和木材中应用了一点点。这对于演示来说足够了，但是对于加工 Bondo 的实际细节，我们必须等到[Eric]的下一个视频。在那之前，你可以看看下面的介绍，或者看看他之前关于未固化 Bondo 的[自由造型的作品。](https://hackaday.com/2018/02/22/skull-cane-proves-bondo-isnt-just-for-dents/)
