# 滑板黑客在 3D 打印轮子上恶作剧

> 原文：<https://hackaday.com/2016/06/22/skateboard-hackers-trick-on-3d-printed-wheels/>

[盲文滑板]团队愿意骑任何东西。本周，他们正在测试 3D 打印的滑板轮子。我们不只是在这里谈论滚动，[盲文]团队把他们的实验带到滑板公园，交给当地人测试。甩尾、跳跃、碰撞和摩擦都被提上了日程。溜冰者有点担心，因为这是他们第三次测试 3D 打印的轮子。

第一盘落地一跳就粉碎。该装置似乎是由填充了约 10%的 PLA 制成的。[第二套是由 NinjaFlex](https://www.youtube.com/watch?v=KW6NQV6VKrg) 制造的，没有碎裂的问题，但是太软了，以至于轮子在骑手的重量下完全压扁了。第三套，印刷由[尼克林登穆斯]工作很大。它们有一点弹性，但不要粉碎。我们猜测这一套要么是 ABS 或一个更奇特的细丝。非常令人惊讶的是，3D 打印机能够吐出轮子，不仅可以处理骑手的负荷，还可以处理跳跃和技巧带来的冲击负荷。
