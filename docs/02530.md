# 从公路上开采白金

> 原文:[https://hack aday . com/2016/06/06/mining-platinum-from-the-road/](https://hackaday.com/2016/06/06/mining-platinum-from-the-road/)

几十年来，所有汽油驱动的机动车辆都必须在其排气系统中配备催化转化器，以满足排放法规的要求。这些催化剂具有高表面积，表面镀有铂、钯和铑，可催化废气的高温分解。

当一辆汽车达到其寿命的尽头时，其催化转换器被再循环，这些金属被回收，但是这种回收并不包括所有的金属。[Cody Reeder]注意到从报废汽车上取下的催化转换器中的铂的重量明显小于新的。一些金属已经逃逸了，那么它们去了哪里？

这个问题的答案是，它已经脱离了转换器，并通过排气管的后部吹出。因此，在有成千上万辆汽车通过的繁忙高速公路周围的区域，必须有合理的铂浓度。休息时间下方的[视频详细描述了【科迪】寻求验证这一理论的过程](https://youtu.be/v5GPWJPLcHg)，视频以他和一个朋友凌晨在高速公路旁清扫灰尘开始。由此产生的袋子含有大量的砾石和轮胎碎片，加上一些烟头和大量非常细的灰尘。他筛去碎片，并在熔炉中用含有氧化铅的混合熔剂加热灰尘样品。他希望当这种氧化物降解为金属铅时，它会溶解所有的铂并沉淀在坩埚底部，事实上，当他倒出产生的炉渣时，有一粒铅珠。把铅拿走，会发现一点不纯的铂，他进一步提纯和化验，以确定铂的百分比，并检测其他催化剂金属。

他最终得出的数字是，每吨经过细筛的路边泥土“矿石”的重量为 6.7 克，正如他指出的那样，如果在矿井中发现这种矿石，这个数字将被认为是非常有价值的。他的过程可能对那些拿着扫地机的人来说有点困难，但是一个现代淘金热降临到他们当地的州际公路上，配备了清扫车的高速公路机构可以在更有利可图的水平上提取金属。
