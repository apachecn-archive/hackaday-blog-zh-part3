# 看世界创造日的许多建筑

> 原文:[https://hack aday . com/2017/04/24/looke-the-many-builds-of-world-create-day/](https://hackaday.com/2017/04/24/behold-the-many-builds-of-world-create-day/)

今年的世界创造日意义重大。周六，来自六大洲的 70 多个不同团体聚集在一起，作为一个全球黑客社区开展项目。

### 学习机器人实验室

迄今为止，也许关于世界创造日最好的记录来自我们在巴基斯坦的朋友。 [LearnOBots 举办了为期一天的盛会](https://hackaday.io/event/20888-hackaday-world-create-day-at-learnobots-labs)，项目涉及从家庭自动化到可穿戴计算的方方面面。

[Haziq]和[Rafay]不仅仅是一起建造了一个物联网照明项目，[他们还花时间在这个精彩的演示视频中展示了他们的作品](https://twitter.com/learnobots/status/855711937027592193)。该产品连接了 Arduino、蓝牙模块和继电器来驱动灯泡，所有这些都由他们与麻省理工学院应用程序 inventor 一起开发的应用程序控制，以帮助一位卧床休息的朋友。

浏览 LearnOBots 发布的事件日志，了解更多相关信息。这张图片展示了可穿戴界面方面的工作。织物标记用于绘制有趣的设计，然后使用导电线和 Lilypad 板赋予其交互性。我们还看到了使用 Raspberry Pi Zero 和 7 英寸屏幕制作的夏令营注册用户界面。其他小组正在使用 Makey Makey 和 Arduino 开发自定义输入项目。本文顶部的图片展示了一些 LearnOBots 的工作人员，他们拿着一张世界创造日的海报，太棒了！

### 北卡罗来纳州纽兰的阿巴拉契亚锻造厂

世界创造日[在阿巴拉契亚锻造厂](https://hackaday.io/event/21035-hackaday-world-create-day-appalachian-forge-works)在世界创造日带来了一个婴儿吉他放大器。基本电路围绕一个 LM386 放大器构建。在[移动到试验板](https://twitter.com/AppForgeWorks/status/856012039684329473)进行原型制作之前，它是使用白板示意图设计的。

对于一些人来说，这可能是足够的黑客会议，但努力并没有就此停止。设计了一个外壳，并用激光从胶合板上切割下来。这包括蚀刻电源按钮和音量旋钮的标签。扬声器格栅甚至还有织物网，看起来非常完美，即使没有[打出一些黑键](https://twitter.com/AppForgeWorks/status/856231026242318341)也是一件展品。

### 巴尔的摩黑客空间爆出焊机

小轮子、大马达和方管——几乎可以开始测试了。[Baltimore hacker space](https://hackaday.io/event/21113-baltimore-hackerspace-world-create-day-ii)的一帮人花了他们的世界创造日来制作赛车系列中的下一个冠军作品。

在这张照片被抓拍后，团队开始为赛车的控制电子设备工作，这些电子设备最终被放置在电机之间的一个透明盒子中。车队没有时间安装驾驶座，但这并不妨碍深夜试车。

### 麦迪逊 Bodgery 缝纫和嵌入式修补的声音

我在威斯康辛州麦迪逊的教堂庆祝了世界创造日。在 meetup 上工作的项目多得惊人，其中至少有三个项目使用了对我来说是新的东西:

 [![wcd2017-the-bodgery-josh-lange-wares](../Images/8b262b8a52c795c6ab26809551693817.png "wcd2017-the-bodgery-josh-lange-wares")](https://hackaday.com/2017/04/24/behold-the-many-builds-of-world-create-day/wcd2017-the-bodgery-josh-lange-wares/)  [![wcd2017-the-bodgery-bob-baddeley-showing-off-costume](../Images/5ad1dc44dfde96ee94caec2fb6e02924.png "wcd2017-the-bodgery-bob-baddeley-showing-off-costume")](https://hackaday.com/2017/04/24/behold-the-many-builds-of-world-create-day/wcd2017-the-bodgery-bob-baddeley-showing-off-costume/)  [![wcd2017-the-bodgery-group](../Images/8517f3241cd45916b9ef407c6d24db32.png "wcd2017-the-bodgery-group")](https://hackaday.com/2017/04/24/behold-the-many-builds-of-world-create-day/wcd2017-the-bodgery-group/) 

[Josh Lange]带来了他一直在设计的驱动板。我很高兴看到项目中使用的电池。我不知道你可以在一个消费者友好的包装中购买 18650 锂电池(像 AA 电池，但更大)，而且还有电池座。我习惯于看到这些从旧笔记本电池中取出来的东西。

哈卡戴自己的(鲍勃·巴德利)就在旁边，狂热地在缝纫机前工作。他正在制作一系列古怪的挥舞充气手臂的地铁人服装。他们用那些有弹性的洗衣篮作为内部骨架。此外，在 Bodgery 正在进行的是 NES 扩展端口项目，该项目将在不改变库存硬件的情况下，利用 NES 视频硬件制作一个托管 Raspberry Pi Zero 的定制盒式磁带。我们还和一位软件工程师一起在嵌入式基础知识方面度过了一段愉快的时光，这位工程师正在加快嵌入式技术的发展。
