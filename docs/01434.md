# 烧烤温度计变得严肃

> 原文：<https://hackaday.com/2016/02/08/bbq-thermometers-get-serious/>

你可以用 50 美分的一次性笔写字。或者你可以用 350 美元的万宝龙来写。话是一样的，但很多人会告诉你万宝龙有些不同。也许这就是阿明对肉类温度计的感觉。他的版本使用了 Raspberry Pi，并且有一个很长的特性列表:

*   8 通道数据记录
*   测绘
*   网络摄像头(USB 或 Raspicam)
*   通过本地寻呼机、网络、WhatsApp 或电子邮件发出警报
*   使用 PID 控制温度和风扇
*   液晶显示器

你甚至可以用一个 Pi 零来表示一个轻量级版本。Hackaday.io 上有大量的信息，尽管完整的细节目前只有德语版本。正如你在下面的视频中看到的，这不是你的一元店肉温度计。

尽管一次性笔和万宝龙的作用相同，但我们大多数人还是更愿意买万宝龙(尽管在我们开始使用 meisterstüCK Solitaire Blue Hour Skeleton 149 之前，Hackday 不得不大幅提高价格)。

我们可能会用 ESP8266 做更多工作，然后在客户端上做更多工作，但我们必须承认，这是一个功能齐全的温度计。我们之前已经看到过使用蓝牙的[更简单的](http://hackaday.com/2015/10/21/bluetooth-thermometer-minds-your-meats/)，以及一些商业单位的[黑客](http://hackaday.com/2015/04/01/reverse-engineering-a-wireless-bbq-thermometer/)。
