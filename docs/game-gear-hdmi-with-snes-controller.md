# 带 SNES 控制器的游戏装备 HDMI

> 原文：<https://hackaday.com/2017/07/23/game-gear-hdmi-with-snes-controller/>

凭借背光彩色屏幕和主系统兼容性，这款游戏装备领先其主要竞争对手数年。主要的缺点是它很快就耗尽了碱性电池，因此更便宜但设备更少的 Game Boy 仍然能够竞争。然而，由于我们生活在未来，游戏设备已经获得了新的生命，并进行了许多修改，以解决其缺点，包括最新的一款增加了 HDMI 输出的。

构建的核心是 FPGA，用于处理像素解码和 HDMI 输出。FPGA 允许足够高的速度来处理所需的所有数据，尽管[Stephen]仍然需要解决一些屏幕填充问题，通过 HDMI 添加声音，并处理一些不同的像素故障。为了把这个黑客变成一个完整的适配器大杂烩，虽然，[斯蒂芬]还增加了一个 SNES 控制器适配器的游戏装备。任天堂在它的许多游戏中都使用了 Sonic，所以尽管我们在 90 年代早期可能不同意，但我们认为世嘉/任天堂的组合不再跨越任何界限。

游戏装备也进行了其他的修改，使得它们作为手持系统比新的时候更有能力。我们还看到它们[变成了控制台系统](http://hackaday.com/2017/06/02/game-gear-console-edition/)(毕竟它们是主系统兼容的)[也完全变成了其他东西](http://hackaday.com/2016/03/15/upgrade-the-graphics-on-a-sega-game-gear-to-mental/)。
