# 汽车闲置警报帮助你停止浪费汽油，而推文

> 原文：<https://hackaday.com/2016/03/29/stop-wasting-gas-while-tweeting-with-this-car-idle-alarm/>

如果你一边打电话聊天或发短信，一边让车空转，你会有意见的。他不喜欢这样，他想让你改掉这种浪费的习惯——因此[他建立了一个闲置阻止系统](https://hackaday.io/project/10313-idle-waste-alarm),他似乎想在每辆汽车的仪表板上安装这个系统。

在下面的视频中，他努力的目标很明确——那些启动汽车的人然后花时间更新 Twitter 或 Instagram。他的闹钟只是一个 Arduino Nano，在汽车启动时启动计时器。彩色编码的发光二极管标记时间，当灯变红时，一个恼人的嘟嘟声开始提醒你继续驾驶。该设备还包括加速度计，当车辆运动时，该加速度计重置计时器；两分钟的超时应该可以防止最长的停车灯触发警报。

[TVMiller]计划围绕 MKR1000 构建一个放大版的设备，该设备将把空闲与移动比率和其他统计数据转储到云。对于我们的口味来说，这有点太老大哥了，但我们可以看到他的观点，即当大量车辆分散开来时，仅仅几分钟的空转是多么浪费。这个黑客可能是一个很好的个人提醒，以纠正浪费行为。它甚至可以集成到直接从 OBD 端口读取加速度和油门位置的东西中，就像我们不久前报道的车联网黑客。
