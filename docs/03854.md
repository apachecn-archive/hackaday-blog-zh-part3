# Hackaday 奖参赛作品:反重力臂漂浮

> 原文：<https://hackaday.com/2016/10/20/hackaday-prize-entry-antigravity-arm-floaties/>

几年前，[Mike]听说了为坐轮椅的人设计的矫形装置，这种装置可以使他们更容易移动手臂。他的女儿有机会演示这些设备中的一个，该设备的结果很好。与保险公司的斗争并不顺利，但这确实是一个可以用几根橡皮筋在 3D 打印机上制作的设备。于是，【迈克】发明了 [3D 打印的反重力臂浮子](https://hackaday.io/project/14809-antigravity-arm-floats)。

这个名字基本上讲述了这个故事——对于肌肉紧张度低的人来说，这些反重力手臂浮子可以很好地抵消重力的拉力。[Mike]的女儿发现专业的、官方的、没有保险的版本很有用，所以[Mike]决定自己造一个。其实没什么大不了的——只是几个 3D 打印的零件连接到轮椅上，用几根橡皮筋给机械连接提供一些阻力。

本着真正的黑客精神，[Mike]采用了这些弹簧臂浮子的基本思想，并对其进行了新的改进。他用链条作为机械装置，允许在 XY 平面内自由移动。这使得该设备稍微好一点，从各方面来看都是对商业版本的改进。这就是当你可以用 3D 打印机快速迭代时所得到的，使这个项目成为我们在 Hackaday 奖的辅助技术部分寻找的一个很好的例子。
