# 一种无点焊的电动自行车电池组

> 原文：<https://hackaday.com/2018/07/03/an-e-bike-battery-pack-without-spot-welding/>

[Make It Extreme]在某种程度上偏离了重金属 mods 的正常票价，正在为电动自行车开发一种电池组，这种电池组有一些有趣的设计特点。

包装的内容和你想象的差不多——回收的 18650 个锂离子电池。他们没有透露细节，但是我们假设 52 个细胞都被检测过了，所有不合格的细胞都被剔除了。排列为 13S4P，电池用激光切割的丙烯酸框架固定。[Make It Extreme]没有点焊端子，而是使用一系列战略定位的插槽，用折叠的镍条制作触点。牢固的接触由穿过上下接触框架之间的有头螺钉保持。一堆电线将每个电池连接到四个 BMS 板中的一个，整个东西被包裹在一个漂亮的丙烯酸框架中。构建和一个简单的测试在下面的视频中。

虽然我们喜欢无焊接设计的简单性，但我们想知道仅通过摩擦保持电池接触，电池组如何经得起振动。考虑到他们以前的电动交通工具，比如[这款越野气垫自行车](https://hackaday.com/2016/07/28/all-terrain-hoverboard-junk-pile-build/)，我们预计这款车将很快接受测试，而且是以极端的方式。
