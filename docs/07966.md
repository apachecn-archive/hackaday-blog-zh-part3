# 数字窑炉

> 原文:[https://hackaday.com/2017/12/16/digital-kiln/](https://hackaday.com/2017/12/16/digital-kiln/)

窑炉或铸造厂往往被视为一件设备，只有在 hackspace 足够幸运地拥有一个或者个人足够投入资金购买一个自己的设备时才可用。[The Thought Emporium]一直是这样想的，直到他采购材料来制作他自己的窑(T1 ),休息后也可以看到。它的价格是商业模型的一半，不包括一个失败的——爆炸的——油漆罐版本。

正如视频中所描述的，这些熔炉不仅仅是制作陶器和软金属饰品的工具。当然，粘土国际象棋会很酷，但是碳纤维、石墨烯、气凝胶和玻璃呢？一些非常热门的科学发生在高温下。

从容器开始，我们对熔炉的每个部分都有了一个很好的了解，这是一个 11 加仑的金属桶，应该为正在建造的窑炉水平设置了标准。为了安全起见，一些硬件配置可以调整，我们坚持认为任何载流螺钉都安全地安装在外壳内，没有工具无法打开外壳。如果使用金属，有一个关于容器接地的好建议。PID 循环的解释可以忽略。

你还能用窑做什么？怎么样[珠宝](https://hackaday.com/2014/12/10/casting-engagement-rings-or-other-small-metal-parts/)、[热处理金属](https://hackaday.com/2013/10/16/ben-krasnow-discusses-the-heat-treatment-of-steel/)，或者把你的[啤酒罐回收成发动机](https://hackaday.com/2012/06/25/melting-beer-cans-and-building-engines/)。
