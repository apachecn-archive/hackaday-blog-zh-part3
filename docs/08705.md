# 本周失败:小超声刀不能

> 原文:[https://hacka day . com/2018/02/27/本周失败的小超声刀-那不可能/](https://hackaday.com/2018/02/27/fail-of-the-week-the-little-ultrasonic-knife-that-couldnt/)

我们都知道这种感觉，当一个想法在我们脑海中盘旋时，听起来很棒，但当我们实际构建这个东西时，却感到失望。这是尝试新事物的自然结果，当它发生时，我们尽可能挽救并继续前进，希望是明智的。

至少半打败[这个老托尼]的事情是[一次尝试制造超声波切割器](https://www.youtube.com/watch?v=pFeek0a8s7Q)，而且进行的并不顺利。在下面的视频中没有任何流血，尽管看起来像是有[老托尼]处理那些 X-Acto 刀片的方式。他的基本方法是从廉价的超声波清洗器中获取换能器和驱动器，然后将其重新分配到一个工具中，以足够快的速度振动刀具，轻松地穿过坚硬的材料。

剧透提示:效果不是很好。我们认为主要问题是使用了一个与商业(和昂贵的)超声波切割工具相比功率大大不足的换能器，但我们怀疑他加工的喇叭可能也没有优化。公平地说，模拟这样的东西的声学性能并不容易，所以我们不能期望太多。但是，看起来切割机可以做得更好。在评论中分享你对如何让 2.0 版本更好的想法。

这个视频很长，但它和[老托尼]的任何视频一样有趣，充满了附带的宝石，比如空化的细节。我们乐在其中，即使结果并不理想。如果你想看一个真正交付的[这个老托尼]项目，看看他漂亮的令人厌烦的头部构造。
