# 柯林·福尔泽被烧伤了

> 原文：<https://hackaday.com/2016/03/05/even-colin-furze-gets-burned/>

就当这是个公益广告吧。[柯林·福尔泽]除了是一个狂怒的疯子之外，似乎还有一只猫的九条命。嗯，[他并不总是那么幸运](https://www.youtube.com/watch?v=6FYZFAuhFV4)，现在我们已经从对结果的厌恶中恢复过来，我们很高兴[科林]发布了这个“失败”的视频。

基本上，他正在启动他的一个喷气发动机，然后有一个大火球。他没有穿任何防护服。这不是剧透——如果你被那些去医生办公室治疗前臂上下可怕的二度烧伤的人恶心到了，请不要看下面的视频。你可能已经从预览图片中学到了这一课。

当然，我们以前也报道过[科林]的视频。他要么非常幸运，要么比他表现出来的要小心一点。我们见过他[玩火不自焚](http://hackaday.com/2014/05/30/furze-sets-fire-to-everything-with-pyro-gloves/)，还有[把喷气发动机装在卡丁车上](http://hackaday.com/2015/05/17/do-not-try-this-at-home-a-jet-powered-go-kart/)。我们不会告诉你该怎么做，但如果是我们，我们至少会穿长袖，戴头盔。
