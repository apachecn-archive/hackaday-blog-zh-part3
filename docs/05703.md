# x 光成像镜头被说服加入微型四分之三相机

> 原文:[https://hack aday . com/2017/04/22/x 射线-成像-相机-镜头-劝进-加入-微四分之三-相机/](https://hackaday.com/2017/04/22/x-ray-imaging-camera-lens-persuaded-to-join-micro-four-thirds-camera/)

任何对摄影感兴趣的人都知道镜头是包里最贵的部分。镜头的光圈或光圈越大，进入的光线就越多，这对光线昏暗的场景更好。因此，大玻璃的价格会在一个人的口袋里烧出一个洞。[Anthony Kouttron]决定使用他在网上找到的 Rodenstock TV-Heligon 镜头，并将其用于他的微型四分之三相机。

该镜头连接到费歇尔成像电视摄像机上，该摄像机被认为是用于 X 射线成像的荧光管系列系统的一部分。我们发现(安东尼)对设备的探索，以及发现未知所有者以前的黑客行为，非常有趣。甚至在他开始为自己的目的加工零件之前，这是他公布的一次史诗般的拆卸。

由于镜头最初安装在黄铜部件上，[Anthony Kouttron]知道加工定制部件以适应标准化的镜头适配器是相当容易的。他详细描述了通过打磨、加工和攻丝清理原始底座的过程。在这个过程中，你会喜欢他关于处理一个零件的技巧，这个零件不是一个外部的完美圆形，而是有一个可怕的安装凸片(他不再需要它)从一侧伸出。

休息后的视频显示了用非常浅的景深拍摄的结果。对于那些已经有手动镜头但没有自动对焦马达的人来说，[一个转换黑客](http://hackaday.com/2012/05/06/converting-a-manual-camera-lens-to-use-motorized-zoom-and-focus/)也很管用。
