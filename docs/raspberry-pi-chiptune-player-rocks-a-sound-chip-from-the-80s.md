# Raspberry Pi Chiptune 播放器摇滚 80 年代的声音芯片

> 原文：<https://hackaday.com/2016/09/04/raspberry-pi-chiptune-player-rocks-a-sound-chip-from-the-80s/>

有时候用低效的方式做一件事是值得的。例如，为了学习新的东西，或者只是为了使用某个特定的部分，这可能是值得的。[Deater]用[Raspberry Pi AY-3-8910 chip tune 播放器](http://www.deater.net/weave/vmwprod/hardware/ay-3-8910/)做到了这一点(用 LED 可视化器！)

古老的通用仪器 AY-3-8910 系列声音芯片在旧硬件中很常见，如家用电脑和游戏控制台以及 Apple II 系列的声卡。他们能够产生具有不同效果的三个通道的方波。开发人员最终通过巧妙的破解挤出了每一点点性能。Raspberry Pi 有足够的能力在软件中完成所有这些，但正如[Deater]所说，使用 80 年代的*实际* AY-3-8910 要有趣得多。一些 LED 条形图和矩阵完善了整个系统。

树莓 Pi AY-3-8910 chiptune 播放器的所有代码都可以在项目的[【deater】的 github 资源库](https://github.com/deater/vmw-meter)中找到。在休息之后嵌入了玩家敲打出一些声音的视频。

[deater]对用于 Apple II 系列声卡的 80 年代的声音芯片感兴趣，因为为 Apple II 制作了一个 Kerbal 太空程序的[端口。AY-3-8910 有一些当代的等价物，其中之一(雅马哈 YMZ294)被用在这个奇怪的](http://www.deater.net/weave/vmwprod/ksp/)[“8 位”口琴](http://hackaday.com/2014/01/28/the-8-bit-harmonica-blows-in-from-japan/)上。
