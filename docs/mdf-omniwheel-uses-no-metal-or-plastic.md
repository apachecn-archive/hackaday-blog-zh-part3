# MDF Omniwheel 不使用金属或塑料

> 原文：<https://hackaday.com/2016/02/20/mdf-omniwheel-uses-no-metal-or-plastic/>

有什么比脚轮更好？全方位轮。这些轮子就像一个大轮子，不同角度的小轮子可以向任何方向滚动。【索诺拉】[用激光切割的中密度纤维板](http://www.instructables.com/id/MDF-Omni-Wheel/)建造了一个全方位轮。中密度纤维板构成了车轮的所有部件。根本没有塑料或金属。

[索诺拉的]轮子更像是一个脚轮的被动设计。有可能在两个方向上驱动轮子通过中心，但是直角滚轮是被动的。

我们以前见过几个带全轮的[机器人。事实上，这个受](http://hackaday.com/2014/10/01/minicut2d-and-omniwheel-robot/)[三脚架启发的机器人](http://hackaday.com/2013/06/08/omniwheel-robot-build-uses-a-bit-of-everything/)也有被动滚轮，三条腿的设计利用了它们(所谓的[猕猴桃驱动](https://en.wikipedia.org/wiki/Omni_wheel))。一些方案将多向轮与常规轮结合起来(通常标准轮在中心)。还有其他多向车轮设计，包括[麦克纳姆车轮](https://en.wikipedia.org/wiki/Mecanum_wheel)。您可以在下面看到一段 MDF 车轮运转的视频。
