# 阿杜·麦克杜伊诺演奏风笛

> 原文:[https://hack aday . com/2017/01/10/ardu-mcduino-plays-the-badgpipes/](https://hackaday.com/2017/01/10/ardu-mcduino-plays-the-bagpipes/)

为了迎接新年，[约翰]决定制造一个演奏风笛的机器人。与我们之前见过的其他乐器演奏机器人不同，这个机器人在解剖学上也是正确的。约翰更进一步，用 3D 打印的手指和手来演奏他的那套笛子。

机器人的大脑由 Arduino Mega 2560 处理，它通过驱动板驱动一组螺线管。手本身是从开源项目[打印出来的，这是一个为匹配的接受者 3D 打印假肢手的组织，特别是那些买不起假肢的人。他必须将手的比例放大 171%才能让它们正确地吹奏笛子，但从那时起，向袋子提供空气(通过一个人)并对 Arduino 进行编程以演奏几首歌曲就变得相当简单了。](http://enablingthefuture.org/)

风笛不是一种特别常见的乐器(至少在世界上不属于苏格兰的地方)，所以看到一个机器人演奏风笛是很有趣的。当然，你的音乐播放机器人也许可以用[一种通常根本不被认为是乐器的东西](http://hackaday.com/2017/01/01/mechano-robotic-flute-made-from-an-old-shotgun/)来制作音乐。如果这些都不能满足你的需求，你也可以随时制作你自己的[专用半机器人仪器](http://hackaday.com/2016/03/03/incredible-marble-music-machine/)。
