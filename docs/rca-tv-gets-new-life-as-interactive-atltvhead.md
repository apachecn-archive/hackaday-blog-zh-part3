# RCA 电视获得新的生命，成为交互式电视

> 原文：<https://hackaday.com/2018/05/27/rca-tv-gets-new-life-as-interactive-atltvhead/>

电视通常是你坐着被动观看的东西。对于[Nate Damen 的] [交互式可穿戴电视头戴项目，又名 Atltvhead](https://hackaday.io/project/34274-atltvhead) ，情况并非如此。如果你在佐治亚州的亚特兰大走来走去，看到他拿着一台电视机走来走去，介绍一下自己吧！或者[登录 Twitch chat 并控制](https://www.natedamen.com/stream)他连接到屏幕上的 led 显示屏上显示的内容。除了是可穿戴技术，它还意味着是一件互动艺术品。

为此，他的第三个版本，电视是一个 1960 年的 RCA 维克多便携式电视。你可以在他的 hackaday.io 页面上看到他为以前的版本找到的一些电视。它们都是真正的古董。他掏空了最新的这个，并在屏幕内以蛇形图案安装了 WS2812 LED 灯条。led 由他的代码和运行在 ESP8266 上的 FastLED 库控制。电源来自四节镍氢 AA 电池，给他 5 伏电压，他将其下调至 3.3 伏。他的手机充当 WiFi 热点。

[Nate]限制命令，以便只显示积极的东西，例如心形。或者你可以通过改变亮度或让 led 闪烁来调整显示的内容。从我们在下面的第一个视频中看到的他吸引的人群来看，我们可以说他的项目取得了巨大的成功。在第二个视频中，Nate 进行了代码演练，并谈到了他的一些设计决策。

WS2812 LED 灯带非常受欢迎，用于各种项目。在这里，它们用来照亮一个旋转的夜光球，在这里，它们是低分辨率 LED 电视的 T2 显示屏。你用它们做什么？
