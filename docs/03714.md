# 派对无人机出现在玫瑰上

> 原文:[https://hackaday.com/2016/10/07/party-drone-comes-up-roses/](https://hackaday.com/2016/10/07/party-drone-comes-up-roses/)

[Nicomedia](一个两人小组)为他们的无人机制造了一个有效载荷，有两个不同的目的:允许它[投掷玫瑰花瓣之类的东西，并燃放烟花](http://www.instructables.com/id/Party-Drone/)。老实说，虽然这是一个很酷的想法，但我们有点担心从高处扔下东西可能不是一个好主意(尽管玫瑰花瓣可能是好的)，从无人机上燃放烟花似乎根本不是一个好主意。如果你想复制这一点，你可能需要确保这些东西在你所在的地方是合法的。

在下面的视频中，你可以看到效果。与无人机控制器相连的伺服系统打开盒子，释放有效载荷。在这种情况下，团队没有备用通道，所以他们使用了一个单独的控制器，但如果在飞行控制器上有一个备用通道，那可能会更好。

烟花部分的描述不是很详细，也没有图片或视频。这或许也无妨，因为这或许不是你应该做的事情。好消息是，典型的无人机不能携带太重的东西掉在人身上或爆炸。

如果你有很多多余的无人机并且想要一个游戏聚会，试试[太空入侵者](https://hackaday.com/2016/09/19/real-life-space-invaders-with-drones-and-lasers/)。如果你只是要点燃烟火，你可能应该读一下关于[这个项目](https://hackaday.com/2014/06/17/triggering-remote-fireworks-with-an-arduino-and-an-android/)。
