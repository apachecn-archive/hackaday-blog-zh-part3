# 迪拜警方测试四轴摩托车

> 原文:[https://hack aday . com/2017/10/16/迪拜-警察-测试-四轴飞行器-摩托车/](https://hackaday.com/2017/10/16/dubai-police-test-quadcopter-motorcycle/)

如果你曾经希望你能在你的四轴飞行器上飞行，你会真的想看看迪拜警察局测试气垫自行车的视频。销售该设备的俄罗斯[公司](https://www.hoversurf.com/) Hoversurf 没有提供很多技术细节，但它看起来相当简单。它基本上是一个摩托车座椅和一个大的四轴飞行器。从有关该设备的视频中，你可以推断出飞行员可以控制它，或者你可以远程驾驶它。你可以在下面看到其中一个视频。

这里有几件事让我们担心。当然，作为飞行员膝盖高度的巨大旋转螺旋桨应该会让你手心冒汗。在演示中，他们甚至展示了试飞前螺旋桨防护装置的拆除，但老实说，这些看起来不像他们会让一个坠落的飞行员远离旋翼。除了炒作之外，我们发现很奇怪的是，演示没有显示飞行员转弯的许多(如果有的话)镜头。像这样的车辆对警察的好处应该是机动性，从我们看到的气垫冲浪仍然有限。

那么这是真的吗？很难说。短视频大多显示垂直或水平飞行，没有机动。很难转吗？电池寿命真的很短吗？另一件怪事:当我们第一次在他们的网站上看到美国专利局的来信时，我们认为他们可能有一些新技术。然而，这封信只是显示他们注册了商标，并没有提到专利。如果有专利我们想知道这里有什么新的和新颖的。

当然，我们知道建造这样一台机器是可能的，因为我们[看到【柯林·福尔泽】用两个转子而不是四个](https://hackaday.com/2016/04/28/colin-furze-flies-the-dangerous-skies/)做到了。美国国防部正在与一家名为 Malloy 的公司合作，还有其他实际的例子。还有一些不太[的实际例子](https://hackaday.com/2017/07/01/manned-multirotor-flies-again-electric-style/)。我们真正在寻找的是一种效果非常好的产品，它将被实际使用。你知道，就像机场警察用的赛格威，对吧？

我们希望 Hoversurf 能把它推向市场，因为我们确实想要一个。没有理由认为他们不能，但我们希望有更多的细节即将到来。
