# 穿着小兔子的 Alexa

> 原文:[https://hackaday.com/2017/08/16/alexa-in-a-bunny-rabbit/](https://hackaday.com/2017/08/16/alexa-in-a-bunny-rabbit/)

树莓派是谷歌 AIY 的完美候选，你可以对着一个装有电子设备的纸箱说话。[BuddyCasino]接受了在 ESP32 中挤压一个[Alexa 客户端的挑战，为了让事情变得有趣，一只小兔子被选为虚拟助手的主人。](https://www.youtube.com/watch?v=xRobZAVO_Io)

几个月前，我们拆除了谷歌 AIY 工具包(T1)，其中,[BuddyCasino]评论说,[成功地将 Echo Dot 客户端移植到了 ESP32(T3)中。果不其然，下面的视频展示了这种构建的运行演示。该项目使用的 MAX98357A 与谷歌 AIY 语音帽中使用的 I2S DAC 相同。对于麦克风，该设备也是 I2S 的一个组件，但与使用 SPH0645LM4H 的谷歌 AIY 套件不同，[BuddyCasino]选择了 ICS-43434。](https://github.com/MrBuddyCasino/ESP32_Alexa)

两个新像素被用作用于各种目的的视觉指示器。这个项目是现代设计变得多么简单和廉价的一个极好的例子。我们希望看到作者在设计中添加更多功能，谁知道我们将来会不会在 ESP32 上看到谷歌助手端口。查看原拆卸更多的灵感。
