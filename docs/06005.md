# 全息图不能太薄

> 原文:[https://hackaday.com/2017/06/03/holograms-cant-be-too-thin/](https://hackaday.com/2017/06/03/holograms-cant-be-too-thin/)

我们已经看到 3D 手机时尚来来去去，像 Evo 3D 这样的设备使用视差屏障来实现自动立体视觉(即不戴眼镜进行 3D 观看)。这些显示器不是全息图，它们只是向你的眼睛显示两种不同的图像，就像 3D 电影或立体视觉仪一样。然而，来自澳大利亚和中国的研究人员希望改变这种情况。他们开发了一种纳米全息图(他们的术语)，大约比一根头发还要细 1000 倍。你可以在下面看到一个关于这项发明的视频。

传统的全息图调制光的相位，以给出三维深度的错觉。但是为了产生所需的相移，这些全息图需要与所涉及的光波长一样厚。研究人员声称全息图制作“简单”，但这取决于你将其与什么进行比较。你需要一些奇异的材料，真空沉积设备，和一个可以产生飞秒长脉冲的激光器。

研究小组用 25 纳米的全息图打破了这一厚度限制。他们的技术依赖于拓扑绝缘体材料，这是一种新型的量子材料，在表面层具有低折射率，但在材料的主体部分具有高得多的折射率。这形成了一个固有的光学谐振腔，它可以增强相移，使全息摄影成为可能。

下一步是开发一种覆盖在 LCD 屏幕上的刚性薄膜。当前版本的像素至少大了十倍，不适合该应用程序，所以这是另一个需要克服的障碍。

多年来，我们已经在《星球大战》等电影中看到了拍摄 3D 图像的屏幕。这还不是它，但它是下一步。想象一下，一部电话、一块手表或一个隐形眼镜可以产生全息图像。或者垃圾桶大小的机器人。
