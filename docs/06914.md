# 打印机网络上闪烁的指示灯

> 原文:[https://hack aday . com/2017/08/25/blinking-LEDs-on-the-internet-of-printers/](https://hackaday.com/2017/08/25/blinking-leds-on-the-internet-of-printers/)

当你询问购买哪种 3D 打印机的建议时，不管价格如何，Ultimaker 始终是首选。这种打印机受欢迎是有原因的——它易于使用，质量极高，并且在引擎盖下的某个地方运行着一个完整的 Linux 系统。最后一点为一些有趣的技术打开了大门，[比如使用 3D 打印机作为 RGB LED](https://www.youtube.com/watch?v=kKGXqZjwn8Y&feature=youtu.be) 。

Doodle3D 团队一直在玩 Ultimaker API，看看他们能否让他们的软件与 Ultimaker 打印机一起工作。Ultimaker 有 RGB LEDs，所以很明显，在使用 API 时，最简单的概念验证就是闪烁几个 led。实际的代码是用 HTML、JavaScript 和 Node 在仅仅两个小时内编写的。作者承认很丑，但是很管用。不会错的。

虽然这只是对 Ultimaker API 的一个简单测试，但当你搜索“Ultimaker API”时，它在谷歌搜索结果中的排名出乎意料地高。真遗憾，因为这台打印机的引擎罩下有很大的能量。如果你有某种你想加入的模式，这里有[黑客日提示线](http://hackaday.com/about/)。
