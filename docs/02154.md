# 黑客 105——更多思维和大脑黑客

> 原文:[https://hack aday . com/2016/04/30/hack let-105-more-mind-and-brain-hacks/](https://hackaday.com/2016/04/30/hacklet-105-more-mind-and-brain-hacks/)

浪费头脑是一件可怕的事情，但要破解它却是一件可怕的事情。我们上次[访问 brain hacks 是在 2015 年](http://hackaday.com/2015/07/17/hacklet-56-brain-hacks/)7 月。Hackaday.io 上的事情发生得很快。错过几天，你就会错过一堆伟大的新项目，包括一些令人敬畏的新生物技术黑客。本周，我们将在 [Hackaday.io](https://hackaday.io) 上盘点一些最好的新头脑和大脑黑客

[![](../Images/13ce4665413f615e2c024bbb476c5116.png)](https://hackaday.io/project/10600) 我们先从【丹尼尔费利佩巴伦西亚五世】和[大脑运动](https://hackaday.io/project/10600)说起。Brainmotic 是(丹尼尔的)2016 年黑客日奖的参赛作品。智能家居和物联网是这些天的热门词汇。[丹尼尔的]项目旨在将这项技术与脑电图(EEG)结合起来。你的思想将能够控制你的家。这对任何人来说都很好，但对残疾人来说尤其重要。Brainmotic 的接口正在使用开放硬件 [OpenBCI](http://openbci.com/) 作为大脑接口。[丹尼尔的]软件和硬件将在这个界面和用户的家之间建立一座桥梁。

[![biofeed1](../Images/dfd36146225c953038076c15d7927612.png)](https://hackaday.io/project/6935) 接下来我们有【安杰利克·贝科】与[串行/无线脑波生物反馈](https://hackaday.io/project/6935)。脑电图过去实施起来非常昂贵。东西已经变得足够便宜，我们现在在市场上有大脑控制的玩具。正在把这些玩具改造成有用的生物反馈工具。这些工具可以用来可视化，甚至控制用户的精神状态。[Angeliki 的]首选武器是 MindFlex 系列玩具。在 PunchThrouch LightBlue Bean 的帮助下，她能够让脑电图耳机通过蓝牙进行通话。PC 端的一点花哨软件可以将 MindFlex 释放的脑电波信号解释为简单的图形。[Angeliki]甚至基于这个项目创造了一个[精神控制的机器人木琴](https://hackaday.io/project/7676)。

[![brainhelmet](../Images/1be86da27fa9f4715f77b93fc25f14b4.png)](https://hackaday.io/project/9831) 接下来是希望用[强化头盔](https://hackaday.io/project/9831)保护大脑的【司徒朗兰】。创伤性脑损伤(TBI)是这些天医疗技术的焦点。尽管情况可能很糟糕，但 TBI 只是自行车或摩托车事故中头部和颈部受伤的几种类型之一。技术的存在是为了减少伤害，包括一些新的头盔。许多这些技术，如 MIPS，都获得了专利。[Stuart]正致力于创造一个更精确的头盔内的头部模型，以及头骨内的大脑模型。根据这些数据，他打算创建一个免许可证的保护系统，该系统可以用于新头盔以及现有硬件的改造。

[![mindwave](../Images/0ef20cffee34fafec662ba41ebb14d1c.png)](https://hackaday.io/project/11146) 最后我们有【汤姆·米恩】，他在 2016 年黑客日奖的参赛作品是[用神经反馈训练你的大脑](https://hackaday.io/project/11146)。[Tom]希望利用神经反馈改善癫痫、自闭症、多动症和其他疾病患者的生活质量。就像上面的[Angeliki ]一样，[Tom]正在黑 NeuroSky 的硬件。在这种情况下，它是 MindWave 耳机。[Tom]目前的目标是从 MindWave 内部的 TAGM1 板上提取数据。一旦他获得了脑电图数据，在 PC 端运行的 Java 应用程序将允许他向用户显示脑电图信息。这是一个全新的项目，更新很快，所以绝对值得一看！
