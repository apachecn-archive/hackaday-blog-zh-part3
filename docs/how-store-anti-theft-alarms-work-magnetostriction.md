# 商店防盗报警器如何工作:磁致伸缩

> 原文：<https://hackaday.com/2015/12/03/how-store-anti-theft-alarms-work-magnetostriction/>

这太不可思议了。应用科学 YouTube 博客[的【Ben Krasnow】在使用](https://www.youtube.com/watch?v=KAm7qAKAXwI)[磁致伸缩](https://en.wikipedia.org/wiki/Magnetostriction)的防盗标签上发布这个视频的前两天，我们[写了一篇关于一家公司使用逆磁致伸缩发电](http://hackaday.com/2015/11/27/physics-youve-never-heard-of-providing-power-from-waves/)的博客。奇怪的同步性！

[Ben]拆开那些矩形塑料安全标签，当销售人员在你离开商店之前忘记消磁时，这些标签最终会让每个人尴尬。里面是两个金属条。一个磁条被磁化和消磁，另一个是磁致伸缩的——这意味着它在磁场中会稍微改变长度。

一个发送器线圈以磁致伸缩带的共振频率(约 58kHz)用脉冲信号撞击磁致伸缩带。磁条随着发送者的磁场膨胀和收缩。当发送者的脉冲停止时，磁条会持续振动一小段时间，发出一个交流磁场，被探测器接收到。你被抓了。

最后一个难题是标签内的可磁化金属条。当它根本没有被磁化，或者磁化太强时，磁致伸缩带对发送者的磁场没有太大的响应。当偏置磁铁被磁化得恰到好处时，另一个磁条就会像它应该的那样响起来。这就是为什么他们在结账时对磁条进行消磁。

我们甚至没有破坏[本]的解释。他在调查这一切方面做得非常出色。他甚至测量了这些小条的长度变化，变化幅度为百万分之十。这是一个伟大的低技术含量的测量，最终是正确的，值得在你的“观察”列表中名列前茅。

既然磁性扭曲已经进入了我们的集体潜意识，那么下一个我们会看到它出现的地方是哪里呢？
