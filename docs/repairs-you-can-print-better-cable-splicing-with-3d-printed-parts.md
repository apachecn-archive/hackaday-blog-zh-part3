# 您可以打印的维修:使用 3D 打印零件实现更好的电缆拼接

> 原文：<https://hackaday.com/2018/02/16/repairs-you-can-print-better-cable-splicing-with-3d-printed-parts/>

不久前，[马里乌斯]遇到了一个问题。他的一个朋友住在雨林中，一个麦克风被一只肮脏油腻的老鼠袭击了。电缆被咬成两半，与外界联系的重要手段也随之消失。固定五芯或六芯电缆的常用方法是热收缩、线路工接头、绝缘带和运气。[Marius]需要比这更好的东西，所以他转向他的 3D 打印机，制作了自己的电线接头外壳。

问题中的麦克风是一个奇特的 Jenal jobbie，电缆中有半打左右的导线。接线盒是解决这个问题的显而易见的办法，一些原型，从矩形到印有标志的精美椭圆形盒子，都在 3D 打印机上打印出来。这些接线盒的两端都有孔，当电缆末端穿过这些孔时，电线可以接合、焊接并相互绝缘。

这种麦克风必须经得起热带雨林和老鼠的严酷考验，所以[马里乌斯]必须包括一些防水措施。这是以热胶枪的形式出现的；只需将熔化的热胶注入接线盒，盖上盖子，然后等待它冷却。像所有好的修复一样，它是有效的，当这种修复最终失效时，麦克风中的其他东西肯定会变坏。

这是一次很好的维修，也是 3D 打印机如何让维修变得容易、简单、便宜，并且几乎和库存零件一样好的一个很好的例子。你可以看看下面修复的几个视频。
