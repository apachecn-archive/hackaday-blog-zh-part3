# 快速和肮脏的冲击手套

> 原文：<https://hackaday.com/2017/05/10/quick-and-dirty-electroshock-gloves/>

[JLaservideo]制作了一些很酷的高压手套，并在 YouTube 上上传了一段视频，向你展示如何自己戴上手套。使用一些非常简单的零件，他设法制造出一些像样的火花。

这个项目的核心是一种新的电弧打火机，它通常使用某种类型的电压倍增器电路来工作。剩下的只是电线、胶水、铝箔、开关和彩弹手套。

用每个手指的尖端作为电极，他接触的任何东西都会接通电路，产生高压电弧。烧穿纸张的演示非常简洁，尽管我们承认我们很难想出用带电的指尖还能玩出什么花样。有人吗？
