# 单芯片视频游戏控制台

> 原文:[https://hack aday . com/2015/09/23/a-单芯片-视频-游戏机/](https://hackaday.com/2015/09/23/a-single-chip-video-game-console/)

准备好感受你的编程技能的不足了吗？已经警告过你了。看看使用 PIC24 的[【Voja 的】单芯片视频游戏机。](https://hackaday.io/project/5574-single-chip-game-console)它产生 VGA 信号、5 声道声音，并采用带有方向板和两个按钮的游戏手柄形式。

他使用 PIC24 已经有一段时间了，现在正在生成 VGA 信号，他决定创建一个 2D 视频游戏可能会很有趣…所以他决定看看他是否可以编写一个旧的频谱游戏“跳跃杰克”的副本(在线玩[这里](http://torinak.com/qaop#!jumpingjack))。

它使用 PIC24EP512GP202 微控制器，配有 512K 闪存、48k 数据和庞大的 28 引脚。这款游戏被很好地记录了下来，展示在他的项目页面上。光是看着它就让我们头晕目眩！与上周的 ArduinoCade 相比，[这是一个伟大的项目。两者都在输出音频和视频方面做得非常出色，同时为游戏的实际运行留出了足够的空间。](http://hackaday.com/2015/09/17/retro-games-on-arduinocade-just-shouldnt-be-possible/)
