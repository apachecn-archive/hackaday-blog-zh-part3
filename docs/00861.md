# 沃尔沃卡车:孩子通过测试，母亲认可

> 原文：<https://hackaday.com/2015/12/04/volvo-trucks-kid-tested-mother-approved/>

如果那看起来像一个四岁的小孩拿着遥控器开着一辆全尺寸的自动倾卸卡车——那就是[因为它就是。](http://www.pastemagazine.com/articles/2015/12/volvo-shows-off-new-trucks-by-letting-a-four-year-.html)作为现场测试系列的一部分，沃尔沃制作了一个荒谬的障碍课程，然后让一个四岁的孩子驾驶他们的重型翻斗车。也许是病毒式广告——但太棒了，不能不分享。

不要担心，有一个黑客参与！卡车上的遥控装置没有那么精致，不可能是商用的“遥控工具”。这意味着某个幸运的黑客为一辆该死的自动倾卸卡车建造了一个远程控制系统。认为我们嫉妒。

令人惊讶的是(也可能不是)，这辆卡车似乎经受住了这个四岁孩子扔过来的任何东西。包括把它滚下山坡，当然还有砸穿整栋大楼。它非常值得一看，让我们笑得合不拢嘴。

当然，有些镜头可能已经上演，但它仍然是一个非常令人印象深刻的显示如何强硬的沃尔沃卡车真的。我们想知道他们在拍摄这个的过程中损失了多少 GoPro…房子里可怜的 go pro 肯定没挺过来。

是的，你可以只使用一个 Arduino 来遥控一辆真正的汽车。
