# 周五黑客聊天:用机器制造机器人

> 原文:[https://hack aday . com/2018/05/02/Friday-hack-chat-building-robotics-with-the-mearm/](https://hackaday.com/2018/05/02/friday-hack-chat-building-robotics-with-the-mearm/)

在每个计算机科学或工程程序使用的储藏室的某个地方，有一个机器人手臂。它就在那里，你可能从未见过，但它就在那里。最初，这个非常昂贵的机械臂旨在成为一个真正非凡的教学工具，让学生学习反向运动学和控制系统。现在，最有可能的是，那个机器人手臂被灰尘覆盖，要么是因为手臂本身坏掉了，要么是因为唯一使用它的教练退休了。

[![](../Images/f21385453d4c0dc6c76206360c117096.png)](https://hackaday.com/wp-content/uploads/2018/05/mearm.png) 如今，几乎每个人都能接触到机械臂。Ben Gray 的 MeArm 是一个受欢迎的机器人手臂，由激光切割丙烯酸制成，由任何人都可以组装的业余爱好伺服系统提供动力。这是最小的可行的机械臂，本周的黑客聊天的[，我们将谈论所有关于机械臂的话题，它们能做什么，以及如何在教育中使用它们。](https://hackaday.io/event/156850-building-robotics-hack-chat-with-mearm)

在这次聊天中，我们将与 Ben 讨论反向运动学和制造机器人套件的来龙去脉。我们还将讨论本目前为让不同背景的人参与机器人教育所做的努力。将涵盖的主题包括:

*   设计和制造工具
*   机械臂
*   机器人套件
*   为黑客制造的机器人
