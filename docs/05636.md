# 使用水银和玻璃器皿的高真空

> 原文：<https://hackaday.com/2017/04/15/high-vacuum-with-mercury-and-glassware/>

如果你想制造自己的真空管，不管是放大管、谢妮管还是阴极射线管，你都需要一个真空。毕竟名字里就有。花上几千美元，你或许可以买到一台二手涡轮分子泵。但是他们以前是怎么制造高真空的呢？爱迪生是如何疏散他的灯泡的？

奇怪的是，你可以做得比在 YouTube 上寻找答案更糟:[Cody]演示了制作一个 Sprengel 真空泵。正如情报人员[BrightBlueJim]写给我们的，这个项目拥有一切:高真空，自制火炬玻璃器皿，以及大量有毒重金属。(不知何故，[吉姆]连续几天错过了水银滑下玻璃管所产生的静电产生的高压。)

[泵本身](https://en.wikipedia.org/wiki/Sprengel_pump)非常简单。汞滴抓住了容器中的气泡并被抽空，最终剩下的只有真空，至少直到汞开始沸腾。[Cody]甚至在液氮中冷却水银以降低其蒸汽压并获得更好的真空的实验。看起来它成功了，但是用他最基本的测量技术我们不能确定。

无论如何，如果你有几美元的玻璃管，几公斤的水银，几个多小时的等待，以及将水银提升到玻璃管顶部的肌肉，你就可以为自己制造一个真空泵，完全适合制造灯泡或谢妮管。[BrightBlueJim]建议将这个项目从汞升级到像镓这样毒性更小但密度仍然很高的物质。我们喜欢这个想法。

因此，如果你羡慕地看着别人制造自己的电子管，甚至 T2 制造自己的完整放大器，你现在就少了一个借口。
