# 美国空军拥有最大的舰队

> 原文：<https://hackaday.com/2016/07/15/retrotechtacular-the-us-air-force-has-the-biggest-fleet/>

当谈到航空的最高级时，有比 C-5 银河更大的飞机。[霍华德·休斯]的云杉鹅翼展最大，苏联和现在乌克兰的安东诺夫 AN-225 Mriya 载货量最大。当它在明年左右飞行时，Scaled Composites strato launch——一种设计用于将火箭拖至 30，000 英尺高的双体飞机——将成为翼展最大、载货量最大的飞机。

这些最高级的说法虽然完全准确，却没有意识到这些巨大的飞机是独一无二的。没有建造第二架同温层发射飞机的计划，而且安-225 的第二架机体正在野外生锈。如果你想找到一个巨大的飞机机群，只有一个竞争者:C5 银河，美国空军库存中最大的飞机。

这段视频来自大约 1968 年的美国空军档案，讲述了 C5 星系的设计、建造和运行。它涵盖了计划的开始，早期飞机的缺点，当然，完全忽略了 C5 最初的问题。

虽然 C5 不是当时最大也不是最重的飞机，但它拥有大量的技术创新。尤其是悬挂在机翼上的巨大的 Ge TF39 发动机。在这种发动机之前，涡轮风扇发动机的涵道比非常低。涡轮风扇发动机的低涵道比意味着低效率。TF39 改变了这一点，其涵道比是之前任何一种涵道比的四倍，引领了现代航空动力装置的发展方向。只要看看几张喷气式客机的老照片，你就会注意到老式飞机使用细长的发动机。这些仍然是涡轮风扇发动机，但效率非常低。看看现代客机，你会看到发动机短舱几乎和机身本身一样大。这些高涵道比涡轮风扇运行得更清洁、更高效，它们最早是在 C5 星系上发现的。

当然，C5 并非没有问题。对机翼的初步测试揭示了结构上的弱点，这导致空军降低了第一代 C5s 的载货量。这是由于机翼中较差的铝锻件造成的，虽然这在后来的机身中得到了解决(以及早期用新翼梁改装的机身)，但这个问题导致 C5 被称为一个巨大的失误。成本超支困扰着这个项目，国会调查了这个问题。

尽管有这些最初的问题，并呼吁取消该计划，C5 是一个非常成功的飞机。很少有其他飞机具有 C5 的有效载荷能力或体积，也没有其他更大的飞机。C5 将至少服役到 2040 年——距其问世已有 70 年——并将作为有史以来最成功的飞机开发项目之一载入史册。当你听说空军在新飞机项目上的最新混乱时，请记住这一点。
