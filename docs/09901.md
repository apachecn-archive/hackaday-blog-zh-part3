# 旋转谷物混合搭配定制早餐

> 原文：<https://hackaday.com/2018/06/28/carousel-of-cereals-mixes-and-matches-custom-breakfast-blends/>

有人认为一碗不起眼的谷类早餐是烹饪成就的顶峰。如果你一定要侧目，但是谷物迷是一种东西，他们有一个观点。神奇之处不仅仅在于往碗里倒一碗牛奶，而是在于知道哪些谷物可以混合在一起。

然而，当你可以自动化和定制你的谷物分配杂务时，谁还需要那些繁琐的混合呢？这就是凯文·奥博曼(Kevin Obermann)和阿德里安·伯恩哈特(Adrian Bernhart)在他们的谷类食品分发机(T1)上采用的方法，即使他们做得有些过火。激光切割的胶合板为现成的干货分配器形成了一个四站传送带，每个传送带都有一个步进电机来代替手腕扭转。最初的电机有点太弱，无法处理更崎岖的早晨选择，最终升级为齿轮电机。支持分配器的平台还包含所有的电子设备，包括一个 ESP32 来运行一切，并托管选择毒药所需的网络应用程序。加上 RGB LEDs，因为早餐应该看起来像锐舞。遗憾的是，该团队用完了 GPIO 引脚，无法运行添加牛奶所需的蠕动泵。不过，总会有 2.0 版本。

如果麦片不是你的自动早餐选择，我们理解。或许一台更像华莱士和阿高的早餐机会更好，或者 T2 一天中任何时候都可以享用机器人花生酱三明治。
