# 把汽车变成电脑鼠标

> 原文:[https://hack aday . com/2018/02/18/将汽车变成电脑鼠标/](https://hackaday.com/2018/02/18/turning-a-car-into-a-computer-mouse/)

[威廉·奥斯曼]和[西蒙娜·吉尔兹]之前都曾为我们的页面增光添彩，两人都有怪异、古怪和精彩的黑客行为，所以毫不奇怪，当他们聚在一起时，他们这样做是为了把西蒙娜的车变成一只电脑鼠标。这比你想象的要复杂。

他们开始用通常用于安全摄像头的镜头替换光学鼠标的镜头。令人惊讶的是，当安装到汽车的前保险杠上时，它竟然工作了！但并不理想。问题在于，要横向移动鼠标光标，你必须横向移动鼠标。然而，汽车不是侧向移动，而是以弧形转弯。现在不要让你的鼠标向侧面移动，让它呈弧形移动，看看会发生什么。屏幕上的鼠标光标在屏幕上上下垂直移动，但不能左右移动。那么如何辨别汽车是否在转弯呢？为此，他们增加了一个磁力计。然后鼠标给出汽车移动的距离，磁力计给出方向或角度。通过一些简单的三角学，他们计算出汽车的坐标。

鼠标点击是用汽车喇叭完成的，但细节是模糊的。

是的，使用 carmouse 就像听起来一样有趣，尽管我们仍然不建议开车时用这种技术发短信。在下面的视频中，观看他们写电子邮件和驾驶汽车自拍。

或许更安全但同样有趣的方法是[通过接入汽车的 CAN 总线，将方向盘、踏板和其他信息转换为操纵杆命令，将你的汽车变成游戏控制器](https://hackaday.com/2017/12/22/turn-a-car-into-a-game-controller/)。
