# 用 3D 打印的离心机提取 DNA

> 原文:[https://hack aday . com/2016/05/03/DNA-extraction-with-a-3d-printed-离心机/](https://hackaday.com/2016/05/03/dna-extraction-with-a-3d-printed-centrifuge/)

[F.Lab]真的很担心我们会在他们的 3D 打印离心机中从唾液、洗碗皂和外用酒精中准备一份 DNA 样本，然后像子弹一样喝掉它。也许他们从一次可怕的经历中学到了什么，也许生物学家有不同的饮食要求。不管怎样，他们的离心机真的很酷。不喝结果就好。(编者注:是外用酒精。)

离心机是按照草图设计的，然后进行 3D 打印。他们注意到要特别注意获得高质量的 3D 打印，这样转子就不会失去平衡。为了获得提取所需的高速，他们使用四轴飞行器上的无刷电机。这与 Arduino 和 ESC 结合使用。在 [Thingiverse](http://www.thingiverse.com/thing:1175393) 上有完整的装配说明。
