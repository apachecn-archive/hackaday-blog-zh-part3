# "闹钟吃了我的羽绒被，这就是我迟到的原因！"

> 原文:[https://hackaday . com/2017/08/03/the-alarm-clock-ate-my-duvet-cover-thas-why-im-late/](https://hackaday.com/2017/08/03/the-alarm-clock-ate-my-duvet-cover-thats-why-im-late/)

有些人就是醒不过来。闹钟不能解决问题，闪光灯不起作用，嘈杂的音乐只会成为一个无法穿透的梦的配乐。也许一个粗鲁地把被子从床上拉下来的闹钟可以解决这个问题。

或者没有，但[向上生活]决定尝试一下。他的机制非常简单——床脚下面的一个大桶，温暖舒适的床上用品可以缠绕在它的周围。闹钟在铃上安装了一个开关，告诉 Arduino 给鼓上弦，将你的睡眠状态暴露在严酷寒冷的世界中。老实说，事实上这是由一个 2000 磅的绞盘驱动的，可以毫不费力地肢解任何被卷入工程的人，这有点可怕。但是我们明白这个项目并不意味着是一个解决睡过头的实用方案；如果是这样的话，使用绞盘拉动底部的床单，将睡眠者从床上完全吐出来，可能会更好。

这款新像素日出钟可能会更温和地满足你睡过头的需求，自然地把你从床上哄起来。
