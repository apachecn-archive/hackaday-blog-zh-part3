# Reactor Forge 承诺为所有人提供感应加热

> 原文：<https://hackaday.com/2016/04/05/reactor-forge-promises-induction-heating-for-all/>

有没有想过在铁匠行业一试身手？建造一个铁匠铺既昂贵又棘手——尤其是如果你住在公寓里！但我们都是科技迷——用感应加热来加热你的金属进行锻造要酷得多。火是给穴居人的！[乔什·坎贝尔]正在开发一套工具，将感应加热带给大众——他称之为[反应堆熔炉。](http://reactorforge.com/)

工具包还没有发布，但是你可以在他的 GitHub 上关注他的进展。感应加热通过磁感应电流进入金属来工作，电阻将电流转化为热量，而无需物理接触。[乔希]建造的反应堆利用 220 伏的电路，在下面的演示中，加热了一段 6 英寸长的 1/2 英寸的钢条。

当酒吧是冷的时候，感应加热器消耗大约 1500 W，但是一旦它开始加热，它消耗越来越多，一旦加热，最高可达 6500 W。

就感应加热器而言，这个项目看起来相当巧妙。但是我们喜欢这个我们几年前报道过的老式感应加热器。它用一个*肯德基鸡桶*作为炉膛的混凝土模具！
