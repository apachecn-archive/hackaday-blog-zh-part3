# Kickstarter 上的虚假声明:有什么新内容？

> 原文：<https://hackaday.com/2017/07/30/false-claims-on-kickstarter-whats-new/>

当谈到“世界上最*(在此插入功能)*的设备”时，Kickstarter 和它的同类看起来像是狂野的西部当你真的可以说你拥有一项设备的世界纪录时，这确实增加了一些特别的东西，而且[梅尔贝尔电子公司]目前正在运行一个 Kickstarter，声称是[世界上最小的 Arduino](https://www.kickstarter.com/projects/melbel/pico-the-worlds-smallest-arduino-board/) 兼容板，名为 Pico。

我们不想过多地抨击他们，他们看起来像是一个合法的 Kickstarter 活动，在撰写本文时，他们已经将目标翻了一番，但在观看了他们的宣传视频，检查了他们的 Kickstarter，并进行了大约几分钟的研究后，他们声称是世界上最小的 Arduino 兼容板似乎已经被揭穿。Pico 的尺寸为令人印象深刻的 0.6 英寸。x 0.6 英寸。总面积为 0.36 平方英寸，这没什么可嗤之以鼻的，但我们在 2014 年写的 [Nanite 85](https://hackaday.com/2014/05/07/the-smallest-attiny85-based-usb-board/) 的尺寸大约为 0.4 英寸。x 0.7 英寸，总面积约为 0.28 平方英寸..在我们生活的这个事后造假的世界里，这真的重要吗？我们在吹毛求疵吗？还是 Pico 团队对事实和真相有点太快太松了？

可能会有更小的 Arduino 兼容板，这只是两者之间的一个案例研究。我们认为，在大胆宣称“世界上最小的”或类似的东西时，进行简单的谷歌搜索只是为了确定可能是一个想法。
