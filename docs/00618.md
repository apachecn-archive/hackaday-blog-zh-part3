# 自折叠石墨烯纸

> 原文:[https://hack aday . com/2015/11/13/自折叠-石墨烯-纸/](https://hackaday.com/2015/11/13/self-folding-graphene-paper/)

折纸，一种将纸折成各种形状的艺术，是最新的自动化工艺。中国的研究人员在科学进展发表了一篇论文，描述了他们如何创造出能够自我折叠的石墨烯纸。根据他们的论文(即他们写的论文，而不是他们的石墨烯论文)，新材料可以采用预定义的形状，行走，甚至转弯。

像形状记忆聚合物这样的活性材料并不新鲜。但是使用这种材料存在许多实际问题。使用 MGMs(宏观石墨烯材料)，研究人员创造了可以基于光改变形状的纸张。温度或湿度。

下面的视频展示了一些用途，包括一个自动折叠的盒子，一个蠕虫状的运动装置，以及一张做抓握动作的手状纸。创造者提到有广泛的应用，包括机器人，人造肌肉和传感设备。看完视频，我们不禁在想，一朵在阳光下开放的纸花会有多酷。

我们已经讲述了如何在家庭实验室里制作石墨烯，甚至在 DVD 刻录机里制作 T2。我们很想知道谁是第一个破解石墨烯纸的人，以及你会用它做什么。
