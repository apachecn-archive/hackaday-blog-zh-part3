# 便携式雅各布的梯子当…你知道…你需要一个便携式雅各布的梯子

> 原文：<https://hackaday.com/2017/06/09/portable-jacobs-ladder-for-when-you-know-you-need-a-portable-jacobs-ladder/>

什么时候需要便携式的雅格布梯子？我们不知道，但显然[mitxela]不想离开家(或实验室)没有一个。因此，他制造了一个便携式装置，可以用电池工作几分钟。在视频中(见下文)，他说他不会冒昧地声称这是有史以来最小的雅各布之梯，但他认为它可能是一个竞争者。

这种电池是一种脂肪电池，虽然它可能持续四分钟，但[mitxela]指出，尽管他安装了散热器，晶体管可能不会持续那么长时间。整个装置 45 毫米见方，17 毫米厚。当然，电线增加了一些高度(总共约 150 毫米)。

我们希望看到更多的内部，但我们认为这使用了一种便宜的高压模块，你可以从通常的远东来源获得，或者至少可以。剩下的只是激光切割和做工。

如果你以前没有遇到过它们(除了老怪物电影)，雅各布的梯子让高压电离狭窄缝隙底部的空气。电离的空气是热的并上升，电流流过它，尽管电极离得越来越远。当然，这意味着你不应该在你的零重力空间站穿上。

你可能会认为便携式雅各布梯子是独一无二的。原来，我 [t 不是](https://hackaday.com/2017/01/08/a-portable-jacobs-ladder/)。如果你想要一些简单的东西(也许没有那么便携)，你不可能比[这个](https://hackaday.com/2016/08/29/really-easy-jacobs-ladder/)更简单了。
