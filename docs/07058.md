# 多普勒模块拆卸揭示了微波电子的怪异世界

> 原文：<https://hackaday.com/2017/09/11/doppler-module-teardown-reveals-the-weird-world-of-microwave-electronics/>

振荡器的元件不与任何东西电连接？PCB 走线仅根据其形状就能充当无源元件？板子上有特定功能的槽孔？欢迎来到微波电子的奇妙世界，通过对多普勒微波收发器模块的拆卸和分析[带给你。](http://www.kerrywong.com/2017/09/01/teardown-and-experiments-with-a-doppler-microwave-transceiver/)

随着频率的增加，传统的电子规则是如何被打破的，对此我们一直很感兴趣。[Kerry Wong]选择打开的多普勒模块是一个微型 X 波段收发器，目前在易贝售价约为 10 美元，内部几乎没有组件。一个晶体管用于本地振荡器，一个用于混频器，大约三个其他无源器件是整个 BOM。LO 是由充当介质谐振器的钛酸钡芯片调谐的，这一点非常有趣，因为 PB 走线可以凭借其尺寸和形状形成完整的滤波器网络。天线通过电路板上的插槽通过气隙耦合到收发器也是一个巧妙的技巧。

[Kerry]在下面的视频中分析了所有这些，并展示了如何将该模块用作传感器。如果你需要更多关于这些模块工作的细节，[我们有一些基本电路](https://hackaday.com/2017/03/31/the-right-circuit-turns-doppler-module-into-a-sensor/)你可以看看。
