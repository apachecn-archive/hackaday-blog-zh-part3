# 地铁上的 3D 打印；或者其他地方！

> 原文：<https://hackaday.com/2017/10/04/3d-printing-on-the-subway-or-anywhere-else/>

3D 打印的可穿戴电子产品正在兴起，然而我们自己的[Naomi Wu]将其翻转过来，制成了一台[可穿戴 3D 打印机](https://www.youtube.com/watch?v=i-ksb-gzfyE)，它不仅便携，而且可以在移动中制造！

该项目从当地购买的婴儿背带开始，多余的脂肪被剪掉，只留下主要的背带和方形框架。这个方形框架保持完整，以便为安装的打印机提供稳定性，并为佩戴者提供一定程度的舒适性。[Naomi]然后在有问题的 delta 打印机上钻一些新孔，幸运的是顶部是塑料做的。使用旋转螺钉和长螺钉，上部与线束连接。上部的插座夹也是 3D 打印的，为机器提供了模块化的刚性夹具。

下部也使用 3D 打印的三角形底座，该底座有一个用于托架的插槽，该托架使用螺钉与三角形的底部相连。该项目通过两节 3 Ah 电池供电，使用 PLA 定制的夹子将电池固定在打印机后面。整个项目在移动中进行，正如下面视频中的[内奥米]所展示的那样。

从解剖婴儿背带到用焊枪加热的螺丝刀在背带上打孔，这个项目有很多 DIY 的成分。对于那些寻找更高效的电动可穿戴设备的人，请查看为残疾人增加触觉反馈的。
