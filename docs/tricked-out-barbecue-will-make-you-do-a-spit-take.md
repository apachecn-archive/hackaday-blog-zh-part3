# 精心设计的烧烤会让你大吃一惊

> 原文：<https://hackaday.com/2018/01/20/tricked-out-barbecue-will-make-you-do-a-spit-take/>

不管外面天气如何，他的朋友都喜欢烧烤。但似乎总有一些事情会打断这种乐趣:要么是该起床翻肉了，要么是因为某人的手机没电了，音乐停止了，要么是夜幕降临，没有足够的手电筒或充电手机。他有了建造瑞士军刀烧烤的想法，[一个解决所有这些问题和更多问题的便携式发电站](https://geektimes.ru/post/296933/) ( [翻译](https://translate.google.com/translate?sl=auto&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=https%3A%2F%2Fgeektimes.ru%2Fpost%2F296933%2F&edit-text=&act=url))。

最重要的是，E-Mangal 可以旋转烤串，均匀烹饪。它通过一个 3D 打印的蜗轮系统来实现这一点，该系统由汽车上的加热器挡板致动器驱动。慢转 25 分钟后，一个声音宣布该吃饭了。[Strn]朋友们永远不会因为预装曲目、蓝牙音频、FM 天线、USB 和 SD 选项之间的音乐选项而受到伤害，这些选项通过 3W 放大器运行。两个 USB 灯照亮夜间烧烤，10 Ah 电池可以做到这一切*和*保持每个人的手机充电。为了安全起见，[Strn]包括一个半升的水箱，通过喷射流熄灭煤炭。一切都由 PIC18F 运行，可以在机器上或通过简单的网络界面进行控制。

我们喜欢这款烧烤控制器的外观，就像喜欢它的功能一样。这些短而有角度的腿的坚固姿态给了它一种中世纪器具的感觉，看到所有展出的内脏总是一种加分。休息过后，请抓紧时间参观。

E-Mangal 在煤箱中有一个热电偶来测量温度，但没有直接控制。如果你对温度选项比对娱乐更感兴趣，[这里有一个项目可以对烤架上的一切进行微观管理](https://hackaday.com/2017/09/28/pitmaster-bbq-dashboard-monitors-your-meat-and-veggies/)。
