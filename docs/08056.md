# 牛肉机器人:你的机器人烧烤师傅

> 原文：<https://hackaday.com/2017/12/30/beefbot-your-robotic-grill-master/>

你有没有因为太忙而无暇顾及牛排的烹饪？别再悲伤了，提醒你的心脏病专家。康奈尔大学的三名学生设计并制造了烤牛排机器人，让你的美味晚餐梦想成为现实。

电子和计算机工程专业的学生乔纳·米特勒、凯尔西·内德和马丁·埃雷拉应该感谢这位机器人厨师。它的工作原理如下:在将牛排串在机器人的叉子上后，BeefBot 将牛排放到烤架上，并以只有经验丰富的烧烤大师才能复制的方式监控内部温度。一旦达到设定的温度，牛排就会翻转——抱歉，这里没有交叉的烤痕——并烹饪到想要的熟透程度。如果你想照看 BeefBot，一个小屏幕会显示温度——牛排翻转后可能需要一些手动调节，以确保烹饪均匀——但除此之外，这是一件不干涉的事情。如果你不介意对着你的屏幕垂涎三尺，休息之后看看项目演示。

乍一看，你可能认为这是 YouTube 的噱头，但这是真正的科学。从设计和制造，到温度校准和调节背后的数学，这本书写得很精致。向饥饿的康奈尔学生致敬，他们在一个热煎锅上辛勤工作，将这个变成了现实！
