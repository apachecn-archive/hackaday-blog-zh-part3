# 浏览器中的开源参数化 CAD

> 原文：<https://hackaday.com/2016/12/24/open-source-parametric-cad-in-your-browser/>

直到最近，计算机辅助设计(CAD)软件实际上只被每年支付数千美元的工程公司使用。可用的软件虽然非常强大，但学习曲线非常高，需要大量的培训和经验才能掌握。但是，随着 3D 打印爱好者的兴起，许多更简单的 CAD 程序变得可用。

虽然这些程序确实帮助制造商进入 3D 建模领域，但大多数都有严重的局限性。只有少数是真正开源的，更少的是既开源又参数化的。参数化 CAD 允许您基于一系列参数创建 3D 模型，例如通过其原点和尺寸定义立方体。这与雕刻风格的 3D 建模软件形成对比，后者的控制更加直观。参数化建模的好处是参数可以在以后更改，并且模型可以动态更新。特征也可以用数学方法来定义，因此它们之间的关系会发生变化。

虽然还处于起步阶段。Sketcher 正在寻求填补这个空白。它是 100%开源的，仅使用 JavaScript 在您的浏览器中运行，并且是完全参数化的(具有约束和可编辑的尺寸)。目前，可用的功能仍然非常有限和简单。您可以:挤出/切割、旋转、抽壳和对实体进行布尔运算。更多的高级功能还不可用，但希望将来会增加。

如果您正在寻找一个可以在浏览器中运行的参数化开源 CAD 程序，这就是它。这已经足够了，您可以使用它进行真实世界(尽管很简单)的建模。然而，计算机辅助设计仍然需要某种类型的空间思维和推理。所以，如果你是 3D 建模世界的新手，可能值得尝试一下更面向学习的工具，比如 BlocksCAD 。
