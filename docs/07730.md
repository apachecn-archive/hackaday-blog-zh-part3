# 你的下一个可穿戴设备可能不需要电力

> 原文：<https://hackaday.com/2017/11/17/your-next-wearable-may-not-need-electricity/>

如果你能用你的衬衫袖子打开一扇门，或者在你的领带上写一条秘密信息，会怎么样？这可能很快就会成为现实，因为华盛顿大学的研究人员创造了一种无需任何电子设备就能存储数据的织物。织物可以洗涤、干燥，甚至熨烫而不会丢失数据。哦，它比 RFID 便宜多了。

通过利用导电线的铁磁性，[陈家]和[希亚姆·戈拉科塔]已经证明了通过磁化存储位串和 2D 图像的能力。该团队使用刺绣机将线分成密集的条状和片状，然后通过用 N 和 S 钕磁铁摩擦线来编码 1 和 0。

他们也没有使用任何特殊的东西，只是使用了这根导电线、一些磁铁和一个 Nexus 5 来读取数据。任何带有磁力计的手机(大多数都是如此)都可以解码这种类型的二进制数据。这些线可靠地保持磁化大约一周，然后开始变弱。然而，他们的试验证明，这些线可以反复重新磁化。

该团队还在一块由导电织物制成的 9 片式磁铁上制作了 2D 图像。这些图像可以由单个磁力计逐个解码，也可以由一组磁力计同时解码。最后，研究小组制作了一只手套，指尖上有一片磁化的线。他们能够让手机以 90%的准确率识别六种独特的手势，即使手机藏在口袋里。休息之后，请观看他们的演示视频。

[磁记忆当然不是什么新概念](https://hackaday.com/2014/10/14/retrotechtacular-core-competencies/)。但对于可穿戴技术前沿来说，这是一个新奇的概念。
