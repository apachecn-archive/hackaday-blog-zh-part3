# 天花板瓷砖放弃它的圣诞 LED 饰品

> 原文：<https://hackaday.com/2016/12/25/ceiling-tiles-give-it-up-for-christmas-blinkies/>

假期的伟大之处在于，它们似乎总是需要一些闪亮的东西。发光的东西的现代版本似乎是发光二极管和技术的进步，他们是什么，我们现在有惊人的可编程发光二极管。可编程发光二极管意味着动画闪亮的东西！几年前，[wpqrek]用分立元件制作了一个 LED 装饰品。今年，他重新审视了他的装饰品，并决定制作一个新的、动画的 RGB 装饰品。

[Wpqrek]的构建基于连接到 Arduino Pro Mini 的大约五个 WS2812b 条。装饰品本身是一块厚厚的泡沫聚苯乙烯天花板，被切割成星形，带有一个红色的木制框架。装饰着小玩意和星星的 LED 灯条从中心开始，在星星的每一点结束。随着每一条并行连接到 Pro Mini，[wpqrek]使用 [Arduino Light Animation](http://yaab-arduino.blogspot.com/p/ala.html) 库来处理动画。

[Wpqrek]说这个结果对他的树来说太大了，所以他把它作为一个独立的装饰品。也许使用更轻的材料会有帮助——或者得到一棵更大的树！查看 [Arduino 照明控制器](https://hackaday.com/2016/12/15/arduino-lighting-controller-with-remote-twist/)或 [Trompe-l'oeil Menorah](https://hackaday.com/2014/11/20/the-trompe-loeil-menorah/) 以获得更多假日秘籍。
