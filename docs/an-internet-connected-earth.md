# 互联网连接地球

> 原文：<https://hackaday.com/2015/10/24/an-internet-connected-earth/>

世界上超过一半的人口没有互联网连接，这并不奇怪。很难得到一个准确的数字，但是没有连接的人数通常被认为是地球人口的 2/3。有一些重量级人物正在研究这个问题，并提出了一些非常有趣的解决方案。

OneWeb 是一个由理查德·布兰森领导的团队，他们计划发射低轨道卫星与地面终端通信。地面终端将转播来自卫星的通信信号，从而为地面终端附近的人产生 2G、3G、LTE 和 WiFi 信号。

SpaceX 在谷歌和富达 10 亿美元的资助下，开始了它的征程。

也许最令人惊讶的是[扎克伯格的]太阳能互联网激光光束无人机。这个想法是，这些激光鸟将在互联网死区上空盘旋，就像秃鹰在垂死的水牛上空盘旋一样(达到？)并为下面的人提供连接。太阳能无人机将在 20 公里的高度飞行，这是一个非常好的高度，据信它们可以一次飞行数月。休息过后，有一位脸书大学的博士在视频中对此进行了解释，谢谢脸书博士。
