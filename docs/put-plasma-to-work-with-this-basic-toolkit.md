# 用这个基本的工具包来使用等离子体

> 原文：<https://hackaday.com/2017/04/21/put-plasma-to-work-with-this-basic-toolkit/>

公平的警告:[Justin Atkin]关于如何制造等离子体、熔丝和磁控管的视频有点长。但它值得观看，因为他正在为一系列等离子体实验奠定基础，看起来会很有趣。

在对等离子体的物理学做了一个很好的介绍后，[Justin]进入了一些关于这个行业的基本工具的细节:高电压和高真空。几个废弃的微波炉变压器、一个桥式整流器和一个电容器提供所需的 2000 伏 DC 输出。这是一个可行的设置，但我们会反对[由【随机之王】推广的极其危险的“scariac”自耦变压器](https://www.instructables.com/id/The-Scariac-Poor-Mans-Variable-Power-Controller/)。当[100 美元就能买到一个 20 安培的自耦变压器](https://www.circuitspecialists.com/variac-tdgc2-2.html)时，冒混合水和线电流的痛苦死亡的风险似乎是愚蠢的。

当然，还需要一个像样的真空泵；也许你可以通过自己制造斯普兰格尔真空泵节省下来的钱可以用于电力预算。真空室也很便宜——带有磨边和钻孔的梅森瓶，用于火花塞等附件。安装在一个腔室下面的磁铁形成了一个基本的磁控管，幸好没有产生微波所需的共振腔。另一个实验尝试氮化钛的气相沉积。

这都是很酷的东西，我们期待更多的细节和结果。在我们等待的时候，请随意查看我们精选的大量等离子项目，从[微型等离子扬声器](http://hackaday.com/2016/03/27/555-plasma-speaker/)到[巨型等离子管](http://hackaday.com/2016/10/28/giant-plasma-tube-makes-a-great-attention-getter/)。
