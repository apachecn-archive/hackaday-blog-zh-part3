# 多少个 led 才算多？

> 原文:[https://hackaday.com/2016/03/12/how-many-leds-are-too-many/](https://hackaday.com/2016/03/12/how-many-leds-are-too-many/)

"你应该回答一个反问句吗？"但不管怎么说，答案是 led 永远不够用。至少 maniacallabs 的亚当·黑尔(Adam Haile)似乎是这么认为的。到目前为止，[他已经达到了 3072](http://maniacallabs.com/2016/02/29/wyomaniacal-display-nova-maker-faire/)。

我们已经在之前[报道过【亚当】的先前大 LED 建造，称为“巨像”。虽然目前的显示器体积更小，但它有更多的 led。这意味着更多，嗯，其他的一切。全开时重量约为 500W，具有 175 部分 3D 打印框架和扩散器元件，由三个驱动移位寄存器的 Teensy 3.2 微控制器驱动，该显示器每秒能够输出 60 帧令人目眩的 RGB LED 亮度。](https://hackaday.com/2015/10/04/building-a-big-led-display/)

设计、适配器板和动画代码将在他们“有机会稍微清理一下”后发布。这里希望这是很快！【编辑:[代码和设计在这里](https://github.com/ManiacalLabs/WyoManiacalDisplay)。谢谢亚当。]

如果你在大华盛顿 DC 地区，你甚至可以去莱斯顿的 [NoVA Maker Faire](http://makerfairenova.com/) 亲自看看。如果有，告诉他们是黑客日派你来的。
