# 危险玩具屋教授消防安全

> 原文:[https://hack aday . com/2017/10/22/hazardous-doll house-teaches-fire-safety/](https://hackaday.com/2017/10/22/hazardous-dollhouse-teaches-fire-safety/)

消防安全从小就灌输给我们。这也是有充分理由的，因为火灾隐患在普通家庭中无处不在。即使是一场小火灾也能在几分钟内变得极其危险。但是，你如何让孩子们真正注意到可怕的(通常是无聊的)成人概念呢？你可以教一个孩子停下来、放下去、打滚，直到你脸色发青，但仍可能无法理解防火的重要性。像这样的主题需要儿童尺寸的视觉教具来激发想象力。

这正是波兰波兹南的消防员与当地软件公司 mlabs 合作所做的事情。他们制作了一个移动的、互动的消防安全教育工具，非常详细地模拟了常见的家庭火灾危险。这无疑是我们见过的最华丽的玩具屋。许多不同的危险场景通过触摸屏使用定制的应用程序进行控制。按下按钮，房子就变成了死亡陷阱。灯光照亮的危险发出逼真的不同强度的光，而且有烟从其中冒出来，触发烟雾探测器。嵌入整个房子的摄像头在附近的监视器上提供了恐怖的第一人称视角。

对于居住在这个复杂的 1:12 比例的住宅中的小雕像家庭来说，几乎没有一个房间是安全的。食物在炉子上烤着，爸爸在厨房里袖手旁观。奶奶坐在楼上的床上，她忘了带的香烟在羽绒被上烧了一个洞。女儿用她所有的小发明在她卧室的电源插座上超载。闷烧的煤从杂物间翻倒的炉子中溢出。

这不是我们看到的第一个[智能玩具屋](https://hackaday.com/2016/09/06/ritas-dolls-probably-live-better-than-you-do/)，但它可能是最有趣的。消防安全玩具屋本周在波兹南的 POL-EKO 系统年度环境博览会上展出。不在波兰附近？休息之后请看视频。
