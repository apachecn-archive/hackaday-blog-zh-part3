# Hackaday 奖参赛作品:一台核磁共振成像仪

> 原文：<https://hackaday.com/2016/06/30/hackaday-prize-entry-an-mri-machine/>

磁共振成像设备是人类有史以来建造的最不可思议的机器之一。他们能够通过磁场翻转质子来产生活组织的三维图像。百分之九十的人不知道这句话是什么意思，然而你可以在美国几乎任何一家设备合理的医院里找到一台核磁共振仪。

为了他的 Hackaday 奖参赛作品， [[Peter Jansen]正在建造一台磁共振成像器](https://hackaday.io/project/12352-low-field-mri-continued)，能够产生与你从医院放射科获得的图像相同类型的图像。这将是一个桌面单元，能够扫描水果和其他类似大小的物体，并且可以使用不比热风枪和激光切割机更先进的工具来制造。

这个项目是去年[Peter]的 Hackaday 奖参赛作品的延续。去年夏天他的事情变得很忙，他退出了 Hackaday 奖，这意味着欢迎他今年继续他的建设。

去年，[Peter] [开发了胶合板机制](https://hackaday.io/project/5030-low-field-mri/log/16020-measuring-magnetic-fields-in-a-volume)，该机制将在扫描体积的直径上旋转磁传感器，旋转待扫描的对象，并提升对象通过该体积。基本上，这是一台奇怪的 3 轴数控机床，但磁性传感器附近的零件不能由金属制成。牙线工作正常，但如果我们碰到[彼得]，我们有几百英尺的光谱钓鱼线。磁共振成像也意味着大线圈，这意味着将线圈缠绕在圆柱体上的繁琐任务是建造的一部分。【彼得】[造了一台机器替他做工作](https://hackaday.io/project/5030-low-field-mri/log/38706-back-at-it-with-coil-winding)。

这不是[彼得]第一次尝试制造成像设备。他建造了一台桌面 CT 扫描仪，速度非常慢，但确实可以通过水果发射辐射来产生图像。他在 Hackaday.io [上的第一个项目是开源科学 Tricorder](https://hackaday.io/project/1395-open-source-science-tricorder) ，它是 Hackaday 奖第一年的前五名决赛选手之一。

[Peter]已经有了一些惊人的成果，产生了无法通过其他方式获得的真实数据。开源 MRI 是 Hackaday 奖公民科学阶段的完美项目，我们很高兴看到他加入这个项目。
