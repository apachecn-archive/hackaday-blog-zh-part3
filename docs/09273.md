Majenta Strongheart [来看看 2018 年](https://www.youtube.com/watch?v=ES4HXvQXYVo)[黑客日奖](https://hackaday.io/prize)第一轮的几个很酷的参赛作品:

[这是一台无限 3D 打印机](https://hackaday.io/project/114738-autonomous-infinite-3d-printing-workhorse-3d)。主力 3D 是我们将 3D 打印大众化的方式。重型 3D 打印机通过在 3D 打印机的底座上添加传送带来实现这一点，允许快速*制造*，而不仅仅是原型制作。[Swaleh Owais]创造了重型 3D 打印机，可以自动开始打印，制造一个对象，然后从打印床上移除该打印，重新开始整个循环。

[查看来自【Yann Guidon】](https://hackaday.io/project/26068-numitron-hexadecimal-display-module)的这个 Numitron 十六进制显示模块。[Yann]正在从头开始构建一台完整的计算机，他需要一种方法来显示总线上各个位的状态。最简单的方法是用一些缓冲芯片和一些发光二极管，但这对[Yann]来说太容易了。他决定使用数码管来计算总线上的位数，从 0 到 f。他没有使用微控制器，而是使用继电器和二极管控制来打开和关闭数码管的这些部分。
