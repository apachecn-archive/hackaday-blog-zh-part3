# 另一个桌面 LED 圣诞树！

> 原文:[https://hack aday . com/2016/12/18/another-desktop-led-xmas-tree/](https://hackaday.com/2016/12/18/another-desktop-led-xmas-tree/)

我们喜欢有人从我们的帖子中获得灵感，并提出自己的见解。[Matthew]非常喜欢他在 Hackaday 上看到的一个构建，[他构建了自己的 LED 桌面圣诞树](https://www.youtube.com/watch?v=HHse2RrE_RY)！

[Matthew]的灵感来自于[designer2k2]在十月份发布的 [DIY 桌面圣诞树](https://hackaday.com/2016/10/30/just-in-time-for-christmas-a-diy-desktop-led-tree/)。首先，他在阿里快递上找到了一组同心的 WS2812 环。五个环总共 93 个 led，加上一个单独的 WS2812 用于树的顶部。他还从 Thingiverse 获得了一个激光切割树模型，并将其切割，在最终产品中结合了 LED 环和树

整个项目在 DigiStump 的 Digispark USB 开发板上运行，与最初的项目相同。视频中没有太多细节，但[Matthew]放了一些链接，指向他从哪里得到戒指和树，激光切割服务，DigiStump 网站的链接，以及[designer2k2]最初的树项目的链接。目前还没有源代码，但是[Matthew]说会有更多的图片和链接。
