# 自动百叶窗打开了我们心灵的窗户

> 原文：<https://hackaday.com/2016/04/21/automated-blinds-open-the-window-to-our-heart/>

[Brian Harms]使用伺服系统、Arduino 和 SmartThings Arduino shield 使他的客厅窗户自动打开和关闭。最重要的是，它与他的亚马逊 Echo 相连，因此只需说“Alexa，打开/关闭百叶窗”就可以打开和关闭它们。

为了完成这一壮举[布赖恩]使用了两个激光切割丙烯酸齿轮；其中一个连接在伺服喇叭上，另一个连接在百叶窗的长杆上。尽管使用了体积庞大的 Arduino 和 shield，但成品并不显眼，呈流线型，单个 Arduino 就控制了客厅的全部三个百叶窗。[Brian]在[Reddit 帖子](https://www.reddit.com/r/DIY/comments/4eb0fh/i_made_my_dumb_living_room_blinds_smart/)上回答了一堆问题。

百叶窗是一种常见的联网家庭黑客，虽然我们过去报道的黑客都不是语音激活的，但我们已经看到了[温度敏感百叶窗](http://hackaday.com/2013/10/07/temp-sensitive-automatic-blinds/)和[基于树莓 Pi 的解决方案](http://hackaday.com/2013/12/07/raspi-ac-and-blinds-controller/)。
