# 由电视柜打造的电动转盘

> 原文：<https://hackaday.com/2017/08/03/motorized-turntable-created-from-tv-stand/>

[Robin Reiter]有一个电动电视支架，只能旋转 20°，因为谁真的需要他们的电视完全旋转？他想把它变成一个用于拍摄视频的[电动转盘](https://hackaday.io/project/26187-diy-motorized-turntable-from-an-old-tv-stand)，但首先他必须黑掉它。

打开后，罗宾发现底座里有数量惊人的电子设备。除了 DC 电机，还有一个连接到齿轮上的电位计来提供反馈，但它被设置为部分旋转，所以必须被拉出来。

还有一个塑料齿轮，它的齿只围绕着内部的一部分。[Robin]拍摄了齿轮的照片并将其放入 Fusion360，在他重新创建齿轮时使用照片作为参考图像。新的那块在四周都有齿。打印出来后，他把它粘在旧变速箱上，现在他已经把他的电视柜变成了一个电动转盘。
