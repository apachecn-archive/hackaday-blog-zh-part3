# 由聚脂薄膜压制而成的抛物面镜

> 原文：<https://hackaday.com/2016/07/26/pressure-formed-parabolic-mirror-from-a-mylar-blanket/>

抛物面反射器是非常方便的装置。无论你是在建造微波天线还是长距离定向麦克风，合适的商用天线都不难找到。但是，一个大而闪亮的镜子来满足你的太阳死亡射线需求是另一回事，这就是这种压力形成的太空毯镜可能派上用场的地方。

压力成型是[夜鹰之光]镜子的绝佳选择。我们之前已经[覆盖了压力成型的塑料圆顶](http://hackaday.com/2016/03/22/quick-and-easy-pressure-forming-makes-plexiglas-domes/)，这个过程是相似的。带有凹进式空气配件的 PVC 板形成压板。金属化聚脂薄膜太空毯，在木制框架上伸展以拉出皱纹和折叠，被应用到压板上的一圈环氧树脂上。固化后，用自行车轮胎打气筒吹几下就形成了曲线，使薄膜拉伸得更加平滑。[NighthawkInLight]第一次尝试用喷涂泡沫绝缘材料支撑薄膜是失败的，但后来尝试用玻璃纤维网效果很好。一个小小的边缘支撑着闪亮的玉米卷外壳，镜子能够达到所需程度的破坏力。

我们怀疑这一过程能否优化到足以生产天文级可见光反射镜，但它仍然有很多潜在的应用。也许玻璃纤维射电天文盘可以用这样的设备直接压力成型？
