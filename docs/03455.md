# 让蠢朋克嫉妒的头盔

> 原文：<https://hackaday.com/2016/09/06/a-helmet-to-make-daft-punk-jealous/>

如果你在过去的几十年里对流行音乐有一点点关注，那么你肯定知道电子音乐二人组*蠢朋克*。当然，他们的成功不仅仅是因为他们的音乐，很大一部分也是因为他们标志性的服装和角色。让这些服装具有标志性的是音乐家们戴的机器人头盔。最初是想遮住他们的脸，结果变成了他们最与众不同的特征。

二人组戴的头盔多年来一直在变，但是由迈克·米凯莱纳设计的一款致敬头盔让他们都相形见绌。它保持了蠢朋克头盔的美学元素，同时在各个方面对技术进行了改进。210 个 RGB LEDs、一个微处理器和 14 安培小时的电池使它完全可定制，可使用 5 小时。

MIDI 序列可以被编程来控制灯光，并可以自动与音乐播放的节拍同步。它甚至创建了自己的 WiFi 热点，以便用户可以用智能手机控制它。正如你在下面的视频中看到的，结果相当壮观。我们在过去展示过其他的蠢朋克头盔，但这是我们见过的最专业、功能最丰富的一款。
