# 简单，模块化的字母数字显示器充满了 Flappy 的好处

> 原文:[https://hack aday . com/2018/02/20/easy-modular-alphanumeric-displays-are-full-of-flappy-goodness/](https://hackaday.com/2018/02/20/easy-modular-alphanumeric-displays-are-full-of-flappy-goodness/)

有很多方法可以制造出远距离可读的大型字母数字显示器。我想到了 LED 招牌，还有大屏幕液晶显示器。但是这样的显示感觉有点没有灵魂，没有什么比由数百个对开显示组成的到达和离开显示板更能捕捉繁忙火车站的气氛了。

为了让这些嘈杂但有趣的显示器对家庭游戏玩家实用，[Scott Bezek]在过去几年里花了[一个简单的模块化分体式显示器单元](https://scottbez1.github.io/splitflap/)，从下面的视频看，它非常接近就绪。[构建日志](https://imgur.com/a/0VAMZ)详细描述了设计过程，从 OpenSCAD 开始，利用脚本语言的参数化特性，在合理的范围内支持任意数量的字符。激光切割的中密度纤维板框架和传动装置保持低成本，廉价的步进机提供运动。字符卡只是带有乙烯基字母的 PVC ID 徽章，一个简单的光传感器可以防止遗漏的步骤和错误的字符。这些模块可以链接在一起成为多字符显示器，声音令人满意地轻快。

[Scott]在这些显示器上花了很多心思，即使它不是我们见过的最简单的分开式显示器，它也确实值得一试。
