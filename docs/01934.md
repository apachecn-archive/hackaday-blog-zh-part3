# 放眼全球，用这些开源回收机器在本地制造

> 原文：<https://hackaday.com/2016/04/03/think-globally-build-locally-with-these-open-source-recycling-machines/>

走在几乎所有的海滩上，或者看着大多数的路边，你会看到瓶子、袋子和聚合物字母汤的废弃碎片——HDPE、PET、ABS、PP、ps。市政回收项目可能会有所帮助，但真正解决问题的是分散回收，这些开源塑料回收机可能会推动这一努力。

[我们两年前关注过【Precious Plastic】](https://hackaday.com/2014/02/06/plastic-recycling-at-home-promises-a-revolution-in-local-plastic-production/)，从那时起，他们的小型塑料回收机开源计划已经取得了很大进展。它们目前包括碎纸机、压塑机、注塑机和长丝挤出机。该计划规定了一些需要定制的部件，如粉碎机的激光切割不锈钢齿，但大多数可以从废品场获得。正如你从休息后的视频中所看到的，金属和电子制造技能是假定的，但构建是大多数黑客力所能及的。更多机器的计划正在进行中，在设计上有足够的扩展和改进空间。

我们认为[珍贵塑料]在这方面有所发现。也许大量的小型回收公司是一个比大规模的市政努力更好的方法，后者似乎没有多大帮助。分散回收可以创造大规模制造业懒得去开发的市场，尤其是在发展中国家。毕竟，我们已经看到[一个用回收零件建造的塑料回收工厂](https://hackaday.com/2015/09/02/recycled-factory-recycles-soda-bottles/)在巴西制造很酷的东西。
