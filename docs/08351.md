# 物联网门卫:脑性瘫痪女孩的眼控门

> 原文:[https://hack aday . com/2018/01/27/IOT-doorwoon-eye-controlled-door-for-a girl with-脑瘫女孩的门/](https://hackaday.com/2018/01/27/iot-doorman-eye-controlled-door-for-a-girl-with-cerebral-palsy/)

凯莉在她的轮椅上有一台眼控电脑，但是像她卧室的门这样简单的东西她仍然够不着…直到现在！[Bill Binko]最近拍摄了一个为患有脑瘫的小女孩制作的自动物联网开门器的演示。[Bill]是 ATMakers 的联合创始人，ATMakers 是一个让对辅助技术感兴趣的制造商与用户合作以提高生活质量的组织。

使用她的眼球追踪平板电脑(PRC 设备)，Kyleigh 有两个新图标，可以向一个网站发出相关呼叫，按下一个简单的命令来打开或关闭她的卧室门。附在门上的设备使用了阿达果 M0 WiFi 羽毛板、DC 步进电机和轮子、UBEC 降压转换器和电位计。

由于其他家庭成员也将打开和关闭门，所以有一个电位计可以测量门的位置，以便下次 Kyleigh 希望使用门时能够正确操作。该装置还保持了一个相当不显眼的轮廓，用于提供帮助——“大脑”被封装在门上的一个小盒子中，电机只在门的底座上稍微大一点。

[Bill]认为该项目存在一些问题，希望在执行打开/关闭流程之前缩短等待时间，并优化打开/关闭速度。你必须看看下面的视频，看看它真的真的有效。我们也很兴奋地看到 Kyleigh 使用她的目光控制与亚马逊回声交谈。[Bill]预见一个门控制的改进，把它和 Alexa 联系起来。而提高这个年轻女孩的生活质量又花了多少钱？$70.
