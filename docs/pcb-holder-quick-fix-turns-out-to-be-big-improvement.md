# PCB 支架快速修复是一大改进

> 原文：<https://hackaday.com/2018/07/01/pcb-holder-quick-fix-turns-out-to-be-big-improvement/>

当某件事情需要改进时，大多数黑客通常会做些小调整来解决问题，而不会改变事情的实际运作方式。其他黑客更深入一层，这就是[菲利克斯·鲁苏]用他的 [3D 打印磁性固定器](https://lowpowerlab.com/2018/06/19/3d-printed-magnetic-holders-for-pick-place/)所做的。最初设计是为了解决他的 LE40V 桌面取放机中 PCB 固定器的缺点，后来证明它们也适用于其他应用，并且很容易修改以使用任何大小的磁铁。

[Felix]抓放器上的 PCB 固定器的问题是，它们通过抓住边缘将电路板悬挂在半空中。电路板被牢牢固定，但面板化 PCB 设计上的高密度器件会导致取放头工作时悬挂电路板振动。当电路板被 v 形刻痕以便稍后容易地折断较小的电路板时，情况甚至更糟；由于压力，它们有时会沿着刻痕线断裂。

大多数人会通过在电路板下放置垫片来解决这个问题，但[Felix]决定更进一步，用一个简单的模型来改变安装系统。电路板现在放在一个金属平板上，他的磁性固定器制作简单，可以轻松固定任何尺寸的 PCB。作为一个额外的奖励，它证明了持有者也做了一个过得去的工作，将工作材料放在激光切割机的蜂窝桌上。下面嵌入了视频概述，设计文件可在 [Thingiverse](https://www.thingiverse.com/thing:2969116) 上获得。

把手的斜边和螺丝很好，但在 3D 打印机普及之前，我们简要地看到了另一种制作磁性支架的方法:放下磁铁，在一些烤箱固化的建模粘土上滴上一滴，根据需要成型，然后在烤箱中烘烤。Sugru(或者它自制的替代品， [Oogoo](https://hackaday.com/2010/10/11/oogoo-a-home-made-sugru-substitute/) )也可能会成功。
