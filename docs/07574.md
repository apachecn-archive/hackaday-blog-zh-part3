# 混合界面将触摸屏带入 Rigol 范围

> 原文：<https://hackaday.com/2017/11/03/hybrid-interface-brings-touchscreen-to-rigol-scope/>

随着智能手机和平板电脑的普及，触摸界面被认为是小液晶屏幕，我们可能都在某个屏幕上戳和捏，却发现它不受我们手势的影响。制造商已经注意到这一点，并开始为数字示波器等仪器添加触摸界面，但触摸界面往往是一种升级功能。但是多亏了[这个混合示波器触摸屏界面](https://github.com/mmbuw/touchscope)，即使是低端示波器也能参与进来。

只有[Matt Heinz]从这个构建中最容易破解的范围之一开始才有意义，这是他的硕士论文项目。使用 Android 平板电脑作为辅助界面，[Matt]能够远程控制示波器的大多数主要功能。收缩和扩展手势被解释为水平和垂直缩放，而拖动显示的波形会改变其位置并控制触发。虽然这不是一个真正的触摸屏范围，代码都是开源的，所以真正的售后 Rigol 触摸屏还会远吗？

Rigol hacks 在这里比比皆是——你可以用 Linux 与他们对话，用 T2 增加带宽，或者用 T4 了解他们的想法。
