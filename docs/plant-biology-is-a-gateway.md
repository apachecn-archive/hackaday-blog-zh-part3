# 植物生物学是一个入口

> 原文：<https://hackaday.com/2018/05/23/plant-biology-is-a-gateway/>

太多的大学生受到教师的帮助，他们认为自己太聪明了，不会被束缚在仅仅教低年级学生上。出于这个原因，[思想仓库]很重要，因为他对学习充满热情，并随时准备自己学习新的东西，并教任何想学习的人。当他发布一个关于[染色和观察植物样本](https://www.youtube.com/watch?v=JKz3t1I6MAk)的视频时，他避开了大学或高中实验室中常见的最大陷阱。他没有死记硬背，而是用有用的拍摄角度和特写镜头带我们走过这些步骤。他不是指着一个瓶子说，“蓝色的”，而是告诉我们里面是什么，为什么它是必不可少的。他没有准确地告诉我们需要看到什么才能获得及格分数，而是让我们思考我们可能会看到什么，并向我们展示让实验看起来令人兴奋的例子。视频也可以在休息下面看。

染色的过程可以在一本生物教科书中找到，有些人通过阅读学得最好，但我们没有读过一本手册，让一个简陋的实验室看起来像纳尼亚的衣柜，所以他得到了信任。诚然，你必须处理一个邪恶的锋利的剃刀，失败的几率从来不是零。事实上，他会告诉你，失败的机会无处不在。通往科学的道路不是刚铺好的，它需要铺路者。

如果生物实验室不在你的个人预算之内，一个[黑客空间可能有一个或者需要一个](http://hackaday.com/2013/08/15/your-hackerspace-needs-a-biology-lab-heres-the-inspiration/)。如果你想知道你以前在哪里听过[思想商场]的声音，那是因为他是像黑客一样对抗乳糖不耐症的[。](https://hackaday.com/2018/02/14/biohacking-lactose-intolerance/)
