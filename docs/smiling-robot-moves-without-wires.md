# 微笑机器人不用电线就能移动

> 原文：<https://hackaday.com/2018/05/22/smiling-robot-moves-without-wires/>

有什么比一个整天围着游戏围栏跑来跑去、面带微笑的小机器人更可爱的呢？2018 年 Hackaday 奖【bobricius】分享了他的[微型磁力机器人](https://hackaday.io/project/154496-2d-actuator-move-micro-robot-in-xy-2d-space)的 2D 致动器。这个名字并不可爱，但是它拥有一个不到十美元的材料清单，所以它应该非常适合教育用途，这也是它被创造出来的原因。

双层电路板隐藏了六个电极。三极垂直运行，三极水平运行。每个磁极都类似于步进电机中的一个绕组。当磁极打开时，磁梭移动到最近的活动磁极。当垂直绕组激活时，可以将梭子锁定到位。随着绕组按顺序激活，可以向左/向右和向前/向后移动。第二个视频完美地演示了这一点。

[bobricius]从一个更可怕的来源找到了[灵感，但希望我们知道这是他的创造，而不是专利侵权。我们不是律师。](https://hackaday.com/2014/04/23/micro-robots-are-scary-awesome/)
