# 艾可，见见麦考夫

> 原文：<https://hackaday.com/2015/09/23/echo-meet-mycroft/>

亚马逊 Echo 试图引入一个新的产品类别。一个听你的，遵从你意愿的盒子。有点像你家里的 Siri 或 Google Now。Kickstarter 的创建者[Joshua Montgomery]喜欢这个想法，但他想用树莓 Pi 和 Arduino 来做这件事。

Kickstarter(本月早些时候达到了融资目标)声称，该设备将使用自然语言访问媒体，控制物联网设备，并将对硬件和软件黑客开放。Kickstarter 页面上说，迈克罗夫特与 Lucid 和 Canonical(Ubuntu 背后的人)有合作关系。此外，他们还增加了拉伸目标，将计算机视觉和 Linux 桌面控制添加到 Mycroft 中。

不管有没有迈克罗夫特，人们都会一起破解这样的东西。如果你梦想能够用命令“计算机”来启动你的茶壶。茶。伯爵茶。热。”那么迈克罗夫特可能是一个很好的开始。我们也看到了[与 Roku](http://hackaday.com/2015/08/01/amazon-echo-orders-the-roku-about/) 甚至 [Nest](http://hackaday.com/2015/07/25/control-nest-devices-with-amazon-echo/) 的 Echo 集成。我们想象一个开放的平台会产生很多有趣的黑客。你可以在下面的视频中了解更多关于迈克罗夫特的计划。
