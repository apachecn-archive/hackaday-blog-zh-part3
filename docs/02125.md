# 分离襟翼式列车显示器使用穿孔卡片；以车站的独创性服务

> 原文：<https://hackaday.com/2016/04/20/split-flap-train-display-uses-punch-cards-serviced-with-station-ingenuity/>

由[Krzysztof Tyszecki]制作的一部简短但非常详细的纪录片探索了波兰łódź卡利斯卡火车站的分体式展示系统，以及工作人员为保持其运行并使用至今所做的努力。分体式显示器可能是旧技术，但许多仍在世界各地使用。但即使按照这些标准，你也不会想到在博物馆外能看到łódź·卡利斯卡的这个单元。一名员工表示:“我怀疑你在其他任何地方都找不到原作。”。它需要不断的维护以保持运转，在原有系统的限制下满足现代空间站不断变化的需求需要一些聪明才智。“总的来说，这种设备的故障率非常高，”他补充道。

这个系统在穿孔卡上运行。你再也买不到它们了，所以当地的印刷商生产它们——每次时间表有变化都需要几百张。打孔钳(也已经买不到了)太旧了，所以他们用当地锁匠定制的钳子来代替别针。读卡器的活动部件有开口销，需要每一两周更换一次——重复运动的压力会磨损它们。除了定期更换，别无他法。该组件需要定期清洁，因为灰尘会积聚在卡上并进入整个组件。清单还在继续…电视台也是。

没有现代意义上的计算——它是一个由人工操作员管理和更新的机电签名系统。它与基于纵横制交换机的电话交换机有更多的共同之处。穿孔卡只是一种快速、准确、重复地将显示器设置到已知状态的方法。

这部短片深入介绍了该系统每个部分的许多细节。卡片本身被详细描述( [1:07](https://youtu.be/FIHpVf_C4wU?t=67) )，操作员的程序也是如此( [2:27](https://youtu.be/FIHpVf_C4wU?t=147) )。我们甚至看到了后端控制器( [9:41](https://youtu.be/FIHpVf_C4wU?t=582) )，还看到了一个分开的襟翼模块，并用一台旧的测试仪进行了测试( [14:33](https://youtu.be/FIHpVf_C4wU?t=873) )，工作人员不确定它是否还能工作——但就像我们看到的其他东西一样，它当然能工作。

翻盖显示器是令人着迷的技术。我们甚至看到人们从零开始制作他们自己的翻盖显示器。
