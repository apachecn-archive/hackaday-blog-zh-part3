# 光荣的履带式“疯狂机甲”开始是纸板

> 原文:[https://hack aday . com/2018/03/11/glorious-body-of-tracked-mad-mech-started-as-cardboard/](https://hackaday.com/2018/03/11/glorious-body-of-tracked-mad-mech-started-as-cardboard/)

[迪克尔]总是喜欢履带式车辆。他[从*疯狂麦克斯:狂暴之路*中的‘和平缔造者’履带车获得灵感，将其复制为*疯狂机甲*T5。车辆是遥控的，坦克踏板部分来自 VEX 机器人坦克踏板套件。通过使用 Arduino 和 NRF24L01 模块的 DIY 无线控制器进行控制。车辆本身使用 Arduino UNO 和 L298N 电机驱动器。能量来自三个锂电池。](https://www.robotshop.com/letsmakerobots/mad-mech-mad-maxs-peacemaker-inspired-tracked-vehicle)

[![](../Images/9d1cba3032316b81f2a44b3e5e42edb2.png)](https://hackaday.com/wp-content/uploads/2018/03/mad-mech-open-featured.jpg) 真正的艺术作品是在身体里。[Dickel]使用了一个名为 [Pepakura](http://www.tamasoft.co.jp/pepakura-en/) 的 papercraft 工具(非免费软件，但[这个 Blender 插件](https://wiki.blender.org/index.php/Extensions:2.6/Py/Scripts/Import-Export/Paper_Model)是一种替代的免费方法)进行设计，用薄纸板制作机身。然后纸板的设计被修改成尽可能与*和平缔造者*的身体相匹配。它被涂上玻璃纤维以增强强度，然后用车身填充物和砂纸打磨以获得光滑的表面。经过一些细节和良好的油漆工作，这是准备滚动。

有很多巨大的努力投入到这个构建中，[Dickel]在他的项目页面和下面嵌入的视频中展示了他的工作和过程。第一段视频展示了完成的 *Mad 机甲*正在进行一些试驾。第二个是显示构建过程关键部分的剪辑。
