# PiCorder: Raspberry Pi 代表石刀和熊皮

> 原文：<https://hackaday.com/2017/08/18/picorder-raspberry-pi-stands-in-for-stone-knives-and-bearskins/>

在《星际迷航》的经典一集中，斯波克被困在 20 世纪 30 年代时，试图用他所谓的“石刀和熊皮”从三录仪中获取数据。实际上，他使用了真空管、几个大线圈和一架雅各布梯。可惜他们不在 2017 年。然后斯波克可以做得像[Directive0]和[用树莓派代替](https://github.com/directive0/picorder)。你可以在下面的视频中看到结果。

构建从钻石选择玩具模型三录机开始。Raspberry Pi、电池、TFT 屏幕和 Pi Sense 帽子构成了该产品的主体。

老实说，这是一个不是电子火箭科学(或者可能是曲速核心工程)的项目。它只是用 Python 代码将一些现成的模块串在一起。但玩具三录仪的改装才是真正的故事，视频展示了这一切是如何进行的细节。

我们已经看到了相当多的[下一代三录仪](https://hackaday.com/2014/06/11/thp-entry-the-improved-open-source-tricorder/)的复制品。我们也见过[声称是三录仪](https://hackaday.com/2017/04/28/im-a-tricorder-not-a-doctor-jim/)的设备，尽管它们看起来并不总是像我们见过的任何设备。我们生活在一个悬浮滑板不会悬浮，人工智能助手也没有那么聪明的世界。所以我想我们可以忽略这些三录仪没有一个能真正做到《星际迷航》中一半的功能。
