# 用一个破熨斗做一个蒸汽清洁器

> 原文：<https://hackaday.com/2018/03/20/make-a-steam-cleaner-from-a-broken-clothes-iron/>

老话说，当生活给你一个破铁，就做一个蒸汽清洁器。或者类似的东西。总之，[意外科学]的[Claudio]有一个破头的熨斗，他决定[把它改装成一个蒸汽吹风机，可以用来清洁 PCB、玻璃器皿、脱脂部件或清除沙发上的污渍](https://www.youtube.com/watch?v=5cvDjPULZWs)。

[Claudio]涵盖了从拆除碎铁到制作新笔尖的所有内容，避免了水滴凝结在他最先使用的黄铜笔尖上的问题。在打捞出头部控制蒸汽的开关后，他雕刻了一个浸泡过并涂有高温树脂的木柄。热端然后被重新安装，整个事情放在一起。

不过，这种构建应该小心谨慎:任何将高压蒸汽与电力混合的东西都有可能以令人不快的方式出错，所以在那里要小心。
