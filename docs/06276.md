# 用尺子做一个绘图仪

> 原文：<https://hackaday.com/2017/06/16/make-a-plotter-out-of-rulers/>

Instructables 用户[lingib]制作了一个聪明而[便宜的笔臂绘图仪](http://www.instructables.com/id/CNC-Dual-Arm-Plotter/)，它使用塑料尺子作为手臂。对于没有一堆机器人零件的人来说，这是一个鼓舞人心的景象，

电子设备很简单，一个 Arduino UNO 和一对简单的驱动器来控制连接到机器人轮子的 NEMA17 步进电机，机器人轮子是尺子的中枢。在手臂的末端，SG90 微型伺服系统按照命令升高和降低笔，用它的角将整个笔组件推离纸张——这是一个古老的 drawbot 问题的优雅解决方案。他甚至编写了一个定制的处理程序，允许他从桌面上控制绘图仪

[lingib]试验了不同种类的绘图机，包括鼓式绘图仪(休息后的视频)、V 型绘图仪以及滚动绘图机器人。

你会发现大量关于所有类型绘图机的黑客帖子，包括[老式绘图仪](http://hackaday.com/2017/05/11/vintage-plotter-handles-chaos-with-ease/)，为[制作电路板的绘图仪](http://hackaday.com/2016/08/31/the-evolution-of-a-diy-circuit-board-plotter/)，甚至还有一个用纸板做成的。
