# 复活——将二战无线电设备重新投入使用

> 原文:[https://hack aday . com/2016/11/09/复活-按压-WW2-无线电-设备-恢复服务/](https://hackaday.com/2016/11/09/resurrection-pressing-ww2-radio-equipment-back-into-service/)

第二次世界大战期间，大规模生产是生存的关键。如此多的东西被制造出来，以至于继续有大量新的未包装的东西留下来，成吨的二手设备以合理的价格出售。这种战争剩余物资的可用性在 20 世纪中期第一次为实验人员提供了获得高性能测试设备、无线电设备和高质量组件的途径。

即使在今天，这种古老的东西仍然激励着年轻一代，因为它的高制造质量、独特的机电方法和整体创新设计在 21 世纪仍然适用。在本帖中，我们将向你展示如何开始恢复二战无线电设备的爱好，并把它放回广播中。

## ARC-5，你的第一个接收器

许多业余无线电操作员在得到一个旧的 [ARC-5](https://en.wikipedia.org/wiki/AN/ARC-5) “命令集”接收器后，对这个爱好产生了兴趣。这些无线电通常用于编队内飞机之间的通信，因此得名“命令集”。这条线路中有许多不同的接收机和发射机，覆盖从 190 KHz 到 146 MHz 的频带，其中大多数能够调谐 AM、CW 和 SSB(通过调整 BFO)。

![ARC-5 command set receiver, this one covering 1.5-3.0 MHz.](../Images/bebcc9efd4e642d3dd6e5b1968ba29d3.png)

Example of an ARC-5 command set receiver, this one covering 1.5-3.0 MHz.

我最近在纽约乔治湖的一家军事古董店购买了上面显示的接收器。这台收音机很有趣有两个原因，它完全是原创的，从未为业余爱好者使用而修改过(多年来，业余无线电爱好者操作这些收音机来适应他们当时的需要)，它的制造日期是 1942 年 2 月。

我没有像修复古董收音机时通常需要的那样更换所有的旧电容器，而是测试了它们中的每一个，只更换了坏的。在我更换的那些罐子中，我重新塞住了原来的罐子，这样看起来就不会有任何改变。

我还修复了旧发电机，这是一种独特的电动发电机设备，在第二次世界大战期间用于从 28 VDC 飞机总线产生高板电压。这些收音机是一个很好的第一次二战收音机的复原，因为它们很简单，有大量的文件，你可以在任何一个集市上花不到 50 美元买到一个 ARC-5 接收器，或者根据我的经验，它们经常被送给任何真正感兴趣的人。
