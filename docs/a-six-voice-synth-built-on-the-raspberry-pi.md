# 一个基于树莓派的六声部合成器

> 原文：<https://hackaday.com/2017/02/07/a-six-voice-synth-built-on-the-raspberry-pi/>

在过去的几十年里，音频合成器越来越不是真正的硬件，而是越来越多的软件仿真。现在我们有了功能强大的微型计算机，它只能降低功耗，那么显而易见的结论是什么？[围绕树莓派](http://r-mono-lab-en.tumblr.com/products/S3-6R)打造的六声部复音合成器。

这款名副其实的“S -6R”合成器是一款六声部相位调制合成器，可以输出非常高分辨率(24 位和 96 kHz)的音频。这是 R-MONO 实验室的产品，他们在过去展示过有趣的音乐设备，如基于录音机的管风琴。这个构建有点复杂，提供了一些令人惊叹的声音，这些声音都是在 Raspberry Pi 3 上生成的。

虽然谈论振荡器和滤波器很棒，但这里真正有趣的是键盘本身。S -6R 使用的是罗兰 K-25m T1，这是一种小型 MIDI 键盘，旨在作为罗兰最近重新发布的经典 [Jupiter 和 Juno synts](https://www.roland.com/ca/promos/roland_boutique/)的一种“坞站”。构建一个 MIDI 键盘无论如何都不容易，使用这个小键盘 dock 是一种将 MIDI 音符传输到任何项目的廉价方式，没有太多麻烦。
