# 使用这款物联网淋浴控制器，以 Arduino 方式开始您的一天

> 原文：<https://hackaday.com/2016/03/04/start-your-day-the-arduino-way-with-this-iot-shower-controller/>

物联网供应商不再满足于增加走廊恒温器的价值或让你的冰箱变得比它应该的更智能，他们正在进入家庭的内殿，即最神圣的地方——浴室。当然，你可以花一大笔钱买一个电子控制阀门，把你的淋浴变成一个远程控制的水疗中心，与云分享你的洗澡习惯，但如果你的预算比较适中，并且有黑客精神，你可能想看看这个具有物联网功能的 DIY 自动淋浴阀。

当我们最后一次遇到[TVMiller]时，他正在[用绝地思维技巧打开大门](http://hackaday.com/2016/02/14/open-sesame-from-a-galaxy-far-far-away/)，在此之前他正在[将一个浮动的高尔夫果岭缩小到可管理的大小](http://hackaday.com/2015/12/28/mobile-mini-green-recreates-coeur-dalenes-floating-14th-hole/)。这种黑客工作起来很累，虽然一个干净的黑客是一个快乐的黑客，但所有这些讨厌的阀门扭曲和温度篡改是如此令人讨厌。因此，利用从废物流中获得的一些零件，如丙烯酸树脂盒、一些回收的伺服系统、冰棒棒和大量填缝剂，[TVMiller]为他现有的淋浴阀拼凑了一个功能丰富的控制器。Arduino MKR1000 读取水温并控制伺服系统，使他能够从手机上开始淋浴。使用 ThingSpeak 将时间和温度数据发送到云端。休息之后，你可以在 NSFW 的视频中看到整个过程。

诚然，这是一个非常简单的设置，但它属于通用黑客免责声明“这只是一个原型。”尽管如此，我们喜欢改造标准淋浴阀的想法，冰棒棒平行四边形增加杠杆是一个巧妙的技巧。我们将拭目以待接下来会发生什么。
