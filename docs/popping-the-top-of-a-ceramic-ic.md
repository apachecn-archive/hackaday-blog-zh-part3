# 打开陶瓷集成电路的顶部

> 原文：<https://hackaday.com/2017/04/07/popping-the-top-of-a-ceramic-ic/>

如果你曾经想打开一个 IC 看看里面有什么，你有几个选择。带有金属盖子的陶瓷包装将屈服于业余爱好刀。那很简单。普通的环氧树脂封装更硬，通常需要混合使用机械研磨和酸(例如发烟硝酸)。[Robert Baruch]想要打开一个全陶瓷包装,所以他使用了 MAP 气体喷枪的“冷却器”部分。如果你喜欢看东西在明火中变热，你可能会喜欢下面的视频。

剧透提示:[罗伯特]艰难地发现去掉热部件不是一个好主意。此外，如果你想做的不仅仅是检查芯片，我们也不确定加热能做什么。在工艺过程中测量芯片上的一个结，看看有多少热量实际传递到器件，这将是一件有趣的事情。

这个过程真的很快:只有 20 秒左右。我们想知道更大的部分是否需要更长的时间。然而，与化学方法相比，这看起来非常快速和简单，只要你不介意高温。

如果你有开始打开部件的冲动，并想实际探测芯片的表面，不要忘记几乎整个芯片上都有一层薄薄的玻璃。这层钝化层相对较厚，通常只在焊盘周围有切口。去除这层需要氢氟酸(讨厌的东西)。你可以通过在焊盘边缘上下聚焦显微镜来判断你什么时候得到的。当你找不到钝化的边缘时，你就完了。
