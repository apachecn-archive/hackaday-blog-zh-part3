# 改良的 Uke 用螺线管保持节拍

> 原文：<https://hackaday.com/2017/12/02/modified-uke-keeps-the-beat-with-a-solenoid/>

一个经典的单人乐队通常有一两个弦乐器，一个免提支架上的口琴，以及某种打击乐器，通常是一个像背包一样佩戴的低音鼓，也许在膝盖之间还有一些钹。音乐家也可以用他们的弹奏手敲击弦乐器的音板，这是古典和弗拉门戈吉他手能够以惊人的范围完成的事情。

音乐家通常必须手动操作每件乐器。谈到打击乐，[吉姆德]有另一个想法:通过用螺线管敲打共鸣板来保持节拍。他构建了一个简单的 Arduino 驱动 MOSFET 电路，将可变 BPM 的敲击声传递给 ukulele 的音板。10kΩ的锅控制米和拍频，声音由桥上的麦克风拾取。到目前为止，它做了 3/4 和 4/4 次，但是[JimRD]让代码可以自由扩展。有人让它做 5/4，因为我们喜欢听[吉姆德]演奏“[拿五个](https://www.youtube.com/watch?v=o2In5a9LDNg)”。

他没有对他的好 uke 这样做，请注意——这是一个他不介意钻孔和胶合的旧搅拌器。我们起初有点怀疑，但共振使螺线管芯的机电敲击变得更加悦耳。那，而且[吉姆德]有一些相当好的排骨。穿过休息时间去听一听。

有一把便宜的尤克里里琴却不知道怎么弹？如果你让火焰从主轴箱中喷出，那也没什么大不了的。没有 ukes？[只需打印一张](https://hackaday.com/2013/12/11/the-3d-printed-ukulele/)。
