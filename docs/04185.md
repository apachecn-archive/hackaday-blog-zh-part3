# commodore Home——您 1983 年的智能家居

> 原文：<https://hackaday.com/2016/11/20/commodore-home-your-smart-home-for-1983/>

物联网是一种可怕的时间浪费，尽管没有人知道它到底是什么。什么能让它变得更好？当然是经典的准将装备。现在，您可以使用 Commodore 64 和 Commodore Home(来自[retro.moe]的最新智能家居框架)来运行您的智能家居。

Commodore Home 具有您期望的标准智能家居功能。家庭照明解决方案是一个点阵式打印机，几个齿轮，和绑在灯开关上的绳子。启动打印机，灯就会打开和关闭。太棒了。菊花链打印机可以控制多个灯开关。

安全在智能家居中很重要，虽然入侵警报功能不全，但未来版本的 Commodore Home 将会拨打调制解调器，登录 BBS，并在授权人员进入您家时留言。

所有这一切都要归功于同样来自[retro.moe]的 UniJoystiCle 技术的进步。该设备采用标准的 ESP8266 WiFi 模块，并将其转变为 Commodore 64 的智能手机到操纵杆的端口桥。

与今天销售的其他物联网硬件不同，Commodore Home 不会在背后的公司破产时停止工作；准将已经死了二十年了。你可以在 Githubs 上下载 Commodore Home [的所有软件，也可以看看下面的视频。](https://github.com/ricardoquesada/c64-home)
