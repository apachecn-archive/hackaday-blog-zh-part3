# 水下机器人使用体长鳍

> 原文：<https://hackaday.com/2018/06/21/festos-underwater-robot-uses-body-length-fins/>

[Festo]已经发明了另一个神奇的机器人，一个会游泳的机器人，这次有一个优雅的推进装置。他们称之为生物电波。两边几乎和身体一样长的两个鳍产生了向后推水的波浪，使机器人向前移动。它是模仿乌贼和尼罗河鲈鱼的。

他们用什么优雅的方法让鱼鳍呈波浪形？九个杠杆臂连接到每个鳍上。这些杠杆臂由两根从车身前部延伸到后部的曲轴控制，每侧一根。然后，伺服电机转动每个曲轴。由于曲轴是独立的，这意味着每个鳍独立运作。这允许通过让一个鳍比另一个鳍移动得更快来转弯。头部的第三个马达使身体弯曲，使机器人上下游动。

头部还有一个压力传感器和一个超声波传感器，用于深度控制和躲避物体和墙壁。虽然这些允许它在丙烯酸管状轨道中自主游泳，但有无线通信记录传感器数据。在下面的视频中，看它毫不费力地绕着跑道游来游去。

这些年来，[Festo]创造了许多这样的奇迹。我们之前报道过他们的[仿生跳跃袋鼠(我们不骗你)](https://hackaday.com/2014/04/08/festo-creates-bionic-kangaroo-steve-austin-unimpressed/)，他们的[外骨骼上印有电路的机器蚂蚁](https://hackaday.com/2017/03/16/robot-ants-wear-circuitry-as-exoskeleton/)，也许还有[有史以来最逼真的扑翼机器鸟](https://hackaday.com/2011/03/30/robotic-bird-flaps-away-last-bits-of-privacy/)。
