# 冷却液软管被要求伸出援助之手

> 原文：<https://hackaday.com/2018/03/04/coolant-hoses-retasked-to-lend-a-helping-hand/>

每个人在商店里都偶尔需要帮手，我们大多数人都走传统路线，买了一个铸铁底座和可调节臂端接鳄鱼夹的小玩意儿。它们很便宜，很容易买到，而且充其量只是“咩”的一声。

在寻求更好的手牌的过程中，[Jeremy S. Cook]想出了这个桌面辅助工具的定制设计，我们对此印象深刻。有商业设计使用相同的柔性冷却剂软管，称为 Loc-Line，经常看到它在金属加工机器上喷射冷却剂，如轧机和车床。但是这些东西很便宜，只要做一点点工作，你就能造出符合你需求的东西，而不是绕过商业设计。[Jeremy]用 CNC 路由器从标准尺寸的木材中切割出底座，但是用简单的手工工具也可以做同样的事情。3D 打印的底座也很容易，尽管它可能需要一些压舱物来防止它在工作台上晃动。Loc-Line 软管很容易改装成鳄鱼夹，我们也可以想象其他配件，如灯和放大镜，甚至是 3D 打印的勺子，通过软管吸入焊接烟雾。

毫无疑问，这是一个简单的项目，但很有用，我们喜欢这个设计。但是不要认为[杰里米]没有宏大的计划——还记得他华丽的聚碳酸酯灯吗？
