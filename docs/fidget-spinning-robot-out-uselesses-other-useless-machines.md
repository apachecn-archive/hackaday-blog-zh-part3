# 坐立不安的旋转机器人胜过其他无用的机器

> 原文：<https://hackaday.com/2017/06/26/fidget-spinning-robot-out-uselesses-other-useless-machines/>

当路易斯维尔 LVL1 黑客空间的[法律邪恶主任]决定通过打印一个坐立不安的旋转器来证明 3D 打印机的无用性时，该空间周二会议的另一名成员接受了挑战，建造了一台机器，其唯一目的是*旋转*坐立不安的旋转器。

[Gary Flispart]使用了一个 Arduino 克隆体，以及一个看起来像是电机驱动器的东西与一个步进电机结合在一起。马达带动一条皮带转动一系列金属废料，作为一个四连杆机构。连杆移动旋转器的销钉，然后让开，这样就不会妨碍玩具的旋转。[数字坐立不安的数字](https://www.youtube.com/watch?v=6niSMCtohIg)，正如[Gary]所称，看起来像是在黑客空间项目的光荣传统中用废金属和随机的木头块建造的。

我们 Hackaday 喜欢来自黑客空间的疯狂项目，比如 i3Detroit 的 [iris 舷窗](http://hackaday.com/2016/07/09/door-iris-porthole-is-the-perfect-fix-for-detroit-hackerspace/)，另一个空间的[终极油烟机](http://hackaday.com/2016/10/08/hackerspace-takes-fume-extraction-seriously/)，以及 LVL1 的 [doomcano](http://hackaday.com/2016/09/18/lvl1-hackerspace-builds-a-doomcano/) 。
