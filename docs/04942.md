# 模块化便携式传送带

> 原文:[https://hack aday . com/2017/02/09/modular-portable-conveyor-belt/](https://hackaday.com/2017/02/09/modular-portable-conveyor-belt/)

在向学生教授工业自动化时，你需要让他们接触到他们在工业中会遇到的东西。大多数科目可以使用计算机程序或模拟器进行教学，例如涵盖 PLC、DCS、SCADA 或 HMI 的主题。但是要教授许多其他概念，您需要手头有实际的硬件，以便能够理解基础知识。例如，机器视觉、传送带、电机速度控制、安全和联锁系统、传感器和外围设备都与上述控制系统接口，通过使用硬件可以更好地理解。[绝对自动化]的团队已经发布了几个旨在帮助解决这个问题的项目。其中之一是带有电机速度控制和显示的 [DIY 传送带](http://www.absolutelyautomation.com/articles/2017/01/30/modular-portable-conveyor-speed-controlled-arduino)。

这更像是一个初始的概念验证项目，有很大的改进空间。构建本身很简单。所有的零件都是标准的，现成的商品——你可以在任何出售 3D 打印机零件的商店里找到这些东西。只需要几个简单的工具就可以把它组装起来。唯一棘手的部分可能是传送带本身。[Absolutelyautomation]提供了一些建议，提到了旧汽车或卡车轮胎和用于治疗/锻炼的弹性阻力带作为选项。

如果您计划复制它，建议做一些更改。8 毫米的滚筒可以覆盖更大的“鼓”——直径大约一两英寸。这有助于防止皮带打滑并改善张力调节。3D 打印附加鼓应该很容易。皮带可能还需要滚轮之间的支撑板来防止下垂。速度显示需要用线性单位——英尺每分钟或米每分钟，而不是电机转速。虽然电子设备包括一个 RS-485 接口，但在组合中添加 RS-232、RS-422 和以太网将有所帮助。

虽然这是一个简单的构建，但它可以形成一系列附加组件和扩展的基础，以帮助学生学习更多关于自动化和控制系统的知识。或者出于某种原因，你想在你的地下室装一个传送带。
