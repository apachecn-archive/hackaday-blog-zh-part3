# 谁说热感相机大众接触不到？

> 原文：<https://hackaday.com/2018/06/08/who-said-thermal-cameras-werent-accessible-to-the-masses/>

热感相机拥有持久的魅力，也是工程师的有用工具。毕竟，谁不想指着长椅周围随机的东西，高兴地笑着发现比预期更暖和或更冷的东西呢？但是它们总是如此昂贵，而且许多试图以低廉的成本提供这种设备的努力都令人失望。

尽管如此，这并没有阻止[Offer],他已经使用基于 M5Stack ESP32 的计算机模块和 AMG8833 热传感器阵列模块在 3D 打印的外壳中制作了一款看起来非常专业的热感相机,该外壳复制了商业设备上的热感相机。模块化的方法使它成为建造者的简单前景，[软件可以在 GitHub](https://github.com/hkoffer/M5Stack-Thermal-Camera-) 上找到，[案例文件存放在 Thingiverse](https://www.thingiverse.com/thing:2907015) 上。正如下面的视频所示，你很快就能在长凳上找到温暖和寒冷的东西。

我们见过的大多数热感相机都以 FLIR 轻子模块为中心，但这是一个仍然昂贵的组件。这个项目向我们展示了热感相机是一种慢慢变得负担得起的技术，并且更好的事情正在到来。
