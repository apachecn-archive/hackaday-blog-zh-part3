# 微小的弹球模拟器非常令人印象深刻

> 原文:[https://hack aday . com/2018/05/14/tiny-pinball-emulator-is-very-impressive/](https://hackaday.com/2018/05/14/tiny-pinball-emulator-is-hugely-impressive/)

我们想知道 Circuitbeard 最近在忙什么。原来[他一直在建造一个迷你弹球柜](https://circuitbeard.co.uk/2018/05/12/rombus-pinball-a-lattepanda-mini-pinball-machine/)来增加他自建游戏的街机。

[Circuitbeard]这次被迫打破他的 Raspi 舒适区。我们很高兴他这样做了，因为这是一个令人印象深刻的建设。发现 Linux 缺乏弹球仿真社区，他转向 LattePanda，一个内置 Arduino Leonardo 的小型 Windows 10 SBC。这真的是完美的主板，因为他需要以最小的麻烦支持多个显示器。Leonardo 可以在可视弹球模拟器中将按钮转换为按键。

3 毫米激光切割胶合板柜完全在 Inkscape 中设计，大小围绕两个屏幕:一个真正的 7 英寸 LattePanda 显示屏用于操场，一个 5 英寸 HDMI 用于后玻璃。主盒装有 Lattepanda、两个 Pimoroni 迷你扬声器和一个风扇，以保持电路板凉爽。

由于[Circuitbeard]对细节的非凡关注，这个小橱柜有很多令人喜欢的地方，你可以在大量图片中亲自看到。仔细观察投币口——它实际上是一个带有自定义图形的发光按钮。如果你想尝试模拟这个模拟器，[所有的代码都在 GitHub](https://github.com/circuitbeard/rombus-pinball) 上。休息时间过后，观看一些现代弹球魔术表演，然后[看看他的迷你跑垒机](https://hackaday.com/2017/08/30/toy-dash-turned-gaming-interface/)。

如果弹球模拟器不能让你得分，[这里有一个全是木头和橡皮筋的](https://hackaday.com/2017/02/12/wood-and-rubber-band-pinball/)。
