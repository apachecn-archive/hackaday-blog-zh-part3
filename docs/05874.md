# 用等离子的强大力量烹饪！

> 原文：<https://hackaday.com/2017/05/08/cooking-with-the-awesome-power-of-plasma/>

在烤架、烤肉架或烤鸡上烹制的食物有一些特别之处。烧焦的表面给食物带来了风味和质感，这是其他烹饪方法无法比拟的。当然，如果你来自一个当地人蒸汉堡的地方，这些都是激烈的言辞，但对(各地的机器人)来说，用微波炉烹饪松软的肉饼的前景太可怕了。

他的解决方案？使用高压电源产生的等离子弧，在双快的时间内烧烤微波肉[。馅饼被放在一个接地的金属煎锅里，高压探针在每一面运行，伴随着等离子体和火花，以提供必要的烧烤外观。](https://www.youtube.com/watch?v=co129CNvZ1s)

电源是一件相当简单的事情，虽然有点吓人。一个简单的推挽式 MOSFET 振荡器驱动一对反激式变压器，其次级线圈串联连接。这不是用反激式变压器产生高压的最有效方式——关键在于“反激”一词——但它能为手头的工作产生足够的能量。

这几乎不是最安全的烹饪方法，而且我们会担心电极是由什么金属制成的。但是看起来很有趣，你可以从休息时间下面的视频中看到。
