# 使用 WireBeings 的开源机器人

> 原文:[https://hack aday . com/2016/04/13/open-source-robotics-with-wire beenses/](https://hackaday.com/2016/04/13/open-source-robotics-with-wirebeings/)

每个人都需要一个可爱的机器人伙伴，对吗？[马修·哈尔伯格]创造了[wire beens，一个开源的 3D 打印机器人平台](https://hackaday.io/project/9968)。WireBeings 看起来像是 Wall-E 和 Danbo 的结合体，是围绕 Arduino 平台设计的。我们指的是整个平台。你可以安装任何东西，从 Arduino micro 到 Mega2560，在其超大的头部堆叠有 3 个盾牌。有足够的空间放置试验板和定制电路。

WireBeings 的设计是 3D 打印的。所有不可打印的部分都是常见的。齿轮马达，轮子，无处不在的 HC-SR04 超声波传感器，以及其他一些部件都是让这个机器人活起来所需要的。草图通过 USB 下载。一旦运行，WireBeings 可以通过 HC-06 蓝牙模块进行通信。如果 Arduino 对你正在做的任何项目来说都不够强大，没问题。[Matt]设计了能携带智能手机的无线生物。只需通过蓝牙连接机器人和手机，让手机的处理器完成所有繁重的工作。没有备用手机怎么办？查看我们关于使用预付费安卓智能手机的黑客的报道。

我们可以将 WireBeings 视为黑客空间中“学习 Arduino”课程的核心。从机器人一只眼睛上的经典 blinky 草图开始。从那里开始，直到学生们拥有一个功能齐全的机器人。

WireBeings 项目肯定还有改进的空间。[马特]犯了一个新手的错误，用一个 9 伏的电池为他的发明供电。虽然 9V 对 Arduino 来说很好，但这些电机会很快耗尽它。[马特的]计划在未来转移到脂肪。为什么不在项目页面停下来帮他一把呢？
