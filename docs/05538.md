# 用你的自拍恢复你坏掉的 SD 卡自拍

> 原文：<https://hackaday.com/2017/04/03/recover-your-selfies-by-your-selfie/>

你可能仍然有机会从你的 SD 卡中取出这些自拍照，即使它不再安装在你的电脑上。[硬盘恢复服务]向我们展示了一个过程，即[直接访问有故障的 micro SD 卡](https://www.youtube.com/watch?v=jjB6wliyE_Y)的 NAND 内存，以恢复那些您想过备份但从未考虑过的珍贵文件。

在一个微型 SD 卡上，你可能已经注意到有两个比其他的稍长的针。这些是 VSS 和 VCC 引脚。只要它们不是两者之间的绝对短路，SD 卡控制器就不会完全报废，我们可以继续前进，进入那个小吸盘。只要有一点专业知识，加上砂纸、漆包线和 NAND 阅读器，再加上一点耐心和一些良好的焊接技巧，就可以恢复丢失数据的图像。

从粒度相对较高的砂纸开始，慢慢打磨掉 SD 卡底部的塑料，直到可以清楚地看到隐藏在内部的铜痕迹。然后将漆包线焊接到小焊盘上，将其连接到 NAND 读卡器，您应该能够读取以前通过传统方法无法读取的数据。当然，您仍然需要了解 NAND 转储。这是另一篇文章的主题。

如果你发现自己需要一个 SD 卡恢复工具，你可以随时推出自己的 DIY NAND 读卡器。我们可能会尝试这个过程，只是为了玩玩这个概念。希望我们永远不需要做 SD 卡恢复！
