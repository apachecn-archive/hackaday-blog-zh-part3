# 黑客日的参赛作品:[纳尔达斯]发射火球

> 原文：<https://hackaday.com/2017/03/31/nardax-shoots-fireballs/>

如果你正在寻找每字节代码的高娱乐价值，[Nardax]已经用他的[可穿戴施法控制器](https://hackaday.io/project/20648-wearable-spellcasting-controller)覆盖了你。不费吹灰之力，他就造出了一个看起来非常有趣的设备，证明了我们一直以来都知道的一点:一点点互动就能大有作为。

[Nardax]原本打算把他那被美化了的肘装电位计作为一个烟花控制器。讽刺的是，他现在用它来投掷虚拟火球。根据他松开手肘前握住手肘的角度，他可以在游戏《魔兽世界》中施放不同的法术。我们一点也不确定这是否有助于他的游戏性，但我们绝对肯定简单地混合不同的键会更有趣。

这里有很大的扩张空间，但问题是你能把它推进多远。有时候最简单的想法是最好的。不过，看起来[Nardax]很享受他的产品测试研究，所以我们会密切关注这个项目的下一次迭代。

我们已经看到了一些高科技竞争对手的[良好的旧动力手套](https://hackaday.com/2017/01/28/the-power-glove-ultra-is-the-power-glove-we-finally-deserve/)，虽然有些是[比绑在肘部的电位计更复杂的](https://hackaday.com/2017/01/31/your-arm-is-the-ideal-controller/)，这个项目让我们微笑。有时候，不仅仅是你有多少技术，而是你如何使用它。毕竟，[DDS pad 只是地毯下的开关集合](http://hackaday.com/2016/11/29/game-controller-cuts-the-rug/)。
