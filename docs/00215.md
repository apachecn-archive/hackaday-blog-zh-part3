# 把任何东西侵入手机

> 原文:[https://hackaday.com/2015/09/29/hack-anything-into-a-phone/](https://hackaday.com/2015/09/29/hack-anything-into-a-phone/)

如果你花了很多时间摆弄电子产品，你可能会从他们的开发板、工具和 PCB 制造服务中听说过[seed studio]。他们最新的 Kickstarter 项目是 RePhone，这是一款开源的模块化手机,允许黑客通过混合 GSM 模块、电池、屏幕和其他库存单元来组装手机，包括基于 Arduino 的处理核心、GPS、NFC 和其他构建模块。

据[Seeed]称，该资助活动已经超过了其目标，预计将于明年交付，基本套件预计为 59 美元。据推测，核心手机模块将获得监管机构的认可，但其他辅助模块不需要严格的测试和认证。

你会用一部便宜的嵌入式手机做什么？模块很小，所以你可以把它们植入很多地方。Seeed 的一些更有趣的想法包括将手机做成手杖、狗项圈或风筝(尽管我们在考虑四轴飞行器)。

当然，我们以前见过 Arduino 的 [GSM 和手机屏蔽。很难想象把它们放在狗项圈里，除非你有一只相当大的狗。如果你是 1960 年代电视的粉丝，很容易想象](http://hackaday.com/2012/09/03/make-cell-phone-calls-with-your-arduino/)[是一个更好的鞋子电话](http://hackaday.com/2009/02/11/cell-phone-shoe/)或者是一个有效的《星际迷航》通讯器。
