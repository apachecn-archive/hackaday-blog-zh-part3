# Raspberry Pi + Wolfram 数据删除

> 原文：<https://hackaday.com/2016/01/04/raspberry-pi-wolfram-data-drop/>

当你想到 Mathematica 和 Wolfram 时，你可能会想到高性能的数字运算。你可能不会想到嵌入式系统。然而，Wolfram 运行在 Raspberry Pi 上，最近有一个视频(如下)显示了[一个 Raspberry Pi，控制 I/O 设备，并使用 Wolfram data drop](https://youtu.be/8YtnOMbYv5M) 与 Web 交互。

下面的第二个视频展示了一些老的例子项目，包括[一个简单的带有 PIR 传感器的家庭报警器](http://community.wolfram.com/groups/-/m/t/226163)。这不是 Wolfram 所熟知的那种东西，但作为一个“hello world”项目来说还是不错的。甚至有一个项目使用 Arduino 进行更多的 I/O。在这两个视频中，您可以很好地了解使用 Pi 和该语言可以完成的事情。

Wolfram 的真正优势在于它在声音生成和求解微分方程等方面的专业特性，这两项对于一个简单的示例都不起作用。数据拖放示例展示了这些功能，例如，可视化 PIR 数据。此外，这种语言有很好的文档。

几年前我们在圆周率上看过 Wolfram。不过，这些例子在显示与 Pi 的摄像头和 Arduino 接口或以有意义的方式可视化传感器数据所获得的价值方面有很大的帮助。甚至还有 [IFTTT](https://ifttt.com/) 集成。
