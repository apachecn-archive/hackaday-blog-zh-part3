# 3D 打印书桌利用 Fusion 360 和 McMaster-Carr 的力量

> 原文：<https://hackaday.com/2018/01/23/3d-printed-desk-harnesses-the-power-of-fusion-360-and-mcmaster-carr/>

黑色管道家具现在很流行，这是有原因的——它有很好的工业美感，很坚固，螺纹配件使它很容易组装起来。但是，如果你最近对配件进行了定价，你就会知道它远不便宜，所以能够 3D 打印你自己的黑色管道配件可以让书桌和桌子更加实惠。

当然，便宜是有代价的，弗拉基米尔·马里亚诺(Vladimir Mariano)不厌其烦地指出，他的桌子是轻型的，可能经不起大量使用。但是由于用于将胶合板顶部连接到腿和脚的法兰配件在当地的家居中心会花费大约 64 美元，所以打印它们是有意义的。再加上在两腿之间安装担架的定制零件，3D 打印的零件构成了一个相当坚固的底座。

但是最终产品并不是下面视频的重点。由于能够在 Fusion 360 中浏览 McMaster-Carr 目录，[Mariano]能够无缝导入合适的铁法兰的 CAD 模型，并根据需要快速修改。这个特性的力量是很难夸大的；您可以浏览工程零件目录，并立即打印出可用的复制品。当然，它不是由金属制成的，但对于设计师来说，能够看到最终产品的外观是一个巨大的福音，尤其是在项目的原型阶段。

不熟悉麦克马斯特-卡尔？这是一个工程师的网上游乐场，不久前我们讨论了与麦克马斯特做生意的来龙去脉。
