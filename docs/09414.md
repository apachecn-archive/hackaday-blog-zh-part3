# 内置唱针的时尚名片

> 原文:[https://hackaday . com/2018/05/11/stylisted-business-card-a-stylophone-built/](https://hackaday.com/2018/05/11/stylish-business-card-with-a-stylophone-built-in/)

如果你从事电子行业，PCB 名片似乎是一个天然的选择。它们可能不实用且昂贵，但它们确实能让你与 Vistaprint 人群中那些无聊的纸质卡片区分开来。但是它们需要对你所做的事情有意义，所以对于一个音乐家和 MIDI 专业人士来说，[这张 MIDI 控制器 stylophone 名片](https://mitxela.com/projects/stylocard)是一个真正的眼睛和耳朵捕捉器。

这张名片是[Mitxela]已经考虑了一段时间的想法，几年前他甚至制作了一个原型。自制卡使用喷漆、激光蚀刻和氯化铁方法制作，作为概念证明效果很好，但边缘有点粗糙，需要 PCB 制造商的专业触摸。我们不得不说，成品卡非常性感，黑色的保护层与浸金垫形成了鲜明的对比。他选了一块 1mm 厚的板，把 USB 连接器做成了单独的小板子；从主板上取下并回流，它将边缘连接器构建到适当的厚度。零件数量很少——只有一个 ATtiny85 和一个电阻梯来对每个键进行编码，一个简单的跳线用作触针。该设备本身只是一个 MIDI 控制器，本身不会产生音乐，但我们仍然认为这是一种非常有创意的挂招牌的方式。

[Mitxela]有很多有趣的构建，对我们的页面来说并不陌生。看看他最近的[伺服弹拨 MIDI 音乐盒](https://hackaday.com/2018/03/23/servos-do-the-plucking-in-this-midi-music-box/)，或者[这些令人惊叹的微型 LED 耳环](https://hackaday.com/2017/02/12/tiny-led-earrings-are-a-miniaturization-tour-de-force/)。
