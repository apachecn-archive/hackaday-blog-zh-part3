# 相机滑块有助于使用电子垃圾控制器拍摄照片

> 原文:[https://hack aday . com/2016/04/23/camera-slider-helps-get-shots-with-e-waste-controller/](https://hackaday.com/2016/04/23/camera-slider-helps-get-the-shots-with-e-waste-controller/)

相机滑盖是真正能拍出镜头的配件。但是，当你的业务是摄影而不是制造相机配件时，快速和肮脏的解决方案往往就足够了。于是就有了[这个相机滑块控制器](http://paulorenato.com/index.php/electronics-diy/192-slider-controller-with-auto-reverse-and-variable-speed-control)的由来。

[Paulo Renato]中提到的摄影师，虽然他的热情可能是摄影，但他似乎对电动推车和滑块很有天赋。该控制器是一种变速、可逆、基于 PIC 的设计，驱动易贝齿轮电机。该电路依靠一块废弃的 perfboard，它与电池和一个降压转换器一起被塞进一个旧 KVM 开关的外壳中。从另一个电子垃圾中回收的按钮充当限位开关，一点点代码就提供了魔力。我们喜欢控制器的黑客性质，但我们想知道使用以前的 KVM 的 USB 端口来连接控制器和传动系统是否明智；在你把一个真正的 USB 设备插入之前，这都是有趣的游戏。总之，虽然，一个好的建筑有好的结果。查看他的其他视频，了解更多关于机械的信息。

相机滑动装置已经为我们的页面增色不少，包括[一个主要由木头制成的](http://hackaday.com/2015/12/26/mostly-wood-motorized-camera-slider/)和[一个由花哨的 iPad 应用程序控制的](http://hackaday.com/2015/09/08/the-ipad-controlled-camera-slider/)。
