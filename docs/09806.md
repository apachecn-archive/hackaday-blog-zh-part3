# 线性轨道使等离子切割整洁简单

> 原文：<https://hackaday.com/2018/06/22/linear-track-makes-plasma-cuts-neat-and-simple/>

没有微控制器，没有显示器，甚至看不到 LED。这就是为什么【波兰制造】决定推出[这种电动线性等离子切割机](https://www.youtube.com/watch?v=V0ahzDSYDxo)，尽管简单，但当有金属需要切割时，它真的能完成工作。

等离子切割使切片和切割重原料成为一项快速的工作，但很容易偏离火炬的路线或改变速度，最终导致边缘不佳。这个工具用方管制作的完全自制的线性滑轨消除了震动。一个装有等离子切割炬的托架在一根螺纹杆上移动，这根螺纹杆卡在一个旧的无绳电钻的残骸上。钻机的原始离合器消除了当滑架撞击滑道任一端时限位开关的需要，我们认为这是一个很好的接触。提供了简单的速度和方向控制，以及与火炬触发器的并联连接。该车的一个很好的特点是能够将焊枪旋转一个角度，使厚钢板上的 V 型坡口焊接变得轻而易举。也不需要一张复杂的床，上面有祭祀用的支撑物和一个水盆——只需要把木头挂在桌子边上，让火花落在它们可能落的地方。

简单有时更好，但数控等离子台可能仍然是你心中的愿望。[我们理解](https://hackaday.com/2014/06/22/cnc-plasma-cutter-build-presented-in-excruciating-detail/)。
