# 构建坚固的远程控制邮箱

> 原文:[https://hack aday . com/2016/05/14/building-a-sturdy-remote-control-mailbox/](https://hackaday.com/2016/05/14/building-a-sturdy-remote-control-mailbox/)

这个 DIY 邮箱几乎什么都有。它不仅防水安全，而且建造精美，不太可能引起邮递员的怀疑或恐惧。

它由 2 毫米厚的金属板制成，并以梅里为特色，梅里是一种原产于西澳大利亚的非常好的红木。[George]the Make It Extreme 建造这个邮箱主要是为了远程控制访问，其想法是他的每个家庭成员都有一个遥控钥匙来打开它。盖子下面有一个输入面板，以防有人丢失或忘记他们的遥控器。

设置很简单。地址编号下的 12V 太阳能电池板连接到太阳能充电控制器，为一个小电池充电。按下遥控钥匙上的 A 按钮会触发锁闩滑动，打开车门。按下 B 按钮可打开室内灯，用于深夜收信。边上的管子是用来放传单和其他邮件的。现在，最酷的功能:当邮件通过插槽时，它会通过打电话通知[乔治]。休息之后，请观看构建/演示视频。

这些年来，我们推出了各种各样的邮箱。这个“支持 wifi”的盒子使用一个亚马逊 Dash 按钮和一个 Pi 在主人的手机上播放 AOL 通知。[当外面的真邮箱里有邮件时，这个可爱的迷你邮箱](http://hackaday.com/2014/11/07/triple-sensor-mailbox-alert-really-delivers/)上的旗子就会升起。
