# 橙色是新的(15 美元)圆周率

> 原文：<https://hackaday.com/2015/09/05/orange-is-the-new-15-pi/>

当 Raspberry Pi 出现时，很难想象你能以如此低的价格获得一个相当完整的 Linux 系统。当然，圆周率已经变得更大了，但是当你试图把圆周率放到一个项目中时，还是会错过一些东西。例如，我想到了 Wifi。你通常做的第一件事是插入一个 Wifi 加密狗，消耗两个 USB 端口中的一个。

Orange Pi 是一个直接的竞争对手，有几个变种。最初，该板的价格约为 30 美元，但配备了 WiFi、1.6 GHz 处理器、8 GB 闪存和 SATA 接口。现在有一款价格约为 15 美元的精简版主板，它删除了闪存和 SATA，以及 WiFi 和原来 4 个 USB 端口中的一个。不过，树莓派没有内置闪光灯。售价 15 美元的 Orange Pi PC 拥有你所期望的 Pi (HDMI 和以太网)以及其他额外功能，如红外接收器和板载麦克风。考虑到它有一个四核处理器，一个 GPU 和 1GB 的内存，15 美元还不错。

Orange Pi PC 兼容为其老大哥设计的操作系统负载，因此你可以运行 Lubuntu、Android、Raspbian、[和更多](http://www.orangepi.org/downloaded/download.html)。你也可以在那里找到一些手册和其他工具。当然，如果你正在寻找社区支持，你可能会找到更多的 Raspberry Pi 用户、论坛和项目——至少现在是这样。

你可以在下面的视频中看到 Debian 在一个非常相似的橙色 Pi 上运行。实际使用的硬件是 Orange Pi Mini 2(T1)，它有一个额外的 USB 端口，但在其他方面与 PC 几乎相同。就个人而言，对于大多数项目，我们宁愿多花一点钱，并获得机载 Wifi(我们覆盖的每个树莓项目)都有 WiFi)。但如果你不需要 WiFi，一台功能强大的小型单板电脑的价格很难超过 15 美元。
