# 用柠檬、孩子和阳光给电动超级跑车充电

> 原文：<https://hackaday.com/2018/06/29/charging-an-electric-supercar-with-lemons-kids-and-the-sun/>

先说第一件事:这个视频中的玩笑，一辆电动超级跑车可以由一个巨大的柠檬电池阵列充电，这就是一个玩笑。尽管如此，这是一个有趣的故事，也是让孩子们接触科学和工程的伟大尝试。

故事是这样的，大众汽车公司找到[Mark]帮助他们为即将到来的派克峰国际爬山赛充电，这是科罗拉多州一年一度的“云端竞赛”。参赛者要在 12.42 英里(20 公里)的赛道上垂直攀爬 4700 英尺(1440 米)，其中有 156 个急转弯。大众的参赛作品是一辆电动超跑，他们给[马克]送了一辆便携式电池车，以他认为合适的最佳方式充电。

与威廉·奥斯曼(William Osman)合作，第一次尝试是一个巨大的柠檬电池阵列，由盛在胶合板框架中的锌和铜条组成。嵌有 1，232 个柠檬的电池表现和你预期的一样好。B 计划很可爱，马克的另一个尝试是增加他“有史以来最有趣的叔叔”的分数。他设计了一种带有再生制动的滑索来给无绳电钻电池充电，然后通过将电池中的能量转化为柠檬水给一群孩子间接收获。糖多的孩子们骑着滑索直到电池充满电。

不过，这仍然是杯水车薪，所以 C 计划看到[马克]在他的屋顶上安装了一个大型太阳能电池阵列；这里的联系是柠檬动力的孩子们要为太阳能电池板设计一个清洁系统。诚然，这是一个薄弱环节，但孩子们玩得很开心，我们不能否认，这辆车在上山时至少会有一部分是柠檬动力的。
