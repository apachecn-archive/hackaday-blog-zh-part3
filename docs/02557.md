# 石蜡油和水点阵显示器

> 原文:[https://hack aday . com/2016/06/02/石蜡油和水点阵显示/](https://hackaday.com/2016/06/02/paraffin-oil-and-water-dot-matrix-display/)

为了准备 Makerfaire，[hwhardsoft]需要做一些演示。所以他们深入挖掘，制作了这个独特的展示。

该显示器使用两个同步蠕动泵推动水和红色石蜡通过一个管子，该管子以可预测的方式自动切换。正如在休息后的视频中可以看到的，水泵持续几分钟，产生了一种看似随机的模式。该模式在最后合并成一个短的文本字符串。不幸的是，即使在对比鲜明的背景下，这篇文章也很难读懂。也许应用紫外线染料会有帮助？

信息显示后，当下一条信息排队时，水和石蜡会落回到储存罐中。正如预期的那样，油和水分离，每种流体水平处的泵将其反馈到系统中。

我们对安装在 DIN 导轨上用于工业环境的 Arduino 产品深感困惑，但随后发现该产品正是[hwhardsoft]制作的演示产品。我们可以看到这种技术在艺术展示中的一些非常酷的变化。
