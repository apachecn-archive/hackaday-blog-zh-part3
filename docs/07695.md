# 寂静之声？Muzo 未能交付

> 原文：<https://hackaday.com/2017/11/13/the-sounds-of-silence-muzo-fails-to-deliver/>

如果你经常坐飞机或者在嘈杂的办公室里工作，你会知道降噪耳机可以帮助你保持清醒。如果你不用耳机就能消除噪音，那不是很好吗？显然，很多人认为这是一个好主意，并资助了一个名为 Muzo 的项目。[Electroboom]借了一个，并困惑于这样一个设备如何工作，开始测试它。一路上，在下面的视频中，你可以看到他在他的衣柜里做了一个两个扬声器互相抵消的整洁演示。

根据(Electroboom 的)测试和其他用户的测试，Muzo 似乎在降低噪音方面做得不多。它可能会添加一些自己的噪音，但这与人们期望该单元做的事情相去甚远。

从理论上讲，这种设备可以感应到一块玻璃或桌面等平坦表面的振动，然后通过振动表面来消除噪音。至少，这是它的一种模式。它还应该能够掩盖你发出的声音，在设备周围创造一种隐私气泡。它还会产生睡眠噪音，当然，这可能会掩盖其他噪音，但不会消除它。

虽然这是一个伟大的想法，但很难想象像这样的设备如何能够任意消除所有听众的复杂声音，只有一个设备位于一定距离之外。有了耳机，声音就直接传入你的耳朵，同时也有很多被动噪声阻挡。“空气中”噪声消除还有其他特殊情况

如果你想坚持耳机，可以被动的得到[好的结果](https://hackaday.com/2014/05/04/block-noise-listen-to-music/)。顺便说一下，亚马逊有一个有趣的[专利申请](https://hackaday.com/2016/08/02/noise-cancelling-speech-recogntion/)与噪音消除相关。
