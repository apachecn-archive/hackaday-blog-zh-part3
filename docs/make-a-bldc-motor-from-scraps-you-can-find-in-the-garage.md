# 用你能在车库里找到的废料做一个 BLDC 汽车

> 原文：<https://hackaday.com/2016/04/10/make-a-bldc-motor-from-scraps-you-can-find-in-the-garage/>

认为你有什么需要建立一个自制无刷电机？正如[JaycubL]向我们展示的，事实证明[一个 bldc 电机可能就在你的眼皮底下生活在碎片中](https://www.youtube.com/watch?v=vnjCrLTMGxQ&feature=share)，这些碎片是我们许多人本来会扔在一边的。为了让我们的头转向，[JaycubL]带我们进入无刷 DC 电机运行的理论。然后，他用螺钉、一个塑料容器、几个轴承、一根金属棒、一些磁铁和少量环氧树脂制造了一台自制的无刷电机。最后，他用现成的电机控制器试了一下。

这不是[JaycubL]第一次涉足自制无刷电机。出于好奇，他还组装了一个带油漆罐外壳的[全功能无刷外转子电机](https://www.youtube.com/watch?v=XeqC7nnae_Y&feature=iv&src_vid=vnjCrLTMGxQ&annotation_id=annotation_3183743077)。

当然，理解原理是一回事，但能够跃入现实世界，从你的废物箱中找到电机的功能开端是一个完全不同的故事！我们敢说，对电机设计原理的精细理解让我们不禁要问:我们还能从苦涩的废料中推出多少其他功能更高的电气和机械元件？为了让你开始，我们将为你指出这台 [CNC 路由器](http://hackaday.com/2016/02/27/trash-can-cnc-gets-a-stepper-upgrade/)的方向，它离五金店只有几步之遥。
