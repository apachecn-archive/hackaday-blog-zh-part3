# 数字风铃

> 原文：<https://hackaday.com/2016/05/22/digital-wind-chimes/>

质量好的风铃不便宜。不管你花多少钱，它们年复一年听起来都一样。如果您对此感到困扰，或许[sensatroniclib]可以提供帮助。他们发布了一个使用 Ototo 音乐发生器的数字风铃的简单设计[。](http://www.instructables.com/id/Sensatronic-Wave-Chimes/)

Ototo 价格合理，并承诺让你用任何东西(嗯，任何有传导性的东西)制作音乐。因为 Ototo 处理所有的音乐制作，所以该项目的唯一真正构建部分是风传感器。传感器是由导电织物制成的，末端有一个用来承重的弹球。

在下面的视频中，你可以看到[Matthew Ward]谈论这款设备，并像弹奏竖琴一样弹奏它。这将是一个很好的学校项目，因为使用 Ototo 很简单，尽管[sensatroniclab]实际上正在进行无障碍音乐项目。

学生面临的一个挑战是制造新型传感器。毕竟，为什么是风铃？为什么不是雨铃或者仓鼠驱动的乐器呢？

当然，音乐设备在黑客时代并不新鲜。我们甚至看了 MIDI 的[黑边，](http://hackaday.com/2015/11/12/black-midi-there-is-no-denser-music/)这对于 Ototo 来说可能有点过了。
