# 丹妮尔·阿普尔斯通:打造 2030 年的劳动力

> 原文：<https://hackaday.com/2017/12/08/danielle-applestone-building-the-workforce-of-2030/>

某天早上你醒来，有一个想法——这个世界不能没有的新东西。你日以继夜地苦干，锁在车库里挥汗如雨，爱迪生说这是你工作的 99%。你启动，你成功，你得到你的原型。新东西的订单源源而来，你在一些旧工厂里得到一个永久的空间，并建立组装工作站。你订购了堆积如山的零件，并把它们放在闪亮的铬合金架上，你已经准备好了——除了一件事。没有人坐在那些漂亮的新工作站前，准备组装你的产品。更糟糕的是，你所有寻找合格人员的尝试都没有结果，你甚至找不到知道拿烙铁哪一头的人。

诚然，烙铁课通常只需要上一次，但这不是初露头角的企业家需要浪费时间的事情。正如 Danielle Applestone 博士在 2017 年 Hackaday 超级大会上所讨论的那样，在 21 世纪找到合格的工人来推动制造业务绝非易事。Applestone 博士知道她在说什么——她是广受欢迎的 Othermill 背后的驱动力，担任 Other Machine Co .的首席执行官，并精心策划了它在桌面铣削领域的崛起。现在更名为 Bantam Tools，该公司有点独特，因为它没有将制造业务转移到国外，而是在加州伯克利的中心组装产品。因此，寻找合格的工人是她每天都非常担心的事情。

### 从服务业到制造业

正如 Danielle 在她的演讲中所阐述的，美国制造业的现状是一个大杂烩。好消息是国内制造业远未消亡，美国仍在世界各国中处于领先地位，美国工人生产的商品占全球的 18%以上。制造商雇佣了 1230 万美国人，这听起来很多，直到你意识到这只是劳动力的 9%。用这么少的工人完成了这么多的工作，这是工业效率的证明，可能与自动化有很大关系，但即便如此，目前仍有约 60 万个制造业岗位等待技术工人。

对当前劳动力缺乏的指责可能很复杂，但像为了准备又一轮国家强制测试而取消高中的车间课程这样的事情肯定于事无补。但是丹妮尔并没有专注于指责，而是在寻找解决方案，91%不在制造业的美国工人是一块成熟的土地。她热情地指着她的一些员工，包括一名针灸师、一名摄像师、一名建筑师、一名零售店员和一名动物园管理员，他们都获得了足够的技术技能，成为制造团队的重要组成部分。

无论他们如何到达那里，丹妮尔的工人来自他们在工作中使用最多技术的地方，即条形码扫描仪或浓缩咖啡机，现在他们组装最先进的数控机器。有些人自己获得了他们需要的技能，有些人在当地黑客空间的帮助下获得了技能，我们可以大胆地说，有些人是在阅读了 Hackaday 后受到启发拿起了烙铁。这些技能让他们有信心超越以前的工作，说:“你知道，我打赌我能做到。”

### 漫长的游戏

虽然丹妮尔通过超越正常的招聘标准解决了劳动力问题，但她不仅仅关注现在。她计划长期从事 it 行业，培养一支能够应对不断变化的制造业格局的员工队伍是长期工作的重要组成部分。为此，丹妮尔提倡让更多的人参与到创客运动中来，在接下来的十年里增加 200 万名新黑客。她认为，解决这个问题的关键是利用公共图书馆，这些图书馆越来越多地提供黑客交易工具，如 3D 打印机。如果仅仅 1000 个图书馆每年“提高”200 人的技能，那么就有 200 万潜在员工可供选择。美国有 120，000 家图书馆，数字开始变得有利。

丹妮尔认为图书馆是扩大创客运动的关键，足以开始吸引那些不会以积极和吸引人的方式接触技术的人。她指出，现在做这件事的硬件和软件都很便宜，所以这只是一个有意愿去做的问题。还有一个问题是，创造新的黑客和让他们意识到这不仅仅是一种爱好，真正的工作可能来自在黑客空间或家庭商店度过的时间之间的闭环。毕竟，在伯克利有很多人做到了这一点。
