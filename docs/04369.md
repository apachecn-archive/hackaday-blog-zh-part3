# Arduino 激光弹球击中目标

> 原文：<https://hackaday.com/2016/12/07/arduino-laser-pinball-is-on-target/>

你有没有想过推出自己的弹球机？这是那种很容易失控的建筑。但是，如果你只是想尝试不同的游戏场地配置，可以从类似于 Arduino 激光弹球这样的东西开始。

它由附有支架的 meccano 部件制成，因此目标很容易在操场上重新排列。[Joe instructables]想要在目标中使用翻转开关，但发现乒乓球太轻，无法启动它们。取而代之的是，每个目标都使用由指向光电池的激光制成的绊网。当乒乓球进入目标时，它打破了光束。这触发了一个螺线管来弹出球，并把它放回比赛中。它还会触发一个场外螺线管，根据目标的难度设置，发出一到三次标准的前台类型的铃声。

脚蹼使用螺线管来拉动由 meccano 制成的杠杆的外侧端，这导致内侧端将球向上推离排水管。偶尔一个脚蹼会卡住，你可以在休息后的演示视频中看到。早期版本有一个 LCD 屏幕来显示比分，但是这个版本不能让它工作。你能帮忙吗？你认为弹力球会启动翻车开关吗？

这不是我们讨论的第一台弹球机。这甚至不是我们报道的第一个用墨西哥玉米制成的。这是[专门为他们准备的一整篇文章](https://hackaday.com/2016/03/26/hacklet-101-pinball-projects/)。还记得[一台 Arduino 让一张旧桌子再次变得伟大的时候吗](https://hackaday.com/2015/07/27/pinball-table-gets-new-lease-of-life-with-arduino/)？
