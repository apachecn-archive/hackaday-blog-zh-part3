# 定制铝车轮教一两件关于铸造

> 原文：<https://hackaday.com/2017/07/01/custom-aluminum-wheels-teach-a-thing-or-two-about-casting/>

对于一些移动项目，如小推车或滚动柜，你的港口货运标准脚轮就可以了。但是一些项目需要大而结实的车轮，这些定制的铸铝车轮无疑是一种声明。主要是，“小心你的脚趾！”

老实说，[布莱恩·奥尔特罗格]的轮子是寻找项目的一个配件，不会很快粉碎双脚。他制造它们只是为了制造它们，但我们对此没有异议。他们有一个伟大的外观，让人回想起重金属完全意味着其他东西的时代，东西是为了持久。当然，用铝铸造会有所不同，但是在家庭铸造中能做的有实际的限制。[Brian]从 CAD 巫术开始，然后为他的模具加工型芯。他没有像消失模或 PLA 那样做，而是从杨木中磨出芯材。他的混合砂比我们通常在自制砂型铸造中看到的要好得多——可以用二氧化碳固化的水玻璃砂。他所有的精心准备意味着倾倒顺利进行，车轮看起来很棒。

我们最近特别介绍了几个金属铸造项目，[一些进展顺利](http://hackaday.com/2016/07/07/the-best-gingery-lathe-video-series-to-date/)和[一些进展不顺利](http://hackaday.com/2017/06/07/fail-of-the-week-sand-casting-copper/)。[Brian]看起来他知道他在做什么，我们很欣赏他在这里展示的工艺。
