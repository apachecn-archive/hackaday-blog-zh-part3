# Hackaday 奖参赛作品:一台物联网显微镜

> 原文:[https://hack aday . com/2016/07/20/hack aday-prize-entry-an-internet-of-things-microscope/](https://hackaday.com/2016/07/20/hackaday-prize-entry-an-internet-of-things-microscope/)

为了进入黑客日奖的公民科学家部分，圣路易斯黑客空间 Arch Reactor 的人们正在建造一台显微镜。不是一般的显微镜——这台显微镜是低成本的，数字化的，有着惊人的高放大倍数和非常好的光学系统。这是物联网显微镜，就像所有为公民科学家准备的好仪器一样，它是教室和发展中国家的非凡工具。

当你想到“教室显微镜”时，你可能会想到储藏室后面的一堆旧光学元件。这些显微镜是纯光学的，没有拍摄数码照片的能力。玻璃是好的，但当你在处理一个满是粘手的八年级学生的教室里制造的 30 年前的设备时，你不会得到一个扫描台。

物联网显微镜包括一个扫描台，它在 X 轴和 Y 轴上移动样本，将数字图像拼接在一起，创建一个非常大的图像。这是廉价数码显微镜的杀手锏，Arch Reactor 的工作人员正在用一些廉价的步进电机和步进电机驱动器来完成这项工作。

其余的电子设备是围绕一个树莓 Pi，树莓 Pi 相机(最近得到了很好的分辨率升级)，以及一些显微镜目镜和物镜构建的。其他一切都是 3D 打印的，这使得这款显微镜非常便宜，非常容易使用，并且具有一些杀手级的功能。
