# 测试一个穷人的矢量分析仪

> 原文：<https://hackaday.com/2018/03/29/putting-a-poor-mans-vector-analyzer-through-its-paces/>

如果有任何关于电子学的东西接近黑魔法的水平，那就是天线理论。致力于该主题的整本书通常只是触及表面，除非你是一个拥有可视化所发生事情所需的所有昂贵测试设备的专业人士，否则你的天线游戏很可能比理论更实用。这并没有什么错——几代人以来，hams 和其他 RF 爱好者一直在使用能够工作的天线，却没有真正理解为什么会这样。

但我们生活在未来，正确分析天线设计的工具现在几乎人人都可以使用。[Andreas Spiess] [最近回顾了一种这样的仪器](https://www.youtube.com/watch?v=ZpKoLvqOWyc)，N1201SA 矢量阻抗分析仪，通常以低于 150 美元的价格从海外购买。[安德烈亚斯]的评论似乎没有得到赞助，所以我们似乎得到了他的直言不讳的意见；剧透警报，他喜欢。而且有充分的理由；虽然不是一个全矢量网络分析仪(VNA ),将打击你的钱包数千美元的洞，这个仪器看起来像一个不可思议的除了你的测试套件。测试单元的工作频率为 137 MHz 至 2.4 GHz，因此它涵盖了 VHF 和 UHF ham 频段以及 LoRa、WiFi、cell、ISM 等。但是当然，[Andreas]不仅仅是回顾这个单元，他还以他平易近人的风格给我们提供了有益的理论。

[那个有瑞士口音的家伙]这些天做了很多很棒的工作，涵盖了从[如何不忘记你的杂务](https://hackaday.com/2018/03/10/iot-chore-reminder-for-the-serially-forgetful/)到[逆向工程物联网盖格计数器](https://hackaday.com/2017/10/08/connecting-a-50-geiger-counter/)的一切。看看他的频道——几乎他做的每件事都值得一看。
