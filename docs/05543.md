# 点击鼠标，从横向变为纵向

> 原文：<https://hackaday.com/2017/04/05/landscape-to-portrait-at-the-click-of-a-mouse/>

现代的 16:9 宽高比显示器可能非常适合在网飞上观看宽屏电影，但对于大多数 pdf、Word 文档和某些网页，横向显示就不行了。但是如果你不打算写下一部伟大的美国小说，也不愿意使用纵向模式，那就不要——制作一个自动旋转的显示器来动态切换你的宽高比。

像我们许多人一样，[Bob]发现某些内容不太适合已经成为显示器标准的电影格式。他的解决方案在概念上很简单，但在工程上有点挑战性。使用一个旋转转盘作为一个巨大的轴承，[鲍勃]建造了一个转体，可以由 NEMA 23 步进器和环形齿轮的 3D 打印部分提供动力。由于转盘顶部和底部之间的间隙很窄，[Bob]不得不费很大的劲才能穿过安装五金件的孔，但最终整个过程非常顺利。

我们唯一的异议是为支架焊接镀锌管，这总是让我们心惊肉跳。但是我们承认，仅仅用一个纸模板，管切口就变得很棒。我们怀疑，如果他使用一个增强的等离子动力管道切槽机会不会好得多。
