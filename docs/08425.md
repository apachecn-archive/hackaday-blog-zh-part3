# 一个落地钟

> 原文：<https://hackaday.com/2018/02/01/a-grandfather-clock-barbot/>

俗话说，某处五点；当时钟终于敲响整点的时候，[那个时钟可以给你倒一杯酒](https://www.reddit.com/r/RASPBERRY_PI_PROJECTS/comments/7tf6wr/raspberry_pi_controlled_cocktail_dispenser_in_a/)感谢 redditor [Diggedypomme]。

这个酒吧时钟可以分配多达四种不同的烈酒和四个混合器的饮料，并通过语音、键盘或网络控制接受订单。一个皮带驱动的饮料装载平台通过弹簧门推出，一旦容器到位并收到订单，蠕动泵就会分配烈酒，同时伺服打开混合器的水龙头——这是一种管理通常为碳酸液体的简单得多的方法。一个树莓派充当这个老计时器的大脑，一个 Arduino 控制灯光，一顶帽子控制伺服系统。[这里有一个关于酒吧后面发生的事情的更深入的游览](https://www.youtube.com/watch?v=hdxV_KKH5MA),但是休息之后看看视频，完整地浏览几个饮料订单！

 [https://www.youtube.com/embed/_GzJ4YeVhNk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent](https://www.youtube.com/embed/_GzJ4YeVhNk?version=3&rel=1&showsearch=0&showinfo=1&iv_load_policy=1&fs=1&hl=en-US&autohide=2&wmode=transparent)



在未来的版本中，[Diggedypomme]希望播放音乐，灯光根据所倒的饮料闪烁不同的图案，并可以选择依次提供多种饮料。倒饮料时的一些飞溅也可以通过调节龙头的高度或安装喷嘴来弥补。

尽管这个老牌饮料机器人很酷，但 barbots 并不局限于酒精，茶也是一个可行的选择。然而，如果你更喜欢你的饮料自动售货机，你可能会惊讶什么[两个马达可以完成](https://hackaday.com/2017/08/31/open-source-barbot-needs-only-two-motors/)。
