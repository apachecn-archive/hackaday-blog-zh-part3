# 烙铁上的俄罗斯方块

> 原文：<https://hackaday.com/2017/07/07/tetris-on-a-soldering-iron/>

我们的评论者都对开源的 TS100 烙铁铅笔赞不绝口:比如“它焊接得很好”。但是我们都有可以很好焊接的烙铁。烙铁上的开源固件可能带来什么额外的价值？【Joric】为我们回答了那个问题——[它会玩俄罗斯方块](https://www.youtube.com/watch?v=Buzew1z1AhQ)。(视频嵌入下方。)

虽然这很酷，但直到我们在 GitHub 上通读了 [README 之后，我们才意识到这个黑客最有趣的部分。每输一局，铁尖温度就升高 10 度。受虐狂的俄罗斯方块？一些可怕的酒吧赌注的形成？我们只是很高兴它是开源的，因为我们没有那么好，它会变得太热而无法快速处理。](https://github.com/joric/ts100tris)

我们还没有尝试过 TS100，但这个黑客几乎把我们推向冲动购买。例如，如果你不喜欢字体，还有其他版本的固件。现在，俄罗斯方块。这会成为你期待已久的热门新游戏平台吗？请在评论中告诉我们。
