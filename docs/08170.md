# 用网 3D 打印可穿戴设备

> 原文:[https://hack aday . com/2018/01/06/3d-printing-wearables-with-a-net/](https://hackaday.com/2018/01/06/3d-printing-wearables-with-a-net/)

如果你想制造可穿戴设备，你需要知道如何缝纫，对吗？也许不是。虽然我们确信它会派上用场，但[Drato](也被称为[RobotMama])展示了她如何在网状织物上直接打印设计。你可以在下面看到这个过程的视频。

休息后的视频显示了一个 Ultimaker，但这台打印机真的没有什么特别的。诀窍是打印几层，暂停，然后在继续打印前将织物插入打印机下。

[Drato]在插入织物后将其压住，并提到您也可以使用胶水将其压住。我们想知道是否一些牛头犬或鳄鱼夹可能有用。我们唯一担心的是，如果这种布料是由合成纤维制成的，它可能无法承受不熔化的热塑料。

[Drato]提到她使用透明硬纱，这是一种常见于婚纱的薄纱。然而，她没有提到她使用的是聚酯、丝绸还是尼龙类型的面料。一项小小的研究表明，聚酯和尼龙面料在 295 摄氏度左右就会融化。丝绸更难追踪，但既然你可以在中等温度下熨烫，那也可能行得通。当然，它融化的温度和它变形到无法使用的温度可能是不同的，所以做一些实验可能是明智的。

真正激起我们兴趣的是无需缝纫就能创造可穿戴设备的应用。我们很想知道您还能找到哪些直接在织物基底上打印的应用。

即使你可以在织网上 3D 打印，你可能还是应该弄清楚[整个缝纫的事情](https://hackaday.com/2016/10/26/why-you-should-own-a-sewing-machine/)。或者，你可以[训练一个机器人来做这件事](https://hackaday.com/2016/09/12/sewbo-robot-sews-up-automated-garment-manufacturing/)。
