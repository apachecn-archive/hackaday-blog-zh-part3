# 从内部学习 555

> 原文：<https://hackaday.com/2018/02/21/learning-the-555-from-the-inside/>

了解 555 定时器如何工作以及如何使用的一个方法是了解这些引脚的含义以及它们的连接方式。一种更令人愉快，也可以说是更有用的学习方法是通过观察每种操作模式下的内部情况。[Dejan Nedelkovski]制作了这样一个视频，他在视频中讲述了 555 定时器 IC 如何从内部工作。

我们特别喜欢他如何立即消除恐惧因素，他首先展示了一个包含所有独立元件的原理图，然后将它们分成不同的组成部分:两个比较器、一个分压器、一个触发器、一个放电晶体管和一个输出级。在将内部提升到一个更高的水平后，他接着通过例子，附加外部元件，为三种工作模式中的每一种工作模式:双稳态、单稳态和非稳态。如果你已经熟悉 555，那么你会喜欢沿着记忆之路的旅程。如果你不熟悉它，那么你很快就会熟悉了。看看下面他的视频。

这不是我们第一次玩弄这个奇妙芯片的内脏。几年前，我们都对这个[巨型分立 555 套件](https://hackaday.com/2013/09/22/whats-inside-a-555/)感到高兴，而最近，这个[拆除了实际的芯片](https://hackaday.com/2016/02/23/555-teardown-and-analysis/)。
