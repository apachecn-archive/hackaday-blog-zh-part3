# 为什么不通过液晶显示器暴露你的多氯联苯？

> 原文:[https://hack aday . com/2017/10/16/why-not-expose-your-PCBs-through-an-LCD/](https://hackaday.com/2017/10/16/why-not-expose-your-pcbs-through-an-lcd/)

大多数涉足过电子构造领域的人都会以某种形式熟悉印刷电路板的生产过程，即在进行蚀刻之前，通过透明掩模曝光紫外线敏感涂层。年长的读者会用醋酸纤维上的绉纱纸带手工制作他们的面具，而年轻的读者可能会从他们的 CAD 软件包开始激光打印。

如何改进工艺，完全去除醋酸纤维面膜？[Ionel Ciobanuc]可能有答案，形式是通过 LCD 屏幕曝光。休息下面的视频显示了它是如何完成的，从在电路板上应用光致抗蚀剂涂层的一个序列(可能有点太长)开始，然后将 LCD 放在 UV 灯的顶部，将电路板放在堆叠的顶部。

这是一个有趣的演示，而且肯定消除了 PCB 创建过程中的一个步骤，因为它将图案直接从计算机带到电路板上，无需中间媒介。是否值得花费在液晶显示器上取决于你，毕竟一张醋酸纤维纸非常便宜，如果你已经有一台激光打印机，你就可以使用了。我们很想知道 LCD 本身的任何塑料组件是否会因长期暴露在强烈的紫外线下而受损。
