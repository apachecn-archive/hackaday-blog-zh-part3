# 2017 年月食:来自死火山的报道

> 原文：<https://hackaday.com/2017/08/24/eclipse-2017-report-from-an-extinct-volcano/>

位置，位置，位置——对房地产来说至关重要的东西对观看日食来说也是至关重要的，没有听起来太自吹自擂，我们这些人在爱达荷州东南部的死火山南梅南巴特山顶上，绝对找到了它。我们不仅有完美的天气，我们有一个极好的露营经历，美味的食物，壮丽的自然环境，以及一个 800 英尺高的栖木，它位于一个向东和向西无尽延伸的大平原上。一切都是为了完美的月食体验而设置的，我们没有失望。

### 快乐的露营者

日食本身仅仅是一个持续不断的周末的高潮，而这真正开始于一月份，当时我开始计划这次旅行。那是我第一次写日食的时候，并宣布我选择了爱达荷州雷克斯堡外的梅南巴特作为我的理想地点。原来，虽然北梅南巴特是公共土地，南梅南巴特是私人财产的一部分，由布伦特冈德森先生拥有。他实际上读了我的 Hackaday 帖子，并利用它来衡量向日食观测者开放他的土地的兴趣。

谢天谢地，他认为这是值得的，他和他的邻居竭尽全力。在周六晚上布伦特在他家草坪上举办的欢迎野餐会上，我和他简短地聊了几句。他显然是个大忙人，但仍设法在食品线上工作，并端上一些我吃过的最好的炸鸡。很明显，他很喜欢见到那些几个月来一直和他通信的人，他们来到了他在他家和斯内克河之间的苜蓿地里建立的露营地。

对于布伦特的热情好客，我不能说太多——有人可能会试图利用绝望的日食观测者来勒索尽可能多的钱，而提供尽可能少的回报，布伦特和他的家人只是不断给予。食物、当地服务指南、为残疾人和那些无法爬到最佳观赏位置的人提供的住宿，甚至 t 恤和日蚀眼镜之类的商品——一切都是免费的，或者价格极其合理，我怀疑这几乎不够他的开销。每个在布伦特家露营的人都欠他一份情，感谢他和他的家人为我们所做的一切。

### 在边缘

当然，这次经历中最重要的是地点本身。梅南孤峰是大约一万年前形成的火山锥，高出斯内克河平原 800 英尺。周日，我爬上了南梅南孤峰，观看日出，并寻找观看的地点；这是一次艰难的攀登，但努力是值得的。从山顶望去，景色非常壮观，斯内克河平原向西北延伸 40 英里，到达莱姆希山脉，向东延伸 20 英里，到达提顿山脉。我选择的地点是边缘北部的一块岩石，这是我能找到的最高点。我做了大量笔记，记下了我探险需要的东西，然后返回营地。

当重要的一天到来的时候，我大约在早上 4:30 出发去边缘。天还是漆黑一片，当我小心翼翼地爬上孤峰时，爱达荷州乡村未受污染的天空给了我一幅美丽的银河图。眼睛从鼠尾草丛中回瞪着我的前灯；或许是鹿？或者郊狼。经过一个小时的艰苦攀登，我到达了我的栖木，立起了我的领地，观看了另一次壮观的日出，等待其他人的到来。我儿子首先想到了克里斯，他是一位业余读者，也从北爱达荷州赶来参加我们周日晚上的聚会。接下来是我的女儿们，然后是我的妻子为我们大家端着早餐三明治。没有什么比在户外进行简单的食物测试更好的了。

我们等待的时候天气很冷——华氏 50 多度，还刮着风。偏科终于在 10 点半左右开始了。那时，我的栖木上已经挤满了其他的观察者，所有人都受到了欢迎。对我来说，关于日食最令人惊讶的事情是通过与完全陌生的人分享它，这种体验得到了极大的提升。我们有一对从圣地亚哥开车过来的夫妇，一个在毯子下挤成一团的年轻家庭，还有一群从斯洛文尼亚远道而来的年轻人，他们在来看日食之前在国家公园进行了一次盛大的旅行。我分享了我的 Hackaday eclipse 眼镜，其中大部分都被立即撕开，它们的滤镜贴在智能手机摄像头上。必要时每个人都是黑客。

### 昨晚

我真的没有准备好偏心的最后时刻和整体的突然出现会是什么样子。我们把摄像机对准了西北方向，等待月亮的影子穿过平原向我们飞来。我急切地希望它不仅仅是逐渐变暗，我没有失望。这是我发病时的原始视频。我把音频留了下来，因为它显示了每个人有多兴奋:

很难描述——也更难用相机捕捉——日全食前的光线质量有多怪异。这看起来几乎像一个糟糕的 CGI 渲染，当太阳接近成为一个点源时，所有的东西都有硬边。相机也没有捕捉到黑暗逐渐逼近的方式，就像海啸席卷海洋地平线一样。虽然没有明确的阴影边缘，我们可以清楚地看到虚假的暮色吞噬了我们面前的平原。

我所听到的关于全食的一切都是真实的，甚至更多。气温骤然下降，风加快并改变了方向，我们周围的生物，如昆虫和以它们为食的燕子，成群结队地出来了。我们看到了 360°的日落，天空变黑，星星和行星出来了。金星明亮地照耀在东南方向的天空，我的相机在被遮住的太阳周围挑出了几颗明亮的星星。

不过，我没想到的是颜色。我不知道我期待什么，但这不是我所看到的。天空是从浅蓝色到天蓝色再到漆黑的渐变，日冕是银白色的光线。这是最丰富多彩的单色，如果这有任何意义的话。我用另一台相机拍下了一些照片:

### 德州 2024！

仅仅用语言是不够的，照相机也不能公正地记录我们的经历，所有这些都被在场的其他人无限放大了，他们同样渴望看到如此简单却如此惊人的东西。既然我已经经历了全食，我无法想象看到日偏食还会感到满意。事实上，这一切结束得如此之快，只会加深我再次看到这一点的需要。

他们说你第一次吃全食后说的第一句话是“下一次是什么时候？”这是千真万确的。幸运的是，再过七年，另一次日全食将席卷美国大片地区。那是我策划和计划下一次旅行的七年时间。我甚至想在 2019 年飞往智利的 T2。从塞罗托洛洛天文台观看日食可能会比这种体验更好。

对于那些不明白为什么大惊小怪的人，我只能说:走吧。你会明白的。
