# 丘巴卡面具黑客欢呼哀嚎伍基

> 原文：<https://hackaday.com/2016/08/15/chewbacca-mask-hack-cheers-up-the-whining-wookie/>

一旦尝试过，即使是成年人也无法摆脱这个滑稽可笑的丘巴卡面具。还有，发声玩具字面上喊着要被黑。厌倦了呜呜叫的伍基，[约翰·派克]开始给朱伊的表达方式带来变化，[将面具的记忆和词汇扩展到前所未有的水平](https://learn.adafruit.com/happy-chewbacca-mask)。

在他的黑客中，[约翰]保留了面具原来的下颚限位开关和扬声器，但用一个 *Adafruit Audio FX 迷你音板*取代了声音生成电子设备。这个来自[Ada 的]锦囊妙计的小玩意从板载 16 MB 闪存回放样本，并且可以由 8 个触发引脚之一或通过其 UART 接口触发。[John]还添加了一个用于在不同声音之间切换的旋转选择器开关，一个 LiPo 电池组和充电器，以及一个小型 D 类放大器。面具装好后，丘巴卡准备好再次大笑。欣赏视频，在视频中[约翰·派克]会带你了解整个过程，或者直接跳到 [13:45](https://www.youtube.com/watch?v=2gveeXyNoqg&t=13m45s) 来看看*快乐的丘巴卡面具*的使用过程。
