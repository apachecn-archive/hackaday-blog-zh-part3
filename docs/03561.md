# Hackaday 奖参赛作品:高端前置放大器

> 原文：<https://hackaday.com/2016/09/16/hackaday-prize-entry-high-end-preamps/>

虽然光盘的受欢迎程度有所上升，这要归功于一种方便的格式，它提供了 16 位深度的无损高质量 44.1 KHz 采样率，但一些人仍在乘坐 2010 年的乙烯基潮流。随之而来的是对最好的硬件的需求，这意味着由知道自己在做什么的人设计昂贵的墨盒和前置放大器。

对于今年的 Hackaday 奖，[skrodahl] [正在为动圈转盘拾音器建造一个非常非常好的前置放大器](https://hackaday.io/project/12483-muffsy-mc-head-amp)。它已经制作完成，经过测试，结果很好:它产生 36 至 46dB 的增益，-110dB 的动态范围，相对于 5mV 输入的信噪比为 79.46。这使得这种前置放大器进入了与序列号、晶格和其他发烧友胡说八道的前置放大器相同的领域。

这个前置放大器的质量来自于设计，就像任何好的开放硬件项目一样，[skrodahl]已经将这个前置放大器的原理图、PCB 和布局[完全开放。](https://hackaday.io/project/12483-muffsy-mc-head-amp/log/43390-muffsy-mc-head-amp-final-design)这是一个很好的前奏，也是 Hackaday 奖的一个很好的参赛作品。
