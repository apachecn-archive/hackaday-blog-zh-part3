# 电脑走向好莱坞

> 原文：<https://hackaday.com/2018/06/21/computers-go-hollywood/>

你有没有在看电视节目或电影时发现一台熟悉的电脑？詹姆斯·卡特照做了，他创建了一个网站来帮助你识别电视节目和电影中出现的旧电脑。我们在研究另一个关于旧电脑的帖子时遇到了这个，想知道它是否是任何旧电影。不是的。

你可以通过电脑或标题进行搜索。还有关于计算机对每一项的可见性、真实性和重要性的评级。该数据库只包含虚构作品，不包含商业广告或纪录片。我们能找到的最古老的条目是 1950 年的*目的地月球*，它由阿格微分分析器主演。嗯，我们想还有约翰·阿彻尔。我们假设 GE 有一个好的代理，因为同一台计算机出现在*地球大战飞碟* (1956)和*当世界碰撞* (1951)中。你可以在下面的*地球大战飞碟*中看到电脑外观的剪辑。

当我们没有看到牛郎星 8800 与 600 万美元的人一起上市时，我们变得很兴奋。但是，唉，[詹姆斯]有一张[的清单，上面列着他还没来得及做的事情](http://www.starringthecomputer.com/movies.txt)，而它就在那张清单上。很难判断哪台电脑拥有最多的屏幕学分，尽管我们很高兴看到巴勒斯 B205 出现的频率，包括在蝙蝠洞里。

我们经常在广播中看到一些设备而不是电脑，但我们还没有找到相关的参考网站。老*号太空堡垒卡拉狄加*号在泰克测试设备上发了大财。如果你还记得 25 世纪的节目*Buck Rogers*，Huer 博士的桌子上有一个高压灭菌器，与此同时，Hackday 的一个成员在他的实验室里也有一个。

我们一直在等待机器人先生在他的平板电脑上打开 Hackaday，但迄今没有任何收获。当然，电脑在屏幕上的使用方式可以从准确的[到荒谬的](https://hackaday.com/2013/04/11/top-10-best-hacking-scenes-in-movies/)。如果你想知道哪个是哪个，似乎[每个人都有自己的看法](https://hackaday.com/2013/03/19/ask-hackaday-what-movies-have-the-bestworst-hacking-scenes/)。
