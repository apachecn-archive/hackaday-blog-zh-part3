# 这是谢妮！是 VFD！不，这是一个定制的 LED 显示管

> 原文:[https://hackaday . com/2018/03/26/its-a-谢妮-its-a-VFD-no-its-a-custom-led-display-in-a-tube/](https://hackaday.com/2018/03/26/its-a-nixie-its-a-vfd-no-its-a-custom-led-display-in-a-tube/)

喜欢 Nixies 的外观，但他们似乎有点过头了？或者你只是不想要高压电源的麻烦？然后，也许[这些仿谢妮 LED“管”显示器](https://hackaday.io/project/85431-boboxie-nixie-vfd-tube-style-desktop-clock)会找到一种方法进入你的下一个时钟制造。

对于他的 2018 年 Hackaday 奖参赛作品，[bobricius]认为世界需要的是一个谢妮，而不是一个谢妮。为此，每个显示器都由七个表面贴装 led 组成，这些 led 焊接在七段形 PCB 上，并嵌入玻璃管中。led 采用 4014 封装，因此只有 4 毫米长，但它们在亮度上弥补了尺寸上的不足。我们不确定这是否是相机的一个小把戏，但 led 肯定会发出蓝色的光，让人想起真空荧光显示器——就像谢妮和 VFD 都卷在一个包装中。目前的情况下，隐藏在 PCB 下部的时钟电路，只是塑料，但这将在一个精致的木箱看起来漂亮。

这可能是另一个从未出现过的数码管杀手吗？也许吧，但无论它最终出现在哪里，我们都喜欢它的外观，我们也很高兴它是[早期黑客奖的参赛作品之一](https://hackaday.io/submissions/prize2018/list)。你有什么东西可以参加地球上最大的硬件竞赛吗？如果没有，开始破解吧！
