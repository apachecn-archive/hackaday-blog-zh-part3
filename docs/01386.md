# 真空？不，今晚我们要向尘螨开战

> 原文:[https://hackaday . com/2016/02/01/vacuum-no-tonight-we-go-war-again-the-dust-mite/](https://hackaday.com/2016/02/01/vacuum-no-tonight-we-go-to-war-against-the-dust-mite/)

对于我们这些有幸拥有 Roomba 的人来说，它让打扫房间里的灰尘变得轻而易举——但它还可以做得更好。这就是为什么[马塞尔]花了他的周末升级他的 Roomba——或者我们应该说， [Doomba。](http://www.marcelvarallo.com/i-dont-vacuum-i-go-to-war-with-dust/)

他开始时意图不大。是什么阻止了他的 Roomba 跑得更快呢？他非常确定自己可以稍微提高一点产量。他穿上白大褂，戴上安全眼镜，开始把这个小机器人升级成更强大的东西。

12 小时后，他回到了清醒的现实中。他不仅升级了他的 Roomba，还把它变成了一个迷你战争机器 Doomba。

升级后的机器人配备了一个 UE 音箱，可以让他在扑向无助的尘螨猎物时播放《女武神的飞行》。现在可以使用 Playstation 2 无线加密狗和 Arduino 来访问远程控制功能。Raspberry Pi 允许网络摄像头录制和 WiFi 功能，特别是远程触发任务。他甚至加入了一组电容器，以防止 SPI 端口断电。

自从 Roomba 出现以来，人们就一直在对其进行修改以适应其他用途——见鬼，甚至 Roomba 公司自己[也发布了一个黑客版本！](http://hackaday.com/2014/12/21/irobot-releases-hackable-roomba-without-the-vacuum/)
