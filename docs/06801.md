# 铁人 TEO 提供的新熨衣服

> 原文:[https://hack aday . com/2017/08/19/fresh-pressed-衣服-teo-the-iron-man 的礼遇/](https://hackaday.com/2017/08/19/fresh-pressed-clothes-courtesy-of-teo-the-iron-man/)

正如许多任务一样，机器人可能很快就会在我们去上班之前帮我们熨衣服。TEO 是由西班牙赫塔菲卡洛斯三世大学机器人实验室的一个团队制造的，它是一个高度关节化的机器人，可以爬楼梯，开门，最近还增加了熨烫技能。

来自 TEO 头部深度感应摄像头的数据被算法梳理，分解成数千个点——0 是平滑的，1 是衣服上确定的线条。将这些点的值与其邻居的值进行比较，使得 TEO 能够识别褶皱，而无需预先知道新熨烫的衣服是什么样子。

一旦识别出令人不快的皱纹，这些信息就会被传递给 TEO 的其他流程，从而完成工作。它还不知道如何区分褶皱和拉链等其他特征，完成这项工作需要一段时间——这两者在熨烫时都可能是一种危险——但这只是时间问题。
