# 精密 DIY 卡尺？那是云纹！

> 原文:[https://hack aday . com/2018/03/07/0-05-mm-precision-thas-a-moire/](https://hackaday.com/2018/03/07/0-05-mm-precision-thats-a-moire/)

莫尔图案是一种艺术，物理，现在工具设计的东西！来自 Mojoptix 的[juldozer]创造性地使用莫尔图案为他定制设计的 [3D 打印卡尺](http://www.mojoptix.com/?p=178)实现了 0.05 毫米的精度目标。他的卡尺被设计用来验证原始 3D 模型的 3D 打印。当选择哪种卡尺最适合某项工作时，他指出了衡量卡尺的两个关键特征，即准确度和精确度，他在他的信息视频中解释了这两个特征的定义。他为自己的设计设定的准确度和精度值分别为 0.5 毫米和 0.05 毫米。

[![](../Images/ff05981a851d905e2db41aa7dc6d3450.png)](https://hackaday.com/wp-content/uploads/2018/03/ezgif-com-crop.gif) 通过实验莫尔条纹的不同参数:一个图案相对于另一个图案的比例，两幅图像上黑线的距离，以及黑白线条的粗细。[juldozer]发现后者是将微小的线性运动放大并转化为突出的视觉测量的最佳方式。使用他提供的 Python 脚本，他通过将线条粗细比例从 50:50 增加到 95:5 来生成莫尔图案的图像，从黑到白创建指向 1/100 毫米的三角形莫尔条纹。厘米和毫米的测量由传统的标尺布局表示。
