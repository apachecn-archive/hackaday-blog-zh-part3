# 尘土归尘土，果冻归果冻:一把非常奇怪的刀的旅程

> 原文:[https://hackaday . com/2018/06/25/dust-to-dust-and-jello-the-journey-of-a-very-strange-knife/](https://hackaday.com/2018/06/25/dust-to-dust-and-jello-to-jello-the-journey-of-a-very-strange-knife/)

你觉得果冻怎么样？品尝起来还不错，但更多的是它变得有多摇晃。如果果冻是一种硬糖，可能没人会吃。它会很快被限制在奶奶似乎总是有的那碗草莓糖上。你觉得刀怎么样？我们在黑客日。这里的大多数人工具箱里都至少有几个。他们中的一些人有不止一对，包括磨刀石。可以肯定地说，他们可能喜欢这个概念。现在，如果你能把两者结合起来会怎么样？两个最爱总比一个好。一把果冻刀，虽然看似不可能，但却相当令人印象深刻,[ kiwami Japan]就是这么做的，还有更多。

他从几十种可爱的果冻小吃(果冻？)，并从最摇晃的基础上，他建立了一个杰作。当务之急是吃掉几个掉队的，同时他决定如何处理剩下的。一点蓝色的食用色素，更多的明胶，以及后来几个牛形碗和水罐的帮助，[kiwami japan]已经融化了幸存者，得到了一个平板。一旦充分冷却，它就成了一个漂亮的刀形果冻坯。

从这一点来看，这看起来很容易。如果你想了解【kiwami japan】在一把果冻刀上投入了多少工作，你可以看看[我们之前关于制作钢刀的文章](https://hackaday.com/2018/06/07/building-a-knife-by-hand-is-just-as-hard-as-you-think/)。有大量的打磨、锉磨和研磨工作要做，而[kiwami japan]用一块风干的脆果冻来完成所有这些工作。一旦他展示了必须的剪纸技巧，他就把果冻重新融化并重新密封到原来的包装里，同时吃了几个。

这本身就够整洁了，但是他的果冻刀甚至不是他制作的第二锋利的非金属刀。这一荣誉属于他的意大利面刀，他还吃了一份美味的自制奶酪酱。不想吃意大利面吗？那么你可能想试试他的[巧克力刀](https://www.youtube.com/watch?v=Jct33SgQofY)。喜欢把食物放在刀上而不是刀里？看看他的[碳纤维刀](https://www.youtube.com/watch?v=TCkmKgUp1ak)，他做过最锋利的。只有[纸板](https://www.youtube.com/watch?v=jvo86AHovFc)躺在周围？用它做一把刀。[铝箔](https://www.youtube.com/watch?v=owseTngZFMI)？[保鲜膜](https://www.youtube.com/watch?v=IvPbQdbsQUI)？[水](https://www.youtube.com/watch?v=r1GzOadFYpg)？基本上，如果你以前看过这个材料，[kiwami japan]已经尝试并成功地用它建造了一个武器。他为什么这样做仍然是一个谜，但它肯定很酷。
