# 摄像机四轴飞行器差点撞上障碍滑雪

> 原文:[https://hack aday . com/2015/12/23/camera-quad copter-about-hits-slalom-skier/](https://hackaday.com/2015/12/23/camera-quadcopter-almost-hits-slalom-skiier/)

在周三的世界杯障碍滑雪锦标赛中，滑雪冠军[希尔舍]差点被一架失控的无人驾驶摄像机击中，这架摄像机在他跑步拍摄时坠毁在他身后。休息后观看嵌入的(恐怖)视频。

根据 Heise.de ( [谷歌翻译链接](https://translate.google.com/translate?sl=auto&tl=en&js=y&prev=_t&hl=en&ie=UTF-8&u=http%3A%2F%2Fwww.heise.de%2Fnewsticker%2Fmeldung%2FKameradrohne-stuerzt-bei-Weltcup-Slalom-auf-Ski-Piste-3055248.html&edit-text=&act=url))上的这篇文章，飞行员获得了认证并被允许飞行四轴飞行器，但只能在没有观众在场的走廊上空飞行。在最初的几次飞行后，飞行员显然偏离了航线，很明显失去了对直升机的控制。

世界杯立即禁止了摄像无人机。根据国际滑雪联合会男子比赛主任的说法，无人机“对安全来说是一件坏事。”斯基耶[赫希尔]说他“不知道那是什么，尽管他感觉到了一些东西。”当时他认为这可能是一些雪或一根柱子倒了。因此，他“保持专注”。不过，从无人机的大小来看，我们非常肯定，如果它早半秒钟落下，他会注意到的。

你怎么想呢?当有人在下面时，携带相机的大型无人机飞行是否太危险？对他们进行登记就足够了吗？还是我们需要更强有力的禁令？或者这只是一个反常的事故，我们看到的是对新技术的过度反应？
