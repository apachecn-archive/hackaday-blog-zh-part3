# 周五黑客聊天:电路板艺术

> 原文：<https://hackaday.com/2018/04/18/friday-hack-chat-circuit-board-art/>

我们现在正处于印刷电路板的黄金时代。不久前，制作自己的电路板要么需要大量资金，要么只需稍少的资金，并使用一些专有的垃圾 PCB 布局工具。现在，每个板房都说格柏语，你花五块钱就能买到十包来自中国的多氯联苯。这种难以置信的成本降低意味着人们正在用印刷电路板制作艺术品。我们见过肖像、风景和迷因。这确实是玻璃纤维和阻焊材料呈现的新艺术媒介的开始。

看看这个闪闪发光的新艺术。有一个关于 PCB 绘画的 facebook 群组，一些 Badgelife 的工作人员依靠[木刻和油毡雕刻技术](https://twitter.com/Team_Missing_No/status/984525945573797888)来创作铜和玻璃纤维的艺术作品。

在本周的黑客聊天中，[我们将讨论 PCB 艺术品](https://hackaday.io/event/112119-pcb-artwork-and-photo-conversions)。我们本周黑客聊天的嘉宾将是[安德鲁·索瓦]，一位电气工程师，KiCad 的倡导者，和[，那个让我赚了更多钱的家伙](https://hackaday.com/2017/03/22/making-more-of-me-money/)。通过简单地从 Hackaday 自己的[Joe Kim]中选取一些精彩的插图，并应用 KiCad 的 bitmap 2 组件工具，就可以创建 Benchoff Nickel。自从镍币问世以来，[Andrew]一直致力于将他的技术扩展到交叉影线、背光和半色调技术。

在这次聊天中，我们将谈论所有的 PCB 艺术品，包括 PCB 艺术的最初阶段，工程师在 Xboxen 和其他消费电子产品的 PCB 中藏了一些复活节彩蛋。涵盖的主题将是位图到 SVG 的转换(在 Inkscape 和 Illustrator 中)，KiCad 足迹创建，以及 PCB 制造的局限性和稍微不同的米色 FR4 色调带来的更多技术方面的内容。
