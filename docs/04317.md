# 一个方便的语音命令指南

> 原文:[https://hackaday . com/2016/12/09/a-handy-tutorial-for-voice-command-awesome/](https://hackaday.com/2016/12/09/a-handy-tutorial-for-voice-command-awesomeness/)

当有人找不到如何完成特定任务的指南时，我们 Hackaday 钦佩那些主动为他人编写指南的人。Instructables 用户[ PatrickD126 ]找不到关于如何连接亚马逊的 Alexa 服务和 Echo 到他的 Raspberry Pi 家庭安全系统的文章，所以[他的便捷教程](http://www.instructables.com/id/Control-Raspberry-Pi-GPIO-With-Amazon-Echo-and-Pyt/?ALLSTEPS)应该可以让你快速了解自己的项目。

[PatrickD126]展示了如何将一些软件加载到 Raspberry Pi 上，同时使 Alexa 能够更直接地与 Pi 通信。接下来，就是用你喜欢的语音命令来配置你的亚马逊网络服务账户，以及你想要访问的 GPIO 管脚。搞定了。[PatrickD126]注意到指南中的说明只能产生一个临时的解决方案，但建议了允许您的项目长期运行的替代方案。
