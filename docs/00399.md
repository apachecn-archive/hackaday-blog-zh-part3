# 豚鼠甲烷动力！

> 原文:[https://hackaday.com/2015/10/18/guinea-pig-methane-power/](https://hackaday.com/2015/10/18/guinea-pig-methane-power/)

在地球的另一边，秘鲁的一对夫妇正在利用天竺鼠的粪便来为他们的农场生产甲烷。即使我们努力也无法弥补。

这对夫妇是一对退休的植物生理学教授，他们在自己的别墅 Casa Blanca 经营一个可持续农业项目。这是一个美丽的花园，配有一个实验室用于研究。但最让人好奇的是他们养的上千只豚鼠。他们有一个特殊的棚子，里面有用砖砌成的小隔间。豚鼠吃专门的植物废物，反过来，每月产生惊人的 3 吨粪便。

他们使用大约 200 公斤的粪便为他们自己的生物消化器提供动力，反过来，每天产生 3 立方米的甲烷，用于为他们的别墅提供动力。其余的用作肥料，卖给当地农场。

这是一些真正的农村独创性——这是他们对此的看法:

“我给秘鲁人民的信息是:不要责怪你的贫穷。使用负担得起的技术和过程来改善你的生活质量和幸福感，改变你的贫困，”Ulises 说，一边吃着用 cuy shit 种植的水果制成的冰淇淋，并用 cuy-shit-power 冷冻。

*cuy 是豚鼠的另一个词。
