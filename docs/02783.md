# Hackaday 奖品入口:ESP 瑞士刀

> 原文:[https://hack aday . com/2016/07/04/hack aday-prize-entry-esp-Swiss-knife/](https://hackaday.com/2016/07/04/hackaday-prize-entry-esp-swiss-knife/)

如果你在需要的时候不在身边，最好的设备也帮不了你。知识、经验和厚脸皮可能会在硬件战场的泥泞中帮助你，但它们不能取代万用表、示波器、逻辑分析仪、串行控制台或 WiFi 接入点。[阿卡迪亚实验室]已经接受了将这些功能中的大部分结合到一个设备中的挑战，开发了黑客的瑞士军刀的等价物:[ESP 瑞士军刀](https://hackaday.io/project/11869-esp-swiss-knife)。

[![esp_swiss_knife_enclosure](../Images/342f08202895bb1fd7f76e746b964ca9.png)](https://hackaday.com/wp-content/uploads/2016/06/esp_swiss_knife_enclosure-e1466978226609.jpg) 就像瑞士军刀首先是一把刀，那把办事的瑞士军刀首先是一把 ESP8266。这意味着它已经是任何类型项目的一个很好的平台，而且[阿卡迪亚实验室]通过添加许多项目中常用的一些有用的功能，增强了普通的 ESP-12E 模块。有一个有机发光二极管显示屏，四个按钮，一个温度传感器和一个带充电模块的锂离子电池，以便在旅途中为设备供电。一个通用的“实用插座”可以断开 ESP8266 剩余的 GPIOs 和电源电压，以便连接更多的外设。

随着硬件的安装和运行，[阿卡迪亚实验室]继续开发几个应用程序，以提供使该设备名副其实的功能。其中包括一个基本的示波器、一个基于数字 NTP 的时钟、一个温度计、一个 WiFi 测试仪、一个气象站和一个 3D 打印机状态监视器。更多的应用正在计划中，如天文钟、计时器、DSLR 间隔计等等。一个保护性的 3D 打印外壳也在工作中。[阿卡迪亚实验室]之前已经参加了 2014 年和 2015 年的 Hackaday 奖，我们很高兴看到另一个伟大的构建即将诞生！
