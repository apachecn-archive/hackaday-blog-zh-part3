# 周五黑客聊天:为教育制造电子产品

> 原文:[https://hack aday . com/2017/07/19/Friday-hack-chat-making-electronics-for-education/](https://hackaday.com/2017/07/19/friday-hack-chat-making-electronics-for-education/)

在本周的 Hack Chat on [Hackaday.io](http://hackaday.io) 中，我们将与 AnnMarie Thomas 谈论[为教育制作电子产品](https://hackaday.io/event/25924-making-electronics-for-education)。电子学和教育之间有着巨大的交集，无论你是为第一团队设计机器人，还是围绕 3D 打印物体设计几何课程，电子学都可以教给学生很多东西。

安玛丽·托马斯是圣托马斯大学工程学院和奥普斯商学院的副教授。她是有趣学习实验室的创始人，她和她的学生一起创造了[柔软的电路](http://courseweb.stthomas.edu/apthomas/SquishyCircuits/)。AnnMarie 是《制造制造者:孩子、工具和创新的未来》一书的作者。基本上，如果你在找一个知道如何制作教育产品的人，你不可能做得更好了。

在本周的 Hack Chat 中，我们将讨论如何定义技术和教育的交集。有很多方法来定义一个概念，制造和销售一个教育产品，以及如何为一个产品找到市场。如果你曾经想知道是什么让学生们一头扎进电子学，这是你不得不参加的黑客聊天。

哦，安玛丽也是今年[黑客日奖](http://hackaday.io/prize)的评委。干净利落。
