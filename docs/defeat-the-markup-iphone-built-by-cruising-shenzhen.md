# 击败加价:巡游深圳打造的 Iphone

> 原文：<https://hackaday.com/2017/04/13/defeat-the-markup-iphone-built-by-cruising-shenzhen/>

[Scotty Allen]来自奇怪的地方，刚刚结束了为期三个月的旅程，这显然是我们在一段时间里看到的最有趣的深圳市场项目之一。我们都听说过令人惊奇的故事，关于这些中国市场的多功能性，以及可供您使用的众多零件、工具和专业知识。但是如果你愿意的话，你能走多远，你能完成的最离谱的项目是什么？为了回答这个问题，[Scotty]决定采购和组装自己的 Iphone 6S，直到组件级别！

旅程从收购大肆宣传的一体式铝制背板开始，与苹果的广告相比，这显然在这些中国市场上得不到同等程度的重视。[Scotty 的] vlog 显示大量这样的衬垫被扔在深圳的街道上。买了合适的后，他需要用激光蚀刻出所有相关的美国变体标记。这显然不是一个问题，当蚀刻店坐落在一个石头扔远，相当简单的楼梯甲板下。

接下来是屏幕组件，为了忠实于最初的原因，以数字化仪、LCD、背光的形式单独购买，然后在另一家商店随意组装，比你穿上洁净室工作服更快，你认为这是完成这样一项工作所需要的。

[Scotty]报告称，寻找和组装逻辑板被证明是这一挑战中最困难的部分。尽管如此，他成功地购买了一个无人居住的 PCB 和所有的硅；焊接他们成功地被证明是一个死胡同，而不是现在，他购买了一个二手的逻辑板。我们认为，如果你拥有合适的工具和经验，这应该是绝对可以克服的。

所有其他的螺栓和哨子都是作为单独的组件获得的，最终的结果在很大程度上与真正的产品没有什么区别，但成本只有 300 美元。这并不令人惊讶，因为苹果公司臭名昭著的标记之前已经在各种拆解中被揭露。

查看[Scotty 的]完整视频，其中包括对这些神秘的深圳市场的许多见解。我们当然爱它的每一点。这是买到便宜货的一种方法。
