# 揭开木环的秘密

> 原文：<https://hackaday.com/2016/09/13/lifting-the-secret-of-the-wooden-rings/>

用环氧树脂和木材制作漂亮的东西恰好是[彼得·布朗]的专长领域。最近，他被要求对加拿大制造商[秘密木材](http://mysecretwood.com)的戒指设计进行逆向工程，这是一种由碎木和环氧树脂组成的独特组合，他取得了令人印象深刻的成果。

事实证明，故意以正确的方式劈开木头比不在切口上贴胶带要困难得多。[Peter]试着在他的膝盖上把木板掰成两半，在老虎钳里，他锯木头使它裂得更直，他试着用薄木板和厚木板。最终，他在一台简易压弯机中折断了一块较厚的山毛榉木板，得到了他想要的结果。

 [![secret-wood-ring-breaking](img/e3ba14135d57d3589849ea0873a6f9f7.png "secret-wood-ring-breaking")](https://hackaday.com/2016/09/13/lifting-the-secret-of-the-wooden-rings/secret-wood-ring-breaking/)  [![secret-wood-ring-pouring](img/e809d92520e4dc89ec8eab2451f4eb9b.png "secret-wood-ring-pouring")](https://hackaday.com/2016/09/13/lifting-the-secret-of-the-wooden-rings/secret-wood-ring-pouring/) 

然后，他用环氧树脂填充裂缝，只加入几滴蓝色染料，并故意加入一些气泡。嵌入的雾效果是通过将两种不同的色调连续倒入模具中实现的。有了秘方，彼得可以开始研究神秘材料的环形形状。他钻了一个中心孔，在带锯上切出粗略的形状，并在砂带磨床上完成切割。

 [![secret-wood-ring-cast](img/8f71d34a46cbc7449df3ec38da99a183.png "secret-wood-ring-cast")](https://hackaday.com/2016/09/13/lifting-the-secret-of-the-wooden-rings/secret-wood-ring-cast/)  [![secret-wood-ring-cutting](img/a2c5b5271c590b41c05f45c8ab86ef10.png "secret-wood-ring-cutting")](https://hackaday.com/2016/09/13/lifting-the-secret-of-the-wooden-rings/secret-wood-ring-cutting/)  [![secret-wood-ring-shaping](img/52730798c4a51ab76c1d429c18490f54.png "secret-wood-ring-shaping")](https://hackaday.com/2016/09/13/lifting-the-secret-of-the-wooden-rings/secret-wood-ring-shaping/)
