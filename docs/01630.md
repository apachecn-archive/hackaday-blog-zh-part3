# 使用可折叠 CNC，成为下一次黑客马拉松的英雄

> 原文:[https://hackaday . com/2016/03/01/be-a-hero-at-your next-hackathon-a-foldable-CNC/](https://hackaday.com/2016/03/01/be-a-hero-at-your-next-hackathon-with-a-foldable-cnc/)

使用这款[可折叠 cnc](http://www.thingiverse.com/thing:1214438) 成为下一次黑客马拉松的英雄。当激光切割机的线有四队深时，你会领先。它甚至可以精确到弹出一个快速电路板。不过，[wwwektor]只是想要一台可以从仓库中取出并在需要时展开的 CNC。把它放在厨房的桌子上，剪下一些装饰品，或者把它挂在前门，刻上房子的地址。谁需要注射成型镀铬号码呢？

它基于管状方式，很像我们已经讨论过的其他 [3D 打印 CNC](https://hackaday.com/2016/01/25/sub-300-cnc-if-you-have-a-3d-printer/)。这种设计的便携性使其具有内在的不稳定性。然而，考虑到设计目标，这是合理的。它使用同步带、步进器和滚珠轴承来移动。框架放置在桌子上的方式应该可以处理大多数布线任务，而不需要调整以保持与其放置的表面在一个平面上。只要不需要方边。

休息之后有一段视频。我们喜欢这些独特的数控设计的尝试。我们永远不知道下一步会看到什么新想法。
