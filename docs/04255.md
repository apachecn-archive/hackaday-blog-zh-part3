# 90 多个视频带你从激光傻瓜到激光冠军

> 原文：<https://hackaday.com/2016/11/24/90-videos-take-you-from-laser-chump-to-laser-champ/>

我们很少有人记录我们的副业项目的进展。对于那些这样做的人来说，这些博士有机会成为一本洞察力的书，这是一名宇航员在一个遥远的星球上发现的“任务日志”，可以告诉我们如何驯服一个残酷和充满敌意的世界。随着[rd works 学习实验室系列](https://www.youtube.com/channel/UCqCyShJXqnElPTUnxX0mD5A)的到来，中国激光切割机终于接受了彻底深入的指导，使其进入专业工作秩序。

在两个系列中，总共超过 90 个视频(还在继续！)退休的钣金机械师[Russ]带我们进行一次改装、特征化和充分利用您最近购买的中国激光切割机的盛大旅行。

对激光物理很好奇？只需查看第二部分[。想知道镜头尺寸如何影响功率输出？试一试](https://www.youtube.com/watch?v=zF9nQHlvGkc)[第 39 部分](https://www.youtube.com/watch?v=WbBPps42iHc)。需要一个辅助视频来进行光束对准吗？查看[第 31 部分](https://www.youtube.com/watch?v=-RgbmAWgWxU)。对于这些机器的每一个未记录在案的怪癖，[Russ]都以数据驱动科学家的分析学科来处理每个问题，用他的工具套件来测量和表征每个怪癖，然后设计出解决该怪癖的解决方案。在某些情况下，这些只是微小的螺丝调整。在其他情况下，[Russ]用一个定制的硬件解决方案(通常也是激光切割)向我们展示了他的机械魔法。[Russ]还为我们带来了经验丰富的机械师的技术洞察力，实施了经典的机械师解决方案，如[销台](https://www.youtube.com/watch?v=kMDRzBZTW9c)来生产边缘清晰的零件，不会受到传统蜂窝床上切割零件的散射激光标记的影响。

坚实的建造日志是很难得到的宝石，而[Russ]的中国激光切割机介绍作为一个参考，将经得起时间的考验。没有空间放置激光切割机？对于微加工专家来说，看看[数控加工、模具制造和树脂浇铸的游击指南](http://hackaday.com/2010/12/22/guerilla-guide-to-cnc-machining/)。
