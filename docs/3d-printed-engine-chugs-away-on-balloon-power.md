# 3D 打印引擎依靠气球动力突突前进

> 原文：<https://hackaday.com/2017/03/10/3d-printed-engine-chugs-away-on-balloon-power/>

因此，3D 打印机所有者购买他们的机器时，往往会承诺将自己从商业制成品的束缚中解放出来，并制作各种精彩而有用的东西来改善他们的生活。然后，他们继续用花哨的彩色细丝印刷一系列好运猫、玩具大象、小拖船和许多其他漂亮但最终无用的物品。

也许这是一个不公平的评价，但是如果你有一种偷偷摸摸的感觉，它可能只是描述了你，那么我们可以给你指出一些东西，虽然它仍然没有什么用处，但至少玩起来很有趣。【Gzumwalt】的[单缸空气发动机](http://www.instructables.com/id/Single-Cylinder-Air-Engine/)顾名思义，就是靠压缩空气运转的活塞式发动机。你不需要商店的压缩机，你的肺或者一个充气的气球就足够了。

这是一个足够简单的设计，但它确实包含了两个连杆，其中一个驱动滑阀。所有的文件都可以下载，我们在广告下方放了一个视频，展示了它从气球中发出的美妙声音。它可能不是最有用的引擎，也不会给你带来好运，但在利益方面，它胜过塑料动物园。
