# 吉普牧马人顶灯模块

> 原文：<https://hackaday.com/2017/12/03/jeep-wrangler-dome-light-mod/>

如果你是一辆 Jeep 牧马人的车主，你可能会对内部顶灯感到沮丧。对于那些不了解情况的人来说，在温暖的日子或为了一点乐趣而移除门可能会导致电池没电。这是因为牧马人的灯一直亮着，除非拆除保险丝或灯，或者添加一个定制的关闭开关——代价是麻烦的布线。你可以说这是辆吉普车。[【Tim Nummy】提供了一个解决方案](https://www.youtube.com/watch?v=RfnhaBNAVuA)，只需做最少的修改。

首先，将开关弹出门外，放在一边。作为替代，[蒂姆·Nummy]设法从一辆旧奔驰汽车上抢救出了一个门灯开关。除了与牧马人的 stock 开关相同的瞬时关闭功能外，新款牧马人的按钮可以拉出并锁定在第二个关闭位置。许多机器和设备在其安全联锁中使用这种相同类型的开关作为服务位置。[Tim]不想切断牧马人的线路，以防线路出现问题，所以现在他已经锉下了一些铲形端子，以插入梅赛德斯插头。他还 3D 打印了一个螺母来很好地固定新开关。休息之后看看他的视频吧！

你有它！新开关无需进一步修改即可工作！最重要的是，当门被替换时，开关恢复正常操作。
