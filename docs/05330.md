# 你会想要建造这艘独木舟

> 原文：<https://hackaday.com/2017/03/19/you-will-want-to-build-this-canoe/>

对于大多数制造商来说，木船有一点应该是很重要的，那就是用比薄木片稍多一点的木头制造出线条流畅、适合在海上、湖上或河上航行的船只的工艺和工作水平。造船大师有持续数年的学徒期，并花费整个职业生涯来完善他们的艺术。

[亚当]，也被称为[做东西的家伙]，不是一个造船大师。用他的话说，他只是一个有一些基本木工知识的家伙，用雪松木条造独木舟。但是从他的作品中你不会知道他没有接受过造船训练，你可以这样做，因为他上传了一些美丽的视频。

我们看到骨架的创建产生了船的基本形状，接着是船头和船尾的创建。然后是精心制作的雪松木条来制作船身，两侧各一层玻璃纤维。虽然涂了胶衣，但你不会知道纤维在那里。最后，我们制作了座椅和内部配件，接着是划过湖面的独木舟。

我们中很少有人会做独木舟。但如果有的话，我们会希望它像这样。
